# v2026.05.20-5 GitHub Pages公開開始

## 日本語 🇯🇵

### 概要

この release は、FinWiki を GitHub Pages で公開し、`finwiki.zksc.io` を custom domain として接続した snapshot です。

### 対象範囲

- GitHub Pages settings。
- Cloudflare DNS `finwiki.zksc.io` CNAME。
- GitHub Pages custom domain `CNAME` file。
- Pages homepage `index.html`。
- README / CHANGELOG の Pages 運用記録。

### 主要変更

- GitHub Pages source を `main` branch `/ (root)` に設定。
- Custom domain を `finwiki.zksc.io` に設定し、GitHub 側の DNS check successful を確認。
- GitHub UI により remote `main` に `CNAME` file が作成されたため、local branch を fast-forward。
- Pages build が成功しても homepage file 不在で 404 になる問題を確認。
- `index.html` を追加し、Pages の first page として README / INDEX / CHANGELOG への導線を提供。
- README に Pages 公開ドメイン、DNS、source、custom domain、HTTPS 注意点を追加。

### 検証

- DNS: `finwiki.zksc.io` は `jasonhnd.github.io` に解決。
- GitHub Pages API: `status: built`, `cname: finwiki.zksc.io`, `source: main /` を確認。
- GitHub Pages Actions: `pages-build-deployment` の build / deploy 成功を確認。
- `index.html` 追加前: `finwiki.zksc.io` が 404 を返すことを確認し、homepage file 不在と判定。
- `index.html` 追加後: push 後に再 build と HTTP response を確認する。

### 既知の注意点

- HTTPS enforce は、GitHub が custom domain certificate を発行するまで unavailable になる場合があります。
- 現時点の Pages は公開入口ページです。Markdown 全文を静的 HTML に変換する dedicated site build はまだ導入していません。

### 次の作業

- HTTPS certificate 発行後に GitHub Pages の Enforce HTTPS を有効化する。
- 必要に応じて Markdown wiki 全体を静的 HTML 化する build pipeline を追加する。
- Pages homepage の情報設計を、README / INDEX の更新に合わせて継続同期する。

## 中文 🇨🇳

### 概要

本 release 是 FinWiki 启动 GitHub Pages，并将 `finwiki.zksc.io` 作为 custom domain 接入的公开快照。

### 范围

- GitHub Pages settings。
- Cloudflare DNS `finwiki.zksc.io` CNAME。
- GitHub Pages custom domain `CNAME` 文件。
- Pages homepage `index.html`。
- README / CHANGELOG 的 Pages 运维记录。

### 主要变更

- 将 GitHub Pages source 设置为 `main` branch `/ (root)`。
- 将 custom domain 设置为 `finwiki.zksc.io`，并确认 GitHub 侧 DNS check successful。
- GitHub UI 自动在 remote `main` 创建 `CNAME` 文件，因此本地分支已 fast-forward。
- 确认 Pages build 成功后仍然 404 的原因是缺少 homepage file。
- 新增 `index.html`，作为 Pages 首页，并提供 README / INDEX / CHANGELOG 导航。
- README 补充 Pages 公开域名、DNS、source、custom domain 和 HTTPS 注意事项。

### 验证

- DNS：`finwiki.zksc.io` 已解析到 `jasonhnd.github.io`。
- GitHub Pages API：确认 `status: built`, `cname: finwiki.zksc.io`, `source: main /`。
- GitHub Pages Actions：确认 `pages-build-deployment` 的 build / deploy 成功。
- 新增 `index.html` 前：确认 `finwiki.zksc.io` 返回 404，并判断原因是缺少首页文件。
- 新增 `index.html` 后：push 后继续确认重新 build 和 HTTP response。

### 已知注意事项

- HTTPS enforce 需要等 GitHub 为 custom domain 签发证书后才能启用。
- 当前 Pages 是公开入口页，还没有导入把整个 Markdown wiki 转为静态 HTML 的专用构建流程。

### 下一步

- GitHub certificate 签发完成后，启用 GitHub Pages Enforce HTTPS。
- 如有需要，增加将 Markdown wiki 全量静态 HTML 化的 build pipeline。
- 随 README / INDEX 的更新持续同步 Pages homepage 的信息架构。
