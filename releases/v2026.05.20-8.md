# v2026.05.20-8 GitHub Pages ポータルステータスカードの三言語化

## 日本語 🇯🇵

### 概要

この release は、GitHub Pages ポータルページ（`index.html`）の下部にあるステータスカード（キー指標）を更新し、以前の「二言語維持・中国語維持」の表記から、今回制定された「日本語・英語・中国語」の三言語維持および並び順（JP-EN-ZH）に基づいた表記へと完全に同期・刷新したスナップショットです。

### 対象範囲

- Pages portal homepage `index.html`。
- ステータスカードの三言語記述同期。
- CHANGELOG / リリースノート。

### 主要変更

- **ポータルステータスカードの三言語化修正**: 
  - 日本語パネル (`#sec-jp`): 「中国語維持・二言語でのドキュメント運用体制」を「三言語維持・日・英・中のドキュメント運用体制」へ変更。
  - 英語パネル (`#sec-en`): 「Chinese Maintained・Trilingual & bilingual synchronization」を「Trilingual Sync・JP, EN, and ZH fully synchronized」へ変更。
  - 中国語パネル (`#sec-zh`): 「中文同步维护・多语言常态化与高度一致性运作」を「三语同步维护・日、英、中三语常态化与高度一致性」へ変更。

### 検証

- **`git diff` & `git status`**: 変更を確認し、不正な末尾空白や無関係なファイルがないことを確認。
- **実機表示確認**: 各言語タブ（日本語・English・中文）で、変更後のカード項目が歪みなく美しくレスポンシブに表示されることを確認。
