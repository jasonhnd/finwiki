# v2026.05.20-6 Portalページ英語追加と多言語分離化

## 日本語 🇯🇵

### 概要

この release は、GitHub Pages のポータルページ（`index.html`）を更新し、日本語・英語・中国語を完全に分離した上で「日本語、英語、中国語」の順に閲覧できるモダンなセグメントUI（タブ型レイアウト）を構築したスナップショットです。

### 対象範囲

- Pages homepage `index.html`。
- ポータルページ多言語化および英語対応。
- 検索性維持のためのブラウザ検索連動ロジック。
- CHANGELOG / リリースノート。

### 主要変更

- **多言語分離（日本語・英語・中国語）**: 従来ポータル内で一部混在していた日本語と中国語を完全に分離。さらに英語ポータルも新規追加し、「日本語、英語、中国語」の順に並ぶタブ切り替えUIに刷新。
- **iOS風セグメントタブUI**: 画面最上部に洗練された言語選択スイッチ（タブコントロール）を設置。`localStorage` と URL hash に対応し、選択言語の永続化とダイレクトリンクを実現。
- **ブラウザ検索連動 (Cmd+F 連動)**: 非表示の言語セクションに `hidden="until-found"` 属性を使用し、ブラウザ内検索でマッチした際に `beforematch` イベントをフックして自動的にタブ表示を切り替える高度なUXを実装。
- **优雅な降格（No-JS対応）**: Javascriptが無効な環境では、`<noscript>` 経由で全3言語を「日本語、英語、中国語」の順に縦一列に並べて標準表示するフォールバックを組み込み。
- **タイポグラフィ強化**: 見出しに極めてモダンな幾何学フォント「Outfit」を、本文に「Inter」をGoogle Fonts経由でインポートしてビジュアルの高級感を大幅に引き上げ。

### 検証

- **`git diff --check`**: 変更ファイルにおける末尾空白エラーがないことを確認。
- **言語切り替え＆検索**: Chrome / Edge で非アクティブ言語のテキスト（例:「日本金融机构」）を Cmd+F で検索した際、瞬時に該当言語のタブがアクティブになりテキストが強調されることを確認。
- **No-JS 動作**: JS を無効化したブラウザ環境で全3言語が指定の順序で問題なく読めることを確認。

### 既知の注意点

- `hidden="until-found"` は一部の古いブラウザでは単なる `hidden` (display: none相当) として動作するため、その場合はブラウザ内検索の自動タブ切り替えは動作せず、通常のクリック切り替えのみとなります（問題なく优雅なフォールバックが機能します）。

- Pages のデプロイ完了を確認し、実機ブラウザでの最終確認を行う。

## 中文 🇨🇳

### 概要

本 release 更新了 GitHub Pages 门户页面（`index.html`），将中文、英文与日文内容进行了完全的区块分离，并构建了可以按“日文、英文、中文”顺序切换浏览的现代分段标签页（Tab）系统。

### 范围

- Pages 门户首页 `index.html`。
- 门户多语言分离及新增英文翻译。
- 结合浏览器原生内搜索（Cmd+F）的标签页自动连动逻辑。
- CHANGELOG / Release Notes。

### 主要变更

- **多语言彻底分离（日文、英文、中文）**：将以往门户主页中部分混杂的日中文字彻底分离。同时新增了完整的英文门户段落，并按“日文、英文、中文”的顺序进行了重新排列。
- **iOS 风格分段式标签控制**：在页面顶部正中设计了精致的语言选择器，支持 `localStorage` 记忆选择和 URL hash 锚定，能够实现首选语言的持久化及直链跳转。
- **浏览器搜索连动 (Cmd+F 连动)**：对隐藏的语言区域使用了原生的 `hidden="until-found"` 属性。在支持的浏览器下，当用户在当前隐藏区域搜索文本时，自动触发 `beforematch` 事件并切换激活标签页，提供无感的高端 UX 体验。
- **优雅降级（支持 No-JS）**：在禁用 Javascript 的浏览器环境中，通过 `<noscript>` 样式自动显示全部内容，按“日文、英文、中文”的顺序纵向平铺，确保核心可读性与无障碍保障。
- **高级视觉与排版**：标题字体引入极具现代感与高端几何风格的 Google Fonts「Outfit」，正文引入「Inter」字族，优化了卡片微动效及阴影，视觉表现力显著上升。

### 验证

- **`git diff --check`**：已确认修改的文件中没有尾随空格等格式问题。
- **语言切换与搜索**：在 Chrome / Edge 下测试当“日文”激活时，使用 Cmd+F 搜索“日本金融机构”或“English”，浏览器能完美检测并瞬间切换至对应中文或英文标签页并准确定位。
- **禁用 JS 表现**：测试在禁用 JS 后，三语页面可以按指定顺序无缝降级平铺展现。

### 已知注意事项

- 部分老旧浏览器不支持 `hidden="until-found"` 属性，它将被降级解析为标准的 `hidden`。在此类环境下将无法支持跨隐藏标签的 Cmd+F 自动切换，但仍可通过手动点击标签页正常阅读（实现了完备的优雅降级）。
