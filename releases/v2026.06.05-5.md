## 日本語

### 公開範囲

- roadmap P2 = JapanFG 拆分域の entity 内容深化。v2026.06.05-3（第 1 批 58 件）に続き、本 release で**残り 50 件**の <250 語「登记簿存根」を深化 → **P2 thin 存根の深化は完了**（合計 108 件）。
- 対象 11 域: `life-insurers` 8 / `securities-firms` 8 / `asset-managers` 7 / `payment-firms` 7 / `trust-banks` 6 / `megabanks` 5 / `card-issuers` 5 + 小域 4（`non-life-insurers` anicom-hd / `leasing-firms` mitsubishi-hc-capital / `consumer-finance` smbc-consumer-finance / `financial-regulators` rcc）。
- 三語 README/CHANGELOG/`releases/v2026.06.05-5.md`、roadmap/backlog 更新、discovery 再生成。

### 主要変更

- **存根 → 実体ページ**: 各社に License/group boundary（保険業法 / 金融商品取引法 / 資金決済法 / 割賦販売法 / 銀行法 / 兼営法 等の免許種別、FSA + 各業界団体）、institution-type の Operating model（生保＝引受・責任準備金・長期資産運用 / 証券＝第一種金商・店頭FX / 投資運用業 / 資金移動業・前払式 / 信託兼営 / 都市銀行 等）、group/parent 文脈、3+ の core-body wikilink を追加。
- **無捏造を厳守**: 親会社 / 免許種別 / 設立 / 本店 / 業務範囲は **web 照合できたものだけ**記載。照合不能な数値（AUM / 会員数 / 取扱高 / 財務）は **shape として hedge または削除**。照合により**既存 stub の誤りを訂正**した例:
  - `securities-firms/paypay-securities` 親会社 → PayPay株式会社（2025）、`mufg-esmart-securities` → 三菱UFJ銀行 100%（2025、旧「三菱UFJ証券HD」は stale）。
  - `payment-firms/rakuten-edy` を 資金移動業 → 正しくは**第三者型前払式支払手段**へ訂正。
  - `trust-banks/orix-bank` を「信託銀行」→ 正しくは**信託兼営の普通銀行**（Daiwa Next Bank への売却 2026-04 公表）へ訂正・hedge。
  - `asset-managers/nikko-asset-management` 親 → SMFG ではなく **Sumitomo Mitsui Trust 100%**（Amova 改称 2025-09）；`smd-am` 株主構成を実照合値へ訂正。
- **split 旧路由の修復**: 50 件すべて開頭 Wiki-route が旧 `[[JapanFG/INDEX]]` を指していた（拆分の取り残し）→ 各域 INDEX へ修正。regulator link も存在しない `[[JapanFG/fsa]]` から実在の `[[financial-regulators/fsa]]` へ統一。
- 並列 subagent（域別 file-scope 隔離）8 体・2 batch。`canonical_anchor` 不要（全て単一領域）。
