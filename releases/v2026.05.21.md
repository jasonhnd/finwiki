# 日本金融機関拡張と registry-index 同期 2026-05-21

## 日本語

### 公開範囲

この release は、2026-05-21 の日本金融機関・政策金融・協同組織金融・決済・証券 registry 拡張をまとめる公開知識庫 snapshot です。

### 主要変更

- `JapanFG/` を 379 entries に拡張し、foreign-bank branch, trust-company registry, insurers, securities / asset managers, payments / PSP / remittance, and missing-institution backlog routes を整理。
- `banking/` を 17 entries に拡張し、JA Bank system, JF Marine Bank system, shinkin / shinkumi / rokin registry indexes, cooperative-banking boundary を同期。
- `policy-finance/` を 12 entries に拡張し、housing finance, SME credit guarantee, agriculture / fisheries guarantee-insurance を ordinary bank pages と分けて整理。
- `payments/` に third-party prepaid payment instrument issuer registry route を追加し、FSA `daisan.xlsx` 803-row population は official workbook を source of truth として保持。
- `securities/` に FIEA operator registry route を追加し、FSA `kinyushohin.xlsx` 1,945-row population は category / jurisdiction counts と material routes に要約。
- 20 legacy entries の `sources` / `tags` hygiene を補完し、public-source only ルールに合わない source labels を公開ソース記述に置換。
- `tools/wiki_link_audit.py` と `wiki-link-improvement-plan.md` を追加し、本文内 `[[wikilink]]` が3件未満の entry と route / peer / system advisory issue を 0 にした。

- Representative phone numbers are intentionally omitted from registry wiki tables.
