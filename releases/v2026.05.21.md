# 日本金融機関拡張と registry-index 同期 2026-05-21/22

## 日本語

### 公開範囲

この release は、2026-05-21〜22 の日本金融機関・政策金融・協同組織金融・決済・証券 registry 拡張をまとめる公開知識庫 snapshot です。

### 主要変更

- `JapanFG/` を 411 entries に拡張し、foreign-bank branch, foreign-bank agency, adjacent foreign-group license routes, insurance license / solvency route, Euroclear / Clearstream ICSD exceptions, India / Thailand / ANZ corridor foreign-bank pages, Australia / Taiwan / Indonesia / Philippines / Brazil corridor anchors, specialty trust-bank / CCI current-name anchors, pet / outdoor / telco-linked specialty insurers, trust-company registry, insurers, securities / asset managers, securities license stack, payments / PSP / remittance, consumer finance / auto finance, and missing-institution backlog routes を整理。
- `banking/` を 23 entries に拡張し、JA Bank system, JF Marine Bank system, shinkin / shinkumi / rokin registry indexes, net-bank competition map, BaaS operating models, trust-bank / custody infrastructure, trust-bank custody operating comparison, regional bank API partnerships, and cooperative-banking boundary を同期。
- `policy-finance/` を 12 entries に拡張し、housing finance, SME credit guarantee, agriculture / fisheries guarantee-insurance を ordinary bank pages と分けて整理。
- `payments/` を 15 entries に拡張し、third-party prepaid payment instrument issuer registry route, funds-transfer service provider registry route, credit / card operator registry route, bank API / electronic payment agency route, bank API incident / fraud-control route, card issuer / acquirer / processor split, card security / authentication controls, BNPL / credit-purchase boundary, account-to-account payment route, account-direct merchant acquiring route, code-payment competitive map, and PSP merchant-settlement-risk control page を追加。
- `securities/` を 18 entries に拡張し、FIEA operator registry route に加えて online brokerage competition, best execution / SOR / PTS, PTS liquidity data guide, margin trading / securities finance route, short-selling / stock-loan controls, stock lending market route, prime brokerage / institutional financing, underwriting market structure, and market-infrastructure map を追加。
- `insurance/` を 11 entries に拡張し、life-insurance big four, non-life big three, earthquake insurance public-private scheme, and insurance agency / brokerage operating-model pages を追加。
- `finance/` を 9 entries に拡張し、Japan tender offer process / TDnet route, MBO / squeeze-out process, post-May-1-2026 large-shareholding disclosure route, acquisition finance, and activist investor / engagement playbook を追加。
- 20 legacy entries の `sources` / `tags` hygiene を補完し、public-source only ルールに合わない source labels を公開ソース記述に置換。
- `tools/wiki_link_audit.py` と `wiki-link-improvement-plan.md` を追加し、本文内 `[[wikilink]]` が3件未満の entry と route / peer / system advisory issue を 0 にした。
