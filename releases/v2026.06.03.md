# 日本語 i18n 翻訳層の全量追加

## 日本語

### 概要

FinWiki の ja i18n 翻訳層を全 1,380 entry に追加しました。これにより `/ja/` は source fallback の日英混在本文ではなく、`site/src/content/i18n/ja/` の機械翻訳 layer を優先して表示します。

### 公開範囲

- `site/src/content/i18n/ja/` の 1,380 件の machine translation entry
- `site/scripts/prep-parallel.mjs` と `site/scripts/commit-translate.mjs` の `--langs` 対応
- README / CHANGELOG / release notes
- AI discovery surface と root `index.html` の snapshot 更新

### 主要変更

- ja 翻訳 layer を全 entry へ追加し、`source_hash` による同期判定、`status: machine`、`fidelity: ok` を付与しました。
- 翻訳準備と収集 scripts が `--langs ja` を受け取れるようになり、既存の zh/en default は維持しました。
- 完了済み worker は閉じ、最終 batch 後に `prep-parallel --langs ja` が 0 jobs を返す状態まで確認しました。

### 検証結果

- `site/src/content/i18n/ja/`: 1,380 files
- `fidelity: needs_review`: 0 files
- `bun scripts/prep-parallel.mjs --workers 10 --size 15 --langs ja`: 0 jobs
- `bun run build`: PASS, 4,219 pages
- `bun tools/check_duplicate_html_ids.ts site/dist`: checked=4219 / duplicate_ids=0
- `bun tools/release.ts --write`: link audit entries=1411 / issues=0; AI discovery regenerated
- `bun tools/release.ts --check --strict`: PASS
- UTF-8 rendered ja samples show Japanese h1 / h2 text and the 「機械翻訳」 badge
- `git diff --check`: PASS

### 既知の注意点

- ja entry には「機械翻訳」badge が表示されます。
- Mask された wikilink の表示文字は、source 側の label をそのまま保持する場合があります。

### 次の作業

- `origin/main` へ push 後、GitHub Actions、公開 `/ja/`、代表 entry、AI discovery surface を確認します。
