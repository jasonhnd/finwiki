# Bun工具体系への全面移行

## 日本語

### 概要

2026-06-02 JST の FinWiki root tooling migration release です。Vercel shadow deployment と Pagefind 静的検索を追加した後に残っていた Python / Node `.mjs` の root tooling entry points を、Bun + TypeScript に統一しました。

### 公開範囲

- Root release / discovery / audit / publish tooling
- GitHub Pages workflow and Vercel build command
- Pre-push release gate
- README / AGENTS operating commands
- Release-generated AI discovery surfaces and static JSON API

### 主要変更

- `tools/generate_ai_discovery.ts` と `lib/markdown_helpers.ts` を追加し、`robots.txt`、`sitemap.xml`、`llms.txt`、`llms-full.txt`、`ai-index.json`、`api/entries/*.json` を Bun で生成します。
- `tools/release.ts` を Bun release gate とし、wikilink audit、README / index count sync、JSON / LF / duplicate HTML id verification を実行します。
- `tools/check_duplicate_html_ids.ts` は Bun 内蔵 `HTMLRewriter` を使い、追加 dependency なしで rendered HTML を検査します。
- `tools/vercel_build.ts` と `tools/assemble_static_publish.ts` により、Vercel と GitHub Pages が同じ Bun publish assembly を使います。
- Root `package.json`、`vercel.json`、GitHub Actions、pre-push hook、AGENTS / README commands を Bun に更新しました。
- 旧 root `tools/*.py` と root `.mjs` build tools を削除しました。

### 検証結果

- `bun tools/generate_ai_discovery.ts`: markdown_files=1466 / public_pages=1465 / sitemap_urls=1466 / domains=23 / link_audited_entries=1411 / api_entries=1404
- `bun tools/release.ts --check --strict`: PASS
- `bun tools/check_duplicate_html_ids.ts site/dist`: checked=4147 / duplicate_id_pages=0 / duplicate_ids=0
- `bun tools/vercel_build.ts`: PASS. Astro built 4147 pages; Pagefind indexed 4147 pages / 3 languages / 136151 words; static assembly wrote astro_files=8697 / root_surface_files=2838.
- Active operational docs / workflow / hooks / tools no longer reference `python tools`, `python3 tools`, `node tools`, or root `tools/*.py` / `tools/*.mjs`.

### 既知の注意点

- Historical CHANGELOG records still mention older Python / Node commands because those entries document what was run at that time.
- Astro-side `.mjs` config and plugins remain site framework files executed through Bun; this release targets root release / discovery / publish tooling.

### 次の作業

- Push to `origin/main`, verify remote HEAD, publish this GitHub Release, and verify public homepage / AI surfaces / search assets after deployment.
