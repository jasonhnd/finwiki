# CI 警告整理と HTML ID 検証ゲート追加

## 日本語

### 概要

2026-06-02 JST の FinWiki CI 品質リリースです。前回 release の残タスクだった Astro duplicate id warning と GitHub Actions Node.js 20 deprecation annotation を整理しました。

### 公開範囲

- 対象: `.github/workflows/deploy.yml`, `tools/check_duplicate_html_ids.py`, `tools/release.py`
- 同期 surface: `README.md`, `CHANGELOG.md`, `releases/v2026.06.02.md`, discovery / footer snapshot
- 公開 wiki 本文と `site/src/content/i18n/{zh,en}` は変更なし

### 主要変更

- `actions/checkout@v6`, `actions/upload-pages-artifact@v5`, `actions/deploy-pages@v5` へ更新し、GitHub Actions の Node.js 24 runtime 世代へ移行しました。
- `tools/check_duplicate_html_ids.py` を追加し、rendered HTML の page-local duplicate `id` を検出できるようにしました。
- GitHub Pages workflow で Astro build 直後に duplicate HTML id gate を実行します。
- `tools/release.py --check` の verification に、既存 `site/dist` がある場合の duplicate HTML id check を追加しました。

### 検証結果

- `bun run build`: 4147 pages built
- `python tools/check_duplicate_html_ids.py site/dist`: checked=4147 / duplicate_id_pages=0 / duplicate_ids=0
- `python tools/release.py --check --strict`: PASS, counts in sync, JSON valid, LF endings OK
- `git diff --check`: whitespace error なし
- GitHub Actions deployment と live site は push 後に確認します

### 既知の注意点

- この release は CI / tooling の品質改善です。wiki 本文、翻訳本文、domain counts は意図的に変更していません。
- `oven-sh/setup-bun@v2` は現行 v2 action が Node.js 24 runtime を使うため維持しています。

### 次の作業

- push 後に GitHub Actions run が成功し、Node.js 20 deprecation annotation が消えたことを確認します。
- `https://finwiki.zksc.io/` と `llms.txt` の反映を確認します。
