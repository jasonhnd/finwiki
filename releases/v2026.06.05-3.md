## 日本語

- 三語 README/CHANGELOG/`releases/v2026.06.05-3.md`、roadmap/backlog 更新、discovery 再生成。

### 主要変更

- **存根 → 実体ページ**: 各銀行に License/group boundary（銀行法/信用金庫法の免許種別、FSA + 地域財務局、地銀協/第二地銀協/信金中金 等の所属）、Operating model（institution-type の業務モデル）、group/system 文脈、4+ の core-body wikilink を追加。
- **無捏造を厳守**: 具体的数値（預金/資産/設立年/証券コード/支店数/親会社）は **web 照合できたものだけ**記載（公式 IR/開示・FSA 登記簿・各協会・公式プレス）。照合不能な記述は agent が**削除**（例: 一部の設立年・業務改善命令の断定を撤回）。照合した例: shonai+hokuto→フィデア銀行 2027-01、shizuoka-bank 1943-03 設立・コード 8355、daiwa-next-bank の Orix Bank 買収 2026-04 等。
- **split 旧路由の修復**: 58 件すべて開頭 Wiki-route が旧 `[[JapanFG/INDEX]]` を指していた（拆分の取り残し）→ `[[regional-banks/INDEX]]` 等へ修正。58 件に `[[JapanFG/INDEX]]` route は **0 件**残存。
- 並列 subagent（domain/range 別 file-scope 隔離）。本 batch は 3 agent（regional A/B/C）+ cooperative（前 batch）。残り ~51 件（11 域）は backlog（P2 継続）。
