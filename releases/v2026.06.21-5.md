### 既知の注意点

- Factual consistency / provenance completeness の監査は baseline review 前の report-only tool です。Default では release gate ではありません。

- Issue body の branch / PR stop protocol は個別 issue 向けです。この branch では reviewer 向けに複数 open issue の実装をまとめています。

### 次の作業

- Reviewer が branch / PR を確認した後、issue closeout evidence を GitHub 側で紐付けます。
- Report-only audit の baseline を review し、false-positive control を調整してから gate 化の是非を判断します。
- Asset-management shortlist から downstream content issue を reviewer 判断で切り出します。
