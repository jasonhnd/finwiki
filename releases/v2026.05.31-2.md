# 公開ルート・カバーホームページ導入

## 日本語

### 概要

2026-05-31 JST の公開サイト入口更新です。`finwiki.zksc.io/` を単純な日本語ページへの redirect から、FinWiki の三語入口、主要領域、現在の翻訳状況、AI discovery surface を一画面で確認できるカバーホームページへ変更しました。

### 公開範囲

- 公開サイトの `/` ルート
- 三語入口: `/ja/`, `/en/`, `/zh/`
- AI discovery surface: `/llms.txt`, `/llms-full.txt`, `/ai-index.json`, `/sitemap.xml`
- 記録更新: `README.md`, `CHANGELOG.md`, this release note

### 主要変更

- `site/src/pages/index.astro` を redirect から standalone cover page に変更。
- `site/public/og-image.png` を追加し、Astro build と公開ルートで OG image を安定配信。
- カバーページに current site statistics、featured domains、language entrances、machine-readable links を追加。
- README の公開入口説明を、Astro cover homepage と root `index.html` の repository discovery 役割に合わせて更新。

### 検証結果

- `bun run build`: pass, 4147 pages built.
- `/` の generated HTML で 1,380 human entries, 23 domains, zh 483 / en 483 translations を確認。
- desktop / mobile viewport で横スクロールなしを確認。
- `python tools/release.py --check --strict`: pass.

### 既知の注意点

- GitHub Pages の実配信反映は `origin/main` push 後の Actions deployment 完了に依存します。
- Human page count は Astro content entries の 1,380、repository-wide link-audited count は release tooling の 1,411 で、用途の違う別口径です。

### 次の作業

- GitHub Pages deployment 後に `https://finwiki.zksc.io/` の root 表示を確認。
- 引き続き未完了 i18n translation queue を GPT/Codex worker で処理。

## English

### Overview

This 2026-05-31 JST release updates the public site entrance. `finwiki.zksc.io/` now serves a real FinWiki cover homepage instead of redirecting directly to the Japanese surface.

### Release Scope

- Public `/` root
- Trilingual entrances: `/ja/`, `/en/`, `/zh/`
- AI discovery surface: `/llms.txt`, `/llms-full.txt`, `/ai-index.json`, `/sitemap.xml`
- Records: `README.md`, `CHANGELOG.md`, and this release note

### Major Changes

- Replaced the Astro root redirect with a standalone cover page.
- Added `site/public/og-image.png` so the root page has a stable public OG image.
- Added live site statistics, featured domains, language entrances, and machine-readable links.
- Updated README entrance wording to distinguish the Astro cover homepage from root `index.html`, which remains a repository discovery content map.

### Validation Results

- `bun run build`: passed, 4147 pages built.
