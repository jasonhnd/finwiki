# GPT i18n 翻訳バッチ継続・1143/1143 進捗

## 日本語

### 概要

2026-06-01 JST の GPT/Codex i18n 継続バッチです。zh/en 1013 の release 後、次の 130 source entries を GPT/Codex worker で処理し、累計進捗を zh 1143 / en 1143 に進めました。

### 公開範囲

- 対象: `site/src/content/i18n/{zh,en}/`
- 作業 batch: 130 source entries, 260 language outputs
- 記録更新: `README.md`, `CHANGELOG.md`, this release note
- AI discovery / counts: `tools/release.py --write` により同期

### 主要変更

- `bun scripts/prep-parallel.mjs --workers 10 --size 13` で 130 jobs を準備しました。
- 10 本の GPT/Codex workers で `.cache/jobs/w0..w9/` を処理しました。
- 未生成の 3 entries は recovery workers / single-file recovery に分割して補完しました。
- `japanfg/ssnb.md` の英訳で placeholder order がずれたため、原文順に合わせて修正しました。
- 完了済み worker / recovery agent は archive し、`.cache/jobs` を次 batch に使える状態に戻しました。

### 検証結果

- `bun scripts/commit-translate.mjs`: ok=260, needs_review=0, missing=0.

- Windows では完了済み agent が worker directory を保持すると `.cache/jobs` cleanup が EBUSY になることがあります。
- 今後も batch 終了ごとに used worker agent を archive します。

### 次の作業

- `prep-parallel` が 0 jobs を返すまで同じ verify-first flow を繰り返します。
