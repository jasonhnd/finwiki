# ホームページ可読性デザイン改善 2026-05-23

## 日本語

### 概要

FinWiki の公開 homepage を、長文中心の説明ページから、数値カード、短い concept cards、展開式 domain cards を持つ読みやすい入口へ調整しました。既存の公開情報ベースの wiki content は変更せず、トップページの初期表示密度を下げています。

### 対象範囲

- `index.html`
- `README.md`
- `CHANGELOG.md`
- `releases/v2026.05.23-3.md`

### 主要変更

- `FinWiki とは` / `What FinWiki Is` / `FinWiki 是什么` の長い introduction を短い lede と 2 つの concept cards に分解。
- 読み方と対象読者の説明を collapsed details に移し、必要な時だけ展開して読める構造に変更。
- 9 つの主要領域を accordion-style domain cards に変更し、初期表示では領域名を優先して見せる設計に整理。
- `concept-grid`, `concept-card`, `read-more`, domain `details` styles を追加し、desktop は domain grid 3 columns、mobile は single column に調整。
- README の homepage 説明を、metric cards / concept cards / expandable domain cards を持つ現在の内容地図として更新。

### 数値スナップショット

- Markdown files: 909
- Topical domains: 20
- Link-audited entries: 881
- Unresolved link issues: 0
- Text volume: 約363.1万字 / 3,631,366 non-space UTF-8 characters
- Word-like tokens: 577,676

### 検証結果

- `python3 tools/wiki_link_audit.py --report wiki-link-improvement-plan.md --fail-on-issues` pass。
- `git diff --check` pass。
- Local preview で desktop / 390px mobile ともに日本語・英語・中文 tabs の horizontal overflow がないことを確認。
- Desktop は concept grid 2 columns / domain grid 3 columns、390px mobile は concept / metric / domain grids 1 column。

### 既知の注意点

- 新しい wiki 本文ページは追加していません。本 release は homepage と公開入口説明の可読性改善です。
- GitHub Pages の反映は `origin/main` push と Pages build 後に確認します。

### 次の作業

- Largest domains ranking、source-category breakdown、official-source ratio、link graph density などを追加すると、数字で見る FinWiki をさらに拡張できます。

## English

### Overview

This release adjusts the public FinWiki homepage from a text-heavy introduction into a more readable entrance with metric cards, short concept cards, and expandable domain cards. Existing public-source wiki content is unchanged; the update reduces initial reading density on the top page.

### Scope

- `index.html`
- `README.md`
- `CHANGELOG.md`
- `releases/v2026.05.23-3.md`

### Major Changes

- Split the long introductions in `FinWiki とは` / `What FinWiki Is` / `FinWiki 是什么` into a short lede plus two concept cards.
- Moved reading guidance and audience notes into collapsed details blocks so readers can expand them only when needed.
- Converted the nine main domains into accordion-style domain cards that prioritize the domain names in the initial view.
- Added `concept-grid`, `concept-card`, `read-more`, and domain `details` styles; desktop uses a three-column domain grid and mobile uses one column.
- Updated README to describe the homepage as a current content map with metric cards, concept cards, and expandable domain cards.

### Numeric Snapshot

- Markdown files: 909
- Topical domains: 20
- Link-audited entries: 881
- Unresolved link issues: 0
- Text volume: ~3.63M chars / 3,631,366 non-space UTF-8 characters
- Word-like tokens: 577,676

### Validation

- `python3 tools/wiki_link_audit.py --report wiki-link-improvement-plan.md --fail-on-issues` passed.
- `git diff --check` passed.
- Local preview confirmed no horizontal overflow in the Japanese, English, and Chinese tabs on desktop and 390px mobile.
- Desktop uses a two-column concept grid and three-column domain grid; 390px mobile uses one-column concept, metric, and domain grids.

### Known Notes

- No new wiki body pages were added. This release focuses on homepage and public-entrance readability.
- GitHub Pages reflection is verified after push to `origin/main` and Pages build completion.

### Next Work

- Largest-domain ranking, source-category breakdown, official-source ratio, and link graph density would make the numeric FinWiki view richer.
