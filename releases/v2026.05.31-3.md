# GPT i18n 翻訳バッチ継続・610/610 進捗

## 日本語

### 概要

2026-05-31 JST の GPT/Codex i18n 継続バッチです。前回の zh/en 483 件から、placeholder verify を通過した訳文だけを公開 i18n surface に追加し、累計を zh 610 / en 610 へ進めました。

### 公開範囲

- 対象: `site/src/content/i18n/{zh,en}/`
- 作業 batch: 130 source entries, 260 language outputs
- 記録更新: `README.md`, `CHANGELOG.md`, this release note
- AI discovery / counts: `tools/release.py --write` により同期

### 主要変更

- `bun scripts/prep-parallel.mjs --workers 10 --size 13` で 10 worker 分の job を準備。
- GPT/Codex worker 10 本で `.cache/jobs/w0..w9/` を分担。
- `bun scripts/commit-translate.mjs` で placeholder verify + unmask を実行。
- 239 language outputs が verify を通過し、21 language outputs は `needs_review` として公開訳文から除外。
- 完了 worker thread はすべて archive 済み。

### 検証結果

- `bun scripts/commit-translate.mjs`: ok=239, needs_review=21, missing=0.
- Cumulative i18n count: zh 610 / en 610.
- `bun run build`: passed, 4147 pages built.
- `python tools/release.py --check --strict`: passed.

### 既知の注意点

- `needs_review` の 21 language outputs は placeholder mismatch のため公開 i18n に含めていません。
- 残り翻訳と `needs_review` は次回の incremental batch で再取得します。

### 次の作業

- 次の GPT/Codex batch で未翻訳分を継続。
- `needs_review` が多い entry は最後にまとめて prompt / protect rule を見直します。
