# human site カラートークンの冷灰テーマ化

## 日本語

### 概要

human site の CSS color token を、暖色紙面・赭色系から、冷灰の紙面、青緑の primary action、低彩度の補助色へ変更しました。金融 reference としてより中立で実務的な見え方に寄せます。

### 公開範囲

- `site/src/styles/global.css` の light / dark theme token
- README / CHANGELOG / release notes
- GitHub Release metadata

### 主要変更

- CSS の実質的な theme template である `:root` と `[data-theme="dark"]` の custom properties を更新しました。
- `--paper` / `--surface-*` / `--ink-*` / `--line-*` を冷灰基調に変更しました。
- primary action を青緑系に寄せ、補助色を低彩度にしました。
- `--on-accent`、`--code-bg`、`--code-ink` を追加し、言語切替、primary button、skip link、code block の hard-coded color を token 化しました。

### 検証結果

- `bun run build`: PASS, 4,219 pages
- 旧 warm palette の代表色は active CSS から除去済み
- layout / routing / wiki content / i18n entry / AI discovery count は変更なし

### 既知の注意点

- 本 release は視覚テーマの token 更新です。ページ構造、本文、公開 entry 数は変更していません。
- CSS template は新規ファイルではなく、既存どおり `site/src/styles/global.css` の custom properties として維持します。

### 次の作業

- push 後に GitHub Actions と公開 `/ja/`、`/en/`、`/zh/` の表示を確認します。
- 必要に応じて後続で visual regression 用 screenshot workflow を追加します。

## English

### Overview

Refreshed the human-site CSS color tokens away from warm paper / ochre tones and toward cool gray surfaces, teal-blue primary actions, and restrained secondary colors. The goal is a more neutral, work-focused financial reference presentation.

### Release Scope

- Light / dark theme tokens in `site/src/styles/global.css`
- README / CHANGELOG / release notes
- GitHub Release metadata

### Major Changes

- Updated the practical theme template: the custom-property blocks under `:root` and `[data-theme="dark"]`.
- Shifted `--paper`, `--surface-*`, `--ink-*`, and `--line-*` to cool gray neutrals.
- Moved primary action color to teal-blue and secondary colors to lower-saturation support tones.
- Added `--on-accent`, `--code-bg`, and `--code-ink`, then routed language switcher, primary button, skip link, and code block hard-coded colors through tokens.

### Validation Results

- `bun run build`: PASS, 4,219 pages
- Representative old warm-palette colors are removed from active CSS
- No layout, routing, wiki content, i18n entry, or AI discovery count change

### Known Notes

- This release changes visual theme tokens only. It does not change page structure, body content, or public entry counts.
- The CSS template remains the existing custom-property system in `site/src/styles/global.css`; no separate template file was added.

### Next Work

- After push, verify GitHub Actions and public `/ja/`, `/en/`, and `/zh/` rendering.
- Add a visual-regression screenshot workflow later if needed.
