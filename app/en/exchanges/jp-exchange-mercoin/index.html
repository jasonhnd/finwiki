株式会社メルコイン — 日本暗号資産交換業者 overview

FSA 登録番号 関東財務局 第00030号 · JVCEA 会員 #1039 · 登録日 2022-06-17

1. 法人・株主

  • 商号：株式会社メルコイン / Mercoin, Inc.
  • 設立：2021年4月28日
  • 本店所在地：東京都港区六本木6丁目10番1号 六本木ヒルズ森タワー
  • 親会社：株式会社メルカリ（東証プライム 4385）の完全子会社
  • 関連会社：株式会社メルペイ（グループ内 fintech 姉妹会社）
  • 資本金・出資比率の公表なし

2. 牌照・登録状況

  • 暗号資産交換業：関東財務局長 第00030号（登録日 2022-06-17）
  • 金融商品取引業：取得なし（デリバティブ取引非対応）
  • 電子決済手段等取引業：取得なし
  • JVCEA 会員：第一種・暗号資産交換業のみ（会員番号 1039）

3. 取扱通貨

  • 取扱銘柄数：3 種（2025-04-24 時点で確認）
    • ビットコイン（BTC）— 2023-03 取扱開始
    • イーサリアム（ETH）— 2024-05-21 追加
    • エックスアールピー（XRP）— 2025-04-24 追加
  • 板取引（取引所形式）は非提供。全銘柄 販売所形式のみ（スプレッドあり、手数料無料）
  • レバレッジ取引・デリバティブ：非対応

4. 業務範囲

  • 現物 spot 販売所：BTC / ETH / XRP の売買
  • 暗号資産つみたて機能：BTC（2024-08-01 開始）、ETH（2024-10-22 開始）。メルペイ残高からの自動引き落とし対応（2024-12）、毎日つみたて機能追加（2025-07-31）
  • ビットコイン決済：メルカリ内ショッピングでのビットコイン支払い（2024-02-15 開始）
  • ポイント付与機能：対象暗号資産を保有するだけで毎月メルカリポイント付与（2024-12-18 開始）
  • 法定通貨入金チャネル：メルペイ残高チャージ（銀行・ATM 振込）；メルカリ売上金・メルカリポイント直接購入が最大の特徴
  • 法人サービス・staking：提供なし（公表範囲内）

5. 市占・取引高

  • 2024-04（サービス開始約1年時点）：利用者数 200万人突破（同社プレスリリース）
  • 2023-10（サービス開始7ヶ月）：利用者数 100万人突破
  • 2024-05：直近1年間の暗号資産口座開設数が業界 No.1 と同社が公表（MMD研究所調査）
  • 月間取引高・預かり資産残高等の詳細数字：公表なし
  • 顧客層：メルカリ既存ユーザー（スマホ利用者）を主軸とした個人 retail。1 円からの少額購入設計
  • 競合内ポジション：口座開設数では industry top クラスと公称。ただしトレーダー向け機能は限定的で、本格的な売買プラットフォームとしては大手専業取引所（bitFlyer / bitbank 等）と異なるセグメントを主戦場とする。リテール経済圏連携は 楽天ウォレット と並んで JP CEX ポイント経済統合 パターンを成す。

7. 戦略動向

  • コインチェックとの業務提携（2025-08-05 締結）：詳細条件は非公開
  • 銘柄拡充ペース：BTC（2023） → ETH（2024-05） → XRP（2025-04）と年 1 銘柄ペースで増加
  • つみたて機能の拡充：BTC → ETH と積み立て対象が拡大、毎日つみたてオプション追加（2025-07）
  • XJPY / XUSD 業界横断ステーブルコイン：2023-11 に Ginco・三菱 UFJ 信託・Progmat との共同検討参画を発表（商用化スケジュール等は未公表）
  • メルカリグループとしての fintech 三本柱（メルペイ：決済送金与信 / メルカード：クレジットカード / メルコイン：暗号資産）の位置付けが明確。メルコインはブロックチェーン・Web3 戦略の中核とされている
