国内 Web3 / 暗号資産 公共政策ボディ層 (METI Web3 政策室 / 自民党 web3 PT / 内閣官房)

国内 Web3 / 暗号資産 公共政策ボディ層 (METI Web3 政策室 / 自民党 web3 PT / 内閣官房)

Wiki route

This entry sits under exchanges index. Read it against JCBA — 日本暗号資産ビジネス協会 for peer / contrast context and FSA 暗号資産交換業登録制度 for the broader system / regulatory boundary.

概要

国内 Web3 / 暗号資産政策は、FSA (金融規制) + METI (産業政策) + 自民党 web3 PT (議員立法 + 党内政策) + 内閣官房 (デジタル戦略横断調整) の四極構造で形成される。FSA が VASP ライセンス・JVCEA 監督・資金決済法 / 金商法を所管するのに対し、METI Web3 政策室は産業育成・トークン税制 (法人課税)・スタートアップ支援・国際標準化を所管。自民党 web3 PT が政治レイヤーで両者を架橋し、内閣官房 / デジタル庁が省庁横断調整を担う。**2022 年「岸田 web3 宣言」**以降、政府全体の web3 推進姿勢が確立し、毎年の「骨太の方針」「新しい資本主義」に web3 / 暗号資産政策が継続的に明記されるようになった。

主要公共政策ボディ

経済産業省 Web3 政策推進室 (METI Web3 Policy Office)

  • 設置：2022-07 (経済産業省商務情報政策局)
  • 位置：経済産業省 商務情報政策局 情報経済課 配下
  • 所管：Web3 産業育成、トークン税制 (法人課税) の制度改正提言、Web3 スタートアップ支援、NFT 産業政策、海外進出支援、国際標準化
  • 公式: https://www.meti.go.jp/policy/it_policy/web3/index.html
  • 主要成果：
    • 「Web3 ホワイトペーパー」シリーズ公表 (年次)
    • 法人保有暗号資産期末時価評価課税の見直し提言 (2024 改正で自社発行 + 第三者発行トークン除外として結実)
    • DAO ガイドラインの整備 (有限責任事業組合 LLC + DAO 構造)
    • NFT ガイドライン (景品表示法 + 賭博罪の境界整理)
    • 海外 web3 イベントへの省派遣 (デンバー Eth Denver / ETHGlobal Tokyo 等)

自民党 web3 プロジェクトチーム (web3 PT)

  • 設置：2022-01 (デジタル社会推進本部の下部 PT)
  • 位置：自民党 政務調査会 デジタル社会推進本部 配下
  • 過去座長 / 顧問：平将明 衆議院議員 (元デジタル副大臣) を中心に、塩崎彰久 / 神田潤一 / 山田太郎 / 川崎ひでと 等の議員が継続参画
  • 主要成果：
    • 「web3 政策提言 2022」「web3 政策提言 2023」「web3 政策提言 2024」「web3 政策提言 2025」を年次公表
    • 個人課税の分離課税化を継続提言 (未実現)
    • 法人税法改正への議員圧力 (2024 改正の駆動力)
    • DAO 法制度化議論
    • Web3 海外人材ビザ緩和
    • 規制サンドボックス活用
  • 業界連携：JCBA + JVCEA と密接に協議、政策提言を共同起草

内閣官房 / デジタル庁 (Digital Agency)

  • デジタル庁：2021-09 発足。マイナンバー / 政府デジタル基盤を所管。Web3 / 暗号資産に直接所管はないが、デジタル ID + ステーブルコイン + CBDC 等で交差
  • 内閣官房 新しい資本主義実現本部：「新しい資本主義」グランドデザイン + 実行計画で web3 / 暗号資産を継続記載
  • 内閣官房 デジタル田園都市国家構想実現会議：地方創生 × web3 / NFT 活用 (新潟県 山古志 NFT 等の前例)

金融庁 (FSA) — 規制サイド

  • 位置：内閣府 外局
  • 所管：VASP 登録 (FSA 登録制度)、JVCEA 認定自主規制監督、ステーブルコイン規制 (改正資金決済法 2023-06)、金商法暗号資産デリバティブ規制
  • 金融審議会 ワーキング・グループ：暗号資産・ステーブルコイン関連 WG を継続開催、改正法案の前段審議
  • METI との関係：規制 vs 産業育成の役割分担、両省局長級協議は継続

日本銀行 (BOJ)

  • 所管：CBDC (中央銀行デジタル通貨) 実証実験 (Stage 1: 2021-04 / Stage 2: 2022-04 / Stage 3 / 試験運用 2024-)
  • 暗号資産との関係：CBDC は暗号資産 ≠ という公式立場、ただし民間ステーブルコイン (→ JPYC) との競合 / 補完論点

実際の政策形成は、業界 (JCBA / JVCEA) → 自民党 web3 PT → METI / FSA → 内閣官房 / 与党税調 → 税制改正大綱 → 国会提出 という流れで進行。

主要政策アジェンダ (2024-2026)

  1. 個人課税分離課税化 — 業界長年要望、与党税調で継続審議。2026 年度税制改正大綱での実現が焦点
  2. DAO 法制度化 — LLC + DAO ハイブリッド設計、定款 + スマートコントラクトの法的位置付け
  3. NFT 二次流通法制 — 著作権 + 景品表示法 + 賭博罪の境界整理
  4. ステーブルコイン拡大 — 三菱 UFJ 信託 Progmat / JPYC / メガバンク発行体の枠組整理
  5. 海外 web3 人材誘致 — ビザ緩和 + 高度専門職在留資格
  6. AML / Travel Rule 強化 — → 犯収法 + FATF Travel Rule 国内実装
  7. トークン税制法人化対応 — 2024 改正の運用、未対応領域 (ガス代・エアドロップ等) の整理

国際比較 / 連携

  • 米国 SEC / CFTC / 財務省 / FIT21 / CARF 1099-DA (→ CARF / 1099-DA)
  • EU MiCA (2024-12 完全施行)
  • UK FCA / HMT crypto regime
  • シンガポール MAS Payment Services Act
  • 韓国 FSC + DAXA + Virtual Asset User Protection Act (2024-07)
  • 香港 SFC VASP licensing

日本の制度は MiCA に近い「包括的・先行型」と評価され、グローバル G7 議長国時 (2023) には web3 / ステーブルコイン国際整合性議論を主導。
