Coinone（코인원）— 韓国 CEX 中堅 / NH農协銀行実名口座 / 独立系 overview

2014-08 設立 · 韓国 spot 市場 #3（推定 3% 前後）· NH 농협은행 実名口座 · DAXA 共同設立 5 社の 1 つ · 2022 上場 청탁 검찰 기소事件で経営陣が司法手続を経た歴史

1. 法人・株主

  • 商号：주식회사 코인원（Coinone Inc.）
  • 設立：2014-08
  • 本店所在地：서울특별시 영등포구
  • 創業者：차명훈（Daniel Lim、Cha Myung-hoon） — 韓国初期暗号通貨業界の主要人物の 1 人として知られる（Coinone 共同創業 / CEO）
  • 主要株主：
    • 차명훈 創業者株式 が中核
    • 데일리금융그룹（Daily Financial Group、초기 투자자）→ 後に持株比率変動
    • 이상준 등 韓国系 IT / 金融投資家
    • 카카오뱅크系の出資検討報道（2018–2019）など、複数の戦略的資本関係模索の歴史
    • 詳細株主構成は非上場のため部分開示
  • IPO 検討：直接的アナウンスはないが、Bithumb の KOSDAQ 上場推進が成功すれば追随可能性

2. 登録・牌照状況（韓国制度）

  • 가상자산사업자 (VASP) 등록：FIU 登録済（2021-09 韓国第 1 期登録クラス）
  • ISMS / ISMS-P 認証：KISA ISMS-P 保有
  • 실명확인 입출금 계정（実名口座）：NH 농협은행（NH NongHyup Bank） との契約
    • NH 농협은행は韓国 5 大銀行の 1 つで、Bithumb も同 NH 농협은행と契約 → 韓国 5 大 CEX のうち NH 농협은행は 2 社（Bithumb + Coinone）と契約 する唯一の銀行
    • 詳細制度は 韓国 5 大 CEX 制度比較 参照

3. 取扱通貨

  • KRW spot 市場（韓国 5 大 CEX の 1 つとして合法）
  • 取扱銘柄数：200+（Upbit / Bithumb と並ぶ中規模ライン）
  • BTC / ETH / 主要アルトコイン中心、Klaytn 系 / WEMIX / XPLA 等韓国系トークンも積極取扱
  • 上場審査：DAXA 共通基準 + Coinone 独自基準

4. 業務範囲

  • 現物 spot 取引（KRW pair 中心）
  • 派生・margin 取引：韓国 VASP 規制下では非提供
  • staking：限定的サービス（Cross：staking / lending 関連商品の歴史的提供、規制制約下で縮小）
  • 法定通貨入金：NH 农协은행 実名口座経由のみ
  • Coinone Pro：プロトレーダー向け UI / API
  • Cross 海外送금：暗号通貨を媒介とした海外送金サービス（部分的展開）

5. 市占・取引高

  • 韓国 spot 市場シェア：約 3%（Upbit 70% / Bithumb 25% に次ぐ #3 ポジション、Korbit / GOPAX を上回る中堅 #3）
  • 24h volume：$20–200M（市場ボラティリティ連動）
  • 顧客層：韓国居住者中心（KYC 完全実施）

7. 戦略動向

  • 独立系経営：Upbit（카카오系・하나금융・한화등）/ Korbit（SK Square）/ GOPAX（Binance 部分関与）と異なり、創業者主導の独立系経営 を継続。これは独立性の優位性と同時に資本調達力の限界も意味する
  • NH 농협은행 関係：Bithumb と同銀行を共有する状況で、NH 농협은행側の AML/CFT 責任分担協議が継続課題
  • DAXA 参画：Upbit / Bithumb / Korbit / GOPAX と共に DAXA で上場基準を共通化。2022 上장 청탁 事件後のガバナンス再構築 において主導的役割
  • 2024 利용자보호법 対応：利용자보호기금 拠出、不공정거래 감시強化、内部統制再構築
  • 韓国系トークン主要上場場：Klaytn / 후속 Kaia、WEMIX、XPLA 等の韓国系 L1 / L2 トークンの主要上場場所として継続
  • 送金サービス Cross：暗号通貨を媒介とした海外送금サービス、規制環境変化対応

8. 業界での位置付け

Coinone は 韓国 spot 市場 #3 の中堅 CEX として、トップ 2（Upbit / Bithumb）と下位（Korbit / GOPAX）の 中間ポジション を保持する。独立系経営という個性と、2022 上장 청탁 事件のガバナンス問題という重荷を同時に抱える。

韓国 5 大 CEX 制度比較 の中で、創業者主導の独立系 という独自ポジションは、韓国 CEX 業界の構造的多様性を維持する役割を果たす。2022 事件以降の DAXA 共通上場基準整備は、Coinone 主導の側面もあり、業界全体のガバナンス底上げへの貢献は無視できない。
