ほくほく FG (Hokuhoku FG)

ほくほく FG (Hokuhoku FG)

TL;DR

北陸 3 県（富山・石川・福井）+ 北海道の 2 拠点広域地銀 FG。2003-09-26 北陸銀行が単独株式移転で持株会社化（旧「ほくぎん FG」）、2004-09-01 北海道銀行と経営統合して ほくほく FG 発足。地理ロングテール戦略（北陸 + 北海道の遠距離 2 地盤）が最大の特徴で、邦銀史上初の「離れた地域の地銀どうしの広域連合」モデル。北陸銀行は富山県地盤の旧第十二国立銀行（1877 創立）系、北海道銀行は札幌地盤の戦後設立（1951）行。北海道では ndfg（北洋銀行） との競合構造、北陸では富山県内 main bank ポジション。

1. 会社概要

正式名：株式会社ほくほくフィナンシャルグループ 英名：Hokuhoku Financial Group, Inc. 証券コード：東証 PRIME 8377 / 札証 8377 設立：2003-09-26（北陸銀行 単独株式移転で「ほくぎん FG」として発足） 現法人発足：2004-09-01（北海道銀行と統合 → ほくほく FG 改称） 本社：富山県富山市堤町通り 1-2-26（北陸銀行本店ビル）/ 札幌市中央区大通西 4-1（北海道銀行本店ビル）2 本社制

主要子会社・持分

ほくほく FG（持株会社・上場 8377）
  ├── 北陸銀行（100%）── 富山県本店・北陸 3 県地盤コア
  │     ├── 富山県（本店地盤、県内 main bank ポジション）
  │     ├── 石川県（金沢中心）
  │     ├── 福井県
  │     └── 関東・関西・北海道・東北 等の県外支店も歴史的に展開
  ├── 北海道銀行（100%）── 札幌本店・北海道地盤
  │     └── 北海道全域、札幌都市圏 + 函館・旭川・帯広・釧路 等
  ├── ほくほく TT 証券（北陸銀行 / 北海道銀行 系証券会社）
  └── 各種リース・カード・コンサル子会社（北陸カード / 道銀カード 等）

合併経緯・前身

  • 旧 北陸銀行：1877 第十二国立銀行 設立（富山）→ 1943 北陸銀行 改称（戦時統合）
  • 旧 北海道銀行：1951 設立（北海道地盤の戦後新設地銀。明治期の旧北海道銀行=植民地銀行とは別法人）
  • ほくぎん FG：2003-09-26 北陸銀行 単独株式移転で持株会社化
  • ほくほく FG：2004-09-01 北海道銀行と統合 → 改称

地理ロングテール戦略

  • 北陸 3 県（富山・石川・福井）+ 北海道 の 2 拠点広域連合
  • 邦銀統合史で「離れた地域の地銀どうしの広域連合」モデルの嚆矢
  • 都市地盤：札幌 / 富山 / 金沢 / 函館 / 福井 等の地方都市
  • 北陸新幹線（2015-03 金沢開業、2024-03 敦賀延伸）→ 北陸 - 首都圏アクセス改善で観光・ビジネスマネー流入機会

競合構造

  • 北海道：ndfg（北洋銀行）との激しい競合。北海道銀行はシェア 2 位ポジション
  • 富山県：県内 main bank ポジション（富山第一銀行 等の第二地銀と差別化）
  • 石川県：北國銀行 / 北國 FH と競合
  • 福井県：福井銀行 / 福邦銀行 と競合

観光・農業・水産金融

  • 北陸：観光（金沢・能登・立山黒部）+ 製造業（富山アルミ・薬・機械）
  • 北海道：観光（インバウンド）+ 農業 + 水産業 + 物流（千歳・苫小牧）
  • 両地盤とも一次産業 / 観光ウェイトが高く、地域経済との連動が深い

4. 規制・政策

  • 主管：金融庁（FSA）+ 北陸財務局 / 北海道財務局
  • 持株会社規制：銀行法 第 52 条の 17
  • 直近政策論点：
    • 2024〜 日銀政策金利正常化 → 地銀マージン改善（北陸銀行 / 北海道銀行 両行で恩恵）
    • 2025〜 地銀再編加速（人口減 + デジタル投資負担）
    • 北陸新幹線延伸（2024-03 敦賀）→ 福井県地盤の経済効果
    • 2024-01 能登半島地震 → 北陸銀行 石川県・富山県地盤への影響と復興金融
