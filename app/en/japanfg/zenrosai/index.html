全労済 / こくみん共済 coop (全国労働者共済生活協同組合連合会 / National Federation of Workers and Consumers Insurance Cooperatives)

全労済 / こくみん共済 coop (全国労働者共済生活協同組合連合会 / National Federation of Workers and Consumers Insurance Cooperatives)

Wiki route

This entry sits under JapanFG index as a workers’ / consumer-cooperative-based mutual-aid (共済) federation outside the JA / JF / shinkin / shinkumi axis. Read with JA 共済連 for the agricultural cooperative kyosai peer, 共栄火災海上保険 for cooperative-system non-life adjacency, insurance domain for the kyosai-vs-insurance regulatory boundary, and 労働金庫連合会 for the workers’ cooperative finance peer in banking.

TL;DR

労働者・消費者向けの生活協同組合系「共済」事業を担う全国連合会。1957 設立、消費生活協同組合法 (生協法) を根拠とする 生協連合会 (農協法ではなく生協法)。**「こくみん共済 coop (旧 こくみん共済)」**ブランドで、生命共済 + 損害共済 (火災・自動車・傷害等) を労働組合・労働者・地域住民を対象に提供。会員 = 全国の地域勤労者生協 + 職域生協 + 都道府県の労働者生協。保有契約・組合員数で 1000 万人超の協同組合系保険事業者として、JA 共済連と並ぶ国内大規模 mutual-aid 組織。主管は厚生労働省 (生協法所管、生活協同組合は厚労省管轄) で、保険業法対象の保険会社とは規制系統が異なる。

「こくみん共済 coop」ブランド戦略

  • 2019 ブランド統一: 「全労済」名称は旧称、対外的に 「こくみん共済 coop」 で広告・販売
  • 「coop」 = 生協連合体としての性格を前面化
  • 若年層・都市部勤労者の取り込み + 知名度向上

主要競合

  • JA 共済連 (ja-kyosairen) — 共済規模最大の同型機関
  • 都道府県民共済 (全国生協連) — 別の生協系共済連合会
  • 生保ビッグ 4 (日本生命 / 第一生命 / 明治安田 / 住友生命) — 生命系商品で競合
  • 3 大損保 (東京海上 / MS&AD / SOMPO) — 自動車・火災で競合
  • ネット保険 (ライフネット生命 等) — 直販モデルで価格競争

労働運動との結合

  • 連合 (日本労働組合総連合会) と緊密関係 — 共済の販売チャネルとして労組経由を活用
  • 公務員労組・教職員労組等の 職域分野 で強い基盤

デジタル戦略

  • Web 申込み・契約管理アプリ

4. 監督・規制

  • 主管: 厚生労働省 (生協法所管) + 金融庁 (金融健全性・消費者保護で協力監督)
  • 法的根拠: 消費生活協同組合法 (昭和 23 年法律第 200 号) + 生協共済規制
  • 直近政策論点:
    • 共済 = 保険の規制ギャップ (保険業法 vs 生協法の競争公平性)
    • 生協共済規制 (厚労省・金融庁協力監督) の強化方向
    • マイナンバー連携 での税控除等のデジタル化
    • ESR 類似指標 の自主開示
