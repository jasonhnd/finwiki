めぶき証券 (Mebuki Securities)

Wiki route

This entry sits under the JapanFG index as the FG-subsidiary securities operating-company anchor for めぶき FG. Read it against 常陽銀行 and 足利銀行 (parent bank subsidiaries) and the regional-bank consolidation pattern for system / regulatory boundary. Adjacent peer is FFG 証券 under fukuoka-fg for the FG-affiliated regional-securities comparison.

TL;DR

株式会社めぶき証券は、めぶき FG (東証 PRIME 7167) 100% 子会社の地銀系証券会社で、旧 常陽証券 (常陽銀行 系) を母体に mebuki-fg 設立後 (2016-10) にグループ統合・再編された。茨城・栃木・首都圏を地盤とする中堅企業 main bank 顧客向け資産運用・公募株引受・私募債仲介・事業承継 advisory を提供。第一種金融商品取引業 (証券業) 免許で、日本証券業協会 加盟。

旧 常陽証券系譜

  • 2007-11-30: 常陽証券株式会社 設立（茨城県水戸市、常陽銀行 系証券子会社）
  • 茨城県内 retail brokerage と中堅企業向け公募 / 私募債仲介を主業
  • 2010 年代〜: 地銀系証券として銀証連携の下で資産運用・私募債仲介の役割を拡大
  • 2017-04-03: めぶき証券株式会社へ商号変更

めぶき FG 設立とグループ統合

  • 2016-10-01: めぶき FG 設立 (常陽銀行 + 足利銀行 共同株式移転)

FG 内ポジション

  • 銀証連携の中核: mebuki-fg 顧客 (個人・中堅企業) の証券窓口
  • 資産運用 cross-sell: 銀行預金 → 投信 / ファンドラップへの顧客移行
  • 事業承継・M&A: 北関東中堅企業オーナー向け advisory サービス

競合構図

  • 大手証券 (野村 / 大和 / smbc-nikko / mizuho-securities): 都内大手企業 / 富裕層では大手が優位、地場顧客では銀証連携で対抗
  • ネット証券 (sbi-securities / rakuten-securities / matsui-sec): 個人デジタル取引では劣勢、対面 advisory で差別化
  • 地銀系証券: FFG 証券 / SMBC日興 等の同型 model 銀証連携相手
  • 県内競合: 七十七銀行 (宮城)・千葉銀行 系証券

強みと弱み

  • 強み: joyo-bank / ashikaga-bank 顧客基盤 + 北関東中堅企業 main bank ネットワーク

7. 規制・政策

  • 主管: 金融庁 (FSA) / 関東財務局 (第一種金融商品取引業者登録)
  • 業界団体: 日本証券業協会 正会員
  • 規制論点:
    • 銀証ファイアウォール規制緩和 (2022-) で銀証連携範囲拡大
    • 顧客本位の業務運営原則 (FD principle) 遵守
    • 高齢者投資勧誘ルール強化
    • 投信販売手数料の透明化・低水準化圧力
