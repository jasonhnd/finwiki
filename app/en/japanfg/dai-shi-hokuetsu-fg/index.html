第四北越 FG

第四北越 FG

TL;DR

新潟県地盤の地銀フィナンシャルグループ（同県内シェア圧倒・人口 220 万県の一県一行型）。2018-10-01 第四銀行 + 北越銀行 の共同持株会社化で発足、2021-01-01 子銀行同士を合併し 第四北越銀行 単一行体制へ移行。県内地盤統合（フルバンキング + 中堅企業 main bank）+ 県外進出（首都圏営業店）2 軸。前身の第四銀行は 1873 年設立で「日本初の国立銀行の一つ」(第四国立銀行) という由緒。観光（佐渡・スキー）・農業（米産地）・製造業（燕三条等）金融が地盤特性。地銀再編の中堅広域連合事例の一つ。

1. 会社概要

正式名：株式会社第四北越フィナンシャルグループ 英名：Dai-shi Hokuetsu Financial Group, Inc. 証券コード：東証 PRIME 7327 設立：2018-10-01（第四銀行 + 北越銀行 共同持株会社化） 本社：新潟県新潟市中央区（第四北越銀行本店所在地）

主要子会社・持分

第四北越 FG（持株会社・上場 7327）
  └── 第四北越銀行（100%）── 新潟県地盤の単一銀行子会社
        ├── 旧 第四銀行（1873〜2020）── 新潟県内最大手地銀
        ├── 旧 北越銀行（1942〜2020）── 県内第二地銀格
        ├── 県内営業基盤：新潟市・長岡市・上越市・佐渡 等
        └── 県外営業店：首都圏（東京）等

合併歴史・前身

  • 旧 第四銀行：1873 年 第四国立銀行として開業（日本初の国立銀行の一つ、ナンバー銀行 4 番）→ 1917 年 株式会社化 → 第四銀行。新潟県内最大手地銀
  • 旧 第六十九国立銀行：1878 年設立、後の北越銀行前身の一つ（ナンバー銀行 69 番）
  • 旧 長岡六十九銀行：第六十九銀行系列、長岡地盤
  • 旧 高田銀行：上越（高田）地盤
  • 旧 柏崎銀行：柏崎地盤
  • 旧 北越銀行：1942 年 長岡六十九銀行 + 高田銀行 + 柏崎銀行 合併で発足、新潟県内第二地銀格

統合背景：新潟県の人口減少 + 低金利マージン圧迫 + メガバンク法人融資競合 → 一県一行に近い広域地盤統合で対応。共同持株会社化（2018-10）後に銀行合併（2021-01）の 2 段階方式。

2. 事業セグメント・マップ

セグメント主要事業者特徴
地盤コア（新潟県内）第四北越銀行県内シェア圧倒、フルバンキング
中堅企業 main bank第四北越銀行製造業（燕三条金属加工等）・農業（米産地）
観光・地場産業金融第四北越銀行佐渡・スキーリゾート・新潟港物流
県外展開第四北越銀行 首都圏営業店東京等への進出（取引先随伴 + 新規開拓）
個人 retail第四北越銀行県内人口 220 万、預金 / 住宅ローン

地盤統合戦略（県内一行型）

  • 新潟県内 一県一行に近い地盤：県内競合が ほぼ存在しない地銀構造（第二地銀格の北越を統合済）
  • 県内人口 220 万を独占的にカバー → 安定的な預金基盤と中堅企業 main bank ポジション
  • 2021-01 銀行合併で 店舗・人員・システムの統合シナジー 顕在化フェーズ
  • 県内競合は信用金庫 / JA バンク / 郵貯（yucho）/ メガ法人融資

県外進出戦略

  • 首都圏営業店：県内顧客の取引先随伴（製造業 + 商社）+ 新規法人開拓
  • メガバンクとの法人融資競合：mufg / smfg / mizuho-fg が県内大手法人を直接アプローチ → main bank 防衛戦
  • 県外展開は規模拡大よりも県内顧客サービス拡充の意味合いが強い

地場産業特性

  • 観光：佐渡（金山・世界遺産候補）・湯沢 / 妙高スキー / 新潟市・長岡 観光振興
  • 農業：米（コシヒカリ ブランド産地）・酒蔵（清酒蔵元数 全国上位）
  • 製造業：燕三条金属加工クラスタ（刃物・洋食器・工具）/ 長岡 機械
  • 物流：新潟港（日本海側拠点港）・新潟空港

地銀再編における位置

  • 単独 FG として中堅広域連合の典型事例（県内一行型）
  • メガ 3 行（mufg / smfg / mizuho-fg）に対して県内 main bank ポジションで対抗
  • ndfg 等の他広域 FG / 隣県（東北・北陸）地銀との将来連合可能性は未確定

4. 規制・政策

  • 主管：金融庁（FSA）+ 関東財務局新潟財務事務所
  • 持株会社規制：銀行法 第 52 条の 17
  • 銀行合併規制：2021-01 第四 + 北越 合併は県内シェア過半超のため公取委審査対象（許認可済）
  • 直近政策論点：
    • 2024〜 日銀政策金利正常化 → 地銀マージン改善（メガより恩恵大）
    • 2025〜 地銀再編加速（人口減 + デジタル投資負担）
    • 新潟県の人口減少（県人口 220 万 → 長期減少トレンド）→ 預貸ボリューム圧迫
