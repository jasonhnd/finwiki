1. 法人・株主

  • 商号：Foxbit Serviços Digitais Ltda.
  • 設立：2014-09（São Paulo, Brazil）
  • 創業者：João Canhada（CEO）等の共同創業者
  • 主要株主：
    • 創業者株式が中核を維持
    • ParaFi Capital 等の暗号通貨専門 VC からの戦略出資（部分的）
    • Genoa Capital（ブラジル現地 VC）
    • 詳細株主構成は非上場のため部分開示
  • IPO 検討：2021–2022 期にブラジル B3 への上場検討報道、暗号市場下落で保留
  • ファンディング規模：Mercado Bitcoin $200M SoftBank ラウンドと比較すると相対的に小規模、独立系成長を継続

2. 登録・牌照状況（ブラジル制度）

  • Banco Central do Brasil (BCB) 認可：Lei 14.478/2022（marco legal de criptoativos, 2022-12 公布、2023-06 施行） 下で BCB の PSAV（Prestadores de Serviços de Ativos Virtuais）認可制度対象
    • 2024–2025 年期：BCB の本格的 PSAV 認可制度運用開始、Foxbit も登録プロセス対象
  • CVM：トークン化証券に該当する商品取扱いは CVM 規制下（Foxbit は主に utility token / 主要暗号通貨に集中、CVM 監督対象範囲は限定的）
  • Receita Federal：暗号通貨課税 / 取引報告義務
  • AML/CFT：COAF（ブラジル版 FIU）への報告義務
  • ABCripto（Associação Brasileira de Criptoeconomia）：ブラジル暗号通貨業界団体の主要メンバー、業界自主規制 / 規制対話に参画

3. 取扱通貨

  • BRL spot 市場（ブラジル国内で長期にわたる流動性提供）
  • 取扱銘柄数：100+（主要暗号通貨中心、銘柄数は Mercado Bitcoin より絞り込み）
  • USDT market 併設
  • 銘柄選定は 保守的 / 主要銘柄中心（リスク管理重視）

4. 業務範囲

  • 現物 spot 取引（BRL pair 中心）
  • 派生・margin 取引：限定的 / 規制対応観察中
  • staking：限定的（カスタディアル staking）
  • 法定通貨入金：PIX（ブラジル中央銀行即時決済システム）統合、TED / DOC 経由
  • 機関向け：OTC desk、API trading 提供
  • Foxbit Pay：暗号通貨決済関連サービス（限定的）

5. 市占・取引高

  • ブラジル spot 市場シェア：#2（推定 10–20%、Mercado Bitcoin の 60–70% に次ぐ位置）
  • 24h volume：$10–100M（BRL 流動性換算、市場ボラティリティ連動）
  • ユーザー数：100 万人超（公式公表ベース）
  • グローバル CEX ランキング：top 100 圏内
  • 海外 offshore CEX（Binance, OKX 等）のブラジル進出により、国内 CEX 全体のシェア圧迫を Mercado Bitcoin と共に経験

7. 戦略動向

  • 保守的銘柄戦略：Mercado Bitcoin の RWA tokenization / 多様な商品展開と異なり、主要暗号通貨 + BRL pair 流動性 に集中する保守的戦略
  • PIX 統合：ブラジル中央銀行即時決済システムの完全統合により、24/7 入出金 UX 提供
  • BCB / CVM 規制対応：Lei 14.478/2022 施行後の PSAV 登録 + CVM トークン化証券規制への二重対応、規制対応コスト負担増
  • ABCripto 業界団体活動：ブラジル暗号通貨業界自主規制 / 規制対話への積極参加、規制環境の安定化に寄与
  • 海外 CEX 競合対応：Binance, OKX, Bybit 等の offshore CEX のブラジル進出に対し、現地規制完全準拠 + PIX 即時決済 + ポルトガル語サポート で差別化
  • 機関向け展開：OTC desk、API trading、custody 関連サービスの段階的拡張

8. 業界での位置付け

Foxbit は ブラジル暗号通貨業界の老舗 #2 CEX として、Mercado Bitcoin と並ぶ二強構造を 2014 年以降維持してきた。Mercado Bitcoin の SoftBank 資金調達 / RWA tokenization 攻勢に対し、Foxbit は 保守的銘柄選定 + PIX 即時決済 UX + 独立系経営 で差別化を継続。

グローバル top 10 には届かないが、ブラジル国内では Mercado Bitcoin と並ぶ二大プラットフォームの一角として、ブラジル暗号通貨市場の構造的二強体制 を構成する。Lei 14.478/2022 + BCB PSAV 認可制度の本格運用は、両社の規制対応コストを引き上げる一方で、海外 offshore CEX 競合に対する 国内 CEX の制度的優位性 を強化する効果を持つ。
