イオン銀行 / イオンフィナンシャルサービス (Aeon Bank / AFS)

イオン銀行 / イオンフィナンシャルサービス (Aeon Bank / AFS)

Wiki route

This entry sits under JapanFG index. Read it against セブン銀行 (Seven Bank) for peer / contrast context and banking index for the broader system / regulatory boundary.

TL;DR

流通系銀行の代表格。総合スーパー aeon-group イオン株式会社が頂点に立ち、中間持株会社 イオンフィナンシャルサービス（AFS、東証 PRIME 8570） が イオン銀行 100% と アジア消費者金融 7 ヶ国網 を束ねる構造。店舗 = イオンモール内ブランチ + 駅前 ATM 約 6,500 台級、プロダクト = イオンカード（国内発行枚数上位）+ WAON 電子マネー + イオン住宅ローン、海外 = タイ・マレーシア・香港等のアジア消費者金融子会社群。seven-bank / lawson-bank と並ぶ流通 3 兄弟だが、ATM 単機能の 7 銀 / L 銀と異なり「総合銀行 + クレカ + アジア」の 3 階建てモデル。

1. 会社概要

正式名：株式会社イオン銀行（Aeon Bank, Ltd.） 設立：2006-10-03 開業：2007-10-29 本店：東京都江東区 親会社：イオンフィナンシャルサービス株式会社（100% 連結子会社） 最終親会社：イオン株式会社（流通最大手、東証 PRIME 8267）

イオンフィナンシャルサービス（AFS）— 中間持株会社

正式名：イオンフィナンシャルサービス株式会社（Aeon Financial Service Co., Ltd.） 証券コード：東証 PRIME 8570 設立：1981-06-20（旧ジャスコクレジット） 現体制：2013-04-01 イオンクレジットサービス → イオンフィナンシャルサービスへ社名変更、同時に持株会社化してイオン銀行を 100% 子会社化 本社：東京都千代田区

グループ構造（概要）

イオン株式会社 (8267, 流通最大手)
        ├── 株式会社イオン銀行 (100%) ── 商業銀行・イオンモール内支店 + ATM
        ├── イオン・アリアンツ生命保険 (Allianz と JV) ── 生保
        ├── イオン保険サービス ── 来店型保険ショップ
        ├── AEON Thana Sinsap (Thailand) PCL ── タイ消費者金融 (SET 上場)
        ├── AEON Credit Service (M) Bhd ── マレーシア消費者金融 (Bursa 上場)
        ├── AEON Credit Service (Asia) Co., Ltd. ── 香港消費者金融 (HKEX 上場)
        ├── AEON Microfinance (Cambodia) ── カンボジア
        ├── PT AEON Credit Service Indonesia ── インドネシア
        ├── ベトナム・ミャンマー子会社 ── 段階的展開
        └── イオン・フィナンシャル系シェアード子会社 (システム・債権回収等)

重要子会社・関連

  • イオン銀行：商業銀行コア。店舗 = イオンモール内有人ブランチ + イオン店舗内 ATM。住宅ローン直販強化。
  • イオン・アリアンツ生命：独 Allianz と JV、来店型販売チャネル
  • AEON Thana Sinsap (Thailand)：タイ SET 上場、東南アジア最大の AFS 海外子会社
  • AEON Credit Service (Asia) / (M)：香港・マレーシアそれぞれ現地上場

3. 沿革

年月事象
1981-06ジャスコクレジット 設立（旧ジャスコ系・現 AFS の前身）
2001イオンクレジットサービス へ社名変更（ジャスコ → イオン グループブランド統合）
2006-10-03株式会社イオン銀行 設立
2007-10-29イオン銀行 開業（流通系銀行の代表事例）
2013-04-01イオンクレジットサービス → イオンフィナンシャルサービス 改称・持株会社化、イオン銀行を 100% 子会社化
2013〜アジア消費者金融 子会社群を AFS 傘下に集約・現地上場維持
2020-03-31AFS がアリアンツ生命保険の株式 60% を取得 → 2020-05-01「イオン・アリアンツ生命保険」へ商号変更
2025イオン・アリアンツ生命を明治安田生命へ譲渡（2025-07-01 明治安田が約 85% 取得 → 2025-10-01「明治安田トラスト生命保険」へ商号変更）

流通系銀行モデル（差別化軸）

  • イオンモール = 店舗網：全国数百拠点のイオンモール内に有人ブランチ + ATM、買い物動線と一体化
  • ATM 6,855 台（2025-03-31）：イオン店舗・ミニストップ等を含む流通系 ATM 網（seven-bank 約 27,000 台、lawson-bank 約 13,000 台と比較すれば中規模だが、自社モール内集中型）
  • 住宅ローン強化：イオン住宅 + 直販。店舗動線で来店誘導

アジア消費者金融という第 2 軸

  • タイ（AEON Thana Sinsap PCL、SET 上場）、マレーシア（Bursa 上場）、香港（HKEX 上場）と 3 ヶ国で現地上場を維持する稀有な構造
  • 加えて カンボジア・インドネシア・ベトナム・ミャンマー に段階展開

競争環境

  • 流通系銀行 3 兄弟：seven-bank（7&i / ATM 単機能特化）/ lawson-bank（KDDI 提携 / ATM + 後発 retail）/ イオン銀行（総合銀行 + クレカ + アジア）— イオン銀行のみ「総合銀行 + 国際展開」フル装備
  • ネット銀行：sony-fg / 楽天銀行 / 住信 SBI ネット銀行 — 住宅ローン金利競争で対抗
  • 流通系電子マネー：WAON vs nanaco（7&i）vs 楽天 Edy vs PayPay
  • クレカ：イオンカード vs 楽天カード vs PayPay カード vs jcb
  • アジア消費者金融：アコム Easy Buy（タイ）/ オリコ / 地場プレイヤーと競合

B2C ブランディング

  • イオングループ統合ブランド（「イオン」赤紫ロゴ）
  • イオンカード会員向けの ときめきポイント、WAON ポイント連携
  • イオンモール来店動線が広告・販売チャネル

5. 規制・政策

  • 主管：金融庁（FSA）— イオン銀行は普通銀行ライセンス
  • AFS は東証 PRIME 上場（8570）：金商法ベースで開示
  • 直近政策テーマ（一般論）：
    • 日銀政策金利正常化 → 国内預貸マージン改善
    • キャッシュレス決済シェア争奪（WAON vs PayPay 等）
