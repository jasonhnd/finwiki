TL;DR

楽天グループ株式会社 (Rakuten Group, Inc., 東証 PRIME 4755) のフィナンシャルテック・セグメント。EC 軸の独自モデル: 楽天市場 + 楽天ポイント を core に、銀行 / 証券 / カード / 保険 / 決済 を「楽天経済圏」として束ねるクロスセル戦略。メガバンク系 (mufg / smfg / mizuho-fg) や通信系 FG (ndfg / au-FH / paypay-fg) と異なり、コマース起点 → 金融 という独自の事業組み立て。2023-04 楽天銀行 IPO、2024 楽天証券へみずほ証券 49% 出資受入。2024 期 FinTech セグメント Non-GAAP 営業利益 1,534 億円 (+37.9%)、グループ IFRS 営業利益 黒字転換 (530 億円)。楽天モバイル赤字 (Non-GAAP 営業損失 2,163 億円) は依然 FinTech 利益を上回るが、改善傾向。創業者・三木谷浩史 (1965-) 会長兼社長は 1997 創業以来の代表者。

ポイント楽天ポイント経済圏 OS・SPU で全事業を束ねる

「楽天経済圏」モデル

  • 楽天ポイント = OS: EC 購買・通信・カード・銀行・証券・保険・トラベル すべてでポイント付与/利用
  • SPU (Super Point Up Program): 楽天市場買い物のポイント倍率を「楽天モバイル契約 / 楽天カード利用 / 楽天銀行給与受取 / 楽天証券投信積立」等で段階アップ → 金融商品クロスセルの強力ドライバー
  • 会員 ID 統一: 1 楽天 ID で全サービス → CRM データ統合の優位

EC 軸の独自性（vs 通信系 / メガバンク系）

競合カテゴリ軸代表
EC 軸コマース → 金融Rakuten FG (本項)
通信軸通信回線 → 金融ndfg (docomo) / au-FH (KDDI) / paypay-fg (Yahoo/Z)
銀行軸商業銀行 → デジタル化mufg / smfg / mizuho-fg
専業ネット軸ネット証券 → 拡張sbi-hd

モバイル赤字 × 金融補填構造

  • 2019-10〜 楽天モバイル MNO 参入 → 基地局投資で 数年単位の大幅赤字 が常態化
  • 2024 期: Mobile Non-GAAP 営業損失 2,163 億円 vs FinTech Non-GAAP 営業利益 1,534 億円 → 依然 FinTech が Mobile 赤字を直接相殺できないが、改善幅 (Mobile +850 億・FinTech +37.9%) で 2024 期 IFRS 営業利益 黒字転換 (530 億円)
  • FinTech セグメント（特に 楽天銀行 / 楽天カード）の安定収益が、グループ連結 EBITDA を下支え
  • 2022-2024 期間: 楽天証券 IPO 計画 → みずほ証券 49% 出資受入で 資金調達 + IPO 撤回 の組換え
  • 2023-04 楽天銀行 IPO 実施 → グループ流動性 + 上場簿価顕在化

みずほ FG との資本業務提携

  • 2023-10 みずほ証券 19.99% 出資 → 2024 段階引上げで 49% へ
  • 楽天証券 IPO は事実上撤回
  • みずほ側にとっては リテール・ネット顧客基盤獲得、楽天側にとっては グループ資金調達 + メガバンク系証券との連携深化

4. 規制・政策

  • 主管: 金融庁（FSA）
  • 業態別規制:
    • 楽天銀行: 銀行法（ネット銀行）
    • 楽天証券: 金融商品取引法（第一種金融商品取引業）
    • 楽天カード: 割賦販売法 + 貸金業法
    • 楽天生命 / 楽天損保: 保険業法（生保・損保各々）
    • 楽天ウォレット: 資金決済法（暗号資産交換業者登録 = JVCEA 加盟）
    • 楽天ペイ / 楽天 Edy: 資金決済法（前払式 / 資金移動業）
  • 持株会社規制: 楽天グループ本体は事業持株会社（純粋持株会社ではない）。各業態子会社が個別に業法規制を受ける
  • 直近政策論点:
    • 楽天モバイル 周波数割当 / 設備投資 → 総務省管轄、ただし金融セグメントへの間接影響
    • 2024〜 新 NISA 制度 → 楽天証券口座増勢
    • ステーブルコイン / 暗号資産規制改定 → 楽天ウォレット への波及
