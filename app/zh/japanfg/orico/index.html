株式会社オリエントコーポレーション (Orico)

TL;DR

mizuho-fg 系・伊藤忠商事系の国内大手信販。1954-12-22 広島で「廣島総合信用」として設立、1962 にオリエントファイナンス、2001 に現社名へ改称。自動車ローン (オートローン) 国内最大手を背骨に、ショッピングクレジット・クレジットカード (Orico Card 各種)・カードローン・法人ファイナンス・不動産担保ローンを展開。2005-12 経営再建で mizuho-fg (旧興銀系) + 伊藤忠商事 + KDDI 等の優先株注入を受け、現在は みずほ FG の持分法適用関連会社 + 伊藤忠商事が主要株主の枠組み。東証 PRIME 8585。競合は jaccs / クレディセゾン / 三井住友カード (smfg 系) / 三菱 UFJ ニコス (mufg 系) / イオン系 aeon-bank。

1. 会社概要

正式名：株式会社オリエントコーポレーション 英名：Orient Corporation 略称：オリコ / Orico 証券コード：東証 PRIME 8585 設立：1954-12-22 (前身「廣島総合信用株式会社」) 現社名改称：2001-04 (オリエントファイナンス → オリエントコーポレーション) 本社：東京都千代田区麹町 5-2-1 麹町センタープレイス 業態：信販 (個品割賦・包括信用購入あっせん)・クレジットカード・カードローン・銀行保証・法人ファイナンス・不動産担保ローン

社名・略称の由来

  • Orient = 旧社名「オリエントファイナンス」(1962-) を継承
  • Orico = Orient Corporation の合成略称
  • 創業地は広島 (旧 廣島総合信用)、後に東京本社へ移転

主要株主 (公開情報ベース)

株主性格備考
みずほ銀行 (mizuho-fg 系)筆頭株主・持分法適用関連会社2010-09 に持分法適用関連会社化。2026-03 末で約 48.66%、2026-05 に 33.79% へ引下げ（後述）
伊藤忠商事旧 大株主（持分法適用外へ）2005 に第三者割当等で 21% 取得し筆頭株主、2024-11 保有株一部売却で伊藤忠側の持分法適用対象外に
信託銀行各社・機関投資家浮動株上場会社のため通常開示

みずほ銀行は 2010-09 以降オリコを 持分法適用関連会社として位置付け（議決権ベースで約 48% 台）。2026-04 にアクティビスト（ストラテジックキャピタル）がみずほ FG に対し「間接保有約 48% は連結子会社に当たる可能性」を指摘し完全子会社化または全株売却を求める株主提案を実施。みずほは 2026-05-15 にオリコ株の 15%（約 260 億円）をアイフル親会社のムニノバ HD へ売却し保有比率を 48.79%→33.79% に引下げ（3 社業務提携も発表）、引き続き持分法適用関連会社とした。

コア差別化

  • 自動車ローン国内 No.1: 自動車販売店との長期チャネル関係が最大の参入障壁。新車・中古車の販社経由割賦で 業界トップシェア
  • みずほ系列クレジット中核: mizuho-fg グループ内のクレジット・信販事業の中核機能 (vs mufg の三菱 UFJ ニコス / smfg の三井住友カード)
  • 伊藤忠連携: 伊藤忠商事の流通・自動車関連ネットワークとの親和性 (商社系顧客基盤・海外進出時の協業)

戦略課題

  • オートローン市場の構造変化: EV シフト・サブスク (KINTO 等)・カーシェアの普及で、従来型「個人購入+割賦」モデルが中長期で変容リスク
  • デジタル化対応: コード決済 (PayPay / 楽天ペイ / d 払い / au PAY) の急成長で物理カード・割賦モデル全体が押される圧力
  • 海外展開: アジア (インドネシア・タイ・ベトナム) の自動車・二輪ファイナンスは中長期成長領域

B2C ブランディング

  • Orico Card THE POINT: 高還元率プロパーカードでの個人客獲得

4. 規制・政策

  • 主管: 経済産業省 (割賦販売法所管) + 金融庁 (貸金業法・資金決済法等)
  • 主要根拠法: 割賦販売法 (個品割賦・包括信用購入あっせん)、貸金業法 (カードローン・カードキャッシング)、資金決済法
  • 業界自主規制団体: 一般社団法人日本クレジット協会 (JCA)、日本貸金業協会
  • 個人情報保護: 個人情報保護法 + クレジット関連個人信用情報機関 (CIC / JICC / 全国銀行個人信用情報センター)
  • 加盟店管理: 改正割賦販売法 (2018〜) で加盟店調査義務強化、PCI DSS 準拠
  • 直近政策論点:
    • キャッシュレス推進政策 (経産省目標 2025 年 40%、2030 年代 80%)
    • 改正個人情報保護法 (信用情報の取扱厳格化)
    • グレーゾーン金利問題の収束 (利息制限法・出資法上限金利の一本化、2010-06 完了)
    • インボイス制度 (2023-10〜) でカード明細・経費精算の取扱変化
