TL;DR

英国 Barclays PLC（G-SIB Bucket 2）の日本拠点。1969 駐在員事務所開設 → 1972 東京支店 → 1986 英銀初の東証上場 → 1987 BZW 証券事務所 → 1990 東証会員 → 2006 バークレイズ・キャピタル証券化 → 2012 バークレイズ証券に商号変更。リーマンショック後の世界再編で、北米はバークレイズが買収、アジア太平洋（含む日本）は nomura-hd が買収したため、日本拠点は Lehman 統合の恩恵を直接受けず、自力での IB / マーケッツ集中戦略を歩む。2016 グローバル再編でアジア株式キャッシュ事業撤退、現在は日本・香港・シンガポール・インドの 4 拠点をコアアジア市場として残す。約 500 名規模、本拠は六本木ヒルズ森タワー 31F。

2. 沿革・重要年表

年事象
1969バークレイズ銀行 東京駐在員事務所 開設（バークレイズ初のアジア拠点）
1972東京支店 に格上げ・営業開始
1986英国の銀行として初めて東京証券取引所に上場
1987BZW 証券会社（バークレイズ・キャピタル証券の前身）の駐在員事務所 → 支店に格上げ
1990BZW 証券会社 東京支店、東証 正会員
1995BZW 証券会社 → BZW ジャパン証券会社 東京支店に改名
1998バークレイズ・キャピタル証券会社 東京支店 に改名
2006バークレイズ・キャピタル証券会社 → バークレイズ・キャピタル証券株式会社（日本法人化）
2008-06-28東証 上場廃止
2008-09リーマン・ブラザーズ 米州事業 を 約 17.5 億ドルで買収（グローバル IB 強化）— ただし アジア太平洋 + 欧州・中東は nomura-hd 買収、日本拠点は直接統合効果なし
2008BGI 信託銀行 → 住友信託銀行（現 sumitomo-mitsui-trust）吸収合併
2009-06親会社 Barclays が BGI 部門全体を BlackRock に約 135 億ドルで売却 → 日本も資産運用から事実上撤退
2010-03大手町 ↔ 六本木 拠点交換（旧 リーマン拠点を継承した 野村 と部分入替、バークレイズが六本木ヒルズ森タワー 31F へ集約）
2012バークレイズ・キャピタル証券 → バークレイズ証券株式会社 に商号変更
2016グローバル再編: アジア株式キャッシュ事業撤退、豪・インドネシア・マレーシア・フィリピン・韓国・台湾・タイから IB 縮小、日本は 4 コア市場として残存。同時にアフリカ撤退
2016アサヒグループ HD の中東欧ビール買収（約 8900 億円）買い手側アドバイザー — 唯一の M&A 大型案件
2021-12アジア事業再活性化方針（Bloomberg 報道）、日本・香港で採用強化
2024-01馬場 直彦 を チーフ・エコノミスト兼調査部長 に任命
2025-01竹村 陽一 を 日本のマクロ・トレーディング本部長 として採用

誤解しやすい点: バークレイズの Lehman 北米買収（2008-09）は日本に直接的な人員・業務拡大効果をもたらしていない。アジア太平洋（含 日本）+ 欧州・中東は nomura-hd が買収した。日本拠点の M&A / IB 拡大は自力で行われた。

3. 事業セグメント・マップ

セグメント主要事業者特徴
投資銀行（IB）バークレイズ証券 + バークレイズ銀行東京支店M&A 助言、ECM、DCM。クロスボーダー M&A 中心
マーケッツバークレイズ証券・東京支店株式（一部）・債券・FX・金利・クレジット・コモディティ
法人銀行業務バークレイズ銀行 東京支店多国籍企業向け融資・トランザクションバンキング
調査（リサーチ）バークレイズ証券 調査部馬場 直彦 がチーフ・エコノミスト
資産運用（撤退）BGI を BlackRock に売却（2009）、信託は住信に吸収（2008）
リテール未展開親会社英国本体は UK Barclaycard を持つが日本展開なし

親会社の戦略文脈

  • 2016 Jes Staley CEO の **「Transatlantic + Consumer + Corporate + IB」**戦略 → 新興国 IB 撤退、アジアは 4 コア市場に絞り込み
  • 2021- アジア再活性化方針への転換、日本・香港でリビルド
  • 2024- IB + マーケッツ集中継続

日本での競争ポジション

  • 強み: グローバル DCM（債券引受）— グローバルリーグテーブルでは Top 10、クロスボーダー金利・FX マーケッツ
  • 弱み: 国内 M&A / ECM リーグテーブル外 — 過去 5 年で 2016 年 16 位（約 1.2 兆円）が最高位
  • 競合: 中堅外資 IB — ubs-japan / deutsche-japan / bnp-paribas-japan / hsbc-japan / standard-chartered-japan / citigroup-japan
  • 大手米系（より上位）: goldman-sachs-japan / morgan-stanley-japan / jpmorgan-japan
  • 国内勢: nomura-hd / smbc-nikko / daiwa-sg / MUMSS（mufg × Morgan Stanley JV）

過去の主要案件

  • 2016 アサヒグループ HD 中東欧ビール買収（約 8900 億円） 買い手側アドバイザー
  • 2013 オリックスのロベコ・グループ買収（約 2400 億円） 売り手側アドバイザー

5. 規制・政策

  • 主管: 金融庁（FSA）+ 関東財務局
  • 証券業ライセンス: 関東財務局長(金商)第143号（バークレイズ証券）
  • 銀行業ライセンス: 外国銀行支店認可（東京支店）
  • 直近政策論点:
    • 2024〜 日銀利上げ → 円金利マーケッツ機会拡大
    • 2025〜 アジア IB 再強化方針 → 日本での採用継続
