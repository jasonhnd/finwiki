九州 FG (Kyushu FG)

九州 FG (Kyushu FG)

Wiki route

TL;DR

九州南部地盤の単独地銀グループ（熊本 + 鹿児島 2 県中核）。2015-10-01 肥後銀行 + 鹿児島銀行 の経営統合で発足。県またぎ「2 強連合」型統合（fukuoka-fg の広域地盤統合・mufg 等メガ広域連結と異なるモデル）。TSMC 熊本進出（2022〜菊陽町）に伴う半導体クラスター関連融資需要急増が直近の最大トピック。九州内では fukuoka-fg・nishi-nippon-fhd と並ぶ拠点 FG だが、地盤を九州南部に絞った棲み分け構造。

1. 会社概要

正式名：株式会社九州フィナンシャルグループ 英名：Kyushu Financial Group, Inc. 証券コード：東証 PRIME 7180 / 福証 7180 設立：2015-10-01（肥後銀行 + 鹿児島銀行 共同持株会社化） 本社：熊本市中央区 + 鹿児島市（2 本社制）

主要子会社・持分

九州 FG（持株会社・上場 7180）
  ├── 肥後銀行（100%）── [[higo-bank]] / 熊本県地盤、県内シェア圧倒
  │     └── 熊本県 main bank ポジション、TSMC 菊陽町関連融資の中核
  ├── 鹿児島銀行（100%）── [[kagoshima-bank]] / 鹿児島県地盤、県内 2 強の一角
  │     └── 鹿児島県 main bank、農林漁業・観光融資
  ├── 九州 FG 証券（100%）── グループ証券
  ├── 九州 FG リース（100%）── リース業務
  └── 関連: ヒゴカード / カゴシマカード 等 ─ 各銀行系クレカ

合併経緯・前身

  • 旧 肥後銀行：
    • 1879 第百三十五国立銀行 設立（熊本、国立銀行条例下）
    • 1925 普通銀行転換 → 肥後銀行 改称
    • 戦後 熊本県を地盤とする地方銀行として再編
  • 旧 鹿児島銀行：
    • 1879 第百四十七国立銀行 設立（鹿児島、国立銀行条例下）
    • 普通銀行転換を経て、1944 戦時統制下で県内銀行合同 → 鹿児島銀行 成立
    • 戦後 鹿児島県を地盤とする地方銀行
  • 九州 FG：2015-10-01 共同持株会社化で発足

統合背景：九州南部の人口減・低金利マージン圧迫 + 隣県の地盤統合需要。肥後銀行（熊本）+ 鹿児島銀行（鹿児島）は地盤が隣接かつ重複が小さい「相互補完型」統合。長崎県のような県内競合圧縮型（fukuoka-fg の十八親和銀行統合 = 公取委審査 3 年遅延）と異なり、独禁法ハードルは相対的に低かった。

TSMC 関連特需（2022〜2024 の最大トピック）★

  • TSMC（台積電）熊本菊陽町進出：第 1 工場（JASM、2024-02 開所）+ 第 2 工場（2027 稼働予定）
  • 肥後銀行が地元 main bank として関連融資の中核：
    • 半導体サプライチェーン企業の進出融資（用地取得 / 設備投資）
    • 地場下請け・関連業者の設備更新融資
    • 不動産・宿舎・商業施設関連融資（菊陽町・大津町周辺の急膨張）

地盤戦略

  • 九州南部 2 県（熊本 + 鹿児島）+ 宮崎一部に集中
  • fukuoka-fg（福岡 + 熊本 + 長崎）と熊本県でオーバーラップ → 肥後銀行が地盤優位、熊本銀行（FFG）は第二地銀的ポジション
  • 鹿児島県は**鹿児島銀行 + 南日本銀行 の 2 強構造**（南日本銀行は第二地銀、規模は鹿児島銀行が圧倒）
  • 観光・農業・漁業金融：南九州固有の地場産業（焼酎・農産物・漁業・観光）への融資ノウハウ

競合・地銀再編

  • 九州内競合：fukuoka-fg（FFG）・西日本 FHD（西日本シティ銀行）と並ぶ九州 3 大 FG だが、地盤の南北分業で直接競合は限定的
  • メガバンク（mufg / smfg / mizuho-fg）：大企業・全国展開は譲るが、地場中堅は地銀優位

4. 規制・政策

  • 主管：金融庁（FSA）+ 九州財務局
  • 持株会社規制：銀行法 第 52 条の 17
  • 熊本地震対応（2016-04）：被災地金融機関として復旧融資・モラトリアム対応の運用知見
  • 直近政策論点：
    • 2022〜 TSMC 進出に伴う経済産業省 / 熊本県の半導体クラスター施策との連携（補助金スキーム / 用地造成 / 人材確保）
    • 2024〜 日銀政策金利正常化 → 地銀マージン改善（メガより恩恵大）
    • 2025〜 地銀再編加速（人口減 + デジタル投資負担）の中で南九州 FG の位置付け
