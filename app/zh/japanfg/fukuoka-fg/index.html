ふくおか FG (FFG)

TL;DR

国内最大級の単独地銀グループ（九州 3 県地盤 + 全国デジタル展開）。2007-04-02 福岡銀行 + 熊本ファミリー銀行（現・熊本銀行）の共同持株会社化で発足。地盤統合（福岡 / 熊本 / 長崎）+ デジタル全国展開（みんなの銀行 BaaS）2 軸。2021-05 開業の みんなの銀行 は日本初のフルクラウドネイティブ・スマホ完結デジタルバンクで、親 FFG にとってデジタル戦略の中核。地銀再編における主導役の一角（SBI 第 4 メガ構想 / 千葉・横浜・コンコルディア等と並ぶ拠点）。

1. 会社概要

正式名：株式会社ふくおかフィナンシャルグループ 英名：Fukuoka Financial Group, Inc. 証券コード：東証 PRIME 8354 / 福証 8354 設立：2007-04-02（福岡銀行 + 熊本ファミリー銀行 共同持株会社化） 本社：福岡県福岡市中央区天神 2-13-1（福岡銀行本店ビル内） 連結子会社・関連会社：27 社 + 1 社（持分法） 連結従業員数：7,928 名（2025-03 期、嘱託・臨時 2,803 人・執行役員 43 人は除く）

主要子会社・持分

FFG（持株会社・上場 8354）
  ├── 福岡銀行（100%）── [[fukuoka-bank]] / 九州最大地銀・地盤コア
  │     └── 福岡県中心、大都市 retail + 中堅企業 main bank
  ├── [[kumamoto-bank|熊本銀行]]（100%）── 旧 熊本ファミリー銀行（旧 肥後ファミリー銀行）
  │     └── 熊本県地盤
  ├── 十八親和銀行（100%）── [[18-shinwa-bank]] / 2020-10-01 十八銀行 + 親和銀行 合併
  │     └── 長崎県地盤（県内シェア圧倒）
  ├── みんなの銀行（100%）── 日本初フルクラウドネイティブ・デジタルバンク（2021-05-28 開業）
  │     └── BaaS 機能を他社提供（決済 API / 口座開設 API / パートナー支店モデル）
  ├── FFG 証券（100%）── グループ証券
  └── iBank マーケティング（100%）── Wallet+ アプリ運営（地域 Fintech）

合併経緯・前身

  • 旧 福岡銀行：1945 設立、九州最大地銀（旧 富士銀行系・旧三和銀行系等の遺産含む）
  • 旧 熊本ファミリー銀行：旧 肥後ファミリー銀行（地方銀行）が前身、後に 熊本銀行 に改称
  • 旧 十八銀行：1877 創立、長崎県地盤（旧 第十八国立銀行）
  • 旧 親和銀行：1939 設立、長崎県地盤（旧 三和銀行系含む）
  • 旧 熊本銀行（第二地銀）：FFG 統合前から存在、熊本ファミリー改称後の現法人 熊本銀行 と区別が必要

統合背景：地銀の人口減少 / 低金利マージン圧迫 → 県をまたぐ広域連合と全国デジタル展開の 2 軸。十八銀行 + 親和銀行 合併は独禁法対応で 3 年遅延（長崎県内シェアが過半超のため公取委審査）。

デジタル戦略（みんなの銀行が中核）

  • みんなの銀行（2021-05-28 開業）：
    • 日本初の「クラウドネイティブ・スマホ完結」デジタルバンク
    • Google Cloud + Accenture 提携でフルクラウド設計
    • 2025-05：130 万口座突破（開業 4 周年）。SNS 経由で若年層獲得が中心
    • 2025-03 末 預金残高 331 億円・貸出金残高 228 億円（前年比 +90%）
    • BaaS パートナー 30 社突破（2025-12 時点）：
      • パートナー支店モデルのみ：3 社
      • API 提供モデルのみ：8 社
      • 両モデル併用：7 社
      • 基本合意書締結（リリース準備中）：12 社
    • 代表的提携先：メルペイ（2025-08 業務提携）等
    • 黒字化目標：当初 2025 年度 → 先行投資 + ローン残高伸び悩みで延期、現在は 2027 年度黒字化が新マイルストーン
    • 2026 年度予定：1 人で最大 5 支店まで口座開設可能に拡張
  • 「地銀発デジタル基盤」のリーダーシップ志向：BaaS によって他地銀 / 異業種に勘定系を提供する野心
  • iBank マーケティング：Wallet+ アプリで地域 Fintech 展開

地盤統合戦略

  • 九州 3 県（福岡 / 熊本 / 長崎）地盤統合 + デジタル全国展開の 2 軸
  • 十八親和銀行 統合（2020-10）で長崎県内シェア圧倒 → 県内 main bank ポジション確立
  • 熊本銀行 子会社化（2007〜）で熊本県進出

地銀再編における位置

  • 単独地銀グループで国内最大級（純利益 854 億円・総資産 33.56 兆円）
  • 地銀再編の主導役（SBI 第 4 メガ構想 / chiba-bank-fg / concordia-fg / mebuki-fg 等と並ぶ広域連合拠点）
  • メガバンク（mufg / smfg / mizuho-fg）に対して地銀統合 + BaaS で対抗

4. 規制・政策

  • 主管：金融庁（FSA）+ 九州財務局
  • 持株会社規制：銀行法 第 52 条の 17
  • 十八銀行統合 独禁法審査：2016 申請 → 2018-12 統合決定（3 年遅延）。地銀統合における公取委審査の典型ケース
  • みんなの銀行 規制：銀行法 普通銀行免許（フルバンキング、ネット専業銀行と異なるフル機能）
  • 直近政策論点：
    • 2024〜 日銀政策金利正常化 → 地銀マージン改善（メガより恩恵大）。FY2025 経常収益 +36.3% は金利上昇の直接反映
    • 2025〜 地銀再編加速（人口減 + デジタル投資負担）
    • みんなの銀行 BaaS 拡大 → 他地銀向け勘定系提供の制度整備
