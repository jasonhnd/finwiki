TL;DR

国内地銀 FG 最大級、預金 19.7 兆円・総資産 24.8 兆円（2025-03 末、地銀 FG 単独首位級）。2016-04-01 横浜銀行 + 東日本銀行 が共同株式移転で経営統合、旧称「コンコルディア・フィナンシャルグループ」(ラテン語「調和」) として発足。2025-10-01 商号変更 →「株式会社横浜フィナンシャルグループ」（中核行ブランドを社名に冠し、国内外認知度向上が目的）。神奈川県（県内シェア圧倒的）+ 首都圏 中堅・中小企業のリテール集合体。証券コード 東証 PRIME 7186。FY2024 (2025-03 期) 純利益 828 億円（前年比 +23.7%）、FY2025 (2026-03 期) 通期見通し 1,065 億円（+28.6%）と日銀利上げ追い風で連続二桁増益。横浜銀行は TSUBASA アライアンス非加盟、千葉銀行と「千葉・横浜パートナーシップ」（2019〜、24-28 年度二期目 250 億円効果目標）。最大の競合は fukuoka-fg / 千葉銀行・resona-hd。

戦略軸

  • 「フィナンシャルプラットフォーム」戦略: 銀行業務に加えて、地域企業向け非金融サービス（事業承継・M&A 仲介・人材紹介等）を統合提供
  • 神奈川 + 首都圏 集中: 全国展開しない代わりに、首都圏 GDP の大きさを最大活用
  • 横浜銀行 = リテール + 中堅 / 東日本銀行 = 都内中小特化 の 役割分担（統合後も子銀行 2 社体制継続）
  • 商号変更による「横浜」ブランド回帰（2025-10-01）: 国際性・多様性・地域ブランド力の活用、認知度向上

統合シナジー・課題

  • 預金 統合効果: 単独で地銀 No.1 級の預金規模 → 大型法人案件取り組み可能
  • システム統合: 横浜銀行系基盤 + 東日本銀行 統合作業の進捗
  • 東日本銀行 不適切融資問題（2018-08）→ 内部統制再構築・コンプライアンス強化中
  • 千葉・横浜パートナーシップ: 一期目（2019-2023）で目標 200 億円→実績 359 億円、二期目（2024-2028）で 250 億円効果目標

B2C ブランディング

  • 横浜銀行 = 「はまぎん」: 神奈川県民の生活インフラとしての認知度 圧倒的
  • 東日本銀行: 都心中小企業向け、B2C ブランド 弱
  • 横浜 FG（旧 CFG）: 持株会社ブランドは個人客には浸透限定的だったが、2025-10 商号変更で「横浜」ブランドを前面化
  • 株主優待（2025-03 基準〜変更）: 定期預金金利上乗せ → 神奈川・東京の特産品カタログギフトに切替、地域への送客強化

4. 規制・政策

  • 主管: 金融庁（FSA）
  • 持株会社規制: 銀行法 第 52 条の 17（銀行持株会社）
  • 業務改善命令歴: 2018-08 東日本銀行に対し 不適切融資・反社チェック不備で命令発出 → 経営陣刷新・内部統制再構築
  • 直近政策論点:
    • 2024〜 日銀政策金利正常化 → 国内貸出金利マージン改善ボーナス（FY2024 純利益 +23.7%、FY2025 +28.6% 見通し）
    • 2024〜 地銀再編期待（金融庁のスタンス: 統合自体は否定しないが当局主導はしない）
    • 2025〜 営業店舗削減・地方支店再編
    • 中小企業向け事業性評価融資（担保・保証に依らない融資）の推進
    • ESG / 地域脱炭素 ファイナンス（TSUBASA 非化石証書共同購入の枠外）
