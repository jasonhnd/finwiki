TL;DR

滋賀県地盤の単独銀行（持株会社化していない地銀の一つ）。1933-10-01 八十三銀行 + 近江銀行 統合により発足、ルーツは 1879 第六十四国立銀行（彦根）。略称「しがぎん」。滋賀県内シェア No.1 + 京都府北部 / 大阪府北摂 / 名古屋圏 への関西・東海境界エリア展開 + 琵琶湖周辺の環境ファイナンス + ESG / サステナビリティ先進性が収益コア。国連責任銀行原則（UN PRB）採択地銀の先駆け（2019、地銀初）として ESG リーダー枠を確立。chiba-bank 主導の TSUBASA アライアンス 加盟行でシステム共同化・海外サポートに参画。隣接する関西地銀 kyoto-fg / resona-hd（関西みらい銀行含む）/ ikeda-senshu-hd / nanto-bank とエリアで接触。

1. 会社概要

正式名：株式会社滋賀銀行 英名：The Shiga Bank, Ltd. 証券コード：東証 PRIME 8366（旧東証一部から 2022-04 移行） 設立：1933-10-01（八十三銀行 + 近江銀行 統合） 本店：滋賀県大津市浜町 1-38 業態：単独銀行（持株会社化していない点が、近隣の 京都 FG・池田泉州 HD 等の関西地銀 FG 化トレンドと対照的）

合併歴史・前身

  • 1879 第六十四国立銀行 設立（彦根、明治期国立銀行条例下、第 64 号で設立）— 滋賀銀行の最古の源流（しがぎんの源流彦根）
  • 戦前期、滋賀県内で 八十三銀行（大津）・近江銀行・第六十四銀行系などの地銀が並立
  • 1933-10-01 滋賀銀行 発足：戦時下「一県一行主義」政策の流れの中で 八十三銀行 + 近江銀行 統合により発足（南都銀行 1934-06-01 と並び戦時統合の先駆例）
  • 戦後、滋賀県内のコアバンクとして長期上場、滋賀県内シェアを維持・拡大
  • 2015 TSUBASA アライアンス加盟（千葉銀行 主導）★ — システム共同化・海外サポート連携
  • 2019 国連責任銀行原則（UN PRB）採択 ★ — 地銀初の採択行、ESG リーダー枠確立
  • 2022-04 東証市場区分見直し → 東証 PRIME 8366（旧東証一部から移行）

ESG リーダー戦略 ★

  • 国連責任銀行原則（UN PRB）採択（2019、地銀初）：ESG 開示・サステナビリティ目標設定・気候関連財務情報（TCFD）対応の地銀リーダー枠を確立
  • ESG リーダー枠は、地銀再編 / アライアンス交渉での差別化要素となる可能性

TSUBASA の戦略意義：

  • システム共同化：勘定系・周辺システムの共同化で IT コスト削減（単独地銀ではスケール不足）
  • 海外サポート：加盟行のシンガポール / 海外拠点を相互利用、滋賀銀行は単独で海外拠点拡張せずに済む
  • 業務連携：信託・証券・リース等の業務連携で異業種展開の自由度を確保
  • 隣県加盟行 京都銀行 との関係：TSUBASA 同盟内では協力、滋賀県・京都府境界エリアでは競合という二面性

県内地盤戦略

  • 滋賀県 = 内陸製造業大県：自動車部品・電機・素材産業の中堅製造業が集積、これらの main bank ポジション
  • 琵琶湖周辺の特殊性：琵琶湖は京阪神 1400 万人の水源、環境保全融資 / ESG ファイナンスのテーマとして県外への発信材料
  • 京都府北部・大阪府北摂・名古屋圏：滋賀県内のみではボリュームに限界があるため、京都市東部 / 北部 + 大阪府高槻・茨木 + 名古屋圏（米原経由）への展開を強化
  • 県内シェア圧倒だが、滋賀県は京阪神ベッドタウン化（南部・大津・草津・栗東）と県北部（彦根・長浜・米原）の人口減という二面性

関西・東海境界での競合構造

  • 関西・東海境界エリアの特殊性：滋賀県は西で京都府・南で三重県（鈴鹿越え）・東で岐阜県・北で福井県・西北で京都府丹後と接し、4 府県境界に位置する。京阪神経済圏と東海経済圏（名古屋）の中間に位置する地理特殊性
  • メガバンク（mufg / smfg / mizuho-fg）との競合：京阪神 / 名古屋都市部の大企業案件は メガに流出する構造、滋賀銀行は滋賀県内 + 隣県中堅・中小製造業 main bank ポジションで対抗
  • 関西地銀群との競合：
    • 京都 FG（京都銀行）：京都府全域 + 滋賀県南部 + 大阪府北部、TSUBASA 同盟兼隣県競合の二面性
    • りそな HD（旧大和銀 + あさひ銀、関西みらい銀含む）：関西最大の地銀グループ、大阪府 / 京都府で間接競合
    • 池田泉州 HD：大阪府北摂エリアで間接競合
    • 南都銀行：奈良県地盤、東で県境接触（東大寺・笠置エリア）
  • 東海地銀群との競合：
    • 大垣共立銀行：岐阜県西部、米原経由で県境接触
    • 十六 FG：岐阜県、関ヶ原エリアで間接競合
    • 三重銀行 / 三十三 FG：三重県、鈴鹿越えで間接競合

単独銀行の戦略意義

  • 持株会社化していない点が、近隣 京都 FG（2009 単独 FG 化）・池田泉州 HD（2009 統合）等の関西地銀 FG 化トレンドと対照的（滋賀銀行は監査役会設置会社の単独上場銀行として存続）
  • 単独銀行のメリット：(1) ガバナンス簡素、(2) 銀行本体に経営資源集中、(3) 持株会社化の追加コスト不要
  • TSUBASA 加盟 + UN PRB 採択 で単独銀行のままスケール / ESG 差別化を獲得する独自モデル

地銀再編における位置

  • 滋賀県の単独地銀リーダーとして独立路線維持（メガとの提携・SBI 第 4 メガ構想からは距離）
  • TSUBASA + UN PRB の二本柱で単独維持の戦略を補強
  • ESG リーダー枠は、再編時の交渉力に寄与する可能性

4. 規制・政策

  • 主管：金融庁（FSA）+ 近畿財務局
  • 銀行業規制：銀行法（単独銀行のため銀行持株会社規制は対象外）
  • 国際枠組み参加：国連責任銀行原則（UN PRB）採択（2019、地銀初）、TCFD 提言準拠の気候関連財務情報開示
  • 直近政策論点：
    • 2024〜 日銀政策金利正常化 → 地銀マージン改善（メガより恩恵が相対的に大）
    • 2025〜 地銀再編加速（人口減 + デジタル投資負担）→ 関西地銀群との連携 / 再編判断
    • 単独銀行のまま維持するか、持株会社化（FG 化）へ移行するかの判断
    • ESG / サステナビリティ開示の高度化（GFANZ / NZBA 等の国際イニシアチブ動向）
    • 琵琶湖環境保全融資の制度的位置付け（環境省 / 滋賀県との連携）
