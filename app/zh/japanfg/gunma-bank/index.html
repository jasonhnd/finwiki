群馬銀行 (The Gunma Bank)

群馬銀行 (The Gunma Bank, Ltd.)

TL;DR

群馬県地盤の地銀（東証 PRIME 8334）。1932-09 群馬大同銀行として設立（大同銀+群馬商工銀+黒岩銀 統合）→ 1955 群馬銀行 改称。持株会社化していない単独銀行体制を維持する首都圏 PRIME 上場地銀の代表格。SUBARU 太田工場・上信越自動車道経済圏を背景に自動車関連サプライチェーン融資が重要顧客基盤。2015 TSUBASA アライアンス参加でシステム共同化・海外サポート連携。

1. 会社概要

正式名：株式会社群馬銀行 英名：The Gunma Bank, Ltd. 証券コード：東証 PRIME 8334（1949 上場・市場区分見直しで PRIME 継続） 設立：1932-09（群馬大同銀行として） 現商号：1955 群馬銀行 改称 本店：群馬県前橋市元総社町 機関設計：監査等委員会設置会社 ^[likely] 業態区分：地方銀行（全国地方銀行協会 加盟）/ 単独銀行（持株会社化していない）

持株会社化していない論点

メガバンク 3 行・有力地銀（mebuki-fg / concordia-fg / fukuoka-fg 等）が持株会社化に進む中、群馬銀行は単独銀行体制を維持。これは:

  • 地域専業・単一県地盤で持株会社化の事業多角化メリット小
  • 子会社（リース・カード・コンサル）は銀行直下子会社で十分管理可能
  • 統合・経営統合相手不在（県内で唯一の PRIME 地銀）

2. 事業セグメント・マップ

セグメント特徴
県内法人自動車関連（SUBARU 太田工場系・部品サプライヤー）・製造業集積
首都圏拡大埼玉北部（熊谷・本庄・深谷）・東京北部 営業展開
個人県内シェア最大、住宅ローン・年金口座中心
銀証連携ぐんぎん証券経由の投資信託・債券・ファンドラップ
リース・カードぐんぎんリース / ぐんぎんカードサービス
海外サポート駐在員事務所（中国・NY・香港 等）→ TSUBASA 経由補完
デジタルTSUBASA 共同インフラ + 自行アプリ

営業エリア地理

  • 本拠: 群馬県全域（前橋・高崎・太田・桐生・伊勢崎・館林 等）
  • 隣接県進出: 埼玉県（熊谷・本庄・深谷・川越）/ 栃木県 / 東京都北部 / 神奈川 / 千葉 / 大阪 / 名古屋
  • 特徴: 上信越自動車道経済圏（関越道 + 上信越道）沿線の物流・製造業 → 自動車・部品・電機 サプライチェーン融資

産業基盤特性

  • SUBARU（富士重工業）太田工場: 群馬県太田市 主力組立工場 → 部品サプライヤー数百社が県内集積 → 群馬銀行 法人融資の中核顧客群
  • 上信越自動車道経済圏: 群馬・長野・新潟 物流動脈 → 倉庫・物流 / EC 関連法人融資
  • 観光: 草津・伊香保 等温泉地、富岡製糸場（世界遺産）→ 観光業ローン
  • 農業: こんにゃく・キャベツ・養蚕（歴史）→ 農業法人融資

TSUBASA アライアンス参加

TSUBASA アライアンス = 千葉銀行 主導の地銀広域連携枠組み。参加行: 千葉銀・第四北越銀・中国銀・伊予銀・東邦銀・北洋銀・武蔵野銀・滋賀銀・琉球銀・群馬銀 等 。

群馬銀行参加メリット:

  • 基幹システム共同化（コスト分散）
  • 海外サポート（参加行の海外拠点活用 → 自前で抱える必要なし）
  • 商品・サービス共同開発（投信・ファンドラップ・サイバー対策）
  • 人材交流・研修

4. 規制・政策

  • 主管: 金融庁（FSA）・関東財務局
  • 業界: 全国地方銀行協会（地銀協）加盟・第二地銀協会 非加盟（第 1 地銀）
  • 資本規制: バーゼル III 国内基準行 ^[likely]
  • 預金保険: 預金保険機構 加盟
  • 直近政策論点:
    • 2024〜 日銀政策金利正常化 → 国内預貸利ザヤ改善 → 地銀収益ボーナス
    • 地銀再編圧力（金融庁・公的資金 / 持株会社化推奨）vs 単独銀行維持判断
    • 自動車産業 EV 転換 → 部品サプライチェーン構造変化リスク（SUBARU 系融資先依存度）
