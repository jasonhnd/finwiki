地方公共団体金融機構 (JFM)

地方公共団体金融機構 (JFM)

Wiki route

This entry sits under JapanFG index. Read it against 日本政策金融公庫 (JFC) for peer / contrast context and banking index for the broader system / regulatory boundary.

TL;DR

地方自治体 (都道府県・市区町村) 向け長期融資に特化した地方共同法人。設立 2008-08-01 (地方公営企業等金融機構として) → 2009-06-01 現名称に改組。旧 公営企業金融公庫 (1957-06 設立、2008-09-30 廃止) の業務を承継。出資者は全 47 都道府県 + 全市区町村による地方共同出資 (国出資なし)。貸付金残高は 約 22.7 兆円 (令和6年度末 / 2025-03末、JFM IR) で、地方財政の長期資金供にインフラ。民間メガバンク・地銀との競合 + 補完関係を持ち、災害復興融資・上下水道などの地方公営企業向け融資で独自の役割。

1. 法人構造

正式名: 地方公共団体金融機構 英名: Japan Finance Organization for Municipalities (JFM) 法人形態: 地方共同法人 (特殊法人。地方公共団体金融機構法に基づく) 設立: 2008-08-01 (地方公営企業等金融機構として) 現名称改組: 2009-06-01 (地方公共団体金融機構に名称変更、業務範囲拡大) 本店: 東京都千代田区麹町 出資者: 全 47 都道府県 + 全市区町村 (地方共同出資、国出資なし)

機関設計

  • 理事長: 地方公共団体側から選任 (歴代に地方自治体経験者多数)
  • 代表者会議: 全国知事会・全国市長会・全国町村会の代表が監督
  • 経営審議委員会: 外部有識者による経営チェック

業務範囲

業務内容
地方債引受都道府県・市区町村の地方債を長期低利で引受・運用
地方公営企業向け融資上下水道・電気・交通・病院などの地方公営企業向け長期融資
災害復興融資大震災・台風・地震などの災害復興のための地方自治体融資
資金調達政府保証なしで国内外の資本市場から債券 (地方金融機構債) 発行
経営支援地方自治体・地方公営企業の財政分析・経営アドバイス

貸付金残高

  • 令和6年度末 (2025-03末) 約 22.7 兆円 (JFM IR / 地方金融機構債 IR 資料)
  • 地方債引受先として 三大資金供に源の一つ (民間銀行・財政融資資金と並ぶ)

公営企業金融公庫時代 (1957-2008)

年月事象
1957-06公営企業金融公庫 設立 (中央政府による地方自治体向け融資、国 100% 出資の特殊法人)
1957-2008上下水道・交通・電気などの地方公営企業に低利長期融資を提供
1980s-90s地方財政膨張期に融資残高拡大、地方インフラ整備の主要資金供に源
2001-12小泉政権の特殊法人改革 → 公営企業金融公庫の廃止方針決定
2008-08-01地方公営企業等金融機構 設立 (前身組織、地方共同出資へ移行)
2008-09-30公営企業金融公庫 廃止 → 業務・資産を新機構が承継

地方公共団体金融機構時代 (2009-現在)

年月事象
2009-06-01地方公共団体金融機構に改組 (業務範囲を地方公営企業から一般地方自治体に拡大)
2011-03-11東日本大震災 → 災害復興融資の主要担い手の一つ
2016-04-14熊本地震 → 災害復興融資対応
2018-07西日本豪雨 → 災害復興融資対応
2019-10令和元年東日本台風 → 災害復興融資対応
2024-01-01能登半島地震 → 災害復興融資対応

3. 事業セグメント map

セグメント主要対象特徴
一般地方債引受都道府県・市区町村民間銀行・財政融資資金と競合 + 補完
上下水道融資地方公営企業長期低利 (民間より低金利)
交通融資地方公営交通 (バス・地下鉄)採算性低い公共交通維持
病院融資公立病院・地方独立行政法人地域医療維持の長期資金
電気・ガス融資地方公営電気・ガス事業一部地方の独立公営事業
災害復興融資被災自治体・公営企業大震災・台風・地震対応特別融資
資金調達地方金融機構債 (JFM 債)政府保証なし。発行体格付 Moody’s A1 / S&P A+ / R&I AA+ (いずれも安定的、JFM IR)

競合・補完関係

地方自治体・地方公営企業の資金調達
  ├── 地方公共団体金融機構 (JFM) ── 長期低利、政策金融
  ├── 財政融資資金 (財務省) ── 国が直接融資、長期低利
  ├── 民間金融機関
  │     ├── [[mufg]] ── 大都市自治体・大規模地方公営企業
  │     ├── [[smfg]] ── 大都市自治体・関西圏自治体
  │     ├── [[mizuho-fg]] ── 大都市自治体・伝統的地方自治体融資
  │     └── 地銀・第二地銀 ── 地元自治体メインバンク
  └── 資本市場 (地方債発行) ── 大都市は市場調達増加傾向

公営企業金融公庫廃止 → 地方共同方式への移行背景

  • 小泉政権特殊法人改革 (2001-) → 国出資型政策金融機関の整理対象
  • 2008-09-30 公営企業金融公庫廃止 → 2008-08-01 新機構設立 で実質的に業務継続

民間金融機関との関係

  • 補完: 採算性低い小規模自治体・公営企業向け長期融資 (民間が積極的でない領域)
  • 金利優位: 地方共同出資 + 高位の発行体格付 (Moody’s A1 / S&P A+ / R&I AA+) による低い調達コストで民間より低金利を提供可能
  • メガバンク側の戦略: mufg smfg mizuho-fg は大都市自治体向けにシンジケートローン・市場性地方債引受で対応

災害復興融資の特殊役割

  • 大震災・台風・地震時に地方自治体・公営企業の資金需要が急増
  • 民間金融機関だけでは対応困難な長期低利融資を JFM が提供
  • 東日本大震災 (2011)・熊本地震 (2016)・能登半島地震 (2024) などで重要な役割

直近政策論点

  • 2024 能登半島地震復興: 地震復興融資の本格対応

5. 監督・規制

  • 根拠法: 地方公共団体金融機構法 (2007 年制定、2008 施行)
  • 監督: 地方公共団体金融機構代表者会議 (全国知事会・市長会・町村会代表) + 経営審議委員会
  • 会計検査: 会計検査院による検査対象 (地方共同法人として)
  • 格付け: Moody’s A1 / S&P A+ / R&I AA+ (いずれも安定的、2024-2025、JFM IR)。政府保証なしながら地方共同出資による信用力で国内公共債発行機関では最高位水準

公的金融機関との関係

  • 日本政策金融公庫 (JFC): 中小企業・農林漁業向け → JFM とは融資対象が異なる
  • 日本政策投資銀行 (DBJ): 民間大企業・インフラ向け → JFM とは融資対象が異なる
  • 国際協力銀行 (JBIC): 海外向け → JFM は国内地方自治体向けで明確に役割分担
