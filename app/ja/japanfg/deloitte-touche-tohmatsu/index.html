有限責任監査法人トーマツ (Deloitte Touche Tohmatsu)

有限責任監査法人トーマツ (Deloitte Touche Tohmatsu)

Wiki route

This entry sits under JapanFG index. Read it alongside EY 新日本 and PwC あらた for peer Big 4 audit context, and 金融ライセンス INDEX for 公認会計士法 / 監査品質管理の規制境界。CPAAOB 監督文脈は 金融庁 監督局 と 金融庁 総合政策局 の所管。

1. 法人概要

  • 正式名：有限責任監査法人トーマツ
  • 英名：Deloitte Touche Tohmatsu LLC
  • 法人形態：有限責任監査法人
  • 本部：東京都千代田区丸の内
  • 加盟ネットワーク：Deloitte Touche Tohmatsu Limited（DTTL）（ロンドン本部）
  • グループ法人：
    • デロイト トーマツ コンサルティング合同会社（戦略 / コンサル）
    • デロイト トーマツ ファイナンシャルアドバイザリー合同会社（FAS / M&A）
    • デロイト トーマツ 税理士法人（税務）
    • デロイト トーマツ リスクアドバイザリー合同会社（リスク / サイバー）

2. 沿革

年月事象
1968等松・青木監査法人 設立（トーマツの直接前身、トーマツ＝等松青木の略）
1975Touche Ross 国際提携
1989Touche Ross + Deloitte Haskins & Sells グローバル合併 → Deloitte Touche Tohmatsu 形成（社名にトーマツが組み込まれた点で日本系の存在感）
2008-07公認会計士法改正により有限責任監査法人化
2003DTT のコンサル部門（旧 デロイト トーマツ コンサルティング）が監査法人トーマツと資本関係を解消し DTT を離脱 → ブラクストン（2003-01）→ アビームコンサルティング（2003-11）として独立。現在の「デロイト トーマツ コンサルティング」は別系統（旧 トーマツ コンサルティングが 2008-07 改称）で、アビームとは直接の関係なし
2020 頃〜デジタルアセット監査 / Travel Rule コンサル先行参入

銀行 / 保険 / 証券

  • トーマツは大手証券・運用・生損保・地銀の監査に関与するが、個別会社の最新監査人は EDINET 一次資料で要確認であり、当エントリでは具体的クライアント名の列挙を行わない

暗号資産 / デジタルアセット

  • ブロックチェーン・暗号資産アドバイザリー グループ
  • 国内クリプト監査ランドスケープ で言及される通り、FSA 登録 VASP の 財務諸表監査 + 分別管理監査 で先行
  • JICPA 業種別委員会実務指針第 61 号(暗号資産関連事業者の監査) 策定に関与
  • 国内大手暗号資産取引所（bitFlyer 系含む公開関連事例）

Travel Rule / AML コンサル

  • JVCEA Travel Rule 実装 に関するコンサルティング
  • AML / 制裁スクリーニング助言
  • リスクアドバイザリー部門（リスクアドバイザリー合同会社）連携

M&A / FAS（監査と分離）

  • デロイト トーマツ ファイナンシャルアドバイザリー（DTFA）が M&A 案件アドバイザリー

4. 主要 listed FG クライアント関係（公開ベース）

トーマツは大手金融関係先の監査に広く関与するが、個別会社の監査人は時点で変動するため、当エントリでは検証可能な範囲に限定する：

  • 大手 VASP — bitFlyer 系等の関連監査実績（公開ベース言及あり）

注：個別会社の監査人は時点で変動。最新は EDINET 各社最新有価証券報告書「監査人の状況」 または JICPA 検索 を一次ソースとして確認。

5. 規制・監督

  • 主管法令：公認会計士法
  • 登録：日本公認会計士協会（JICPA）
  • 監督：金融庁 公認会計士・監査審査会（CPAAOB） が品質管理レビュー / 検査
  • CPAAOB 検査結果：年次「公認会計士・監査審査会の活動状況」報告書で大手監査法人検査結果概要を公表

過去の処分歴

  • 個別の行政処分歴は CPAAOB 検査結果・金融庁公表ベースで都度確認すべき
