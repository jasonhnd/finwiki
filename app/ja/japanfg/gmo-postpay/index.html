GMO後払い (GMO Postpay)

GMO後払い (GMO Postpay)

Wiki route

This entry sits under the JapanFG index as a PSP-side BNPL product. Read it against the Japan BNPL landscape system note, with the GMO ペイメントゲートウェイ sibling explaining the merchant-acquiring stack and the Net Protections HD / Paidy peers showing pure-play BNPL. The BNPL and credit-purchase boundary frames the regulatory question, the METI credit-purchase registry anchors registration, and the Japan payment scheme economics matrix places it next to card / code economics.

TL;DR

GMO後払い は GMO ペイメントサービス株式会社 (GMO インターネット G、GMO ペイメントゲートウェイ の関連会社) が運営する EC 後払い決済。PSP 出身グループが BNPL を内製・運用している事例で、加盟店契約 / 与信 / 請求書発行 / 集金代行を一括で提供する。Net Protections の NP 後払い、ヤマトの クロネコ代金後払い と並ぶ EC 後払いの主要選択肢。

2. Business role

  • GMO ペイメントゲートウェイ (GMO PG) が提供する カード決済 PSP ラインに対し、GMO ペイメントサービス は 後払い と コンビニ収納代行 を担う兄弟ライン。EC 加盟店は GMO 1 社で多様な決済手段をワンストップで提供できる。
  • 与信判断 + 請求書発行 + 督促 / 回収を加盟店側に代行することで、純粋 PSP に与信機能を上乗せした PSP 系 BNPL カテゴリの代表例。
  • BtoC EC（特に通販・サブスク EC）が中核セグメント。

3. Why this standalone page matters

GMO ペイメントゲートウェイ 本体ページは カード決済 PSP が主題のため、後払い事業は別エンティティ (GMO ペイメントサービス) として独立ページを置くことで、PSP / 後払い / コンビニ収納 のライセンス境界が明示できる。

4. 規制・政策

  • 資金決済法 隣接: 収納代行業 / 前払式支払手段の境界が PSP 系 BNPL では複雑化しやすい。
  • 個人情報保護法: 与信データと EC 購買履歴の取扱い。
