日本カストディ銀行 (CBJ)

日本カストディ銀行 (CBJ)

Wiki route

This entry sits under JapanFG index. Read it against 日本マスタートラスト信託銀行 for peer / contrast context and banking index for the broader system / regulatory boundary.

TL;DR

日本最大級の信託カストディ専業銀行（信託財産総額 約 476 兆円、口座管理業務等を含む総預かり資産 約 696 兆円、いずれも 2025-03 末時点）。2020-07-27 日本トラスティ・サービス信託銀行 (JTSB) + 資産管理サービス信託銀行 (TCSB) 合併発足。sumitomo-mitsui-trust（三井住友トラストグループ）33.3% + mizuho-fg 27.0% + りそな銀行 16.7% + dai-ichi-life 第一生命 8.0% + 朝日生命 5.0% + 明治安田生命 4.5% + かんぽ生命 3.5% + 富国生命 2.0% の合弁（2020-07-27 現在）。上場企業の主要株主欄に「日本カストディ銀行 (信託口)」として常連登場 ★。日本マスタートラスト信託 master-trust-bank（MUFG mufg + nippon-life 系）と並ぶ国内 2 強の信託口株主（証券保管 2 強体制）。

重要性 ★

  • **「日本カストディ銀行 (信託口)」**として日本の上場企業ほぼ全社の主要株主欄に常連登場
  • 日本マスタートラスト信託 master-trust-bank と並ぶ「信託口株主」の 2 強
  • 信託財産総額 約 476 兆円、口座管理業務等を含む総預かり資産 約 696 兆円（いずれも 2025-03 末時点、CBJ ディスクロージャー誌 2025）
  • 国内年金（厚生年金 / 私的年金 / 確定拠出年金 等）の中央保管・運営機関の一翼

2. 事業セグメント・マップ

セグメント主要事業特徴
信託カストディ上場株式・債券・投信の保管信託口名義での議決権行使
年金信託厚生年金基金・確定給付企業年金・確定拠出年金信託受託・運用報告・給付事務
投資信託受託公募投信・私募投信の信託契約投信協会会員ファンドの保管・計算
特定金銭信託 (特金)機関投資家・事業法人の運用ビークル受託・運営
資産管理サービス機関投資家向け Master Custody投資指図受託・運用報告

専業ポジション

  • 信託カストディ + 年金信託 専業：商業銀行業務・証券業務・小売は持たない
  • 親会社の SuMi TRUST sumitomo-mitsui-trust / みずほ信託 mizuho-fg がフロント（運用・営業）、CBJ がバックオフィス（保管・計算・事務）という棲み分け

競合・国内 2 強体制

  • マスタートラスト信託銀行 master-trust-bank（MUFG mufg + nippon-life 日本生命 系）：最大競合
  • 日本の証券保管・信託口株主は CBJ + マスタートラスト の 2 強体制
  • 両者で日本の機関投資家・年金資産の保管バックオフィスを実質寡占

信託口株主としての影響力

  • 上場企業の主要株主欄に「日本カストディ銀行 (信託口)」として登場
  • 議決権行使は運用機関（信託元本所有者の指図）に従うのが原則（CBJ 自身は議決権を独立行使しない）

統合シナジー

  • JTSB + TCSB 合併（2020-07）の主要目的：システム統合・規模効果・運営コスト削減
  • 親会社別の重複していたカストディシステム / 投信計算システムを CBJ で一元化

4. 規制・政策

  • 主管：金融庁（FSA）
  • 法的位置付け：銀行法 + 信託業法（信託兼営）
  • 預金保険：預金保険機構加盟（信託カストディ専業のため一般預金者向け預金は限定的）
  • 間接規制：信託協会・投資信託協会・日本証券業協会
  • 直近政策論点：
    • 2024〜 NISA 拡充 → 投信保管資産の増加
    • 2025〜 確定拠出年金制度改革（拠出限度額拡大）→ 信託資産の拡大圧力
