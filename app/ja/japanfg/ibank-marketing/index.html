iBank マーケティング (iBank Marketing)

iBank マーケティング (iBank Marketing)

Wiki route

This entry sits under the JapanFG index as the FG-subsidiary fintech / wallet-app operating-company anchor for ふくおか FG (FFG). Read it against parent operating bank 福岡銀行 / 十八親和銀行 / 熊本銀行 and the wallet-app peer PayPay FG / au フィナンシャル HD / 楽天 FG context. For broader system route see regional-bank API / digital partnership route and banking index.

TL;DR

株式会社 iBank マーケティングは、ふくおか FG (FFG) 100% 子会社の地銀発フィンテック子会社で、Wallet+ (ウォレットプラス) スマホアプリの企画・運営を担当。地銀発のスマホファースト個人金融サービスとして、福岡銀行 / 十八親和銀行 / 熊本銀行 顧客向けに 目的別貯金 (myCoin)・家計管理・プリペイドカード Wallet+ Card・地域連携クーポン 等を提供。地銀フィンテックの代表例で、PayPay / au PAY / 楽天 等の通信系 / EC 系ウォレットとは異なる「銀行口座連携 + 目的別貯金」軸。

iBank マーケティング 設立

  • 2016: FFG が iBank マーケティング 設立 (地銀発フィンテック子会社の先駆例)
  • 「銀行を再発明する」コンセプトで、スマホ完結・若年層向け金融サービスを企画
  • 福岡銀行 口座連携を基本に、家計管理 + 目的別貯金機能を中核に据える

Wallet+ アプリ 展開

  • 2016: Wallet+ アプリ リリース
  • myCoin 機能: 目的別貯金 (旅行・結婚・住宅頭金等の目的を設定し、月次定額で貯金)
  • Wallet+ Card: JCB ブランドのプリペイドカード (チャージ式)
  • 地域連携: 福岡市内・九州各地の店舗・観光地連携クーポン

グループ拡大

  • 2018-2020: 十八親和銀行 / 熊本銀行 連携追加 → 九州 3 県カバー

FG 内ポジション

  • 若年層獲得チャネル: fukuoka-fg 銀行 3 行の若年層 main bank 化の入り口
  • デジタル戦略の入口: みんなの銀行 (デジタルバンク)・FFG 銀行アプリと並ぶデジタル接点
  • 地域経済活性化: 福岡市内・九州各地の店舗連携クーポンで地域内消費循環

競合構図

  • 通信系ウォレット (PayPay / au PAY / d 払い / 楽天ペイ): 全国規模・決済中心、Wallet+ は銀行連携 + 目的別貯金軸で差別化
  • 銀行アプリ (メガバンク・他地銀): UI/UX で差別化、myCoin 等独自機能
  • 家計簿アプリ (マネーフォワード / freee 系個人): 銀行連携範囲は広いが、地域連携 / プリペイドは Wallet+ 独自
  • 他地銀フィンテック: 北國銀行 (CCI グループ) や山陰合同銀行系の地銀発フィンテックと類似 model

強みと弱み

  • 強み: fukuoka-fg 銀行 3 行口座顧客基盤 + 九州地盤の地域連携深さ + 目的別貯金 myCoin の独自性

7. 規制・政策

  • 主管: 金融庁 (FSA) (アプリ事業会社単独としては銀行業免許不要、連携先銀行 fukuoka-bank / プリペイド発行者経由)
  • 規制論点:
    • プリペイド発行業務は資金決済法上の前払式支払手段発行者規制
    • 銀行 API 連携は銀行法上の電子決済等代行業 (連携先銀行が登録)
    • 個人情報保護法 + マイナンバー法 (家計データ取扱い)
    • 反社チェック・マネロン対策 (連携先銀行と共同実施)
