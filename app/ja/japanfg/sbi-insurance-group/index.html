SBI インシュアランスグループ (SBI Insurance Group)

SBI インシュアランスグループ (SBI Insurance Group)

Wiki route

This entry sits under JapanFG index. Read it against SBI証券 (SBI Securities) for peer / contrast context and banking index for the broader system / regulatory boundary.

TL;DR

sbi-hd の保険事業統括中間持株会社。2017-04 設立、ネット直販型保険ビジネスのプラットフォーム。傘下に SBI 損害保険（自動車保険メイン）/ SBI 生命保険（旧 PCA 生命、定期死亡 + 医療）/ SBI いきいき少額短期保険（高齢者向け）/ SBI リスタ少額短期保険（地震費用保険）を擁する。グループ戦略は「米 Geico / Progressive モデルの日本版」= 低価格・ダイレクト型 + SBI 証券・sbi-shinsei-bank・住信 SBI ネット銀とのクロスセル。2017-09 東証マザーズ上場 → 2023-09 SBI HD による完全子会社化（TOB スクイーズアウト）で再非公開化。

1. 会社概要

正式名: 株式会社 SBI インシュアランスグループ 英名: SBI Insurance Group Co., Ltd. 設立: 2017-04-03（SBI HD の保険事業統括会社として） 本社: 東京都港区六本木 1-6-1（泉ガーデンタワー、SBI HD と同所） 親会社: sbi-hd（100% 完全子会社） 上場履歴: 2018-09-27 東証マザーズ 上場 → 2022-04 東証市場再編で 東証グロース市場へ移行（証券コード 7326、現在も上場継続）。sbi-hd は連結子会社として保有（2025-03-31 時点で議決権比率 59.67%、完全子会社化は未実施） 業態: 保険持株会社（少額短期保険持株 + 損害保険・生命保険子会社統括）

主要子会社

  └── SBI インシュアランスグループ（中間持株・100%）

        ├── SBI リスタ少額短期保険（100%）── 地震費用保険専業

2. 事業セグメント・マップ

子会社業態主力商品特徴

SBI 生命保険生保定期死亡 + 医療 + 就業不能旧 PCA、シンプル保障特化
SBI いきいき少短少短高齢者医療・葬儀シニア層特化
SBI リスタ少短少短地震費用保険ニッチ・地震災害特化

コアコンセプト

  • ネット直販保険（ダイレクト型）= 代理店コスト排除 → 低価格訴求
  • 米 Geico / Progressive モデル: 自動車保険を起点に多商品クロスセル
  • SBI 経済圏クロスセル: SBI 証券 1300 万口座 + 住信 SBI ネット銀 + sbi-shinsei-bank の顧客基盤共有
  • 4 商品群（損保 + 生保 + 高齢者少短 + 地震少短）でライフサイクル全領域カバー

提携・買収戦略

  • 完全子会社化（2023-09）: 上場維持コスト削減 + 経営判断スピード向上 + sbi-hd 経済圏一体運用
  • SBI 経済圏内連携: 証券口座開設時に保険提案、新生銀ローン契約に火災保険セット 等のシナジー設計

4. 規制・政策

  • 主管: 金融庁（FSA）
  • 規制法: 保険業法（損保・生保子会社）+ 保険業法第 272 条（少額短期保険業者）
  • 健全性: ソルベンシー・マージン比率（連結 + 個別）
  • 2024〜 政策論点:
    • 個人情報保護（ネット直販ゆえデータ依存度高）
    • サイバーセキュリティ（金融庁モニタリング強化）
