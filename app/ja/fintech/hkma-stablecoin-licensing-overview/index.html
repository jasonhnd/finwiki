HKMA ステーブルコイン・ライセンス · 香港 Stablecoin Licensing Regime 概観

Wiki route

[!info] TL;DR 香港 HKMA(Hong Kong Monetary Authority)ステーブルコイン立法は 2025-08 採択、アジア初の完成された SC ライセンス・フレームワーク。初回審査結果は 2026-05-21 前後に公表予定(メディアの推定、HKMA は日付を公式確認していない);9 社申請者のうち市場予測では実際 2 件のみ(HSBC + StanChart Anchorpoint)、その他は遅延。HK ライセンス = 三円 MRA 第三圏候補 + 中国本土で唯一取得可能なルート。

Key facts

  • HKMA SC 立法:2025-08 採択
  • 香港 HKD-stable + USD-stable の二軌ライセンス
  • 京東 JD-HKD = 中国本土系大手による香港での初の SC 発行申請
  • 準備金 100%(HKD / USD 短期国債 + 中銀預金)+ 第三者カストディ + 月次監査
  • HKD 100M+ 資本要件

Mechanism / How it works

HKMA SC ライセンスのコア要件:100% 準備金(HKD / USD 短期国債 + 中銀預金)+ 準備金第三者カストディ + 月次監査 + 発行体が HK にエンティティ + HKD 100M+ 資本要件 + KYC + AML/CFT + Travel Rule + OFAC SDN との協調(間接的に USD 準備金経由)+ PBoC / e-CNY との境界画定(中国本土リテール顧客非受入)。香港 VASP / OTC 側のもう一つの規制軸は HK SFC VASP ライセンス概観 を参照。

Origin & evolution

HKMA は 2022-12 に SC consultation を開始、2023-12 第二回 CP で Sandbox リストを確立。2025-08 Stablecoin Ordinance 採択(立法に約 3 年)。2026-05-21 初回決定が重要なシグナルを発する:HKMA は「低政治リスク + 高コンプライアンス背景」を持つグローバル大手銀行(HSBC / StanChart)を優先発行、中国本土系大手 + グレー市場プレイヤーは遅延。準備金 FRTB 資本計量フレームワークは HK FRTB 準備金概観、アジア全景比較は グローバル VASP 規制比較マトリクス を参照。
