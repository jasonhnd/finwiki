金融商品取引業 関東財務局長（金商）第123号 · JVCEA 会員 #1037（第一種 · 暗号資産デリバティブ取引のみ）

1. 法人・株主

  • 商号：トレイダーズ証券株式会社（英文名 TRADERS SECURITIES CO., LTD.）
  • 設立：平成11年11月（1999年11月）東京都港区南麻布にて資本金1億5,000万円で設立
  • 本店：〒150-6028 東京都渋谷区恵比寿4丁目20-3 恵比寿ガーデンプレイスタワー28階（令和5年4月移転）
  • 親会社：トレイダーズホールディングス株式会社（東証スタンダード市場上場 · 証券コード 8704）
    • 平成18年10月に持株会社制へ移行。トレイダーズ証券は100%子会社として証券・外国為替取引事業を承継
  • 資本金：2,324,285,000円（平成30年2月現在）
  • 従業員数：72名（令和7年3月31日現在）

2. 牌照・登録状況

  • 金融商品取引業：関東財務局長（金商）第123号（第一種・第二種・投資助言代理業）
  • 暗号資産関連：JVCEA 第一種会員 #1037 — 暗号資産デリバティブ取引のみ。現物の暗号資産交換業（FSA 暗号資産交換業登録）は保有していない。
  • 加入協会：日本証券業協会 / 一般社団法人 金融先物取引業協会 / 一般社団法人 第二種金融商品取引業協会 / 一般社団法人 資産運用業協会 / 一般社団法人 日本暗号資産等取引業協会（JVCEA）/ 日本投資者保護基金

3. 取扱通貨

  • 暗号資産デリバティブ取引（「みんなのコイン」「LIGHT FX コイン」）にて取扱い。同社ウェブサイト上のサービスページ（min-fx.jp）では BTC・ETH・XRP・LTC・BCH の主要5銘柄が言及されている。
  • 現物取扱なし。全取引がレバレッジ2倍上限の CFD（暗号資産証拠金取引）。
  • 銘柄数の公式明示なし（公表なし）。

4. 業務範囲

  • 主業（FX）：みんなのFX / LIGHT FX — 国内 FX 専業に近いポジション。FX が売上の大半を占める。
  • 暗号資産CFD：みんなのコイン（令和4年1月 2022-01 開始）/ LIGHT FX コイン（同時開始）。現物保有なし、全額信託保全対応、レバレッジ2倍。
  • その他：みんなのシストレ（自動売買）、みんなのオプション（店頭外国為替オプション）
  • 法定通貨入金：銀行振込（三菱UFJ銀行・三井住友銀行・みずほ銀行等）
  • NFT / staking / 貸暗号資産：なし

5. 市占・取引高

  • 口座数（FX 2ブランド合算）：667,900口座（2027年3月期 4月時点 = 2026-04 公表）
  • 預り資産（同）：133,611,648千円（≒1,336億円）（2026-04 公表）
  • 暗号資産 CFD 単独の口座数・取引高は公表なし。
  • 競合内ポジション：FX 中堅証券（みんなのFX は国内 FX 口座数トップ10圏内と推定されるが公式ランキング出典なし）。暗号資産デリバティブは同業13社のうち規模は中位と推定（公表なし）。同業他社には マネックス証券、SBI 証券、SBI FXトレード があり、上場企業の対比は JP 上場 CEX 関連企業マトリクス 参照。
