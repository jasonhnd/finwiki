JBA — 日本ブロックチェーン協会 (Japan Blockchain Association)

JBA — 日本ブロックチェーン協会 (Japan Blockchain Association)

概要

一般社団法人 日本ブロックチェーン協会 (JBA / Japan Blockchain Association) は、日本の暗号資産業界における最古参の業界団体の一つ。前身は 2014-09 設立の「日本価値記録事業者協会 (JADA)」 で、2016-04 に現名称へ改称しブロックチェーン技術全般を所管する協会として再編。会員は VASP・ブロックチェーンインフラ事業者・大学・研究機関を含む。取引所部会は 2018-04 に JCBA と統合し JVCEA を発足。以降 JBA は技術・研究・教育・国際連携に軸足を移し、ブロックチェーン技術団体として活動継続。

沿革

  • 2014-09 — JADA (日本価値記録事業者協会) 設立。Mt.Gox 崩壊直後、業界自主規制の中核として国内 VASP が発足
  • 2016-04 — 「日本ブロックチェーン協会 (JBA)」に改称。ブロックチェーン技術全般を所管する協会へ再編
  • 2017-04 — 改正資金決済法施行 (仮想通貨交換業登録制度) を受け、取引所部会が自主規制機能を強化
  • 2018-04 — JBA 取引所部会 + JCBA → 統合により JVCEA が発足 (FSA 認定自主規制協会)
  • 2018 以降 — JBA は技術団体・研究団体としての性格を強化、JCBA は産業団体、JVCEA は認定自主規制と三分の役割分担が定着

現在の活動領域

  • 技術検証 / R&D：ブロックチェーン技術の標準化・相互運用性検証、ユースケース PoC 共有
  • 教育 / 人材育成：ブロックチェーン教育セミナー、大学・研究機関との連携
  • 国際連携：GBBC・ISO/TC 307 (ブロックチェーン国際標準化) 等への日本側窓口
  • 政策提言：技術中立的観点からの政策レビュー (JCBA とは異なる切り口)
  • 会員向け勉強会：DeFi・L2・ZK・ステーブルコイン技術・ノードオペレーション等の技術討議

組織

  • 法人形態：一般社団法人
  • 本部：東京
  • 代表理事：会員企業 / 有識者から輪番選出 (公式サイトで随時公表)
  • 会員区分：正会員 (法人) / 個人会員 / 賛助会員 / 学術会員

JCBA / JVCEA との棲み分け (三層構造)

団体役割会員中心法的位置付け
**[[exchanges/jvcea-self-regulatory-overviewJVCEA]]**認定自主規制執行FSA 登録 VASP
**[[exchanges/jcba-japan-crypto-business-associationJCBA]]**産業政策・税制提言暗号資産ビジネス全般
JBA技術・研究・教育ブロックチェーン技術コミュニティ一般社団法人 (任意団体)

多くの主要 VASP は 3 団体に重複加盟 (bitFlyer・Coincheck・bitbank 等)。技術団体としての JBA の歴史的価値は、JVCEA 発足前の自主規制ルール形成期に国内 VASP が自律的にアンチマネロン・コールドウォレット運用基準を策定した遺産にある。これは現行 JVCEA Travel Rule 実装 や VASP セキュリティ監査基準 のベースとして継承されている。

著名な歴史的会員 / 代表者

  • 初期 JADA は Mt.Gox 後の国内 VASP (bitFlyer・Coincheck・QUOINE・bitbank 等) が中心
  • 改称後の JBA 代表理事には bitFlyer 加納裕三氏らが歴任
