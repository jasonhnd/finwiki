2. 牌照・登録状況

  • 暗号資産交換業: 近畿財務局長 第00004号（登録日 2021-06-18）
  • JVCEA 第一種会員: #1034（2021年1月に第二種で加入 → 同年6月に第一種へ種別変更）
  • 金融商品取引業・電子決済手段等取引業・資金移動業の登録なし（公表情報の範囲内）
  • 関西所在の暗号資産交換業 VASP は 26 社中 2 社のみ（Zaif: 近畿財務局長 第00001号 と当社、参考は FSA 暗号資産交換業登録制度）

3. 取扱通貨

  • 取扱銘柄数: 3銘柄（2026年5月現在の公表情報）
  • 対応銘柄: BTC（ビットコイン）、ETH（イーサリアム）、ADA（カルダノ）
  • 特色: 通貨の種類は少なく BTC 中心だが、ADA 対応は国内 VASP でも珍しい。ETH は 2022年12月一時停止→2025年12月取り扱い再開の経緯あり。

4. 業務範囲

  • 主力サービス: 暗号資産自動両替機 BTM（Bitcoin Teller Machine）— 日本国内唯一の暗号資産 ATM として展開（2022年8月開始、対比は COINHUB の ATM 業態）
  • BTM: 日本円 ↔ BTC / ETH 購入・売却、ADA 購入のみ対応。1取引最大10万円。
  • WEB 両替サービス: BTM 不要のオンライン両替（24時間受付、BTC / ETH / ADA の購入・売却対応）
  • インバウンド対応: 利用者登録なし・ICAO 準拠 IC パスポートのみで ADA 売却以外の両替が可能（外国人観光客向け）
  • BTM 機体販売: パートナー向けに BTM 機体の販売制度あり
  • 法定通貨入金チャネル: 現金紙幣（ATM 投入）または銀行口座振込
  • margin / derivative / staking / NFT / 貸暗号資産: 公表情報なし

5. 市占・取引高

  • 口座数・取引高は非公表
  • 規模は小規模 niche に分類（従業員15名、資本金2,990万円）
  • BTM 設置台数: 2022年8月に国内1台目（大阪）→ 9月2台目（東京）→ 11月3台目を設置。2026年5月1日時点で一時利用停止中（理由は公表情報の範囲内では未詳）
  • 顧客層: retail（BTM 利用者）＋インバウンド観光客

7. 戦略動向

  • BTM 機体のパートナー販売制度を提供し、他社・他事業者への BTM 設置展開を模索
  • 外国人観光客（インバウンド）向け本人確認不要両替の開始（パスポート照合方式）は、インバウンド需要を取り込む独自路線
  • 2026年5月現在の BTM 一時利用停止は戦略・規制上の意味合いが不明（公表情報の範囲内では未詳）
  • 日本自動販売システム機械工業会への加盟は、BTM を「自動販売機」カテゴリとして業界横断的に展開する意図を示す
