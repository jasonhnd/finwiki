# Agent 主体性辩论：四个阵营


## Wiki route

This entry sits under [[agent-economy/skill-market-monetization|Skill 市场收费化]]. Read it with [[agent-economy/2026-04-08_agent-payment-infrastructure-research|Agent 支付基础设施研究报告]] for adjacent context and [[INDEX|FinWiki index]] for the broader system boundary.

> [!info] TL;DR
> AI Agent 是否构成新型经济主体，存在四个立场阵营：

**阵营 A（已经是）**：参与自发秩序的价格形成即构成行动者（哈耶克）。

**阵营 B（制度可赋予）**：技术能力已就绪，但需构成性规则赋予独立责任和信用体系才能成为制度性主体（斯密、科斯、塞尔、德·索托）。

**阵营 C（不是，也不需要是）**：不能独立承受不可逆后果就不是主体（中本聪）。争定义是在逃避找切入点（蒂尔）。不必给它名分但必须治理（韩非子）。

**阵营 D（问题框架有误）**：主体/工具二分法本身需被超越（拉图尔）。Agent 是第一个不需要人类同意就在行动的准实体（赫拉利）。真正的问题是谁拥有 Agent、谁拿走剩余价值（马克思）。

**跨阵营共识**：构建信用评分和身份注册体系，本身就是在为 Agent 创造制度性角色——这是制度设计选择，不是价值中立的技术问题（塞尔）。在 [[agent-economy/skill-market-monetization|Skill 收费化]]推动下，A2A 微交易规模将使这套制度从可选变为必需。

**实用结论**：白皮书不应以争议性定义为论证起点。采用"功能先于定义"立场——Agent 的经济功能先于其法律定义而存在，[[agent-economy/2026-04-08_agent-payment-infrastructure-research|基础设施需求]]不依赖于定义之争的结论。
