# Payward Asia (Kraken Japan) — 日本暗号資産交換業者（dormant / 撤退）overview


## Wiki route

This entry sits under [[exchanges/INDEX|exchanges index]]. Read it against [[exchanges/jp-exchange-coinbase-japan|Coinbase 株式会社 — 日本暗号資産交換業者]] for the closest parallel pattern (外資系 retail exit) and [[exchanges/fsa-vasp-registration-system|FSA 暗号資産交換業登録制度]] for the broader system / regulatory boundary.

> Kraken は日本市場に **2 度参入し 2 度撤退** した稀有な外資系取引所。1 回目 2014–2018、2 回目 2020–2023。現在は dormant。

**foreign-exchange-exit / 2 度の日本市場撤退 / FSA 登録歴あり → 抹消**

## 1. 法人・株主

- 商号（最終）：Payward Asia Inc.（日本支店ベース）／旧 Kraken Japan 部門
- 親会社：Payward, Inc.（米国・サンフランシスコ、ブランド名 Kraken）
  - 創業 2011年、Jesse Powell 創業 CEO
  - 2025-04 NASDAQ 上場準備フェーズ報道（公開アナウンス済）
- 日本における事業形態：当初 Payward 米国本社からの直接サービス提供（2014–2018）、その後 Payward Asia による現地登録法人での再進出（2020–2023）

## 2. 牌照・登録状況（歴史）

- 第 1 期（2014–2018）：FSA 暗号資産交換業登録制度（2017-04 施行）前後にサービス停止。登録未取得のまま日本居住者向けサービスを停止
- 第 2 期（2020–2023）：Payward Asia 名義で関東財務局 暗号資産交換業者として登録（2020 年中）。JVCEA 第一種会員加入
- 第 3 期（2023-01 以降）：日本撤退発表。FSA 登録抹消手続実施。JVCEA 退会
- 2026-04 snapshot 時点：FSA 登録一覧に Payward Asia の記載なし（抹消確認）

## 3. 取扱通貨

- 第 1 期サービス停止前：BTC、ETH、BCH、LTC、XRP 等を中心に米国本社準拠ラインナップ
- 第 2 期再参入時：日本向けに上場銘柄を絞り込み、JVCEA ホワイトリスト準拠の主要銘柄に限定
- 撤退後：取扱い銘柄ゼロ（dormant）

## 4. 業務範囲

- 現物 spot 取引のみ（リテール向け）。デリバティブ・margin・staking の公開情報は限定的
- 法定通貨入金：銀行振込（JPY）
- 第 2 期は OTC や法人サービスへの拡張は限定的にとどまり、リテール撤退とともに全面停止

## 5. 市占・取引高

- 公開数値なし。両期とも口座数・取引高ともに国内市場で目立つシェアを獲得することはなく、外資系として bitFlyer / Coincheck / GMO コインなど国内大手の後塵を拝した
- 顧客層：日本居住者リテール中心（法人向けは限定的）
- 業界的位置付け：外資系取引所の日本市場撤退パターンとして [[exchanges/jp-exchange-coinbase-japan|Coinbase Japan]] と並んで論評されることが多い

## 6. 沿革・重大事件

| 時期 | 事項 |
|------|------|
| 2014 | Kraken（米国本社直接運営）が日本居住者向けサービス提供開始 |
| 2017-04 | 改正資金決済法施行（FSA 暗号資産交換業登録制度成立） |
| 2018-04 | Kraken Global Blog で「日本居住者向けサービスを 2018-06 までに停止」と発表。コスト高・規制環境を理由に挙げる |
| 2018-06 | 第 1 期サービス完全停止。日本居住者の口座閉鎖 |
| 2019–2020 | 日本再参入準備期間。Payward Asia 名義で FSA 登録申請 |
| 2020 年中 | Payward Asia として関東財務局 暗号資産交換業者 登録完了。JVCEA 第一種会員加入 |
| 2020–2022 | 第 2 期サービス運営。BTC/ETH 等主要銘柄でリテール spot 提供 |
| 2023-01 | Kraken 本社が日本事業の撤退を正式発表（背景：暗号資産市場低迷、グローバル人員削減 30%、規制環境負担） |
| 2023-04 | 第 2 期サービス停止・顧客資産返還完了。FSA 登録抹消手続実施 |
| 2026-04 | FSA 登録一覧 snapshot に Payward Asia 不在を確認（dormant 状態継続） |

## 7. 戦略動向

第 2 期撤退（2023-01 発表）は、親会社 Payward の以下の戦略要因によるもの：

1. 2022 年の暗号資産市場低迷とグローバルコスト削減（人員削減 30%）
2. 日本市場における外資系の上場銘柄制約（JVCEA ホワイトリスト準拠）と KYC/AML 対応コスト
3. 米国本拠地での SEC 監督強化と国内市場集中の戦略選択

撤退後、Payward 本社は **2025–2026 年に NASDAQ 上場準備** を進めており、日本再々参入の公開アナウンスは 2026-05 時点で存在しない。日本市場でアクセス可能な外資系 spot CEX は [[exchanges/jp-exchange-binance-japan|Binance Japan]] と [[exchanges/jp-exchange-bittrade|BitTrade]] に限定される。

**業界的位置付け**：[[exchanges/jp-exchange-coinbase-japan|Coinbase Japan]]（2023-02 撤退）と並ぶ「外資系 retail CEX 撤退パターン」の代表例。両社の同時期撤退は、JVCEA ホワイトリスト・FSA 規制負担・実名口座要件（[[exchanges/jvcea-self-regulatory-overview|JVCEA 自主規制]] 参照）が外資系参入の構造的バリアーとなったことを示す。比較対照は [[exchanges/jp-vasp-ma-consolidation-history|JP VASP M&A コンソリデーション史]] 参照。

## 8. 関連人物

- **Jesse Powell**：Payward / Kraken 創業者・元 CEO（2023-09 退任）。2018 年第 1 期撤退時の日本市場発信は Powell 自身が複数行った
- **Dave Ripley**：Payward 現 CEO（2023-09 就任）。2023 年第 2 期撤退決定時の経営トップ
- 日本法人代表取締役：公開情報なし（2026-05 時点で公式サイト消失）
