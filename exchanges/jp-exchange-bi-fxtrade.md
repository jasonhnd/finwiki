# SBI FXトレード株式会社（BI FXTRADE）— 日本暗号資産デリバティブ業者 overview

> FSA 関東財務局長（金商）第2635号 · JVCEA 第一種会員 #1026（**暗号資産デリバティブ取引のみ**） · 設立 2011年11月

## 1. 法人・株主

- **商号（日）**: SBI FXトレード株式会社（JVCEA 登録名: BI FXTRADE Co., Ltd.）
- **商号（英）**: SBI FXTRADE Co., Ltd.
- **設立**: 2011年11月
- **本店**: 〒106-6019 東京都港区六本木1-6-1 泉ガーデンタワー（SBI VC トレード・SBI 証券と同住所）
- **資本金等**: 9億6,000万円（資本準備金含む）
- **株主**: SBIリクイディティ・マーケット株式会社 100%（SBI HD 連結子会社）
- **上場状態**: 非上場（SBI HD グループ完全子会社）

**注記**: JVCEA 会員一覧では "BI FXTRADE Co., Ltd." と表記されているが、コーポレートサイト・金商登録（第2635号）・所在地（泉ガーデンタワー）が一致しており、SBI FXトレード株式会社と同一法人と確認される。

## 2. 牌照・登録状況

- **金融商品取引業**: 関東財務局長（金商）第2635号（第一種金融商品取引業）
- **暗号資産交換業（FSA）**: **登録なし**（現物取引・交換業ライセンス保有せず）
- **JVCEA 第一種会員 #1026**: 暗号資産デリバティブ取引（Cryptocurrency derivatives trading）のみ
- **加入協会**: 一般社団法人金融先物取引業協会（会員番号 1588）、一般社団法人日本暗号資産等取引業協会（会員番号 1026）

## 3. 取扱通貨

- **取扱銘柄数**: 6銘柄（デリバティブのみ。現物取引なし）
- **銘柄**: BTC/JPY, XRP/JPY, ETH/JPY, BTC/USD, XRP/USD, ETH/USD
- **特色**: 対円・対ドル双方を提供する暗号資産 CFD（差金決済）。現物保有・ウォレットなし。FX 主業者による暗号資産デリバ付加サービスの位置付け。

## 4. 業務範囲

同社は FX（外国為替証拠金取引）専業会社であり、暗号資産は FX と同一口座基盤に付加したデリバティブサービス（**SBIビットコインFX**）として提供する。

| サービス | 提供状況 |
|---------|---------|
| FX（店頭外国為替証拠金取引） | ✓ 主業・34通貨ペア |
| つみたて外貨（積立 FX） | ✓ 外貨積立サービス |
| SBIビットコインFX（暗号資産 CFD） | ✓ 6銘柄・レバレッジ最大2倍（個人） |
| 暗号資産現物取引 | ✗ なし |
| NEOBANK | ✓ 銀行代理業（住信SBIネット銀行） |
| 貸暗号資産・NFT・staking | ✗ なし |
| 電子決済手段等取引業 | ✗ なし |

**入金チャネル（暗号資産 CFD）**: クイック入金（住信SBIネット銀行、楽天銀行、三菱UFJ銀行、三井住友銀行、ゆうちょ銀行、みずほ銀行、PayPay銀行）、SBI FXトレード NEOBANK からの即時入金。**コンビニ払い・仮想通貨入金なし**。

**暗号資産 CFD の主要スペック**: 取引種類＝店頭暗号資産証拠金取引、レバレッジ＝個人口座2倍（法人は別途）、ロスカット水準＝証拠金維持率50%以下、取引手数料・ロスカット手数料・口座維持費 すべて無料、ファンディングレート変動制。

## 5. 市占・取引高

- **取引高・口座数**: 公表なし（暗号資産 CFD 単体の定期開示なし）
- **顧客層**: 主に FX 既存顧客（リテール中心）。機関投資家向け専用プログラムの記載なし。
- **ポジション**: 暗号資産 CFD の取引高ランキング公表なし。FX 部門では「スプレッド業界最狭水準」「オリコン顧客満足度 FX 初心者部門 第1位」を訴求。FX 専業かつ SBI グループの知名度を背景に、主に小売 FX トレーダー層が顧客基盤。

## 7. 戦略動向

- **2026年4月**: 「外貨宅配サービス」開始（FX 口座顧客向けの外貨現金配達）
- **2026年4月**: 設立15周年特設ページ公開、周年キャンペーン展開
- **2026年5月**: 新通貨ペア（トルコリラ/円）の基準スプレッド縮小（1.28銭）
- 暗号資産 CFD は 2020年8月開始以降、銘柄拡充（BTC→XRP→ETH の順）は完了済みで、対ドル銘柄も追加済み。直近12ヶ月で暗号資産部門の新規拡張発表は確認されず。
- SBI グループ全体の暗号資産戦略はグループ統括会社（SBI HD）および [[exchanges/jp-exchange-sbi-vc-trade|SBI VCトレード]] が主導しており、同社はリテール FX 基盤での補完的位置付け。
