# ビットトレード株式会社 — 日本暗号資産交換業者 overview

## 1. 法人・株主

- **商号**: ビットトレード株式会社（英文: BitTrade Inc.）
  - 旧商号: ビットトレード株式会社 → フォビジャパン株式会社（2018-12-17 改称）→ ビットトレード株式会社（2023-04-21 改称）
- **設立**: 2016年9月
- **本店所在地**: 〒105-0003 東京都港区西新橋1-1-1
- **資本金**: 1億円（2023年2月末時点）
- **上場状態**: 非上場
- **主要株主（公表分）**:
  - SINOHOPE SG PTE. LTD.（筆頭とみられる）
  - Sinohope Technology Holdings Limited
  - 東海東京フィナンシャル・ホールディングス株式会社
  - 株式会社FPG

## 2. 牌照・登録状況

- **暗号資産交換業**: 関東財務局長 第00007号
- **第一種金融商品取引業**: 関東財務局長（金商）第3295号
- **JVCEA 会員**: 第一種会員 #1007（暗号資産交換業 + 暗号資産デリバティブ取引 両方対象）
- **JCBA 会員**: 一般社団法人日本暗号資産ビジネス協会 正会員
- **Fintech 協会**: ベンチャー会員
- **日本STO協会 / 日本セキュリティトークン協会**: 賛助会員

## 3. 取扱通貨

- **取扱銘柄数**: 全49銘柄（2026年5月現在）
  - ※ 2026年5月8日に BSV（Bitcoin SV）取扱廃止予定通知あり
- **主要銘柄（例示）**: BTC / ETH / XRP / SOL / ADA / DOGE / BNB / DOT / LTC / BCH / LINK / TON / APT / SUI / MATIC / ARB / OP 他
- **特色**:
  - アルトコインラインナップが国内最多水準（取引開始当初「銘柄数国内No.1」と標榜）
  - 独自・プロジェクト系銘柄（JOC, UPC, CRTS, ZPG など）も複数取り扱い
  - IEO（Initial Exchange Offering）プラットフォームを設置

## 4. 業務範囲

| サービス | 提供状況 |
|---------|---------|
| 販売所（スポット） | ✓ |
| 取引所（板取引） | ✓ |
| レバレッジ取引（CFD） | ✓（第一種金商業の根拠） |
| 積立（定期積立） | ✓ |
| 貸暗号資産 | ✓ |
| IEO | ✓（専用一覧ページあり） |

| API | WebSocket 規格（ミリ秒対応） |

- **法定通貨入金チャネル**: クイック入金（ペイジー除き無料）/ 振込入金 / ペイジー入金 / コンビニ入金
- **取引銀行**: PayPay 銀行・楽天銀行・住信SBIネット銀行・りそな銀行・ゆうちょ銀行・GMOあおぞらネット銀行
- **セキュリティ**: サービス開始以来ハッキング件数ゼロ（公式発表）、暗号資産100%コールドウォレット管理、24時間365日システム監視

## 5. 市占・取引高

- **口座数・取引高**: 公表なし（開示情報に数値記載なし）
- **市場内ポジション**: 銘柄数の豊富さを主な差別化軸とする中堅取引所。JVCEA Type 1 会員26社のうち、暗号資産交換業とデリバティブ取引の双方を保有するのは13社に限られ、同社はその一つ（詳細は [[exchanges/jvcea-self-regulatory-overview|JVCEA 自主規制体系]] / [[exchanges/jp-vasp-derivative-license-system|JP VASP デリバティブライセンス制度]] 参照）。アルトコイン軸の競合は [[exchanges/jp-exchange-bitbank|bitbank]]、フルレンジ業界比較は [[exchanges/jp-exchange-binance-japan|Binance Japan]] が参考になる。
- **顧客層**: 主に個人リテール。法人向けサービスも展開。

## 6. 沿革・重大事件

| 年月 | 事項 |
|------|------|
| 2016年9月 | ビットトレード株式会社として設立 |
| 2017年9月29日 | 暗号資産交換業 登録（関東財務局 第00007号） |
| 2018年12月17日 | Huobi グループと提携強化に伴い、サービスブランドを「Huobi Japan（フォビジャパン）」へ変更 |
| 2019年10月 | 「販売所」サービスリリース。FPG株式会社へ第三者割当増資 |
| 2019年12月 | 東海東京フィナンシャル・ホールディングスと資本業務提携 |
| 2019年4月 | 金融庁・ロイター報道によれば立入検査の実施（行政処分には至っていない模様） |
| 2020年6月 | Huobi Token（HT）を上場 |
| 2023年2月15日 | ブランド名を「ビットトレード」に変更 |
| 2023年4月21日 | 法人名を「ビットトレード株式会社」に正式改称（フォビジャパン株式会社より） |
| 2023年3月31日 | ウェブドメイン変更（huobi.co.jp → bittrade.co.jp） |
| 2025年11月 | サーキットブレーカー制度開始 |
| 2026年5月 | BSV・SXP など一部銘柄の取扱廃止通知 |

**ハック・重大インシデント**: 公式発表によれば設立以来ハッキングなし（公表なし）。

## 7. 戦略動向（直近 12 ヶ月）

- **2025年7月**: JTB・NTTテクノクロスと「推し活」による地域活性化に向けた共同研究開始（プレスリリース）
- **2025年8月**: 株式会社北紡と暗号資産・Web3.0関連ビジネスに関する基本合意書締結
- **2025年11月**: ENI Foundation と Web3.0 領域における協業推進に合意（プレスリリース）
- **2026年1月**: オーストラリア不動産開発会社 Avantaus Development（ADG）と RWA（現実資産）活用の戦略的提携締結
- **IEO 展開**: 国内独自銘柄（JOC 等）の定期情報開示を継続、新規上場申請窓口も公開中
- **第一種金商業活用**: CFD（暗号資産デリバティブ）プラットフォームを提供中。自己資本規制比率を四半期ごとに開示（2026年3月末開示分まで公表済み）
