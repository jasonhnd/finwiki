# JVCEA Type 2 (第二種会員) 制度 — 賛助会員 4 社分析

## 1. 制度概要

JVCEA ([[exchanges/jvcea-self-regulatory-overview]]) は会員区分を 2 種類設定:

- **第一種会員 (Type 1)** — 暗号資産交換業 + 暗号資産デリバ取引業者。FSA 登録必須。
- **第二種会員 (Type 2 / 賛助会員)** — FSA 登録 不要。 (a) FSA 登録準備中、(b) 関連事業者、(c) 機関向け事業者 等 多様な業態を許容。

## 2. Type 2 加盟意義

- **JVCEA 自主規制への準拠表明** — 業界スタンダードへの自発的アライメント
- **業界情報共有 + ロビーアクセス** — FSA / 金融庁 制度動向の早期把握
- **将来の Type 1 移行準備** — 登録申請前の関係構築
- **取引所と機関顧客接続経路** — 第一種会員 + 機関投資家との接続点

- **#2027 サクソバンク証券** (Saxo Bank A/S デンマーク親) — FSA 暗号資産交換業登録ゼロ・加盟理由公表されず・海外 prime broker としての準備?  → [[exchanges/jp-exchange-saxo-bank-japan]]
- **#2033 Gaudiy Financial Labs** (Gaudiy Group · ソニーG + バンダイナムコ + 集英社 + 講談社 134 億累計調達) — Web3 IP ファンエコノミープラットフォーム・NFT 中心  → [[exchanges/jp-exchange-gaudiy]]
- **#2034 Trek Labs Japan** (Backpack グローバル親 · 元 FTX 系 Armani Ferrante 創業) — FSA 登録申請準備中・日本進出初期段階  → [[exchanges/jp-exchange-trek-labs-japan]]
- **#2037 Laser Digital Japan** (野村 HD スイス子 100%) — Komainu Board 連結 ([[exchanges/jp-custody-komainu]])・機関カストディ + 取引基盤  → [[exchanges/jp-exchange-laser-digital-japan]]

## 4. 業界含意

- **海外大手の準備拠点化** — 米/欧/HK 系プレイヤーが Type 2 で「日本市場前哨基地」を構築 (Trek Labs / Saxo / Laser Digital)
- **エコシステム企業** — Web3 / IP / NFT 系 (Gaudiy) も加盟可能、業態の柔軟性
- **Type 1 移行ハードル** — FSA 登録は数億円コスト + 半年〜数年審査。 Type 2 で関係維持しつつ準備

## 5. 比較 — Type 1 vs Type 2

Type 1 が "事業者" であれば Type 2 は "業界エコシステム"。 韓国 KISA 認証や香港 SFC ライセンスのような閾値型 (取得/不取得 の二値) ではなく、業界関与の柔軟さを許容する日本特有の構造。 詳細パレント関係は [[exchanges/jp-vasp-parent-company-map]] 参照。
