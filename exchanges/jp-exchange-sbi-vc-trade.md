# SBI VCトレード株式会社 — 日本暗号資産交換業者 overview

## 1. 法人・株主

- **商号**：SBI VCトレード株式会社
- **英語表記**：SBI VC Trade Co., Ltd.
- **設立**：2017年5月26日
- **本店所在地**：〒106-6021 東京都港区六本木1-6-1 泉ガーデンタワー
- **資本金**：13億5,000万円（資本準備金含む）
- **株主**：SBIグループ 100%（親会社：SBIホールディングス株式会社、東証プライム上場・証券コード 8473）

## 2. 牌照・登録状況

同社は現時点で日本唯一の「3種牌照同時保有業者」であり、暗号資産交換業・金融商品取引業（暗号資産デリバティブ）・電子決済手段等取引業を単独で持つ。

| 業種 | 登録番号 | 備考 |
|------|---------|------|
| 暗号資産交換業 | 関東財務局長 第00011号 | 登録 2017-12-01 |
| 第一種金融商品取引業（暗号資産デリバティブ） | 関東財務局長（金商）第3247号 | OTC 派生対応 |
| 電子決済手段等取引業 | 関東財務局長 第00001号 | 2025-03-04 登録完了、USDC 取扱用 |

- **JVCEA 会員種類**：第一種（会員番号 1011）
- **加入協会**：日本暗号資産等取引業協会（JVCEA）、日本暗号資産ビジネス協会（JCBA）、日本デジタル経済連盟、日本デジタル分散型金融協会、JPCrypto-ISAC

### 電子決済手段 / USDC cross-link

金融庁の電子決済手段等取引業者登録一覧では、SBI VCトレードが関東財務局長 第00001号の登録業者として記載され、取扱電子決済手段は USDC とされている。したがって同社は単なる暗号資産交換業者ではなく、日本の電子決済手段 / stablecoin 実装を読む際の primary route でもある。

Related stablecoin routes: [[fintech/japan-ecisb-license]], [[fintech/japan-stablecoin-regulatory-landscape]], [[fintech/stablecoin-channel-japan-sbi-jpyc-ring]], [[JapanFG/jpyc]], [[JapanFG/progmat]].

## 3. 取扱通貨

2026年5月時点で現物取引対象は約40銘柄。主要銘柄：BTC・ETH・XRP・LTC・BCH・SOL・ADA・DOGE・DOT・LINK・XLM・AVAX・MATIC・SHIB・TON・SUI など。加えて電子決済手段（EP 手段）として **USDC** および JPY 安定性資産（ZPG・DAI）を取扱う。

特色：
- **USDC**（ユーエスディーシー）を 2025年3月25日に国内一般向け取引開始。電子決済手段等取引業登録（第00001号）を生かした日本円⇔USDC スポット交換および **USDCレンディングサービス**（2026年3月18日開始・国内初）を提供。
- **ステーキング対応銘柄**（ETH・SOL 等）あり。
- **NIDT**（ニッポンアイドルトークン）・**FLR**（フレア）・**XDC** など国内他社に比べやや幅広い上場ラインナップ。

## 4. 業務範囲

| サービス | 概要 |
|---------|------|
| 現物販売所 | 対日本円スプレッド取引 |
| 現物取引所（板取引） | Maker/Taker 手数料制、PC 向け VCTRADE Web 提供 |
| レバレッジ取引（OTC デリバティブ） | 最大 2 倍（個人）、JVCEA 定める倍率（法人） |
| 貸コイン（レンディング） | 個人・法人向け暗号資産消費貸借 |
| 積立 | 定額自動積立 |
| ステーキング | ETH 等 PoS 銘柄 |
| SBI Web3 ウォレット | セルフカストディ連携 |
| SBIVC for Prime | 大口・法人向け OTC 専用サービス |
| USDC レンディング | 国内初（2026-03-18 開始）、雑所得扱い |

**法定通貨入金チャネル**：銀行振込・クイック入金（ペイジー等）。口座管理費・年会費・円入出金手数料・暗号資産入出庫手数料は無料。

## 5. 市占・取引高

公開指標：
- **自己資本規制比率** 354.0%（固定化されていない自己資本 76億9,800万円）※2026年3月末時点・同社 IR 開示
- 口座数・月間取引高の公開数値は確認できず（公表なし）

ポジション：3種牌照保有・DMM Bitcoin 顧客資産移管（推定数十万口座規模）・SBI グループ百%子会社という点から、国内大手 5–6 社に位置すると見られるが、取引高・口座数の公的比較ランキングに同社独自の数字は公表されていない。顧客層は個人リテール中心に大口・法人向け Prime サービスを併設。

## 7. 戦略動向

- **3 種牌照構造の戦略的意義**：暗号資産交換業（現物）・金融商品取引業（デリバティブ）・電子決済手段等取引業（ステーブルコイン）を単一法人に集約するのは国内唯一。特に電子決済手段等取引業（USDC 取扱）は 2025-03 に登録完了し、Circle 社 USDC の日本円建スポット交換・レンディングを展開中。
- **SBI グループ内シナジー**：SBI 証券（金商 第44号、JVCEA #1041）・BI FXTRADE（金商 第2635号、JVCEA #1026）など同グループ傘下の金融商品取引業者との連携が可能な体制。SBI ホールディングスは Ripple Labs（XRP 関連）への出資・提携関係を持つ。
- **ビットポイントジャパン合併**（2026-04 効力発生）は、金融商品取引法による暗号資産規制強化を見据えたグループ内経営資源集中を目的とする（公式発表より）。合併後も VCTRADE と BITPOINT の 2 ブランドを当面維持。
- **MTGOX 債権者向け対応窓口**を設置しており、マウントゴックス清算絡みの資産返還にも関与（詳細は公開情報の範囲内）。
