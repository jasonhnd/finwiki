- **商号**：ビットバンク株式会社（英文：Bitbank, Inc.）
- **法人番号**：1010801024625（国税庁法人番号 DB 確認済み）

- **本店所在地**：東京都中央区八重洲2-2-1 東京ミッドタウン八重洲 八重洲セントラルタワー10F（2026年2月6日付で品川区西五反田から移転）
- **親会社 / 株主構成**：独立系。非上場。詳細な株主構成は公表なし。
- **コーポレートサイト**：https://bitbank.cc/

## 2. 牌照・登録状況

- **電子決済手段等取引業 / 資金移動業**：公表なし
- **JVCEA 会員種類**：第一種・暗号資産交換業のみ（JVCEA 会員一覧にて確認。デリバティブ交換業の認証なし）

## 3. 取扱通貨

- **取扱銘柄数**：公式サイトへの直接アクセス不可のため正確な最新数は未確認
- **主要銘柄**（公知情報）：BTC, ETH, XRP, LTC, BCC（BCH）, MONA, XLM, QTUM, BAT, IOST, XEM ほか多数
- **特色**：XRP をはじめとするアルトコイン銘柄数が国内取引所の中で多く、アルトコイン取引高において国内上位とされる。spot 専業（margin / derivative 取扱いなし）。

## 4. 業務範囲

- **Spot 取引**：現物売買（取引所・販売所）
- **Margin / Derivative**：未取扱（JVCEA 会員種別にデリバティブ交換業の認証なし）
- **貸暗号資産**：公表なし
- **NFT / staking**：公表なし
- **法定通貨入金チャネル**：銀行振込・クイック入金（住信 SBI ネット銀行等）対応（公式情報 direct 確認不可）

## 5. 市占・取引高

- **取引高**：JVCEA 月次統計にて会員個社別の取引高が公表されているが、2026-05-18 時点では当該 Excel への直接アクセス不可のため具体数字は未記入。
- **口座数**：公表なし
- **顧客層**：retail（個人投資家）中心。法人サービスも提供。
- **競合内ポジション**：アルトコイン取引高において国内上位とされるが、根拠となる JVCEA 統計の具体数値は要別途確認。BTC 中心の [[exchanges/jp-exchange-bitflyer|bitFlyer]]・[[exchanges/jp-exchange-coincheck|Coincheck]] に対し、XRP など多通貨の取引高が相対的に高い構造を持つ。スポット取引量の業界比較は [[exchanges/jvcea-spot-volume-statistics-analysis|JVCEA スポット取引量統計分析]] 参照。

- **FTX Japan 吸収（2024年）**：破綻 exchange の顧客保護型引き継ぎを実施（[[exchanges/ftx-japan-100pct-return-case-study|FTX Japan 100% 返還ケーススタディ]] と [[exchanges/jp-vasp-ma-consolidation-history|JP VASP M&A コンソリデーション史]] 参照）。口座数・取引通貨の拡充が見込まれる。
- **本店移転（2026年2月）**：東京ミッドタウン八重洲へ移転。大手交流ハブエリアへの集約と見られる。
