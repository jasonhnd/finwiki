# 台湾 FSC VASP licensing 制度

## 制度概要

台湾 FSC (金融監督管理委員會) は 2021-07 AML 法改正で VASP (虛擬資産服務業者) 制度を確立。既存 6 業種 (証券/銀行/保険/期貨/信託/投信) と並行する独自 framework として位置づけ、2024-2025 にかけて専門法 (虛擬資產服務法草案) の立法準備を進めている。アジアで最も柔軟な枠組だが、近い将来構造化が予想される過渡期。

## 現行制度 (AML 中心)

- **AML 登記必須**: VASP は財務犯罪防制法・洗錢防制法を遵守する義務
- **顧客実名確認**: KYC + EDD (Enhanced Due Diligence) 実装
- **疑わしい取引届出**: 法務部 調查局 (MJIB) 経由で報告
- **取扱品目制限なし** (現行): 業者裁量で上場銘柄を決定 · 将来専門法で明確化予定

## 主要事業者 (2024-2026)

- **MaiCoin / MAX (現 MaiCoin Exchange)** — 国内最大手 · 蔡振平 創業
- **BitoPro (BitoEX)** — 鄭光泰 創業 · 2014 設立 · ローカル老舗
- **ACE Exchange** — 多通貨対応 · 2018 設立
- **Hoo Exchange Taiwan** — 海外系 · 現在停止状態
- **Rybit / Kronos Research** — 機関 OTC trading 中心

## 2024 立法動向

- **2024-09** 虛擬資產服務法草案 公開
- **2025-2026** 国会審議継続中
- **想定要件**: ライセンス制 + 取扱品目審査 + コールド保管義務 + 顧客資産分別管理 — 日本/香港モデルに接近する見込み
