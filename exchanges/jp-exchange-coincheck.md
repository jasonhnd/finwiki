# コインチェック株式会社 — 日本暗号資産交換業者 overview

> FSA 登録番号 関東財務局 第00014号 · JVCEA 会員 #1017 · 登録日 2019-01-11

## 1. 法人・株主

- **商号**：コインチェック株式会社 / Coincheck, Inc.（日本法人）
- **設立**：2012年（Wonder株式会社として設立。2014年にコインチェック株式会社へ商号変更）
- **本店**：東京都渋谷区桜丘町1番4号 渋谷サクラステージ SHIBUYAサイド 27F
- **法人番号**：1010001148860
- **親会社**：マネックスグループ株式会社（東証プライム 8698）
  - 2018年4月 マネックスグループが株式取得により完全子会社化（取得価額 約36億円）
- **上場持株会社**：Coincheck Group N.V.（オランダ公開有限責任会社、アムステルダム本拠）
  - Nasdaq 上場（ティッカー CNCK / CNCKW）。コインチェック株式会社の直接の親会社。
  - マネックスグループが Coincheck Group N.V. の筆頭株主（合併完了後も過半を保持）

## 2. 牌照・登録状況

- **暗号資産交換業**：関東財務局長 第00014号（登録日 2019-01-11）
- **JVCEA 会員**：第一種会員 #1017（暗号資産交換業のみ；デリバティブ取引業は非認可）
- 電子決済手段等取引業・金融商品取引業・資金移動業の登録は JVCEA 公開情報の範囲内では確認されていない

## 3. 取扱通貨

- **取扱銘柄数**：2022年9月時点で18銘柄（+1 QoQ）。以後も随時拡充継続
- **主要銘柄**：BTC、ETH、ETC、XRP、NEM（XEM）、LTC、XLM、MONA、BAT、QTUM、LSK、XEM、ENJ、SAND、AXS など
- **特色**：国内で早期に多銘柄対応。IEO（Initial Exchange Offering）機能を国内初導入。NFT マーケットプレイス（Coincheck NFT）を運営。現物中心で、証拠金・デリバティブ取引は提供していない

## 4. 業務範囲

- **現物取引**：取引所（板取引）+ 販売所（OTC）の双方提供
- **貸暗号資産**（Coincheck Lending）：顧客から暗号資産を借り入れ、カバー取引に活用
- **NFT マーケットプレイス**：Coincheck NFT (β)。IP 保有者との 1次販売（BtoC）+ ユーザー間 2次売買（CtoC）
- **IEO**：「Coincheck IEO」として国内初の IEO サービスを展開
- **法定通貨入金**：銀行振込・クイック入金（コンビニ・ペイジー等）
- **Web3 / メタバース**：Coincheck Labs（スタートアップ支援・投資）、メタバース向け NFT LAND 販売（The Sandbox、Otherside、Decentraland 等）
- 法人向けサービスも提供（OTC 大口取引等）

## 5. 市占・取引高

- **アプリダウンロード数**：国内暗号資産取引アプリ No.1（5年連続 2019–2023年、AppTweak 調べ）
- **認証ユーザー数**：175万人（2022年9月末）→ 183万人（2023年6月末）
- **預かり資産（コールドウォレット）**：
  - 2022年9月末：約2,796億円
  - 2023年6月末：約3,465億円
  - 2024年12月末：約1兆305億円（Coincheck Group N.V. 財務開示 2025-02）
- **取引高**：現物取引高 2Q FYE Mar.2022（3ヶ月）7,695百万円（当時）。市場低迷期（2023年頃）は大幅縮小。市況回復後の2024年度は大幅拡大。個別月次データは JVCEA 統計参照
- **競合内ポジション**：認証口座数ベースで国内最大規模。ビットフライヤー・GMOコイン等とともに上位3社の一角

## 6. 沿革・重大事件

| 年月 | 事項 |
|------|------|
| 2012-08 | Wonder株式会社として設立。暗号資産ウォレット・取引サービスを開始 |
| 2014頃 | コインチェック株式会社に商号変更 |
| **2018-01-26** | **NEM（XEM）不正流出事件**：ホットウォレットに保管されていた XEM 約5億2,630万 NEM（当時時価約580億円相当）が外部ハッカーにより不正送金される。同日深夜に同社が公式発表。日本史上最大規模の暗号資産流出事件（詳細は [[exchanges/coincheck-nem-hack-detailed-analysis|Coincheck NEM ハック詳細分析]] および [[security/forensic-identity-anchor-chain|forensic identity anchor chain]] 参照） |
| 2018-03 | 金融庁より業務改善命令（複数回）受領。FSA 検査・改善要求に対応 |
| 2018-04 | マネックスグループが全株式を約36億円で取得し完全子会社化。経営体制刷新 |
| 2019-01-11 | 暗号資産交換業登録（関東財務局 第00014号）取得 |
| 2020頃 | NFT マーケットプレイス「Coincheck NFT（β）」開始 |
| 2022-03 | Nasdaq 上場 SPAC Thunder Bridge Capital Partners IV, Inc.（THCP）との Business Combination Agreement 締結を公表（マネックスグループ IR 開示） |
| 2024-10 | Business Combination Agreement 第3次修正 |
| **2024-12-05** | Thunder Bridge IV 株主特別決議で合併承認 |
| **2024-12-10** | 合併完了（Closing）。Coincheck Group N.V. が THCP の後継法人として成立 |
| **2024-12-11** | **Nasdaq 上場開始**：ティッカー CNCK（普通株式）・CNCKW（ワラント）。総調達額 約3,160万ドル（リデンプション後ネット額）。日本の暗号資産交換業者として初の Nasdaq 上場 |

## 7. 戦略動向

- **グローバル資本市場への接続**：Nasdaq 上場後、米国の機関投資家向け IR 活動を強化。Coincheck Group N.V. として F-1 / F-3 / 6-K 等の SEC 継続開示を実施（CIK: 0001913847）
- **Web3 / メタバース展開**：国内 NFT プラットフォームの No.1 ポジション維持を目指し、IP ホルダーとの提携を継続
- **IEO 拡大**：国内初の IEO サービスを活かし、プロジェクト誘致を継続
- **Coincheck Labs**：ブロックチェーン関連スタートアップへの投資・インキュベーション継続
- **取扱銘柄拡充**：現物・新規銘柄の追加。2024-2025年の市場回復期に口座開設・預かり資産が急増

## 8. 関連人物

- **松本大（Oki Matsumoto）**：マネックスグループ代表執行役社長 CEO、Coincheck Group N.V. Executive Chairman（SEC 開示にて確認。NEM事件後の2018年4月買収時より Coincheck 統治に関与）
- **有定誠（Satoshi Hasuo）**：コインチェック株式会社 代表取締役社長（2022-2023年時点の SEC 開示にて "President: Satoshi Hasuo" として確認）
- **Gary Simanson**：Thunder Bridge IV 元 President & CEO → 合併後 Coincheck CEO（SEC 開示にて確認）
- **創業者**：和田晃一良（Koichiro Wada）・大塚雄介（Yusuke Otsuka）が共同設立（公知情報）。NEM事件後の経営体制刷新に伴い、マネックス主導の経営陣へ移行
