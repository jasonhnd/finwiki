# HKMA 稳定币牌照 · 香港 Stablecoin Licensing Regime 概览

> [!info] TL;DR
> 香港 HKMA(Hong Kong Monetary Authority)稳定币立法 2025-08 通过,是亚洲第一个完整 SC 牌照框架。2026-05-21 首批 9 家申请方决定(实际只发 2 张:HSBC + Anchorpoint)。HK 牌照 = 三圆 MRA 第三圈候选 + 中国大陆唯一可获牌途径。

## Key facts

- HKMA SC 立法:2025-08 通过 ^[extracted]
- 9 家申请 2026-05-21 决定 ^[extracted]
- 实际只发 2 张(HSBC + Anchorpoint)· 其他延迟 ^[extracted]
- 香港 HKD-stable + USD-stable 双轨牌照 ^[extracted]
- 京东 JD-HKD = 中国大陆背景大厂在港首次申请发行 SC ^[extracted]
- §501(d) 候选第三波(2027-H2)· 入榜条件:与 e-CNY 划界 + 不接收 SDN 客户 ^[inferred]
- 储备金 100%(HKD / USD 短期国债 + 央行存款)+ 第三方托管 + 月度审计 ^[extracted]
- HKD 100M+ 资本要求 ^[extracted]

## Mechanism / How it works

**HKMA SC 牌照核心要求**:100% 储备金(HKD / USD 短期国债 + 央行存款)+ 储备金第三方托管 + 月度审计 + 发行方在 HK 有实体 + HKD 100M+ 资本要求 + KYC + AML/CFT + Travel Rule + 与 OFAC SDN 协同(间接通过 USD 储备)+ 与 PBoC / e-CNY 划界(不接收中国大陆零售客户)。

**9 家申请方(2026-05-21 决定)**:
| 申请方 | 背景 | 牌照决定 |
|---|---|---|
| HSBC | 全球大行 | ✓ 获批 |
| Standard Chartered Anchorpoint(StanChart + Animoca + HKT) | 联合体 | ✓ 获批 |
| 圆币科技 | HK 本地 | 待定 |
| 京东 JD-HKD | 中国大陆背景 | 待定(敏感) |
| 蚂蚁国际 | 中国大陆背景 | 待定(敏感) |
| 中银(BOCHK) | 中国背景 | 待定(敏感) |
| ZA Bank | 数字银行 | 待定 |
| Tether | 灰色市场美元龙头 | 待定(关键) |
| Conflux | 中国背景公链 | 待定 |

**关键发现**:HKMA 第一批实际只发 2 张,不是预测的 9 张。其他 7 家延迟或被拒。

## Origin & evolution

HKMA 2022-12 启动 SC consultation,2023-12 第二轮 CP 确立 Sandbox 名单。2025-08 通过 Stablecoin Ordinance(立法历时近 3 年)。2026-05-21 首批决定释放重要信号:HKMA 优先发给"低政治风险 + 高合规背景"的全球大行(HSBC / StanChart),中国大陆背景大厂 + 灰色市场玩家延迟。

## Counterpoints

- 第一批只发 2 张 · 显著低于市场预期 · 暗示后续步伐谨慎
- 中国大陆背景大厂全部延迟 · 反映"e-CNY 划界"硬条款执行严
- Tether HK 申请决定可能拖到 2026-Q4 之后
