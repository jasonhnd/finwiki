# HKMA Project Ensemble 概览


## Wiki route

This entry sits under [[fintech/INDEX|fintech index]]. Read it with [[fintech/japan-financial-regulation|日本金融規制 — トークン・暗号資産・決済に関する法体系]] for adjacent context and [[fintech/japan-stablecoin-regulatory-landscape|日本 Stablecoin 法制度の三層構造（JPYC・USDC・Project Pax）]] for the broader system boundary.

> [!info] TL;DR
> Project Ensemble 是香港金管局(HKMA)2024.03 启动的代币化货币沙盒,核心测试代币化银行存款 + wholesale CBDC 在香港境内的应用。首批 HSBC / Standard Chartered / BoCHK / Hang Seng;2024.08 Anchorpoint(非传统银行)加入。Ensemble + mBridge 双轨使香港成为亚洲唯一同时拥有本地代币化结算 + 跨境 CBDC 走廊的金融中心。

## Key facts

- 2024.03 启动,首批 4 家银行(HSBC / Standard Chartered / BoCHK / Hang Seng)+ HKEX + PolyU ^[extracted]
- 2024.08 Anchorpoint 加入,扩展至非传统银行参与方 ^[extracted]
- 沙盒不向 retail 开放,纯机构间实验 ^[extracted]
- HKMA 同步推进 e-HKD 零售 CBDC(独立项目) ^[extracted]
- 4 大试验主题:代币化资产结算 / PvP / DvP / 绿色金融 ^[extracted]
- 与 BIS Agorá 互联尚未公开,但可能 2026 后接入 ^[inferred]

## Mechanism / How it works

Ensemble 架构以"代币化港元存款"为核心,商业银行在 HKMA 受监管沙盒中发代币化存款 token,机构间通过该 token 完成结算。**4 大试验主题**:
1. 代币化资产结算 — 债券 / 基金二级市场代币化交易
2. PvP(Payment-vs-Payment)— 港元代币化存款 vs 外币原子互换
3. DvP(Delivery-vs-Payment)— 代币化资产 vs 代币化港元结算
4. 绿色与可持续金融 — 碳信用代币化与气候融资

**Anchorpoint 战略意义**:由前 HSBC / Goldman 高管创立,定位代币化基础设施服务商。其入选 Ensemble 标志 HKMA 接纳"金融科技 + 传统银行"混合参与 → 沙盒不仅是大行实验场,也是新型 infrastructure provider 的入场券。与 JPM Kinexys / Citi Token Services 形成"独立 vs 银行内嵌"的代币化路径对照(deposit token 论题见 [[fintech/institutional-stablecoin-deposit-token-thesis|机构 SC 与 deposit token 论题]])。

## Origin & evolution

2021 HKMA 与 BIS 共同启动 [[fintech/mbridge-bis-multi-cbdc-overview|mBridge]] → 2024.10 BIS 退出后 HKMA + PBoC 接管。2024.03 Ensemble 启动 → HKMA 实现"境内 + 跨境"双轨战略:Ensemble 让本地金融机构掌握代币化存款 know-how,mBridge 让港元成为跨境 CBDC 走廊的"必经货币"。2024.08 Anchorpoint 加入扩展至非银参与方。2025-2026 [[fintech/hkma-stablecoin-licensing-overview|HKMA Stablecoin Ordinance]] 发牌后,Ensemble 沙盒成为持牌前的预热场 —— 持牌稳定币发行方可通过 Ensemble 测试与代币化港元存款的互操作。

## Counterpoints

- 沙盒 2+ 年仍处实验阶段,商用化时间表未公开
- 不向 retail 开放限制了规模化验证(机构间流量量级有限)
- HKMA 同时推 e-HKD(零售)+ Ensemble(机构)+ mBridge(跨境)三线并进,资源分散
- Anchorpoint 路径若被大行排挤,"独立 infrastructure"叙事可能失败
