# 司法管辖区清单作为货币保护主义工具


## Wiki route

This entry sits under [[fintech/INDEX|fintech index]]. Read it against [[fintech/three-circles-stablecoin-mra-framework|美 / 欧 / 日 "三个大圆" 稳定币全球合规架构（MRA 互认）]] for peer / contrast context and [[fintech/japan-financial-regulation|日本金融規制 — トークン・暗号資産・決済に関する法体系]] for the broader system / regulatory boundary.

> [!info] TL;DR
> GENIUS Act §501(d) 要求美国财政部于 2026-Q3 前发布**"允许互操作的境外司法管辖区"清单**。**入榜的司法管辖区**获得 12-18 个月跨境合规独占窗口；**入榜的稳定币 / 链**获得机构准入"通行证"。这是新型"白名单货币保护主义"，与 1980s 关税壁垒 / 2010s 数据本地化 同源但更精细。

## 法律机制

§501(d) 实质等效测试要求：
- 与美国 GENIUS Act 框架对齐的本地立法
- OFAC 制裁兼容的执法能力
- AML/CFT 框架与 FATF Travel Rule 一致（详见 [[fintech/fatf-travel-rule-cross-border-stack|FATF Travel Rule 跨境 stack]]）

**MRA（双边等效协议）是入榜前置条件**：
- U.S.-Japan MRA: 2026-02 已签
- U.S.-EU MRA: 2026-Q3 预计
- U.S.-UK MRA: 2026-Q4 预计
- U.S.-SG / UAE / HK MRA: 谈判中

## 第一波入榜预测

| Tier | 司法管辖区 | 概率 |
|---|---|---|
| 1（2026-Q3）| 日本 + 欧盟 + 英国 | 75-85% |
| 2（2026-Q4 - 2027-Q1）| 新加坡 + UAE | 60-70% |
| 3（2027 H1）| HK + 澳大利亚 + 加拿大 + 韩国 + 瑞士 | 30-50% |
| 4（永远不入榜）| 中国大陆 / 俄罗斯 / 伊朗 / 朝鲜 | < 5% |

## 入榜的连锁反应

### 对司法管辖区
- 12-18 个月**先发优势**
- 资本流入（美机构资金通过合规 SC 进入）
- 金融中心地位提升（HK / SG 抢"亚洲合规稳定币枢纽"）

### 对合规 SC 发行人
- 机构准入 = 真正的"机构 USDC 类地位"
- **第一波获取定价权**（类似第一波 OCC charter 拿到者）

### 对未入榜
- 跨境支付**成本上升**（需逐笔 OFAC license）
- 合规 SC 项目**估值压缩**
- **加速去美元化或转向 e-CNY 阵营**

## 货币保护主义的历史先例

| 时代 | 工具 | 现代等效 |
|---|---|---|
| 1934 美国关税法 | 白名单贸易伙伴享互惠 | §501(d) jurisdiction list |
| 1995 GATT MFN | 最惠国待遇 | 互操作等效认定 |
| 2015 EU GDPR 第 45 条 | 充分性认定决议 | 数据流动 / 货币流动结构相似 |
| 2018 CFIUS 反向投资审查 | 战略行业准入 | 稳定币基础设施作为战略行业 |

**结构性共同点**：用"白名单 + 等效认定"取代"普遍准入"，制造**合规圈内的特权地带**。

## 应用 / 转用

可用于分析：
- **CBDC 跨境互操作**（如 mBridge 是"中国版 jurisdiction list"）
- **数据流动协议**（GDPR adequacy decisions 与此结构对偶）
- **AI 治理框架**（FIDO Alliance / AAIF 类似的协议白名单）
- **跨境清算系统**（BIS Project Agorá 类似）

**未来 5 年的连锁推断**：
- 欧盟会发布"欧盟版 §501(d)"清单（2027-2028）
- 中国会通过 mBridge 隐性发布"中国版友好清单"
- UAE / SG 等会努力"双重入榜"（美 + 欧 + 亚）
