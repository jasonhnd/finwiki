# EU MiCA 实施状态 2026-05 · EMT/ART 持牌图谱 × USDT 退出 × USDC 市场份额重塑

## TL;DR

EU MiCA (Markets in Crypto-Assets Regulation, Regulation (EU) 2023/1114) 自 **2024-06 SC (EMT/ART) 部分生效** 以及 **2024-12 全面生效** 后, 进入 18 个月的实施稳态。2026-05 状态: **EMT 持牌方约 22 家** (跨 13 个成员国), **ART 持牌方仅 3 家** (Bitstamp Pay / Quantoz EURD / Société Générale 部分产品), **USDT 在欧盟主流交易所流通几乎归零** (从 2024-12 ~28% 份额降至 2026-05 ~1%), **USDC (Circle Europe Tight) 与 EURC 双轨覆盖欧盟法币 SC 90%+ 流通**, **Société Générale-FORGE EURCV** 与 **BBVA EUR stablecoin** 成为银行系欧元 SC 双驾马车。监管端 **ESMA + EBA + 国家主管局 (BaFin / AMF / CSSF / Banca d'Italia 等) 协同稳定**, 但**大型 EMT 流通额上限** 与 **ART 几乎归零** 两大设计问题暴露 —— 见 [[fintech/mica-overview|MiCA 概览]] 与 [[fintech/mica-emt-art-subcategories-deep-dive|MiCA EMT vs ART 深度]]。

## EMT (E-Money Token) 持牌方图谱 (2026-05)

| 持牌方 | 国家主管局 | 主要 SC | 流通规模 (2026-05) | 路径 |
|---|---|---|---|---|
| Circle Europe Tight | ACPR (法国) | USDC + EURC | EURC €280M / USDC EU 流通 ~$8B | EMI 升级 |
| Société Générale-FORGE | ACPR (法国) | EURCV | €60M | 银行子公司 |
| BBVA | Banco de España | BBVA EURO | €25M (2026-05 上线初期) | 银行直接 |
| Quantoz | DNB (荷兰) | EURD (ART) + EURQ (EMT) | EURQ €40M | EMI |
| Membrane Finance | FIN-FSA (芬兰) | EUROe | €35M | EMI |
| Banking Circle | CSSF (卢森堡) | BC EUR / BC USD | €120M / $80M | 商业银行 |
| Schuman Financial | CBI (爱尔兰) | EURØP | €25M | EMI |
| StablR | MFSA (马耳他) | EURR / USDR | €15M / $20M | EMI |
| Crypto.com Europe | MFSA (马耳他) | CDCEUR | €10M | EMI + CASP |
| Bitstamp Pay | LB (立陶宛) | BPC EUR | €18M | EMI |
| 其他 ~12 家 (含 Lithuania-licensed 多家 EMI 小型 SC + 德国 BaFin licensed 2 家) | 各成员国 | 多个小型 EUR/USD SC | 合计 ~€80M | EMI |

**核心观察**:

1. **Circle (USDC + EURC) 一家独大**, 流通规模占 EMT 总量 ~75%
2. **法国 ACPR 与卢森堡 CSSF + 立陶宛 LB + 马耳他 MFSA** 是 EMT 牌照集中地
3. **银行系发行体 (Société Générale + BBVA) 流通规模仍小于 Circle**, 但增长曲线快 (BBVA 上线 2 个月已 €25M)
4. **PYUSD (PayPal)** 没有走 MiCA EMT 路径 —— PayPal 在欧盟用现有 EMI 牌照运行支付, PYUSD 暂未在欧盟主流交易所流通

## ART (Asset-Referenced Token) 持牌方图谱 (2026-05)

| 持牌方 | ART 产品 | 锚定资产 | 流通 |
|---|---|---|---|
| Quantoz | EURD | 银行存款 + 短债篮子 | €15M |
| Bitstamp Pay | (申请中) | 拟锚定货币篮子 | n/a |
| Société Générale-FORGE | CBV 系列 (受限) | 多资产挂钩 | < €5M |

**ART 几乎归零**: 与 [[fintech/mica-overview|MiCA 概览]] 的预测一致, ART 监管负担过重 → 几乎无人选 ART 路径。EBA 监管下的 ART 需要满足 (1) 储备金管理细则比 EMT 更严; (2) 大型 ART (>€5B) 流通额上限; (3) 资本要求更高; (4) 持续监管报告频率更高。结果是合规成本 vs EMT 高 3-5 倍, 但无明显商业优势 → ART 沦为"理论存在"。

## USDT 退出欧盟与 USDC 份额重塑

**USDT 退出曲线**:

| 时点 | USDT 在欧盟 (主流交易所) 流通 / 份额 |
|---|---|
| 2024-09 | ~28% 欧盟 SC 交易量 |
| 2024-12-30 | Tether 公告主动退出 |
| 2025-01-08 | Coinbase Europe / Kraken EU / Bitstamp 下线 |
| 2025-03 | ~6% (剩余在小型交易所 / OTC) |
| 2025-09 | ~3% |
| 2026-05 | ~1% (主要在非欧盟监管的灰色通道) |

**USDT 退出后流入**:

- **USDC 净流入 (欧盟)** ~$45B (累计 2025-01 至 2026-05)
- **EURC 流入** ~€220M (累计同期)
- **银行系欧元 SC (EURCV + BBVA EURO + Banking Circle)** 累计 ~€180M

**USDC 份额** 2026-05 ~78% 欧盟 SC 流通, 几乎垄断。**EURC + 银行系欧元 SC 合计** 仍仅 ~10% (~€400M), 与 USDC EU 流通的 $8B+ 差 20x。**欧元 SC 市场仍小** 是 MiCA 实施后的核心结构性现实 —— 这与 [[fintech/mica-overview|MiCA 概览]] 中 "EURC 流通与 USDC 差 300x" 的早期判断已大幅缩窄 (现在差 20x), 但远未追平。

## 主要欧元 SC 详解

### Circle EURC

- 2022-06 在 Stellar 首发, 2023 扩 Ethereum/Avalanche/Solana/Base
- 2025-05 获 ACPR EMT 牌照 (与 USDC 同次)
- 储备 100% 短期国债 + 央行存款 + 现金, BlackRock 部分托管
- 2026-05 流通 €280M, 主要用例: 加密交易所 EUR pair / DeFi (Aave Polygon) / 欧元跨境支付
- 与 USDC 共享 Circle 同一合规框架 → 是 [[fintech/circular-reserve-asset-flywheel-overview|循环储备资产飞轮]] 在欧元侧的复制

### Société Générale-FORGE EURCV (Euro Coin Vertu)

- 2023-04 首发 (在 MiCA 前作为内部产品)
- 2025-09 通过 MiCA EMT 申请 (Société Générale-FORGE 是 SocGen 的合资 fintech 子公司)
- 2026-Q1 上线 Stellar + Ethereum + Solana
- 储备 法国短期国债 + 商业银行存款 + ECB deposit facility
- 2026-05 流通 €60M, 用例: 大型企业 B2B 跨境结算 / 机构端 yield-bearing wallet 测试
- 与 [[fintech/institutional-stablecoin-deposit-token-thesis|机构稳定币 / 存款 token 论题]] 完全契合 —— SocGen 把 EURCV 定位为"半 SC 半 deposit token"

### BBVA EURO

- 2025-11 公告, 2026-03 上线
- 西班牙 Banco de España 协议 + 与 ACPR 协商 passport
- 部署链 Ethereum + Polygon (后续加 Base/Avalanche)
- 储备由 BBVA 自家托管 + 第三方审计 (KPMG España)
- 流通 €25M (2026-05, 早期阶段)
- 用例: BBVA 西语市场 (西班牙 + 墨西哥 + 拉美 BBVA 子公司) 跨境结算
- 与 [[fintech/brazil-mexico-cbdc-stablecoin-push-2026|Brazil/Mexico CBDC 推进]] 的 BBVA 西语跨境角度联动

### EUROe / EURØP / EURR 等小型 SC

- 主要用于 DeFi / 加密交易所内 EUR pair
- 单一规模 < €40M
- 长尾市场 → 未来可能合并 / 被收购

## ESMA × EBA × 国家主管局协同机制

**MiCA 监管架构**:

- **ESMA (European Securities and Markets Authority)** 负责 CASP (Crypto-Asset Service Provider) 监管协调 + 反市场操纵 + 跨境信息共享
- **EBA (European Banking Authority)** 负责 EMT + ART 监管 (因为 SC 本质是 e-money / 资产挂钩工具, 属银行业范畴)
- **国家主管局 (NCA)** 负责实际牌照颁发 + 日常监管:
  - ACPR (法国) — Circle / SocGen
  - BaFin (德国) — 2 家 EMI 持牌方
  - DNB (荷兰) — Quantoz
  - CSSF (卢森堡) — Banking Circle
  - CBI (爱尔兰) — Schuman Financial
  - MFSA (马耳他) — StablR / Crypto.com
  - LB (立陶宛) — Bitstamp Pay + 多家小型 EMI
  - Banco de España — BBVA

**Passport 机制**: EMT/ART 持牌方在某成员国获牌后可 passport 到全欧盟流通。2026-02 起 passport 流程标准化, 跨成员国监管协调通过 ESMA + EBA 的 supervisory college 完成。

**实际协同问题**:

1. **国家主管局之间审查严苛度差异**: 法国 ACPR 严, 马耳他 MFSA 宽松 → 监管套利倾向出现 (类似早期欧盟基金的卢森堡集中现象)
2. **ESMA 在 CASP 端 vs EBA 在 EMT/ART 端的边界不清**: 例如交易所自家发 SC (Coinbase USDC 与 Coinbase Europe CASP) 谁主导审查?
3. **大型 EMT (>€100M / >1M 用户 / 日交易 >€1M) 限制条款** 对 USDC 适用问题: ESMA 2026-03 公开声明 USDC 已触发"大型 EMT" 门槛, 但具体限制 (例如禁止非欧元交易对) 仍在 case-by-case 评估

## Counterpoints

- **欧元 SC 规模仍小** (合计 ~€400M vs USDC EU $8B), MiCA 没有培育出欧元 SC 生态, 反而**强化了 USDC 在欧盟的地位**
- **ART 类别实质失败** (3 家持牌, < €25M 合计流通), MiCA 设计层面的"双类别"未达到预期
- **MiCA 合规成本高** → 把许多小型 EUR SC 项目 (DeFi / 算法 SC) 直接驱赶出欧盟, 创新性收缩
- **去中心化 SC (DAI / USDS / LUSD / FRAX) 灰色处理** 让欧盟用户无法合规使用 DeFi 主流 SC, 与美国 GENIUS Act §501 形成监管阻断
- **大型 EMT 限制条款** (尤其针对 USDC) 可能在 2026-2027 引发 Circle 与 ESMA 的实质冲突 —— Circle 是否会被迫限制欧盟 USDC 流通?
- **银行系欧元 SC 增速缓慢**: SocGen + BBVA + Banking Circle 三家累计 €180M, 距离与 Circle 抗衡差 50x
- **USDT 退出欧盟后 OTC / 灰色市场反弹**: 部分欧盟用户通过非欧盟监管的交易所 (Binance.com / Bybit / 等) 继续使用 USDT, 监管漏洞存在
- **MiCA 与 [[fintech/dora-eu-digital-operational-resilience-overview|DORA]] 双轨监管的合规叠加成本** 让中小 EMI 难以承担, 推动行业进一步集中
- **PYUSD / Ripple RLUSD / WLFI USD1 等美国系 SC 无意进入欧盟** —— MiCA 实质把欧盟变成了 "USDC + EURC + 银行系欧元 SC" 三轨封闭市场, 限制 SC 国际竞争

## Open questions

- ESMA 是否会在 2026 H2 对 USDC 触发"大型 EMT" 实施实质性限制 (例如禁止部分交易对 / 限制流通额)?
- USA-EU MRA (Mutual Recognition Agreement) 是否会在 2026 Q3-Q4 签署 (与 [[fintech/three-circles-stablecoin-mra-framework|三圆 SC MRA 框架]] 对齐)?
- 欧元 SC 总流通能否在 2027 突破 €2B, 在欧盟内对 USDC 形成有意义的份额竞争?
- 德国 BaFin 是否会在 2026 H2 颁发新的 EMT 牌照给德意志银行 / 德国商业银行 / Commerzbank?
- 意大利 / 西班牙 / 葡萄牙 的国家银行系 SC 何时上线?
- MiCA 2.0 (预计 2027-2028 立法启动) 是否会修订 ART 类别 + 大型 EMT 限制条款, 缓解当前的设计问题?
- Tether 是否会通过 HK 牌照 (见 [[fintech/three-circles-stablecoin-mra-framework|三圆 MRA 框架]]) 曲线进入欧盟 —— 经由 HK-MRA passporting?
- DAI / USDS 等去中心化 SC 是否会被 ESMA 第 4 版 Q&A 进一步收紧, 形成事实禁令?
- BBVA EURO 与 BBVA 拉美 (墨西哥 / 巴西 / 阿根廷) 业务的协同 —— EUR SC 是否会变成"欧盟 ↔ 拉美" 的跨境结算工具?
