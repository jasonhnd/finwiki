# Finance Domain

## Overview

This domain keeps public finance reference pages for capital markets, M&A, acquisition finance, shareholder activism / engagement, tender offers, MBO / squeeze-out process, large-shareholding disclosure, fair disclosure / insider trading controls, shareholder proposal / AGM voting, tax-jurisdiction comparison, and general retirement-math concepts. It excludes private financial plans, personal income examples, private account topology, and non-public advisory cases.
