# 株式会社ジャックス (Jaccs)

## Wiki route

This entry sits under [[card-issuers/INDEX|card-issuers INDEX]]. Read it against [[card-issuers/orico|株式会社オリエントコーポレーション (Orico)]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

北海道発祥のクレジット・信販大手。**1954-07 北海道信用販売株式会社として函館で設立**、1968 に「ジャックス」に改称。業態は信販・個品割賦・カードローン・集金代行・家賃保証・自動車ローン軸。**[[mufg]] グループの持分法適用関連会社**（三菱 UFJ 銀行が主要株主）として位置付け、MUFG のクレカ・消費者金融軸（[[mufg]] 内の 三菱 UFJ ニコス・アコム）と並ぶ「**第三の信販軸**」を担う。東証 PRIME 8584 上場。競合は [[orico]]・[[credit-saison]]。海外はベトナム・カンボジア・インドネシアでファイナンス事業を展開。

## 1. 会社概要

**正式名**: 株式会社ジャックス（Jaccs Co., Ltd.）
**証券コード**: 東証 PRIME **8584**
**設立**: **1954-07**（旧 北海道信用販売株式会社、函館発祥）
**改称**: 1968 年「ジャックス」へ
**上場**: 1971 東京証券取引所一部上場
**本社**: 東京都港区（登記）/ 函館・札幌に主要拠点
**親会社**: **[[mufg]]（持分法適用関連会社、三菱 UFJ 銀行が主要株主）**

### 業態カテゴリ

信販・クレジット業界で **Orico（[[orico]]）に次ぐ 2〜3 位グループ**。
- 大手 3 強: [[orico]] / **Jaccs** / [[credit-saison]]
- 業界軸: 個品割賦 + クレジットカード + 集金代行 + 保証

### 資本系列の歴史

```
1954-07 北海道信用販売（函館）
   ↓ 1968 商号変更
ジャックス
   ↓ 1971 東証一部上場
   ↓ 1990s 経営難 → 三和銀行支援
   ↓ 2005-10 MUFG 発足（[[mufg]] 親会社）
   ↓ 2007-04 三菱 UFJ 銀行系列入り（[[mufg]] 持分法適用）
現在: [[mufg]] グループ・東証 PRIME 8584
```

## 2. 事業セグメント・マップ

| セグメント | 主要事業 | 特徴 |
|---|---|---|
| **個品割賦（ショッピングクレジット）** | 自動車ローン / 教育ローン / リフォームローン | **主力 3 大ローン軸**、加盟店経由の販売金融に強み |
| **クレジットカード** | ジャックスカード（VISA / Mastercard / JCB 提携）| カード事業を消費者接点として展開 |
| **カードローン** | 個人向け無担保ローン | 自社チャネル・銀行保証提携 |
| **集金代行** | 公共料金・継続課金の口座振替代行 | BtoB ストック収益 |
| **家賃保証** | 賃貸住宅向け保証サービス | 不動産業界連携 |
| **法人ファイナンス** | リース・割賦の法人取扱 | 補完事業 |
| **海外**（ASEAN） | ベトナム・カンボジア・インドネシア消費者金融 | 二輪車ローン・耐久財ローン中心 |

### 自動車ローン軸の強み

Jaccs は **自動車販売店（中古車含む）の加盟店ネットワーク** を持ち、店頭での「ローン提案」フローで信販業界上位。教育ローン・リフォームローンも同様の加盟店経由モデル。これが [[orico]] と直接競合する主戦場。

### 海外戦略（ASEAN）

- **ベトナム**: 2014 年頃から本格進出、二輪車・家電ローン
- **カンボジア**: 現地法人で個人向け割賦
- **インドネシア**: 二輪車・耐久消費財ローン

ASEAN は MUFG 全体の海外戦略（Krungsri / Bank Danamon / VietinBank、[[mufg]] 参照）と整合する。Jaccs は親会社の銀行ネットワークを補完する**消費者ファイナンス層**を担う。

## 3. 沿革

| 年 | 事象 |
|---|---|
| **1954-07** | **北海道信用販売株式会社 設立（函館）** |
| 1968 | 商号を「ジャックス」に変更 |
| **1971** | **東京証券取引所一部上場** |
| 1980s | 全国展開・本社機能を東京へ |
| **1990s** | **経営難** → 三和銀行（後の UFJ → [[mufg]]）の支援 |
| 2001 | UFJ ホールディングス発足、Jaccs は UFJ 系列に |
| **2005-10-01** | **[[mufg]] 発足**、Jaccs は MUFG 関連会社化の流れ |
| **2007-04** | **三菱 UFJ 銀行 系列入り**（持分法適用関連会社） |
| 2014〜 | **海外進出加速**（ASEAN：ベトナム・カンボジア・インドネシア） |
| 2022-04 | 東証市場区分見直し → **東証 PRIME 8584** |
| 2024〜 | 中期経営計画（公開分）始動 |

### グループ内ポジショニング

[[mufg]] グループ内の消費者金融・クレカ・信販 3 軸：

| 機能 | 担当会社 | [[mufg]] からの位置付け |
|---|---|---|
| メガバンク・カード | 三菱 UFJ ニコス | 100% 連結子会社・三大クレカ |
| 消費者金融（無担保） | アコム | 持分法 39.6%・東証 STD 8572 |
| **信販・個品割賦** | **Jaccs** | **持分法適用関連会社・東証 PRIME 8584** |

つまり Jaccs は「[[mufg]] が直接の銀行融資ではカバーしにくい**販売金融＝加盟店経由の割賦**」を担当する位置。

### 競合構造

| 競合 | 強み | Jaccs との差 |
|---|---|---|
| [[orico]] | 業界最大手・伊藤忠 + みずほ系列 | 規模では先行、Jaccs は北海道・自動車・教育で対抗 |
| [[credit-saison]] | カード × ストック型・流通系 | カードに偏重、Jaccs は割賦に厚い |
| イオンフィナンシャル | 流通系・WAON | 流通基盤に依存、Jaccs は加盟店多様 |
| 楽天カード | デジタル・EC | デジタル軸で先行、Jaccs はオフライン加盟店で対抗 |

### 中期戦略（公開された方向性）

- **3 大ローン（自動車・教育・リフォーム）の深耕**
- **加盟店ネットワーク拡大**：ディーラー・教育機関・住宅リフォーム会社
- **デジタル申込フローの拡充**：紙の与信から Web 完結へ
- **ASEAN 消費者金融**：現地法人を通じた二輪・耐久財ローン
- **MUFG 系列としての補完機能**：[[mufg]] 銀行・ニコス・アコムと役割分担

## 5. 規制・政策

- **主管**: 金融庁（FSA）
- **登録**: 割賦販売法・貸金業法に基づく登録業者
- **業界団体**: 日本クレジット協会（JCA）加盟
- **主要規制**:
  - 割賦販売法：包括信用購入あっせん業者・個別信用購入あっせん業者
  - 貸金業法：総量規制（年収 1/3 規制）対象
  - 個人情報保護法・改正資金決済法（暗号資産関係は対象外）
- **直近政策論点**:
  - 2024〜 与信モデルへの AI / 機械学習導入
  - キャッシュレス化政策（カード手数料・加盟店規制）
  - 2025〜 改正割賦販売法・本人確認の高度化
