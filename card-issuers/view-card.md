# 株式会社ビューカード (Viewcard)


## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[card-issuers/jal-card|株式会社JALカード]] for peer / contrast context and [[payments/credit-purchase-card-operators-japan-index|credit purchase / card operators index]] for the broader regulatory / registry boundary.

## TL;DR

株式会社ビューカード（Viewcard Co., Ltd.）は **JR 東日本の完全子会社で、「ビューカード」ブランドのクレジットカードを発行・運営する会社**。2009年10月設立（旧 JR 東日本クレジットカード事業部からの分社化）、本店 東京都品川区大崎、資本金 50 億円、従業員 590 名（2024-04 時点）。**「Suica」一体型カードの主力イシュア**として、JR 東日本グループのキャッシュレス・モビリティ経済の中核を成す。**銀行代理業者登録**を保有する点が他のカード会社と異なり、Suica チャージ・カードローン中継等を含む幅広い金融機能を持つ。競合は [[card-issuers/jal-card|JAL カード]]、ANA カード（航空系）、[[card-issuers/credit-saison|セゾン]]・[[card-issuers/epos-card|エポス]]（流通系）等。

## 1. 法人・株主

- **商号**：株式会社ビューカード（英文：Viewcard Co., Ltd.）
- **設立**：2009年10月1日（東日本旅客鉄道株式会社のクレジットカード事業部を分社化）
- **本店所在地**：〒141-8540 東京都品川区大崎1-5-1 大崎ニューシティ1号館
- **資本金**：50 億円
- **法人番号**：2010701023263
- **親会社**：東日本旅客鉄道株式会社（JR 東日本、東証 PRIME 9020）100% 出資の完全子会社
- **代表者**：代表取締役社長
- **従業員数**：590 名（2024 年 4 月 1 日時点）

## 2. ライセンス・登録

- **割賦販売法**：包括信用購入あっせん業者として METI 登録
- **クレジットカード番号等取扱契約締結事業者**：METI 登録
- **銀行代理業者**：JR 東日本グループに関連する銀行代理業務を担う（特徴的なライセンス、他カード会社にない）
- **業界団体**：[[card-issuers/jcb|日本クレジット協会]] (JCA) 会員
- **国際ブランド提携**：JCB・VISA・Mastercard
- **発行協力**：自社発行のほか、[[card-issuers/jr-east-financial|JR東日本ファイナンシャル]] 系列のカード機能も束ねる

## 3. 商品ライン

- **ビュー・スイカカード（VIEW Suica）**：JR East 全駅で Suica チャージ可能、オートチャージ機能、JRE POINT 還元
- **「ビュー・カード」**（基本ベーシック）
- **「ビック・カメラ Suica カード」** — Bic Camera 提携カード
- **JRE CARD / JRE CARD ゴールド** — JRE POINT 還元強化
- **JAL カード Suica**（[[card-issuers/jal-card|JAL カード]] との共同発行、航空 + 鉄道マイル統合）
- **大人の休日倶楽部ジパング** — シニア向け JR 割引カード
- **ビューゴールドプラスカード** — ゴールド級
- **ビューカード プレミアム** — 上位層

## 4. 主要 KPI（公表ベース）

- **会員数**：約 600 万人超（JR East グループ統合報告書ベース）
- **Suica チャージ取引高**：オートチャージ + JRE POINT 還元の主力チャネル
- **JRE POINT 経済圏会員**：1,900 万人超（JR East 全体、Suica + Viewcard 統合 ID）
- **取扱高**：非公開（親会社 JR East の「IT・Suica 事業」セグメントに統合）

## 5. 業界ポジション

- **鉄道系アフィニティーカード**として **ICOCA（JR 西日本）・PASMO 提携カード（私鉄各社）・TOICA（JR 東海）** と並ぶ交通系カード経済圏の中核
- **JR East の Suica + JRE POINT エコシステム** の "唯一の専属クレジットカードイシュア" として独自ポジション
- **JR East の MaaS（モビリティ・アズ・ア・サービス）戦略** において重要な決済ハブ
- 競合：[[card-issuers/jal-card|JAL カード]]（航空）、[[card-issuers/credit-saison|セゾン]]・[[card-issuers/epos-card|エポス]]・[[card-issuers/seven-card-service|セブンカード]]（流通）、[[card-issuers/aeon-financial-service|イオン]]（流通系）
- **モバイル Suica** の普及・拡大に伴い、決済プラットフォームとしての重要性が拡大中

## 6. 沿革

| 年月 | 事項 |
|------|------|
| 1980 年代後半 | JR 東日本がクレジットカード事業を内製化 |
| 2001-11 | Suica サービス開始 — ビューカードと Suica の一体型カード商品が拡大 |
| 2007-03 | モバイル Suica サービス開始 |
| 2009-10-01 | 株式会社ビューカード設立（JR 東日本クレジットカード事業部の分社化） |
| 2010 | JRE POINT 制度の前身「ビューサンクスポイント」運用拡充 |
| 2016-06 | JRE POINT 統合 — Suica + Viewcard + JRE POINT が 1 つの ID 経済圏に統合 |
| 2019-10 | モバイル Suica 機能拡張、Apple Pay / Google Pay 対応進展 |
| 2022 | ビューカード プレミアム カードローン拡充（銀行代理業者ライセンス活用） |
| 2024-04 | 従業員数 590 名、JR East 統合 ID に Suica + Viewcard + JRE POINT が連結 |

## 7. 関連企業・取引先

- **親会社**：[[card-issuers/jr-east-financial|JR East Financial]] / 東日本旅客鉄道
- **国際ブランド**：JCB・VISA・Mastercard
- **共同発行カード**：[[card-issuers/jal-card|JAL カード Suica]]
- **加盟店パートナー**：Bic Camera（ビックカメラ Suica カード）、JR East エキナカ加盟店ネットワーク（NewDays・エキュート等）
- **Suica チャージ・モバイル Suica**：JR East のシステム基盤と連動

## 8. なぜ独立ページとして必要か

JR East の本体 [[card-issuers/jr-east-financial|JR East Financial]] ページからカード発行体としてのライセンス境界・銀行代理業者ライセンス・JRE POINT エコシステム・Suica 一体型カード戦略を分離して独立比較ベンチマークが必要。**「銀行代理業者を保有するカード会社」** という珍しい複合ライセンスを正確に記録するためにも必須。
