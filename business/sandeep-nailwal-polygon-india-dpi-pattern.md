## Mechanism / How it works

国家级抱团的"反对手"机制:**1) 监管栏**(ARC 是政府许可,与 RBI LRS 兼容,仅企业账户可铸造)→ 抵御美国 GENIUS Act 跨境压力;**2) 大企业栏**(Reliance Jio 独家合作)→ 提供 4.5 亿用户底盘,其他链(Ethereum / Solana / TON)无法企及;**3) 主权叙事栏**(反"事实美元化",代表 G20 新兴市场对美元稳定币的反制立场)→ 把项目命运绑在国家利益,获得长期政治保护。

诊断模板:当某 L1 / L2 链方在"全球市场失意"(无 Tier-1 美国机构客户),开始 pivot 到"国家级抱团"路径时,**监控指标 = 该国监管机构的 explicit endorsement + 国家级大企业独家合作 + 主权叙事在 G20 / BIS 场合发声**。三栏齐到 = pivot 成功(Polygon 案例);仅一栏 = 短期回光返照。

## Origin & evolution

源头 2017 Polygon(原 Matic Network)在印度创立,2021 unicorn + decacorn。但 2023-2025 zkEVM 路线失败 + Solana / Base 抢市场 → Polygon 全球估值压力。2025-06 Sandeep 接任 Foundation CEO,宣布"零到一"战略重启:弃 zkEVM、All-in AggLayer、PoS 升级 GigaGas。同时启动**印度抱团**:2025-01 Reliance Jio 独家合作公布,2026-01 Polygon Open Money Stack 发布,2026-Q1 ARC 卢比稳定币上线。Sandeep 个人 X 粉丝 250 万+ 是印度 Web3 最大 KOL,把项目转型与国家叙事绑定。复制候选:CELO + 非洲 / VeChain + 越南?

## Counterpoints

- "国家级抱团"代价 = 在该国之外的市场吸引力进一步下降(印度抱团 = 美国 / 欧盟客户难拓展)
- 单一国家叙事高度依赖该国政治稳定 + 大企业关系 → Reliance / RBI 政策变可能瞬间逆转
- 反美元叙事在 G20 内部存在分裂(印度反美元 vs 日本 / 韩国对美元友好)→ 联盟规模受限
