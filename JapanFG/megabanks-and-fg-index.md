# JapanFG megabanks and financial groups sub-index

## Overview

This sub-index routes the **mega-bank and financial-group holding-company** layer of the JapanFG namespace: the three mega-bank groups, telecom / platform financial groups, the postal financial group, and the independent internet financial group. It is a discovery surface over entity pages that remain in [[JapanFG/INDEX|JapanFG]].

Use this page when the question is **"which megabanks and fg entities does JapanFG cover?"** It lists 19 entity pages by institution type. Each link points to a page that physically lives in `JapanFG/`; this sub-index adds discoverability without moving any file.

## Placement Rule

- **Entity pages stay in `JapanFG/`.** This sub-index only links to them with bare `[[JapanFG/slug]]` wikilinks; it does not move, rename, or duplicate any entity file.
- Membership here is derived from each entity page's frontmatter `tags:` taxonomy. An entity may legitimately also appear in an adjacent sub-index when its activity spans types.
- This is a navigation surface, not a registry of record. Where an FSA registry index already exists for a type, treat that registry page as the source of truth.
