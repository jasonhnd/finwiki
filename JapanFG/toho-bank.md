# 東邦銀行 (Toho Bank)

## TL;DR

**福島県唯一の第一地方銀行・福島県内シェア No.1**、本店福島市、東証 PRIME 8346。1941-11-21 戦時統合で福島県内 5 行を統合して設立、1949 東証上場。**単独銀行体制**を維持し、2015 [[chiba-bank]] 主導の **TSUBASA アライアンスに参加**してシステム共同化と広域連携を進める。**2011-03-11 東日本大震災 + 福島第一原発事故の二重被災**を経験し、復興融資と原発事故関連帰還困難区域取引の中核を担う ★。略称「とうほう」。^[extracted]

## 1. 公司结构

**正式名**: 株式会社東邦銀行 ^[extracted]
**英名**: The Toho Bank, Ltd. ^[extracted]
**証券コード**: 東証 PRIME 8346 ^[extracted]
**設立**: **1941-11-21**(戦時統合で福島県内 5 行統合)^[extracted]
**本店**: 福島県福島市大町 3-25 ^[extracted]
**機関設計**: 監査役会設置会社 ^[ambiguous]
**主要監督**: 金融庁、東北財務局、日銀福島支店

### 業態の特徴

- **単独銀行・持株会社化せず** ^[extracted]
  - [[77-bank]]・[[chiba-bank]]・[[JapanFG/shizuoka-fg|shizuoka-bank]] と同様、持株会社化を選ばない単独路線
  - 隣県の青森銀行→[[proclair-hd]]・足利銀行→[[mebuki-fg]]・常陽銀行→[[mebuki-fg]] が持株会社統合した中で独立を維持
- **TSUBASA アライアンス加盟行** ^[extracted]
  - 2015 [[chiba-bank]] 主導設立の地銀アライアンス、11 行協業
  - システム共同化・海外サポート・商品共通化で持株会社化を回避しつつスケールメリット獲得
- **福島県内シェア圧倒** ^[inferred]
  - 福島県内預金シェア No.1、商業銀行として唯一の県内第一地銀

## 2. 重要历史

### 戦時統合期: 一県一行政策下の発足

| 年月 | 事象 |
|---|---|
| 1872 | 国立銀行条例公布 |
| 1878〜1898 | 福島県内に複数の国立銀行設立(後の前身行群)|
| 1936-1945 | 戦時統合・一県一行政策 |
| **1941-11-21** | **株式会社東邦銀行 設立**(福島県内 5 行統合)^[extracted] |
| **1949** | **東京証券取引所 上場** ^[extracted] |

### 戦後〜バブル期

| 年月 | 事象 |
|---|---|
| 1950s〜1980s | 高度成長期に福島県全域に支店網拡大、首都圏・宮城・茨城等の隣県進出 |
| 1990s | バブル崩壊後の地銀業界再編期も単独路線維持 |

### 東日本大震災 + 原発事故(最重要)★

| 年月 | 事象 |
|---|---|
| **2011-03-11** | **東日本大震災発生** ★ 福島県沿岸部支店が津波被害、内陸部も地震被害 ^[extracted] |
| **2011-03-12** | **東京電力福島第一原発事故** ★ 双葉郡・浜通り北部の支店が帰還困難区域・避難指示区域に指定、長期休業 ^[extracted] |
| 2011-03〜04 | 義援金・緊急融資、被災者向け預金払戻し特例措置(通帳・印鑑なし対応)^[extracted] |
| 2011-04 以降 | 復興支援室設置、原発事故関連の二重ローン問題対応 ^[inferred] |
| 2012-2015 | 復興 5 年集中期間: 東日本大震災復興特別貸出制度、グループ補助金、原子力損害賠償関連融資 |
| **2015** | **TSUBASA アライアンス参加**([[chiba-bank]] 主導の 11 行アライアンス発足時メンバー)^[extracted] |
| 2017 以降 | 福島イノベーション・コースト構想関連融資、復興公営住宅取引 ^[inferred] |
| **2022-04** | **東証 PRIME 上場**(市場区分見直し)^[extracted] |
| 2024〜 | 日銀政策金利正常化への対応、ポスト復興フェーズの人口減対策 ^[inferred] |

## 3. 战略与竞争

### 地理的地盤

- **メイン**: 福島県全域(中通り・浜通り・会津の 3 地域)^[extracted]
- **隣県**: 宮城・山形・栃木・茨城・新潟各県に支店配置 ^[inferred]
- **首都圏**: 東京・関東各支店(福島出身企業・個人向け)^[inferred]

### 競合構図

| 競合 | 本拠 | 関係性 |
|---|---|---|
| **大東銀行** | 福島県郡山市 | 福島県第二地銀、中小・小規模法人で競合 |
| **福島銀行** | 福島県福島市 | 福島県第二地銀、SBI 系([[sbi-regional-bank-hd]] 関連)^[inferred] |
| **[[77-bank]]** | 仙台 | 隣県宮城の東北 No.1 地銀、首都圏・東北縦断取引で競合 |
| **[[jimoto-hd]]** | 仙台・山形 | 仙台銀+きらやか銀、隣県地銀グループ |
| **[[mebuki-fg]]** | 茨城・栃木 | 常陽銀行+足利銀行、隣県統合地銀 |
| **メガバンク**([[mufg]] / [[smfg]] / [[mizuho-fg]])| 東京 | 大企業・首都圏取引で競合、地元中小では東邦優位 |
| **信金・信組**(福島信金等)| 各地 | 地元零細法人・個人 |

### 戦略の柱(推定)^[inferred]

- **福島県内シェアの圧倒的維持**: 個人預金・地元中小法人取引
- **復興 → ポスト復興フェーズへの転換**: 震災・原発事故復興融資のリーダーから、人口減・原発事故影響地域の構造転換金融へ ^[inferred]
- **TSUBASA アライアンス活用**: システム共同化・海外サポート・商品共通化で単独銀行のままスケールメリット獲得
- **首都圏支店戦略**: 福島出身企業の本社機能・福島縁故個人向け
- **DX / 営業店再編**: 全国地銀同様、店舗削減・タブレット完結型化
- **福島イノベーション・コースト構想**: 国家プロジェクト関連の地元金融機関としての関与 ^[inferred]

### 福島原発事故関連の特殊事情

- **帰還困難区域の取引**: 双葉郡(大熊・双葉・浪江・富岡等)の旧支店顧客の口座管理・遠隔対応 ^[inferred]
- **二重ローン問題**: 震災前ローンと再建後ローンの並存問題への対応 ^[inferred]
- **東京電力からの原子力損害賠償**: 個人・法人賠償金の入金窓口役 ^[inferred]
- **廃炉関連企業取引**: 廃炉作業関連の地元法人取引 ^[inferred]

- **直近政策論点**:

  - 2011〜 福島復興特別法・福島復興再生特別措置法下の地元金融機関としての役割
  - 2020s〜 福島イノベーション・コースト構想・水素・再エネ・廃炉産業集積関連融資
  - 2024〜 地銀統合再編圧力下で単独路線継続の妥当性 ^[inferred]

## Counterpoints

- 「単独路線の限界論」: 人口減・原発事故影響・IT 投資負担増の中で持株会社化や広域統合を選ばない選択の合理性 vs 規模の経済論 ^[inferred]
- 「TSUBASA で代替可能」観: 持株会社化せずとも TSUBASA で実質的なグループ機能 → 千葉銀モデル踏襲 ^[ambiguous]
- 「原発事故関連リスク」: 帰還困難区域・廃炉関連の貸出ポートフォリオは長期 unknowable リスクを内包 ^[inferred]
- 「福島県人口減と復興出口の構造課題」: 復興期から人口減フェーズへの移行で、地元金融需要の構造変化 ^[inferred]
- 「[[sbi-regional-bank-hd]] 系・福島銀行との競合」: 県第二地銀の SBI 連携で県内競争環境が変化 ^[ambiguous]
