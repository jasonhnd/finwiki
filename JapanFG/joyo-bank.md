# 株式会社常陽銀行 (Joyo Bank)

## TL;DR

常陽銀行は茨城県を地盤とする [[JapanFG/mebuki-fg|めぶき FG]] の中核銀行。常磐銀行と五十銀行が合併して 1935年7月30日に設立され、2016年10月に足利 HD との経営統合でめぶき FG を発足させた。茨城・栃木・首都圏北部、製造業・農業・筑波研究学園都市・日立系サプライチェーンの地域金融を見るための anchor。

## 1. 公司结构

| 項目 | 内容 |
|---|---|
| 正式名 | 株式会社 常陽銀行 |
| 本店 | 茨城県水戸市南町2丁目5番5号 |
| 創立 | 1935年7月30日 |
| 親会社 | [[JapanFG/mebuki-fg|めぶきフィナンシャルグループ]] |
| 統合 | 2016年10月、足利 HD と経営統合 |
| 主地盤 | 茨城県、北関東、首都圏北部 |

公式・親会社資料では、2025-09-30 時点で資本金 851億円、従業員 3,112人、国内 181 店舗、海外 4 駐在員事務所、総資産 12兆8,287億円、預金 10兆4,059億円、貸出金 7兆6,250億円、連結自己資本比率 13.13% とされる。

| 业务线 | 内容 | 重要性 |

| グループ連携 | 足利銀行・めぶき証券・めぶきリース | 北関東広域化 |

## 3. 战略与竞争

- **北関東広域モデル**: 常陽銀行は茨城、足利銀行は栃木を主地盤に、めぶき FG として県境を越えた地銀グループを作る。
- **筑波・日立・製造業接点**: 茨城には研究機関、工業団地、日立系・素材・食品・農業関連企業が集まるため、法人支援の厚みが重要。
- **首都圏隣接地銀**: 東京圏に近いことで、メガバンク・ネット銀行・証券会社との競争も受ける。
- **七十七銀行との比較**: [[JapanFG/77-bank|77-bank]] が東北中核の単独大手であるのに対し、常陽は広域持株会社の一角として再編済み。

## 4. 监管与政策

- 銀行本体は銀行法、親会社は銀行持株会社規制の対象。
- 上場持株会社 [[JapanFG/mebuki-fg|mebuki-fg]] を通じて東証・金商法開示の対象。
- 地域金融政策では、北関東広域の事業承継、産業転換、自治体連携、地域 DX 支援が論点。

## Counterpoints

- 足利銀行との統合シナジーは、顧客基盤が県別に分かれるため過大評価されやすい。
- 首都圏近接の利点は、同時にメガバンク・ネット金融との競争圧力でもある。
- 茨城県内の人口・産業分布は広く、店舗・営業人員の最適配置が難しい。
