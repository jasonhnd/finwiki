# ハナ銀行 在日支店


## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[JapanFG/woori-bank-japan|ウリィ銀行 東京支店]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

ハナ銀行 在日支店は、金融庁の外国銀行支店一覧に掲載される Republic of Korea / Hana Bank の在日銀行支店。旧 Korea Exchange Bank / Hana 統合後の Japan branch route として、東京・大阪・福岡の韓国系 banking corridor を担う。

## 2. Operating model

Hana's Japan branch site provides public information for Japan-branch products, deposit notices, remittance, policy disclosures, and branch operations. JBA lists the bank with the Japan code and public support contacts. The useful boundary is a foreign-bank branch network, not a Japanese bank subsidiary.

Because Hana's Japan site is hosted under the legacy KOEX domain, wiki aliases should preserve both Hana and KEB Hana names.
