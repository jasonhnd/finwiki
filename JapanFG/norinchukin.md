# 農林中央金庫 (Norinchukin Bank)


## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[JapanFG/mufg|三菱 UFJ FG (MUFG)]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

**世界最大級の協同組合系金融機関**。1923 産業組合中央金庫として設立、1943「農林中央金庫」改称、1986/2001 農林中央金庫法改正で現代的銀行業務へ。**株式会社ではない特殊法人**で、出資者は JA（農協）・JF（漁協）・森林組合等の系統団体。[[banking/ja-bank-system-japan|JA Bank system]] と [[banking/jf-marine-bank-system-japan|JF Marine Bank system]] の national central institution layer を担い、系統信用事業の頂点 + **国内屈指の機関投資家**という二重性格を持つ。総資産約 80-100 兆円規模 ^[ambiguous]。**2008 リーマン・ショックで米サブプライム関連約 1.5 兆円損失**、**2024-06 米国債等の含み損対策で出資 JA から 1.2 兆円規模の資本増強**を発表 ^[extracted] — 2 度の大規模機関投資損失がガバナンス論点として浮上。

## 1. 会社概要

**正式名**：農林中央金庫 ^[extracted]
**英名**：The Norinchukin Bank
**略称**：農林中金 / 農中 / Nochu
**設立**：1923（産業組合中央金庫として）^[extracted]
**現法人発足**：1943-03（戦時統合で「農林中央金庫」改称）
**根拠法**：農林中央金庫法（昭和 61 年法律第 93 号、1986 全面改正、2001 大幅改正）^[extracted]
**形態**：**協同組合系金融機関の特殊法人**（株式会社ではない、上場もしていない）^[extracted]
**本店**：東京都千代田区有楽町 1-13-2
**機関設計**：理事長 + 経営管理委員会（株式会社の取締役会に相当）^[inferred]

### 出資構造（系統団体出資）^[extracted]

- **主要出資者**：JA（農業協同組合・連合会）・JF（漁業協同組合・連合会）・森林組合等の系統団体
- **株式会社の「株主」に相当するのが「出資者」**：JA 全中・JA 全農・JA 共済連等の上部団体経由で出資
- **配当先**：系統団体への配当が本業の一つ（系統還元）
- 上場はしていない（**東証/JPX 上場銘柄ではない**）

### 主要拠点

```
農林中央金庫（本店：東京・有楽町）
  ├── 国内支店：札幌・仙台・名古屋・大阪・福岡 等
  ├── 海外拠点：
  │     ├── ニューヨーク支店
  │     ├── ロンドン支店
  │     └── シンガポール支店
  ├── 系統団体：
  │     ├── JA バンク（農業協同組合・全国連合会含む系統）
  │     ├── JF マリンバンク（漁業協同組合系統）
  │     └── JForest（森林組合系統）
  └── 子会社・関連会社：
        ├── 農中信託銀行
        ├── 農林中金全共連アセットマネジメント（共同出資）^[ambiguous]
        └── その他金融サービス子会社
```

## 2. 事業セグメント・マップ

| セグメント | 主要事業者 | 特徴 |
|---|---|---|
| **系統信用事業（中央銀行機能）**| 農林中央金庫 本体 | JA バンク・JF マリンバンク・JForest 系統の中央銀行・最終貸し手 |
| **機関投資業務** | 本店 + NY/London/Singapore | **本業並みの収益源** — 国内外債券・株式・オルタナティブ・証券化商品 |
| 系統団体向け融資 | 本体 | 農業協同組合・漁業協同組合・森林組合への資金供給 |
| 農林水産業向け融資 | 本体 + 系統経由 | 農業法人・漁業者・林業者への直接 + 系統経由融資 |
| 国際業務 | NY/London/Singapore | 外貨資金調達・海外投資・国際決済 |
| 信託・年金 | 農中信託銀行 | 系統団体年金・運用受託 |
| アセットマネジメント | 系統共同出資 AM | 系統運用商品・公募投信 |
| クレジットカード等 | （過去）三菱 UFJ ニコス保有歴 | 2017-10 MUFG に売却 |

## 3. 戦略・競争

### 二重性格：系統中央銀行 + 国内屈指の機関投資家

- **「上流」：JA バンク系統の中央銀行** — 全国の JA から集まる**潤沢な貯金（系統預け金）を一手に集約**し、運用する仕組み
- **「下流」：機関投資家として国際運用** — 国内農業向け資金需要だけでは消化しきれない巨大資産を、国内外の証券・債券・オルタナティブで運用
- → **「農業金融」と称しながら実態は世界最大級の機関投資家**という構造的特徴

### 国際分散投資戦略

- **米国債・社債・MBS** 中心の外貨建て運用
- ニューヨーク・ロンドン・シンガポール 3 拠点で運用・調達
- 2008 サブプライム / 2024 米金利急騰の **2 度とも米国市場で大規模損失**

### 2024-2025 ポートフォリオ再構築 ^[extracted]

- 2024-06 公表の中計：米国債等の含み損実現売却 + 資本増強 + リスク資産再配分
- 出資 JA 等への配当圧縮 → 系統内ガバナンス論点
- 中期的に外債依存度を下げ、円債・国内クレジット・代替資産へシフト ^[inferred]

### 系統金融再編

- JA バンク全国システム（2002〜）の中央プレーヤー
- 都道府県信連（信用農業協同組合連合会）との役割分担再編
- 漁協系統（JF マリンバンク）・森林組合系統（JForest）でも同様の中央銀行機能

### 競争ポジション

- **メガバンク（[[mufg]] / [[smfg]] / [[mizuho-fg]]）との比較**：商業銀行業務は限定的、本業は系統金融 + 機関投資
- **生保（[[nippon-life]] / [[dai-ichi-life]]）との比較**：機関投資家としては類似ポジション、運用残高で同じ序列
- **政策金融機関（[[jfc]] / [[dbj]]）との比較**：農林水産業向け融資で部分的に競合 / 補完

## 4. 規制・政策

- **主管**：金融庁（FSA）+ **農林水産省**（系統金融部分）— 二重監督が特徴 ^[extracted]
- **根拠法**：農林中央金庫法（昭和 61 年法律第 93 号）
- **預金保険**：農水産業協同組合貯金保険機構（一般金融機関の預金保険機構とは別建て）^[inferred]
- **国際規制**：バーゼル III 自己資本比率規制対象（メガバンクと同様の規制水準）^[inferred]
- **直近政策論点**：
  - 2024-2025 米国債含み損問題 → 金融庁・農水省の合同モニタリング強化 ^[inferred]
  - 出資 JA への配当・系統還元水準の妥当性
  - 系統金融再編（JA バンク統合・信連との役割分担）
  - 機関投資家としての ESG / 気候リスク開示

### 代表役職（理事長）^[ambiguous]

- 理事長は経営管理委員会で選任。歴代理事長は農水省・財務省 OB と内部昇格の組み合わせが多いと一般に知られる ^[inferred]
- 具体的氏名・任期は公表資料を参照（本ノートでは個別氏名は記載せず）

## Counterpoints

- **「世界最大の協同組合銀行」vs「実態は国際機関投資家」**：表向きは農業金融、実態の収益源とリスクは国際市場運用に集中 ^[extracted]
- **2008 サブプライム + 2024 米債含み損の 2 度の大規模機関投資損失** → ガバナンス・リスク管理体制の論点 ^[ambiguous]
- **出資 JA 配当圧迫リスク**：機関投資損失 → 出資 JA への配当縮小 → 系統団体財務逼迫の連鎖懸念 ^[inferred]
- **金利感応度の高さ**：保有債券残高が巨大なため、金利上昇局面でデュレーション・ミスマッチが直撃する構造 ^[extracted]
- **二重監督（金融庁 + 農水省）の効率性**：政策金融的性格 vs 商業金融的実態のミスマッチ ^[inferred]
- **出資者ガバナンスの限界**：JA 全中等の上部団体経由出資 → エンドユーザー（個別 JA）からの直接的経営監視が弱い構造 ^[inferred]
