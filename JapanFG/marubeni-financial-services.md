# 丸紅フィナンス (Marubeni Finance Corporation)

## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[JapanFG/mizuho-lease|みずほリース]] (the listed leasing arm where 丸紅 became a strategic shareholder in 2024) and [[JapanFG/fuyo-lease|芙蓉総合リース]] (the 芙蓉グループ peer leasing arm — 丸紅 is part of the 芙蓉 community). For peer 商社系 group-treasury arms, see [[JapanFG/mitsubishi-corp-finance|三菱商事FS]], [[JapanFG/mitsui-bussan-financial-services|三井物産FS]], [[JapanFG/sumitomo-corp-financial-management|住友商事FM]], [[JapanFG/itochu-finance|伊藤忠ファイナンス]]. See [[banking/INDEX|banking index]] for the broader regulatory boundary.

## TL;DR

丸紅フィナンス は **丸紅 (8002)** 系の **グループ内金融サービス会社**。丸紅グループの **CMS / 一括ファクタリング / グループファイナンス / 為替・金利リスクヘッジ仲介 / 海外送金** を担う。リース系 (外向け asset finance) は、2024 年に 丸紅 が戦略株主として参画した [[JapanFG/mizuho-lease|みずほリース]] と、芙蓉グループ系の [[JapanFG/fuyo-lease|芙蓉総合リース]] が中心。丸紅は「**外向けリース = みずほリース + 芙蓉総合リース / 内向け = 丸紅フィナンス**」の構造。^[inferred]

### 丸紅と芙蓉グループ

丸紅は **旧 富士銀行系 (現 [[JapanFG/mizuho-fg|みずほ FG]]) の芙蓉グループ** の中核商社の一つ。同じ芙蓉グループには [[JapanFG/fuyo-lease|芙蓉総合リース]]・日産・キヤノン・東京海上 ([[JapanFG/tokio-marine|東京海上HD]])・日清製粉 等が並ぶ。^[inferred] よって丸紅系金融機能は、単独の「丸紅閉じ」ではなく、芙蓉グループ内連携の文脈でも理解する必要がある。

### 丸紅系金融機能の配置

| 層 | 主体 | 役割 |
|---|---|---|
| **外向け銀行 (主取引)** | [[JapanFG/mizuho-fg|みずほ FG]] / [[JapanFG/mizuho-bank|みずほ銀行]] | 商業銀行・対外金融サービス |
| **外向けリース (主)** | [[JapanFG/mizuho-lease|みずほリース]] | 2024 年に 丸紅 が戦略株主として参画 |
| **外向けリース (芙蓉)** | [[JapanFG/fuyo-lease|芙蓉総合リース]] (8424) | 芙蓉グループ系、丸紅も歴史的株主 |
| **損保 (芙蓉系)** | [[JapanFG/tokio-marine|東京海上HD]] | 芙蓉グループ系主要損保 |
| **内向け / グループ金融** | **丸紅フィナンス (本ページ)** | グループ各社の internal banking / CMS / 内部ファクタリング |

### 2024 年 丸紅 + みずほリース 資本関係拡大

2024-05 に 丸紅 が [[JapanFG/mizuho-lease|みずほリース]] に対する戦略株主としての出資を拡大することが発表された。これは みずほ FG (旧 IBJ + 旧 富士 + 旧 第一勧銀 の 3 遺伝子) と 丸紅 (芙蓉グループ) という旧 富士銀行系の **資本紐帯の再構築** の文脈で理解できる。^[inferred] みずほリース が 「みずほ系 + 丸紅 (商社系) の hybrid platform」へ進化することで、丸紅本体のリース機能委託先としての位置付けがより明確になる。

## 3. 战略与竞争

### 丸紅の事業構造と金融機能

丸紅は **電力 / プラント / 自動車・モビリティ / 航空宇宙 / 食料・アグリ / 生活産業 / 金属・鉱物 / エネルギー / 化学品** といった事業ポートフォリオを持つ。特に **電力 IPP / 海外プラント** で世界的なポジションを持ち、長期与信・プロジェクトファイナンス案件が多い。丸紅フィナンス はこれらの案件における内部金融補助 + 為替/金利リスクヘッジ集約を担う。^[inferred]

## 4. 监管与政策

- **主管**：金融庁 (貸金業 / 為替業務関連)
- **業界団体**：(商社系内部金融としてリース事業協会への加盟は限定的) ^[inferred]
- **規制論点**：
  - グループ内金融取引の **移転価格税制** (特に海外 IPP / プラント案件)
  - グループファクタリングと **割賦販売法 / 貸金業法** の境界
  - 連結会計上の **内部取引消去**
- **直近政策論点**:
  - 2024〜 [[JapanFG/boj-monetary-policy|日銀政策金利正常化]] → グループ内金利付与の再評価
  - 2024〜 みずほリース 資本関係拡大 → 外向け asset finance 機能の強化
  - 海外電力 IPP 案件の脱炭素圧力 (石炭火力撤退 / 再エネ shift) に伴う与信構造変化 ^[inferred]

## Counterpoints

- 「内部金融子会社」はガバナンスが形式的になりがち ^[inferred]
- 外部開示が限定的、業績規模は推測ベース ^[inferred]
- 丸紅は芙蓉グループ + みずほ FG + (一部) 三井系 と複数の銀行系列に跨る関係を持ち、内向け金融の優位性は他商社より相対的に低い可能性 ^[ambiguous]
- 2024 みずほリース 戦略株主化により、丸紅フィナンス の役割は (相対的に) 内向き機能に集中する方向か、それとも他金融機能も統合する方向か未確定 ^[ambiguous]
