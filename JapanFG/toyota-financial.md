# トヨタファイナンシャルサービス (TFS)


## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[JapanFG/mufg|三菱 UFJ FG (MUFG)]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

**トヨタ自動車 100% 子会社、グローバル自動車金融統括会社**（2000-07-03 設立）。世界 **30 ヶ国超**で自動車ローン・リース・保険・クレカを展開。**米 Toyota Financial Services Americas（旧 TMCC）は米国大手自動車金融**として Ford Motor Credit・Ally Financial に伍する規模。国内コアは **トヨタファイナンス株式会社**（TS キュービックカード・自動車ローン）。新車販売 + ローン + リース + 保険 + クレカの**総合パッケージ**でディーラー網と一体化、Toyota ブランドの収益源として 2024 年以降は EV / コネクテッドカー連動金融へシフト中。非上場（トヨタ自動車 連結子会社）。

## 1. 公司结构

**正式名**：株式会社トヨタファイナンシャルサービス
**英名**：Toyota Financial Services Corporation
**設立**：2000-07-03
**本社**：愛知県名古屋市西区（トヨタ自動車 グループ拠点近接）
**上場**：非上場 — トヨタ自動車（東証 PRIME 7203）100% 連結子会社
**親会社**：トヨタ自動車株式会社（[[manufacturing/toyota-motor|toyota-motor]]）
**位置付け**：トヨタグループの**金融事業統括持株会社**

### 業態カテゴリ

- **自動車ローン**: 新車購入・中古車購入向けクレジット
- **リース**: オペレーティングリース・残価設定型リース（残クレ）
- **保険**: 自動車保険・生命保険の代理販売・持株
- **クレジットカード**: TS キュービックカード（トヨタディーラー会員向け）
- **法人向けフリート金融**: 大口法人車両管理リース

## 4. 战略与竞争

### 総合パッケージ戦略

トヨタの**販売店金融（dealer-financed sales）**モデル — 新車購入時に **車両 + ローン + リース + 自動車保険 + TS キュービックカード**をワンストップ提案。ディーラー網と一体化した送客モデルで、銀行系自動車ローンや独立系金融に対する**販売チャネル優位**を維持。残価設定型ローン（残クレ）は中古車市場と連動し、TFS 側でリセール価値リスクを抱える構造。

### グローバル分散

- **米州（TMCC）**: 規模最大、米国自動車金融市場で Ford Motor Credit・Ally Financial に伍する
- **欧州（Toyota Kreditbank）**: 独中心、EU 全域カバー
- **アジア・豪州・中南米**: トヨタ販社の進出と連動、新興国市場で先行展開
- 通貨分散・地域景気分散により、単一市場依存リスクを低減

### EV / コネクテッド連動金融（2024〜）

EV 普及で**残価評価モデルが従来 ICE 車と乖離**（バッテリー劣化・急速変化する技術世代）→ 残価設定の精緻化が課題。コネクテッドカーの走行データを活用した**使用量連動ローン / 保険**の試行（テレマティクス UBI）。充電インフラ事業者との連携金融商品も模索中。

### 競合

| 競合 | 親会社 | 特徴 |
|---|---|---|
| 日産 NMA（NMAC）| 日産自動車 | 北米中心、日産ディーラー連動 |
| ホンダファイナンス | ホンダ | 国内 + 北米、Honda Cars 系 |
| [[orix-corp]] | 独立系（持株分散） | 法人リース・個人カーリース総合 |
| 三菱 HC キャピタル | [[mufg]] グループ | 旧三菱 UFJ リース + 日立キャピタル合併、大型法人 |
| [[tokyo-century]] | みずほ + 伊藤忠 | リース・モビリティ、Avis 関連等 |
| 銀行系自動車ローン | メガバンク・地銀 | 低金利だが販社送客力で劣勢 |

トヨタ販社チャネル × 残クレ商品設計の組合せで、独立系・銀行系に対する**販売現場での優位**を維持。

## 5. 监管与政策

- **国内主管**: 金融庁（FSA）— 貸金業法・割賦販売法・保険業法
- **米国**: 連邦消費者金融保護局（CFPB）・各州金融監督
- **欧州**: 各国金融監督（独 BaFin 等）+ ECB / 欧州銀行監督
- **アジア**: 各国金融監督（タイ BoT・インドネシア OJK 等）
- **直近政策論点**:
  - 2024〜 EV 残価評価モデルの透明化（消費者保護論点）
  - 2025〜 コネクテッドカーデータ × 金融の個人情報保護整理
  - 2025〜 米国 CFPB の自動車金融 APR 開示強化動向

## Counterpoints

- 「販社送客モデルの持続性」: 直販 / D2C・オンライン中古車プラットフォーム（米 Carvana 等）台頭でディーラー経由率低下リスク ^[inferred]
- 「残価リスクの構造的増大」: EV 技術世代交代 + 中古車相場ボラ拡大 → 残クレの TFS 側残価損リスク ^[inferred]
- 「地域分散の収益貢献」: 新興国は規模拡大中だが、為替・カントリーリスクで利益ボラ拡大 ^[ambiguous]
- 「銀行系・独立系との金利競争」: 銀行系オートローンは低金利で TFS の販社優位を浸食する余地 ^[inferred]
