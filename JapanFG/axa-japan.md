# アクサ・ジャパン (AXA Japan)


## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[JapanFG/prudential-japan|プルデンシャル・ジブラルタ・ファイナンシャル (Prudential Japan)]] for peer / contrast context and [[insurance/INDEX|insurance index]] for the broader system / regulatory boundary.

## TL;DR

日本最大級の外資系生損保グループ。親会社は仏 **AXA SA**（世界最大級の保険・運用グループ、保険料収入グローバル Top 3）。前身は 1932 年創業の **日本団体生命保険**で、1994 年に旧 UAP 系（後の AXA グループ）が出資参画、1999 年に「アクサ生命」へ改称。同年 **アクサ損害保険** を新設し、損保業界初の本格的ネット直販モデルを投入。生損保＋ネット直販生保（アクサダイレクト生命）の **3 チャネルハイブリッド** が特徴。

## 1. グループ構造

**最終親会社**：AXA SA（仏、Euronext Paris 上場、CAC 40 構成銘柄）
**日本持株会社**：アクサ・ホールディングス・ジャパン株式会社（旧称: アクサジャパンホールディング）
**主な日本拠点本社**：東京都港区白金 1-17-3 NBF プラチナタワー（アクサ生命）
**従業員規模**：日本グループ全体で約 8,000 〜 9,000 名（営業職員含む、^[inferred]）

### 重要历史（年表）

| 年月 | 事象 |
|---|---|
| 1932 | **日本団体生命保険** 創業（団体生保専業として戦前発足）|
| 1994 | 旧 UAP 系（後の AXA グループ）が日本団体生命に資本参加 |
| 1996 | 仏 UAP と AXA が合併、世界最大級の保険グループ「AXA-UAP」（後 AXA）誕生 |
| **1999** | **日本団体生命 → アクサ生命保険 改称**、同年 **アクサ損害保険 設立**（ネット自動車保険参入）|
| 2000 | グループ生損保再編、損保事業は ネット直販（テレマーケティング＋ Web）に特化 |
| 2000s 中盤 | **ニッセイ・ドット・コム生命** 等を経て **アクサダイレクト生命** へ統合 — ネット直販生保ブランド確立 |

| 2023〜2024 | グループ「Unlock the Future」戦略下で日本も健康・予防（ヘルスケア）領域へ拡張 |

## 2. 業務ライン

| セグメント | 主要事業者 | 特徴 |
|---|---|---|
| 個人生保（営業職員）| アクサ生命 | 終身・医療・がん・収入保障・変額。営業職員「ファイナンシャル・プラン・アドバイザー」 |
| 団体生保 | アクサ生命 | 創業以来の主力（旧 日本団体生命由来）。企業の従業員保障・退職年金 |

| ネット直販生保 | アクサダイレクト生命 | 医療・定期・収入保障に特化。月額数百円〜の低価格帯 |
| ネット直販損保 | アクサ損保 | 自動車保険・バイク保険・海外旅行保険。テレマ＋ Web |
| 資産運用 | AXA IM 日本拠点 | 機関投資家向け運用、グローバル AXA IM のローカルブランチ |
| 海外アシスタンス | アクサ・アシスタンス | 旅行保険付帯の現地救援サービス |

## 3. 戦略と競争

### ポジショニング

- **外資系生損保で最大級**：プルデンシャル系（[[prudential-japan]]）、メットライフ生命、マニュライフ生命、アフラック生命（[[aflac-japan]]）と並ぶ「外資系大手 5 社」の一角 ^[inferred]
- **ハイブリッドチャネル戦略**：日系大手（[[dai-ichi-life]] / [[nippon-life]] / 住友生命 / 明治安田生命）が営業職員中心、ネット系専業（[[JapanFG/lifenet|lifenet-insurance]]）がオンライン一本足なのに対し、アクサは **営業職員 + 銀行窓販 + ネット直販** の 3 チャネルを並存
- **損保はあえてネット直販**：[[JapanFG/tokio-marine|tokio-marine-hd]] / [[JapanFG/sompo|sompo-hd]] / [[JapanFG/msad|ms-and-ad-insurance]] の代理店チャネル中心の国内 3 メガ損保と差別化、ソニー損保・セコム損保・SBI 損保と同じ「ダイレクト系損保」セグメント

### グローバルレバレッジ

- **AXA グループの再保険プラットフォーム**：大型契約・カタストロフィリスクをグループ内再保険で吸収
- **グローバル資産運用**：AXA IM（運用残高 約 8000 億ユーロ規模、グループ全体）のグローバル債券・オルタナを日本契約者勘定に活用 ^[inferred]
- **健康・予防領域**：AXA Group の「Payer to Partner」戦略（単なる保険金支払者から、健康支援パートナーへ）を日本でも展開

## 4. 規制と政策

- **主管**：金融庁（FSA）監督。生損保ともに保険業法ベース
- **国際規制**：親会社 AXA SA は **G-SII（Global Systemically Important Insurer）** 指定歴があり、グループベースの ICS（保険資本基準）整備の影響を受ける ^[inferred]
- **ソルベンシー**：日本子会社単体は日本のソルベンシー・マージン比率規制、グループ連結は仏監督当局 ACPR ベース
- **直近政策論点**：
  - **2025〜 経済価値ベース ソルベンシー規制（ESR）導入**：日系大手と同様、内部モデル整備が論点
  - **代理店・営業職員チャネルのコンプライアンス**：高齢者向け外貨建一時払終身の説明責任強化
  - **ヘルスケア領域進出**：金融サービス仲介業 + 保険周辺サービスの境界整理

## Counterpoints

- **「外資系最大」言説の検証**：保険料収入で「外資系最大」とされる時期と社が変動。プルデンシャル系（ジブラルタ+プルデンシャル+プルデンシャル ジブラルタ 合算）、メットライフ、アフラックそれぞれが「最大」を主張した時期あり ^[ambiguous]
- **ネット直販生保の規模感**：アクサダイレクト生命は商品ラインナップが医療・収入保障に絞られており、保険料規模では [[JapanFG/lifenet|lifenet-insurance]] と並ぶ「ネット直販生保第二集団」レベル。第一集団は依然として営業職員・代理店チャネル ^[inferred]
- **損保事業の単独収益性**：アクサ損保はネット直販自動車保険市場でソニー損保に次ぐポジションとされるが、代理店系大手と比べると規模は二桁小さい ^[inferred]
- **営業職員規模の縮小傾向**：グループ全体としては営業職員チャネルの効率化（人数削減・1 人当たり生産性向上）方向。日系大手の数万人規模の営業職員ネットワークとは桁が違う ^[inferred]
