# 東京きらぼし FG (Tokyo Kiraboshi FG)

## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[JapanFG/fukuoka-fg|ふくおか FG (FFG)]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

**東京都内特化** の珍しい「東京専業地銀」グループ。2014-10-01 **東京都民銀行 + 八千代銀行** の経営統合で「東京 TY フィナンシャルグループ」発足 → 2016-04 **新銀行東京** を統合（旧石原都政の負の遺産処理）→ 2018-05-01 **3 行合併で「きらぼし銀行」誕生 + FG も「東京きらぼし FG」に改称**。地盤は東京都内（神奈川・千葉一部）に絞り込んだ希少な地銀モデル。中小企業金融 + 都心立地の富裕層 + 「きらぼしデジタル銀行」「UI 銀行」等のデジタル戦略 + BaaS / iBank 提携 で「東京エリアの中小・スタートアップ・富裕層」ニッチを取りに行く戦略 ^[extracted]。

## 1. 会社概要

**正式名**：株式会社東京きらぼしフィナンシャルグループ ^[extracted]
**英名**：Tokyo Kiraboshi Financial Group, Inc.
**証券コード**：東証 PRIME 7173 ^[extracted]
**設立**：2014-10-01（東京 TY フィナンシャルグループとして、東京都民銀行 + 八千代銀行 統合）^[extracted]
**現商号移行**：2018-05-01（東京 TY FG → 東京きらぼし FG 商号変更）^[extracted]
**本店**：東京都港区南青山 3-10-43 ^[extracted]
**地盤**：東京都内全域 + 神奈川・千葉一部 ^[extracted]

### 統合背景：3 行合併の経緯

- **東京都民銀行**（1951 設立）：戦後の中小企業金融特化地銀、東京都出資レガシー
- **八千代銀行**（1924 無尽 → 1951 相互 → 1990 普銀）：相互銀行 → 普通銀行転換組、第二地銀
- **新銀行東京**（2003 設立）：石原都政の象徴的政策銀行 → 不採算で経営問題化 → 都政の負の遺産処理として 2016-04 統合
- 統合方式：2014-10 持株会社方式（TY FG）で都民 + 八千代を経営統合 → 2016-04 新銀行東京を吸収合併 → 2018-05 3 行を「きらぼし銀行」として完全合併、FG 名も「きらぼし」に統一 ^[extracted]

### 名称由来

- **「TY」（旧名）**: 東京（**T**okyo）+ 八千代（**Y**achiyo）の頭文字、合併時の暫定名

## 2. 事業セグメント・マップ

| セグメント | 主力プロダクト | 特徴 |
|---|---|---|
| 中小企業金融 | 事業性融資、シンジケート、私募債 | **コア事業** — 旧 都民銀行以来の伝統領域 |
| 富裕層 | プライベートバンキング、相続、不動産 | 都心立地多数 — 港区・千代田・中央 |
| 個人ローン | 住宅ローン、カードローン | 都内通勤層中心 |
| **デジタル（UI 銀行）** | **スマホ専業バンキング** | **2022-01 開始** — Z 世代・若年層獲得 ★ |
| スタートアップ金融 | ベンチャー融資、CVC（きらぼしキャピタル）| 東京の起業エコシステム取り込み |
| 不動産 | 都内不動産融資、賃貸不動産 | 旧 八千代銀行系強み |
| 証券 | きらぼし証券・ライフデザイン証券 | リテール証券 |

### 「東京特化地銀」モデル

- **地盤を東京都内に絞り込む** — 全国の地銀の中で「東京都を主戦場」とする希少なポジション ^[extracted]
- 「メガに対する小回り + 地銀の地縁ネットワーク」を融合したニッチ

### デジタル戦略

- **UI 銀行**（2022-01 サービス開始）★：きらぼし銀行運営の個人向けスマホ専業バンキング、Z 世代・若年層獲得チャネル ^[extracted]
- 「きらぼしデジタル銀行」ブランド：法人向けデジタル支店

### 旧 新銀行東京レガシー処理

- **新銀行東京 統合（2016-04）は単純な事業統合ではなく、東京都の政策的負担の引き取り** ^[extracted]

### 提携・買収戦略

- **スタートアップ連携**：きらぼしキャピタル経由 CVC + 東京の VC ネットワーク

## 4. 規制・政策

- **主管**：金融庁（FSA）+ 関東財務局
- **持株会社規制**：銀行法 第 52 条の 17
- **直近政策論点**：
  - 2024〜 日銀政策金利正常化 → 国内金利マージン改善（地銀全般のボーナス）
  - 2025〜 中小企業の事業承継 / M&A 支援 — 政府の地銀役割期待
