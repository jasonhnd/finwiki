# JF 全漁連 (全国漁業協同組合連合会 / National Federation of Fisheries Cooperative Associations)

## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]] as the **JF Group national representative federation**, parallel to [[JapanFG/ja-zenchu|JA 全中]] in the agricultural cooperative system. Read with [[JapanFG/norinchukin|農林中央金庫]] for the central financial institution, [[banking/jf-marine-bank-system-japan|JF Marine Bank system]] for the credit system boundary, and [[banking/cooperative-banking-japan|cooperative banking domain]] for the broader cooperative finance scope.

## TL;DR

**JF (漁業協同組合) グループの全国代表組織**。1952 設立、水産業協同組合法 (水協法) を根拠とする **全国連合会**。JA グループの 4 大全国連 ([[JapanFG/ja-zenchu|全中]] + [[JapanFG/ja-zen-noh|全農]] + [[JapanFG/ja-kyosairen|共済連]] + [[JapanFG/norinchukin|農林中金]]) と異なり、漁協グループでは **JF 全漁連が代表 + 経済事業 (販売・購買・系統水産加工等) を兼務**、別途 **JF 共水連 (全国共済水産業協同組合連合会)** が共済事業、信用事業は **都道府県の [[JapanFG/jf-shingyoren|信漁連]] + 中央 [[JapanFG/norinchukin|農林中央金庫]]** が担う構造。**会員 = 全国の漁協 (JF) 71 組合 + 都道府県漁連 + 関連法人** ([[banking/jf-marine-bank-system-japan|JF Marine Bank system]] と同じ母集団)。事業は **(1) 水産物販売 (魚市場・流通)**、**(2) 購買 (漁具・燃料)**、**(3) 漁業政策運動・代表機能**、**(4) 系統水産加工・輸出促進**。**規模は JA 全農の数 % 水準** (漁業生産額が農業比小)。^[extracted]

### JF グループ全国組織の役割分担 ^[extracted]

| 全国連 | 領域 | 性格 |
|---|---|---|
| **JF 全漁連** | **代表・経済事業 (販売・購買)・政策運動** | 連合会 (代表 + 経済事業) ★本ページ |
| **JF 共水連** | 共済事業 | 連合会 (共済事業) |

### 重要年表 ^[extracted]

| 年 | 事象 |
|---|---|
| 1948 | 水産業協同組合法制定、戦後漁協再編開始 |
| **1952** | **全国漁業協同組合連合会 (JF 全漁連) 設立** |
| 1960〜70 年代 | 漁協数ピーク (約 2000 組合) — 沿岸漁業の村落漁協が主体 |
| 1990〜2000 年代 | 漁業生産額減少 + 漁業者高齢化 → 漁協数大幅減少 |
| 2000 年代 | **「JF (Japan Fishery cooperatives)」ブランド統一化** |
| 2015〜 | **漁協改革** — 規制改革会議で「漁協の競争力強化」議論 |
| **2018〜** | **漁業法改正** (70 年ぶり) — 漁業権配分の見直し、漁協役割再定義 ^[extracted] |
| 2020〜 | **JF Marine Bank system** 一体運営強化 ([[banking/jf-marine-bank-system-japan|JF Marine Bank]] の boundary) |
| 2024 | **2025-04-01 時点 71 JF + 10 信漁連 + 1 農林中央金庫** ([[JapanFG/norinchukin|農林中金]] 公開資料による) ^[extracted] |

| **政策運動** | 水産政策・漁業権・資源管理に関するロビイング |
| **系統水産加工** | 加工流通の系統取引 |
| **教育・情報提供** | JF 役員研修・水産情報誌等 |

### JF Marine Bank system との関係 ^[extracted]

- **JF 全漁連 = 代表 + 経済事業**
- **[[banking/jf-marine-bank-system-japan|JF Marine Bank system]] = 信用事業の枠組み** — [[JapanFG/jf-shingyoren|JF 信漁連]] (都道府県) + [[JapanFG/norinchukin|農林中央金庫]] (中央) で構成
- JF 全漁連は **JF Marine Bank の直接運営者ではない** が、グループ全体代表として関与

### 漁業改革と漁協存在意義

- **2018 漁業法改正**: 70 年ぶりの抜本改正、**漁業権配分の自由化** → 漁協以外の事業者参入余地拡大 ^[extracted]
- JF 全漁連は **「漁協が地域漁業のインフラ」**として存在意義を主張 — 食料安全保障 + 地方創生フレームで対抗
- ただし **若年漁業者の漁協離れ** + **大規模水産会社 (マルハニチロ・日本水産 等) の事業拡大** で漁協基盤縮小圧力

### 水産物輸出促進

- **「水産物輸出 5 兆円目標」**の一翼として、JF 系統経由の輸出強化
- **中国向け禁輸 (2023〜) 問題**で代替市場開拓が課題

### 漁業資源管理

- **TAC (漁獲可能量) 制度** の運用協力
- **資源回復計画**・漁業 IT 化への対応

### JA 全農との比較 ^[inferred]

| 軸 | JA 全農 | JF 全漁連 |
|---|---|---|
| 事業規模 (取扱高) | 4〜5 兆円規模 | 数千億円規模 (1/10〜数 %) |
| 会員組合数 | 497 JA | **71 JF** |
| 中央機関構造 | 全中 + 全農 (分業) | 全漁連 (兼務) |
| 政治影響力 | 強 (自民党農林族) | 中 (水産族議員) |
| 海外子会社 | CGB 等大規模 | 小規模 |

### 強み・弱み

**強み**
- **全国漁協を集約**する唯一の組織 — 漁業政策での代表性
- 水産庁との **密接関係**
- 食料安全保障フレームでの政治的重み

**弱み**
- **漁業生産額自体が縮小** (1980 年代ピーク 1200 万トン → 2020 年代 400 万トン台)
- 若手漁業者の **JF 離脱**圧力 (大規模水産会社直接取引志向)
- 大規模水産会社 (上場企業) との競合で **規模劣勢**

## 4. 監督・規制

- **主管**: 水産庁 (農林水産省内) + 内閣府規制改革推進室 (改革論議)
- **法的根拠**: 水産業協同組合法 (水協法) + 漁業法 (2018 改正)
- **直近政策論点**:
  - **漁業法改正の運用** (漁業権配分・新規参入)
  - **水産物輸出促進** (5 兆円目標)
  - **資源管理 (TAC) と漁協の役割**
  - **食料安全保障計画 (2026〜)** での JF 全漁連の機能再定義
  - **中国向け輸出停止** (2023〜) への対応
