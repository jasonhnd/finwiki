# 丸三証券 (Marusan Securities)

## TL;DR

丸三証券は **1910 年創業の老舗中堅独立証券** (東証 PRIME **8613**)、創業 110 年超で **対面リテール主軸 + コンサルティング営業** に特化する独立系証券会社。FSA 金融商品取引業者一覧では「丸三証券株式会社」、関東財務局長(金商)第167号として掲載される。**JSDA 正会員**。メガバンク・通信系には属さず、株式・債券・投資信託の対面販売を中核とする伝統的中堅証券モデルを維持し、競合は [[JapanFG/aizawa-sg|アイザワ証券]] / [[JapanFG/okasan-sg|岡三証券グループ]] / [[JapanFG/iwai-cosmo-hd|岩井コスモ HD]] / [[JapanFG/tokai-tokyo-fh|東海東京 FH]] 等の中堅独立証券枠。

## 2. 公司結構

**正式名**：丸三証券株式会社

**業態**：金融商品取引業 (第一種金融商品取引業者) — 中堅独立系証券

### 業態の特徴

丸三証券は持株会社化を経ていない**単一事業会社型**の老舗中堅独立証券。グループ内に銀行・保険・暗号資産事業を持たず、証券業 (特に対面リテール株式・債券・投資信託) に純粋特化するシンプルな事業構造が特徴。これは [[JapanFG/aizawa-sg|アイザワ証券グループ]] (持株会社化済) や [[JapanFG/okasan-sg|岡三証券グループ]] (持株会社化済) と異なる組織形態。

## 4. 战略与競争

### 老舗中堅独立証券としてのポジション

丸三証券は 1910 年創業の老舗で、110 年超の歴史を持つ中堅独立証券。メガバンク系証券 ([[JapanFG/smbc-nikko|SMBC 日興]] / [[JapanFG/mufg-mums|MUMSS]] / [[JapanFG/mizuho-securities|みずほ証券]]) や大手対面 ([[JapanFG/nomura-hd|野村]] / [[JapanFG/daiwa-sg|大和]])、ネット専業 ([[JapanFG/sbi-securities|SBI 証券]] / [[JapanFG/rakuten-securities|楽天証券]] / [[JapanFG/monex-group|マネックス]]) とは異なる立ち位置で、**対面リテール × 中堅独立** のニッチを維持する。中京・関西の [[JapanFG/okasan-sg|岡三]] / [[JapanFG/iwai-cosmo-hd|岩井コスモ]] / [[JapanFG/tokai-tokyo-fh|東海東京]] とは地域基盤 (東京 vs 中京・関西) が異なる。

### 競争環境 ([[securities/japan-online-brokerage-competition|japan-online-brokerage-competition]] 文脈)

- ネット証券の手数料無料化トレンドに対して、対面コンサルティング営業で差別化
- 富裕層・シニア層向けの「教育型営業」「対面相談」のニッチを維持
- 大手対面証券との競合は規模差で不利、しかし中堅独立証券枠での独自ブランド維持

## 5. 监管と政策

- **主管**：金融庁 (FSA)・関東財務局
- **業態**：第一種金融商品取引業 (証券業)
- **規制**：金融商品取引法・日本証券業協会 ([[JapanFG/jsda|JSDA]]) 自主規制
- **直近政策論点**：
  - 2024〜 新 NISA 拡充：対面証券として富裕層向け NISA 商品ラインナップ強化 ^[inferred]
  - 2025〜 顧客本位の業務運営 (FD 原則) 強化：対面コンサル型営業の正当化要素
  - 中堅独立証券業界の再編論議 (規模の劣後問題) ^[ambiguous]

## Counterpoints

- 「対面リテール特化の持続性」：ネット証券の手数料無料化と若年層流入が進む中、対面コンサル型の収益モデルが中長期的に維持できるかは継続論点 ^[ambiguous]
- 「規模の劣後」：大手 5 大証券 + 上位ネット 4-5 行と比較して資本・人員・システム投資規模で不利 ^[inferred]
- 「中堅独立証券業界の再編リスク」：同枠の競合 ([[JapanFG/aizawa-sg|アイザワ]] / [[JapanFG/okasan-sg|岡三]] / [[JapanFG/iwai-cosmo-hd|岩井コスモ]]) との統合・連携シナリオは公開情報からは確定できない ^[ambiguous]
