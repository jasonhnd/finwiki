# ノーザン・トラスト・ジャパン (Northern Trust Japan)

> [!note] 命名 note
> このページは当初 "orient-trust-japan" として作成依頼された slot だが、日本に「Orient Trust」という独立法人は確認できないため（「オリエントコーポレーション」[[JapanFG/orico]] は信販会社で別カテゴリー、「東洋信託銀行」は旧 UFJ 系で現 [[JapanFG/mitsubishi-ufj-trust-bank|三菱 UFJ 信託銀行]] に統合済み）、外資系トラスト子会社の代替として **ノーザン・トラスト・ジャパン** を充当。

## TL;DR

米 **Northern Trust Corporation**（NASDAQ: **NTRS**、シカゴ本社）の日本拠点群。親会社は **米国 3 大カストディ・アセット・サービシング銀行の 1 つ**（首位 [[JapanFG/bny-mellon-japan|BNY Mellon]] / 2 位 [[JapanFG/state-street-japan|State Street]] に次ぐ、世界 AUC/A 約 16 兆ドル規模、2024）で、機関投資家サービス（カストディ・アセット・サービシング・ファンドアドミ）と富裕層プライベートバンキングの両輪が特徴。日本では **ノーザン・トラスト・グローバル・サービシング・ジャパン**（信託業ライセンスを軸に GPIF・公的年金・運用会社向けバックオフィスサービスを提供）+ **東京支店** + **ノーザン・トラスト・セキュリティーズ・ジャパン**（FIEA 業者）の組み合わせで運営。**邦銀リテール非関与・純粋ホールセール B2B 機関投資家特化** の外資カストディ第 3 軸。^[inferred]

## 6. 戦略

### 6.1 米 3 大カストディの末尾としての差別化

- 規模では [[JapanFG/bny-mellon-japan|BNY Mellon]] / [[JapanFG/state-street-japan|State Street]] に劣後
- 差別化軸: **富裕層プライベートバンク機能との統合**（米本国主軸）+ **選別された大口機関投資家への深いサービス**
- 日本では富裕層プライベートバンク本格展開はせず、機関投資家サービス（GPIF・大手生損保・大手運用会社）に絞り込み ^[likely]

### 6.2 GPIF・公的年金受託機関としての位置

- GPIF の海外資産カストディは複数の外資カストディ（[[JapanFG/state-street-japan|State Street]] / [[JapanFG/bny-mellon-japan|BNY Mellon]] / Northern Trust 等）に分散委託
- 公的年金・企業年金の海外運用比率上昇 → 外資カストディ全体への追い風

### 6.3 邦銀信託との棲み分け

- [[JapanFG/master-trust-bank|マスタートラスト信託]]・[[JapanFG/mitsubishi-ufj-trust-bank|三菱 UFJ 信託銀行]]・[[JapanFG/sumitomo-mitsui-trust|住信トラスト]] 等 **邦銀系信託** は国内資産のカストディ・運用受託で圧倒的シェア
- Northern Trust 等の外資カストディは **海外資産の専門受託** で棲み分け
- グローバル分業: 邦銀信託（国内）× 外資カストディ（海外）の組み合わせが日本機関投資家の標準アーキテクチャ ^[inferred]

### 6.4 富裕層プライベートバンク（米本国）

- Northern Trust の **創業以来のコア事業** はシカゴ富裕層向けプライベートバンク（1889 開業以来）
- 米国超富裕層向けのウェルスマネジメント・信託・遺産管理で深い顧客関係
- 日本では UBS [[JapanFG/ubs-japan]] / [[JapanFG/lgt-wealth-management-trust|LGT]] / [[JapanFG/smbc-trust-bank|SMBC 信託（PRESTIA）]] 等のプライベートバンク勢と直接競合せず、機関投資家チャネルに集中

## Counterpoints

- 「米 3 大カストディ第 3 軸」: 規模で BNY Mellon / State Street に劣後、シェア拡大より選別顧客との深い関係維持が現実路線 ^[inferred]
- 「富裕層プライベートバンク」: 米本国主力事業の日本展開度は限定的、UBS / LGT 等との競合を避けた戦略 ^[likely]
- 「邦銀信託との棲み分け」: 海外資産カストディの主力役は外資 3 強だが、邦銀信託が海外運用受託機能を内製化する流れもあり、長期的な役割再編リスク ^[inferred]
- 「日本拠点規模」: 邦銀信託の従業員数千名 vs 外資カストディの数百名規模で、構造的に B2B 特化型小規模ホールセール ^[likely]
- 「公開情報の限定性」: 米本国上場会社 + 日本では拠点ベース、Japan-specific 開示は親会社 10-K の片隅程度 ^[likely]
