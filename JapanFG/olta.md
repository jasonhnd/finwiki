# OLTA (オルタ)

## Wiki route

This entry sits under the [[JapanFG/INDEX|JapanFG index]] as a **B2B factoring fintech** that sits next to BNPL in the wider working-capital stack. Read it together with the [[JapanFG/bnpl-landscape|Japan BNPL landscape]] system note, the [[JapanFG/net-protections-hd|Net Protections HD]] BtoB BNPL peer (NP 掛け払い), and the [[JapanFG/money-forward|Money Forward]] / Biz Forward sibling for the SaaS-led factoring comparator. The [[payments/japan-bnpl-credit-purchase-boundary|BNPL and credit-purchase boundary]] frames the question of whether factoring counts as credit, the [[payments/credit-purchase-card-operators-japan-index|METI credit-purchase registry]] is a related baseline, and the [[payments/japan-payment-scheme-economics-matrix|Japan payment scheme economics matrix]] situates the working-capital economics next to card / code rails.

## TL;DR

OLTA は **オンライン完結型クラウドファクタリング** をリードする独立系 fintech。中小企業の **未回収売掛債権** を即時に買い取って資金化することで、銀行融資や信販ファイナンスと並ぶ第三の運転資金チャネルを提供する。BtoB の文脈での "BNPL 的キャッシュフロー前倒し" カテゴリに該当し、[[JapanFG/net-protections-hd|Net Protections HD]] の NP 掛け払い、[[JapanFG/money-forward|Money Forward]] の Biz Forward、Yup、各銀行系ファクタリングと競合する。

## 2. Business role

- 中小企業が保有する売掛債権をオンライン申込・即日審査ベースで買い取り、資金化を提供する。銀行融資より審査が短期で、信用情報基準も異なる。^[extracted]
- 銀行・地銀の与信ラインで拾われにくい中小事業者の **キャッシュフロー前倒し** ニーズに応える。
- BtoB BNPL（[[JapanFG/net-protections-hd|Net Protections HD]] の NP 掛け払い等）と異なり、債権買取ベースなので **加盟店 → 取引先 → 弁済** の流れではなく **売り手 → ファクター** の二者間 / 三者間契約で成立する。^[inferred]

## 3. Why this standalone page matters

[[JapanFG/bnpl-landscape|Japan BNPL landscape]] は B2C 後払いを主軸に整理されているが、BtoB の運転資金ファクタリングは **同じ "支払いを遅らせる / 入金を前倒す" 経済学** を共有しており、JapanFG 上で独立して扱う価値がある。銀行融資 / 信販 / BNPL / ファクタリングの **代替・補完関係** を読むためのアンカー。

## 4. 监管与政策

- **ファクタリングの法的位置付け**: 債権譲渡として民法に基づくのが原則。「給与ファクタリング」や「実質貸付」と判定された事例で金融庁・最高裁が貸金業法該当を認定した経緯あり。^[inferred]
- **金融庁の注意喚起**: 悪質ファクタリング業者への注意喚起が継続的に発出されており、合法ファクターは透明な手数料・契約設計が必要。^[inferred]
- **個人情報保護法 / 反社チェック**: 中小事業者の財務データ取得と取引先与信判断における個人情報・与信情報の取扱い。
- **2025〜 中小企業 DX 政策**: 経産省・中小企業庁の電子インボイス（Peppol）/ 適格請求書制度との連携で、ファクタリング基盤の電子化が進展。^[inferred]

## Counterpoints

- 「ファクタリング業界の社会的評価」: 給与ファクタリング・悪質業者問題で業界全体のイメージが毀損された経緯あり、純正ファクターも信用回復が継続課題。^[ambiguous]
- 「銀行融資 / 信販融資との競合」: [[JapanFG/jaccs|Jaccs]] / [[JapanFG/orico|Orico]] / [[JapanFG/credit-saison|Credit Saison]] / 各銀行系の中小企業向け運転資金商品との直接競合。^[inferred]
