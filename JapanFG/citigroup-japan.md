# シティグループ・ジャパン (Citigroup Japan)


## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[JapanFG/hsbc-japan|HSBC ジャパン (HSBC Japan)]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

米 **Citigroup Inc.**（NYSE: C）の日本拠点群。**世界最大級の G-SIB（FSB Bucket 3 相当）**の一部として、東京を**アジア圏内のホールセール金融ハブ**の一つに位置付ける。日本では **1902 年に旧 ナショナル・シティ銀行 が東京支店を開設**（邦銀史上最初期の外資系プレゼンス）して以来 120 年超の歴史を持つが、**2014-09 リテール事業を SMBC 信託銀行へ売却**（2015-11 完了、現「PRESTIA」ブランド）し、現在は **法人銀行 + 投資銀行 + GTS（グローバル・トランザクション・サービシズ）+ 機関投資家サービス** に集中。日系メガバンクのグローバル化加速期に**日興コーディアル証券を SMFG へ売却（2009-10）**、続いてリテール撤退と、**「日本リテールを 2 度たたんだ外資 IB」**としての位置付けが特徴。^[inferred]

## 1. 会社概要

**最終親会社**：Citigroup Inc.（米デラウェア州法人、NYSE: C、本店ニューヨーク市）
**設立**（親）：1998（Citicorp + Travelers Group 合併、後に Travelers 保険は分離）
**前身**：1812 City Bank of New York → National City Bank of New York（1865 国法銀行転換）→ Citicorp（1967 持株会社化）→ Citigroup（1998 合併）
**規制資本ステータス**：FSB G-SIB **Bucket 3**（世界最大級カテゴリの一つ、追加 CET1 +2.0% 要件）^[extracted]

### 日本の主要法人（公開情報ベース）

```
Citigroup Inc.（米 NYSE: C・連邦準備制度監督）
  ├── シティバンク, N.A. 東京支店 ── 米国法人の在日支店、法人融資・トランザクション・銀行
  ├── シティグループ証券株式会社 ── 投資銀行・機関投資家サービス（日本登録金商業者）
  ├── シティグループ・グローバル・マーケッツ・ジャパン株式会社 ── 株式・債券・デリバティブ業務
  └── （かつて）シティバンク銀行株式会社 ── 個人リテール → 2015-11 SMBC 信託に売却 → 現 PRESTIA
```

**特徴**：
- 日本での主体は **「在日支店（シティバンク, N.A. 東京支店）+ 国内登録法人（シティグループ証券）」のハイブリッド型**。日系メガバンクが「銀行 + 信託 + 証券」を国内子会社で揃えるのに対し、Citi は**米本店直結の支店を中核に置く**伝統的な外資 IB スタイル ^[inferred]
- リテール撤退後、雇用規模・支店網は大きく縮小したが、**ホールセール領域では引き続き有力プレイヤー**

## 2. 沿革（年表抜粋）

| 年 | 事象 |
|---|---|
| **1902** | **ナショナル・シティ銀行 東京支店 開設**（邦銀史上最初期の外資系プレゼンスの一つ）^[extracted] |
| 1960s〜70s | 日本の高度成長期に法人・トランザクション銀行を拡大 |
| 1980s | **シティバンク銀行 リテール展開**（外資系個人銀行として独自ブランディング、24h ATM・外貨預金で都市部富裕層を取り込み）|
| 1998 | 親 Citicorp + Travelers Group 合併 → **Citigroup 発足** |
| 2007〜2008 | サブプライム危機 → Citi 大規模損失計上 |
| **2008-11/2009** | リーマン・ショック余波で Citi **米政府公的資金注入**（TARP 経由 250 億ドル + 追加支援、ストレステスト対象）/ 実質国有化局面 |
| **2009-10** | **日興コーディアル証券を SMFG に売却**（→ 後の [[smbc-nikko]]）。Citi 日本ホールセール再建の一環 |
| 2010s 前半 | グローバル「Citi Holdings」非中核資産整理スキーム下で日本リテールも見直し対象に |
| **2014-09** | **シティバンク銀行 リテール → SMBC 信託銀行へ売却 発表**（個人顧客・支店・PB 事業）|
| **2015-11** | **リテール売却完了** → SMBC 信託銀行傘下で **「PRESTIA」ブランド** として継続。シティバンク銀行（法人格）は消滅 |
| 2014〜 | 日本拠点を **法人銀行 + 投資銀行 + GTS + 機関投資家** に再集中。リテール完全撤退 |
| 2020s | グローバル戦略「Citi Strategy Refresh」下、Wealth と機関投資家サービスを軸にリストラクチャリング継続 ^[inferred] |

> 注: 売却金額・人員規模など個別細目は当時の公表ベース。最新スキームは Citi 公式 IR / 金融庁開示を参照のこと。^[inferred]

## 3. 事業セグメント・マップ

| セグメント | 在日担い手 | 特徴 |
|---|---|---|
| 法人銀行（コーポレートバンキング）| シティバンク, N.A. 東京支店 | 多国籍企業の日本進出支援、日系大企業のグローバル取引、シンジケートローン |
| **GTS（Global Transaction Services）** | シティバンク, N.A. 東京支店 | 多国籍企業向けキャッシュマネジメント・トレードファイナンス・カストディ — Citi グローバル収益柱の一つ |
| 投資銀行 IB（M&A / DCM / ECM）| シティグループ証券 | 日系企業のクロスボーダー M&A アドバイザリー、外債発行、グローバル IPO |
| 機関投資家サービス | シティグループ・グローバル・マーケッツ・ジャパン | 株式・債券・FX・デリバティブ売買、機関投資家リサーチ |
| カストディ・証券管理 | Citi グローバル GTS の一部 | 海外機関投資家の日本株保管、本邦年金の海外資産管理 |
| 個人リテール | **なし**（2015-11 撤退、SMBC 信託「PRESTIA」へ承継）| — |

## 4. 戦略・競争

### 日本でのポジショニング

- **リテール撤退 → 法人特化の純粋外資 IB / トランザクションバンク**：日系メガバンク（[[mufg]] / [[smfg]] / [[mizuho-fg]]）と異なり、個人・中小企業への裾野を持たず、**多国籍企業・大企業・機関投資家のみがターゲット** ^[inferred]
- **GTS で他外資 IB と差別化**：純粋 IB（ゴールドマン / モルガン・スタンレー）が証券・アドバイザリー中心なのに対し、Citi は**「銀行口座 + 決済 + 貿易金融」の世界規模ネットワーク**を強み（世界 90 か国超に拠点を持つ商業銀行ライセンスは外資の中では希少）^[inferred]
- **米 G-SIB Bucket 3 のグローバル一部**：地政学イベント・米利上げ・ドル流動性のグローバル波及を通じて日本拠点の業務にも影響

### 主要競合

| 領域 | 主要競合 |
|---|---|
| 外資系 IB / M&A | [[goldman-sachs-japan]] / [[morgan-stanley-japan]] / JP モルガン日本 / バークレイズ |
| GTS / トランザクションバンク | JP モルガン / HSBC / Deutsche Bank / BNP Paribas |
| カストディ | State Street / BNY Mellon / JP モルガン / HSBC |
| 日系メガバンクのクロスボーダー | [[mufg]] / [[smfg]] / [[mizuho-fg]]（Morgan Stanley JV 含む）|

### グローバル戦略との接続

- **Citi グローバル「5 つのコアビジネス」**（Services / Markets / Banking / Wealth / U.S. Personal Banking）のうち、日本は **Services + Markets + Banking の 3 軸**に集中（Wealth は撤退、U.S. Personal Banking は対象外）^[inferred]
- 日本リテール売却（2015）と同時期、グローバルでも消費者銀行 14 市場撤退（アジア・東欧）を進行 — 日本撤退はその先駆例

## 5. 監督・規制

- **主管**：金融庁（FSA）— 銀行支店・金商業者として
- **米側監督**：FRB（連邦準備制度理事会）/ OCC（通貨監督庁）/ FDIC、グローバルストレステスト DFAST 対象
- **G-SIB Bucket 3**：FSB 認定（+2.0% CET1 追加資本要件）^[inferred]
- **直近の論点**：
  - 米 Citi のグローバル組織再編（Services / Markets / Banking 分離管理）→ 日本拠点ガバナンス連動
  - **米中・地政学リスク**：日本拠点の対中・対韓フローは米コンプライアンスの影響を強く受ける
  - 外為法・対内直接投資審査（2020 改正）→ 機微業種における外資 IB の助言業務に規制

## Counterpoints

- **「リテール撤退は失敗」言説 vs 「資本効率化として合理的」言説の両論** ^[ambiguous]：シティバンク銀行は外資系個人銀行ブランドとして 1990s〜2000s 前半に**都市富裕層から強い支持**を集めたが、収益性とリーマン後の資本制約から維持困難。撤退は「グローバル戦略上の選択集中」とも評価可能
- **PRESTIA ブランド継続の解釈** ^[inferred]：旧シティバンク銀行の店舗・口座は SMBC 信託で PRESTIA として継続しており、「Citi 日本リテール」のレガシーは**ブランド名としては消えたが、サービス・顧客基盤としては SMBC 信託に承継**された側面がある
- **日興コーディアル売却の戦略的位置付け** ^[ambiguous]：2009-10 の SMFG への売却は当時「金融危機下の緊急処分」とされた一方、結果的に SMFG にとっては国内 5 大証券の一角（[[smbc-nikko]]）を獲得する歴史的買収となった
- **「外資 IB は撤退の歴史」観**：日本における Citi の歩み（リテール売却 → 証券売却）は、外資系金融の日本市場縮退の象徴例として語られることが多いが、**ホールセール領域では今も主要外資 IB の一角**である事実は見落とされやすい ^[inferred]
