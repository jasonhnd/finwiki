# 地方公共団体金融機構 (JFM)


## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[JapanFG/jfc|日本政策金融公庫 (JFC)]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

地方自治体 (都道府県・市区町村) 向け長期融資に特化した**地方共同法人**。設立 2008-08-01 (地方公営企業等金融機構として) → 2009-06-01 現名称に改組。旧 **公営企業金融公庫** (1957-06 設立、2008-09-30 廃止) の業務を承継。出資者は全 47 都道府県 + 全市区町村による地方共同出資 (国出資なし)。融資残高は概算 **13-15 兆円規模** で、地方財政の長期資金供にインフラ。**民間メガバンク・地銀との競合 + 補完関係**を持ち、災害復興融資・上下水道などの地方公営企業向け融資で独自の役割。^[ambiguous]

## 1. 法人構造

**正式名**: 地方公共団体金融機構 ^[extracted]
**英名**: Japan Finance Organization for Municipalities (JFM) ^[extracted]
**法人形態**: 地方共同法人 (特殊法人。地方公共団体金融機構法に基づく) ^[extracted]
**設立**: 2008-08-01 (地方公営企業等金融機構として) ^[extracted]
**現名称改組**: 2009-06-01 (地方公共団体金融機構に名称変更、業務範囲拡大) ^[extracted]
**本店**: 東京都千代田区麹町 ^[extracted]
**出資者**: 全 47 都道府県 + 全市区町村 (地方共同出資、国出資なし) ^[extracted]
**主務官庁**: 総務省 (地方自治体所管のため) + 財務省 ^[inferred]

### 機関設計 ^[extracted]

- **理事長**: 地方公共団体側から選任 (歴代に地方自治体経験者多数)
- **代表者会議**: 全国知事会・全国市長会・全国町村会の代表が監督
- **経営審議委員会**: 外部有識者による経営チェック

### 業務範囲 ^[extracted]

| 業務 | 内容 |
|---|---|
| 地方債引受 | 都道府県・市区町村の地方債を長期低利で引受・運用 |
| 地方公営企業向け融資 | 上下水道・電気・交通・病院などの地方公営企業向け長期融資 |
| 災害復興融資 | 大震災・台風・地震などの災害復興のための地方自治体融資 |
| 資金調達 | 政府保証なしで国内外の資本市場から債券 (地方金融機構債) 発行 |
| 経営支援 | 地方自治体・地方公営企業の財政分析・経営アドバイス |

### 融資残高 (概算) ^[ambiguous]

- 2024 時点 約 **13-15 兆円規模** (正確な数値は公式 IR 要確認)
- 地方債引受先として 三大資金供に源の一つ (民間銀行・財政融資資金と並ぶ)

## 2. 沿革

### 公営企業金融公庫時代 (1957-2008)

| 年月 | 事象 |
|---|---|
| 1957-06 | **公営企業金融公庫 設立** (中央政府による地方自治体向け融資、国 100% 出資の特殊法人) ^[extracted]
| 1957-2008 | 上下水道・交通・電気などの地方公営企業に低利長期融資を提供 |
| 1980s-90s | 地方財政膨張期に融資残高拡大、地方インフラ整備の主要資金供に源 |
| 2001-12 | **小泉政権の特殊法人改革** → 公営企業金融公庫の廃止方針決定 ^[extracted]
| 2007 | 廃止 + 後継組織の枠組み議論 (国出資維持か地方共同出資か) ^[inferred]
| 2008-08-01 | **地方公営企業等金融機構 設立** (前身組織、地方共同出資へ移行) ^[extracted]
| 2008-09-30 | **公営企業金融公庫 廃止** → 業務・資産を新機構が承継 ^[extracted]

### 地方公共団体金融機構時代 (2009-現在)

| 年月 | 事象 |
|---|---|
| **2009-06-01** | **地方公共団体金融機構に改組** (業務範囲を地方公営企業から一般地方自治体に拡大) ^[extracted]
| 2011-03-11 | 東日本大震災 → 災害復興融資の主要担い手の一つ |
| 2011-2014 | 復興融資ピーク (被災自治体・公営企業向け) ^[inferred]
| 2016-04-14 | 熊本地震 → 災害復興融資対応 |
| 2018-07 | 西日本豪雨 → 災害復興融資対応 |
| 2019-10 | 令和元年東日本台風 → 災害復興融資対応 |
| 2020-2022 | コロナ禍下の地方財政支援 (税収減・医療費増対応) ^[inferred]
| **2024-01-01** | **能登半島地震** → 災害復興融資対応 ^[extracted]
| 2024-2025 | 地方インフラ老朽化 + 人口減少時代の地方財政支援役割再定義 ^[inferred]

## 3. 事業セグメント map

| セグメント | 主要対象 | 特徴 |
|---|---|---|
| 一般地方債引受 | 都道府県・市区町村 | 民間銀行・財政融資資金と競合 + 補完 |
| 上下水道融資 | 地方公営企業 | 長期低利 (民間より低金利) |
| 交通融資 | 地方公営交通 (バス・地下鉄) | 採算性低い公共交通維持 |
| 病院融資 | 公立病院・地方独立行政法人 | 地域医療維持の長期資金 |
| 電気・ガス融資 | 地方公営電気・ガス事業 | 一部地方の独立公営事業 |
| 災害復興融資 | 被災自治体・公営企業 | 大震災・台風・地震対応特別融資 |
| 資金調達 | 地方金融機構債 (JFM 債) | 政府保証なし、AAA 格付け相当 ^[ambiguous]

### 競合・補完関係

```
地方自治体・地方公営企業の資金調達
  ├── 地方公共団体金融機構 (JFM) ── 長期低利、政策金融
  ├── 財政融資資金 (財務省) ── 国が直接融資、長期低利
  ├── 民間金融機関
  │     ├── [[mufg]] ── 大都市自治体・大規模地方公営企業
  │     ├── [[smfg]] ── 大都市自治体・関西圏自治体
  │     ├── [[mizuho-fg]] ── 大都市自治体・伝統的地方自治体融資
  │     └── 地銀・第二地銀 ── 地元自治体メインバンク
  └── 資本市場 (地方債発行) ── 大都市は市場調達増加傾向
```

## 4. 戦略・競争

### 公営企業金融公庫廃止 → 地方共同方式への移行背景

- **小泉政権特殊法人改革** (2001-) → 国出資型政策金融機関の整理対象
- 「国主導から地方主導へ」のスローガン → 地方共同出資モデルへ移行 ^[inferred]
- 業務継続のための地方自治体側からの要請 (廃止だけでは地方インフラ資金供にに支障) ^[inferred]
- **2008-09-30 公営企業金融公庫廃止 → 2008-08-01 新機構設立** で実質的に業務継続

### 民間金融機関との関係

- **補完**: 採算性低い小規模自治体・公営企業向け長期融資 (民間が積極的でない領域)
- **競合**: 大都市自治体・大規模公営企業の融資ではメガバンク・地銀と競合 ^[inferred]
- **金利優位**: 地方共同出資 + AAA 格付け相当の調達コストで民間より低金利提供可能 ^[ambiguous]
- **メガバンク側の戦略**: [[mufg]] [[smfg]] [[mizuho-fg]] は大都市自治体向けにシンジケートローン・市場性地方債引受で対応

### 災害復興融資の特殊役割

- 大震災・台風・地震時に地方自治体・公営企業の資金需要が急増
- 民間金融機関だけでは対応困難な長期低利融資を JFM が提供
- 東日本大震災 (2011)・熊本地震 (2016)・能登半島地震 (2024) などで重要な役割

### 直近政策論点

- **人口減少時代の地方財政**: 税収減・社会保障費増 → 地方財政逼迫 → JFM の役割重要化 ^[inferred]
- **インフラ老朽化**: 上下水道・道路・橋梁の更新需要 → 長期資金供に需要 ^[inferred]
- **2024 能登半島地震復興**: 地震復興融資の本格対応 ^[extracted]
- **デジタル地方財政**: 自治体 DX・スマートシティ投資への長期資金供に ^[inferred]

## 5. 監督・規制

- **主務官庁**: 総務省 (地方自治体所管) + 財務省 ^[inferred]
- **根拠法**: **地方公共団体金融機構法** (2007 年制定、2008 施行) ^[extracted]
- **監督**: 地方公共団体金融機構代表者会議 (全国知事会・市長会・町村会代表) + 経営審議委員会
- **会計検査**: 会計検査院による検査対象 (地方共同法人として)
- **格付け**: AAA 格付け相当 (政府保証なしながら地方共同出資による信用力) ^[ambiguous]

### 公的金融機関との関係

- **日本政策金融公庫 (JFC)**: 中小企業・農林漁業向け → JFM とは融資対象が異なる
- **日本政策投資銀行 (DBJ)**: 民間大企業・インフラ向け → JFM とは融資対象が異なる
- **国際協力銀行 (JBIC)**: 海外向け → JFM は国内地方自治体向けで明確に役割分担
- **財政融資資金 (FILP)**: 国が直接融資、JFM と並ぶ地方自治体向け 2 大長期資金源 ^[inferred]

## 6. Counterpoints

- **「地方共同出資の独立性」観**: 国出資なしでも実質的に総務省・財務省の強い監督下にあり、独立性は限定的 ^[ambiguous]
- **「民間金融機関との競合」観**: 民間メガバンク・地銀から見れば「政策金融による民間圧迫」の側面 vs 「採算性低い領域の補完」の側面、両論あり ^[inferred]
- **「公営企業金融公庫廃止の意義」**: 廃止 → 地方共同方式への移行が実質的な業務継続だったため、「特殊法人改革の実効性」に疑問 ^[ambiguous]
- **「人口減少時代の融資戦略」**: 過疎自治体への長期融資はリスク管理上どう対応するか、明確な戦略は限定的 ^[inferred]
- **「災害復興融資のリスク」**: 大震災時の融資集中はリスク管理上の負担。再保険・国補填の制度設計議論あり ^[inferred]
