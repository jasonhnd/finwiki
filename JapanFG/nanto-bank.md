# 株式会社南都銀行 (The Nanto Bank)


## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[JapanFG/ikeda-senshu-hd|池田泉州ホールディングス]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

奈良県地盤の単独銀行（**持株会社化していない地銀**の一つ）。1934-06-01 六十八銀行 + 八木合同銀行 + 御所銀行 + 吉野銀行 + 大和実業銀行 + 第三十国立銀行系などの戦時統合で発足。略称「**なんと**」。**奈良県内シェア圧倒（県内 No.1）+ 大阪府東部・京都府南部への関西 3 県境界エリア展開 + 観光業（奈良公園・東大寺・薬師寺・春日大社・法隆寺）融資 + 不動産融資**が収益コア。関西経済圏の競合は [[resona-hd]] / [[kyoto-fg]] / [[ikeda-senshu-hd]]、隣接県境では紀陽銀行（和歌山）/ 滋賀銀行 / [[tomony-hd]](旧大正銀行)と競合。^[ambiguous]

## 1. 公司结构

**正式名**：株式会社南都銀行
**英名**：The Nanto Bank, Ltd.
**証券コード**：東証 PRIME 8367
**創業**：1934-06-01（六十八銀行 + 八木合同銀行 + 御所銀行 + 吉野銀行 + 大和実業銀行 等 戦時統合）
**本店**：奈良県奈良市橋本町 16
**業態**：単独銀行（**持株会社化していない**点が、近隣の [[kyoto-fg|京都 FG]]・[[ikeda-senshu-hd|池田泉州 HD]]・[[tomony-hd|トモニ HD]] 等の関西地銀 FG 化トレンドと対照的）^[ambiguous]

### 合併歴史・前身

- 1878 **第六十八国立銀行 設立**（奈良、明治期国立銀行条例下、第 68 号で設立）— 南都銀行の最古の源流
- 戦前期、奈良県内で六十八銀行・八木合同銀行・御所銀行・吉野銀行・大和実業銀行 等の複数地銀が並立
- **1934-06-01 南都銀行 発足**：戦時下「一県一行主義」政策の先駆けとして 6 行統合で奈良県内地銀統一を実現
- 1949 東証上場
- 戦後の県内コアバンクとして長期上場、奈良県内シェアを維持
- **2022-04 東証市場区分見直し** → 東証 PRIME 8367（旧東証一部から移行）
- **2024 中期経営計画** 策定 ^[inferred]：地銀デジタル化 + 関西経済圏拡張のフェーズ

> 統合背景：奈良県は戦前から東京・大阪・京都の経済圏に挟まれた中規模県であり、複数地銀が並立する余地が小さかった。1934 の早期 6 行統合は、戦時統制経済以前の自発的統合に近い側面があり、戦後の「一県一行」体制の先駆け例として位置付けられる ^[inferred]。

## 3. 战略与竞争

### 県内地盤戦略

- **奈良県 = 観光業大県**：奈良公園（東大寺・春日大社）・薬師寺・唐招提寺・法隆寺・吉野（桜・修験道）等の世界遺産級観光資源を県内に多数保有。観光ホテル / 寺社関連事業者 / 土産物業者への融資が地銀ポートフォリオの重要セグメント
- **不動産融資（大阪府東部）**：奈良県内のみではボリュームに限界があるため、大阪府東部（東大阪・八尾・柏原・大東 等の中小企業集積地）への展開を強化。製造業中小企業の main bank ポジションを獲得
- **京都府南部（木津川市・精華町・けいはんな学研都市）**：奈良県との県境エリアは生活圏が一体化しており、けいはんな学術研究都市の研究関連企業や住宅地開発への融資が伸びる余地 ^[inferred]
- **県内シェア圧倒**だが、奈良県は人口減（特に若年層の大阪流出）が全国平均より顕著で中長期の構造課題

### 関西経済圏での競合構造

- **関西 3 県境界エリアの特殊性**：奈良県は西で大阪府・北で京都府・南で和歌山県・東で三重県・滋賀県と接し、5 府県境界に位置する。県内完結型の地銀運営は困難で、必然的に **関西経済圏全体での競合** に巻き込まれる構造
- **メガバンク（[[mufg]] / [[smfg]] / [[mizuho-fg]]）との競合**：大阪都市部の大企業案件は メガに流出する構造、南都銀行は奈良県内 + 大阪府東部の中堅・中小企業 main bank ポジションで対抗
- **関西地銀群との競合**：
  - [[resona-hd|りそな HD]]（旧大和銀 + あさひ銀、関西みらい銀含む）：関西最大の地銀グループ、大阪府全域で南都銀と直接競合
  - [[kyoto-fg|京都 FG]]（京都銀行）：京都府全域 + 大阪府北部 + 滋賀県へ拡張、京都府南部で南都銀と接触
  - [[ikeda-senshu-hd|池田泉州 HD]]：大阪府北部（旧池田銀行）+ 大阪府南部（旧泉州銀行）、大阪府全域で競合
  - 紀陽銀行（和歌山県）：南で県境接触、和歌山北部 / 五條エリアで競合
  - 滋賀銀行：東で県境接触、東大寺 / 笠置エリアで競合
  - [[tomony-hd|トモニ HD]] 傘下の旧大正銀行：大阪府で直接競合

### 単独銀行の戦略意義

- **持株会社化していない**点が、近隣 [[kyoto-fg|京都 FG]]（2009 単独 FG 化）・[[ikeda-senshu-hd|池田泉州 HD]]（2009 統合）等の関西地銀 FG 化トレンドと対照的 ^[ambiguous]
- 単独銀行のメリット：(1) ガバナンス簡素、(2) 銀行本体に経営資源集中、(3) 持株会社化の追加コスト不要
- 単独銀行の制約：(1) 銀行法 制約で新業態（証券・リース等）取得は持株会社経由が必要、(2) 異業種 M&A の自由度が低い、(3) デジタル子会社設立の柔軟性が劣る ^[inferred]
- 中長期では、関西地銀再編の中で持株会社化 / 統合方針の判断局面が来る可能性 ^[inferred]

### 地銀再編における位置

- **奈良県の単独地銀リーダーとして独立路線維持**（メガとの提携・SBI 第 4 メガ構想からは距離）
- 関西 3 県境界という地理特殊性により、単独で関西経済圏全体をカバーすることが構造的に困難 → **関西地銀連携 / 再編における立ち位置**が中長期論点 ^[inferred]
- 観光業 + 不動産融資という収益コアは関西の中でも独自性が高く、再編時の交渉力に寄与する可能性

## 4. 监管与政策

- **主管**：金融庁（FSA）+ 近畿財務局
- **銀行業規制**：銀行法（単独銀行のため銀行持株会社規制は対象外）
- **直近政策論点**：
  - 2024〜 日銀政策金利正常化 → 地銀マージン改善（メガより恩恵が相対的に大）
  - 2025〜 地銀再編加速（人口減 + デジタル投資負担）→ 関西地銀群との連携 / 再編判断
  - 単独銀行のまま維持するか、持株会社化（FG 化）へ移行するかの判断
  - 観光業ポートフォリオ（インバウンド回復の追い風 vs 外的ショック脆弱性）への対応
  - 大阪府東部・京都府南部 への営業エリア拡張継続

## Counterpoints

- 「関西 3 県境界の戦略困難性」：関西 5 府県の境界に位置するため、県内完結型運営が困難で、関西経済圏全体での競合（[[resona-hd]] / [[kyoto-fg]] / [[ikeda-senshu-hd]] / メガ）に巻き込まれる構造リスク ^[inferred]
- 「単独銀行維持 vs 持株会社化」：[[kyoto-fg|京都 FG]]・[[ikeda-senshu-hd|池田泉州 HD]] 等の近隣関西地銀 FG 化トレンドに対し南都銀行は単独銀行維持、中長期の方針転換可能性 ^[ambiguous]
- 「観光業ポートフォリオの脆弱性」：奈良観光業向け融資はコロナ危機（2020〜2022）で経験した通り、外的ショック（パンデミック・地震・経済危機等）に脆弱 ^[inferred]
- 「不動産融資集中リスク」：大阪府東部の不動産融資への依存度が高い場合、関西不動産市況の変動に脆弱 ^[inferred]
- 「人口減 + 若年層流出」：奈良県の人口減トレンドは全国平均より顕著（特に大阪流出）、中長期の県内預貸ボリューム縮小は構造リスク ^[inferred]
- 「奈良観光のインバウンド集中度」：外国人観光客比率が高い場合、為替 / 国際情勢に連動するリスク ^[inferred]
