# 株式会社オリエントコーポレーション (Orico)

## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[JapanFG/jaccs|株式会社ジャックス (Jaccs)]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

**[[mizuho-fg]] 系・伊藤忠商事系の国内大手信販**。1954-12-22 広島で「廣島総合信用」として設立、1962 にオリエントファイナンス、2001 に現社名へ改称。**自動車ローン (オートローン) 国内最大手**を背骨に、ショッピングクレジット・クレジットカード (Orico Card 各種)・カードローン・法人ファイナンス・不動産担保ローンを展開。2005-12 経営再建で [[mizuho-fg]] (旧興銀系) + 伊藤忠商事 + KDDI 等の優先株注入を受け、現在は **みずほ FG の持分法適用関連会社 + 伊藤忠商事**が主要株主の枠組み。東証 PRIME 8585。競合は [[jaccs]] / クレディセゾン / 三井住友カード ([[smfg]] 系) / 三菱 UFJ ニコス ([[mufg]] 系) / イオン系 [[aeon-bank]]。^[extracted]

## 1. 会社概要

**正式名**：株式会社オリエントコーポレーション ^[extracted]
**英名**：Orient Corporation ^[extracted]
**略称**：オリコ / Orico
**証券コード**：東証 PRIME **8585** ^[extracted]
**設立**：**1954-12-22** (前身「廣島総合信用株式会社」) ^[extracted]
**現社名改称**：**2001-04** (オリエントファイナンス → オリエントコーポレーション) ^[extracted]
**本社**：東京都千代田区麹町 5-2-1 麹町センタープレイス ^[extracted]
**業態**：信販 (個品割賦・包括信用購入あっせん)・クレジットカード・カードローン・銀行保証・法人ファイナンス・不動産担保ローン

### 社名・略称の由来

- **Orient** = 旧社名「オリエントファイナンス」(1962-) を継承
- **Orico** = **Ori**ent **Co**rporation の合成略称
- 創業地は広島 (旧 廣島総合信用)、後に東京本社へ移転

### 主要株主 (公開情報ベース) ^[extracted]

| 株主 | 性格 | 備考 |
|---|---|---|
| **みずほ銀行** ([[mizuho-fg]] 系) | 大株主・持分法適用関連 | 旧 興銀系列の流れを継承、2005-12 優先株引受 |
| **伊藤忠商事** | 大株主 | 2005-12 優先株引受、流通・自動車販社チャネル親和 |
| (旧)**KDDI** | 過去 出資 | 2005-12 出資、後に持株比率変動 ^[ambiguous] |
| 信託銀行各社・機関投資家 | 浮動株 | 上場会社のため通常開示 |

> みずほ FG 側の連結子会社化は過去 (2010 年前後) に検討された経緯あり ^[ambiguous]、現状は持分法適用関連会社の位置付け (Wikipedia 公開情報ベース)。

### コア差別化

- **自動車ローン国内 No.1**: 自動車販売店との長期チャネル関係が最大の参入障壁。新車・中古車の販社経由割賦で **業界トップシェア** ^[extracted]
- **みずほ系列クレジット中核**: [[mizuho-fg]] グループ内のクレジット・信販事業の中核機能 (vs [[mufg]] の三菱 UFJ ニコス / [[smfg]] の三井住友カード)
- **伊藤忠連携**: 伊藤忠商事の流通・自動車関連ネットワークとの親和性 (商社系顧客基盤・海外進出時の協業)

### 戦略課題

- **オートローン市場の構造変化**: EV シフト・サブスク (KINTO 等)・カーシェアの普及で、従来型「個人購入+割賦」モデルが中長期で変容リスク
- **デジタル化対応**: コード決済 (PayPay / 楽天ペイ / d 払い / au PAY) の急成長で物理カード・割賦モデル全体が押される圧力
- **海外展開**: アジア (インドネシア・タイ・ベトナム) の自動車・二輪ファイナンスは中長期成長領域

### B2C ブランディング

- **Orico Card THE POINT**: 高還元率プロパーカードでの個人客獲得

## 4. 規制・政策

- **主管**: 経済産業省 (割賦販売法所管) + 金融庁 (貸金業法・資金決済法等)
- **主要根拠法**: 割賦販売法 (個品割賦・包括信用購入あっせん)、貸金業法 (カードローン・カードキャッシング)、資金決済法
- **業界自主規制団体**: 一般社団法人日本クレジット協会 (JCA)、日本貸金業協会
- **個人情報保護**: 個人情報保護法 + クレジット関連個人信用情報機関 (CIC / JICC / 全国銀行個人信用情報センター)
- **加盟店管理**: 改正割賦販売法 (2018〜) で加盟店調査義務強化、PCI DSS 準拠
- **直近政策論点**:
  - キャッシュレス推進政策 (経産省目標 2025 年 40%、2030 年代 80%)
  - 改正個人情報保護法 (信用情報の取扱厳格化)
  - グレーゾーン金利問題の収束 (利息制限法・出資法上限金利の一本化、2010-06 完了) ^[extracted]
  - インボイス制度 (2023-10〜) でカード明細・経費精算の取扱変化
