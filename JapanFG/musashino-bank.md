# 武蔵野銀行 (The Musashino Bank, Ltd.)


## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[JapanFG/fukuoka-fg|ふくおか FG (FFG)]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

埼玉県地盤の単独・第一地銀（東証 PRIME **8336**、本店さいたま市大宮区）。1952-03-06 戦後の埼玉県中小企業金融補完目的で **戦後新設地銀** として設立、1973 東証上場。略称「**むさしの**」。県内競合は **埼玉りそな銀行**（[[resona-hd]] 子会社、シェア No.1）。**TSUBASA アライアンス**（[[chiba-bank]] 主導の地銀広域連合）参加で IT・海外・市場運用を共同化。持株会社化していない単独銀行体制が継続中。^[extracted]

## 3. 战略与竞争

### 地盤特性 — 埼玉県という商圏

埼玉県は **人口約 730 万人**（全国 5 位級）、首都圏ベッドタウン経済の代表県。**さいたま市・川越市・川口市・所沢市・越谷市・春日部市** などのベッドタウンと **熊谷・本庄** などの中堅製造業集積を持つ。^[inferred]

商圏としての特徴:
- 個人ローン（住宅ローン）需要が安定して厚い
- 中小製造業（自動車部品・機械・食品）取引基盤
- 東京通勤者層の預金・運用ニーズ
- 県内 GDP は政令市単独（さいたま市）+ 県全体で大手地銀並み市場

### 県内競合構造 ★

| 競合 | 親会社 | 県内ポジション |
|---|---|---|
| **埼玉りそな銀行** | [[resona-hd]] 100% 子会社 | **県内シェア No.1**・旧協和埼玉銀ベース・店舗網最大 ^[inferred] |
| **武蔵野銀行**（本項）| 単独 | 県内シェア No.2 級・地元密着 ^[inferred] |
| **群馬銀行** [[gunma-bank]] | 単独 | 県北エリア競合 ^[inferred] |
| **千葉銀行** [[chiba-bank]] | 単独・TSUBASA 幹事 | 越谷以東・草加 等 県東エリア競合 ^[inferred] |
| **コンコルディア FG** [[concordia-fg]] | 横浜銀＋東日本銀傘下 | 川口・草加 等 県南エリア競合 ^[inferred] |
| メガバンク 3 行 | - | さいたま市 等大都市部支店 |

> **特異点**: 埼玉県は **県名と同名の地銀（埼玉銀行）がりそな統合で消滅** した稀有なケース。旧協和埼玉銀 → りそな統合 → 埼玉りそな分社化（2003-03）の経緯で、現在は「埼玉りそな銀行」が県名冠の地銀ポジション。武蔵野銀行はこの構造で **県内 No.2 ながら独立資本の地元銀** という独自ポジションを維持。^[inferred]

### TSUBASA アライアンス参加（2015〜）

**TSUBASA アライアンス** は [[chiba-bank]] 主導の地銀広域連合で、武蔵野銀行は 2015 年参加。前史として 2003 年から千葉銀との業務提携が存在。^[extracted]

主な共同領域:
- **基幹システム共同化**（TSUBASA 基幹系）
- **海外進出支援ネットワーク**
- **市場運用 / リスク管理 高度化**
- **デジタルサービス共同開発**（アプリ・API 等）
- **人材交流 / 研修**

地銀単独では到達困難な IT 投資・海外拠点・市場運用ノウハウを共同インフラで補完するモデル。^[inferred]

### 持株会社化していない理由（推測）

メガバンク・[[fukuoka-fg]] [[concordia-fg]] [[hokuhoku-fg]] 等は持株会社体制を採用するが、武蔵野銀行は **単独銀行体制を継続**。理由として推測されるのは:

- 県内単一商圏完結型で、複数銀行統合・地理的多角化の必要性が低い
- 子会社（リース・カード・保証）規模が小さく持株会社のスコープメリットが薄い
- TSUBASA で広域連携を満たし、資本統合まで踏み込まない緩い同盟が経済合理的

^[ambiguous]

## 4. 监管与政策

- **主管**: 金融庁（FSA）/ 関東財務局
- **加盟**: 全国地方銀行協会（地銀協）/ 預金保険機構
- **適用規制**: 銀行法・金商法 等 標準的地銀規制
- **直近政策論点**（地銀共通）^[inferred]:
  - 2024〜 日銀政策金利正常化 → 地銀金利マージン改善ボーナス（プラス材料）
  - 2025〜 地銀再編圧力（独占禁止法特例法・地銀共倒れ抑制）
  - 2025〜 持続可能性開示・気候関連リスク開示強化
  - 県内人口動態（高齢化・東京流出）→ 中長期預金量への影響

## Counterpoints

- 「**埼玉りそな に永遠の 2 位**」観: 県内シェア No.1 は埼玉りそな（[[resona-hd]] 子会社）で、武蔵野銀行は店舗網・預金量で構造的に 2 位。一方、独立資本ゆえ意思決定が速く地元密着深度では優位という見方もある ^[ambiguous]
- 「**TSUBASA 依存リスク**」: 基幹システム・海外網を [[chiba-bank]] 主導 TSUBASA に依存 → 千葉銀方針変更時の影響波及。一方、単独投資不可能領域を補完する合理的選択肢 ^[inferred]
- 「**持株会社化しない＝再編余地温存**」観: 単独銀行体制は将来的な大手地銀との経営統合・持株会社化選択肢を温存している、と読める。一方、現状の経営効率は十分との見方も ^[ambiguous]
- 「**首都圏ベッドタウン経済の中長期不確実性**」: 県内人口動態（高齢化・東京流出・少子化）と住宅ローン需要鈍化リスク ^[inferred]
