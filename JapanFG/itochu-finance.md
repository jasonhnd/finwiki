# 伊藤忠ファイナンス (ITOCHU Finance Corporation)

## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[JapanFG/tokyo-century|東京センチュリー]] (the same-parent **外向けリース・asset finance** platform — 伊藤忠 + みずほ FG の 2 軸株主構造、上場 8439) and [[JapanFG/mitsubishi-corp-finance|三菱商事フィナンシャルサービス]] (the closest peer 商社系 group-treasury arm). For the broader peer set, see [[JapanFG/mitsui-bussan-financial-services|三井物産FS]], [[JapanFG/marubeni-financial-services|丸紅フィナンス]], [[JapanFG/sumitomo-corp-financial-management|住友商事FM]]. See [[banking/INDEX|banking index]] for the broader regulatory boundary.

## TL;DR

伊藤忠ファイナンス は **伊藤忠商事 (8001)** 系の **グループ内金融サービス会社**。伊藤忠商事グループの **CMS / 一括ファクタリング / グループファイナンス / 為替・金利リスクヘッジ仲介 / 海外送金 / 信用補完** を担う。リース系 (外向け asset finance) は [[JapanFG/tokyo-century|東京センチュリー]] (伊藤忠商事 + [[JapanFG/mizuho-fg|みずほ FG]] の 2 軸株主、上場 8439) が担うため、伊藤忠は「**外向けリース = 東京センチュリー / 内向け = 伊藤忠ファイナンス**」の二層構造。^[inferred]

### 伊藤忠系金融機能の構造

伊藤忠商事の金融機能は以下のように分散している：^[inferred]

| 層 | 主体 | 役割 |
|---|---|---|
| **外向け銀行 (主取引)** | [[JapanFG/mizuho-fg|みずほ FG]] / [[JapanFG/mufg|MUFG]] | 商業銀行・対外金融サービス |
| **外向けリース (上場)** | [[JapanFG/tokyo-century|東京センチュリー]] (8439) | 大型リース・航空機 (米 ACG)・海外案件・モビリティ — 伊藤忠 + みずほ FG 2 軸 |
| **内向け / グループ金融** | **伊藤忠ファイナンス (本ページ)** | グループ各社の internal banking / CMS / 内部ファクタリング |
| **生損保 (持分法)** | [[JapanFG/aioi-nissay-dowa-insurance|あいおいニッセイ同和損保]] 等への部分出資 ^[inferred] | 損保 |
| **証券・AM (限定)** | 部分出資 ^[inferred] | - |

### 「伊藤忠商事 + みずほ FG」連合の金融構造

伊藤忠は メガバンク に対しては みずほ FG との関係性が深く、東京センチュリー の 2 軸主要株主構造はその象徴。^[inferred] 一方で MUFG とも一定の取引関係を持ち、商社として「単一銀行系列」に閉じない柔軟な金融戦略をとる。伊藤忠ファイナンス はこの構造の **内向き機能** を担う。

## 3. 战略与竞争

### 伊藤忠商事の事業構造との関係

伊藤忠商事は **繊維・機械・金属・エネルギー・化学品・食料・住生活・情報・金融** の事業ポートフォリオを持つ。各セグメントが固有の与信・為替・コモディティ価格リスクを抱えるなか、伊藤忠ファイナンス はこれらを集約管理する内部金融機能を提供。^[inferred]

特に **食料 / 住生活 / 情報** セグメントは中小サプライヤーとの取引が多く、CMS / 一括ファクタリングを通じてサプライヤー側のキャッシュフロー安定化と本社側の支払いコスト最適化を同時実現する典型例。

## 4. 监管与政策

- **主管**：金融庁 (貸金業 / 為替業務関連)
- **業界団体**：(商社系内部金融としてリース事業協会への加盟は限定的) ^[inferred]
- **規制論点**：
  - グループ内金融取引の **移転価格税制**
  - グループファクタリングと **割賦販売法 / 貸金業法** の境界
  - 連結会計上の **内部取引消去**
- **直近政策論点**:
  - 2024〜 [[JapanFG/boj-monetary-policy|日銀政策金利正常化]] → グループ内金利付与の再評価
  - 2025〜 海外送金 / クロスボーダー資金移動の AML/CFT 強化
  - 伊藤忠が注力する DX / デジタル金融 (Family Mart 経済圏・[[JapanFG/famima-digital-one|ファミマデジタルワン]] 等) とのシナジー ^[inferred]

## Counterpoints

- 「内部金融子会社」はガバナンスが形式的になりがち ^[inferred]
- 外部開示が限定的、業績規模は推測ベース ^[inferred]
- 伊藤忠系金融機能は [[JapanFG/tokyo-century|東京センチュリー]] (上場・大型・グローバル) との分担関係が明確ではない部分がある (どこまでが内向け、どこからが外向けか) ^[ambiguous]
- 伊藤忠本体が CITIC グループとの戦略提携を持つ中、伊藤忠ファイナンスがアジア / 中国向け金融機能で関与するかは不透明 ^[ambiguous]
