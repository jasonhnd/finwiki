# JapanFG securities and asset management sub-index

## Overview

This sub-index routes the **securities and asset-management** layer of the JapanFG namespace: securities firms and groups, online / discount brokers and FX operators, and asset-management companies. It complements the legal route in [[JapanFG/legal-financial-licenses/securities-license-stack|Japan securities license stack]]. Entity pages stay in [[JapanFG/INDEX|JapanFG]].

Use this page when the question is **"which securities and asset management entities does JapanFG cover?"** It lists 50 entity pages by institution type. Each link points to a page that physically lives in `JapanFG/`; this sub-index adds discoverability without moving any file.

## Placement Rule

- **Entity pages stay in `JapanFG/`.** This sub-index only links to them with bare `[[JapanFG/slug]]` wikilinks; it does not move, rename, or duplicate any entity file.
- Membership here is derived from each entity page's frontmatter `tags:` taxonomy. An entity may legitimately also appear in an adjacent sub-index when its activity spans types.
- This is a navigation surface, not a registry of record. Where an FSA registry index already exists for a type, treat that registry page as the source of truth.
