# 極東証券 (Kyokuto Securities)

## TL;DR

極東証券は **1949 年創業の中堅独立証券** (東証 STANDARD **8706**)。FSA 金融商品取引業者一覧では「極東証券株式会社」、関東財務局長(金商)第65号として掲載される。**外国債券 (外債) 仲介に強み** を持つ独立系中堅で、東京を本拠に対面リテール + 外債販売の伝統的モデルを維持。**日本証券業協会 ([[JapanFG/jsda|JSDA]]) 正会員**。同枠の独立中堅証券 ([[JapanFG/marusan-securities|丸三証券]] / [[JapanFG/aizawa-sg|アイザワ証券]] / [[JapanFG/tachibana-securities|立花証券]]) と並ぶが、**外債特化** のニッチで差別化されるのが特徴。

**正式名**：極東証券株式会社

**本社**：東京都中央区日本橋茅場町 (兜町証券街隣接)
**業態**：金融商品取引業 (第一種金融商品取引業者) — 中堅独立系証券、外債仲介に強み

### 業態の特徴

極東証券は東証 STANDARD 上場の中堅独立証券で、**外国債券 (外債) 仲介** に強みを持つ。日本では大手証券 ([[JapanFG/nomura-hd|野村]] / [[JapanFG/daiwa-sg|大和]] / [[JapanFG/smbc-nikko|SMBC 日興]]) も外債販売を扱うが、極東証券は**規模は小さいが外債プロダクトラインに特化**することで富裕層 / シニア層向けの安定収益基盤を維持。

## 4. 战略与競争

### 外債特化型ニッチ

極東証券の差別化軸は **外国債券 (外債) 仲介に特化したリテール販売**。1990 年代以降の円高局面で外債商品ニーズが高まる中、中堅独立証券として外債プロダクトラインを厚く整備し、富裕層 / シニア層の安定収益基盤を確保。これは大手証券の総合プロダクトに対し、**プロダクト深耕型** の差別化戦略。

### 中堅独立証券枠でのポジション

- 同枠の [[JapanFG/marusan-securities|丸三証券]] (1910 年創業) / [[JapanFG/aizawa-sg|アイザワ証券]] (1918 年創業) より新興 (1949 年創業) だが、外債特化軸で独自ポジション確立
- [[JapanFG/aizawa-sg|アイザワ証券]] の「アジア新興国株式」と並ぶ「プロダクト特化型独立中堅証券」モデル
- 大手証券 ([[JapanFG/nomura-hd|野村]] / [[JapanFG/daiwa-sg|大和]]) の外債商品とは富裕層対面販売の信頼関係で差別化 ^[inferred]

### NISA 拡充の影響 ([[securities/nisa-2024-flow|nisa-2024-flow]] 文脈)

- 2024〜 新 NISA 拡充で、債券・外債商品の NISA 対象拡大期待 ^[inferred]
- 米国債等の外債商品を NISA 対応口座で扱うことで、シニア層 + 中長期保有層への訴求強化 ^[inferred]

## 5. 监管と政策

- **主管**：金融庁 (FSA)・関東財務局
- **業態**：第一種金融商品取引業 (証券業)
- **規制**：金融商品取引法・日本証券業協会 ([[JapanFG/jsda|JSDA]]) 自主規制
- **外債販売関連**：外国証券業務に関する自主規制 (顧客説明義務・通貨リスク開示等)
- **直近政策論点**：
  - 2024〜 新 NISA 拡充：外債 NISA 対応の商品ラインナップ強化 ^[inferred]
  - 顧客本位の業務運営 (FD 原則) 強化：外債商品の通貨・信用リスク開示の厳格化
  - 金利上昇局面での外債価格変動リスク顕在化への対応

## Counterpoints
