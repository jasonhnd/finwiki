# いよぎんリース (Iyogin Leasing)

## Wiki route

This entry sits under the [[JapanFG/INDEX|JapanFG index]] as the FG-subsidiary leasing operating-company anchor for [[iyogin-hd|いよぎん HD]] グループ ([[iyo-bank|伊予銀行]] 母体)。Read it against parent operating bank [[iyo-bank|伊予銀行]] and sibling subsidiaries いよぎん DC カード / いよぎん地域経済研究センター (IRC) / いよぎんキャピタル. Peer comparison: [[mebuki-lease|めぶきリース]] (北関東) / [[yokohama-bank-leasing|横浜銀リース]] (神奈川) / [[chibagin-leasing|ちばぎんリース]] (千葉) / [[yamaguchi-leasing|山口リース]] (中国地方西部) / [[hokkaido-lease|北海道リース]] and major lessor [[mitsubishi-hc-capital|三菱 HC キャピタル]] / [[mizuho-lease|みずほリース]] / [[tokyo-century|東京センチュリー]]. For broader system route see [[banking/regional-bank-consolidation-pattern|regional-bank consolidation pattern]], [[banking/regional-bank-api-digital-partnership-route|regional-bank API/digital partnership route]] and [[banking/INDEX|banking index]]. Investable-universe context: [[finance/japan-listed-financial-groups-investable-universe|Japan listed financial groups investable universe]].

## TL;DR

株式会社いよぎんリースは、[[iyogin-hd|いよぎん HD (東証 PRIME 5830)]] グループの地銀系リース会社で、中核子会社 [[iyo-bank|伊予銀行]] (1878 第二十九国立銀行設立、愛媛県シェア No.1) の中堅企業 main bank 顧客基盤を背景に、**愛媛県 + 四国全域 + 大阪・東京・広島等本州拠点**で設備リース・ファイナンスリース・オペレーティングリース・自治体リース・医療リース・**海事クラスター (今治造船 + 海運) 設備リース**・農業機械リース・オートリース等を提供。四国 4 県の地銀系リースとしては最大級で、[[iyogin-hd]] 2022-10-03 持株会社化後にグループ並列子会社の一角として再配置された。^[extracted]

```

  ├── [[iyo-bank|伊予銀行]] (100%) ── 中核子会社・愛媛県シェア No.1・四国最古級銀行 (1878-)
```

### 収益構造の特徴 ^[inferred]

- 主要収益源は **リース料収入 - 減価償却費 - リース債権取得コスト**
- [[iyo-bank|伊予銀行]] 中堅企業 main bank ネットワーク + 海事クラスター・製紙業など愛媛県特有産業集積がリース受託入口
- **海事クラスター連携**は他地銀系リースとの差別化要因 ([[iyo-bank]] の今治造船融資特性を活かす)
- 業績は親 [[iyogin-hd]] 連結セグメント情報経由で部分開示 ^[ambiguous]

## 3. 親会社・グループ内ポジション

いよぎん HD グループ内では、中核子会社 [[iyo-bank|伊予銀行]] の **愛媛県シェア No.1 + 四国全域 + 海事クラスター main bank 顧客基盤に対するリース業務窓口** という位置づけ。**いよぎん HD 2022-10-03 持株会社化**後、いよぎんリースは持株会社直下並列子会社として再配置され、[[mebuki-fg]] / [[fukuoka-fg]] / [[shizuoka-fg]] と同型の「持株会社下兄弟会社」model に移行 ^[inferred]。

### グループ内クロスリンク

| グループ会社 | 連携内容 |
|---|---|
| [[iyo-bank|伊予銀行]] | 法人顧客紹介、海事クラスター融資 + リース組合せ提案、四国広域営業 |
| いよぎん DC カード | 法人カード + 経費精算 + リース料決済 cross-sell ^[inferred] |
| いよぎん IRC | 愛媛・四国業界レポート (造船 / 製紙 / 農業) 提供 |
| いよぎんキャピタル | スタートアップ向けリース + 投資 一体提案 ^[inferred] |
| いよぎんビジネスサービス | BPO 経由のリース管理業務効率化 ^[inferred] |

### TSUBASA アライアンス参加 (伊予銀)

[[iyo-bank|伊予銀行]] は [[chiba-bank|千葉銀行]] 主導の **TSUBASA アライアンス** (2015-) 参加 11 行 (千葉銀 + 北洋 + 武蔵野 + 滋賀 + 第四北越 + 中国 + 伊予 + 東邦 + 北國 + 京都 + 群馬) の一員。これによりいよぎんリースは [[chibagin-leasing|ちばぎんリース]] / 群馬リース / 京都リース 等アライアンス参加行系リースとの情報共有・共同案件組成余地あり ^[inferred]。

### 大手リース・専業リースとの競合

- **大手リース**: [[mitsubishi-hc-capital|三菱 HC キャピタル]] / [[smfl|SMFL]] / [[mizuho-lease|みずほリース]] / [[fuyo-lease|芙蓉リース]] / [[tokyo-century|東京センチュリー]] / [[ricoh-leasing|リコーリース]] - 大型・全国・複雑案件で優位 (海事リースでは三菱 HC が今治造船関連にも強い)
- **メーカー系リース**: [[toyota-finance|トヨタファイナンス]] / [[nec-leasing|NEC キャピタルソリューション]] / [[showa-leasing|昭和リース]] / [[ja-mitsui-leasing|JA 三井リース]] - 製品連動で優位
- **県内競合**: [[ehime-bank|愛媛銀行]] (第二地銀) 系リース / 大手リース愛媛支店

### 強み・弱み

- 強み: [[iyo-bank|伊予銀行]] 愛媛県シェア No.1 + 四国広域営業 + **海事クラスター差別化** (今治造船連携) + 製紙業・農業など愛媛特有産業
- 弱み: 単独規模では大手リースに大幅劣後、航空機 / 大型不動産 / 海外複雑リースは対応困難、四国 = 人口減少先行地域 ^[inferred]

## 5. 規制・会計

### 規制枠組み

- **主管**: 金融庁 (FSA) ・経済産業省
- **法的根拠**: リース業 (銀行法上の銀行業免許不要)、貸金業登録 (ファクタリング領域で必要に応じて取得)
- **業界団体**: リース事業協会 加盟 ^[inferred]
- **親 HD ガバナンス**: [[iyogin-hd]] 連結子会社として持株会社ガバナンス下、銀行業以外子会社規制 (主要業務 / 付随業務 / 業務範囲規制) で位置づけられる
- **地方財務局**: 四国財務局 (高松) 管轄

### 規制論点

- **リース会計基準変更 (IFRS16 / 日本基準改訂)**: 借手のオンバランス化 (使用権資産・リース負債) 対応支援
- **中小企業向けリース税制**: 中小企業投資促進税制 / 経営強化税制活用支援
- **環境リース (再エネ設備 / EV)**: 脱炭素税制対応・グリーンリース商品開発
- **海事リース特有規制**: 船舶リース・船舶部品リースは海事法 (船員法・船舶法) との整合 ^[inferred]
- **割賦販売法 / 貸金業法**: ファクタリング領域での適用整理
- **反社チェック・マネロン対策**: [[iyogin-hd]] グループ共同実施
- **EDINET / 有価証券報告書**: いよぎんリース単体は非上場で開示限定的、[[iyogin-hd]] 連結セグメント情報で間接開示 ^[ambiguous]

### 会計上の取扱い

- **連結対象**: [[iyogin-hd]] 100% 子会社として連結対象、銀行業セグメント以外の「その他」セグメントに集約されるのが典型 ^[inferred]
- **単体財務 KPI**: 親 [[iyogin-hd]] 有価証券報告書注記 / セグメント情報で部分開示 ^[ambiguous]
- **リース債権**: 連結 BS 上「リース債権及びリース投資資産」勘定で表示 ^[inferred]
- **報告通貨**: 円建て、日本基準 (リース会計基準) 適用 ^[inferred]

## Counterpoints

- 「地銀系リース = 中途半端」論: 大手リースに規模で劣後 vs 銀リース連携の優位性 ^[ambiguous]
- 「HD 化 = 形式だけ」論: [[iyogin-hd]] 2022-10-03 持株会社化後、いよぎんリースの実質的な戦略変化は限定的との見方 vs 持株会社化により資本効率改善 / M&A 余地確保 ^[inferred]
- 「単独存続困難」論: リース業界寡占化進展、地銀系リース統合・売却シナリオ ^[inferred]
- 「FG 内利益相反」: [[iyo-bank|伊予銀行]] 融資 vs いよぎんリース提案の使い分け基準 ^[inferred]
- 「四国 = 人口減少リスク」: 四国は全国でも人口減少先行地域 ([[iyogin-hd]] 中期経営計画でも認識)、リース需要長期縮小懸念 ^[inferred]
- 「海事クラスター集中 = 海運市況リスク」: 今治造船 / 海運業界の市況変動で需要変動大 ^[inferred]
