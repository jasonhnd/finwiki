# ちばぎんリース (Chibagin Leasing)

## Wiki route

This entry sits under the [[JapanFG/INDEX|JapanFG index]] as the FG-subsidiary leasing operating-company anchor for [[chiba-bank|千葉銀行 (Chiba Bank)]]. Read it against parent operating bank [[chiba-bank|千葉銀行]] (no-holding 単独銀行 model) and sibling subsidiary [[chibagin-securities|ちばぎん証券]]. Peer comparison: [[mebuki-lease|めぶきリース]] (北関東地銀系リース) / [[yokohama-bank-leasing|横浜銀リース]] (神奈川地銀系リース) / [[hokkaido-lease|北海道リース]] / [[yamaguchi-leasing|山口リース]] / [[iyogin-leasing|いよぎんリース]] and major lessor [[mitsubishi-hc-capital|三菱 HC キャピタル]] / [[mizuho-lease|みずほリース]] / [[tokyo-century|東京センチュリー]]. For broader system route see [[banking/regional-bank-consolidation-pattern|regional-bank consolidation pattern]], [[banking/regional-bank-api-digital-partnership-route|regional-bank API/digital partnership route]] and [[banking/INDEX|banking index]]. Investable-universe context: [[finance/japan-listed-financial-groups-investable-universe|Japan listed financial groups investable universe]].

## TL;DR

株式会社ちばぎんリースは、[[chiba-bank|千葉銀行 (東証 PRIME 8331)]] 100% 子会社の地銀系リース会社で、千葉県・首都圏東部を地盤に**設備リース・ファイナンスリース・オペレーティングリース・自治体リース・医療リース・ファクタリング・オートリース**等を提供。地銀系リース上位の一角で、千葉銀の千葉県内シェア No.1 + 預金 18-20 兆円規模の中堅企業 main bank 顧客基盤を背景とする。**親 [[chiba-bank]] が持株会社化していない単独銀行 (no-holding model)** のため、ちばぎんリースは銀行本体直下にぶら下がる構造で、[[mebuki-lease]] (持株会社下) や [[yokohama-bank-leasing]] (銀行連結子会社) とは法的位置が異なる。^[extracted]

### 収益構造の特徴 ^[inferred]

- 主要収益源は **リース料収入 - 減価償却費 - リース債権取得コスト + ファクタリング手数料 + オートリース月額収入**
- 千葉県地盤の中堅製造業 / 物流 / 医療 / 自治体 main bank ネットワークがリース受託の入口
- 大手リース ([[mitsubishi-hc-capital]] / [[smfl|SMFL]] / [[mizuho-lease]] / [[fuyo-lease]] / [[tokyo-century]] / [[ricoh-leasing]]) との競合では地銀連携で差別化
- 業績は親 [[chiba-bank]] 連結セグメント情報経由で部分開示 ^[ambiguous]

## 3. 親会社・グループ内ポジション

千葉銀グループ内では、銀行本体の **預金 18-20 兆円規模・千葉県シェア No.1 の中堅企業 main bank 顧客基盤に対するリース業務窓口** という位置づけ。**銀リース連携** ([[chiba-bank]] 法人顧客紹介 → リース提案 → 銀行融資 + リース 組合せ提案) がコアバリュー。千葉銀が持株会社化していない単独銀行 (no-holding model) のため、ちばぎんリースは法的に銀行本体の子会社 (直下) で、[[mebuki-fg]] の [[mebuki-lease]] (FG 直下の兄弟会社) や [[concordia-fg]] / [[yokohama-bank]] の [[yokohama-bank-leasing]] (銀行連結子会社) とはガバナンス構造が異なる ^[inferred]。

### グループ内クロスリンク

| グループ会社 | 連携内容 |
|---|---|
| [[chiba-bank]] | 法人顧客紹介、銀行融資 + リース組合せ提案、京葉工業地帯製造業 main bank 連携 |
| [[chibagin-securities]] | 中堅企業 main bank クロス提案 (リース + 私募債 + 株式公開) |
| ちばぎんカード | 法人カード + 経費精算 + リース料決済 cross-sell ^[inferred] |
| ちばぎん総研 | 千葉県業界レポート (製造業設備投資動向) 提供 |
| ちばぎんキャピタル | スタートアップ向けリース + 投資 一体提案 ^[inferred] |
| ちばぎん保証 | リース債権保証 (与信補完) ^[inferred] |

### TSUBASA アライアンス連携余地

千葉銀が 2015 主導設立した **TSUBASA アライアンス** (11 行協業: 千葉銀 + 北洋 + 武蔵野 + 滋賀 + 第四北越 + 中国 + [[iyo-bank|伊予]] + 東邦 + 北國 + 京都 + 群馬) を背景に、ちばぎんリースもアライアンス参加行系リース ([[iyogin-leasing|いよぎんリース]] / 群馬リース / 京都リース 等) との情報共有・共同案件組成余地あり ^[inferred]。

## 5. 监管与会计

### 規制枠組み

- **主管**: 金融庁 (FSA) ・経済産業省
- **法的根拠**: リース業 (銀行法上の銀行業免許不要)、貸金業登録 (ファクタリング / 集金代行領域で必要に応じて取得)
- **業界団体**: リース事業協会 加盟 ^[inferred]、社団法人リース信用機構 加盟 ^[inferred]
- **親銀行ガバナンス**: [[chiba-bank]] が単独銀行のため、銀行法上の銀行業以外子会社規制 (主要業務 / 付随業務 / 業務範囲規制) で位置づけられる

### 規制論点

- **リース会計基準変更 (IFRS16 / 日本基準改訂)**: 借手のオンバランス化 (使用権資産・リース負債) 対応支援
- **中小企業向けリース税制**: 中小企業投資促進税制 / 経営強化税制活用支援
- **環境リース (再エネ設備 / EV)**: 脱炭素税制対応・グリーンリース商品開発
- **割賦販売法 / 貸金業法**: ファクタリング / 集金代行領域での適用整理
- **反社チェック・マネロン対策**: [[chiba-bank]] グループ共同実施
- **EDINET / 有価証券報告書**: ちばぎんリース単体は非上場で開示限定的、[[chiba-bank]] 連結セグメント情報で間接開示 ^[ambiguous]

### 会計上の取扱い

- **連結対象**: [[chiba-bank]] 100% 子会社として連結対象、銀行業セグメント以外の「その他」セグメントに集約されるのが典型 ^[inferred]
- **単体財務 KPI**: 親 [[chiba-bank]] 有価証券報告書注記 / セグメント情報で部分開示 ^[ambiguous]
- **リース債権**: 連結 BS 上「リース債権及びリース投資資産」勘定で表示 ^[inferred]
- **報告通貨**: 円建て、日本基準 (リース会計基準) 適用 ^[inferred]

## Counterpoints

- 「地銀系リース = 中途半端」論: 大手リースに規模で劣後 vs 銀リース連携の優位性 ^[ambiguous]
- 「no-holding model = 戦略的柔軟性低下」論: [[mebuki-fg]] / [[fukuoka-fg]] のような持株会社化により子会社並列配置 / 資本効率改善余地を取れないリスク vs 千葉銀の単独銀行 model は迅速意思決定のメリット ^[inferred]
- 「単独存続困難」論: リース業界寡占化進展 ([[mitsubishi-hc-capital]] / [[smfl]] / [[mizuho-lease]] の上位 3 強)、地銀系リース統合・売却シナリオ ^[inferred]
- 「FG 内利益相反」: [[chiba-bank]] 銀行融資 vs ちばぎんリース提案の使い分け基準 ^[inferred]
- 「千葉県集中 = 人口減少リスク」: 千葉県は首都圏東部位置で他県より緩やかだが、長期人口減少局面で設備投資需要縮小懸念 ^[inferred]
- 「TSUBASA アライアンス内競合 / 協業」: 11 行系リース子会社の地理的競合 (例: 群馬リース等北関東地盤と) vs 共同案件組成余地 ^[ambiguous]
