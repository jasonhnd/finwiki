# 札幌中央信用組合 (Sapporo Chuo Credit Cooperative)

## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]] as a **Hokkaido 地域型 (regional) shinkumi anchor**, peer to [[JapanFG/iwaki-shinkumi|いわき信用組合]] (Tohoku) within the 143-cooperative population catalogued in [[banking/credit-cooperative-registry-japan|Japan credit cooperative registry]]. Contrast with [[JapanFG/sapporo-shinkin|札幌信用金庫]] for the shinkin-vs-shinkumi distinction in the Sapporo market.

## TL;DR

札幌中央信用組合は、北海道札幌市中央区南2条西に本店を置く **札幌都市圏の中小企業者・住民を組合員基盤とする地域型信用組合**。FSA registry No. 3 (北海道財務局所管)、法人番号 5430005003132。北海道 7 信組のうち札幌都心部を本拠とする代表行で、[[JapanFG/zenshin-kumi|全国信用協同組合連合会 (全信組連)]] 会員、[[JapanFG/dic|預金保険機構]] 対象。**地域 + 業域**の二重組合員資格、札幌市内 SME 融資・住宅ローン・商店街金融が事業の中核。北海道経済の中心地札幌でメガバンク・[[JapanFG/hokuyo-bank|北洋銀行]]・[[JapanFG/hokkaido-bank|北海道銀行]] と棲み分けるニッチ金融機関。^[extracted]

### 営業地区 ^[inferred]

- **本店**: 札幌市中央区南2条西 (大通公園・狸小路アーケード・すすきの隣接エリア、商業中心地)
- **支店網**: 札幌市内中心 (中央区・北区・西区・東区・南区・豊平区・厚別区・手稲区・清田区・白石区) + 道央近郊

### 組合員資格 ^[extracted]

地域型信組のため、組合員資格は **地区 + 業域**の二層構造 (中小企業等協同組合法準拠):

1. **地区要件**: 札幌中央信用組合の定款で定める地区 (主に札幌市) に在住・在勤・事業所を有する個人・法人
2. **規模要件**: 中小企業者 — 従業員 300 人以下または資本金 3 億円以下 (業種により異なる)

### 北海道信組ブロック (registry比較)

北海道には FSA registry 上 7 信用組合 (registry No. 1-7):

| 信組 | 本店 | 類型 |
|---|---|---|
| ウリ信用組合 | 札幌市中央区大通西 | 民族系 (在日韓国・朝鮮人系) |
| 釧路信用組合 | 釧路市 | 道東地域型 |
| **札幌中央信用組合** ★本ページ | **札幌市中央区南2条西** | **道央札幌地域型** |
| 空知商工信用組合 | 美唄市 | 道央内陸・商工系 |
| 十勝信用組合 | 帯広市 | 道東十勝地域型 |
| 函館商工信用組合 | 函館市 | 道南・商工系 |
| 北央信用組合 | 札幌市中央区 | 道央・地域型 |

> 注: 札幌市内には **札幌中央信組 + 北央信組 + ウリ信組**の 3 信組が並存。札幌中央が中央区南2条西、北央が中央区南1条西8、ウリ信組が中央区大通西12と、いずれも中央区中心部に集中する密集配置。

### 全信組連内の位置付け

- **[[JapanFG/zenshin-kumi|全信組連]]** の北海道ブロック会員行
- 北海道 7 信組の中核札幌都市圏代表
- 系統 ATM 連携・余資運用は全信組連経由

### ALM・資金フロー ^[inferred]

```

  → 預金 → 札幌中央信用組合
              → 札幌都心 SME 融資 + 住宅ローン
              → 余資預け入れ → [[JapanFG/zenshin-kumi|全信組連]]
                                  → 機関投資 + 系統還元
```

## 6. 監督・規制

- **主管**: 金融庁・北海道財務局
- **法的根拠**: 中小企業等協同組合法 + 協同組合による金融事業に関する法律 (協金法)
- **預金保険**: [[JapanFG/dic|預金保険機構]] 一般金融機関枠
- **直近政策論点**:
  - **DX 対応** (組合員向けスマホアプリ・電子契約)
  - **AML/CFT 強化**対応コスト
  - **札幌都心人口集中**の継続 vs **道内地方部過疎化**との相反トレンド
