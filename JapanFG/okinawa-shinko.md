# 沖縄振興開発金融公庫 (ODFC)


## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[JapanFG/jfc|日本政策金融公庫 (JFC)]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

沖縄県限定の政策金融特殊法人。**1972-05-15 沖縄本土復帰と同時に設立**された県内唯一の総合政策金融機関で、本土の [[jfc]]（国民・中小企業・農林漁業）+ [[dbj]]（中堅・大企業）+ 住宅金融支援機構 の機能を**沖縄県内で一機関に統合**したユニークな存在。国 100% 出資の特殊法人で、内閣府沖縄担当部局が主管。**沖縄振興特別措置法**に基づく特別金利・特別保証で県内事業者・住宅取得者・離島住民を支援。2008 政策金融再編で本土公庫が JFC に統合された際にも、**「沖縄の特殊事情」を理由に単独存続が決定**された希少な機関。^[extracted]

## 1. 会社概要

**正式名**：沖縄振興開発金融公庫 ^[extracted]
**英名**：Okinawa Development Finance Corporation (ODFC)
**根拠法**：沖縄振興開発金融公庫法（昭和 47 年法律第 31 号）^[extracted]
**設立**：1972-05-15（沖縄本土復帰と同時、琉球政府金融機関を承継）^[extracted]
**形態**：特殊法人（株式会社形態ではなく、本土の JFC とは別組織体）^[extracted]
**本店**：沖縄県那覇市おもろまち 1-2-26（那覇市新都心地区）^[likely]
**出資者**：日本政府 100%（一般会計出資 + 財投借入）^[extracted]
**主管**：内閣府（沖縄担当部局）+ 財務省（共管）^[extracted]

### 設立背景 ^[extracted]

1972-05-15 沖縄本土復帰時、琉球政府下の以下 4 金融機関を統合承継：

| 統合前機関（琉球政府下）| 承継機能 |
|---|---|
| 琉球開発金融公社 | 産業基盤・中堅企業向け開発融資 |
| 民政府工業開発金融公庫 | 工業振興向け融資 |
| 大衆金融公庫 | 中小・小規模事業者向け融資 |
| 琉球住宅公庫 | 住宅取得・建設融資 |

→ 本土復帰と同時に**沖縄振興開発金融公庫**として再編・統合発足

### ガバナンス・機関設計

- **理事長**：内閣総理大臣の認可を経て主務大臣が任命 ^[likely]
- 役員：理事長 + 副理事長 + 理事 + 監事（特殊法人ガバナンス標準）
- 経営委員会 / 業務委員会で事業運営
- **国会報告義務**：毎年度業務概況・決算を国会に提出 ^[likely]
- **会計検査院**：政府全額出資のため常時検査対象

### 事業部門（県内完結型・一機関統合）

```
ODFC（沖縄振興開発金融公庫・100% 国出資）
  ├── 中小企業事業
  │     └── 県内中小企業向け運転・設備資金（本土 JFC 中小企業事業 + DBJ 機能の沖縄版）
  ├── 産業開発事業
  │     └── 観光・製造・農水産加工・物流向け開発融資（本土 DBJ 機能の沖縄版）
  ├── 住宅事業
  │     └── 個人住宅ローン・賃貸住宅建設（本土 住宅金融支援機構 機能の沖縄版)
  ├── 農林水産業事業
  │     └── 県内農家・畜産・漁業向け融資(本土 JFC 農林水産事業 の沖縄版)
  ├── 国民生活事業
  │     └── 県内個人事業主・小規模事業者向け融資(本土 JFC 国民生活事業 の沖縄版)
  ├── 災害復興事業
  │     └── 台風・地震・津波等の災害復興特別貸付
  └── 教育貸付事業
        └── 沖縄県内大学生・専修学校生向け奨学融資・教育ローン
```

### 県内完結型の特殊性

- **本土の 3 公庫機能を 1 機関で提供**：JFC は本土で「国民生活 + 中小企業 + 農林水産」と部門分けされるが、ODFC は沖縄県内のみでこれらに加え DBJ 機能 + 住宅金融支援機構機能まで**全て統合**して提供
- **沖縄振興特別措置法に基づく特別金利**：本土 JFC より一段低い政策金利水準を県内事業者に適用可能
- **島嶼地域(離島)対応**：宮古・八重山・大東諸島など県内離島の事業者・住民に対する独自融資メニュー

## 3. 戦略・競争

### 民間補完原則(沖縄県内版)

ODFC の存在意義は本土 JFC 同様「**民間金融機関が貸せないリスクをカバー**」する点にあるが、**沖縄県内 3 行寡占構造**との補完関係が独特：

- **[[okinawa-fg]] 沖縄銀行**(県内預貸金シェア約 30% 首位)
- **[[ryukyu-bank]] 琉球銀行**(同 No.2、県指定金融機関を沖縄銀行と隔年受託)
- **沖縄海邦銀行**(同 No.3)

→ この 3 行プロパー融資 + ODFC 政策融資の協調融資が県内中小企業金融の標準形

### 現行戦略の 4 軸 ^[likely]

1. **観光業復興・拡張支援**：那覇空港第 2 滑走路 + JUNGLIA テーマパーク(2025〜開業)+ MICE 拠点整備等、コロナ後の観光業 V 字回復を支える融資供給
2. **米軍基地跡地再開発**：返還予定地(西普天間・キャンプキンザー 等)の都市再開発・産業用地化に伴うプロジェクト融資
3. **離島振興・島嶼地域支援**：宮古・八重山・大東諸島など本土と異なる経済構造を持つ離島の事業者・住民を支援
4. **災害復興・防災**:台風・地震・津波等の自然災害が多い県情を反映した復興融資体制の常時整備

### 第 6 次沖縄振興計画(2022〜2031)との連動 ^[extracted]

2022-2031 年度の第 6 次沖縄振興計画では以下を重点分野とし、ODFC が中核資金提供機関として位置付け：

- **観光関連産業の高度化**
- **情報通信関連産業・IT 拠点強化**
- **物流・国際ハブ機能(那覇空港・那覇港)**
- **健康医療・バイオ関連産業**
- **農林水産業の高付加価値化(サトウキビ・マンゴー・水産加工 等)**
- **離島振興・条件不利地域支援**

### 民間金融機関との関係

- **協調融資の主役**：[[okinawa-fg]] / [[ryukyu-bank]] / 沖縄海邦銀行とのプロパー併用が県内中小・観光業融資の標準形
- **県内地銀との補完原則**:民業圧迫を避けるため、創業期・無担保無保証人・離島・災害復興など民間が貸しにくい層を優先
- **本土公庫との棲み分け**:[[jfc]] / [[dbj]] / 住宅金融支援機構 とは「沖縄県内案件は ODFC、本土案件は本土公庫」と地域で分業

## 4. 規制・政策

- **主管**：内閣府(沖縄担当部局)+ 財務省 共管 ^[extracted]
- **根拠法**:沖縄振興開発金融公庫法(昭和 47 年法律第 31 号)
- **基本法**:沖縄振興特別措置法(平成 14 年法律第 14 号、5 年ごと改正)
- **会計検査院**:政府全額出資のため常時検査対象
- **国会報告**:毎年度業務概況・決算を国会に報告
- **資金調達**:財投借入 + 政府保証債 + 自己資金 ^[likely]

### 2008 政策金融再編での単独存続決定 ^[extracted]

2008-10-01 本土の 4 政策金融機関(国民生活金融公庫 + 中小企業金融公庫 + 農林漁業金融公庫 + JBIC 国内部門)が JFC に統合された際、ODFC についても統合の議論はあったが、以下の理由で**単独存続**が決定：

- **沖縄県固有の歴史的経緯**(本土復帰特別措置の継承)
- **沖縄振興特別措置法に基づく特別金利体系**の維持必要性
- **島嶼地域・離島対応**の独自ノウハウ
- **米軍基地・基地跡地再開発**という他県にない政策課題

→ 政策金融改革の中で例外措置として ODFC は単独で残された

### 直近政策論点

- 2024〜 コロナ特別貸付の**返済期到来**に伴う県内観光業者・中小企業のデフォルト懸念
- 2025〜 観光業 V 字回復に伴う設備投資資金需要の拡大対応
- 2025〜 米軍基地跡地再開発(西普天間・キャンプキンザー 等)の長期プロジェクト融資
- 2031 第 6 次沖縄振興計画終了後の機構・特別措置法の継続可否議論

## Counterpoints

- 「**本土 JFC との二重構造の非効率**」: 同じ国民・中小企業・農林水産機能を本土 JFC と ODFC で別組織運営することの組織コスト・人材分散批判 ^[ambiguous]
- 「**沖縄県内 3 地銀との競合・圧迫**」: 沖振法特別金利で ODFC が低利融資すると、民間 3 行(沖縄銀行・[[ryukyu-bank]]・沖縄海邦銀行)のプロパー融資機会が圧迫されるとの批判 ^[inferred]
- 「**観光業集中リスク**」: 県内産業構造が観光業に過度依存しており、ODFC 融資ポートフォリオも観光業エクスポージャー過大 → 次の感染症・地政学リスクで再度の壊滅シナリオ ^[inferred]
- 「**米軍基地跡地再開発の長期リスク**」: 返還スケジュールの不確実性 + 跡地利用計画の政治化 → プロジェクト融資の回収不安 ^[ambiguous]
- 「**沖振法 5 年ごと改正の政治依存**」: ODFC の存続基盤が沖縄振興特別措置法に依拠しており、本土政権の沖縄政策スタンス変化で組織存続が揺らぐ構造リスク ^[inferred]

## Open questions

- 第 6 次沖縄振興計画(2022-2031)終了後、ODFC は単独存続を維持するか、JFC への統合議論が再燃するか?
- コロナ特別貸付の県内最終的な代位弁済・損失額は本土 JFC と比較してどう推移するか? (観光業集中リスク)
- JUNGLIA テーマパーク開業(2025〜)に伴う県内観光業の融資需要拡大は ODFC ポートフォリオにどう影響するか?
- 米軍基地跡地再開発(西普天間・キャンプキンザー 等)のプロジェクト融資は ODFC vs [[dbj]] のどちらが主担当となるか?
- 離島・島嶼地域(宮古・八重山)の人口減少・産業空洞化に対する ODFC の独自支援策は?
