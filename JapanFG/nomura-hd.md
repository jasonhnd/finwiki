# 野村ホールディングス (Nomura HD)

## TL;DR

日本最大の独立系証券グループ（純営業収益ベース）。1925-12-25 大阪野村銀行証券部から独立した「野村証券」を起源とし、2001-10-01 持株会社化で発足。**国内リテール（営業店約 130）+ ホールセール（IB / Markets）+ 投資マネジメント（野村 AM、国内 No.2 クラス）+ Instinet（米電子取引）4 軸**。2008-10 リーマン・ブラザーズのアジア / 欧州 / 中東部門を約 2 億ドルで買収しグローバル化を加速、メガバンク系証券（[[mufg]] MUMSS、[[smfg]] SMBC 日興、[[mizuho-fg]] みずほ証券）と異なる「独立系」ポジションを維持。^[extracted]

### 主要子会社・持分

```
野村 HD（持株会社・上場 8604）
  ├── 野村証券（100%）── 国内リテール + 国内ホールセール中核
  │     └── 全国営業店 約 130 / 顧客資産残高 約 130 兆円規模
  ├── 野村アセットマネジメント（100%）── 国内公募投信 No.2 クラス・ETF 強み
  ├── Nomura International plc（100%、ロンドン）── 欧州 IB/Markets 中核
  ├── Nomura Securities International, Inc.（100%、ニューヨーク）── 米国機関投資家向け
  ├── Instinet, LLC（100%、2007 買収 → 2019 完全統合）── 米電子取引・アルゴ
  ├── Nomura Asia Holding N.V. / Nomura Singapore Ltd. 他 ── アジア拠点
  ├── 野村信託銀行（100%）── ラップ口座・年金信託
  ├── 野村バブコック&ブラウン（100%）── 不動産・インフラファンド管理
  ├── 野村フィデューシャリー・リサーチ&コンサルティング ── 年金コンサル
  ├── 野村ファシリティーズ（100%）── 不動産管理
  └── ジョインベスト証券（旧）→ 野村証券に統合済
```

> リテール: 野村証券の **営業店約 130**（メガ系証券に対し最多級、3 大 IFA ネットワークと棲み分け）^[extracted]
> ホールセール: グローバル Markets（株式・債券・FX・デリバティブ）+ Investment Banking（M&A 助言・DCM/ECM）
> Investment Management: 野村 AM（国内 No.2 クラス）+ アジア・欧州投信

### 創業・前身

- 1872 年 野村徳七（初代）大阪で両替商「野村商店」創業
- 1904 野村徳七（二代目）家業継承
- 1918 大阪野村銀行 設立（現 [[JapanFG/resona-hd|rsh]] りそな HD の遠祖の一系列）
- **1925-12-25** 大阪野村銀行 **証券部** が独立 → **野村証券** 設立（二代目野村徳七）^[extracted]
- 1946 戦後再建、財閥解体対象外（証券業のため）
- 1961 東京証券取引所 上場（旧野村証券）
- 2001-10-01 株式移転で野村 HD 設立、野村証券完全子会社化

## 2. 业务线 map

> **2025-04-01 部門再編**: 従来の 4 部門（営業 / ホールセール / Investment Management / その他）に **バンキング部門** を新設、**5 部門制** に移行 ^[extracted]

| セグメント | 主要事業者 | 特徴 |
|---|---|---|
| 国内リテール（営業部門）| 野村証券 国内 | 営業店約 130、対面 + オンライン併用、富裕層・退職世代厚い |
| ホールセール - Global Markets | 野村証券 / Nomura International plc / Nomura Securities International | 株式・債券・FX・デリバ、東京 / ロンドン / NY 3 拠点運営 |
| ホールセール - Investment Banking | 同上 | M&A 助言・DCM/ECM、邦銀系 IB との直接競合 |
| **バンキング部門（2025-04 新設）** | 野村信託銀行 + 関連子会社 | **法人 / 富裕層向け銀行業務集約、Group 横断の資金 / 与信機能を統合** ^[extracted] |
| 電子取引 | Instinet, LLC | 米株アルゴ・ダークプール、機関投資家向け |
| Investment Management | 野村アセットマネジメント | 公募投信 No.2 クラス、ETF（NEXT FUNDS）強み、**Macquarie 米欧資産運用事業買収（2025-04 契約、18 億 USD）でグローバル運用残高拡大** ^[extracted] |
| 信託・年金 | 野村信託銀行 / 野村フィデューシャリー | ラップ口座・年金コンサル |

## 3. 战略与竞争

### ホールセール再生

- **「One Nomura」戦略**（2013-04〜）: 部門縦割りを廃し、グローバル統合運営。リーマン買収後の文化摩擦への対処
- **欧米 Equity 縮小 → 債券 / FX / Macro 集中**（2016〜）: コスト削減 10 億ドル規模、欧州人員大幅削減
- **Archegos 損失後の再構築**（2021〜）: プライムブローカレッジ縮小、リスク管理 governance 強化
- **アジア集中**: シンガポール / 香港 / インドが成長軸。リーマン買収で承継したアジア網が基盤

### リテール

- **営業店 約 130**: 邦証最大級。メガ系証券（MUMSS 等）と顧客資産で競合
- **新 NISA（2024-01〜）**: 個人マネー流入の追風、運用残高拡大
- **オンライン × 対面 ハイブリッド**: 野村ネット&コール、対面営業店併用
- **富裕層・退職世代**: 対面チャネル強み、若年層は [[sbi-hd]] 楽天証券に劣後 ^[inferred]

### Investment Management

- **野村 AM**: 公募投信 国内 No.2 クラス、ETF（NEXT FUNDS）国内首位級
- **★ Macquarie 米欧資産運用事業 買収**（2025-04-22 契約、**18 億 USD**）: 豪 Macquarie Asset Management の米国 + 欧州 公募 / プライベート資産運用事業を承継、野村 AM のグローバル運用残高を一気に拡大。日系 AM では稀有な「欧米同時参入」案件 ^[extracted]
- **Wealth Management**: 富裕層向け一任、SMA / ラップ口座を伸長
- **代替投資**: PE / インフラ / 不動産ファンドを Group 内マーチャント・バンキングで展開

### バンキング部門（2025-04 新設）

- **2025-04-01 5 部門制移行**: 従来 4 部門に **バンキング部門** を新設、Group 横断の銀行業務（信託 / 法人融資 / 富裕層向け資金供与）を集約 ^[extracted]
- 銀行系証券（MUMSS / SMBC 日興 / みずほ証券）との競合上、**「証券 × 銀行」両輪化** を独立系として独自確立する戦略 ^[inferred]
- 野村信託銀行 + 関連子会社が中核。MUFG / SMFG 銀行のような商銀機能ではなく、**プライベート・バンキング寄り** ^[inferred]

### グローバル 3 拠点モデル

- **東京（本社）**: 日本最大証券、リテール + ホールセール本部
- **ロンドン**: Nomura International plc、欧州 / 中東 IB / Markets ハブ
- **ニューヨーク**: Nomura Securities International、米国機関投資家、Instinet 電子取引
- **アジア**: シンガポール / 香港 / インド / インドネシア、リーマン承継網ベース

## 4. 监管与政策

- **主管**: 金融庁（FSA、証券業）+ 日本証券業協会（自主規制）
- **海外規制**: 米 SEC / FINRA、英 FCA / PRA、香港 SFC、シンガポール MAS 等
- **G-SIB 指定**: なし（メガバンクと異なり持株会社レベルでは未指定）^[inferred]
- **直近政策論点**:
  - 2024〜 新 NISA 拡大（投資枠倍増）→ 個人マネー流入
  - 2025〜 ステーブルコイン / トークン化証券規制
  - 2025〜 顧客本位 業務運営（FSA「顧客本位の業務運営に関する原則」改訂）

## Counterpoints

- **独立系の宿命**: 銀行系証券（MUMSS / SMBC 日興 / みずほ証券）は法人顧客プールを持つ。野村は純粋に証券業として競合 ^[inferred]
- **欧米 IB 撤退論**: 2016 米欧 Equity 縮小・2021 Archegos 損失を踏まえ、「リーマン買収は失敗」評価も。一方で債券 / FX / Macro / アジアでは黒字定着 ^[ambiguous]
- **若年層デジタル弱**: [[sbi-hd]] / 楽天証券 / マネックスの台頭、リテールの主戦場が変質。野村の対面 + ネット&コール ハイブリッドは中間世代向け ^[inferred]
- **1997 不祥事 / 2012 インサイダー / 2021 Archegos**: 約 10 年周期の大型ガバナンス事案、構造的体質論の指摘 ^[ambiguous]
- **NRI 持分** ^[ambiguous]: 持株比率は変動（資本提携継続中、独立上場グループ）
