# あおぞら銀行 (Aozora Bank)


## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[JapanFG/sbi-shinsei-bank|SBI 新生銀行]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

旧 **日本債券信用銀行（日債銀, NCB）** 系譜の中堅独立銀行。**1998-12 経営破綻 → 特別公的管理（一時国有化）** を経験した邦銀再生史の代表例。2000-09 ソフトバンク + オリックス + 東京海上連合に売却 → 2001-01「あおぞら銀行」改称 → 2003 米 PE **サーベラス（Cerberus Capital Management）** 主要株主化 → 2006-11 東証一部再上場 → サーベラス段階的売却完了。直近最大論点は **2023-2024 期 米国 CRE（商業用不動産）融資巨額損失** → 2024-05 通期赤字発表 → 業績下方修正・経営戦略再構築フェーズ。メガ系列にも地銀系列にも属さない「中堅独立銀行」モデル ^[extracted]。

**正式名**：株式会社あおぞら銀行 ^[extracted]
**英名**：Aozora Bank, Ltd.
**証券コード**：東証 PRIME 8304 ^[extracted]
**設立**：1957-04（日本不動産銀行として設立）^[extracted]
**現法人名移行**：2001-01-04（旧 日本債券信用銀行 → あおぞら銀行 商号変更）^[extracted]

**業態**：旧長期信用銀行 → 普通銀行転換組（2006-04 普通銀行転換）^[inferred]

## 3. 战略与竞争

### 「中堅独立銀行」モデル

- **メガ系列にも地銀系列にも属さない独立ポジション** ^[extracted]
- 規模（総資産）はメガの 1/10 以下 ^[inferred]、ニッチ法人融資 + 不動産 + 国際で差別化
- 個人リテールは BANK ネット支店中心、店舗網は最小限
- 旧 サーベラス時代に「日本人向け retail よりも資本効率重視」の収益構造が定着 ^[inferred]

### 米国 CRE 損失問題（直近最大論点）

- **2018〜2023 米国オフィス CRE 向け融資拡大**（国内低金利環境下での利回り追求）^[extracted]
- 2022〜2024 米国オフィス需要構造変化（リモート定着）+ Fed 利上げ → 担保価値急落
- **2024-05 通期赤字 約 320 億円発表 → 過去最大級の損失計上** ^[extracted]
- 経営戦略再構築：米 CRE エクスポージャー圧縮、引当強化、国内シフト ^[inferred]

### 過去の統合検討歴

- **2010 旧 新生銀行（現 SBI 新生）との統合検討 → 撤回** ^[extracted]
  - 両行とも公的資金注入歴 + 海外 PE 出口問題を共有 → シナジー期待されたが破談
  - 旧 新生銀行は後に [[sbi-hd]] が TOB（2021-12〜2022-02）→ 完全子会社化（2023）

### 戦略的位置付け

- 「メガバンクのスポンサー化（将来的再編可能性）」は ^[ambiguous]
- 旧 サーベラス出口後の安定株主構成は機関投資家分散型 ^[inferred]
- [[sbi-hd]] の SBI 新生銀行モデル（PE 出口後の戦略パートナー化）との対比軸

## 4. 监管与政策

- **主管**：金融庁（FSA）
- **業態**：普通銀行（2006-04 長期信用銀行から転換）^[inferred]
- **業態の歴史**：旧長期信用銀行 3 行（日本興業銀行 → みずほ系 [[mizuho-fg]] / 日本長期信用銀行 → 旧新生銀行 → [[sbi-hd]] / 日本債券信用銀行 → あおぞら）の唯一の独立存続行
- **預金保険機構**：1998-12 破綻時の特別公的管理執行主体
- **公的資金注入歴**：1998-1999 預金保険機構経由で資本注入 → 後返済完了 ^[inferred]
- **直近政策論点**：
  - 米 CRE 損失に関する金融庁モニタリング ^[inferred]
  - 日銀政策金利正常化（2024〜）→ 国内金利マージン改善ボーナス
  - サーベラス出口後の中堅独立銀行ガバナンス枠組み

## Counterpoints

- 「米 CRE 損失は構造的か一時的か」: 2023-2024 損失は米国オフィス市況連動だが、利回り追求戦略自体が独立中堅行の構造的弱みを露呈した可能性 ^[ambiguous]
- 「旧 サーベラス出口後の安定株主構成」: 機関投資家分散型と推測されるが、戦略的安定株主の不在は将来再編リスクの裏返し ^[inferred]
- 「メガバンク・SBI 等との将来的統合可能性」: 旧 新生銀行（→ SBI 新生）統合撤回（2010）の歴史あり、規模効率の観点で再燃可能性は ^[ambiguous]
- 「中堅独立銀行モデルの持続可能性」: 国内低金利長期化 + 米 CRE 損失の二重打撃で「独立貫徹 vs 系列入り」分岐点 ^[ambiguous]
