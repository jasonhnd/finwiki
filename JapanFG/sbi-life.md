# SBI生命保険


## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[JapanFG/dai-ichi-frontier-life|第一フロンティア生命保険]] for peer / contrast context and [[insurance/INDEX|insurance index]] for the broader system / regulatory boundary.

## TL;DR

SBI生命保険株式会社は、金融庁の生命保険会社免許一覧に掲載される [[sbi-insurance-group]] / [[sbi-hd]] 系の life-insurance operating company。SBI group の銀行・証券・資産運用・保険をつなぐ retail financial ecosystem の中で、死亡保障・医療保障などの保険 underwriting boundary を担う。

## 2. Operating model

SBI生命は、SBI group のオンライン金融導線と接続しやすい life insurer として位置付けられる。銀行、証券、FX、暗号資産、資産運用まで広がる [[sbi-hd]] の customer base に対して、保険商品の direct / partner distribution を組み合わせられる点が比較軸になる。

FinWiki では、[[lifenet]] の純ネット life-insurance model、[[rakuten-life]] の ecosystem insurer、[[sony-life]] の advisory / planner-led model と比較するページとして扱う。
