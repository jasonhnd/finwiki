# NTT ドコモ・フィナンシャルグループ (NDFG)

## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[JapanFG/rakuten-insurance-hd|楽天インシュアランスホールディングス]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

NTT ドコモが **2026-07-01 事業開始**（吸収分割により金融領域の事業・関連株式を承継）する **金融事業統括中間持株会社「株式会社 NTT ドコモ・フィナンシャルグループ」（ドコモ FG）**。証券（マネックスグループ）+ 銀行（住信 SBI ネット銀行 / docomo Neobank ブランド）+ カード・決済（dカード / d 払い / dスマートバンク）を 1 枚屋根下に再編。位置付けは [[JapanFG/au-fh|au-fg]] / [[paypay-fg]] / [[rakuten-fg]] と並ぶ **「通信系 FG」第 4 極**。親会社は NTT ドコモ（NTT 100%）。^[extracted]

## 1. 会社概要

**正式名**：株式会社 NTT ドコモ・フィナンシャルグループ（略称 ドコモ FG）^[extracted]
**英名**：NTT DOCOMO Financial Holdings（公式英名は未公表、慣用表記）
**事業開始**：**2026-07-01**（吸収分割の効力発生日、公式発表ベース）^[extracted]
**親会社**：株式会社 NTT ドコモ（日本電信電話株式会社 100%）^[extracted]
**本社**：東京都千代田区永田町 2-11-1 山王パークタワー ^[extracted]

### 設立スキーム（公開資料ベース）^[extracted]

NTT ドコモが既存の金融関連子会社・出資持分を中間持株会社 NDFG に集約する分社/移管スキームを公表。NDFG は NTT ドコモの 100% 子会社として位置付けられ、上場予定は公表されていない。グループ全体の最上位親会社は引き続き日本電信電話（NTT、東証 PRIME 9432）。

### 親会社チェーン

```
日本電信電話（NTT、東証 PRIME 9432）
```

### 通信系 FG「第 4 極」の位置付け

- **既存 3 極**: [[JapanFG/au-fh|au-fg]]（KDDI / auじぶん銀行売却後の再編期）、[[paypay-fg]]（SoftBank / Z Holdings 系列、PayPay 銀行 + PayPay 証券 + PayPay カード）、[[rakuten-fg]]（楽天銀行 / 楽天証券 / 楽天カード / 楽天ペイ）

### 金融サービス統合の狙い（公開発表趣旨）^[extracted]

1. 証券・銀行・カード・ポイントの**運営主体を分散管理から統括管理に変更**し、グループ KPI と ROE 管理を明確化
2. NTT ドコモ本体（通信）と金融事業の**経営責任を分離**（規制対応・ガバナンス）
3. dポイント経済圏 + 金融プロダクトの**クロスセル設計**を中間 HD レベルで一元化

### 提携・買収戦略

- **マネックスグループ**: 資本業務提携契約（2023-10-04）→ 2024-01-04 連結子会社化完了（ドコモマネックス HD 経由、ドコモ議決権約 49%）。証券セグメントの中核 ^[extracted]
- **住信 SBI ネット銀行**: docomo Neobank ブランド提供から発展し、2025-05-30 開始の TOB（2025-09 成立）で **2025-10-01 付 NTT ドコモ連結子会社化**。資本再編後 ドコモ 55.37%：三井住友信託 44.63%（議決権 50%／50% 共同経営）、2026-08-03 商号「ドコモ SMTB ネット銀行」へ ^[extracted]
- **Coincheck Group**: マネックス傘下の暗号資産事業として継承 ^[extracted]

### B2C ブランディング

- **dブランド統合**: dカード・dスマートバンク・dポイント・d 払い の「d」プレフィックス共通化を継続 ^[extracted]

## 4. 規制・政策

- **主管**: 金融庁（FSA）
  - 第一種金融商品取引業（マネックス証券）
  - 銀行業（住信 SBI ネット銀行）
  - 包括信用購入あっせん業（dカード）
  - 資金移動業 / 前払式支払手段（d 払い・dポイント）
  - 暗号資産交換業（Coincheck）
  - 銀行業（ドコモ連結子会社化済みの住信 SBI ネット銀行 → ドコモ SMTB ネット銀行）
