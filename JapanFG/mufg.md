# 三菱 UFJ FG (MUFG)


## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[JapanFG/smfg|三井住友 FG (SMFG / SMBC グループ)]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

日本最大金融集团（総資産 **431.7 兆円**、純利益 **2 兆 4300 億円** / **2026-03 期 (FY2026) 実績・邦銀史上最高益**、2026-05-15 公表）。2005-10-01 三菱東京 FG + UFJ HD 合併発足。**商業銀行 + 信託 + 証券（Morgan Stanley 戦略提携）+ 海外（アジア中心、Krungsri 等）4 軸**。G-SIB Bucket 1。グループスローガン「世界が進むチカラになる」。略称由来: UFJ = United Financial of Japan（社名に「フィナンシャル」が 2 つ含まれる）。^[extracted]

### 合併歴史・前身

- 旧 **三菱銀行**：1880 設立（岩崎弥太郎 三菱財閥創設者）
- 旧 **東京銀行**：1946〜（旧 横浜正金銀行ベース）外国為替専門銀行
- **東京三菱銀行**：1996-04-01 三菱銀行 + 東京銀行 合併
- 旧 **三和銀行**：1933 鴻池・山口・三十四 3 行合併（前身の両替商 鴻池は 1656 創業）
- 旧 **東海銀行**：1941 名古屋・愛知・伊藤 3 行合併
- **UFJ ホールディングス**：2002-01 三和銀行 + 東海銀行 + 東洋信託銀行
- **MUFG**：2005-10-01 三菱東京 FG + UFJ HD 合併

> 統合背景: UFJ 銀行の不良債權膨張 → 当初 UFJ 信託 を 住信 に売却予定（2004-06）→ 2004-07 三菱東京 FG との経営統合に方針転換。住信は東京地裁に統合差止仮処分申請。2006-08-11 東京高裁が住信申請を斥下、2006-11-20 住信に和解金 25 億円で和解成立。^[extracted]

## 3. 战略与竞争

### 数字化转型

- **GO（グローバル・オープン）戦略**: API 開放・パートナー連携
- **Insight as a Service**: 法人客向けデータ・AI 分析
- **Japan Digital Design**（2017-10-02 設立）: フィンテック子会社
- **Global Open Network Japan**（2019-04 Akamai と JV → 2022-02 廃業）: ブロックチェーン決済挫折歴
- **MUFG × Google 戦略提携**（2026-05-07）★: 個人向け金融サービス AI エージェント、2026 年度中実用化目標 ^[extracted]
- **Progmat Coin**: ステーブルコイン基盤（MUFG 主導、[[smfg]] 等加盟）

### 合纵连横

- **Morgan Stanley 戦略提携**（2008-09-22〜）: 当初 90 億ドル（約 9000 億円）出資、発行済株 21% 取得 → 後 ~20% 維持。邦銀最深いグローバル IB タイアップ。2010-05 日本証券事業統合
- **Google 戦略提携**（2026-05-07〜）: AI エージェント領域
- **Krungsri TOB**（2013-12）: タイ最大銀行、約 5360 億円
- **Bank Danamon 連結子会社化**（2019-04）: インドネシア、約 3970 億円段階的買増し
- **Union Bank 売却 → U.S. Bancorp 株切替**（2022-12）: 米 retail 撤退・IB 集中シフト
- **KDDI / auじぶん銀行 出資解消**（2025-01-31）: auじぶん全株を KDDI へ、対価で 三菱 UFJ eスマート証券（旧カブドットコム証券）100% 取得

- **モビット解消**（2013-05〜2014）: 三菱 UFJ + SMFG 合弁解消、SMBC 側に売却
