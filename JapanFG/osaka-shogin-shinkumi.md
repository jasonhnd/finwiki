# 大阪商銀信用組合 (Osaka Shogin Credit Cooperative)

## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]] as the **first standalone shinkumi (credit cooperative) operating-company anchor** in FinWiki, closing the long-flagged 143-shinkumi audit gap. Read it against [[JapanFG/osaka-shinkin|大阪信用金庫]] for the shinkin-vs-shinkumi distinction in the same Osaka market, [[JapanFG/zenshin-kumi|全信組連 / Zenshinkumiren]] for the cooperative system parent, and [[banking/cooperative-banking-japan|cooperative banking domain]] for the broader regulatory boundary.

## TL;DR

大阪商銀信用組合 (商銀信組大阪) は、**在日韓国・朝鮮人の中小事業者を母体とする「商銀系」信用組合の関西代表行**。中小企業等協同組合法を根拠とし、業域ベース (商工業者) + 民族ベース (在日韓国・朝鮮人系) の **二重組合員資格**で営業する。**2002 年の関西商銀信組危機 (商銀信組の連鎖破綻問題) 後の再編で生まれた現行体制**で、関西の在日商工人 SME 融資・預金・送金・在日コミュニティ向け生活金融を主軸とする。[[JapanFG/zenshin-kumi|全国信用協同組合連合会 (全信組連)]] の会員として系統金融に組み込まれ、預金保険機構 ([[JapanFG/dic|DIC]]) の対象機関でもある。**信用組合 = 信用金庫より小規模 + 組合員資格がより厳格**という業態特徴を、地域・民族・業域の三層で具現する代表的事業者。^[extracted]

### 商銀系信用組合の歴史的経緯 ^[extracted]

| 年代 | 事象 |
|---|---|
| 1950〜60 年代 | 在日韓国・朝鮮人商工人向けの相互扶助金融として **「商銀」**系信用組合が各地に設立 |
| 1970〜80 年代 | 大阪・東京・神奈川・愛知ほか主要都市で商銀信組ネットワーク形成 |
| 1990 年代 | バブル崩壊で **不動産・建設業向け融資**の不良債権化が顕在化 |
| **2002〜2003** | **関西商銀信組連鎖破綻問題** — 関西地区の商銀信組数行が経営破綻、預金保険機構による資金援助 + 受け皿信組への事業譲渡 ^[extracted] |
| 2003〜 | 関西地区での **商銀信組の再編・統合**進行 — 現行体制への移行 |
| 2010 年代 | 在日コミュニティの世代交代 + 顧客層変化への対応 |
| 2020 年代 | 系統内 DX + AML/CFT 対応強化 (跨境送金監視等) |

### 主要業務 ^[extracted]

| 業務 | 内容 |
|---|---|
| **中小企業向け融資** | 在日商工人 SME の運転資金・設備資金 |

| **住宅ローン** | 組合員向け住宅取得資金 |

### 規模感 (推定) ^[ambiguous]

- **総資産**: 数千億〜1 兆円規模 (商銀系最大手の関西代表として)
- **預金残高**: 数千億円規模
- **店舗網**: 大阪・京都・兵庫・奈良ほか関西広域に数十店舗
- **常勤職員**: 数百人

> 注: 正確な KPI は組合員向け事業報告書 + 金融庁信組免許一覧を要参照 ^[ambiguous]

### 全信組連内の位置付け

- **[[JapanFG/zenshin-kumi|全信組連]]** の主要会員行の一つ
- 関西地区における **商銀系信組ネットワーク**の核
- 系統 ATM・送金網への参加 + 全信組連経由のリスク分散運用

### 2002〜2003 関西商銀信組危機の教訓 ^[extracted]

- 関西地区で商銀信組数行が **不良債権・経営難**で破綻
- **預金保険機構による資金援助 + 受け皿信組への事業譲渡**で組合員預金保護
- **大阪商銀信用組合**は危機を経た **再編後の存続・統合後体制**として現在に至る
- この経験が **商銀系 = 民族系信組** のリスク管理・ガバナンス強化の出発点

## 5. 監督・規制

- **主管**: 金融庁・近畿財務局
- **法的根拠**: 中小企業等協同組合法 (信用協同組合条項) + 協同組合による金融事業に関する法律 (協金法)
- **預金保険**: [[JapanFG/dic|預金保険機構]] 一般金融機関枠 (信組も対象、限度額 1,000 万円 + 利息)
- **直近政策論点**:
  - **信組業界の更なる再編・統合**
  - **AML/CFT 強化** (跨境送金監視)
  - **デジタル化対応** (組合員向けスマホアプリ等)
  - **食料安全保障 + 中小企業セーフティネット**フレームでの信組存在意義の再評価
