# JF 共水連 (全国共済水産業協同組合連合会 / Kyosuiren)

## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]] as the **JF Group cooperative-insurance national federation**, the JF-side parallel to [[JapanFG/ja-kyosairen|JA 共済連]] in the agricultural cooperative system. Closes the audit gap where JA 共済連 was deeply documented but the JF-side equivalent (JF 共水連) had no anchor. Read with [[JapanFG/jf-zengyoren|JF 全漁連 (JF グループ代表 + 経済事業)]] for the JF group representative federation, [[JapanFG/jf-shingyoren|JF 信漁連]] for the JF credit federation layer, [[JapanFG/jf-marine-bank-hokkaido|JF Marine Bank 北海道]] for prefectural credit anchor, [[JapanFG/norinchukin|農林中央金庫]] for the JF Marine Bank system central institution, and [[insurance/INDEX|insurance domain]] for the cooperative-vs-stock insurance form contrast.

## TL;DR

**JF 共水連 (全国共済水産業協同組合連合会) = JF (漁業協同組合) グループの共済事業中核**。水産業協同組合法を根拠とする **全国連合会**、JA グループでの [[JapanFG/ja-kyosairen|JA 共済連]] に相当する漁協系共済事業者。**会員 = 全国 JF (漁業協同組合) + 都道府県漁連**で、組合員 (漁業者・地域住民) 向けの **生命共済・年金共済・建物更生共済・自動車共済・漁船・漁業設備共済**等を提供。**JA 共済連 (200 兆円規模) に比べ規模は数 %〜10 % 程度** (漁業生産額が農業比小) だが、**漁業者向け固有共済 (漁船共済・漁業者災害共済等)** で独自性を持つ。JF グループの三本柱 ([[JapanFG/jf-zengyoren|JF 全漁連 = 代表 + 経済]] / **JF 共水連 = 共済 ★本ページ** / [[JapanFG/jf-shingyoren|JF 信漁連 = 信用]]) の中で **共済事業を専担**する全国連合会。^[extracted]

### 漁船共済 = JF 共水連の固有強み ^[inferred]

- **漁船共済**: 漁船 (沿岸〜沖合〜遠洋) の海難・火災・座礁等の被害補償
- 民間損保でも漁船保険は提供されるが、**漁業者組合員向け団体共済**としての規模感は JF 共水連が最大
- **漁業者災害共済**: 漁業中の人身事故・行方不明等の補償
- 漁業者の **「海上での職業リスク」**に特化した独自商品群

### 販売チャネル ^[inferred]

- **販売チャネル**: 全国 71 JF + 都道府県漁連 + 共水連直営
- 漁業者組合員世帯への **対面・訪問販売** が中心
- **取扱高は生命系 + 損害系合計で数千億〜数兆円規模** ^[ambiguous]

### ALM・資金フロー ^[inferred]

```
組合員 (漁業者・地域住民)
  → 共済掛金 → 全国 71 JF + 都道府県漁連
                  → JF 共水連
                      → 機関投資 (国債・地方債・外債)
                      → 共済給付 → 組合員へ
```

### JA 共済連との規模対比

- **JA 共済連 = 200 兆円規模 vs JF 共水連 = 数千億〜数兆円規模** = 漁業 vs 農業の経済規模差を反映
- ただし **漁船共済等の固有商品**で民間損保との差別化を維持

### 漁業者組合員特性

- **海上での職業リスク** (海難・行方不明・天候) → 共済需要の特殊性
- **季節集中型収入** (漁期に応じた所得変動) → 共済掛金支払いのタイミング工夫
- **高齢化と後継者問題** → 共済加入者の世代変化対応

### 共栄火災海上保険との関係 ^[inferred]

- [[JapanFG/ja-kyosairen|JA 共済連]] が [[JapanFG/kyoritsu-fire-marine|共栄火災海上保険]] の 74.20% 株主

### デジタル戦略 ^[inferred]

- **共済アプリ**: 漁業者組合員向け契約照会・請求 (簡素版)
- **AI 査定**: 自動車事故・建物被害の査定支援 (JA 共済連のシステム類似)
- **マイナンバー連携** + 確定申告対応

### 競合構造 ^[inferred]

- **生命保険業界**: 日本生命・第一生命・明治安田・住友生命 - 漁業者世帯のうち高所得層を巡る競合
- **損保業界 3 大 ([[JapanFG/tokio-marine|東京海上]] / [[JapanFG/msad|MS&AD]] / [[JapanFG/sompo|SOMPO]])**: 自動車・漁船 (大型) で部分競合
- **共栄火災**: JA 共済連子会社 → JF 共水連と一部協働
- **[[JapanFG/ja-kyosairen|JA 共済連]]**: 農協系の peer (規模圧倒的)、地域競合は限定
- **[[JapanFG/zenrosai|全労済]] / 都道府県民共済**: 部分競合

## 5. 監督・規制

- **主管**: **農林水産省** (主) + **金融庁** (協力監督) + **水産庁** (漁業政策)
- **法的根拠**: 水産業協同組合法 (共済事業条項) + 共済事業ガイドライン
- **直近政策論点**:
  - **共済 = 保険の規制ギャップ** 議論 (JA 共済連と同論点)
  - **経済価値ベース・ソルベンシー (ESR)** の共済への類似規制適用論
  - **漁船共済法制度** の運用と JF 共水連の役割
  - **食料安全保障計画 (2026〜)** での漁業共済の機能再定義
  - **2018 漁業法改正** + 漁業者構造変化への対応
