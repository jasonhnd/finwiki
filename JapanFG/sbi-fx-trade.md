# SBI FXトレード


## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[JapanFG/traders-securities|トレイダーズ証券]] for peer / contrast context and [[securities/INDEX|securities index]] for the broader system / regulatory boundary.

## TL;DR

SBI FXトレードは [[JapanFG/sbi-hd|SBI group]] の FX 専門 FIEA operator。FSA 金融商品取引業者一覧では「SBI FXトレード株式会社」、関東財務局長(金商)第2635号として掲載される。SBI証券とは別法人で、FX / 積立FX / option FX / crypto CFD adjacency を担う。

## 1. 法人・ライセンス境界

| Item | Reading |
|---|---|
| Legal name | SBI FXトレード株式会社 |
| FSA registration | 関東財務局長(金商)第2635号 |
| FIEA scope | 第一種金融商品取引業 |
| Parent boundary | SBIリクイディティ・マーケット 100% subsidiary |
| Core products | FX, 積立FX, option FX, crypto CFD-adjacent services |

## 2. Business role

- FX 専門会社として、少額・透明性・SBI グループ連携を打ち出す。
- [[JapanFG/sbi-securities|SBI証券]] の総合 broker function とは別に、FX execution / retail FX product を担う。
- [[exchanges/jp-exchange-sbi-vc-trade|SBI VC Trade]] と crypto / CFD / EPI 周辺で隣接する。

## 3. Why standalone page matters

SBI group は証券・銀行・暗号資産・FX が密接に絡む。SBI FXトレードを独立させることで、FX 専業会社、FIEA 第一種、crypto CFD adjacency、NEOBANK 連携を混同せず追える。
