# 京都フィナンシャルグループ

## TL;DR

京都府を地盤とする **関西の名門地銀 FG**。証券コード **東証 PRIME 5844**。**2023-10-02** に京都銀行の **単独株式移転** による持株会社化で発足（地銀の中では持株会社化が遅かった例）。**京都銀行は京都府内シェア No.1** を維持し、京都府本社の任天堂・京セラ・村田製作所・日本電産（現ニデック）等の **世界的優良企業** との取引・**政策保有株式** で著名。関西では [[resona-hd]]（関西みらい銀）・[[ikeda-senshu-hd]]・滋賀銀行・南都銀行・紀陽銀行 等と直接競合。メガ 3 行（[[mufg]] / [[smfg]] / [[mizuho-fg]]）に対しては京都地盤と上場企業株式持合いの **資金力** で独自の存在感。^[extracted]

### 合併歴史・前身

- 旧 **京都銀行**（京都府地盤）
  - **1941-10-01** 京都銀行 設立（戦時統制下の地銀統合。**京都中央信用組合 + 西陣銀行 + 京北銀行 + 京都銀行（旧）** 等の統合に由来）^[extracted]
  - 戦後、京都市中心部から京都府全域・関西広域へ営業基盤拡大
  - **1969** 大阪証券取引所・東京証券取引所 第二部 上場
  - **1971** 東証一部 上場（旧証券コード 8369）^[extracted]
  - 京都本社の世界的企業（任天堂・京セラ・村田製作所・日本電産/ニデック 等）と長期取引・**株式持合い** 関係構築 → 巨額の **政策保有株式 含み益** が同行の代名詞に
- **京都 FG 発足**: **2023-10-02**（京都銀行 単独株式移転、HD 化。地銀の中では持株会社化が遅かった例の 1 つ）^[extracted]

## 3. 戦略与競争

### 戦略軸

- **京都府シェア No.1 地銀**: 京都府内預金・貸出シェアで圧倒的 No.1。地元密着 + 京都ブランドの活用 ^[extracted]
- **政策保有株式 含み益 戦略** ★: 任天堂・京セラ・村田製作所・日本電産（ニデック）等、京都本社の世界企業との **長期株式持合い** で蓄積した **数千億円規模の含み益** が同行最大の財務特徴。資本充実度・信用力の源泉 ^[extracted]
- **関西広域展開**: 京都府を地盤としつつ、大阪・滋賀・兵庫・奈良・愛知の一部にも染み出し、関西広域での中堅地銀ポジション ^[extracted]
- **「京銀ダイレクトファイナンス」DX**: デジタル投資、法人向けインターネットバンキング、フィンテック提携 ^[extracted]
- **京都ブランド活用**: 観光・伝統産業・京都本社企業との繋がりを差別化資産として活用 ^[extracted]
- **2024 中計**: 持株会社化（2023-10）後の **銀証一体運営・地域企業ソリューション強化** が主軸 ^[ambiguous]

### 京都本社の世界企業との関係 ★

- **任天堂**（京都市南区）: 京都銀行は安定株主として長期保有、株価上昇局面で巨額含み益 ^[extracted]
- **京セラ**（京都市伏見区）: 同上、長期株式持合い ^[extracted]
- **村田製作所**（京都府長岡京市）: 同上、長期株式持合い ^[extracted]
- **日本電産 / ニデック**（京都市南区）: 同上、長期株式持合い ^[extracted]
- **オムロン**（京都市下京区）・**SCREEN HD**（京都市上京区）・**島津製作所**（京都市中京区）等の京都本社上場企業群とも取引・株式持合い関係 ^[inferred]
- これら **政策保有株式 含み益** が京都銀の **資本充実度** と **信用力** を下支え、地銀の中でも独自のバランスシート構造 ★ ^[extracted]

### 統合シナジー・課題

- **持株会社化シナジー**: 2023-10-02 発足からまだ間がなく、銀証一体運営・地域企業ソリューション強化の効果は中長期で発現 ^[inferred]
- **京都府人口動態**: 京都府は政令市の中では比較的安定だが、京都市以外の地方部は人口減少。京都市内の観光・大学需要 + 京都本社企業の雇用が下支え ^[inferred]
- **収益多角化**: 預貸金利息収入 + 株式配当 / 売却益の二本柱、株式売却益の年度変動が大きい点が課題 ^[extracted]
- **政策保有株式の縮減圧力**: 金融庁・東証の **政策保有株式 縮減要請** に対する対応が課題。「持合い」の合理性説明と縮減ペースの調整 ^[extracted]

### B2C ブランディング

- **京都銀行 = 「京銀」**: 京都府民での認知度は圧倒的、京阪神広域でも一定の認知 ^[extracted]
- **京都 FG**: 持株会社ブランドは個人客には浸透限定的、銀行ブランド一本化 ^[inferred]
- **「京都の銀行」** という ブランド資産: 観光・伝統産業・世界企業との繋がりが地銀ブランドを差別化 ^[extracted]

## 4. 監管与政策

- **主管**: 金融庁（FSA）
- **持株会社規制**: 銀行法 第 52 条の 17（銀行持株会社）^[extracted]
- **直近政策論点**:
  - **2024〜 日銀政策金利正常化**: 国内貸出金利マージン改善ボーナス（地銀メリット大、利上げで地銀収益改善期待）
  - **政策保有株式 縮減要請**: 金融庁・東証のスチュワードシップ・コード / コーポレートガバナンス・コード遵守、政策保有株式の縮減ペースが市場の関心 ★ ^[extracted]
  - **2024〜 地銀再編期待**: 金融庁の地銀再編スタンスは「統合自体は否定しないが当局主導はしない」→ 京都 FG は **単独路線維持** が現状 ^[ambiguous]
  - **2025〜 営業店舗削減**: 地銀標準のリストラ路線、店舗網のスリム化と機能集約
  - **中小企業向け事業性評価融資**: 担保・保証に依らない融資の推進
  - **ESG / 地域脱炭素 ファイナンス**: 京都発の脱炭素転換支援（京都議定書発祥地としてのブランド活用）^[inferred]
  - **大阪・関西万博（2025-04〜10）**: 関西経済圏の活性化期待 → 京都の観光・サービス・建設業への follow-tail 効果

## Counterpoints

- 「政策保有株式 含み益 強み論」 vs 縮減圧力: 任天堂・京セラ・村田・ニデック 等の含み益は資本充実度に寄与する反面、ガバナンス・コードの縮減要請との両立が課題 ★ ^[ambiguous]
- 「京都府シェア No.1」 vs 関西広域での中位置: 京都府内では圧倒的だが、関西広域では大手地銀の 1 つに過ぎず、メガとの中堅企業取引競合は厳しい ^[ambiguous]
- 「持株会社化遅延」観: 地銀の中で持株会社化が遅かった例（2023-10）→ 銀証一体・グループ多角化での後発スタート、効果発現には時間 ^[extracted]
- 「京都本社世界企業との関係」: 株式持合いによる安定株主機能と、本業（融資・銀行業務）での実利の乖離。配当 / 株価依存度が高い収益構造 ^[inferred]
- 「デジタル投資の規模劣後」: PayPay 銀行 / 楽天銀行 / 住信 SBI ネット銀行 等に対する個人デジタル領域での投資規模は限定的 ^[inferred]
- 「関西地銀再編シナリオ」観: 京都 FG + 滋賀銀 + [[ikeda-senshu-hd]] 等の組み合わせが時折議論されるが、京都 FG は単独路線維持 ^[inferred]
