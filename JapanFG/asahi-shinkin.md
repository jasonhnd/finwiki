# 朝日信用金庫

## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[JapanFG/jonan-shinkin|城南信用金庫]] for peer / contrast context inside the Tokyo shinkin block and [[banking/shinkin-bank-registry-japan|Japan shinkin bank registry]] for the broader system / regulatory boundary maintained by [[JapanFG/shinkin-central|信金中央金庫]].

## TL;DR

朝日信用金庫は、東京都台東区に本店を置く、東京都心部（台東・中央・千代田・墨田など）と埼玉県・千葉県の一部を主たる営業地盤とする大手都市型信用金庫。中小企業・問屋・商店・個人事業主向けの融資を中核とする協同組織金融機関。

## 2. Operating model

朝日信用金庫は、東京都心の問屋街（浅草橋・蔵前・日本橋など）の中小卸売業・小売業・個人事業主、ならびに台東・中央・千代田・墨田区の中小製造業・サービス業向け融資を中核業務とする。東京の shinkin landscape では [[JapanFG/jonan-shinkin|城南信用金庫]] と並ぶ大手都市型 shinkin の一つで、預金規模で全国信用金庫の上位に位置する。
