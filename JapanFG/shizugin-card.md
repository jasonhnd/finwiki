# しずぎんカード (Shizugin Card / Shizuoka Bank Card)

## TL;DR

株式会社しずぎんカード (Shizugin Card / Shizuoka Bank Card) は、[[shizuoka-fg|しずおか FG (東証 PRIME 5831)]] グループのクレジットカード子会社で、中核子会社 [[shizuoka-bank|静岡銀行]] (静岡県シェア No.1 / 製造業集積地 main bank) の個人 / 法人顧客基盤を背景に、**静岡県地盤の銀行発クレジットカード** (JCB / VISA / Master ブランド) ・ローン・キャッシング・収納代行・電子マネー・QR 決済関連サービスを提供。地銀系クレカ会社の典型 model で、銀行口座連動・地域加盟店ネットワーク・地域経済圏内決済の機能を担う。[[shizuoka-fg]] 2022-10-03 持株会社化後にグループ並列子会社の一角として再配置された。^[extracted]

### 持株ツリー (しずおか FG グループ簡略図)

```
[[shizuoka-fg|しずおか FG]] (持株会社・上場 5831、2022-10-03 設立)
  ├── [[shizuoka-bank|静岡銀行]] (100%) ── 中核子会社・静岡県シェア No.1・製造業集積地 main bank
  │     └── 自動車 (スズキ・ヤマハ発動機) / 楽器 (ヤマハ・ローランド) / 食品・お茶 main bank

  ├── しずぎんカード (100%、本ページ) ── グループクレジットカード
  │     ├── 静岡県地盤の銀行発クレカ (JCB / VISA / Master)
  │     ├── 静岡銀行口座連動カード
  │     └── 地域加盟店ネットワーク (静岡県内中小小売)

  ├── 静銀信用保証 (100%) ── 信用保証
```

### 収益構造の特徴 ^[inferred]

- 主要収益源は **加盟店手数料 (interchange) + 年会費 + キャッシング金利 + リボ手数料 + ショッピングクレジット手数料**
- [[shizuoka-bank|静岡銀行]] 個人 / 法人顧客 (預金口座連動) からのカード発行が成長 driver
- 静岡県内 製造業 (スズキ / ヤマハ発動機 / ヤマハ等) の従業員給与振込口座 → 銀行系カード自動発行 → 加盟店手数料収入の cross-sell flow ^[inferred]
- 業績は親 [[shizuoka-fg]] 連結セグメント情報経由で部分開示 ^[ambiguous]
- ネット決済 / コード決済普及で「銀行系カード」の存在感は相対的に低下 (vs [[rakuten-card]] / [[paypay-card]] / [[smbc-card]]) ^[inferred]

## 3. 親会社・グループ内ポジション

しずおか FG グループ内では、中核子会社 [[shizuoka-bank|静岡銀行]] の **静岡県シェア No.1 + 製造業集積地 main bank + 県内人口 360 万 (全国 10 位前後) の個人 / 法人顧客基盤** に対するクレジットカード / リテール決済窓口という位置づけ。**地域加盟店ネットワーク** (静岡県内中小小売・飲食・サービス業) の card acquiring 機能と組み合わせて、地域経済圏内決済の循環を担う。**しずおか FG 2022-10-03 持株会社化**後、しずぎんカードは持株会社直下並列子会社として再配置され、[[mebuki-fg]] / [[fukuoka-fg]] / [[iyogin-hd]] と同型の「持株会社下兄弟会社」model に移行 ^[inferred]。

### グループ内クロスリンク

| 静銀ティーエム証券 | 富裕層クレカ + 証券口座 cross-sell ^[inferred] |

### 地域連携 (静岡県経済圏)

しずぎんカードは静岡県の **地域加盟店ネットワーク** (中小小売・飲食・観光・ホテル) の card acquiring 機能を担い、静岡県地域経済圏内決済の循環を支える。これは [[mebuki-fg]] のめぶきカード (北関東) や [[fukuoka-fg]] の FFG カード (九州) と同型の「地域経済圏内決済 model」^[inferred]。

### 強み・弱み

- 強み: [[shizuoka-bank|静岡銀行]] 静岡県シェア No.1 + 預金顧客基盤 + 製造業従業員給与振込口座連動 + 地域加盟店ネットワーク + 製造業集積地 (スズキ / ヤマハ等) のオーナー / 役員富裕層
- 弱み: 単独規模では大手カードイシュアー ([[smbc-card]] / [[credit-saison]] / [[rakuten-card]]) に大幅劣後、QR 決済 / コード決済の急速普及で銀行系カードの存在感低下、デジタル投資余力限定的 ^[inferred]

## 5. 监管与会计

### 規制枠組み

- **主管**: 経済産業省 (割賦販売法所管) ・金融庁 (FSA, 貸金業法・資金決済法所管)
- **法的根拠**:
  - **割賦販売法 (割販法)**: クレジットカード発行・ショッピングクレジット業務の本体規制 (包括信用購入あっせん業者登録)
  - **貸金業法**: キャッシング・カードローン業務 (貸金業登録)
  - **資金決済法**: 電子マネー / プリペイドカード / QR 決済関連業務
  - **個人情報保護法 / マイナンバー法**: カード会員情報管理
- **業界団体**: [[j-credit|日本クレジット協会 (JCA)]] 加盟 ^[inferred]、JCB / VISA / Master ブランド加盟銀行ライセンス
- **親 FG ガバナンス**: [[shizuoka-fg]] 連結子会社として持株会社ガバナンス下、銀行業以外子会社規制 (主要業務 / 付随業務 / 業務範囲規制) で位置づけられる

### 規制論点

- **割販法改正 (2018-)**: 加盟店契約者 (acquirer) の義務強化、セキュリティ対策義務 (PCI DSS 準拠)
- **改正割販法 (2020-)**: 利用者保護強化 (与信審査・利用上限管理・苦情処理体制)
- **個人情報保護法 (2022-改正)**: クレジットカード会員情報の越境移転・適正取得義務
- **キャッシュレス推進**: 経産省キャッシュレスビジョン 2025・「2025 年キャッシュレス比率 40%」目標下で加盟店手数料水準論争
- **コード決済 (PayPay / 楽天ペイ / au PAY 等) 競争**: QR 決済の加盟店手数料低水準化圧力 → 銀行系カード加盟店手数料への圧力 ^[inferred]
- **マネロン対策・反社チェック**: [[shizuoka-fg]] グループ共同実施
- **EDINET / 有価証券報告書**: しずぎんカード単体は非上場で開示限定的、[[shizuoka-fg]] 連結セグメント情報で間接開示 ^[ambiguous]

### 会計上の取扱い

- **連結対象**: [[shizuoka-fg]] 100% 子会社として連結対象、銀行業セグメント以外の「その他」セグメントに集約されるのが典型 ^[inferred]
- **単体財務 KPI**: 親 [[shizuoka-fg]] 有価証券報告書注記 / セグメント情報で部分開示 ^[ambiguous]
- **カード債権**: 連結 BS 上「割賦売掛金」「貸付金 (カードローン)」勘定で表示 ^[inferred]
- **報告通貨**: 円建て、日本基準適用 ^[inferred]

## Counterpoints

- 「地銀系クレカ = 中途半端」論: 大手カードイシュアー ([[smbc-card]] / [[credit-saison]] / [[rakuten-card]]) にもコード決済 ([[paypay-card]] / au PAY) にも勝てない中間ポジション vs 地域加盟店ネットワーク・銀行口座連動の優位性 ^[ambiguous]
- 「銀行系カードの構造的衰退」論: QR コード決済 / 後払い ([[bnpl-landscape|BNPL]]) / モバイル決済の普及で銀行系プラスチックカードの相対的地位低下 ^[inferred]
- 「HD 化 = 形式だけ」論: [[shizuoka-fg]] 2022-10-03 持株会社化後、しずぎんカードの実質的な戦略変化は限定的との見方 vs 持株会社化により資本効率改善 / M&A 余地確保 ^[inferred]
- 「単独存続困難」論: クレカ業界寡占化進展、地銀系カード統合・大手カード会社 OEM 化シナリオ ^[inferred]
- 「FG 内利益相反」: [[shizuoka-bank|静岡銀行]] カードローン vs しずぎんカードキャッシングの使い分け基準 ^[inferred]
- 「静岡県集中 = 人口減少リスク」: 静岡県は若年層流出続く、リテール決済需要長期縮小懸念 ^[inferred]
