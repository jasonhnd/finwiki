# めぶきリース (Mebuki Lease)

## Wiki route

This entry sits under the [[JapanFG/INDEX|JapanFG index]] as the FG-subsidiary leasing operating-company anchor for [[mebuki-fg|めぶき FG]]. Read it against parent operating banks [[joyo-bank|常陽銀行]] / [[ashikaga-bank|足利銀行]] and sibling subsidiary [[mebuki-securities|めぶき証券]]. Peer comparison: [[hokkaido-lease|北海道リース]] (北洋・北海道銀行系) and major lessor [[mitsubishi-hc-capital|三菱 HC キャピタル]] / [[mizuho-lease|みずほリース]]. For broader system route see [[banking/regional-bank-consolidation-pattern|regional-bank consolidation pattern]] and [[banking/INDEX|banking index]].

## TL;DR

株式会社めぶきリースは、[[mebuki-fg|めぶき FG]] (東証 PRIME 7167) 100% 子会社の地銀系リース会社で、**1974-09 に [[joyo-bank|常陽銀行]] が設立した「常陽リース」を母体とし、2017-04-03 に めぶき FG が全株式を取得して「めぶきリース」へ商号変更**して発足した（公式沿革・法人登記で確認。両行リース会社の「対等合併」ではなく、常陽リースの単独改称）。茨城・栃木・首都圏の中堅企業 main bank 顧客向けに設備リース・ファイナンスリース・オペレーティングリース・ベンダーリース等を提供。地銀系リース上位の一角。^[extracted]

### 旧 常陽リース系譜（母体）

- **1974-09**: [[joyo-bank|常陽銀行]] がリース・ファイナンス部門の補完事業として「常陽リース株式会社」を設立（公式沿革で確認）
- [[joyo-bank|常陽銀行]] 系列リース会社として、茨城県内中堅企業向け設備リースを主業
- 戦後地銀系リースの典型: 銀行融資ではなく設備リース形態での中小企業設備投資支援
- **2017-04-03**: めぶき FG が全株式を取得し「めぶきリース株式会社」へ商号変更（法人登記で確認）

### 足利銀行系リース機能との関係 — 要再確認

- 旧版は「足利リース ([[ashikaga-bank|足利銀行]] 系) を対等統合」と記述していたが、めぶきリース公式沿革・法人登記には **足利リース という独立会社の合併記録も、足利信用保証のリース事業承継の記録も確認できなかった**（2026-05-30 時点）。「常陽リース + 足利リース の対等合併」という framing は誤り。栃木側（[[ashikaga-bank]] / 足利信用保証）のリース機能を めぶきリースへ承継・統合したか否かは一次資料で未確認のため **要再確認**。
- [[ashikaga-bank|足利銀行]] は 2003-11 一時国有化局面を経て 2016-10 めぶき FG に統合された経緯があり、栃木県内中堅企業向けリース機能の所在・移管経路の精査が必要。

### めぶき FG 設立とグループ統合

- **2016-10-01**: [[mebuki-fg|めぶき FG]] 設立 ([[joyo-bank]] + [[ashikaga-bank]] 共同株式移転)
- **2017-04-03**: めぶき FG が常陽リースの全株式を取得し「めぶきリース」へ改称 — 銀行子会社 2 行の顧客基盤に対するグループ共通リース機能として再編

### FG 内ポジション

- **銀リース連携の中核**: [[mebuki-fg]] 銀行 2 行の中堅企業 main bank 顧客のリース需要受託
- **設備投資支援**: 銀行融資補完 + 簿外処理ニーズへの対応
- **県内自治体・公的セクター連携**: 茨城県・栃木県の市町村向けリース案件

### 競合構図

- **大手リース**: [[mitsubishi-hc-capital|三菱 HC キャピタル]] / [[smfl|SMFL]] / [[mizuho-lease|みずほリース]] / [[fuyo-lease|芙蓉リース]] / [[tokyo-century|東京センチュリー]] / [[ricoh-leasing|リコーリース]] - 大型・全国・複雑案件で優位
- **地銀系リース**: [[hokkaido-lease|北海道リース]] / 千葉興業リース / 群馬リース 等 - 同型 model
- **メーカー系リース**: [[toyota-finance|トヨタファイナンス]] / [[nec-leasing|NEC キャピタルソリューション]] - 製品連動
- **県内競合**: [[tsukuba-bank|筑波銀行]] 系・[[tochigi-bank|栃木銀行]] 系リース

### 強みと弱み

- 強み: [[joyo-bank]] / [[ashikaga-bank]] 顧客基盤 (茨城・栃木の中堅企業 main bank) + 地域密着の現場対応

## 7. 規制・政策

- **主管**: 金融庁 (FSA) (リース業単独としては銀行業免許不要、貸金業登録は対応領域に応じて)
- **規制論点**:
  - リース会計基準変更 (IFRS16 / 日本基準) 対応支援
  - 中小企業向けリース税制活用 (中小企業投資促進税制)
  - 環境リース (再エネ設備リース) の脱炭素税制対応
  - 反社チェック・マネロン対策 ([[mebuki-fg]] グループ共同実施)
