# 株式会社八十二銀行 (The Hachijuni Bank)


## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[JapanFG/fukuoka-fg|ふくおか FG (FFG)]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

長野県地盤の旧単独銀行 page。1931-08-01 第十九銀行 + 第六十三銀行 合併で発足、社名は **19 + 63 = 82** に由来する。2023-06 長野銀行を完全子会社化し、2026 年の FSA bank-license list では current legal-name entity が [[hachijuni-nagano-bank|八十二長野銀行]] として掲載される。以後、本ページは旧八十二銀行の historical / predecessor analysis とし、current operating-company boundary は [[hachijuni-nagano-bank]] が持つ。

## 1. 公司结构

**旧正式名**：株式会社八十二銀行
**英名**：The Hachijuni Bank, Ltd.
**証券コード**：東証 PRIME 8359
**創業**：1931-08-01（第十九銀行 + 第六十三銀行 合併）
**本社**：長野県長野市中御所岡田 178-8（本店ビル）
**現行境界**：[[hachijuni-nagano-bank|株式会社八十二長野銀行]] に統合後の bank-license operating company が移行。

### 主要子会社・持分

```
八十二銀行（旧単独銀行・上場 8359）
  ├── 長野銀行（2023-06-01 完全子会社化）
  └── [[hachijuni-nagano-bank]] ── current operating-company boundary
  ├── 八十二証券 ── 県内リテール証券
  ├── 八十二リース ── グループ金融子会社
  ├── 八十二信用保証 ── 住宅ローン等の信用保証
  ├── 八十二キャピタル ── ベンチャー / M&A 関連投資
  ├── 八十二ビジネスサービス / 八十二システム開発 等 ── システム・事務子会社
  └── 海外駐在員事務所（上海・香港・バンコク・ニューヨーク 等）── 県内製造業の海外進出フォロー ^[inferred]
```

> 子会社構成の詳細・出資比率は本版未確定（要 八十二銀行 公式 IR / 有価証券報告書直接確認）。^[inferred]

### 合併歴史・前身

- 1877 **第六十三国立銀行**（明治期国立銀行条例下、第 63 号で設立）
- 1878 **第十九国立銀行**（同、第 19 号で設立）
- 戦前期、長野県内で第十九銀行と第六十三銀行が並立、共に県内有力地銀として発展
- **1931-08-01 八十二銀行 発足**：第十九銀行 + 第六十三銀行 合併、社名は両前身行の番号合計（**19 + 63 = 82**）に由来する命名 — 国立銀行ナンバー由来命名の代表例
- 戦後の地銀コアとして長期上場、1949 東証上場
- **2022-04 東証市場区分見直し** → 東証 PRIME 8359（旧東証一部から移行）
- **2023-06-01 長野銀行を株式交換で完全子会社化**：県内第二地銀の長野銀行（旧長野相互銀行 → 1989 普通銀行転換）を傘下化
- **2026 bank-license snapshot**：金融庁の地方銀行一覧では current legal name が [[hachijuni-nagano-bank|株式会社八十二長野銀行]] として確認できる。

> 統合背景：長野県内では従来「八十二銀行 vs 長野銀行 vs 信用金庫群」の 3 つ巴構造が長く続いていたが、人口減と地銀デジタル投資負担の中で、長野銀行側が独立維持を断念して八十二銀行傘下入りを選択。完全子会社化（2023-06）から current legal-name entity [[hachijuni-nagano-bank|八十二長野銀行]] への移行は、旧長野銀行従業員・顧客・地域経済への影響を緩和する 2 段階アプローチだったと推測される ^[inferred]。

## 3. 战略与竞争

### 県内地盤戦略

- **長野県 = 電子産業集積地**：長野県内には [[manufacturing/seiko-epson|セイコーエプソン]]（諏訪本社、プリンタ / ウォッチ / 半導体）、ミネベアミツミ（旧諏訪精工舎系譜の電子部品大手）等の有力中堅製造業が集積。これらの main bank ポジションが収益基盤
- **観光業融資**：軽井沢（高級リゾート）・白馬（スキー / 訪日インバウンド）・諏訪（温泉 / 諏訪湖）・上田 / 佐久（歴史観光）・善光寺周辺など、長野県は観光産業大県でもあり、観光ホテル / リゾート融資が地銀ポートフォリオの重要セグメント
- **県内シェア圧倒**だが、長野県は人口減・若年層流出が全国平均より顕著で中長期の構造課題。長野市・松本市・諏訪・上田・佐久・北信（飯山等）の県内 5〜6 圏域全てに営業ネットワークを展開

### 県内同業との関係：3 つ巴 → 1 強体制へ

- **従来構造**：八十二銀行 vs [[#長野銀行統合]] vs 信用金庫群（長野信用金庫・松本信用金庫等）の 3 つ巴
- **長野銀行統合**（2023-06 完全子会社化後、[[hachijuni-nagano-bank|八十二長野銀行]] の current legal-name boundary へ移行）で「旧八十二銀行 + 旧長野銀行」の 1 強体制が完成 → 旧第二地銀との対立解消
- **信用金庫群との棲み分け**：信用金庫は中小零細企業・個人預金中心で住み分けは比較的明確、八十二銀行は地銀として中堅企業 main bank ポジションを確立
- **県外大手（メガバンク・[[mufg]] / [[smfg]] / [[mizuho-fg]]）との競合**：県内有力企業の大型融資 / グローバル業務はメガに流出する構造、八十二銀行は地域密着型サービスで対抗

### 単独銀行の戦略意義

- **持株会社化していない**点が、しずおか FG（2022-10 単独 FG 化）・[[fukuoka-fg|FFG]] 等の単独地銀 FG 化トレンドと対照的
- 単独銀行のメリット：(1) ガバナンス簡素、(2) 銀行本体に経営資源集中、(3) 持株会社化の追加コスト不要
- 単独銀行の制約：(1) 銀行法 制約で新業態（証券・リース等）取得は持株会社経由が必要、(2) 異業種 M&A の自由度が低い、(3) デジタル子会社設立の柔軟性が劣る ^[inferred]
- 長野銀行統合後の持株会社化の有無は中長期の判断局面と推測される ^[inferred]

### 地銀再編における位置

- **長野県の単独地銀リーダーとして独立路線維持**（[[mufg]] 等メガとの提携・SBI 第 4 メガ構想からは距離）
- **県内 1 強体制完成**（長野銀行吸収後）→ 県内市場での競争圧力が低下する反面、独占禁止法・地域金融機関再編ガイドラインの観点で当局審査が論点となった経緯あり ^[inferred]
- 北信越地域（長野・新潟・富山・石川・福井）の地銀連携は [[hokuhoku-fg|ほくほく FG]]・[[dai-shi-hokuetsu-fg|第四北越 FG]] 等を中心とした他軸が形成されており、八十二銀行は長野県集中路線

## 4. 监管与政策

- **主管**：金融庁（FSA）+ 関東財務局
- **銀行業規制**：銀行法（単独銀行のため銀行持株会社規制は対象外）
- **長野銀行統合関連審査**：公正取引委員会 + 金融庁の地域金融機関再編に関する独占禁止法特例法（2020 施行）下での審査が論点となった ^[inferred]
- **直近政策論点**：
  - 2024〜 日銀政策金利正常化 → 地銀マージン改善（メガより恩恵が相対的に大）
  - 2025〜 地銀再編加速（人口減 + デジタル投資負担）→ 長野銀行吸収完了後の次の戦略フェーズ
  - 単独銀行のまま維持するか、持株会社化（FG 化）へ移行するかの判断
  - 県内電子産業（セイコーエプソン等）の構造転換（プリンタ需要減・半導体シフト）への main bank としての対応

## Counterpoints

- 「長野銀行統合の実効性」：完全子会社化（2023-06）から current legal-name entity への移行後、システム統合・店舗統廃合・人員整理が円滑に進行したかは観察事項。地銀合併で旧第二地銀側の顧客流出・店舗閉鎖の反発は典型的論点 ^[inferred]
- 「県内 1 強体制の独禁法論点」：完全子会社化時点で地域金融機関再編特例法（2020 施行）下での当局審査が論点となった経緯あり、吸収合併後の県内市場集中度は継続観察 ^[inferred]
- 「単独銀行維持 vs 持株会社化」：しずおか FG・[[fukuoka-fg|FFG]] 等の単独地銀 FG 化トレンドに対し八十二銀行は単独銀行維持、長野銀行吸収完了後の方針転換可能性 ^[inferred]
- 「電子産業集積の構造転換リスク」：セイコーエプソンのプリンタ需要減・ミネベアミツミの構造再編等、県内電子産業の中長期構造課題は main bank としてのリスク要因 ^[inferred]
- 「観光業ポートフォリオの脆弱性」：軽井沢・白馬等のリゾート / 観光ホテル融資はコロナ危機（2020〜2022）で経験した通り、外的ショックに脆弱 ^[inferred]
- 「人口減 + 若年層流出」：長野県の人口減トレンドは全国平均より顕著、中長期の県内預貸ボリューム縮小は構造リスク ^[inferred]
