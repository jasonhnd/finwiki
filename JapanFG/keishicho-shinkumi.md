# 警視庁職員信用組合 (Metropolitan Police Employees Credit Cooperative)

## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]] as the **首都圏 (Tokyo) 職域 (occupation-based) shinkumi anchor** for the police sector, complementing [[JapanFG/tokyo-shobo-shinkumi|東京消防信用組合]] (fire dept) and [[JapanFG/tokyo-to-shokuin-shinkumi|東京都職員信用組合]] (metropolitan civil servant) within the 143-cooperative population catalogued in [[banking/credit-cooperative-registry-japan|Japan credit cooperative registry]].

## TL;DR

警視庁職員信用組合は、東京都千代田区霞が関 (警視庁本部内) に本店を置く **警視庁所属職員 (現職警察官 + 職員 + OB) を組合員とする職域 (occupation-based) 信用組合**。FSA registry No. 40 (関東財務局所管)、法人番号 5010005002102。組合員資格を **警視庁職員 + その配偶者・親族**に限定する典型的な職域信組で、組合員向けの **生活資金ローン・住宅ローン・冠婚葬祭融資・退職金運用**を主要業務とする。[[JapanFG/zenshin-kumi|全国信用協同組合連合会 (全信組連)]] 会員、[[JapanFG/dic|預金保険機構]] 対象。警察組織という閉鎖的職域コミュニティの相互扶助金融として、戦後の地方公務員・警察組織発展と並走して成長した特殊形態。^[extracted]

## 2. 営業地区・組合員資格

### 組合員資格 ^[extracted]

職域信組のため、組合員資格は厳格な **職域 (occupation) 限定**:

1. **基本要件**: 警視庁所属職員 (現職警察官・事務職員・技術職員等)
2. **退職者**: 警視庁を退職した OB / OG (一定期間内継続加入が一般的)
3. **配偶者・親族**: 組合員の配偶者・同一世帯の家族 (定款による)
4. **員外利用**: 厳格に制限 (信組法準拠)

### 営業拠点 ^[inferred]

- **本店**: 警視庁本部 (千代田区霞が関) 内
- **支店・出張所**: 警視庁の各方面本部・大規模警察署内に出張窓口を設置する形態が一般的 ^[inferred]
- **オンライン**: 系統 (全信組連) 経由 ATM ネットワーク + インターネットバンキング
- **物理店舗網は最小限**で、組合員 (警視庁職員) の職場アクセス前提

## 3. 系統内位置

### 全信組連内の位置付け

- **[[JapanFG/zenshin-kumi|全信組連]]** の関東ブロック会員行
- 関東 49 信組のうち東京 19 信組の一翼
- **職域信組グループ**の代表行の一つ — 全国の警察職員信組 (大阪府警察信用組合・愛知県警察信用組合・兵庫県警察信用組合・福岡県警察信用組合等) と連携 ^[inferred]
- 系統 ATM 連携・余資運用は全信組連経由

## 4. 業務 map

### 主要業務 ^[extracted]

| 業務 | 内容 |
|---|---|
| **個人ローン** | 組合員向けカードローン・フリーローン |
| **住宅ローン** | 組合員世帯向け住宅取得資金 |
| **教育ローン** | 組合員子女向け教育資金 |
| **マイカーローン** | 組合員向け自動車購入資金 |
| **冠婚葬祭融資** | 組合員家庭の慶弔費 |
| **定期・流動性預金** | 組合員世帯向け預金 |
| **退職金運用** | 組合員定年退職者向け資産運用相談 |
| **共済関連** | 警察共済 ([[JapanFG/police-kyosai|警察共済組合]]) との補完的金融サービス ^[inferred] |

### ALM・資金フロー ^[inferred]

```
組合員 (警視庁職員・OB + 家族)
  → 預金 → 警視庁職員信用組合
              → 組合員向け住宅・生活ローン
              → 余資預け入れ → [[JapanFG/zenshin-kumi|全信組連]]
                                  → 機関投資 + 系統還元
```

### 警察共済組合との関係 ^[inferred]

- **警察共済組合**は地方公務員共済組合法に基づく **共済年金 + 福祉事業 + 貸付事業**を担う制度上の機関
- 警視庁職員信用組合は **共済組合と並立する独立金融機関**で、共済組合の貸付事業を補完する位置付け
- 組合員視点では、共済組合 + 信用組合の **二重金融アクセス**が確保される構造

## 6. 監督・規制

- **主管**: 金融庁・関東財務局
- **法的根拠**: 中小企業等協同組合法 + 協同組合による金融事業に関する法律 (協金法)
- **預金保険**: [[JapanFG/dic|預金保険機構]] 一般金融機関枠
- **直近政策論点**:
  - **デジタル化対応** (組合員向けスマホアプリ・電子契約)
  - **AML/CFT 強化** (職域信組は相対的に低リスク扱い)
  - **共済組合との機能調整** (年金・貸付・福祉事業分担)
  - **信組業界全体での再編・統合圧力** (職域信組は固有性が高く、地域信組とは異なる存続論理) ^[inferred]

## Counterpoints

- **職域信組の独立性 vs スケールメリット**: 組合員基盤が警視庁職員に限定されるため **規模拡張余地が構造的に小さい**が、独立性・密着性・低不良債権率が強み ^[inferred]
- **共済組合との機能重複**: 警察共済組合の貸付事業との **役割分担が組合員にとって不透明**なケース ^[inferred]
- **DX 投資余力**: 系統共同化に依存、自前 DX 投資余力は限定的 ^[inferred]
- **若年警察官の信組離れ**: スマホネイティブ世代がメガバンク・ネット銀行を選好する傾向で、組合員浸透率の世代別格差 ^[inferred]
