# プロクレアホールディングス (ProCrea HD)

## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[JapanFG/fidea-hd|フィデア HD (Fidea Holdings)]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

青森県地盤の地銀 2 行（**青森銀行 + みちのく銀行**）が 2022-04-01 経営統合して発足した持株会社（東証 PRIME **7384**）。**独占禁止法特例法（地域基盤企業特例法）初適用案件**として国の「お墨付き」で県内 2 行統合を実現した、**人口減少地域における地銀生残モデルケース**。**2025-01-01 に両子銀行は合併し「青森みちのく銀行」が発足**（総預金残高 5 兆円超で東北 3 位、青森県内融資シェア 7 割超）。社名「ProCrea」は「Progress（前進）」と「Creation（創造）」の合成、北東北（青森・岩手・秋田）広域戦略を掲げる。

## 1. 会社概要

**正式名**：株式会社プロクレアホールディングス
**英名**：ProCrea Holdings, Inc.
**証券コード**：東証 PRIME **7384**
**設立**：**2022-04-01**（青森銀行 + みちのく銀行 共同株式移転による経営統合）
**本社**：青森県青森市（青森銀行本店所在地ベース）

### 主要子会社（統合直後の構造）

```
プロクレアホールディングス（持株会社・上場 7384）
  ├── 株式会社青森銀行（100%）── 県内シェア最大手・北東北中核
  │     └── 関連会社（リース・カード・コンサル等）
  └── 株式会社みちのく銀行（100%）── 県内 2 位・北海道道南にも店舗
        └── 関連会社（リース・カード・道銀との提携等）

→ 2025-01-01 合併（実施済み）:
  プロクレアホールディングス
  └── 株式会社青森みちのく銀行（100%）
```

### 統合の経緯（重要）

- **2021-05** 青森銀行・みちのく銀行が経営統合を発表
- **2022-04-01** プロクレア HD 発足（共同株式移転）
- 県内 2 行統合のため独禁法上の「地理的市場」問題が発生 → **独占禁止法特例法**で公正取引委員会の合併許可
- 本件は同特例法の **初適用案件**（地銀統合のモデルケースとなる）
- **2025-01-01** 両子銀行を合併、商号「**青森みちのく銀行**」が発足（頭取に青森銀・石川啓太郎、会長にみちのく銀・藤沢貴之、従業員約 2,300 人・店舗 148）

### 青森銀行 系譜

| 年 | 事象 |
|---|---|
| **1879**（明治 12）| **第五十九国立銀行** 創立（弘前、青森県下初の銀行、青森銀行最古の前身）|
| **1897**（明治 30）| 国立銀行営業満期前特別処分法により**株式会社第五十九銀行**へ改組 |
| **1943-10-01**（昭和 18）| **第五十九・八戸・津軽・板柳・青森の 5 行合併で青森市に青森銀行を新設**（一県一行政策下の集約）|
| 戦後〜2022 | 青森県内シェア最大の地方銀行として営業 |
| 2022-04-01 | プロクレア HD 子会社化 |
| 2025-01-01 | みちのく銀行と合併 → 青森みちのく銀行 |

### みちのく銀行 系譜

| 年 | 事象 |

| 2025-01-01 | 青森銀行と合併 → 青森みちのく銀行 |

## 3. 事業セグメント・マップ

| セグメント | 主要事業者 | 特徴 |
|---|---|---|
| 県内法人 | 青森銀行・みちのく銀行 | 青森県内事業者向け main bank、合併後はシェア突出 |
| 県内個人 | 両行 | 預金・住宅ローン・年金口座 |
| 北東北広域 | 青森銀行（岩手・秋田にも店舗）| 北東北 3 県広域戦略 |
| 北海道道南 | みちのく銀行 | 函館等、北海道金融機関との接点 |
| 一次産業金融 | 両行 | **りんご・米・水産（青森は日本一のりんご産地、漁業も主要産業）** |
| 観光金融 | 両行 | ねぶた祭・弘前さくら・新青森駅・奥入瀬・十和田湖 |

### 統合シナジー

- **県内シェア集中**: 合併後の青森みちのく銀行は青森県内貸出・預金シェアでほぼ独占的地位 → 独禁法特例法で許容
- **システム統合**: 大規模 IT 投資の重複回避
- **人件費合理化**: 重複管理部門の集約

### 地域戦略

- **北東北戦略**: 青森・岩手・秋田の北東北 3 県を一体市場として捉える
- **道南接続**: みちのく銀行の函館拠点が新幹線（新青森〜新函館北斗）開通で価値化
- **観光・農業・漁業**: りんご輸出（台湾・香港向け）・ホタテ・サクラマス・新青森駅周辺再開発等

### 課題（人口減少地域モデル）

- **青森県人口減少**: 全国でも上位の減少率、若年層流出（東京・仙台へ）
- **デフレと高齢化**: 預貸ギャップ拡大（預金は集まるが貸出先が乏しい）
- **地元企業の縮小**: 廃業・後継者不在 → 事業承継支援が主要業務化
- 統合の本質は「成長戦略」より「**収益基盤の整理統合による持続可能性確保**」

## 5. 規制・政策

- **主管**: 金融庁（FSA）東北財務局
- **持株会社規制**: 銀行法 第 52 条の 17
- **独禁法特例**: **独占禁止法特例法**（地域基盤企業特例法、令和 2 年法律第 32 号、2020-11 施行、10 年時限立法）
  - **2021 年 公取委認可** → **同特例法 初適用案件** ★
  - 県内シェア集中による競争制限懸念を「持続可能性」優先で許容した政策転換
  - 後続地銀統合（ふくおか FG 系等）のモデルとなる

## 7. Open questions

- **独禁法特例法 10 年時限**（〜2030）後の制度延長・恒久化議論
- 北東北広域戦略の実質的進展（岩手・秋田での貸出シェア拡大は実現するか）
- **TSUBASA アライアンス等広域提携**との関係：県境を越えた更なる経営統合シナリオ
- 北海道道南拠点（みちのく銀函館等）の戦略的位置付け（北洋銀行・北海道銀行との関係）
- **地銀再編の次のドミノ**：プロクレア HD モデルを他県地銀（特に九州・四国・北陸）が追随するか
