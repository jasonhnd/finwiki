# 豊田通商フィナンス (Toyota Tsusho Finance)

## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[JapanFG/toyota-financial|トヨタファイナンシャルサービス (TFS)]] (the Toyota-group **captive auto-finance** holding, distinct from this 商社系 internal-treasury arm) and [[JapanFG/toyota-finance|トヨタファイナンス]] (the auto-loan / Toyota dealer finance operating company). For peer 商社系 group-treasury arms, see [[JapanFG/mitsubishi-corp-finance|三菱商事FS]], [[JapanFG/mitsui-bussan-financial-services|三井物産FS]], [[JapanFG/sumitomo-corp-financial-management|住友商事FM]], [[JapanFG/itochu-finance|伊藤忠ファイナンス]], [[JapanFG/marubeni-financial-services|丸紅フィナンス]]. See [[banking/INDEX|banking index]] for the broader regulatory boundary.

## TL;DR

豊田通商フィナンス は **豊田通商 (8015)** 系の **グループ内金融サービス会社**。豊田通商グループの **CMS / 一括ファクタリング / グループファイナンス / 為替・金利リスクヘッジ仲介 / 海外送金** を担う。豊田通商は **トヨタグループ系商社** で、トヨタ自動車のサプライチェーン (鋼材 / 自動車部品 / 製造設備 / 海外モビリティ) + 非自動車事業 (アフリカ・食品・化学品・エネルギー) の二軸で構成され、グループ金融機能はこの両軸を支える。なお、トヨタ車の販売金融 (dealer financing / オートローン) は [[JapanFG/toyota-financial|TFS]] / [[JapanFG/toyota-finance|トヨタファイナンス]] が担い、本ページの 豊田通商系 finance とは別系統である点に注意。^[inferred]

### トヨタグループ金融機能の三系統

トヨタグループには **三つの異なる金融機能** が並列している：^[inferred]

| 系統 | 主体 | 役割 |
|---|---|---|
| **顧客販売金融 (auto-finance)** | [[JapanFG/toyota-financial|TFS]] (持株) → [[JapanFG/toyota-finance|トヨタファイナンス]] 等 | トヨタ車のオートローン・dealer financing・クレジットカード ([[JapanFG/jcb|JCB]] 系 TS3 等) |
| **金融市場業務 (treasury / 為替 / 投資)** | トヨタファイナンシャル株式会社 (持株傘下) | 本体 + 海外子会社の金融市場業務 |
| **商社系 internal banking (本ページ)** | **豊田通商フィナンス** | 豊田通商グループ内の CMS / 一括ファクタリング / グループファイナンス |

この三系統は **トヨタ自動車 (本体) / 豊田通商 (商社) / 各販社 (国内 / 海外ディーラー)** という機能分業に対応する。

### 豊田通商系銀行関係

豊田通商の主取引銀行は伝統的に **東海銀行 → UFJ → [[JapanFG/mufg|MUFG]] 系** + [[JapanFG/smfg|SMFG]] 系 + [[JapanFG/mizuho-fg|みずほ FG]] 系の複数銀行関係。^[inferred] 商社系内部金融子会社としては、メガバンク 3 行と並列的に協業する形が中心。

## 3. 战略与竞争

### 豊田通商の事業構造と金融機能

豊田通商は **金属 / グローバル部品・ロジスティクス / 自動車 / 機械・エネルギー・プラントプロジェクト / 化学品・エレクトロニクス / 食料・生活産業 / アフリカ (CFAO)** の事業ポートフォリオを持つ。^[inferred] 特徴：

- **トヨタ自動車サプライチェーン** での鋼材・部品ロジスティクス機能
- **アフリカ事業** (旧 トヨタ自動車のアフリカ販社 + 2012 年買収の フランス系 CFAO による全アフリカ 54 ヶ国カバー)
- **再生可能エネルギー / リチウム** 等の新領域

豊田通商フィナンス はこれらの分野でのサプライヤーファイナンス / 海外通貨ヘッジ / 国境跨ぎ送金 等を支える。

豊田通商の特徴は、**外向けリース専業子会社を独立化せず**、トヨタグループ内の [[JapanFG/toyota-financial|TFS]] 系列と機能を補完しあう形をとる点。これは「トヨタ車関連の auto-finance / dealer financing は TFS、商社サプライチェーン金融は 豊田通商フィナンス」という棲み分け。^[inferred]

## 4. 监管与政策

- **主管**：金融庁 (貸金業 / 為替業務関連)
- **業界団体**：(商社系内部金融としてリース事業協会への加盟は限定的) ^[inferred]
- **規制論点**：
  - グループ内金融取引の **移転価格税制** (特にアフリカ・新興国向け取引)
  - グループファクタリングと **割賦販売法 / 貸金業法** の境界
  - 連結会計上の **内部取引消去**
  - **アフリカ向け金融サービス** の現地通貨規制 + AML/CFT 対応 ^[inferred]
- **直近政策論点**:
  - 2024〜 [[JapanFG/boj-monetary-policy|日銀政策金利正常化]] → グループ内金利付与の再評価
  - EV / リチウム / アフリカ事業の拡張に伴うグループ金融機能の負担増 ^[inferred]
  - 海外送金 / クロスボーダー資金移動の AML/CFT 強化 (アフリカ 54 ヶ国対応の特殊性)

## Counterpoints

- 「内部金融子会社」はガバナンスが形式的になりがち ^[inferred]
- 外部開示が限定的、業績規模は推測ベース ^[inferred]
- 豊田通商系 finance は トヨタグループ内 [[JapanFG/toyota-financial|TFS]] / [[JapanFG/toyota-finance|トヨタファイナンス]] と機能区分が外部からはわかりにくく、混同されやすい ^[ambiguous]
- アフリカ事業の急拡大で、現地通貨リスク / 国境跨ぎ送金リスク / 政治リスクが集中する可能性 ^[inferred]
