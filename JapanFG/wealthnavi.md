# ウェルスナビ株式会社 (WealthNavi)

## TL;DR

国内ロボアドバイザリー型資産運用サービス最大手。**預かり資産約 1 兆円 ^[ambiguous]**、運用者数約 40 万人規模 ^[ambiguous]。2015-04-28 柴山和久（元 マッキンゼー、財務省出身）創業、2016-07 サービス開始。2020-12-22 東証マザーズ上場（証券コード **7342**、現 グロース）。**2024-12-12 三菱 UFJ 銀行 [[mufg]] が公開買付（TOB）で完全子会社化発表 ★** ── 国内ロボアドのパイオニアがメガバンク AI 金融エコシステムに統合される転換点。^[extracted]

## 2. 业务线 map

| セグメント | サービス | 特徴 |
|---|---|---|
| 個人向け資産運用 | WealthNavi（自社）| 全自動・長期分散投資 / 米国 ETF 中心 |
| 提携（B2B2C）| WealthNavi for SBI / マネックス / ソニー銀行 / 横浜銀行 等 | ホワイトラベル供給、提携先顧客基盤活用 |
| NISA 対応 | おまかせ NISA | つみたて投資枠 + 成長投資枠 一括運用 |
| 機能オプション | 自動積立 / リバランス / DeTAX（税最適化）| 米国ロボアド準拠機能 |
| アドバイザリー | ライフプラン診断 / 目標金額シミュレーション | 投資未経験者向けオンボーディング |

### 運用方針

- **モダンポートフォリオ理論（MPT）**ベースの分散投資
- **米国上場 ETF 中心**（VTI / VEA / VWO / AGG / GLD / IYR / TIP 系の組み合わせ）
- 顧客のリスク許容度に応じた **5 段階のポートフォリオ**
- **自動リバランス**（半年に 1 回 + 乖離発動時）
- **DeTAX**: 含み損銘柄を意図的に売却し税最適化（米 Wealthfront の Tax-Loss Harvesting に相当）^[inferred]

### MUFG 子会社化（2024-12-12 発表）★

- **公開買付者**: 三菱 UFJ 銀行（MUFG Bank、[[mufg]] 100% 子会社）
- **目的**: WealthNavi を MUFG グループの個人向け資産運用 DX エンジンに統合 ^[inferred]
- **戦略的意義**:
  - MUFG は個人デジタル領域で メガ 3 行中最弱と言われ（vs [[smfg]] Olive・[[rakuten-fg]]・PayPay）、ロボアド No.1 取得で巻き返しを狙う ^[inferred]
  - 2026-05-07 発表の MUFG × Google AI 提携と組み合わせ、AI 金融エコシステム化を加速 ^[inferred]
  - 三菱 UFJ eスマート証券（旧 auカブドットコム証券、2025-01 KDDI から取得）と組み合わせたデジタル証券・ロボアド連携シナリオ ^[inferred]
- **競争影響**:
  - SBI / マネックス / ソニー銀行 等の既存ホワイトラベル提携の継続可否が論点 ^[ambiguous]
  - 特に [[sbi-hd]] 提携（WealthNavi for SBI 証券）は競合 SBI ラップ と並列、MUFG 子会社化後の整理が不透明 ^[ambiguous]

### ビジネスモデル特性

- **手数料一律 1.1%（税込）**: 投信信託報酬とは別、預かり資産連動のストック型収益
- **損益分岐は預かり資産規模に依存**: 預かり資産 1 兆円超で年間収益 ~110 億円規模（粗利ベース）^[inferred]
- **顧客獲得コスト（CAC）**: ホワイトラベル提携で抑制、自社直販はデジタル広告中心
- **チャーン**: 長期投資前提で低水準だが、相場下落局面で解約増 ^[inferred]

## 4. 监管与政策

- **主管**: 金融庁（FSA）
- **業登録**: 第一種金融商品取引業者・投資運用業者
- **加入協会**: 日本証券業協会、一般社団法人 日本投資顧問業協会
- **顧客資産保全**: 信託保全（三井住友信託銀行 等で分別管理）^[extracted]
- **直近政策論点**:
  - **2024-01〜 新 NISA 制度**: ロボアド業界全体に追い風（つみたて + 成長投資枠 拡充）
  - **2025〜 顧客本位の業務運営原則**: 手数料水準・運用パフォーマンス開示の透明性要求強化
  - **2024〜 金融サービス仲介業 制度**: ホワイトラベル提携の規制整理

### Counterpoints

- **「手数料 1.1% は高い」批判**: 米 Wealthfront（0.25%）/ Betterment（0.25%）と比べ約 4 倍。為替・税制差異もあるが、長期リターンを圧迫する可能性 ^[inferred]
- **「ロボアドのリターンは凡庸」批判**: S&P500 連動 ETF 単純積立と比較してアウトパフォーム不明、リバランス・分散効果は相場局面依存 ^[ambiguous]
- **「MUFG 子会社化＝独立性喪失」観**: メガバンク傘下で投信・銀行商品との利益相反リスク、提携先（SBI 等）からの離反リスク ^[inferred]
- **「ロボアド市場の頭打ち」観**: 大手ネット証券のラップ参入（SBI ラップ / 楽天）でコモディティ化、差別化要因が薄れる ^[inferred]
- **「DeTAX の実効性」**: 日本の税制下では米 Tax-Loss Harvesting ほど節税効果が出にくいとの指摘 ^[ambiguous]
