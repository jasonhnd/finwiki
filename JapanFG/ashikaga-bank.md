# 足利銀行


## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[JapanFG/yokohama-bank|横浜銀行]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

株式会社足利銀行は、金融庁の銀行免許一覧に掲載される栃木県本店の地方銀行で、[[mebuki-fg]] 傘下の operating bank。栃木県の first-regional bank anchor として、茨城側の [[joyo-bank]] とともに Mebuki group の two-bank operating model を構成する。

## 2. Operating model

足利銀行は、栃木県を中心に retail, SME, public-sector, housing-loan, and regional-development finance を担う regional bank。2003 年の一時国有化 history は [[mebuki-fg]] page で詳述し、この page は現行の bank-license and operating-company boundary を担当する。Kanto regional-bank consolidation context は [[banking/regional-bank-consolidation-pattern]] を参照。
