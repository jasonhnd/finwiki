# 双日フィナンス (Sojitz Finance)

## TL;DR

双日フィナンス は **双日 (2768)** 系の **グループ内金融サービス会社**。双日は 2004 年に **ニチメン + 日商岩井** の合併で発足した 7 大商社の一角 (5 大商社の次のレイヤー)。グループ規模は 5 大商社 (三菱商事 / 三井物産 / 住友商事 / 伊藤忠 / 丸紅) より小さいが、CMS / グループファイナンス / 一括ファクタリング / 為替・金利ヘッジ仲介 等の **内向け金融機能** はやはり子会社化されている。リース系 (外向け asset finance) の専業大手子会社は持たず、外部リース会社との取引が中心。^[inferred]

### 双日と銀行系列

双日は 2004 年に旧 ニチメン (日綿実業 → ニチメン、関西系・繊維発祥) と旧 日商岩井 (日商 + 岩井産業、関西系) が合併して発足。^[inferred] よって資本系列としては：

- **旧 ニチメン**：三和銀行 (現 [[JapanFG/mufg|MUFG]] 三和系) との関係
- **旧 日商岩井**：旧 第一勧業銀行・旧 IBJ (現 [[JapanFG/mizuho-fg|みずほ FG]]) との関係

合併後は **複数メガバンクとの並列取引** が中心で、単一銀行系列に閉じない柔軟な戦略。^[inferred]

## 3. 戦略・競争

### 双日の事業構造と金融機能

双日は **自動車 / 航空産業 / インフラ・ヘルスケア / 金属・資源・リサイクル / 化学品 / 食料・アグリビジネス / 生活産業 / リテール** の事業ポートフォリオを持つ。^[inferred] 7 大商社の中では中堅規模だが、**自動車 (海外 ASEAN・ロシア・中央アジア 等のディーラー網) / 航空機部品 / 化学品** で固有の強みを持つ。

双日フィナンス はこれらのセグメントでのサプライヤーファイナンス / 海外通貨ヘッジ / 国境跨ぎ送金 等を支える。中堅商社ゆえに金融機能のスケールは 5 大商社より小さいが、機能セットは類似。

## 4. 規制・政策

- **主管**：金融庁 (貸金業 / 為替業務関連)
- **業界団体**：(商社系内部金融としてリース事業協会への加盟は限定的) ^[inferred]
- **規制論点**：
  - グループ内金融取引の **移転価格税制**
  - グループファクタリングと **割賦販売法 / 貸金業法** の境界
  - 連結会計上の **内部取引消去**
- **直近政策論点**:
  - 2024〜 [[JapanFG/boj-monetary-policy|日銀政策金利正常化]] → グループ内金利付与の再評価
  - 中堅商社ゆえにスケールメリットが限定的、外部リース会社・銀行とのバランス再構築の余地 ^[inferred]
  - ロシア・中央アジア事業の地政学リスク (双日の自動車事業は同地域露出が比較的高い) ^[inferred]

## Counterpoints

- 「内部金融子会社」はガバナンスが形式的になりがち ^[inferred]
- 外部開示が限定的、業績規模は推測ベース ^[inferred]
- 中堅商社の内部金融子会社は、5 大商社レベルのスケールメリットを得られず、独立子会社化のコスト対効果が議論される可能性 ^[inferred]
- 双日は合併商社 (ニチメン + 日商岩井) で組織文化が複雑、内部金融の統合度がどこまで進んだかは不透明 ^[ambiguous]
