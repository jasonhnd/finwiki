# 池田泉州ホールディングス

## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[JapanFG/kyoto-fg|京都フィナンシャルグループ]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

大阪府を地盤とする **関西中堅地銀 FG**。証券コード **東証 PRIME 8714**。2009-10-01 **池田銀行 + 泉州銀行** の経営統合により発足し、翌 2010-05-01 に両子銀行が合併して **池田泉州銀行** が成立。大阪府 **北摂・池田エリア（旧池田銀）** + **泉州エリア = 大阪府南部（旧泉州銀）** + **兵庫県南東部** を 3 つの主軸地盤とし、京阪神メガ経済圏の中堅・中小企業金融に特化。関西では [[resona-hd]]（関西みらい銀）・京都 FG・滋賀銀行・南都銀行 等と直接競合。メガ 3 行（[[mufg]] / [[smfg]] / [[mizuho-fg]]）に対しては中堅・中小レイヤーで棲み分け。^[extracted]

## 1. 会社概要

**正式名**：株式会社池田泉州ホールディングス ^[extracted]
**英名**：Ikeda Senshu Holdings, Inc.
**証券コード**：東証 PRIME **8714**（旧 池田銀行 8370 + 旧 泉州銀行 8407 の上場廃止と引換、2009-10-01 上場）^[extracted]
**設立**：**2009-10-01**（池田銀行 + 泉州銀行 共同株式移転による経営統合）^[extracted]
**本社**：大阪市北区茶屋町 18-14（池田泉州銀行本店併設）^[extracted]

### 主要子会社・組織

```
池田泉州 HD（持株会社・東証 PRIME 8714）
  ├── 株式会社 池田泉州銀行（100%）── 大阪府北摂 + 泉州 + 兵庫南東部 地盤
  │     ├── 営業店舗 約 130 店規模（大阪府が中心、兵庫・京都・奈良・和歌山一部）
  │     ├── 中堅・中小企業向け融資（製造業・物流・サービス・不動産）
  │     ├── 住宅ローン・個人預金（北摂エリア中産階層）
  │     └── 法人 IB / シ・ローン 参画（メガ主導案件に subordinate 参加）
  ├── 池田泉州キャピタル ── ベンチャー投資 / 事業承継支援
  ├── 池田泉州 JCB ── クレジットカード（JCB 提携）
  ├── 池田泉州ビジネスサービス ── BPO / 事務代行
  └── 関連シンクタンク・調査機能（地域経済分析）
```

### 合併経緯・前身

- 旧 **池田無尽 → 池田銀行**（北摂地盤）
  - **1949** 池田無尽 設立（無尽会社・大阪府池田市）
  - **1951** 池田相互銀行 改称（相互銀行法施行に伴う転換）
  - **1989** 池田銀行 改称（普通銀行への転換、旧証券コード 8370）
- 旧 **泉州銀行**（大阪府泉州地区地盤）
  - **1942** 泉州銀行 設立（戦時統制下の地方銀行統合・大阪府泉州地区）
  - 戦後、堺市・岸和田市・泉佐野市等 泉州エリアで中小企業金融特化（旧証券コード 8407）
- **池田泉州 HD 発足**: 2009-10-01
- **池田泉州銀行 合併**: 2010-05-01（池田銀行 + 泉州銀行 → 池田泉州銀行）

### 戦略軸

- **地域密着 + 中堅・中小特化**: 大阪府の中堅・中小企業 main bank としての位置確立。メガに対しては小口・地場案件で棲み分け ^[extracted]
- **北摂（旧池田銀）+ 泉州（旧泉州銀）の地理的補完**: 大阪府の北部と南部を 1 行でカバー、府内 GDP の取り込み拡大 ^[extracted]
- **2024 中計**: 中期経営計画で **地域密着 + デジタル + 法人 BS ソリューション** を主軸とする方針 ^[ambiguous]

### 統合シナジー・課題

- **池田銀（北摂）+ 泉州銀（南部）の地理的補完性 高**: 大阪府を北部・南部両端で押さえる地銀は他になく、単一行統合のシナジー大 ^[extracted]
- **京阪神メガ経済圏での「中堅地銀」生残**: メガ 3 行 + [[resona-hd]] + 京都銀 等との顧客争奪 ^[extracted]

## 4. 規制・政策

- **主管**: 金融庁（FSA）
- **持株会社規制**: 銀行法 第 52 条の 17（銀行持株会社）^[extracted]
- **直近政策論点**:
  - **2024〜 日銀政策金利正常化**: 国内貸出金利マージン改善ボーナス（地銀メリット大、利上げで地銀収益改善期待）
  - **2024〜 地銀再編期待**: 金融庁の地銀再編スタンスは「統合自体は否定しないが当局主導はしない」→ 関西地銀再編の主体候補の 1 つとして言及されることあり ^[ambiguous]
  - **2025〜 営業店舗削減**: 地銀標準のリストラ路線、店舗網のスリム化と機能集約
  - **中小企業向け事業性評価融資**: 担保・保証に依らない融資の推進
  - **ESG / 地域脱炭素 ファイナンス**: 中小企業向け脱炭素転換支援
  - **大阪・関西万博（2025-04〜10）**: 関西経済圏の活性化期待 → 地銀の取引先（建設・物流・観光・サービス）に follow-tail 効果
