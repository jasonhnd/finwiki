# アモーヴァ・アセットマネジメント (旧 日興アセット)


## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[JapanFG/asset-management-one|アセットマネジメントOne]] for peer / contrast context and [[securities/INDEX|securities index]] for the broader system / regulatory boundary.

## TL;DR

アモーヴァ・アセットマネジメントは旧「日興アセットマネジメント」。2025-09-01 に商号変更し、FSA 金融商品取引業者一覧では「アモーヴァ・アセットマネジメント株式会社」、関東財務局長(金商)第368号として掲載される。[[JapanFG/sumitomo-mitsui-trust|三井住友トラストグループ]] 系の global asset manager。

## 1. 法人・ライセンス境界

| Item | Reading |
|---|---|
| Current legal name | アモーヴァ・アセットマネジメント株式会社 |
| Former name | 日興アセットマネジメント株式会社 |
| FSA registration | 関東財務局長(金商)第368号 |
| FIEA scope | 第二種、投資助言・代理業、投資運用業 |
| Parent boundary | [[JapanFG/sumitomo-mitsui-trust|三井住友トラストグループ]] |

## 2. Business role

- 日本に本社を置く global asset manager として、投資信託・ETF・機関投資家向け運用を扱う。
- 旧 Nikko AM ブランドから Amova へ移行したため、旧名リンクと現名リンクの両方が必要。
- [[JapanFG/smtb|三井住友信託銀行]] / [[JapanFG/sumitomo-mitsui-trust|SMTG]] の trust / asset-management strategy と接続する。

## 3. Why standalone page matters

日興アセットの旧名で資料・ファンド名・販売会社資料が残る一方、現行登録名は Amova。wiki では旧名・現名を同じページに束ね、FSA registration と brand transition を同時に追う。
