# JA 信連 (信用農業協同組合連合会 / Prefectural Credit Federations of Agricultural Cooperatives)

## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]] as the **prefectural credit-federation layer** of the [[banking/ja-bank-system-japan|JA Bank system]], sitting between local JA (497 cooperatives) and the national central institution [[JapanFG/norinchukin|農林中央金庫]]. Read with [[JapanFG/jf-shingyoren|JF 信漁連]] for the fishery-side equivalent and [[banking/cooperative-banking-japan|cooperative banking domain]] for the wider context.

## TL;DR

**JA Bank system の都道府県中間層 = 信用農業協同組合連合会 (信連)**。農業協同組合法を根拠とする **都道府県単位の信用事業連合会**。[[banking/ja-bank-system-japan|JA Bank system]] 公式資料 (2025-04-01 時点) では **31 都道府県信連** が稼働 (一部都道府県は信連を経由せず JA → 農林中金 直結方式に移行)。役割は **(1) 域内 JA からの余資の集約 (預金)**、**(2) 余資の中央機関 ([[JapanFG/norinchukin|農林中央金庫]]) への預け入れ + 自主運用**、**(3) 域内 JA への融資・流動性供給**、**(4) 域内 JA への経営指導・システム提供**。**信用金庫 → [[JapanFG/shinkin-central|信金中央金庫]]**、**信用組合 → [[JapanFG/zenshin-kumi|全信組連]]** の二段階構造と類似だが、JA Bank の場合は **「497 JA → 31 信連 → 農林中金」の三段構造** (一部「JA → 農林中金 直結」もあり)。^[extracted]

## 1. 組織構造

| 項目 | 内容 |
|---|---|
| 通称 | JA 信連 / 信連 / 県信連 |
| 英名 | Prefectural Credit Federation of Agricultural Cooperatives (Shinnoren) |
| 法的形態 | **農協法に基づく都道府県連合会** (信用事業連合会) |
| 設立 | 1948 以降、各都道府県別に設立 |
| 信連数 | **31 信連** (2025-04-01 時点) ^[extracted] |
| 監督官庁 | 金融庁 (信用事業) + 農林水産省 (協同組合監督) |

### JA Bank system の三段構造 ^[extracted]

```
JA Bank system (3 層構造)
  Layer 1: 全国 497 JA (個別農業協同組合)
                ↓ 余資預け入れ
  Layer 2: 31 信連 (都道府県中間層) ★本ページ
                ↓ 余資預け入れ
  Layer 3: 1 農林中央金庫 (中央) → [[norinchukin]]
                ↓ 機関投資・系統還元
```

注: 一部都道府県では **「JA → 農林中金 直結」方式** に移行 (信連が解散 / 統合済み) — 沖縄・東京等 ^[inferred]

### 信連数の推移 ^[extracted]

| 年代 | 信連数 | 備考 |
|---|---|---|
| 1990 年代後半 | 47 信連 (各都道府県別) | ピーク |
| 2000〜2010 年代 | 段階的減少 (合併・統合) | JA Bank system 統合化と並行 |
| **2025-04-01** | **31 信連** | [[JapanFG/norinchukin|農林中金]] 公開資料 |

### 重要年表 ^[extracted]

| 年 | 事象 |
|---|---|
| 1947 | 農業協同組合法制定、信連の法的根拠整備 |
| 1948〜 | 各都道府県別に信連設立 |
| 1990 年代 | バブル崩壊・不良債権で信連経営課題 |
| **2002〜** | **[[banking/ja-bank-system-japan|JA Bank system]] 稼働** — 全国 JA + 信連 + 農林中金 一体運営化 ^[extracted] |
| 2010 年代〜 | **信連の合併・統合進行** — 都道府県別事情で段階的に整理 |
| 2020〜 | 一部都道府県で **「JA → 農林中金 直結」**移行 (信連を介さない方式) ^[inferred] |
| 2025-04 | 31 信連 + 497 JA + 1 農林中金 体制 |

## 2. 機能・業務

### 4 大機能 ^[extracted]

| 機能 | 内容 |
|---|---|
| **余資集約** | 域内 JA からの余資 (組合員預金の超過分) を集約 |
| **中央への預け入れ** | 集約した余資を [[JapanFG/norinchukin|農林中央金庫]] への系統預け金として上納 |
| **自主運用** | 一部は信連自身で機関投資 (国債・地方債等) |
| **域内 JA 支援** | 経営指導・流動性供給・システム共同化 |

### JA Bank system 内の資金フロー ^[inferred]

```
組合員 (農家・地域住民)
  → 預金 → 全国 497 JA
              → 余資預け入れ → 31 信連
                                   → 余資預け入れ → 農林中央金庫
                                                       → 機関投資 (国債/外債/オルタナ)
                                                       → 系統還元 (配当)
                                                       → 信連 → JA → 組合員へ
```

### 信連の規模感 ^[ambiguous]

- 大規模信連 (北海道・愛知・福岡等) は **総資産数兆円** クラス
- 小規模信連 (山陰・四国一部等) は **数千億〜1 兆円** クラス
- 業界全体預金規模 (信連単独集計) は **JA Bank 預金 100 兆円超のうち相当部分** ^[inferred]

### 都道府県別の主要信連 (例) ^[inferred]

| 都道府県信連 | 規模感 | 備考 |
|---|---|---|
| 北海道信連 | 大規模 | JA 北海道大集約 |
| 愛知県信連 | 大規模 | JA あいち管轄 |
| 福岡県信連 | 大規模 | 九州主要 |
| 静岡県信連 | 中規模 | - |
| (沖縄等) | 直結方式へ移行 | 信連を介さず |

> 注: 個別信連の規模は agriregistry / 金融庁資料 + 各信連 IR 要確認 ^[ambiguous]

## 3. 戦略・競争

### JA Bank system 内の位置付け

- **中間層としての存在意義**: 都道府県単位の経営指導・流動性供給は **農林中金の中央機能では細かく対応不可** な部分を担当 ^[extracted]
- ただし **IT 共同化・JA 統合進行**で **中間層を「省略可能」**との議論も — 直結方式への移行論 ^[inferred]
- 「**信連存続派 vs 直結派**」の論争が JA Bank system 内に存在 ^[inferred]

### 統合・再編動向

- **複数県信連の合併** = 例として中央事例 + 西日本での統合事例 ^[inferred]
- 経営難信連の **農林中金への業務移管 / 直結方式移行**
- 域内 JA 数縮小に伴う **信連の規模縮小・効率化** 圧力

### 競合・対照組織

- **[[JapanFG/shinkin-central|信金中央金庫]]**: 信用金庫業界の中央 (信金 → 信金中金 の 2 段) — 信連 + 農林中金の 3 段とは構造が異なる
- **[[JapanFG/zenshin-kumi|全信組連]]**: 信用組合業界の中央 (信組 → 全信組連 の 2 段)
- **[[JapanFG/rokin-renraku|労金連]]**: 労働金庫業界の中央 (労金 → 労金連 の 2 段)
- **[[JapanFG/jf-shingyoren|JF 信漁連]]**: 漁協系の対応国機関 (JF → 信漁連 → 農林中金 の 3 段)

### システム共同化

- **JA バンク中央センター** (基幹系・勘定系の共同運営) ^[inferred]
- 全国 JA + 信連で **JASTEM (JA バンク統合電算システム)** 共同利用 ^[inferred]
- ATM 共同ネットワーク (JA カード相互利用)

## 4. 監督・規制

- **主管**: 金融庁 (信用事業の健全性) + 農林水産省 (協同組合監督)
- **法的根拠**: 農業協同組合法 (信用事業連合会条項)
- **預金保険**: [[JapanFG/yokin-hoken-kiko|預金保険機構]] への加入は **農水産業協同組合貯金保険機構経由** (一般金融機関とは別建て) ^[inferred]
- **直近政策論点**:
  - **信連の更なる統合・直結方式移行**
  - **JA Bank 全体のシステム統合** (JASTEM 高度化)
  - **2024〜農林中金米国債含み損** の信連への配当影響
  - **食料安全保障計画 (2026〜)** での信連の機能再定義

## Counterpoints

- **「中間層の存在意義」論争**: 全国 497 JA + 中央農林中金で十分との立場 vs 都道府県単位の経営指導・流動性が必要との立場 — JA Bank system 内の **長年の論点** ^[ambiguous]
- **規模の不均一**: 大規模信連 (北海道・愛知) と小規模信連 (山陰等) で **業務効率・収益性の格差** — 統合圧力 ^[inferred]
- **農林中金 2024 米債損失の波及**: 中央の損失 → 信連への配当縮小 → 域内 JA への配当縮小 → 組合員 (農家) への還元縮小 の **連鎖懸念** ^[inferred]
- **信連 vs 直結方式のコスト比較**: 信連を介する場合の管理コスト vs 直結方式の信連解散コスト — **中長期での合理化判断** ^[ambiguous]
- **協同組合的ガバナンス**: 都道府県別組合員代表性 vs 全国一体運営の効率性のトレードオフ ^[inferred]
