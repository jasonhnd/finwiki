# EY 新日本有限責任監査法人 (EY ShinNihon)

## TL;DR

**EY 新日本有限責任監査法人**（Ernst & Young ShinNihon LLC）は **Big 4 系列の最大規模監査法人** の一角で、**EY グローバルネットワーク** の日本メンバーファーム。上場会社監査クライアント数は国内首位級（公開ベース、JICPA 公開リストに基づく）^[inferred]。前身は朝日新和会計社（朝日監査法人）と新日本監査法人の合流系譜で、現法人化（有限責任化）は **2008-07** 公認会計士法改正後の各 Big 4 同時期。**金融サービス本部** が銀行・保険・証券・資産運用・暗号資産を横断カバーし、**大手金融グループの会計監査人** として複数のメガ系・地銀・生損保案件を抱える ^[extracted]。

## 1. 法人概要

- **正式名**：EY 新日本有限責任監査法人 ^[extracted]
- **英名**：Ernst & Young ShinNihon LLC ^[extracted]
- **法人形態**：有限責任監査法人（公認会計士法 第 34-2 条以下、2008-07 制度導入）^[inferred]
- **本部**：東京都千代田区有楽町（東京ミッドタウン日比谷等）^[extracted]
- **代表者**：理事長（任期制）^[extracted]
- **加盟ネットワーク**：**EY（Ernst & Young）グローバル**（ロンドン本部）^[extracted]
- **提携法人**：EY 税理士法人、EY ストラテジー・アンド・コンサルティング、EY 新日本 サステナビリティ等 ^[extracted]

### 規模（公開ベース概算）^[inferred]

| 指標 | 概算 |
|---|---|
| 公認会計士（社員 + 職員） | 5,500 名超 |
| 上場会社監査クライアント数 | 800 社級（国内首位級）|
| 大手金融機関監査 | メガ系・大手生損保・大手証券を含む複数案件 |
| 拠点 | 全国主要都市 |

## 2. 沿革・系譜

| 年月 | 事象 |
|---|---|
| 1968 頃 | 朝日新和会計社 / 新和監査法人系譜のルーツ ^[inferred] |
| 1980s | 新日本監査法人 設立（中央青山系等との分離・再編期）^[inferred] |
| **2007-07** | **カネボウ粉飾事件・中央青山監査法人解体に伴う再編期**（PwC 系の中央青山が業務停止 → 業界再編へ）^[extracted] |
| **2008-07** | **公認会計士法改正により有限責任監査法人制度導入** → 大手は順次有限責任化 ^[extracted] |
| 2014-06 | **東芝 不適切会計事件 第三者委員会報告（後年）** に先行する新日本監査時代の論点（東芝は当時新日本クライアント）^[extracted] |
| **2015-12** | **金融庁が新日本監査法人に対し業務改善命令 + 課徴金処分（21 億円）** — 東芝 不適切会計関連 ★ ^[extracted] |
| 2018 頃〜 | **「EY 新日本」ブランドへ統合** — EY グローバル ブランド整理 ^[extracted] |
| 2020s | 金融サービス本部の暗号資産・サステナビリティ・IFRS チーム強化 ^[extracted] |

## 3. 金融サービス本部（Financial Services Office, FSO）

EY 新日本の **金融サービス本部** は **銀行・保険・証券・資産運用・暗号資産** を横断カバーする FS 専業組織。EY グローバルの FSO ネットワークと連携 ^[extracted]。

### 主要セクター ^[extracted]

| サブセクター | 主な対象 |
|---|---|
| **メガバンク・大手銀行** | [[JapanFG/mufg]] 系会計監査関連業務、地銀グループの一部 |
| **生損保** | 大手生命保険・損害保険会社、外資系生損保 |
| **証券・資産運用** | 大手証券、運用会社、信託銀行 |
| **暗号資産・Web3** | FSA 登録 VASP の財務諸表監査・分別管理監査（[[exchanges/japan-crypto-audit-firm-landscape]] 参照）|
| **規制対応** | IFRS、米国基準、J-SOX、SOC レポート |

### サービスライン

- 法定監査（会社法 + 金融商品取引法）
- 分別管理監査（VASP 向け、資金決済法）
- 内部統制監査（J-SOX）
- SOC 1 / SOC 2 レポート
- IFRS / US-GAAP 助言
- IPO 支援（証券取引所準備）
- サステナビリティ保証（SSBJ / ISSB 関連）^[extracted]

## 4. 主要 listed FG クライアント関係（公開ベース）

EDINET 上の **有価証券報告書 監査人開示** および各社開示資料から確認可能な範囲で、新日本（EY 系）が監査人を務めた / 現在務める大手金融機関の代表例：

- 大手生保 / 損保数社（具体的監査人組み合わせは各社有価証券報告書 監査人欄参照）^[inferred]
- 大手証券・運用会社の複数 ^[inferred]

注：監査人交代は会社法 + 金商法上頻繁にあり得るため、最新の監査人就任状況は **EDINET 上の各社最新有価証券報告書「監査人の状況」セクション** または **JICPA 公開検索** を一次ソースとして確認すべき。当エントリは時点情報を保証しない ^[ambiguous]。

## 5. 規制・監督

- **主管法令**：公認会計士法（昭和 23 年法律第 103 号）^[extracted]
- **登録**：日本公認会計士協会（JICPA）に監査法人として登録 ^[extracted]
- **監督**：**金融庁 公認会計士・監査審査会（CPAAOB）** が品質管理レビュー / 検査を実施 ^[extracted]
- **業界自主規制**：JICPA 業種別委員会実務指針（暗号資産については第 61 号）^[extracted]

### CPAAOB 検査・処分歴 ^[extracted]

- **2015-12 業務改善命令 + 約 21 億円の課徴金**（東芝 不適切会計を契機）★
  - 金融庁公表処分。新日本史上最大規模の行政処分
  - その後の品質管理体制・パートナー任期制度等の強化につながった ^[inferred]
- それ以降は CPAAOB 検査結果報告書で個別法人名指摘は限定的（公開ベース）^[ambiguous]

## Counterpoints

- **「Big 4 寡占リスク」**：日本の上場会社監査は Big 4（新日本・トーマツ・あずさ・あらた）で過半が占められる。監査人選択の競争性低下・利益相反リスクは長期的論点 ^[ambiguous]
- **「コンサル兼業の利益相反」**：EY ストラテジー・アンド・コンサルティング等の関連法人とのファイアウォール運用は SOX 同等規制下にあるが、米国 EY 分割案（2022〜2023 撤回）等のグローバル議論と連動 ^[extracted]
- **「東芝事件の教訓持続性」**：2015 課徴金以降の品質管理改革が真に定着したかは ^[ambiguous]
- **「監査報酬の長期低水準」**：日本の監査報酬水準は欧米比で低く、品質確保とのバランスは継続論点 ^[inferred]
