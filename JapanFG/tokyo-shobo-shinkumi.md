# 東京消防信用組合 (Tokyo Fire Department Credit Cooperative)

## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]] as the **東京消防庁職員 occupation-based shinkumi anchor**, completing the Tokyo emergency/civil-service shinkumi trio with [[JapanFG/keishicho-shinkumi|警視庁職員信用組合]] (police) and [[JapanFG/tokyo-to-shokuin-shinkumi|東京都職員信用組合]] (metropolitan civil servant) within the 143-cooperative population catalogued in [[banking/credit-cooperative-registry-japan|Japan credit cooperative registry]].

## TL;DR

東京消防信用組合は、東京都千代田区大手町 (東京消防庁本部内) に本店を置く **東京消防庁所属職員 (消防官 + 事務職員 + OB) を組合員とする職域信用組合**。FSA registry No. 49 (関東財務局所管)、法人番号 2010005002121。組合員資格を **東京消防庁職員 + その配偶者・親族**に限定する小規模職域信組で、組合員向けの **生活資金ローン・住宅ローン・退職金運用**を主要業務とする。[[JapanFG/zenshin-kumi|全国信用協同組合連合会 (全信組連)]] 会員、[[JapanFG/dic|預金保険機構]] 対象。東京消防庁約 1.8 万人 (消防官 + 事務職員) という比較的小規模な組合員基盤で運営される、職域信組のニッチ代表例。^[extracted]

## 2. 営業地区・組合員資格

### 組合員資格 ^[extracted]

職域信組のため、組合員資格は厳格な **東京消防庁職員限定**:

1. **基本要件**: 東京消防庁所属職員 (消防官・救急隊員・事務職員・技術職員等)
2. **退職者**: 東京消防庁を退職した OB / OG (一定期間内継続加入が一般的)
3. **配偶者・親族**: 組合員の配偶者・同一世帯の家族 (定款による)
4. **員外利用**: 厳格に制限
5. **組合員除外**: 区部以外の消防団 (各市町村消防本部) は対象外、東京消防庁所管エリア (23 区 + 多摩地区委託) の本庁職員に限定 ^[inferred]

### 営業拠点 ^[inferred]

- **本店**: 東京消防庁本部 (千代田区大手町) 内
- **支店・出張所**: 主要消防方面本部・救急救命東京研修所等に出張窓口を設置する形態 ^[inferred]
- **オンライン**: 系統 (全信組連) 経由 ATM ネットワーク + インターネットバンキング
- **物理店舗網は最小限**で、組合員の職場アクセス前提

### 東京都内 emergency/civil-servant 職域信組三巨頭 (registry比較)

| 信組 | 本店 | 組合員基盤 | registry No. |
|---|---|---|---|
| 警視庁職員信用組合 | 千代田区霞が関 警視庁内 | 警視庁職員 約 4.6 万人 | 40 |
| **東京消防信用組合** ★本ページ | **千代田区大手町 東京消防庁内** | **東京消防庁職員 約 1.8 万人** | **49** |
| 東京都職員信用組合 | 新宿区西新宿 東京都庁内 | 東京都職員 約 16 万人 | 50 |

> 注: **警察 / 消防 / 一般行政**の三系統が独立信組を持つのは東京都だけの特殊事情。他の道府県は警察 + 一般行政 (県庁) のみが多く、消防は独立信組を持たず共済組合貸付で代替が一般的 ^[inferred]。

## 3. 系統内位置

### 全信組連内の位置付け

- **[[JapanFG/zenshin-kumi|全信組連]]** の関東ブロック会員行
- 関東 49 信組のうち東京 19 信組の一翼
- **職域信組グループ** + 中でも **公務員系職域信組**の代表
- 系統 ATM 連携・余資運用は全信組連経由

### 全国消防系職域信組ネットワーク ^[inferred]

東京消防信組は **全国唯一の消防専属職域信組** ^[ambiguous]:

|---|---|---|
| **東京消防信用組合** ★本ページ | 東京都 | **49** |
| (他道府県消防は独立信組なし、共済組合貸付で代替) | — | — |

> 注: 大阪市消防局・名古屋市消防局・横浜市消防局等の主要消防本部も独立信組は持たず、自治体職員共済組合の中で消防分野が含まれる形が一般的 ^[inferred]。東京消防庁の規模・歴史的経緯が独立信組を可能にした特殊例。

## 4. 業務 map

### 主要業務 ^[extracted]

| 業務 | 内容 |
|---|---|
| **住宅ローン** | 組合員世帯向け住宅取得資金 (主要業務) |
| **個人ローン** | カードローン・フリーローン |
| **教育ローン** | 組合員子女向け教育資金 |
| **マイカーローン** | 組合員向け自動車購入資金 |
| **冠婚葬祭融資** | 組合員家庭の慶弔費 |
| **定期・流動性預金** | 組合員世帯向け預金 |
| **退職金運用** | 組合員定年退職者向け資産運用相談 |
| **団信付き団体扱い** | 給与天引き融資 |
| **共済補完** | [[JapanFG/police-kyosai|地方公務員共済組合]] (消防共済) との補完的金融サービス ^[inferred] |

### ALM・資金フロー ^[inferred]

```
組合員 (東京消防庁職員・OB + 家族, 約 1.8 万人母集団)
  → 預金 → 東京消防信用組合
              → 組合員向け住宅・生活ローン (低リスク・高安定)
              → 余資預け入れ → [[JapanFG/zenshin-kumi|全信組連]]
                                  → 機関投資 + 系統還元
```

### 消防職員信組の業務特性 ^[inferred]

- **信用リスクが極めて低い**: 組合員 (消防官・救急隊員) は **安定収入・終身雇用に近い身分**で延滞率が低い
- **住宅ローン中心**: 消防官の持ち家ニーズが組合員サービスの主軸
- **24 時間 3 交代制対応**: 組合員の勤務形態に配慮した窓口時間・WEB 完結手続
- **災害時継続性**: 大規模災害時も組合員アクセス継続性が重要 (消防官は災害時最前線)

## 6. 監督・規制

- **主管**: 金融庁・関東財務局
- **法的根拠**: 中小企業等協同組合法 + 協同組合による金融事業に関する法律 (協金法)
- **預金保険**: [[JapanFG/dic|預金保険機構]] 一般金融機関枠
- **直近政策論点**:
  - **DX 対応** (スマホアプリ・電子契約・WEB 完結ローン)
  - **災害時継続性確保** (BCP・事業継続計画)
  - **共済組合との機能調整**
  - **OB / OG 組合員管理**
  - **信組業界全体での再編・統合圧力** (職域信組は固有性が高く、地域信組とは異なる存続論理) ^[inferred]

## Counterpoints

- **規模の構造的制約**: 組合員基盤が東京消防庁 1.8 万人に限定されるため **規模拡張余地が極めて限定的**、業務多角化も困難 ^[inferred]
- **DX 投資余力**: 系統共同化に依存、自前 DX 投資余力は限定的 (小規模信組共通の制約) ^[inferred]
- **共済組合との機能重複**: 地方公務員共済組合 (消防分野) の貸付事業との **役割分担が組合員にとって不透明**なケース ^[inferred]
- **若年消防官の信組離れ**: スマホネイティブ世代がメガバンク・ネット銀行を選好する傾向 ^[inferred]
- **再編リスク**: 業界再編圧力下で、消防専属信組の存続論理が問われる可能性 (他の自治体職員信組との統合シナリオ等) ^[inferred]
