# 荘内銀行


## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[JapanFG/hokuto-bank|北都銀行]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

株式会社荘内銀行は、金融庁の銀行免許一覧に掲載される山形県地盤の地方銀行で、[[fidea-hd]] 傘下の operating bank。公式会社概要では 1878 年創業の第六十七国立銀行を前身とし、[[hokuto-bank]] と 2027-01 に合併して「フィデア銀行」となる予定の片翼として扱う。

## 2. Operating model

荘内銀行は、山形県の retail, SME, agriculture, tourism, inheritance / wealth support, and public finance を担う regional bank。県内では [[yamagata-bank]] と競合しつつ、[[fidea-hd]] の山形・秋田広域モデルでは秋田側の [[hokuto-bank]] と組み合わせて整理する。
