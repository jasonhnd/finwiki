# 東京海上ホールディングス (Tokio Marine Holdings)

## TL;DR

国内損保 **3 メガ筆頭**(東京海上 / [[JapanFG/msad|ms-ad]] / [[JapanFG/sompo|sompo-hd]] のうち時価総額 No.1)。1879 年創業の **日本初の損害保険会社** 東京海上保険会社を起源とし、2002-04 ミレアホールディングス設立 → 2008-07 東京海上 HD に改称。**国内損保(東京海上日動) + 国内生保(あんしん生命) + 海外(米 4 大プラットフォーム:Philadelphia / Delphi / HCC / Pure)** の 3 軸。**純利益ベース海外比率 50% 超**で損保業界 No.1 の海外プレゼンス。東証 PRIME 8766。

**業態**:保険持株会社(損保・生保・再保険)

### 合併歴史・前身

- **1879-08** 東京海上保険会社 創業(**日本初の損害保険会社**、岩崎弥之助・渋沢栄一 等が発起人)
- **1944-03** 東京海上火災保険(社名変更)
- **1996-09** 東京海上 + 日動火災 業務提携
- **2001-04** 東京海上火災 + 日動火災 経営統合発表
- **2002-04-02** **ミレアホールディングス** 設立(東京海上 + 日動 共同持株会社)
- **2004-10-01** 東京海上火災 + 日動火災 → **東京海上日動火災保険** 合併発足
- **2008-07-01** ミレア HD → **東京海上ホールディングス** に改称(ブランド統一)
- **2008-12** Philadelphia Consolidated 買収
- **2012-05** Delphi Financial Group 買収
- **2015-10** HCC Insurance Holdings 買収
- **2020-02** Pure Group 買収

## 3. 战略与竞争

### 海外 M&A 主導の成長戦略

- **「4 大米国プラットフォーム」構築**(Philadelphia / Delphi / HCC / Pure):
  - 中堅 P&C(Philadelphia)・障害(Delphi)・スペシャルティ(HCC)・富裕層 P&C(Pure)の 4 領域で米国市場のニッチ最適化
  - 2008〜2020 の 12 年間で **累計買収額 約 2 兆円超**
- **海外比率 50% 超**(純利益ベース):損保業界で国内 No.1 の海外比率、[[JapanFG/msad|ms-ad]] / [[JapanFG/sompo|sompo-hd]] より高い
- **収益分散効果**:国内自然災害集中リスクを米国・欧州・アジアで分散

### 国内戦略

- **東京海上日動**:自動車保険シェア国内首位級、企業保険(海上・火災・賠責)に強み
- **あんしん生命**:第三分野(医療・がん)注力、銀行窓販チャネル活用
- **代理店網**:全国 約 5 万店、地銀 / 自動車ディーラー / 旅行代理店 等

### ESG / 気候変動

- **業界 ESG リーダー**:TCFD / NZIA 主要メンバー
- **自然災害リスク対応**:台風 / 集中豪雨 / 地震 の再保険スキーム高度化
- **2050 ネットゼロ** コミット(自社運営・引受ポートフォリオ両面)
- **生物多様性 / TNFD** 早期対応

### デジタル / イノベーション

- **InsurTech 出資**:米英 InsurTech へのコーポレートベンチャー投資
- **AI 査定**:自動車保険の事故査定 AI 活用
- **データ事業**:保険データを活用した法人向けリスクアドバイザリー

## 4. 监管与政策

- **主管**:金融庁(FSA)
- **保険業法**:第 271 条の 18 等(保険持株会社規制)
- **海外規制**:NAIC / 各州保険監督官(米国)、PRA / FCA(英国)、MAS(シンガポール)、OIC(タイ)等
- **IFRS 17**:2023〜 国際保険会計基準対応
- **直近政策論点**:
  - 2023〜 **ビッグモーター不正請求問題**:損保業界(東京海上日動も関連報道)で代理店ガバナンス・損保カルテル疑惑論点化、金融庁・公取委 調査
  - 2024〜 **政策保有株削減** 圧力:損保 3 メガに対し金融庁が政策保有株(企業向け株式持ち合い)解消を強く要請
  - 2024〜 **企業向け火災保険カルテル疑惑**:大規模物件の保険料調整疑惑で公取委 立入検査

## Counterpoints

- **「海外集中の通貨・地政学リスク」**:米国比率高 → ドル円・米国景気・米国訴訟環境(クラスアクション)依存。地震 / ハリケーン年は損害集中
- **「HCC 9400 億円買収のプレミアム評価」**:当時の損保 海外 M&A としては高額、のれん減損リスク論点(2015 以降の収益は概ね計画ライン)^[ambiguous]
- **「ビッグモーター・カルテル問題の業界余波」**:東京海上日動も代理店関与・カルテル疑惑当事者として報道される。レピュテーション影響範囲は継続観察 ^[ambiguous]
- **「政策保有株 vs 関係性ビジネス」**:法人保険は持ち合い株式と関係深く、解消は短期収益にプラスでも長期顧客関係に影響しうる ^[inferred]
- **「Pure Group(富裕層 P&C)の市場成長性」**:米国富裕層市場のニッチ性 → 規模拡大ペースに限界 ^[inferred]
