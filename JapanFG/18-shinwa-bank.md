# 株式会社十八親和銀行 (Eighteenth Shinwa Bank)


## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[JapanFG/fukuoka-bank|福岡銀行]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

十八親和銀行は長崎県を地盤とする [[JapanFG/fukuoka-fg|ふくおか FG]] 傘下銀行。1877年創立の十八銀行と、1879年起源・1939年設立の親和銀行が合併し、2020年10月に現在の十八親和銀行として発足した。長崎県の地域金融、FFG の九州広域戦略、地銀合併後のブランド統合を見るための entry。

## 1. 会社概要

| 項目 | 内容 |

| 正式名 | 株式会社十八親和銀行 |
| 本店 | 長崎市銅座町1番11号 |
| 代表者 | 取締役頭取 山川信彦 |
| 親会社 | [[JapanFG/fukuoka-fg|ふくおかフィナンシャルグループ]] |
| 発足 | 2020年10月、十八銀行 + 親和銀行合併 |
| 主地盤 | 長崎県全域、佐世保・長崎・県北・離島 |

公式会社概要では、2025-03 末時点で資本金 368億円、総資産 7兆4,927億円、預金 5兆5,269億円、貸出金 3兆8,965億円、従業員 1,835名、店舗 188（長崎県内 164）とされる。

## 2. 事業セグメント・マップ

| 事業セグメント | 内容 | 重要性 |
|---|---|---|

| 地域創生 | 観光、事業承継、産業支援 | 人口減少・地方創生対応 |

## 3. 戦略・競争

- **県内 2 行統合の実例**: 同一県内の十八銀行・親和銀行統合により、重複店舗・重複システムを整理する地銀再編モデル。
- **FFG 九州戦略**: [[JapanFG/fukuoka-fg|fukuoka-fg]] は福岡銀行を中心に、熊本・長崎・福岡中央・デジタル銀行を組み合わせる。十八親和銀行は長崎県担当の中核。
- **第8次中期経営計画**: 2025年4月開始。長崎を「もっと」元気にする地域共創銀行という方向性を掲げる。
- **長崎固有リスク**: 造船・観光・離島・人口減少という構造テーマが、地銀の融資ポートフォリオに反映される。

## 4. 規制・政策

- 銀行法上の銀行であり、金融庁・財務局監督下。
- 親会社 FFG は銀行持株会社として監督対象。
- 同一県内統合は競争政策・地域金融行政の観点で重要な先例。

## Counterpoints

- 県内 2 行統合はコストシナジーが見込める一方、顧客選択肢の低下・店舗統廃合への地域反発が起きやすい。
- 長崎県は人口減・離島コスト・観光依存が強く、通常の店舗効率改善だけでは限界がある。
- FFG グループの中で長崎県向け投資が福岡中心戦略に劣後しないかは観察点。
