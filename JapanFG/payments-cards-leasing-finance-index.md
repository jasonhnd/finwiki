# JapanFG payments, cards, leasing and non-bank finance sub-index

## Overview

This sub-index routes the **payments, cards, leasing and non-bank finance** layer of the JapanFG namespace: card issuers and brands, wallets / code-payment / prepaid / PSPs, funds transfer and remittance, crypto-asset and stablecoin infrastructure, leasing and corporate finance, and consumer finance. It complements the legal route in [[JapanFG/legal-financial-licenses/payment-license-stack|Japan payment license stack]]. Entity pages stay in [[JapanFG/INDEX|JapanFG]].

Use this page when the question is **"which payments cards leasing finance entities does JapanFG cover?"** It lists 111 entity pages by institution type. Each link points to a page that physically lives in `JapanFG/`; this sub-index adds discoverability without moving any file.

## Placement Rule

- **Entity pages stay in `JapanFG/`.** This sub-index only links to them with bare `[[JapanFG/slug]]` wikilinks; it does not move, rename, or duplicate any entity file.
- Membership here is derived from each entity page's frontmatter `tags:` taxonomy. An entity may legitimately also appear in an adjacent sub-index when its activity spans types.
- This is a navigation surface, not a registry of record. Where an FSA registry index already exists for a type, treat that registry page as the source of truth.
