# ハナ信用組合 (Hana Credit Cooperative)

## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]] as a **首都圏 (Tokyo) 在日コリアン民族系 shinkumi anchor**, peer to [[JapanFG/osaka-shogin-shinkumi|大阪商銀信用組合]] (Kansai 商銀系)・[[JapanFG/yokohama-kogin-shinkumi|横浜幸銀信用組合]] (Kanagawa 幸銀系)・[[JapanFG/aichi-shogin-shinkumi|信用組合愛知商銀]] (Chukyo 商銀系) within the broader Korean-business shinkumi sub-population catalogued in [[banking/credit-cooperative-registry-japan|Japan credit cooperative registry]].

## TL;DR

ハナ信用組合は、東京都渋谷区千駄ヶ谷に本店を置く **在日韓国・朝鮮人を組合員基盤とする民族系信用組合**で、**朝銀系信用組合の再編後の受け皿信組**として 2000 年代以降の信組危機後に発足した経緯を持つ ^[inferred]。FSA registry No. 53 (関東財務局所管)、法人番号 4011005001243。商号「ハナ (HANA)」は朝鮮語で「一つ・統合」を意味し、複数の旧朝銀信組の事業統合を象徴する命名 ^[inferred]。[[JapanFG/zenshin-kumi|全国信用協同組合連合会 (全信組連)]] 会員、[[JapanFG/dic|預金保険機構]] 対象。首都圏在日コリアンコミュニティの SME 融資・住宅ローン・跨境送金・コミュニティ金融が事業の中核。^[extracted]

## 2. 営業地区・組合員資格

### 営業地区 ^[inferred]

- **本店**: 東京都渋谷区千駄ヶ谷 (新宿駅・代々木駅近郊エリア)
- **支店網**: 東京 23 区内 + 首都圏在日コリアンコミュニティ集中地区 (新宿区・荒川区・足立区・葛飾区等) ^[inferred]
- **広域営業**: 神奈川 (川崎)・埼玉・千葉等の関東広域を含む可能性 ^[inferred]

### 組合員資格 ^[extracted]

民族系信組のため、組合員資格は **業域 + 民族コミュニティ**の二重構造 (中小企業等協同組合法準拠):

1. **業域要件**: 商工業者・自営業者中心 (員外利用厳格)
2. **民族コミュニティ**: 在日韓国・朝鮮人 + その配偶者・親族・関連事業者
3. **地区要件**: 関東圏を中心とする定款指定地区に在住・在勤・事業所を有する者

## 3. 系統内位置・朝銀系信組再編史

### 全信組連内の位置付け

- **[[JapanFG/zenshin-kumi|全信組連]]** の関東ブロック会員行
- 関東 49 信組のうち東京 19 信組の一翼
- **民族系信組グループ** + 中でも **旧朝銀系再編後の代表的受け皿信組** ^[inferred]
- 系統 ATM 連携・余資運用は全信組連経由

### 朝銀系信用組合の歴史的経緯 ^[extracted/inferred]

| 年代 | 事象 |
|---|---|
| 1950〜60 年代 | 在日朝鮮人 (主に北朝鮮支持系) 商工人向けの相互扶助金融として **「朝銀」**系信用組合が各地に設立 |
| 1970〜80 年代 | 全国主要都市に朝銀信組ネットワーク確立 |
| 1990 年代 | バブル崩壊で **不動産・建設業向け融資**の不良債権化が深刻化 |
| 2001〜2002 | **朝銀信組連鎖破綻問題** — 全国の朝銀信組数行が経営破綻、預金保険機構による大規模資金援助 |
| 2002〜 | **受け皿信組への事業譲渡 + 統合再編**進行 |
| 2003 以降 | **ハナ信用組合・ウリ信用組合・ミレ信用組合等**の受け皿信組として再編体制発足 ^[inferred] |
| 2010 年代〜 | 在日コミュニティの世代交代対応 + AML/CFT 強化対応 |

### 朝銀危機の規模 ^[inferred]

- 2001〜2002 年の朝銀信組破綻に伴う **預金保険機構からの公的資金投入規模は数千億円**規模 ^[ambiguous] — 在日朝鮮人系金融機関への大規模資金援助として議論を呼んだ
- 業界全体での **責任問題・送金問題**が政治問題化
- 現在の民族系信組 (ハナ・ウリ・ミレ等) はこの再編後の **クリーンスレート体制** ^[inferred]

### ALM・資金フロー ^[inferred]

```
組合員 (在日コリアン商工人・世帯 + 関連事業者)
  → 預金 → ハナ信用組合
              → 首都圏 SME 融資 + 住宅ローン
              → 余資預け入れ → [[JapanFG/zenshin-kumi|全信組連]]
                                  → 機関投資 + 系統還元
```

## 6. 監督・規制

- **主管**: 金融庁・関東財務局
- **法的根拠**: 中小企業等協同組合法 + 協同組合による金融事業に関する法律 (協金法)
- **預金保険**: [[JapanFG/dic|預金保険機構]] 一般金融機関枠
- **直近政策論点**:
  - **AML/CFT 強化** (跨境送金監視) — 民族系信組固有規制対応コスト + **対北朝鮮制裁関連**の制裁監視要件
  - **組合員世代交代** (在日 2 世・3 世以降の信組離れリスク)
  - **韓国系銀行 (新韓銀行東京支店等)** との競合
  - **政治的注目度の高さ** — 在日朝鮮人系信組ゆえに政治・国際関係環境変化の影響を受けやすい ^[inferred]
  - **信組業界全体での再編・統合圧力** ^[inferred]

## Counterpoints

- **政治的リスク**: 在日朝鮮人系金融機関ゆえに、**日朝関係・南北関係・国際制裁**の影響を直接受けやすい構造 ^[inferred]
- **過去の朝銀危機の長期影響**: 2001〜2002 年の朝銀信組破綻に伴う公的資金投入の **歴史的負荷** + **業界信認の再構築**プロセス継続 ^[inferred]
- **AML/CFT 規制対応コスト高**: 跨境送金監視 + 対北朝鮮制裁関連スクリーニングで規制対応コストが他信組より大きい ^[inferred]
- **コミュニティ縮小**: 在日朝鮮人コミュニティの世代交代 + 帰化進行 + 韓国国籍取得進行で組合員基盤の長期縮小リスク ^[inferred]
- **韓国系銀行参入競合**: 新韓銀行東京支店・KEB ハナ銀行東京支店等の韓国系商業銀行が同セグメントに直接参入 ^[inferred]
