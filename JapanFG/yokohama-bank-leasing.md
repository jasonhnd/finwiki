# 横浜銀リース (Yokohama Bank Leasing)

## Wiki route

This entry sits under the [[JapanFG/INDEX|JapanFG index]] as the FG-subsidiary leasing operating-company anchor for [[concordia-fg|横浜フィナンシャルグループ (旧コンコルディア FG)]]. Read it against parent operating bank [[yokohama-bank|横浜銀行]] and sibling subsidiary [[higashi-nippon-bank|東日本銀行]]. Peer comparison: [[mebuki-lease|めぶきリース]] (北関東地銀系) and major lessor [[mitsubishi-hc-capital|三菱 HC キャピタル]] / [[mizuho-lease|みずほリース]] / [[tokyo-century|東京センチュリー]]. For broader system route see [[banking/regional-bank-consolidation-pattern|regional-bank consolidation pattern]] and [[banking/INDEX|banking index]].

## TL;DR

株式会社横浜銀リースは、[[yokohama-bank|横浜銀行]] (現 [[concordia-fg|横浜フィナンシャルグループ]] 子会社) 連結子会社の地銀系リース会社で、神奈川県・首都圏を地盤に設備リース・ファイナンスリース・オペレーティングリース・自治体リース・医療リース等を提供。地銀系リースの中では上位規模で、[[concordia-fg]] (旧 CFG、2025-10 商号変更) の地銀 FG 最大級の預金顧客基盤を背景とする。^[extracted]

## 3. 合併歷史・前身

### 横浜銀リース 設立

- [[yokohama-bank|横浜銀行]] 系列リース会社として、神奈川県内中堅企業向け設備リースを主業
- 横浜銀行の県内 No.1 圧倒シェアを背景に、リース業務でも神奈川県内シェア上位
- 戦後地銀系リースの典型: 銀行融資ではなく設備リース形態での中小企業設備投資支援

### CFG → YFG 商号変更 (2025-10-01)

- 親持株会社 [[concordia-fg]] が 2016-04 [[yokohama-bank]] + [[higashi-nippon-bank]] 共同株式移転で発足
- **2025-10-01**: 旧 コンコルディア FG → **横浜フィナンシャルグループ** 商号変更
- 横浜銀リース 自体は商号変更の影響を受けず、引き続き [[yokohama-bank]] 連結子会社として運営

## 6. 戦略与競争

### FG 内ポジション

- **銀リース連携の中核**: [[yokohama-bank]] の神奈川県内中堅企業 main bank 顧客のリース需要受託
- **設備投資支援**: 銀行融資補完 + 簿外処理ニーズへの対応
- **県内自治体・公的セクター連携**: 神奈川県・横浜市・川崎市等 自治体リース案件
- **横浜銀行ブランドのリース展開**: [[concordia-fg|YFG]] 「横浜」ブランド回帰 (2025-10) との整合性

### 競合構図

- **大手リース**: [[mitsubishi-hc-capital|三菱 HC キャピタル]] / [[smfl|SMFL]] / [[mizuho-lease|みずほリース]] / [[fuyo-lease|芙蓉リース]] / [[tokyo-century|東京センチュリー]] / [[ricoh-leasing|リコーリース]] - 大型・全国・複雑案件で優位
- **地銀系リース**: [[mebuki-lease|めぶきリース]] (北関東) / [[hokkaido-lease|北海道リース]] / 千葉興業リース 等 - 同型 model
- **メーカー系リース**: [[toyota-finance|トヨタファイナンス]] / [[nec-leasing|NEC キャピタルソリューション]] - 製品連動
- **県内競合**: 横浜銀行系・神奈川県内信金リース・大手リース支店

### 強みと弱み

- 強み: [[yokohama-bank]] 預金 15 兆円超 (単体) の中堅企業 main bank ネットワーク + 首都圏 GDP 規模 + 神奈川県内 No.1 認知度
- 弱み: 単独規模では大手リースに大幅劣後、航空機 / 不動産複雑リースは対応困難 ^[inferred]

## 7. 监管与政策

- **主管**: 金融庁 (FSA) (リース業単独としては銀行業免許不要、貸金業登録は対応領域に応じて)
- **業界団体**: リース事業協会 加盟 ^[inferred]
- **規制論点**:
  - リース会計基準変更 (IFRS16 / 日本基準) 対応支援
  - 中小企業向けリース税制活用 (中小企業投資促進税制)
  - 環境リース (再エネ設備リース) の脱炭素税制対応
  - 反社チェック・マネロン対策 ([[concordia-fg]] グループ共同実施、2018-08 東日本銀行 業務改善命令後の内部統制強化と整合)

## Counterpoints

- 「地銀系リース = 中途半端」論: 大手リースに規模で劣後 vs 銀リース連携の優位性 ^[ambiguous]
- 「単独存続困難」論: リース業界寡占化進展、地銀系リース統合・売却シナリオ ^[inferred]
- 「FG 内利益相反」: 銀行融資 vs リース提案の使い分け基準 ^[inferred]
- 「神奈川集中 = 人口減少リスク」: 神奈川は他県より緩やかだが、長期人口減少局面で設備投資需要縮小懸念 ^[inferred]
