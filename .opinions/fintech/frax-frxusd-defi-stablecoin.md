# 主観内容の隔離 — Frax frxUSD · DeFi ネイティブ・ステーブルコインの「半コンプライアンス化」ルート

> 出典エントリー: `fintech/frax-frxusd-defi-stablecoin.md`

## Counterpoints

frxUSD $1.2B は §501 時代でも依然ニッチプロダクト (USDC $60B / USDS $8.5B / USDT $145B)。veFXS ガバナンスロック量は高い (流通の 45%) がアクティブアドレスは少なく、実際のガバナンスは少数のコア保有者に主導されており、「分散化」の名目が実質を上回る。Fraxtal L2 TVL $450M は Base $20B / Arbitrum $30B との差が巨大で、内部循環規模は小さい。**法的リスク**:SEC/Treasury が 2027-2028 に「issuer = 実質支配者」定義を veFXS DAO 上に置けば、Frax は強制コンプライアンスとなる。sfrxUSD の高 APY の一部は Pendle / Convex 二次市場レバレッジ由来であり、ストレステストでは脆弱。

## Open questions

Frax の BUIDL 直接保有は BlackRock からの KYC / Reg D 506(c) 追加制限を引き起こすか (vault は契約であって機関投資家ではない)?Fraxtal L2 は 2027 に $2B TVL を突破して内部循環規模を形成できるか?Liquity LUSD / GHO / crvUSD 等の小型分散型 SC は Frax の「半コンプライアンス化」ルートに追従するか?
