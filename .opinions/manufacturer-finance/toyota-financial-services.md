# 主観内容の隔離 — Toyota Financial Services (トヨタファイナンシャルサービス 製造業視点アンカー)

> 出典エントリー: `manufacturer-finance/toyota-financial-services.md`

## Counterpoints

- **販社送客モデルの持続性**: 直販 / D2C・オンライン中古車プラットフォーム (米 Carvana 等) が台頭し、ディーラー経由率が低下する可能性。
- **残価リスクの構造的増大**: EV 技術世代交代 + 中古車相場ボラ拡大で、残クレ・リースの RV 損失リスクが TFS 側に蓄積する設計。
- **地域分散の収益貢献**: 新興国は規模拡大中だが、為替・カントリーリスクで利益ボラ拡大。
- **銀行系・独立系との金利競争**: 銀行系オートローンは低金利で、TFS の販社優位を金利面で浸食する余地。
- **OEM 本体と金融子会社の利益相反**: 新車販売を最大化する販売価格と、残価リスクを最小化する金融設計の間でトレードオフが発生する。
- **モビリティ転換の収益化スピード**: KINTO 等サブスクの収益化は段階的で、従来型ローン・リースの収益を即座に補完できるかは未確定。

## Open questions

- TFS 単体の連結純利益・総資産規模 (親トヨタ有報「金融事業セグメント」開示の精緻化必要)。
- EV 残価設定モデルの具体的方法論 (バッテリー劣化評価・電池リースとの組合せ)。
- TS キュービックカード の会員数・流通額・トヨタ純正部品購買連動のシナジー定量。
- 米 TMCC の業界シェア (vs Ford Motor Credit・Ally・GM Financial) の最新ポジション。
- KINTO 等モビリティサブスクの収益化進捗と、従来 captive ビジネスとのカニバリゼーション度合い。
- グローバル統合 IT 基盤 (与信モデル・データ基盤) の整備状況。
- 自動車 ABS 発行体としての年間発行額・スプレッド・投資家ベースの変化。
