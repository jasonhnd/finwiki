# 主観内容の隔離 — Japan crypto exchange overview — FSA / JVCEA regulated VASP market

> 出典エントリー: `exchanges/jp-crypto-exchange-overview.md`

## Counterpoints

- 登録制は投資者保護に有効だが、銘柄上場速度とプロダクト多様性では海外市場に劣りやすい。
- JVCEA 会員数と金融庁登録業者数は一致しない。現物、デリバティブ、電子決済手段等取引業の区分を混同しない。
- 登録業者でも流動性・セキュリティ・親会社信用・停止/廃業リスクは個別に異なる。

## Open questions

- 金商法移行・申告分離課税の議論は、現物 VASP の収益構造をどう変えるか。
- 国内 CEX は EPI / stablecoin / deposit token とどのように接続するか。
- 海外系日本法人は、登録取得後にグローバル流動性をどこまで日本市場に持ち込めるか。
