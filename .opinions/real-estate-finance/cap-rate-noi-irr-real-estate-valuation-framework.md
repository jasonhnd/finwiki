# 主観内容の隔離 — Cap rate NOI IRR real-estate valuation framework

> 出典エントリー: `real-estate-finance/cap-rate-noi-irr-real-estate-valuation-framework.md`

## Counterpoints

- Cap rate is a static first-year metric; it does not capture cash-flow timing or growth — IRR is needed for that
- NOI definitions vary across surveys; "Tokyo office cap rate 3.2%" is not a single number until you specify the NOI definition
- Levered IRR is sensitive to assumed cost of debt; locked-in fixed-rate debt vs floating-rate exposure changes the analysis materially
- Hold-period sensitivity is rarely fully stress-tested in fund underwriting; LP-disclosed IRR may overstate base-case probability
- Appraisal-lag mechanics make appraised-NAV stale at inflection points
- Private-fund IRR can be window-dressed via early-exit selection bias and J-curve management
- Cap rate compression / widening is a market-clearing outcome, not a forecast input — using historical compression as a base case projects past dynamics forward

- Do data-centre and life-sciences niche asset classes establish their own cap-rate / IRR conventions, distinct from traditional four-class real estate?
