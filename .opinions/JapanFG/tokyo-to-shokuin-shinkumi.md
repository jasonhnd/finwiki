# 主観内容の隔離 — 東京都職員信用組合 (Tokyo Metropolitan Government Employees Credit Cooperative)

> 出典エントリー: `JapanFG/tokyo-to-shokuin-shinkumi.md`

## 推論 (from TL;DR)

東京都職員信用組合は、東京都新宿区西新宿 (東京都庁内) に本店を置く **東京都所属職員 (現職 + OB) を組合員とする職域信用組合**で、自治体職員系信組として **日本最大規模クラス** ^[inferred]。FSA registry No. 50 (関東財務局所管)、法人番号 8011105001107。組合員資格を **東京都職員 + その配偶者・親族**に限定し、住宅ローン・教育ローン・退職金運用を主要業務とする。[[cooperative-banks/zenshin-kumi|全国信用協同組合連合会 (全信組連)]] 会員、[[financial-regulators/dic|預金保険機構]] 対象。都職員約 16 万人 (教員・警察・消防除く知事部局 + 公営企業) という巨大組合員基盤を背景に、職域信組としては突出した規模を持つ。^[extracted]

## Counterpoints

- **職域信組の独立性 vs スケールメリット**: 組合員基盤が都職員に限定されるため **規模拡張余地が構造的に小さい**が、低不良債権・低コスト構造が強み ^[inferred]
- **共済組合との機能重複**: 地方職員共済組合 (短期貸付) との **役割分担が組合員にとって不透明**なケース ^[inferred]
- **若年都職員の信組離れ**: スマホネイティブ世代がメガバンク・ネット銀行 ([[regional-banks/sony-bank|ソニー銀行]] 等) を選好する傾向で、組合員浸透率の世代別格差 ^[inferred]
- **東京都の財政影響**: 都の財政・人員政策 (採用抑制・指定管理者化) が組合員基盤の長期サイズに影響 ^[inferred]

### ALM・資金フロー ^[inferred]

```
組合員 (東京都職員・OB + 家族, 約 16 万人母集団)
  → 預金 → 東京都職員信用組合
              → 組合員向け住宅・生活ローン (低リスク・高安定)
              → 余資預け入れ → [[cooperative-banks/zenshin-kumi|全信組連]]
                                  → 機関投資 + 系統還元
```

### 自治体職員信組の業務特性 ^[inferred]

- **信用リスクが極めて低い**: 組合員 (都職員) は **安定収入・終身雇用に近い身分**で延滞率が低い
- **住宅ローン中心**: 都職員の持ち家ニーズが組合員サービスの主軸
- **団信付き団体扱い**: 給与天引きによる団体扱いで事務コストが低い
