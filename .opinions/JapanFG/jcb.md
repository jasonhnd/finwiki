# 主観内容の隔離 — 株式会社ジェーシービー (JCB)

## Counterpoints

- **「アジア強い」評価の限界**: 欧州・南米・アフリカでは加盟店網薄く、「グローバル」を名乗るには Visa / Mastercard に大差 ^[inferred]
- **「日本唯一の国際ブランド」象徴性 vs 実態の経営規模**: ブランド存在感は大きいが、Visa / Mastercard との売上規模差は数桁レベル ^[inferred]
- **国内シェア圧迫**: 三井住友カード (Visa) / 楽天カード等の積極展開で国内 JCB ブランドシェアは漸減傾向 ^[ambiguous]
- **コード決済勢の侵食**: PayPay / 楽天ペイ / d 払い 等の急成長で「物理カード+タッチ決済」モデル全体が中長期で構造変化リスク ^[inferred]
- **非上場ゆえの情報限定**: 詳細な業績・シェア・株主構成等の公開情報が限定的、外部評価困難 ^[ambiguous]
- **Discover 提携の依存**: 米国カバレッジは Discover 加盟店網に依存 → 提携解消リスクの単一点 ^[inferred]
