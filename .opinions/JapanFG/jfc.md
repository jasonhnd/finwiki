# 主観内容の隔離 — 日本政策金融公庫 (JFC)

> 出典エントリー: `JapanFG/jfc.md`

## Counterpoints

- 「**コロナ特別貸付の不良債権化リスク**」: 20 兆円超の実質無利子無担保貸付が 2024〜2026 に返済期到来 → 代位弁済増加 → 国民負担懸念 ^[inferred]
- 「**民業圧迫批判**」: 信金・信組サイドから「JFC が低利で先に貸すと民間プロパー機会が減る」との批判が継続。協調融資原則は建前か実態か議論あり ^[ambiguous]
- 「**JBIC 分離の正否**」: 2012 分離後、中小企業の海外展開支援機能が JFC 中小企業事業に残ったが、JBIC との連携・棲み分けは時に不明瞭 ^[ambiguous]
- 「**4 機関統合のシナジー**」: 2008 統合から 18 年経つも事業部門間のシステム / 人事 / 営業文化は旧 3 公庫を色濃く残す。統合効果は限定的との見方 ^[inferred]
- 「**政策金利と財投金利の鞘**」: 民間金利上昇局面で JFC 政策金利が硬直的だと逆ザヤ懸念 vs 政策意義優先 ^[ambiguous]
