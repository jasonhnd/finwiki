# 主観内容の隔離 — 沖縄振興開発金融公庫 (ODFC)

> 出典エントリー: `JapanFG/okinawa-shinko.md`

## Counterpoints

- 「**本土 JFC との二重構造の非効率**」: 同じ国民・中小企業・農林水産機能を本土 JFC と ODFC で別組織運営することの組織コスト・人材分散批判 ^[ambiguous]
- 「**沖縄県内 3 地銀との競合・圧迫**」: 沖振法特別金利で ODFC が低利融資すると、民間 3 行(沖縄銀行・[[ryukyu-bank]]・沖縄海邦銀行)のプロパー融資機会が圧迫されるとの批判 ^[inferred]
- 「**観光業集中リスク**」: 県内産業構造が観光業に過度依存しており、ODFC 融資ポートフォリオも観光業エクスポージャー過大 → 次の感染症・地政学リスクで再度の壊滅シナリオ ^[inferred]
- 「**米軍基地跡地再開発の長期リスク**」: 返還スケジュールの不確実性 + 跡地利用計画の政治化 → プロジェクト融資の回収不安 ^[ambiguous]
- 「**沖振法 5 年ごと改正の政治依存**」: ODFC の存続基盤が沖縄振興特別措置法に依拠しており、本土政権の沖縄政策スタンス変化で組織存続が揺らぐ構造リスク ^[inferred]
