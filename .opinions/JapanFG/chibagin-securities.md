# 主観内容の隔離 — ちばぎん証券 (Chibagin Securities)

> 出典エントリー: `JapanFG/chibagin-securities.md`

## 推論 (from 1. 会社概要)

| 設立 | 旧 中央証券系譜 (戦後証券会社) → 千葉銀行買収・ちばぎん証券改称 ^[inferred] |
| 本社 | 千葉県千葉市中央区 (千葉銀行本店周辺) ^[inferred] |

## 推論 (from 持株ツリー (千葉銀グループ簡略図))

> 千葉銀は持株会社化を選択せず単独銀行のまま TSUBASA アライアンスを 2015 主導設立。証券子会社・リース子会社等のグループ会社は銀行本体直下にぶら下がる構造で、これは [[mebuki-fg]] (持株会社下に常陽銀・足利銀・めぶき証券・めぶきリース 並列配置) や [[fukuoka-fg]] (FFG 下に福岡銀・FFG 証券・FFG リース 並列配置) とは法的位置が異なる。^[inferred]

## 推論 (from 3. 親会社・グループ内ポジション)

千葉銀グループ内では、銀行本体の **預金 18-20 兆円規模の顧客基盤の証券窓口** という位置づけ。千葉銀が持株会社化していない単独銀行 (no-holding model) であるため、ちばぎん証券は法的に銀行本体の子会社 (直下) であり、[[mebuki-fg|めぶき FG]] の [[mebuki-securities|めぶき証券]] (FG 直下の兄弟会社) や [[fukuoka-fg]] の [[ffg-securities|FFG 証券]] (FFG 直下の兄弟会社) とはガバナンス構造が異なる。実務面では銀証連携 (法人紹介 + 顧客紹介 + 共同店舗) の depth は持株会社 model と同等ないし上回るとされる ^[inferred]。

## 推論 (from TSUBASA アライアンス連携)

千葉銀が 2015 主導設立した **TSUBASA アライアンス** (11 行協業: 千葉銀 + 北洋 + 武蔵野 + 滋賀 + 第四北越 + 中国 + [[iyo-bank|伊予]] + 東邦 + 北國 + 京都 + 群馬) を背景に、ちばぎん証券もアライアンス参加行系証券 (例: [[iyogin-leasing]] と並ぶ伊予銀系証券、群馬銀系証券 等) との情報共有・人材交流余地あり ^[inferred]。

## 推論 (from 強み・弱み)

- 弱み: 単独規模では大手証券に大幅劣後、デジタル投資余力限定的、千葉県集中 = 人口減少・首都圏東部の若年層流出リスク ^[inferred]

## 推論 (from 会計上の取扱い)

- **連結対象**: [[chiba-bank]] 100% 子会社として連結対象、銀行業セグメント以外の「その他」セグメントに集約されるのが典型 ^[inferred]
- **報告通貨**: 円建て、IFRS ではなく日本基準 ^[inferred]

## Counterpoints

- 「地銀系証券 = 中途半端」論: 大手証券にもネット証券にも勝てない中間ポジション vs 銀証連携の優位性 ^[ambiguous]
- 「no-holding model = 戦略的柔軟性低下」論: [[mebuki-fg]] / [[fukuoka-fg]] のような持株会社化により子会社並列配置 / 資本効率改善余地を取れないリスク vs 千葉銀の単独銀行 model は迅速意思決定のメリット ^[inferred]
- 「単独存続困難」論: 預かり資産 / 手数料収益規模で大手 IB に劣後、TSUBASA アライアンス内証券再編 / 大手証券への売却シナリオ ^[inferred]
- 「親銀行優先 = 証券独立性」: [[chiba-bank]] 銀行融資優先 vs 証券 advisory 独立性のバランス ^[inferred]
- 「デジタル投資余力」: 地銀系証券単独で対面 → デジタルへの transition 投資は限定的 vs ネット証券との手数料競争 ^[inferred]

- 主要収益源は **委託手数料 + 投信代行手数料 + 引受手数料 + ラップ口座フィー**
