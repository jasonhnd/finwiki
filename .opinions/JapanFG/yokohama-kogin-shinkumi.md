### 営業地区 ^[inferred]

- **本店所在地**: 横浜市中区尾上町 (関内・伊勢佐木町エリア)
- **支店網**: 横浜市内 (中区・鶴見区) + 川崎市 (川崎区) + 神奈川県内主要市の在日コミュニティ集中地区
- **コリアタウン拠点**: 鶴見区 (在日コリアン集中地区)・川崎区 (戦前期から在日朝鮮人居住)・横浜中華街周辺

### 幸銀系信用組合ネットワーク ^[inferred]

**幸銀**系は **商銀**系と並ぶ在日韓国・朝鮮人系信組グループの一系統:

| 系統 | 主要信組 (推定) |
|---|---|
| **商銀系** | [[JapanFG/osaka-shogin-shinkumi|大阪商銀信用組合]]、信用組合愛知商銀、信用組合広島商銀、宿毛商銀信用組合、朝銀西信用組合等 |
| **幸銀系** | **横浜幸銀信用組合** ★本ページ ほか系統内 |
| その他民族系 | ハナ信用組合 (旧朝銀系統)、ミレ信用組合 (大阪)、ウリ信用組合 (北海道) |

### 2000 年代信組危機の影響 ^[inferred]

- 1990 年代末〜2000 年代初頭の **在日系信組連鎖破綻問題** (関西商銀危機 + 朝銀信組破綻問題) を経て、関東神奈川エリアの民族系信組は受け皿信組への事業譲渡 + 統合再編を経験

### ALM・資金フロー ^[inferred]

```
組合員 (在日韓国・朝鮮人商工人・世帯)
  → 預金 → 横浜幸銀信用組合
              → 域内 SME 融資 + 住宅ローン
              → 余資預け入れ → [[JapanFG/zenshin-kumi|全信組連]]
                                  → 機関投資 + 系統還元
```
