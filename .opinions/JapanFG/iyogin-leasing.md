## 推論 (from 3. 親会社・グループ内ポジション)

いよぎん HD グループ内では、中核子会社 [[regional-banks/iyo-bank|伊予銀行]] の **愛媛県シェア No.1 + 四国全域 + 海事クラスター main bank 顧客基盤に対するリース業務窓口** という位置づけ。**いよぎん HD 2022-10-03 持株会社化**後、いよぎんリースは持株会社直下並列子会社として再配置され、[[regional-banks/mebuki-fg]] / [[regional-banks/fukuoka-fg]] / [[regional-banks/shizuoka-fg]] と同型の「持株会社下兄弟会社」model に移行 ^[inferred]。

| いよぎんビジネスサービス | BPO 経由のリース管理業務効率化 ^[inferred] |

## 推論 (from TSUBASA アライアンス参加 (伊予銀))

[[regional-banks/iyo-bank|伊予銀行]] は [[regional-banks/chiba-bank|千葉銀行]] 主導の **TSUBASA アライアンス** (2015-) 参加 11 行 (千葉銀 + 北洋 + 武蔵野 + 滋賀 + 第四北越 + 中国 + 伊予 + 東邦 + 北國 + 京都 + 群馬) の一員。これによりいよぎんリースは [[leasing-firms/chibagin-leasing|ちばぎんリース]] / 群馬リース / 京都リース 等アライアンス参加行系リースとの情報共有・共同案件組成余地あり ^[inferred]。

## 推論 (from 会計上の取扱い)

- **連結対象**: [[regional-banks/iyogin-hd]] 100% 子会社として連結対象、銀行業セグメント以外の「その他」セグメントに集約されるのが典型 ^[inferred]
- **リース債権**: 連結 BS 上「リース債権及びリース投資資産」勘定で表示 ^[inferred]
- **報告通貨**: 円建て、日本基準 (リース会計基準) 適用 ^[inferred]

## Counterpoints

- 「地銀系リース = 中途半端」論: 大手リースに規模で劣後 vs 銀リース連携の優位性 ^[ambiguous]
- 「HD 化 = 形式だけ」論: [[regional-banks/iyogin-hd]] 2022-10-03 持株会社化後、いよぎんリースの実質的な戦略変化は限定的との見方 vs 持株会社化により資本効率改善 / M&A 余地確保 ^[inferred]
- 「単独存続困難」論: リース業界寡占化進展、地銀系リース統合・売却シナリオ ^[inferred]
- 「FG 内利益相反」: [[regional-banks/iyo-bank|伊予銀行]] 融資 vs いよぎんリース提案の使い分け基準 ^[inferred]
- 「四国 = 人口減少リスク」: 四国は全国でも人口減少先行地域 ([[regional-banks/iyogin-hd]] 中期経営計画でも認識)、リース需要長期縮小懸念 ^[inferred]
- 「海事クラスター集中 = 海運市況リスク」: 今治造船 / 海運業界の市況変動で需要変動大 ^[inferred]
