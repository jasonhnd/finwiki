# foreign-financial-institutions — Foreign banks & securities/AM in Japan index

## Overview

`foreign-financial-institutions/` contains **56 entries** (excluding INDEX, 58 total with the registry page) covering **foreign banks' Japan branches and subsidiaries** and **foreign securities / asset-management houses operating in Japan** — Tokyo branches of overseas commercial banks, foreign investment banks, international central securities depositories (ICSDs), and foreign-group Japan subsidiaries, grouped by home-country corridor. Each entry is a source-backed entity page that physically lives in `foreign-financial-institutions/`. Use this domain when a banking, securities, or market-infrastructure page needs the Japan footprint of an overseas bank, the corridor it serves (Korea / Taiwan / mainland China / ASEAN / Europe / North America), or its custody / settlement role.

> **Scope / disambiguation.** This domain holds foreign-institution **entity** pages. For the thematic *concepts* — the foreign-bank Japan retreat / refocus pattern, the banking-system and license boundary, and how foreign branches sit in the licence tier map — see [[banking/INDEX|banking]]; for the foreign investment-bank / securities-services and ICSD interface see [[securities/INDEX|securities]]. For the broader financial-group *entity* umbrella, see [[JapanFG/INDEX|JapanFG]]. The pages here are the foreign-entity complement to those homes.
