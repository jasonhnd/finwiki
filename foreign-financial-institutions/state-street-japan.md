## 1. 会社概要

**最終親会社**：State Street Corporation（米マサチューセッツ州法人、NYSE: **STT**、本店マサチューセッツ州ボストン）
**親会社設立**：1792（The Union Bank として、米国 2 番目に古い継続営業の銀行の一つ）^[extracted]
**現体制発足**：1969（State Street Boston Financial Corporation 持株会社化）→ 1978 改称 State Street Corporation ^[extracted]
**ティッカー**：NYSE: STT

### グローバル財務（2025〜2026Q1 期）

| 項目 | 規模 |
|---|---|
| AUC/A（受託資産 + 管理資産）| **約 49 兆ドル**（2025 2Q）→ **約 53.8 兆ドル**（2025 4Q、過去最高）^[extracted] |
| 世界順位（カストディ）| **No.2**（首位 [[foreign-financial-institutions/bny-mellon-japan]] 約 59.3 兆ドル）^[extracted] |
| 直近新規受託純増 | 約 1.1 兆ドル（2025 期内、主に Asset Owners / Asset Managers 経由）^[extracted] |

### 日本の主要法人（公開情報ベース）

```
State Street Corporation（米 NYSE: STT・FRB / OCC 監督）
  ├── ステート・ストリート信託銀行株式会社 ── 日本法人、信託業ライセンス保有
  │     ├── 東京本店: 港区虎ノ門 1-23-1 虎ノ門ヒルズ森タワー 25F
  │     ├── 福岡営業所: 福岡市博多区綱場町 (2012 開設、2016 時点で 100 名超)
  │     └── 横浜営業所 (2012 開設 → 2015 閉鎖)
  ├── ステート・ストリート銀行 東京支店 (State Street Bank and Trust Company 在日支店)
  │     └── 米本店直結のグローバルカストディ・トランザクションバンキング窓口
  ├── ステート・ストリート・グローバル・アドバイザーズ株式会社 (SSGA)
  │     ├── 1998-02 設立、関東財務局長 (金商) 第 345 号
  │     └── SPDR ETF シリーズ運用元 (SPY = 世界最大 ETF を含む)
  └── チャールズリバー デベロップメント インク (Charles River Development Japan)
        └── 2007 設立、運用会社向けトレーディング / ポートフォリオ管理システム
```

**特徴**：
- **リテールバンキング・個人預金は一切扱わない**。完全に B2B / B2 機関投資家特化
- **SSGA の存在が BNY Mellon との差別点**：SPDR ETF（世界最大の SPY 含む）を日本でも展開し、ETF 運用 + カストディの両輪を持つ ^[extracted]
- **ステート・ストリート信託銀行 資本金 25 億 1000 万円、従業員 426 名（2024-04-30 現在）、R&I 格付 AA-** ^[extracted]
- **代表者**：信託銀行社長 **寺田恵子**、東京支店長 **若林憲弘**、SSGA 社長 **越前谷通博** ^[extracted]

## 2. 沿革（年表抜粋）

| 年 | 事象 |
|---|---|
| **1792** | **The Union Bank（State Street 前身）設立**（米国 2 番目に古い銀行系譜の一つ）^[extracted] |
| 1969 | State Street Boston Financial Corporation 持株会社化 ^[extracted] |
| 1978 | State Street Corporation に改称 ^[extracted] |
| **1986-03** | **ステート・ストリート信託銀行の前身 マニュファクチュラース・ハノバー信託銀行株式会社 設立**（米マニュファクチュラース・ハノバー社 100% 出資）^[extracted] |
| **1990** | **ステート・ストリート銀行 東京支店 開設**（State Street Bank and Trust Company の在日支店として）^[extracted] |
| **1991-12** | **マニュファクチュラース・ハノバー信託銀行 → ステート・ストリート信託銀行に商号変更**（State Street 傘下入り）^[extracted] |
| **1998-02** | **ステート・ストリート・グローバル・アドバイザーズ株式会社 (SSGA) 設立**（資産運用ビジネス本格化、SPDR シリーズの日本展開基盤）^[extracted] |
| **2006** | **ステート・ストリート銀行 東京支店に行政処分**（詳細は公開情報、当時の業法違反）^[extracted] |
| **2007** | **Charles River Development 日本拠点設立**（運用会社向け OMS/PMS の提供拡大）^[extracted] |
| **2012** | **ステート・ストリート信託銀行 福岡営業所開設 + 横浜営業所開設**（拠点拡張期）^[extracted] |
| 2015 | 横浜営業所閉鎖 ^[extracted] |
| 2016 | 福岡営業所人員 100 名超に拡大（バックオフィス機能集約）^[extracted] |
| **2025 2Q** | **AUC/A 約 49 兆ドル（過去最高）達成** ^[extracted] |
| **2026-01-16** | **2025 4Q 決算で AUC/A 53.8 兆ドルの新記録更新** ^[extracted] |

## 3. 事業セグメント・マップ

| セグメント | 在日担い手 | 特徴 |
|---|---|---|
| **グローバルカストディ** | ステート・ストリート銀行 東京支店 + 信託銀行 | 米国 ADR・海外株式・海外債券・オルタナ資産の保管・決済・コーポレートアクション処理 — 親会社の主力収益柱 |
| **国内信託（カストディ周辺）** | ステート・ストリート信託銀行 | 信託業ライセンス下での国内資産管理・受託業務、GPIF の運用受託機関の一角 ^[extracted] |
| 機関投資家サービス | 信託銀行 + 東京支店 | 邦銀・年金基金・運用会社向けバックオフィス・ファンドアドミ・トランスファーエージェント |
| **資産運用（AM）** | **SSGA（ステート・ストリート・グローバル・アドバイザーズ）** | 公的・私的年金、機関投資家、投資信託向け運用 / **SPDR ETF シリーズ運用元**（SPY = 世界最大 ETF）^[extracted] |
| **運用システム** | Charles River Development Japan | 運用会社向け OMS/PMS/EMS（買付・運用管理システム）SaaS — BNY Mellon にはない独自業態 |
| 個人リテール | **なし** | — |

### 日本でのポジショニング

- **GPIF・大手年金・邦銀の海外投資カストディ主要担い手**：日本マスタートラスト [[trust-banks/master-trust-bank]] と並ぶ GPIF 資産管理機関の一角 ^[extracted]
- **[[megabanks/mufg]] 株主構成 4.51%（2025-03-31）の意味**：これは **機関投資家の海外保管口座経由の名義集約**（カストディアン名義集積）であり、State Street 自身による戦略保有ではない ^[extracted]

### グローバル戦略との接続

- **SPDR シリーズ**：S&P 500 連動 SPY は世界最大 ETF、日本でも SSGA 経由で機関投資家向けに展開 ^[extracted]

## 5. 監督・規制

- **主管**：金融庁（FSA）— 信託銀行・在日支店・SSGA（金商業者）として
- **米側監督**：FRB（連邦準備制度理事会）/ OCC（通貨監督庁）/ FDIC、グローバルストレステスト DFAST 対象
- **直近の論点**：
  - **GPIF・公的年金の海外資産運用比率上昇** → カストディ需要構造的拡大
  - **円安・海外投資シフト** → 邦銀・運用会社の海外資産積み増しによる受託資産増
  - **2025〜2026 AUC/A 過去最高更新**：53.8 兆ドルだが、コスト上昇に伴い「収益パラドックス」懸念（市場の冷静視）^[extracted]
