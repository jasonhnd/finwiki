# アモーヴァ・アセットマネジメント (旧 日興アセット)

## Wiki route

This entry sits under [[asset-managers/INDEX|asset-managers INDEX]] (its domain route). Read it against [[asset-managers/smt-am|三井住友トラスト・アセットマネジメント (SMT-AM)]] for the same-group trust-affiliated 運用会社 peer / contrast, and against [[financial-regulators/fsa|the FSA]] plus [[securities/financial-instruments-business-operators-japan-index|FSA 金商業者 index]] for the supervisory / registry boundary over 投資運用業 operators.

## TL;DR

アモーヴァ・アセットマネジメント株式会社は旧「日興アセットマネジメント」。2025-09-01 に商号変更し、FSA 金融商品取引業者一覧では現名「アモーヴァ・アセットマネジメント株式会社」、関東財務局長(金商)第368号として掲載される。金融商品取引法上の **投資運用業** 登録を持つ運用会社で、[[trust-banks/sumitomo-mitsui-trust|三井住友トラストグループ (SuMi TRUST)]] の **完全子会社** (公開発表上、改称で資本・統治構造は不変)。日本に本社を置く global asset manager として公募投信・ETF・機関投資家向け運用を扱う。

## 2. Operating model

投資運用業者として、公募投信の **投資信託委託** (組成・運用)、ETF、**投資一任** (機関投資家向け discretionary) を担う、日本発の global asset manager。販売は [[trust-banks/smtb|三井住友信託銀行]] / [[trust-banks/sumitomo-mitsui-trust|SuMi TRUST]] の信託・銀行チャネルおよび証券会社の販売網。旧 Nikko AM ブランドの資料・ファンド名が残るため、wiki では **旧名 (Nikko) と現名 (Amova) を同一ページに束ねる**。監督は FSA + 自主規制機関の二層構造。同じ SuMi TRUST 系の [[asset-managers/smt-am|SMT-AM]] とは役割 (Amova = グローバル運用 / SMT-AM = 年金・公募投信) で分かれる。

## 3. Why this page matters

- 旧 Nikko 名の資料・ファンド・販売会社資料と、現行登録名 Amova の **brand transition** を同時に追う。
