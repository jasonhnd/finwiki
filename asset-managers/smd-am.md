# 三井住友DSアセットマネジメント

## Wiki route

This entry sits under [[asset-managers/INDEX|asset-managers INDEX]] (its domain route). Read it against [[asset-managers/asset-management-one|アセットマネジメントOne]] for the other multi-shareholder JV 運用会社 peer / contrast, and against [[financial-regulators/fsa|the FSA]] plus [[securities/financial-instruments-business-operators-japan-index|FSA 金商業者 index]] for the supervisory / registry boundary over 投資運用業 operators.

## TL;DR

三井住友DSアセットマネジメント株式会社は、金融商品取引法上の **投資運用業** 登録を持つ運用会社で、[[megabanks/smfg|SMFG]] を筆頭株主とする **multi-shareholder JV**。FSA 金融商品取引業者一覧では「三井住友DSアセットマネジメント株式会社」、関東財務局長(金商)第399号として掲載される。2019-04 に三井住友アセットマネジメントと大和住銀投信投資顧問の合併で発足し、公募投信・投資一任・機関投資家向け運用を担う。

## 2. Operating model

投資運用業者として、公募投信の **投資信託委託** (組成・運用)、**投資一任** (機関投資家・年金 discretionary)、投資助言を担う SMBC グループの資産運用部門中核。販売は [[megabanks/sumitomo-mitsui-banking-corp|三井住友銀行]] の銀行チャネルと [[securities-firms/smbc-nikko|SMBC日興]] / [[securities-firms/daiwa-sg|大和証券]] の証券チャネルを横断する。**単一メガバンク完全子会社ではなく、銀行 + 証券 + 損保 + 生保の multi-shareholder JV** である点が、MUFG 完全子会社の [[asset-managers/mufg-asset-management|三菱UFJ AM]] や Mizuho + 第一生命 JV の [[asset-managers/asset-management-one|アセマネOne]] と対照される。監督は FSA + 自主規制機関の二層構造。
