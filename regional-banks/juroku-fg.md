# 十六 FG (Juroku Financial Group)

## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[regional-banks/fukuoka-fg|ふくおか FG (FFG)]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

岐阜県地盤の地銀 FG。**2021-10-01 十六銀行の単独株式移転で持株会社化**して発足。中核子銀行は **十六銀行 (1877-10 第十六国立銀行として設立)** で岐阜県内 No.1 シェアを長期維持。社名「十六」は明治期の国立銀行ナンバリング (第十六国立銀行) 由来で、現存メガ・地銀の中でも「番号銀行」名称を残す数少ない例。中京経済圏の**北辺**に位置し、県内二強の [[ogaki-kyoritsu-bank]] (大垣共立銀行) と岐阜県内シェアを分け合う構図。地銀単独 HD 化としては全国的に遅い時期 (2021) の発足。^[extracted]

## 1. 会社概要

**正式名**: 株式会社十六フィナンシャルグループ
**英名**: Juroku Financial Group, Inc.
**証券コード**: 東証 PRIME 7380
**設立**: 2021-10-01 (十六銀行 単独株式移転により持株会社方式に移行) ^[extracted]
**本社**: 岐阜県岐阜市
**主要子会社**: 十六銀行 (中核子銀行) ^[extracted]

### 主要子会社・関連会社

```
十六 FG (持株会社・東証 PRIME 7380)
  ├── 十六銀行 (100%) ── 中核子銀行、1877 設立、岐阜県 No.1 地銀
  │     ├── 岐阜県内営業基盤 (岐阜・大垣・多治見・高山 等全域)
  │     └── 愛知県西部 (一宮・名古屋圏北辺) 進出
  ├── 十六信用保証 ── 個人ローン保証
  ├── 十六リース ── 設備リース
```

### 旧 十六銀行系 (源流)

- **1877-10**: **第十六国立銀行** 設立 (岐阜)、明治期国立銀行条例下のナンバリング 16 番目
- **1896**: 株式会社化 (国立銀行から普通銀行への転換期)
- **1898**: **十六銀行** 改称 (普通銀行転換に伴う社名整理)
- **1949**: 東証上場 (戦後地銀の上場波)

### 持株会社化

- **2021-10-01**: **十六 FG 設立** ─ 十六銀行 単独株式移転により共同持株会社方式に移行 ^[extracted]
  - 単独持株会社化は地銀再編の流れの中では**比較的遅い**時期 (メガ・大手地銀は 2000 年代〜2010 年代に持株会社化完了)
- **2024**: 中期経営計画策定 (詳細は十六 FG 公式 IR 要参照)

### 周辺関係

- 旧 **岐阜銀行** (別法人、経営難の歴史あり) との関係 — **十六銀行が救済・吸収した相手**
  - 岐阜銀行は 1990 年代の経営難で旧東海銀行の親密行となった後、2009-02 に十六銀行へ資本支援を要請。十六銀行が優先株 25 億円を引き受け
  - **2010-12-22**：十六銀行が（三菱東京 UFJ 銀行の協力を得て）岐阜銀行株式を全取得し**完全子会社化**
  - **2012-09-18**：十六銀行が岐阜銀行を**吸収合併**（金融庁が銀行法第 30 条に基づき認可）、岐阜銀行は消滅。岐阜県内地銀構図は「十六 vs 大垣共立」の二強に集約

## 3. 事業セグメント・マップ

| セグメント | 特徴 |
|---|---|
| 県内法人 | 岐阜県内中小企業・地場産業 (繊維・陶磁器・木工・自動車部品) |
| 中京圏法人 | 愛知県西部 (一宮・尾西) + 名古屋圏北辺への融資拡大 |
| 個人 | 岐阜県内個人預金・住宅ローン・資産運用 |
| 観光関連 | 高山・下呂温泉・白川郷等の観光業融資 |
| 製造業 | 自動車部品・電子部品 (中京自動車産業圏の北辺) |
| 信用保証 | 十六信用保証経由の個人ローン保証拡大 |
| リース | 十六リース経由の中小企業設備投資ファイナンス |

### 地理ポジション

- **岐阜県全域**: 岐阜・大垣・多治見・高山等の主要都市で店舗網展開
- **愛知県西部進出**: 一宮・尾西・名古屋圏北辺への営業基盤拡大
- **中京経済圏の北辺**: 名古屋を中心とする中京圏の周辺部に位置し、愛知県地銀勢 ([[mufg]] 旧東海銀行系統 + 名古屋銀行 + 愛知銀行) との競合・棲み分け

### 中京経済圏での競合

- **名古屋銀行・愛知銀行** ([[regional-banks/aichi-fg|aichi-bank]] 等) との愛知県西部での競合

### 産業基盤

- **製造業**: 自動車部品 (中京自動車産業圏の周辺サプライヤー)、電子部品、繊維・陶磁器 (美濃焼)
- **観光**: 高山 (古い町並み)、下呂温泉、白川郷 (世界遺産)、長良川鵜飼 等
- **木工**: 飛騨家具等の地場産業

### TSMC 進出など中部経済圏の半導体特需

- TSMC 熊本工場 (JASM) の波及効果は中京圏全体の半導体エコシステム拡大として認識

## 5. 規制・政策

- **主管**: 金融庁 (FSA) + 東海財務局
- **持株会社規制**: 銀行法 第 52 条の 17
- **直近政策論点**:
  - 2024〜 日銀政策金利正常化 → 地銀収益マージン改善ボーナス
  - 地方創生・人口減少下の地銀再編圧力 (岐阜県人口減少は全国平均並み)
  - 中京経済圏内の地銀広域連携の方向性
  - DX 投資・勘定系刷新の地銀共同利用
