# UI銀行 (UI Bank)


## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[megabanks/saitama-resona-bank|埼玉りそな銀行 (Saitama Resona Bank)]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

UI銀行は [[regional-banks/kiraboshi-fg|東京きらぼし FG]] 100% 出資の digital-bank operating company。2022 年に開業し、対面・非対面のハイブリッド、生活パートナー型金融、API 連携を打ち出す地域金融グループ発のスマホ銀行。

## 2. Business role

- きらぼしグループの DX の中心的役割として開業したスマホ銀行。
- 「対面・非対面のハイブリッド」「デジタルの生活パートナー」「外部 API 連携」をコンセプトにする。
- 地銀グループが既存銀行とは別に digital bank を持つモデルとして、[[banking/minna-bank-baas-model|みんなの銀行]] や [[regional-banks/01-bank|01銀行]] と比較できる。

## 3. Why standalone page matters

UI銀行は東京きらぼし FG の単なるデジタルチャネルではなく、独立した銀行免許を持つ operating company。地銀 DX、スマホ銀行、地域金融グループの全国展開モデルを分析するために分けておく。
