# 株式会社琉球銀行

## Wiki route

This entry sits under [[regional-banks/INDEX|regional-banks INDEX]]. Read it against [[regional-banks/okinawa-fg|おきなわ FG (Okinawa FG)]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

沖縄県内 No.2 地銀（[[okinawa-fg]] 傘下の沖縄銀行に次ぐ規模）。**1948-05-01 米国軍政府主導で設立、当初は琉球列島の中央銀行 + 商業銀行 機能を兼任**という極めて珍しい出自を持つ。1972-05 沖縄本土復帰で普通銀行化、同年東証上場（現 PRIME 8399）。略称「**りゅうぎん**」。沖縄県全域 + 本土主要都市 + 海外駐在事務所体制。観光業・米軍基地経済・離島金融という沖縄特有の事業環境に密着。他のメガバンク・地銀と異なり、**持株会社化していない単独銀行** という構造を維持。

## 1. 会社概要

**正式名**：株式会社琉球銀行
**英名**：The Bank of The Ryukyus, Limited
**証券コード**：東証 PRIME 8399
**設立**：1948-05-01（琉球列島米国民政府布令第 1 号「琉球銀行設立に関する布令」による設立）
**本店**：沖縄県那覇市久茂地 1-11-1
**形態**：単独上場銀行（持株会社化していない、メガバンク・大手地銀の多くが採用する FG 構造を取らず）

### 設立の特殊性

琉球銀行は通常の民間銀行とは大きく異なる出自を持つ：

- **米国軍政府（USCAR / 琉球列島米国民政府）の布令により設立**
- **当初の機能**：商業銀行業務 + **中央銀行業務（通貨発行・銀行監督）** を兼務
- 1948 当時、琉球列島では B 円 → 米軍発行軍票 → 米ドル へと通貨が転換する過渡期で、琉球銀行は事実上「琉球の日銀」役を担った
- 1972-05-15 沖縄本土復帰により、中央銀行機能は日本銀行那覇支店に移管、琉球銀行は普通銀行となる

この歴史的経緯により、沖縄県内における琉球銀行の存在感・象徴性は本土地銀と比較できない独自の重さを持つ。

### 主要子会社・関連会社（概要）

```
琉球銀行（単独上場 8399）
  ├── りゅうぎんビジネスサービス（事務代行・本店内）
  ├── りゅうぎんジェーシービー（クレジットカード、JCB 提携）
  ├── りゅうぎんディーシー（クレジットカード、三菱 UFJ ニコス 系 DC 提携）
  ├── りゅうぎん総合研究所（沖縄経済シンクタンク、県内有数）
  ├── りゅうぎんリース（リース）
  ├── りゅうぎん保証（信用保証）
  └── 関連: 沖縄県内ベンチャー出資・地域 PE ファンド出資 等
```

シンクタンク子会社「**りゅうぎん総合研究所**」は沖縄県内の経済統計・観光統計の主要発信機関の一つ。

### 中期経営計画（直近）

- 2024 開始の中計を遂行中
- 重点：**観光業 V 字回復対応**（コロナ後の観光復活）、**県内 DX 支援**、**離島金融維持**、**サステナビリティ（脱炭素・洋上風力等の沖縄案件）**
- 人的資本：県内雇用維持と本土・アジア人材の双方向流動

### 競合構造（沖縄県内）

| 行 | 親会社 | 位置付け |
|---|---|---|

| **琉球銀行**（8399）| 単独上場 | 県内 No.2、単独銀行を維持 |
| **[[okinawa-kaiho-bank|沖縄海邦銀行]]**（8390）| 単独上場 | 県内 No.3 第二地銀、規模はかなり小さい |

→ **「沖銀・琉銀の県内二強体制」**が長年継続。両行は本店も那覇市内で至近距離。地公体指定金や観光大型案件で住み分け / 競合。第二地銀の海邦銀行は規模差が大きく、二強への挑戦は限定的。

### 沖縄経済との連動

琉球銀行の業績は沖縄県経済の以下要素と強く連動：

1. **観光客数**（インバウンド + 国内）— ホテル・リゾート融資・関連事業者預貸
2. **米軍関連支出**（基地交付金 + 在日米軍人 個人消費）— 県内 GDP の重要構成
3. **公共投資**（沖縄振興予算）— ゼネコン・建設向け融資
4. **離島人口・経済**— 県内離島の人口減・高齢化は構造リスク
5. **沖縄県人口**（本土と異なり、長らく自然増だったが近年は出生率低下傾向）

### 提携・他行連携

- **広域地銀ネットワーク**: 全国地銀協加盟、本土地銀との相互 ATM 提携
- **対アジア**: 香港駐在員事務所、台湾・東南アジア取引銀行とのコルレス
- **メガバンクとの関係**: メガ 3 行（[[mufg]]・[[smfg]]・[[mizuho-fg]]）とは協調・与信補完関係
- **デジタル**: 地銀共同センター系勘定系（NTT データの地銀共同センター利用、※詳細は IR 要確認）

## 4. 規制・政策

- **主管**: 金融庁（FSA）
- **地域監督**: 福岡財務支局 沖縄財務事務所
- **特殊規制**:
  - 沖縄振興特別措置法に紐づく金融優遇制度（沖縄型特定免税店・経済特区関連）
  - 沖縄県信用保証協会との連携
- **国際**: 米軍関係取引・ドル建て口座保持に関する米国制裁法令（OFAC）対応
- **直近政策論点**:
  - 2024〜 日銀利上げ → 預金マージン回復は地銀全体の追い風
  - 沖縄県観光人口 V 字回復 → 観光融資資産質改善
  - 離島人口減 → 店舗統廃合・ATM 維持コスト
  - 沖縄振興予算の中長期動向（本土復帰 50 周年後の振興策の見直し）
