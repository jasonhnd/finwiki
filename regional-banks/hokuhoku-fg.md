# ほくほく FG (Hokuhoku FG)

## Wiki route

This entry sits under [[regional-banks/INDEX|regional-banks INDEX]]. Read it against [[regional-banks/fukuoka-fg|ふくおか FG (FFG)]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

北陸 3 県（富山・石川・福井）+ 北海道の **2 拠点広域地銀 FG**。Hokuhoku FG の corporate data は、2026-03-31 現在、社名を Hokuhoku Financial Group, Inc.、設立日を 2003-09-26、上場市場を東証 Prime / 札証、事業目的を Banking Act 上許容される子会社管理等と説明する。^[source:hokuhoku-overview-2026] グループ表では中核子会社として The Hokuriku Bank と The Hokkaido Bank を並列に置き、両行とも主業を Banking とする。^[source:hokuhoku-group-2026] 北陸銀行と北海道銀行は FSA の銀行免許 workbook でも確認できる。^[source:fsa-ginkou-2026]

## 1. 会社概要

**正式名**：株式会社ほくほくフィナンシャルグループ
**英名**：Hokuhoku Financial Group, Inc.
**証券コード**：東証 PRIME 8377 / 札証 8377
**設立**：2003-09-26（北陸銀行 単独株式移転で「ほくぎん FG」として発足）
**現法人発足**：2004-09-01（北海道銀行と統合 → ほくほく FG 改称）
**本社**：富山県富山市堤町通り 1-2-26。^[source:hokuhoku-overview-2026]
**トップ構成**：President は Hiroshi Nakazawa（President, Hokuriku Bank）、Deputy President は Yuji Kanema（President, Hokkaido Bank）で、二つの中核銀行を前提にした持株会社運営が明示されている。^[source:hokuhoku-overview-2026]

### 主要子会社・持分

```
ほくほく FG（持株会社・上場 8377）
  ├── 北陸銀行（100%）── 富山県本店・北陸 3 県地盤コア
  │     ├── 富山県（本店地盤、県内 main bank ポジション）
  │     ├── 石川県（金沢中心）
  │     ├── 福井県
  │     └── 関東・関西・北海道・東北 等の県外支店も歴史的に展開
  ├── 北海道銀行（100%）── 札幌本店・北海道地盤
  │     └── 北海道全域、札幌都市圏 + 函館・旭川・帯広・釧路 等
  ├── ほくほく TT 証券（北陸銀行 / 北海道銀行 系証券会社）
  └── 各種リース・カード・コンサル子会社（北陸カード / 道銀カード 等）
```

### Dual-bank operating model

- Hokuhoku FG 公式グループ表は、Hokuhoku Financial Group の配下に The Hokuriku Bank と The Hokkaido Bank を並列に掲げ、Hokuriku Bank は Toyama City に本店を置く Banking 子会社、Hokkaido Bank は Sapporo に main branch / head office を置く Banking 子会社として記載する。^[source:hokuhoku-group-2026]
- 公式 corporate governance は、holding company としての group governance を強めるため、主要子会社である Hokuriku Bank と Hokkaido Bank の人材を相互に取締役として任命し、相互理解とチェック・アンド・バランスを促進すると説明する。^[source:hokuhoku-governance-2026]
- したがって本ページでは、単一地銀の広域支店網ではなく、**北陸銀行 + 北海道銀行の二銀行を上場持株会社が統合管理するモデル**として扱う。

### 合併経緯・前身

- 旧 **北陸銀行**：1877 第十二国立銀行 設立（富山）→ 1943 北陸銀行 改称（戦時統合）
- 旧 **北海道銀行**：1951 設立（北海道地盤の戦後新設地銀。明治期の旧北海道銀行=植民地銀行とは別法人）
- **ほくぎん FG**：2003-09-26 北陸銀行 単独株式移転で持株会社化
- **ほくほく FG**：2004-09-01 北海道銀行と統合 → 改称

## 2. 事業セグメント・マップ

| セグメント | 主要事業者 | 特徴 |
|---|---|---|
| 北陸地盤 | 北陸銀行 | 富山県 main bank、石川・福井へ展開 |
| 北海道地盤 | 北海道銀行 | 札幌都市圏中心、道内 [[megabanks/ndfg]]（北洋銀行）と競合 |
| 県外ネットワーク | 北陸銀行（関東・関西・北海道）| 戦後早期から県外展開、東京・大阪に大型店 |
| 証券・カード・リース | ほくほく TT 証券 / 北陸カード / 道銀カード 等 | グループ表は securities、card、leasing、consulting、servicer、venture capital 等の周辺金融会社を列挙する。^[source:hokuhoku-group-2026] |
| 観光・農業金融 | 両行 | 北陸新幹線開業（2015）+ 北海道観光 / 農業 / 水産 |

### 地理ロングテール戦略

- **北陸 3 県（富山・石川・福井）+ 北海道** の 2 拠点広域連合
- 邦銀統合史で「離れた地域の地銀どうしの広域連合」モデルの嚆矢
- 都市地盤：札幌 / 富山 / 金沢 / 函館 / 福井 等の地方都市
- 北陸新幹線（2015-03 金沢開業、2024-03 敦賀延伸）→ 北陸 - 首都圏アクセス改善で観光・ビジネスマネー流入機会

### 競合構造

- **北海道**：[[megabanks/ndfg]]（北洋銀行）との激しい競合。北海道銀行はシェア 2 位ポジション
- **富山県**：県内 main bank ポジション（富山第一銀行 等の第二地銀と差別化）
- **石川県**：北國銀行 / 北國 FH と競合
- **福井県**：福井銀行 / 福邦銀行 と競合

### 観光・農業・水産金融

- 北陸：観光（金沢・能登・立山黒部）+ 製造業（富山アルミ・薬・機械）
- 北海道：観光（インバウンド）+ 農業 + 水産業 + 物流（千歳・苫小牧）
- 両地盤とも一次産業 / 観光ウェイトが高く、地域経済との連動が深い

## 4. 規制・政策

- **主管**：金融庁（FSA）+ 北陸財務局 / 北海道財務局
- **持株会社規制**：銀行法 第 52 条の 17
- **ライセンス境界**：銀行免許 workbook で北陸銀行・北海道銀行を確認。Hokuhoku FG 自体は公式 corporate data 上、子会社管理等を目的とする上場持株会社。^[source:fsa-ginkou-2026][source:hokuhoku-overview-2026]
- **直近政策論点**：
  - 2024〜 日銀政策金利正常化 → 地銀マージン改善（北陸銀行 / 北海道銀行 両行で恩恵）
  - 2025〜 地銀再編加速（人口減 + デジタル投資負担）
  - 北陸新幹線延伸（2024-03 敦賀）→ 福井県地盤の経済効果
  - 2024-01 能登半島地震 → 北陸銀行 石川県・富山県地盤への影響と復興金融
