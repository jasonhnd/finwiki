# regional-banks — Japan regional and second-regional banks index

## Scope

`regional-banks/` is a **standalone top-level domain**, split out of the former `JapanFG` entity umbrella. It covers Japan's **regional banks (地銀) and second-regional banks (第二地銀)**: regional-bank holding companies (地銀 FG / HD), digital / net / retail-platform banks, and prefecture-based operating banks grouped by region. It holds **135 entity pages** (excluding this INDEX), each physically resident in `regional-banks/`.

This page is the index of record for the domain. List the question as **"which regional and second-regional bank entities does FinWiki cover?"** and route from here.

## Disambiguation

- For **thematic banking concepts** (regional-bank consolidation patterns, the regional-bank API / digital-partnership route, net-bank competition map) see [[banking/INDEX|banking]].
- For the **entity-umbrella discovery surface** spanning every Japanese financial group, see [[JapanFG/INDEX|JapanFG]].
- Sibling **bank domains** split from the same umbrella: [[megabanks/INDEX|megabanks]] (3 メガ + FG holding companies) and [[cooperative-banks/INDEX|cooperative-banks]] (信金 / 信組 / JA・JF / 労金). Trust banks live in [[trust-banks/INDEX|trust-banks]].

## Placement Rule

- **Entity pages live in `regional-banks/`** and are linked with `[[regional-banks/slug]]` wikilinks. This INDEX does not duplicate entity content.
