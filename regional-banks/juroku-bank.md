# 十六銀行

## Wiki route

This entry sits under [[regional-banks/INDEX|regional-banks INDEX]] (its domain route). ## TL;DR

株式会社十六銀行 (The Juroku Bank) は、銀行法上の銀行免許を持ち金融庁の銀行免許一覧に掲載される岐阜県岐阜市本店の地方銀行 (第一地銀)。[[regional-banks/juroku-fg|十六フィナンシャルグループ]] の core operating bank であり、FG page とは別に bank-license entity として整理する。事業の中身は岐阜県・中京圏での預金受入と中小企業・個人・地方公共団体向け貸出という地方銀行の標準モデル。

## 2. Operating model

十六銀行は、岐阜県を中心に retail deposits, SME finance, corporate lending, 地方公共団体向け融資, wealth / securities referral, and Chukyo-area corporate support を担う第一地方銀行。[[regional-banks/juroku-fg|十六FG]] page は holding-company strategy を扱い、本ページは banking-license operating boundary を持つ。中京圏では [[regional-banks/aichi-bank|あいち銀行]]、[[regional-banks/nagoya-bank|名古屋銀行]]、[[regional-banks/ogaki-kyoritsu-bank|大垣共立銀行]] と peer boundary を持つ。

## 3. Regulatory & system positioning

- **Supervision**: 銀行法のもとで金融庁 ([[financial-regulators/fsa|FSA]]) と東海財務局が監督。自己資本比率規制（国内基準）等で健全性が規律される。
- **Industry body**: 全国地方銀行協会（地銀協）に属する第一地方銀行。全国銀行資金決済ネットワーク（全銀システム）に参加し、業界団体としては [[financial-regulators/zenginkyo|全国銀行協会（全銀協）]] の枠組みに連なる。
- **Depositor protection**: 預金は預金保険機構（DIC）の付保対象。
- **Consolidation context**: 中部地銀の再編・統合圧力の一般文脈は [[banking/regional-bank-consolidation-pattern|regional-bank consolidation pattern]] を参照。
