# FFG 証券 (FFG Securities)

## Wiki route

This entry sits under the [[securities-firms/INDEX|securities-firms INDEX]] as the FG-subsidiary securities operating-company anchor for [[fukuoka-fg|ふくおか FG (FFG)]]. Read it against parent operating bank [[fukuoka-bank|福岡銀行]] and sibling subsidiaries [[18-shinwa-bank|十八親和銀行]] / [[kumamoto-bank|熊本銀行]]. For peer FG-securities comparison see [[mebuki-securities|めぶき証券]] under [[mebuki-fg]] and the broader [[banking/regional-bank-consolidation-pattern|regional-bank consolidation pattern]] for system / regulatory boundary.

## TL;DR

株式会社 FFG 証券は、[[fukuoka-fg|ふくおか FG (FFG, 東証 PRIME 8354)]] 100% 子会社の地銀系証券会社で、九州地盤 ([[fukuoka-bank|福岡銀行]] / [[18-shinwa-bank|十八親和銀行]] / [[kumamoto-bank|熊本銀行]]) の中堅企業 main bank 顧客向け資産運用・公募株引受・私募債仲介・事業承継 advisory を提供。第一種金融商品取引業 (証券業) 免許で、[[jsda|日本証券業協会]] 加盟。九州地盤の銀証連携モデルの代表例。^[extracted]

### FFG 設立とグループ統合

- **2007-04-02**: [[fukuoka-fg|FFG]] 設立 ([[fukuoka-bank|福岡銀行]] + 熊本ファミリー銀行 共同持株会社化)
- **2007-10**: 親和銀行 を FFG 完全子会社化 (長崎県進出)
- **2020-10-01**: 十八銀行 + 親和銀行 → [[18-shinwa-bank|十八親和銀行]] (長崎県地盤統合完了)
- **2021-05-28**: みんなの銀行 開業 (日本初フルクラウドネイティブ・スマホ完結デジタルバンク)

### FFG 証券 設立・展開

- 1940-06-06 前田良三商店 創業 → 1944-07-07 前田証券株式会社 設立
- 2012-04 福岡銀行が完全子会社化し「ふくおか証券」へ商号変更
- 2018-05 「FFG 証券株式会社」へ商号変更
- **2022-04 福岡銀行 100% 子会社から ふくおかフィナンシャルグループ 直接 100% 子会社へ移管**（FFG グループ唯一の証券機能として再編）
- FFG グループ証券として、九州 3 県地盤 + 親 [[fukuoka-fg]] のデジタル戦略との連動を志向

### FG 内ポジション

- **銀証連携の中核**: [[fukuoka-fg]] 顧客 (個人・中堅企業) の証券窓口
- **資産運用 cross-sell**: 銀行預金 → 投信 / ファンドラップへの顧客移行
- **事業承継・M&A**: 九州中堅企業オーナー向け advisory サービス

### 競合構図

- **大手証券** ([[nomura-hd|野村]] / [[daiwa-sg|大和]] / [[smbc-nikko]] / [[mizuho-securities]]): 都内大手案件では大手が優位、九州地場顧客では銀証連携で対抗
- **ネット証券** ([[sbi-securities]] / [[rakuten-securities]] / [[matsui-sec]]): 個人デジタル取引では劣勢、対面 advisory で差別化
- **地銀系証券**: [[mebuki-securities]] (北関東) / [[okasan-sg]] (中京) 等の同型 model
- **九州競合**: [[nishi-nippon-fhd|西日本シティ銀]] 系・[[kyushu-fg|九州 FG]] 系証券子会社・[[hirogin-hd|広島]] 系証券との地域広域競合

### 強みと弱み

- 強み: [[fukuoka-fg]] 3 県 21.9 兆円預金顧客基盤 (FY2025) + 九州中堅企業 main bank ネットワーク

## 7. 規制・政策

- **主管**: 金融庁 (FSA) / 福岡財務支局 (第一種金融商品取引業者登録)
- **業界団体**: [[jsda|日本証券業協会]] 正会員
- **規制論点**:
  - 銀証ファイアウォール規制緩和 (2022-) で銀証連携範囲拡大
  - 顧客本位の業務運営原則 (FD principle) 遵守
  - 高齢者投資勧誘ルール強化
  - 投信販売手数料の透明化・低水準化圧力
