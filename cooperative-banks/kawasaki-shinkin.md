# 川崎信用金庫

## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[cooperative-banks/yokohama-shinkin|横浜信用金庫]] for peer / contrast context inside the Kanagawa shinkin block and [[banking/shinkin-bank-registry-japan|Japan shinkin bank registry]] for the broader system / regulatory boundary maintained by [[cooperative-banks/shinkin-central|信金中央金庫]].

## TL;DR

川崎信用金庫は、神奈川県川崎市川崎区に本店を置く、川崎市と東京都南部の一部を主たる営業地盤とする地域信用金庫。京浜工業地帯の中小製造業・商店向け融資、住宅ローン、個人預金を中核業務とする協同組織金融機関。

## 2. Operating model

川崎信用金庫は、京浜工業地帯（川崎区・幸区など）の中小製造業・サプライヤー向け SME 融資・運転資金、ならびに川崎北部のベッドタウン地域の住宅ローン・個人預金を中核業務とする。神奈川 shinkin landscape では [[cooperative-banks/yokohama-shinkin|横浜信用金庫]] と地理的分担関係にあり、東京都南西部に営業基盤を持つ [[cooperative-banks/jonan-shinkin|城南信用金庫]] とは境界地域で競合する都市型 shinkin。
