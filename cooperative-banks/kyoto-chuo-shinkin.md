# 京都中央信用金庫

## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[cooperative-banks/osaka-shinkin|大阪信用金庫]] for peer / contrast context inside the Kansai shinkin block and [[banking/shinkin-bank-registry-japan|Japan shinkin bank registry]] for the broader system / regulatory boundary anchored by [[cooperative-banks/shinkin-central|信金中央金庫]].

## TL;DR

京都中央信用金庫（中信）は、京都府京都市下京区に本店を置く、業界 deposits 規模で全国トップクラスの大手信用金庫。京都市・京都府・滋賀県・大阪府の一部を営業地盤とし、京都の中小企業・伝統産業・観光関連事業者を中心に取引する。

## 2. Operating model

京都中央信用金庫は、京都の中小企業金融・住宅ローン・伝統産業（西陣織・京焼など）取引先・観光関連 SME を主たる顧客とする協同組織金融機関。預金規模では信用金庫業界の上位常連で、京都市内において [[regional-banks/kyoto-bank|京都銀行]] と並ぶ存在感を持つ。地元事業承継・M&A コンサル・地域 PE 連携など中信特有の地域密着サービスも展開する。
