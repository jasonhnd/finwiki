# JA 全農 (全国農業協同組合連合会 / National Federation of Agricultural Cooperative Associations)

## Wiki route

This entry sits under [[cooperative-banks/INDEX|cooperative-banks INDEX]] as the **JA Group economic-business federation**, parallel to [[cooperative-banks/norinchukin|農林中央金庫 (信用)]] / [[non-life-insurers/ja-kyosairen|JA 共済連 (共済)]] / [[cooperative-banks/ja-zenchu|JA 全中 (代表)]]. Read with [[banking/ja-bank-system-japan|JA Bank system]] for the credit-side adjacency and [[banking/cooperative-banking-japan|cooperative banking domain]] for the system boundary.

## TL;DR

**JA グループ 4 大全国連の経済事業中核 = 全国規模の購買 + 販売の事業連合会**。1972 設立 (全販連 + 全購連 合併)、農業協同組合法を根拠とする **連合会**。事業内容は **(1) 購買 = 飼料・肥料・農薬・農業機械・燃料・生活物資の調達と供給**、**(2) 販売 = 米・麦・園芸・畜産物の集荷と流通**、**(3) 加工・物流・直営店舗 (A コープ等)**。**取扱高は年間数兆円規模**、日本最大級の農産物流通組織。[[cooperative-banks/norinchukin|農林中金]] は金融、JA 全農は **「モノ」のフロー**を担当する補完関係。全国 JA → JA 全農 → 市場・大手流通 → 消費者 のフロー、および **逆方向の購買フロー (メーカー → JA 全農 → 全国 JA → 農業者)** が二大事業ライン。**自身は預金や保険を扱わない非金融事業体**だが、JA Bank と同じ JA 全国 497 組合を顧客基盤とする。^[extracted]

### 統合史 ^[extracted]

| 年 | 事象 |
|---|---|
| 1948 | **全販連** (全国販売農業協同組合連合会) 設立 |
| 1948 | **全購連** (全国購買農業協同組合連合会) 設立 |
| 1949 | 業界別に各種連合会発足 (米麦・園芸・畜産等) |
| **1972-03** | **全販連 + 全購連 合併で「全国農業協同組合連合会 (JA 全農)」発足** |
| 1992 | 「JA」ブランド統一に伴い **JA 全農** 通称化 |
| 2002〜 | [[banking/ja-bank-system-japan|JA Bank]] 統合稼働、JA 全農と金融は事業として分離維持 |
| 2014 | **規制改革会議で JA 全農の「株式会社化」が政治論点化** |
| 2015〜 | **JA 全農 自己改革** 進行 (流通効率化・コスト削減) |
| 2017 | 米国向け農産物輸出促進 (米輸出社設立等) |

#### 購買事業 (Buying / Supply) ^[extracted]

| 領域 | 内容 |
|---|---|
| **生産資材** | 飼料・肥料・農薬・農業機械・種苗 |
| **燃料** | A コープ系列 SS・JA-SS でガソリン・軽油・LP ガス供給 |
| **生活物資** | A コープ (生協並み店舗) で食品・日用品供給 |
| **その他** | 自動車・住宅資材・農業ハウス資材等 |

#### 販売事業 (Selling / Marketing) ^[extracted]

| 領域 | 内容 |
|---|---|
| **米穀** | 全国 JA からの米集荷、政府入札 + 民間流通 |
| **園芸** | 野菜・果実の集出荷・市場流通 |
| **畜産** | 牛・豚・鶏の集出荷 + 食肉加工 (全農チキンフーズ等) |
| **海外** | 米国子会社 (CGB 等) でカーゴ・船積み事業 |

### 金融補完関係

- **モノのフロー** = JA 全農、**カネのフロー** = [[cooperative-banks/norinchukin|農林中央金庫]] + [[cooperative-banks/ja-shinnoren|信連]] + JA Bank
- 農業者の購買代金・販売代金は **JA 預金口座経由** で決済 → JA Bank の預金基盤を厚くする構造 ^[extracted]
- 大手商社 (三井物産・住友商事・伊藤忠商事等) が JA 全農の競合 + 取引先

### 自己改革と外圧

- **2014 規制改革会議**: 全農「株式会社化」「事業統廃合」「生産資材コスト引下げ」を強く要求 ^[extracted]
- **2015〜 自己改革年次計画**: 飼料・肥料コスト 10% 引下げ、流通効率化、海外輸出強化 ^[extracted]
- 株式会社化は最終的に **見送り** — 連合会形態維持、ただし事業効率化は継続

### 海外展開

- **米国子会社 CGB** (Consolidated Grain and Barge): 米国穀物集荷・物流大手
- **海外輸出促進**: 米・牛肉・青果の海外販売 (米・アジア・EU 市場)

### 競合構造

- **大手商社**: 三井物産・住友商事・伊藤忠商事の生産資材 + 食品流通事業との競合
- **大手食品メーカー**: 米穀流通で大手米卸 (パールライス系・神明等) と競合
- **流通大手**: 直営 A コープは地方スーパー・コンビニ (セブン・イオン) と競合

## 4. 監督・規制

- **主管**: 農林水産省 (連合会監督) + 公正取引委員会 (独占禁止法 — 取扱集中度の論点)
- **法的根拠**: 農業協同組合法 (連合会条項) + 連合会への組合員加入規制
- **直近政策論点**:
  - **生産資材コスト引下げ** (改革会議継続論点)
  - **農産物輸出促進**
  - **食料安全保障計画 (2026〜)** での JA 全農の役割再定義
  - **デジタル農業 (スマート農業)** への対応
