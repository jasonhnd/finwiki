# cooperative-banks — Japan shinkin / shinkumi / JA-JF cooperative finance index

## Scope

`cooperative-banks/` is a **standalone top-level domain**, split out of the former `JapanFG` entity umbrella. It covers Japan's **cooperative finance**: shinkin banks (信用金庫), shinkumi credit cooperatives (信用組合), JA / JF agricultural and fisheries cooperative finance, labour banks (労金 / Rokin), and their central institutions and system federations — including [[cooperative-banks/norinchukin|Norinchukin]] and [[cooperative-banks/shinkin-central|信金中央金庫]]. It holds **45 entity pages** (excluding this INDEX), each physically resident in `cooperative-banks/`.

This page is the index of record for the domain. List the question as **"which cooperative-finance entities does FinWiki cover?"** and route from here.

- For **thematic banking concepts** (cooperative-banking structure, the JA Bank system, the JF Marine Bank system, shinkin / credit-cooperative / rōkin registries) see [[banking/INDEX|banking]].
- For the **entity-umbrella discovery surface** spanning every Japanese financial group, see [[JapanFG/INDEX|JapanFG]].
- Sibling **bank domains** split from the same umbrella: [[megabanks/INDEX|megabanks]] (3 メガ + FG holding companies) and [[regional-banks/INDEX|regional-banks]] (地銀 / 第二地銀 / 地銀 FG / ネット銀). Trust banks live in [[trust-banks/INDEX|trust-banks]].
