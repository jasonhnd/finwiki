# 多摩信用金庫

## TL;DR

多摩信用金庫（たましん）は、東京都立川市に本店を置く、東京都多摩地域（立川・武蔵野・八王子・三鷹・調布など）を主たる営業地盤とする大手地域信用金庫。多摩地区の中小企業・住宅ローン・個人預金を中核とし、ベンチャー支援・産学連携・地域 CRE 金融でも知られる協同組織金融機関。

## 2. Operating model

多摩信用金庫は、多摩地域の中小企業向け SME 融資・住宅ローン・個人預金を中核業務としつつ、ベンチャー支援（多摩信用金庫ビジネスフェアなど）、地元大学（中央大・東京農工大など）との産学連携、地域 CRE/不動産金融に強みを持つ。東京 23 区中心の [[cooperative-banks/jonan-shinkin|城南信用金庫]]・[[cooperative-banks/asahi-shinkin|朝日信用金庫]] と地理的に役割分担する Tama 地区アンカー shinkin。
