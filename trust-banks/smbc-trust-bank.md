# SMBC信託銀行 (SMBC Trust Bank)

## TL;DR

SMBC信託銀行 (SMBC Trust Bank Ltd.) は [[megabanks/sumitomo-mitsui-banking-corp|三井住友銀行]] の 100% 子会社で、[[megabanks/smfg|SMBC / SMFG グループ]] の trust-bank operating company。**銀行法の銀行免許に加え兼営法に基づく信託業務の兼営**を持つ信託兼営金融機関で、PRESTIA ブランドを軸に外貨・富裕層・クロスボーダー個人金融と信託・不動産機能を担う。名称の似た [[trust-banks/smtb|三井住友信託銀行 (SMTB)]] とは別グループ・別法人である点が重要。

## 2. Operating model

信託兼営金融機関として、SMBC信託銀行は次を担う：

- **PRESTIA リテール**：2015 年にシティバンク銀行のリテールバンク事業を統合し、PRESTIA ブランドで外貨預金・富裕層・クロスボーダー個人金融に強い。
- **信託・不動産**：信託業務、不動産関連併営業務。
- **登録金融機関業務**：投資信託・金融商品仲介。

旧外資系信託銀行の系譜を持ち、2013 年に三井住友銀行の子会社となり SMBC信託銀行へ商号変更した経緯がある。[[megabanks/sumitomo-mitsui-banking-corp|三井住友銀行]] が商業銀行中核、[[securities-firms/smbc-nikko|SMBC日興証券]] が証券を担うのに対し、本行は affluent / private-banking 寄りの機能を担う。監督は [[financial-regulators/fsa|金融庁]]。
