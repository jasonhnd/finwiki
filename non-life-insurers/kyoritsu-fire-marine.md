# 共栄火災海上保険

## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[non-life-insurers/sompo|SOMPO ホールディングス (Sompo Holdings)]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

JA 共済連を筆頭株主（**74.20%**）とする中堅損害保険会社。1942-07-01 戦時統合（大東海上 + 大福海上）で創業、1946 年に**損保業界では珍しく相互会社化**、2003-04 株式会社転換。3 大損保（[[tokio-marine]] / [[msad]] / [[sompo]]）には属さず**協同組合・協同組織系列**の独立路線。北海道・東北・北陸・九州で相対的に強いシェアを持ち、JA 共済では扱えない領域（自動車・火災等）を補完する位置付け。2025-04 から新中計「**SHIFT〜持続的成長に向けて〜**」開始。^[extracted]

## 1. 会社概要

**正式名**：共栄火災海上保険株式会社 ^[extracted]
**英名**：The Kyoei Fire and Marine Insurance Company, Limited
**設立**：1942-07-01（大東海上 + 大福海上 合併） ^[extracted]
**本社**：東京都港区新橋 1-18-6（〒105-8604） ^[extracted]
**資本金**：525 億円（2023-03-31 現在） ^[extracted]
**従業員**：2,809 名（2023-03-31 現在） ^[extracted]
**上場区分**：**非上場**（協同組合・協同組織系列が株主）

> 注: 当初ブリーフでは「三井住友海上 約 14%」が主要株主とされていたが、公開情報（Wikipedia「共栄火災海上保険」株主構成）では **MS&AD 系列の株主入りは確認できない**。筆頭株主は JA 共済連 74.20%、次いで農林中央金庫 6.63%・信金中央金庫 6.08% で、いずれも協同組織系。2001 年にミレア保険グループ（現 [[tokio-marine]] HD 前身）参加を一度決定したが、2002-08 に離脱して JA 共済連を筆頭株主とする株式会社化を選択しており、現体制は **JA 共済連・協同組織系列で完結**。

## 2. 事業セグメント・マップ

| セグメント | 主要商品 | 特徴 |
|---|---|---|
| 自動車保険 | 任意保険・自賠責 | 主力。ディーラー・整備工場代理店経由 |
| 火災保険 | 住宅火災・地震保険 | JA 共済補完領域 |
| 傷害保険 | レジャー・スポーツ・交通傷害 | - |
| 海上保険 | 貨物・船舶 | 創業領域 |
| 賠償責任保険 | 個人・法人 PL・施設賠 | - |
| 新種・その他 | 動産総合・利益保険等 | - |
| JA 連携商品 | JA 共済が扱えない領域の損保補完 | **農協顧客向け独自チャネル** |

### 販売チャネル ^[extracted]

- **協同組合・協同組織系代理店**（JA、信金、漁協等）
- **企業・ディーラー・整備工場代理店**
- **共栄プロクラブ**（プロ代理店ネットワーク）
- **直販社員**

### 地域特性 ^[extracted]

**北海道・東北・北陸・九州で加入シェアが相対的に高い**。農協基盤が強い地域と一致しており、JA 共済連を筆頭株主とする構造が販売ネットワークに反映されている。

#### GO TO THE NEXT STAGE（2022〜2024 年度）^[extracted]
- 「お客様にもっと選ばれる会社へ」
- 2024 年度が最終年度

#### **SHIFT〜持続的成長に向けて〜（2025〜）**★ ^[extracted]
- 「SHIFT」= 単なる変化ではなく、環境変化に適応しながら**重心を移していく**
- 重点テーマ:
  1. 協同組合・協同組織諸団体や親交の深い企業・ディーラー・整備工場との**信頼関係強化**
  2. 共栄プロクラブ・直販社員チャネルの強化
  3. **人財投資・システム投資**などの成長投資を持続的に実現できる事業構造の確立
  4. 不確実性の高い時代を生き抜くための組織づくり

## 4. 規制・政策

- **主管**: 金融庁（FSA）
- **業界団体**: 日本損害保険協会
- **規制**: 保険業法、損害保険会社向け監督指針
- **直近政策論点**:
  - 自動車保険料率（損害率上昇 → 料率引上げ局面）
  - 自然災害（台風・豪雨・地震）と再保険コスト
  - JA 共済連系列の協同組合金融制度の議論
  - 損保業界カルテル問題（2023〜2024 大手 4 社）の中堅損保への波及
