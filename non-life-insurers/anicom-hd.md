# アニコム ホールディングス

## Wiki route

This entry sits under [[non-life-insurers/INDEX|non-life-insurers INDEX]] (its domain route). ## TL;DR

アニコム ホールディングス株式会社は、金融庁の **保険持株会社認可一覧**に掲載される pet-insurance group holding company。グループの中核は損害保険業免許を持つ [[non-life-insurers/anicom-insurance|アニコム損害保険]] で、HD はその親会社・group strategy anchor。HD 自身は保険を引き受けず、傘下の損保事業・動物医療データ・周辺サービスの資本/戦略統括を担う。**HD (listed parent) と operating insurer (免許保有主体) は別 entity** として読む。

## 1. License / group boundary

| Item | Notes |
|---|---|
| Legal name | アニコム ホールディングス株式会社 / Anicom Holdings, Inc. |
| License route | 保険持株会社 (保険業法上の認可); FSA insurance holding-company list。2007 年に保険持株会社認可を取得。 |
| Core operating company | [[non-life-insurers/anicom-insurance|アニコム損害保険]] (損害保険業免許, 2007 年取得) |
| Group surface | アニコム損保 + アニコム パフェ + アニコム先進医療研究所 + 周辺事業 |
| Wiki role | Pet-insurance holding-company / group strategy page |

HD は保険業法上の **保険持株会社**であり、損害保険会社そのものではない。実際の引受 (少額短期ではなく正規の損害保険業免許) は子会社のアニコム損保が行う。

## 2. Operating model

The group operates a pet **non-life insurance** franchise: 子会社のアニコム損保がペット保険を引き受け、HD がその上に動物医療データ・先進医療研究・周辺サービスを重ねる。Holding-company analysis should focus on how underwriting data (請求・診療データ) and animal-health services reinforce each other — このデータ循環がモノライン損保各社 ([[non-life-insurers/dai-ichi-ipet-insurance|アイペット損保]]、[[non-life-insurers/pet-and-family-insurance|ペット&ファミリー損保]]) との差別化軸になる。

## 3. Why this page matters

- アニコムの operating-company / holding-company split を完成させ、免許保有主体 ([[non-life-insurers/anicom-insurance|アニコム損保]]) と資本・戦略の HD を分離する。
- FSA 保険持株会社一覧に載る pet-insurance HD の anchor を提供する。
- specialty insurance（ペット・データネットワーク型損保）の分析ノードになる。
