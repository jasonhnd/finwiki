# SOMPO ホールディングス (Sompo Holdings)

## Wiki route

This entry sits under [[non-life-insurers/INDEX|non-life-insurers INDEX]]. Read it against [[non-life-insurers/msad|MS&AD インシュアランス グループ ホールディングス]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

国内 3 大損保の一角（[[non-life-insurers/tokio-marine|東京海上 HD]]・[[non-life-insurers/msad|MS&AD HD]]・SOMPO HD）。**2010-04 旧 [[non-life-insurers/sompo-japan-insurance|損保ジャパン]] + 日本興亜損保 経営統合**で NKSJ HD として発足、2014-09 ソンポ HD → 2016-10 SOMPO HD 改称。**「テーマパーク戦略」**で 損保 + 生保 + **介護（SOMPO ケア = 国内介護最大手）** + デジタル + 海外保険（**Sompo International**, 2017 約 6300 億円で旧 Endurance 買収）を 5 本柱。**2023-2024 ビッグモーター不正修理問題**で損保ジャパンが金融庁から業務改善命令（2024）を受けた。東証 PRIME 8630。^[extracted]

## 1. 会社概要

**正式名**：SOMPO ホールディングス株式会社 ^[extracted]
**英名**：Sompo Holdings, Inc.
**証券コード**：東証 PRIME 8630 ^[extracted]
**設立**：2010-04-01（NKSJ ホールディングス株式会社として）^[extracted]
**改称歴**：NKSJ HD（2010-04）→ ソンポ HD（2014-09）→ **SOMPO HD（2016-10）** ^[extracted]
**本社**：東京都新宿区西新宿（損保ジャパン本社ビル）^[extracted]
**代表**：桜田謙悟（会長 ※2024-03 CEO 退任後）/ 奥村幹夫（SOMPO HD 社長）/ 石川耕治（損保ジャパン社長, 2024 就任）^[extracted]
**業態**：保険持株会社（損保 + 生保 + 介護 + 海外）

### 主要子会社・持分

```
SOMPO HD（持株会社・上場 8630）
  ├── 損害保険ジャパン株式会社（SOMPO 損害保険, 100%）── 国内損保コア
  │     └── 旧 損保ジャパン + 旧 日本興亜損保 2014-09 合併由来
  ├── SOMPO ひまわり生命保険（100%）── 国内生保
  ├── SOMPO ケア株式会社（100%）── 国内介護最大手
  │     └── 旧 ワタミの介護 + 旧 メッセージ 2019 統合
  ├── Sompo International Holdings Ltd.（100%）── バミューダ拠点・特殊保険/再保険
  │     ├── 旧 Endurance Specialty（2017 約 6300 億円買収）
  │     └── 北米・南米・アジア・欧州事業統括
  ├── DC Capital（英ロイズ・シンジケート運営）
  ├── Sompo Insurance (US / Canada / アジア / 欧州 各拠点)
  ├── Sompo Sigorta（トルコ損保）
  └── Sompo Brasil（ブラジル損保）
```

### 合併歴史・前身

- 旧 **安田火災海上保険**：旧安田財閥系（戦前 共済生命系）

- 旧 **大成火災海上保険**：2001 経営破綻 → 損保ジャパンに統合（2002）
- **損保ジャパン**：2002-07 安田火災 + 日産火災 + 大成火災 合併発足 ^[extracted]
- 旧 **日本興亜損害保険**：2001-04 日本火災海上 + 興亜火災海上 合併（2001-04 設立）^[extracted]
- **NKSJ HD**：2010-04-01 損保ジャパン + 日本興亜損保 共同持株会社方式で経営統合 ^[extracted]
- **損害保険ジャパン日本興亜**：2014-09-01 損保ジャパン + 日本興亜損保 合併・改称（持株 NKSJ HD → ソンポ HD と同時改称）^[extracted]
- **損害保険ジャパン（再改称）**：2020-04-01 「損害保険ジャパン日本興亜」→「損害保険ジャパン」に再改称 ^[extracted]

| セグメント | 主要事業者 | 特徴 |

### テーマパーク戦略

- SOMPO 経営の代名詞。**損保 + 生保 + 介護 + デジタル + 海外保険** を「保険を超えた安心・安全・健康のテーマパーク」として位置付ける ^[extracted]
- 桜田謙悟（会長）が長年主導してきたコンセプト
- 介護（SOMPO ケア）を非保険セグメントの戦略中核に据える点は国内 3 大損保で最も明確

### 介護事業

- **SOMPO ケア**：旧ワタミの介護（2015 ワタミから 210 億円で買収）+ 旧メッセージ（2016 約 270 億円で TOB）統合 ^[extracted]
- 2019 SOMPO ケアとして一本化 → 居室数で国内介護最大手規模

### 海外戦略

- **Sompo International**（旧 Endurance Specialty）：2017-03 約 6300 億円買収 → バミューダ系・特殊保険 + 再保険ハブ ^[extracted]
- **北米中心**：米国損保 + バミューダ再保険
- **新興国**：Sompo Sigorta（トルコ）/ Sompo Brasil / アジア拠点で小口分散
- **DC Capital**：英ロイズ・シンジケート運営

### デジタル

- **SOMPO Digital Lab**（東京・シリコンバレー・テルアビブ）
- **Palantir 戦略提携**（2020）：SOMPO HD が Palantir Technologies に約 5 億米ドル（約 540 億円）出資。SOMPO HD・Palantir・Palantir Japan の 3 者で Palantir Foundry を基盤とする「安心・安全・健康のリアルデータプラットフォーム (RDP)」を立ち上げ、介護・健康分野を起点にグループ内外データを統合分析
- 介護データ + 損保データの統合プラットフォーム構想

### ビッグモーター事件以降のガバナンス改善 (FY2024 進捗) ^[extracted]

- **2024-01-26**：損保ジャパン 白川儀一 社長 辞任
- **2024-03**：SOMPO HD **桜田謙悟 CEO 退任**（会長職へ）→ 経営トップ刷新
- **後任**：石川耕治 損保ジャパン社長就任 / 奥村幹夫 SOMPO HD 社長体制
- **2024 業務改善命令**（金融庁・損保ジャパン宛て）→ 内部統制報告・経営責任明確化
- **FY2024 内部統制評価：「有効」**（再建後初の総合評価で改善を公式確認）
- 取締役会改革・社外取締役比率引き上げ
- 中古車修理工場依存型営業モデルの見直し（特定整備工場リスク管理強化）
- DRS Group（車両修理プラットフォーム）/ Zego（英 insurtech）等との 2026 戦略提携 → ガバナンス再建後の海外・デジタル攻勢へ転換 ^[extracted]
- 3 大損保カルテル疑惑（企業向け保険料事前調整、独占禁止法違反）も別途公正取引委員会調査 → 業界全体で構造改革圧力継続 ^[extracted]

## 4. 規制・政策

- **主管**：金融庁（FSA）
- **持株会社規制**：保険業法（保険持株会社）
- **国内損保業界規制**：保険業法、独占禁止法（カルテル疑惑関連）
- **海外規制**：
  - バミューダ金融庁（BMA, Sompo International）
  - 米州 各州保険監督庁
  - 英 PRA / FCA（DC Capital / ロイズ関連）
  - トルコ SEDDK（Sompo Sigorta）
- **直近政策論点**:
  - **2024 業務改善命令**（損保ジャパン宛て、ビッグモーター案件）→ 内部統制報告・経営責任明確化
  - **独占禁止法違反疑い**（3 大損保カルテル疑惑、企業向け保険料調整）→ 公取委調査
  - 政策保有株式削減（金融庁 株式持合い解消方針）
  - 自然災害多発化 → 火災保険料率改定圧力（損害保険料率算出機構経由）
