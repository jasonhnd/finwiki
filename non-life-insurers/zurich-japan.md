# チューリッヒ保険会社 (Zurich Japan)

## Wiki route

This entry sits under [[non-life-insurers/INDEX|non-life-insurers INDEX]]. Read it against [[life-insurers/axa-japan|アクサ・ジャパン (AXA Japan)]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

スイス Zurich Insurance Group の日本拠点。**1986 年 日本支店開設** で本邦進出した外資系損保のパイオニアの一社。日本では「**海外旅行保険**」と「**ネット自動車保険（ダイレクト型）**」の 2 大プロダクトで認知を獲得し、2010 年には **チューリッヒ生命** を別法人として設立、医療保険・がん保険のネット直販へ事業領域を拡張。営業店舗を持たず、コールセンター + ウェブを中核とした **直販モデル** が一貫した特徴で、国内ダイレクト系損保では [[megabanks/sony-fg]] のソニー損保、[[life-insurers/axa-japan]] のアクサ損保、セコム損保、[[megabanks/sbi-hd]] の SBI 損保 と並ぶ主要プレイヤー。

## 1. グループ構造

**最終親会社**：Zurich Insurance Group AG（スイス Zurich、SIX Swiss Exchange: ZURN、SMI 構成銘柄）
**日本における主要法人**：
- **チューリッヒ保険会社 日本支店**（損害保険、外国保険会社の日本支店形態）
- **チューリッヒ生命保険株式会社**（生命保険、日本法人）

### 主要法人ストラクチャ

```
Zurich Insurance Group AG（スイス、SIX: ZURN）
  └── 日本における主要拠点
        ├── チューリッヒ保険会社 日本支店（外国保険会社支店）
        │     ├── 海外旅行保険 ── 1980〜90 年代に旅行代理店経由で確立
        │     ├── ネット自動車保険（個人用自動車）
        │     ├── ペット保険・傷害保険等の個人向け損保
        │     └── 一部法人向け損保（駐在員保険・賠償責任等）
        └── チューリッヒ生命保険株式会社（日本法人、2010 設立）
              ├── 医療保険（終身医療・引受緩和型を含む）
              ├── がん保険
              └── 定期・収入保障保険
```

### 重要历史（年表）

| 年月 | 事象 |
|---|---|
| 1872 | スイスで **Zurich Insurance** 創業（海運保険から出発）|
| **1986** | **チューリッヒ 日本支店 開設**（外資系損保として本邦進出）|
| 1990 年代 | **海外旅行保険** でブランド確立（旅行代理店経由 + テレマで個人客にリーチ）|
| 2000 年代前半 | **ネット自動車保険（ダイレクト型）参入** ─ ソニー損保・アクサ損保等と「ダイレクト損保」セグメント形成 |
| 2002 頃 | 海外旅行保険でテレマーケティング × ウェブ申込のチャネルミックス確立 |
| 2000 年代後半〜 | 自動車保険のリスク細分化（走行距離・年齢・地域別料率）でダイレクト型の競争力強化 |
| **2010** | **チューリッヒ・ライフ・インシュアランス・カンパニー・リミテッド 日本支店** から「**チューリッヒ生命保険株式会社**」設立（日本法人化）── 医療・がん・収入保障のネット直販に本格参入 |
| 2010 年代 | 引受緩和型医療保険（高齢者・既往症あり層向け）でラインアップ拡張 |
| 2020 年代 | グループ Zurich Insurance Group の中期戦略（顧客中心主義・デジタル変革）下で日本も UX 改善・契約管理デジタル化を推進 |

## 2. 業務ライン

| セグメント | 主要事業者 | 特徴 |
|---|---|---|
| 海外旅行保険 | チューリッヒ保険 日本支店 | 旅行代理店経由 + ネット直販。日本拠点の創業期からの主力プロダクト |
| ネット自動車保険 | チューリッヒ保険 日本支店 | 個人用自動車保険のダイレクト型。リスク細分化型料率 |
| ペット保険 | チューリッヒ保険 日本支店 | 個人向けニッチ損保ラインの一つ |
| 傷害・賠償責任 | チューリッヒ保険 日本支店 | 個人賠償・スポーツ傷害等 |
| 医療保険 | チューリッヒ生命 | 終身医療・引受緩和型医療。ネット直販主体 |
| がん保険 | チューリッヒ生命 | 一時金・診断給付特化型 |
| 収入保障・定期 | チューリッヒ生命 | 比較的低価格帯の保障性商品 |

### ポジショニング

- **「直販一本足ではない」**：完全ネット専業の [[life-insurers/lifenet]]（ライフネット生命）と異なり、海外旅行保険では旅行代理店チャネルも活用するハイブリッド型
- **生損保兼営型外資**：[[life-insurers/axa-japan]] が「生損保 + ネット直販生保」の 3 法人体制で営業職員チャネルも持つのに対し、Zurich は **対面営業職員チャネルを持たず、直販モデルに一貫**

### グローバルレバレッジ

- **Zurich Insurance Group の再保険プラットフォーム**：大型契約・自然災害リスクをグループ内再保険で吸収
- **グローバル海外旅行保険ネットワーク**：日本人の海外渡航時の現地救援・医療搬送ネットワークを Zurich Assistance（グループ機能）で提供

## 4. 規制と政策

- **主管**：金融庁（FSA）。
  - **チューリッヒ保険会社 日本支店**：保険業法 第 185 条（外国保険業者の日本支店免許）ベース
  - **チューリッヒ生命保険株式会社**：保険業法上の生命保険会社（日本法人）として通常の免許
- **ソルベンシー**：日本支店・日本生命保険会社それぞれ、日本のソルベンシー・マージン比率規制が適用。**2025〜 経済価値ベース ソルベンシー規制（ESR）導入** への対応は他生損保と同じ論点
- **直近政策論点**：
  - **自動車保険の自動運転・先進安全装備対応**：リスク料率体系の再設計、テレマティクス保険の展開度合い
  - **ネット直販生保の説明責任**：オンライン申込みフローでの告知義務・募集規制の遵守
