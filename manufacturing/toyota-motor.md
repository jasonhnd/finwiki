# Toyota Motor Corporation (トヨタ自動車)

## TL;DR

トヨタ自動車は 1937-08-28 設立の日本最大級自動車メーカーで、[[JapanFG/toyota-financial|トヨタファイナンシャルサービス]] の親会社。FinWiki では「製造業本体が金融子会社・販売金融・残価リスク・EV/ハイブリッド戦略をどう支えるか」を見るための manufacturing anchor として使う。

## 1. 公司结构

| 項目 | 内容 |
|---|---|
| 正式名 | Toyota Motor Corporation / トヨタ自動車株式会社 |
| 設立 | 1937年8月28日 |
| 本社 | 1 Toyota-Cho, Toyota City, Aichi Prefecture 471-8571, Japan |
| 資本金 | 6,350億円（2025-03-31） |
| 従業員 | 単体 71,515人、連結 383,853人（2025-03-31） |
| 主業 | Motor vehicle production and sales |
| 金融子会社 | [[JapanFG/toyota-financial|Toyota Financial Services]] |

Toyota は自動車製造・販売を中核に、金融、モビリティ、部品、物流、販売網を含む巨大グループを形成する。金融子会社は販売金融・リース・保険・クレジットカードを担い、車両販売と残価管理を一体化する。

## 2. 业务线 map

| 业务线 | 内容 | 金融・地域金融との接点 |
|---|---|---|
| 自動車製造 | Toyota / Lexus / 関連ブランド車両 | 設備投資、部品サプライヤー金融 |
| 販売 | 国内外販売網、ディーラー | ディーラー金融、在庫金融 |
| ハイブリッド / EV | HEV, PHEV, BEV, FCEV の多経路戦略 | 残価設定・充電インフラ金融 |
| 金融 | [[JapanFG/toyota-financial|TFS]] | ローン、リース、保険、カード |
| グローバル生産 | 北米、欧州、アジア等 | 為替、貿易金融、サプライチェーン |

FY2025 Integrated Report では、FY2025 の総販売台数は 9,362,410 台、売上収益は 48,036.7 billion yen とされる。Toyota Financial Services は 35 以上の国・地域をカバーする。

## 3. 战略与竞争

- **多経路電動化**: Toyota は BEV 一本足ではなく、HEV / PHEV / BEV / FCEV を組み合わせる。金融面では残価設定と中古車価格見通しが重要。
- **販売金融との一体化**: [[JapanFG/toyota-financial|toyota-financial]] は車両販売、リース、保険、ディーラー網を接続し、製造業本体の販売サイクルを支える。
- **サプライチェーン金融**: トヨタ系部品会社、地域サプライヤー、設備投資は [[JapanFG/mufg|mufg]]、[[JapanFG/smfg|smfg]]、[[JapanFG/mizuho-fg|mizuho-fg]]、地銀の法人金融と接続する。
- **為替・通商リスク**: グローバル生産・輸出企業として、円安/円高、関税、米中・米日通商政策の影響が大きい。

## 4. 监管与政策

- 上場会社として東証・金商法・会社法の開示対象。
- 自動車安全・環境規制、CO2 排出規制、各国 EV 補助金・関税政策の影響を受ける。
- 金融子会社は各国金融監督、消費者信用、リース、保険販売規制の対象。

## Counterpoints

- BEV 移行が急加速した場合、Toyota の多経路戦略は強みではなく遅れと評価される可能性がある。
- 残価設定型ローン・リースは中古車価格と電池技術更新に左右され、金融子会社側のリスクになる。
- 巨大サプライチェーンは効率性が高い一方、品質不正・部品供給停止・地政学リスクの伝播も大きい。

## Open questions
