# 日本の株主提案と 株主総会 議決権行使ルート

## 概要

日本の株主提案と 株主総会 議決権行使記録は、会社法上の株主権、招集通知、参考書類、書面投票、電子投票、コーポレートガバナンス開示、委任状投票、スチュワードシップ開示、総会後の議決権行使結果を接続する。このページは、上場会社のガバナンス・イベントで参照される公開記録項目を整理する。

このページは [[finance/INDEX|finance domain]] に属し、[[finance/japan-activist-investor-playbook|Japan activist investor playbook]]、[[finance/japan-large-shareholding-disclosure|large-shareholding disclosure]]、[[finance/japan-fair-disclosure-and-insider-trading-controls|fair disclosure and insider trading controls]]、[[finance/japan-mbo-and-squeeze-out-process|Japan MBO and squeeze-out process]]、[[finance/japan-tender-offer-process|Japan tender offer process]]、[[securities/tokyo-stock-exchange|Tokyo Stock Exchange]]、[[financial-regulators/japan-exchange-group|Japan Exchange Group]] と接続する。

## ルートマップ

| レイヤー | 公開ソースのルート | 公開記録項目 |
|---|---|---|
| 株主総会決議 | 会社法 298  条 | 日時、場所、会議の目的、書面投票、電子投票。 |
| 招集通知 | 会社法 299  条 | 通知期間と書面通知ルート。 |
| 書面投票書類 | 会社法 301  条 | 参考書類と議決権行使書面。 |
| 電子投票書類 | 会社法 302  条 | 参考書類と電子投票フォーム情報。 |
| 株主提案権 | 会社法 303-305  条 | 議題追加、総会当日の提案、通知への提案要旨掲載。 |
| 電子投票の効力 | 会社法 312  条 | 電子的に行使された議決権は出席株主の議決権として算入。 |
| 上場会社ガバナンス | JPX コーポレートガバナンス code / レポrts | 株主総会 円滑化とガバナンス開示項目。 |
| 議決結果 | 発行体開示 / TDnet / ガバナンス資料 | 賛成率、議案結果、取締役選任結果。 |

## 会社法上の項目

| 条文 | 公開項目 |
|---|---|
| 298  条 | 取締役が総会の日時・場所、会議の目的事項、書面または電子投票を認めるかを決定する。 |
| 299  条 | 株主総会の招集通知は原則として総会日の二週間前までに発送する。 |
| 299  条 | 非公開会社の一定の場合には一週間前通知ルートが適用される。 |
| 299  条 | 書面 / 電子投票を採用する場合、または取締役会設置会社では書面通知が必要。 |
| 301  条 | 書面投票を採用する場合、参考書類と議決権行使書面を招集通知に同封する。 |
| 302  条 | 電子投票を採用する場合、参考書類を招集通知とともに送付し、議決権行使書面情報を電子的に提供する。 |
| 303  条 | 取締役会設置会社では、1% の議決権または 300  議決権を六か月以上保有する株主が、目的事項への追加を請求できる。 |
| 303  条 | 取締役会設置会社での請求期限は、総会日の八週間前まで。 |
| 304  条 | 株主は、法定制限の範囲で、すでに会議の目的に含まれる事項について総会で提案できる。 |
| 305  条 | 要件を満たす株主は、招集資料に提案要旨を掲載するよう請求できる。 |
| 305  条 | 取締役会設置会社での提案要旨請求にも、同じ 1% / 300  議決権、六か月保有、八週間前期限が適用される。 |
| 312  条 | 会社の承認および政令 / 法務省令ルートにより電子投票が可能で、電子的に行使された議決権は出席株主の議決権として扱われる。 |

## 株主総会 資料ルート

| 公開資料 | 記録項目 |
|---|---|
| 招集通知 | 開催日、場所 / 形式、議題、議決権行使期限、議決権行使方法。 |
| 参考書類 | 会社提案、株主提案、取締役会意見、取締役 / 監査役候補者情報。 |
| 議決権行使書面 | 書面投票ルートと議案ごとの選択。 |
| 電子投票ページ | 電子投票方法、認証ルート、議決権行使期限。 |
| コーポレートガバナンス報告書 | 取締役会構成、委員会構成、独立取締役、ガバナンスコード上の説明。 |
| 議決結果開示 | 賛成率、議案別の議決結果、取締役選任結果、株主提案の結果。 |

## JPX ガバナンス報告書項目

JPX の適時開示ガイドブック資料は、上場会社の 株主総会 / 議決権行使円滑化項目をコーポレートガバナンス報告書ルートで特定している。ソースパックでは、次の公開開示項目が記録されている。

| JPX ガバナンス報告書項目 | 公開項目 |
|---|---|
| 招集通知の早期掲載 | 招集通知公表の時期と方法。 |
| 集中日を避けた開催 | 株主総会 日程集中 / scheduling 項目。 |
| 議決権の電子行使 | 電子投票採用項目。 |
| 機関投資家向け議決権行使プラットフォーム | ICJ などの機関投資家向けプラットフォームへの参加。 |
| 英文招集通知 / 要約 | 英語版通知または要約項目。 |
| その他の円滑化措置 | 発行体が開示する追加的な 株主総会 円滑化項目。 |
