# 日本のクロスボーダーM&A

## 要約

日本のクロスボーダーM&Aには、外国企業・投資家による日本企業への取得・投資であるインバウンド、日本企業による海外企業の取得であるアウトバウンド、そしてジョイントベンチャー・戦略提携という三つの方向がある。これらを一つの市場として扱ってはいけない。それぞれ、取締役会、資金調達、規制、外国為替、独禁法、労務、統合の論点が異なる。

FinWiki では、このページは [[securities-firms/goldman-sachs-japan]]、日本のメガバンク系証券会社、[[finance/japan-ib-league-table]] の背後にある取引文脈レイヤーである。

## 方向性マップ

| 方向 | 意味 | 主要な問い |
|---|---|---|
| インバウンド / OUT-IN | 外国企業または外国PEが日本企業に投資する、または取得する。 | 外国資本は成長、ガバナンス、事業承継、再編の問題を解くものか。 |
| アウトバウンド / IN-OUT | 日本企業が海外事業を取得する。 | 日本側買収者は経営、人材、コンプライアンス、海外成長を統合できるか。 |
| クロスボーダーJV / 提携 | 日本企業と外国企業が完全買収なしに能力を組み合わせる。 | 支配関係は十分に明確で、インセンティブは持続可能か。 |

## 規制マップ

| レイヤー | 当局 / 情報源 | 確認事項 |
|---|---|---|
| 対内直接投資審査 | FEFTA に基づく財務省および事業所管省庁 | 対内直接投資とセンシティブ業種に関する事前届出 / 審査。 |
| 企業結合 / 独禁法 | 公正取引委員会 | 取引が競争を実質的に制限し得るか、どの届出 / 審査手続が適用されるか。 |
| 証券開示 | FSA / EDINET / TDnet / 取引所規則 | TOB、大量保有、上場会社開示、資金調達開示。 |
| 業種別規制 | 事業所管省庁または規制当局 | 銀行、保険、通信、防衛、エネルギー、医療、データ、インフラの感応度。 |
| 資金調達 | 銀行 / 証券会社 | 買収ファイナンス、ブリッジローン、社債発行、エクイティ・オファリング、ヘッジ。 |

## METI のインバウンドM&A読解

METI の 2023  インバウンドM&A事例研究プロジェクトは、外国資本を、日本企業が経営課題を解決し成長を加速するための選択肢の一つとして位置づけている。有用な読み方は「外国買収者は常に良い」ではなく、次の点である。

- 外国資本は、グローバル販路、経営ノウハウ、技術、成長投資をもたらし得る。
- 日本側売り手にはなお、経済安全保障、ガバナンス、従業員、統合に関する保護策が必要である。
- インバウンドM&Aは、窮境売却だけでなく、事業承継や企業改革の戦略的選択肢にもなり得る。

## JapanFG との関係

- [[securities-firms/goldman-sachs-japan]] やその他の外資系IBは、クロスボーダー・アドバイザリーで強く競争することが多い。
- [[securities-firms/nomura-hd]]、[[securities-firms/daiwa-sg]]、[[securities-firms/smbc-nikko]]、[[securities-firms/mizuho-securities]]、[[securities-firms/mufg-mums]] は、国内発行体へのアクセス、資金調達関係、執行支援を通じて競争する。
- 買収が資源安全保障、インフラ、国家産業政策を支える場合、[[financial-regulators/jbic]]、[[policy-finance/INDEX|policy-finance INDEX]]、輸出信用 / 政策金融ページが関係する。

## デューデリジェンス・チェックリスト

1. 取引方向: インバウンド、アウトバウンド、またはJV。
2. 買収者タイプ: 事業会社、PE、ソブリン、銀行支援、経営陣、またはコンソーシアム。
3. 支配: 少数持分、過半取得、完全買収、TOB、またはカーブアウト。
4. 規制審査: FEFTA、JFTC、業種別承認、取引所規則。
5. 資金調達: 現金、借入、ブリッジ、エクイティ、ハイブリッド、売主ファイナンス。
6. 統合: ガバナンス、言語、人事、IT、コンプライアンス、顧客契約。
7. 通貨: 買収価格、ヘッジ、利益換算、債務通貨。
