# 日本のプライベートエクイティ・ファンド構造マトリクス

## 要約

日本のプライベートキャピタル領域は単一の資産クラスではない。少なくともバイアウト、グロース、セカンダリー、継続、インフラ、プライベートクレジットという6つの異なるサブ戦略と、グローバル大手 PE の日本チーム、国内独立系 GP、政策連動 / 国家関連 GP という3つの GP 類型があり、それぞれ異なるファンド構造を運用している。このマトリクスは、**ファンド種別、対象会社プロファイル（規模 / セクター）、レバレッジ前提、保有期間、キャリーと管理報酬の経済性（通常 1/20, 1.5/20, または 2/20）、LP 構成（ソブリン、年金、保険会社、ファミリーオフィス）、ビークル法域（Japan investment LPS、Cayman ELP、Delaware LP）**を横並びで比較するための面である。これはルートマップであり、投資助言、推奨、資金募集の勧誘、GP の推奨ではない。

## このマトリクスが重要な理由

「Japan PE」に関する外部コメントの多くは、グローバル大手 PE の日本チーム（KKR Japan、Bain Japan、Carlyle Japan、EQT Japan、CVC Japan）と、国内独立系（Advantage Partners、J-STAR、Integral、Polaris）、さらに 国家関連 / ソブリン隣接ビークル（JIC Capital、INCJ / JIC、DBJ / DBJC）を一括りにしている。これらは同じビジネスモデルではない。

実務家には、次の点を横並びで見えるようにする必要がある。

- **ファンド種別**（バイアウト / グロース / セカンダリー / 継続 / インフラ / プライベートクレジット）;
- 企業価値 帯とセクター別の**対象プロファイル**;
- **レバレッジ前提**（低レバレッジの グロース・エクイティ、5-7x EBITDA の バイアウト、無レバレッジ インフラ の違い）;
- **保有期間**（グロース は3-5 年、バイアウト は4-7 年、インフラ は7-10+ 年）;
- **フィー / キャリー配分**（クレジット は1/20 、新興マネージャー は1.5/20 、バイアウト標準 は2/20 ）;
- **LP 構成**（ソブリン / 年金 / 保険会社 / ファミリーオフィス / ファンド・オブ・ファンズ）;
- **ビークル法域**（Japan investment LPS / 投資事業有限責任組合、Cayman exempted LP、Delaware LP、場合により並行構造）。

このマトリクスは、そうした問いに対する比較面である。個別ファンドの仕組みは、GP の公開ウェブサイト開示と LP の年次報告書に現れる。

## ファンド種別の分類

| サブ戦略 | 短い定義 |
|---|---|
| **バイアウト** | 成熟企業の過半 / 100%支配権を取得し、レバレッジを用いる。 |
| **グロース・エクイティ** | 成長段階企業への少数持分 / 共同支配投資。レバレッジは限定的。 |
| **セカンダリー** | 既存 LP から LP 持分またはポートフォリオを購入する。 |
| **継続ファンド** | GP 主導セカンダリー。既存 GP が1社以上のポートフォリオ企業を新ビークルに移す。 |
| **インフラ** | 長期保有でキャッシュ利回りを生む資産（公益事業、再生可能エネルギー、交通、デジタルインフラ）。 |
| **プライベートクレジット** | 直接貸付、メザニン、ディストレスト、資産担保金融。 |

バイアウト IRR 分解 は[[finance/japan-leveraged-buyout-economics]]、オペレーション改善レイヤー は[[finance/japan-private-equity-operating-model]]とあわせて読む。

## ファンド種別ごとの構造

### バイアウト

| 次元 | 典型的な読み方 |
|---|---|
| 対象会社プロファイル | 成熟企業。典型的な EBITDA は¥1-50bn帯、EV は¥10-500bn帯。セクター横断だが ヘルスケア、インダストリアル、ソフトウェア、消費者向け に集中しやすい。 |
| Sector | インダストリアル、ヘルスケア、ソフトウェア / IT サービス、消費者向け、金融サービス。 |
| レバレッジ前提 | 5-7x EBITDA が典型。安定キャッシュフロー事業ではより高い場合もある。LBO loan は[[JapanFG/mufg-bank]]、[[JapanFG/mizuho-bank]]、[[JapanFG/sumitomo-mitsui-banking-corp|SMBC]]、[[JapanFG/smtb|SMTB]]で club される。 |
| 保有期間 | 4-7 年 が典型（グローバル大手 PE の日本チーム では3-5 年、国内独立系では5-8 年 の例もある）。 |
| フィー / キャリー | 2% / 20%（グローバル大手 PE の日本チーム の標準）、1.5% / 20%（一部の newer / 国内）。8% ハードル / 優先リターン 超。国内では 欧州型ウォーターフォール、global では 案件別 が多い。 |
| LP 構成 | global 年金（CalPERS、CPP、OTPP、GPIF（一部））、ソブリン・ウェルス（ADIA、GIC、KIA）、日本の megabank 財務部門、生命保険会社、foundation、ファミリーオフィス、ファンド・オブ・ファンズ。 |
| ビークル法域 | グローバル大手 PE の日本チーム: Cayman ELP を主ビークルとし、非課税 LP 向けに Japan parallel を置く。国内独立系: Japan investment LPS（投資事業有限責任組合）を主ビークルとし、場合により Cayman parallel を置く。 |
| 出口ルート | strategic への 事業会社への売却、スポンサー間売却 セカンダリー、TSE Prime / Growth での IPO、配当リキャップ。 |
| 案件ソーシング | TSE からの非公開化、上場親会社からのカーブアウト、事業承継 / ファミリービジネス、スポンサー間売却。 |

非公開化メカニクス は[[finance/japan-mbo-and-squeeze-out-process]]、IRR 分解 は[[finance/japan-leveraged-buyout-economics]]、上場対象会社ユニバース は[[finance/japan-listed-financial-groups-investable-universe]]とあわせて読む。

### グロース・エクイティ

| 次元 | 典型的な読み方 |
|---|---|
| 対象会社プロファイル | 高成長で、しばしば 黒字化前 または 黒字化初期。EV は¥5-50bn帯が典型。tech、ヘルスケア、消費者向け brand。 |
| Sector | Software / SaaS, フィンテック, ヘルスケア, D2C 消費者向け, 気候 / エネルギー転換. |
| レバレッジ前提 | 低い、またはゼロ。グロース・エクイティ は主に エクイティ資金中心 である。 |
| 保有期間 | 3-5 年 が典型（純粋な グロース 投資ではより長い場合もある）。 |
| フィー / キャリー | 2% / 20% が標準。小規模 / 初期ファンドでは2.5% / 20%の場合もある。欧州型ウォーターフォール。 |
| LP 構成 | ソブリン・ウェルス、年金、保険会社、ファンド・オブ・ファンズ、ファミリーオフィス。投資チケット が小さいため mega-年金 は相対的に少ない。 |
| ビークル法域 | 国内向けは Japan investment LPS が主。クロスボーダーでは Cayman ELP。 |
| 出口ルート | TSE Growth / Prime での IPO、strategic への 事業会社への売却、スポンサー間売却。 |
| 案件ソーシング | 創業者 / 経営陣との接点、VC からの卒業、初期段階 VC からの セカンダリー、創業者-led カーブアウト。 |

国内グロース GP には Polaris Capital Group（旧 Polaris Holdings）、Integral Corporation、NSSK、JIC Capital（Japan Investment Corporation 傘下）が含まれる。いずれも公開ウェブサイト開示を行っている。

### セカンダリー

| 次元 | 典型的な読み方 |
|---|---|
| 対象 | PE / VC ファンドの既存 LP 持分、セカンダリー・ポジションのポートフォリオ、GP 主導再編。 |
| セクター | セクター横断。セカンダリー買い手は裏付けポートフォリオ構成を引受審査する。 |
| レバレッジ前提 | ファンドレベルの信用枠（NAV ベースまたはサブスクリプションライン）。ポートフォリオ企業のレバレッジは裏付けファンドを引き継ぐ。 |
| 保有期間 | 2-5 年が典型。プライマリーより残存ファンド期間が短いため短めになる。 |
| フィー / キャリー | 1% / 10%（一部）、1.25% / 12.5%、または 1.5% / 15%。期待リターンと期間が低め / 短めであるため、通常はバイアウトより低い。 |
| LP 構成 | 年金、保険会社、ソブリン、ファンド・オブ・ファンズなど、セカンダリー市場の専門性を持つ投資家。 |
| ビークル法域 | Cayman ELP が標準。欧米 LP 基盤向けに Luxembourg SCSp または Delaware LP を使う場合もある。 |
| ルート | LP 主導（LP が持分を売却）、GP 主導（継続ファンド）、ストラクチャード / 優先型セカンダリー。 |

日本で活動するグローバル・セカンダリー GP には、Coller Capital、Ardian、Strategic Partners（Blackstone）、Lexington Partners（Franklin Templeton）、HarbourVest、Pantheon、Goldman Sachs Vintage が含まれる。日本特化のセカンダリー活動は米国 / EU より薄いが拡大している。

### 継続ファンド

| 次元 | 典型的な読み方 |
|---|---|
| 対象 | 同一 GP の既存ポートフォリオ企業 1 社以上を、既存ファンドから新しいビークルへ移す。 |
| セクター | セクターは裏付けとなるポートフォリオ企業を引き継ぐ。 |
| レバレッジ前提 | ポートフォリオ企業のレバレッジを引き継ぐ。移管時にリファイナンスされる場合もある。 |
| 保有期間 | 移管後3-5 年が典型。 |
| フィー / キャリー | 移管時にリセット。通常は1% / 10%または1.5% / 15%。単一資産の場合は案件レベルのフィーになる場合もある。 |
| LP 構成 | 既存 LP（ロールオーバー選択肢）と新規 LP（資本注入）。ILPA Continuation Fund Guidance は利益相反とプロセスに適用される。 |
| ビークル法域 | Cayman ELP が標準。非課税 LP 基盤向けに日本籍バリアントも存在する。 |
| 利益相反管理 | LPAC 同意、公正性意見、GP コミットメント、既存 LP の現状維持オプション。 |

継続ファンドは日本では比較的新しい構造であり、利益相反管理プロセスは ILPA guidance と高まりつつある国内 LP の期待に沿う。

### インフラ

| 次元 | 典型的な読み方 |
|---|---|
| 対象 | 長期キャッシュ利回り資産: 再生可能エネルギー（太陽光 / 風力 / 水力）、デジタルインフラ（データセンター、タワー、光ファイバー）、交通（有料道路、空港）、公益事業、規制ネットワーク。 |
| セクター | エネルギー転換、デジタルインフラ、交通、社会インフラ。 |
| レバレッジ前提 | 資産レベルの資金調達（プロジェクトファイナンス、インフラローン、インフラ債）。ファンドレベルは中程度。総レバレッジは資産価値の50-70%が典型。 |
| 保有期間 | 7-10+ 年が典型（一部のコアプラスは7-10 年、コアは10-15+ 年、スーパーコアは15-25 年）。 |
| フィー / キャリー | 1% / 10%（コア）、1.25% / 15%（コアプラス）、1.5% / 20%（バリューアッド）。6-8%の優先リターン超。 |
| LP 構成 | 年金、保険会社、ソブリン、インフラ重視のファンド・オブ・ファンズ。長期志向の LP 基盤。 |
| ビークル法域 | グローバルでは Cayman ELP、EU LP 基盤では Luxembourg SCSp、国内インフラ TK-GK では日本特化構造。 |
| 収益プロファイル | 利回り志向: 4-8% キャッシュ利回り + キャピタルゲイン. |

日本 / APAC インフラで活動するグローバル・インフラ GP には、Macquarie Asset Management、Global Infrastructure Partners（BlackRock）、KKR Infrastructure、Brookfield Infrastructure、I Squared Capital、Stonepeak が含まれる。日本国内インフラには[[JapanFG/dbj|DBJ]]関連の DBJ Capital、JOIN（Japan Overseas Infrastructure Investment Corporation）、複数の日本上場再エネ / インフラファンドが含まれる。

### プライベートクレジット

| 次元 | 典型的な読み方 |
|---|---|
| 対象 | スポンサー支援企業への直接貸付、メザニン債務、ディストレスト / 特殊状況、資産担保貸付、不動産クレジット。 |
| セクター | 直接貸付ではセクター横断。ディストレスト / 資産担保では特化型。 |
| レバレッジ前提 | ファンドレベルの信用枠（NAV ベース）で0.5-1.5x レバレッジが典型。裏付けローン自体は「レバレッジ付き」ではないが、借り手のレバレッジは4-6x。 |
| 保有期間 | 3-5 年 ローンごとに典型的; ファンド期間 5-8 年. |
| フィー / キャリー | 1% / 10% または 1.25% / 12.5%; エクイティ戦略より低い; 超 5-7% ハードル. |
| LP 構成 | 保険会社（大口）、年金、ソブリン、ファミリーオフィス。保険会社 LP は特に利回りに惹かれやすい。 |
| ビークル法域 | Cayman ELP が標準。米国 LP 基盤向けに Delaware LP、EU 向けに Luxembourg SCSp / RAIF を使う場合もある。 |
| 収益プロファイル | 利回り + スプレッド。シニア直接貸付では変動金利が中心。 |

日本で活動するグローバル・プライベートクレジットには、Apollo、Blackstone Credit（BXCI）、Ares Japan、Oaktree、Sixth Street、Bain Capital Credit、KKR Credit が含まれる。日本国内のプライベートクレジットは発展途上であり、メガバンクのローンポートフォリオと信託銀行のローン参加が隣接領域を占める。

## ビークル法域の読み方

| Vehicle | 根拠法 / 法域 | 典型的な用途 |
|---|---|---|
| **投資事業有限責任組合 (Japan investment LPS)** | Japan Investment LPS Act | 国内 GP の主ビークル。税務上パススルー。日本 LP 基盤。 |
| **Cayman Exempted LP (ELP)** | Cayman Islands ELP Law | 標準的なグローバル・ビークル。税務上パススルー。Cayman 所在。 |
| **Delaware LP** | Delaware Revised Uniform LP Act | 米国 LP 基盤の選好。税務上パススルー。 |
| **Luxembourg SCSp / RAIF** | Luxembourg AIFMD | EU LP 基盤の選好。AIFMD パスポート。 |
| **TK-GK (匿名組合 / 合同会社)** | Japan Commercial Code (TK) + Companies Act (GK) | TK 投資家を用いる日本の不動産 / インフラ投資構造。 |
| **GK-TK** | 同上 | 逆方向の組み合わせ。日本 REIT 型またはインフラ投資で用いられる。 |

日本で活動するグローバル GP の多くは、Cayman ELP を主ビークルとし、非課税の日本 LP 投資家向けに Japan parallel LPS を併用する。国内独立系は Japan LPS を主導し、海外 LP 向けに Cayman を組み合わせる。

## 境界事例

上記の6ファンド種別の分類は、現実のすべての戦略をきれいに分類するものではない。一般的な境界事例は次のとおり。

- **バイアウト + グロースのハイブリッド**: 支配権案件と少数持分案件の両方を扱う日本国内 GP など、ハイブリッド・マンデートを明示的に運用するファンドがある。LPA 上のキャリー経済性は案件分類によって異なる場合がある。

- **単一資産の継続ファンド**: 1社のポートフォリオ企業のみを保有する GP 主導セカンダリー。ILPA Continuation Fund Guidance はこれを、公正性意見、現状維持オプション、GP ロールオーバーなど、より厳格な利益相反審査の対象として扱う。

- **共同投資ビークル**: 多くの大口 LP（特にソブリン）は、低い、またはゼロのフィー / キャリーで並行共同投資権を交渉する。これは主ファンド LPA の外側に置かれる。

- **個別運用口座（SMA）**: 大口 LP は個別条件の専用マンデートを交渉する。多くは、別個のキャリー / フィー構造を持つ単一 LP ビークルである。

- **オープンエンド型エバーグリーン・ファンド**: 特にプライベートクレジットとインフラで用いられる。ビンテージ付きファンド期間ではなく、定期的な償還窓口を持つ恒久資本である。

- **クロスアセット・プラットフォーム**: Apollo、Blackstone、Brookfield、KKR など一部のグローバル企業は、PE + クレジット + インフラ + 不動産を統合したプラットフォームを運営する。LP コミットメント間のクロスプラットフォーム配分は、ファンドレベルの経済性と同じくらい重要である。

- **政府連動 / 政策連動のキャリー上限**: JIC Capital や一部 DBJ ファンドなど、日本の政策連動ビークルは、上限付きキャリー、政策連動の投資基準、共同投資義務を持つ場合があり、市場標準の LPA 経済性とは区別される。

- **PE 型の長期保有を行うアクティビスト・ヘッジファンド**: Effissimo、Strategic Capital、村上系、Oasis など複数の日本特化アクティビストファンドは、上場会社に対して PE 型の長期保有を行う。正式な PE GP ではないが、日本の上場対象案件フローの一部では機能的に競合する。[[finance/japan-activist-investor-playbook]]参照。

- **不動産 / インフラ向け TK-GK 構造**: 日本特化の TK（匿名組合）と GK（合同会社）構造は、不動産およびインフラ投資で税務パススルーのために用いられる。キャリーとウォーターフォール・メカニクスは標準的な LP / LPS 構造と異なる。

- **日本投資 LPS と Cayman ELP の税務処理比較**: Japan investment LPS は LPS レベルで税務上パススルーである。Cayman ELP も税務上パススルーだが、日本居住 LP は、みなし PE / 管理支配テストによって異なる取扱いを受ける。複数法域の接点は[[finance/multi-jurisdiction-identity-tax-leverage]]を参照。

- **地域銀行貸出と PE 案件フロー**: 地域銀行再編が続くなか（[[banking/regional-bank-consolidation-pattern]]参照）、地域銀行は限定的な LP 投資家であると同時に、中小企業承継を狙う国内 PE GP の案件ソーシング・パートナーにもなっている。

- **継続ファンドとプライマリーファンドの緊張関係**: Fund III にコミットした LP が、Fund III の資産を継続ファンドにロールすることへの同意を求められる場合がある。ILPA Continuation Fund Guidance はプロセスを定めており、LPAC 同意と LP 過半の承認が典型である。

## 実務家向け検証チェックリスト

上記の各セルに依拠する前に、次を確認する。

1. 出典にリンクされた所在地 / 企業ページを含め、対象 GP の公開ウェブサイト開示を開く。
2. GP の最新年次報告またはファンドクローズ発表に照らして、ファンド規模、ビンテージ、戦略を確認する.
3. 開示されている場合は、LP 構成を LP 自身の年次報告に照らして確認する (e.g. CalPERS, OTPP, CPP, GPIF 透明性報告).
4. 日本ビークルの状況を、該当する場合は EDINET の投資 LPS 届出、および Cayman / Luxembourg / Delaware の公開登録に照らして確認する。
5. 関連 FinWiki 項目を読む: [[finance/japan-private-equity-operating-model]], [[finance/japan-leveraged-buyout-economics]], [[finance/japan-acquisition-finance]], [[finance/japan-mbo-and-squeeze-out-process]].
6. アドバイザー・フランチャイズを照合する [[finance/japan-ib-league-table]].
7. 検証ステップに日付を付す.

## 注意点

- これは公開情報面のルートマップであり、投資助言、資金募集の勧誘、GP の推奨、推奨ではない。
- セル単位の記述は分類上のものに限られ、業界一般の公開情報上の慣行を反映している。個別ファンドの LPA 条件は大きく異なり、公開されていない。
- GP の公開ウェブサイト開示はマーケティングページであり、完全な LPA 開示ではない。実際のファンド条件については、読者は GP の完全な Form ADV（SEC 登録がある場合）、募集覚書、LPA を GP の投資家向け窓口を通じて直接確認すべきである。
- LP 構成の記述は、CalPERS、CPP 透明性報告など、公開された LP 配分に基づくものであり、GP 開示ではない。
- ビークル法域の慣行は変化する。租税条約や BEPS / Pillar Two の進展により、Cayman / Luxembourg / Delaware の選好が変わる可能性がある。
- 名前を挙げた一部の GP は複数の日本特化戦略を運用する場合がある。このマトリクスは主要な戦略パターンのみを示す。
- キャリー、フィー、ハードル、キャッチアップ、ウォーターフォール構造は LPA レベルの秘密条件である。このマトリクスは市場標準レンジのみを示す。
- アクティビストファンド、PE 型保有を行うヘッジファンド、ファミリーオフィス、政策連動ビークルは、6つのファンド種別分類にきれいには当てはまらない。境界事例セクションでこの外縁を扱う。
