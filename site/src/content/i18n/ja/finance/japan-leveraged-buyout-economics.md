# 日本のレバレッジド・バイアウト経済性

## ウィキ上の位置づけ

このページは [[finance/INDEX|finance domain]] の配下にある。ファイナンス・スタックの概観は [[finance/japan-acquisition-finance|Japan acquisition finance]]、非公開化の中核構造は [[finance/japan-mbo-and-squeeze-out-process|Japan MBO and squeeze-out process]]、公開開示レイヤーは [[finance/japan-tender-offer-process|Japan tender offer process]]、GP-LP 経済性の文脈は [[finance/japan-private-equity-operating-model|Japan private equity operating model]] とあわせて読む。

## 要約

日本の LBO は、負債とエクイティを階層化したスタックで資金調達するスポンサー主導の買収である。構造上の論点は、SPC ビークル、スポンサー・エクイティ、メガバンク LBO ローン、メザニン債務（任意）、ブリッジ・ファシリティ（任意）、担保パッケージ、財務コベナンツ、バスケット、EBITDA アドバック、リファイナンス余地、配当リキャップの可能性である。米国／欧州に比べ、日本の LBO レバレッジ水準は通常より保守的で、コベナンツ・パッケージはより厳しく、配当リキャップは攻撃性が低い。メガバンク 貸し手（[[megabanks/mufg|MUFG]]、[[megabanks/smfg|SMFG]]、[[megabanks/mizuho-fg|Mizuho FG]]）がシニア債務市場の中核を担う。

## レバレッジの源泉

| レイヤー | 提供者 | 典型的な役割 |
|---|---|---|
| スポンサー・エクイティ | PE ファンド、場合により共同投資 LP と経営陣ロールオーバー | 最初に損失を負担する資本、支配、ガバナンス |
| メガバンク LBO ローン（シニア） | MUFG 銀行、SMBC、みずほ銀行 | 最大の債務トランシェ、シンジケートまたはクラブディール |
| 信託銀行トランシェ | 三井住友信託銀行（SMTB）、三菱 UFJ 信託 | シニアまたはストレッチド・シニアへの参加 |
| DBJ トランシェ | [[financial-regulators/dbj|Development Bank of Japan]] | 戦略／政策整合的なシニア参加 |
| メザニン債務 | 専門メザニン・ファンド、地銀コンソーシアム、生保のプライベートデット部門 | 劣後トランシェ、レバレッジの不足分を補完 |
| ベンダー・ファイナンス／セラーノート | 対象会社の売り手 | 価値評価ギャップを埋め、支払いを繰り延べる |
| ブリッジ・ファシリティ | 同じメガバンク・アレンジャー | 短期。債券、タームローン、資本注入で借り換え |
| リボルビング・クレジット／運転資金枠 | 同じアレンジャー | クロージング後の OpCo 流動性 |
| ヘッジ・カウンターパーティ | メガバンク／グローバル銀行 | 金利スワップ、為替ヘッジ |

シニア LBO ローンが主要な資金源である。日本でのメザニン利用は安定しているが、米国／欧州より選別的である。ハイイールド債によるテイクアウトはまれで、リファイナンスは通常、延長シニアローン、アメンド・アンド・エクステンド、またはエクイティ注入で行われる。

## レバレッジ水準とプライシング

日本の LBO レバレッジは、米国／欧州の同種市場より通常保守的である。公開ソースの価格データは限られるため、構造的レンジは FSA／BOJ 統計、メガバンク開示、JSLA 公表資料から推定する。

| 項目 | 日本での典型的レンジ |
|---|---|
| 総レバレッジ / EBITDA | ミドルマーケットでは多くが 4-6x、安定キャッシュフローの大型案件では 6-7x の場合もある |
| シニアレバレッジ / EBITDA | 多くが 3-5x |
| エクイティ拠出 | 企業価値の 30-50 % となることが多い |
| 基準金利上乗せのシニア・マージン | スプレッドは案件により異なるが、米国 TLB 市場よりタイト |
| メザニン・クーポン | キャッシュ + PIK 構造、オールイン利回りは 1 桁台半ばから後半 |
| 手数料 | アップフロント・アレンジメント、コミットメント、エージェント、期限前返済手数料 |

マージン・スプレッドは 2022 以降、世界的な金利変動に沿って拡大したが、米国／欧州の広範なシンジケート市場の水準を下回っている。変動金利の基準は、JPY LIBOR 廃止後は通常 TONA（Tokyo Overnight Average Rate）である。ヘッジの仕組みは [[finance/japan-corporate-fx-and-rate-hedge-policy|Japan corporate FX and rate hedge policy]] を参照。

## コベナンツ・パッケージ

日本の LBO ローンは通常、米国大型 TLB より厳しいコベナンツ・パッケージを持つ。コベナントライトはそれほど普及していない。

### 財務コベナンツ

| コベナンツ | 機能 |
|---|---|
| レバレッジ比率 | 総債務 / EBITDA を四半期ごとにテスト。ローン期間中に低下するカーブ |
| インタレスト・カバレッジ比率 | EBITDA / 支払利息。返済能力を確認 |
| デットサービス・カバレッジ比率 | EBITDA - capex / 約定債務返済 |
| capex 制限 | 繰越／繰戻しバスケット付きの年間 capex 上限 |
| 最低 EBITDA | 調整後 EBITDA の下限 |
| エクイティ・キュア | スポンサーがエクイティを注入して財務コベナンツ違反を治癒可能 |

キュア権は通常、期間ごとおよび累計で制限され、二重取りを防ぐ文言が入る場合がある。

### 作為・不作為コベナンツ

| コベナンツ領域 | 日本での典型的な定式化 |
|---|---|
| 情報提供義務 | 監査済み年次、未監査四半期、月次管理会計 |
| 許容債務 | 劣後メザニン、キャピタルリース、運転資金枠 |
| 許容担保 | 既存担保、法定担保権、運転資金優先権 |
| 制限支払い | 配当、自社株買い、劣後債務の期限前返済を制限 |
| 許容買収 | レバレッジ／規模バスケットの範囲内でのボルトオン買収 |
| 資産処分 | 再投資バスケット付きの処分上限 |
| 支配権変更 | 強制期限前返済トリガー |

### EBITDA定義とアドバック

調整後 EBITDA の定義は、最も交渉される技術的論点である。典型的なアドバックは次のとおり。

| アドバック | 交渉ポイント |
|---|---|
| 買収／リストラクチャリング費用 | 年ごとまたは区分ごとの上限 |
| シナジー | ランレート／予想シナジーは EBITDA 比率で上限設定され、18-24 か月に期間制限されることが多い |
| 非経常項目 | 一回限りの訴訟、規制対応、退職関連費用 |
| 株式報酬 | 通常アドバックされる |
| 為替換算影響 | 一貫した扱い |
| リース会計移行 | IFRS 16 ／相当する調整 |

攻撃的なアドバック・パッケージは、米国大型 TLB ほど日本では一般的ではないが、アッパーミドル案件には入り込みつつある。

### バスケット

| バスケット | 機能 |
|---|---|
| 一般許容債務バスケット | ハードキャップまで追加債務を許容 |
| Available amount／ビルダー・バスケット | 留保利益とともに増加し、制限支払いを許容 |
| 許容投資バスケット | 中核事業外への投資を許容 |
| 制限支払いバスケット | 配当と株主分配に上限を設定 |
| インクリメンタル・ファシリティ（アコーディオン） | レバレッジ・テストを条件に追加シニア債務を許容 |

ビルダー・バスケットは米国大型 TLB より寛大ではない。Available amount の仕組みは存在するが、通常は上限があり、レバレッジ・テストの対象となる。

## 担保パッケージ

日本の LBO におけるシニアローン担保には通常、次が含まれる。

| 資産 | 担保形態 |
|---|---|
| OpCo の SPC 株式 | 株式質（kabushiki shichi） |
| OpCo 銀行口座 | 預金質（yokin shichi） |
| OpCo 債権 | 債権譲渡（chiken jōto） |
| OpCo 不動産 | 抵当権（teitoken） |
| OpCo 知的財産／商標 | 知的財産質（chiteki zaisan shichi） |
| 在庫 | セキュリティ・トラストを通じた浮動担保相当 |
| 保険金請求権 | 損害保険金受取人指定 |
| スポンサー保証 | 存在する場合は限定／上限付き保証 |

クロスボーダー案件では、法域ごとの担保対抗要件が加わる。信託銀行の担保エージェント（例: SMTB）が、クロスコラテラル・パッケージを調整することが多い。

## リファイナンスの力学

日本の LBO ローンは通常、5-7 年のシニア・ファシリティで、満期一括または一部アモチゼーション構造を持つ。リファイナンスの選択肢は次のとおり。

| ルート | 説明 |
|---|---|
| アメンド・アンド・エクステンド | 同じ 貸し手 グループで期間を延長し、場合により価格条件を引き締め |
| シニア・リファイナンス | 新たなシニアローン団が既存ファシリティを借り換える |
| 配当リキャップ | 会社を再レバレッジしてスポンサー分配を支払う |
| メザニン・リファイナンス | メザニンをシニアまたは新たなメザニン・トランシェで置き換える |
| 債券テイクアウト | 社債またはスポンサー支援債を発行してシニアを借り換える。日本ではまれ |
| エクイティ主導リファイナンス | 新規エクイティ調達または IPO 資金で期限前返済 |
| トレードセール・テイクアウト | 事業会社または後継 PE への売却で債務を返済 |

リファイナンス頻度は米国 TLB より低い。メガバンク 貸し手 が満期保有を好むこと、債券テイクアウト市場が薄いことが一因である。

## 配当リキャップ

日本にも配当リキャップは存在するが、米国 PE ほど攻撃的ではない。典型的な構造項目は次のとおり。

| 項目 | 詳細 |
|---|---|
| トリガー | 案件後の強いデレバレッジ、低い統合リスク、支援的なマクロ環境 |
| 規模 | 通常は元のスポンサー・エクイティの一部を回収。リキャップ後レバレッジ・コベナンツで上限 |
| 貸し手 の意欲 | メガバンク 貸し手 は慎重。信託銀行と DBJ はさらに慎重 |
| 必要な余裕 | リキャップ後レバレッジは通常、当初案件の範囲内に残る |
| 公開上の可視性 | プライベート市場。上場対象会社の文脈でない限り開示はまれ |
| レピュテーション面 | 対外的な事業運営が続く中でスポンサーへ資本を返す見え方 |

メガバンクが中心の 貸し手 市場では、配当リキャップへの意欲は 貸し手 の保有志向とリレーションシップ力学によって抑制される。

### 東芝の非公開化（Japan Industrial Partners 主導コンソーシアム）

| 項目 | 公開情報上の観察 |
|---|---|
| プロセス | 長期化した公開の取締役会／アクティビスト・プロセス後の非公開化 TOB |
| スポンサー | メガバンクと JIC が参加する JIP 主導コンソーシアム |
| 公開開示 | EDINET 公開買付届出書、JPX TDnet 対象会社意見表明、TOB 後スクイーズアウトと上場廃止 |
| 貸し手 構成 | 公開報道上はメガバンク主導のシニアにメザニン・トランシェを加えた構成 |
| 意義 | 近年最大級の非公開化。ジャンボ LBO に対する日本メガバンクの供給力を試した |

開示の中核構造は [[finance/japan-mbo-and-squeeze-out-process|MBO process]] と [[finance/japan-tender-offer-process|tender offer process]] を相互参照する。

### Lawson buyout

| 項目 | 公開情報上の観察 |
|---|---|
| プロセス | 親会社である三菱商事と提携した KDDI による TOB |
| 構造 | 純粋なスポンサー LBO ではなく、戦略的パートナーシップ主導の非公開化 |
| 公開開示 | EDINET TOB、JPX TDnet 取締役会意見表明 |

複数の [[finance/japan-mbo-and-squeeze-out-process|MBO and going-private]] プロセスでは、EDINET／JPX TDnet の提出資料からスポンサーと 貸し手 のスタックを確認できる。FinWiki は公開開示資料に基づいて個別案件分析を位置づける。

| 銀行 | 典型的なLBO上の役割 |
|---|---|
| [[megabanks/mufg|MUFG]] / MUFG 銀行 | シニア MLA、エージェント、ヘッジ・カウンターパーティ |
| [[megabanks/smfg|SMFG]] / SMBC | シニア MLA、エージェント、ヘッジ・カウンターパーティ |
| [[megabanks/mizuho-fg|Mizuho FG]] / みずほ銀行 | シニア MLA、エージェント、ヘッジ・カウンターパーティ |
| [[financial-regulators/dbj|DBJ]] | シニアまたは戦略的参加、時にメザニン |

LBO アレンジャー役割のリーグテーブル上のクレジットは、ベンダーが公表するローン／LBO 区分では [[finance/japan-ib-league-table|league-table sources]] を通じて確認される。

## アクティビストと開示の接点

LBO と非公開化プロセスは、特に少数株主がプレミアムやプロセス保護を不十分と見る場合、アクティビストの監視を受けやすくなっている。[[finance/japan-activist-investor-playbook|Japan activist investor playbook]] と [[finance/japan-fair-disclosure-and-insider-trading-controls|Japan fair disclosure and insider trading controls]] を参照。[[finance/japan-shareholder-proposal-and-agm-voting-route|shareholder proposal route]] は競合提案の出現や特別委員会の再構成を誘発し得る。

[[finance/japan-large-shareholding-disclosure|Large shareholding disclosure]] は、TOB 前の株式取得、共同保有者関係、競合 bidder のポジショニングを追跡するための主要な公開情報源である。
