# 日本 M&A ディールプロセス比較マトリクス

## 要約

日本M&Aは単一のプロセスではない。少なくとも七つの異なるディール類型別プロセスであり、法定の基礎構造を共有しつつ、買い手類型、支配権取得メカニズム、規制審査の積み上げ、資金調達形態、タイムラインが分岐する。このマトリクスは、**インバウンド**（外国買い手による日本対象会社の買収）、**アウトバウンド**（日本買い手による外国対象会社の買収）、**国内戦略買収**、**国内PE LBO**、**MBO / スクイーズアウト**、**JV組成**、**TOB義務取引**を、法定支配権取得メカニズム（合併 / 株式交換 / TOB / 事業譲渡 / 三角合併）、規制審査（[[finance/japan-fair-disclosure-and-insider-trading-controls|FSA disclosure]] / [[finance/japan-tender-offer-process|MOF FEFTA]] / [[finance/cross-border-m-a-japan|JFTC antitrust]]）、資金調達構造、フェアネス・オピニオン実務、少数株主の株式買取請求権、法定閾値（5% / 30% / 50.1% / 66.67% / 90%）にわたり横並びで示す。これはルートマップであり、法務・税務・投資助言ではない。

## ウィキ上の位置づけ

この項目は [[finance/INDEX|finance index]] の配下に位置する。デット・スタックの接点は [[finance/japan-acquisition-finance]]、公開買付メカニクスは [[finance/japan-tender-offer-process]]、経営陣主導ルートは [[finance/japan-mbo-and-squeeze-out-process]]、スポンサーのキャッシュフロー計算は [[finance/japan-leveraged-buyout-economics]]、アドバイザー側フランチャイズの厚みは [[finance/japan-ib-league-table]] と照合して読む。領域横断リンクは [[corporate-strategy/INDEX|corporate-strategy index]] に接続し、[[corporate-strategy/japan-kaisha-bunkatsu-tax-regime|kaisha bunkatsu tax]] と [[corporate-strategy/partial-spinoff-tax-deferral|partial spinoff]] の課税繰延べ接点を扱う。

## このマトリクスが重要な理由

日本M&Aに関する外部解説の多くは、ディール類型、法定メカニズム、規制審査を一括りにしてしまう。実務上、TSE上場対象会社を買収する外国事業会社は、義務的TOB、FEFTA事前届出、JFTC企業結合届出、上場対象会社のフェアネス・オピニオン、株式買取請求権という、日本スポンサーによるMBOとは異なるゲートの積み上げに直面する。実務家には横並びの視点が必要である。

- どの**買い手類型**がどの**規制審査**を発動するか。
- 法令上どの**支配権取得メカニズム**が利用でき、どれが [[corporate-strategy/japan-kaisha-bunkatsu-tax-regime|kaisha bunkatsu tax]];
- **TOBが義務**となる場合と**任意**にとどまる場合。
- **フェアネス・オピニオン**が実質的に必要となる場合と市場慣行にとどまる場合。
- **株式買取請求権**が少数株主にとって実効的なキャッシュアウト手段となる場合。
- どの**法定持株比率閾値**がどのメカニズムを開くか。

このマトリクスは、これらの問いに対する比較面である。詳細なメカニクスはリンク先の各プロセスページに置かれている。

## ディール類型の分類

| ディール類型 | 短い定義 |
|---|---|
| **インバウンド** | 外国（日本国外）の買い手が日本の対象会社を買収する。 |
| **アウトバウンド** | 公開資料に基づく分類上の記述。該当する法令、指針、当局通知で確認する。 |
| **国内戦略買収** | 日本の買い手 買収する a 日本の対象会社 for industrial / 事業会社 reasons. |
| **国内PE LBO** | 国内PEスポンサー 買収する a 日本の対象会社 using レバレッジ. |
| **MBO / スクイーズアウト** | 現経営陣 + スポンサー take a listed 日本の対象会社 private. |
| **JV組成** | 二者以上が資産または現金を新しい共同ビークルに拠出する。 |
| **TOB義務取引** | FIEAにより公開買付けが義務付けられる取引（例: 市場外で 1/3  を超える取得）。 |

この段落は、案件類型、法定メカニズム、規制審査、資金調達、少数株主保護を日本M&A実務の観点から整理する。 [[finance/cross-border-m-a-japan|cross-border M&A Japan]]

## 規制審査の積み上げ

| 審査項目 | 当局 | 発動条件 |
|---|---|---|
| **TOB規制** | [[finance/japan-tender-offer-process|FSA / Local Finance Bureau]] | 公開資料に基づく分類上の記述。該当する法令、指針、当局通知で確認する。 5% / 1 / 3  / 50% |
| **大量保有報告 (大量保有報告)** | [[finance/japan-large-shareholding-disclosure|EDINET]] | Beneficial ownership ≥ 5% of 上場発行体; 変更報告書 on ≥ 1% movement. |
| **インサイダー取引管理** | [[finance/japan-fair-disclosure-and-insider-trading-controls|FSA / Stock Exchange Surveillance]] | 重要な未公表情報 during 公表前期間. |
| **FEFTA事前届出** | MOF / sector regulators | 公開資料に基づく分類上の記述。該当する法令、指針、当局通知で確認する。 1% |
| **企業結合届出** | JFTC | 公開資料に基づく分類上の記述。該当する法令、指針、当局通知で確認する。 20 / 5 |
| **業法規制当局** | BOJ, FSA, METI, MIC, MHLW, MLIT, etc. | Banking, 保険, 放送, 通信, 医療, etc. require 業法承認. |
| **外国競争法** | EU Commission, US DOJ/FTC, PRC SAMR, KFTC, CMA, others | If アウトバウンド or クロスボーダー deal crosses 外国法上の閾値. |
| **CFIUS / 国家安全保障審査相当制度** | 相手方法域 | 公開資料に基づく分類上の記述。該当する法令、指針、当局通知で確認する。 |

審査の積み上げは順序が重要である。FEFTA事前届出は外国投資家が実行する**前**にクリアされなければならない。競争法届出は並行して進められる。TOB期間の順序はFIEAにより規定される。プロセスのタイムラインに依拠する前に [[finance/japan-tender-offer-process]] を開く。
