# 日本国庫短期証券 (TDB / 国庫短期証券)

## ウィキ上の位置づけ

このエントリは [[money-market/INDEX|money-market index]] 配下に位置する。ピア比較・対比の文脈では [[money-market/japan-cp-commercial-paper-market|Japan CP market]]、より広い制度・規制境界では [[money-market/jgb-repo-market-japan|JGB repo market]] と照らして読む。

## 要約

Treasury Discount Bills (TDB / 国庫短期証券) は、日本の財務省 (MOF) が 3 か月、6 か月、1 年の満期で発行する短期ゼロクーポン国債である。TDB は円短期金融市場におけるソブリンの基礎ブロックであり、[[money-market/japan-cp-commercial-paper-market|CP]]、[[money-market/japan-ncd-negotiable-cd-market|NCD]]、円短期レポ価格にとって自然な無リスク・ベンチマークである。

TDB は MOF の通常の JGB 入札カレンダーに載り、日銀ネット国債振替決済制度を通じて無券面で保有され、日銀オペや相対レポにおいて最も清浄度の高い円担保として受け入れられる。また、海外投資家が円短期金利エクスポージャーを取る際の主要な入口でもある。

## 発行メカニクス

| 要素 | 読み方 |
|---|---|
| 満期 | 3 か月、6 か月、1 年。財政サイクルの平準化のために資金繰り債が発行されることもある。 |
| 形態 | ゼロクーポン割引債。投資家は満期に額面を受け取る。 |
| 発行方法 | 通常の利回り競争入札。結果は MOF / 日銀が開示する。 |
| 頻度 | 年限に応じた通常の週次 / 月次ペース。MOF の国債入札カレンダーを参照。 |
| 登録 | 日銀ネット国債振替決済制度。完全無券面。 |
| 税務 | 特定の海外投資家非課税制度の下で、非居住者保有者への源泉徴収はゼロ。 |

## 入札ペース

MOF は、利付国債と TDB の双方を含む国債入札カレンダーを公表している。TDB の入札サイクルは、日本国債市場で最も安定した発行リズムの一つである。短期満期では 3か月物のほぼ週次発行が必要になり、6か月物と 1年物のオペレーションも定期的に行われる。財政サイクル上必要な場合には、資金繰り債の発行が追加される。

入札は単一価格方式または通常方式で、参加者は利回りで応札する。国債市場のプライマリーディーラーが主な応札者だが、ディーラー仲介を通じた非ディーラーの応札経路も存在する。結果は同日に公表され、きれいな短期金利の目印を提供する。

## 投資家構成

TDB の保有者は、円短期金融商品としては異例に多様である。

- **海外投資家**: TDB は、円短期エクスポージャー、円資金管理、為替ヘッジ・ベーシス取引のために、歴史的に海外投資家の高い比率を集めてきた。海外保有比率は、為替ヘッジ後利回り差によって大きく変動してきた。
- **日銀**: 日銀は緩和局面の買入オペを通じて TDB を保有してきた。保有額は政策レジームによって変動する。
- **国内銀行**: 銀行は、HQLA、日銀担保、円短期利回りのために、銀行勘定で TDB を保有する。
- **MMF / MRF / 円短期ファンド**: TDB はこれらのビークルの典型的な保有資産である。
- **保険・信託勘定**: TDB は短期デュレーションのソブリンエクスポージャーを提供する。

MOF は JGB と TDB の投資家別保有内訳を公表しており、最新値はそれを参照すべきである。

## 日銀と TDB

日銀は複数のオペレーション面で TDB を利用する。

| 日銀機能 | TDB との関係 |
|---|---|
| 買入 | 日銀は過去の枠組みで、資金供給オペの一部として TDB を買い入れてきた。 |
| レポ / 担保 | TDB は、共通担保資金供給オペやその他の有担保オペで清浄度の高い円担保として受け入れられる。 |
| 日銀ネット | TDB 決済は日銀の国債振替決済レールを通る。 |
| 政策波及 | TDB 利回りは、[[money-market/boj-post-2024-floor-system-complementary-deposit-facility|post-2024 floor system]] 政策金利と相互作用するきれいな短期金利指標を提供する。 |

## TDB 利回りの読み方

| 視点 | 注記 |
|---|---|
| ソブリン短期金利 | 無リスク円短期金利に最も近い市場読み。 |
| OIS スプレッド | TDB 利回りと TONA OIS の差は、担保 / 希少性プレミアムを示す。 |
| CP / NCD スプレッド | TDB は [[money-market/japan-cp-commercial-paper-market|CP]] と [[money-market/japan-ncd-negotiable-cd-market|NCD]] の信用スプレッドの基準になる。 |
| 為替ヘッジ後利回り | USD / EUR ヘッジ後の海外投資家リターンであり、限界的な海外需要を動かす。 |
| 日銀金利スプレッド | TDB 利回りと補完当座預金制度利率との差は、担保希少性を示す。 |

## 2024 後の文脈

日銀がマイナス金利を終了した 2024 年3月のレジーム変更後、TDB 利回りはプラス圏に転じ、入札カレンダーは引き続き定期的な短期金利指標を提供している。読み方のルールは以下の通りである。

- 政策金利決定と実際の TDB 落札利回りを分けて読む。
- ヘッジ後利回りの算術で動く海外投資家比率の変化を見る。
- 日銀が買っているのか、保有しているのか、満期落ちに任せているのかを確認する。
- TDB-OIS ベーシスは、単一原因の指標ではなく、担保と資金調達の有用なシグナルとして扱う。

## 境界事例

- **TDB と短期利付国債**: TDB はゼロクーポンで、3M / 6M / 1Y のみで発行される。同じ残存年限の短期利付国債は別の商品であり、流動性特性も異なる。
- **TDB と CP / NCD**: TDB はソブリンで無リスクである。CP / NCD は発行体信用リスクを持ち、スプレッドで取引される。
- **TDB と資金繰り債**: 資金繰り債は財政サイクルの平準化のために随時発行され、通常入札スケジュールの一部ではない。
