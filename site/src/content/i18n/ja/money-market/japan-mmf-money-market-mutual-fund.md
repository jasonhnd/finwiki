# 日本の MMF / MRF (マネー・マーケット・ミューチュアル・ファンド)

## ウィキ上の位置づけ

この項目は [[money-market/INDEX|money-market index]] の配下に位置する。ピア / 対比の文脈については [[money-market/japan-ncd-negotiable-cd-market|Japan NCD market]] と、より広い制度 / 規制境界については [[money-market/boj-post-2024-floor-system-complementary-deposit-facility|BoJ post-2024 floor system]] と照らして読む。

## 要約

日本には、「MMF」として混同されることがある二つの異なる短期円ファンド商品群がある。

- **MRF (Money Reserve Fund)** — 証券口座内の未投資現金を保有するための証券スイープ商品。MRF は、日本で持続している大衆向け短期円ファンド商品である。
- **旧来型の個人向け MMF** (公社債投信 series) — かつて有力だった個人向け短期債券ファンドのカテゴリであり、2001 Enron 関連の元本割れ、2008 Lehman 危機時の解約、長期のゼロ金利 / マイナス金利環境の後、主力商品としては実質的に姿を消した。

FinWiki では、このページが歴史的経路、MRF と旧 MMF の区別、現在の代替商品 (中期国債ファンド型の短期債券ファンド等)、2024 後のフロアシステムが短期円ファンドカテゴリに与えた影響を記録する。

## 商品マップ

| 商品群 | 目的 | 状態 |
|---|---|---|
| MRF (Money Reserve Fund) | 証券口座内に保有される証券スイープ現金。取引決済のための同日流動性を提供する。 | 現役かつ支配的。投資信託純資産カテゴリの中でも最大級の一つ。 |
| 旧来型の個人向け MMF | 個人貯蓄者向けに準貨幣代替として販売された短期円投資信託。 | 主力商品としては実質的に退場。既存残高は流出。 |
| 中期国債ファンド | 短期 / 中期 JGB ファンドカテゴリ。代替商品かつ先行商品。 | 形を変えて残存するが、ピーク時ほど目立たない。 |
| 短期債 / 短期円 ETF・オープンエンドファンド | 短期円エクスポージャーの現代的代替商品。 | 現役だが、MRF に比べると小さい。 |

## 歴史的経路

| 年 | 出来事 | MMF / MRF への影響 |
|---|---|---|
| 1980s-1990s | 高利回り時代に、中期国債ファンドと個人向け MMF が銀行預金代替として発展。 | 個人向け短期円ファンドカテゴリが家庭向け商品となる。 |
| 2001 | Enron 破綻により、Enron CP を保有していた一部の日本 MMF が元本割れ。 | 個人向け MMF カテゴリに大きな評判被害。大量解約。商品が後景化。 |
| 2008 | 一部 MMF が保有していた Lehman Brothers 破綻債がさらなる元本毀損を招き、MMF / 短期円ファンド全体で大量解約圧力が発生。 | 個人向け MMF カテゴリは主力商品として機能的に崩壊。 |
| 2010s | 長期ゼロ金利環境により、多くの短期円ファンドで手数料控除後のプラス利回りが不可能になる。 | 旧来型の個人向け MMF は実質退場。MRF はスイープ現金商品として存続。 |
| 2016 | BoJ が NIRP を導入。 | マイナス金利環境により、多くの MMF / MRF 商品が特殊構造と例外に追い込まれ、一部 MMF は償還された。 |
| 2024 年 3 月 | BoJ が NIRP と YCC を終了。短期金利がプラスに戻る。 | MRF と短期円ファンドカテゴリはプラス利回りを生み出せるようになり、商品の再マーケティングが始まった。 |
| 2024-2025 | 新 NISA が開始。家計資産シフトプログラムが続く。 | MRF は証券 / [[securities/nisa-2024-flow|NISA]] 口座のキャッシュスイープ層にとどまり、他の短期円ファンドは低ボラティリティ代替として再位置づけられた。 |

## MRF と旧 MMF

| 項目 | MRF | 旧来型個人向け MMF |
|---|---|---|
| 主な用途 | 証券口座内の現金。取引決済スイープ。 | 銀行預金代替として販売された独立型短期円投資信託。 |
| 流動性 | 決済連動のスイープ機構により同日換金。 | 日次換金で t+決済だが、ストレス時には遅くなる。 |
| 元本安全性 | 安定 NAV を維持する設計。法的には投資信託であり元本割れはありうるが、広範なリスク管理と格付がある。 | 2001 および 2008 ストレス事象で元本割れを示した。 |
| 顧客基盤 | ほぼすべての個人証券口座。証券口座開設に結び付く。 | 預金代替を求める個人貯蓄者。 |
| 状態 | 現役で純資産規模も大きい。 | 主力商品としては実質退場。 |

## 現在の代替商品

旧来型の個人向け MMF の後退後、短期円利回りを求める個人貯蓄者は次のルートを使う。

- **MRF**: 証券口座を開くと MRF スイープが必要になるため、デフォルトのルート。
- **中期国債ファンド型商品**: 短期 / 中期 JGB ファンド。ピーク時より小さいが、主要投信グループからなお提供される。
- **短期債 ETF**: [[securities/tokyo-stock-exchange|TSE]] に上場するパッシブな短期デュレーション JGB / 社債 ETF。
- **銀行預金代替**: 手数料控除後利回り差が小さいため、単に銀行預金に現金を置く。
- **証券会社経由の TDB 直接購入**: 国債割引ルートについては [[money-market/japan-tbill-treasury-discount-bill|TDB page]] を参照。

機関投資家にとっての同等の商品層は、機関投資家向けマネーファンドまたは短期債券ファンドである。日本の機関投資家向け商品は米国より小さい。

## 投資保有資産

MRF と短期円ファンドのポートフォリオは、標準的な短期円資産スタックを保有する。

- 短期 JGB と [[money-market/japan-tbill-treasury-discount-bill|TDBs]]。
- [[money-market/japan-ncd-negotiable-cd-market|NCDs]] (銀行発行)。
- [[money-market/japan-cp-commercial-paper-market|commercial paper]] (事業会社 / ノンバンク発行)。
- [[money-market/jgb-repo-market-japan|JGB repo]] および短期有担保貸付。
- 適格銀行への銀行預金。

資産側の構成は、目論見書上の格付 / 満期ルールおよび投資信託協会 (JITA / 投資信託協会) の自主規制によって制約される。

## ゼロ金利時代の影響

2016-2024 NIRP / YCC 期は、短期円ファンドにとって構造的に厳しい環境だった。短期金利がゼロ近傍またはマイナスである中、手数料控除後利回りはしばしばマイナスとなり、ファンド運営者は次の対応を迫られた。

- 旧 MMF シリーズの閉鎖または併合。
- 顧客向け利回りのマイナス化を避けるための MRF の一時的な手数料免除。
- 受託銀行の BoJ 口座に置かれる MRF 現金についての特殊な法的構造と例外の利用。
- 個人向け獲得ストーリーを、利回りからスイープの利便性と取引執行支援へ移す。

2024 年 3 月の、NIRP と [[money-market/boj-post-2024-floor-system-complementary-deposit-facility|floor system shift]] の終了を含むレジーム変更後、短期円ファンドカテゴリは再びプラス利回りを生み始め、MRF と復活した MMF 代替商品の商品レベルの経済性は徐々に改善している。

## 境界事例

- **MRF vs 銀行預金**: MRF は法的な元本割れリスクを持つ投資信託である。銀行預金には適用限度まで預金保険がある。
- **MMF vs MRF**: 頻繁に混同されるが、販売レールとリスクプロファイルが異なる別の商品カテゴリである。
- **MMF vs 短期債ファンド**: 2008 後、短期債および JGB ファンドが MMF のマーケティング上の役割を置き換えた。構造と流動性は異なる。
- **安定 NAV vs 変動 NAV**: 日本の MRF は安定した解約価額の維持を目指すが、法的に保証しているわけではない。

## 読解チェックリスト

1. 問いが MRF (スイープ) なのか、独立型の短期円ファンドなのかを特定する。
2. ポートフォリオ構成を想定する前に、JITA の現在の自主規制ルールを確認する。
3. 元本安全性設計と法的保証を区別する。
4. ファンドフローのデータを [[money-market/boj-post-2024-floor-system-complementary-deposit-facility|post-2024 floor system]] 利回り計算の文脈で読む。
5. MRF 残高を証券口座開設と [[securities/nisa-2024-flow|NISA]] 資金流入に結び付ける。
