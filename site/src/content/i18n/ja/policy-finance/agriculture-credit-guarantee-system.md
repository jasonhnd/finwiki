# Agriculture credit guarantee system in Japan

## Wiki route

This entry sits under [[policy-finance/INDEX|policy-finance index]]. Read it against [[policy-finance/japan-credit-guarantee-system|Japan credit guarantee system]] for SME-side contrast, [[policy-finance/fisheries-credit-guarantee-system|fisheries credit guarantee system]] for the sister sector system, and [[policy-finance/japan-policy-finance-system|Japan policy finance system]] for the broader public-credit perimeter. The deposit-and-lending counterpart for farmers is [[banking/ja-bank-system-japan|JA Bank system]] and the cooperative apex is [[cooperative-banks/norinchukin|Norinchukin]].

## TL;DR

日本の農業信用保証保険制度は、農業者および農業関連の借り手の信用を、二層構造によって補完する：(1) 都道府県単位の **農業信用基金協会**（Agriculture Credit Fund Associations、47 都道府県の主体）が農業者／農業法人の貸付金融機関向け債務を保証し、(2) 全国単位の **独立行政法人農林漁業信用基金**（Agriculture, Forestry and Fisheries Credit Foundations、JAFFIC）の農業信用保険業務によって再保険される。これは [[banking/ja-bank-system-japan|JA Bank system]] に隣接するが、預金／貸付の経路ではなく保証保険の経路である。[[policy-finance/japan-credit-guarantee-system|the general SME credit guarantee system]] と比較すると、農業の制度はセクター固有であり、農林水産省（MAFF）が監督し、一般の地方銀行ではなく農業協同組合システム（系統金融）と統合されている。

## 1. 系統金融との位置関係

日本の農業金融は **系統金融**（cooperative system finance）スタックを通じて流れる：

| Layer | Function | FinWiki route |
|---|---|---|
| 農家 / 農業法人 | 借り手（農業者、農業法人、農業従事者、アグリビジネス）。 | System route here |
| 単位農協 (JA / 単協) | 単位協同組合；農業者の預金を受け入れ、農業・家計向け信用を供与。 | [[banking/ja-bank-system-japan]] |
| 信連 (JA信連 / 都道府県信用農業協同組合連合会) | 都道府県の農業協同組合の信用連合会；JA の預金／貸付を仲介。 | [[banking/ja-bank-system-japan]] / prefectural-信連 layer |
| 農林中央金庫 (Norinchukin) | JA／信連／漁業・林業協同組合の全国アペックスバンク；余剰の協同組合預金をグローバルな投資家として運用。 | [[cooperative-banks/norinchukin]] |
| 農業信用基金協会 | 農業向け貸付の都道府県保証主体；デフォルト時に代位弁済を行う。 | This page |
| 独立行政法人農林漁業信用基金 (JAFFIC) | 都道府県の農業・漁業保証協会向けの全国単位の信用保険／再保険主体；直接の融資支援も行う。 | This page |
| MAFF (農林水産省) | 協同組合システム + 信用保証保険制度の政策監督者。 | [[policy-finance/INDEX]] |

したがって、保証制度は **協同組合バンキング・スタックの上に重ねられた並行する公的信用インフラ** である ── JA／信連／農林中金が預金と貸付を扱い、農業信用基金協会 + JAFFIC がその貸付を裏付ける保証・再保険層を扱う。

## 2. 機関概要 — 農業信用基金協会

| 資金源 | 国 / 都道府県 / 系統金融機関の出資・補助 |
| 連合会 | 全国農業信用基金協会連合会 (national federation of prefectural agriculture credit fund associations) |
| 再保険 | 農林漁業信用基金 (JAFFIC) 農業信用保険業務 |

## 4. 主要保証商品

- **農業近代化資金保証** — 農機具投資、施設近代化（温室、搾乳施設、貯蔵庫）向けの近代化資金保証。
- **農業改良資金保証** — イノベーション、新作物導入、有機転換向けの改良資金保証。
- **農業経営基盤強化資金 (スーパーL資金) 保証** — JFC の農林水産事業を通じて供給される大規模な農業経営統合資金向けの保証。
- **就農支援資金保証** — 新規就農者支援保証。
- **担い手・農業法人向け保証** — 大規模農業者／法人農場保証。
- **災害復旧保証** — 災害復旧農業保証（台風被害、洪水被害、雪害、家畜疾病）。
- **6次産業化 / 加工流通関連保証** — 農業者主導の食品加工・直接流通向けの保証（「第 6 次産業化」の政策レーン）。
- **借換 / 経営改善資金保証** — 借換および経営改善保証。

## 5. Operating logic — guarantee, subrogation, reinsurance

1. **申込** — 農業者または農業法人が貸付金融機関（典型的には JA／信連／農林中金／地方銀行／JFC の農業部門）から借り入れる。貸し手は保証信用を要し、都道府県の農業信用基金協会に申し込む。
2. **保証** — 都道府県協会は貸付を評価し、保証を引き受け、保証料（保証料 — 典型的には商品により異なる、残存元本に対する低い一桁の年率）を徴収する。
3. **貸付** — 貸し手は保証が設定された状態で貸付を実行する。
4. **デフォルト** — 借り手がデフォルトした場合、貸し手は保証に基づき請求する；都道府県協会は貸し手に代位弁済（代位弁済）を行い、その後、借り手／担保に対して回収（求償）を追求する。
5. **再保険** — JAFFIC の農業信用保険業務が都道府県協会の保証エクスポージャーを再保険する；該当する保険契約に基づき、保険金が JAFFIC から都道府県協会へ支払われる。
6. **残余損失** — 回収および JAFFIC 保険の後、残余損失は都道府県協会の準備金が吸収し、国／都道府県／協同組合の拠出がこれを補完する。

したがって、この制度には **3 つの損失吸収層** がある：(a) 借り手からの回収、(b) 都道府県協会の準備金 + 全国 JAFFIC の再保険、(c) 制度を下支えする公的／協同組合の資本拠出。

## 6. 保証料 economics

保証料の体系は、MAFF のガイダンスの範囲内で各都道府県の農業信用基金協会が設定し、典型的には次のように構成される：

- 残存元本に対する年率で、上位の政策資金（例：スーパーL資金）については低い一桁のベーシスポイント帯、よりリスクの高い商品についてはより高い。
- 商品（近代化資金／改良資金／就農支援 etc.）および借り手カテゴリー（担い手の優先割引）別に階層化。
- 優先政策カテゴリー（新規就農者、災害復旧、第 6 次産業化）について補助。
- 保証料の経済性は、市場相当の商業保証価格を意図的に下回る ── そのスプレッドは公的／協同組合の補助によって吸収される。

## 7. Year evolution

| Year | Event |
|---|---|
| 1961 | 農業信用保証保険法が制定；都道府県の農業信用基金協会が設立。 |
| 1972 | 農業信用保険公庫が設立（後に JAFFIC に統合）。 |
| 1986 | 林業信用基金が設立。 |
| 2003-10-01 | 独立行政法人農林漁業信用基金 (JAFFIC) が農業・林業・漁業の信用保険主体を統合して創設。 |
| 2007 | 農業政策改革のもとで担い手優先保証プログラムが拡充。 |
| 2010s | 6次産業化保証が追加；農業者主導の加工／流通が明示的な保証支援を獲得。 |
| 2014–2016 | TPP／FTA の農業への圧力；農業近代化への適応を支援するために保証制度が用いられる。 |
| 2018 | 卸売市場法改正／種子法廃止／漁業法改正 etc. ── 農業政策再編の時代。 |
| 2020–2022 | COVID 期の災害保証が農業サプライチェーンの混乱（特に乳製品の過剰供給、外食の崩壊が和牛／園芸に与えた影響）に対して発動。 |
| 2022–2023 | エネルギー／肥料／飼料の価格ショック ── 保証制度が畜産および施設園芸セクターの流動性を支援。 |
| 2024 | スマート農業／smart-agriculture の融資商品が保証支援を獲得。 |
| 2024–2025 | 農業改革（基本法改正の道筋、食料安全保障の重視）── 保証制度が食料安全保障投資支援に向けて位置づけられる。 |
| 2025 | 担い手の承継 + 法人化の優先 ── 保証制度が法人農場／承継主導の構造改革に整合。 |

## 8. Comparison vs general SME credit guarantee system

| Dimension | Agriculture credit guarantee system | [[policy-finance/japan-credit-guarantee-system\|General SME credit guarantee system]] |
|---|---|---|
| Borrower target | 農家 / 農業法人 / 農業関連事業者 | SME / small business (broad sectors) |
| Local guarantor | 農業信用基金協会 (47 prefectural) | 信用保証協会 (51 prefecture / city) |
| National federation | 全国農業信用基金協会連合会 | [[policy-finance/national-federation-credit-guarantee-corporations\|全国信用保証協会連合会 (JFG)]] |
| National reinsurer | 農林漁業信用基金 (JAFFIC) | [[financial-regulators/jfc\|JFC]] credit-insurance |
| Supervisor | MAFF (農林水産省) | METI / SME Agency (中小企業庁) |
| Lender partners | JA / 信連 / Norinchukin / banks / JFC ag-arm | Regional banks / shinkin / credit cooperatives / JFC |
| Crisis-era flagship | Disaster / livestock-disease / 食料安全保障 | セーフティネット / 危機関連 (ZeroZero 2020–2022) |
| Funding economics | 公的 / 系統金融 subsidies | 公的補助 + 信用保険 reinsurance |

この 2 つの制度は、異なるセクターターゲット、異なる監督者、異なる協同組合バンキングのインターフェースを持つ **並行する公的信用保証インフラ** である。両者は借り手を奪い合わない ── 農業者への貸付は農業の制度を通り、小規模製造業者への貸付は中小企業の制度を通る。

## 9. Boundary cases

- **JA バンクではない** — JA バンクは貸し手の経路であり、本ページはその上に重ねられた保証保険のメカニズムである。
- **中小企業保証制度ではない** — 中小企業保証は [[policy-finance/japan-credit-guarantee-system]] に経路される。
- **漁業ではない** — 漁業の保証保険は [[policy-finance/fisheries-credit-guarantee-system]]（同じ JAFFIC を親とする姉妹セクター）に属する。
- **林業ではない** — 林業の信用支援も JAFFIC の業務だが、MAFF の林業政策のもとで別途経路される。
- **JFC の農業貸付ではない** — JFC の [[financial-regulators/jfc|agriculture-forestry-fisheries arm]] は農業者に直接貸し付ける；本ページは農業者の借入に対する保証保険についてであり、保証付き貸付が JFC によって組成される場合には重複しうる。
