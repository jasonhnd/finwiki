#日本の農業信用保証制度

## ウィキ上の位置づけ

このエントリは [[policy-finance/INDEX|policy-finance index]] の下にあります。中小企業側の対比については [[policy-finance/japan-credit-guarantee-system|Japan credit guarantee system]]、姉妹部門システムについては [[policy-finance/fisheries-credit-guarantee-system|fisheries credit guarantee system]]、より広範な公的信用境界については [[policy-finance/japan-policy-finance-system|Japan policy finance system]] と照らし合わせてお読みください。農家向けの預金・貸出の対応項目は [[banking/ja-bank-system-japan|JA Bank system]]、協同組合系統の頂点は [[cooperative-banks/norinchukin|Norinchukin]] です。

## 要約

日本の農業信用保証制度 (農業信用保証保険制度) は、2 層の階層構造を通じて農家および農業関連の借り手の信用を補完します: (1) 都道府県レベルの **農業資金信用協会** (農業信用基金協会、47 都道府県団体)。農家/農業法人の債務を融資機関に保証し、(2) 国レベルの **独立行政法人農業信用基金**によって再保険されます。 （財団法人農林水産信用財団、JAFFIC）農業信用保険事業 [[banking/ja-bank-system-japan|JA Bank system]]に隣接していますが、保証・保険ルートであり、預金・貸付ルートではありません。 [[policy-finance/japan-credit-guarantee-system|the general SME credit guarantee system]] と比較すると、農業システムはセクター固有であり、農林水産省の監督下にあり、一般の地方銀行ではなく農業協同組合システム (系統金融) と統合されています。

## 1. 系統金融との位置関係

日本の農業金融は **系統金融** (協同組合システム金融) スタックを介して流れます。

| レイヤー | 機能 | FinWiki ルート |
|---|---|---|
| 農家 / 農業法人 | 借り手（農家、農業法人、農業従事者、アグリビジネス）。 | ここのシステムルート |
| 単位農協 (JA / 単協) | 第一次協同組合。 農民の預金を受け取ります。 農業と家計の信用を拡大します。 | [[banking/ja-bank-system-japan]] |
|県信用農業協同組合連合会、 JAの預金・貸付を仲介する。 | [[banking/ja-bank-system-japan]] / 県信連層 |
| 農林中央金庫 | JA/信連/漁業/森林組合の全国頂点銀行。 グローバル投資家として協同組合の余剰預金を管理。 | [[cooperative-banks/norinchukin]] |
| 農業信用基金協会 | 農業融資に対する県の保証機関。 デフォルト時に代位を実行します。 | このページ |
| 独立行政法人農林漁業信用基金 (JAFFIC) | 都道府県農業漁業保証協会の全国規模の信用保険・再保険機関。 直接融資サポートも。 | このページ |
| MAFF（農林水産省） | 組合制度＋信用保証保険制度の政策責任者。 | [[policy-finance/INDEX]] |

したがって、保証システムは**協同組合銀行スタックの上に重ねられた並行した公的信用インフラ**であり、JA / 信連 / 農林中金が預金と融資を処理し、農業信用基金協会 + JAFFICがその融資を裏付ける保証と再保険の層を処理します。

## 4. 主要保証商品

- **農業近代化資金保証** — 農業設備投資、施設近代化（温室、酪農場、倉庫）に対する近代化融資保証。
- **農業改良資金保証** — イノベーション、新作物の導入、有機転換のための改良融資保証。
- **農業経営基盤強化資金（スーパーL資金）保証** — 日本政策金融公庫の農林水産業部門を通じた大規模な農業経営統合融資に対する保証。
- **就農支援資金保証** — 新規参入農家のサポート保証。
- **緊急・農業法人向け保証** — 大規模農家/法人農場保証。
- **災害復旧保証** — 災害復旧農業保証（台風被害、水害、雪害、家畜疾病）。
- **6次産業化 / 加工流通関連保証** — 農家主導の食品加工および直接流通に対する保証 (「6次産業化」政策レーン)。
- **借換 / 経営改善資金保証** — 借り換えおよび経営改善保証。

## 5。 運用ロジック — 保証、代位、再保険

1。 **申し込み** — 農家または農業法人が金融機関 (通常は JA / 信連 / 農林中央金庫 / 地方銀行 / 日本政策金融公庫の農業部門) から資金を借ります。 貸し手は保証信用が必要で、県農業信用基金協会に申請する。
2。 **保証** — 都道府県協会はローンを評価し、保証を受け入れ、保証料 (保証料 — 通常、未払い元本に対する 1 桁台前半の年率で、商品によって異なります) を請求します。
3。 **融資** — 貸し手は保証を付けてローンを実行します。
4。 **債務不履行** — 借り手が債務不履行になった場合、貸し手は保証に基づいて請求を行います。 県協会は貸し手に代位弁済を支払い、その後、借り手・担保に対して回収を求めます。
5。 **再保険** — JAFFIC の農業信用保険事業は、県協会の保証エクスポージャーを再保険します。 保険金は、適用される保険契約に基づいて JAFFIC から都道府県協会に支払われます。
6。 **残余損失** — 回収およびJAFFIC保険の後、残余損失は県協会の積立金によって吸収され、国/県/組合の拠出金によって補われます。

したがって、このシステムには **3 つの損失吸収層**があります: (a) 借り手の回収、(b) 都道府県協会の積立金 + 国の JAFFIC 再保険、(c) システムを支える公共/協同組合の出資。

## 6. 保証料 economics

保証料スケジュールは、農林水産省のガイダンスに従って各都道府県農業信用基金協会によって設定されており、通常は次のように構成されています。

- 未払い元本に対する年率は、上級保険ローン（スーパーL資金など）では一桁ベーシスポイント台前半の範囲にあり、高リスク商品ではそれより高くなります。
- 商品別（近代化資金 / 改良資金 / 就農支援など）および借り手カテゴリー別（緊急優先割引）。
- 優先政策カテゴリー（新規参入農家、災害復興、6次産業化）に対して補助金を交付。
- 手数料の経済性は市場同等の商業保証価格を意図的に下回っています。スプレッドは公的/協同組合の補助金によって吸収されます。

## 7。 年の進化

| 年 | イベント |
|---|---|
| 1961 | 農業信用保証保険法制定。 都道府県農業信用基金協会設立。 |
| 1972 | 農業信用保険公庫設立（後にJAFFICに統合） |
| 1986 |林業信用基金を設立|
| 2003-10-01 |農林漁業信用保険機関が統合して誕生したJAFFIC。 |
| 2007 | 農政改革のもとで優先保障制度が拡充された。 |
| 2010s | 6次産業化保証が追加されました。 農家主導の加工/流通には明示的な保証サポートが得られます。 |
| 2014–2016 | 農業に対するTPP/FTAの圧力。 農業近代化への適応をサポートするために使用される保証システム。 |
| 2018 | 卸売市場法改正／種子法廃止／漁業法改正など――農政再編の時代。 |
| 2020–2022 | 新型コロナウイルス時代の災害保証は、農業サプライチェーンの混乱（特に乳製品の供給過剰、和牛や園芸に影響を与える食品サービスの崩壊）に対して発動されました。 |
| 2022–2023 | エネルギー/肥料/飼料価格ショック — 保証システムは畜産および温室園芸部門の流動性をサポートします。 |
| 2024 | スマート農業・スマート農業ローン商品に保証サポートがつきます。 |
| 2024–2025 | 農業改革（基本法改正の方向性、食料安全保障への焦点） — 食料安全保障への投資支援に位置付けられた保証制度。 |
| 2025 | 後継者優先 + 法人化優先 — 企業経営/後継者主導の構造改革に合わせた保証制度。 |

## 8。 一般的な中小企業信用保証制度との比較

| 寸法 | 農業信用保証制度 | [[policy-finance/japan-credit-guarantee-system\|General SME credit guarantee system]] |
|---|---|---|
|借り手のターゲット |農家・農業法人・農業関連事業者 |中小企業・小規模事業者（広範な分野） |
| 現地保証人 | 農業信用基金協会 (47 県) | 信用保証協会 (51 県/市) |
| National federation | 全国農業信用基金協会連合会 | [[policy-finance/national-federation-credit-guarantee-corporations\|全国信用保証協会連合会 (JFG)]] |
|国営再保険会社 |農林漁業信用基金（JAFFIC） | [[financial-regulators/jfc\|JFC]] 信用保険 |
| Supervisor | MAFF (農林水産省) | METI / SME Agency (中小企業庁) |
| 貸し手パートナー | JA / 信連 / 農林中央金庫 / 銀行 / 日本政策金融公庫 ag-arm | 地方銀行 / 信用金庫 / 信用組合 / 日本政策金融公庫 |
| 危機時代の旗艦 | 災害・家畜疾病・食糧安全保障 | セーフティネット / 危機関連 (ZeroZero 2020–2022) |
| 資金調達の経済学 | 公的 / 系統金融補助金 | 公的補助＋信用保険再保険 |

この 2 つのシステムは、異なるセクター、異なる監督機関、および異なる協同組合銀行業務インターフェースを備えた **並行する公的信用保証インフラストラクチャ**です。 借り手をめぐって競合することはありません。農家への融資は農業システムを経由し、小規模製造業者への融資は中小企業システムを経由します。

## 9。 境界ケース

- **JA バンクではありません** — JA バンクは融資ルートです。 このページは、最上位に階層化された保証保険メカニズムです。
- **SME 保証制度ではありません** — SME は [[policy-finance/japan-credit-guarantee-system]] までのルートを保証します。
- **漁業ではありません** — 漁業保証保険は [[policy-finance/fisheries-credit-guarantee-system]] (同じ JAFFIC 親会社の下にある姉妹部門) に属します。
- **林業ではありません** — 林業信用支援も JAFFIC の事業ですが、農林水産省の林業政策の下で別個にルーティングされます。
- **JFC の農業融資ではありません** — JFC の [[financial-regulators/jfc|agriculture-forestry-fisheries arm]] は農家に直接融資します。 このページは、日本政策金融公庫が保証融資を行う場合に重複する可能性のある農家融資に対する保証保険についてのページです。
