# 全国信用保証協会連合会

## ウィキ上の位置づけ

この項目は、日本の信用保証制度における**連合会 / 調整機関**ページとして [[policy-finance/INDEX|policy-finance index]] の下に位置づけられる。[[policy-finance/japan-credit-guarantee-system|Japan credit guarantee system]] (メカニズムページ) と [[policy-finance/japan-policy-finance-system|Japan policy finance system]] (より広い外縁) と組み合わせる。[[policy-finance/agriculture-credit-guarantee-system|agriculture]] と [[policy-finance/fisheries-credit-guarantee-system|fisheries]] の保証を対象とするセクター別連合会と役割を比較する。地方の会員 CGC には [[policy-finance/hokkaido-credit-guarantee-corp|Hokkaido CGC]]、[[policy-finance/tokyo-credit-guarantee-corp|Tokyo CGC]]、[[policy-finance/osaka-credit-guarantee-corp|Osaka CGC]]、[[policy-finance/aichi-credit-guarantee-corp|Aichi CGC]]、[[policy-finance/hyogo-credit-guarantee-corp|Hyogo CGC]] が含まれる。

## 要約

全国信用保証協会連合会 (一般社団法人全国信用保証協会連合会, **JFG** / **全信保連**) は、日本の 51 地方信用保証協会 (47 都道府県 + 4 市 CGC — 横浜 / 川崎 / 名古屋 / 岐阜) の全国連合会である。これは**保証制度そのものではない**。51 地方 CGC の上に位置し、[[JapanFG/jfc|JFC]] 信用保険と METI / 中小企業庁政策に接続する、調整、調査、研修、システム管理、金融支援の主体である。運用上最も重要な機能は、**責任共有制度**の全国的な金融フロー管理である。2020–2022 の ZeroZero ローン展開時、JFG は 51 CGC 全体に統一的な運用基準を設定する連合会レベルの調整主体であり、現在は代位弁済の波が 2023–2026を通過する中でポスト COVID 出口枠組みを調整している。

## 2. 51 会員 CGC list

51 会員 CGC は、すべての 47 都道府県と 4 指定都市をカバーする。

- **47都道府県**: 北海道, 青森, 岩手, 宮城, 秋田, 山形, 福島, 茨城, 栃木, 群馬, 埼玉, 千葉, 東京, 神奈川 (Yokohama/Kawasaki city CGCs は別会員のため除く), 新潟, 富山, 石川, 福井, 山梨, 長野, 岐阜 (Gifu city CGC は除く), 静岡, 愛知 (Nagoya city CGC は除く), 三重, 滋賀, 京都, 大阪, 兵庫, 奈良, 和歌山, 鳥取, 島根, 岡山, 広島, 山口, 徳島, 香川, 愛媛, 高知, 福岡, 佐賀, 長崎, 熊本, 大分, 宮崎, 鹿児島, 沖縄.
- **4 市**: 横浜市信用保証協会, 川崎市信用保証協会, 名古屋市信用保証協会, 岐阜市信用保証協会.

4 市 CGC が存在するのは、制度創設の歴史的時点で、四市が都道府県レベル CGC から独立して信用保証協会を支えられるほど大きく独自の産業基盤を持っていたためである。都道府県 CGC (Kanagawa CGC, Aichi CGC, Gifu CGC) との併存は運用上通常であり、市 CGC と都道府県 CGC は地理的 / 行政境界で区別される。

## 3. JFG 役割マップ

| 役割 | 詳細 |
|---|---|
| 会員支援 | 51 地方 CGC への金融 / 運用 / IT / 研修支援。標準運用手順。リスク管理調整。 |
| 責任共有制度 administration | CGC と貸し手の間の 80% / 20% 責任共有金融フローを全国管理し、中央で照合する。 |
| 調査 / 出版 / 統計 | CGC 集計統計 (保証残高、代位弁済、求償、求償権回収率、セクター別内訳)、業界調査、SME 金融出版物。 |
| 連合会プール資金 | 制度を支える連合会レベルのプール型金融資源 (資本注入、緊急支援、損失プール) を管理する。 |
| 貸し手調整 | 全国銀行協会、信金中央金庫、NCB (信組)、その他貸し手機関と、制度全体の商品 / プロセス変更を調整する。 |
| JFC 連携 | 信用保険契約管理と運用調整に関する、[[JapanFG/jfc\|JFC]] との主要な連合会レベル接点。 |
| METI / 中小企業庁連携 | 政策実施の接点。全国 SME 金融政策における 51 CGC の中央的な声。 |
| 危機モード調整 | 危機期プログラム (1998 安定化特別保証, 2008 緊急保証, 2011 災害関係, 2020 ZeroZero) の連合会レベル調整主体。 |
| 標準化商品設計 | 創業、事業承継、ABL などの全国展開向け共通テンプレートプログラムを設計する。 |
| 情報システム / 連合会システム | CGC 業務を支える連合会レベルの IT システムを運用する。 |

## 4. ZeroZero 全国展開 2020 → 出口枠組み 2025

JFG は、**2020–2022 ZeroZero loan** の展開 (実質無利子・無担保融資) における連合会レベルの運用調整主体だった。制度レベルの仕組みは [[policy-finance/japan-credit-guarantee-system]] を参照。JFG の役割:

| フェーズ | JFG の対応 |
|---|---|
| プログラム設計 (2020-04) | ZeroZero 申請書、貸し手-CGC 運用インターフェース、適格性確認、書類基準を連合会レベルで標準化。 |
| 急速展開 (2020-05 → 2020-12) | 51-CGC の処理能力を連合会レベルで監視。制度全体のボトルネック調整。指標標準化。 |
| 継続運用 (2021) | 連合会レベルの統計集計。派生プログラム (延長、地域上乗せプログラム) の調整。 |
| 組成終了 (2021-03 → 2022) | プログラム終了メカニクスの連合会レベル調整。巻き戻しに向けた移行計画。 |
| ワークアウト / 巻き戻し枠組み (2022–2025) | 借り手ごとのワークアウト (借換 / 経営改善 / 事業承継 / 廃業メニュー) に関する連合会レベルの指針。「粘り強い支援」ドクトリン。 |
| 代位弁済波の調整 (2023–2026) | 代位弁済量の連合会レベル統計集計。信用保険請求処理について JFC と調整。残存政策支援について METI と調整。 |
| 破産データ連携 | ZeroZero 関連破産分類について、破産 / 再建データ提供者と連合会レベルで調整。 |

ポスト COVID 出口枠組み (おおむね 2022以降) は、執行主導の回収ではなく、**借り手ごとの粘り強いワークアウト**を重視する。これは、(a) 巻き戻し期間を通じて存続可能な SME を守る政策方向と、(b) 大量デフォルトシナリオが CGC と JFC の処理能力を圧倒するという認識の双方を反映する。この枠組みには、JFG が 51 CGC 全体で調整する、借換保証 / 経営改善 / 事業承継 / 廃業の具体的な商品メニューが含まれる。

## 5. 保証残高による会員 CGC ランキング

51 会員 CGC のうち、保証残高で最大級の協会には通常以下が含まれる。

| ランク層 | CGC (例示) | 理由 |
|---|---|---|
| 上位 5 (都市・産業の重量級) | [[policy-finance/tokyo-credit-guarantee-corp\|Tokyo CGC]], [[policy-finance/osaka-credit-guarantee-corp\|Osaka CGC]], [[policy-finance/aichi-credit-guarantee-corp\|Aichi CGC]] (+ Nagoya City CGC overlap), Kanagawa CGC (+ Yokohama / Kawasaki City CGCs), Saitama CGC | 最大の都市 SME 人口、高い産業集積、ZeroZero 期の厚い組成。 |
| 中位層 (大規模地域 / 地域工業地帯) | Hyogo, Chiba, Fukuoka, Shizuoka, [[policy-finance/hokkaido-credit-guarantee-corp\|Hokkaido]] | 中密度の工業 / 商業 / 第一次産業ミックスを持つ大きな SME 基盤。 |
| 小規模県層 | Tōhoku rural (Akita, Iwate など), Shikoku, 日本海側県 | SME 人口は小さく、絶対保証残高は低いが、一人あたり政策的重要性は高い。 |

具体的な順位は年とプログラム構成で変動する。上位 5 の顔ぶれ (Tokyo / Osaka / Aichi / Kanagawa / Saitama) は安定しており、Hyogo / Chiba / Fukuoka が近接順位で入れ替わる。ZeroZero 期の組成は、2020–2022, に相対残高の順序へ大きく影響し、2023–2026 を通じた代位弁済の波が現在はローリング残高推移に見えている。

## 6. JFC 信用保険との相互作用

**JFC 信用保険業務**は、CGC 代位弁済損失の残余を吸収する再保険レイヤーである。JFG と JFC の接点:

- 51 CGC 全体に適用される信用保険契約条件の連合会レベル調整。
- 保険請求処理基準とタイミングの調整。
- JFC の保険数理 / 準備金評価へ入力される連合会レベルの統計集計。
- 危機関連保証 100% 保証設計と対応する信用保険条件など、信用保険の経済性に連合会レベルの整合が必要な危機モード商品設計の調整。
- CGC + JFC 制度経済に関する METI との共同政策接点。

JFC は、前身の中小企業信用保険公庫 (1958) から信用保険業務を承継した。同公庫は政策金融再編の一環として 2008 に JFC へ統合された。信用保険業務は現在、JFC の直接 SME / 小規模事業 / 農林漁業 / 国際融資と並んで JFC 内で運営されている。[[JapanFG/jfc]] を参照。

## 7. 年次進化

| 年 | 出来事 |
|---|---|
| 1937 | 東京信用保証協会 — 最初の信用保証協会。 |
| 1953 | 信用保証協会法制定。全国 CGC 枠組み。 |
| 1951 → | 全国連合会の前身組織。 |
| 1958 | 中小企業信用保険公庫設立 (JFC 信用保険の前身)。 |
| 1998–1999 | 安定化特別保証 — アジア危機時の緊急 100% 保証プログラム。JFG が全国展開を調整。 |
| 2007-10 | 責任共有制度導入。JFG が金融フローを管理。 |
| 2008–2009 | 緊急保証 — GFC 緊急プログラム。 |
| 2008-10 | JFC 設立。信用保険が JFC に統合。JFG-JFC 接点が再編。 |
| 2011 | Tōhoku earthquake。災害関係保証が広く発動。JFG が連合会レベル対応を調整。 |
| 2020-05 | ZeroZero プログラム開始。JFG が急速な全国展開を調整。 |
| 2021-03 | ZeroZero 組成終了。 |
| 2022–2025 | ポスト COVID 出口枠組み — 粘り強い支援ドクトリン。 |
| 2023–2026 | 代位弁済の波。JFG が連合会レベルのワークアウト / 再建を調整。 |
| 2024–2025 | 経営者保証ガイドラインに沿った保証、事業承継保証、創業保証への継続的注力。 |

## 8. 境界事例

- **JFC ではない** — JFC は信用保険と直接融資を運営する。JFG は連合会である。両者は接続するが別個の主体である。
- **地方 CGC ではない** — 地方 CGC は借り手向け保証を実行する。JFG はそれらを支援 / 調整する。
- **農業 / 漁業ではない** — 農業 / 漁業の信用保証制度には、JFG / JFC とは別に独自のセクター連合会と JAFFIC 再保険がある。
- **METI / 中小企業庁ではない** — METI は政策監督者である。JFG は 51 CGC を代表して制度を管理する連合会である。
- **貸し手ではない** — JFG は借り手を直接保証しない。地方 CGC が保証する。
