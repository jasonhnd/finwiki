# 地方債協会 (Japan Local Government Bond Association)

## 要約

Japan Local Government Bond Association（公益社団法人地方債協会）は、日本の地方債市場について調整、調査、情報発信を行う公益社団法人である。地方債市場の構造、発行統計、共同発行の枠組み、信用制度の概要に関する標準的な公開情報源であり、FinWiki の [[policy-finance/local-government-bond-market|local government bond market]] ページを読む際の事実上の参照機関でもある。

この項目は [[policy-finance/INDEX|policy-finance index]] 配下にあり、地方債市場の業界団体アンカーとして位置づけられる。これは、[[policy-finance/national-federation-credit-guarantee-corporations|JFG]] が 51 地方信用保証協会を支える位置づけと並行する。

## 1. 制度上の境界

| 項目 | 読み方 |
|---|---|
| 日本語名 | 公益社団法人地方債協会 |
| 英語名 | Japan Local Government Bond Association |
| 法的形態 | 公益社団法人 |
| 設立 | 1948 （戦後再編を経た連続的な系譜） |
| 会員 | 地方債発行に関わる地方公共団体、金融機関、証券会社 |
| 監督経路 | 一般的な社団法人監督。地方財政事項については総務省（MIC）との非公式な調整 |
| FinWiki上の扱い | [[policy-finance/INDEX|policy-finance]] 配下の協会ページであり、公的信用機関ではない |

## 2. 機能マップ

| 機能 | 重要性 |
|---|---|
| 地方債市場の調査・刊行物 | 地方債の発行統計、市場構造、共同発行制度に関する標準的な参照資料を公表する。 |
| 共同発行プログラムの調整 | 共同発行市場公募地方債の枠組みにおいて調整役を担う。 |
| 研修と情報共有 | 発行体職員、引受金融機関、証券会社向けのセミナーや説明会を開催する。 |
| 地方債政策に関する業界の声 | 総務省や財務省との政策議論において、地方債エコシステムの組織化された声として機能する。 |
| [[policy-finance/japan-finance-organization-municipalities|JFM]] と FILP 資料への相互参照 | 資本市場での発行、[[policy-finance/japan-finance-organization-municipalities|JFM]] 融資、財政融資資金の相対的役割を理解するための公開文脈を提供する。 |

## 3. FinWikiにおける重要性

同協会の公表資料は、以下について主要な公開情報源である。

- **地方債市場の構造**。[[policy-finance/local-government-bond-market|local government bond market]] と [[JapanFG/local-bond-market|JapanFG local-bond market]] で扱う内容。
- **共同発行市場公募地方債の仕組み**。連帯債務構造や価格形成を含む。
- **発行体類型別の内訳**（都道府県 / 政令指定都市 / 一般市 / 町 / 村）。
- **統計集計**。[[policy-finance/japan-finance-organization-municipalities|JFM]] 融資データや財務省 FILP 統計と組み合わせて読むためのもの。

[[policy-finance/japan-policy-finance-system|Japan policy finance system]] 分析において、同協会は制度内の他の業界団体と同じ構造的位置にある。貸し手でも規制当局でもなく、市場を読み解けるようにする文書化・調整レイヤーである。

## 4. 制度内の位置づけ

| レイヤー | 役割 | FinWiki上の位置づけ |
|---|---|---|
| 発行体 | 都道府県、政令指定都市、市町村 | [[JapanFG/local-govt-finance]] |
| 発行アレンジャー / 引受会社 | メガバンク、地域銀行、証券会社 | 銀行 / 証券会社ページ |
| 公的信用貸付機関 | [[policy-finance/japan-finance-organization-municipalities|JFM]]、FILP | 公的信用機関ページ |
| 業界団体 / 文書化 | Japan Local Government Bond Association | このページ |
| 規制・監督 | 総務省（地方財政）+ 財務省（FILP / 債券協議） | 政府政策資料 |

## 5. 境界事例

- **信用格付機関ではない**: 同協会は市場構造と統計を公表するが、地方公共団体の発行体に信用格付を付与しない。
- **[[policy-finance/japan-finance-organization-municipalities|JFM]] ではない**: JFM は地方公共団体向けの共同貸付法人である。同協会は業界調整団体である。
- **政府機関ではない**: 国の行政機関ではなく、公益社団法人である。その調整力は規制権限ではなく、会員参加に由来する。
- **全国地方銀行協会ではない**: 地方銀行の業界団体は別組織であり、地方債協会は地方債発行に特化している。

## 6. 未解決の論点

- より多くの小規模地方公共団体が共同発行へのアクセスを求めるなかで、同協会の共同発行調整機能はどう変化するか。
- 将来のグリーン / トランジション地方債の枠組みにおいて、同協会はどのような役割を担うか。
- 同協会の公表統計は、財務省 FILP 統計や [[policy-finance/japan-finance-organization-municipalities|JFM]] 開示とどのように整合されるか。
