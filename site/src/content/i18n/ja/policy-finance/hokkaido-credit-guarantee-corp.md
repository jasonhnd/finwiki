# 北海道信用保証協会 (Hokkaido Credit Guarantee Corporation)

## ウィキ上の位置づけ

この項目は [[policy-finance/INDEX|policy-finance index]] 配下にあり、日本の信用保証制度における 51  都道府県・市区町村レベルの構成員の一つである。制度全体の概要である [[policy-finance/japan-credit-guarantee-system|Japan credit guarantee system]]、調整機関である [[policy-finance/national-federation-credit-guarantee-corporations|Japan Federation of Credit Guarantee Corporations]] とあわせて読む。[[policy-finance/tokyo-credit-guarantee-corp|Tokyo CGC]] および [[policy-finance/osaka-credit-guarantee-corp|Osaka CGC]]（いずれも都市部 / 工業地帯の協会）と比較するとよい。北海道信用保証協会は構造的に異なり、日本で最も面積が大きく人口密度が最も低い都道府県を対象とし、製造業よりも農業、水産業、食品加工、観光への比重が高い。

## 要約

Hokkaido Credit Guarantee Corporation（北海道信用保証協会）は、日本の 51 協会からなる地域信用保証制度における北海道の構成機関である。その保証ポートフォリオは都市部の都府県協会とは構造的に異なる。北海道の中小企業経済は、農業・食品（乳製品、牛肉、じゃがいも、小麦、てん菜、玉ねぎ）、水産・水産加工（太平洋サケ、ホタテ、昆布、ウニ）、観光（インバウンド、札幌 / ニセコのスキー、函館 / 小樽ルート）、そして相対的に薄い製造業基盤に集中しているためである。主要な協調融資先は [[regional-banks/hokkaido-bank|Hokkaido Bank (Hokuhoku FG)]] と [[regional-banks/hokuyo-bank|Hokuyo Bank]] であり、これに北海道内の広範な信用金庫・信用組合網と農業協同組合（[[banking/ja-bank-system-japan|JA Bank]]）の重なりが加わる。

## 1. 機関概要

| 項目 | 内容 |
|---|---|
| 設立 | 1948年（信用保証協会法 1953年施行後に再編） |
| 所管 | 中小企業庁 / 北海道 |
| 法令 | 信用保証協会法 (1953) |
| 本店 | 札幌市中央区（北海道札幌市） |
| 保証残高 | 千億円〜兆円規模 — 連合会全国 51 協会中の中位上位 |
| 加盟金融機関 | 約 30 行（地域銀行、メガバンク道内支店、信用金庫、信用組合、農業協同組合） |
| 連合会加盟 | [[policy-finance/national-federation-credit-guarantee-corporations|全国信用保証協会連合会 (Japan Federation of Credit Guarantee Corporations)]] の会員 |
| 信用保険 | [[financial-regulators/jfc|JFC]] の信用保険業務により再保険される |
| 地理境界 | 北海道は約 83,000 km² に及ぶ一つの都道府県（日本最大）であり、単一の保証協会が全 179  市町村を対象とする |

## 2. 主要保証商品

- **一般保証** — 責任共有制度の下での標準的な中小企業向け銀行融資保証。
- **マル経** — 小規模事業者経営改善資金融資の保証。
- **セーフティネット保証** — 業種不況や指定事由に対応する保証（水産資源ショック、観光崩壊、乳価変動など）。
- **危機関連保証** — 全国的な緊急対応枠組みに基づく危機関連保証（100% 保証）。
- **創業保証** — 創業 / 新規事業保証。観光、フードテック、札幌 / ニセコのホスピタリティ分野で活用が目立つ。
- **事業承継保証** — 事業承継保証。家族経営の水産業者や地方小売中小企業で需要が厚い。
- **経営革新保証** — 経営革新 / 事業転換保証。
- **借換保証** — 借換 / 債務一本化保証。
- **食品加工 / 観光関連特別保証** — 北海道庁と連携した食品加工・観光セクター向け制度。

## 3. 道内産業構造との関係

北海道の中小企業経済は他の日本の都府県とは構造的に異なり、北海道信用保証協会の保証ポートフォリオにもそれが反映されている。

1. **農業 (Agriculture)** — 北海道は日本の乳製品、小麦、てん菜、じゃがいも、玉ねぎの大半を生産し、大豆と牛肉でも大きな比率を占める。中小企業の裾野には、農機販売業者、農薬 / 種苗業者、農業物流、食品グレードの冷蔵倉庫が含まれる。なお、農業者への直接信用の多くは [[banking/ja-bank-system-japan|JA Bank]] / [[cooperative-banks/norinchukin|Norinchukin]] の協同組合制度を通じて流れ、北海道信用保証協会の主たる相手方ではないが、農業周辺のサプライチェーン中小企業は重要な利用者である。
2. **水産業 (Fisheries and marine processing)** — 北海道は漁獲量・金額の両面で日本最大の水産県であり、太平洋サケ、ホタテ、昆布、ウニ、ズワイガニ、ニシンに集中している。水産加工の中小企業クラスター（冷凍、切り身加工、燻製、缶詰、輸出包装）は、漁獲と在庫サイクルの季節性を背景に、保証付き運転資金を利用する。水産業者への直接信用も、別系統の [[policy-finance/fisheries-credit-guarantee-system|fisheries credit guarantee system]] と JF マリンバンク網（[[banking/jf-marine-bank-system-japan|JF Marine Bank system]]）を通じて流れる。
3. **観光 (Tourism)** — 札幌、函館、小樽、旭川、富良野、ニセコのスキークラスターは、インバウンドと国内観光の大きな中小企業群を生み出す。ホテル、旅館、飲食店、スキー事業者、交通、小売、食品土産製造などである。このセグメントは景気循環性が高く、COVID 期の国境閉鎖で最も急激な影響を受けた。
4. **食品加工 (Food processing)** — 北海道の食品加工中小企業の密度は日本で最も高い。乳製品、菓子、ラーメン、ビール原料、水産加工品が含まれ、菓子では Royce' / Ishiya / Rokkatei のようなブランドから、地域のラーメン / ビール / 海産スナックの厚いクラスターまで広がる。
5. **薄い製造業 (Thin manufacturing)** — 北海道の一般製造業基盤は本州の都府県と比べて相対的に薄い。食品以外で最大の製造業集積は苫小牧（製紙、石油化学）と室蘭（鉄鋼）である。

## 4. 協調融資と JFC の重なり

北海道信用保証協会の保証は、通常、北海道で支配的な二つの地域銀行、すなわち [[regional-banks/hokkaido-bank|Hokkaido Bank]]（北陸銀行と並ぶ [[regional-banks/hokuhoku-fg|Hokuhoku FG]] 子会社）と [[regional-banks/hokuyo-bank|Hokuyo Bank]] の融資と並走する。密度の高い北海道内の信用金庫・信用組合網は、長い地方の裾野をカバーしている。本州に比べると北海道でのメガバンクの存在感はかなり薄く、[[megabanks/mufg|MUFG]]、[[megabanks/smfg|SMFG]]、[[megabanks/mizuho-fg|Mizuho]] はそれぞれ札幌支店を置くものの、中小企業カバレッジは二つの地域銀行と信用金庫の裾野が中心である。

農業協同組合系統の銀行制度、すなわち [[banking/ja-bank-system-japan|JA Bank Hokkaido]] / [[cooperative-banks/norinchukin|Norinchukin]] は並行して運営され、農業サプライチェーン中小企業セグメントで重なる。[[banking/regional-bank-consolidation-pattern|regional-bank consolidation pattern]] は部分的に関連する。北海道銀行と北陸銀行によるほくほく FG の統合は、同一都道府県内の救済合併ではなく、県境をまたぐ地理的組み合わせだった。

公的金融の協調融資は [[financial-regulators/jfc|JFC]]（政府系金融機関）を通じて流れる。JFC の中小企業、小規模事業、農林水産の各部門は、北海道の一次産業比重のため、いずれも道内の融資窓口を維持している。[[financial-regulators/shoko-chukin|商工中金 (Shoko Chukin)]] は、特に食品加工と観光において、協同組合が組成する中小企業グループに参加する。

## 5. 代位弁済と求償

保証付きの北海道中小企業がデフォルトすると、北海道信用保証協会は保証契約に基づき、貸付金融機関に代位弁済を行い、その後に求償を進める。回収後の残余損失は、[[financial-regulators/jfc|JFC]] により再保険される連合会プール型の信用保険レイヤーに流れ、[[policy-finance/national-federation-credit-guarantee-corporations|全国信用保証協会連合会]] が 51  協会全体の損失分担と運用基準を調整する。

北海道の代位弁済額には三つの特徴的なショック経路がある。(1) 観光崩壊（国境閉鎖、自然災害、スキーシーズンの天候不順）、(2) 一次産業ショック（水産資源の減少、乳価下落、作柄不振）、(3) 観光イベント起因のエネルギー / 燃料費ショックがホスピタリティの利益率を圧迫する経路である。地方・沿岸部の商業用不動産の二次市場が薄く、回収価値が押し下げられるため、回収も都市部の都府県より構造的に難しい。

## 6. 危機期の対応

COVID-19  の緊急事態（2020–2021）の間、北海道信用保証協会は、人口あたりで見ると実質無利子・無担保（ゼロゼロ融資 / ZeroZero loan）の特別保証プログラムに最も強くさらされた協会の一つだった。観光 / ホスピタリティ / インバウンド供給の中小企業セグメントが、国境閉鎖と国内旅行停止により構造的な打撃を受けたためである。2020–2022,  の間、北海道の観光経済は本州平均より鋭く縮小し、特別保証プログラムはホテル / 旅館 / 飲食店 / 土産製造業の資金繰りショックを吸収した。北海道庁はその上に、北海道信用保証協会を通じて追加の地域別緊急プログラムを重ねた。

2023–2025  にかけてのポスト COVID 正常化では、2020 年組成の ZeroZero 元本返済猶予が終わるにつれ、想定どおり代位弁済の波が生じた。北海道の代位弁済損失プロファイルで注目すべきなのは、観光回復が力強かった一方（インバウンド来訪者数は 2023–2025 にかけて急回復）、事業者規模によってばらつきがあった点である。資本余力のある大規模事業者はギャップを吸収したが、小規模な季節事業者のすべてが完済まで生き残ったわけではない。

危機関連保証の枠組みは、エネルギー価格ショック（2022–2023）に対して再発動された。これは、エネルギー集約的な食品加工業と、暖房依存度の高い地方小売基盤にとりわけ強く影響した。2025  までに、北海道信用保証協会のポートフォリオは、一般保証 / 借換保証 / 事業承継保証のバランスへ戻りつつあり、ニセコ地域への外国資本流入と札幌のスタートアップ活動に結びついた創業保証の利用が増えている。

## 7. 連合会制度におけるアンカー

[[policy-finance/japan-credit-guarantee-system|Japan credit guarantee system]] のアーキテクチャの中で、北海道信用保証協会は独自の運用上の役割を持つ。一次産業比重が高く、観光循環性の強い都道府県に対する連合会の参照テンプレートである。[[policy-finance/national-federation-credit-guarantee-corporations|全国信用保証協会連合会]] が観光崩壊シナリオ（例: 2020  COVID 国境閉鎖ショック）に向けた緊急対応商品の設計を調整する際、北海道信用保証協会の運用経験はテンプレートに反映される。これは、[[policy-finance/hyogo-credit-guarantee-corp|Hyogo CGC]] の 1995 地震対応ドクトリンが連合会の災害対応テンプレートに反映されるのと似ている。

[[financial-regulators/jfc|JFC]] にある連合会プール型の信用保険レイヤーは、責任共有制度の下で北海道信用保証協会が初期損失として行う代位弁済後の残余損失を吸収する。北海道の回収価値問題、すなわち地方商業用不動産の二次市場の薄さにより、絶対的な回収率が都市部の都府県の同業協会より構造的に低く、JFC 再保険バックストップへの比重が大きくなるため、これは運用上重要である。[[banking/ja-bank-system-japan|JA Bank]] / [[cooperative-banks/norinchukin|Norinchukin]] の農業協同組合レイヤーと [[banking/jf-marine-bank-system-japan|JF Marine Bank]] / 水産協同組合レイヤーは、北海道信用保証協会の直接の中小企業保証範囲外にあるセグメントのための並行する公的信用インフラとして機能する。

## 8. 同業都府県信用保証協会との比較

| 観点 | Hokkaido CGC | [[policy-finance/tokyo-credit-guarantee-corp|Tokyo CGC]] | [[policy-finance/aichi-credit-guarantee-corp|Aichi CGC]] |
|---|---|---|---|
| 支配的な産業構成 | 農業 / 水産 / 観光 / 食品加工 | サービス / 小売 / 金融 / IT / ホスピタリティ | 自動車サプライチェーン |
| 地理的課題 | 面積最大の都道府県（約 83,000 km²）。地方北海道では信用金庫カバレッジが薄い | 最も高密度な都市集積 | 中規模県で、工業回廊が密 |
| 一次産業の比重 | 主要 CGC の中で最高 | ごく小さい | ごく小さい |
| 観光ショック感応度 | 非常に高い（国境閉鎖 + スキーシーズンの天候） | 中程度（都市観光） | 低い |
| 協同組合銀行との重なり | 高い（[[banking/ja-bank-system-japan|JA Bank]] / [[banking/jf-marine-bank-system-japan|JF Marine Bank]]） | 低い | 中程度 |
| 回収価値の課題 | 地方資産の二次市場が薄い | 都市部の回収価値が高い | 中都市型の回収価値 |

この比較は、同じ [[policy-finance/japan-credit-guarantee-system|Japan credit guarantee system]] メカニズムの下で運営される都市部の都府県信用保証協会と比べ、北海道信用保証協会の運用ドクトリンと商品構成がなぜ意味あるほど異なるのかを示している。
