# JASSO (日本学生支援機構)

## ウィキ上の位置づけ

この項目は、日本の国家金融システムにおける教育金融 / 人的資本ノードとして [[policy-finance/INDEX|policy-finance index]] の下に位置づけられる。家計側で JFC の国民生活事業 教育一般貸付と重なる家計教育ローン隣接領域は [[financial-regulators/jfc|日本政策金融公庫 (JFC)]]、JFC 教育ローン商品の運用メカニクスは [[policy-finance/jfc-kokumin-life-finance-division|JFC 国民生活事業]]、JASSO が外国人留学生受入で接する留学生 / 海外協力レーンは [[policy-finance/jica|JICA]]、家計向け政策金融 / 証券化支援カテゴリーの同業機関は [[policy-finance/japan-housing-finance-agency|JHF]]、沖縄固有の学生支援の扱いは [[policy-finance/okinawa-development-finance-corp|ODFC]]、より広いツールキットは [[policy-finance/japan-policy-finance-system|Japan policy finance system]]、九機関比較軸は [[policy-finance/japan-policy-finance-institution-mandate-matrix|Japan policy-finance institution mandate matrix]]、JASSO ローン返済行動が家計バランスシートと相互作用する家計信用 / 消費者金融外縁は [[banking/INDEX|banking index]] と照らして読む。

## 要約

JASSO (独立行政法人日本学生支援機構 / Japan Student Services Organization) は、日本の主要な教育金融・学生支援機関であり、文部科学大臣が主管する。2004-04-01に、日本育英会 (Japan Scholarship Foundation、奨学金事業は 1943以来) と、国際学友会 (留学生宿舎 / 交流)、関西国際学友会、内外学生センター、日本国際教育協会を含む複数の学生支援機能を統合して設立された。機関は四つの機能レーンを運営する。**奨学金事業** (貸与型と給付型の学生支援を含む旗艦事業)、**留学生支援事業** (JASSO 管理宿舎と日本人学生の海外留学支援を含む)、**学生生活支援事業**、**統計 / 調査 / 開示機能** (学生支援に関する調査研究 / 情報提供) である。奨学金レーンは三つの類型を中心に整理される。**第一種奨学金** (Type I — 無利子貸与型)、**第二種奨学金** (Type II — 返済期間中の上限が 3.0% の有利子貸与型、実務上は通常かなり低い)、**給付型奨学金** (返済不要の奨学金。2017 から拡充され、2020 「高等教育無償化」/ Higher Education Free-Access Reform で大幅に規模拡大) である。資金は、貸与型奨学金元本向けの FILP 借入、給付型奨学金と運営費向けの一般会計繰入、既存貸与ポートフォリオからの返済キャッシュフロー (数兆円規模) の組み合わせで流れる。JASSO は借り手数で日本最大の単一家計信用機関であり、日本の大学生のおおむね三人に一人が JASSO から借りている。政策金融上の重要性は、2020 改革以降、さらに 2024-2025 以降の「次世代育成 / 子育て世帯支援」/ 高等教育費負担軽減の政策方向 (政治的に追跡されている反転授業料 / 授業料後払い論を含む) の下で急速に高まった。第二種貸与のデフォルト率は、所得連動返還の導入、猶予・救済メカニズム、近年の 2024 改革を伴う複数年の改革サイクルにおいて、繰り返し政策金融上の論点となってきた。

### 1.1 JASSO が行うこと — 行わないこと

- **行うこと**: 貸与型・給付型奨学金事業の運営、返済 / デフォルト / 救済メカニズムの管理、宿舎、EJU (日本留学試験) 運営、日本人学生の海外留学支援を含む留学生支援、学生生活支援 / 障害学生支援 / 相談、国内高等教育在籍、奨学金利用、学生金融に関する全国統計の公表。
- **行わないこと**: 日本の高等教育授業料や入学政策の決定 (これは MEXT 直接 + 個別大学)、大学の設備投資向け融資 (これは民間銀行または私立大学等経常費補助金の経路)、国際開発教育援助の運営 (これは [[policy-finance/jica|JICA]])、日本企業向け教育関連金融の提供 (これは民間 / JFC 教育一般貸付)。

家計教育側の JFC ↔ JASSO の境界は構造的に重要である。JFC 国民生活事業 教育一般貸付は、学生の**親**が家計信用に基づいて借り、授業料 / 生活費の一括資金に使う家計ローン商品である。JASSO 奨学金貸与は、**学生**が将来の稼得能力に基づいて借り、卒業後に返済を繰り延べる。二つの商品経路は代替ではなく補完として設計されており、多くの家計は双方を使う。

## 2. 事業ライン別マンデート

### 2.1 奨学金事業の階層構造

旗艦商品。三つの階層がある。

| 階層 | 種類 | 優遇性 | 適格性 | 返済 |
|---|---|---|---|---|
| **第一種奨学金** | 貸与型、無利子 | 最も優遇的 (0%利息、元本のみ返済) | 所得審査 + 学業成績審査。第二種より適格性が厳しい | 卒業後に返済開始 (通常は猶予期間あり)。標準的に ~15-20 年の返済スケジュール |
| **第二種奨学金** | 貸与型、有利子 | 優遇性は低い。返済中の金利上限は 3.0% だが、実務上は通常かなり低い (近年の低金利環境ではしばしば 1%未満) | より広い所得審査適格性。第一種の対象外家計向けの標準商品 | 卒業後に返済開始。~15-20 年スケジュール。卒業時から利息発生 |
| **給付型奨学金** | 給付型 (返済不要) | 完全優遇 (返済なし、利息なし) | 厳格な所得 / 家計資産審査。2017導入。2020 に高等教育無償化枠組みで大幅拡大 | N/A (返済なし) |

2020 後の給付拡大は、政策金融上の構造的転換だった。2017, 以前、JASSO はほぼ学生ローン運営者だった。2020 高等教育無償化 (Higher Education Free-Access Reform) は、低所得世帯について国公立大学授業料の相当部分と私立大学授業料の一部をカバーする、所得審査付き給付階層を導入した。これにより JASSO の役割は純粋な貸与運営から貸与 + 給付の複合運営へ広がり、財政会計も根本的に変わった。給付支出は一般会計歳出であり、FILP の貸付元本ではない。

### 2.2 留学生支援

- **JASSO 管理の留学生宿舎** (東京国際交流館、大阪国際交流センターなど) — 日本の大学で学ぶ外国人留学生向け宿舎。
- **EJU (日本留学試験)** の運営 — JASSO は、日本の学部課程に出願する外国人留学生が利用するこの試験を運営する。
- **日本人学生の海外留学支援** — 海外で学ぶ日本人学生向けの奨学金プログラムと情報。
- **受入留学生奨学金の調整** (MEXT 奨学金および大学直接奨学金と並行)。

### 2.3 学生生活支援

相談、障害学生支援、ハラスメント防止 / メンタルヘルス支援について日本の大学横断で調整する。学生生活状況に関する統計 / 調査 / 開示、災害時の緊急学生支援調整 (例: 東北 2011 後の学生支援調整) も行う。

### 2.4 統計・調査開示

JASSO は、日本の高等教育在籍、階層別・世帯所得別奨学金利用、返済状況、留学生の受入 / 送出フローに関する全国統計の主要公表主体である。JASSO の開示ページは、日本の高等教育金融分析における標準的なデータソースである。

## 5. 兄弟ページとの比較

このスナップショット時点で、JASSO には FinWiki 内に専用の運用メカニクス兄弟ページがない。[[policy-finance/japan-policy-finance-institution-mandate-matrix|the mandate matrix]] に含まれる九つの政策金融機関のうち、運用メカニクスの深掘りがない唯一の機関である。最も近い比較基点は以下である。

- [[policy-finance/jfc-kokumin-life-finance-division|JFC 国民生活事業]] — 家計教育ローンを運営する JFC 部門 (借入人 = 親 / 保護者)。本土 JFC 構造で最も近い運用メカニクス上の同業。
- [[policy-finance/japan-housing-finance-agency|JHF]] — FILP 資金による家計信用運用を持つ、もう一つの家計向け政策金融機関。制度比較に有用。
- [[policy-finance/japan-policy-finance-institution-mandate-matrix|Mandate matrix]] — JASSO が JFC、DBJ、JBIC、JICA、NEXI、JOGMEC、JHF、ODFC と並ぶ Institution 9 row。

### JFC 教育ローン商品 (最も近い同業) との比較軸

| 軸 | JASSO | [[financial-regulators/jfc\|JFC]] 国民生活事業 教育一般貸付 |
|---|---|---|
| 監督 | MEXT | 四大臣共管 (資本 / ガバナンスは MoF、生活関連業務は MHLW) |
| 法的形態 | 独立行政法人 | 特殊会社 |
| 借り手 | 学生 (将来の稼得能力に基づく) | 親 / 保護者 (家計信用に基づく) |
| 優遇性 | 第一種 0%、第二種は法定上限 ≤ 3.0% (実務上 1%未満)、給付型 0% | 商業銀行より低く、JASSO より高い |
| 返済開始 | 卒業後まで繰延 | 実行時から標準的な元利償還 |
| テナー | 卒業後の典型スケジュールは 15-20 年 | 典型的に ~15 年 |
| 所得連動オプション | あり (第一種 所得連動返還) | なし |
| デフォルト時の帰結 | 猶予 / 救済オプションを持つ法定枠組み | 標準的な JFC 回収 / リストラクチャリング |
| 年間実行規模 | 貸与 ~¥1-1.2 trillion + 給付数千億円 | より小さい (国民生活事業ポートフォリオの家計ローン部分) |

二つの商品経路は補完的に設計されている。JASSO は学生側の直接借入をカバーし、JFC は親 / 保護者の家計信用をカバーする。多くの家計は双方を使う。

## 6. 反論・留意点

- **社会政策金融 ≠ 商業信用。** JASSO の貸与事業は社会政策金融であり、銀行型の商業信用ではない。リスクプロファイルは、返済支援、所得連動メカニズム、政府支援、家計所得条件に依存する。JASSO のデフォルト率を商業的な消費者金融デフォルト率と比較するのは誤解を招く。
- **デフォルトと家計バランスシート集中。** ~¥9-10 trillion の JASSO 貸与簿は、若年成人 / キャリア初期の家計に集中している。デフォルトと延滞率は、家計金融動向、家族形成時期 (卒業債務負担による家計形成遅延は繰り返し社会政策論点となっている)、消費支出能力に直接影響する。構造的な批判は、キャリア初期の JASSO 貸与償還が住宅ローン余力形成、家計形成、裁量的消費支出と競合するというものである。これは貸付のみの財務諸表視点では捉えられないマクロ経済効果である。
- **給付拡大と貸与代替をめぐる議論。** 2020 給付拡大は、貸与を完全には代替していないと批判されてきた。多くの家計は給付と並行してなお貸与を受けている。反対側の議論は、低所得世帯について給付が構造的に貸与を置き換えるべきか (完全な高等教育無償化へ向かう) それとも貸与を補完的に残すべきかである。
- **第二種の金利フロア。** 第二種は法定上 3.0% を上限とするが、実務上は JGB / FILP 資金コスト近辺で価格付けされる。金利正常化環境 (2024 後の BoJ によるマイナス金利と YCC からの退出、その後の 2024-2026利上げ) では、第二種の実効金利は上昇する。近年の借り手は 2015-2020 借り手より高い実効金利に直面する。
- **民営化論。** DBJ (民営化が議題) と異なり、JASSO の独立行政法人形態は構造的に民営化の議題に乗らない。家計教育政策金融は民営化可能な機能ではない。
- **マンデート拡張批判。** 給付拡大と所得連動返還拡大は、JASSO のマンデートを純粋な貸与管理から、所得分配契約に似た特徴を持つ貸与 + 給付の複合運営へ広げた。批判は制度マンデートの漂流であり、反論は高等教育費用環境の変化への政策適応である。
- **BIS Basel III との相互作用 (最小限)。** JASSO 自体は BIS 規制対象ではない。NEXI / JBIC / JFC のようにメガバンク協調融資が Basel III との相互作用面を作る機関と異なり、JASSO はほぼ完全に直接借り手関係の機関として運営される。したがって Basel III との相互作用は最小限である。ただし、大規模な JASSO 貸与償還が家計バランスシートに与える影響は、より広い銀行システム全体の住宅ローン組成量に影響する。
- **反転授業料をめぐる議論。** 2024-2025 政策議論上の「反転授業料」(授業料後払い / 所得分配契約型授業料) は、現在政治的に追跡されているが未実施である。反転授業料制度の下では、JASSO の制度的役割は、卒業後の所得連動拠出を通じて高等教育の公的資金を管理する方向へ移る。これは構造的に異なる制度商品である。政策方向は不確実である。

## 7. 未解決の論点

- JASSO 貸与ポートフォリオの償還軌道が、2020 後の給付拡大コホートと 2017 前の貸与のみコホートの間で、家計金融 / 住宅ローン組成能力にどう影響するか。
- 給付型奨学金が低所得世帯向け支援を構造的に貸与から置き換えているのか、それとも併存しているのか。
- 日本の労働 / 入管政策 (2023 後の入管枠組み、2024 後の高度人材 / 特定技能拡大) に伴い、留学生支援がどう変化するか。
- 国会立法が進む場合、反転授業料 / 所得分配契約改革の実施経路と時期。
- 金利正常化環境下における第二種のデフォルト / 延滞率の軌道。
- 家計教育金融の二重計上と家計信用負担測定に関する [[policy-finance/jfc-kokumin-life-finance-division|JFC 国民生活事業]] 教育一般貸付との調整。
- 2020 後の給付拡大が、低所得世帯の高等教育進学に測定可能な変化を生むか (意図された政策成果)、それとももともと進学したであろう世帯への代替になるか。
