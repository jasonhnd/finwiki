# 日本における漁業信用保証制度

## Wiki ルート

このエントリは [[policy-finance/INDEX|policy-finance index]] の下に位置する。SME サイドの対比については [[policy-finance/japan-credit-guarantee-system|Japan credit guarantee system]]、姉妹セクター（同じ JAFFIC 親）については [[policy-finance/agriculture-credit-guarantee-system|agriculture credit guarantee system]]、より広範な公的信用の境界については [[policy-finance/japan-policy-finance-system|Japan policy finance system]] と対照して読まれたい。漁業の預金・貸出の対応物は [[banking/jf-marine-bank-system-japan|JF Marine Bank system]] であり、協同組合のアペックスは [[cooperative-banks/norinchukin|Norinchukin]]（農業・林業・漁業協同組合を共同で担う）である。

## TL;DR

日本の漁業信用保証制度（漁業信用保証保険制度）は、二層構造を通じて中小漁業者の信用を補完する：(1) 漁業者の債務を貸出機関に対して保証する都道府県・広域レベルの**漁業信用基金協会**（Fisheries Credit Fund Associations）で、その全国連合体としての**全国漁業信用基金協会連合会**があり、(2) 国レベルの**独立行政法人農林漁業信用基金**（JAFFIC）の漁業信用保険業務によって再保険される。この制度は [[policy-finance/agriculture-credit-guarantee-system|agriculture credit guarantee system]] と並行するが、セクター固有の借り手ターゲット — 漁船金融、養殖、水産加工 — を持ち、地理的には北海道、九州（長崎 / 鹿児島）、東北太平洋岸、三陸海岸が保証ボリュームの大半を占めて日本の沿岸都道府県に集中している。

## 1. 系統金融との位置関係（fisheries variant）

日本の漁業金融は、農業のスタックと並行する**漁業系統金融**のスタックを通じて流れる：

| 層 | 機能 | FinWiki ルート |
|---|---|---|
| 漁業者 / 漁業法人 / 加工業者 | 借り手（漁業者、漁業法人、養殖業者、水産加工業者、適格協同組合）。 | システムルートはここ |
| 単位漁協（JF / 単漁協） | 一次漁業協同組合；組合員の預金を受け入れる；漁業関連の信用を供与する；水揚げ港の物流と魚市場の機能を運営する。 | [[banking/jf-marine-bank-system-japan]] |
| 信漁連（都道府県信用漁業協同組合連合会） | 漁業協同組合の都道府県信用連合会。 | [[banking/jf-marine-bank-system-japan]] |
| 農林中央金庫（Norinchukin） | JA / 信連 / JF / 信漁連 / 森連の協同組合の国レベルのアペックス銀行；農業・漁業・林業の協同組合を共同で担う。 | [[cooperative-banks/norinchukin]] |
| 漁業信用基金協会 | 漁業ローンのための都道府県 / エリアの保証主体；デフォルト時に代位弁済を行う。 | このページ |
| 全国漁業信用基金協会連合会 | 漁業信用基金協会の全国連合体。 | このページ |
| 独立行政法人農林漁業信用基金（JAFFIC） | 国レベルの信用保険 / 再保険主体；農業システムと同じ JAFFIC 親。 | このページ |
| 水産庁 / MAFF | 漁業協同組合システム + 保証保険システムの政策監督者。 | [[policy-finance/INDEX]] |

## 2. 機関概要 — 漁業信用基金協会

| 項目 | 内容 |
|---|---|
| 設置 | 都道府県 / 広域 — 全国の漁業信用基金協会数は agriculture と異なり、複数都道府県を統合した広域協会を含む |
| 法令 | 中小漁業融資保証法（1952） |
| 法的形態 | 特別法上の法人 |
| 所管 | 水産庁（Fisheries Agency）/ 都道府県 |
| 主要業務 | 漁業者 / 漁業法人 / 関連事業者 への融資保証 |
| 連合会 | 全国漁業信用基金協会連合会（zengyoki.or.jp） |
| 再保険 | 農林漁業信用基金（JAFFIC）漁業信用保険業務 |

## 3. 機関概要 — JAFFIC 漁業信用保険業務

農業と同じ親機関（独立行政法人農林漁業信用基金 / JAFFIC）。JAFFIC 内に別個の**漁業信用保険**勘定があり、別個の再保険プールと別個の都道府県協会インターフェースを持つ。農業勘定の対応物については [[policy-finance/agriculture-credit-guarantee-system]] を参照されたい。

## 4. 主要保証商品

- **漁業近代化資金保証** — 漁船建造、エンジン交換、漁具のアップグレード、冷蔵、船上加工のための近代化ローン保証。
- **漁業経営改善支援資金保証** — 漁業経営改善ローン保証。
- **沿岸漁業改善資金保証** — 沿岸漁業改善ローン保証。
- **新規就業者支援資金保証** — 新規就業漁業者 / 養殖業者支援保証。
- **養殖関連融資保証** — 養殖施設 / 飼料 / 種苗の運転資金保証（ホタテ / カキ / ブリ / 昆布 / 真珠のセグメントで多用）。
- **水産加工 / 流通融資保証** — 水産加工および流通の運転資金保証。
- **災害復旧保証** — 災害復旧保証（台風被害、海洋汚染、赤潮、船舶の喪失、漁港被害）。
- **燃油価格高騰対策保証** — 燃油価格ショックの流動性保証。
- **借換保証** — リファイナンス保証。

## 5. 製品ミックスの特徴

漁業保証ブックは、農業ブックとは異なる製品ミックスのプロファイルを持つ：

- **漁船金融**は重く担保されているが、減価する資産価値を伴う；保証条件は船舶のライフサイクルと、専門化された船舶の再販市場の薄さを反映する。
- **養殖金融**は地理的に高度に集中し（北海道と三陸のホタテ / 昆布；広島と三重のカキ / 真珠；鹿児島と愛媛のブリ；和歌山と長崎のマグロ養殖）、環境ショック（赤潮、海洋熱波、疾病の発生）にさらされる。
- **加工金融**は一般的な SME の運転資金需要と重なるが、漁獲サイクルと輸出市場向けの在庫積み増しに結びついた季節性を伴う。

## 6. 地理的集中

ほぼすべての都道府県に地理的に分布する農業とは異なり、漁業保証制度は**沿岸都道府県に重く集中している**：

| 地域 | 漁業の性格 | 保証ブックの重点 |
|---|---|---|
| 北海道 | 太平洋サケ、ホタテ、コンブ、ウニ、カニ — ボリュームと価値で日本最大の漁業都道府県。 | 漁船金融 + ホタテ / コンブ養殖 + 加工。 |
| 三陸海岸（岩手、宮城） | 沿岸漁業；カキ、ホタテ、コンブ養殖；サケ；サンマ。 | 2011後の地震復興保証 + 養殖。 |
| 九州（長崎、鹿児島、福岡、佐賀） | マグロ養殖、ブリ養殖、沿岸漁業、遠洋漁船。 | 養殖 + 漁船金融 + 輸出加工。 |
| 日本海（鳥取、島根、富山） | ズワイガニ、ホタルイカ、ブリ。 | 漁船 + 季節加工保証。 |
| 太平洋中部（千葉、静岡、高知） | カツオ、マグロ、サバ；遠洋漁船。 | 遠洋漁船のための漁船金融。 |
| 内海（広島、愛媛、三重） | カキ、真珠、ブリ。 | 養殖に重い保証ブック。 |

地理的集中は、漁業保証制度のポートフォリオリスクが、単一の沿岸エリアの複数の保証に同時に影響しうる**地域的な海洋環境ショック**にさらされることを意味する。

## 7. 年表の進化

| 年 | 事象 |
|---|---|
| 1952 | 中小漁業融資保証法 制定；都道府県漁業信用基金協会 設立。 |
| 1972 | 漁業信用基金 が国レベルで設立。 |
| 1990年代 | 200海里 EEZ 制度 / 遠洋漁業の再構築；保証の再構築によって支援された船舶削減政策。 |
| 2003-10-01 | 独立行政法人農林漁業信用基金（JAFFIC） が農業・林業・漁業の信用保険主体を統合して創設。 |
| 2011-03-11 | 東北地震 / 津波が三陸沿岸漁業を壊滅 — 漁業保証制度が復興融資の中核に。 |
| 2018 | 漁業法 70年ぶりの抜本改正 — TAC / IQ の資源管理近代化；養殖ライセンス改革；保証制度が新しい資源フレームワークに適応。 |
| 2020–2022 | COVID 時代の外食崩壊がマグロ / ブリ / ホタテ / カキの輸出市場を直撃 — 保証制度が流動性を支援。 |
| 2022 | エネルギー / 燃油価格ショック；燃油価格高騰保証が発動；沿岸漁業の営業マージンが崩壊。 |
| 2023 | 福島 ALPS 処理水放出 → 中国の海産物輸入禁止 → ホタテ / ナマコの輸出ショック — 保証制度が影響を受けた事業者（特に北海道 / 青森）の運転資金ブリッジを支援。 |
| 2024 | 継続する ALPS 関連の輸出市場の回復；保証ブックが進行中の復興を吸収。 |
| 2024–2025 | コンブ / ホタテ / 貝類に影響する海洋熱波ショック；保証制度が営業損失のブリッジファイナンスをカバー。 |
| 2025 | 食料安全保障戦略としての養殖拡大（国産養殖振興）への継続的な政策フォーカス；保証制度が整合。 |

## 8. 農業信用保証制度との比較

| 次元 | 漁業信用保証制度 | [[policy-finance/agriculture-credit-guarantee-system\|Agriculture credit guarantee system]] |
|---|---|---|
| 借り手ターゲット | 漁業者 / 漁業法人 / 養殖業者 / 水産加工業者 | 農家 / 農業法人 / 関連事業者 |
| ローカルな保証人 | 漁業信用基金協会（都道府県 / 広域） | 農業信用基金協会（47 都道府県） |
| 全国連合体 | 全国漁業信用基金協会連合会 | 全国農業信用基金協会連合会 |
| 全国再保険者 | JAFFIC 漁業信用保険業務 | JAFFIC 農業信用保険業務 |
| 監督者 | 水産庁 / MAFF | MAFF |
| 貸出パートナー | JF / 信漁連 / Norinchukin / 銀行 / JFC の農林漁業部門 | JA / 信連 / Norinchukin / 銀行 / JFC の農林漁業部門 |
| 地理的範囲 | 沿岸都道府県に集中 | 全国 |
| 資産の特徴 | 船舶（減価、専門化）+ 養殖（環境にさらされる） | 土地 + 農機具 + 家畜 |
| 危機のベクトル | 資源崩壊 / 赤潮 / 台風 / 油流出 / 輸出市場ショック（ALPS 水事案） | 天候 / 疾病 / TPP-FTA / 食料価格 |
| 典型的な規模 | より小さい個別ローンサイズ；集中したポートフォリオ | より広い借り手基盤；より多様なローンサイズの分布 |

両システムは同じ JAFFIC 再保険親を共有するが、異なるアクチュアリアル前提と異なる都道府県ネットワークのカバレッジを持つ**別個の保険勘定**を運用する。

## 9. 境界事例

- **JF マリンバンクではない** — JF マリンバンクは貸出ルートである；このページはその上に重ねられる保証保険のメカニズムである。
- **農業ではない** — 農業の保証保険は [[policy-finance/agriculture-credit-guarantee-system]]（姉妹セクター、同じ JAFFIC 親）に属する。
- **通常の SME 保証ではない** — 通常の SME 保証は [[policy-finance/japan-credit-guarantee-system]] に属する。
- **森林（forestry）ではない** — 林業の信用支援も JAFFIC のもとだが、別個の勘定である。
- **JFC の漁業貸出ではない** — JFC の [[financial-regulators/jfc|agriculture-forestry-fisheries arm]] は直接貸し出す；このページは漁業の借入に対する保証保険についてである。
