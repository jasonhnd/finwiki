# 日本の漁業信用保証制度
## ウィキ上の位置づけ
この項目は [[policy-finance/INDEX|policy-finance index]] の下にあります。中小企業側の対比は [[policy-finance/japan-credit-guarantee-system|Japan credit guarantee system]]、姉妹セクター (同じ JAFFIC 親会社) は [[policy-finance/agriculture-credit-guarantee-system|agriculture credit guarantee system]]、より広範な公的信用境界線は [[policy-finance/japan-policy-finance-system|Japan policy finance system]] と対比して読み取ってください。漁業預貸先は[[banking/jf-marine-bank-system-japan|JF Marine Bank system]]、協同組合頂点は[[JapanFG/norinchukin|Norinchukin]]（農林漁協共同）です。
## 要約
日本の漁業信用保証制度（漁業信用保証保険制度）は、二層構造を通じて中小漁業者の信用を補っています: (1) 都道府県および広域レベルの **信用基金協会** (漁業信用基金協会) は、融資機関に対する漁業の借り手の債務を保証し、**全国漁業信用基金協会連合会**を全国連合として、全国レベルの機関 (2) によって再保険されています。 **独立行政法人農林漁業信用基金** (JAFFIC) 漁業信用保険事業。この制度は[[policy-finance/agriculture-credit-guarantee-system|agriculture credit guarantee system]]と類似しているが、融資対象者が漁船ファイナンス、養殖業、水産加工業など分野別であり、地理的には北海道、九州（長崎・鹿児島）、東北太平洋岸、三陸沿岸など日本の沿岸部に集中しており、保証額の大部分を占めている。
## 1。 系統金融との位置関係 (漁業バリアント)
日本の漁業金融は、農業スタックと並行する **漁業系統金融** スタックを介して流れています。

## 2。 機関概要 — 漁業信用基金協会
| 項目 | 内容 |
|---|---|
| 設置 | 都道府県 / 広域 — 全国の漁業信用基金協会数は農業と異なる、複数都道府県を統合した広域協会を含む |
| 法令 | 中小漁業融資保証法 (1952) |
| 法的形態 | 特別法上の法人 |
| 所管 | 水産庁 / 都道府県 |
| 主な業務 | 漁業者 / 漁業法人 / 関連項目事業者への融資保証 |
| 連合会 | 全国漁業信用基金協会連合会 (zengyyoki.or.jp) |
| 再保険 | 農林漁業信用基金 (JAFFIC) 漁業信用保険業務 |
## 3。 機関概要 — JAFFIC漁業信用保険業務
農業と同じ母体機関（独立行政法人農林漁業信用基金/JAFFIC）。 JAFFIC 内の別個の **漁業信用保険** アカウント。別個の再保険プールと別個の県協会インターフェースを備えています。農業アカウントに対応するものについては、[[policy-finance/agriculture-credit-guarantee-system]] を参照してください。
## 4。 主要保証商品
- **漁業近代化資金保証** — 漁船の建造、エンジン交換、装備のアップグレード、冷凍、船上処理に対する近代化融資保証。
- **漁業経営改善支援資金保証** — 漁業経営改善融資保証。
- **沿岸漁業改善資金保証** — 沿岸漁業改善融資保証。
- **新規参入者支援資金保証** — 新規参入漁業者/養殖業者サポート保証。
- **養殖関連項目保証保証** — 水産養殖施設 / 飼料 / 稚魚の運転資金保証 (ホタテ貝 / 牡蠣 / ブリ / 昆布 / 真珠のセグメントに多用)。
- **水産加工/流通融資保証** — 水産加工および流通の運転資本保証。
- **災害復旧保証** — 災害復旧保証（台風による被害、海洋汚染、赤潮、船舶の損失、港湾の損傷）。
- **燃料価格ショック対策保証** — 燃料価格ショック流動性保証。
- **借換保証** — 借り換え保証。
## 5。 プロダクトミックスの特徴
漁業保証簿には、農業保証簿とは異なる製品構成プロファイルがあります。
- **漁船ファイナンス**には多額の担保がかけられていますが、資産価値は下落しています。保証条件は、船舶のライフサイクルと特殊船舶の再販市場の薄さを反映しています。
- **水産養殖金融**は地理的に非常に集中しており（北海道と三陸のホタテ貝/昆布、広島と三重のカキ/真珠、鹿児島と愛媛のブリ、和歌山と長崎のマグロ養殖）、環境ショック（赤潮、海洋熱波、疾病の発生）にさらされている。
- **処理金融**は一般的な中小企業の運転資本のニーズと重複しますが、漁獲サイクルと輸出市場向けの在庫構築に関連項目した季節性を伴います。
## 6。地理的集中
地理的にほぼすべての都道府県に分布している農業とは異なり、漁業保証システムは**沿岸県に大きく集中しています**。
| 地域 | 水産キャラクター | 保証本の強調 |
|---|---|---|
| 北海道 | パシフィックサーモン、ホタテ、昆布、ウニ、カニ - 量と金額で日本最大の水産県。 | 船舶金融＋ホタテ・昆布養殖＋加工。 |
| 三陸海岸（岩手県、宮城県） | 沿岸漁業。カキ、ホタテ、昆布の養殖。鮭;さんま。 | 2011後～震災復興保証＋養殖。 |
| 九州（長崎、鹿児島、福岡、佐賀） | マグロの養殖、ブリの養殖、沿岸漁業、外洋漁船団。 | 養殖+船舶ファイナンス+輸出加工。 |
| 日本海（鳥取・島根・富山） | ズワイガニ、ホタルイカ、ブリ。 | 器+季節加工保証。 |
| 太平洋中部（千葉県、静岡県、高知県） | カツオ、マグロ、サバ。遠洋艦隊。 | 遠洋艦隊向けの船舶ファイナンス。 |
| 内海（広島、愛媛、三重） | 牡蠣、真珠、ブリ。 | 養殖に特化した保証書。 |
地理的な集中は、漁業保証システムのポートフォリオ リスクが、単一の沿岸地域の複数の保証に同時に影響を与える可能性がある**地域的な海洋環境ショック**にさらされていることを意味します。
## 7。年の進化
| 年 | イベント |
|---|---|
| 1952 | 中小漁業融資保証法制定。都道府県漁業信用基金協会設立。 |
| 1972 | 国家レベルで設立された漁業信用基金。 |
| 1990 | 200海里EEZ体制/遠洋漁業の再編。船舶削減政策は保証の再構築によって支えられている。 |
| 2003-10-01 | 独立行政法人農林漁業信用保険基金（JAFFIC）は、農林漁業信用保険機関を統合して設立されました。 |
| 2011-03-11 | 東北地方の地震と津波で三陸沿岸漁業が壊滅的打撃を受ける―復興融資の中核となる漁業保証制度。 |
| 2018 | 漁業法 70 年根本改訂 — TAC / IQ 資源管理の近代化。養殖ライセンスの改革。保証システムは新しいリソースフレームワークに適応します。 |
| 2020～2022 | 新型コロナウイルス時代の食品サービス崩壊がマグロ、ブリ、ホタテ、牡蠣の輸出市場を直撃 - 保証制度が流動性を支えている。 |
| 2022 | エネルギー/燃料価格のショック。 燃焼油価格保証が有効化されています。沿岸漁業の営業利益率は崩壊する。 |
| 2023 | 福島ALPS処理水放出→中国水産物輸入禁止→ホタテ・ナマコ輸出ショック―影響を受けた事業者（特に北海道・青森）の運転資本の橋渡しを保証制度で支援。 |
| 2024 | ALPS関連項目の輸出市場の回復が継続。ギャランティブックは進行中の再建を吸収します。 |
| 2024～2025 | 昆布/ホタテ/貝類に影響を与える海洋熱波ショック。営業損失のつなぎ融資を保証する制度です。 |
| 2025 | 食料安全保障戦略として水産養殖の拡大（国産養殖振興）に重点を置く政策を継続。保証制度も充実。 |
## 8。農業信用保証制度との比較
| 寸法 | 漁業信用保証制度 | [[policy-finance/agriculture-credit-guarantee-system\|Agriculture credit guarantee system]] |
|---|---|---|
| 借り手のターゲット | 漁業者 / 漁業法人 / 養殖業者 / 水産加工業者 | 農家 / 農業法人 / 関連項目事業者 |
| 現地保証人 | 漁業信用基金協会（都道府県・広域） | 農業信用基金協会（47 県） |
| 全国連盟 | 全国漁業信用基金協会連合会 | 全国農業信用基金協会連合会 |
| 国営再保険会社 | JAFFIC漁業信用保険事務 | JAFFIC農業信用保険事務 |
| スーパーバイザー | 水産庁 / 農林水産省 | 農林水産省 |
| 貸し手パートナー | JF / 信漁連 / 農林中央金庫 / 銀行 / JFC 農林漁業部門 | JA / 信連 / 農林中央金庫 / 銀行 / JFC 農林漁業部門 |
| 地理的範囲 | 沿岸部集中県 | 全国 |
| 資産の特性 | 船舶（減価償却、特殊化） + 水産養殖（環境にさらされた） | 土地+農機具+家畜 |
| 危機ベクトル | 資源崩壊／赤潮／台風／原油流出／輸出市場ショック（ALPS水事件） | 天気 / 病気 / TPP-FTA / 食料価格 |
| 典型的なスケール | 個人向けローンの規模が小さい。集中ポートフォリオ | 借り手の裾野が広がる。より多様なローンサイズ分布 |
どちらのシステムも同じ JAFFIC 再保険親会社を共有していますが、保険数理上の前提と都道府県ネットワークの補償範囲が異なる **別々の保険口座**を運営しています。
## 9。境界事例
- **JF マリン バンクではありません** — JF マリン バンクは融資ルートです。このページは、最上位に階層化された保証保険メカニズムです。
- **農業ではありません** — 農業保証保険は [[policy-finance/agriculture-credit-guarantee-system]] (姉妹部門、同じ JAFFIC 親会社) に属します。
- **通常の中小企業保証ではありません** — 通常の中小企業保証は [[policy-finance/japan-credit-guarantee-system]] に属します。
- **森林（林業）ではありません** — 森林信用サポートも JAFFIC の下にありますが、別のアカウントです。
- **JFC 漁業融資ではありません** — JFC の [[JapanFG/jfc|agriculture-forestry-fisheries arm]] が直接融資します。漁業融資保証保険についてのページです。
