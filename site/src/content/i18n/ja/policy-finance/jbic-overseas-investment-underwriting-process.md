# JBIC 海外投資融資の審査プロセス

## ウィキ上の位置づけ

この項目は[[policy-finance/INDEX|policy-finance index]]の下に置かれる、[[financial-regulators/jbic|国際協力銀行 (JBIC)]]に関する審査プロセスの深掘りである。[[policy-finance/japan-project-finance-stack-diagram|Japan project finance stack diagram]]では JBIC 融資が[[policy-finance/nexi|NEXI]]保険、[[policy-finance/jogmec|JOGMEC]]エクイティ、メガバンク商業債務とどのように並ぶかを扱い、[[policy-finance/oecd-export-credit-arrangement|OECD Export Credit Arrangement]]では JBIC が運用される期間上限、[[policy-finance/jica|JICA]]では政策分岐の反対側にある ODA 金融の外縁、[[policy-finance/japan-eximbank-history|Japan Eximbank history]]では2012 以前の制度史、[[finance/japan-acquisition-finance|Japan acquisition finance]]では同じメガバンク・シンジケートが民間クロスボーダー案件をどう組成するかを読む。

## 要約

JBIC の審査は、**一般業務勘定**と**特別業務勘定**という二分法の勘定区分を軸に組み立てられ、その上に**輸出バイヤーズクレジット / サプライヤーズクレジット**、**資源輸入向け輸入融資**、**海外投資融資（OIL）**、**アンタイドローン**、選択的な**出資**まで広がる商品分類が重ねられる。特別業務勘定（2016創設）は、一般業務では価格付けできない政策戦略上重要で高リスクの取引について、JBIC が損失を取る明示的な権限を与える。半分は政策銀行、半分は国家資金の merchant bank である。審査フロー自体は概ね六段階で進む。**(1) 政策 / パイプライン・スクリーニング**では METI / MOFA / MAFF の優先事項と JBIC 中期経営計画に照らす。**(2) 初期信用・カントリーリスク審査**では JBIC の国別枠とソブリン格付に照らす。**(3) 詳細な信用委員会作業**では技術、環境、法務アドバイザーを使い、プロジェクトファイナンスでは標準的なレンダーズエンジニア / 市場コンサルタント / 法務デューデリジェンスのスタックを動員する。**(4) **[[policy-finance/oecd-export-credit-arrangement|OECD Arrangement]]** 適合性レビュー**では最低プレミアム、返済条件、気候 / 石炭セクター了解を確認する。**(5) インタークレディターおよび協調融資の組成**では、メガバンク（[[megabanks/mufg|MUFG]] / [[megabanks/smfg|SMFG]] / [[megabanks/mizuho-fg|Mizuho FG]]）、[[policy-finance/nexi|NEXI]]、必要に応じて peer ECA（US EXIM、K-EXIM、UKEF、KfW IPEX、Bpifrance）と組む。**(6) 総裁承認および取締役会決裁**を経て、common-terms / loan agreement と security trust を実行する。JBIC の政策姿勢は2022 以降、**GX（エネルギー / 脱炭素）、重要鉱物サプライチェーン、friend-shoring、半導体、選択的なアウトバウンド M&A**支援へ大きく移り、新規石炭火力向け金融からの撤退と LNG の気候フレーミングの厳格化が並行して進んでいる。

## 1. 機関 / 部門位置

| 項目 | 詳細 |
|---|---|
| 親エンティティ | [[financial-regulators/jbic|JBIC]]（株式会社国際協力銀行）、100%国有の特殊会社、唯一の株主は財務大臣 |
| 法的根拠 | 株式会社国際協力銀行法（平成 23 年法律第 39 号） |
| 前身の歴史 | 日本輸出銀行（1950）→ 日本輸出入銀行（1953）→ OECF との統合（1961）→ 旧 JBIC（1999）→ JFC 統合（2008-10）→ 再独立（2012-04） — [[policy-finance/japan-eximbank-history|Japan Eximbank history]]参照 |
| 勘定区分 | 標準的な政策金融融資向けの**一般業務勘定**と、高リスクの政策戦略取引向けの**特別業務勘定**。2016 創設 |
| 業務ライン | 輸出信用（バイヤー / サプライヤー）、輸入信用（資源輸入）、海外投資融資、アンタイドローン、出資、保証、債券取得ファシリティ |
| 調達基盤 | FILP 借入、政府保証債発行、FILP 外債プログラムによる外貨調達。基礎貸出帳簿の通貨を反映し、バランスシートの通貨構成は主に USD |
| 監督当局 | 財務大臣（株主・主管）、産業政策調整の METI、二国間 / 地域調整の MOFA |

一般業務と特別業務の区分は、審査目的では最も重要な構造的特徴である。一般業務は通常の政策銀行としての信用規律に制約される。特別業務は、他の形では金融化できない政策戦略上の取引で JBIC が損失を取れるよう、明示的に創設された。

### 2.1 商品分類

| 商品ライン | 借手 | 典型用途 | OECD Arrangement の適用 | 勘定 |
|---|---|---|---|---|
| **輸出バイヤーズクレジット** | 日本の物品 / サービスの海外買い手 | 日本製プラント、船舶、鉄道車両、衛星、設備の海外購入 | あり — Arrangement 適用 | 一般 + 特別 |
| **輸出サプライヤーズクレジット** | 日本の輸出者 | 長期輸出契約上の債権金融 | あり | 一般 |
| **輸入融資** | 日本の輸入者 / コモディティ買い手 | 日本のエネルギー / コモディティ安全保障に結びつく LNG、石油、金属の長期資源輸入契約 | あり（輸出信用相当条件の場合） | 一般 + 特別 |
| **海外投資融資（OIL）** | 日系海外 SPV / 日本親会社 | 海外プロジェクト・子会社向けの出資、劣後金融、運転資金 | 部分的（輸出信用相当部分に Arrangement 適用） | 一般 + 特別 |
| **アンタイドローン** | ソブリン / ソブリン隣接の海外借手 | 製品紐づけ契約なしの産業開発、インフラ、資源安全保障支援 | あり（アンタイドローン規律） | 一般 + 特別 |
| **出資** | プロジェクト SPV / 産業エンティティ | 戦略プロジェクト（エネルギー、サプライチェーン、重要鉱物、GX）への少数持分出資 | Arrangement 外（エクイティであり融資ではない） | 主に特別 |
| **保証** | 商業銀行 / 債券保有者 | 商業銀行融資、債券発行の保証 | 輸出信用相当条件には適用 | 一般 + 特別 |
| **債券取得** | 外国国家発行体 / 国際機関 | 外国国家または国際機関債券の直接取得 | 中核 ECA 枠外 | 特別 |

「JBIC 海外投資審査」が最も一般的に指す旗艦商品は、**海外投資融資（OIL）**である。これは、日系海外 SPV または海外出資のための日本親会社に対する長期融資である。

### 2.2 ODA 円借款 vs 特別業務 — 分岐線

重要な境界として、**ODA 円借款**は JBIC ではなく[[policy-finance/jica|JICA]]が運営する。1999-2008 の法的前身は、旧 JBIC の ODA 部門（OECF 系譜）と政策金融部門を合わせていた。その後、2003-10 の ODA の JICA 移管と、2008  / 2012 の JFC 期を経て、商業グレードの海外金融は ODA の無償 / 譲許的融資から分離された。特別業務勘定は**ODA ではない**。これは、貧困削減基準に基づく譲許的 ODA 金融ではなく、潜在損失に対する国会予算を背景に認められたリスク負担型の政策金融融資および出資活動である。

審査上の含意は、フロンティア国のインフラプロジェクトには、ODA で賄われる部分（技術協力、プロジェクトに隣接する公共インフラ — JICA が金融）と、JBIC OIL + NEXI カバー + メガバンクトランチによる商業グレードのプロジェクトファイナンスが併存し得る、ということである。二つのレーンは文書、ガバナンス、財政上の姿勢が異なり、JBIC の審査委員会は JICA 金融部分を自らの信用判断の一部ではなく外生的文脈として扱う。

### 2.3 頻出するセクターカテゴリ

実務上、JBIC の海外投資融資ブックは、繰り返し現れるセクター縦軸に偏る。

- **エネルギー（LNG、石油・ガス上流、中流、IPP 発電）。** 歴史的には最大のエクスポージャー。2021 以降、気候フレーミングで大きく再編。
- **インフラ（鉄道 / メトロ、港湾、水、通信）。** 鉄道車両や設備輸出が結びつくことが多く、バイヤーズクレジット / サプライヤーズクレジットの重ね合わせが出る。
- **重要鉱物とバッテリーサプライチェーン。** 2022 後に比重が上昇。銅、ニッケル、リチウム、コバルト、希土類に加え、バッテリー / EV 下流。
- **半導体とハイテク産業サプライチェーン。** Friend-shoring の重ね合わせ。
- **日本企業のアウトバウンド M&A。** 中大型の日本企業によるクロスボーダー買収で、JBIC OIL 部分がメガバンクのシンジケート買収トランチを補完する。[[finance/japan-acquisition-finance|Japan acquisition finance]]で説明する構造と接続するが、JBIC 部分は政策整合的なチケットを持つ。
- **再エネ、水素 / アンモニア、CCS。** 2022後の GX 枠組みで成長が大きい領域。
- **航空宇宙と防衛隣接。** 選択的で、主に輸出バイヤーズクレジット経由。

セクターミックスは、JBIC 中期経営計画（現在は GX、サプライチェーン強靭化、friend-shoring、アウトバウンド M&A を重視）と METI / MOFA の優先事項によって形づくられる。

## 3. 審査 / underwriting プロセス

典型的な大型海外投資融資またはアンタイドローンのエンドツーエンドの審査フローは、六つの段階を通る。

### 3.1 フェーズ 1 — 政策 / パイプライン・スクリーニング

- 日本スポンサー、海外借手、peer ECA からの紹介による問い合わせ、または JBIC の産業 / 地域担当者による能動的カバレッジからオリジネーションする。
- JBIC **中期経営計画**の政策優先リスト、および METI / MOFA 主導のセクター別マスタープラン（エネルギー戦略、重要鉱物戦略、インフラ輸出戦略）と照合する。
- 特別業務勘定の候補案件では、取引が損失を生み得るが政策上正当化されるかどうかを早期に判断する（勘定選択を左右する）。
- 気候感応的セクターでは、JBIC の公表済み気候金融姿勢（例: 新規海外石炭火力発電なし、LNG 金融条件の厳格化）に早期照合する。

### 3.2 フェーズ 2 — 初期信用・カントリーリスク審査

- [[policy-finance/oecd-export-credit-arrangement|OECD Arrangement]]のカントリーリスク分類（Country Risk Classification 0–7）に整合する JBIC 内部の国別格付を用いたカントリーリスク分類。
- JBIC ポートフォリオ限度に対する国別エクスポージャー限度チェック。
- スポンサー / 借手の予備的信用審査。プロジェクトファイナンスではスポンサー構成（日本の商社、日本の電力会社、ホスト国 NOC / 国営鉱山会社、国際メジャー）に注目し、日本親会社向け OIL では親会社格付と連結レバレッジに注目する。
- 初期の構造見立て。単一名義のコーポレートローンか、プロジェクトファイナンスか、ソブリン支援付きアンタイドローンかが、後続のアドバイザー動員を決める。

### 3.3 フェーズ 3 — 詳細な信用委員会作業とデューデリジェンス

プロジェクトファイナンス OIL では、標準的なアドバイザースタックを動員する。

- **レンダーズ・テクニカルアドバイザー** — エンジニアリングと建設リスクのレビュー。
- **レンダーズ・マーケットコンサルタント** — オフテイク / 市場価格モデリング（LNG ネットバック、鉱山オフテイク経済性、電力購入契約キャッシュフロー安定性）。
- **レンダーズ・保険ブローカー** — [[policy-finance/nexi|NEXI]]カバーと民間政治リスク保険の重ね合わせを含む bind 構造。
- **レンダーズ・リーガルカウンセル** — 日本側およびホスト国カウンセル。文書は通常、国際プロジェクトファイナンスの LMA 型先例に整合。
- **環境社会 DD** — JBIC の**環境社会配慮確認のためのガイドライン**（「JBIC Guidelines」）に基づき評価。OECD Common Approaches と Equator Principles に整合。
- 内部信用モデリング（DSCR 予測、ダウンサイドケース・ストレス、loan-life coverage ratio、カントリーリスク上乗せ）。
- 内部の業界 / セクターレビュー（例: LNG では長期ガス需要シナリオへの感応度、重要鉱物では価格暴落への感応度）。

単一名義のコーポレート OIL（例: 海外買収資金を賄う日本親会社向け融資）では、アドバイザー動員は通常より軽く、焦点は親会社格付分析、構造的信用補完、買収後モニタリングコベナンツへ移る。これは[[finance/japan-acquisition-finance|Japan acquisition finance]]で説明するメガバンク買収ローンの流れに近い。

### 3.4 フェーズ 4 — OECD Arrangement 適合性レビュー

- カントリーリスク分類に基づく minimum premium rate（MPR）計算。
- 返済期間上限チェック（現在、通常の輸出信用では最大10 年、再エネ / 気候適合取引では14 年、一部プロジェクトファイナンスカテゴリではセクター了解に基づき18 年）。
- 頭金最低額（通常15%）と構造化返済プロファイルの適合性。
- **Coal Sector Understanding（2021 改定）** — OECD Arrangement に従うメンバーにとって、新規石炭火力金融を事実上閉じる。
- 再エネ、水、一定の気候適合プロジェクト向けの **Climate Change Sector Understanding**。
- アンタイドローンについては Arrangement 上のアンタイドローン規律。
- 特別業務勘定取引では、勘定固有の適合性と高リスク政策正当化文書。

この適合性レビューは重要である。一般業務の下で Arrangement パラメータを満たさない取引は、明確な政策正当化がある場合に限り、特別業務へ回せる可能性がある。

### 3.5 フェーズ 5 — インタークレディター / 協調融資の組成

- **メガバンク協調融資の組成。** ほぼすべての大型 OIL 案件は、[[megabanks/mufg|MUFG]]、[[megabanks/smfg|SMFG]]、[[megabanks/mizuho-fg|Mizuho FG]]が提供する並行商業トランチを持ち、通常は一行が**エージェント / ファシリティ銀行**となる。
- **NEXI の重ね合わせ組成。** 保険カバーは商業トランチの一部、またはエクイティエクスポージャーの一部に巻かれることがある。構造選択は JBIC と[[policy-finance/nexi|NEXI]]の間で共同交渉される。
- **Peer-ECA 協調融資。** 大型 LNG、鉱山、鉄道 / メトロ案件では、US EXIM、K-EXIM、UKEF、KfW IPEX、Bpifrance、EDC と並ぶ ECA 協調融資が一般的である。JBIC は peer ECA と共同で common-terms agreement と intercreditor agreement に参加する。
- **国際開発金融機関との協調。** ADB / World Bank / AIIB の参加は、common terms 上で JBIC OIL と構造的に両立する。
- **Security trust とインタークレディター・メカニクス。** すべてのシニア貸手に共通する共同担保エージェント、プロジェクト担保の持分、議決閾値、執行メカニクス、エクイティ cure 権は、common-terms 文書を通じて交渉される。

### 3.6 フェーズ 6 — 承認と実行

- **内部信用委員会**による適切な閾値での承認（部長 → 上級管理職 → 総裁 → 取締役会へ、チケットサイズと政策上の重要性に応じてエスカレーション）。
- 標準委員会閾値を超える取引の**総裁 / 役員承認**。
- 最大規模取引の**取締役会決裁**。
- 協調融資当事者と並行した**common-terms agreement**および**facility agreement**の実行。
- ファイナンシャルクローズ後の**ドローダウンと継続的コベナンツモニタリング**。

典型的な大型 OIL 案件サイクルは、初期マンデートからファイナンシャルクローズまで、プロジェクトファイナンスで12–24 カ月、単一名義コーポレート OIL で6–12 カ月である。政治的に敏感な案件や初物取引では、タイムラインは大幅に長くなる。

## 4. 民間金融機関との co-financing / 連携

JBIC の構造的な共同相手は日本のメガバンクである。標準的なシンジケーションの幾何は以下の通り。

- **日本のメガバンク一行がエージェント銀行。** [[megabanks/mufg|MUFG]]、[[megabanks/smfg|SMFG]]、または[[megabanks/mizuho-fg|Mizuho FG]]が、インタークレディター・メカニクス、ドローダウン・ウォーターフォール、クロージング後モニタリングを運営する。
- **JBIC 直接トランチと並ぶメガバンク商業トランチ。** メガバンクからの純粋な商業ローン + JBIC OIL 部分 + 多くの場合 NEXI カバー付き部分。JBIC と NEXI の相互作用は構造的である。JBIC が直接貸手、NEXI が商業部分を包む保険者として位置づく全体の重ね合わせは[[policy-finance/japan-project-finance-stack-diagram|Japan project finance stack diagram]]参照。
- **地域銀行 / 信金 / 農林中金の参加。** アウトバウンド M&A または国内に軸を置くクロスボーダー構造の日本所在 SPV 金融では、[[cooperative-banks/norinchukin|Norinchukin]]や大手地域銀行が参加することがある。
- **外資系銀行の参加。** 大型 LNG・鉱山案件では、フランス、シンガポール、オーストラリア、米国、中国の銀行が日常的に協調融資する。
- **シンジケート内の peer ECA。** US EXIM、K-EXIM、UKEF、KfW IPEX、Bpifrance、EDC、Cesce — common-terms 文書、調和された保険 / 直接融資構造。

NEXI と JBIC の重なりは構造的であり、競合的であることはまれである。JBIC は直接貸手で、NEXI は保険者である。同一プロジェクトに JBIC 直接融資と NEXI カバー付き商業融資がある場合、両国策機関は重複、先行条件、情報共有を調整する。

ECA 関与のない純粋な民間アウトバウンド買収シンジケートでは、同じメガバンクが別のシンジケーション幾何を運営する。これは[[finance/japan-acquisition-finance|Japan acquisition finance]]で説明される。JBIC アンカーのスタックは、同じエージェント銀行を使っていても、純商業スタックとは構造的に異なる。

## 5. 政策目標と政府関与

JBIC の海外投資政策姿勢は以下によって形づくられる。

- **中期経営計画**（現在は GX、サプライチェーン強靭化、friend-shoring、アウトバウンド M&A、インフラ輸出を重視）。
- **内閣および METI 主導のセクター別マスタープラン** — エネルギー基本計画、重要鉱物戦略、GX 推進戦略、Asia Zero Emission Community（AZEC）構想、インフラ輸出計画。
- **MOFA の二国間・地域調整** — 特にソブリン相手方のアンタイドローンや大型インフラ金融。
- **財政投融資（FILP）予算** — 国会承認の年次調達上限。
- **特別業務勘定（2016）** — 高リスク政策戦略取引向けに区分された権限と準備。

JBIC の年次融資量と政策姿勢は各中期計画サイクルで変化し、セクターミックスは METI と内閣の優先事項の変化に応じて、エネルギー / 資源、インフラ、製造、サプライチェーン金融の間で移る。

## 6. 経済安全保障 / 最近の方針シフト

2022 頃以降、JBIC の政策姿勢は**経済安全保障**を軸に再構成されている。

- **重要鉱物サプライチェーン。** 銅、ニッケル、リチウム、コバルト、希土類 — オーストラリア、インドネシア、カナダ、チリ、南米、アフリカの日本関係サプライチェーンプロジェクトへの出資、OIL、アンタイドローン支援。
- **バッテリーと EV サプライチェーン。** 重要鉱物から下流の正極 / 負極材料、セル製造へ。
- **半導体とハイテク産業サプライチェーン。** 友好法域における日本関係の半導体材料・設備能力拡張の friend-shoring。拡張プロジェクトへの協調融資を含む。
- **GX / エネルギー転換。** 再エネ、洋上風力、水素 / アンモニア、CCS、AZEC 枠組み下の地域火力発電所におけるアンモニア混焼。
- **LNG の気候フレーミング。** LNG を定義された文脈で移行燃料として支援し続けつつ、新規 LNG 金融を気候適合性枠組みに照らしてより厳格に審査。
- **新規海外石炭火力金融なし。** 2021 G7 コミットメントと2021 OECD Coal Sector Understanding 更新に整合する公的姿勢。
- **アウトバウンド M&A 支援。** 日本企業の大型クロスボーダー買収金融における OIL 部分。
- **国別姿勢の変化。** 経済安全保障または制裁上の考慮が適用される一定法域への新規金融を実質的に抑制。

特別業務勘定は、一般業務の信用規律では届かない政策戦略取引で、JBIC が損失負担リスクを取ることを可能にする制度的ビークルである。

## 8. ファイナンシャルクローズ後のモニタリング

ファイナンシャルクローズ後、典型的な大型 OIL における JBIC の継続的役割は以下の通りである。

1. **先行条件（CP）管理。** エージェント銀行と連携し、初回ドローダウン前に最終 CP 充足を確認する。
2. **ドローダウン管理。** NEXI カバー付き商業トランチとスポンサーエクイティに並ぶ按分ドローダウン。JBIC のドローダウン規律は、レンダーズ・テクニカルアドバイザーによる建設マイルストーン認証と整合することが多い。
3. **コベナンツ・モニタリング。** プロジェクト DSCR、市場条件テスト、スポンサーサポート義務、JBIC Guidelines に基づく環境社会コンプライアンス。
4. **信頼性テスト管理。** COD（commercial operation date）時に、プロジェクトはスポンサーサポート義務から純粋なプロジェクトキャッシュフロー返済へ移行する前に、貸手の信頼性テストに合格する必要がある。
5. **リストラクチャリング管理。** ストレス時（コスト超過、市場悪化、安全保障事案を含む不可抗力）には、common-terms 債権者グループ全体で再編を調整し、商業トランチがカバーされている場合は NEXI の請求処理が並行する。
6. **リファイナンス対応。** COD 後に商業トランチを債券またはサステナビリティリンク市場へリファイナンスすることは一般的である。JBIC の直接融資トランチは通常、予定満期まで残る。

単一名義コーポレート OIL では、モニタリングはコーポレートローンのコベナンツ追跡に近い。レバレッジ、インタレストカバレッジ比率の遵守、制限支払テスト、報告義務が中心となる。
