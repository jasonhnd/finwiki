# Japan Export-Import Bank (historical predecessor of JBIC)

## TL;DR

日本輸出入銀行（日本輸出入銀行 / JEXIM）は、1950 から 1999までの間、日本の主要な国家輸出信用・対外投資融資機関であった。1999 に海外経済協力基金（海外経済協力基金 / OECF）と合併し、初代の国際協力銀行（JBIC）を形成した。JEXIM を理解することは、日本の戦後の輸出主導型工業化、ODA アーキテクチャ、そして [[financial-regulators/jbic|JBIC]] と [[policy-finance/jica|JICA]] の間の現代的な分割についての歴史的分析にとって重要である。

本ページは、現代の JBIC / JICA / NEXI / DBJ という公的金融マップの歴史的アンカーとして [[policy-finance/INDEX|policy-finance index]] の下に位置する。

## 2. Why this page exists separately

1999 の合併と 2008 の組織再編により、現代の日本の政策金融マップにはきれいな機関分割が残された — [[financial-regulators/jbic|JBIC]] が対外事業金融を、[[policy-finance/jica|JICA]] が ODA を担う — が、JEXIM のレガシーは以下の点で依然として重要である：

- 「Eximbank Japan」または「JEXIM」という用語を用いる古いプロジェクトファイナンス文書を読むうえで。
- 日本の現代的な輸出信用アーキテクチャが、US EXIM、Korea EXIM、China EXIM、または KfW IPEX と構造的に異なる理由を理解するうえで — 1999 の合併は、他国が別々の機関に残した機能を統合した。
- 現代の [[financial-regulators/jbic|JBIC]] 事業ラインと [[policy-finance/japan-project-finance-stack-diagram|Japanese project-finance stack]] の系譜をたどるうえで。
- OECD アレンジメント適格の輸出信用（JBIC の後継）を、ODA の譲許的融資（JICA の後継）と区別するうえで。

## 3. Function map (historical)

| Function | Description |
|---|---|
| Export buyer credit | 日本製の資本財（プラント、船舶、輸送機器）の外国の買い手への長期融資。 |
| Export supplier credit | 延払輸出契約をファイナンスするための日本の輸出企業への融資。 |
| Overseas investment lending | 日本企業による海外株式投資へのファイナンス。 |
| Untied loan（アンタイドローン） | 特定の日本の輸出に紐付かない外国のソブリンおよび主体への長期融資で、しばしば資源輸入を支援する。 |
| Import financing | 日本の輸入企業への長期融資、特にエネルギーおよび原材料向け。 |

これらの機能はすべて今日 [[financial-regulators/jbic|JBIC]] の下で継続しており、現代の [[policy-finance/oecd-export-credit-arrangement|OECD Arrangement]] 規律の枠内で枠付けされている。

## 4. Historical trajectory

| Year | Event |
|---|---|
| 1950-12 | 戦後の輸出主導型回復を支援するため日本輸出銀行（日本輸出銀行）が設立された。 |
| 1953-04 | 輸入金融の権限が追加され、日本輸出入銀行に改称された。 |
| 1957–1970年代 | 戦後の工業化期；プラント輸出、造船、重工業輸出をファイナンスした。 |
| 1980年代 | 円の国際化と日本企業の対外投資拡大；JEXIM は対外投資融資を拡大した。 |
| 1990年代 | バブル後の時代；資源安全保障と新興市場エクスポージャーが拡大した。 |
| 1999-10-01 | OECF と合併し初代の国際協力銀行（旧 JBIC）を形成。 |
| 2003-10 | 改革のもとで OECF の ODA 借款ポートフォリオが [[policy-finance/jica|JICA]] に移管された。 |
| 2008-10-01 | 初代 JBIC が [[financial-regulators/jfc|JFC]] にその国際金融部門として吸収された。 |
| 2012-04-01 | 国際金融部門が現行の [[financial-regulators/jbic|JBIC]]（特殊株式保有会社）として分離・スピンアウトされた。 |

## 5. Institutional significance

JEXIM の存在とその 1999 の合併は、現代の日本の政策金融に持続する 3 つの特徴を説明しており、そのすべてが [[policy-finance/japan-policy-finance-system|Japan policy finance system]] に見られる：

1. **統合された対外金融ビークル**：EXIM Bank と OPIC / DFC の間の米国の分離とは異なり、日本は輸出信用と対外投資金融を単一の機関に統合し、メガバンク（[[megabanks/mufg|MUFG]]、[[megabanks/smfg|SMFG]]、[[megabanks/mizuho-fg|Mizuho FG]]）との [[policy-finance/japan-project-finance-stack-diagram|project-finance stack]] における協調融資を簡素化した。
2. **遅れた ODA 分離**：ODA 借款の管理が完全に [[policy-finance/jica|JICA]] へ移ったのは、機関の歴史が始まってから 2008, 年も経った後であった。これが、現代の JBIC は ODA に隣接していないにもかかわらず、古い教科書が JEXIM を ODA 隣接として記述する理由である。
3. **WTO 以前の輸出信用規範**：JEXIM は、現代の [[policy-finance/oecd-export-credit-arrangement|OECD Arrangement]] セクターの理解が ECA 規律を厳格化する以前に運営されていた；JBIC が承継したレガシーの帳簿には、より緩いルールのもとで成立した取引が含まれる。

## 6. Boundary cases

- **現代の JBIC ではない**：本ページは歴史的なものである。現行の対外政策金融は [[financial-regulators/jbic|JBIC]] にルーティングされる。
- **JICA ではない**：ODA / 開発協力の歴史は主に OECF に、そして 2008 から [[policy-finance/jica|JICA]] に属し、JEXIM に直接属するものではない。
- **US EXIM ではない**：この機関は US EXIM と英語の「Eximbank」というラベルを共有するが、対外投資融資やアンタイドローンを含むより広範な機能を持っていた。
- **[[financial-regulators/dbj|DBJ]] ではない**：DBJ の前身（日本開発銀行 / 日本開発銀行）は **国内** の開発銀行のカウンターパートであった；JEXIM はその **対外** のカウンターパートであった。両者はピアの特殊目的国家銀行であった。

## 7. Open questions

- JEXIM の 1990年代後半の対外投資帳簿のうち、どの程度が初代 JBIC のバランスシートへ、そして現代の JBIC へと引き継がれたのか？
- 1999 の合併は、ピア（K-EXIM、US EXIM、KfW IPEX）に対する国際輸出信用における日本の競争上のポジションを実質的に変えたのか？
- 2008–2012 の JFC 吸収・再スピンアウトのサイクルは、政策金融の機関的記憶にどう影響したのか？
