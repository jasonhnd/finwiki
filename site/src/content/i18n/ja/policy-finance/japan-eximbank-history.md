# 日本輸出入銀行（JBICの歴史的前身）

## 要約

日本輸出入銀行（日本輸出入銀行 / JEXIM）は、1950 から1999まで、日本の主要な国家輸出信用・海外投資融資機関だった。1999 には海外経済協力基金（OECF）と統合され、初代の国際協力銀行（JBIC）となった。JEXIMを理解することは、日本の戦後輸出主導型工業化、ODAアーキテクチャ、そして現代の[[financial-regulators/jbic|JBIC]]と[[policy-finance/jica|JICA]]の分離を歴史的に分析するうえで重要である。

このページは、現代のJBIC／JICA／NEXI／DBJを含む公的金融マップの歴史的アンカーとして、[[policy-finance/INDEX|policy-finance index]]配下に位置づけられる。

## 1. 制度上の境界

| 項目 | 読み方 |
|---|---|
| 日本語名 | 日本輸出入銀行（当初: 日本輸出銀行1950–1953） |
| 英語名 | Export-Import Bank of Japan / Japan Export-Import Bank |
| 設立 | 1950-12 にExport Bank of Japanとして設立 |
| 改称 | 1953-04 に輸入金融を追加し、Japan Export-Import Bankへ改称 |
| 解散 | 1999-10-01。OECFと統合され、初代JBICを形成 |
| 法的形態 | 特殊法人、100%政府出資 |
| 監督ルート | 大蔵省 |
| 承継レーン | [[financial-regulators/jbic|JBIC]]が輸出金融と海外投資金融を承継。OECFのODA円借款は、後に[[policy-finance/jica|JICA]]へ2008 に移管 |

## 2. このページを独立させる理由

1999 の統合と2008 の再編により、現代の日本の政策金融マップは、海外事業金融の[[financial-regulators/jbic|JBIC]]とODAの[[policy-finance/jica|JICA]]という明確な制度分離を持つようになった。しかしJEXIMの遺産は、次の点で引き続き重要である。

- "Eximbank Japan"または"JEXIM"という用語を使う古いプロジェクトファイナンス文書を読むため。
- 日本の現代の輸出信用アーキテクチャが、US EXIM、Korea EXIM、China EXIM、KfW IPEXと構造的に異なる理由を理解するため。1999 の統合によって、他国では別機関に残された機能が統合された。
- 現代の[[financial-regulators/jbic|JBIC]]の事業ラインと[[policy-finance/japan-project-finance-stack-diagram|Japanese project-finance stack]]の系譜をたどるため。
- OECDアレンジメント適格の輸出信用（JBICの承継領域）と、ODA譲許的融資（JICAの承継領域）を区別するため。

## 3. 機能マップ（歴史的）

| 機能 | 説明 |
|---|---|
| 輸出バイヤーズクレジット | 日本の資本財（プラント、船舶、輸送機器）の外国買い手に対する長期融資。 |
| 輸出サプライヤーズクレジット | 延払輸出契約を資金化するための日本輸出者向け融資。 |
| 海外投資金融 | 日本企業による海外エクイティ投資への金融。 |
| アンタイドローン | 特定の日本輸出にひも付かない、外国政府・事業体向けの長期融資。資源輸入を支えることが多い。 |
| 輸入金融 | とくにエネルギーや原材料を対象とする、日本輸入者向けの長期融資。 |

これらの機能はすべて、現在の[[financial-regulators/jbic|JBIC]]の下で、現代の[[policy-finance/oecd-export-credit-arrangement|OECD Arrangement]]規律の枠内に位置づけられて継続している。

## 4. 歴史的な軌跡

| 年 | 出来事 |
|---|---|
| 1950-12 | 戦後の輸出主導型復興を支えるため、日本輸出銀行（日本輸出銀行）が設立された。 |
| 1953-04 | 輸入金融の権限が追加され、日本輸出入銀行へ改称。 |
| 1957–1970年代 | 戦後工業化期。プラント輸出、造船、重工業輸出を金融面で支援。 |
| 1980年代 | 円の国際化と日本企業の海外投資拡大。JEXIMは海外投資金融を拡大。 |
| 1990年代 | バブル後期。資源安全保障と新興市場エクスポージャーが拡大。 |
| 1999-10-01 | OECFと統合され、初代Japan Bank for International Cooperation（旧JBIC）を形成。 |
| 2003-10 | 改革により、OECFのODA円借款ポートフォリオが[[policy-finance/jica|JICA]]へ移管。 |
| 2008-10-01 | 初代JBICが[[financial-regulators/jfc|JFC]]へ吸収され、その国際金融部門となる。 |
| 2012-04-01 | 国際金融部門が再び分離され、現在の[[financial-regulators/jbic|JBIC]]（特殊会社）となる。 |

## 5. 制度的意義

JEXIMの存在と1999 の統合は、現代の日本の政策金融に残る3つの特徴を説明する。いずれも[[policy-finance/japan-policy-finance-system|Japan policy finance system]]に見られる。

1. **統合された海外金融ビークル**: 米国がEXIM BankとOPIC / DFCを分離しているのと異なり、日本は輸出信用と海外投資金融を単一機関に統合した。これにより、メガバンク（[[megabanks/mufg|MUFG]]、[[megabanks/smfg|SMFG]]、[[megabanks/mizuho-fg|Mizuho FG]]）との[[policy-finance/japan-project-finance-stack-diagram|project-finance stack]]上での協調融資が簡素化された。
2. **遅いODA分離**: ODA円借款管理が完全に[[policy-finance/jica|JICA]]へ移ったのは2008, であり、制度史の開始から数十年後だった。このため、現代のJBICはODA機関ではないにもかかわらず、古い教科書ではJEXIMがODA隣接として説明される。
3. **WTO以前の輸出信用規範**: JEXIMは、現代の[[policy-finance/oecd-export-credit-arrangement|OECD Arrangement]]セクター了解がECA規律を厳格化する前に活動していた。JBICが承継したレガシー帳簿には、より緩いルールの下で組成された案件も含まれる。

## 6. 境界事例

- **現代のJBICではない**: このページは歴史的項目である。現在の海外政策金融は[[financial-regulators/jbic|JBIC]]へ接続する。
- **JICAではない**: ODA／開発協力の歴史は主にOECFに属し、2008 からは[[policy-finance/jica|JICA]]に属する。JEXIMに直接属するものではない。
- **US EXIMではない**: 同機関はUS EXIMと英語の"Eximbank"ラベルを共有するが、海外投資金融やアンタイドローンを含む、より広い機能を持っていた。
- **[[financial-regulators/dbj|DBJ]]ではない**: DBJの前身（Japan Development Bank / 日本開発銀行）は**国内**開発銀行の対応物であり、JEXIMはその**海外**対応物だった。両者は目的別の特殊国策銀行として並立していた。

## 7. 未解決の論点

- JEXIMの1990年代後半の海外投資帳簿のうち、どれだけが初代JBICのバランスシートへ、さらに現代JBICへ引き継がれたのか。
- 1999 の統合は、国際輸出信用における日本の競争上の地位を、ピア（K-EXIM、US EXIM、KfW IPEX）との比較で実質的に変えたのか。
- 2008–2012 のJFC吸収・再分離サイクルは、政策金融の制度的記憶にどのような影響を与えたのか。
