# JICA (Japan International Cooperation Agency)

## ウィキ上の位置づけ

この項目は、日本の国家金融制度におけるODA実施ノードとして [[policy-finance/INDEX|policy-finance index]] 配下に位置する。同じ海外周辺領域の輸出信用 / 海外投資の相手方は [[financial-regulators/jbic|国際協力銀行 (JBIC)]]、JICAのODA融資業務が隣に位置する商業グレードのレーンにおける貸し手側フローは [[policy-finance/jbic-overseas-investment-underwriting-process|JBIC overseas-investment underwriting process]]、商業部分の政治リスク保険は [[policy-finance/nexi|NEXI]]、ODAグレードの開発案件が先行することのある資源安全保障上流エクイティは [[policy-finance/jogmec|JOGMEC]]、より広いツールキットは [[policy-finance/japan-policy-finance-system|Japan policy finance system]]、9機関比較軸は [[policy-finance/japan-policy-finance-institution-mandate-matrix|Japan policy-finance institution mandate matrix]]、[[policy-finance/japan-eximbank-history|Japan Eximbank history]] では旧JBICのODA部門がJICAへ移った 2008 再編、JICAが明示的に入らない商業側の境界線は [[policy-finance/oecd-export-credit-arrangement|OECD Export Credit Arrangement]]、ODA金融が外側に位置する貿易促進 / 保険 / 金融の3本柱比較は [[trade/jetro-vs-nexi-vs-jbic-comparison|JETRO vs NEXI vs JBIC]] と照らして読む。

## 要約

JICA（独立行政法人国際協力機構）は、日本の主要なODA実施機関であり、MOFA監督下（外務大臣主管 + 融資条件について財務省 / METI / MOFA が調整）にある。現在の形は 2008-10 に、旧JBICのODA部門（OECF系譜の円借款）が従来のJICA（技術協力 / 無償資金協力機関）へ統合されたことで成立した。現在、同機関は3つの法定商品レーン、すなわち**ODA loans（有償資金協力 / 円借款）**、**technical cooperation（技術協力）**、**grant aid（無償資金協力）**を運営し、加えて4つ目の小規模レーンである **Private Sector Investment Finance（PSIF / 海外投融資）** を持つ。PSIF は、ODA適格の民間セクター案件の狭い領域で [[financial-regulators/jbic|JBIC]] と競合する。資本構造は政府資金に基づく。独立行政法人として、一般会計からの出資と、融資ポートフォリオ向けのFILP（財政投融資）資金を受ける。融資条件は制度的に譲許的であり、長期（通常 30 年以上）、据置期間、低いクーポン（多くの場合 1%未満）、円建てまたは米ドル建てである。この点で、JBICの市場金利またはOECDアレンジメントMPRに沿う商業グレードの海外融資とは明確に異なる。DAC報告境界が重要な分析上の識別子である。JICAの融資はOECD-DAC基準上ODAに算入されるが、JBICの融資は通常算入されない。近年（2022-2025）の政策方向は、**経済安全保障フレーミング**へ回転している。ASEAN / インド太平洋におけるフレンドショアリング型インフラ、サプライチェーン強靭化助成、水素 / 脱炭素化の技術協力、ODAと日本企業の海外投資をつなぐ橋としてのPSIFの選択的拡大であり、これに「オファー型協力」（日本が能動的に案件形成を主導するODA）を明文化した 2023 ODA大綱改定が重なる。

法的形態（独立行政法人）は構造的に重要である。JICA はJBIC、JFC、DBJのような特殊会社ではない。国家が株式資本を所有するという同じエクイティ商品上の意味での株式資本は存在しない。JICA は政府予算とFILP借入を受ける。そのガバナンスは、MOFAを主たる評価者とする独立行政法人評価の枠組みを通じて流れる。開示のタイミングと連結会計は、特殊会社機関に適用される会社法 / 金融商品取引法の開示制度ではなく、独立行政法人通則法に支配される。

### 1.1 JICA が行うこと、行わないこと

- **行うこと**: 融資、無償資金協力、技術協力、PSIFにまたがる日本のODAを実施する。専門家とJOCVボランティアを派遣する。MOFAと国別プログラム計画を行う。多国間機関（World Bank、ADB、IDB、AfDB）と共同融資を調整する。海外研修員を日本へ受け入れる国内研修プログラムを運営する。
- **行わないこと**: 商業グレードの海外プロジェクトファイナンスを書く（それは [[financial-regulators/jbic|JBIC]] の領域）。政治リスクを保険する（それは [[policy-finance/nexi|NEXI]]）。資源上流エクイティを取る（それは [[policy-finance/jogmec|JOGMEC]]）。日本のODA政策や国別配分を決める（それはMOFA / 内閣）。外交青書や制裁リストを管理する（MOFA）。

JICA / JBIC 境界は、日本の海外金融スタックで最もよく問われる制度上の論点である。短く言えば、**JICA = ODA = 譲許的、MOFA政策主導、DAC報告対象。JBIC = 商業グレード、MoF政策主導、OECDアレンジメント準拠、DAC報告対象外。** 単一の海外インフラ案件に、JICA資金によるODA融資部分（公共インフラ、能力構築）、JBIC OIL部分（日本関係SPVへのエクイティ注入金融）、NEXIカバーレイヤー（メガバンクトランチの政治リスク保険）が並存する場合がある。2つの金融レーンは、ドキュメンテーション、ガバナンス、財政上の姿勢が異なる。JICAの案件判断は商業JBICトランチを外生的に扱い、その逆も同様である。

## 2. 事業ライン別マンデート

### 2.1 商品レーン概要

| レーン | 法定根拠 | 資金源 | 典型的な譲許性 | 取引相手 | 国の適格性 |
|---|---|---|---|---|---|
| **ODA loans（有償資金協力 / 円借款）** | JICA法 + 二国間融資合意（E/N + L/A） | FILP借入による円資金（選択的に米ドル） | 長期（通常 30-40 年）、据置 10 年、標準条件で 1%未満のクーポン。STEP条件（本邦技術活用条件）は日本技術紐付け案件向けにさらに低利 | ソブリンまたはソブリン隣接（例: ソブリン保証付きSOE） | DACリスト上の低中所得国 / 低所得国。高所得国はDAC規則により不適格 |
| **Technical cooperation（技術協力）** | JICA法 | 一般会計予算 | 100%譲許的（融資ではない）。専門家派遣 + JOCV + 研修員受入 + プロジェクト型協力 | 政府 / 公的機関 / 研修機関 | DACリスト適格。一部上位中所得国も適格 |
| **Grant aid（無償資金協力）** | JICA法 + 二国間無償合意 | 一般会計予算 | 100%譲許的。プロジェクト無償とプログラム無償 | 政府 / ソブリン隣接 | 主にDACリスト上の低所得国 / 低中所得国 |
| **Private Sector Investment Finance（PSIF / 海外投融資）** | JICA法（2010 に再開、1999 停止後）。2012 以降に対象範囲変更 | FILP + 準備金 | DACリスト国における日本民間セクター海外投資案件向けの譲許的または準市場条件。エクイティ投資 + 融資。少数持分 | 日本関係海外SPV / 民間セクター案件 | 民間資金ギャップがあるDACリスト国 |

### 2.2 ODA loans（円借款）

旗艦商品である。開発途上国のソブリンまたはソブリン隣接借入人向けの長期譲許的円建て（または米ドル建て）融資であり、日本政府と相手国政府の **E/N（Exchange of Notes）**、続くJICAと借入人の **L/A（Loan Agreement）** を中心に組成される。標準条件は、国分類と事業セクターに応じて 30-40 年の期間、10年据置、0.5-1.5%クーポンである。気候 / 人間の安全保障カテゴリは低利になる。

- **STEP（本邦技術活用条件）階層**: 日本技術 / 施工業者に紐づく調達の見返りとして超低金利（歴史的に 0.1-0.3%レンジ）を提供する。メニューの中で最も譲許的な階層である。
- **セクター集中**: 歴史的なポートフォリオでは交通インフラ（鉄道、高速道路、都市交通、港湾、空港）が支配的である。発電・送電網、水・衛生、農業・農村開発、教育 / 保健システム強化も含む。
- **国別集中**: India、Vietnam、Indonesia、Bangladesh、Philippines、Egypt、Kenya は高頻度の大口受入国である。2022以降、国別構成はインド太平洋のフレンドショアリング案件へ傾いている。
- **共同融資**: World Bank / ADB / IDB / AfDB の案件構造と共同融資されることが多い。

### 2.3 Technical cooperation（技術協力）

専門家派遣（短期 / 長期 専門家）、JOCV（青年海外協力隊）とシニアボランティア派遣、日本での研修員受入（集団 / 個別研修）、複数の手段を国別プログラムテーマの周辺に組み合わせるプロジェクト型技術協力からなる。このレーンは、後続のODA融資や無償資金協力の実施を可能にする二国間関係資本を築く。

### 2.4 Grant aid（無償資金協力）

MOFA主導の国別配分とJICA実施の調達を通じて提供されるプロジェクト無償とプログラム無償である。セクターには、保健システム強化（病院建設、ワクチン・コールドチェーン支援、母子保健プログラム）、教育インフラ（小学校建設）、水・衛生、食料安全保障、災害復旧 / 人道対応が含まれる。

### 2.5 Private Sector Investment Finance（PSIF / 海外投融資）

PSIF は、[[financial-regulators/jbic|JBIC]] の領域と重なるJICAの狭いレーンである。DACリスト国で商業金融ギャップが存在する日本関係海外民間セクター案件に対し、譲許的または準市場条件の資金を提供する。商品メニューは以下を含む。

- 日本企業スポンサーと開発インパクトの文脈を持つ海外SPVへの**エクイティ参加**。
- 海外SPV / 日本スポンサー民間セクター案件への、譲許的または準市場条件の**融資**。

PSIF は当初プログラムの損失を受けて 1999 に停止され、その後、より厳格な審査基準の下で 2010 に再開された。チケットはJBIC OILに比べ小さく、通常は案件あたり数十億円までである。PSIF のナラティブ上の機能は、JBICの商業グレード海外投資融資商品と正面から競合するのではなく補完し、日本スポンサーがODA適格国市場へアクセスしやすくするJICAブランドの開発金融シグナルを提供することである。

## 4. 年次別の進化

| 年 | 出来事 |
|---|---|
| 1954  | コロンボ・プラン枠組みの下で、日本初の戦後ODA実施が始まる |
| 1961-03  | OECF（Overseas Economic Cooperation Fund）がEPA下のODA融資実施機関として設立 |
| 1962-06  | 海外技術協力事業団（OTCA）が設立（技術協力の前身） |
| 1974-08  | OTCAがJICA（旧JICA）へ再編。MOFA下の技術協力 + 無償資金協力 + JOCV機関 |
| 1999-10  | OECFが日本輸出入銀行に統合され、旧JBIC（Japan Bank for International Cooperation）となる。ODA融資部門は輸出信用部門と並んで旧JBIC内に置かれる |
| 2003-10  | JICAが独立行政法人国際協力機構法に基づく独立行政法人形態へ再編 |
| 2008-10  | 旧JBICのODA融資部門がJBIC（当時はJFC国際部門に組み込まれていた）からJICAへ移管される。JICAはODA融資 + 技術協力 + 無償資金協力を単一機関に吸収（「New JICA」）。ODA融資業務はMOFA監督下に統合される |
| 2010  | PSIF（Private Sector Investment Finance）が、1999 停止後の新審査基準の下で再開 |
| 2012-04  | JBICがJFCから再分離（特殊会社形態へ戻る）するが、ODA融資はJICAに残る。2008 ODA / 商業金融分離が維持される |
| 2015  | 開発協力大綱改定。「非軍事」原則と相手国の自助努力重視を明文化 |
| 2022-04  | Tanaka Akihiko が Kitaoka Shinichi の後任としてJICA理事長に就任 |
| 2023-06  | 開発協力大綱が再改定。「オファー型協力」、経済安全保障フレーミング、GX支援を明文化 |
| 2024-2025  | インド太平洋のフレンドショアリング型インフラプログラムが拡大。サプライチェーン強靭化助成、半導体 / 鉱物の能力構築技術協力、水素 / アンモニア / 脱炭素化技術協力案件がASEANで拡大 |

## 5. 兄弟ページとの比較

この時点のFinWikiでは、JICAにはJBIC（[[policy-finance/jbic-overseas-investment-underwriting-process|JBIC OIL underwriting]]）、NEXI（[[policy-finance/nexi-export-credit-insurance-products|NEXI insurance products]]）、JFC（[[policy-finance/jfc-sme-division-operating-model|JFC SME Division]]）、JOGMEC（[[policy-finance/jogmec-equity-and-offtake-mechanics|JOGMEC equity / offtake]]）と異なる単一の専用運営メカニクス兄弟ページはない。現在の最も深い比較アンカーは、[[policy-finance/japan-policy-finance-institution-mandate-matrix|the mandate matrix]]（機関横断軸）、[[policy-finance/japan-project-finance-stack-diagram|the project finance stack diagram]]（海外案件上でODAグレードと商業グレードのレイヤーがどう並ぶか）、[[policy-finance/japan-eximbank-history|the eximbank history]]（2008 ODAのJICA移管が中心的な制度ピボットとなる箇所）である。

### JBIC（最も近い同業）との比較軸

| 軸 | JICA | [[financial-regulators/jbic\|JBIC]] |
|---|---|---|
| 主務大臣 | MOFA（外務大臣） | MoF（財務大臣） |
| 法的形態 | 独立行政法人 | 特殊会社（株式会社） |
| 資本構造 | 政府予算。株式資本なし | MoF が保有する株式資本 100% |
| 資金基盤 | FILP + 一般会計予算 | FILP + 政府保証債 |
| 融資条件 | 譲許的（長期、低クーポン） | 商業グレード（OECDアレンジメント準拠） |
| DAC ODA報告 | あり（ODAに算入） | なし（商業グレード） |
| OECDアレンジメント適用 | 対象外（ODAは除外） | あり（アレンジメント準拠） |
| 主な取引相手 | ソブリン / ソブリン隣接 | 日本関係SPV / 外国買い手 / アンタイド融資の外国ソブリン |
| 共同融資パートナー | World Bank / ADB / IDB / AfDB / 二国間DFI | メガバンク（[[megabanks/mufg\|MUFG]] / [[megabanks/smfg\|SMFG]] / [[megabanks/mizuho-fg\|Mizuho FG]]）+ 同業ECA |
| 地理的範囲 | 約150 のDACリスト国 | OIL / エクイティでは高所得国を含むクロスボーダー |
| 典型セクター | 公共インフラ、保健、教育、水、農業 | エネルギー、資源、プラント、交通、半導体、M&A金融 |

両機関は代替できないように設計されている。一方に適合する案件が、同一条件で他方にも適合することは稀である。

## 6. 反論・留意点

- **譲許的であることは補助金なしを意味しない。** JICA融資は制度的に譲許的だが、DAC報告と譲許性計算は時代とともに変化してきた（DACの 2014-2018 ODA近代化はグラント・エレメント計算を再定義した）。JICA融資条件を数十年単位で比較するには、どの譲許性手法が適用されるかを確認する必要がある。
- **マンデート拡張批判。** PSIF拡大は、JBICや一部政策分析者から、JBICの商業グレード海外投資融資商品と重なり、2008 再編が明示的に維持しようとしたJICA / JBIC制度分離を曖昧にすると批判されてきた。2010 後に再開されたPSIFには、この批判を避けるための慎重な審査基準があるが、上位中所得国のODAグレード対準市場案件では境界が争点になる。
- **経済安全保障フレーミング批判。** 「オファー型協力」と経済安全保障フレーミングを明文化した 2023 ODA大綱改定は、一部の開発NGOや学術界から、ODAを貧困削減目的から日本企業の市場アクセス目的へ動かすものとして批判されている。反論は、インド太平洋のインフラ・フレンドショアリングはDACリスト国における開発と日本の戦略的利益の双方に整合する、というものである。
- **DAC対非DAC競争。** 日本のODA金融は、多くの国で非DAC国家金融（特にCDBやChina Eximbankを通じた中国政策銀行金融）と競合している。これらはOECD-DAC譲許性枠組みの外側にあり、異なる条件で報告される。JICAへの含意は、案件条件が制約のない期間を持つ非DAC代替と競争力を持たなければならないことである。
- **BIS Basel III との相互作用。** JICA自体はBIS規制対象ではない（預金取扱銀行ではなく政府機関である）が、インフラ案件でJICA ODA融資の隣に置かれることの多いJBIC / メガバンク共同融資構造はBasel規制対象である。商業部分が存在する場合、メガバンクシンジケートの需要は、長期クロスボーダー・プロジェクトファイナンスのBasel IIIリスク加重資産処理に制約される。JICAのODA融資はこれに影響されないが、より広いプロジェクトファイナンス一式は影響を受け得る。
- **民営化論議。** DBJ（長く延期されている民営化計画を持つ）や一部の特殊会社機関と異なり、JICAの法的形態（独立行政法人）は構造的に民営化候補ではない。MOFA監督下で二国間ODAを実施しており、民営化は政策課題ではない。

## 7. 未解決の論点

- 2023 大綱実施後、JICAの経済安全保障フレーミング付きODAポートフォリオがインド太平洋でどう進化し、案件ミックスが実際に半導体 / 鉱物 / 水素の能力構築へ回転するか。
- PSIFの実際のチケットサイズ推移と、JICAのPSIFポートフォリオがJBIC OILと実質的に重なる規模まで成長するか、あるいは小規模 / 補完的なままか。
- JICA / JBIC / NEXI / JOGMEC が、2022 後のフレンドショアリング方向の下で、インド太平洋の複雑な複数手段案件パッケージ（特にIndia、Vietnam、Indonesia、Bangladesh、Philippines）を運用上どう調整するか。
- JICAがSTEP（本邦技術活用条件）階層の利用をセクター別・国別にどう報告し、STEP条件が水素 / アンモニア / GX領域へ拡大しているか。
- 近年の危機サイクル（Lebanon、Sri Lanka、Pakistan、Egyptのソブリンストレス期、2022 後のロシア制裁がレガシー案件に及ぼす影響）を通じたODA融資ポートフォリオの累積貸倒引当とカントリーリスク評価損。
