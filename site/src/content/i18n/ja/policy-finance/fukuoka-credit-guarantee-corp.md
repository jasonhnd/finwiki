# 福岡県信用保証協会 (Fukuoka Credit Guarantee Corporation)

## ウィキ上の位置づけ

この項目は、日本の信用保証制度における51 の都道府県 / 市レベル構成員の一つとして、[[policy-finance/INDEX|policy-finance index]]の下に位置づけられる。制度全体の概説である[[policy-finance/japan-credit-guarantee-system|Japan credit guarantee system]]、調整主体である[[policy-finance/national-federation-credit-guarantee-corporations|Japan Federation of Credit Guarantee Corporations]]とあわせて読む。残高でより大きい[[policy-finance/tokyo-credit-guarantee-corp|Tokyo CGC]]および[[policy-finance/osaka-credit-guarantee-corp|Osaka CGC]]と同業比較する。Fukuoka CGCは、九州地域における中小企業金融の重心として、九州7県経済全体にハブ・アンド・スポーク型の配置を支える点に特徴がある。

## 要約

Fukuoka Credit Guarantee Corporation（福岡県信用保証協会）は、日本の51協会から成る地域信用保証制度における福岡県の構成員である。その保証ブックは、九州の中小企業金融のアンカーとして構造的に重要である。福岡県には、九州最大の人口、金融サービス中心地（博多 / 天神）、TSMCを起点とする半導体クラスター（北九州工業地帯 + 熊本からの波及）、博多港物流、東アジアからのインバウンド観光の玄関口、農業 / 食品加工の厚い裾野が集中している。主要な協調融資先は[[JapanFG/fukuoka-bank|Fukuoka Bank ([[JapanFG/fukuoka-fg|Fukuoka FG]])]]、[[JapanFG/nishi-nippon-city-bank|Nishi-Nippon City Bank ([[JapanFG/nishi-nippon-fhd|Nishi-Nippon FHD]])]]、[[JapanFG/fukuoka-chuo-bank|Fukuoka Chuo Bank]]、そして密な九州の信金 / 信組ネットワークである。

## 2. 主要保証商品

- **一般保証** — 責任共有制度の下での標準的な中小企業向け銀行融資保証。
- **マル経** — 小規模事業者経営改善資金融資保証。
- **セーフティネット保証** — 半導体サイクルの反転、インバウンド観光ショック、港湾交通量の落ち込みなど、業況悪化の指定事象に対応する保証。
- **危機関連保証** — 全国的な緊急枠組みに基づく危機関連保証（100%保証）。
- **創業保証** — スタートアップ / 新事業保証。福岡市の成長するスタートアップ生態系で利用が活発。
- **事業承継保証** — 事業承継保証。
- **経営革新保証** — 経営革新 / 事業転換保証。
- **借換保証** — 借換 / 一本化保証。
- **半導体関連サプライチェーン特別保証** — TSMCを起点とする半導体クラスターのサプライヤー基盤に向けた、福岡県庁と連携した制度。

## 3. 県内産業構造との関係

福岡県は九州経済の構造的中心であり、Fukuoka CGCの保証ブックは複数クラスター型の姿を反映している。

1. **半導体 (Semiconductor cluster)** — 北九州 / 鞍手 / 宗像の工業ベルトは、2021–2024 に近隣の熊本で行われたTSMC-JASM（Japan Advanced Semiconductor Manufacturing）投資が地域サプライチェーン構築を触発して以降、主要な半導体サプライヤー基盤になっている。化学品、ガス、包装、精密加工、設備保守、エンジニアリングサービスを供給する福岡の中小企業は、capex拡大に保証付き信用を使う。
2. **物流 / 港湾 (Logistics and ports)** — 博多港（コンテナ）と北九州港（バルク + ro-roフェリー）は、貨物フォワーディング、倉庫、通関業務の厚い中小企業裾野を支える。博多は韓国向けフェリー / クルーズの主要玄関口であり、インバウンドの重要な入口でもある。
3. **観光 (Tourism)** — 福岡は、国内（博多 / 福岡市都市観光、太宰府、柳川、門司港）とインバウンド（韓国 / 台湾 / 中国からの直行便とフェリー）の双方で九州の観光ハブである。ホテル、飲食、小売、交通の中小企業裾野は密で、2020–2022の国境閉鎖で大きな打撃を受けた。
4. **食品 / 農業 (Food and agriculture)** — 福岡には、博多ラーメン / もつ鍋 / 明太子ブランド、清酒 / 焼酎醸造、農産物（いちご、柿、柑橘）を含む食品加工中小企業が相当集中し、東アジア向け食品輸出の流れも活発である。
5. **金融 / サービス (Financial and services)** — 天神と博多は、九州の金融サービスと専門サービスの中心地であり、多くの金融機関、専門職事務所、IT / スタートアップ事業者の九州本部が集まる。創業保証は、福岡市の成長するスタートアップ生態系で多用されている。
6. **重工業 (Heavy industry)** — 北九州には、日本製鉄八幡製鉄所に代表される歴史的な鉄鋼、化学、機械基盤が残る。戦後ピークより規模は縮小しているが、Tier-2/3 の中小企業サプライヤー裾野はなお稼働している。

Fukuoka CGCは機能的にも**九州地域の中小企業金融の重心**として働く。九州各県にはそれぞれCGCがあるものの、福岡クラスターの銀行・企業集中により、九州広域で活動する中小企業の信用は福岡拠点の銀行支店とFukuoka CGC保証を経由することが多い。

## 4. 協調融資とJFCの重ね合わせ

Fukuoka CGCの保証は通常、[[JapanFG/fukuoka-bank|Fukuoka Bank]]（主要地域銀行であり、熊本銀行、長崎の[[JapanFG/18-shinwa-bank|Eighteenth-Shinwa Bank]]、みなと銀行業務も含む[[JapanFG/fukuoka-fg|Fukuoka FG]]持株会社）、[[JapanFG/nishi-nippon-city-bank|Nishi-Nippon City Bank]]（[[JapanFG/nishi-nippon-fhd|Nishi-Nippon FHD]] / NNFH傘下）、[[JapanFG/fukuoka-chuo-bank|Fukuoka Chuo Bank]]（小規模な地域銀行）からの融資と並ぶ。九州の信金・信組ネットワークが長い中小企業裾野を支える。メガバンクの参加も重要で、[[JapanFG/mufg|MUFG]]、[[JapanFG/smfg|SMFG]]、[[JapanFG/mizuho-fg|Mizuho]]はいずれも博多 / 天神に支店を持ち、中堅から大きめの中小企業と法人顧客に対応している。

[[banking/regional-bank-consolidation-pattern|regional-bank consolidation pattern]]は九州で特に目立つ。Fukuoka FGの複数県ロールアップ（福岡 + 熊本 + 長崎）は、日本国内でも最大級の地域銀行持株会社構成の一つである。福岡県内では、強い博多の経済アンカーにより統合圧力は抑えられている。

公的金融の協調融資は[[JapanFG/jfc|JFC]]（政府系金融機関）を通じる。JFCの中小企業・小規模事業部門は、保証付き地域銀行トランシェと協調融資を行い、JFC信用保険が保証レイヤーを再保険する。[[JapanFG/shoko-chukin|商工中金 (Shoko Chukin)]]は、特に半導体サプライヤークラスターや食品輸出コンソーシアム周辺で、協同組織化された中小企業グループに参加する。

## 5. 代位弁済と求償

保証付きの福岡の中小企業がデフォルトした場合、Fukuoka CGCは保証契約に基づき融資金融機関へ代位弁済を行い、その後、求償を進める。回収後の残余損失は[[JapanFG/jfc|JFC]]が再保険する連合会プール型の信用保険レイヤーへ流れ、[[policy-finance/national-federation-credit-guarantee-corporations|全国信用保証協会連合会]]が51 協会全体の損失分担を調整する。

福岡の代位弁済量には特徴的なショック経路がある。(1) インバウンド観光の崩れ（国境閉鎖、近隣国との地政学的緊張）、(2) 世界的な半導体サイクル反転（サプライヤー基盤のcapexストレス）、(3) 北九州重工業の需要サイクル、(4) 博多港の交通量ショックである。福岡市都市部の回収価値は日本の地方県としては妥当であり、健全な求償回収率を支えている。

## 6. 危機期対応

COVID-19 緊急期（2020–2021）には、Fukuoka CGCは全国の実質無利子・無担保（ゼロゼロ融資 / ZeroZero loan）特別保証制度に大きく参加した。韓国 / 台湾 / 中国との国境閉鎖で主要なインバウンド流入が完全に止まり、インバウンド観光の中小企業セグメントに特に大きな打撃が及んだ。また、博多 / 天神の密な飲食・宿泊基盤にも強く影響した。福岡県庁は、Fukuoka CGCを通じて県内の緊急制度を重ねた。

2023–2025 を通じたポストCOVID正常化では、2020年型ZeroZero元本返済猶予が終了するにつれ、予想された代位弁済の波が生じた。一方で、インバウンド観光セグメントは強く回復し、韓国・台湾からの訪問者フローは2023–2024を通じて急速に戻った。危機関連保証の枠組みは、エネルギー価格ショック（2022–2023）で再起動された。2024–2025 を通じて、Fukuoka CGCのブックは一般保証 / 借換保証 / 事業承継保証のバランスへ正常化しつつあり、地域のTSMC-JASM生態系構築により半導体サプライヤーcapexトランシェが増えている。

## 7. 連合会制度上のアンカー

[[policy-finance/japan-credit-guarantee-system|Japan credit guarantee system]]アーキテクチャの中で、Fukuoka CGCは保証残高で上位5位層に入り、[[policy-finance/national-federation-credit-guarantee-corporations|全国信用保証協会連合会]]調整レイヤーにおける九州地域の中小企業金融ハブとして機能する。連合会が地域アンカーの運用テンプレートを評価する際、Fukuoka CGCの複数クラスター型ブック（半導体 / 物流 / インバウンド観光 / 食品 / 重工業）は、産業と観光の循環性を地域ハブ責任と組み合わせる県にとって有用な参照例となる。

[[JapanFG/jfc|JFC]]の連合会プール型信用保険レイヤーは、Fukuoka CGCのファーストロス代位弁済後の残余損失を吸収する。半導体サプライヤーcapexトランシェは運用上興味深い。TSMC-JASM熊本投資の福岡への波及により、従来の北九州重工業裾野とは異なるリスクサイクルを持つ保証セグメントが生まれるためである。半導体capexサイクルは単一会計年度内でも大きく振れる可能性があり、連合会再保険とJFC信用保険の平準化機能により大きな重みがかかる。[[JapanFG/fukuoka-fg|Fukuoka FG]]の複数県持株会社形態に見える[[banking/regional-bank-consolidation-pattern|regional-bank consolidation pattern]]は、保証関連リスクを県別にきれいに帰属させるうえで追加の複雑さを生む。

## 8. 同業県CGCとの比較

| 観点 | Fukuoka CGC | [[policy-finance/tokyo-credit-guarantee-corp|Tokyo CGC]] | [[policy-finance/hokkaido-credit-guarantee-corp|Hokkaido CGC]] |
|---|---|---|---|
| 支配的な産業ミックス | 半導体 + 物流 + インバウンド観光 + 食品 + 重工業 | サービス / 小売 / 金融 / IT / ホスピタリティ | 農業 / 漁業 / 観光 / 食品 |
| 地域的役割 | 7 県の九州中小企業金融の重心 | 全国ハブ | 北方の一次産業アンカー |
| インバウンド観光の地政学感応度 | 非常に高い（韓国 / 台湾 / 中国の玄関口） | 中程度（グローバルインバウンド） | 中程度（東アジアのスキー / 自然インバウンド） |
| 半導体エクスポージャー | 新しく成長中（TSMC-JASM波及） | 限定的 | 限定的 |
| 支配的地域銀行 | [[JapanFG/fukuoka-bank|Fukuoka Bank]] / [[JapanFG/fukuoka-fg|Fukuoka FG]]が支配的、[[JapanFG/nishi-nippon-city-bank|NNCB]]が二番手 | 支配的地域銀行なし | [[JapanFG/hokkaido-bank|Hokkaido Bank]] / [[JapanFG/hokuyo-bank|Hokuyo Bank]]の複占 |
| 複数県FG波及 | [[JapanFG/fukuoka-fg|Fukuoka FG]]持株会社が3 県（福岡 + 熊本 + 長崎）にまたがる | n/a | [[JapanFG/hokuhoku-fg|Hokuhoku FG]]が北海道 + 北陸にまたがる |

この比較は、Fukuoka CGCが[[policy-finance/japan-credit-guarantee-system|Japan credit guarantee system]]枠組みの下で単一県ブックというより、地域金融ハブとして独自の位置を持つことを示している。
