# 公的支援輸出信用に関する OECD アレンジメント

## ウィキ上の位置づけ

この項目は[[policy-finance/INDEX|policy-finance index]]の配下に置かれ、日本の公的輸出信用・投資支援スタックを制約する国際ルールブックである。アレンジメント上の最低プレミアム / 期間上限に制約される日本の保険機関は[[policy-finance/nexi|NEXI]]、アレンジメント適合カバーの商品側詳細は[[policy-finance/nexi-export-credit-insurance-products|NEXI export-credit insurance products]]、アレンジメント適合貸付条件に制約される日本の直接貸し手は[[JapanFG/jbic|国際協力銀行 (JBIC)]]、アレンジメント上限内で動く貸し手側フローは[[policy-finance/jbic-overseas-investment-underwriting-process|JBIC overseas-investment underwriting process]]と併せて読む。アレンジメントの範囲**外**にある上流エクイティ活動（エクイティは輸出信用ではない）は[[policy-finance/jogmec|JOGMEC]]、下流のアレンジメント規律付きファイナンスに先行することが多い上流ファイナンスは[[policy-finance/jogmec-equity-and-offtake-mechanics|JOGMEC equity and offtake mechanics]]、明示的に範囲**外**である ODA グレードのレーンは[[policy-finance/jica|JICA]]、より広いツールキットは[[policy-finance/japan-policy-finance-system|Japan policy finance system]]、九機関比較は[[policy-finance/japan-policy-finance-institution-mandate-matrix|Japan policy-finance institution mandate matrix]]、重層的なプロジェクトファイナンス・スタックの可視化は[[policy-finance/japan-project-finance-stack-diagram|Japan project finance stack diagram]]、日本の ECA 枠組みがアレンジメントと並行して発展した制度史は[[policy-finance/japan-eximbank-history|Japan Eximbank history]]、貿易促進 / 保険 / 融資の三本柱比較は[[trade/jetro-vs-nexi-vs-jbic-comparison|JETRO vs NEXI vs JBIC]]で読む。

## 要約

公的支援輸出信用に関する OECD アレンジメント（一般に「OECD Arrangement」または「OECD Consensus」）は、返済期間二年以上の公的支援輸出信用を規律する国際ルールブックである。OECD Trade and Agriculture Directorate が **Participants to the Arrangement** グループを通じて運営するが、法的拘束力のないジェントルマンズ・アグリーメントとして機能する。参加者間では法的には非拘束だが、OECD の通知・レビュー手続を通じて政治的・行政的に執行される。枠組みは四つの中核規律を置く。**1 最大返済期間**（セクターと国カテゴリで異なる。一般ルールでは Category I 国が概ね8.5-10 年まで、Category II 国が10 年まで。船舶 / 航空機 / 再エネ / 原子力 / 気候の Sector Understandings により上方修正）、**2 最低プレミアム率（MPR）**（OECD の **Country Risk Classification (CRC)** 尺度に連動。0 が最低リスク、7 が最高リスク）、**3 最低金利**（固定金利の公的貸付に使う Commercial Interest Reference Rates / CIRRs 制度）、**4 ガバナンス規律**（贈収賄防止、環境・社会デューデリジェンス、低所得国向け債務持続可能性）である。一般条件を修正する現行の **Sector Understandings** は五つある。**Ship Sector Understanding (SSU)**、**Aircraft Sector Understanding (ASU)**、**Nuclear Power Sector Understanding (NSU)**、**Climate Change Sector Understanding (CCSU)**、**Coal-Fired Electricity Generation Sector Understanding (CFSU)** である。2021 の Coal Sector Understanding 強化は、参加者間で新規の排出削減措置なし石炭火力輸出金融をほぼ禁止する効果を持ち、構造的に重要な気候政策上の結果である。参加者は十一。**Japan、EU（27 加盟国をブロックとして代表）、United States、United Kingdom、Canada、Korea、Australia、New Zealand、Norway、Switzerland、Türkiye** である。China、Russia、India、Brazil、その他非 OECD の国家金融は含まれない。この非対称性が、2020年代の戦略金融文脈（半導体、重要鉱物、新興市場インフラ、水素 / アンモニアサプライチェーン）で競争を形づくる。2023-2025 には、2009 Tokyo Action Statement 以来最大級の改革があった。**CCSU 2023 modernisation**（再エネ / 水素 / アンモニア / CCS / 一部鉄道・地下鉄などで気候友好型期間を多くのカテゴリで最長22 年へ拡大）、石炭火力発電金融を制限する **CFSU 2021**、および **Critical Minerals Sector Understanding** の可能性をめぐる継続議論である。

## 1. ルール枠組み

| 項目 | 詳細 |
|---|---|
| 正式名称 | Arrangement on Officially Supported Export Credits |
| フォーラム | OECD Trade and Agriculture Directorate / Trade Committee — Participants to the Arrangement 作業部会 |
| 法的地位 | 参加者間のジェントルマンズ・アグリーメント（政治的には拘束、法的には非拘束）。紛争解決ではなく、通知、共通条件からの逸脱レビュー、ピア透明性で執行される。 |
| 範囲 | 返済期間が**二年以上**の公的支援輸出信用。二年未満の短期貿易金融は除外。 |
| 対象 | **直接貸付**（例：JBIC OIL、US EXIM direct loan、K-EXIM）と**保険 / 保証**（例：NEXI、UKEF cover、Coface、EDC、KfW IPEX、Bpifrance）の両方。 |
| 参加者 | 11: Japan、EU（27 加盟国）、US、UK、Canada、Korea、Australia、New Zealand、Norway、Switzerland、Türkiye |
| 注目すべき非参加者 | China（CDB、China Eximbank、Sinosure）、Russia、India、Brazil、Saudi Arabia（Saudi EXIM を通じて ECA 活動が増加）、UAE。いずれの国家金融もアレンジメント条件に拘束されない。 |
| 現行版 | 定期的に更新。現在の統合本文と Sector Understandings は、定期改訂日のスタンプ付きで OECD ウェブサイトに掲載。 |
| 通知制度 | 参加者は範囲内取引を相互に、また OECD 事務局に通知しなければならない。共通条件から外れる取引は通知と、場合により common-line derogation が必要。 |

### 1.1 アレンジメントが規律するもの、しないもの

**アレンジメントが適用されるもの**:
- 長期（≥ 2 年）の公的支援輸出信用。バイヤーズクレジット、サプライヤーズクレジット、輸出信用相当のタイド・ソブリンローン。
- ECA による直接貸付と、商業銀行貸付への保険 / 保証カバー。
- 参加国の物品・サービス購入に紐づく外国借り手向けタイド貸付。
- アンタイド貸付であっても、公的支援が輸出信用相当の性格を持つもの。

**アレンジメントが適用されないもの**:
- 短期（< 2 年）貿易金融（Berne Union の短期協力枠組みで別途扱われる）。
- DAC の譲許性基準を満たす ODA ローン。JICA 実施の ODA ローンは範囲外。
- エクイティ投資。JOGMEC の上流資源プロジェクトへのエクイティ参加は構造的に範囲外。
- 公的支援のない純粋な商業銀行貸付。
- 国内政策金融。JFC の SME 金融、DBJ の国内ストラクチャードファイナンス、JHF の Flat 35 証券化支援は完全に範囲外。
- 非参加国の国家金融。中国政策銀行融資、Sinosure カバー、Russian VEB / Eximbank、Indian Exim Bank などは拘束されない。

アレンジメントに拘束されるか自由かの境界は、実務上最も重要な区別である。単一の複雑なクロスボーダープロジェクトでも、JBIC OIL（アレンジメント対象）、JOGMEC エクイティ（範囲外）、JICA ODA ローン（範囲外）、NEXI カバー（対象）、メガバンク商業債務（純商業銀行形態では範囲外。ただし NEXI カバーを伴うトランシェは公的支援分類を通じて対象に戻る）が併存し得る。

## 2. 中核規律

### 2.1 最大返済期間

期間は国カテゴリとセクターで制約される。

| 国カテゴリ | 一般ルール上の最大期間 |
|---|---|
| Category I（高所得 OECD） | 最長5 年（特定の場合は common-line で延長可能） |
| Category II（その他すべての国） | 最長10 年 |

Sector Understandings は特定セクターで期間を大きく延ばす。

| Sector Understanding | セクタールール上の最大期間 |
|---|---|
| Ship Sector Understanding (SSU) | 船舶について引渡しから最長12 年 |
| Aircraft Sector Understanding (ASU) | 大型民間航空機について最長12 年 |
| Nuclear Power Sector Understanding (NSU) | 原子力発電所輸出について最長18 年 |
| Climate Change Sector Understanding (CCSU) | 再生可能エネルギー / 水素 / アンモニア / CCS / 一部鉄道・地下鉄 / 気候友好型の水・貯蔵について最長22 年 |
| Coal-Fired Electricity Generation Sector Understanding (CFSU) | 2021 に強化。参加者間で新規の排出削減措置なし石炭火力輸出金融を実質的に禁止 |
| Railway Infrastructure | 気候友好基準を満たすプロジェクトで最長14 年 |

### 2.2 Minimum Premium Rate (MPR) — Country Risk Classification (CRC) 連動

OECD は **Country Risk Classification (CRC)** という単一尺度を**0 から7**まで維持している。0 は最低リスク（通常は高所得 OECD 国）、7 は最高リスク（差し迫ったデフォルトリスクを持つ重債務国）を示す。CRC は、参加者の国リスク専門家グループにより、定量モデルと定性レビューに基づき定期的（通常年に数回）に見直される。

| CRC | おおよその国例（例示。現在の OECD レビューで変わり得る） |
|---|---|
| 0 | 高所得 OECD（Japan、US、UK、Germany、France、Canada、Korea、Australia、Singapore など） |
| 1 | 低リスク新興市場（Taiwan、Czech、UAE、Saudi Arabia など。時点により異なる） |
| 2 | 各種新興市場（Indonesia、India、Malaysia、Mexico、Brazil など。時点により異なる） |
| 3-4 | 中位新興市場（Philippines、Egypt、Türkiye、Vietnam など。時点により異なる） |
| 5 | 高リスク新興市場 |
| 6 | 債務持続可能性に大きな懸念がある高リスク国 |
| 7 | 差し迫った / 実際のデフォルトリスクを持つ最高リスク国 |

MPR は、CRC、期間、取引構造（ソブリン対ソブリン近接対企業）、信用リスク緩和（担保、第三者保証）の関数として計算される。MPR は**下限**であり、ECA は MPR を上回る価格設定はできるが、下回ることはできない。MPR-MTRO（Most Restrictive Other Terms）規律は、他の条件レバーを緩めて MPR を相殺することを防ぐ。

### 2.3 最低金利 — Commercial Interest Reference Rate (CIRR)

公的な**固定金利**貸付（JBIC の通常の OIL 価格のような市場金利貸付とは別）では、参加者は対象通貨・期間の **Commercial Interest Reference Rate (CIRR)** 以上を課さなければならない。CIRR は OECD 事務局が、国債利回りにマージンを加えて毎月更新する。CIRR 制度は、固定金利の公的貸付が市場を下回る価格で出されないようにする。

### 2.4 ガバナンス規律

- **贈収賄防止**: 公的支援輸出信用取引で贈収賄に対する措置を取る明示的義務。
- **環境・社会デューデリジェンス**: OECD Recommendation on Common Approaches for Officially Supported Export Credits and Environmental and Social Due Diligence（"Common Approaches"）。環境 / 社会に悪影響を及ぼし得るプロジェクトの共通レビュー基準を定める。
- **低所得国向け債務持続可能性**: Principles and Guidelines to Promote Sustainable Lending Practices in the Provision of Official Export Credits to Low-Income Countries。IMF/IDA 適格低所得国向け貸付に特定の規律を置く。

## 3. Sector Understandings 詳細

### 3.1 Ship Sector Understanding (SSU)

船舶輸出の輸出信用を規律する。引渡しから最長12年、最低20%頭金、セクター固有 MPR を定める。造船業の経済性に合わせて一般ルールを調整する。日本の造船会社（[[JapanFG/INDEX|Imabari Shipbuilding, Japan Marine United, Mitsui E&S, others]]）と、JBIC / NEXI と並ぶメガバンク船舶ファイナンスに関係する。

### 3.2 Aircraft Sector Understanding (ASU)

大型民間航空機（Airbus、Boeing、Embraer、COMAC）の輸出信用を規律する。最長12年。ASU MPR 枠組みによるリスクベース価格設定が非常に精緻である。ASU は世界的に金額ベースで最も多く使われる Sector Understanding である。ASU における日本の役割は、外国メーカーから大型機を発注する日本所有航空会社（ANA、JAL）向けの JBIC / NEXI 支援と、選択的には日本供給の航空宇宙部品支援である。

### 3.3 Nuclear Power Sector Understanding (NSU)

原子力発電所輸出について最長18年を認める。原発プロジェクトの長い建設 / 試運転期間を反映する。日本の原子力輸出では、2017 前期、Westinghouse Chapter 11前の Toshiba-Westinghouse、2020 前期、Hitachi が UK Wylfa などから撤退する前の Hitachi-GE に歴史的に関係した。現在は、福島後の日本国内の原子力輸出姿勢により日本 ECA での利用は少ないが、制度としては有効である。

### 3.4 Climate Change Sector Understanding (CCSU) — 構造的な気候政策レバー

CCSU は2012 に設立され、2023, に大幅拡張された。気候移行時代で最も重要なアレンジメント修正である。CCSU の下で最長22年の期間対象となるカテゴリは次のとおり。

- 再生可能エネルギー発電（太陽光、風力、地熱、バイオマス、小水力、海洋）。
- 水素 / アンモニアの生産、貯蔵、輸送。
- Carbon capture, utilisation, and storage（CCUS / CCS）。
- 定義された閾値を上回るエネルギー効率プロジェクト。
- 気候友好型の水管理と貯蔵。
- 気候友好基準を満たす一部の鉄道 / 都市交通プロジェクト。
- 一部のバッテリー貯蔵と送配電近代化プロジェクト。

2023 CCSU modernisation は、2009 Tokyo Action Statement 以来の最重要アレンジメント改革であった。対象カテゴリと最大期間の両方を拡大し（一部カテゴリは現在22 年まで）、一般アレンジメント期間上限では実行困難だった長期の再エネ / 水素 / アンモニアファイナンスを NEXI / JBIC と他国 ECA が引き受けられるようにした。

### 3.5 Coal-Fired Electricity Generation Sector Understanding (CFSU) — 2021 の気候政策強化

CFSU は2015に設立された。2021 に大幅強化され、アレンジメント参加者間で新規の排出削減措置なし石炭火力輸出金融を実質的に禁止した。これは JBIC / NEXI の引受に大きな下流影響を持つ構造的な気候政策上の結果である。

- 2021前: Indonesia、Vietnam、Bangladesh、India などで、日本 ECA による石炭火力発電所支援は継続的で政治的に争われる政策方向だった。
- 2021 CFSU 強化: 新規の排出削減措置なし石炭火力発電所輸出金融について、アレンジメント適合チャネルを実質的に閉じた。
- 2021後: 日本 ECA の排出削減措置なし石炭ファイナンス案件パイプラインは閉鎖。CCS などの削減措置を統合する案件に限り選択的に継続。JBIC OIL パイプライン構成と「新規海外石炭なし」に関する日本の政策声明に大きな制度的影響。

## 4. Country-Risk Classification (CRC) メカニズム

CRC 制度は MPR 規律の運用上の中核である。CRC は次の性質を持つ。

- 参加者の国リスク専門家グループにより**定期的に見直される**（通常、年に数回）。
- マクロ経済 / 財政 / 対外収支 / 債務持続可能性指標に政治リスク上乗せを加えた**定量モデルに基づく**。
- IMF Article IV、World Bank Country Risk、Standard & Poor's / Moody's / Fitch ソブリン格付、OECD Economic Outlook など、二国間・多国間分析と照合して**レビューされる**。
- OECD ウェブサイトで公開参照リストとして**公表される**。各国の CRC 0-7 が定期更新される。
- 各参加国 ECA により**運用される**。NEXI / JBIC / US EXIM / K-EXIM / UKEF は各国取引の MPR 算定に CRC を直接使う。

CRC はアレンジメント遵守を超えた分析用途を持つ。国をまたいで一貫した手法を持つ数少ない公開ソブリンリスク分類の一つであり、一部の商業銀行や格付会社もソブリンリスク・ベンチマークのクロスリファレンスとして使う。

## 6. 気候移行アップデート（2023-2025）

2023-2025 期は、十年以上で最も活発なアレンジメント改革期である。

- **2023-04 CCSU modernisation**: 構造的に重要。再エネ / 水素 / アンモニア / CCS / 気候友好型水・鉄道地下鉄について、最長22年の長期 ECA 支援を可能にする。ASEAN、中東、Australia、Latin America の日本関連再エネ / 水素 / アンモニア案件に対する JBIC OIL と NEXI カバーへ直接影響する。
- **2021 CFSU 強化（2025まで実施効果が継続）**: 新規の排出削減措置なし石炭パイプラインを引き続き制約する。日本の「新規海外石炭なし」政策声明はアレンジメントの方向と整合した。
- **Critical Minerals Sector Understanding（議論中）**: 2026半ば時点で未採択。非アレンジメント国家金融と競争するため、リチウム、コバルト、ニッケル、レアアースなど重要鉱物プロジェクトの期間延長または MPR 修正を行うかが議論されている。
- **気候整合型 MPR 議論**: 気候移行プロジェクトのリスクプロファイルを反映するよう MPR を修正すべきかが議論されている。一部の再エネ / 水素 / CCS プロジェクトは、ソブリン CRC だけでは捉えにくい技術リスク / 市場発展リスクを持つ。

## 7. China と非アレンジメント国家金融 — 中心的な非対称性

アレンジメントには China が含まれない。中国国家金融（貸付は CDB / China Eximbank、保険は Sinosure）はアレンジメント規律の外で運用される。実務上の含意は次のとおり。

- **より長い期間**: 中国国家金融はアレンジメント上限に制約されず、例えば20-30 年のプロジェクトファイナンスを提示できる。これは、CCSU 適格外でアレンジメント参加者が10-12 年に制限される市場で問題になる。
- **より低い価格**: 中国国家金融は MPR 下限を下回る価格設定ができる。
- **異なる条件性**: 中国国家金融は Common Approaches の環境 / 社会デューデリジェンスを同じ形では遵守しない（ただし中国には独自のエクエーター原則隣接枠組みがある）。
- **異なるガバナンス**: 中国国家金融には通知 / ピアレビュー規律がない。

これは、日本 ECA と中国国家金融が類似プロジェクトで競合する市場で非対称競争を生む。特に ASEAN インフラ、アフリカ資源・インフラ、Latin American 資源案件、2022後の friend-shoring / サプライチェーン競争で顕著である。OECD 側の対応には、非 OECD 国家金融にアレンジメント規律参加を求める声、G7 / G20 枠組みを通じた多国間調整、戦略的重要プロジェクトでアレンジメント適合条件を競争力あるものにする選択的近代化（CCSU 拡張、Critical Minerals Sector Understanding の可能性）が含まれる。

## 9. 反論点

- **国家金融の不完全な地図。** アレンジメントはすべての国家金融の完全な地図ではない。非アレンジメント商品、エクイティ投資、ODA、非参加者支援を多く除外する。競争力学を評価する前に、「アレンジメントの境界」は最初の分析論点である。
- **ジェントルマンズ・アグリーメントの執行可能性。** アレンジメントは法的には非拘束である。執行は通知、common-line-derogation レビュー、政治的ピア圧力を通じて行われる。参加者は原則として、通知し政治的コストを引き受けることで共通条件から逸脱できる。
- **短期除外。** アレンジメントは返済期間 ≥ 2年を対象とする。短期貿易金融は範囲外であり、Berne Union の短期協力で別扱いされる。2 年という線引きは、長期化する貿易金融と短期プロジェクトファイナンスの時代にはやや恣意的だと批判されてきた。
- **気候政策レバーか障壁か。** CCSU 拡張は気候移行金融にとって構造的に重要だが、CFSU の石炭強化は、石炭火力が現実的なエネルギーミックスの一部であり続ける一部新興市場文脈で批判されてきた。反論は、参加者が石炭火力輸出金融を集合的に厳格化する決定は Paris Agreement の温度目標整合経路と整合する、というものである。
- **China 非対称性批判。** 2020年代のアレンジメントに対する中心的な構造批判は、OECD 参加者を規律する一方で中国国家金融が同じ規律外で動き、戦略金融文脈で競争非対称性を生むことにある。反論は、非参加国との競争が不均一であっても、アレンジメント規律は補助金競争の底辺への競争を防ぎ、公的輸出信用の長期持続可能性を守るというものである。
- **民営化論争（制度文脈）。** アレンジメントは民営化される機関ではなく、ルールブックである。ただし、その規律は ECA（NEXI、US EXIM、K-EXIM、UKEF など）が準商業的主体として動くか、政策補助ビークルとして動くかの中心にある。MPR / CIRR / 期間規律は、ECA を開かれた補助金ではなく、コスト回収 / 商業水準価格へ押しやる。
- **BIS Basel III との相互作用。** アレンジメントは ECA 保険 / 保証付きエクスポージャーの Basel III 取扱いを規制しない。しかし Basel 枠組み上の取扱い、特に無資金 ECA 保証の信用リスク削減 / CRM と、ECA 保険付きローンのリスクウェイト資産計算が、メガバンクのクロスボーダープロジェクトファイナンス貸付意欲を ECA カバーに実質的に依存させるメカニズムである。Basel 枠組みは、ECA 保証 / 保険を借り手のリスクウェイトから ECA の通常より低いリスクウェイトへ置換するものとして扱う。これが、アレンジメント適合 ECA カバーに対するメガバンク需要の構造的理由である。
- **マンデート拡大批判（Sector Understanding の増殖）。** Sector Understandings がなかった1980年代前から、五つの有効な SSU / ASU / NSU / CCSU / CFSU と六番目の Critical Minerals 議論へ増えたことは、「Sector Understanding の増殖」と呼ばれてきた。反論は、セクター固有の経済性が本当にセクター固有の期間 / 価格ルールを必要とするというものである。

## 10. 未解決の論点

- 日本の海外プロジェクトのうち、アレンジメント適合の JBIC / NEXI 支援を使うものと、範囲外の JOGMEC エクイティ / JICA ODA レーンを使うものの切り分け。
- CCSU 2023 modernisation が2025-2030 にかけて日本の公的金融構成を実際にどう変えるか。長期の再エネ / 水素 / アンモニア / CCS パイプラインは実質的に拡大するのか。
- Critical Minerals Sector Understanding が進むか、進む場合の期間 / 価格構造はどうなるか。
- 非 OECD 国家金融、特に China が G7 / G20 / 多国間枠組みを通じてアレンジメント相当の規律へ向かうことがあるか。
- Sri Lanka、Pakistan、Lebanon、Egypt、Argentina、Ghana、Zambia などのソブリンストレス事例が CRC 調整にどう流れ、ECA カバーの MPR に何をもたらすか。
- Russia 制裁後の執行インフラ。参加者が今後、制裁上乗せとアレンジメント遵守をどう調整するか。
- アレンジメント近代化が新しい取引類型（例：鉱物オフテイク担保型ストラクチャード貿易金融、水素 / アンモニア長期供給契約に組み込まれた ECA カバー）を認識するか。
- CRC の精度が実際のデフォルト / リストラクチャリング結果に対してどこまで保たれるか。CRC と将来のソブリン信用イベントのバックテスト。
