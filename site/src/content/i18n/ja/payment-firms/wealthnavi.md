# ウェルスナビ株式会社 (WealthNavi)

## ウィキ上の位置づけ

このエントリは [[payment-firms/INDEX|payment-firms INDEX]] の配下に位置する。ピア / 対比の文脈には [[payment-firms/rakuten-fg|楽天グループ金融事業 (Rakuten FG)]]、より広い制度 / 規制境界には [[fintech/japan-stablecoin-regulatory-landscape|日本 Stablecoin 法制度の三層構造（JPYC・USDC・Project Pax）]] と照らして読む。

## 要約

国内ロボアドバイザリー型資産運用サービス最大手。**預かり資産 2 兆円突破（2026-05-12 公表）**、運用者数約 41.6 万人（2024-09 末）。2015-04-28 柴山和久（元 マッキンゼー、財務省出身）創業、2016-07 サービス開始。2020-12-22 東証マザーズ上場（証券コード **7342**、現 グロース）。**2024-12-12 三菱 UFJ 銀行 [[megabanks/mufg]] が公開買付（TOB）で完全子会社化発表 ★** ── 国内ロボアドのパイオニアがメガバンク AI 金融エコシステムに統合される転換点。^[extracted]

### 運用方針

- **モダンポートフォリオ理論（MPT）**ベースの分散投資
- **米国上場 ETF 中心**（VTI / VEA / VWO / AGG / GLD / IYR / TIP 系の組み合わせ）
- 顧客のリスク許容度に応じた **5 段階のポートフォリオ**
- **自動リバランス**（半年に 1 回 + 乖離発動時）

### ビジネスモデル特性

- **手数料一律 1.1%（税込）**: 投信信託報酬とは別、預かり資産連動のストック型収益
- **顧客獲得コスト（CAC）**: ホワイトラベル提携で抑制、自社直販はデジタル広告中心

## 4. 規制・政策

- **主管**: 金融庁（FSA）
- **業登録**: 第一種金融商品取引業者・投資運用業者
- **加入協会**: 日本証券業協会、一般社団法人 日本投資顧問業協会
- **顧客資産保全**: 信託保全（三井住友信託銀行 等で分別管理）^[extracted]
- **直近政策論点**:
  - **2024-01〜 新 NISA 制度**: ロボアド業界全体に追い風（つみたて + 成長投資枠 拡充）
  - **2025〜 顧客本位の業務運営原則**: 手数料水準・運用パフォーマンス開示の透明性要求強化
  - **2024〜 金融サービス仲介業 制度**: ホワイトラベル提携の規制整理
