# ジェーシービー・インターナショナル (JCB International)

## ウィキ上の位置づけ

この項目は [[JapanFG/INDEX|JapanFG index]] の配下に位置づける。親会社 / スキーム運営者の関係を含む比較対象・対照文脈は [[card-issuers/jcb|JCB]]、より広い制度・規制上の境界は [[payments/japan-card-issuer-acquirer-processor-split|Japan card issuer / acquirer / processor split]] とあわせて読む。

## 要約

JCB International Co., Ltd. (JCBI) は [[card-issuers/jcb|JCB]] の **海外事業を担う 100% 子会社**。国際ブランド「JCB」の **海外発行 / 海外加盟店開拓 / ブランド・ライセンス** の 3 機能を一手に担う。日本国内発行は親会社 JCB が直接担うのに対し、海外ライセンシー銀行 (アジア・米州・欧州) との提携、海外加盟店ネットワーク (Discover / UnionPay との相互利用含む) の運営はすべて JCBI 経由。

## 2. 日本での事業ライン

JCBI 自身は **海外向け機能を持つが法人本店は東京 (港区南青山)** に置かれ、人員・本社機能は日本に集中。日本国内の活動は主に以下：

- **海外加盟店ネットワーク運営**: アジア (中国・韓国・台湾・東南アジア)、米州 (Discover 提携経由)、欧州主要観光地の加盟店契約・サポート。
- **訪日外国人 (インバウンド) 加盟店向けサポート**: 海外発行 JCB カード保有者の訪日決済を国内加盟店が受け入れる際の処理を担当。
- **海外ライセンシー銀行とのブランド契約**: アジアの提携銀行 (Bangkok Bank / Bank of Communications 等) への JCB ブランドライセンス供与。
- **Discover / UnionPay 等他ブランドとの相互利用契約**: JCB カードが海外で Discover 加盟店・UnionPay 加盟店で使える/その逆の業務契約管理。

国内消費者向け発行は親 [[card-issuers/jcb|JCB]] が担当するため、JCBI は国内一般消費者との直接接点はほぼ無い。

## 3. JCB / 国内スキームとの戦略・競争上の位置づけ

JCB は **3者間スキーム**（Amex 型・自社が発行+加盟店契約+ブランドを兼ねる）に近い構造を持つが、海外では **ライセンシー銀行を介した 4者間スキーム** のハイブリッド方式を採る。この海外展開モデルの実行主体が JCBI。Visa / Mastercard が完全 4者間の純粋スキーム運営者であるのに対し、JCB は「自社直接発行+ライセンス供与」を両方使い分ける。海外加盟店カバレッジは Visa / Mastercard に大差で劣るが、**Discover (米) / UnionPay (中) との相互利用提携** で穴を埋める戦略を取り、JCBI がその提携契約・運用の窓口となる。国内発行会社の視点では、[[card-issuers/smbc-card|三井住友カード]] や [[card-issuers/mufg-nicos|三菱UFJニコス]] のような Visa / Mastercard 主体の発行会社と異なり、JCB ブランド発行会社（楽天カード等の JCB 版・JCB プロパー）は JCBI 経由で海外加盟店網を共有する仕組み。

## 4. このページが重要な理由

- JCB の「海外ブランド事業」と「国内発行事業」を **法人レベルで分離整理** するための境界ページ
- Discover / UnionPay 相互利用契約の **契約主体** を明示 (JCB ではなく JCBI)
- 国際ブランド比較で「JCB は 3-party か 4-party か」という曖昧さを整理する
