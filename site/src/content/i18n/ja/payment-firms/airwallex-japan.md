# Airwallex Japan


## ウィキ上の位置づけ

この項目は [[JapanFG/INDEX|JapanFG index]] の下に位置する。ピア / 対比の文脈は [[payment-firms/wise-payments-japan|Wise Payments Japan]]、より広いシステム / 規制境界は [[payments/INDEX|payments index]] と照らして読む。

## 要約

Airwallex Japan株式会社は、Airwallex group の日本における B2B 決済およびクロスボーダー資金移動オペレーターである。公式発表は、第二種資金移動業者登録と、日本企業向けの onboarding / local payout / collection service 展開を示している。

## 1. 法人・ライセンス境界

| 項目 | 読み方 |
|---|---|
| 法人 | Airwallex Japan株式会社 |
| グループ | Airwallex global financial platform |
| 主要レーン | B2B global account、クロスボーダー送金、payout / collection、acquiring / payment acceptance |
| 金融庁登録ルート | 金融庁資金移動業者一覧：関東財務局長 第00100号、Airwallex Japan株式会社。 |

## 2. 事業上の役割

- 消費者送金だけでなく、グローバル化する SME とプラットフォーム事業者を対象にする。
- 決済受入、global accounts、FX、送金、法人支出インフラを組み合わせる。
- [[payment-firms/wise-payments-japan|Wise Business]]、[[payment-firms/payoneer-japan|Payoneer]]、[[payment-firms/paypal-pte-ltd-japan|PayPal]]、銀行の国際送金商品と比較する際に有用である。

## 3. この独立ページが重要な理由

Airwallex は金融庁の資金移動業者登録における比較的新しい日本参入者であり、そのビジネスモデルは PSP、acquirer、FX、treasury software と重なる。
