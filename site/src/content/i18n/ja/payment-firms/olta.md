# OLTA（オルタ）

## ウィキ上の位置づけ

本項目は[[JapanFG/INDEX|JapanFG index]]配下に置き、より広い運転資金スタックの中でBNPLに隣接する **B2Bファクタリングfintech** として扱う。[[payment-firms/bnpl-landscape|Japan BNPL landscape]]のシステムノート、BtoB BNPLの比較対象である[[payment-firms/net-protections-hd|Net Protections HD]]（NP 掛け払い）、SaaS主導型ファクタリング比較対象である[[payment-firms/money-forward|Money Forward]] / Biz Forwardと併せて読む。[[payments/japan-bnpl-credit-purchase-boundary|BNPL and credit-purchase boundary]]はファクタリングが信用供与に当たるかという論点を枠づけ、[[payments/credit-purchase-card-operators-japan-index|METI credit-purchase registry]]は関連する基準線、[[payments/japan-payment-scheme-economics-matrix|Japan payment scheme economics matrix]]はカード / コード決済レールに隣接する運転資金の経済性を位置づける。

## 要約

OLTA は **オンライン完結型クラウドファクタリング** をリードする独立系 fintech。中小企業の **未回収売掛債権** を即時に買い取って資金化することで、銀行融資や信販ファイナンスと並ぶ第三の運転資金チャネルを提供する。BtoB の文脈での "BNPL 的キャッシュフロー前倒し" カテゴリに該当し、[[payment-firms/net-protections-hd|Net Protections HD]] の NP 掛け払い、[[payment-firms/money-forward|Money Forward]] の Biz Forward、Yup、各銀行系ファクタリングと競合する。

## 2. 事業上の役割

- 中小企業が保有する売掛債権をオンライン申込・即日審査ベースで買い取り、資金化を提供する。銀行融資より審査が短期で、信用情報基準も異なる。^[extracted]
- 銀行・地銀の与信ラインで拾われにくい中小事業者の **キャッシュフロー前倒し** ニーズに応える。

## 3. この独立ページが重要な理由

[[payment-firms/bnpl-landscape|Japan BNPL landscape]] は B2C 後払いを主軸に整理されているが、BtoB の運転資金ファクタリングは **同じ "支払いを遅らせる / 入金を前倒す" 経済学** を共有しており、JapanFG 上で独立して扱う価値がある。銀行融資 / 信販 / BNPL / ファクタリングの **代替・補完関係** を読むためのアンカー。

## 4. 規制・政策

- **個人情報保護法 / 反社チェック**: 中小事業者の財務データ取得と取引先与信判断における個人情報・与信情報の取扱い。
