# FTX Japan 顧客資産 100% 返還ケース — 世界初の即時全額返還事例


## ウィキ上の位置づけ

本項目は [[exchanges/INDEX|exchanges index]] 配下に位置づける。同業比較・対照文脈は [[exchanges/jp-vasp-incident-history|国内 VASP 暗号資産流出事件史 (2014-2026)]]、より広い制度・規制境界は [[exchanges/fsa-vasp-registration-system|FSA 暗号資産交換業登録制度 — 番号体系・財務局管轄・登録要件]] と併読する。

## 1. 経緯

2022-11-11、FTX Trading Ltd. 等の Chapter 11 申請。グローバルでは Alameda Research との流動性危機 + 顧客資金の自社運用への混同により、世界中の FTX 顧客資産が凍結。日本法人 FTX Japan (旧 [[exchanges/jp-exchange-custodiem|QUOINE]]) は[[exchanges/jp-vasp-cold-storage-segregation-rules|分別管理義務]] (信託銀行への信託保管) を遵守していたため、顧客資産が無傷で残存していた点が決定的に異なる。

## 2. 返還 timeline

- **2022-11-10** — FSA、FTX Japan に業務停止命令 (一部) + 業務改善命令を発出
- **2022-11-30** — FSA が FTX Japan の顧客資産分別管理を確認 (信託保管が機能)
- **2023-02-20** — 顧客資産返還スキーム承認 ([[exchanges/jp-exchange-bitflyer|bitFlyer]] 提携ベースの引き出し UI 提供)
- **2023-02-21** — 顧客引き出し開始 — 世界初の即時 100% 返還を達成

## 3. 制度的意味

- 日本の分別管理義務 + 信託保管要件 が破綻時に実際に機能した実証ケース
- [[exchanges/fsa-business-improvement-orders-history|FSA 業務改善命令制度]] + JVCEA 自主規制 + 信託保管の三重構造の有効性
- 世界の VASP 監督モデル (米国 SEC・EU MiCA・HK SFC 等) で引用される事例
- 「自社カストディ + 顧客資金混同」を許す制度設計の危険性を逆照射

## 4. その後の経緯

2024-07、bitFlyer Holdings が FTX Japan (= QUOINE) を完全子会社化。2024-08 に [[exchanges/jp-exchange-custodiem|Custodiem]] へ改称し、クリプトカストディ事業へ転換中。[[exchanges/jp-vasp-ma-consolidation-history|VASP M&A 集約]]の象徴的事例の一つ。

## 5. 海外比較

米国 FTX International は債権者 100% 弁済を 2025-06 に達成 — 3 年以上を要した。一方、日本は破綻から約 3 ヶ月で全額返還を実現。制度設計差の劇的証明であり、分別管理 + 信託保管モデルの実効性を裏付ける。[[exchanges/jp-vasp-incident-history|JP VASP インシデント史]]上、最も成功した破綻処理ケース。
