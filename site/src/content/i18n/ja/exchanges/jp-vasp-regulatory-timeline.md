# 国内暗号資産 VASP 規制タイムライン (2014-2026)

## 概要

日本の暗号資産規制は、**国内大型インシデント** (Mt.Gox・Coincheck) と **国際協調圧力** (FATF Travel Rule・MiCA・GENIUS Act 等) の両方をトリガーに段階的に積層されてきた。世界で最も早く包括的な VASP 法体系を持つジュリスディクションのひとつで、現在は **資金決済法 + 金商法 + 自主規制 (JVCEA)** の三層構造で運用されている。詳細な登録制度は [[exchanges/fsa-vasp-registration-system]]、自主規制詳細は [[exchanges/jvcea-self-regulatory-overview]] を参照。

## 主要規制マイルストーン (年代順)

- **2014-02** — Mt.Gox 破綻 (詳細 [[exchanges/jp-vasp-incident-history]])。規制立法の動機を形成。
- **2016-05** — 改正資金決済法 成立。**世界初** の VASP 法的位置づけ。
- **2017-04** — 改正資金決済法 施行 → 「仮想通貨交換業」登録制度発足。
- **2018-01** — Coincheck NEM 580 億円流出事件 → セキュリティ規制の強化契機。
- **2018-04** — JVCEA (日本仮想通貨交換業協会) 設立。
- **2018-10** — JVCEA → **認定資金決済事業者協会** に格上げ。自主規制権限付与。
- **2019** — 金商法 + 資金決済法 改正成立。名称「仮想通貨」→「**暗号資産**」へ統一。
- **2020-05** — 改正金商法 施行 → 暗号資産デリバ取引業が金融商品取引業に編入 ([[exchanges/jp-vasp-derivative-license-system]])。
- **2020-05** — ホットウォレット **5% 制限** / コールド **95% 義務** / 顧客資産分別管理義務。
- **2021** — [[fintech/fatf-travel-rule-overview]] 国内実装開始。
- **2022-06** — 改正資金決済法 成立 → ステーブルコイン制度 ([[fintech/japan-epi-three-types-overview]]) 法制化。
- **2023-06** — 改正資金決済法 SC 規制 施行 ([[fintech/japan-stablecoin-regulatory-landscape]])。
- **2024-09** — JVCEA 会員数 50 社突破 (推定)。
- **2025+** — App Store 削除要請 ([[exchanges/fsa-foreign-exchange-warning-system]]) + JPYC 資金移動業 SC 発行開始。
- **2026** — Travel Rule 強化 + トラベルルールデータ共有体制整備。

## 規制の三層構造 (2026 時点)

- **第一層 (法律)**: 資金決済法 (現物 / SC) + 金商法 (デリバ) + 犯収法 (AML/CFT)
- **第二層 (省令・監督指針)**: FSA 監督指針 + 内閣府令
- **第三層 (自主規制)**: JVCEA 規則 + WhiteList 制度 (上場審査)

上位概念は [[fintech/japan-financial-regulation]] 参照。
