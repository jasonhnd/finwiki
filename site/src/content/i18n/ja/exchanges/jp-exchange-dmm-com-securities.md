# 株式会社DMM.com証券 — 日本金融商品取引業者（暗号資産デリバティブ）概要


## ウィキ上の位置づけ

本項目は [[exchanges/INDEX|exchanges index]] 配下に位置する。隣接する文脈は [[exchanges/fsa-vasp-registration-system|FSA 暗号資産交換業登録制度 — 番号体系・財務局管轄・登録要件]]、より広いシステム境界は [[fintech/japan-financial-regulation|日本金融規制 — トークン・暗号資産・決済に関する法体系]] とあわせて読む。

> FSA 関東財務局長（金商）第1629号 · JVCEA 第一種会員 #1043（暗号資産デリバティブ取引業のみ）· 第一種金融商品取引業登録日 2007-09-30
> **注意**：DMM Bitcoin（株式会社DMM Bitcoin · 廃業）とは**別法人**。同社は暗号資産現物交換業を担い 2025-03 廃業。本法人は FX/CFD/株式を主力とし、暗号資産は**デリバティブ（CFD）**登録のみ。→ [[exchanges/jp-exchange-dmm-bitcoin]]

## 1. 法人・株主

- **商号**: 株式会社DMM.com証券（英文: DMM.com Securities Co.,Ltd.）
- **設立**: 2006年12月6日
- **本店所在地**: 〒103-6010 東京都中央区日本橋2-7-1 東京日本橋タワー
- **資本金**: 98億円（資本準備金 73億9千万円）
- **代表取締役**: 谷川 龍二
- **親会社**: 合同会社 DMM.com（DMM.com グループ）— 上場なし
- **法人番号**: 公表なし（公式サイト非掲載）

## 2. 牌照・登録状況

- **第一種金融商品取引業**: 関東財務局長（金商）第1629号（登録日 2007年9月30日）
- **第二種金融商品取引業**: 同 第1629号に含む（2017年6月27日登録）
- **商品先物取引業**: 同 第1629号に含む（2011年1月1日登録）
- **JVCEA 第一種会員 #1043**: 暗号資産デリバティブ取引業のみ（現物 spot / 暗号資産交換業の登録なし）
- 電子決済手段等取引業・資金移動業：登録なし（公表なし）
- **加入協会**:
  - 日本証券業協会（協会員番号1105）
  - 一般社団法人 金融先物取引業協会（協会員番号1145）
  - 日本商品先物取引協会
  - 一般社団法人 第二種金融商品取引業協会（協会員番号480）
  - 一般社団法人 日本暗号資産等取引業協会（JVCEA 協会員番号1043）
- **自己資本規制比率**: 419.6%（令和8年3月末 = 2026-03）

## 3. 取扱通貨（暗号資産）

- 暗号資産**現物取引なし**。JVCEA 登録は「暗号資産デリバティブ取引業」のみ。
- 公式 CFD サービス（DMM CFD）の取扱22銘柄は **株価指数8銘柄＋商品CFD14銘柄** で構成。2026-05-18現在、暗号資産 CFD 銘柄の掲載は公式サイト上に確認できず。
- 会社概要ページには「暗号資産関連店頭デリバティブ取引」が事業内容に明記されており、JVCEA 登録と整合。暗号資産 CFD の実際の提供状況は「登録済・非公開サービス（または停止中）」と判断（公表なし）。

## 4. 業務範囲

同社の主力事業は以下の4本柱：

| サービス | 内容 |
|---------|------|
| DMM FX | 店頭外国為替証拠金取引 |
| DMM CFD | 株価指数8銘柄 + 商品CFD14銘柄 = 計22銘柄（店頭デリバティブ） |
| 株式取引（DMM株）| 国内株・米国株・IPO対応の上場有価証券等取引 |
| DMMバヌーシー | 競走用馬ファンド（商品投資関連業務） |

- 暗号資産現物（spot）：**なし**（別法人 DMM Bitcoin が担当していたが 2025-03 廃業）
- レバレッジ取引（暗号資産）：JVCEA 登録あり、ただし公式サービスページで暗号資産 CFD 銘柄は非掲載（2026-05-18現在）
- 法定通貨入金チャネル：銀行振込・クイック入金（全国約340金融機関対応）
- ステーキング・レンディング・NFT：非対応
- 法人サービス：公開情報なし

## 5. 市占・取引高

- 口座数・月間取引高：公表なし（開示資料非公開）
- 自己資本規制比率 419.6%（2026-03末）— 財務の健全性は高水準を維持
- 競合内ポジション：FX・CFD 分野では GMO クリック証券・外為どっとコムと並ぶ大手。暗号資産デリバティブ取引業として JVCEA 第一種会員 33 社のうち、現物なし・デリバティブ専業の事業者区分（13社のうちの1社）に属する。暗号資産 CFD の実取扱量は公表なし。

## 6. 沿革・重大事件

| 年月 | 事項 |
|------|------|
| 2006-12 | 設立（証券会社として事業開始） |
| 2007-09-30 | 第一種金融商品取引業として FSA 登録（関東財務局長（金商）第1629号） |
| 2011-01-01 | 商品先物取引業 追加登録 |
| 2017-06-27 | 第二種金融商品取引業 追加登録 |
| 時期不明 | JVCEA 第一種会員として「暗号資産デリバティブ取引業」登録（JVCEA 会員番号1043） |
| 2024-05-31 | **関係グループ会社 DMM Bitcoin（別法人）にて約482億円相当の暗号資産不正流出発生** |
| 2024-12-02 | DMM Bitcoin が廃業を発表・SBI VCトレードへの移管を発表。DMM グループの暗号資産現物交換業が消滅 |
| 2025-03-08 | DMM Bitcoin サービス全面終了。DMM.com証券は DMM Bitcoin とは別法人のため直接の業務影響は公表なし（公式声明なし） |
| 2026-03 | 自己資本規制比率 419.6% を開示（財務健全） |

**DMM Bitcoin（別法人）廃業との関係**：

DMM.com証券と DMM Bitcoin は別法人（住所は同じ東京日本橋タワーだが法人格・登録番号は別）。DMM Bitcoin の廃業（2024-12 発表・2025-03 完了）が DMM.com証券の JVCEA 暗号資産デリバティブ登録・事業継続に直接影響したとの公式発表は確認できない（2026-05-18現在）。ただし DMM グループとしての暗号資産現物取引業務は消滅したため、グループ全体の暗号資産事業は大幅に縮小した。

## 7. 戦略動向

- 直近の公開アナウンスは FX・CFD・株式取引の機能拡充が主。暗号資産分野の新規サービス公表は確認できず（2026-05-18現在）。
- DMM CFD は 2024年以前の14銘柄から22銘柄へ拡充（商品 CFD 8銘柄追加）。暗号資産 CFD の追加予定は公式サイトに記載なし。
- DMMバヌーシー（競走用馬ファンド）は引き続き他社に類のない特色サービスとして継続。
