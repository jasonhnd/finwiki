# BTCボックス株式会社 — 日本暗号資産交換業者概要


## ウィキ上の位置づけ

本項目は [[exchanges/INDEX|exchanges index]] の配下に位置する。隣接文脈は [[exchanges/fsa-vasp-registration-system|FSA 暗号資産交換業登録制度 — 番号体系・財務局管轄・登録要件]]、より広いシステム境界は [[fintech/japan-financial-regulation|日本金融規制 — トークン・暗号資産・決済に関する法体系]] とあわせて読む。

> FSA 登録番号 関東財務局 第00008号 · JVCEA 会員 #1008 · 登録日 2017-09-29

## 1. 法人・株主

- **商号**: BTCボックス株式会社
- **英文名**: BtcBox Co.,Ltd.
- **設立**: 2014年3月6日（David Zhang が資本金500万円で設立）
- **本店**: 〒103-0025 東京都中央区日本橋茅場町2丁目8番1号 BRICK GATE 茅場町 5階（2018年1月17日に旧住所・京橋1丁目から移転）
- **資本金**: 22億2704万円（資本準備金含む）—— 2019年以降の増資後の公表数値
- **親会社**: 株式会社TTXHoldings（2019年3月に David Zhang が全株式を譲渡し完全親会社化。TTXHoldings 自体は非上場の非公開会社）
- **上場状態**: 非上場
- **法人番号**: 7020001104824（国税庁法人番号公表サイト。番号指定 2015年10月5日）

## 2. 牌照・登録状況

- **暗号資産交換業**: 関東財務局長 第00008号（FSA、登録日 2017年9月29日）
- **JVCEA 会員**: 第一種会員 #1008（暗号資産交換業のみ。デリバティブ取引業に係る第二種相当のライセンス非保有）
- **日本資金決済業協会**: 会員番号 第00745号

## 3. 取扱通貨

- 取扱銘柄数: 7銘柄（2026-05-18 時点、公式ブログ沿革より）
- 主要銘柄: BTC · ETH · BCH · LTC · DOGE（2022年4月追加）· DOT（2022年6月追加）· TRX（2022年7月追加）
- **特色**: BTC 中心の spot 専業取引所。BTCは2014年4月サービス開始当初から取扱。BCH は国内でいち早く2017年8月に取引開始した実績を持つ。アルトコインの追加ペースは比較的緩慢で、小規模 spot 特化の方針を維持している。
- 販売所（かんたん売買）: 2020年10月よりサービス開始

## 4. 業務範囲

- **Spot 現物取引所**: 取引所形式（板取引）
- **販売所（かんたん売買）**: 2020年10月開始
- **デリバティブ / 信用取引**: なし（JVCEA 第一種・暗号資産交換業のみ）
- **暗号資産融資（貸付）**: 2016年2月開始。総資産最大3倍のBTCを借りられるサービス
- **あんしん定期購入**: 2021年9月開始（積立サービス）
- **法定通貨入金**: 銀行振込（住信SBIネット銀行からは手数料無料）、即時入金（2020年2月開始）
- **eKYC**: スピーディー本人確認 2020年4月開始
- **サポート言語**: 日本語・英語・中国語（3カ国語）

## 5. 市占・取引高

- **口座数・月間取引高**: 公表なし（2026-05-18 時点）
- **過去の市場地位**: 2014年4月サービス開始後、国内BTC/JPY取引量をほぼ独占し、2015年11月まで国内1位を継続（Wikipedia 記述、出典はbitcoin日本語情報サイト）
- **2019年4月時点**: CoinMarketCap「調整取引量・仮想通貨取引所 Top100」において日本の取引所の中で1位を記録（Wikipedia 記述、出典はCoinMarketCap）
- **現在のポジション**: 取引高・口座数の最新公表データなし。[[exchanges/jp-exchange-bitflyer|bitFlyer]]・[[exchanges/jp-exchange-gmo-coin|GMOコイン]]・[[exchanges/jp-exchange-coincheck|Coincheck]] 等の大手と比較すると niche / mid に位置する独立系老舗として業界内で認知されている（[[exchanges/jvcea-spot-volume-statistics-analysis|JVCEA スポット取引量統計分析]] 参照）。
- **顧客層**: 国内個人（海外個人・法人口座も開設可）

## 6. 沿革・重大事件

| 年月 | 事項 |
|------|------|
| 2014-03 | BTCボックス株式会社設立（David Zhang、資本金500万円） |
| 2014-04 | BTC/JPY 取引所「BTCBOX」サービス開始（同時に LTC も取扱開始）。現存する日本の暗号資産取引所の中で最も古くからサービスを継続している取引所の一つ |
| 2015-07 | BTC月間取引量が50万BTCを超える |
| 2016-02 | 暗号資産融資サービス開始 |
| 2017-08 | BCH を国内でいち早く取引開始 |
| 2017-09 | 暗号資産交換業者として FSA 登録完了（第00008号） |
| 2017-11 | ETH の取扱開始 |
| 2017-12 | 茅場町の新オフィスへ移転 |
| 2018-10 | 株式会社TTXHoldings 設立（代表取締役 David Zhang、資本金500万円） |
| 2018-10 | Liquid Network（Blockstream のビットコインサイドチェーン）参加を表明 |
| 2019-03 | David Zhang が全株式を TTXHoldings に譲渡 → TTXHoldings が完全親会社化 |
| 2019-05 | 親会社引受増資 2億円。社長交代：David Zhang 会長就任、辻治俊 新社長就任 |
| 2019-06 | 追加増資 3億円（親会社引受） |
| 2020-02 | 即時入金サービス開始 |
| 2020-04 | スピーディー本人確認（eKYC）開始 |
| 2020-10 | かんたん売買（販売所）サービス開始 |
| 2021-09 | あんしん定期購入サービス開始 |
| 2022-04 | DOGE 取扱開始 |
| 2022-06 | DOT 取扱開始 |
| 2022-07 | TRX 取扱開始 |
| 2025-04 | サービス開始から10周年を迎える |

**重大事件**: 2014年4月サービス開始以来、ハッキング等のセキュリティ事件は公表されていない（公式ブログ・Wikipedia の記述、2019年時点）。FSA による行政処分は公表記録で確認されていない（2026-05-18 時点）。

## 7. 戦略動向

- 直近の公開リリースは blog.btcbox.jp（情報コラム中心）に限定。新規銘柄追加・大型資本施策の公開発表は 2022年以降確認されていない（公表なし）。
- spot 専業に特化した小規模独立系路線を維持。デリバティブや staking 等の機能拡張については公表計画なし。
- サービス開始10年超（2025年4月時点）を強調したブランド訴求を継続。

## 8. 関連人物

- **辻治俊（つじ はるとし）**: 代表取締役社長（2019年5月就任）
- **David Zhang（張大為）**: 創業者、元代表取締役。中国出身のエンジニア。2014年3月設立。2019年5月会長就任。2019年3月に全株式を自ら設立した TTXHoldings に移管。
- **顧問法律事務所**: アンダーソン・毛利・友常法律事務所
- **会計監査**: ゼロス有限責任監査法人
