# Mt.Gox 破綻処理タイムライン（2014-2026）


## ウィキ上の位置づけ

この項目は [[exchanges/INDEX|exchanges index]] の下位に位置づけられる。同業比較・対照の文脈は [[exchanges/jp-vasp-incident-history|国内 VASP 暗号資産流出事件史 (2014-2026)]]、より広いシステム・規制境界は [[exchanges/fsa-vasp-registration-system|FSA 暗号資産交換業登録制度 — 番号体系・財務局管轄・登録要件]] とあわせて読む。

## 1. 概要

2014-02-28、Mt.Gox (株式会社 MTGOX、東京・渋谷) が約 850,000 BTC + 約 28 億円の流出を発表し民事再生を申立て。その後破産手続へ移行し、2018-06 に世界初となる「暗号資産事業者の民事再生再開」決定を受けた。2024-06 にようやく現物弁済が開始され、2026 年現在も継続中 — 12 年に及ぶ世界最長級の暗号資産破綻処理事例。[[exchanges/jp-vasp-incident-history|JP VASP インシデント史]]の起点であり、日本の VASP 制度設計の最大トリガーとなった。

## 2. タイムライン 主要日付

- **2014-02-07** — 出金停止 (technical issue を公式理由)
- **2014-02-28** — 民事再生申立 · 約 850,000 BTC + 28 億円流出を発表
- **2014-03〜10** — 流出 BTC 一部回収 (約 200,000 BTC)
- **2014-04-24** — 民事再生手続中止 · 破産手続開始決定 (東京地裁)
- **2018-06-22** — 民事再生手続再開決定 — 世界初の暗号資産事業者再生事例
- **2019-04** — 民事再生計画案提出
- **2021-11-16** — 民事再生計画案認可決定
- **2024-06-24** — 弁済開始 (5-7 月 集中弁済期間)
- **2025-2026** — 残債権・継続弁済中

## 3. 弁済構造

- **CRC (Crypto Receiver Coupon)** vs **JPY 早期一括弁済** の選択制 — 債権者ごとに選択可
- **BTC + BCH** 現物弁済 (時価ベース · 2014 当時の保有比率を反映)
- 弁済プラットフォーム提携: **Bitstamp / Kraken / [[exchanges/jp-exchange-bitflyer|bitFlyer]] / [[exchanges/jp-exchange-sbi-vc-trade|SBI VC トレード]]**
- 国内・海外債権者を各登録プラットフォーム経由で振込・出庫する分散モデル

## 4. 制度的意味

- 2014 流出 → 2017 改正資金決済法 (VASP 登録制度) 立法の最大トリガー — [[exchanges/jp-vasp-regulatory-timeline|JP VASP 規制 timeline]]の起点
- 民事再生 → 破産 → 民事再生再開の 3 段階転換は世界初 (法解釈上、ビットコインを「金銭債権」ではなく「物」として扱う判例形成)
- 複数取引所による弁済プラットフォーム提携モデルは [[exchanges/ftx-japan-100pct-return-case-study|FTX Japan (2023)]] の bitFlyer 提携返還スキームの先例
- [[exchanges/fsa-business-improvement-orders-history|FSA 業務改善命令制度]] · [[exchanges/jp-vasp-cold-storage-segregation-rules|分別管理義務]]はすべて Mt.Gox 教訓の反射

## 5. グローバル比較

米国 [[exchanges/ftx-japan-100pct-return-case-study|FTX International]] は 2022-11 破綻 → 2025-06 に 100% 弁済達成 (約 3 年)。一方 Mt.Gox は 12 年 (2014→2026 継続中) — 規制設計差ではなく、訴訟複雑性 (流出 BTC の所有権争訟 + Karpelès 個人責任訴訟 + 海外債権者調整) の差。「破綻処理の迅速性は事業者の分別管理体制で決まる」という制度設計命題を逆照射する事例。
