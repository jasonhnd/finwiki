# 日本 認定NPO法人 — 認定ルートと寄付者の税制優遇

## ウィキ上の位置づけ

この項目は [[non-profit/INDEX|non-profit INDEX]] 配下に位置づけられ、[[non-profit/japan-npo-houjin-overview|Japan NPO法人 overview]] で記述される基本形態に対する **税のアップグレード・コンパニオン** である。それが解き放つ寄付者控除の褒賞は、[[non-profit/japan-koeki-vs-general-zaidan-comparison|公益財団 vs 一般財団 vs 一般社団 vs NPO法人 comparison]] で扱う 公益 の利点と並行し、[[non-profit/japan-research-grant-map-2026|Japan research grant map 2026]] にマップされる公的資金調達と研究助成のパターンを追求する組織にとって最も重要である。

## TL;DR

単なる [[non-profit/japan-npo-houjin-overview|NPO法人]] はその寄付者に **税控除を付与しない**。**認定NPO法人 (Certified NPO)** ステータスは、それを行う別個のアップグレードである —— それは寄付を個人について **所得税控除可能**（所得控除 か 税額控除 の選択）、法人寄付者について **損金算入**-適格、相続税について控除可能にする。ゲートウェイは **Public Support Test (PST)** である：組織が単一の篤志家ではなく公衆によって広く支援されているという証明。

## 誰が認定するか、そしてどの法律の下で

| 属性 | 詳細 |
|---|---|
| 法律 | 特定非営利活動促進法 (NPO Act)；認定 ルートは **2011/2012 改革** で追加、ガバナンスは **2024** に厳格化 |
| 認定者 (所轄庁) | **都道府県知事 / 指定都市市長** —— 2012 年改革で国税庁から分権化 |
| 前提条件 | すでに設立された **NPO法人** でなければならない |
| 標準認定有効期間 | **5 年**、更新可能（更新は失効の 6-3 ヶ月前に申請） |

歴史的に重要な転換：認定権限は 2012 年に **NTA から 所轄庁 へ移行** し、認定者を、基礎となる NPO をすでに監督する機関と整合させた。しかし税の *恩典* は、依然として NTA が管理する国の税法を通じて流れる。

## The Public Support Test (PST)

PST は問う：**この NPO は公衆によって広く支援されているか？** 申請者は三つの代替基準の **いずれか一つ** を満たさなければならない：

| 基準 | 閾値 |
|---|---|
| **相対値基準 (relative value)** | 寄付が評価期間にわたる経常収入 (経常収入) の **1/5 (20%)** 以上 |
| **絶対値基準 (absolute value)** | 評価期間にわたり、それぞれ **¥3,000 以上** を拠出する平均 **年間 100 人以上の寄付者** |
| **条例個別指定 (ordinance designation)** | 住民福祉に貢献する団体として地方自治体の条例で指定（個人住民税控除の対象） |

PST は認定 NPO 制度の分析的中核である：それは「広範な公衆支援」を測定可能な財務テストに変換し、単一の大口パトロンによって資金提供される組織よりも **多くの小口寄付者** によって資金提供される組織を意図的に優遇する。

## 特例認定 —— 若い NPO のための PST 免除

設立から **5 年以内** の NPO については、**特例認定 (special certification)** が **一度** 利用可能であり、健全なガバナンスと信頼できる公益基盤を依然として要求しつつ **PST を免除** する：

- **有効期間：3 年**、**更新不可**。
- ブリッジとして設計されている：寄付者の実績のない若い NPO が、後に完全な PST を通過するために必要な寄付者基盤を構築する間も、寄付者に税控除を提供できる。

これは「暫定的」公益ステータスの NPO 側のアナログであり、新しい市民組織のコールドスタートの障壁を下げる。

## 寄付者の税制優遇 —— 非対称の褒賞

恩典は **寄付者** に発生し、エンティティにではない。これがまさに 認定 ステータスが組織の資金調達基盤を広げる理由である：

| 寄付者の種類 | 認定NPO法人 の下での恩典 |
|---|---|
| 個人所得税 | **所得控除 (income deduction) OR 税額控除 (tax credit)** —— 寄付者がより有利なものを選択 |
| 個人相続税 | 認定NPO に寄付された相続資産は課税相続財産から控除可能 |
| 法人寄付者 | 寄付は一般の寄付上限を超える **拡大した 損金算入枠** に適格 |
| 住民税 (個人住民税) | 地方自治体が条例で NPO を指定する場合に利用可能 |
| NPO 自身 | 収益事業 から非営利活動への移転について **みなし寄付金 (deemed-donation)** 取扱い |

**税額控除** の選択肢は、課税所得を単に減らすのではなく税負担を直接減らすため、小口の個人寄付者にとって特に強力である —— 草の根の寄付を広げる意図的な政策レバーである。

## これが 公益 ステータスとどう比較されるか

| レバー | 認定NPO法人 | 公益財団 / 公益社団 |
|---|---|---|
| 基本形態 | NPO法人（別個の NPO 法律） | 一般財団 / 一般社団（一般法人法 family） |
| 認定者 | 所轄庁（都道府県 / 指定都市） | 公益認定等委員会（内閣府 / 都道府県） |
| ゲーティングテスト | **PST**（広範な公衆支援の財務テスト） | 公益認定（公益 + 50%-支出 + ガバナンス + 資産 テスト） |
| 寄付者所得税控除 | あり（所得控除 / 税額控除） | あり（所得控除 / 税額控除） |
| 許可活動 | 20 NPO カテゴリー | 列挙された公益カテゴリー |

両ルートは異なる法律上の出発点から **同じ寄付者控除の目的地** に到達する。両者の間の選択は上流にあり —— それは [[non-profit/japan-koeki-vs-general-zaidan-comparison|the cross-form comparison]] における **NPO法人 vs 一般社団/財団** の形態決定である —— 後付けの税の選択ではない。市民的正統性と低い設立コストのために NPO 形態を選んだ組織は 認定 経由で登り；財団/社団を選んだ組織は 公益認定 経由で登る。

## 戦略的読解

- 認定 ステータスは **見栄のラベルではない** —— それは **資金調達経済のゲート** である。それなしには、公的資金調達を行う NPO は寄付について控除可能な 公益 や 認定 の同業者と構造的不利の下で競争する。
- **絶対値基準（100 人の寄付者 × ¥3,000）** のパスは、真に草の根の組織にとってしばしば最も達成可能である；**相対値基準（収入の 20%）** のパスは寄付が支配的な資金調達モデルに適する。どの PST の枝に向けてエンジニアリングするかを選ぶことは、コアな初期段階の財務決定である。
- 金融・政策研究のテーマに隣接する NPO については、認定-ステータスの資金調達を [[non-profit/japan-research-grant-map-2026|research grant map]] のプログラム助成および [[policy-finance/INDEX|policy-finance INDEX]] の制度的文脈とペアにすることが、最も強靭な資金ミックスを与える。
