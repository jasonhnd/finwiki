# 日本 一般社団法人 vs 公益社団法人 — 一般 vs 公益 の社団法人

## ウィキ上の位置づけ

この項目は [[non-profit/INDEX|non-profit INDEX]] 配下に位置づけられ、基本財産ベースの財団ページ [[non-profit/japan-general-foundation-establishment-requirements|Japan 一般財団法人 establishment requirements]] に対する **会員ベース (社団)** のコンパニオンである。[[non-profit/japan-koeki-vs-general-zaidan-comparison|公益財団 vs 一般財団 vs 一般社団 vs NPO法人 comparison]] のクロス形態テーブルの 社団 列を拡張し、[[non-profit/japan-npo-houjin-overview|Japan NPO法人 overview]] の別個の法律の市民形態と対比する。

## TL;DR

**社団 (association)** は会員によるガバナンスである：議決権を持つ 社員 が総会で方向性を決定するのに対し、**財団 (foundation)** は会員を持たない基本財産によるガバナンスである。一般 と 公益 の社団形態の両方は、その財団のいとこと **同じ法律ファミリー** に存在する：

| 形態 | 法律 | 監督庁 | 寄付者税控除 |
|---|---|---|---|
| 一般社団法人 | 一般社団・財団法人法 (2006) | 法務局（登記のみ） | なし（一般則） |
| 公益社団法人 | 公益法人認定法（一般 法の上に） | 内閣府 / 都道府県知事（公益認定等委員会） | あり |

その関係は **段階的** である：まず登記によって 一般社団法人 を設立し、その後オプションで **公益認定** を申請して 公益社団法人 となり、それが重い公益テストと継続的監督と引き換えに寄付者側の税控除を解き放つ。

## 一般社団法人 —— 基本形態

| 属性 | ルール |
|---|---|
| 法定根拠 | 一般社団法人及び一般財団法人に関する法律 (2006, 法律第48号) |
| 最低会員数 (社員) | 設立時 **2 以上** |
| 理事 (理事) | **1 以上**（小規模協会では 理事会 / board はオプション） |
| 監事 (監事) | 理事会 が組成される場合 **のみ** 必要 |
| 会員-理事の重複 | **許可される**（社員 は 理事 も兼ねうる） |
| 最低資本 / 基本財産 | **なし** —— ¥3M-最低の 財団 形態との重要な対比 |
| 定款の公証人認証 | 必要（¥50,000） |
| 設立登録税 | ¥60,000 |
| 許可活動 | **あらゆる適法な目的**（NPO の 20-カテゴリーリストよりはるかに広い） |
| 設立期間 | 通常 **1-2 ヶ月**（登記のみのルート） |

一般社団法人 は低摩擦・広範スコープの非営利形態である：財団 に比べて安価（基本財産なし）、速い（登記のみ）、目的において無制限。それは業界団体、専門職団体、業界グループ、および NPO の 20-カテゴリーリストに適合しない会員主導の市民活動のデフォルトのビークルである。

### 非営利型 vs 普通法人扱い (tax sub-types)

一般財団法人 と同様に、一般社団法人 は二つの税務上の取扱いに分かれる：

- **非営利型法人** —— 法定の非営利条件（利益分配条項なし、解散時の残余財産条項、理事会構成ルール）を満たす；**収益事業 のみが課税される**。
- **普通法人扱い** —— 非営利型条件に不合格；通常の会社のように **すべての営利活動について広く課税される**。

大半の稼働中の 一般社団 は法人税の露出を制限するために 非営利型 を選ぶ。

## 公益社団法人 —— 公益のアップグレード

公益社団法人 となるには、既存の 一般社団法人 が 公益社団法人及び公益財団法人の認定等に関する法律 (2006) の下で **公益認定** を申請する。実質的なバーは高く、公益財団 テストを反映する：

- **公益活動テスト** —— 活動は列挙された公益カテゴリー内に該当しなければならない。
- **財務テスト** —— おおよそ公益活動への **支出の 50%**（公益目的事業比率）。
- **ガバナンステスト** —— 独立した理事、関係者取引の制限、報酬限度。**2024 年改革（2024 年 4 月施行）** は、特別利害関係（配偶者、三親等内親族）にある理事/監事が理事会の **1/3** を超えられないというルールを厳格化した。
- **資産テスト** —— 保持資産の限度。

その代わりにエンティティは以下を得る：**公益活動への法人税免除**、保護された **「公益社団法人」名称**、そして —— 非対称の褒賞 —— **寄付者側の税控除**。

## 横並び比較

| 次元 | 一般社団法人 | 公益社団法人 |
|---|---|---|
| 構造基盤 | 会員 (社員) | 会員 (社員) |
| 設立 | 法務局での登記 | 一般社団 設立 **後** の 公益認定 |
| 最低会員数 | 2+ | 2+（継承） |
| 許可活動 | あらゆる適法な目的 | 列挙された公益活動を中心としなければならない |
| エンティティ税 | 非営利型: 収益事業 のみ課税；それ以外は広く | 公益活動について非課税 |
| 寄付者所得税控除 | なし | あり（所得控除 か 税額控除） |
| 法人寄付者の恩典 | 一般の寄付上限のみ | 拡大した 損金算入枠 |
| 継続的監督 | なし（商業登記のみ） | 公益認定等委員会 + 内閣府 / 都道府県 |
| 開示 | 軽い（登記された定款/役員） | 重い（年次届出、公的登記） |
| 設立期間 | 1-2 ヶ月 | 6-12 ヶ月（公益認定 レビューを含む） |

## 社団 vs 財団 —— 軸を選ぶ

最初の決定は **会員 vs 基本財産** であり、一般 vs 公益 ではない：

- 定義された構成員 —— 会員 —— が投票を通じて方向性を駆動すべきとき **社団** を選ぶ。業界団体、専門職協会、会員資金の市民グループ。
- 専用の財産プール（基本財産）が、会員なしで、評議員と理事によって固定された目的に向けて管理されるべきとき **財団** を選ぶ。助成と長期ホライズンのフィランソロピー。[[non-profit/japan-general-foundation-establishment-requirements|Japan 一般財団法人 establishment requirements]] を参照。

その軸を固定した後にのみ、**一般 vs 公益** の問い（寄付者控除 vs 監督負担）が適用され —— かつそれは 社団 と 財団 の両方に対称的に適用される。

## 戦略的読解

- **一般社団 → 公益社団** の階段は、**一般財団 → 公益財団** の階段の会員側のアナログである。中間の 一般 フェーズは、公益認定等委員会 のレビュアーが公益ステータスを付与する前に期待する事実上の実績期間である。
- 助成を受ける、または資金調達を行う団体については、公益 ステータスの下での寄付者控除が対応可能な寄付者基盤を実質的に広げ、これが成熟した法人フィランソロピーと研究ビークルがしばしば 公益 に登る理由である —— [[non-profit/japan-research-grant-map-2026|Japan research grant map 2026]] にマップされた資金エコシステムに関連する。
- 公益アームを運営する上場金融グループは、アームズレングスのガバナンスのために 公益 形態を典型的に好む；そのようなグループのエンティティレベルのカバレッジは [[non-profit/INDEX|non-profit INDEX]] に、規制の背景は [[fintech/japan-financial-regulation|Japan financial regulation]] にある。
