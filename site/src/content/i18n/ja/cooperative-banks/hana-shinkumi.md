# ハナ信用組合

## ウィキ上の位置づけ

この項目は [[JapanFG/INDEX|JapanFG index]] の配下で、**首都圏（東京）の在日コリアン民族系信用組合アンカー**として位置づける。[[cooperative-banks/osaka-shogin-shinkumi|大阪商銀信用組合]]（関西商銀系）、[[cooperative-banks/yokohama-kogin-shinkumi|横浜幸銀信用組合]]（神奈川幸銀系）、[[cooperative-banks/aichi-shogin-shinkumi|信用組合愛知商銀]]（中京商銀系）と比較し、[[banking/credit-cooperative-registry-japan|Japan credit cooperative registry]] に整理される在日コリアン系信用組合群の中で読む。

### 組合員資格 ^[extracted]

民族系信組のため、組合員資格は **業域 + 民族コミュニティ**の二重構造 (中小企業等協同組合法準拠):

1. **業域要件**: 商工業者・自営業者中心 (員外利用厳格)
2. **民族コミュニティ**: 在日韓国・朝鮮人 + その配偶者・親族・関連事業者
3. **地区要件**: 関東圏を中心とする定款指定地区に在住・在勤・事業所を有する者

### 朝銀系信用組合の歴史的経緯 ^[extracted/inferred]

| 年代 | 事象 |
|---|---|
| 1950〜60 年代 | 在日朝鮮人 (主に北朝鮮支持系) 商工人向けの相互扶助金融として **「朝銀」**系信用組合が各地に設立 |
| 1970〜80 年代 | 全国主要都市に朝銀信組ネットワーク確立 |
| 1990 年代 | バブル崩壊で **不動産・建設業向け融資**の不良債権化が深刻化 |
| 2001〜2002 | **朝銀信組連鎖破綻問題** — 全国の朝銀信組数行が経営破綻、預金保険機構による大規模資金援助 (ハナ信組の承継対象には朝銀東京・朝銀千葉・朝銀新潟・朝銀長野・朝銀関東の 5 組合があり、事業再生に約 4,000 億円の公的資金が投入された) |
| 2002-03-20 | **ハナ信用組合設立** (同年 12-30 開業)。商号「ハナ」は朝鮮語で「一つ・統合」を意味し、複数の旧朝銀信組の事業統合を象徴 |
| 2010 年代〜 | 在日コミュニティの世代交代対応 + AML/CFT 強化対応 |

## 6. 監督・規制

- **主管**: 金融庁・関東財務局
- **法的根拠**: 中小企業等協同組合法 + 協同組合による金融事業に関する法律 (協金法)
- **預金保険**: [[financial-regulators/dic|預金保険機構]] 一般金融機関枠
- **直近政策論点**:
  - **AML/CFT 強化** (跨境送金監視) — 民族系信組固有規制対応コスト + **対北朝鮮制裁関連**の制裁監視要件
  - **組合員世代交代** (在日 2 世・3 世以降の信組離れリスク)
  - **韓国系銀行 (新韓銀行東京支店等)** との競合
