# JA三井リース

## ウィキ上の位置づけ

本項目は [[cooperative-banks/INDEX|cooperative-banks INDEX]] の配下に置く。[[leasing-firms/fuyo-lease|芙蓉総合リース]] を同業・対比の文脈として、[[banking/INDEX|banking index]] をより広い制度・規制境界として併読する。JA / 農林中金 + 三井系の合弁体として、JA 側の文脈は [[cooperative-banks/norinchukin|農林中央金庫]] とあわせて読む。

## 要約

JA三井リース は 2008-04 に **JAリース** (農林中央金庫 / JA 全農系列) と **三井リース事業** (三井物産系) が合併して発足した準大手総合リース会社。**JA グループ** (農林中央金庫 [[cooperative-banks/norinchukin|農林中金]] / JA 全農 / 各 JA) と **三井グループ** (三井物産・三井住友信託銀行 等) の二大資本系列を併せ持つ稀有な構造で、JA グループ向け農業機械リース・物流施設リースの 専属金融機能と、一般総合リース機能の双方を担う。非上場 (大株主は農林中金 + 三井物産 + 三井住友信託 等の戦略株主構成)。

## 2. 事業ライン

- **農業関連リース**：トラクター・コンバイン・農業機械リース (JA 各支店経由)
- **物流施設リース**：JA 系倉庫・物流施設 + 一般物流施設
- **一般産業機器リース**：工作機械・産業機械
- **不動産リース・ファイナンス**：オフィスビル・商業施設
- **環境・再エネ**：太陽光・バイオマス等の PJ ファイナンス
- **航空機リース**：オペレーティングリース・JOL (日本語 Operating Lease)
- **海外**：アジア (中国・タイ・ベトナム等) 拠点
- **医療・介護**：医療機器・介護施設

## 3. 戦略と競争上の位置づけ

JA三井リースの最大の差別化要因は **JA グループの全国エンドユーザー基盤**。日本全国の JA (約 600 弱の単位農協) を通じて農業機械・物流施設・農業関連設備へのアクセスを持ち、これが他のリース会社にない独自チャネル。

[[leasing-firms/smfl|SMFL]] (SMFG + 住友商事 JV) や [[leasing-firms/fuyo-lease|芙蓉総合リース]] (旧 富士銀行系) と比べ、規模では準大手の中位だが、JA / 農業セクターでの圧倒的シェアを持つ。三井物産経由で航空機リース (JOL)・グローバル案件への参画も行い、業容バランスは「JA 向け専属金融 + 一般リース + グローバル案件」の三本柱。

[[cooperative-banks/norinchukin|農林中金]] とは資本関係 + 案件ルートの両面で密接連携。農林中金が JA グループの資金運用機関として巨額の金融資産を運用するなか、JA三井リースは「リース・実物アセット」の役割を分担。

[[finance/japan-acquisition-finance|日本アクイジションファイナンス]] の文脈では、農業 M&A や食品セクターの設備投資ファイナンスで関与することがある。

## 4. このページが重要な理由

- JA グループの「リース・実物アセット」機能を理解する単一ノード
- 二大系列 (JA + 三井) の ハイブリッド 構造は他の大手リースには見られない稀少構造
- [[cooperative-banks/norinchukin|農林中金]] のリスク資産配分 (運用機関) → JA三井リース (実物アセット運用) という JA グループ内分業の理解
- 三井系内で [[leasing-firms/smfl|SMFL]] と 同業（SMFG / 住友商事 系）かつ対比対象（JA / 三井物産 系） という対比対象
