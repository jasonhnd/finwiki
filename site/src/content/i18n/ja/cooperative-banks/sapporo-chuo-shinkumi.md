# 札幌中央信用組合

## ウィキ上の位置づけ

この項目は [[cooperative-banks/INDEX|cooperative-banks INDEX]] の配下で、[[cooperative-banks/iwaki-shinkumi|いわき信用組合]]（東北）と並ぶ **北海道の地域型信組アンカー**として位置づける。143の信用組合群は [[banking/credit-cooperative-registry-japan|Japan credit cooperative registry]] に収録されており、札幌市場における信金と信組の違いは [[cooperative-banks/sapporo-shinkin|札幌信用金庫]] と対比する。

## 要約

札幌中央信用組合は、北海道札幌市中央区南2条西に本店を置く **札幌都市圏の中小企業者・住民を組合員基盤とする地域型信用組合**。FSA 登録番号 3 (北海道財務局所管)、法人番号 5430005003132。北海道 7 信組のうち札幌都心部を本拠とする代表行で、[[cooperative-banks/zenshin-kumi|全国信用協同組合連合会 (全信組連)]] 会員、[[financial-regulators/dic|預金保険機構]] 対象。**地域 + 業域**の二重組合員資格、札幌市内 SME 融資・住宅ローン・商店街金融が事業の中核。北海道経済の中心地札幌でメガバンク・[[regional-banks/hokuyo-bank|北洋銀行]]・[[regional-banks/hokkaido-bank|北海道銀行]] と棲み分けるニッチ金融機関。^[extracted]

### 組合員資格 ^[extracted]

地域型信組のため、組合員資格は **地区 + 業域**の二層構造 (中小企業等協同組合法準拠):

1. **地区要件**: 札幌中央信用組合の定款で定める地区 (主に札幌市) に在住・在勤・事業所を有する個人・法人
2. **規模要件**: 中小企業者 — 従業員 300 人以下または資本金 3 億円以下 (業種により異なる)

### 北海道信組ブロック (登録比較)

北海道には FSA 登録 上 7 信用組合 (登録番号 1-7):

| 信組 | 本店 | 類型 |
|---|---|---|
| ウリ信用組合 | 札幌市中央区大通西 | 民族系 (在日韓国・朝鮮人系) |
| 釧路信用組合 | 釧路市 | 道東地域型 |
| **札幌中央信用組合** ★本ページ | **札幌市中央区南2条西** | **道央札幌地域型** |
| 空知商工信用組合 | 美唄市 | 道央内陸・商工系 |
| 十勝信用組合 | 帯広市 | 道東十勝地域型 |
| 函館商工信用組合 | 函館市 | 道南・商工系 |
| 北央信用組合 | 札幌市中央区 | 道央・地域型 |

> 注: 札幌市内には **札幌中央信組 + 北央信組 + ウリ信組**の 3 信組が並存。札幌中央が中央区南2条西、北央が中央区南1条西8、ウリ信組が中央区大通西12と、いずれも中央区中心部に集中する密集配置。

## 6. 監督・規制

- **主管**: 金融庁・北海道財務局
- **法的根拠**: 中小企業等協同組合法 + 協同組合による金融事業に関する法律 (協金法)
- **預金保険**: [[financial-regulators/dic|預金保険機構]] 一般金融機関枠
- **直近政策論点**:
  - **DX 対応** (組合員向けスマホアプリ・電子契約)
  - **AML/CFT 強化**対応コスト
  - **札幌都心人口集中**の継続 vs **道内地方部過疎化**との相反トレンド
