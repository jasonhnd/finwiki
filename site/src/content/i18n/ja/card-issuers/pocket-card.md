# ポケットカード


## ウィキ上の位置づけ

本項目は[[JapanFG/INDEX|JapanFG index]]配下に置く。[[card-issuers/epos-card|エポスカード (EPOS Card)]]とは同業・対比の文脈で、[[payments/INDEX|payments index]]とはより広い制度・規制上の境界を読むために参照する。

## 要約

ポケットカードは リテール / 提携クレジットカード領域の重要な運営会社。P-one / ファミマ / 旧流通系カードの文脈を持ち、銀行系・EC系・通信系カードとは異なる提携カード戦略を読むためのページ。

| 項目 | 読み方 |

| 法人 | ポケットカード株式会社 |

| 比較ページ | [[card-issuers/credit-saison|Credit Saison]], [[payment-firms/famima-digital-one|Famima Digital One]], [[card-issuers/jcb|JCB]] |
| 規制ルート | METIの割賦販売法登録事業者一覧および公式会社概要。 |

## 2. 事業上の役割

- 提携カードは、小売関係をリボルビング信用と決済データに変換する。
- 非メガバンク系カード競争を読むうえで有用な事例である。
- 小売決済アライアンスとロイヤルティ経済を整理する際に、同社の境界が重要になる。

## 3. この独立ページが重要な理由

金融持株会社だけを追うウィキでは見落としやすい会社である。独自の公開プロフィールと明確なリテールカード戦略を持つ、名のあるカード発行会社として扱う。
