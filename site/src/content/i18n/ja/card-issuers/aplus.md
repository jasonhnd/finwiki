# アプラス (APLUS)


## ウィキ上の位置づけ

この項目は [[card-issuers/INDEX|card-issuers INDEX]] の下に位置づける。隣接する文脈は [[banking/INDEX|banking index]]、より広いシステム境界は [[securities/INDEX|securities index]] とあわせて読む。

## 要約

アプラスは [[regional-banks/sbi-shinsei-bank|SBI新生銀行]] グループの総合信販オペレーターである。ショッピングクレジット、カード、ペイメント事業を中心に、銀行グループと消費者信用・割賦信用をつなぐ接点になる。

## 1. 法人・ライセンス境界

| 項目 | 読み方 |
|---|---|
| 法人格 | 株式会社アプラス |
| グループ | [[regional-banks/sbi-shinsei-bank|SBI Shinsei Bank]] / SBI グループ |
| 主な領域 | ショッピングクレジット、カード、ペイメントサービス、割賦金融 |
| 規制上の確認ルート | METI の割賦販売法登録事業者一覧と SBI 新生グループの公式会社ページ。 |

## 2. 事業上の役割

- SBI 新生の消費者金融スタックの中で、割賦信用とカード機能を提供する。
- 銀行貸出、保証、ペイメントサービスを販売金融の関係で補完する。スキーム経済性は [[payments/japan-payment-scheme-economics-matrix]] に置く。
- SBI 統合後も銀行グループが既存の消費者信用資産をどう活用するかを読むうえで有用である。イシュア / アクワイアラ / プロセッサの分解は [[payments/japan-card-issuer-acquirer-processor-split]] を参照する。

## 3. この単独ページが重要な理由

APLUS は単なるブランド名ではない。預金銀行や証券とは異なる動きをするショッピングクレジットとペイメント商品の運営会社である。
