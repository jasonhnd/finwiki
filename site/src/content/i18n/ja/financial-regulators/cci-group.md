# CCIグループ (CCI Group)

## Wiki 内の位置づけ

CCI Group は、旧 [[regional-banks/hokkoku-fhd|Hokkoku Financial Holdings]] ルートに対応する、現行名の銀行持株会社ページである。[[regional-banks/hokkoku-bank|Hokkoku Bank]]、[[banking/regional-bank-api-digital-partnership-route|regional bank API / digital partnership]]、[[fintech/jp-stablecoin-dcjpy|deposit-token / digital-money routes]]、[[banking/INDEX|banking domain]] と併せて読まれたい。

このページは、持株会社の戦略、現行の法人名、グループの拡大、デジタル / クロスボーダー / コンサルティング事業、および FSA 銀行持株会社の境界のために用いる。事業会社としての銀行そのものについては [[regional-banks/hokkoku-bank|Hokkoku Bank]] を用いる。

## 概要

CCI Group は、北國銀行を中核として構築された現行の上場銀行持株会社である。公開会社概要によれば、当グループは 2021-10-01 に 北國フィナンシャルホールディングス として設立され、2025-10-01 に 株式会社CCIグループ へ商号変更した。

このページが重要なのは、その改称が単なる表面的なものではないからである。公式の会社概要は銀行持株会社としての目的を記す一方、グループ自身は金融、コンサルティング、BPO、システム開発、資産運用、地域活性化、海外事業にまたがって自らを位置づけている。それにより CCI Group は、単なる「旧 北國 FHD」ページではなく、現行名の地域金融プラットフォームとなる。

## 公開コンテキスト

CCI Group は、地方銀行持株会社が伝統的な預金・貸出の枠を超えて拡大しようとする、より明示的な例の一つである。公開会社概要は、グループの価値を銀行、コンサルティング、投資、システム、地域活性化、海外機能に結びつけている。

これは [[banking/regional-bank-api-digital-partnership-route|regional bank API / digital partnership]] で捉えられた、より広範な地方銀行のプレイブックに合致する。地方銀行は地元に根ざしたまま、テクノロジー、アドバイザリー、決済、クロスボーダー支援を収益化しようとすることができる。

Microsoft / オープンシステム / デジタル地方銀行のリーダーシップをめぐる旧来の [[regional-banks/hokkoku-fhd|Hokkoku FHD]] の物語は依然として関連性を持つが、現行の CCI という名称は、グループのリブランドされたプラットフォームの野心をより的確に捉えている。

## 規制 / 調査上の注記

- 現行の法人名の境界が重要な場合は、FSA 銀行持株会社一覧を用いる。
- 2025  の改称およびグループ事業カテゴリについては、会社概要 / IR ページを用いる。
- [[regional-banks/hokkoku-bank|Hokkoku Bank]] は持株会社とは分けて扱う；事業会社としての銀行は [[financial-regulators/zenginkyo|全国銀行協会 (Zenginkyō)]] がマッピングする業界団体レイヤーの内側に位置する。
- 預金トークンやデジタルマネーの主張を分析する際は、ブランディングから商品範囲を想定するのではなく、[[fintech/jp-stablecoin-dcjpy|tokenized deposit / DCJPY context]] および公開会社開示へルートする。
- 地域金融の変革を分析する際は、[[regional-banks/fukuoka-fg|Fukuoka FG]]、[[regional-banks/shizuoka-fg|Shizuoka FG]]、[[regional-banks/kyushu-fg|Kyushu FG]] と比較する。

## 留意点

- 公開されたグループのカテゴリは、各事業が実質的な収益規模を持つことを意味しない。
- より広いグループブランドは、銀行、コンサルティング、資産運用、地域開発の境界を曖昧にしうる；主張を行う前に事業体を特定すること。
- このページは投資助言ではない。
