# CCIグループ (CCI Group)

## ウィキ上の位置づけ

CCI Group は、旧 [[regional-banks/hokkoku-fhd|Hokkoku Financial Holdings]] ルートに対応する現在名称の銀行持株会社ページである。[[regional-banks/hokkoku-bank|Hokkoku Bank]]、[[banking/regional-bank-api-digital-partnership-route|regional bank API / digital partnership]]、[[fintech/jp-stablecoin-dcjpy|deposit-token / digital-money routes]]、[[banking/INDEX|banking domain]] とあわせて読む。

このページは、持株会社戦略、現在の正式名称、グループ拡張、デジタル / クロスボーダー / コンサルティング事業、金融庁の銀行持株会社境界を扱うために使う。事業銀行そのものは [[regional-banks/hokkoku-bank|Hokkoku Bank]] を使う。

## 概要

CCI Group は Hokkoku Bank を中核にした現在上場中の銀行持株会社である。公開会社概要によれば、グループは 2021-10-01 に北國フィナンシャルホールディングスとして設立され、2025-10-01 に株式会社CCIグループへ商号変更した。

このページが重要なのは、商号変更が単なる見た目の変更ではないためである。公式プロフィールは銀行持株会社としての目的を示しつつ、グループを金融、コンサルティング、BPO、システム開発、資産運用、地域活性化、海外事業にまたがる存在として位置づけている。そのため CCI Group は、単なる「旧 Hokkoku FHD」ではなく、現在名称で読むべき地域金融プラットフォームである。

## 公開情報上の文脈

CCI Group は、従来の預金・貸出を超えて領域拡大を図る地域銀行持株会社の中でも、比較的明示的な例である。公開プロフィールは、グループ価値を銀行、コンサルティング、投資、システム、地域活性化、海外機能に結びつけている。

これは [[banking/regional-bank-api-digital-partnership-route|regional bank API / digital partnership]] で整理する、より広い地域銀行のプレイブックに合う。地域銀行は地域に根ざしたまま、テクノロジー、助言、決済、クロスボーダー支援を収益化しようとする。

Microsoft / オープンシステム / デジタル地域銀行リーダーシップをめぐる旧 [[regional-banks/hokkoku-fhd|Hokkoku FHD]] の文脈はなお関連するが、現在の CCI 名称のほうが、ブランド再構築後のプラットフォーム志向をよりよく表す。

## 規制・調査メモ

- 現在の正式名称境界が重要な場合は、金融庁の銀行持株会社一覧を使う。
- 2025  の名称変更とグループ事業区分には、会社概要 / IR ページを使う。
- [[regional-banks/hokkoku-bank|Hokkoku Bank]] は持株会社と分けて扱う。
- 預金トークンやデジタルマネーの主張を分析する場合、ブランド名から商品範囲を推定せず、[[fintech/jp-stablecoin-dcjpy|tokenized deposit / DCJPY context]] と公開会社開示へ誘導する。
- 地域金融変革を分析する場合は、[[regional-banks/fukuoka-fg|Fukuoka FG]]、[[regional-banks/shizuoka-fg|Shizuoka FG]]、[[regional-banks/kyushu-fg|Kyushu FG]] と比較する。

## 注意点

- 公開されているグループ区分は、各事業に重要な収益規模があることを意味しない。
- 広いグループブランドは、銀行、コンサルティング、資産運用、地域開発の境界を曖昧にし得るため、主張する前に運営主体を特定する。
- このページは投資助言ではない。
