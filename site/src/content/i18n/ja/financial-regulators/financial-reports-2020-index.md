# JapanFG 財務報告 2020+ インデックス

## 要約

本ページは [[JapanFG/INDEX|JapanFG namespace]] 中の金融会社 / 金融機関の **2020 年以降の公式財務報告、統合報告書、ディスクロージャー誌、有価証券報告書、親会社年次報告書入口** を一つのインデックスにまとめている。

方針は **archive-first**：2020 年以降の全レポートをカバーできる公式 archive / backnumber ページを優先記録；公式ページに完全な archive がない場合のみ、直接 PDF サンプル、親会社年次報告書、規制開示、または「公開年次報告書ギャップ」を記録する。

## 対象範囲

- 対象：JapanFG 下の銀行、FG、政策金融、信託、証券、保険、信販、リース、決済、fintech、暗号資産、短資会社、外資在日法人 / 支店。
- 除外：会社横断ページ [[payment-firms/bnpl-landscape|bnpl-landscape]]、[[financial-regulators/boj-monetary-policy|boj-monetary-policy]]、[[card-issuers/installment-sales-act-2020-amendment|installment-sales-act-2020-amendment]]、[[financial-regulators/japan-policy-finance-map|japan-policy-finance-map]]、[[financial-regulators/jsda|jsda]]、[[financial-regulators/local-bond-market|local-bond-market]]、および deprecated 誤リンク [[regional-banks/ssjbank|ssjbank]]。
- 日付基準：2026-05-20 JST。2026-03 期 年次報告書 / 統合報告書 が未公開の場合は無理に補完しない。
- 「2020+」は 2020 年以降追跡可能な年次報告書リンクを指す；2020 年以降に設立された会社の場合、表中で前身会社 / 親会社境界を明記する。

## ステータス凡例

| ステータス | 意味 |
|---|---|
| 完了 | 公式 archive / backnumber が 2020-2025 をカバー、または設立が遅いが前身境界が明確 |
| 境界 | 単一上場主体ではなく、親会社、前身会社、子会社、または日本実体境界の説明が必要 |
| 部分 | 公式入口は存在するが、年ごとの PDF 展開が必要、または 2020 ギャップあり |
| ギャップ | 公開 年次報告書 / disclosure archive が見つからず、会社、格付、規制、または親会社入口のみ掲載可能 |

## 次段階のダウンロード方針

1. このページを URL 台帳の信頼できる起点として使う。
2. `部分` と `境界` の行だけを対象に第2パスを実行し、正確な 2020-2025 PDF URL を抽出する。
3. ダウンロードした PDF はまず wiki 外に保存する。例: `~/Downloads/JapanFG-financial-reports-2020-plus/`, grouped by slug.
4. ハッシュとファイル名を正規化してから、このページまたは各社ページへ PDF マニフェストをリンクする。
5. 日本ページで必要な場合を除き、外国銀行親会社の報告書をすべて JapanFG に取り込まない。多くはグローバル 300-500 page 年次報告書s.
