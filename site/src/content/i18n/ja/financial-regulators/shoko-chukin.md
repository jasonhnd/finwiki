# 商工中金 (Shoko Chukin Bank)

## Wiki ルート

本項目は [[financial-regulators/INDEX|financial-regulators INDEX]] に属する。同業 / 対比の文脈については [[megabanks/mufg|三菱 UFJ FG (MUFG)]] を、より広い体系 / 規制の境界については [[banking/INDEX|banking index]] を併せて参照のこと。

## TL;DR

旧 政策金融機関の **民間商業銀行化版**。1936 (昭和 11 年) 商工組合中央金庫法に基づく特殊会社として設立、2008-10 株式会社化、**2023-10 政府保有株式売却完了 → 完全民営化**。中小企業組合・組合員向け融資 + 一般中小企業融資に特化。2016-2017 **危機対応融資不正事件** (約 4,800 件規模) で金融庁から 2 度の業務改善命令を受け、経営陣総入れ替え・ガバナンス改革を経て民営化完了に至る。メガバンク・地銀・信金との中小企業マーケットでの三つ巴競合下、規模で劣る中で「組合員密着 + 事業承継/M&A」差別化を模索する民営化後フェーズ。^[extracted]

## 1. 会社概要

**正式名**：株式会社商工組合中央金庫 ^[extracted]
**英名**：The Shoko Chukin Bank, Ltd. ^[extracted]
**略称**：商工中金 / SCB
**設立**：1936-11-30 (商工組合中央金庫法による特殊会社) ^[extracted]
**株式会社化**：2008-10-01 (商工組合中央金庫法廃止 + 株式会社商工組合中央金庫法 施行) ^[extracted]
**完全民営化**：**2023-10 政府保有株式売却完了** ^[extracted]
**本店**：東京都中央区八重洲 2-10-17 ^[extracted]
**上場**：非上場 (株主は中小企業組合等 + 民間)

### 主要株主 (民営化後)

- 中小企業組合 + 組合員企業 (歴史的に商工中金の出資母体)
- 民間金融機関・事業会社 (2023-10 政府売却後の引受先)
- ※ 2023-10 まで政府が一定比率を保有していたが、売却完了で **政府保有 0%** に ^[extracted]

### 重要年表 ^[extracted]

| 年月 | 事象 |
|---|---|
| 1936-11-30 | 商工組合中央金庫法 制定 → 特殊会社として設立 |
| 戦後〜1990s | 中小企業金融の三大公的金融機関 (商工中金・中小公庫・国民公庫) の一角として位置付け |
| 2002-04 | 政策金融改革議論本格化 |
| **2008-10-01** | **株式会社化** — 商工組合中央金庫法廃止 + 新法施行、政策金融再編 (政策投資銀行・日本政策金融公庫等と並行) |
| 2008-2016 | 危機対応融資 (リーマン後・震災後・金融危機等) を政府委託で実行 |
| **2016-10** | **危機対応融資不正事件発覚** — 鹿児島支店端緒で内部告発 |
| **2017-05-09** | **金融庁 第 1 次業務改善命令** — 不正取扱い 760 件確認時点 |
| 2017-10-25 | **金融庁 第 2 次業務改善命令** — 全件調査結果 **約 4,800 件・職員 444 名関与** 判明 |
| 2017-10 | 安達健祐 社長 (元経産事務次官) 引責辞任 |
| 2018-06 | 関根正裕 新社長就任 (民間出身)・ガバナンス改革本格化 |
| 2020-04 | 中期経営計画「価値共創フェーズ I」 |
| 2022-04 | 中期経営計画「価値共創フェーズ II」 — 民営化を見据えた事業モデル転換 |
| **2023-10** | **政府保有株式売却完了 → 完全民営化** ^[extracted] |
| 2024-04 | 民営化後の新中期経営計画 — 一般融資比率拡大・事業承継/M&A 強化 |

## 2. 事業セグメント・マップ

| セグメント | 内容 | 位置付け |
|---|---|---|
| **組合員融資** | 中小企業組合 + 組合員企業向け融資 | 創業以来のコア・歴史的アイデンティティ |
| **一般中小企業融資** | 組合員外の中小企業向け融資 | 民営化後拡大方針 |
| **事業承継・M&A サポート** | 中小企業 M&A 仲介・後継者支援 | 民営化後の差別化領域 |
| **預金・決済** | 組合員預金 + 一般法人決済 | 補完業務 |

### 民営化後の事業モデル転換

- 旧政策金融機関時代の **政府委託モデル** → **自立的商業銀行モデル** への移行
- 中小企業マーケットで メガバンク (法人取引基盤) / 地銀 (地域密着) / 信金信組 (零細密着) と三つ巴

### ガバナンス改革 (2017-2018)

- 経営陣総入れ替え (社長交代 + 役員刷新)
- 内部統制・コンプライアンス体制再構築
- 危機対応融資の取扱手続厳格化
- 業務改善計画 → 金融庁定期報告

### 政府との関係

- 2023-10 政府保有株式売却で資本関係終了
- 完全民間化と「準公的役割」の二重構造

### 競合構図

| 競合先 | 重なる領域 |
|---|---|
| メガバンク ([[megabanks/mufg]] / [[megabanks/smfg]] / [[megabanks/mizuho-fg]]) | 中堅企業融資・海外進出支援 |
| 地方銀行 | 地域中小企業融資・事業承継 |
| 信金中央金庫・信用金庫 | 中小企業組合・零細企業融資 |
| [[financial-regulators/jfc|日本政策金融公庫 (JFC)]] | 危機対応融資・政策金融領域 |

## 4. 規制・政策

- **主管**: 金融庁 (FSA) — 一般銀行として銀行法に基づく監督
- **特殊性**: 旧商工組合中央金庫法廃止後も、危機対応融資の指定金融機関として **株式会社商工組合中央金庫法** が継続適用 ^[extracted]
- **業務改善命令の経緯**:
  - 2017-05-09 第 1 次業務改善命令 (760 件)
  - 2017-10-25 第 2 次業務改善命令 (約 4,800 件)
  - 業務改善計画の継続報告義務
- **G-SIB / D-SIB 指定**: なし (規模ではメガ・大手地銀以下)
