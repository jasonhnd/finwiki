# 有限責任 あずさ監査法人

## ウィキ上の位置づけ

この項目は [[financial-regulators/INDEX|financial-regulators INDEX]] に属する。[[financial-regulators/ey-shinnihon|EY 新日本]] と [[financial-regulators/deloitte-touche-tohmatsu|トーマツ]] は Big 4  監査法人としての同業文脈、[[financial-licenses/INDEX|金融ライセンス INDEX]] は公認会計士法 / 監査品質管理の規制境界として併せて読む。CPAAOB 監督文脈は [[financial-regulators/fsa-supervision-bureau|金融庁 監督局]] と [[financial-regulators/fsa-strategy-bureau|金融庁 総合政策局]] の所管。

## 1. 法人概要

- **正式名**：有限責任 あずさ監査法人 ^[extracted]
- **英名**：KPMG AZSA LLC ^[extracted]
- **法人形態**：有限責任監査法人 ^[extracted]
- **本部**：東京都新宿区津久戸町（東京 KPMG タワー）^[extracted]
- **加盟ネットワーク**：**KPMG International Cooperative**（アムステルダム本部）^[extracted]
- **グループ法人（KPMG ジャパン）**：
  - KPMG コンサルティング株式会社
  - KPMG FAS 株式会社（M&A / 事業再生）
  - KPMG 税理士法人
  - KPMG あずさ サステナビリティ株式会社 ^[extracted]

## 2. 沿革

| 年月 | 事象 |
|---|---|
| **2003-12** | **あずさ監査法人 設立**（朝日 + 太田昭和系合流）^[extracted] |
| **2010-07** | **公認会計士法改正に基づき有限責任あずさ監査法人へ移行** ^[extracted] |
| 2020s | KPMG ジャパン ブランド統合・サステナビリティ法人新設 ^[extracted] |

### 金融機関監査（最大強み）^[extracted]

- **メガ系列の一部**、**大手地銀グループ**、**大手生損保**、**大手証券** を多数監査
- **J-SOX（金融商品取引法 内部統制報告制度）** 対応の方法論成熟
- **SOC 1 / SOC 2 レポート** 発行体制

### 暗号資産・デジタルアセット ^[extracted]

- **「KPMG ジャパン デジタルアセット」グループ**
- **内部統制（J-SOX）+ クリプト保管統制の連携監査**
- **SOC2 / ISMS 準備支援**
- [[exchanges/jp-crypto-bank-credit-facilities|銀行系クリプト与信検討]] の助言（金融機関 + 暗号資産双方の業務知見）
- [[exchanges/japan-crypto-audit-firm-landscape]] 言及通り、メガ系 / 地銀系暗号資産関連企業の助言に強み

### 地方銀行 / 地方公共団体

- 地方公共団体の包括外部監査も担当

### サステナビリティ保証

- **KPMG あずさ サステナビリティ株式会社** で SSBJ / ISSB 関連保証 ^[extracted]

## 4. 主要上場 FG クライアント関係（公開ベース）

あずさは大手金融機関監査に強みを持つが、個別会社の最新監査人は EDINET 各社最新有価証券報告書「監査人の状況」セクションで都度確認すべき一次情報であり、当エントリでは具体的クライアント名の列挙を行わない。

## 5. 規制・監督

- **主管法令**：公認会計士法 ^[extracted]
- **登録**：日本公認会計士協会（JICPA）^[extracted]
- **監督**：**金融庁 公認会計士・監査審査会（CPAAOB）** が品質管理レビュー / 検査 ^[extracted]

### 過去の処分歴 ^[extracted]

- **オリンパス 不適切会計事件**（2011 発覚）：オリンパスの監査は 2009-03 期まであずさ、その後 新日本が担当。2012-07-06、金融庁は監査引き継ぎ等の問題を理由に **あずさ・新日本の両法人に業務改善命令** を発出。あずさに対しては巨額買収案件を上級審査の対象としなかった点も指摘された
  - 課徴金（1 億 9181 万円）は **オリンパス本体** に課されたもので、あずさ法人自体への課徴金処分には至らなかった
