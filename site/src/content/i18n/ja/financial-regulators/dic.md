# 預金保険機構 (DIC)

## ウィキ上の位置づけ

この項目は [[financial-regulators/INDEX|financial-regulators INDEX]] の配下に置く。並行するセーフティネット比較対象として [[financial-regulators/sonpo-hosho-kiko|損害保険契約者保護機構]] と [[financial-regulators/seimei-hokensha-hogo-kiko|生命保険契約者保護機構]]、より広いシステム境界・規制境界として [[financial-regulators/fsa|金融庁]] / [[financial-regulators/boj|日本銀行]] とあわせて読む。

## 要約

日本の **預金保険・破綻処理を担う唯一の特殊法人**。1971-07 預金保険法に基づき設立。**保険金支払（pay-off）** + **資金援助（financial assistance）** + **ブリッジバンク** + **特定回収困難債権買取** + **破綻金融機関の経営管理・資産買取** までを一手に担う。出資者は **政府 + 日本銀行 + 民間金融機関の三者出資**（各 1/3 相当）。理事長は財務大臣・金融庁長官・日本銀行総裁による協議任命。2003 [[megabanks/resona-hd|りそな]]、2003 [[regional-banks/ashikaga-hd|足利銀行]] への公的資金注入・一時国有化など、平成期の銀行危機で実質的に金融セーフティネットの最終防衛線として機能。

## 1. 法的根拠・設立

**正式名**: 預金保険機構
**英名**: Deposit Insurance Corporation of Japan (DIC)
**法的根拠**: **預金保険法**（昭和 46 年法律第 34 号）
**設立**: **1971-07-01**
**本部**: 東京都千代田区有楽町 1-12-1
**業態**: 認可法人（特殊法人類似）
**主管**: 金融庁・財務省・日本銀行（三位一体ガバナンス）

### 出資構造

```
政府        ── 1/3 相当 ── 預金保険機構
日本銀行    ── 1/3 相当 ──
民間金融機関 ── 1/3 相当 ── （メガバンク・地銀・信金・信組等）
```

### 重要年表

| 年月 | 事象 |
|---|---|
| **1971-07** | 預金保険法施行・**預金保険機構 設立**（米 FDIC を参考に整備） |
| 1986 | **資金援助方式** 導入（破綻処理に救済合併等のスキーム追加） |
| 1996 | 住専処理法成立 → 住宅金融債権管理機構（住管機構）設立 |
| **1998-10** | 金融再生法 / 早期健全化法成立 → 公的資金注入枠組み確立 |
| **2000-04** | 整理回収機構（[[financial-regulators/rcc|RCC]]）発足（住管機構＋整理回収銀行統合・DIC 子会社化） |
| 2001-04 | **ペイオフ凍結解除** 第 1 段階（定期性預金）|
| **2002-04** | ペイオフ全面解禁 → 翌年延期 |
| **2003-05** | **[[megabanks/resona-hd|りそな]] 公的資金注入** 約 2 兆円（事実上の一時国有化） |
| **2003-11** | **[[regional-banks/ashikaga-hd|足利銀行]] 一時国有化**（特別危機管理銀行第 1 号）|
| 2005-04 | **ペイオフ全面実施**（決済用預金以外は元本 1,000 万円 + 利息まで保護） |
| 2010-09 | [[financial-regulators/japan-shinko-bank|日本振興銀行]] 破綻 → **日本で初のペイオフ発動** |
| 2014 | 預金保険法改正 → 秩序ある処理（金融システム上重要な金融機関等の処理） |
| 2020〜 | コロナ危機対応・金融機能強化法の継続運用 |

## 2. 業務領域

| 業務 | 内容 | 発動例 |
|---|---|---|
| **保険金支払（pay-off）** | 破綻金融機関の預金者に元本 1,000 万円 + 利息まで直接支払 | 2010-09 [[financial-regulators/japan-shinko-bank|日本振興銀行]] |
| **資金援助** | 救済金融機関に金銭贈与・資金貸付・資産買取等を実施 | 1990s 後半多数 |
| **特別危機管理銀行** | 破綻金融機関を一時的に国有化、預保が株主として経営管理 | 2003 足利銀行 |
| **ブリッジバンク** | 受皿候補が見つかるまでの暫定承継銀行を設立 | 第二日本承継銀行 等 |
| **公的資金注入** | 金融機能強化法等に基づき優先株・劣後債で資本注入 | 2003 りそな、平成金融危機諸案件 |
| **資産買取** | RCC を通じた不良債権・特定回収困難債権の買取 | [[financial-regulators/rcc|RCC]] 経由 |
| **金融機能強化** | 中小金融機関の自主的資本増強支援（金融機能強化法）| 2008-09 リーマン後 |
| **金融商品取引業者特例** | 投資者保護基金との連携領域 | 限定的 |

### 2003 りそな救済（最大級の公的資金注入）

- [[megabanks/resona-hd|りそなホールディングス]] の自己資本比率が 4% 国内基準を割り込む寸前
- 預金保険法第 102 条第 1 項第 1 号措置（資本注入）発動
- 約 **1.96 兆円** の公的資金注入 → 実質国有化
- 2015 までに公的資金完済（利益込みでは納税者に対し純益確定）

### 2003 足利銀行 一時国有化

- 栃木県地盤の地方銀行が債務超過認定
- 預金保険法第 102 条第 1 項第 3 号措置（**特別危機管理**）発動
- **預保が全株式取得 → 経営管理 → 2008 野村系受皿に売却**
- 日本で唯一の「銀行国有化 → 完全民営化」完走事例

### 2010 日本振興銀行 ペイオフ初発動

- 木村剛 元日銀審議委員設立の振興銀行が破綻
- **日本で初のペイオフ発動**（戦後初）
- 1,000 万円超預金者は一部カット → 「決済用預金以外は元本 + 利息 1,000 万円まで」原則の現実適用例

## 4. 国際的位置付け

- **米 FDIC（Federal Deposit Insurance Corporation）** を 1970 年代に参考導入したが、米と異なり**保険機能 + 破綻処理機能 + 不良債権処理機能 を一元化**している点が特徴
- **IADI（国際預金保険協会）** メンバー
- 危機時には [[financial-regulators/boj|日本銀行]] 特別融資（最後の貸し手機能）と相補的に機能
