# V Point (SMBC × CCC) — 銀行主導の共通ポイント事例

## 要約

現在の形の V Point（2024-04-22 以降）は、日本における **銀行主導の共通ポイント事例** です。SMBC 既存の V Point と CCCMK Holdings の T Point が V Point ブランドの下で単一ポイントに統合され、[[JapanFG/smfg|SMFG]] の [[JapanFG/smbc-card|SMBC Card]]（SMCC）が発行金融基盤となり、[[payments/INDEX|payments]] / [[retail/INDEX|retail]] のパートナーネットワークは T Point から引き継がれました。通信主導（dポイント）、EC 主導（Rakuten Points）、決済アプリ主導（PayPay points）の共通ポイントシステムに対する、最も信頼性の高い対抗軸です。

この種の統合において「ポイント負債」が実際に何を意味するかという会計上の見方は [[loyalty/point-liability-accounting-boundary]] を参照してください。より広い競争地図は [[loyalty/japan-points-landscape]] と [[loyalty/INDEX|loyalty index]] を参照してください。

## タイムライン

| 日付 | 事象 |
|---|---|
| 2003 | T Point が CCC（Culture Convenience Club）運営の日本初の全国共通ポイントとして開始。中核提携先は Tsutaya、FamilyMart、ENEOS。 |
| 2020 | Vポイントが SMBC Card 独自のクレジットカード還元ポイントとして開始。多くの会員でワールドプレゼントを置き換え。 |
| 2022-10 | CCC と SMFG / SMBC が T Point と V Point の統合方針を発表。 |
| 2023-03 | CCC が T Point 事業を CCCMK Holdings に再編。SMFG グループと共同保有するジョイントベンチャー車両となる。 |
| 2023-06 | CCCMK Holdings 設立（SMFG 側が資本参加）。 |
| 2024-04-22 | T Point と Vポイントが「Vポイント」ブランドの下で統合。共通ポイント事業としての T Point ブランドは終了。 |
| 2024- | SMBC Olive Account が、銀行の旗艦リテール商品の中核ロイヤルティ層として V Point 付与を統合。 |
| 2025- | 旧 Tカード会員の Vカードデザインへの移行が順次進み、提携加盟店ネットワークも V Point 対応へ移管。 |

## プレイヤー

- **[[JapanFG/smfg|SMFG]] / [[JapanFG/smbc-card|SMBC Card]]（Sumitomo Mitsui Card Company, SMCC）** — 銀行側のアンカー。SMBC のクレジットカードと Olive Account の還元層として V Point を発行し、信用供与と決済レールを提供する。
- **CCC / CCCMK Holdings** — 歴史的な共通ポイント運営者。T Point の提携加盟店ネットワーク（FamilyMart、ENEOS、Tsutaya、ドラッグストア、レストラン、ホテル）、約 70  百万人規模の Tカード会員基盤、データマーケティング運営を持ち込む。
- **Tカード → Vカード移行** — 定められた移行期間中、旧 Tカードは参加加盟店で V Point を貯め続ける。新規発行は Vカードブランドになる。
- **Olive Account** — SMBC の「オールインワン」リテール商品（デビット + クレジット + 決済 + 投資）であり、ロイヤルティ層として V Point を使う。2023,  に開始し、2024-2025 にかけて拡大。

## 仕組み

| 獲得面 | V Point の貯まり方 |
|---|---|
| SMBC Card / SMCC クレジットカード利用 | 基本還元率に加え、コンビニエンスストア、ファストフードなどのボーナスカテゴリ。「Vポイントアッププログラム」対象加盟店では高還元。 |
| Olive Account | 銀行口座の利用、デビット利用、口座振替設定、給与受取、その他「選べる特典」の選択がポイント獲得につながる。 |
| 提携加盟店での POS スキャン | FamilyMart、ENEOS、ドラッグストア、レストランなどで、T Point から引き継いだ物理カード / アプリのスキャン導線を使う。 |
| V Point アプリ | コード決済、スキャンして貯める導線、キャンペーン還元。 |
| 投資プロモーション | SMBC + SBI 提携により、SBI証券協力の下で一部投資信託購入に V Point を使える。SMBC 系顧客にとって、Rakuten Points / dポイントに直接相当するものがない機能。 |

## 規模と提携加盟店ネットワークの移管

2024  の統合における戦略的要点は、単なる足し算ではありませんでした。コンビニエンスストア、ガソリンスタンド、ドラッグストア、レストランに深く入り込んでいた T Point の提携加盟店ネットワークを、銀行発行の共通ポイントに結びつけることでした。統合後の V Point は次を引き継ぎます。

- SMBC と CCCMK の歴史的な両側を合わせた、数千万人規模とされる総会員基盤。
- 2024 前の SMBC 単独 V Point よりも広かった実店舗での受入ネットワーク。
- CCC が歴史的に運営してきたマーケティングデータ基盤、すなわち T Point を商業的に価値あるものにしていた顧客 ID グラフ。

きれいな形で見れば、移行カレンダー上で T Point 対応加盟店はすべて V Point 対応加盟店になります。一方で、摩擦を含めて見ると、一部のロングテール加盟店は参加を維持せず、一部チェーンは参加条件を再交渉しました。

## 競争上の位置づけ

| 共通ポイントシステム | アンカー種別 | 最も強いチャネル | V Point の相対的位置 |
|---|---|---|---|
| Rakuten Points（[[JapanFG/rakuten-fg|Rakuten FG]]） | EC 主導 | オンラインショッピング + Rakuten Card + 楽天ペイ + Rakuten Mobile | V Point はオンラインで弱く、実店舗アンカー加盟店で強い。 |
| dポイント（[[JapanFG/ndfg|NDFG]] / NTT docomo） | 通信主導 | モバイル回線 ID、d払い、dカード | V Point は通信会社に縛られず、非 docomo ユーザーに強みがある。 |
| PayPay Points（[[JapanFG/paypay-fg|PayPay FG]]） | 決済アプリ主導 | コード決済頻度 + キャンペーン | V Point はキャンペーン依存が小さく、1円あたりの経済性がより持続的。 |
| Ponta（[[JapanFG/au-fh|au FH]] / Loyalty Marketing） | 通信 + 小売連合 | Lawson、KDDI / au PAY、Hot Pepper | V Point はコンビニでの獲得が重なるが、銀行側アンカーを持つ。 |
| WAON POINT（[[retail/aeon-group|Aeon Group]] / [[JapanFG/aeon-bank|AEON Bank / AFS]]） | 小売グループ主導 | AEON モール / スーパー | 位置づけが異なる。店舗頻度ロイヤルティ vs 横断型共通ポイント。 |
| nanaco point（[[retail/seven-and-i-hd|Seven & i]] / [[JapanFG/seven-bank|Seven Bank]]） | コンビニ主導 | Seven-Eleven / Ito-Yokado | 位置づけが異なる。コンビニロイヤルティ vs 横断型共通ポイント。 |

V Point の戦略的新規性は、銀行主導の形にあります。

- **通信主導**（dポイント、Ponta-au）: ID はモバイル回線アカウントから生まれ、ロイヤルティが決済へ広がる。
- **EC 主導**（Rakuten）: ID はショッピングカートから生まれ、ロイヤルティが金融クロスセルを促す。
- **決済アプリ主導**（PayPay）: ID はウォレットアプリから生まれ、ロイヤルティはキャンペーン原資の獲得手段になる。
- **小売グループ主導**（WAON、nanaco）: ID は店舗から生まれ、ロイヤルティが来店頻度を深める。
- **銀行主導**（V Point / SMBC / Olive）: ID は銀行口座とカードから生まれ、ロイヤルティがクレジットカード経済圏から提携加盟店ネットワークへ広がる。

銀行主導の形は、SMBC + SBI 経由の投資・貯蓄クロスセルや、Olive 経由の「メインバンク口座」ワークフローに最も自然につながります。一方で、Rakuten ほどのオンライン商取引の引力や、PayPay ほどのコード決済キャンペーン力は弱いというトレードオフがあります。

## JapanFG 分析で重要な理由

- V Point は、日本における **メガバンク主導の共通ポイント流通層** の最も明確な現行事例です。これを抜きに [[JapanFG/smfg|SMFG]] を読むと、SMBC のリテール戦略を過小評価します。
- SMBC × SBI 提携の下で、SBI証券側の投資商品に対する自然なクロスセルレールになります。証券販売分析に関係します。
- 加盟店アクワイアリング / PSP 分析では、V Point の提携加盟店ネットワークは Rakuten と dポイントに並ぶ第三の重要な共通ポイント接点です。
- 消費者向け銀行競争分析では、Olive Account + V Point の束ね方は、日本のメガバンクによる最も目立つ「メインバンクをフィンテックアプリ化する」試みであり、[[JapanFG/mufg|MUFG]] と [[JapanFG/mizuho-fg|Mizuho FG]] のアプローチと対照されます。
