# PayPay証券


## ウィキ上の位置づけ

この項目は [[JapanFG/INDEX|JapanFG index]] の配下に位置づける。隣接する文脈は [[securities/INDEX|securities index]] と併せて読み、より広い制度上の境界は [[financial-licenses/INDEX|JapanFG index]] を参照する。

## 要約

PayPay証券は [[megabanks/paypay-fg|PayPay financial ecosystem]] に属するスマホ証券事業者。FSA 金融商品取引業者一覧では「PayPay証券株式会社」、関東財務局長(金商)第2883号として掲載される。PayPay アプリ内の資産運用導線を担う点が、従来 オンライン証券 と違う。

## 2. 事業上の役割

- スマホ証券、少額投資、PayPay アプリ内の資産運用導線を担う。
- 決済・ポイント・証券を接続する 組み込み型証券モデル。
- PayPay銀行、PayPayカード、PayPayアプリとの 導線 が分析上の中心。

## 3. この項目が存在する理由

PayPay証券は「証券会社」だけでなく、決済スーパーアプリ に証券機能を埋め込む実験でもある。手数料・商品ラインアップだけでなく、PayPay 残高 / ポイント / 決済アプリから投資行動への転換を見る必要がある。スマホ証券 / 第一種金融商品取引業ライセンス比較は [[securities/financial-instruments-business-operators-japan-index]] を、資産運用業界全体の比較は [[securities/japan-asset-manager-landscape-matrix]] を参照。
