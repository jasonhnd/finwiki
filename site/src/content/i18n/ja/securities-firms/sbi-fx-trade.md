# SBI FXトレード

## Wiki ルート

この項目は [[securities-firms/INDEX|securities-firms INDEX]]（そのドメインルート）の配下に位置する。最も近い FX 専業のピア／対照として [[securities-firms/traders-securities|トレイダーズ証券]] と、FIEA（金商法）の監督／自主規制境界として [[financial-regulators/fsa|the FSA]] および [[financial-regulators/jsda|JSDA]] と照らし合わせて読むこと。総合証券会社の兄弟会社は [[securities-firms/sbi-securities|SBI証券]]。

## TL;DR

SBI FXトレード株式会社は、[[megabanks/sbi-hd|SBI group]] の FX 専門 FIEA operator。金融商品取引法上の **第一種金融商品取引業者** として金融庁の金融商品取引業者一覧に「SBI FXトレード株式会社」（関東財務局長(金商)第2635号）で掲載される。公開会社概要によれば SBIリクイディティ・マーケット株式会社の 100% 子会社で、[[securities-firms/sbi-securities|SBI証券]] とは別法人として 店頭FX / 積立FX / option FX / crypto CFD adjacency を担う。

## 2. 事業モデル

SBI FXトレードは、FX 専門会社として少額・透明性・SBI グループ連携を打ち出す。[[securities-firms/sbi-securities|SBI証券]] の総合 broker function とは別に、FX execution / retail FX product を担う。crypto / CFD 周辺では [[exchanges/jp-exchange-sbi-vc-trade|SBI VC Trade]] と隣接する。retail 店頭FX 市場では [[securities-firms/dmm-com-securities|DMM.com証券]]・[[securities-firms/gmo-click-securities|GMOクリック証券]]・[[securities-firms/traders-securities|トレイダーズ証券]] と横並び比較される。

## 3. このページが重要な理由

- SBI group は証券・銀行・暗号資産・FX が密接に絡む。SBI FXトレードを独立させることで、FX 専業会社・FIEA 第一種・crypto CFD adjacency・NEOBANK 連携を混同せず追える。
- [[financial-regulators/fsa|FSA]] 監督下の FX 専門 第一種 operator anchor を確定し、親会社（SBIリクイディティ・マーケット）境界を記録する。
