# DMM.com証券

## Wiki 経路

この項目は [[securities-firms/INDEX|securities-firms INDEX]]（そのドメイン経路）の下に位置する。最も近いリテール FX / CFD の peer / 対比としては [[securities-firms/gmo-click-securities|GMOクリック証券]] と、FIEA の監督・自主規制境界としては [[financial-regulators/fsa|the FSA]] および [[financial-regulators/jsda|JSDA]] と併せて読む。暗号資産事業者としての視点は [[exchanges/jp-exchange-dmm-com-securities|the DMM exchange page]] に別途ある。

## 要点

株式会社DMM.com証券は、DMM グループの証券 / 店頭FX / CFD 事業者。金融商品取引法上の **第一種金融商品取引業者** として金融庁の金融商品取引業者一覧に「株式会社DMM.com証券」（関東財務局長(金商)第1629号）で掲載される。同じ法人の暗号資産文脈は [[exchanges/jp-exchange-dmm-com-securities|exchanges 側のページ]] が扱い、本ページは FIEA 事業者として切り出す。

## 2. 事業モデル

DMM.com証券は、DMM 株・DMM FX・CFD などのリテール取引サービスを提供する。FIEA 事業者としての登録と、暗号資産事業者としての登録が交差するため、ネームスペースの分離が重要。リテール FX / CFD 市場では [[securities-firms/gmo-click-securities|GMOクリック証券]]・[[securities-firms/sbi-fx-trade|SBI FXトレード]]・[[securities-firms/traders-securities|トレイダーズ証券]]・[[securities-firms/saxo-bank-securities|サクソバンク証券]] などと比較対象になる。

## 3. このページの意義

- 同じ法人が証券・FX・CFD・暗号資産周辺を横断するため、1 ページにまとめると規制論点が混ざる。securities-firms 側では FIEA / 証券事業者、exchanges 側では暗号資産交換業者として扱う。
- [[financial-regulators/fsa|FSA]] 監督下の第一種事業者アンカーを確定し、店頭FX 競合との比較基点を提供する。
