# 楽天証券

## 要約

楽天証券は [[payment-firms/rakuten-fg|Rakuten's financial ecosystem]] と結びつくオンライン証券部門。金融庁の金融商品取引業者一覧では、楽天証券株式会社が関東財務局長（金商）第 195 号として掲載されている。楽天の 2024 月リリースでは、楽天証券の総合証券口座数が 11 百万口座を突破したとされ、その成長環境を刷新後の 2024 NISA 制度と関連づけている。

このページは、オンライン証券、リテール NISA 獲得、ポイント / カード / エコシステム経由の販売、家計の資産形成フローを見るために使う。

## 戦略上の意味

楽天証券は従来型の店舗証券ではない。強みは販売導線にある。

- 楽天 ID / エコシステムを通じた顧客獲得。
- 楽天カードと現金決済による投資信託積立ルート。
- ポイント投資とロイヤルティ統合。
- デジタル優先の NISA 口座開設導線。
- 低手数料のオンライン証券競争。

リスクも販売導線と結びつく。オンライン証券競争は手数料を圧迫し、ポイント / カード還元を獲得補助として使う場合、エコシステム特典のコストが重くなりうる。
