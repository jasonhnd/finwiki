# アモーヴァ・アセットマネジメント (旧 日興アセット)

## Wiki 上の位置づけ

本項は [[asset-managers/INDEX|asset-managers INDEX]]（そのドメインルート）の下に置かれる。同一グループの信託系列の 運用会社 ピア／対比として [[asset-managers/smt-am|三井住友トラスト・アセットマネジメント (SMT-AM)]] と照らし合わせ、投資運用業 事業者に対する監督・登録の境界として [[financial-regulators/fsa|the FSA]] および [[securities/financial-instruments-business-operators-japan-index|FSA 金商業者 index]] と照らし合わせて読むこと。

## 要点

アモーヴァ・アセットマネジメント株式会社は旧「日興アセットマネジメント」。2025-09-01 に商号変更し、FSA 金融商品取引業者一覧では現名「アモーヴァ・アセットマネジメント株式会社」、関東財務局長(金商)第368号として掲載される。金融商品取引法上の **投資運用業** 登録を持つ運用会社で、[[trust-banks/sumitomo-mitsui-trust|三井住友トラストグループ (SuMi TRUST)]] の **完全子会社** (公開発表上、改称で資本・統治構造は不変)。日本に本社を置く グローバル・アセットマネージャー として公募投信・ETF・機関投資家向け運用を扱う。

## 1. ライセンス／グループ境界

| 項目 | 注記 |

| 旧名 | 日興アセットマネジメント株式会社 (Nikko Asset Management) |

| 監督 | [[financial-regulators/fsa|FSA]] + 投資信託協会 / 日本投資顧問業協会 |

## 2. 事業モデル

投資運用業者として、公募投信の **投資信託委託** (組成・運用)、ETF、**投資一任** (機関投資家向け discretionary) を担う、日本発の グローバル・アセットマネージャー。販売は [[trust-banks/smtb|三井住友信託銀行]] / [[trust-banks/sumitomo-mitsui-trust|SuMi TRUST]] の信託・銀行チャネルおよび証券会社の販売網。旧 Nikko AM ブランドの資料・ファンド名が残るため、wiki では **旧名 (Nikko) と現名 (Amova) を同一ページに束ねる**。監督は FSA + 自主規制機関の二層構造。同じ SuMi TRUST 系の [[asset-managers/smt-am|SMT-AM]] とは役割 (Amova = グローバル運用 / SMT-AM = 年金・公募投信) で分かれる。

## 3. このページが重要な理由

- 旧 Nikko 名の資料・ファンド・販売会社資料と、現行登録名 Amova の **ブランド移行 (brand transition)** を同時に追う。
- 「**メガバンクから独立した信託専業 FG**」傘下のグローバル運用会社という出自を確定する。
- 同グループの [[asset-managers/smt-am|SMT-AM]] および証券系ピア ([[asset-managers/nomura-asset-management|野村アセット]]) との比較起点を提供する。
