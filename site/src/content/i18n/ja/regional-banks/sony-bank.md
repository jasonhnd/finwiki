# ソニー銀行


## ウィキ上の位置づけ

本項目は[[JapanFG/INDEX|JapanFG index]]配下に置く。[[regional-banks/daiwa-next-bank|大和ネクスト銀行 (Daiwa Next Bank)]]とは同業・対比の文脈で、[[banking/INDEX|banking index]]とはより広い制度・規制上の境界を読むために参照する。

## 要約

ソニー銀行は [[megabanks/sony-fg|Sony FG]] 傘下のインターネット銀行運営会社。FSA の銀行免許リスト上は「ソニー銀行株式会社」として掲載され、円預金、外貨預金、投資信託、住宅ローン、Sony Bank WALLET などを扱う個人向け資産運用銀行。

## 1. 法人・ライセンス境界

| 項目 | 読み方 |

| 法人名 | ソニー銀行株式会社 |

| 親会社境界 | [[megabanks/sony-fg|Sony Financial Group]] 100% 銀行子会社 |
| 中核モデル | インターネット起点の個人向け資産形成銀行 |

## 2. 事業上の役割

- インターネットを活用した個人向け資産運用銀行として、外貨・投信・住宅ローン・デビットカードを組み合わせる。
- [[megabanks/sony-fg|Sony FG]] の生命保険・損害保険との 顧客基盤上の隣接性 がある。
- 楽天銀行、PayPay銀行、auじぶん銀行、住信SBIネット銀行とは違い、ソニーグループ / Sony FG の金融ブランド力を背景にする。

## 3. この独立ページが重要な理由

Sony FG は保険色が強い金融グループだが、ソニー銀行は預金・決済・住宅ローン・外貨・投信を扱う別ライセンスの銀行法人。デジタル銀行比較、保険グループの銀行戦略、外貨預金 / デビットカードの分析で独立ページが必要。
