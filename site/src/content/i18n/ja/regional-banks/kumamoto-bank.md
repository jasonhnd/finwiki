# 熊本銀行


## ウィキ上の位置づけ

この項目は [[regional-banks/INDEX|regional-banks INDEX]] 配下に置かれる。[[regional-banks/higo-bank|肥後銀行]] を同業・対照の文脈として、[[banking/INDEX|banking index]] をより広い制度・規制境界として読む。

## 要約

株式会社熊本銀行は、金融庁の銀行免許一覧に掲載される熊本県本店の第二地方銀行。[[regional-banks/fukuoka-fg]] 傘下の熊本側営業銀行であり、[[regional-banks/higo-bank]] とは別法人の比較対象として扱う。

| 項目 | 注記 |

| 正式名称 | 株式会社熊本銀行 |

| 主な地盤 | 熊本県 / 熊本市 |

## 2. 運営モデル

熊本銀行は、熊本県内の個人預金、中小企業金融、個人向け銀行業務、FFG グループの販売網を担う第二地方銀行。[[regional-banks/higo-bank]] は Kyushu FG の中核銀行境界を扱い、本ページは FFG 側の熊本県内ライセンス境界を扱う。九州の地域銀行再編の文脈は [[banking/regional-bank-consolidation-pattern]] を参照。

## 3. このページが重要な理由

- 金融庁掲載名に対応する熊本銀行のページを補完する。
- FFG の熊本側営業銀行を [[regional-banks/higo-bank]] や [[regional-banks/kyushu-fg]] から分けて整理する。
- [[regional-banks/fukuoka-fg]]、[[regional-banks/higo-bank]]、[[regional-banks/kyushu-fg]] の分析に使う比較アンカーを提供する。
