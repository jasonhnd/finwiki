# ゆうちょ銀行（株式会社ゆうちょ銀行 / 7182）事業プロフィール

## ウィキ上の位置づけ

この項目は [[regional-banks/INDEX|regional-banks INDEX]] の配下で、[[regional-banks/yucho|ゆうちょ銀行 (Japan Post Bank) entity anchor]] のエンティティ・アンカーに対応する事業プロフィールとして位置づける。メガバンク級資産規模の比較対象として [[megabanks/mufg|MUFG]]・[[megabanks/smfg|SMFG]]・[[megabanks/mizuho-fg|Mizuho FG]]、上場非メガバンクのユニバーサルバンク比較として [[megabanks/resona-hd|りそなホールディングス]]、メガバンク外で次に大きい預金量システムとして [[banking/cooperative-banking-japan|cooperative banking Japan]]、上場リテール銀行 universe の中でゆうちょ銀行がどこに位置するかの構造的読みとして [[banking/post-megabank-positioning|post-megabank positioning]] を併読する。残存する政府金融の境界は [[policy-finance/japan-policy-finance-system|Japan policy finance system]]、デジタル販売の比較対象は [[banking/japan-net-bank-competition-map|Japan net bank competition map]] と組み合わせる。

### 資産サイド: 預金で調達するが貸出主導ではない

| 側面 | メガバンクの典型 | ゆうちょ銀行 |
|---|---|---|
| 貸出 | 資産の50〜60%程度を、法人・住宅ローン・海外貸出の組み合わせが占める。 | 資産のごく一部にとどまる。直接の個人向け貸出は限定的で、大規模な法人貸出簿はない。 |
| 預貸率 | おおむね70〜90%の範囲 | 構造的に低い。ゆうちょ銀行は貸出銀行ではなく証券ポートフォリオ銀行であり、資産の多くは貸出ではなくJGB・有価証券に置かれる。 |

「貯金」という語は郵便貯金時代の名称上の名残である。銀行法上、これらの負債は銀行預金（「預金」）だが、ゆうちょ銀行の商品ブランドでは歴史的な用語を維持している。^[extracted]

### 大きな貸出簿を持たない理由

理由は三つ重なる。

1. **民間補完原則。** 郵政民営化法の枠組みでは、ゆうちょ銀行が民間銀行を置き換えないことが期待されている。新しい貸出業務には郵政民営化委員会の認可が必要で、個人向け無担保ローンや限定的な住宅ローン対応などは段階的に認められてきたが、同銀行にはメガバンク型の法人貸出能力はない。
2. **法人銀行フランチャイズがない。** ゆうちょ銀行は、大企業向けのメガバンク型リレーションシップバンキング基盤を持ってこなかった。その案件創出エンジンがなければ、貸出簿をゼロから構築する経済合理性は低い。
3. **郵便局窓口を中心にした業務設計。** 約24,000 の郵便局販売網は、大量・小口の預金、送金、基本的な金融商品販売に最適化されており、与信審査やリレーションシップバンキングには向いていない。

### 有価証券ポートフォリオと政策金利エクスポージャー

ゆうちょ銀行の有価証券ポートフォリオは、主要な収益源であると同時に主要なリスク集中でもあった。長期化したマイナス金利・ゼロ金利期には、JGB偏重の構成が利回りを圧縮し、リバランス余地を限定した。2024〜2026 の日銀金利正常化サイクルは、相反する二つの軸で重要な意味を持つ。

- **時価評価上の逆風**: 利回り上昇により、既存の長期JGBポートフォリオに評価損圧力がかかる。
- **NIM tailwind** as new investment runs in at higher reinvestment yields and as policy-rate-linked deposit-side spreads decompress.

全体の純効果は、デュレーション構成、ヘッジ、ポートフォリオ入れ替えの速度に左右される。ゆうちょ銀行IRは主要な有価証券構成と評価損益を開示しており、細かな追跡はウィキページではなくIR開示サイクルで扱う。

### 郵便局窓口の実際の機能

ゆうちょ銀行は、顧客接点となる窓口の多くを自社で保有していない。取引の流れは次の通り。

```
郵便局窓口の顧客
      ↓  (deposit / withdrawal / remittance / investment-product order)
日本郵便 (Japan Post) — 窓口・ATM運営者
      ↓  (契約に基づくゆうちょ銀行の銀行代理業者)
ゆうちょ銀行 (Japan Post Bank) — 口座保有主体・バランスシート保有主体
```

| 接点 | 運営者 | 顧客体験 |
|---|---|---|
| 全国約24,000 の郵便局窓口 | 銀行代理業者としての日本郵便 | 同じ窓口で郵便、荷物、ゆうちょ銀行取引を扱う。 |
| 約30,000超のゆうちょATM | 郵便局、駅、小売施設に設置されたゆうちょブランドATM | ゆうちょブランドの直接接点。[[banking/INDEX|banking index]] の銀行間ネットワークを通じて他行ATM網とも相互接続する。 |
| 直営店 | 235 の直営店（営業所、2025-03-31時点） | 主要都市に集中し、より手厚いサービスを担う。 |
| ゆうちょ通帳アプリ + Web | ゆうちょ銀行のデジタルチャネル | 残高、明細、振込に対応するが、機能の厚みは [[banking/rakuten-bank|Rakuten Bank]]・[[banking/paypay-bank|PayPay Bank]]・[[banking/au-jibun-bank|au Jibun Bank]] より薄い。 |

### 代理店モデルが強みであり制約でもある理由

- 24,000規模の窓口到達範囲は、他の日本の銀行では構造的に**比肩しにくい**。メガバンク支店網が届かない地方・高齢者層を含め、最も深い地理的カバレッジを与える。
- 一方でコスト構造は日本郵便（郵便・荷物）と共有され、ゆうちょ銀行は兄弟会社に内部手数料を支払うため、低金利期には預金スプレッドの経済性が吸収されやすい。
- 顧客層はメガバンクやネット銀行より高齢寄りであり、金利比較による流出が少なく預金基盤を安定させる一方、住宅ローン、投資商品、カードなど高採算商品の利用転換を制約する。

| 観点 | パターン |
|---|---|
| 総資産 | ¥233.6tn（2025-03-31） |
| 預金基盤 | ¥190.5tn（2025-03-31）。個人預金中心。 |
| NIM | JGB偏重の構成と歴史的なゼロ金利・マイナス金利環境により、構造的に圧縮されてきた。 |
| 経費率 | [[megabanks/japan-post|Japan Post]] へのグループ内手数料を通じ、郵便局窓口の代理店コストを負担する。 |
| 資本 | 自己資本比率は銀行法上の最低水準を継続的に上回る。G-SIBには指定されておらず、グローバルなクロスボーダーシステム上重要銀行ではなく、大型国内金融機関である。 |

時点数値は最新のゆうちょ銀行決算短信・有価証券報告書サイクルを参照する。このページは変動する主要数値ではなく、フランチャイズの**形**を追跡する。

### ゆうちょ銀行が競争する領域

- **預金の粘着性**はゆうちょ銀行の構造的な堀である。1875以降の移行期を通じて残った個人貯蓄は、金利面で不利でも移動しにくい。
- **窓口到達範囲**は、メガバンクが追随しにくい形で地方部を押さえている。
- **債券ポートフォリオ利回りの上昇**は、2024〜2026 の正常化サイクルで複数年にわたる一度きりの再投資追い風となる。

### ゆうちょ銀行が競争しない領域

- **法人銀行、IB、信託。** この領域はメガバンクが押さえている。
- **デジタルネイティブのメインバンク機能。** [[banking/rakuten-bank|Rakuten Bank]]、[[banking/paypay-bank|PayPay Bank]]、[[banking/au-jibun-bank|au Jibun Bank]]、[[regional-banks/ssnb|住信SBI / NEOBANK]] はアプリ中心UXとエコシステム連動の顧客獲得で先行している。ゆうちょ銀行のデジタルチャネル（ゆうちょ通帳アプリ、mijika）は明確に後れを取る。
- **大規模な住宅ローン組成。** 民間補完の枠組みの下で、ゆうちょ銀行の対応余地は限られる。

### 経営上の戦略レバー

1. **有価証券ポートフォリオの分散。** JGBから外国債券、オルタナティブ、プライベートファンドへの継続的な入れ替え。中期計画に基づく複数年の取り組み。
2. **郵便局チャネルのコスト規律。** 郵便物量が減少する中、日本郵便との内部手数料設計を調整する。
3. **デジタルチャネル投資。** ゆうちょ通帳アプリとmijika関連施策により、顧客基盤の高齢化を緩和する。
4. **手数料ビジネス隣接領域の限定的拡大。** 投信販売、カード発行提携（JP BANKカード。Visa/Mastercardは三井住友カード保証、JCB版はJCBとの共同発行）、かんぽ生命との保険販売重複。
5. **民営化段階への対応。** 日本郵政がゆうちょ銀行株式を減らすにつれ、郵政民営化委員会の枠組みで追加業務の認可可能性が広がる。

### 重層的な規制

| レイヤー | 法令・規制当局 | ゆうちょ銀行への意味 |
|---|---|---|
| 銀行 | 銀行法、金融庁監督指針 | 資本、流動性、ガバナンス、IT、AMLに関する標準的な普通銀行監督。 |
| 郵政民営化 | 郵政民営化法、郵政民営化委員会 | 新規業務の認可枠組みと親会社保有比率低下の順序づけ。 |
| 郵政制度政策 | 総務省（郵政行政） | 窓口網政策、ユニバーサルサービス義務。直接にはゆうちょ銀行ではなく日本郵便に影響。 |
| 預金保険 | 預金保険機構 | ゆうちょ銀行は加盟金融機関であり、標準制度に基づく預金保護が適用される。 |
| 証券監督 | 金融商品取引法（投信販売・証券関連業務） | 投信販売行為について標準ルールに基づく監督を受ける。 |
| 消費者保護 | 消費者契約法 + 業態別ガイダンス | かんぽ問題後、JPグループ全体で重要性が高まった。 |

### 民間補完原則という業務制約

郵政民営化法は、ゆうちょ銀行の業務拡大が既存の民間金融機関を置き換えるべきではないという「民間補完」の考え方を組み込んでいる。実務上は、無担保個人ローンの拡大、住宅ローン対応、法人向け貸出の試行など、新商品・新業務の認可を郵政民営化委員会が個別に判断することで運用される。この原則こそ、ゆうちょ銀行が預金基盤の規模に見合う貸出簿を構築してこなかった構造的理由である。政府保有比率の低下に伴い、この原則を緩和すべきかどうかは反復的な政策論点になっている。

### かんぽ問題後のガバナンス上の重荷

2019 のかんぽ生命不適切販売問題（日本郵便窓口を通じた保険商品の不適切販売）は、郵便局でのクロスセルモデルへの監視強化を含む、グループ全体のコンプライアンス・ガバナンス改革を引き起こした。ゆうちょ銀行の投信販売実務も並行して厳格化され、新商品のクロスセル拡大ペースはこの影響で抑制されている。
