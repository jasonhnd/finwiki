# SBI新生銀行

## ウィキ上の位置づけ

この項目は [[regional-banks/INDEX|regional-banks INDEX]] の配下に置く。比較・対照の文脈は [[regional-banks/aozora-bank|あおぞら銀行 (Aozora Bank)]] と照合し、より広い制度・規制上の境界は [[banking/INDEX|banking index]] とあわせて読む。

## 要約

旧 **日本長期信用銀行（LTCB）→ 1998-10 経営破綻 → 一時国有化 → 2000 リップルウッド連合へ売却 → 「新生銀行」改称・再生**の流れを経て、**2023-09 SBI HD [[sbi-hd]] の TOB 完了で完全子会社化・上場廃止、2023-12 「SBI 新生銀行」に商号変更**した普通銀行。SBI HD が掲げる「**第 4 のメガバンク構想**」の銀行コア。最大の経営論点は **旧 LTCB 国有化時の公的資金未返済（約 3500 億円規模）**。リテール（新生 + アプラスフィナンシャル）+ 法人融資 + 不動産・ストラクチャードファイナンスの 3 軸。^[extracted]

## 1. 会社概要

**正式名**: 株式会社 SBI 新生銀行 ^[extracted]
**英名**: SBI Shinsei Bank, Limited
**証券コード**: 旧 東証 PRIME 8303（2004-02 再上場 → **2023-09 上場廃止**）^[extracted]
**設立**: 1952-12（日本長期信用銀行として）^[extracted]
**現法人沿革**: 2000-06 「新生銀行」改称 → 2023-12 「SBI 新生銀行」商号変更 ^[extracted]
**親会社**: SBI ホールディングス [[sbi-hd]]（連結子会社・実質 100%）^[extracted]

### 業態の特殊性 ^[extracted]

- 旧 **長期信用銀行**（長期信用銀行法・1952 制定の特殊銀行）から **普通銀行**転換組。現在「**普通銀行**」として銀行法傘下
- 兄弟筋: [[aozora-bank]]（旧 日本債券信用銀行、同じく国有化 → 民間再生組）
- 「3 行長信銀」: 旧 興銀（→ みずほコーポレート / [[mizuho-fg]]）/ 旧 LTCB（→ 新生）/ 旧 日債銀（→ あおぞら）— うち 2 行が破綻・国有化

### 主要子会社（連結）

```
SBI ホールディングス [[sbi-hd]]（上場 8473）
  └── SBI 新生銀行（100%, 旧 8303・2023-09 上場廃止）
        ├── アプラスフィナンシャル（信販, 旧 東証 STD 8589）── 信販・カードローン・割賦
        ├── 新生フィナンシャル（旧 GE キャピタル系, 消費者金融）── レイク ALSA ブランド
        ├── 新生信託銀行 ── 信託機能
        └── 昭和リース ── リース・ファイナンス
```

### 公的資金未返済問題 ^[extracted]

- 旧 LTCB 国有化時の公的資金注入残 **約 3500 億円**規模（預金保険機構保有株起源、SBI 完全子会社化後も継続論点）

## 2. 事業セグメント・マップ

| セグメント | 主要事業者 | 特徴 |
|---|---|---|
| リテール銀行 | SBI 新生銀行本体 | 都市部中産・富裕層、SBI 証券との連携 |
| 信販・カード | アプラスフィナンシャル | 自動車ローン・割賦・カードローン |
| 消費者金融 | 新生フィナンシャル（レイク ALSA） | 旧 GE キャピタル系の無担保ローン |
| 信託 | 新生信託銀行 | 信託機能（規模は中堅信託レンジ） |
| リース | 昭和リース | リース・ファイナンス |
| 法人融資 | SBI 新生銀行本体 | 中堅企業・不動産・ストラクチャードファイナンス |
| グループ連携 | SBI 証券 / SBI 生命 / 住信 SBI ネット銀行 等 | SBI HD [[sbi-hd]] 経由のクロスセル |

### 「第 4 のメガバンク構想」

- SBI HD 北尾吉孝主導の構想: メガ 3 行（[[mufg]] / [[smfg]] / [[mizuho-fg]]）+ [[ndfg]] に続く第 4 極を目指す
- **コア銀行 = SBI 新生銀行**、衛星に地方銀行への出資網（島根銀行・福島銀行・筑邦銀行・清水銀行 等への少数出資）^[extracted]

### SBI 系列入りの戦略意義

- SBI 証券との顧客送客・口座連携
- アプラス・新生フィナンシャルでリテール金融商品ライン拡充

### 競合・参照点

- [[aozora-bank]]: 同じ「旧長期信用銀行 → 国有化 → 民間再生」組、業態が最も近似。MUFG [[mufg]] が一時期戦略保有
- [[mizuho-fg]]: 旧 興銀の系譜を引く唯一残存の長期信用銀行系メガ
- メガ 3 行 vs SBI 新生: 総資産・収益規模で 1 桁差、リテール網も都市部限定

## 4. 規制・政策

- **主管**: 金融庁（FSA）
- **規制**: 銀行法（普通銀行）。長期信用銀行法による特殊銀行枠は 2000 年代に普通銀行転換で離脱 ^[extracted]
- **預金保険機構**: 旧 LTCB 公的資金未返済の対カウンターパーティ
- **G-SIB 指定**: なし（規模未達）
- **直近政策論点**:
  - 公的資金返済スキーム（SBI HD との交渉、上場廃止後の市場ルート不在）
  - 「第 4 のメガバンク構想」の地銀ネット出資に対する金融庁の地銀再編政策との整合性
  - 2024〜 日銀政策金利正常化局面での利ザヤ改善寄与
