# アニコム ホールディングス

## ウィキ経路

本項目は [[non-life-insurers/INDEX|non-life-insurers INDEX]]（そのドメイン経路）の下に位置する。持株会社 / 事業会社の分離として自社の事業子会社 [[non-life-insurers/anicom-insurance|アニコム損害保険]] と、その他のペット保険引受会社 [[non-life-insurers/pet-and-family-insurance|ペット&ファミリー損保]] および [[non-life-insurers/dai-ichi-ipet-insurance|アイペット損保]] とのピア / 対比として、そして保険持株会社に対する監督の境界として [[financial-regulators/fsa|the FSA]] と対照して読むこと。

## 要約

アニコム ホールディングス株式会社は、金融庁の **保険持株会社認可一覧**に掲載される pet-insurance group holding company。グループの中核は損害保険業免許を持つ [[non-life-insurers/anicom-insurance|アニコム損害保険]] で、HD はその親会社・group strategy anchor。HD 自身は保険を引き受けず、傘下の損保事業・動物医療データ・周辺サービスの資本/戦略統括を担う。**HD (listed parent) と operating insurer (免許保有主体) は別 entity** として読む。

## 1. ライセンス / グループ境界

| 項目 | 注記 |
|---|---|
| 法人名 | アニコム ホールディングス株式会社 / Anicom Holdings, Inc. |
| ライセンス経路 | 保険持株会社 (保険業法上の認可); FSA insurance holding-company list。2007 年に保険持株会社認可を取得。 |
| 中核事業会社 | [[non-life-insurers/anicom-insurance|アニコム損害保険]] (損害保険業免許, 2007 年取得) |
| グループの全体像 | アニコム損保 + アニコム パフェ + アニコム先進医療研究所 + 周辺事業 |
| ウィキ上の役割 | Pet-insurance holding-company / group strategy page |

HD は保険業法上の **保険持株会社**であり、損害保険会社そのものではない。実際の引受 (少額短期ではなく正規の損害保険業免許) は子会社のアニコム損保が行う。

## 2. 運営モデル

このグループはペット **損害保険** フランチャイズを運営する: 子会社のアニコム損保がペット保険を引き受け、HD がその上に動物医療データ・先進医療研究・周辺サービスを重ねる。持株会社の分析は、引受データ (請求・診療データ) と動物医療サービスがどのように相互強化し合うかに焦点を当てるべきである — このデータ循環がモノライン損保各社 ([[non-life-insurers/dai-ichi-ipet-insurance|アイペット損保]]、[[non-life-insurers/pet-and-family-insurance|ペット&ファミリー損保]]) との差別化軸になる。

## 3. このページの意義

- アニコムの operating-company / holding-company split を完成させ、免許保有主体 ([[non-life-insurers/anicom-insurance|アニコム損保]]) と資本・戦略の HD を分離する。
- FSA 保険持株会社一覧に載る pet-insurance HD の anchor を提供する。
- specialty insurance（ペット・データネットワーク型損保）の分析ノードになる。
