# 全労済 / こくみん共済 coop (全国労働者共済生活協同組合連合会 / National Federation of Workers and Consumers Insurance Cooperatives)

## ウィキ上の位置づけ

このエントリは以下にあります[[JapanFG/INDEX|JapanFG index]]JA / JF / しんきん / しんくみ軸の外の**労働者/生協をベースとした互助会連合会**として。一緒に読む[[non-life-insurers/ja-kyosairen|JA 共済連]]農協共済ピアの場合、[[non-life-insurers/kyoritsu-fire-marine|共栄火災海上保険]]協力システムの損害隣接の場合、[[insurance/INDEX|insurance domain]]共済と保険の規制境界線について[[cooperative-banks/rokin-renraku|労働金庫連合会]]労働者協同組合金融の銀行業界の同業者向け。

## 要約

**労働者・消費者向けの生活協同組合系「共済」事業を担う全国連合会**。1957 設立、消費生活協同組合法 (生協法) を根拠とする **生協連合会** (農協法ではなく生協法)。**「こくみん共済 coop (旧 こくみん共済)」**ブランドで、生命共済 + 損害共済 (火災・自動車・傷害等) を労働組合・労働者・地域住民を対象に提供。**会員 = 全国の地域勤労者生協 + 職域生協 + 都道府県の労働者生協**。**保有契約・組合員数で1000 万人超の協同組合系保険事業者**として、JA 共済連と並ぶ国内大規模 mutual-aid 組織。**主管は厚生労働省** (生協法所管、生活協同組合は厚労省管轄) で、保険業法対象の保険会社とは規制系統が異なる。^[extracted]

### 販売チャネル

- **都道府県労済** (47 都道府県組織) — 直営支店 + 代理店
- **労働組合経由** — 連合・各産別労組と緊密関係
- **コンビニ収納・自動振替** — 掛金収納の効率化

### 規模感（2024 年度決算 /2025-05-31学期末）

- **会員数**:58 会員 (47 都道府県労済 +8 職域労済 +3 生協連合会)
- **契約件数**:2,893 万件 (契約口数38.7 億口）
- **契約高**: 約791 兆円
- **総資産**:4 兆 27 億円 (4,002,681 百万円)
- **受入共済掛金**:5,299 億円 (529,928 百万円)

### 「こくみん共済 coop」ブランド戦略

- **2019 ブランド統一**: 「全労済」名称は旧称、対外的に **「こくみん共済 coop」** で広告・販売^[extracted]
- 「coop」 = 生協連合体としての性格を前面化
- 若年層・都市部勤労者の取り込み + 知名度向上

### 主要売上

- **JA 共済連** ([[non-life-insurers/ja-kyosairen]]) — 共済規模最大の同型機関
- **都道府県民共済 (全国生協連)** — 別の生協系共済連合会
- **生保ビッグ4** ([[life-insurers/nippon-life|日本生命]] / [[life-insurers/dai-ichi-life|第一生命]] / [[life-insurers/meiji-yasuda|明治安田]] / [[life-insurers/sumitomo-life|住友生命]]) — 生命系商品で競合
- **3 大損保** ([[non-life-insurers/tokio-marine|東京海上]] / [[non-life-insurers/msad|MS&AD]] / [[non-life-insurers/sompo|SOMPO]]) — 自動車・火災で競合
- **ネット保険** ([[life-insurers/lifenet|ライフネット生命]]等) — 直販モデルで価格競争

### 労働運動との結合

- **連合 (日本労働組合総連合会)** と緊密関係 — 共済の販売チャネルとして労組経由を活用
- 公務員労組・教職員労組等の **職域分野** で強い基盤

### デジタル戦略

- Web 申込み・契約管理アプリ

## 4. 監督・規制

- **主管**: **厚生労働省** (生協法所管) + 金融庁 (金融健全性・消費者保護で協力監督)
- **法的根拠**: 消費生活協同組合法 (昭和 23 年法第 200 号) + 生協共済規制
- **直近政策論点**:
  - **共済 = 保険の規制ギャップ** (保険業法 vs 生協法の競争公平性)
  - **生協共済規制** (厚労省・金融庁協力監督) の強化方向
  - **マイナンバー連携** での税控除等のデジタル化
  - **ESR 類似指標** の自主開示
