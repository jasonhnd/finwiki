# ナショナルオーストラリア銀行日本

## ウィキ上の位置づけ

ナショナルオーストラリア銀行日本は [[JapanFG/INDEX|JapanFG index]] の下に位置づけられ、[[JapanFG/foreign-bank-branches-japan-index|the foreign-bank branch registry]] におけるオーストラリア・日本間のホールセール銀行経路である。[[JapanFG/commonwealth-bank-japan|Commonwealth Bank Japan]]、[[JapanFG/anz-bank-japan|ANZ Bank Japan]]、[[JapanFG/legal-financial-licenses/foreign-financial-group-adjacent-licenses|foreign financial group adjacent licenses]]、[[securities/financial-instruments-business-operators-japan-index|FIEA operator registry]] とあわせて読む。

本ページは、NAB の日本支店、日本・オーストラリア間のコーポレートファイナンス、資本市場、マーケッツ商品、コルレス銀行業務、銀行支店面と証券エンティティ面の関係を扱う問いに使う。

## 概要

NAB は、日本拠点について、オーストラリアおよびニュージーランドの企業と日本をつなぎ、日本からオーストラリアおよびニュージーランドへの直接・間接投資フローを支援すると説明している。公開されている日本ページでは、NAB が 1969 に東京駐在員事務所を開設し、1985に支店へ格上げしたとされる。

この支店は、[[JapanFG/anz-bank-japan|ANZ]] および [[JapanFG/commonwealth-bank-japan|CBA]] と並ぶオーストラリア系銀行コリドーの基準点の一つであるため、独立して扱う価値がある。公開資料では、リテール銀行業務ではなく、コーポレートファイナンス、マーケッツ、コルレス / 代理銀行業務、機関投資家向けソリューションが強調されている。

## エンティティ・免許境界

| レイヤー | 読み方 |
|---|---|
| 外国銀行支店 | National Australia Bank の金融庁銀行免許上の経路。 |
| 日本拠点 | 法人・金融機関顧客にサービスを提供する東京支店。 |
| 証券隣接領域 | NAB Japan Securities の登録は別個の資本市場面である。 |
| リテール境界 | NAB は日本を含むアジア拠点では個人向け商品を提供していないと説明している。 |
| コリドー | オーストラリア / ニュージーランドから日本、日本からオーストラリア / ニュージーランドへのアウトバウンド投資。 |

| 機能 | 日本における意味 |

| コルレス / 代理銀行業務 | 法域をまたぐ銀行関係と顧客サービス。 |
| 資本市場 | 債券、デリバティブ、レポ、資本市場商品では証券エンティティ / 登録経路が重要。 |
| 運転資金 / トランザクションバンキング | 法人顧客向けの事業管理・キャッシュフローソリューション。 |

## 公開情報上の文脈

日本企業にとって、オーストラリアは単なる資源・エネルギー供給地ではない。インフラ、実物資産、農業、LNG、再生可能エネルギー、金融投資の対象でもある。そのため NAB の日本経路は、[[finance/cross-border-m-a-japan|cross-border finance]]、プロジェクトファイナンス、機関投資家の資金フローに関係する。

NAB のページを一般的なオーストラリア銀行のまとめに統合してはならない。[[JapanFG/anz-bank-japan|ANZ]]、[[JapanFG/commonwealth-bank-japan|CBA]]、NAB は、商品上の重点、免許、証券隣接面が異なる。
