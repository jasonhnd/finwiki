# SBIレミット


## ウィキ上の位置づけ

この項目は [[JapanFG/INDEX|JapanFG index]] の配下に置かれる。比較対象として [[JapanFG/wise-payments-japan|Wise Payments Japan]]、制度全体や規制上の境界を把握する入口として [[payments/INDEX|payments index]] とあわせて読む。

## 要約

SBIレミットは [[JapanFG/sbi-hd|SBI group]] の国際送金事業者である。公式企業情報は第二種資金移動業者登録番号を示し、外国人材・地域金融機関・銀行代理業との接続で重要性が高い。

## 1. 法人・ライセンス境界

| 項目 | 読み方 |
|---|---|
| 法的主体 | SBIレミット株式会社 |
| グループ | [[JapanFG/sbi-hd|SBI Holdings]] / [[JapanFG/sbi-shinsei-bank|SBI Shinsei adjacency]] |
| 主な領域 | 国際送金, 在留外国人向け送金, 銀行代理 / 地域金融機関連携 |
| FSA登録ルート | FSA資金移動業者一覧: 関東財務局長 第00008号, SBIレミット株式会社。 |

## 2. 事業上の役割

- 日本発の国際送金を提供し、外国人労働者や在留外国人の利用場面を持つ。
- 銀行および地域金融機関との提携を通じて展開している。
- ISO 20022  や AML/CFT 圧力の後に、銀行とフィンテックの送金協力を追跡するうえで重要である。

## 3. この単独ページが重要な理由

SBIの銀行、証券、暗号資産、地域銀行関連ページだけでは送金事業者を説明できない。SBIレミットは、独自の免許を持つ資金移動業者である。
