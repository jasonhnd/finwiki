# メルペイ (Merpay)


## ウィキ上の位置づけ

この項目は次の配下に位置づけられる: [[JapanFG/INDEX|JapanFG index]]. 次と併読する: [[payments/INDEX|payments index]] for adjacent context and [[payments/funds-transfer-vs-prepaid-boundary|Funds transfer vs prepaid boundary in Japan]] for the 制度全体の境界.

## 要約

メルペイは [[JapanFG/mercari-hd|メルカリ]] の決済・与信 operating company。公式 About は資金移動業、前払式支払手段、信用購入あっせん、貸金業などの登録を示しており、marketplace balance / wallet / credit の接続点として独立ページが必要。

## 1. 法人・ライセンス境界

| 項目 | 読み方 |
|---|---|
| Legal entity | 株式会社メルペイ |
| Group | [[JapanFG/mercari-hd|Mercari]] |
| 主なレーン | Merpay ウォレット、Mercari 売上残高決済、資金移動、信用・貸付隣接領域 |
| FSA registry route | FSA funds-transfer list: 関東財務局長 第00057号, 株式会社メルペイ. |

## 2. 事業上の役割

- Mercari のマーケットプレイス残高とユーザー ID を、決済および信用利用へ転換する。
- C2C のマーケットプレイスデータ、コード決済、後払い、銀行口座引落を橋渡しする。イシュア、アクワイアラ、プロセッサの分担は [[payments/japan-card-issuer-acquirer-processor-split]] を参照する。
- PayPay、楽天ペイ、au PAY、d払いとの比較対象であり、[[payments/funds-transfer-service-providers-japan-index]] にも現れる。

## 3. この単独ページが重要な理由

Mercari のマーケットプレイス事業と、Merpay の規制対象となる決済・信用事業では、リスクとコンプライアンス上の論点が異なる。このページでは金融サービス上の境界を明示する。
