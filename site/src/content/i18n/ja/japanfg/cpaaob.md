# 公認会計士・監査審査会 (CPAAOB)

## ウィキ上の位置づけ

この項目は [[JapanFG/INDEX|JapanFG index]] 配下に置く。監督境界は [[JapanFG/fsa-supervision-bureau|FSA 監督局]] と [[JapanFG/fsa-inspection-bureau|FSA 検査局 (historical)]]、ルール形成・政策接点は [[JapanFG/fsa-strategy-bureau|FSA 総合政策局]] / [[JapanFG/fsa-planning-coordination-bureau|FSA 企画市場局]] と併読する。監査法人側は [[JapanFG/ey-shinnihon|EY 新日本]] · [[JapanFG/deloitte-touche-tohmatsu|トーマツ]] · [[JapanFG/kpmg-azsa|KPMG あずさ]] · [[JapanFG/pwc-aarata|PwC あらた]]（Big 4）および [[JapanFG/taiyo-grant-thornton|太陽グラントソントン]] · [[JapanFG/bdo-sanyu|BDO 三優]]（準大手）をアンカーにし、免許範囲は [[JapanFG/legal-financial-licenses/INDEX|金融ライセンス INDEX]] 経由で確認する。

## 1. 法的位置づけ

- **根拠法**：公認会計士法第 35 条 (審査会の設置)、金融庁設置法 ^[extracted]
- **所属**：**金融庁** (内閣府の外局) の附属組織 — 監督局・総合政策局・企画市場局と並列ではなく、独立した審査会として位置づけられる ^[extracted]
- **本部所在地**：東京都千代田区霞が関 (金融庁本庁舎内) ^[extracted]
- **委員構成**：**会長 1 名 + 委員 9 名以内**、両議院の同意を得て内閣総理大臣が任命 — 公認会計士、法律実務家、企業会計実務家、学識経験者から構成 ^[extracted]

### 2.1 公認会計士試験 (CPA Examination)

- **短答式試験** (年 2 回) + **論文式試験** (年 1 回) の実施 ^[extracted]
- 試験委員の選任、試験問題の作成、合格判定 — 日本の会計プロフェッション 入口-point gate を独占的に運営 ^[extracted]

### 2.2 監査法人・公認会計士検査 (Audit Firm Inspection)

- **検査対象**：
  - **大手監査法人** (Big 4：[[JapanFG/ey-shinnihon|EY 新日本]] · [[JapanFG/deloitte-touche-tohmatsu|トーマツ]] · [[JapanFG/kpmg-azsa|あずさ]] · [[JapanFG/pwc-aarata|あらた]]) — 原則 **3 年ごとの定期検査** ^[extracted]
- **検査範囲**：品質管理体制、個別監査業務、独立性、不正対応、IT 統制 ^[extracted]
- **検査結果の公表**：『監査事務所検査結果事例集』『公認会計士監査検査結果事例集』を年次公表 ^[extracted]

### 2.3 監査品質管理レビュー (Quality Control Review)

- **JICPA の品質管理レビュー (ピアレビュー)** の結果について、CPAAOB がさらに**モニタリング**を実施する二段階構造 ^[extracted]

### 2.4 国際連携

- **IFIAR (International Forum of Independent Audit Regulators)** メンバー — グローバル 監査監督 当局のネットワーク ^[extracted]

## 3. 検査結果の処分との連動

CPAAOB の検査・建議 → FSA Commissioner の処分判断 のフローで、主要処分事例には以下がある (各事例詳細は 監査法人 個別ページを参照):

- **中央青山監査法人** (カネボウ粉飾事件を受け 2006-07-01〜08-31 の 2 か月業務停止 → 同年 09-01 みすず監査法人へ改称、2007-07 解散。PwC は受け皿として あらた監査法人を 2006-06 設立) — CPAAOB 発足直後の象徴的事案
- **新日本有限責任監査法人** (現 [[JapanFG/ey-shinnihon|EY 新日本]]) の東芝事件関連処分 (2015-12-22、新規契約締結業務 3 か月停止 [2016-01-01〜03-31] + 業務改善命令 + 課徴金 約 21 億円 — 監査法人への課徴金は初)
- **準大手・中小監査法人**の品質管理体制不備に伴う改善命令多数

## 4. JICPA との役割分担

| 機能 | CPAAOB | JICPA (日本公認会計士協会) |
|---|---|---|
| CPA 試験 | **実施・合格判定** | 試験委員推薦 |
| 登録 | 監督 | **登録事務** |
| 品質管理レビュー | **モニタリング (二次)** | **ピアレビュー (一次)** |
| 倫理・自主規制 | 監督 | **規律 (自主規制)** |
| 検査 | **権限を持つ** | 規律調査 |
| 国際連携 | **IFIAR** | **IFAC** |

## 5. ガバナンス上の位置

- **FSA 監督局** ([[JapanFG/fsa-supervision-bureau|監督局]]) の銀行・証券・保険検査とは**対象が異なる** — CPAAOB は監査品質、監督局は金融機関本体 ^[extracted]
- **FSA 検査局** ([[JapanFG/fsa-inspection-bureau|検査局 (historical)]]) は 2018 年に監督局に統合されたが、CPAAOB は**統合対象外** — 独立した審査会としての地位を維持 ^[extracted]
- **企業会計審議会** (内閣府の別審議会) との関係 — 監査基準を企業会計審議会が設定し、CPAAOB が監査品質を検査するという棲み分け ^[extracted]
