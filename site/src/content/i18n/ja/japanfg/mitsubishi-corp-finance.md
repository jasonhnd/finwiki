この項目は [[JapanFG/INDEX|JapanFG index]] 配下に置かれる。同じ親会社内の資産運用隣接領域は [[JapanFG/mitsubishi-corp-asset-management|三菱商事アセットマネジメント]]、最も近い商社系グループ財務部門の比較対象は [[JapanFG/mitsui-bussan-financial-services|三井物産フィナンシャルサービス]] と合わせて読む。このウィキで既に扱うリース系の比較対象は、[[JapanFG/mitsubishi-hc-capital|三菱HCキャピタル]]（三菱側の上場リース基盤）、[[JapanFG/smfl|SMFL]]（住友商事 + SMFG）、[[JapanFG/tokyo-century|東京センチュリー]]（伊藤忠系）、[[JapanFG/ja-mitsui-leasing|JA三井リース]]（三井物産 + JA）。より広い制度・規制境界は [[banking/INDEX|banking index]] を参照。

### 三菱グループ金融機能の三層構造

| 層 | 主体 | 役割 |
|---|---|---|
| **外向け銀行** | [[JapanFG/mufg|MUFG]] グループ ([[JapanFG/mufg-bank|MUFG Bank]] / [[JapanFG/mitsubishi-ufj-trust-bank|MUTB]]) | 商業銀行・信託・対外金融サービス |

## 2. 事業セグメント・マップ

| セグメント | 内容 |
|---|---|
| **CMS (キャッシュマネジメントサービス)** | 三菱商事グループ各社の決済集中・資金プール・グループ内振替 |
| **一括ファクタリング** | 三菱商事グループ各社の売掛債権買取・支払い代行 |
| **グループファイナンス** | グループ内会社への short-term / mid-term ローン提供 |
| **為替・金利リスクヘッジ仲介** | グループ各社向けのデリバティブ取引仲介 (本社財務部門 の代行) |
| **海外送金サービス** | グループ各社の海外向け送金・回収 |
| **信用補完 / 保証** | グループ各社の取引先信用補完 |

### 商社系 group-treasury の典型像

- **資金効率化**：グループ内の余剰資金を不足会社に充当 (intra-group netting)
- **取引コスト削減**：銀行手数料 / 為替手数料 / 送金コストの集約
- **リスク管理集約**：為替・金利リスクを本社財務部門 で集中管理
- **信用集中**：単一の信用ライン (三菱商事本体) で銀行交渉力を確保

この機能を独立子会社化することで、グループ内金融サービスの専門性 + ガバナンス + 会計分離 + (一部) 規制対応 を確保している。

## 4. 規制・政策

- **主管**：金融庁 (一部の貸金業 / 為替業務関連登録範囲)
- **規制論点**：
  - グループ内金融取引の **移転価格税制** (海外子会社との取引価格)
  - 連結会計上の **内部取引消去** 範囲
- **直近政策論点**:
  - 2024〜 [[JapanFG/boj-monetary-policy|日銀政策金利正常化]] → グループ内金利付与 / 移転価格再評価
  - 2025〜 海外送金 / クロスボーダー資金移動の AML/CFT 強化
