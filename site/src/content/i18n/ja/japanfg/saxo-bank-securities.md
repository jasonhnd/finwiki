# サクソバンク証券


## ウィキ上の位置づけ

本項目は[[JapanFG/INDEX|JapanFG index]]配下に置く。[[JapanFG/gmo-click-securities|GMOクリック証券]]とは同業・対比の文脈で、[[securities/INDEX|securities index]]とはより広い制度・規制上の境界を読むために参照する。

## 要約

サクソバンク証券はデンマークの Saxo Bank A/S を親会社に持つ 日本のFIEA事業者。FSA 金融商品取引業者一覧では「サクソバンク証券株式会社」、関東財務局長(金商)第239号として掲載される。外国株式、FX、CFD、デリバティブ系商品の 日本からのアクセス窓口。

## 1. 法人・ライセンス境界

| 項目 | 読み方 |
|---|---|
| 法人名 | サクソバンク証券株式会社 |
| FSA登録 | 関東財務局長(金商)第239号 |
| FIEA上の範囲 | 第一種、第二種 |
| 親会社境界 | Saxo Bank A/S 100% subsidiary |
| 中核商品 | 外国株式, FX, CFDs, 上場 / 店頭デリバティブ |

## 2. 事業上の役割

- グローバル オンライントレーディングプラットフォーム の日本法人として、多市場・多商品 アクセス を提供する。
- 国内 オンライン証券 と比べ、外国株式・オプション・CFD / FX の横断性が強い。主要 リテールFX 国内競合は [[JapanFG/gmo-click-securities|GMOクリック証券]] と [[JapanFG/dmm-com-securities|DMM.com 証券]]、横並びの [[JapanFG/sbi-fx-trade|SBI FXトレード]] が定番ベンチマーク。
- 外資系 FIEA 運営会社 として、外国銀行支店 とは別に扱う。

## 3. この独立ページが重要な理由

サクソバンク証券は通常の国内ネット証券というより、グローバルトレーディングプラットフォーム の 日本規制下ゲートウェイ。商品範囲、取引インフラ、外資系 broker regulation を見るために独立ページ化する。
