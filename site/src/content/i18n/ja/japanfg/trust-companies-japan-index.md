# 日本の信託会社レジストリ索引

## 要約

FSA の信託会社一覧には、**38 社の信託会社が 2026-03-31 時点で** 含まれる。このページは、独立系 / グループ系信託会社の面に対するレジストリルートである。[[JapanFG/mitsubishi-ufj-trust-bank]]、[[JapanFG/mizuho-trust-bank]]、[[JapanFG/smtb]]、[[JapanFG/smbc-trust-bank]]、[[JapanFG/nomura-trust-bank]]、[[JapanFG/nochu-trust-bank]] などの免許を受けた信託銀行とは別に扱う。

信託会社が戦略的に重要でない限り、38 standalone company pages を作成しないためにこのページを用いる。

## 境界

| 境界 | 取扱い |
|---|---|
| 信託銀行 | [[JapanFG/INDEX]] 配下の独立した銀行 operating-company ページとして維持する。 |
| 信託会社 | FSA 一覧全体をここに維持する。重要なグループ、上場会社、フィンテック、相続、不動産、トークン化の関連性がある場合のみ独立ページへ昇格する。 |
| グループ連動の信託会社 | グループページがすでに存在する場合はグループページへルートする。ただし、グループページがすべての信託会社詳細を網羅しているとは含意しない。 |
| ロングテール信託会社 | 昇格させる公開ソース上の理由が出るまではレジストリのみとする。 |

## 独立昇格ルール

次のいずれかに該当する場合のみ、信託会社をレジストリのみから独立ページへ昇格する。

- 上場会社グループの子会社で、重要な戦略的関連性を持つ。
- トークン化証券、相続設計、倒産隔離、エスクロー、不動産金融インフラに結び付いている。
- JapanFG / 証券 / 不動産金融分析に繰り返し登場する。
- 公開提出書類、公式開示、規制当局資料が、出典に裏づけられた別ページを作るのに十分な内容を提供している。
