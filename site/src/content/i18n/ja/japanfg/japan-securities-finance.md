# 日本証券金融 (JSF)

## ウィキ上の位置づけ

この項目は [[JapanFG/INDEX|JapanFG index]] の配下に位置づける。[[JapanFG/japan-exchange-group|日本取引所グループ (JPX)]] は比較対象の文脈、[[securities/INDEX|securities index]] は制度全体と規制上の境界を確認するために併読する。

## 要約

日本証券金融は、日本の上場証券金融会社であり、信用取引、株券貸借、証券担保金融を支える重要なホールセール層である。公式会社資料では、金融商品取引法上の証券金融業として、貸借取引、証券金融、証券投資業務を事業内容に掲げている。

FinWiki では、JSF は [[securities/INDEX|securities]] 側で、個人の信用取引、機関投資家の株券貸借、証券会社の資金調達、PTS・取引所決済の実務インフラを結びつける。[[JapanFG/matsui-sec|松井証券]] のような比較対象や、[[securities/japan-securities-clearing-corp|JSCC]] の中央清算インフラとも隣接する。

## 2. 事業セグメント・マップ

| レイヤー | 機能 |
|---|---|
| 貸借取引 | 信用取引のための標準化された資金・株券貸借レイヤー。 |
| 証券金融 | 上場証券に紐づく貸付、借入、担保金融。 |
| 証券投資 | 投資および資金運用業務。 |
| 信託銀行との隣接 | JSF グループには日証金信託銀行を通じた信託銀行機能が含まれる。 |

## 3. 戦略・競争

JSF は目立ちにくいインフラ企業である。重要性が高まる局面は次のとおり。

- 個人の信用取引が拡大する局面。
- 株券貸借と空売り需要が増える局面。
- PTS 市場が貸借と清算連動の決済支援を必要とする局面。
- 担保と証券金融の条件がストレス指標になる局面。

## 4. 規制・政策

JSF は FIEA エコシステムの証券金融レイヤーで事業を行う。取引所ではないが、取引所市場の流動性、信用取引規制、証券会社のリスク管理、証券貸借と密接につながっている。
