# コメルツ銀行 東京支店


## ウィキ上の位置づけ

この項目は [[JapanFG/INDEX|JapanFG index]] の下に位置する。ピア / 対比の文脈は [[JapanFG/credit-agricole-bank-japan|クレディ・アグリコル銀行 東京支店]]、より広いシステム / 規制境界は [[banking/INDEX|banking index]] と照らして読む。

## 要約

コメルツ銀行 東京支店は、金融庁の外国銀行支店一覧に掲載される Germany / Commerzbank AG の在日銀行支店である。日本では corporate / institutional clients、foreign trade、FX、transaction banking、export-oriented Japanese corporate support を中心にした wholesale branch として扱う。

## 2. 事業モデル

Commerzbank は、1961 に東京駐在員事務所を開設し、1977に支店へ転換したと説明している。日本ページは、東京支店を、日本の法人・機関投資家顧客および世界各地の輸出志向の日本企業を支える、現地設立済みでグローバルに接続されたプラットフォームとして位置づけている。

現在の Commerzbank worldwide PDF は、Tokyo Branch を Toranomon Hills Station Tower に掲載し、法人および金融機関デスクの双方を示している。これは外国銀行支店の典型に合う。リテール上の関連性は限定的だが、B2B、貿易、FX、機関投資家接続では意味を持つ。
