# 三菱UFJ信託銀行 (MUTB)


## ウィキ上の位置づけ

この項目 sits under [[JapanFG/INDEX|JapanFG index]]. あわせて読む と照合して [[JapanFG/smbc-trust-bank|SMBC信託銀行 (SMBC Trust Bank)]] のために 比較・対照 文脈 と [[banking/INDEX|banking index]] のために the より広い制度・規制境界.

## 要約

三菱UFJ信託銀行は [[JapanFG/mufg|MUFG]] の trust-bank 運営 会社。商業銀行の [[JapanFG/mufg-bank|三菱UFJ銀行]] とは別の銀行法人で、信託財産、年金、不動産、証券代行、受託財産領域を担う。

## 2. 事業上の役割

- 信託財産、年金、資産管理、不動産、証券代行を扱う MUFG の trust platform。
- [[JapanFG/master-trust-bank|日本マスタートラスト信託銀行]] など asset-administration institutions との関係を見る時の基点になる。
- [[JapanFG/mufg-bank|三菱UFJ銀行]] が預貸・外為・商業銀行網、MUTB が信託・受託・不動産・年金を担う。

## 3. 独立ページとして重要な理由

stablecoin / security token / trust beneficiary right / 資産管理 の分析では、商業銀行と信託銀行のライセンス差が重要。MUTB を独立ページにしておくと、[[fintech/jp-stablecoin-progmat|Progmat]] や EPI / tokenized deposit 論点からリンクしやすい。
