# 三井住友トラストグループ (SuMi TRUST)

## ウィキ上の位置づけ

この項目は [[JapanFG/INDEX|JapanFG index]] の配下に位置する。比較・対照の文脈では [[JapanFG/master-trust-bank|日本マスタートラスト信託銀行]]、より広い制度上・規制上の境界については [[banking/INDEX|banking index]] とあわせて読む。

## 要約

メガバンクから独立した唯一の信託専業持株会社、**3 大信託の一角**（三菱 UFJ 信託 / 三井住友信託 / みずほ信託）。2002-02 中央三井トラスト HD 設立 → 2011-04 住友信託銀行と統合 → 三井住友トラスト・ホールディングス発足 → **2024-10-01 三井住友トラストグループに商号変更**。年金・不動産・証券代行・遺言信託に特化。FY2024 純利益 **2,576 億円**（前期比 3.3 倍、過去最高）、総資産 **78.2 兆円**、**信託財産残高 263.3 兆円（業界首位）**。**注意：[[smfg]] (三井住友 FG) とは別法人** — 「三井住友」名称が共通だが、SuMi TRUST は旧三井家 + 旧住友家系の **信託** ルーツ、SMFG は旧太陽神戸 + 旧住友銀行系の **銀行** ルーツで資本関係なし。SBI HD と [[ssnb]] (SBI 住信ネット銀行) を JV 運営。

### 合併歴史・前身

- **旧 三井信託銀行**：1924 設立（旧三井財閥系信託、日本初の信託会社系列のひとつ）
- **旧 住友信託銀行**：1925 設立（旧住友財閥系信託）
- **旧 中央信託銀行**：戦後設立、後に三井信託と合併
- **中央三井信託銀行**：2000 三井信託銀行 + 中央信託銀行 合併
- **中央三井トラスト HD**：2002-02 設立（中央三井信託 + 中央三井アセット信託の持株会社）
- **三井住友トラスト HD**：2011-04 中央三井トラスト HD + 住友信託銀行 統合

> **統合背景**：住友信託銀行は単独路線を志向していたが、リーマンショック後の業界再編圧力と信託専業独立維持の戦略一致から、2009 年に中央三井トラスト HD との経営統合方針を発表。2011-04 株式交換方式で統合、信託専業 FG として最大規模を確立。^[likely]

### 信託専業独立路線

- **メガから独立**：3 大信託のうち、三菱 UFJ 信託は [[mufg]] 傘下、みずほ信託は [[mizuho-fg]] 傘下。**SuMi TRUST のみメガから独立した持株会社で、信託専業**。これがアイデンティティかつ差別化軸。
- **「三井住友」名称の混同リスク**：[[smfg]]（三井住友 FG = SMBC 系）と名称が共通だが、別法人・別資本。SMFG は旧太陽神戸銀行 + 旧住友銀行系の銀行系列、SuMi TRUST は旧三井信託 + 旧住友信託の信託系列。

### 富裕層・法人運用シフト

- **個人富裕層**：遺言信託・資産承継・相続対策で旧三井家・旧住友家以来の顧客基盤を活用
- **法人**：年金資産受託（DB / DC）・不動産信託・証券代行で大企業顧客基盤
- **機関投資家**：日興アセット + 三井住友トラスト AM で投信運用を 2 ブランド体制

### 提携・買収戦略

- **SBI HD との JV** ([[ssnb]] / SBI 住信ネット銀行)：50% 出資、SBI HD と JV、BaaS / ネット銀行領域参入の主要チャネル
- **パナソニックとの JV**（三井住友トラスト・パナソニックファイナンス）：リース・ファイナンス
- **日興アセットマネジメント取得**（2009 シティから段階的）：投信運用領域強化

### 中期経営計画「Trust 2030」（2024）

- 信託機能を軸に、年金・不動産・資産承継・サステナビリティ関連受託の拡大
- BaaS 領域は SBI 住信ネット銀行経由で間接展開 ^[likely]

## 4. 規制・政策

- **主管**：金融庁（FSA）
- **持株会社規制**：銀行法 第 52 条の 17（信託銀行の持株会社にも準用）
- **信託業法**：信託銀行業務全般
- **G-SIB**：MUFG と異なり G-SIB 指定なし（規模・グローバル連関性が相対的に小さい）^[likely]
- **直近政策論点**：
  - 2024〜 日銀政策金利正常化 → 信託報酬・運用利回り環境の変化
  - 2025〜 BaaS / ネット銀行領域での 住信 SBI ネット銀行（[[ssnb]]）の動向
  - 2025〜2026 資産承継・相続関連サービスの規制環境変化
