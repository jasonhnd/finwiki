# SBI FXトレード


## ウィキ上の位置づけ

この項目は [[JapanFG/INDEX|JapanFG index]] の配下に置く。比較・対照の文脈では [[JapanFG/traders-securities|トレイダーズ証券]]、より広い制度・規制境界では [[securities/INDEX|securities index]] とあわせて読む。

## 要約

SBI FXトレードは [[JapanFG/sbi-hd|SBI group]] の FX 専門の金商法上の事業者。金融庁の金融商品取引業者一覧では「SBI FXトレード株式会社」、関東財務局長(金商)第2635号として掲載される。SBI証券とは別法人で、FX / 積立FX / オプションFX / 暗号資産CFD隣接領域を担う。

## 1. 法人・ライセンス境界

| 項目 | 読み方 |
|---|---|
| 正式名 | SBI FXトレード株式会社 |
| 金融庁登録 | 関東財務局長(金商)第2635号 |
| 金商法上の範囲 | 第一種金融商品取引業 |
| 親会社境界 | SBIリクイディティ・マーケットの 100% 子会社 |
| 中核商品 | FX, 積立FX, オプションFX、暗号資産CFD隣接サービス |

## 2. 事業上の役割

- FX 専門会社として、少額・透明性・SBI グループ連携を打ち出す。
- [[JapanFG/sbi-securities|SBI証券]] の総合 証券ブローカー機能 とは別に、FX執行・個人向けFX商品 を担う。
- [[exchanges/jp-exchange-sbi-vc-trade|SBI VC Trade]] と 暗号資産 / CFD / EPI 周辺で隣接する。

## 3. 独立ページが重要な理由

SBIグループ は証券・銀行・暗号資産・FX が密接に絡む。SBI FXトレードを独立させることで、FX 専業会社、FIEA 第一種、暗号資産CFD隣接領域、NEOBANK 連携を混同せず追える。
