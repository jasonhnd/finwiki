# 興銀リース（旧称・現みずほリース）

## ウィキ上の位置づけ

この項目は [[JapanFG/INDEX|JapanFG index]] の下に置く。[[JapanFG/mizuho-lease|Mizuho Leasing / みずほリース]]（2019  統合後の承継エンティティ）と、より広い制度・規制上の境界を扱う [[banking/INDEX|banking index]] を併読する。みずほ側の歴史的エンティティとして、親グループ文脈では [[JapanFG/mizuho-fg|Mizuho FG]] と合わせて読む。

## 要約

**興銀リース (IBJ Leasing)** は 旧 日本興業銀行 (IBJ) 系列のリース会社として 1969 年に設立され、長年にわたり みずほ系 / 旧 IBJ 系列の代表的なリース会社として活動してきた。**2019-04-01 に「みずほリース」と社名変更 → 実質的に [[JapanFG/mizuho-lease|Mizuho Leasing / みずほリース]] となり、独立企業体としての「興銀リース」は終了した。** よってこのページは **歴史エンティティ + 命名統合ノート** の位置付け。現行業務 / 財務状況は [[JapanFG/mizuho-lease|みずほリース]] を参照。

## 2. 統合史と旧事業ライン

旧 興銀リースは みずほ系列内で **不動産・航空機・大型 PJ ファイナンス** に強みを持つ準大手リース会社として、芙蓉グループ系の [[JapanFG/fuyo-lease|芙蓉総合リース]] と業容が一部重複しながら共存していた。旧 IBJ の **大企業 / プロジェクトファイナンス** 系の DNA を継承し、不動産・船舶・航空機・海外 PJ といった大型案件で強みを発揮。

主要旧業務領域：

- 一般機器リース・産業機器リース
- 不動産リース・ファイナンス (大型物件)
- 航空機・船舶のオペレーティングリース・JOL
- 海外 (アジア中心) 拠点
- 環境・再エネ PJ ファイナンス

### 2019-04 みずほリース統合の背景

- みずほ FG 内のリース機能は 興銀リース + 芙蓉総合リース等 で重複構造があり、グループ内再編圧力が存在
- 興銀リース → みずほリース 社名変更 (2019-04-01) は、みずほ FG の ノンバンク 機能整理の一環
- 2024 年には [[JapanFG/mizuho-lease|みずほリース]] 自体に対する 丸紅 の戦略株主参画が発表され、みずほ系 + 商社系 の ハイブリッド・プラットフォーム へと進化

## 3. この移行の戦略的意義

旧 興銀リースの統合は、みずほ FG が「複数のリース子会社を保持するメリットがない」と判断した結果。一方で、芙蓉グループ系の [[JapanFG/fuyo-lease|芙蓉総合リース]] は独立した上場会社として並列存続しており、みずほ系内のリース統合は完全には行われていない。

この経緯は [[JapanFG/mizuho-fg|Mizuho FG]] の ノンバンク 戦略 (旧 IBJ・旧 富士・旧 第一勧業 の 3 つの遺伝子の融合プロセス) を理解する重要な事例。

[[finance/japan-acquisition-finance|日本アクイジションファイナンス]] の文脈では、旧 興銀リースの大型 PJ ノウハウは みずほリースに継承され、現在もみずほ系の LBO / M&A ファイナンスで活用されている。

## 4. このページが重要な理由

- **命名 / 検索の 曖昧性解消**：「興銀リース」「IBJ Leasing」での検索で現行エンティティ ([[JapanFG/mizuho-lease|みずほリース]]) にナビゲートできる
- 旧 IBJ レガシーの理解 (みずほ FG 内の 3 遺伝子のうちの一つ) の補助ノード
- リース業界の M&A / 再編史の事例
- 同じみずほ系内でも 芙蓉総合リース ([[JapanFG/fuyo-lease|芙蓉総合リース]]) との統合がなされず並列存続している事実の解釈
