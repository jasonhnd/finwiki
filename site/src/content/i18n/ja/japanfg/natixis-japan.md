# ナティクシス 東京支店


## ウィキ上の位置づけ

## 要約

ナティクシスは、金融庁の外国銀行支店一覧に掲載されるフランス / Groupe BPCE 系 Natixis CIB の在日銀行支店。2025 年に銀行支店化を公表した比較的新しい 0  世代の外国銀行支店アンカーであり、貸付専業の拠点から、より広い銀行免許業務へ広げる局面として扱う。

## 2. 運営モデル

Natixis CIB は、2025-07-01から日本の銀行支店を設立すると公表し、2022以降に貸金業ライセンスの下で運営してきた日本拠点を拡張した。公式リリースは、資本市場、M&A、投資銀行、資産担保ファイナンスに加え、インフラ、エネルギー・再生可能エネルギー、保険、輸送、ヘルスケア、通信、テクノロジー分野の能力を支店の軸としている。

日本での実務上の役割はリテールではなくホールセール / 機関投資家向けである。日本で活動するグローバル顧客、日本企業の国内・海外活動、保険会社、銀行、金融スポンサー、公的部門組織、クロスボーダー CIB カバレッジが対象となる。
