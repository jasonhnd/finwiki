# Progmat株式会社

## ウィキ上の位置づけ

この項目は [[JapanFG/INDEX|JapanFG index]] の下に位置づけます。隣接する文脈では [[fintech/japan-stablecoin-regulatory-landscape|日本 Stablecoin 法制度の三層構造（JPYC・USDC・Project Pax）]]、より広い制度上の境界では [[payments/INDEX|payments index]] とあわせて読んでください。

## 要約

株式会社Progmatは、デジタルアセットの発行・管理プラットフォームを運営する会社。ステーブルコイン領域では、信託型の Progmat Coin / Project Pax 系基盤として重要だが、同社自身が銀行・信託会社・電子決済手段等取引業者になるわけではない。発行体の役割は [[JapanFG/mitsubishi-ufj-trust-bank|三菱UFJ信託銀行]] など、規制対象となる信託銀行側に残る。

## 1. 法人・ライセンス境界

| 項目 | 読み方 |
|---|---|
| 法人 | 株式会社Progmat |
| 起源 | 三菱UFJ信託銀行の社内プロジェクトを起源とし、2023-10-02 から独立会社化 |
| 主な領域 | ST / UT / SC プラットフォーム、Progmat Coin、DCC コンソーシアム運営 |
| 規制上の境界 | プラットフォーム提供者。トークン発行者・受託者・販売者の役割は、参加する規制対象事業者が担う。 |

## 2. 事業上の役割

- セキュリティトークン、ユーティリティトークン、ステーブルコイン基盤向けのトークン化プラットフォームを提供する。
- [[JapanFG/mufg|MUFG]] の信託銀行ノウハウ、他のメガバンク参加者、NTT Data、[[JapanFG/japan-exchange-group|JPX]]、Datachain 型の相互運用性の取り組みを接続する。
- [[exchanges/jp-exchange-jpyc|JPYC]] の資金移動業者発行モデルに対する、機関投資家向けステーブルコイン / トークン化資産プラットフォームとして機能し、[[exchanges/INDEX|exchanges index]] で追跡するノンバンク交換面とは異なる。

## 3. この単独ページが重要な理由

既存の詳細ページは `fintech/` にあるが、JapanFG では会社アンカーが必要になる。Progmat は大手銀行株主を持ち、ステーブルコインとの関連性が高い、日本の金融インフラ事業者として名指しされる会社だからである。
