# UI銀行 (UI Bank)


## ウィキ上の位置づけ

この項目は [[JapanFG/INDEX|JapanFG index]] 配下に置く。競合・比較対象の文脈は [[JapanFG/saitama-resona-bank|埼玉りそな銀行 (Saitama Resona Bank)]]、より広い制度・規制境界は [[banking/INDEX|banking index]] と照合して読む。

## 要約

UI銀行は [[JapanFG/kiraboshi-fg|東京きらぼし FG]] 100% 出資のデジタル銀行事業会社。2022 年に開業し、対面・非対面のハイブリッド、生活パートナー型金融、API 連携を打ち出す地域金融グループ発のスマホ銀行。

## 2. 事業上の役割

- きらぼしグループの DX の中心的役割として開業したスマホ銀行。
- 「対面・非対面のハイブリッド」「デジタルの生活パートナー」「外部 API 連携」をコンセプトにする。
- 地銀グループが既存銀行とは別にデジタル銀行を持つモデルとして、[[banking/minna-bank-baas-model|みんなの銀行]] や [[JapanFG/01-bank|01銀行]] と比較できる。

## 3. 独立ページとする理由

UI銀行は東京きらぼし FG の単なるデジタルチャネルではなく、独立した銀行免許を持つ事業会社。地銀 DX、スマホ銀行、地域金融グループの全国展開モデルを分析するために分けておく。
