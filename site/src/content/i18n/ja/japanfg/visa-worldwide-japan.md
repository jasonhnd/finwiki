# ビザ・ワールドワイド・ジャパン (Visa Worldwide Japan)

## ウィキ上の位置づけ

この項目は [[JapanFG/INDEX|JapanFG index]] の下に位置づける。比較・対照の文脈では [[JapanFG/jcb|JCB]] とあわせて読む。Visa はグローバルな純粋スキームであり、JCB は日本発のハイブリッド型である。より広い制度・規制上の境界については [[payments/japan-card-issuer-acquirer-processor-split|Japan card issuer / acquirer / processor split]] を参照する。

## 要約

ビザ・ワールドワイド・ジャパンは、米 Visa Inc. (NYSE:V) の **日本拠点法人**であり、日本国内における Visa ブランドのライセンス供与とスキーム運営を担う。日本では国際カードブランドとして **最大シェアと推定** され ^[likely]、[[JapanFG/smbc-card|三井住友カード]] (Visa 提携を主軸とする発行会社)、[[JapanFG/mufg-nicos|三菱UFJニコス]]、[[JapanFG/rakuten-card|楽天カード]] などの主要 issuer に Visa ブランドを供与している。歴史的には **VJA (Visa Japan Association、ビザ・ジャパン協会)** という Visa ブランド国内 issuer 連合体が、三井住友カードを事務局として運営されてきた経緯がある。ブランド窓口 (Visa Worldwide Japan) と協会組織 (VJA) の 2 階建て構造が独特である。

## 1. ライセンス・グループ境界

| 項目 | 注記 |

| 正式名 | ビザ・ワールドワイド・ジャパン株式会社 (Visa Worldwide Japan Co., Ltd.) |

## 2. 日本での事業ライン

- **ブランドライセンス供与**: SMBC 系 (三井住友カード主軸)、MUFG 系 ([[JapanFG/mufg-nicos|三菱UFJニコス]])、楽天カード、イオンカードなどの主要 issuer に Visa ブランドを供与する。
- **スキーム運営**: VisaNet を通じ、国境をまたぐオーソリゼーション / クリアリング / セトルメントを提供する。
- **インターチェンジ料率の設定・モニタリング**: 日本市場向けの料率を設定する。詳細は [[payments/japan-interchange-and-merchant-fee-stack]] を参照。
- **VJA との協調事業**: Visa ブランドの全国共通プロモーション、海外旅行保険などの共同付帯サービス。VJA は協会会員 issuer 横断のキャンペーンを実施する。
- **加盟店戦略・テクノロジー導入**: Visa タッチ決済 (NFC 非接触)、Visa Click to Pay、トークン化、3-D Secure 2.0  などの国内普及。
- **B2B・コマーシャル・クロスボーダー**: 法人 purchasing card、Visa B2B Connect などを提供する。

ブランド窓口としては「日本における Visa の顔」だが、消費者向けのカード発行 / 加盟店契約は一切行わない。つまり、経済産業省の割賦販売法上の登録業者ではない。

## 3. JCB / 国内スキームに対する戦略・競争上の位置

Visa は **日本国内シェア首位の国際ブランド** である ^[likely]。圧倒的な国内 issuer 数、伝統的に 100 行超の地銀・信金などを含む VJA 加盟網、加盟店受容性、グローバル相互利用性で他ブランドに先行する。[[JapanFG/jcb|JCB]] が「日本発ブランド」「アジアで強い」ことを訴求するのに対し、Visa は「世界 200 ヵ国超で使える」「最大の加盟店ネットワーク」を訴求する。[[JapanFG/mastercard-japan|Mastercard Japan]] とは 4 者間スキーム同士のグローバル覇権争いだが、日本では Visa が先行している。楽天カードや Olive ([[JapanFG/smbc-card|三井住友カード]]) のような新規プロダクトでも Visa 採用が多いことが優位性の源泉である。VJA の存在は日本特有で、Visa グローバル本体がブランド供与を集約管理する一方、国内 issuer 連合が共同事業を別途運営する「2階層」構造を成している。

## 4. このページが重要な理由

- 「**Visa Worldwide Japan**」(ブランド窓口) と「**VJA**」(国内 issuer 連合) の **2 階建て構造** を整理する。
- 国内最大シェアの国際ブランドについて、**法人エンティティの境界** を明示する。
- カード issuer ページ ([[JapanFG/smbc-card]] など) との上下関係を整理する起点になる。
