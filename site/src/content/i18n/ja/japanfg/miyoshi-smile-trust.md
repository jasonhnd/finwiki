# Miyoshi Smile Trust / 三好スマイル信託

ウィキ上の位置づけ: [[JapanFG/INDEX]] -> [[JapanFG/trust-companies-japan-index]] -> [[JapanFG/miyoshi-smile-trust]]

## 概要

Miyoshi Smile Trust は、三好不動産グループに紐づく福岡財務支局管内の信託会社行である。 FinWiki では、九州・福岡の不動産および家族資産信託ルートとして位置づける。

## 対象範囲

このページは、信託会社の運営主体としての三好スマイル信託株式会社を対象にする。広い三好不動産グループ、不動産仲介、資産活用コンサルティングのページとは統合しない。

## 事業上の役割マップ

| ルート | 読み方 |
|---|---|
| Real-estate trust | 不動産管理・処分および土地開発信託のルート。 |
| Family asset planning | 個人信託、相続、贈与隣接の文脈。 |
| Regional route | 九州・福岡の信託会社カバレッジ。 |
| Registry control | 次の行: [[JapanFG/trust-companies-japan-index]]. |

## 公開情報上の文脈

Miyoshi Smile Trust が重要なのは、福岡財務支局管内の行を埋め、地域不動産グループが規制対象の信託会社ルートを運営できることを示すためである。 次を補完する: [[JapanFG/kainuma-trust-real-estate]] and [[JapanFG/kiriu-real-estate-trust]].
