# 朝日生命保険


## ウィキ上の位置づけ

この項目は [[JapanFG/INDEX|JapanFG index]] の下に置かれる。 [[JapanFG/fukoku-life|富国生命保険]] をピア比較・対照の文脈として、[[insurance/INDEX|insurance index]] をより広い制度・規制境界としてあわせて読む。

## 要約

朝日生命保険相互会社は、金融庁の生命保険会社免許一覧と生命保険協会会員一覧に掲載される日本の生命保険オペレーティング会社。[[nippon-life]]、[[meiji-yasuda]]、[[sumitomo-life]]、[[fukoku-life]] と同じく相互会社形態をとるため、上場会社や銀行系子会社とは異なる ガバナンス / 資本境界 を持つ。

## 2. 運営モデル

朝日生命は、独立系の伝統的な生命保険会社として、死亡保障、医療・介護、年金 / 貯蓄性商品を扱う対顧客保険会社。FinWiki では「持株会社」ではなく、契約者に直接保険を引き受ける 免許を持つオペレーティング会社 として扱う。

相互会社形態は、株式会社型の保険会社と比較して、契約者ガバナンス、外部株主資本、M&A / 再編時の選択肢が異なる。したがって、単に「大手生保」の一社としてではなく、[[insurance/mutual-vs-stock-life-insurer]] と接続するページにしておく価値がある。

## 3. このページが重要な理由

- 従来は JapanFG の単独エンティティとして表現されていなかった主要な伝統的生命保険会社。
- [[dai-ichi-life]] や [[sony-life]] のような上場 / 株式会社型生命保険会社との有用な比較点。
- FSA 登録一覧と JapanFG の保険カバレッジの整合性を保つ。
