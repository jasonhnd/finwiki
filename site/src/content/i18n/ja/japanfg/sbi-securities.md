# SBI証券


## ウィキ上の位置づけ

本項目は[[JapanFG/INDEX|JapanFG index]]配下に置く。[[JapanFG/paypay-securities|PayPay証券]]とは同業・対比の文脈で、[[securities/INDEX|securities index]]とはより広い制度・規制上の境界を読むために参照する。

## 要約

SBI証券は [[JapanFG/sbi-hd|SBI HD]] のオンライン証券中核会社。FSA 金融商品取引業者一覧では「株式会社SBI証券」、関東財務局長(金商)第44号として掲載される。日本の リテール証券 / NISA / iDeCo / オンライントレーディング導線 を見る上で、SBI グループ本体とは分けて扱うべき 運営会社。

## 1. 法人・ライセンス境界

| 項目 | 読み方 |
|---|---|
| 法人名 | 株式会社SBI証券 |
| FSA登録 | 関東財務局長(金商)第44号 |
| FIEA上の範囲 | 第一種、第二種、投資助言・代理業 |
| 親会社境界 | [[JapanFG/sbi-hd|SBI HD]] オンライン証券部門 |

## 2. 事業上の役割

- 国内株式、投信、外国株式、債券、FX、先物・オプション、NISA / iDeCo を扱うオンライン総合証券。
- [[JapanFG/sbi-hd|SBI HD]] の「証券・銀行・保険・暗号資産」金融 エコシステム の入口。
- [[JapanFG/sbi-fx-trade|SBI FXトレード]] や [[exchanges/jp-exchange-sbi-vc-trade|SBI VC Trade]] と隣接するが、FIEA上のブローカー・ディーラー としての主語は SBI証券。

## 3. この独立ページが重要な理由

SBI HD は金融持株・投資グループ、SBI証券は リテール証券 の 運営会社。NISA 資金流入、手数料ゼロ化、銀行連携、ポイント連携、オンラインブローカー競争 を追うには、SBI証券を独立ページにする必要がある。
