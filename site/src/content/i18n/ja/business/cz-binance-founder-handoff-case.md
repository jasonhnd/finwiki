# Changpeng Zhao(CZ)Binance 創業者引き継ぎ事例 — DOJ $4.3bn 和解 + 退任 + Richard Teng CEO 就任 + 保有持分の維持

## ウィキ上の位置づけ

本項目は [[business/INDEX|business INDEX]] の下に、公開された人物ケーススタディとして位置づけられる(注:Binance は非上場だが、創業者の移行は公的記録に基づく事例である)。米国上場の既存大手の対照的な経路については [[business/brian-armstrong-coinbase-public-company-template|Brian Armstrong Coinbase exchange-as-public-company template]]、既存銀行 CEO の姿勢転換の並行事例については [[business/jamie-dimon-anti-crypto-pivot-case|Jamie Dimon anti-crypto pivot case]]、米国規制当局側の文脈については [[business/hester-peirce-sec-regulatory-pivot-case|Hester Peirce SEC regulatory pivot]] と対照して読むこと。グローバル CEX 比較については [[exchanges/INDEX|exchanges INDEX]] と併せること。

## 要約

Changpeng Zhao(CZ)の **2023-11-21 の答弁合意と Binance CEO からの退任**は、**創業者 CEO が個人の刑事責任について DOJ と和解し、運営主体の所有権を維持しつつ、規制当局に好意的なリーダーシップから登用された新 CEO に運営の支配を引き継ぐ**という、典型的な事例を画する。取引内容:$4.3bn の和解金(Binance)+ $50mn の個人罰金(CZ)+ BSA / AML 違反についての有罪答弁 + CEO からの退任 + Richard Teng(元アブダビ・グローバル・マーケットの規制当局者)の CEO 就任 + CZ への 4 ヶ月の禁固刑(2024-04,  に量刑、2024 後半までに服役・釈放)。決定的に重要な点として、CZ は運営の支配を移譲しながらも、Binance の**過半数の所有権**と戦略的方向性への影響力を**維持**した。

これは「制裁を受けた取引所の創業者の引き継ぎ」のテンプレートであり — 創業者ピボットマトリクスで言及される [[business/founder-pivot-outcome-template-matrix|FTX / Bankman-Fried failure pattern]] のような完全な崩壊事例とは切り離せる。本事例は、主体が規制措置を生き延び、創業者が法的役員の地位を失っても経済的なプリンシパルであり続ける、実行可能な経路を示している。

## 1. 和解前のポジション

| 要素 | 2023-11  前の状況 |
|---|---|
| CZ の役割 | 創業者 + CEO + 過半数所有者(Binance) |
| Binance のポジション | スポット + デリバティブの取引量で世界第 1 位の暗号資産取引所 |
| 規制上のエクスポージャー | 複数年にわたる DOJ + FinCEN + OFAC の調査(BSA / AML / 制裁) |
| 運営上の本社 | 長く議論された;事実上無管轄 |
| 個人の居住地 | UAE(ドバイ)、公に報道済み |
| 米国での存在 | Binance.US(別主体、SEC の精査の対象でもある) |

累積的な圧力は 2021-2023  にかけて SEC、CFTC、DOJ の並行する案件とともに高まり、2023  年 11 月の解決で頂点に達した。

## 2. 2023  年 11 月の和解アーキテクチャ

2023-11-21 の発表は、並行する複数の解決を組み合わせたものであった:

| 当局 | 措置 | 金額 / 帰結 |
|---|---|---|
| **DOJ** | Binance が有罪答弁(BSA / AML / 制裁違反) | $4.3bn の和解金総額 |
| **DOJ** | CZ が個人として有罪答弁(BSA 第 1956 条) | $50mn の個人罰金 + CEO からの退任 |
| **FinCEN** | 民事制裁金 | ($4.3bn の総額に含まれる) |
| **OFAC** | 民事制裁の和解 | ($4.3bn の総額に含まれる) |
| **CFTC** | より早期の民事解決 | (別途) |
| **SEC** | 民事訴訟は別途継続(2023-11  の取引の一部ではない) | 係争中の訴訟 |

総額 $4.3bn は、当時、米国史上、暗号資産企業に対する金融制裁の和解として最大であった。和解により、Binance は複数年にわたって米国政府承認のコンプライアンス・モニターを設置し、特定の事業 / 地域から撤退することが求められた。

## 3. 創業者引き継ぎのメカニズム

引き継ぎは意図的に構成された:

1. **CZ が有罪答弁** — 個人の刑事責任を受け入れ、長期裁判のリスクを回避
2. **CEO からの退任** — 和解により必須;今後の役員レベルの責任経路を取り除く
3. **所有権の維持** — 和解は CZ の Binance における経済的利益の売却を求めなかった
4. **後継者の事前指名** — Richard Teng が同時に CEO に就任し、継続性と規制当局に好意的な信任を提供
5. **コンプライアンスの抜本的見直し** — Binance は米国承認のモニターと構造的なコンプライアンス変更にコミット
6. **地理的 / 事業の再構築** — Binance は一部の市場から撤退し、他を再編
7. **個人への量刑** — 2024-04,  に CZ は米国連邦刑務所で 4  ヶ月の禁固刑を言い渡された(DOJ の勧告範囲と整合)

**所有権を維持しつつ運営の支配を譲る**ことにより、CZ は長期的な経済的アップサイドを保全し、その間も主体は運営を続けた。

## 4. Richard Teng — 規制当局から CEO への信任

Richard Teng のプロフィールは引き継ぎの成功にとって重要であった:

| キャリアの区分 | 役割 |
|---|---|
| Binance 以前の規制当局 | アブダビ・グローバル・マーケット(ADGM)金融サービス規制庁(FSRA)CEO |
| Binance 以前の企業 | シンガポール金融管理局(MAS)コーポレートファイナンス部長 |
| Binance 以前の取引所 | シンガポール取引所(SGX)での各種役職 |
| Binance への参入 | 2021  に地域責任者として参加、その後グローバルの地域市場責任者へ |
| 昇進経路 | 各種シニア職 → 地域市場責任者 → CEO 就任 2023-11 |

ストーリー:元 MAS / ADGM のシニア規制当局者が最大の暗号資産取引所でグローバル CEO の役割を担うことは、Binance が今後コンプライアンスに真剣であることを世界の規制当局に示した。これは [[business/christine-moy-talent-signal-jpm-apollo|Christine Moy JPM → Apollo case]] における人材シグナルのパターンに並行する、意図的な**規制当局から CEO への人材フロー**であった。

## 5. 引き継ぎ後の軌跡(2024-2025)

| 時期 | 展開 |
|---|---|
| 2024-Q1 | Richard Teng が世界の規制当局と公に関与;Binance はコンプライアンスチームの拡大を発表 |
| 2024-04-30 | CZ に連邦刑務所 4  ヶ月の量刑 |
| 2024-Q3 | CZ が釈放;公の講演 / 業界関与を再開(役員ではない立場で) |
| 2024-2025 | 和解の逆風にもかかわらず Binance の市場シェアは底堅い;スポット支配を維持 |
| 2025 | CZ が業界の方向性、AI、暗号資産規制について公にコメント |
| 継続中 | SEC 民事訴訟が継続;複数管轄のコンプライアンス姿勢が進化 |

決定的に重要な点:CZ の維持された所有権と継続的な公の存在は、役員レベルのガバナンスがリセットされても、創業者と取引所の間の**ブランドと経済の結びつき**が無傷で残ったことを意味した。

## 6. 他の創業者危機テンプレートとの比較

| 創業者 / 企業 | 危機 | 創業者の帰結 | 主体の帰結 |
|---|---|---|---|
| **CZ / Binance(本事例)** | DOJ 和解 | CEO 退任、所有権維持、4 ヶ月の量刑 | 運営継続、$4.3bn を支払い、コンプライアンス抜本見直し |
| SBF / FTX | 詐欺罪 | 有罪、25 年の量刑 | 破産、顧客資産返還プロセス |
| Do Kwon / Terra | 複数管轄の詐欺罪 | 拘束、引き渡しプロセス | 主体の崩壊 |
| Su Zhu + Kyle Davies / 3AC | 破産 | 個人の逃避の試み | 清算 |
| Brian Armstrong / Coinbase | SEC エンフォースメントを争う | CEO を継続(個人の刑事責任なし) | [[business/brian-armstrong-coinbase-public-company-template|Coinbase template]] 参照 |
| Jamie Dimon / JPM | 危機なし;姿勢転換 | CEO を継続 | [[business/jamie-dimon-anti-crypto-pivot-case|Dimon pivot case]] 参照 |

CZ テンプレートの際立った価値:**深刻な DOJ の解決を通じて、創業者の経済的利益と主体の両方が生き延びる**ことを示している点である。これは暗号資産の刑事解決事例における稀な「ソフトランディング」テンプレートである。

## 7. 保有持分維持のインサイト

法的に重要な動き:和解の文言は CZ に売却を求めなかった。彼は CEO および役員を退任し、個人罰金を支払い、刑に服した — しかし彼の株式保有は維持された。

これがテンプレートにとって重要である:

- 同様のエンフォースメントに直面する創業者にとって:**運営支配の譲歩 + 金銭的罰則 + 刑事答弁 + 所有権維持**は、DOJ が受け入れうる既知の解決の形である
- 投資家 / カウンターパーティにとって:創業者は、主体の成功との経済的整合性を失うことなく、ガバナンスから「除去」されうる
- 規制当局にとって:和解後のモニター体制のもとで、主体がプリンシパル役員の刑事措置を生き延びうるという先例を設定する
- 競合取引所にとって:最悪のエンフォースメントでさえ、適切に構成すれば致命的である必要はないというシグナル

## 8. 日本 / アジアの取引所運営者への読み替え

日本の VASP 規制([[exchanges/fsa-vasp-registration-system|FSA VASP registration system]] 参照)は、Binance がくぐり抜けた米国の枠組みよりも厳格な事前ライセンスのルールのもとで運営される。CZ テンプレートが日本で直接的に当てはまりにくいのは以下の理由による:

- 日本の取引所(例:[[JapanFG/bitflyer|bitFlyer]]、[[JapanFG/gmo-coin|GMO Coin]]、[[JapanFG/bittrade|BitTrade]])は金融庁の事前承認のもとで運営される;同等のエンフォースメント措置は通常、和解ではなくライセンス取消につながる
- 日本の AML / FATF コンプライアンス制度はより予防的 — 和解前の段階でライセンスが危機に瀕する
- 日本の企業ガバナンス規範は、CZ 型の創業者支配ほど個人に集中していない

ただし、本テンプレートは、子会社が米国の域外エンフォースメントの対象となった場合に世界的にどのような解決の形が利用可能でありうるかについて、日本上場の暗号資産隣接の運営者(例:[[JapanFG/sbi-hd|SBI HD]] のデジタル資産エクスポージャー)に示唆を与える。

## 9. 反論

- CZ の事例は解決までに 4+ 年の調査を要した — 「ソフトランディング」の帰結は速くも安くもない
- 維持された所有権は継続的な運営コンプライアンスを条件とする;さらなるエンフォースメントが状況を変えうる
- 一部の論者は、$4.3bn の和解が Binance の収益基盤に対して違反を過小評価していたと主張する — 異なる DOJ の姿勢であれば主体の解散を要求しえた
- 4 ヶ月の量刑は、ありうる帰結の下限であった;比較可能な事例(FTX)ははるかに長い刑を受けた
- CZ は役員でなくても非公式に影響力を維持している — 「引き継ぎ」は実務上部分的である
- Richard Teng の成功は、米国の政治環境とともに変わりうる規制当局の好意に依存する

## 10. 未解決の問い

- CZ の維持された株式保有は、将来の規制措置のもとで強制売却の対象となるか?
- Binance / CZ に対する SEC の民事訴訟は、追加の金銭的 / 構造的帰結をもたらすか?
- 規制当局から CEO へというパターン(Teng)は、他の主要取引所でも再現されるか?
- CZ の引き継ぎは、その波が展開する中で、他の主要取引所運営者(例:Bybit、OKX)に対する将来のエンフォースメントとどう比較されるか?
- コンプライアンス・モニター期間が終了したら、CZ は(例えば新たなベンチャーを通じて)公開会社の取引所の役割に戻るか?
