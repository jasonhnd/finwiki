# ソフトバンク・ビジョン・ファンド + Arm IPO のテンプレート — ポートフォリオ企業の上場を通じたファンド組成型の持分マネタイズ

## ウィキ上の位置づけ

本項目は [[business/INDEX|business INDEX]] の下に、人材・ポートフォリオのクラスターにおける上場企業の戦略ケースとして位置する。ポートフォリオ企業の上場に関するピア / 対比の文脈については [[business/brian-armstrong-coinbase-public-company-template|Brian Armstrong Coinbase exchange-as-public-company template]]、対照的な日本のFG独立ルートのケースについては [[business/kitao-yoshitaka-sbi-independent-strategy-case|Kitao Yoshitaka SBI independent strategy]]、より広い資本市場の文脈については [[finance/INDEX|finance INDEX]] と照らし合わせて読まれたい。ビジョン・ファンドの構造上の親会社については [[JapanFG/INDEX|JapanFG INDEX]] を参照。

## 要約

ソフトバンクグループ(SBG、東証 9984)による 2023-09 年の **Arm Holdings** のNasdaq(ARM)への再上場は、**ポートフォリオ持分のIPOを、親会社のマネタイズおよびバリュエーションの結晶化のツールとして用いる** 典型的なケースである。SBGは(2020-2022 年のファンドレベルの再配置の後、ビジョン・ファンドIではなくソフトバンクグループのエンティティを通じて)2016, 年に $32億でArmを取得し、その後 2023 年9月に、約 $54.5億のバリュエーションでArmのおよそ 10% をNasdaqに上場させ、IPO後に〜90%を保有した。この取引は、意味のあるプライマリーの調達を行わなかった — それは、SBGのバランスシート上で資産をマークし、Arm株式を担保とするマージンローンを支え、ビジョン・ファンド世代の資産が規模をもってIPO可能であることを投資家にシグナルするために用いられた、バリュエーション結晶化のイベントであった。

その後の株価の上昇(Armは数か月以内に $100 を上回り、2024年までに $130 を上回って取引された)は、SBGの報告されたNAV(一株当たり)を押し上げ、SBGをArm持分の時価評価で評価すべきか、それとも保守的な流動性調整後の条件で評価すべきかについて、活発な批判 / 擁護を引き起こした。

## 1. ビジョン・ファンドの構造とArmのそこを通る経路

| ビークル | 設定 | Armの保有経路 |
|---|---|---|
| ビジョン・ファンドI(VF I) | 2017, 年、〜$100億、アンカーLPはPIF + Mubadala + SBG | 2016 年のSBGによる取得後、Armのマイノリティを保有(ローテーションで組み入れ) |
| ビジョン・ファンドII(VF II) | 2019, 年、SBG単独の資本〜$50億 | 重要なArmのポジションを保有せず |
| SBGバランスシート直接 | 2016年以降、ビジョン・ファンドIのアンワインド後 | 2023 年のIPOまでに最大のArm保有者 |
| LatAmファンド | 2019年以降、〜$8億 | 別個のLPの焦点 |

2023 年のIPOの時点までに、Armは主に **ソフトバンクグループの直接のバランスシート** 上にあった(ビジョン・ファンドIの部分的なエクスポージャーが 2020-2022年にわたってSBG自体へ再配置された後)。これは税務および報告にとって重要である:IPO後のマークに対する利益は、ビジョン・ファンドのLPではなく、SBGの公開株主に帰属する。

## 2. 取引のアーキテクチャ(2023年9月)

| 要素 | 詳細 |
|---|---|
| 発行体 | Arm Holdings plc(英国法人、米国上場) |
| 取引場所 | Nasdaq Global Select Market(ティッカー:ARM) |
| プライマリー / セカンダリーの構成 | 圧倒的にセカンダリー(SBGの売出し)、控えめなプライマリー |
| 浮動株 | 株式の〜10% |
| IPO価格 | 一株 $51  |
| IPOバリュエーション | 〜$54.5億 |
| 主幹事 | Barclays、Goldman Sachs、JPMorgan、Mizuho |
| IPO後のSBGの持分 | 〜90% |
| ロックアップ期間 | 180 日(標準) |

意図的な 15% 未満の浮動株は、ARMをSBGに密接に保有させ、公開市場のバリュエーションを結晶化させつつ、将来のセカンダリーの売出しに対するオプション性を維持した。

## 3. なぜこのIPOは現金調達よりもマーカーとして重要だったのか

IPO前、Armは議論の余地のある簿価でSBGのバランスシート上にあった。批判者は、SBGが 2016年に過大に支払ったと主張した。SBGは、ArmのCPUのIPが、AIサーバー、IoT、自動車、そして継続的なモバイルSoCの優位を通じて複利的に成長すると主張した。

10% をIPOすることで、SBGは次を達成した:

1. **公開市場の価格発見** — Armは、私的なバリュエーションではなく、上場された時価を持つようになった
2. **マージンローンの担保** — SBGは、はるかに改善されたLTV比率でArm株式を担保に借り入れることができた
3. **NAVのマークアップ** — SBGの報告された「推定株式価値」が、より高いArmのマークで擁護可能になった
4. **LPへのシグナル** — ビジョン・ファンド世代の資産が規模をもって上場可能であることを示し、将来のファンドのマーケティングを支えた
5. **オプション性の維持** — 90% の保有維持により、価格の上昇に応じた将来の売出しが可能になる

現金調達の側面は副次的であった — SBGはすでに財務上の流動性と継続的なマージンローンの能力を持っていた。

## 4. IPO後のSBGの戦略

Armの上場の後:

| イニシアチブ | ロジック |
|---|---|
| AIテーゼへの転換 | SBGは、Arm + ポートフォリオのAI投資を通じて、自らを「AIに最も多くエクスポーズした持株会社」と位置づけた |
| 自社株買い | SBGは、上昇したNAVを用いて自社株買いを資金手当した(NAVと時価総額のディスカウントの縮小) |
| ビジョン・ファンドII / III の一時停止からの再開 | 2022 年の評価減の後、デプロイメントを減速させ、その後、選択的なAIの賭け(OpenAIなど) |
| 継続的なArmの保有 | 売却よりも保有という戦略的決定 — Armは「ポートフォリオ」ではなく「コア」とみなされる |
| マージンローンのスタック | AI投資を資金手当するためにArmを担保に借り入れ — [[finance/japan-leveraged-buyout-economics|leverage economics]] との比較を参照 |

## 5. 比較:Arm IPO 対 他のビジョン・ファンドのエグジット

| エグジット | タイプ | SBGの結果 | テンプレートの教訓 |
|---|---|---|---|
| **Arm 2023** | 高い保有を伴うIPO | バリュエーションのマーカー + NAVの結晶化 | IPOをエグジットではなくマークとして用いる |
| Uber 2019 | IPO、その後の売出し | まちまち — 高値で買い、部分的に売却 | ポートフォリオIPOのエグジットにおけるタイミングリスク |
| DoorDash 2020 | IPO、部分的な売出し | 勝ち — パンデミック期のモメンタムを捉えた | ウィンドウのタイミングの懸念 |
| WeWork 2019 → 2021 年のSPAC | 失敗したIPO → 低いマークでのSPAC | 負け — 大幅に評価減 | SPACルートを通じたダウンラウンド |
| Coupang 2021 | ピークでのIPO | まちまち — 初日は強いが、その後は弱い | ロックアップの解除を通じて分配 |
| Better.com 2023 年のSPAC | ダウンラウンドでのSPAC | 負け | ダウンラウンドのテンプレート |

Armのテンプレートが際立つのは、バリュエーションのマーカー機能を優先して **最大のプライマリー調達を明示的に回避した** からである。Arm以前の大半のビジョン・ファンドのIPOは、上場をエグジットのイベントとして扱った;Armは上場を時価評価のイベントとして扱った。

## 6. 日本のコングロマリットのカーブアウトへの読み替え

SBG-Armのテンプレートは、いくつかの重要な点で [[business/sony-fg-partial-spinoff-case|Sony FG partial spinoff]] と異なる:

| 次元 | SBG-Arm IPO | ソニーFGの部分スピンオフ |
|---|---|---|
| メカニズム | IPOによる部分的な売出し | 株式分配による現物分配 |
| 親会社の保有 | 〜90%(数年かけてゼロになりうる) | <20%(制度上義務付け) |
| 税務上の取扱い | 売却部分についてのキャピタルゲイン | 課税繰延(制度適格) |
| 株主が受け取るもの | 市場を通じて間接的に現金 | 新会社の株式を直接 |
| 主たるユースケース | マークの結晶化、オプション性の維持 | 事業の分離、コングロマリット・ディスカウントの解消 |
| 規制ルート | 外国発行体のIPO(Arm英国 / Nasdaq米国) | 日本の部分スピンオフ制度([[corporate-strategy/partial-spinoff-tax-deferral|see regime]]) |

両者ともバリュエーションの結晶化をもたらすが、非常に異なる親会社の目的のためである。SBGはオプション性とAIテーゼのために保有し、ソニーはポートフォリオを合理化するために分離する。

## 7. 反論

- 90% の保有持分は、SBGの実際の売出しを圧倒するArm株式への時価評価エクスポージャーを生む — SBGの報告されたNAVは、Armの上場価格に対して極めて敏感である
- Armのバリュエーションは、まだ固まっていないAI / データセンターのIPロイヤルティの軌道に依存する — 倍率は急激に圧縮されうる
- ビジョン・ファンドのビークルは、2022-2023年に他のポートフォリオ銘柄で繰り返し紙の損失を計上した;Arm IPOはVF I / VF II のパフォーマンスを遡及的に正当化するものではない
- 批判者は、Armの本源的価値はその上場価格より大幅に低いと主張する;上場価格を担保として用いることは、Armが下落した場合にマージンコールのリスクを生む
- このテンプレートは、ポートフォリオ資産が規模をもってIPO可能であることを要する — 大半のビジョン・ファンドのポートフォリオ企業(とりわけ失敗した賭け)は、このアプローチで再現できない

## 8. 未解決の問い

- AIコンピュートからのArmのロイヤルティの成長は、現在の上場バリュエーションを正当化するのか、それとも倍率の圧縮が起きるのか?
- Armのテンプレートは、[[JapanFG/sbi-hd|SBI HD]]、[[business/gmo-internet-group|GMO Internet Group]]、または過小にマークされた子会社を持つ他の持株会社構造について、日本上場の戦略にどのような示唆を与えるのか?
- Arm IPOの成功は、他のビジョン・ファンド / SBGのポジション(例:特定のAIポートフォリオ銘柄)の同様の部分的なIPOを解き放つのか?
- NAVのマークと供給オーバーハングがArmの株価に与える影響のバランスをとる、最適な売出しのペースは何か?
