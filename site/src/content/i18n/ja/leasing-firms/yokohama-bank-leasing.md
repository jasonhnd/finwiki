# 横浜銀リース (Yokohama Bank Leasing)

## ウィキ上の位置づけ

このエントリは、[[leasing-firms/INDEX|leasing-firms INDEX]] 配下に位置する、[[regional-banks/concordia-fg|横浜フィナンシャルグループ (旧コンコルディア FG)]] に対応する FG 子会社リース事業会社アンカーである。親事業銀行 [[regional-banks/yokohama-bank|横浜銀行]] および兄弟子会社 [[regional-banks/higashi-nippon-bank|東日本銀行]] と照らして読む。ピア比較は [[leasing-firms/mebuki-lease|めぶきリース]] (北関東地銀系) および大手リース会社 [[leasing-firms/mitsubishi-hc-capital|三菱 HC キャピタル]] / [[leasing-firms/mizuho-lease|みずほリース]] / [[leasing-firms/tokyo-century|東京センチュリー]]。より広い制度ルートについては [[banking/regional-bank-consolidation-pattern|regional-bank consolidation pattern]] と [[banking/INDEX|banking index]] を参照。

## 要約

株式会社横浜銀リースは、[[regional-banks/yokohama-bank|横浜銀行]] (現 [[regional-banks/concordia-fg|横浜フィナンシャルグループ]] 子会社) 連結子会社の地銀系リース会社で、神奈川県・首都圏を地盤に設備リース・ファイナンスリース・オペレーティングリース・自治体リース・医療リース等を提供する。地銀系リースの中では上位規模で、[[regional-banks/concordia-fg]] (旧 CFG、2025-10 商号変更) の地銀 FG 最大級の預金顧客基盤を背景とする。^[extracted]

### 横浜銀リース 設立

- [[regional-banks/yokohama-bank|横浜銀行]] 系列リース会社として、神奈川県内中堅企業向け設備リースを主業
- 横浜銀行の県内 No.1 圧倒シェアを背景に、リース業務でも神奈川県内シェア上位
- 戦後地銀系リースの典型: 銀行融資ではなく設備リース形態での中小企業設備投資支援

### CFG → YFG 商号変更 (2025-10-01)

- 親持株会社 [[regional-banks/concordia-fg]] が 2016-04 [[regional-banks/yokohama-bank]] + [[regional-banks/higashi-nippon-bank]] 共同株式移転で発足
- **2025-10-01**: 旧 コンコルディア FG → **横浜フィナンシャルグループ** 商号変更
- 横浜銀リース 自体は商号変更の影響を受けず、引き続き [[regional-banks/yokohama-bank]] 連結子会社として運営

### FG 内ポジション

- **銀リース連携の中核**: [[regional-banks/yokohama-bank]] の神奈川県内中堅企業メインバンク顧客のリース需要受託
- **設備投資支援**: 銀行融資補完 + 簿外処理ニーズへの対応
- **県内自治体・公的セクター連携**: 神奈川県・横浜市・川崎市等 自治体リース案件
- **横浜銀行ブランドのリース展開**: [[regional-banks/concordia-fg|YFG]] 「横浜」ブランド回帰 (2025-10) との整合性

### 競合構図

- **大手リース**: [[leasing-firms/mitsubishi-hc-capital|三菱 HC キャピタル]] / [[leasing-firms/smfl|SMFL]] / [[leasing-firms/mizuho-lease|みずほリース]] / [[leasing-firms/fuyo-lease|芙蓉リース]] / [[leasing-firms/tokyo-century|東京センチュリー]] / [[leasing-firms/ricoh-leasing|リコーリース]] - 大型・全国・複雑案件で優位
- **地銀系リース**: [[leasing-firms/mebuki-lease|めぶきリース]] (北関東) / [[leasing-firms/hokkaido-lease|北海道リース]] / 千葉興業リース 等 - 同型モデル
- **メーカー系リース**: [[card-issuers/toyota-finance|トヨタファイナンス]] / [[leasing-firms/nec-leasing|NEC キャピタルソリューション]] - 製品連動
- **県内競合**: 横浜銀行系・神奈川県内信金リース・大手リース支店

### 強みと弱み

- 強み: [[regional-banks/yokohama-bank]] 預金 15 兆円超 (単体) の中堅企業メインバンクネットワーク + 首都圏 GDP 規模 + 神奈川県内 No.1 認知度

## 7. 規制・政策

- **主管**: 金融庁 (FSA) (リース業単独としては銀行業免許不要、貸金業登録は対応領域に応じて)
- **規制論点**:
  - リース会計基準変更 (IFRS16 / 日本基準) 対応支援
  - 中小企業向けリース税制活用 (中小企業投資促進税制)
  - 環境リース (再エネ設備リース) の脱炭素税制対応
  - 反社チェック・マネロン対策 ([[regional-banks/concordia-fg]] グループ共同実施、2018-08 東日本銀行 業務改善命令後の内部統制強化と整合)
