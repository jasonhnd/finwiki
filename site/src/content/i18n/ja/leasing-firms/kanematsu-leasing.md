# 兼松リース

## ウィキ上の位置づけ

この項目は [[leasing-firms/INDEX|leasing-firms INDEX]] 配下に置かれる。[[cooperative-banks/ja-mitsui-leasing|JA三井リース]] は同業比較の文脈（いずれも商社系リース）、[[banking/INDEX|banking index]] はより広い制度・規制境界の文脈として読む。[[leasing-firms/tokyo-century|東京センチュリー]]（伊藤忠系）と合わせると、商社 x リースの広いパターンを把握しやすい。

## 要約

兼松リース は 兼松 (旧 兼松江商、中堅総合商社) の キャプティブ / 関連リース機能を担う中堅リース会社。三大リースや SMFL のような大規模商社 x リース プラットフォーム と異なり、兼松本体の事業領域 (電子・デバイス・食品・鉄鋼・繊維 等) に紐づいた捕食型 / 業界特化型リースが主体。商社系リースとしては小型ながら、兼松の海外ネットワーク (東南アジア・中国・米州) を活用したクロスボーダー案件で存在感を持つ。

## 2. 事業領域

- **商取引付随リース**：兼松本体の輸出入商取引に紐づく機器リース・輸送機器リース
- **電子デバイス・IT 機器リース**：兼松の電子デバイス事業部門との連動
- **食品関連リース**：兼松の食品事業 (食肉・水産等) 関連設備リース
- **鉄鋼・産業機器リース**：兼松の鉄鋼事業との連動
- **海外リース**：兼松の海外拠点 (東南アジア・中国・米州) との連携
- **環境関連**：再エネ機器・省エネ設備
- **不動産関連リース**：限定的

## 3. 戦略と競争上の位置づけ

兼松リースは「総合商社 x リース」モデルの小型版。大手商社系リース ([[leasing-firms/smfl|SMFL]] = 住友商事 + SMFG、[[leasing-firms/tokyo-century|東京センチュリー]] = 伊藤忠系の歴史、[[cooperative-banks/ja-mitsui-leasing|JA三井リース]] = 三井物産系) のように 銀行 x 商社 の合弁体ではなく、兼松グループ単独 キャプティブ 性が強い。

兼松本体の業容 (電子・デバイス・食品・鉄鋼・繊維) に紐づいた専門性が強み。商社の海外ネットワーク (特に東南アジア・米州) との連動でクロスボーダーリース案件にアクセスできる点は中堅リースとしての差別化軸。

業容規模では準大手 ([[leasing-firms/fuyo-lease|芙蓉総合リース]] / リコーリース / NECキャピタルソリューション 等) には及ばず、独立系・中堅リース層のなかでも兼松グループの内部 キャプティブ 色が強い。

[[finance/japan-acquisition-finance|日本アクイジションファイナンス]] の文脈では、兼松関連の事業承継 / クロスボーダー M&A での補助的役割。

## 4. このページが重要な理由
