# 日本の地震保険・官民共同スキーム

## 概要

日本の家計向け地震保険は、通常の民間損害保険商品ではない。民間損害保険会社、[[non-life-insurers/japan-earthquake-reinsurance|Japan Earthquake Reinsurance]]、日本政府が関与する官民共同スキームである。通常は家計向け火災保険に付帯され、住宅建物と家財を対象に、法定の商品制約、支払区分、政府が支える再保険構造を備える。

このページは [[insurance/natcat-reinsurance-japan|nat-cat reinsurance Japan]]、[[insurance/japan-nonlife-big-three|Japan non-life big three]]、[[non-life-insurers/japan-earthquake-reinsurance|Japan Earthquake Reinsurance]]、[[non-life-insurers/toa-reinsurance|Toa Reinsurance]]、[[non-life-insurers/tokio-marine|Tokio Marine]]、[[non-life-insurers/msad|MS&AD]]、[[non-life-insurers/sompo|Sompo]] と併読する。

## スキームマップ

| レイヤー | 役割 |
|---|---|
| 保険契約者 | 火災保険に付帯された家計向け地震補償を購入する。 |
| 元受損害保険会社 | 保険を販売・管理し、スキーム経路を通じて保険金を支払う。 |
| 日本地震再保険 | 民間保険会社から再保険責任を引き受け、スキーム内でリスクを再配分する。 |
| 民間損害保険業界 | 定められたレイヤー・限度額まで責任の一部を分担する。 |
| 政府 | 再保険の裏付けを提供し、極端な損失レイヤーの大きな部分を吸収する。 |
| 財務省 / 公的財政レイヤー | 政府責任に対応する特別会計・財政経路を維持する。 |

公共政策上の目的は、安定的な提供可能性を確保することにある。地震リスクは相関が高く、深刻で、地理的に集中した損失を生み、民間保険会社だけで吸収することが難しい。そのため、このスキームは公的再保険を用いて、家計向け地震補償を社会的に受け入れ可能な水準で提供し続ける。

## 商品境界

| 問い | スキーム上の答え |
|---|---|
| 家計は単独の地震保険を購入できるか | 標準的な経路は火災保険への付帯であり、地震保険だけを単独購入する形は通常の構造ではない。 |
| 何が補償対象か | 住宅建物と家財であり、すべての商業用財産リスクではない。 |
| どの危険が対象か | 地震、噴火、津波による損害で、商品ルール上の関連する火災・損壊・埋没・流失区分を含む。 |
| 地震火災は火災保険だけで足りるか | 通常の火災保険は一般に地震に起因する火災損害を補償しないため、そのエクスポージャーには地震保険が経路となる。 |
| 商品は完全に民間か | いいえ。政府が再保険と責任分担を通じて参加する。 |

## 再保険メカニクス

日本地震再保険は公開されたスキーム図を公表している。民間損害保険会社は地震保険責任をスキームへ出再する。日本地震再保険は、その責任を自社、民間保険会社、政府の間で標準化・再配分する。総支払限度額とレイヤー別分担は、日付付きの公開データである。

レイヤー金額と総支払限度額は、日本地震再保険 / 財務省資料の出典日付とともに記録する。

## 自然災害再保険との違い

| 論点 | 地震保険スキーム | より広い自然災害再保険 |
|---|---|---|
| 主な被保険対象 | 家計の住宅建物 / 家財 | 商業用不動産、自動車、海上、海外P&C、事業中断、スペシャルティライン |
| リスク移転設計 | 法定の官民共同フレームワーク | 民間再保険、再々保険、cat bond、グループ保有、資本市場 |
| 公的役割 | 政府の直接的な再保険参加 | 通常は間接的。ただし公的災害財政や政策措置を除く。 |
| FinWiki上の経路 | このページ | [[insurance/natcat-reinsurance-japan]] |

家計向け地震保険スキームから、[[non-life-insurers/tokio-marine|Tokio Marine]]、[[non-life-insurers/msad|MS&AD]]、[[non-life-insurers/sompo|Sompo]] の災害エクスポージャー全体を推定してはいけない。大手損保グループは、この官民共同の家計向けスキームの外側で、商業、海外、地震以外の災害リスクも引き受けている。

## スキームの効果

官民共同スキームには、次の構造的効果がある。

- 民間保険会社は、テールリスク全体を単独で抱えずに家計向け地震補償を提供できる。
- 政府の再保険参加は、災害リスクに対する公的財政の一部である。
- 再保険構造は、レイヤーごとの民間部門の保有と保険金支払責任を変える。
- 保険料、補償限度額、支払区分、再保険責任は公共政策上の変数である。
- 官民のリスク分担は、この経路を保険だけでなく [[policy-finance/INDEX|policy finance]] とも接続する。
