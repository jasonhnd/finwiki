# ESR（経済価値ベースのソルベンシー比率）


## ウィキ上の位置づけ

この項目は [[insurance/INDEX|insurance index]] の下に置かれる。 [[insurance/economic-value-based-solvency|Economic value-based solvency regulation]] をピア比較・対照の文脈として、[[JapanFG/INDEX|JapanFG index]] をより広い制度・規制境界としてあわせて読む。

## 要約

ESR は、利用可能な経済価値ベース資本が必要リスク資本に対して十分かを示す、保険会社レベルの経済的ソルベンシー比率である。正確な算式や名称は会社・規制フェーズによって異なるため、FinWiki では ESR を単一の普遍的な会計数値ではなく、資本十分性のシグナルとして扱う。

制度そのものは [[insurance/economic-value-based-solvency|economic value-based solvency regulation]] を参照し、本ページは会社比較に用いる。

## 読み方のルール

| 問い | ESR の読み方 |
|---|---|
| 保険会社は経済価値ベースで十分に資本化されているか | ESR が高いほど経済価値ベースの資本余力が大きいことを通常は意味するが、会社ごとの定義と移行前提を確認した後で読む。 |
| 経営の自由度は制約されているか | ESR 低下は配当、自社株買い、商品価格、ALM、再保険、M&A 意欲に圧力をかけ得る。 |
| 会社間で比較できる比率か | 部分的にしか比較できない。表面上のパーセンテージを比べる前に、方法論注記を比較する。 |
| ESR はソルベンシー・マージン比率と同じか | 違う。ソルベンシー・マージンは従来型の規制比率で、ESR は経済価値ベースの視点である。移行期には両者が併存し得る。 |

## JapanFG での関連性

- [[JapanFG/dai-ichi-life]]: グループ規制 ESR をめぐる資本十分性とガバナンス変更の開示は、上場株式の評価に直接関係する。
- [[JapanFG/nippon-life]]: 日本生命は業績資料で経済価値ベースのソルベンシー / ESR を用いる。相互会社であるため、投資家目線の論点は自社株買いだけでなく、資本配分と契約者還元である。
- [[JapanFG/tokio-marine]], [[JapanFG/msad]], [[JapanFG/sompo]]: 損保 ESR は、巨大災害リスク、再保険価格、海外スペシャルティ・エクスポージャー、契約者保護とあわせて読むべきである。

## 分析上の用途

1. **資本政策**: ESR は配当、自社株買い、劣後債、M&A のゲートとなる。
2. **ALM**: 長期負債を持つ生命保険会社は、金利、予定利率保証商品、デュレーション・ミスマッチに敏感である。
3. **商品戦略**: 保障性商品、貯蓄性商品、外貨建て商品、年金商品は資本消費の仕方が異なる。
4. **再保険**: 再保険はリスク資本を減らし得る一方、カウンターパーティ、コスト、利用可能性の制約を加える。

## 注意点

- 表面上の ESR は、市場変動、リスク削減、新契約の採算、方法論変更によって改善し得るため、改善要因が重要である。
- 会社開示の ESR は、最終的な規制 ESR と同一とは限らない。
- 非常に高い比率も、未活用資本、弱い成長、過度なリスク削減を意味するなら自動的に良いとはいえない。
