# 日本の医療保険特約商品マトリクス

## TL;DR

日本の医療保険は、より広範な生命保険業界の中の専門フランチャイズであり、一つの外資系（[[life-insurers/aflac-japan|Aflac Japan]]）が歴史的に新規がん保険市場のおよそ 70% を保有し、他のいくつかの外資系（[[life-insurers/aia-life-japan|AIA Japan]]、[[life-insurers/manulife-japan|Manulife Japan]]、[[life-insurers/prudential-japan|Prudential Japan]]）が大規模な医療 / 保障の保有契約を運営し、損保大手三社の生保子会社プラットフォーム（[[life-insurers/tokio-marine-nichido-life|Tokio Marine & Nichido Life]] / あんしん生命系譜、[[life-insurers/sompo-himawari-life|Sompo Himawari Life]]、[[life-insurers/mitsui-sumitomo-aioi-life|Mitsui Sumitomo Aioi Life]]）が医療 / 特約のシェアを争い、生保大手四社の相互会社（[[life-insurers/nippon-life|Nippon Life]]、[[life-insurers/dai-ichi-life|Dai-ichi Life]]、[[life-insurers/sumitomo-life|Sumitomo Life]]、[[life-insurers/meiji-yasuda|Meiji Yasuda]]）が医療特約を終身・定期のフランチャイズにバンドルしている。商品の提供は、競争上のポジションを決定する少数の軸に還元できる：（1）主力ライン — がん特化（アフラックの優位）、医療（入院日額＋手術一時金）、または特約（基礎となる終身・定期に付帯）；（2）保険料払込構造 — 終身払対定期払対短期払の設計；（3）給付タイプ — 一時金、入院日額、実損填補、またはその組合せ；（4）先進医療特約 — ほぼ普遍的だが補償上限の差がある；（5）チャネルミックス — 専属営業職員、代理店、銀行窓販、または直販オンライン；そして（6）解約返戻金の設計 — 貯蓄型（解約返戻金あり）対純保障（解約返戻金なし）。競争のヒートマップは、がん（アフラック対ソンポひまわり対 AIA 対 MS あいおい）、純医療（アフラック対ソンポひまわり対東京海上あんしん対オリックス / メディケア型のマトリクス外プレイヤー）、特約付帯医療（生保大手 4  対外資系）に位置する。

## なぜこのマトリクスが重要なのか

日本の世帯の医療保障の選択を読むということは、「医療保険」という傘の下にバンドルされた少なくとも三つの異なる商品アーキテクチャの中から選ぶことを意味する：

1. **がん特化（がん保険）** — 別個の引受、別個の待機期間、別個の保険料カーブを持つ別個の商品ライン。アフラックは 1970 年代に現代の日本のがん保険市場を構築し、数十年にわたりこのセグメントの新規業務でおよそ 70% のシェアを保持してきた。
2. **一般医療（医療保険）** — 入院日額給付＋手術一時金＋特約メニュー（先進医療、三大疾病、入院一時金）。競争が最も熱いのはここである。
3. **終身または定期保障に付帯する特約（医療特約 / 入院特約）** — 生保大手 4  相互会社、および保障バンドルプランを運営する外資系（ソニー生命のライフプランナー、プルデンシャル、マニュライフ）から、基礎となる生命保険契約にバンドルされる。

以下のマトリクスは各軸を分解する。具体的な商品名、保険料率、チャネル手数料、引受限度は日付特定であり、各キャリアの商品パンフレット、金融庁の募集ルール開示、およびキャリアの年次 / ディスクロージャー文書と照合して確認すべきである。

## アフラック生命 — がん特化の優位と現代的な医療メニュー

[[life-insurers/aflac-japan|Aflac Japan]]（Aflac Life Insurance Japan / アフラック生命保険）は、日本のがん保険における歴史的かつ現在のリーダーである。構造的特徴：長年にわたるゆうちょ銀行の提携を含む広範な代理店・銀行窓販の販売を伴う、がん（がん保険）に特化した専門商品ライン、がん給付をバンドルするか単独で成立する補足的な医療（医療保険）商品、専属営業職員 / 代理店のハイブリッド販売。アフラックは 1974  にオリジナルの日本のがん保険商品を構築し、現代のメニューは特約で最も差別化されている（がん治療一時金、がん治療中の入院日額給付、先進医療、がん通院治療給付）。保険料払込構造は通常、終身払または短期払であり、その選択が平準保険料カーブに影響する。

## AIA 生命 — マルチラインの保障フランチャイズ

[[life-insurers/aia-life-japan|AIA Japan]]（AIA グループの日本子会社、旧 AIG スター生命の遺産）は、がん、医療、定期、終身、特約メニューを備えた完全な保障フランチャイズを運営している。日本における AIA の競争上のポジションは、代理店販売、提携銀行の銀行窓販、ファイナンシャル・アドバイザー / ライフプランナー隣接の営業職員を組み合わせている。医療 / がんの保有契約は新契約価値への主要な貢献者である。

## 東京海上あんしん系譜 — 損保大手 3  生保アフィリエイト

[[life-insurers/tokio-marine-nichido-life|Tokio Marine & Nichido Life]]（あんしん生命のレガシー）は、[[non-life-insurers/tokio-marine|Tokio Marine Holdings]] 損保グループの生保アフィリエイトである。医療 / がんのフランチャイズは歴史的に「メディカル Kit R」シリーズと、医療でアフラックと、クロスライン販売でソンポひまわりと直接競合した後継商品ファミリーによってアンカーされていた。損保親会社の全国的な代理店ネットワークが構造的な販売上の優位である。

## ソンポひまわり生命 — 損保大手 3  生保アフィリエイト

[[life-insurers/sompo-himawari-life|Sompo Himawari Life]]（損保ジャパン日本興亜ひまわり生命 / 現ソンポひまわり生命）は、[[non-life-insurers/sompo|Sompo Holdings]] の生保子会社である。医療における構造的特徴は、がん、三大疾病保障、女性特有医療、先進医療の特約を備えた強力な特約商品フランチャイズである。販売は専属営業職員、代理店、提携銀行のチャネルを組み合わせている。

## 三井住友海上あいおい生命 — MS&AD 生保アフィリエイト

[[life-insurers/mitsui-sumitomo-aioi-life|Mitsui Sumitomo Aioi Life]] は [[non-life-insurers/msad|MS&AD]] グループの生保子会社である。医療 / 特約のフランチャイズは、[[non-life-insurers/mitsui-sumitomo-insurance|MSI]] と [[non-life-insurers/aioi-nissay-dowa-insurance|ADI]] の両方の代理店ネットワークへのアクセスを持つ MS&AD の販売エコシステムの内部に位置する。商品メニューは医療、がん特約、定期、終身、個人年金をカバーする。

## アフラックと AIA を超える外資系 — マニュライフ、プルデンシャル、メットライフ、AXA

[[life-insurers/manulife-japan|Manulife Japan]] と [[life-insurers/prudential-japan|Prudential Japan]] は、ライフプランナー / FA 型の営業職員を備えた保障主導のフランチャイズを運営している。医療 / 特約の商品メニューは、単独の医療やがん商品ではなく、医療特約付きの終身・定期保障によってアンカーされている。[[life-insurers/axa-japan|AXA Japan]]（旧アクサ生命）はマルチチャネルの医療 / 保障フランチャイズを運営している。メットライフ生命、AIG 生命、およびより小規模な外資系が競争セットを締めくくる。

## 生保大手 4  相互会社 — 終身内のバンドル医療特約

四つの大手相互会社 — [[life-insurers/nippon-life|Nippon Life]]、[[life-insurers/dai-ichi-life|Dai-ichi Life]]、[[life-insurers/sumitomo-life|Sumitomo Life]]、[[life-insurers/meiji-yasuda|Meiji Yasuda]] — は、専属営業職員を通じて、主に終身または定期の基礎契約への特約として医療保障を販売する。住友生命の Vitality ブランドのウェルネス連動型商品は、四社の中で最も差別化された構造的特徴である；他社は特約メニューの幅、営業職員の信頼、企業グループのクロスセルで競争する。各社は、専門保障を好むセグメント向けに、代理店および銀行窓販ルートを通じて単独の医療 / がん商品を運営している。

## 歴史的・構造的文脈

現在の日本の医療保険構造は、商品と規制の数十年にわたる進化を反映している：

- **1974  アフラックのがん保険参入** は、日本市場に専門のがん保険を導入し、このセグメントを構築した。アフラックの販売は企業 / 雇用主グループ販売と代理店を組み合わせた。
- **1995  保険業法改正** は、免許制度を近代化し、保障 / 医療専門における外資系の拡大の基礎を築いた。
- **1996–2001  第三分野の自由化** は、医療 / がん / 介護の専門を、対等な競争条件で外資系と損保グループの生保子会社に開放した。第三分野は、生命（第一分野）と損害（第二分野）の間に位置する医療 / がん / 介護商品の規制カテゴリーである。
- **2001  以降のゆうちょ銀行販売アライアンス** は、ゆうちょ銀行の窓口ネットワークを通じてアフラックのがん保険のリーチをスケールさせた。
- **2000 年代後半以降の先進医療特約の標準化** は、先進医療特約を高い実費上限を伴う業界標準の付加保障とした。
- **2010 年代の短期払商品の波** は、終身保障が継続しながら退職前に保険料払込を終える 60 払 / 65 払の平準保険料構造を提供した。
- **2010 年代の外貨建バンドルの拡大** は、[[life-insurers/dai-ichi-frontier-life|Dai-ichi Frontier Life]]、[[life-insurers/ms-primary-life|MS Primary Life]]、マニュライフ等が販売する、銀行窓販を通じた医療特約付き外貨建終身を組み合わせた。
- **2015  以降の引受基準緩和の拡大** は、標準引受の下では謝絶されていたであろう高齢 / 軽度の既往症のある申込者に医療 / がん保障を開放した。
- **2020 年代のウェルネス連動型商品**（[[life-insurers/sumitomo-life|Sumitomo Life]] Vitality 等）は、保険料割引を健康行動と生体データに連動させる。
- **2025  ESR の導入** は、金融庁の経済価値ベースのソルベンシー制度の下で、すべての金融庁境界のキャリアの医療 / がん負債キャッシュフローの資本処理に影響する — [[insurance/economic-value-based-solvency|economic-value-based solvency]] と [[insurance/esr-economic-value-solvency|ESR]] を参照。

## 意思決定での利用

このマトリクスは、日本の医療保険の競争セットを、より広範な生命保険業界に対してではなく、互いに対して読むときに使用する。実務的な問い：

- **がん専門対医療バンドル。** アフラックは単独がんで優位；複数のキャリアが医療で競争する。がん特化保障を購入する世帯は、一般医療を購入する世帯とは異なる競争セットに直面する。
- **解約返戻金の設計。** 現代の医療 / がん商品は、有意な解約返戻金のない純保障として設計されている。古い契約には解約返戻金が組み込まれている場合がある。その選択は保険料カーブと税務処理に影響する。
- **終身対短期払。** 短期払商品は、初期年度の保険料が高くなる代わりに、退職前に保険料払込を終える。解約返戻金の軌道と保障コストの経済性が異なる。
- **外貨建医療バンドル。** 医療特約付き USD 建終身は為替リスクを契約者に移転するが、より高い解約返戻金予測の文法を提供する。銀行窓販の販売ルートが主要チャネルである。
- **チャネルの適合性。** 専属営業職員は生保大手 4  の医療特約の優位なチャネル；代理店 / ショップは外資系の単独医療 / がんの優位なチャネル；銀行窓販は外貨建バンドルで優位；直販オンラインは [[life-insurers/lifenet|Lifenet]] / [[life-insurers/sbi-life|SBI Life]] のレーンである。
- **共済境界の代替。** 多くの世帯は、金融庁境界の医療 / がん保障と共済医療特約（JA共済 / 全労済 / 県民共済 / COOP共済）の両方を保有している。協同組合境界のオーバーレイについては [[insurance/japan-kyosai-vs-fsa-insurance-perimeter-matrix|the kyosai vs FSA perimeter matrix]] を参照。
- **厚労省の国民健康保険との相互作用。** 日本の医療保険商品は、厚労省が高額療養費制度を通じて自己負担に上限を設ける [[insurance/medical-cancer-insurance-product-economics|public health insurance system]]（公的医療保険）の上に位置する。補足保障の商品設計はこのベース層を前提とする。

## 境界ケース / 注意点

- **数字は概念的である。** 市場シェア（特にアフラックの約 70% のがんシェア）、保険料率、給付上限、特約の利用可能性は、各商品サイクルと開示期間で変化する。出典は金融庁、生保業界の刊行物、各キャリアの商品パンフレットと統合 / ディスクロージャー文書、および公的医療保険に関する厚労省の刊行物である。
- **アフラックのがんシェア。** 約 70% の数字は、新規がん保険市場シェアの広く引用される業界の略記であり、長年にわたり保持されてきた。具体的な最近のヴィンテージは、アフラックの開示と生保業界のデータと照合して確認すべきである。
- **キャリアの命名史。** いくつかのキャリアは改称または合併している（あんしん生命 → 東京海上日動あんしん生命の系譜；損保ジャパン日本興亜ひまわり生命 → ソンポひまわり生命；AIG スター生命 → AIA 生命の系譜；住友生命 Vitality は別個のキャリアではなくウェルネス連動型商品ライン）。このマトリクスは現在のエンティティ名を使用する。
- **第三分野。** 医療、がん、介護の商品は規制上の目的で第三分野に位置する — 生命（第一分野）と損害（第二分野）の間。すべての金融庁境界の生保キャリア（および生保子会社経由の損保キャリア）が参加する。
- **外貨建商品。** 医療特約付き外貨建終身は為替リスクを契約者に移転する。行為規制は、特に高齢の顧客に対して、これらの商品の専用の適合性と説明を要求する。[[insurance/bancassurance-economics-japan|bancassurance economics Japan]] を参照。
- **オンライン直販の対比。** [[life-insurers/lifenet|Lifenet]] と [[life-insurers/sbi-life|SBI Life]] は、上記のがん専門マトリクスでは主要競合として現れないが、より低い保険料水準での純医療 / がん専門で成長中のプレイヤーである。[[insurance/internet-life-insurance-business-model|the internet life model]] を参照。
- **ウェルネス連動型商品。** Vitality 等のウェルネス連動型商品メニューは、行動を保険料割引で報いる。この構造的特徴は長期的なプライシングと継続率に影響するが、商品ラインの分類を変えるものではない。
- **解約返戻金と税。** 医療 / がん商品の解約返戻金は一般に設計上抑制されている；終身ベース契約の解約返戻金の税務処理は標準の所得税法に従う。
- **公的医療保険のベース。** 日本の補足的医療商品は公的医療保険のベースを前提とする。高額療養費の上限ルールの変更は補足商品の経済性に影響しうる。
- **かんぽ生命の商品メニュー。** 特別法の認可制約を受ける。[[insurance/kampo-japan-post-insurance|the Kampo entry]] を別途読むこと。
