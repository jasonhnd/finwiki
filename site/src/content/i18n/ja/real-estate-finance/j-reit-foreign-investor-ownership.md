# J-REIT外国人投資家保有

## 要約

外国人投資家は、金額ベースでJ-REIT投資口総発行高の約 4 分の 1 から 3 分の 1 を保有し、毎週のフローは JPX 投資家タイプの J-REIT 取引統計で公的に追跡可能です。 外国人投資家ベースは構造的に 3 つの側面から構成されています: (1) グローバル REIT 指標追跡またはアクティブピック義務の範囲内で J-REIT エクスポージャーを含むグローバルREIT専門ファンド (米国、ヨーロッパ、アジア太平洋)。 (2) ソブリン・ウェルス・ファンドと大手年金基金が、注目市場に対して日本に対してコアまたはコアプラス不動産エクスポージャーを取得している。 (3) 方向性取引またはキャップレート/保有スプレッド取引を実行するヘッジファンドおよびトレーディングスタイルの投資家。 それは、JGBカーブ上のJ-REIT分配金の見通しの上昇と、JPY-USD (または JPY-EUR / JPY-AUD) のヘッジコストです。 2022 後のFRB/ECBの引き締めにより、円と収益のヘッジコストが非常に高くなり、株式投資家に対して完全にヘッジされたJ-REIT銘柄はヘッジされていないキャリーを下回りました。 国内REITの外国人所有権 (~10-15% 範囲) と比較すると、J-REITは構造的に外国投資家の需要により大きく依存していることがわかります。

| メトリック | 読書 |

| 外国人投資家の買い越し・売り越し（週次フロー） | J-REITのJPX投資家タイプ統計を公的に追跡 |
| 国内個人シェア | 重要だが指数レベルでは外国人投資家のシェアより小さい |

25～35% の範囲は、日本のほとんどの上場市場セグメントに比べて大きいです。 外国人投資家は、J-REIT のシェアで最大の保有者ではありません（国内の生命保険会社、年金基金、資産運用会社が合計すると、より多くの保有をしています）が、最も活発なマージンフローです。つまり、マージンでの価格設定は海外からのフローに大きく影響されます。

### シェアの観察方法

JPXでは、株式市場統計とは別に、J-REIT取引に関する投資家別統計を毎週公表しています。 投資家のカテゴリには、個人 (個人)、投信 (投資信託)、銀行 (銀行)、生損保 (生命保険会社および損害保険会社)、信託銀行 (信託銀行 - 多くの場合、外国および国内の委任先の候補者として機能します)、自己 (独自)、外国人 (外国人) などが含まれます。 外国人投資家の買い越し/売り越しは週単位で追跡できます。 信託銀行の指名ポジション ([[banking/master-trust-bank-operating-model|MTBJ]] や [[trust-banks/custody-bank|CBJ]] などの「信託口」表面) には国内と外国の受益所有権が混在しているため、保有株式の読み取り値をリアルタイムで抽出することは困難です。

### セグメント 1  — グローバル REIT 専門ファンド

グローバル REIT インデックスに連動するアクティブ ファンドには、先進国 REIT ユニバースの一部として J-REIT エクスポージャーが含まれています。 パブリック出典の構造:

| セグメント | 読書 |
|---|---|
| 資金 | 米国、欧州、アジア太平洋地域の大手資産運用会社が運営するグローバル REIT ファンド |
| スタイル | 指数連動型とアクティブ — アクティブ戦略は指数に対して J-REIT をオーバーウェイトまたはアンダーウェイト |
| FXトリートメント | 多くの場合、基本通貨 (USD、EUR、AUD) に部分的にヘッジまたは完全にヘッジされます。
| 保有集中 | 流動性と ADR / クロスリストの基準を満たす最大の [[real-estate-finance/top-10-j-reit-overview-matrix|top-10 J-REIT]] 名を優先 |

### セグメント 2 — 政府系資産と多額の年金

政府系ファンドや大手年金基金は、広範な不動産配分の一環として J-REIT のポジションを取得しています。

| セグメント | 読書 |
|---|---|
| 資金 | ソブリン・ウェルス（アジア太平洋、中東）および米国/欧州/アジア太平洋の年金 |
| スタイル | 多くの場合、長期で部分的にヘッジされています。 J-REIT は多くの場合、直接資産または [[real-estate-finance/gk-tk-bond-real-estate-spv|GK-TK SPV]] / [[real-estate-finance/private-reit-japan-vs-listed-j-reit-comparison|private-REIT]] の保有を補完します。
| 保有集中 | 大型株の J-REIT 名全体。 パッシブインデックスチルト |
| レポートの可視性 | 限定; 外国の年金と SWF の多額のポジションが、クロスした場合に大量の保有報告 5% ルールの開示で表面化することがあります。

### セグメント 3 — ヘッジファンドとトレーディングスタイルの投資家

ヘッジファンドとトレーディングスタイルの口座は、J-REIT に対してディレクショナル戦略とスプレッド戦略を実行します。

| 戦略 | 読書 |
|---|---|
| 方向 | キャップレート、リースロールオーバー、または資産タイプのビューに基づいた J-REIT の長い名前と短い名前 |
| キャップレートスプレッド | J-REIT のロングと日本国債のショート（またはその他の金利エクスポージャー） — キャップレート / 金利スプレッドの観点を反映 |
| セクターペア | J-REIT サブセクターにおけるオフィスの長期化と物流の短期化 (またはその逆) |
| クロスアセット | J-REIT と株式 REIT プロキシ (例: 日本の主要な開発株) |
| ヘッジ | アクティブなFXヘッジ管理 — 時にはヘッジ自体が取引になる |

### ヘッジコストが支配的な理由

J-REIT への米ドルベースの外国投資家の場合、オールインリターンはおよそ次のようになります。

```
米ドルの合計リターン = (J-REIT 利回り + J-REIT 価格リターン)
- (ヘッジ部分のJPY-USDヘッジコスト)
+ (ヘッジされていない部分のJPY-USD為替リターン)
```

日本円のヘッジコストは、クロスカレンシーベーシススワップと組み差によって設定されます。 FRB 2022 後の引き締めにより、USD-JPYのヘッジコストはピーク時に年間 4% を超え、ほとんどのJ-REITの分配金相場よりも大幅に高騰しました。

| シナリオ | 読書 |
|---|---|
| 米ドル投資家に対する完全ヘッジされたJ-REIT利回り（FRB高金利環境） | 多くの場合、ヘッジベースではマイナスになります。つまり、分配金利回りからヘッジコストを差し引いた額 < 0% |
| 米ドル投資家に対するヘッジなしの J-REIT 利回り | JPY-USD FX による円建ての分配利回り |
| 意味 | ヘッジコストが高い場合、ヘッジ対象のJ-REITエクスポージャーに対する外国投資家の需要は急激に圧縮される |
| 逆流 | 円利回りをヘッジする投資家（円の機関投資家）は、同じ J-REIT を魅力的だと感じると同時に、海外のヘッジ投資家が魅力的ではないと考える可能性があります。

### クロスカレンシーベーシススワップ

クロスカレンシーベーシススワップ（JPY-USDベーシス）では、金利差に数十ベーシスポイントが加算または減算されます。 米ドル受け取り/円支払いのカウンターパーティ（外国投資家の J-REIT ヘッジに関連する側）にとってベーシスはマイナスになる可能性があり、ヘッジ利回りがさらに低下します。 クロス通貨ベースの公的出典データは、日銀および主要な金利データプロバイダーから入手できます。

### J-REIT の価格設定への影響

日本円とヘッジコストのサイクルが外国投資家の J-REIT フローの変動を引き起こします。 公的情報源の観察:

1。 FRB/ECB 金利が低く、円米ドルベースが良好な場合、ヘッジされた J-REIT の利回りは外国人投資家にとって魅力的であり、外国人投資家のフローは純プラスになります。
2。 FRB / ECB 金利が上昇し、円と米ドルのベーシスが拡大すると、ヘッジされた J-REIT 利回りは低下またはマイナスになります。 外国人投資家の流入は純マイナスに転じる。
3。 外国人投資家の売買シェアが大きいため、J-REIT ユニットに対する限界価格の影響は重要です。

## 4。 米国REIT外国人保有との比較

| 側面 | J-REIT | 米国REIT |
|---|---|---|
| 外国人投資家の売買高シェア | 多くの場合、毎週40-60% | 取引量に占める割合が低い |
| 国内機関アンカー | 生命保険会社、年金会社、資産運用会社、[[insurance/japan-life-insurance-alm-overview|life ALM book]] | 年金基金、投資信託、保険、小売 |
| 国内小売シェア | 材料はあるが外国人投資家の流れより小さい | 資料 — 米国 REIT は小売証券口座で広く保有されている |
| 為替ヘッジコストが要因 | 主なドライバー | 関連なし (国内通貨市場) |
| クロスリスト | 限定的（少数の J-REIT 名が ADR 経由でクロスリストされている） | 該当なし |
| 外国人投資家の委任ルート | グローバル REIT ファンド、SWF、年金、ヘッジファンド | グローバル REIT ファンドと国際直接配分 |

### J-REIT が外国依存度が高い理由

J-REIT の外国人所有割合が米国 REIT よりも高い理由は、いくつかの構造的要因によって説明されています。

1。 **国内小売ベースが小さい** — 日本の個人投資家が保有する上場 REIT は、米国の個人投資家が米国 REIT に比べて他の円資産に比べて少ない。
2。 **国内の円利回り環境** — J-REIT 時代のほとんどにおいて、円利回り投資家は J-REIT の分配利回りが日本国債と比べて魅力的であると感じていました。 これにより需要の一部は吸収されたが、限界フロー水準で外国投資家が参入する余地は残された。
3。 **世界の REIT インデックスへの組み入れ** — 日本の大規模な上場 REIT 市場は、世界の REIT インデックスの中でかなりの比重を占めており、構造的な外国人投資家の需要を生み出しています。

### 生命保険会社の流れ

[[insurance/japan-life-insurance-alm-overview|Japanese life insurers]] は、代替資産/不動産スリーブの一部として J-REIT を保有しています。 負債は日本円であるため、日本円のエクスポージャーをヘッジする必要はありません。 外国人投資家が（ヘッジコストの圧力により）J-REITを売り越すとき、J-REITの価格設定が国債よりも魅力的な円利回りの上昇を生み出すのであれば、生命保険会社は買い越す可能性がある。

### 年金基金の流れ

[[banking/master-trust-bank-operating-model|Master trust bank]] の保管データは、GPIF と企業年金が株式委任と専用の不動産スリーブを通じて重要な J-REIT 配分を保有していることを示しています。 年金資金の流れは、配分目標の構造により、一般に外国資金やヘッジファンドの流れよりも安定しています。

### 地方銀行の流れ

地方銀行は利回りを求めてJ-REITを国債ポートフォリオに保有している。 地方銀行の資金の流れは、絶対的な円利回りと全体的な円曲線の形状に敏感です。

## 6。 保管および所有権チャネルの仕組み

外国人投資家は通常、グローバルカストディ会社である日本のサブカストディチェーンを通じてJ-REITを保有しています。

```
外国人投資家
    │
│ 監護権
    ▼
グローバルカストディアン（BNYメロン/ステートストリート/JPM/シティ）
    │
│ 副監護権の委任
    ▼
日本のサブカストディアン（通常はグローバルカストディアンの日本支社）
    │
│ 保振振替
    ▼
機構 ────► 外国人投資家の受益保有を計上
国内信託銀行ノミネート下
(「トラストポート」サーフェス — MTBJ または CBJ)
```

これは、外国投資家による J-REIT の有益な保有が、国内機関投資家による保有と同じ MTBJ / CBJ ノミニー表面の下に位置することを意味します。 J-REIT取引統計における投資家の種類の内訳は取引側のデータから観察可能ですが、保有側の内訳は決済側の分析が必要です（詳細な表には公開されていません）。

信託銀行の指名メカニズムについては [[banking/master-trust-bank-operating-model|Master Trust Bank of Japan operating model]] を、より広範なインフラストラクチャ マップについては [[banking/japan-master-trust-and-custody-bank-landscape|Japan master-trust and custody bank landscape]] を参照してください。
