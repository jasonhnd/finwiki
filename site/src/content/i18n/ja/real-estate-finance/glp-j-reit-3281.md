# GLP J-REIT（3281）

## 要約

**GLP J-REIT（TSE J-REIT 3281）**は、日本の**近代的物流J-REIT**を代表する2銘柄の一つである。スポンサーは、シンガポールを拠点とするグローバル物流不動産プラットフォームの**GLP（Global Logistic Properties / GLP Pte Ltd）**である。GLP J-REITは2012 年12月に、日本初の外資系スポンサーによる物流J-REITとして上場し、2012 後の近代的物流J-REITセグメント拡大をけん引した主要銘柄だった。ポートフォリオは、**首都圏および関西圏の物流サブマーケット**に所在する**大規模クラスA近代的物流施設**に集中している。具体的には、24/7 トラックアクセス、高い天井高、スプリンクラー設備、大きなフロアプレートを備えたマルチテナント型クロスドック倉庫であり、スポンサーであるGLPが開発し、スポンサー・パイプライン型の資産リサイクル機構を通じてJ-REITへ組み入れる施設である。

GLP J-REITの構造的なアイデンティティは、3つの柱に支えられている。(i) **スポンサー・パイプライン**: GLPが自社バランスシート上で近代的物流資産を開発し、安定稼働した物件を優先供給契約に基づいてGLP J-REITへ供給する。(ii) **能動的な資産入替**: 近代的物流仕様に合わなくなった物件を定期的に売却する。(iii) **ECを軸にしたテナント需要**: Amazon Japan、Rakuten、Yodobashi.com、Mercari物流需要など、日本のECの構造的成長の受益者である。GLP J-REITは、AAA級の近代的物流テナント基盤をめぐって[[real-estate-finance/nippon-prologis-reit-3283|Nippon Prologis REIT (NPR, 3283)]]と正面から競合する。

## 2. スポンサー・プラットフォーム: GLP

| 項目 | GLPの詳細 |
|---|---|
| 本社 | シンガポール |
| 範囲 | 日本、中国、米国、欧州、ブラジル、ベトナム、インドで事業を展開するグローバル物流不動産プラットフォーム |
| 日本事業 | 米Prologisと並ぶ、日本の主要な近代的物流開発業者の一つ |
| スポンサー・パイプライン上の役割 | GLPがバランスシート上で近代的物流施設を開発し、物件を安定稼働させ、選別した安定稼働資産をGLP J-REITへ供給する |
| 投資家基盤 | GLP自体は、中国ソブリンウェルス（歴史的）や各種グローバル機関投資家を含むグローバルLP基盤を持つ |
| 比較対象 | [[real-estate-finance/nippon-prologis-reit-3283|Nippon Prologis REIT]]のスポンサーである米Prologisが最も近いグローバル比較対象 |

GLP J-REITにとって、シンガポール本社の外資系スポンサーであることは特徴的である。多くのJ-REITのスポンサーは、Mitsui Fudosan、Mitsubishi Estate、Nomura Real Estateなどの日本の上場デベロッパーである。外資系スポンサーは、グローバルな物流資産の専門性と、スポンサー段階でのグローバルLP資本へのアクセスをもたらす。

## 5. 資産入替戦略

GLP J-REITの特徴的な運営上の機能は、**能動的な資産入替**である。

| 入替活動 | 理由 |
|---|---|
| 古い物件または仕様に合わない資産の売却 | 進化する近代的物流仕様に合わない物件を売却する |
| 新しいスポンサー・パイプライン資産の取得 | GLPの開発パイプラインから、より新しく、より大型で、現行仕様に近い資産へ置き換える |
| 資本効率 | リサイクルした資本を、長期的なテナント需要がより強い新しい資産へ再投資する |
| ポートフォリオ更新 | 平均築年数と仕様の鮮度を長期的に維持する |
| キャップレート管理 | 通常、売却は取得より高いインプライド・キャップレートで行われ、そのスプレッドが投資主に利益をもたらす |

この資産入替中心のモデルは、オフィスJ-REITが歴史的に採ってきた買い持ちモデルとは対照的であり、近代的物流J-REITがオフィスJ-REITより高いAUM成長テンポを示してきた主要な理由の一つである。

## 6. テナント経済性

| テナント動態 | 読み方 |
|---|---|
| テナント集中 | オフィスJ-REITより高いことが多い。シングルテナントまたは2テナントの施設もあり、マルチテナント施設でも複数の大口3PLまたはECテナントが入る |
| 賃貸借期間 | 近代的物流では通常、オフィスより長い（5-10年以上） |
| 賃料改定 | オフィスより段階的な増額は少ない。テナントが拠点に投じた設備投資のため、テナント粘着性は高い |
| テナント業種 | 3PL（Yamato、Sagawa、Yusen、Nittsu）、EC（Amazon Japan、Rakutenなど）、小売業、製造業 |
| スポンサーのテナント性 | GLP自体はテナントではない。テナントは独立した物流事業者と事業会社ユーザーである |

## 7. GLP J-REITとNippon Prologis REITの比較

| 軸 | GLP J-REIT（3281） | Nippon Prologis REIT（3283） |
|---|---|---|
| スポンサー | GLP Pte Ltd（シンガポール） | 米Prologis |
| 上場日 | 2012 年12月 | 2013 年2月（その直後） |
| 資産フォーカス | 近代的物流、クラスAマルチテナント | 近代的物流、Prologis開発のAAA級施設 |
| 地理的フォーカス | 首都圏／関西圏 | 首都圏／関西圏 |
| スポンサー・プラットフォーム | グローバル物流不動産プラットフォーム | グローバル物流不動産プラットフォーム |
| 資産入替 | 能動的な入替 | 能動的な入替 |
| 外国人投資家プロフィール | 高い。外資系スポンサーとの結びつき | 高い。外資系スポンサーとの結びつき |

両銘柄は、AAA級の近代的物流テナント基盤と、外国人投資家による物流J-REIT配分をめぐって競合する。GLPとPrologisの組み合わせは、日本における**GLP対Prologisというグローバル物流競争の上場プロキシ**である。
