# 日本の CMBS・RMBS 証券化市場

## 要約

日本のモーゲージ担保証券化市場は、二つの系統に明確に分かれる。RMBS（住宅ローン担保証券）は、[[policy-finance/japan-housing-finance-agency|Japan Housing Finance Agency (JHF)]] の月次 Monthly Pass-Through MBS 発行が支配的であり、民間銀行チャネルで組成された Flat 35  固定金利住宅ローンを裏付けとする。これにメガバンクや信託銀行オリジネーターによる小規模な民間 RMBS 系統が加わる。CMBS（商業用不動産ローン担保証券）は構造的に小さく、循環性が高い。2008  後の市場では単一借入人 CMBS が中心であり、コンデュイット CMBS は、日本不動産に対するノンリコース CMBS ローンで大きな評価損が発生した 2008-2010  のストレスサイクル後、実質的に休眠している。アレンジャー機能は、メガバンク系証券子会社、[[JapanFG/mitsubishi-ufj-trust-bank|MUFG Trust]]、[[JapanFG/sumitomo-mitsui-trust|SMTB]]、[[JapanFG/mizuho-trust-bank|Mizuho Trust]] の信託銀行証券化部門、そして残存する外資系ハウスに集中する。格付けカバレッジは国内機関の [[JapanFG/jcr|JCR]] と [[JapanFG/rating-and-investment|R&I]] が中心で、[[JapanFG/sp-global-ratings-japan|S&P]] と [[JapanFG/moodys-japan|Moody's]] がクロスボーダー投資家が必要とするシニアトランシェのグローバルスケール格付けを供給する。

## ウィキ上の位置づけ

この項目は [[real-estate-finance/INDEX|real-estate-finance index]] 配下にあり、日本の不動産金融における証券化側を支える。上場エクイティ・ビークルについては [[real-estate-finance/j-reit-market-overview|J-REIT market overview]]、ウェアハウス / 私募ビークル層については [[real-estate-finance/gk-tk-bond-real-estate-spv|GK-TK bond real-estate SPV]]、証券化前のウェアハウス層については [[real-estate-finance/real-estate-bridge-fund-japan|real-estate bridge fund]]、シニア LTV とトランシェ厚を左右する引受キャップレート入力については [[real-estate-finance/japan-real-estate-appraisal-methodology|Japan real-estate appraisal methodology]] とあわせて読む。公共部門 RMBS エンジンは [[policy-finance/japan-housing-finance-agency|JHF]]、SPV キャッシュフローを運営する受託者インフラは [[banking/japan-master-trust-and-custody-bank-landscape|Japan master-trust and custody bank landscape]]、資産管理と証券化受託の運営分担は [[banking/trust-bank-custody-operating-comparison|trust-bank custody operating comparison]] と組み合わせる。クロスドメインのアンカーは [[insurance/japan-life-insurance-alm-overview|Japan life insurance ALM overview]] である。生命保険会社は、資産クラスのデュレーションと JGB 対比の利回り上乗せがクレジットカーブ分析を補うため、円サイドでシニア RMBS トランシェとシニア CMBS ノートの最大級の機関投資家である。

### 規模が大きく異なる二つの系統

以下は桁感のみである。年ごとの詳細な数値は、出典に挙げた JSDA 証券化市場の動向調査（月次 / 年次 Excel）と JHF MBS 開示にある。JHF MBS だけで累計発行額 ¥26.3  兆 / 残高 ¥12.7  兆（FY2017  末）であり、RMBS 残高は約 ¥15  兆（2016  頃）で、下のレンジを挟む。

| 系統 | 年間発行額（公開情報ベースの概算） | 残高（公開情報ベースの概算） | 支配的な発行体モデル |
|---|---|---|---|
| **RMBS** | 年 ¥1.5-3  兆（JHF MBS ≈ 年 ¥2-3  兆） | ¥10-15  兆 | JHF Monthly Pass-Through MBS が支配的。民間 RMBS は全体の一部 |
| **CMBS** | 活発な年でも ¥500 bn 未満。休眠期はほぼゼロ | ¥500 bn-¥1  兆 | 2010 後は単一借入人 CMBS が中心。コンデュイット CMBS は実質ゼロ |

RMBS と CMBS の非対称性は構造的である。日本の住宅金融制度では、固定金利住宅ローン組成の大きな部分が Flat 35,  を通じて流れ、JHF がほぼ月次で購入・証券化する。これに対し CMBS は、(a) [[real-estate-finance/j-reit-market-overview|J-REIT]] による公募エクイティ資金調達、(b) 銀行バランスシート上のノンリコースローン、(c) 私募ファンド債務と競合し、これらが本来なら公募 CMBS に流れうる機関投資家向け商業用不動産の大半を吸収する。

### 歴史的な流れ

- **1998-2007:** 両系統の立ち上げ。SPC 法（1998）、信託法改正、資産流動化法が、SPV ベース証券化の法的足場を作った。CMBS コンデュイット発行は 2000 年代半ばにかけて、複数借入人の分散プールで拡大した。
- **2008-2010:** 世界金融危機は、日本の CMBS に JP ストラクチャードファイナンス資産クラスの中で最も大きな打撃を与えた。多くのコンデュイット CMBS でテールトランシェの評価損が発生し、バルーン満期のノンリコース CMBS ローンのリファイナンス失敗が、不動産の投げ売り処分を引き起こした。コンデュイット CMBS の投資家基盤は実質的に撤退した。

### JHF Monthly Pass-Through MBS

JHF Monthly Pass-Through MBS はアンカー商品である。公開情報上の構造上の特徴は次のとおり。

| 特徴 | JHF MBS の読み方 |
|---|---|
| 発行体 | [[policy-finance/japan-housing-finance-agency|Japan Housing Finance Agency]] |
| 担保 | 民間金融機関が組成し JHF が購入する Flat 35  固定金利住宅ローン |
| 構造 | パススルー。元利金を月次でパススルーし、信用面でのトランチングはない |
| 信用補完 | 暗黙 / 明示の政策金融的信用支援に加え、独立行政法人としての JHF 自身の信用力 |
| 決済 | [[securities/japan-securities-depository-center|JASDEC]] を通じた振替 |

### 民間 RMBS

民間 RMBS 発行は、メガバンクや信託銀行オリジネーターが自己勘定の固定金利または変動金利住宅ローンポートフォリオの一部を証券化する形で行われる。公開情報上の特徴は次のとおり。

- シニア / メザニン / 劣後トランチングを行い、オリジネーターはリスク保持目的で縦持ちまたは横持ちの持分を残すことが多い。
- プールは通常、シーズニング済み住宅ローンの静的またはほぼ静的なポートフォリオである。
- シニアトランシェは [[JapanFG/jcr|JCR]] と [[JapanFG/rating-and-investment|R&I]] から投資適格格付けを受け、AAA トランシェには [[JapanFG/sp-global-ratings-japan|S&P]] または [[JapanFG/moodys-japan|Moody's]] のクロスボーダー投資家向け格付けが付くことがある。
- プールファクターの償却は、予定償却に小幅な繰上返済を加えたものに沿う。

### RMBS の格付け機関基準

| 要素 | 基準上の重点 |
|---|---|
| プールのシーズニング | 古いローンほど支払い実績が示され、期待損失が低くなる |
| 組成時 LTV | 低い LTV はエクイティバッファーを厚くし、デフォルト時損失を低くする |
| DTI | 収入対債務。返済能力指標 |
| 地理的集中 | 東京 / 大阪集中と地域分散の比較 |
| オリジネーターの質 | メガバンク / [[JapanFG/japan-post-bank|Japan Post Bank]] / 地域銀行とノンバンク・オリジネーターの比較 |
| ローン商品 | Flat 35 （JHF 購入）と通常の銀行住宅ローン、アパートローン（1 棟アパート）の比較 |
| サービシング | マスターサービサー / バックアップサービサー体制 |
| マクロ上乗せ | 失業、家計所得、住宅価格シナリオ |

### 2010

 後の単一借入人 CMBS

2008,  後、コンデュイット CMBS モデルは日本から実質的に撤退し、単一借入人 CMBS に置き換わった。公開情報上の構造上の特徴は次のとおり。

| 特徴 | 単一借入人 CMBS の読み方 |
|---|---|
| 借入人 | 一つまたは少数の関連物件を保有する単一の不動産所有 SPV（多くは [[real-estate-finance/gk-tk-bond-real-estate-spv|GK-TK SPV]] または TMK） |
| ローン | 物件を担保とするノンリコースローン |
| 証券化 | ローンを発行 SPV 信託に移転し、CMBS ノートをトランシェ発行する |
| トランシェ | シニア / メザニン / 劣後。LTV に基づく劣後厚で調整 |
| 満期 | 通常 5-7 年で、満期時にリファイナンスリスクがある |
| 格付け | シニアトランシェは [[JapanFG/jcr|JCR]] と [[JapanFG/rating-and-investment|R&I]] による IG。AAA トランシェはクロスボーダー投資家向けに [[JapanFG/sp-global-ratings-japan|S&P]] または [[JapanFG/moodys-japan|Moody's]] の二重格付けを持つことがある |
| 物件タイプ | オフィス（東京 Grade-A）、物流、ホテル、商業施設、住宅アパートブロック。開発リスク資産はまれ |
| 受託者 | 発行 SPV の信託銀行受託者 |

### コンデュイット CMBS — 休眠

コンデュイット CMBS モデル（複数ローン・プール、分散した借入人基盤、マスターサービサー運営）は、2008-2010  のストレスサイクル後、日本では実質的に休眠している。公開情報から読める理由は次のとおり。

2. 2013  後の円金利カーブ圧縮により、銀行バランスシート上のノンリコースローンが CMBS 調達ローンより安くなり、コンデュイット CMBS 組成を支えたスプレッド裁定が消えた。
3. [[real-estate-finance/j-reit-market-overview|J-REIT]] エクイティが 2010  後に急拡大し、コンデュイット CMBS の借入人基盤になりえた機関投資家向け商業用不動産の多くを吸収した。

### CMBS の格付け機関基準

| 要素 | 基準上の重点 |
|---|---|
| 鑑定 | DCF + 直接還元 + 取引事例比較を含む [[real-estate-finance/japan-real-estate-appraisal-methodology|JREI-compliant appraisal]] |
| キャップレート | [[real-estate-finance/japan-real-estate-appraisal-methodology|JREI cap-rate survey]] と直近の取引実績に照らしたストレステスト |
| DSCR | 債務サービスカバレッジ比率。リファイナンスリスクシナリオでストレステスト |
| テナント集中 | 単一テナントと複数テナント、賃貸借満期プロファイル、テナント信用 |
| 物件タイプ | オフィス、物流、ホテル、商業施設、住宅。それぞれ格付け機関の損失前提が異なる |
| 地理的集中 | 東京 CBD / 大阪 / 地方 |
| スポンサーの質 | スポンサー SPV のエクイティ層、スポンサーの実績、スポンサーのリファイナンス能力 |
| リファイナンスリスク | キャップレートと金利カーブの変化下での満期バルーンリファイナンスシナリオ |

### RMBS オリジネーターの地図

| オリジネーター | RMBS での役割 |
|---|---|
| [[policy-finance/japan-housing-finance-agency|JHF]] | Flat 35  の購入 + Monthly Pass-Through MBS を通じた中核 RMBS 発行体 |
| メガバンク | JHF 向け Flat 35  の直接オリジネーター。自己ポートフォリオ再循環のための民間 RMBS 発行体 |
| 地域銀行 | Flat 35  の組成パートナー。自己ポートフォリオを対象に、時折民間 RMBS を発行 |
| 信託銀行 | 自己ポートフォリオ住宅ローンのオリジネーター。JHF MBS と民間 RMBS の双方で受託者にもなる |
| ノンバンク住宅ローン・オリジネーター | 銀行チャネルに比べて限定的な規模 |

### CMBS オリジネーターの地図

| オリジネーター | CMBS での役割 |
|---|---|
| メガバンクの企業不動産ファイナンスデスク | 後に証券化されるノンリコースローンの組成 |
| 信託銀行の不動産ファイナンス部門 | オリジネーションとアレンジャーを兼ねる |
| 外国銀行日本支店の不動産ファイナンスデスク | クロスボーダー投資家基盤向けオリジネーション。外資系銀行撤退サイクル後は縮小 |
| メガバンク系証券子会社 | 発行時のアレンジャー / ディーラー役 |

### アレンジャー / ディーラー機能

日本の CMBS / 民間 RMBS におけるアレンジャー / ディーラー機能は集中している。

| ディーラー | フランチャイズの読み方 |
|---|---|
| [[JapanFG/mufg-morgan-stanley-securities|Mitsubishi UFJ Morgan Stanley Securities]] | MUFG 側。[[JapanFG/mitsubishi-ufj-trust-bank|MUFG Trust]] を通じた深い信託銀行連携 |
| [[JapanFG/smbc-nikko-securities|SMBC Nikko Securities]] | SMBC 側。[[JapanFG/sumitomo-mitsui-trust|SMTB]] との連携は同一グループ内ではなく、グループ横断的 |
| [[JapanFG/mizuho-securities|Mizuho Securities]] | Mizuho 側。[[JapanFG/mizuho-trust-bank|Mizuho Trust]] を通じた深い信託銀行連携 |

### 信託銀行受託の経済性

信託銀行の受託レイヤーは構造的に重要である。日本の証券化 SPV は、発行 SPV への直接的なローン資産移転ではなく、通常、信託受益権を用いるためである。信託銀行は信託契約の下で原資産（ローンまたは不動産）を保有し、その受益権が移転され、トランシェ化される。これが、JP CMBS / RMBS の信託銀行フランチャイズが [[JapanFG/master-trust-bank|MTBJ]] や [[JapanFG/custody-bank|CBJ]] のようなカストディ専業型信託銀行ではなく、メガバンクグループの信託銀行に置かれる理由の一つである。証券化受託は、単なる資産管理ユーティリティ業務ではなく、手数料を生むフロント業務である。運営分担は [[banking/trust-bank-custody-operating-comparison|trust-bank custody operating comparison]] を参照。

### 国内スケールとグローバルスケール

| 観点 | 国内スケール格付け | グローバルスケール格付け |
|---|---|---|
| 参照母集団 | JP 発行体 / 商品ユニバース | グローバル発行体 / 商品ユニバース |
| ソブリン上限 | 実質的に JGB の上限（AA+ / AAA スケール） | グローバルなソブリン比較可能性 |
| 典型的用途 | 円投資家のリミット管理、JBA / 投信協会報告 | クロスボーダー投資家のリミット管理、バーゼルのリスクウェイト下限 |
| JP CMBS / RMBS でのカバレッジ | 高い（JCR + R&I が支配的） | 選択的（シニアトランシェのみ、かつクロスボーダー販売がコストを正当化する場合のみ） |

### RMBS 投資家基盤

| 投資家タイプ | 需要の読み方 |
|---|---|
| 生命保険会社 | 中核買い手。JHF MBS とシニア民間 RMBS は、ALM ポートフォリオの円長期資産枠に合いやすい。[[insurance/japan-life-insurance-alm-overview|Japan life insurance ALM overview]] を参照 |
| メガバンク（自己勘定） | 自己勘定のトレジャリー・ポートフォリオ向け買い手。JHF MBS のリスクウェイト処理は有利 |
| 地域銀行 | 円ポートフォリオの利回り向上のための買い手 |
| 資産運用会社 | 円スプレッド商品を対象とする債券運用枠の買い手 |

### CMBS 投資家基盤

| 投資家タイプ | 需要の読み方 |
|---|---|
| 生命保険会社 | シニアトランシェの買い手。メザニンはクレジット資産スリーブに入る |
| 地域銀行 | シニアトランシェの選択的買い手。メザニンへの選好はまちまち |
| 資産運用会社 | クレジット運用枠の分散目的の買い手 |
| 外国人投資家 | 選択的。通常はクロスボーダー向け発行でグローバルスケール格付けがある場合のみ |
