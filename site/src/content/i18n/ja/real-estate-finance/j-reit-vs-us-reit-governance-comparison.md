# J-REIT vs US REIT ガバナンス比較

## 要約

[[real-estate-finance/j-reit-market-overview|J-REIT]]とUS REITの最大の構造的差異は運用モデルである。J-REITは外部運用構造を使う。投資法人が、通常はデベロッパー・スポンサーに所有される別法人の資産運用会社へ資産運用を委託する。US REITは、1990年代と2000年代の「内部化」取引の波を経て、圧倒的に内部運用型になった。REIT自身が運用チームを雇用し、外部スポンサーへ別個の運用報酬を支払わない。

この差は、ガバナンス、利益相反、報酬構造、資産取得の透明性、投資主保護へ連鎖する。関連当事者取引のレイヤーは[[real-estate-finance/j-reit-sponsor-structure-conflict|sponsor structure and conflict of interest]]、制度的文脈は[[real-estate-finance/j-reit-market-overview|J-REIT market overview]]とあわせて読む。

## ウィキ上の位置づけ

この項目は[[real-estate-finance/INDEX|real-estate-finance index]]の下に位置づけられる。J-REITとUS REITはいずれも税務上のパススルー投資ビークルであるにもかかわらず構造が異なる理由を理解するため、[[real-estate-finance/j-reit-market-overview|J-REIT market overview]]の後にこの比較を使う。日本固有の関連当事者取引レーンは[[real-estate-finance/j-reit-sponsor-structure-conflict|sponsor structure and conflict of interest]]、スポンサー別マッピングは[[real-estate-finance/top-10-j-reit-overview-matrix|top 10 J-REIT overview matrix]]へ進む。上場デベロッパーとのガバナンス対比には[[finance/japan-listed-financial-groups-investable-universe|Japan listed financial groups investable universe]]、日本の投資法人内の受託者役割には[[banking/trust-bank-custody-operating-comparison|trust bank custody operating comparison]]を使う。

## 1. 横比較

| 軸 | J-REIT | US REIT |
|---|---|---|
| 法的形態 | 投資信託法上の投資法人（投資法人） | 法人、事業信託、または組合（IRC §856に基づくREIT選択） |
| 運用モデル | スポンサー系資産運用会社による外部運用 | 内部運用（UPREIT / DownREITの運営パートナーシップ） |
| 直接雇用者 | ほぼなし。法定役員のみ | REITがフルの運用チームを雇用 |
| 運用者の主体 | 資産運用会社（FSA登録） | REIT自身 |
| 運用者の交代可能性 | 投資主総会で可能だが、実務上は稀 | 内部運用のため実質的に構造固定 |
| 報酬経路 | 資産運用会社への運用報酬 | REIT内部の直接報酬 |
| 利益相反の露出 | スポンサー・パイプライン / 関連当事者取引への構造的露出が高い | 内部化後は低い。外部運用者との利益相反を除去 |
| パイプライン源 | スポンサー（デベロッパー）が資産パイプラインを供給 | オープンマーケットとREIT子会社保有の開発資産 |
| 投資主議決ガバナンス | 投資主総会、役員会 / 監督役員 | 株主が選任する取締役会 / 受託者会 |
| 税務パススルー条件 | おおむね90%の利益分配ルール | 90%の課税所得分配 + 分散 + 所有テスト |
| 分配メカニクス | DPU（1口当たり分配金）。通常は半期 | 配当。多くは四半期 |
| 規制当局 | FSA、JPX | SEC、NYSE / NASDAQ |
| 業界団体 | ARES (Association for Real Estate Securitization) | Nareit |

## 2. 外部運用 vs 内部運用 — 何が問題か

| 観点 | J-REIT外部運用モデル | US REIT内部運用モデル |
|---|---|---|
| 利害の一致 | 資産運用会社の報酬が投資主リターンから一部切り離される。スポンサー持分は助けになるが完全には解決しない | 直接一致。報酬はREIT業績と株式に結びつく |
| 運営コスト項目 | 外部運用者へ運用報酬を支払う（資産連動 + 取得報酬 + 譲渡報酬が典型） | 内部G&A項目。外部運用報酬なし |
| 取得パイプライン | スポンサーの優先交渉 / 優先供給契約が一般的 | REITがオープンマーケットで競争 |
| 取引間価格 | 関連当事者取引方針と投資主保護ルールが必要 | 運営パートナーシップ内の内部帳簿移転のみ |
| 不振資産の売却 | 資産運用会社はAUM連動報酬を減らす処分を勧める誘因が弱い | 内部チームの誘因はREIT全体の業績に連動 |
| 運用者交代 | 投資主総会で理論上可能だが、稀で混乱が大きい | 該当なし（内部運用） |
| 規制上の注視 | FSAは関連当事者取引ガバナンスの欠陥を定期的に指摘 | SECは主に開示とREIT資格テストを見る |

## 3. US REITが内部化へ移った理由

初期のUS REIT産業（1990年代以前）は、現在のJ-REITと同様に外部アドバイザーを使っていた。1990年代のREIT IPOの波と「現代REIT時代」（1991 年のKimco IPO、1992 年のTaubman UPREIT以降）が、ガバナンスを内部化された運用へ動かした。主な要因は次のとおりである。

1. **報酬漏れへの投資家圧力** — 外部アドバイザー報酬は構造的な足かせとして批判された。
2. **UPREIT / DownREITによる課税繰延出資** — 運営パートナーシップ構造により、デベロッパーは不動産を税効率よくREITへ拠出しながら、運用チームをREIT内部に置けた。
3. **株式と連動した報酬** — 内部化により、経営チームへ直接株式報酬を付与できた。
4. **開示の簡素化** — 関連当事者取引の開示が減り、公開会社ガバナンスが明瞭になった。

日本市場は2001 年の開始時点から規制上すでに外部運用モデルを採っていたため、この進化は再現されなかった。J-REITを内部化すべきとの圧力は断続的に浮上してきたが、[[real-estate-finance/j-reit-sponsor-structure-conflict|sponsor structure and conflict of interest]]で記録されるスポンサー・アンカー型パイプラインの構造を考えると、構造的に難しい。

## 4. 利益相反 — どこに現れるか

| 取引タイプ | J-REITの露出 | US REITの露出 |
|---|---|---|
| スポンサーから物件を取得 | 高い。関連当事者取引承認、第三者鑑定、投資主開示が必要 | 実質的に発生しない。REITが自らパイプラインを保有 |
| スポンサーへ物件を売却 | 高い。同じ統制が必要 | 発生しない |
| スポンサーとの共同投資（共同投資 / ブリッジファンド） | 一般的。関連当事者取引統制が必要 | 稀 |
| 運用報酬体系 | 取得報酬構造が取引量を報いる | 内部G&A。取得報酬なし |
| スポンサー貸付または資金調達 | あり得る。関連当事者取引統制が必要 | 稀 |
| スポンサーのJ-REIT投資口保有 | 一般的（スポンサーサポート持分。通常数％） | 同じ意味では該当しない |

## 5. パイプラインの可視性

| J-REIT pattern | US REIT pattern |
|---|---|
| スポンサー・パイプラインリストは取得時またはIR資料で公表 | REIT取得パイプラインは内部管理され、決算で開示 |
| 取得判断における資産運用会社の役割 | REIT投資委員会の役割 |
| 優先供給（first-look / right-of-first-refusal）はIPO書類で一般的 | 標準的な構造特徴ではない |
| AUM成長経路はスポンサーのペースに一部依存 | AUM成長経路はREIT戦略の内部にある |
| 外資系スポンサーのパイプライン（GLP、Prologis）はグローバルスポンサー戦略を反映 | 複数のUS REITは専門分野（物流、データセンター、ヘルスケア）に集中 |

## 6. 投資主 vs 株主保護

J-REITの投資主は、運用チームから一つ構造レイヤーを隔てた位置にいる。投資信託法の下では、役員会と監督役員、資産運用会社の内部統制枠組み、信託銀行の受託者役割（[[banking/trust-bank-custody-operating-comparison|trust bank custody operating comparison]]参照）が主な保護レイヤーである。投資主総会は、資産運用会社の交代を含む重要事項について議決できる。

US REITの株主は、法人 / 信託ガバナンスの連鎖の中で直接位置づけられる。株主は内部運用チームを監督する取締役会 / 受託者会を選任する。株主訴訟、アクティビスト・キャンペーン、委任状争奪は標準的なガバナンス圧力手段であり、外部運用型J-REITより内部化REITの方が道筋は単純である。

## 7. バリュエーション上の意味

| 投資家の問い | 含意 |
|---|---|
| なぜJ-REITのNAVディスカウント / プレミアムはUS REITと異なるのか。 | 外部運用報酬負担とパイプライン依存が終価前提に影響する。 |
| なぜJ-REITではアクティビスト・キャンペーンが稀なのか。 | 外部運用構造とスポンサーサポート持分がアクティビストの経路を複雑にする。US REIT並みのアクティビズム強度は構造的に難しい。 |
| なぜJ-REITの合併は通常スポンサー内で起きるのか。 | スポンサーをまたぐ合併には資産運用会社の変更が必要で、US REITのM&Aより構造的に重い。 |
| なぜ低い[[money-market/INDEX\|Japan rates]]でもJ-REIT利回りがUS REIT利回りを上回りがちなのか。 | 外部運用報酬負担、スポンサー利益相反ディスカウント、日本小型REIT名への外国人投資家層の薄さ、市場構造を反映する。 |
