# 日本の不動産キャップレート圧縮 2026

## 要約

日本の不動産キャップレートは、BoJのNIRP／YCC期を通じて各資産クラスで圧縮され、絶対水準として歴史的な低位に達した。2026, 時点の期待（フォワード）キャップレートのレンジは、おおむね東京プライムCBDオフィス3.0 - 3.5%、大阪プライムオフィス3.5 - 4.0%、東京／首都圏物流3.8 - 4.5%、東京プライム住宅3.5 - 4.2%である。これらのレンジは[[real-estate-finance/INDEX|ARES / JREI]]の公表調査から導いたクラス記述であり、利用前に最新公表資料で再確認すべきである。NIRP後の正常化経路、外国人買い手の引受規律、構造的に低い国内リスクフリーレートが、現在のキャップレートの床を形づくっている。キャップレートは投資助言ではない。このページは経路とリンクのみを示す。

このページは[[real-estate-finance/INDEX]]配下にあり、[[real-estate-finance/bank-commercial-real-estate-lending-japan|bank CRE lending]]、[[real-estate-finance/real-estate-private-credit-japan|private-credit mezz / preferred-equity pricing]]、[[real-estate-finance/j-reit-market-overview|J-REIT acquisition yield]]、[[real-estate-finance/j-reit-dividend-yield-vs-jgb-spread|J-REIT yield-spread analysis]]の評価上の背景である。公表キャップレート調査を生み出す鑑定手法は[[real-estate-finance/japan-real-estate-appraisal-methodology]]で説明している。キャップレートの床を形づくる外国人買い手の存在は[[real-estate-finance/j-reit-foreign-investor-ownership]]で整理している。不動産利回り資産への競合需要としての生命保険会社需要は[[insurance/japan-life-insurance-alm-overview]]につながる。キャップレート対負債スプレッドを考える資金調達コスト参照レートは[[money-market/japan-money-market]]と[[banking/INDEX|banking sector reading]]に接続する。

## キャップレートの定義（簡易版）

| 用語 | 定義 |
|---|---|
| キャップレート（NOI利回り） | 純営業収益を取得価格で割ったもの。初年度安定稼働ベース。 |
| 期待キャップレート | 鑑定調査やJ-REITの取得価格形成で用いられる将来見通しの利回り期待。 |
| 割引率（DCF） | 収益還元法の評価で使うリスク調整後の割引率。キャップレートに成長率とリスクプレミアム要素を加味する。 |
| 復帰／ターミナル・キャップ | DCFのターミナル価値において、N+1 年目NOIに適用するキャップレート。 |
| キャップレート・スプレッド | キャップレートからリスクフリー参照レート（10年JGB）を引いたもの。不動産収入に対するエクイティリスクプレミアムの代理指標。 |

正確なNOI定義は調査によって異なる。総収入から運営費を控除したもの、固定資産税控除後、減価償却前などの違いがあるため、比較時には該当するJREI／ARES調査手法を確認する。

## BoJ NIRP期の低位との比較

NIRP期（おおむね2016 - 2022）には、キャップレートは歴史的低位まで圧縮された。

| 資産クラス | NIRP期の低位クラス | 2026 期待クラス | 含意される経路 |
|---|---|---|---|
| 東京プライムCBDオフィス | ランドマーク取引のスポット価格では3.0%未満 | 3.0 - 3.5% | 絶対的な底からは小幅に拡大。 |
| 東京物流 | 3.5 - 4.0%の底 | 3.8 - 4.5% | リスクフリーレート上昇に伴い小幅拡大。 |
| 東京プライム住宅 | 3.3 - 3.8%の底 | 3.5 - 4.2% | 小幅拡大。構造的な外国人買い手需要が緩衝材となった。 |
| 大阪プライムオフィス | 3.3 - 3.7%の底 | 3.5 - 4.0% | 小幅拡大し、東京より幅は狭い。 |

NIRP期の底は、次の要因で補強された。

- BoJのマイナス政策金利とYCCにより、10年JGBがゼロ近傍に固定されたこと。
- 円調達によるグローバルなキャップレート・アービトラージが東京に有利に働いたこと。
- 国内リスクフリー参照レートが構造的に低かったこと。
- 生命保険会社／年金のエクイティ需要に支えられたJ-REIT取得パイプラインが豊富だったこと。
- TMK／GK-TK構造を通じた外国GP資本の展開。

## NIRP後の正常化の影響

NIRP解除とYCC巻き戻しにより、10年JGB参照レートはゼロ近傍の底から大きく上昇した。キャップレートへの機械的な期待効果は次の通りである。

| ドライバー | 方向 |
|---|---|
| リスクフリーレート（10年JGB）の上昇 | リスクフリーに対するスプレッドを一定とすれば、キャップレートの床を押し上げる。 |
| シニア債務コスト（プロジェクトファイナンス参照レート）の上昇 | エクイティ投資家の要求リターンを上方に圧迫し、キャップレートを押し上げる。 |
| 国内預金／貯蓄金利の上昇 | 代替資産と比べた不動産利回りの相対的魅力を低下させる。 |
| 外国人買い手需要の維持 | 純粋な国内金利ロジックが示す水準より低いキャップレートの床を支える。 |
| 安定したインフレ期待 | 実質賃料成長の前提を支え、名目キャップレート拡大の価格影響を和らげる。 |
| 円安 | USD／EUR建ての参入価格を下げる効果があり、キャップレートの床を支える。 |

ネットの2026 の読み方としては、NIRP期の底からのキャップレート拡大は緩やかであり、急激ではない。外国人買い手の引受上の床と、構造的な国内リスクフリーのアンカーがともに保たれているためである。同じ論理をスプレッド主導で読む場合は[[real-estate-finance/j-reit-dividend-yield-vs-jgb-spread|J-REIT yield-spread analysis]]を参照する。

## 外国人買い手によるキャップレートの床

外国機関投資家資本は、東京・大阪のプライム物件におけるキャップレートの床を支える構造的な買い手である。外国人買い手の引受計算は次の通りである。

| 構成要素 | 外国人買い手の読み方 |
|---|---|
| 引受上のリスクフリー | 資本の出所によってはJGBではなく、米国債／欧州ソブリンを参照することが多い。 |
| FX | 円安はUSD／EUR建ての参入価格を低下させ、許容可能に見えるキャップレートを引き下げる。 |
| 引受上の成長仮定 | 東京／大阪プライムの実質賃料成長が、キャップレートを上回るIRR上乗せを支える。 |
| 要求IRR | プライムではアンレバードIRR8 - 12%が多く、キャップレート、賃料成長、レバレッジで達成する。 |
| トータルリターンの論理 | キャップレートは一要素にすぎない。賃料成長と出口キャップの仮定が総リターンを決める。 |

この外国人買い手の論理により、国内金利が正常化しても、ヘッドラインのキャップレートが圧縮されたままでいられる理由が説明できる。[[real-estate-finance/real-estate-private-credit-japan|private credit]]や直接エクイティルートを通じた外国GPの展開も、買い手側の厚みに寄与する。上場J-REITにおける外国人保有パターンも、[[real-estate-finance/j-reit-foreign-investor-ownership]]で見る通りこの動態を示している。

## キャップレート構成要素の分解

キャップレート水準を考える際の一般的な分解は次の通りである。

```
キャップレート = リスクフリー参照レート + リスクプレミアム - 期待NOI成長率
```

| 構成要素 | 2026 の読み方（東京プライムCBDオフィス、例示） |
|---|---|
| リスクフリー参照（10年JGB） | およそ1.0 - 1.5%のクラス。 |
| リスクプレミアム（資産、立地、テナント） | プライムCBDオフィスではおよそ2.5 - 3.0%のクラス。 |
| 期待NOI成長控除 | 小幅な実質賃料成長を仮定し、およそ0.5 - 1.0%のクラス。 |
| 含意されるキャップレート | およそ3.0 - 3.5%。 |

この分解は例示にすぎない。実際に市場を均衡させるキャップレートは、より広い限界買い手の引受フレームによって決まる。正確な10年JGB水準は、[[money-market/japan-money-market]]およびBoJデータページで確認できる。

## J-REIT NAVにおけるキャップレート感応度

キャップレートの変動は、J-REITのNAVにレバレッジを伴う影響を与える。その理由は次の通りである。

- エクイティが相当量のシニア債務の下位にある。
- 25 - 50bpのキャップレート拡大でも、NAVをミッドからハイシングル桁パーセント動かし得る。
- 上場J-REITの価格対NAV倍率は、鑑定NAVの公表より速く調整する。

この感応度は、上場J-REIT構造についての[[real-estate-finance/j-reit-market-overview]]、および取引価格対NAVギャップと重なって効く遅行鑑定効果についての[[real-estate-finance/japan-real-estate-appraisal-methodology|appraisal methodology]]とあわせて読む。

## 資産クラス別キャップレート・ドライバー（2026 の読み方）

| 資産クラス | 圧縮／拡大ドライバー |
|---|---|
| オフィス（プライムCBD） | インバウンド外国人買い手需要と、ハイブリッドワーク後に残る空室懸念の対立。 |
| 物流 | EC需要の成熟と新規供給パイプライン。 |
| 住宅 | 東京23 区への人口集中と、賃料規制／規制変更。 |
| 商業（プライム都市型） | インバウンド観光回復と、構造的な小売フォーマット変化。 |
| 商業（郊外） | 核テナントの信用力とEC代替。 |
| ホスピタリティ | RevPAR／ADR回復と新規供給の吸収。 |
| データセンター／ニッチ物流 | 伝統的なキャップレートレンジに対するプレミアムを伴う新資産クラスの形成。 |

データセンター、ライフサイエンス、コールドチェーン物流などの専門／ニッチ資産クラスは、主要4カテゴリとは異なる独自のキャップレート論理を持つ。
