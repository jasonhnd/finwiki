# 日本の銀行による商業用不動産融資

## 要約

日本の商業用不動産（CRE）融資は、複数レイヤーからなる制度である。メガバンクは大規模プロジェクトファイナンスと J-REIT スポンサー向けローンの中核を担う。信託銀行は、バランスシート貸出を不動産信託機能、鑑定、資産管理と接続する。地域銀行は中核的な純金利マージンの圧縮を補うため CRE 集中を拡大しており、これは繰り返し [[banking/INDEX|banking supervisor]] の懸念となっている。審査実務は LTV 上限と DSCR 目標を中心に据え、2008  後の監督強化、パンデミック後の集中モニタリング、NIRP 後の金利正常化ストレスが、いずれも [[banking/INDEX|FSR / FSA]] のコメントに見える。このページは経路とリンクを示すものであり、具体的なエクスポージャー数値や審査条件は、使用前にメガバンク IR と BoJ Financial System Report で確認する必要がある。

## ウィキ上の位置づけ

この項目は [[real-estate-finance/INDEX]] 配下に置かれ、[[banking/INDEX]] 上の銀行側カウンターパーティへ接続する。上場 J-REIT ユニバースへのスポンサー側銀行貸出は [[real-estate-finance/j-reit-market-overview]]、ノンバンクのプライベートクレジット代替は [[real-estate-finance/real-estate-private-credit-japan]]、LTV / DSCR 比率を動かす評価背景は [[real-estate-finance/real-estate-cap-rate-compression-2026]]、CRE 審査で用いられる鑑定の下限は [[real-estate-finance/japan-real-estate-appraisal-methodology]] と照合する。メガバンク側のアンカーは [[megabanks/mufg-bank]]、[[megabanks/sumitomo-mitsui-banking-corp]]、[[megabanks/mizuho-bank]]、信託側のアンカーは [[trust-banks/sumitomo-mitsui-trust]] と [[trust-banks/mitsubishi-ufj-trust-bank]]、政策金融のアンカーは [[financial-regulators/dbj]] である。地域銀行の集中要因は [[banking/regional-bank-consolidation-pattern]] に接続する。保険会社の競合 / 協調貸し手マップは [[insurance/japan-life-insurance-alm-overview]] にある。資金調達コストの背景は [[money-market/japan-money-market]] にある。

## 銀行側の構造マップ

| 貸し手クラス | 機能的役割 | CRE での位置づけ |
|---|---|---|
| メガバンク（[[megabanks/mufg-bank]]、[[megabanks/sumitomo-mitsui-banking-corp]]、[[megabanks/mizuho-bank]]） | 大規模バランスシート、シンジケート型プロジェクトファイナンス、J-REIT スポンサー貸出、海外 CRE 部門。 | 東京 / 大阪のプライムオフィス、物流、ホスピタリティ、大型複合用途の中核引受行。 |
| 信託銀行（[[trust-banks/sumitomo-mitsui-trust]]、[[trust-banks/mitsubishi-ufj-trust-bank]]） | 信託勘定貸出に加え、不動産仲介、鑑定、資産管理を担う。 | バランスシート貸出、不動産信託スキーム（TMK / GK-TK）、年金 / 機関投資家向け配分の橋渡し。 |
| 地域銀行 | 地元中小企業信用、不動産担保付きタームローン、アパートローンや東京 CRE 参加の拡大。 | CRE が貸出ポートフォリオに占める比率を上げ、国内 NIM 圧縮を補う局面で集中リスクが生じる。 |
| 政策 / 開発銀行（[[financial-regulators/dbj]]） | 政策連動のプロジェクトファイナンス、インフラ、都市再開発の協調融資。 | 政策整合性を持つ大規模都市再開発・物流案件の協調貸し手 / 主幹事。 |
| 外国銀行 | クロスボーダー CRE ファイナンス、J-REIT シニアトランシェ、メザニン、ウェアハウスファシリティ。 | ニッチ。多くの場合、[[real-estate-finance/real-estate-private-credit-japan]] で扱う外国プライベートクレジット GP と組み合わされる。 |

## プロジェクトファイナンスとコーポレートファイナンス

| 観点 | プロジェクトファイナンス | コーポレートファイナンス |
|---|---|---|
| 借入人 | 定義された資産 / ポートフォリオを保有する特別目的ビークル（TMK、GK-TK、または LLP）。 | 事業会社のバランスシート。 |
| リコース | スポンサー支援レターや完工保証を伴い、プロジェクトのキャッシュフローと資産に限定。 | すべての無担保資産を含むフル・コーポレートリコース。 |
| 審査指標 | LTV、DSCR、ICR、デットイールド。 | 企業レバレッジ（純有利子負債 / EBITDA）、インタレストカバレッジ、バランスシートの質。 |
| 期間 | 5 - 10  年が典型で、バレット / 部分償却。 | リボルバー、ターム、または債券型で、リボルバーは短めのことが多い。 |
| 価格 | 参照金利上乗せのシニアマージン。企業向けより高いことが多く、劣後トランシェはさらに広い。 | 企業信用スプレッド。プライム企業ではよりタイト。 |
| 用途 | 単一資産取得、J-REIT スポンサーのウェアハウス、大規模再開発、物流ポートフォリオ。 | 上場デベロッパー / REIT の企業ファシリティ、運転資金、債券スタックの借換。 |

CRE プロジェクトファイナンスは、J-REIT スポンサー貸出、[[real-estate-finance/real-estate-private-credit-japan|private-credit-co-invested]] 案件、大規模再開発で中心的である。上場デベロッパー / REIT のコーポレートファイナンスは債券・ローン側に寄っており、スポンサー LBO / 非公開化ロジックが関係する場合は [[finance/japan-private-equity-fund-structure-matrix]] と照合する。

## LTV 審査レンジ

| 資産クラス | シニア LTV クラス | 総 LTV（シニア + メザニン） | レンジの理由 |
|---|---|---|---|
| 東京プライムオフィス（CBD 中核） | 55 - 65% | 70 - 75% | 低いキャップレート、厚い流動性、外国買い手の支援。シニア貸し手はやや高い LTV を許容する。 |
| 東京 / 大阪物流 | 55 - 65% | 70 - 75% | 長期 WAULT の機関投資家向けテナント、EC 主導の需要、安定したキャッシュフロー。 |
| 東京住宅（賃貸） | 60 - 70% | 75 - 80% | 長期の稼働安定性、分散したテナント基盤、低いキャッシュフロー変動性。 |
| 大阪 / 地方都市オフィス | 50 - 60% | 65 - 70% | 買い手層が薄く、キャップレートが高く、テナント集中リスクがある。 |
| ホスピタリティ（ホテル、旅館） | 45 - 55% | 60 - 70% | 運営キャッシュフローの変動性、RevPAR 感応度、COVID 期の記憶。 |
| 商業施設（都市部プライム） | 50 - 60% | 65 - 70% | テナントミックスリスクと賃料改定サイクル。 |

これらはクラス記述にすぎない。実際の審査 LTV は、スポンサー、資産品質、賃貸借満期、[[real-estate-finance/real-estate-cap-rate-compression-2026]] に基づく現在のキャップレート環境によって異なる。

## DSCR と ICR の下限

| 指標 | 典型的な下限 | 読み方 |
|---|---|---|
| DSCR（シニア） | 1.30 - 1.50x | 運営キャッシュフローとシニア債務サービスの比較。高変動資産では厳しくなる。 |
| DSCR（総額） | 1.10 - 1.25x | 運営キャッシュフローと、シニアにメザニン / 優先エクイティサービスを加えたものの比較。 |
| ICR | 2.0 - 3.0x | 運営キャッシュフローと利払いのみの比較。バレット構造案件で重要。 |
| デットイールド | 8 - 10% | NOI / 総債務。キャップレートから独立した貸し手側ストレス指標。 |

DSCR の下限は金利正常化をストレステストする。NIRP 解除後、参照金利はマイナス / ゼロ圏から上昇しており、仮想的な +100  から +200 bp のショック下で DSCR を検証することが、[[banking/INDEX|FSR]] のコメントにあるとおり標準的なコベナンツレビューになっている。資金調達コストの参照経路は [[money-market/japan-money-market]] とあわせて読む。

## 2008  後の監督強化

2008 - 2009  サイクルは、三つの構造問題を露出させた。

1. 不動産貸出が慎重な閾値を超えて貸出ポートフォリオに占める地域銀行の CRE 集中リスク。
2. キャップレート拡大に伴う不動産担保の時価変動。
3. ワークアウトを複雑にするクロス担保化と複雑な GK-TK / TMK 構造。

2009 - 2015 （BoJ FSR、FSA 検査重点）にかけての監督対応は、次を引き締めた。

| 領域 | 引き締め |
|---|---|
| 集中限度 | 総貸出に占める CRE 比率に関する内部限度設定への期待。 |
| 鑑定規律 | 保守的な鑑定下限と定期的な再評価要件。 |
| ストレステスト | キャップレート拡大、空室ショック、金利ショックに対する銀行内部ストレスシナリオ。 |
| プロジェクトファイナンス・コベナンツ | LTV / DSCR トリガーとキャッシュトラップ機構の厳格化。 |
| メザニン / 優先エクイティ開示 | ノンバンク・トランシェを含む総レバレッジの可視性向上。 |

これらは現在のサイクルに持ち越されたベースラインの期待であり、[[megabanks/mufg-bank]]、[[megabanks/sumitomo-mitsui-banking-corp]]、[[megabanks/mizuho-bank]] のメガバンク IR リスク開示にも見える。

## パンデミック後の CRE 集中

COVID 期のストレスは次に集中した。

| 資産クラス | ストレスのパターン |
|---|---|
| ホスピタリティ | 渡航制限中のキャッシュフロー崩壊。監督上の猶予とリストラクチャリングの波。 |
| 商業施設 | テナント賃料のリストラクチャリング、来店客数回復のばらつき。 |
| オフィス（地方） | ハイブリッドワーク後の残存需要不確実性。準プライムオフィスのキャップレート拡大。 |
| ホスピタリティ回復 | 2023  からのインバウンド観光回復が急回復を支え、プライムホテルのキャップレートは再び引き締まった。 |

地域銀行では、次の要因により集中モニタリングが強化された。

- 国内 NIM 圧縮が CRE 拡大を促した。
- 地域銀行による東京 / 県外 CRE 参加が監督上の注目を高めた。
- アパートローン・ポートフォリオには、2015 - 2018  年組成に由来する過去の審査脆弱性が残った。

これらの構造圧力に部分的に駆動される地域銀行再編は、[[banking/regional-bank-consolidation-pattern]] にマッピングされている。

## メガバンク CRE エクスポージャーの形

メガバンクの IR 開示は、CRE エクスポージャーを「不動産業向け貸出」「不動産ノンリコースローン」「不動産関連信用」などのカテゴリーで公表する。通常の構成は次のとおりである。

| カテゴリー | 含まれるもの |
|---|---|
| 不動産業向け貸出 | 上場デベロッパー、不動産事業者、J-REIT スポンサーへの貸出。 |
| 不動産ノンリコースローン | 特定資産またはポートフォリオを担保とするプロジェクトファイナンス / SPV 貸出。 |
| アパートローン | 個人 / 中小企業のアパート投資家向けローン。 |
| 建設業向け貸出 | 不動産セクターへのエクスポージャーを持つ建設業者への貸出。 |
| 海外 CRE | 外国支店の CRE エクスポージャー（米国 / EU / アジアのオフィス、物流、住宅）。 |

三つのメガバンクは、意味のあるほど異なる構成を持つ。

- [[megabanks/mufg-bank]] は、米国 / EU 支店網を通じた大きな海外 CRE と、国内不動産スポンサー向け大口ポートフォリオを持つ。
- [[megabanks/sumitomo-mitsui-banking-corp]] は、焦点を絞った国内 CRE ポートフォリオと選別的な海外プロジェクトファイナンスを運営する。
- [[megabanks/mizuho-bank]] は、J-REIT や上場デベロッパーとの長い関係を背景に、国内 / 海外を混合したプロファイルを持つ。

信託銀行アンカーである [[trust-banks/sumitomo-mitsui-trust]] と [[trust-banks/mitsubishi-ufj-trust-bank]] は、信託銀行が受益権構造を保有し、貸出と並んで資産管理を提供する不動産信託の次元を加える。

正確な CRE エクスポージャー数値と集中比率は四半期開示ごとに変わるため、使用前に該当するメガバンク IR ページで確認する。

## J-REIT スポンサー貸出

J-REIT スポンサーおよび個別 J-REIT ビークルへの銀行貸出は、中核的な CRE 事業ラインである。典型的な構造は次のとおり。

| 構造 | 説明 |
|---|---|
| スポンサー支援ローン | ドロップダウン前に取得資産をウェアハウスする J-REIT スポンサーへの直接貸出。 |
| J-REIT レベルのシニアローン | 上場 J-REIT エンティティでのタームローン / 債券購入。 |
| ブリッジファシリティ | エクイティ調達または借換を待つ取得資金の短期ファイナンス。 |
| ウェアハウスファシリティ | スポンサーバランスシート上の物件取得 / 売却に使うリボルビング信用。 |

J-REIT ビークルへの銀行貸出は、クロスボーダー所有の次元について [[real-estate-finance/j-reit-market-overview|J-REIT market overview]] と [[real-estate-finance/j-reit-foreign-investor-ownership]] と照合して読む。J-REIT の資金調達構成には、銀行借入の代替として [[real-estate-finance/j-reit-dividend-yield-vs-jgb-spread|bond / equity issuance]] も含まれる。

## NIRP 後の金利正常化ストレス

BoJ の NIRP 終了と YCC 後の正常化経路は、次を引き上げた。

- シニア変動金利 CRE ローンの債務サービスコスト。
- 借換時の旧来バレット構造案件の DSCR ストレス。
- リスクフリーレート上昇に伴うキャップレート下限圧力（[[real-estate-finance/real-estate-cap-rate-compression-2026]] で扱う）。
- 銀行が新規オリジネーションのスプレッドを調整することによる競争的な再価格付け。

2025 - 2026  年組成の CRE ローンの審査基準には、より高い参照金利前提、より保守的な借換リスクモデル、改訂された DSCR 下限が組み込まれている。資金調達コスト参照の詳細は [[money-market/japan-money-market]]、生命保険会社の競合バランスシートにおける並行的な ALM 再価格付けロジックは [[insurance/japan-life-insurance-alm-overview]] とあわせて読む。
