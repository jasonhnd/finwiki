# Nippon Prologis REIT (3283)

## 要約

**Nippon Prologis REIT, Inc.（NPR、TSE J-REIT 3283）**は、**Prologisがスポンサーを務める現代型物流J-REIT**である。米国本社の**Prologis, Inc.**（NYSE: PLD）は、世界最大の上場物流REITであり、日本の現代型物流プラットフォームの二大勢力の一つである（もう一方は[[real-estate-finance/glp-j-reit-3281|GLP J-REIT]]）。GLP J-REITの直後、2013 年2月に上場したNPRは、以後の外資系スポンサーJ-REITのテンプレートとなる正式な**スポンサーサポート契約**構造を確立した。スポンサーの日本開発パイプラインへの優先供給権、スポンサーによる投資主持分サポート、稼働中物流施設の資産入替権を組み合わせる構造である。

NPRのポートフォリオは、**AAAグレードの現代型物流施設**に集中する。これはPrologis Japanがスポンサーのバランスシート上で開発した、**首都圏、大阪、愛知 / 名古屋の物流サブマーケット**にある高仕様のマルチテナント倉庫である。AAAグレードとは、広い床面、多層ランプアクセス、マルチテナント型クロスドック、高い天井高、スプリンクラー、BCP対応設計（地震・洪水対策）、テナント向けアメニティといった施設仕様を指す。NPRは、外国人投資家が日本物流へ配分する際の**ベンチマーク銘柄**であり、AAAグレードのテナント基盤をめぐって[[real-estate-finance/glp-j-reit-3281|GLP J-REIT]]と正面から競合する。

## 3. スポンサーサポート契約の構造

Prologis-NPRのスポンサーサポート契約は、J-REIT市場で最も明確に開示された正式なスポンサー・パイプライン機構の一つである。主要構成要素は次のとおり。

| 構成要素 | 内容 |
|---|---|
| 優先交渉権 | NPRは、Prologisが日本で開発し安定稼働した物件を取得するための優先交渉権を持つ |
| 情報権 | NPRはスポンサーのパイプライン開発進捗に関する情報を受け取る |
| ウェアハウジング支援 | スポンサーはNPRの取得時期まで安定稼働資産を一時保有できる |
| 資産入替協力 | スポンサーはNPRのポートフォリオ更新戦略に協力する |
| スポンサー投資主持分 | Prologisは通常数％の意味ある投資主持分を維持し、スポンサーと投資主の利害をそろえる |
| AM会社協力 | 資産運用会社（Prologis REIT Management K.K.）はPrologis Japanの人員と支援を受ける |
| 関連当事者取引ガバナンス | スポンサーからの取得には第三者鑑定、独立委員会承認、投資主開示が必要 |

この正式なスポンサーサポート契約テンプレートは、他の外資系スポンサーJ-REITや、スポンサー関係を正式化しようとする国内スポンサーJ-REITにより、修正を加えながら再利用されてきた。

## 7. 現代型物流セグメントを外資スポンサーが支える理由

日本の現代型物流J-REITセグメントは、**外資スポンサーが支配的**である点で特異である。Prologis（米国）とGLP（シンガポール）がこのセグメントを支え、国内デベロッパー・スポンサー（三井不動産、三菱地所、野村不動産）は、オフィスや住宅と比べると現代型物流での役割は小さい。構造的理由は次のとおり。

| 要因 | 読み方 |
|---|---|
| 現代型物流仕様の専門性 | PrologisとGLPは、日本のデベロッパーが同等のプラットフォームを構築する前に、グローバル規模でAAAグレード現代型物流の専門性を蓄積していた |
| 土地取得テンポ | 外資スポンサーは2010年代を通じて、主要サブマーケットの物流用途地域の土地を積極的に確保した |
| 資本効率 | 外資スポンサーのグローバルLP資本により、国内バランスシート金融だけの場合より速い資本展開が可能だった |
| テナント関係 | 国際物流テナント（Amazon、グローバル3PLなど）は、世界的に知る相手方を好む |
| 妥当な反論 | 国内デベロッパーも、総合型J-REIT内の物流スリーブなど、物流焦点のJ-REIT参入で対応している |
