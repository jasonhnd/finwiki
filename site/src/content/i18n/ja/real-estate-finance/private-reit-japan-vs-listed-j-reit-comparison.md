# 日本の私募リートと上場 J-REIT の比較

## 要約

Private REIT (私募リート / 非上場REIT) と上場 [[real-estate-finance/j-reit-market-overview|J-REIT]] は、投資信託法上は構造的に同じ種類の投資法人である。しかし、上場の有無は実務上の使われ方をほぼすべて変える。上場 J-REIT は JPX で継続的に取引され、継続開示義務と幅広い投資家基盤を持つ。私募リートは機関投資家と年金投資家へ私募で販売され、市場取引ではなく構造化されたオープンエンド型のスケジュールで償還し、連続的な市場価格ではなく四半期ごとの NAV ベース価格を提供する。

私募リートは、[[insurance/INDEX|life insurers]]、[[insurance/INDEX|corporate pension plans]]、地域銀行、その他の機関投資家プールにとって、上場 REIT の価格変動を避けながら鑑定評価ベースの不動産エクスポージャーを得る安定的な配分経路に成長してきた。トレードオフは、外部観察者にとって透明性が低い一方で、長期負債や年金配分方針にはより適合しやすい点である。ARES (Association for Real Estate Securitization) は J-REIT と私募リートの双方について業界枠組みを維持し、会員統計を公表している。上場対私募のルーティング層として、このページは [[real-estate-finance/j-reit-market-overview|J-REIT market overview]] の後に使う。

## ウィキ上の位置づけ

この比較は、上場 J-REIT のルーティングページである [[real-estate-finance/j-reit-market-overview|J-REIT market overview]]、[[real-estate-finance/j-reit-vs-us-reit-governance-comparison|US/JP governance comparison]]、[[real-estate-finance/j-reit-sponsor-structure-conflict|sponsor structure and conflict of interest]]、[[real-estate-finance/top-10-j-reit-overview-matrix|top 10 J-REIT overview matrix]] の後に使う。ここから、生命保険会社の配分文脈は [[insurance/INDEX|insurance domain]]、金融レイヤーは [[banking/INDEX|banking domain]]、上場デベロッパー・スポンサー文脈は [[finance/INDEX|finance domain]] へ進む。私募不動産ファンドの内部にしばしば置かれる証券化配管については、[[real-estate-finance/INDEX|real-estate-finance index]] で予定されている GK-TK SPV クラスターへ進む。

## 1. 横並び比較

| 軸 | 上場 J-REIT | Private REIT (私募リート) |
|---|---|---|
| 法的形態 | 投資信託法上の投資法人 | 投資信託法上の投資法人 |
| 上場市場 | JPX REIT セグメント | 非上場。私募 |
| 取引 | JPX で継続的に市場取引 | NAV に基づく定期申込 / 償還枠 (多くは四半期) |
| 価格参照 | 市場価格 (連続的) | 一口あたり NAV (鑑定評価に基づく) |
| 開示 | 継続的な公衆開示 (有価証券報告書)、IR、規制提出 | 適格投資家向け開示のみ。ARES 業界統計がセグメントを集計 |
| 投資家基盤 | 個人 + 機関 + 海外 | 主に機関、年金、生命保険 |
| 流動性 | 高い (上場市場) | 限定的。償還枠ベース |
| スポンサー構造 | 同じ外部運用モデル + スポンサー | 同じ外部運用モデル + スポンサー |
| 税務処理 | 税務パススルー (分配支払要件を満たす場合) | 税務パススルー (分配支払要件を満たす場合) |
| 規制当局 | FSA, JPX | FSA |
| 業界団体 | ARES | ARES |
| ファンド数 | 上場銘柄は約 60  | 数十本。時期により変動するため ARES で追跡 |

## 2. オープンエンド償還メカニクス

日本の私募リートは通常、連続取引ではなく、構造化された申込 / 償還枠を持つオープンエンド型ファンドとして運営される。仕組みは運用会社ごとに異なるが、一般的なパターンは以下である。

| ステップ | パターン |
|---|---|
| 申込 / 償還枠 | 一口あたり NAV に基づき定期的 (多くは四半期) |
| NAV 算定 | 独立鑑定評価ベースで、[[real-estate-finance/INDEX\|JREI methodology]] に基づく |
| 償還上限 | 強制的な資産売却からファンドを守るため、枠ごとにソフト / ハードキャップを設定 |
| 停止条項 | ストレス市場環境で償還を停止する権利 |
| 分配方針 | 賃貸 NOI からフィー、金融費用、資本的支出リザーブを差し引いた定期分配 |
| 報告 | 適格投資家向けの四半期 NAV と詳細な資産別報告 |

この償還構造により、私募リートは年金と生命保険の負債に適合しやすい。すなわち、長期資本、控えめな流動性ニーズ、市場価格変動ではなく鑑定評価ベースの平準化を受け入れる姿勢に合う。同じ構造は、私募リートが日次取引で吸収できる上場 J-REIT と同種の資金流入 / 流出ショックを受け止められないことも意味する。

## 3. フィー構造のトレードオフ

| フィー項目 | 上場 J-REIT | 私募リート |
|---|---|---|
| 資産運用報酬 | 取得報酬 + 資産連動 + 場合により成果報酬 | 資産連動 + 取得報酬 + 譲渡報酬 |
| マーケティング / 販売報酬 | なし (継続的な市場流動性) | コミットメント時の一回限りの販売報酬 |
| 保管 / 受託報酬 | 信託銀行報酬 | 信託銀行報酬 |
| 監査 + コンプライアンス | 継続開示コンプライアンス費用 | 定期 NAV / 監査費用 |
| 流動性プレミアム | 上場市場が流動性を無料で提供 | 投資家が非流動性ディスカウントを受け入れる |

総フィー負担は運用会社により異なる。投資家の関心は歴史的に、見出し上のフィー水準だけでなく、償還メカニクスとの整合性と鑑定評価の独立性に置かれてきた。

## 4. 年金・機関投資家の配分用途

私募リートは、上場 J-REIT では十分に満たせない特定の投資家ニーズに応える。

| 投資家タイプ | 私募リートを使う理由 |
|---|---|
| 企業年金 | 長期負債との適合、鑑定評価で平準化されたリターン、上場 J-REIT より低い表面上のボラティリティ |
| 生命保険会社 ([[insurance/INDEX\|insurance domain]]) | 不動産スリーブ内の分散。鑑定評価に基づく NAV が責任準備金の枠組みに合う |
| 地域銀行 ([[banking/INDEX\|banking domain]]) | 銀行の直接 CRE 融資エクスポージャー外の不動産スリーブ |
| 公的年金・年金ファンドオブファンズ | 長期の不動産配分方針に合うビークル |
| スポンサー関連投資家 | スポンサーの不動産戦略と整合するグループ財務配分 |

私募リートの償還メカニクスと年金負債デュレーションが適合することが、上場 J-REIT が成長した後もこのセグメントが意味を持ち続ける構造的理由である。

## 5. 透明性のトレードオフ

| 公開観察者が見られるもの | 上場 J-REIT | 私募リート |
|---|---|---|
| 市場規模合計 | あり (JPX, ARES) | 合計のみ (ARES 私募リート統計) |
| ファンド別 AUM | あり | 合計のみ |
| ファンド別 DPU 利回り | あり | 合計のみ |
| 資産別詳細 | あり (IR 資料) | なし (適格投資家向け開示のみ) |
| スポンサー持分 | あり | なし (非公開) |
| LTV | あり | 合計 / レンジのみ |
| 海外投資家比率 | あり (JPX 投資部門別売買) | 該当なし |

透明性のトレードオフが中心的な緊張点である。上場 J-REIT 分析は公開情報から可能だが、適格投資家の外から私募リートを分析する場合、ARES 業界統計とスポンサー側の公開開示に限られる。したがってこのページは、集計 / セグメント水準にとどめる。

## 6. ARES 会員エコシステム

ARES (Association for Real Estate Securitization, 不動産証券化協会) は、上場 J-REIT、私募リート、GK-TK 私募不動産ファンドを対象とする業界団体である。ARES の会員企業には以下が含まれる。

- スポンサー (デベロッパー、金融グループ、外資系資産運用会社)。
- J-REIT と私募リートを運用する資産運用会社。
- 受託者として機能する信託銀行 ([[banking/trust-bank-custody-operating-comparison|trust bank custody operating comparison]])。
- 鑑定評価会社 (JREI 系およびその他)。
- 証券化業界の法律 / 会計 / アドバイザリー会社。

ARES の J-REIT と私募リートに関する集計統計は、適格投資家向け開示領域に入らずにセグメント動向を分析するうえで有用な参照点である。

## 7. GK-TK 私募不動産ファンドとの関係

私募リートは、日本の私募不動産金融ビークルという点では GK-TK SPV 構造と重なるが、両者は別物である。

| 特徴 | 私募リート | GK-TK SPV |
|---|---|---|
| 法的形態 | 投資信託法上の投資法人 | 合同会社 (GK) + 匿名組合 (TK) の匿名組合 SPV |
| オープンエンド性 | オープンエンド (定期申込 / 償還) | クローズドエンドの単一資産または単一戦略 SPV |
| 投資家基盤 | 複数の機関 / 年金 | より少数の TK 匿名組合投資家 |
| 税務処理 | 分配支払要件を満たせば税務パススルー | TK レベルの税務処理。TK 投資家へパススルー |
| 用途 | 長期の鑑定評価ベース不動産スリーブ | 単一案件または単一戦略の証券化 (単一ビル、ポートフォリオ切出し、ブリッジファンド) |
| 開示 | 適格投資家向け開示 | TK 投資家への私的契約開示 |

両形態は日本の不動産証券化業界で併存している。ARES は双方を対象とする。GK-TK 構造はより案件固有で、[[real-estate-finance/INDEX|CMBS-style securitization]] と J-REIT に流入するブリッジファンド・パイプラインの典型的なビークルである。

## 8. 重要な理由

- 投資家配分: 上場 J-REIT だけを理解しても、JPX に現れない年金 / 生命保険 / 地域銀行の私募リート経路を見落とす。
- 市場サイクルの読み方: 私募リート NAV は上場 J-REIT 価格より遅く動くため、上場 REIT のインプライド利回りと私募リート NAV 利回りの大きな乖離は不動産サイクルのシグナルである。
- スポンサー戦略: 多くのスポンサーは上場 J-REIT と私募リートを並行して運用しており、[[real-estate-finance/j-reit-sponsor-structure-conflict|sponsor conflict]] の議論に加えて追加的な配分問題を生む。
- 海外投資家の読み方: 海外投資家は通常、私募リートではなく上場 J-REIT にアクセスするため、海外フローデータはより大きな基礎市場のうち上場部分だけを映す窓である。
