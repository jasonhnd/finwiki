# Japan Real Estate Investment（JRE、J-REIT 8952）

## 要約

**Japan Real Estate Investment Corporation（JRE、TSE J-REIT 8952）**は、**Mitsubishi Estate がスポンサーとなる旗艦オフィス J-REIT**であり、[[real-estate-finance/nippon-building-fund-j-reit-8951|Nippon Building Fund (NBF, 8951)]]と構造的に対になる存在で、**Mitsubishi Estate の丸の内オフィス地所**への上場 proxy である。JRE は**日本で最初に上場した REIT の一つ**（2001 年9月、NBF と同時）であり、AUM で常に J-REIT の最上位層に位置する。ポートフォリオは、Mitsubishi Estate が一世紀以上にわたり制度的に保有してきた土地バンクの中心である**丸の内、大手町、有楽町**を軸とし、丸の内アンカー外の東京 CBD オフィスビルと、一部の地方中核都市オフィスで補完される。

上場 REIT レイヤーでは、JRE 対 NBF という形で表れる**MEC 対 Mitsui**の競争関係が決定的な特徴である。両者はいずれも保守的な LTV（低位から中位の40%台）、AA 格、プレミアム利回り（J-REIT 利回り分布の低位）、海外投資家のベンチマークとなるオフィス J-REIT である。主要な対比は、**スポンサーのパイプライン源泉**（Mitsubishi Estate の丸の内および隣接 CBD と、Mitsui Fudosan の大手町・日本橋・渋谷・新宿のフットプリント）と、**資産特性**（JRE の丸の内アンカーは、丸の内固有のキャップレート動向への最も直接的な上場エクスポージャーを与える。[[real-estate-finance/mitsubishi-estate-financing-model|Mitsubishi Estate financing model]]参照）にある。

## ウィキ上の位置づけ

この項目は[[real-estate-finance/INDEX|real-estate-finance index]]の下に置かれる、**Mitsubishi Estate オフィス J-REIT のアンカー**である。スポンサー側の丸の内地所と資産リサイクルのレイヤーは[[real-estate-finance/mitsubishi-estate-financing-model|Mitsubishi Estate financing model]]、構造的な双子 / Mitsui との競争対比は[[real-estate-finance/nippon-building-fund-j-reit-8951|Nippon Building Fund (NBF, 8951)]]、J-REIT 横断の位置づけは[[real-estate-finance/top-10-j-reit-overview-matrix|Top 10 J-REIT overview matrix]]と合わせて読む。ガバナンスの枠組みは[[real-estate-finance/j-reit-vs-us-reit-governance-comparison|J-REIT vs US REIT governance comparison]]および[[real-estate-finance/j-reit-sponsor-structure-conflict|J-REIT sponsor structure and conflict]]である。分散ポートフォリオとの対比は[[real-estate-finance/nomura-real-estate-master-fund-nmf|Nomura Real Estate Master Fund]]、物流との対比は[[real-estate-finance/glp-j-reit-3281|GLP J-REIT (3281)]]および[[real-estate-finance/nippon-prologis-reit-3283|Nippon Prologis REIT (3283)]]を使う。

## 2. ポートフォリオ構成

| 軸 | JRE のパターン |
|---|---|
| 資産クラス | オフィス（圧倒的） |
| 地域集中 | 東京23 区が大きな比重、地方中核都市は小規模 |
| 東京サブマーケットの焦点 | **丸の内 / 大手町 / 有楽町**（Mitsubishi Estate の中核テリトリー）に加え、その他の CBD サブマーケット |
| 物件規模 | 大型マルチテナントオフィスビル。丸の内地所内または隣接するものが多い |
| 取得パイプライン源泉 | 主に Mitsubishi Estate のスポンサー・パイプライン（資産リサイクル） |
| 物件築年 | 丸の内エリアの築年が古く安定した資産と、新しい再開発竣工ビルの混合 |

丸の内アンカーは最も特徴的である。丸の内に意味のある固有エクスポージャーを持つ上場 J-REIT は非常に少ない（Mitsubishi Estate が丸の内の支配的土地所有者である）。したがって JRE は、丸の内のキャップレート変動に対する上場 proxy として機能する。

## 3. 資本とレバレッジ

| 項目 | JRE のパターン |
|---|---|
| LTV 方針帯 | 保守的 — 通常は低位から中位の40%台 |
| 債務構成 | 銀行借入（メガバンク + 信託銀行シンジケート。三菱グループとの関係から[[megabanks/mufg-bank|MUFG Bank]]が主導することが多い）と投資法人債 |
| 債券カーブ | AA ゾーンの国内格付によるタイトなスプレッド。NBF と並ぶオフィス J-REIT セグメントのベンチマーク発行体 |
| スポンサーサポート持分 | Mitsubishi Estate がスポンサーサポートとして投資口保有を維持（通常は一桁%） |
| 分配方針 | 半期 DPU。J-REIT の90%パススルー |
| 海外投資家比率 | 高い — グローバル REIT マンデートにおける日本オフィスのベンチマーク銘柄 |

## 4. 上位テナントへのエクスポージャー

| テナント集中指標 | JRE のパターン |
|---|---|
| スポンサーがテナントか | 限定的。Mitsubishi Estate はスポンサーおよびデベロッパーであり、主要な賃料支払テナントではない |
| テナント業種ミックス | 分散。丸の内エクスポージャーにより金融サービスが特に重要（メガバンク本社や金融セクターのテナントは丸の内・大手町エリアに集積） |
| テナント賃貸借構造 | 定期的な賃料改定を伴う標準的な日本の定期建物賃貸借 |

## 5. JRE vs NBF — MEC vs Mitsui 競争の proxy

| 軸 | JRE（Mitsubishi Estate） | NBF（Mitsui Fudosan） |
|---|---|---|
| ティッカー | 8952 | 8951 |
| スポンサー | [[real-estate-finance/mitsubishi-estate-financing-model|Mitsubishi Estate]] | [[real-estate-finance/mitsui-fudosan-financing-model|Mitsui Fudosan]] |
| 上場日 | 2001 年9月（同時最初期上場） | 2001 年9月（同時最初期上場） |
| 東京サブマーケットのアンカー | **丸の内 / 大手町 / 有楽町** | 大手町 / 日本橋 / 渋谷 / 新宿 |
| パイプライン | Mitsubishi Estate の資産リサイクル | Mitsui Fudosan の資産リサイクル |
| AUM 規模 | 最上位層 | 最上位層 |
| LTV | 低位から中位の40%台 | 低位から中位の40%台 |
| DPU 利回り帯 | プレミアム | プレミアム |
| 海外投資家プロファイル | 日本オフィスのベンチマーク | 日本オフィスのベンチマーク |
| 特徴 | 丸の内アンカー（Mitsubishi Estate の制度的土地バンク） | 再開発竣工物件の寄与を伴う、より広い CBD フットプリント |

両銘柄は、投資法人債発行、海外投資家の配分、AUM ランキングをめぐって正面から競合する。保守的な LTV / AA 格 / プレミアム DPU という似たプロファイルは、両スポンサーが適用する並行した構造モデルの結果である（[[real-estate-finance/mitsui-fudosan-financing-model|Mitsui Fudosan financing model]]および[[real-estate-finance/mitsubishi-estate-financing-model|Mitsubishi Estate financing model]]参照）。

## 6. 丸の内エクスポージャーが重要な理由

丸の内は、いくつかの構造的特徴を持つ独立した東京オフィスサブマーケットである。

- メガバンク本社（隣接する大手町の[[megabanks/mufg-bank|MUFG Bank]]、[[megabanks/mizuho-bank|Mizuho Bank]]本社）、大手生命保険会社本社、主要日本企業本社の集積。
- 丸の内アンカー地所周辺における、実質的に単一地主の所有パターン（Mitsubishi Estate が支配的）。
- トロフィー資産としての性格から、より広い東京 CBD よりキャップレートがタイトに振る舞うことが多い。
- 住所のプレステージと集積効果により、テナント維持率が構造的に高い。
- Mitsubishi Estate の丸の内エリア・マスタープランによる再開発パイプラインの見通し。

これらの特徴への JRE のエクスポージャーにより、同社の NAV と DPU は他の上場オフィス J-REIT より丸の内固有のダイナミクスに直接反応しやすい。
