# J-REIT市場概観

## 要約

J-REIT（Japan Real Estate Investment Trust）は、日本の上場不動産市場レイヤーである。運営会社としての不動産会社ではなく、投資信託法上の投資法人として組成される。東証REIT指数は、60 前後の上場J-REIT銘柄を対象とし、合計時価総額は¥15–20 兆円規模、取引は [[finance/japan-listed-financial-groups-investable-universe|JPX REIT segment]] を通じて行われる。日本市場は 2001 年9月に開始され、Nippon Building Fund と Japan Real Estate が最初の2銘柄として上場した。つまり20年以上の運営実績を持つ成熟市場である。

決定的な構造上の特徴は、外部運用モデルと背後にいるデベロッパースポンサーである。このページは [[real-estate-finance/INDEX|real-estate-finance domain]] におけるJ-REITのルーティング面として使い、次段階の詳細は [[real-estate-finance/j-reit-vs-us-reit-governance-comparison|US/JP governance comparison]]、[[real-estate-finance/j-reit-sponsor-structure-conflict|sponsor conflict]]、[[real-estate-finance/top-10-j-reit-overview-matrix|top 10 matrix]]、[[real-estate-finance/private-reit-japan-vs-listed-j-reit-comparison|private vs listed REIT comparison]] へ進む。

## ウィキ上の位置づけ

上場J-REITレイヤーはここから始める。米国REITとのガバナンス対比は [[real-estate-finance/j-reit-vs-us-reit-governance-comparison|governance comparison]]。スポンサー / 資産運用会社の利益相反の詳細は [[real-estate-finance/j-reit-sponsor-structure-conflict|sponsor conflict]]。私募REIT（非上場REIT）と年金配分の比較は [[real-estate-finance/private-reit-japan-vs-listed-j-reit-comparison|private vs listed REIT comparison]]。デベロッパーのバランスシートと上場会社の不動産エクスポージャーは、Mitsui Fudosan、Mitsubishi Estate、Sumitomo Realty に関する [[finance/INDEX|finance domain]] 項目を使う。住宅金融と政策MBSについては [[policy-finance/japan-housing-finance-agency|JHF]] を [[policy-finance/INDEX|policy-finance]] 内で使う。[[banking/INDEX|banking domain]] 企業の商業不動産融資エクスポージャーは同領域索引を使う。

## 1. 制度上の境界

| 項目 | 読み方 |
|---|---|
| 法的形態 | 投資信託法上の投資法人 |
| 資産運用形態 | 登録済み資産運用会社（多くはスポンサー保有）が外部運用 |
| 上場市場 | 東京証券取引所REITセグメント（J-REIT市場） |
| 指数 | 東証REIT指数（価格指数 + トータルリターン版） |
| 初回上場 | Nippon Building Fund、Japan Real Estate（2001-09） |
| 上場銘柄数 | 約 60 （JPX REITセグメントページで公開） |
| 合計時価総額 | 10兆円台半ばのレンジ。JPXで日次公開 |
| 規制当局 | 投資信託法に基づくFSA。利益の約90%を分配すれば税務パススルー |
| FinWiki上のレーン | [[real-estate-finance/INDEX\|real-estate-finance]]（[[finance/INDEX\|finance]] のデベロッパーページではない） |

J-REIT は事業会社ではない。投資法人が信託受益権を通じて資産を保有し、運営を資産運用会社へ委託する。これが、[[real-estate-finance/j-reit-sponsor-structure-conflict|sponsor conflict]] と [[real-estate-finance/j-reit-vs-us-reit-governance-comparison|governance comparison]] で分析されるJ-REIT固有のガバナンスと利益相反の問いを生む構造上のアンカーである。

## 2. 市場構造

| 要素 | 公開上の見え方 |
|---|---|
| 発行体 | 投資口を発行する投資法人 |
| 運営者 | 資産運用会社（通常はスポンサー関連） |
| 受託者 | 原資産の名義を保有する信託銀行（多くは [[trust-banks/mitsubishi-ufj-trust-bank\|Mitsubishi UFJ Trust Bank]]、[[trust-banks/sumitomo-mitsui-trust\|SMTB]]、[[trust-banks/mizuho-trust-bank\|Mizuho Trust]]）— [[banking/trust-bank-custody-operating-comparison\|trust bank custody operating comparison]] を参照 |
| 監査人 | 上場会社監査人 |
| 資産保管会社 / 事務管理者 | 資産保管受託者として機能する信託銀行 |
| 税務処理 | 分配比率テスト（約90%）などの条件を満たせば税務透明 |
| レバレッジ | 銀行借入 + 投資法人債。LTV はスポンサーごとに異なる（一般に 40–50%レンジ、公表） |
| 分配 | 年2回のDPU（1口当たり分配金）が典型パターン |
| 取引 | JPXで継続取引。JASDECのインフラを通じ T+2 決済（[[banking/trust-bank-custody-operating-comparison\|trust bank custody operating comparison]] を参照） |

|---|---|---|
| オフィス | Nippon Building Fund, Japan Real Estate, Daiwa Office Investment, KDX Tokyo REIT, Global One Real Estate, Mori Hills REIT | 歴史的に最大カテゴリ。東京CBDオフィス空室率に敏感。 |
| 住宅 | Advance Residence, Nippon Accommodations Fund | 安定したキャッシュフローだが、賃料成長プロファイルは限定的。 |

| ホテル | Japan Hotel REIT, Hoshino Resorts REIT, Invincible Investment | ホテルREITのキャッシュフローは営業レバレッジが高く、観光循環に左右される。 |
| 物流 | Nippon Prologis REIT, GLP J-REIT, Industrial & Infrastructure Fund, Mitsui-Soko Logistics REIT | Eコマース需要がこのカテゴリの比重を大きく押し上げた。 |
| ヘルスケア | Healthcare & Medical Investment Corp, Nippon Healthcare Investment | 病院 / 高齢者住宅中心。小さめのセグメント。 |
| 総合型 | Nomura Real Estate Master Fund, Orix JREIT, Hulic REIT, United Urban Investment | 複数資産型で、単一物件集中が低い。 |

## 4. スポンサー類型

| スポンサー類型 | 例 | コメント |
|---|---|---|
| デベロッパースポンサー（大手不動産） | Mitsui Fudosan（Nippon Building Fund）、Mitsubishi Estate（Japan Real Estate）、Mori Building（Mori Hills）、Sumitomo Realty、Nomura Real Estate（NMF） | パイプライン主導のスポンサー・モデル。[[real-estate-finance/j-reit-sponsor-structure-conflict\|sponsor conflict]] の構造的な利益相反パターンに最も近い。 |
| 金融スポンサー | Orix（Orix JREIT）、Daiwa House（Daiwa House REIT）、Kenedix（KDX Tokyo REIT、KDX Office、KDX Residential）、Hulic（Hulic REIT） | 金融 / 資産運用と物件供給の組み合わせ。 |
| 外資スポンサー | GLP（GLP J-REIT）、Prologis（Nippon Prologis REIT）、Lasalle、Invesco | 物流とオフィスが厚い。スポンサー・パイプラインは日本国外グループに依存する。 |
| 独立 / 非スポンサー | 小規模で希少 | 専用スポンサー・パイプラインを欠くため、外部取得主導となる。 |

スポンサー類型は、パイプラインの見通し、資産取得ペース、[[real-estate-finance/j-reit-sponsor-structure-conflict|sponsor conflict]] で論じる関連当事者リスクを決める。

## 5. 重要性

- J-REIT は日本の商業不動産に入る公開市場の入口である。評価、レバレッジ、DPU利回りに関して最も透明性の高い面である。
- J-REIT取引における海外投資家比率は歴史的に大きく、原資産が日本国内であっても、J-REITセグメントはグローバルなキャップレート循環と結びつく。
- 上場J-REITのIR開示は、日本のCBDオフィス、物流、商業不動産のファンダメンタルズを知る数少ない公開窓口であり、[[finance/INDEX|finance domain]] の上場デベロッパー開示や [[retail/aeon-group|AEON Group]] / [[retail/seven-and-i-hd|Seven & i HD]] の不動産フットプリントを補完する。
- J-REITのレバレッジは銀行借入 + 社債に基づくため、[[banking/INDEX|banking domain]] の商業不動産融資姿勢とつながる。
- 資産運用会社 / スポンサー / 受託者構造により、J-REIT は実際の投資法人文脈における [[banking/trust-bank-custody-operating-comparison|trust bank custody operating comparison]] の有用なケーススタディとなる。

## 6. 境界事例

- **J-REIT vs 上場デベロッパー**: J-REIT投資法人は収益不動産のみを保有する。一方、Mitsui Fudosan、Mitsubishi Estate、Sumitomo Realty、Daiwa House、Tokyu Fudosan、Hulic、Nomura Real Estate（[[finance/INDEX|finance domain]] 参照）のような上場デベロッパーは、開発、賃貸、仲介、ホテル、REITスポンサー、資産運用など、より広い不動産事業を営む。
- **J-REIT vs 私募REIT（非上場REIT）**: [[real-estate-finance/private-reit-japan-vs-listed-j-reit-comparison|private vs listed REIT comparison]] を参照。私募REITは機関投資家 / 年金向けのオープンエンド商品であり、JPXには上場しない。
- **J-REIT vs GK-TK SPV / 私募ファンド**: GK-TK SPV（合同会社・匿名組合の匿名組合SPV）は単一資産または単一戦略の私募証券化ビークルであり、J-REITではない。
- **東証REIT指数 vs TOPIX**: J-REIT投資口はTOPIXに含まれない。独自のセグメントと東証REIT指数を持つ。
- **DPU利回り vs 配当利回り**: DPU は1口当たり分配金であり、企業配当ではない。この経済的相違は、税務や [[finance/japan-listed-financial-groups-investable-universe|listed-financial / listed-developer dividend yields]] との比較で重要である。
