# みずほ FG (Mizuho FG)

## ウィキ上の位置づけ

この項目 sits under [[megabanks/INDEX|megabanks INDEX]]. あわせて読む と照合して [[megabanks/mufg|三菱 UFJ FG (MUFG)]] のために 比較・対照 文脈 と [[banking/INDEX|banking index]] のために the より広い制度・規制境界.

## 要約

日本第 3 大金融集団・3 大メガバンクの一角（総資産 **283.3 兆円**、純利益 **8854 億円**（過去最高）/ 2025-03 期実績）^[extracted]。2003-01-08 に**第一勧業銀行 + 富士銀行 + 日本興業銀行 3 行統合**を起源とする持株会社。3 行合併由来の構造的複雑性と**2002 / 2011 / 2021 連発の大規模システム障害**が長らくブランド毀損要因。**「One MIZUHO」**戦略でグループ一体運営へ。**楽天証券との資本業務提携**（2022-11-〜）・**PayPay 銀行への戦略出資**・**サイバーエージェントとの合弁（CA Mobile / その他）**で B2C デジタル弱点をパートナー連携で補完。G-SIB Bucket 1。^[extracted]

### 合併経緯・前身

- 旧 **第一勧業銀行 (DKB)**: 1971 第一銀行 + 日本勧業銀行 合併。旧第一銀行は 1873 渋沢栄一創設の日本最古の銀行
- 旧 **富士銀行**: 1948〜（旧 安田銀行 / 安田財閥系）戦後 GHQ 財閥解体で改名
- 旧 **日本興業銀行 (IBJ)**: 1902 設立、特殊銀行として長期信用金融担当 → 戦後普通銀行化、その後**長期信用銀行**として法人金融特化
- **2000-09-29** みずほホールディングス設立（3 行が完全子会社入り、株式移転）^[extracted]
- **2002-04-01** みずほ銀行 + みずほコーポレート銀行 発足（個人 / 中小 vs 大企業 / 国際の機能別分割）^[extracted]
- **2003-01-08** みずほ FG 発足（HD 上位の純粋持株会社化）^[extracted]
- **2013-07-01** みずほ銀行 + みずほコーポレート銀行 → 新「みずほ銀行」**再統合**（機能別分割 11 年で見直し）^[extracted]

## 2. 事業セグメント・マップ

| セグメント | 主要事業者 | 特徴 |

### デジタル戦略

- **「One MIZUHO」戦略**: 3 行カルチャー統合・グループ一体運営の長期テーマ ^[extracted]
- **e-Mizuho**: 個人向けデジタルバンキング ^[extracted]
- **MINORI**（みずほ次期勘定系システム）: 2019 年完全稼働、4500 億円超投資の超大型システム刷新。だが 2021 年連発障害の震源地ともなった ^[extracted]
- **AI / ChatGPT 業務活用**: 2023-06 に国内ほぼ全社員（約 3.5 万人）向けの対話型 AI 利用を開始。日本マイクロソフトと共同開発、資料作成・要約・翻訳・プログラミング等に活用。**メガバンクで初**の全社員業務導入（2023-04 に検討公表）

### 提携・買収戦略

- **楽天証券 資本業務提携**（2022-10〜）★: みずほ証券が楽天証券株 約 49% 取得（後段階増資で約 50% 弱）、約 800 億円規模。**メガ初の外部ネット証券深掘り出資**。リテール顧客チャネルと商品プラットフォーム共通化 ^[extracted]
- **米 Greenhill 買収**（2023-09 完了、約 770 億円 / 5.5 億ドル）: グローバル M&A アドバイザリー独立系を傘下化、邦銀初の独立系投資銀行買収 ^[extracted]
- **第一生命との JV**: アセットマネジメント One（51:49） ^[extracted]
- **PayPay 証券 出資**（旧 One Tap BUY）: みずほ証券が SoftBank と共同で 2020-10 に合弁化、みずほ証券は約 24.8% を出資。2025-02 に PayPay が同社を連結子会社化した後もみずほ証券は協業パートナーとして出資継続
- **サイバーエージェント等との業務提携**: みずほ銀行・サイバーエージェント・フライウィールが 2024-01-26 にマーケティングプラットフォーム事業の戦略的業務提携を締結、2024-10 にデジタルサイネージ広告等の広告事業を開始
- **丸紅との JV**: みずほリース（持分法 23.5%、丸紅 23%）^[extracted]
- **Bain 資本金 との PE ファンド組成**（2025-09 発表）^[extracted]

### 海外戦略

- **米州**: NY 本拠の Mizuho Americas（旧みずほコーポレート銀行 NY 支店）+ Greenhill（独立系 M&A）
- **アジア**: 中国（みずほ銀行中国）/ タイ / インドネシア / シンガポール 等
- **欧州**: ロンドン IBD（旧興銀ロンドン由来）

### B2C ブランディング

- **コーポレートカラー**: みずほブルー（信頼 / 知性）+ 稲穂モチーフのロゴ ^[extracted]

## 4. 規制・政策

- **主管**: 金融庁（FSA）
- **持株会社規制**: 銀行法 第 52 条の 17
- **海外規制**: 米連邦準備制度（FRB / OCC）、ECB / BoE、中国 CBIRC、タイ BoT 等
- **業務改善命令**: **2002 / 2011 / 2021 と 3 度の大規模システム障害で業務改善命令を複数回受領**。2021-11 命令は史上 2 度目の銀行業務改善命令（過去含む）^[extracted]
- **直近政策論点**:
  - 2024〜 日銀政策金利正常化 → 国内金利マージン改善ボーナス（メガ 3 行共通）
  - 2025〜 中計 2024-2026 進捗・ROE 8% 目標達成
  - サステナブルファイナンス・脱炭素融資ポリシー強化
