# 三菱UFJ銀行 (MUFG Bank)

## Wiki 経路

本項目は [[megabanks/INDEX|megabanks INDEX]]（そのドメイン経路）の配下に位置する。ピア／対比（もう一方の 都市銀行 メガバンク operating company）については [[megabanks/sumitomo-mitsui-banking-corp|三井住友銀行 (SMBC)]]、監督／規制の境界については [[financial-regulators/fsa|the FSA]] と対照して読むこと。グループ／親会社の視点は [[megabanks/mufg|MUFG]] にあり、信託の兄弟会社は [[trust-banks/mitsubishi-ufj-trust-bank|三菱UFJ信託銀行]] にある。

## 要点

三菱UFJ銀行 (MUFG Bank, Ltd.) は [[megabanks/mufg|MUFG]] の商業銀行中核法人。FSA の銀行免許リストでは「株式会社三菱UFJ銀行」(都市銀行) として掲載され、銀行コードは 0005。[[trust-banks/mitsubishi-ufj-trust-bank|三菱UFJ信託銀行]]、証券 HD、カード / 消費者金融子会社とは別の bank-license layer を持つ。

## 2. 運営モデル

- MUFG グループの預金受入・貸出・国内/外国為替・決済・海外商業銀行網を担う中心法人。
- アジア商業銀行投資、法人決済、外国銀行代理業務、信託契約代理など、銀行単体ライセンスに紐づく業務が多い。
- bank-license layer は、[[trust-banks/mitsubishi-ufj-trust-bank|三菱UFJ信託銀行]] (trust / asset administration / real estate / pension) と証券子会社の investment-banking / brokerage layer から機能的に分離される。

## 3. スタンドアロン・ページが重要な理由

[[megabanks/mufg|MUFG]] は上場持株会社で、連結決算・資本政策・Morgan Stanley alliance を見る単位。三菱UFJ銀行は銀行法上の免許、預金基盤、法人決済、海外支店、外国銀行代理、信託代理を追う単位。特に stablecoin / Progmat / EPI 周辺では、商業銀行と信託銀行の境界を分けて分析する必要がある。
