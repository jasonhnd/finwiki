# イオングループ (AEON Group)

## 要約

AEON は、金融サービス上の価値が**日常生活の流通網**から生まれる小売グループである。スーパーマーケット、モール、クレジットカード、WAON 電子マネー、AEON Pay、保険カウンター、[[JapanFG/aeon-bank|Aeon Bank / AEON Financial Service]] を組み合わせている。FinWiki において、AEON は日本の**小売から金融へつながるオペレーティングシステム**の最も強い例である。

金融上の問いは「AEON は銀行を所有しているか」ではない。**小売頻度がどのように金融商品の流通と信用データを生むか**である。

## 構造

| イオンフィナンシャルサービス | AEON 小売に根ざした上場総合金融サービスグループ | クレジットカード、銀行代理、プリペイド、アジア消費者金融、保険 / 決済隣接領域 |
| イオン銀行 | AFS 配下のリテール銀行 | 預金、住宅ローン、ATM、投資信託 / 保険販売、店舗連動型銀行業務 |
| AEON Card | クレジットカード商品ライン | カード利用、リボ / 分割、WAON POINT と AEON 小売特典 |

## ビジネスモデル

AEON の金融モデルには3つの優位性がある。

1. **純オンライン型 CAC に依存しない流通**: 店舗、モール、スーパーマーケットのカウンターが獲得コストを下げる。
2. **反復的な少額データ**: 食料品や日常の買い物は、住宅ローンや証券より高頻度の行動データを生む。
3. **閉ループ特典**: AEON Card、WAON、WAON POINT、AEON Pay は、グループ店舗の利益率とキャンペーンに合わせて調整できる。

## 戦略的な読み方

AEON が JapanFG にとって重要なのは、金融サービスが**小売の運営リズム**に埋め込まれ得ることを示すためである。

- 食料品購入 -> ポイント付与 -> アプリクーポン -> カード / WAON 利用 -> 銀行口座 / 住宅ローン / 保険のクロスセル。
- モール来訪 -> 金融サービスカウンター -> 住宅ローンまたは保険リード。
- 海外小売網 -> アジアでの消費者金融拡大。

これは、貸借対照表から始まるメガバンク金融や、アプリから始まるコード決済金融とは構造的に異なる。
