# イオングループ

## TL;DR

イオンは、その金融サービスの価値が **日常生活の流通** に由来する小売グループである：スーパーマーケット、モール、クレジットカード、WAON 電子マネー、AEON Pay、保険カウンター、そして [[card-issuers/aeon-bank|Aeon Bank / AEON Financial Service]]。FinWiki にとって、イオンは日本の **小売から金融への OS** の最も強力な事例である。

金融上の問いは「イオンは銀行を所有しているか?」ではない。それは **小売の頻度がどのように金融流通と信用データを生み出すか** である。

## 構造

| レイヤー | 役割 | 金融上の関連性 |
|---|---|---|
| イオン株式会社 | グループ親会社および小売持株会社 | 金融商品が収益化する消費者トラフィックとブランドの信頼を所有 |
| イオンフィナンシャルサービス | イオン小売に根ざした上場総合金融サービスグループ | クレジットカード、銀行代理、プリペイド、アジア消費者金融、保険 / 決済隣接 |
| イオン銀行 | AFS 傘下の小売銀行 | 預金、住宅ローン、ATM、投資信託 / 保険流通、店舗連動バンキング |
| イオンカード | クレジットカード商品ライン | カード支出、リボ / 分割、WAON POINT およびイオン小売特典 |
| WAON | プリペイド電子マネー | ストアドバリュー決済、ポイント、イオン店舗の頻度、プリペイド規制境界 |
| AEON Pay / iAEON | アプリ決済 / グループアプリレイヤー | 店舗アプリ、決済、クーポン、ポイント、顧客 ID の接続を試みる |

## ビジネスモデル

イオンの金融モデルには 3 つの優位性がある：

1. **純粋なオンライン CAC なしの流通**：店舗、モール、スーパーマーケットのカウンターが獲得コストを下げる。
2. **反復的な低単価データ**：食料品と日常の買い物が、住宅ローンや証券よりも頻繁な行動データを生み出す。
3. **クローズドループ特典**：イオンカード、WAON、WAON POINT、AEON Pay をグループ店舗のマージンとキャンペーンを中心に調整できる。

## 戦略的読解

イオンが JapanFG にとって重要なのは、金融サービスが **小売の運営リズム** に組み込まれうることを示すからである：

- 食料品購入 -> ポイント発行 -> アプリクーポン -> カード / WAON 利用 -> 銀行口座 / 住宅ローン / 保険のクロスセル。
- モール来店 -> 金融サービスカウンター -> 住宅ローンまたは保険のリード。
- 海外小売拠点 -> アジアにおける消費者金融の拡大。

これは、バランスシートから始まるメガバンク金融とも、アプリから始まるコード決済金融とも構造的に異なる。
