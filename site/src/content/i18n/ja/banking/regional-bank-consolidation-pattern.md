# 地方銀行再編パターン

## TL;DR

日本の地方銀行再編は、地域の人口動態圧力、狭隘な純利鞘、店舗・システムコスト、デジタル投資需要、そして縮小する地域における基礎的な金融サービスの維持の必要性によって駆動される。金融庁の枠組みは、強制的な再編ではなく、顧客課題の解決、金融仲介機能の質、そして自主的な経営判断を重視する。

このページは [[regional-banks/tomony-hd]]、[[regional-banks/chiba-bank]]、[[regional-banks/fukuoka-fg]]、[[regional-banks/concordia-fg]]、[[regional-banks/18-shinwa-bank]]、[[regional-banks/akita-bank]]、[[regional-banks/iwate-bank]]、[[regional-banks/joyo-bank]] に用いる。

## 再編の形態

| 形態 | 意味 | 典型的な理由 |
|---|---|---|
| 持株会社統合 | 複数の銀行が銀行持株会社の傘下に子会社として残る。 | 地域ブランドを維持しつつ、資本・戦略・一部機能を統合する。 |
| 法的合併 | 銀行が一つの銀行法人に合併する。 | より深いコスト削減、システム統一、より強固なバランスシート。 |
| 株式持合い / アライアンス | 完全合併に至らずに銀行が協業する。 | 実行リスクが低い。地域の政治やシステムが合併を困難にする場合に有用。 |
| システム / バックオフィス共同化 | IT、オペレーション、コンプライアンス、デジタル、商品プラットフォームを共有する。 | 顧客向けの独立性を保ちつつ固定費を削減する。 |
| 地域プラットフォーム拡大 | より強固な地域グループが地域の金融プラットフォーム提供者として機能する。 | 非金利収益と地域の法人支援能力を構築する。 |

## なぜ起こるのか

構造的圧力は単一の変数ではない:

- 人口減少と地域の借り手の縮小;
- 伝統的な預金 / 貸出の低成長;
- 上昇するコンプライアンス、AML、サイバー、デジタルコスト;
- 事業承継、再構築、中小企業向けアドバイザリー支援の必要性;
- すべての地域銀行で専門人材を採用・維持することの困難さ;
- 地域の店舗・決済サービスを維持する圧力。

## 判断における用途

ある地方銀行が以下のいずれであるかを問う際に、このページを用いる:

- 買収者 / 再編主体となる可能性が高いか;
- 買収対象となる可能性が高いか;
- 地域サービス維持の受け皿か;
- 中小企業アドバイザリー / 事業承継のプラットフォームか;
- バランスシート問題に偽装されたシステムコスト問題か。
