# 地方銀行再編パターン

## 要約

日本の地方銀行再編は、地域の人口動態圧力、狭い純金利マージン、店舗 / システムコスト、デジタル投資ニーズ、そして縮小する地域で基本的な金融サービスを維持する必要性によって駆動される。FSA のフレーミングは、強制的な再編ではなく、顧客の課題解決、金融仲介の質、自主的な経営判断を強調する。

このページは、[[JapanFG/tomony-hd]]、[[JapanFG/chiba-bank]]、[[JapanFG/fukuoka-fg]]、[[JapanFG/concordia-fg]]、[[JapanFG/18-shinwa-bank]]、[[JapanFG/akita-bank]]、[[JapanFG/iwate-bank]]、および [[JapanFG/joyo-bank]] に使用すること。

## 再編の形態

| 形態 | 意味 | 典型的な理由 |
|---|---|---|
| 持株会社統合 | 複数の銀行が銀行持株会社の下で子会社として残る。 | 資本、戦略、一部機能を統合しながら地域ブランドを維持する。 |
| 法的合併 | 銀行が 1 つの銀行エンティティに合併する。 | より深いコスト削減、システム統一、より強固なバランスシート。 |
| 株式持合い / 提携 | 完全合併なしに銀行が協力する。 | 実行リスクが低い。地域政治やシステムが合併を困難にする場合に有用。 |
| システム / バックオフィス共有 | IT、オペレーション、コンプライアンス、デジタル、製品プラットフォームを共有する。 | 顧客向けの独立性を維持しながら固定費を削減する。 |
| 地域プラットフォーム拡大 | より強固な地域グループが地域金融プラットフォーム提供者として機能する。 | 非金利収益と地域の法人支援能力を構築する。 |

## なぜ起こるか

構造的圧力は単一の変数ではない：

- 人口減少と地域の借り手の縮小。
- 伝統的な預金 / 貸出の低成長。
- コンプライアンス、AML、サイバー、デジタルコストの上昇。
- 事業承継、再構築、中小企業助言支援の必要性。
- すべての地域銀行で専門スタッフを採用・維持することの困難さ。
- 地域の店舗と決済サービスを維持する圧力。

## 意思決定での利用

このページは、ある地方銀行が以下のいずれであるかを問うときに使用すること：

- 買収者 / 再編主体となる可能性が高いか。
- 買収対象となる可能性が高いか。
- 地域サービス維持のための受け皿か。
- 中小企業助言 / 事業承継プラットフォームか。
- バランスシート問題に見せかけたシステムコスト問題か。
