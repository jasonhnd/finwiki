# 日本の地方銀行 M&A 統合のファミリーツリーマトリクス

## 要約

日本における 1990 以降の地方銀行統合は、それぞれが一つ以上の第一地銀(「地銀」)および / または第二地銀(「第二地銀」)の前身行を、場合によっては信用協同組合 / 新銀行の前身を組み合わせた、銀行持株会社の階層的なマップを生み出した。このマトリクスは、2026時点で日本において実質的に活動しているすべての地方銀行 FG について、存続する FG 名、FG 設立年、前身行、都道府県のカバレッジ、資本規模、持株会社の構造を集めたものである。これは [[banking/regional-bank-consolidation-pattern|Regional bank consolidation pattern]](なぜ)および [[banking/japan-banking-license-tier-comparison-matrix|Japan banking-license tier comparison matrix]](どの種類のライセンスか)のファミリーツリー版である。

## ウィキ上の位置づけ

この項目は [[banking/INDEX|banking index]] の下に位置する。構造的なドライバー(人口、NIM、システムコスト、AML、デジタル)については [[banking/regional-bank-consolidation-pattern|Regional bank consolidation pattern]] と、デジタルレール層については [[banking/regional-bank-api-digital-partnership-route|Regional bank API and digital partnership route]] と、地銀対第二地銀対協同組合のライセンス階層の区分については [[banking/japan-banking-license-tier-comparison-matrix|Japan banking-license tier comparison matrix]] と、最大級の地銀対メガバンクの境界事例については [[banking/post-megabank-positioning|Post-megabank positioning]] とあわせて読むこと。FG 持株会社の母集団は [[JapanFG/INDEX|JapanFG index]] である。このマトリクスの基礎となる銀行持株会社の完全な FSA リストは FSA 銀行持株会社一覧 PDF である。

## なぜこのファミリーツリーが重要か

地方銀行統合は階層的である。多くの言及は、話者が次のどれを意味するのかを特定せずに「X銀行」と言う:

- **元の前身行**(例:第四銀行 1873-2020);
- **合併後の承継行**(例:[[JapanFG/daishi-hokuetsu-bank|第四北越銀行]]);
- 営業行の上にある **持株会社**([[JapanFG/dai-shi-hokuetsu-fg|第四北越 FG]] 7327);
- 持株会社が属する **アライアンス / コンソーシアム**(例:TSUBASA、[[JapanFG/sbi-regional-bank-hd|SBI 地銀 HD]])。

明示的なファミリーツリーのマップがなければ、同じブランド名が四つの異なる法的事体に用いられる。このマトリクスは四つのレベルすべてを一つの一貫したビューに収め、いかなるバランスシートまたは戦略の主張の前にも「自分が実際に話しているのはどの事体か」という問いに答えられるようにする。

具体的には、このマトリクスは次を浮かび上がらせる:

- どの FG が **単一県・単一行**(例:[[JapanFG/iyogin-hd|いよぎん HD]]、[[JapanFG/shizuoka-fg|しずおか FG]]、[[JapanFG/hirogin-hd|ひろぎん HD]]、[[JapanFG/juroku-fg|十六 FG]])か ── それらは持株会社が本質的に単一行のラッパーである単独株式移転のパターンである;
- どの FG が **県をまたぐ二行以上**([[JapanFG/mebuki-fg|めぶき FG]]、[[JapanFG/concordia-fg|横浜 FG (旧コンコルディア)]]、[[JapanFG/kyushu-fg|九州 FG]]、[[JapanFG/fukuoka-fg|FFG]]、[[JapanFG/hokuhoku-fg|ほくほく FG]])か ── それらは県の線をまたいで銀行を組み合わせる共同株式移転のパターンである;
- どの FG が **救済 M&A** によって形成されたか([[JapanFG/jimoto-hd|じもと HD]] が 2024-06に RCC の過半数を取得;[[JapanFG/tomony-hd|トモニ HD]] が 2017に大正銀行を吸収;[[JapanFG/mebuki-fg|めぶき FG]] が Resona / DICJ により処理された足利銀行の系譜に加わる);
- どの FG が同一階層の銀行のペアではなく **第一地銀 + 第二地銀** の混合を組み合わせているか;
- どの FG が **より広いコンソーシアム** に属しているか([[JapanFG/sbi-regional-bank-hd|SBI 地銀 HD]] のエクイティアライアンスの輪;[[JapanFG/kansai-mirai-bank|関西みらい銀行]] および [[JapanFG/minato-bank|みなと銀行]] をめぐる Resona グループのクラスター)。

このファミリーツリーは、ライセンス階層マトリクス [[banking/japan-banking-license-tier-comparison-matrix|Japan banking-license tier comparison matrix]](ここの各行は Tier 2 第一地銀または Tier 3 第二地銀にある)、ならびに協同組合銀行マトリクス [[banking/japan-cooperative-banking-unified-federation-matrix|Japan cooperative-banking unified federation matrix]](協同組合銀行の統合はここで述べる持株会社の経路とは別の軸にある)に直接つながる。

## このマトリクスの読み方

1. **まず FG(持株会社)の行を選ぶ。** FG は、傘下の銀行の持株を発行し、M&A 文書に署名し、FSA の銀行持株会社監督のもとで連結自己資本充実度を報告する法的事体である。これは戦略が実行される単位である。
2. **前身行の列を読む。** これは FG が単一行ラッパー(例:[[JapanFG/iyogin-hd|いよぎん HD]]、[[JapanFG/shizuoka-fg|しずおか FG]])か真の複数行統合([[JapanFG/fukuoka-fg|FFG]]、[[JapanFG/concordia-fg|横浜 FG]])かを教えてくれる。
3. **営業行のパターンの列を確認する。** 2行アンブレラは一つの FG の下に二つの営業行を生かしておく;1営業行のパターンは構成銀行が法的にすでに合併済みであることを意味する([[JapanFG/dai-shi-hokuetsu-fg|DHFG]] の下の [[JapanFG/daishi-hokuetsu-bank|第四北越銀行]])。
4. **都道府県カバレッジの列を確認する。** 単一県の FG([[JapanFG/shizuoka-fg|しずおか]]、[[JapanFG/hirogin-hd|ひろぎん]]、[[JapanFG/juroku-fg|十六]])はローカルプラットフォームの戦略である;県をまたぐ FG([[JapanFG/mebuki-fg|めぶき]]、[[JapanFG/hokuhoku-fg|ほくほく]]、[[JapanFG/kyushu-fg|九州]])は地域プラットフォームの戦略である;全国カバレッジの FG([[JapanFG/sbi-regional-bank-hd|SBI 地銀 HD]] コンソーシアム)はデジタルプラットフォームの戦略である。
5. **[[banking/regional-bank-api-digital-partnership-route|regional-bank API and digital partnership route]] とクロスチェックして**、同じ FG がデジタルレールの軸でどう位置するかを見る([[JapanFG/fukuoka-fg|FFG]] のみんなの銀行 BaaS、[[JapanFG/hokkoku-fhd|CCI / Hokkoku]] の Tonari、[[JapanFG/kiraboshi-fg|きらぼし]] の UI 銀行)。

## 転換点としての主要な統合イベント

- **2003-09 → 2004-09 [[JapanFG/hokuhoku-fg|ほくほく FG]] (北陸 + 北海道)。** 日本の銀行史上初の「離れた地域の地銀どうしの広域連合」モデル。
- **2003-11-29 足利銀行 破綻 → 一時国有化 → 2008-07 民間譲渡 (野村ファンド + ネクスト・キャピタル等) → 2008-12 足利 HD → 2016-10 [[JapanFG/mebuki-fg|めぶき FG]] 合流。** 後に FG アンブレラの下で健全なパートナーに加わった破綻後の再生の弧。
- **2007-04 [[JapanFG/fukuoka-fg|FFG]] 発足 (福岡 + 熊本ファミリー)。** 1地銀主導の県をまたぐ FG のテンプレートを設定した。
- **2009-10 [[JapanFG/fidea-hd|フィデア HD]] (荘内 + 北都)。** 東日本初の県をまたぐ FG。
- **2010-04 [[JapanFG/tomony-hd|トモニ HD]] (徳島 + 香川)。** 第二地銀の県をまたぐ FG、後に 2017 大正銀の救済により拡大 → 2020-01 徳島大正銀行 誕生。
- **2014-10 → 2016-04 → 2018-05 [[JapanFG/kiraboshi-fg|東京きらぼし FG]]。** 都民 + 八千代 → +新銀行東京 → 3行合併 (都政の新銀行東京の帰結を組み合わせた東京都市圏の特殊事例)。
- **2015-10 [[JapanFG/kyushu-fg|九州 FG]] (肥後 + 鹿児島)。** 南九州の県をまたぐ 2強連合。
- **2016-04 [[JapanFG/concordia-fg|横浜 FG (旧コンコルディア)]] (横浜 + 東日本)。** 最大の単一 FG クラスのための階層をまたぐ (1地銀 + 2地銀) 県をまたぐアンブレラ。
- **2016-10 [[JapanFG/mebuki-fg|めぶき FG]] (常陽 + 足利 HD)。** 破綻後処理のパートナーとの県をまたぐ組み合わせ。
- **2018-04 → 2021-05 [[JapanFG/san-jusan-fg|三十三 FG]] (三重 + 第三 → 三十三銀行)。** 完全な法的合併を伴う 2地銀 + 2地銀。
- **2018-10 → 2021-01 [[JapanFG/dai-shi-hokuetsu-fg|第四北越 FG]] (第四 + 北越 → 第四北越銀行)。** 新潟の新しい一県一行モデル。
- **2020-12 [[JapanFG/sbi-regional-bank-hd|SBI 地銀 HD]] 設立。** 「第4のメガバンク構想」の非 FG アライアンスモデル。
- **2022-04 → 2025-01 [[JapanFG/proclair-hd|プロクレア HD]] (青森 + みちのく)。** 県単位の独占認可のための **独禁特例法 (令和2年法律第32号) 初適用**。
- **2024-06-20 [[JapanFG/jimoto-hd|じもと HD]] RCC 議決権 63.52% 取得。** 2地銀 FG に対する公的資金監督が強化された。
- **2025-10 [[JapanFG/concordia-fg|横浜 FG]] 商号変更** (旧 コンコルディア → 横浜 FG)。
- **2025-10 [[JapanFG/hokkoku-fhd|北國 FHD]] → [[JapanFG/cci-group|CCI グループ]] 商号変更。** 商号に反映されたデジタルプラットフォームへの転換。

| 共同 IT ベンダー経由でシステム共有する地方銀行 | 複数の地方銀行が日立 / NTT データ / IBM の勘定系プラットフォームを共有 | FG をまたぐ広範な加盟 |
| [[JapanFG/sbi-regional-bank-hd\|SBI 地銀 HD]] エクイティアライアンス | 「第4のメガバンク構想」の非 FG 資本 + 商品協力 | パートナー行リスト:筑邦, 清水, 大東, 福島, [[JapanFG/shimane-bank\|島根]], 仙台 ([[JapanFG/jimoto-hd\|じもと]]), 筑波, 東和, きらやか ([[JapanFG/jimoto-hd\|じもと]]) + [[JapanFG/sbi-shinsei-bank\|SBI 新生銀行]] 商業銀行コア |

これらのアライアンスはファミリーツリーの全体像を複雑にする:地方銀行は一つの FG にいると同時に、一つ以上の FG をまたぐアライアンスにもいられる。特に TSUBASA Alliance は複数の FG にまたがり(千葉銀は独立を維持、[[JapanFG/dai-shi-hokuetsu-fg|第四北越]] FG、北洋グループなど)、FG 層の上にある重要なシステムコスト層である。

## 持株会社の FSA 許可プロセス

銀行法のもとでの FSA 銀行持株会社の許可は次を伴う:

1. 新たな銀行持株会社を形成するための **銀行法の事前認可**(自動的な権利はない;FSA の審査)。
2. 連結ベースで要求される **自己資本充実度**(バーゼル III 連結報告;国際的に活動するか否かに応じて BIS / 国内基準)。
3. 持株会社の取締役会の構造、内部監査の枠組み、グループリスク管理を含む **ガバナンス審査**。
4. 各子会社銀行の役割を含む **グループ事業戦略** の開示。
5. **完了までの期間**:通常は発表から FG 設立まで 6-12 か月;県をまたぐ事例や競争上のクリアランスを経た事例ではより長いこともある([[JapanFG/18-shinwa-bank|十八親和 銀行合併]] は公取委の審査のため 3 年延期された)。
6. **FG 後の継続監督**:FSA は持株会社と営業行子会社をあわせて検査する;連結自己資本充実度の報告は持株会社で行われる。

2021 の銀行法改正は、地方銀行が子会社(地域活性化、デジタルバンキング、コンサルティング)に投資することをやや容易にした ── これは非銀行投資のために持株会社の構造を用いる際の業務上の摩擦を低減した。複数の FG([[JapanFG/concordia-fg|横浜]]、[[JapanFG/fukuoka-fg|FFG]]、[[JapanFG/hokkoku-fhd|北國 / CCI]]、[[JapanFG/kiraboshi-fg|きらぼし]])はデジタル / コンサルティング / ベンチャー投資の子会社のために拡大された範囲を積極的に用いてきた。

## なぜ 2015

以降に統合が加速したか

複数の構造的なドライバーが、持株会社統合を通じて地方銀行の母集団を圧縮した:

- 東京-大阪回廊外の都道府県における **人口減少** が預金 / 融資の基盤を縮小させる。
- **マイナス金利環境**(日銀 2016-)が地方銀行のネット金利マージンを圧縮した。
- 小規模な地方銀行が単独で賄えない **増大するコンプライアンス / AML / サイバー / デジタルのコスト**。
- グループレベルの共有プラットフォームに規模を合わせる **システム投資要件**。
- **公共政策の後押し**:FSA の監督姿勢;同一県の合併に対する独占禁止上の障害を低減する 2020 独禁特例法;子会社投資の範囲を拡大する 2021 の銀行法。
- **銀行労働市場への人口圧力**:独立した地方銀行での専門スタッフの採用の難しさ。
- **中小企業顧客における事業承継の需要**:グループレベルでよりよく規模を合わせるコンサルティング能力を必要とする。
- [[JapanFG/hokuhoku-fg|ほくほく FG]] のような **県をまたぐ協力の先例** が地理的非隣接の実行可能性を証明した。

これらのドライバーの組み合わせが、このマトリクスに記録された FG の波を生み出した ── その大半は 2014-2022に集中している。波は続いており、[[JapanFG/fidea-hd|フィデア HD]] が次のマイルストーンの一つとして 2027-01 の銀行合併を計画している。

## 境界事例と留意点

- **T&D HD (8795) ≠ TD HD / 第四北越。** [[JapanFG/td-hd|T&D Holdings]] は生命保険持株会社(太陽 + 大同)である。user 仕様に従い、このマトリクスは地方銀行の文脈について [[JapanFG/dai-shi-hokuetsu-fg|第四北越 FG]] を指すラベルとして「TD HD」を用いる。実世界での「TD HD」への言及は、明示的に別途述べられない限り、ほぼ常に生命保険事体である。
- **[[JapanFG/sapporo-hokuyo-bank|札幌北洋 (Sapporo Hokuyo)]] は現在 [[JapanFG/north-pacific-bank-hd|北洋 HD]] / [[JapanFG/ndfg|札幌北洋 HD lineage]] の下の [[JapanFG/north-pacific-bank|北洋銀行]] である**;歴史的な札幌 + 北海道の都市銀行統合は、このマトリクスの主な焦点よりやや早い波に属するが、事体は同じ地方銀行の母集団にある。
- **[[JapanFG/resona-hd|Resona HD]] 対地方銀行 FG。** [[JapanFG/resona-hd|Resona]] はライセンス階層マトリクスではメガバンクの境界 / D-SIB 指定の事体であるが、地方銀行の営業子会社として [[JapanFG/kansai-mirai-bank|関西みらい銀行]] および [[JapanFG/minato-bank|みなと銀行]] を運営する。これは同じグループがメガバンク階層(親会社 [[JapanFG/resona-hd|Resona]] / [[JapanFG/resona-bank|Resona Bank]] / [[JapanFG/saitama-resona-bank|Saitama Resona Bank]])と地方銀行階層(関西の営業行)の両方にまたがることを意味する。
- **[[JapanFG/sbi-regional-bank-hd|SBI 地銀 HD]] はすべてのパートナーにとって伝統的な FSA の意味での銀行持株会社ではない。** それは [[JapanFG/sbi-shinsei-bank|SBI 新生銀行]](2023-09以降、法的に完全所有の普通銀行)に支えられた戦略的アライアンスおよびエクイティ投資のプラットフォームである。パートナー地方銀行(筑邦 / 清水 / 大東 / 福島 / [[JapanFG/shimane-bank|島根]] / 仙台 / 筑波 / 東和 / きらやか)は独立した FSA ライセンスの法的事体を維持しており、通常の銀行持株会社のベースで SBI 地銀 HD に連結されていない。公開開示(Wikipedia「SBI地銀ホールディングス」)によれば、SBI の所有持分はパートナーによって大きく異なる ── おおむね低い方で 1%(東和銀)から高い方で ~34%(福島銀)および筆頭株主のポジション(島根銀 ~21%、じもと HD ~12%)まで、いくつかのパートナー(清水銀 ~2.5%、大東銀 ~2.8%、東北銀 ~2.9%、筑邦銀 ~2.9%)は一桁台前半の範囲にある。
- **営業行対持株会社:名称の再利用。** [[JapanFG/concordia-fg|横浜 FG]] は持株会社である;[[JapanFG/yokohama-bank|横浜銀行]] はその下の営業行である。[[JapanFG/dai-shi-hokuetsu-fg|第四北越 FG]] は持株会社である;[[JapanFG/daishi-hokuetsu-bank|第四北越銀行]] は営業行である。「横浜銀行」または「第四北越銀行」と言う言及は、FG ではなく営業行を意味する。
- **一県一行対複数行アンブレラ。** 単独株式移転の単一行ラッパーは、共同株式移転の複数行アンブレラと同じビジネスモデルではない。単一行ラッパーは M&A の選択肢と資本配分の柔軟性を維持するが、それ自体では支店 / システムコストを低減しない;複数行アンブレラは、構成銀行が後に法的に合併すれば、システム / 支店統合の可能性をもたらす。
- **保留中または計画中の合併ステップ。** いくつかの FG は弧の途中にある:[[JapanFG/fidea-hd|フィデア HD]] は 2027-01に荘内 + 北都 → フィデア銀行を計画している;[[JapanFG/proclair-hd|プロクレア HD]] は 2025-01に青森 + みちのく → 青森みちのく銀行を完了した。これらの計画されたイベントは営業行のパターンの列を時とともに変化させる。
- **銀行持株会社の公式リスト。** FSA が公表する完全な銀行持株会社一覧 PDF が、任意の時点における FG 事体リストの権威ある情報源である。このマトリクスを引用する前に、それを用いて事体名の正準性を検証すること。
