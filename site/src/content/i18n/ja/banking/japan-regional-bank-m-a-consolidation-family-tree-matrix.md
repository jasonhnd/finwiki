# 日本 地方銀行 M&A 統合 ファミリーツリー・マトリクス

## TL;DR

1990 以降の日本の地方銀行統合は、それぞれが一つ以上の第一地銀（"地銀"）および／または第二地銀（"第二地銀"）の前身、場合によっては信用協同組合／新銀行の前身を組み合わせた銀行持株会社の階層的マップを生み出してきた。本マトリクスは、2026時点で日本において実質的に活動するすべての地方銀行 FG について、存続する FG 名、FG 設立年、前身銀行、都道府県カバレッジ、資本規模、持株会社構造を収集する。これは [[banking/regional-bank-consolidation-pattern|Regional bank consolidation pattern]]（なぜ）および [[banking/japan-banking-license-tier-comparison-matrix|Japan banking-license tier comparison matrix]]（どの種類の免許か）に対するファミリーツリー版である。

## ウィキ経路

本項目は [[banking/INDEX|banking index]] の配下に位置する。構造的ドライバー（人口、NIM、システムコスト、AML、デジタル）として [[banking/regional-bank-consolidation-pattern|Regional bank consolidation pattern]] を、デジタルレール層として [[banking/regional-bank-api-digital-partnership-route|Regional bank API and digital partnership route]] を、地銀 vs 第二地銀 vs 協同組織の免許層区分として [[banking/japan-banking-license-tier-comparison-matrix|Japan banking-license tier comparison matrix]] を、最大級地銀 vs メガバンク辺縁のケースとして [[banking/post-megabank-positioning|Post-megabank positioning]] を併せて読むこと。FG 持株会社群は [[banking/INDEX|banking INDEX]] である。本マトリクスを支える完全な FSA 銀行持株会社一覧は FSA 銀行持株会社一覧 PDF である。

## このファミリーツリーが重要な理由

地方銀行統合は階層的である。多くの参照は「X銀行」と言うが、話者が次のいずれを意味するかを明示しない：

- **元の前身銀行**（例：第四銀行 1873-2020）；
- **合併後の存続銀行**（例：[[regional-banks/daishi-hokuetsu-bank|第四北越銀行]]）；
- **オペレーティング銀行の上位の持株会社**（[[regional-banks/dai-shi-hokuetsu-fg|第四北越 FG]] 7327）；
- 持株会社が属する**アライアンス／コンソーシアム**（例：TSUBASA、[[megabanks/sbi-regional-bank-hd|SBI 地銀 HD]]）。

明示的なファミリーツリー・マップがなければ、同じブランド名が四つの異なる法律実体に対して使われてしまう。本マトリクスは、いかなる貸借対照表上または戦略上の主張の前にも「自分は実際にどの実体について話しているのか」という問いに答えられるよう、この四つの層すべてを一つの一貫したビューに落とし込む。

具体的には、本マトリクスは次を可視化する：

- どの FG が**単一都道府県・単一銀行**か（例：[[regional-banks/iyogin-hd|いよぎん HD]]、[[regional-banks/shizuoka-fg|しずおか FG]]、[[regional-banks/hirogin-hd|ひろぎん HD]]、[[regional-banks/juroku-fg|十六 FG]]）—これらは持株会社が本質的に単一銀行のラッパーである 単独株式移転 パターン；
- どの FG が**複数都道府県・二行以上**か（[[regional-banks/mebuki-fg|めぶき FG]]、[[regional-banks/concordia-fg|横浜 FG (旧コンコルディア)]]、[[regional-banks/kyushu-fg|九州 FG]]、[[regional-banks/fukuoka-fg|FFG]]、[[regional-banks/hokuhoku-fg|ほくほく FG]]）—これらは都道府県の線をまたいで銀行を組み合わせる 共同株式移転 パターン；
- どの FG が**救済 M&A**で形成されたか（[[regional-banks/jimoto-hd|じもと HD]] が 2024-06 に RCC 過半数を取得；[[regional-banks/tomony-hd|トモニ HD]] が 大正銀行 2017 を吸収；[[regional-banks/mebuki-fg|めぶき FG]] が Resona / DICJ により処理された 足利銀行 系譜に加わる）；
- どの FG が**第一地銀 + 第二地銀**のミックスか、同層の銀行のペアか；
- どの FG が**より広範なコンソーシアム**に属するか（[[megabanks/sbi-regional-bank-hd|SBI 地銀 HD]] の資本アライアンス・リング；[[regional-banks/kansai-mirai-bank|関西みらい銀行]] と [[regional-banks/minato-bank|みなと銀行]] を中心とする Resona グループのクラスター）。

このファミリーツリーは、免許層マトリクス [[banking/japan-banking-license-tier-comparison-matrix|Japan banking-license tier comparison matrix]]（ここの各行は Tier 2 第一地銀 または Tier 3 第二地銀 にある）、ならびに協同組織銀行マトリクス [[banking/japan-cooperative-banking-unified-federation-matrix|Japan cooperative-banking unified federation matrix]]（協同組織銀行の統合は、ここで述べる持株会社経路とは別の軸にある）に直接つながる。

## このマトリクスの読み方

1. **まず FG（持株会社）の行を選ぶ。** FG は、傘下銀行の持株を発行し、M&A 文書に署名し、FSA の銀行持株会社監督の下で連結自己資本比率を報告する法律実体である。これが戦略が実行される単位である。
2. **前身銀行の列を読む。** これは、その FG が単一銀行ラッパー（例：[[regional-banks/iyogin-hd|いよぎん HD]]、[[regional-banks/shizuoka-fg|しずおか FG]]）か、真の複数銀行統合（[[regional-banks/fukuoka-fg|FFG]]、[[regional-banks/concordia-fg|横浜 FG]]）かを教えてくれる。
3. **オペレーティング銀行パターンの列を確認する。** 2-bank umbrella は一つの FG の下で二つのオペレーティング銀行を存続させる；1-operating-bank パターンは、構成銀行が既に法的に合併済み（[[regional-banks/daishi-hokuetsu-bank|第四北越銀行]] は [[regional-banks/dai-shi-hokuetsu-fg|DHFG]] の下）であることを意味する。
4. **都道府県カバレッジの列を確認する。** 単一都道府県 FG（[[regional-banks/shizuoka-fg|しずおか]]、[[regional-banks/hirogin-hd|ひろぎん]]、[[regional-banks/juroku-fg|十六]]）はローカル・プラットフォームの打ち手；複数都道府県 FG（[[regional-banks/mebuki-fg|めぶき]]、[[regional-banks/hokuhoku-fg|ほくほく]]、[[regional-banks/kyushu-fg|九州]]）は地域プラットフォームの打ち手；全国カバレッジ FG（[[megabanks/sbi-regional-bank-hd|SBI 地銀 HD]] コンソーシアム）はデジタル・プラットフォームの打ち手である。
5. **[[banking/regional-bank-api-digital-partnership-route|regional-bank API and digital partnership route]] と相互確認**して、同じ FG がデジタルレール軸上でどこに位置するかを見る（[[regional-banks/fukuoka-fg|FFG]] の みんなの銀行 BaaS、[[regional-banks/hokkoku-fhd|CCI / Hokkoku]] の Tonari、[[regional-banks/kiraboshi-fg|きらぼし]] の UI 銀行）。

## 転換点としての主要統合イベント

- **2003-09 → 2004-09 [[regional-banks/hokuhoku-fg|ほくほく FG]] (北陸 + 北海道).** 日本の銀行史上初の「離れた地域の地銀どうしの広域連合」モデル。
- **2003-11-29 足利銀行 破綻 → 一時国有化 → 2008-07 民間譲渡 (野村ファンド + ネクスト・キャピタル等) → 2008-12 足利 HD → 2016-10 [[regional-banks/mebuki-fg|めぶき FG]] 合流.** 破綻後の再建の弧を経て、後に FG 傘下で健全なパートナーに加わった。
- **2007-04 [[regional-banks/fukuoka-fg|FFG]] 発足 (福岡 + 熊本ファミリー).** 1地銀 主導の複数都道府県 FG のテンプレートを設定した。
- **2009-10 [[regional-banks/fidea-hd|フィデア HD]] (荘内 + 北都).** 東日本 初の複数都道府県 FG。
- **2010-04 [[regional-banks/tomony-hd|トモニ HD]] (徳島 + 香川).** 第二地銀の複数都道府県 FG、後に 2017 大正銀 救済により拡大 → 2020-01 徳島大正銀行 誕生。
- **2014-10 → 2016-04 → 2018-05 [[regional-banks/kiraboshi-fg|東京きらぼし FG]].** 都民 + 八千代 → +新銀行東京 → 3行合併（都政の新銀行東京 の帰結を組み合わせた東京都市圏の特例）。
- **2015-10 [[regional-banks/kyushu-fg|九州 FG]] (肥後 + 鹿児島).** 九州南部の複数都道府県 2強連合。
- **2016-04 [[regional-banks/concordia-fg|横浜 FG (旧コンコルディア)]] (横浜 + 東日本).** 最大の単一 FG クラスのための、クロス層（1地銀 + 2地銀）の複数都道府県傘。
- **2016-10 [[regional-banks/mebuki-fg|めぶき FG]] (常陽 + 足利 HD).** 破綻処理後のパートナーとの複数都道府県の組み合わせ。
- **2018-04 → 2021-05 [[regional-banks/san-jusan-fg|三十三 FG]] (三重 + 第三 → 三十三銀行).** 完全な法的合併を伴う 2地銀 + 2地銀。
- **2018-10 → 2021-01 [[regional-banks/dai-shi-hokuetsu-fg|第四北越 FG]] (第四 + 北越 → 第四北越銀行).** 新潟 の新しい 一県一行 モデル。
- **2020-12 [[megabanks/sbi-regional-bank-hd|SBI 地銀 HD]] 設立.** 「第4のメガバンク構想」非 FG アライアンスモデル。
- **2022-04 → 2025-01 [[regional-banks/proclair-hd|プロクレア HD]] (青森 + みちのく).** 都道府県内独占のクリアランスのための**独禁特例法 (令和2年法律第32号) 初適用**。
- **2024-06-20 [[regional-banks/jimoto-hd|じもと HD]] RCC 議決権 63.52% 取得.** その 2地銀 FG への公的資金監督が強化された。
- **2025-10 [[regional-banks/concordia-fg|横浜 FG]] 商号変更**（旧 コンコルディア → 横浜 FG）。
- **2025-10 [[regional-banks/hokkoku-fhd|北國 FHD]] → [[financial-regulators/cci-group|CCI グループ]] 商号変更.** デジタル・プラットフォームへの転換が社名に反映された。

## アライアンス・システム共有のオーバーレイ（非 FG 協力）

## 持株会社の FSA 許可プロセス

銀行法に基づく FSA 銀行持株会社の許可には次が含まれる：

1. 新たな銀行持株会社の設立に対する**銀行法の事前認可**（自動的な権利はない；FSA の審査）。
2. 連結ベースで要求される**自己資本比率**（Basel III 連結報告；国際的に活動するかどうかにより BIS / 国内基準）。
3. 持株会社の取締役会構造、内部監査の枠組み、グループのリスク管理を含む**ガバナンス審査**。
4. 各子会社銀行の役割を含む**グループ事業戦略**の開示。
5. **完了までの期間**：公表から FG 設立まで通常 6-12 ヶ月；複数都道府県または競争クリアランスのケースでは時にそれより長い（[[regional-banks/18-shinwa-bank|十八親和 銀行合併]] は 公取委 審査のため 3 年延期された）。
6. **FG 設立後の継続監督**：FSA は持株会社とオペレーティング銀行子会社を一体として検査する；連結自己資本比率の報告は持株会社で行われる。

2021 年の銀行法 改正 は、地方銀行が子会社（地域活性化、デジタルバンキング、コンサルティング）に投資することをやや容易にした—これは、非銀行投資に持株会社構造を用いる運営上の摩擦を低下させた。複数の FG（[[regional-banks/concordia-fg|横浜]]、[[regional-banks/fukuoka-fg|FFG]]、[[regional-banks/hokkoku-fhd|北國 / CCI]]、[[regional-banks/kiraboshi-fg|きらぼし]]）が、デジタル / コンサルティング / ベンチャー投資の子会社のために拡大された範囲を積極的に活用してきた。

## なぜ統合が 2015

以降に加速したか 複数の構造的ドライバーが、持株会社統合を通じて地方銀行群を圧縮した：

- 東京・大阪回廊外の都道府県の**人口減少**が、預金 / 貸出の基盤を縮小する。
- **マイナス金利環境**（BoJ 2016-）が、地方銀行の純利鞘を圧縮した。
- 小規模な地方銀行が自前で賄えない**上昇するコンプライアンス / AML / サイバー / デジタルコスト**。
- グループレベルの共有プラットフォームへとスケールする**システム投資要件**。
- **公共政策のナッジ**：FSA の監督姿勢；2020 独禁特例法 による同一都道府県合併への独占禁止上の障害の軽減；2021 銀行法 による子会社投資範囲の拡大。
- **銀行労働市場への人口圧力**：独立の地方銀行で専門スタッフを採用することの困難。
- **SME 顧客の事業承継需要**：グループレベルでより良くスケールするコンサルティング能力を要する。
- [[regional-banks/hokuhoku-fg|ほくほく FG]] のような**複数都道府県協力の先例**が、地理的な非隣接の実現可能性を証明した。

これらのドライバーの組み合わせが、本マトリクスに記録された FG の波を生み出した—その大半は 2014-2022 に集中している。波は続いており、[[regional-banks/fidea-hd|フィデア HD]] が次のマイルストーンの一つとして 2027-01 銀行合併 を計画している。

## 境界ケースと注意点

- **T&D HD (8795) ≠ TD HD / Daishi-Hokuetsu.** [[life-insurers/td-hd|T&D Holdings]] は生命保険持株会社（太陽 + 大同）である。user 仕様により、本マトリクスは地方銀行の文脈で「TD HD」を [[regional-banks/dai-shi-hokuetsu-fg|第四北越 FG]] を指すラベルとして用いる。現実世界での「TD HD」への言及は、明示的に別途述べられない限り、ほぼ常に生命保険実体である。
- **[[regional-banks/sapporo-hokuyo-bank|札幌北洋 (Sapporo Hokuyo)]] は現在 [[regional-banks/north-pacific-bank-hd|北洋 HD]] / [[megabanks/ndfg|札幌北洋 HD lineage]] の下の [[regional-banks/north-pacific-bank|北洋銀行]] である**；歴史的な 札幌 + 北海道 の都市銀行統合は、本マトリクスの主たる焦点よりやや早い波に属するが、その実体は同じ地方銀行群に位置する。
- **[[megabanks/resona-hd|Resona HD]] vs 地方銀行 FG.** [[megabanks/resona-hd|Resona]] は免許層マトリクスではメガバンク辺縁 / D-SIB 指定の実体だが、地方銀行オペレーティング子会社として [[regional-banks/kansai-mirai-bank|関西みらい銀行]] と [[regional-banks/minato-bank|みなと銀行]] を運営する。これは、同じグループがメガバンク層（親 [[megabanks/resona-hd|Resona]] / [[megabanks/resona-bank|Resona Bank]] / [[megabanks/saitama-resona-bank|Saitama Resona Bank]]）と地方銀行層（関西オペレーティング銀行）の双方にまたがることを意味する。
- **[[megabanks/sbi-regional-bank-hd|SBI 地銀 HD]] は、すべてのパートナーにとって伝統的な FSA の意味での銀行持株会社ではない。** それは [[regional-banks/sbi-shinsei-bank|SBI 新生銀行]]（法的には 2023-09 以降の完全子会社である 普通銀行）を中心とする戦略的アライアンス・株式投資プラットフォームである。パートナー地方銀行（筑邦 / 清水 / 大東 / 福島 / [[regional-banks/shimane-bank|島根]] / 仙台 / 筑波 / 東和 / きらやか）は、独立した FSA 免許の法律実体を保持しており、通常の銀行持株会社ベースでは SBI 地銀 HD に連結されていない。公開開示（Wikipedia「SBI地銀ホールディングス」）によれば、SBI の保有比率はパートナーにより大きく異なる—低い側のおよそ 1%（東和銀）から、高くて ~34%（福島銀）および筆頭株主のポジション（島根銀 ~21%、じもと HD ~12%）まで、いくつかのパートナー（清水銀 ~2.5%、大東銀 ~2.8%、東北銀 ~2.9%、筑邦銀 ~2.9%）は一桁台前半の範囲にある。
- **オペレーティング銀行 vs 持株会社：名称の再利用。** [[regional-banks/concordia-fg|横浜 FG]] は持株会社；[[regional-banks/yokohama-bank|横浜銀行]] はその下のオペレーティング銀行である。[[regional-banks/dai-shi-hokuetsu-fg|第四北越 FG]] は持株会社；[[regional-banks/daishi-hokuetsu-bank|第四北越銀行]] はオペレーティング銀行である。「横浜銀行」または「第四北越銀行」と言う参照は、FG ではなくオペレーティング銀行を意味する。
- **一県一行 vs 複数銀行傘。** 単独株式移転 の単一銀行ラッパーは、共同株式移転 の複数銀行傘と同じビジネスモデルではない。単一銀行ラッパーは M&A オプショナリティと資本配分の柔軟性を保つが、それ自体では支店 / システムコストを削減しない；複数銀行傘は、構成銀行が後に法的に合併すれば、システム / 支店統合の潜在力をもたらす。
- **保留中または計画中の合併ステップ。** いくつかの FG は弧の途中にある：[[regional-banks/fidea-hd|フィデア HD]] は 2027-01 に 荘内 + 北都 → フィデア銀行 を計画；[[regional-banks/proclair-hd|プロクレア HD]] は 2025-01 に 青森 + みちのく → 青森みちのく銀行 を完了した。これらの計画されたイベントは、時間の経過とともにオペレーティング銀行パターンの列を変える。
- **銀行持株会社の公式一覧。** FSA が公表する完全な 銀行持株会社一覧 PDF が、いかなる時点でも FG 実体リストの権威ある出典である。本マトリクスを引用する前に、それを用いて実体名の正準性を検証すること。
