# 日本における JF マリンバンク システム

## TL;DR

JF マリンバンクは、信用事業を営む JF（漁業協同組合）、JF 信漁連、および [[cooperative-banks/norinchukin|Norinchukin Bank]] によって構成される漁業協同組合の信用事業システムである。単一の普通銀行や上場企業グループではない。JF マリンバンクの公式基本方針は、本システムを漁業金融のための一体的な運営フレームワークとして説明しており、破綻防止機能と一体運営機能を備える。

本ページは [[banking/ja-bank-system-japan|JA Bank system]] の漁業版カウンターパートとして用いること。

## 1. システムの境界

| レイヤー | 役割 | FinWiki 経路 |
|---|---|---|
| 地域 JF / 漁業協同組合の信用事業 | 信用事業を営む漁業協同組合を通じた、組合員向けの預金・貸出・決済、および漁業地域金融。 | システム経路はここ；戦略的に重要でない限り、各地域 JF を個別ページとして作成しない。 |
| JF 信漁連 | 漁業協同組合の信用事業を支援・監督する都道府県 / 地域の信用連合会レイヤー。 | システム経路はここ；公式リスト作業が必要になった場合に限り、将来のレジストリ経路を設ける。 |
| 農林中央金庫 | 農業・漁業・林業協同組合金融の全国中央機関；JF マリンバンクの中央機関でもある。 | [[cooperative-banks/norinchukin]] |
| JF マリンバンク システム | JF マリンバンク基本方針の下での一体運営フレームワーク。 | 本ページ |
| 隣接する JF グループ機能 | 信用事業以外の漁業協同組合の販売、加工、共済、産業支援。 | 金融に関連する場合のみ用いる；非銀行の協同組合機能をバンキングマップに混在させない。 |

農林中金の公式特徴ページは、2025-04-01 JF マリンバンクのレイヤー数を 71 JF、10 JF 信漁連、1 農林中金機関として示している。

## 2. 運営ロジック

- 地域 JF の信用事業は、漁業世帯、漁業事業者、地域コミュニティを預金 / 貸出サービスに結びつける。
- JF 信漁連は、都道府県または地域の信用事業運営を調整し、会員 JF を支援する。
- [[cooperative-banks/norinchukin|Norinchukin]] が全国中央銀行および支援レイヤーを提供する。
- JF マリンバンク基本方針は、会員が一体的な信用事業システムとして運営することを目的に設計されている。
- 本システムは、通常の商業銀行グループのガバナンスとは別個の、独自の破綻防止および経営改善メカニズムを備える。

## 3. なぜ重要か

JF マリンバンクは、[[banking/ja-bank-system-japan|JA Bank system]] に次いで2つ目の協同組合システムのギャップを埋める。

- 漁業協同組合金融を農業協同組合金融から分離する；
- 地域 JF / JF 信漁連が通常の銀行事業会社と誤読されることを防ぐ；
- [[cooperative-banks/norinchukin]] に農業 + 漁業の協同組合金融の完全なマップを与える；
- 漁業信用保証や政策金融のページを追加する前に、明確な境界を作る。

## 4. 境界ケース

- **単一の銀行ではない**：JF マリンバンクはシステム名称であり、[[cooperative-banks/norinchukin]] は全国中央機関のページである。
- **JA バンクではない**：両システムは中央機関として農林中金を共有するが、JA バンクは農業協同組合金融であり、JF マリンバンクは漁業協同組合金融である。
- **政策金融ではない**：漁業近代化資金や公的信用支援は [[policy-finance/INDEX]] に接続しうるが、JF マリンバンク システム自体は協同組合金融である。
- **ロングテールの地域 JF ページ**：特定の JF / JF 信漁連が戦略的に重要な場合、または公式レジストリ索引が必要な場合に限り作成する。
