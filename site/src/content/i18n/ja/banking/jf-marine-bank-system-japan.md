# 日本の JF マリンバンクシステム

## 要約

JF マリンバンクは、信用事業 JF 協同組合、JF 信漁連、および [[JapanFG/norinchukin|Norinchukin Bank]] によって構成される、漁業協同組合の信用事業システムである。単一の普通銀行や上場企業グループではない。公式の JF マリンバンク基本方針は、本システムを、破綻防止機能および統一運営機能を備えた、漁業金融のための統合運営フレームワークとして説明している。

本ページは [[banking/ja-bank-system-japan|JA Bank system]] の漁業版として用いること。

## 1. システム境界

| 層 | 役割 | FinWiki ルート |
|---|---|---|
| 地域 JF / 漁業協同組合の信用事業 | 信用事業を営む漁業協同組合を通じた、会員向けの預金、貸出、決済、漁業コミュニティ金融。 | システムルートはここ;戦略的に重要でない限り、すべての地域 JF を独立ページとして作成しないこと。 |
| JF 信漁連 | 漁業協同組合の信用事業を支援・監督する都道府県 / 地域の信用連合会層。 | システムルートはここ;公式リストの作業が必要になった場合に限り、将来的に登録簿ルートを設ける。 |
| 農林中央金庫 | 農業、漁業、林業の協同組合金融のための全国中央機関;JF マリンバンクの中央機関でもある。 | [[JapanFG/norinchukin]] |
| JF マリンバンクシステム | JF マリンバンク基本方針のもとの統合運営フレームワーク。 | 本ページ |
| 隣接する JF グループ機能 | 信用事業の外側にある、漁業協同組合の販売、加工、共済、業界支援。 | 金融に関連する場合にのみ用いること;非銀行の協同組合機能を銀行マップに混ぜないこと。 |

農林中央金庫の公式の特徴ページは、2025-04-01 の JF マリンバンクの層の数を、71 JF、10 JF 信漁連、1 農林中央金庫機関としている。

## 2. 運営ロジック

- 地域 JF の信用事業は、漁業世帯、漁業事業者、地域コミュニティを預金 / 貸出サービスに結びつける。
- JF 信漁連は、都道府県または地域の信用事業運営を調整し、会員 JF を支援する。
- [[JapanFG/norinchukin|Norinchukin]] が全国中央銀行および支援層を提供する。
- JF マリンバンク基本方針は、会員を統合された信用事業システムとして運営させるよう設計されている。
- 本システムは、通常の商業銀行グループのガバナンスとは別個の、独自の破綻防止および経営改善のメカニズムを持つ。

## 3. 重要な理由

JF マリンバンクは、[[banking/ja-bank-system-japan|JA Bank system]] に続く 2 つ目の協同組織システムのギャップを埋める:

- 漁業協同組合金融を農業協同組合金融から分離する;
- 地域 JF / JF 信漁連が普通銀行の運営会社として誤読されるのを防ぐ;
- [[JapanFG/norinchukin]] に農業 + 漁業の完全な協同組織金融マップを与える;
- 漁業の信用保証や政策金融のページを追加する前に、明快な境界を作る。

## 4. 境界事例

- **単一の銀行ではない**:JF マリンバンクはシステム名であり、[[JapanFG/norinchukin]] は全国中央機関のページである。
- **JA バンクではない**:両システムは中央機関として農林中央金庫を共有するが、JA バンクは農業協同組合金融であり、JF マリンバンクは漁業協同組合金融である。
- **政策金融ではない**:漁業近代化融資や公的信用支援は [[policy-finance/INDEX]] に接続しうるが、JF マリンバンクシステム自体は協同組織金融である。
- **ロングテールの地域 JF ページ**:特定の JF / JF 信漁連が戦略的に重要な場合、または公式の登録簿インデックスが必要な場合にのみ作成すること。
