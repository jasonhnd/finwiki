# ポストメガバンクのポジショニング

## 要約

ポストメガバンクのポジショニングとは、日本の三大メガバンクの下位に位置しつつ、狭い地域やフィンテックのニッチよりは上位に位置する銀行および銀行グループを指す。これらは、フルなグローバルユニバーサルバンクの規模ではなく、リテール銀行、地域への集中、信託 / カストディ、プラットフォームバンキング、消費者金融、または地域の法人取引に特化することで競争することが多い。

JapanFG における主要なアンカーは [[megabanks/resona-hd]] である:単純な地方銀行として扱うには大きく戦略的に重要すぎるが、MUFG / SMFG / みずほ型のメガバンクではない銀行グループである。

## ポジショニングマップ

| ポジション | 事業ロジック | 例となる経路 |
|---|---|---|
| メガバンク | 証券、信託、法人、マーケット、海外、グループ金融の規模を備えた、グローバル / 全国的なユニバーサルバンク。 | [[megabanks/mufg]]、[[megabanks/smfg]]、[[megabanks/mizuho-fg]] |
| ポストメガバンク | 焦点を絞ったリテール / SME / 信託 / 決済戦略を持つが、メガバンクのフルな広がりは持たない、全国的または広域的な銀行。 | [[megabanks/resona-hd]]、[[regional-banks/sbi-shinsei-bank]] |
| 強い地域グループ | 統合、持株会社、地域の法人取引戦略を持つ、地域フランチャイズ。 | [[regional-banks/chiba-bank]]、[[regional-banks/fukuoka-fg]]、[[regional-banks/concordia-fg]] |
| プラットフォーム / ネット銀行 | 支店主導のユニバーサルバンキングではなく、デジタル獲得とエコシステムバンキング。 | [[banking/rakuten-bank]]、[[banking/paypay-bank]]、[[banking/au-jibun-bank]] |
| 協同組合 / 相互の地域金融 | 株主銀行のロジックではなく、会員または協同組合のロジック。 | [[banking/cooperative-banking-japan]] |

## なぜこのカテゴリーが重要か

メガバンクの下位にあるすべてを「地域」と呼んでしまうと、分析は重要な戦略を見落とす:

- りそな型のリテール / SME 特化は、都道府県の地方銀行業務とは異なる。
- SBI 新生型のグループ戦略は、銀行、証券、プラットフォーム金融を組み合わせる。
- ネット銀行は、歴史的な支店フランチャイズではなく、口座獲得とエコシステムフローを用いる。
- 強い地域グループは、統合の対象ではなく統合の主体として行動しうる。

## 規制・監督上の文脈

FSA の主要行等向けの監督指針は、主要行、銀行持株会社、グループ監督、外国銀行支店、新規参入をカバーする。全国銀行協会もまた、都市銀行 / 地方銀行の区別は厳密な法的カテゴリーではなく慣習的なものであると指摘している。これにより「ポストメガバンク」は、免許上のカテゴリーではなく分析上のカテゴリーとなる。
