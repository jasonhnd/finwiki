# JIP コンソーシアムによる東芝 TOB 2023-2024  事例 — 日本の大型株の非公開化スクイーズアウトテンプレート

## ウィキ上の位置づけ

本項目は [[corporate-strategy/INDEX|corporate-strategy INDEX]] の下に位置し、取引ファイナンスのオーバーレイについては [[finance/INDEX|finance INDEX]] へとルートする。手続き上の仕組みについては [[finance/japan-mbo-and-squeeze-out-process|Japan MBO / squeeze-out process]]、TOB の枠組みについては [[finance/japan-tender-offer-process|Japan tender offer process]]、代替経路の文脈については [[corporate-strategy/spinoff-decision-tree-japan|spinoff decision tree Japan]]、そしてこの帰結を導いた取引前のアクティビストの力学については [[finance/japan-activist-investor-playbook|Japan activist investor playbook]] と併せて読むこと。

## 要約

Japan Industrial Partners(JIP)が率いるコンソーシアムによる**東芝株式会社(当時は東証プライム 6502)の 2023  の非公開化**は、最近の日本の大型株の非公開化の典型的な事例である。本取引 — ~¥2兆円の TOB を 1 株あたり ¥4,620  で、2023,  年 9 月に完了、20,  年 12 月 2023  に上場廃止 — は、会計不正(2015  の開示)、ウェスチングハウス破綻の波及、外国アクティビストの圧力、繰り返されたガバナンスの失敗によって引き起こされた、日本で最も著名な産業グループの一つにおける複数年の危機を終わらせた。

本事例はいくつかの異なる要素のテンプレートとなる:(1)慢性的なアクティビスト / ガバナンスの摩擦に対する**解決メカニズムとしての非公開化**、(2)**国内主導のコンソーシアムによるファイナンス**(経産省が懸念した外資バイアウトの優勢に対するもの)、(3)**大規模なスクイーズアウトの実行**(¥2兆円の取引規模)、(4)分割 / 再集中 / 再上場の計画のための**非公開化後の再構築の滑走路**。SBG-Arm IPO テンプレート([[business/softbank-vision-fund-arm-ipo-template|here]])と並べて読むと、逆の事例である — 東芝は再構築のために非公開化し、Arm は価値を顕在化するために上場する。

## 1. 危機前の文脈(2015-2022)

| 時期 | 出来事 |
|---|---|
| 2015 | 会計不正の開示(複数年の利益過大計上) |
| 2017 | ウェスチングハウス原子力部門の破綻;東芝は数十億規模の減損 |
| 2017-2018 | メモリ事業(東芝メモリ → キオクシア)を ~¥2兆円でベインキャピタル主導のコンソーシアムに売却 |
| 2017-2018 | 外国アクティビストファンド(エフィッシモ、エリオット、キングストリート等)からの大規模な資本注入 |
| 2020 | エフィッシモおよび関連アクティビストが合計 ~30%+ を保有 |
| 2020 | CVC キャピタル・パートナーズによる非公開化の試みが失敗(1 株あたり ¥5,000+ のシグナルだが不十分) |
| 2021 | 複数の経営陣交代;3 分割計画を発表 |
| 2022 | 3 分割が株主に否決される |
| 2022-Q4 | 新たな入札者を求める戦略的レビュープロセスを開始 |
| 2023-Q1 | JIP 主導のコンソーシアムが優先交渉先として選定 |
| 2023-03 | TOB を発表 |

危機前の 10 年間は基盤的である — 東芝は、上場という地位が増幅した、継続的なアクティビスト・外国株主との摩擦の中にあった。非公開化は、この循環を断ち切る唯一の残された手段として位置づけられた。

## 2. 2023  の TOB アーキテクチャ

| 要素 | 詳細 |
|---|---|
| 入札者 | Japan Industrial Partners(JIP)、国内 PE ファーム |
| 入札者コンソーシアム | JIP + オリックス + 中部電力 + その他 17  の国内金融 / 産業主体 |
| ファイナンス | マルチトランシェの銀行ローンスタック(みずほ、MUFG、SMBC 等)+ コンソーシアムからのエクイティ |
| TOB 価格 | 1 株あたり ¥4,620 |
| 取引総額 | ~¥2兆円(国内主導の PE 取引として最大) |
| 最低応募条件 | 株式の ~2/3 (スクイーズアウトのトリガー) |
| TOB 期間 | 2023  年 3 月 → 2023  年 8 月(複数回延長) |
| TOB 結果 | ~78% が応募;コンソーシアムが 2/3  の閾値をクリア |
| スクイーズアウト | TOB 後に株式併合 / プットの権利を通じてスクイーズアウト |
| 上場廃止 | 20,  年 12 月 2023 (東証プライム 6502) |

ファイナンススタックは意図的に**国内のみ**とされ、経産省の安全保障および中核技術保全の懸念に対応した — 東芝の防衛、原子力、半導体、インフラの事業は政治的に機微である。

## 3. なぜ代替案ではなく TOB 非公開化が選ばれたか

2023  前の戦略的レビューは 5 つの経路を評価した:

| 経路 | 却下された理由 |
|---|---|
| 3 分割(2021  計画) | 株主投票が失敗;再構築回避と見なされた |
| 外資 PE への売却 | 経産省 / 政治的抵抗;安全保障の懸念 |
| 戦略的取得者(日立等)への売却 | 期待される価格での現実的な戦略的買い手が不在 |
| 子会社の IPO(キオクシアモデル) | キオクシアの経路は既に尽き、残る事業は IPO の準備が整っていない |
| **国内 PE 主導の非公開化** | **選定** — 再構築を可能にし、国内所有を維持 |

選ばれた経路は、以下を同時に可能にした:(a)アクティビスト株主との摩擦の終結、(b)国内所有 / 支配の保全、(c)公開会社の四半期精査の外での大規模な再構築の滑走路の創出。

## 4. スクイーズアウトの仕組み

TOB が ~2/3  の閾値をクリアした後、コンソーシアムは [[finance/japan-mbo-and-squeeze-out-process|Japan MBO / squeeze-out process]] に基づく標準的な日本のスクイーズアウトのシーケンスを実行した:

1. ~78% 保有での **TOB クリア**
2. 臨時株主総会での **特別決議**(2/3  の閾値をコンソーシアムが満たす)
3. **株式併合** — 残る株主の端株が TOB 同等価格で現金に転換される
4. **反対株主の株式買取請求権** — 少数株主が公正価格の裁判所による決定を求める権利
5. **上場廃止** — 東証プライム 6502  が 20,  年 12 月に上場廃止。2023

 TOB に応募した株主は ¥4,620 で現金を受領。スクイーズアウトで取り込まれた非応募者は、同等価格での現金(反対株主の権利の対象)。反対株主は会社法第 182-4  条以下に基づく買取価格決定の訴訟ルートを持つ。

## 5. 非公開化後の再構築の滑走路

非公開化は、コンソーシアムに(典型的には 3-7  年の)滑走路を与え、以下を可能にした:

| ワークストリーム | ありうる行動 |
|---|---|
| ポートフォリオの再集中 | 非中核事業の売却(複数のより小さい売却) |
| コストの合理化 | 四半期業績のプレッシャーなしに組織的に再構築 |
| 戦略的再集中 | 防衛、エネルギー、インフラ、高マージンの専門分野に集中 |
| 再 IPO の経路 | 最終的に東芝を 5-7  年の時間軸で(部分的または全面的に)再上場 |
| 子会社 IPO | クリーンアップされた子会社の個別 IPO の可能性 |

モデル:控えめなプレミアムで非公開化 → 3-7  年再構築 → より高いバリュエーションで再上場。[[business/softbank-vision-fund-arm-ipo-template|Arm's public mark-to-market]] 戦略の逆である。

## 6. 最近の日本の大型株の非公開化事例の比較

| 事例 | 年 | 価格(¥/株) | 取引規模 | 買い手 | 帰結 |
|---|---|---|---|---|---|
| **東芝** | 2023 | 4,620 | ~¥2兆円 | JIP 主導の国内コンソーシアム | 2023  年 12 月に上場廃止 |
| 日立金属 | 2021-2022 | 2,181 | ~¥800十億円 | ベインキャピタル / 日立の売却 | 2023  に上場廃止 |
| 新光電気工業 | 2024 | 5,920 | ~¥800十億円 | JIC 主導のコンソーシアム | 上場廃止が進行中 |
| JSR | 2024 | 4,350 | ~¥900十億円 | JIC + 外国 LP | 2024  に上場廃止 |
| ベネッセホールディングス | 2024 | 2,600 | ~¥260十億円 | EQT 主導 + 創業家 | 2024  に上場廃止 |

パターン:国内 PE / 戦略的投資家のコンソーシアム(しばしば準政府の JIC の参加を伴う)が大型株の非公開化を率いる。機微なセクターについては、外資のみのバイアウトはますます稀になっている。

## 7. アクティビスト解決テンプレート

東芝は、**アクティビストの摩擦の解決メカニズムとしての**非公開化の教科書的な事例である:

1. **上場という地位がアクティビストのレバレッジを増幅する** — エフィッシモ、エリオット、キングストリートは株主総会決議、臨時総会、ガバナンスレビューを強いることができた
2. **公開会社のガバナンス改革ツール**は尽きていた(取締役会の変更、計画投票等)
3. コンソーシアムと売却するアクティビスト株主の間の**非公開所有のコンセンサス**が循環を終わらせた
4. **現金による退出**により、アクティビストは受け入れ可能な価格でマネタイズし、きれいに退出できた
5. **非公開所有のもとでの再構築**が、四半期開示のプレッシャーから経営陣を解放した

このテンプレートは以下の場合に当てはまる:(a)公開会社が慢性的なアクティビストの摩擦を抱える、(b)公開会社のガバナンスがアクティビストのアジェンダによって実質的に制約される、(c)再構築が複数年の時間軸を要する、(d)現金バイアウトのコンソーシアムがファイナンス可能である。

## 8. 2023 制度前の部分スピンオフ経路との比較

部分スピンオフ制度([[corporate-strategy/partial-spinoff-tax-deferral|see regime]])が 2021-2022,  に、より成熟していたら、東芝はそれを代わりに用いられたか? 主な相違:

| 次元 | 東芝 TOB 非公開化 | 代替案:部分スピンオフ経路 |
|---|---|---|
| アクティビストの退出 | TOB を通じた現金 | 子会社株式を受領(現金ではない) — 現金を望むアクティビストを満足させない |
| 再構築の滑走路 | 非公開所有のもとで複数年 | なし — 親会社と子会社の双方が公開のまま |
| 売却株主への課税 | 現金 → 課税対象 | 株式分配 → 繰延 |
| 買い手に必要な資本 | 多額の現金(¥2兆円) | なし — 純粋に株式構造の再編成 |
| ガバナンスのリセット | あり(非公開の取締役会) | なし(公開会社の取締役会が継続) |

東芝の状況(現金退出を求めるアクティビスト、ガバナンスの抜本見直しが必要)にとっては、TOB 非公開化はスピンオフより構造的に優れていた。ソニー FG(アクティビストの圧力なし、緊急の再構築なし)にとっては、部分スピンオフが TOB より構造的に優れていた。完全なオプション適合の枠組みについては [[corporate-strategy/spinoff-decision-tree-japan|spinoff decision tree Japan]] を参照。

## 9. 経産省「公正な M&A の在り方に関する指針」との相互作用

2023  の取引は、経産省の改訂された「公正な M&A の在り方に関する指針」(2019  の公表、継続的な精緻化)のもとで行われ、これは以下を制約する:

- 公正性を評価する独立社外取締役による特別委員会が必要
- 複数の入札者プロセスが必要(さもなければ理由を説明)
- 独立した財務アドバイザーによるフェアネス・オピニオン
- 利益相反の開示
- 少数株主保護のメカニズム

東芝の TOB プロセスはこれらの指針に準拠した(特別委員会を組成、複数入札者プロセスを実施、フェアネス・オピニオンを取得)。準拠はソフトな要件だが、不準拠は株主訴訟と損害賠償を招く。

## 10. 反論

- ¥4,620  の TOB 価格は、一部の株主から分割価値に比して不十分だと批判された — 買取請求権の訴訟は数年に及びうる
- 国内 PE コンソーシアムモデル(JIC + JIP + オリックス等)は、準政府の資本によって部分的に補助されている;純粋な民間取引であれば、これほど魅力的に価格づけされなかったかもしれない
- 複数年の再構築の滑走路はコンソーシアムの規律を要する;PE 主導の日本の産業再構築の先例はまちまちである
- アクティビストの退出という動機は、非公開化を強いることを狙ったさらなるアクティビストのキャンペーンを招く先例となりうる
- 再 IPO のタイミングは不確実である;市場環境が悪化すれば、コンソーシアムは十分なバリュエーションでの退出に苦労しうる

- JIP 主導のテンプレートは、日本の上場企業に対するアクティビストの圧力に駆動された非公開化をさらに促すか?
