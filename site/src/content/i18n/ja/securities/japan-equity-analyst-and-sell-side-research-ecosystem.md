# 日本株アナリストとセルサイドリサーチエコシステム

## ウィキ上の位置づけ

このページは、[[securities/INDEX|securities index]] および [[securities/japan-underwriting-market-structure|Japan underwriting market structure]] のセルサイドリサーチピアとして [[securities/japan-asset-manager-landscape-matrix|Japan asset manager landscape matrix]] に属します。発行者の開示（[[securities/tokyo-stock-exchange|TSE]]のTDnet/EDINET隣接）、流通（[[securities/japan-best-execution-sor-pts|best execution / SOR / PTS]]のブローカー/オンラインブローカールート）、機関需要（[[securities/japan-asset-manager-landscape-matrix|AM landscape]]および[[securities/japan-prime-brokerage-and-institutional-financing|prime brokerage]]）の間に位置するアナリストのカバレッジ地理について説明します。 JapanFG オペレーター アンカーは、[[JapanFG/nomura-hd|Nomura HD]]、[[JapanFG/smbc-nikko|SMBC Nikko]]、[[JapanFG/daiwa-sg|Daiwa SG]]、[[JapanFG/mizuho-securities|Mizuho Securities]]、[[JapanFG/goldman-sachs-japan|Goldman Sachs Japan]]、[[JapanFG/morgan-stanley-japan|Morgan Stanley MUFG]]、[[JapanFG/jpmorgan-japan|JPMorgan Japan]]、および [[JapanFG/citigroup-japan|Citigroup Japan]] です。日本証券アナリスト協会 (SAAJ / JSAA) と共同の自主規制アンカーは [[JapanFG/jsda|JSDA]] です。

## 要約

日本のセルサイド株式リサーチ市場は、国内では野村、SMBC日興証券、大和証券、みずほ証券、三菱UFJモルガン・スタンレー証券（MUMSS）が主導する5極の寡占であり、ゴールドマン・サックス・ジャパン、モルガン・スタンレーMUFG（MSMSの消費者向けブランド）、JPモルガン証券、日本BofA証券、シティグループ・グローバル・マーケッツ・ジャパン、およびUBS証券の並行グローバルIB層も存在する。そしてバークレイズジャパン。これらの既存企業以外にも、規模は小さいが意味のある独立系調査部門（アストリス・アドバイザリー、日本株リサーチ、Sumeru、さらに小規模な局や個々の公認アナリスト・ブティック）がある。日本のエコシステムは、MiFID II 後のヨーロッパの研究市場とは構造的に異なります。これは、日本が EU の誘導アンバンドル制度を研究の支払いと執行委員会の間の厳密な分離に完全に移行していないためです。その代わりに、FSA の顧客最善の利益の原則 (顧客本位の業務運営) と JSDA 自主規制規則は、英国/EU の同業者よりも明示的な分離が緩やかな原則に基づく行動フレームワークを適用します。リサーチは主にブルームバーグ、リフィニティブ・エイコン/ワークスペース、アルファセンス、企業独自の機関顧客ポータル、および直接のPDF電子メールを通じて配信されます。日本語のセルサイドに関する有意義な調査は、英語のベンダー世界の外にも広まっています。

## このエコシステムが重要な理由

セルサイド株式調査は、発行者の開示と資産運用会社の配分決定の間に位置する分析レイヤーです。

1. **カバレッジは注目を意味します。** セルサイドのアナリストが取り上げない銘柄はマインドシェアが低く、多くの場合、複数の割引で取引され、構造的に高い資本コストに直面します。東証スタンダード/グロース市場名の日本のロングテールは、体系的にカバー率が低いという問題に悩まされています。
2. **イニシエーションおよび格付け変更レポートがフローを推進します。** イニシエーションおよび目標価格の変更は株価を動かします。ブローカー/IB の経済性は、この成果の信頼性、幅広さ、適時性に依存します。
3. **企業アクセス サービスは連携しています。** ほとんどの大企業では、ロードショーの管理、カンファレンス主催、NDR (契約なしロードショー) の予約が経済的に研究制作と結びついています。
4. **引受リーグテーブルの経済性は相互作用します。** あるセクターに関する強力な調査報道が、そのセクターにおける引受義務を裏付けています（これは、FSA/JSDA の行為監督が管理するために存在する対立表面です）。
5. **スチュワードシップ/エンゲージメントのワークフローは調査へのアクセスに依存します。** 資産運用会社のエンゲージメント チームは、議決権行使および対話の優先事項へのインプットの 1 つとしてセルサイド カバレッジを使用します。

### 野村

- **事業体**: [[JapanFG/nomura-hd|Nomura Holdings]] / 野村證券株式会社;研究は、国境を越えたコンテンツについては野村證券 (NS) および野村インターナショナルの下で公開されます。
- **アナリスト数（日本株カバレッジ）**: 国内最大級のアナリスト名簿。日本の住宅の中で最も広範な東証プライム/スタンダード/グロースの名前をカバーします。正確な従業員数は公表されていません。ファームディレクトリの現在の名前を確認してください。
- **対象セクター**: 金融 (銀行 / 保険 / ノンバンク金融)、テクノロジー、自動車 / 工業、ヘルスケア、インターネット / メディア、不動産 / REIT、通信、消費者、エネルギー / 公益事業、中小型株を含む包括的。
- **マクロ/戦略**: 長年確立されているマクロおよび株式戦略チーム。資産を横断したカバレッジ。
- **特徴**：国内最大規模の研究工場。歴史的セクターの深い記憶。生産物を分配するための組織の営業力の規模。

### SMBC日興

- **エンティティ**: [[JapanFG/smbc-nikko|SMBC Nikko Securities Inc.]] (SMFG グループ)。
- **アナリスト数**: 国内トップクラス。包括的な日本株カバレッジ。意味のある戦略とマクロの出力。
- **対象分野**: 広範囲。銀行/金融（SMFGグループとの隣接性により慎重な紛争管理が必要）、不動産/REIT、自動車、消費者に強い。
- **特徴的な機能**: SMFG の銀行グループとの隣接性により、企業銀行業務由来のセクター インテリジェンスが提供されます。行為管理フレームワークにより、研究と銀行業務が分離されます。

### ダイワ

- **主体**: [[JapanFG/daiwa-sg|Daiwa Securities Group]] / 大和証券株式会社
- **アナリスト数**: 国内トップクラス。幅広い宇宙をカバーします。
- **対象分野**: 包括的。産業/自動車、銀行、不動産における歴史的な強み。
- **特徴**: 独立系証券グループ (銀行所有ではない) の伝統。機関投資家との長年にわたる関係。米国/ヨーロッパ/アジアへの有意義な国境を越えた流通。

### みずほ証券

- **事業体**: [[JapanFG/mizuho-securities|Mizuho Securities Co., Ltd.]] (みずほFGグループ)。
- **アナリスト数**: 国内トップクラス。株式と並んで強力なマクロと金利/為替戦略。
- **対象分野**: 広範囲。金融、エネルギー/公益事業、自動車、不動産に強み。
- **特徴**: みずほFG銀行グループとの隣接性。ストラクチャードプロダクト/クロスアセットリサーチの統合。

### MUMSS / 三菱UFJモルガン・スタンレー

- **事業体**: 三菱 UFJ モルガン・スタンレー証券株式会社 (MUMSS) は、[[JapanFG/morgan-stanley-japan|Morgan Stanley]] と MUFG の合弁会社として運営される総合ホールセール/機関投資家向け証券事業です。消費者/小売ブランドはモルガン・スタンレー MUFG 証券です。
- **アナリスト数**: トップクラス。モルガン・スタンレーの世界的なアナリストの統合と国内のMUFGグループのリーチの両方から恩恵を受けることができます。
- **対象分野**: 広範囲。モルガン・スタンレーのプラットフォームを介してグローバル/日本ビューを統合。
- **特徴**: 国内唯一のグローバルIB JV体制を持つ5本柱住宅。モルガン・スタンレーのシームレスなグローバルコンテンツ配信と日本固有の出力を可能にします。

## 独立した研究部門

独立した（セルサイドと提携していない）リサーチは、日本では規模は小さいですが成長している分野です。

| 会社 | キャラクター |
|---|---|
| アストリスアドバイザリージャパン | 長年にわたる独立した日本株調査。テーマ別の管理指向の作品で尊敬されています。 |
| 日本株リサーチ（JER） | 日本株に焦点を当てた独立したファンダメンタルズ調査。 |
| スメル | 日本とアジアのテーマ/分野の研究をカバーする独立した調査局。 |
| 嵐の研究 | 独立したグローバル/日本株調査。 |
| Smartkarma の寄稿者 (日本中心) | 日本取材協力者によるプラットフォームベースの独立系洞察プロバイダー。 |
| ブティック公認アナリスト（証券アナリスト）の実務 | 個人または小規模チームがサブスクリプション/マンデートベースで機関顧客にサービスを提供します。 |

日本における独立系セグメントの経済的制約は、機関投資家にリサーチ費用の別途支払いを強制する強制的なMiFIDスタイルのアンバンドリングルールがないことである。強制的な分離がなければ、機関投資家バイヤーはセルサイドの無償補償に頼ることができ、EUのバイヤーのように独立系リサーチのために個別に予算を組む必要はないかもしれない。

## 規制の境界: 誘導と最良執行の背景

日本のセルサイドリサーチの規制枠組みは、重要な点で EU の MiFID II 誘導制度とは異なります。

| 次元 | EU MiFID II | 日本 |
|---|---|---|
| 研究を実行委員会から厳密に分離する | はい (SME 研究向けに後でソフト化されます) | EU の必須形式ではありません |
| 誘導ルール | 詳細で規範的な支払いベースの分離が必要 | 原則に基づく。金融庁の顧客最善の利益の期待（顧客本位の業務運営）とJSDA自主規制の枠組み |
| クライアントへの開示 | 詳細な定量的研究予算の内訳 | ゆるめに。ブローカーポリシーレベルの開示 |
| 研究支払いアカウント (RPA) | 確立された慣行 | 標準ではありません |

実際的な影響: 日本の資産運用会社は通常、MiFID 後の欧州の AM のように個別の研究予算を維持しません。セルサイドのリサーチ消費はブローカー関係に組み込まれ、約定手数料、プライムブローカー手数料、企業アクセスサービスを通じて効果的に支払われます。行為監督は、ハードペイメントの分離ではなく、紛争管理（例えば、非公開の方法で引受業務のサポートに研究が使用されることの防止）に重点を置いています。

国境を越えた複雑な問題は、EUの顧客と取引する日本の資産運用会社、またはEUのビークルを通じて業務を行う日本の資産運用会社は、EU側のMiFID II由来の誘導制約の対象となる可能性があり、その活動部分については別個の研究予算の取り扱いが必要となる可能性がある。したがって、日本のブローカーのリサーチデスクは、通常の日本の顧客との関係と並行して、MiFID 準拠の価格設定メニューを維持しています。

## JSAA / 証券アナリスト資格

日本証券アナリスト協会 (公益社団法人日本証券アナリスト協会、SAAJ / JSAA) は、日本のセルサイドおよびバイサイドのアナリストの間で最も有力なアナリスト資格である証券アナリスト (認定会員アナリスト) 資格を管理しています。関連性:

- CMA / CIIA 資格は、標準化されたアナリスト トレーニングと倫理規定フレームワークを提供します。
- この資格は、[[securities/japan-asset-manager-landscape-matrix|domestic asset managers]] およびブローカー/IB 研究チーム内で広く保持されています。
- SAAJ は懲戒手続きと CPE 要件を実行します。これは、世界的な CFA 指定に最も近い日本の機能的類似物です (相互主義の取り決めがいくつかの形式で存在します)。
- SAAJ は、業界調査、アナリストのガイドライン、スチュワードシップ関連の資料も発行しています。

## 順位表と競争力のある評価

日本のセルサイドリサーチリーグ表は、以下によって定期的に発行されます。

- **機関投資家 (II) 全国調査チーム調査** — バイサイド投票によるセクター別の年次ランキング。
- **日経ヴェリタスのアナリストランキング** — 日本語のバイサイド調査。
- **Greenwich Associates / Coalition / その他の業界分析** — 研究支出のシェア / 機関顧客の投票指標。

注意: 調査方法は異なります。方法論を調整せずに、調査間のランキングを組み合わせるべきではありません。

## 紛争と監督

主な競合パターン:

1. **リサーチ銀行との対立** 発行会社が引受顧客である株式の格付けを発行するリサーチアナリスト。緩和策: 中国の壁の手順、出版前のレビュー管理、制限リストの管理。
2。 **調査とプライム・ブローカレッジと約定の競合。** カバレッジの強度は、対象銘柄の最大の機関保有者からの約定手数料およびプライム・ブローカー手数料の流れと相関します。 「[[securities/japan-prime-brokerage-and-institutional-financing|prime brokerage]]隣」を参照してください。
3. **企業アクセスの競合** 効果的に誘導として機能する方法で、発行会社 NDR / 経営者会議へのアクセスを優先機関顧客に割り当てる。
4. **発行者と関係の競合** 対象となる名前を格下げしたアナリストは、発行者管理へのアクセスを失います。市場規律への反応は、企業文化によって異なります。
5. **グループ銀行と保険引受関連会社の競合** 銀行グループと提携している国内の 5 極会社の場合、法人銀行の顧客と株式調査の対象が重複する可能性があります。

金融庁の顧客最善の利益の原則、JSDA の自主規制規則、および各院の文書化された研究方針が一緒になって管理フレームワークを形成します。金融庁/JSDA の検査や懲戒結果において、公的監視が表面化することがあります。
