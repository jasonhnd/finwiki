# NISA 2024 flow


## ウィキ上の位置づけ

このエントリは [[securities/INDEX|securities index]] の下にあります。ピア/コントラストのコンテキストについては [[securities/japan-securities-clearing-corp|日本証券クリアリング機構 (JSCC)]]、より広範なシステム/規制の境界については [[securities/financial-instruments-business-operators-japan-index|financial instruments business operators japan index]] と照らし合わせて読み取ってください。

## 要約

2024 NISA 改革は、日本の家計の非課税投資口座を恒久的な資産形成制度に変えました。 FSA の NISA サイトによると、新しい制度は 2024  1 月に開始され、恒久化されたとのことです。公式ガイドブックでは、「つみたて投資枠」と「成長投資枠」の2つの利用可能なバケットを設定しており、年間投資枠を合わせて3.6 00万円、生涯非課税保有枠を18 00万円としている。

FinWiki の場合、NISA は証券流通と家計資産フローのページです。 [[JapanFG/rakuten-securities]]などの個人証券会社、[[JapanFG/nomura-hd]]などの総合証券会社、[[JapanFG/mizuho-securities]]、[[JapanFG/mufg-mums]]などの銀行系証券部門を接続します。

## システムマップ

| レイヤー | 2024 NISA 読み取り |
|---|---|
| 政策目標 | 家計の金融資産を長期的な資産形成にシフトする。 |
| アカウント層 | 金融機関を通じて対象者1人につき1つのNISA口座が開設されます。 |
| 製品層 | 長期分散投資信託のつみたて枠。上場株式・投資信託の成長枠（除外あり）。 |
| 税レイヤー | 新制度では非課税保有期間は無期限となる。 |
| ディストリビューション層 | 証券会社、銀行、オンライン証券会社は、口座開設、定期投資、資産保有をめぐって競争しています。 |

## 主要パラメータ

| 項目 | 新 NISA の読み方 |
|---|---:|
| 年間つみたてノルマ | 1.2 百万円 |
| 年間成長ノルマ | 2.4 百万円 |
| 合計年間割り当て量 | 3.6 百万円 |
| 生涯非課税保有限度額 | 18 百万円、書籍原価ベース |
| 成長クォータのサブリミット | 12 万円（生涯限度額内） |
| 非課税保有期間 | 無期限 |
| システム用語 | 永久的 |
