# 通貨ベーシス・スワップ(日本に焦点)

## 要約

通貨ベーシス・スワップ(CCBS)は、2つの相手方が2つの異なる通貨で変動金利の利払いを交換し、当初に元本も交換され満期に再交換されるOTCデリバティブである。「ベーシス」とは、カバー付き金利平価からの乖離を調整するために一方のレッグ(慣行上は非USDレッグ)に提示されるスプレッドである。JPY-USDベーシスは、世界の市場で最も注目されるFX調達のシグナルの一つである。なぜなら、それは日本の投資家(保険会社、銀行、アセットマネージャー)がUSD資産を円にヘッジして戻す際に直面するコスト、およびグローバルな借り手が円の調達にアクセスする際のコストを反映するからである。

本項目は、[[derivatives/INDEX|derivatives index]] の中の [[derivatives/corporate-fx-hedge-policy-japan-listed|corporate FX hedge policy]] の、より長期の調達側のいとこであり、より広いベーシス・カーブの見方については [[derivatives/yen-basis-swap-market|yen basis swap market]] と密接に関連する。

## ウィキ上の位置づけ

本項目は [[derivatives/INDEX|derivatives index]] の中に、通貨ベーシス・スワップの解説として位置する。ベーシス・カーブの見方については [[derivatives/yen-basis-swap-market|yen basis swap market]]、生命保険会社の需要側については [[insurance/japan-life-insurance-alm-overview|Japan life-insurance ALM overview]]、短期の調達の文脈については [[money-market/japan-money-market|Japan money market]] と照らし合わせて読まれたい。

## メカニクス

標準的なJPY-USD 5年のCCBSは次のように機能する:

| ステップ | アクション |
|---|---|
| t=0 | 相手方Aが相手方Bに円の想定元本を渡す;Bは、その時のスポットレートでUSDの想定元本をAに渡す。 |
| t=0 → T | AはUSD変動(通常 3か月のUSD SOFR + スプレッド)をBに支払う;BはJPY変動(通常 3か月のJPY TONA / LIBOR相当 + ベーシス・スプレッド)をAに支払う。 |
| t=T | 元本が当初のスポットレートで再交換される(満期のレートではない)。 |

ベーシス・スプレッドは、スワップを当初にゼロNPVにする、非USD変動指標の上に適用される調整である。慣行上、JPY-USDベーシスはJPYレッグの支払者側で支払われるスプレッドとして提示される:-50bpの「マイナスのベーシス」は、JPYレッグの支払者がJPY変動から 50bp 少なく支払うことを意味し、これは、JPYの調達者がUSDを手に入れるために 50bp の利回りを諦める必要があることに対応する。ベーシスが広い(よりマイナスである)ほど、円が豊富なエンティティがヘッジ後ベースで円をドルにスワップするのはコストが高くなる。

[[derivatives/yen-basis-swap-market|yen basis swap market]] ページは、期間をまたぐ期間構造をより詳細に扱う。

### 想定元本、担保、信用の交換

典型的なバイラテラルのISDAに基づいて文書化されたJPY-USD CCBSでは、キャッシュフローのタイムラインは次を含む:

- t=0 での当初の元本交換(スポットレート)。
- 各レッグでの定期的な利払い(通常は四半期、関連する変動の参照に連動)。
- t=Tでの最終的な元本の再交換、当初の交換レートで(満期時のスポットレートではない)。
- MTMを担保化するための、取引の存続期間を通じたCSA駆動の変動証拠金の移動。

元本交換が当初のスポットレートを用いるため、取引は、相手方が契約の存続期間中にデフォルトした場合に相当な元本再構築コストを持つ。CSAの担保化はこのリスクを緩和するが、オペレーション上および流動性上の要求を導入する。担保通貨の選択(USD対JPY)および再担保化の権利は、商業的に重要な条件である。

## ベーシスがゼロから乖離する理由

カバー付き金利平価(CIP)は、裁定がない下では、ベーシスはゼロであるべきだと述べる。実際には、世界金融危機後、CIPは非常に不完全にしか成立してこなかった。持続的なJPY-USDベーシス(過去10年の大半でマイナス)は次を反映する:

1. FXヘッジ後のUSD資産を必要とする日本に拠点を置く保険会社、銀行、アセットマネージャーからの構造的なUSD需要。
2. バーゼルIII / LCR / NSFR後のUSDの貸し手に対するバランスシートの制約。
3. 四半期末および年末のバランスシート・ウィンドウの効果。
4. 米国の銀行ディーラーに対するレバレッジ比率の上限による、裁定チャネルを通じた限定的なUSD供給。
5. 異なる 2008 年以降の金融政策のレジーム(日銀がほぼゼロ金利である一方、Fedが正常化した;キャリー対ヘッジコストのミスマッチ)。

BISのQuarterly Reviewシリーズは、CIPの乖離を広範に追跡してきた。このパターンはEUR-USD、GBP-USD、AUD-USDのベーシスにわたって繰り返されるが、JPY-USDは、USDのリターンを見出す必要のある円建ての貯蓄の深いプールがあるために、構造的に最も持続的に広いものの一つである。

## JPY-USD 対 EUR-USD のベーシスのダイナミクス

両ベーシスとも持続的にマイナスである(USD需要がCIPが含意する供給を超える)が、JPY-USDはEUR-USDよりも持続的かつ広い傾向がある。その差は次を反映する:

- ユーロ圏の同等のものと比べた、USD資産を求める日本の円の貯蓄の絶対的なプールの大きさ。
- ECB / ユーロシステムのオペレーションと日銀のオペレーションは範囲が異なる;[[money-market/boj-open-market-operations|BoJ open market operations]] 参照。
- 法域をまたぐ生命保険会社のALMフレームワークにおける、FXヘッジ後キャリーの異なる規制上の取扱い。

両ベーシスは、世界的なUSD調達のストレス・エピソード(例:2020 年3月のコロナ)の間に共に動くが、JPY-USDベーシスは最も急激に拡大し、最大の中央銀行のスワップライン引出しを要したものであった。

### ベーシスの期間構造

JPY-USDベーシスの期間構造が平坦であることはまれである。典型的なパターン:

| 期間 | 典型的な相対水準 |
|---|---|
| 1週-1か月 | 最もボラタイル;ローカルな調達イベントで急激に動く。 |
| 3か月 | ベンチマーク;大半の市場データフィードで報告される。 |
| 1年 | ロールフォワードのヘッジコストを反映;生命保険会社の財務部門が注目する。 |
| 3年-5年 | 企業 / ソブリンの発行体からの構造的な調達需要を反映。 |
| 7年-10年 | 流動性が低い;長期の保険 / 年金のヘッジに駆動される。 |
| 20年-30年 | 薄い流動性;主にオーダーメイドの取引。 |

ストレス時には形状が反転しうる:短期が急激に拡大する一方、長期はよりタイトなまま保たれる。このパターンは、生命保険会社が短期ローリング(通常は安いが、ストレス時には高い)でヘッジするか、長期ロック(通常は高いが、ストレス時には安定)でヘッジするかを決める上で意味がある。

## 年末 / 四半期末の拡大

CCBSのベーシスは、四半期末、とりわけ年末に、ディーラー銀行が報告日に向けてバランスシートの提供を減らすため、常習的に拡大する。パターンは次のとおり:

| 期間 | 典型的なJPY-USD 3か月ベーシスの挙動 |
|---|---|
| 四半期の半ば | 現に存在する構造的水準の周りでレンジ取引。 |
| 四半期の最後の 2 週間 | ディーラーが引き上げるにつれて急激な拡大(よりマイナスに)。 |
| 新四半期の最初の週 | 構造的水準へ向けてスナップバック。 |
| 年末(12月) | 暦年の最大の拡大。 |

このパターンはBISのQuarterly Reviewの研究で十分に記録されており、スポット対フォワード含意のベーシス・カーブに反映される。日本の銀行(とりわけ [[megabanks/mufg-bank|MUFG Bank]]、[[megabanks/sumitomo-mitsui-banking-corp|Sumitomo Mitsui Banking Corp]]、[[megabanks/mizuho-bank|Mizuho Bank]])はJPY-USDベーシス市場の支配的な相手方であり、これらのダイナミクスの両側を感じる。

## 中央銀行のスワップライン

Fedと少数の中央銀行のグループ(日銀、ECB、BoE、SNB、カナダ銀行)との間の恒久的 / 常設のFXスワップラインは、USD調達のストレスに対する最後の手段としてのシステムのバックストップである。メカニカルには:

- Fedは、OISに連動するレートで、JPY担保に対して日銀にUSDを貸す。
- 日銀は、オークション(プールされた担保に対する米ドル資金供給オペ)を通じて、日本の銀行にUSDを転貸する。
- このメカニズムは、日本の銀行が代替のUSD供給源を持つため、ストレス時にCCBSのベーシスが実際にどれだけ拡大しうるかに上限を設ける。

日銀-Fedのスワップラインの引出しは、2008-2009 年の危機の間、そして再び 2020 年3月-4月のコロナの調達ショックの間に急増した。各エピソードの後、引出しはゼロに正常化し、常設のファシリティはバックストップの地位に戻った。

[[money-market/INDEX|money-market index]] は、米ドル資金供給オペを含む、より広い日銀の流動性管理フレームワークへの入口である。

### 日銀の米ドル資金供給オペのオペレーション上のメカニクス

発動されると、このオペは通常次のように機能する:

| ステップ | アクション |
|---|---|
| 1 | 日銀がプレスリリースを通じてオークションのパラメータ(規模、期間、頻度)を公表する。 |
| 2 | 適格な日本の相手方(主にメガバンクと主要な地方銀行 / 信託銀行)が、プールされたJGB担保に対して応札する。 |
| 3 | 応札レートに対する配分(またはストレスモードでは固定レート、フルアロットメント)。 |
| 4 | コルレス銀行の取決めを通じて、入札者の口座にUSDが引き渡される。 |
| 5 | 満期にUSDが返済される(ストレス期には通常 7日、84日、または 3か月の期間)。 |

このファシリティは 2020-Q1 年 / Q2 年に、7日、14日、84日のオペで盛んに用いられた;総残高は正常化の前に USD 200億超でピークに達した。このファシリティの存在自体が、JPY-USDベーシスがストレス時に無制限に拡大することを許されるのではなく「アンカーされている」理由の一部である。

## USDプロジェクトのための法人の円調達

円建ての社債を発行する非日本の多国籍企業は、CCBSを用いて、元本のFXリスクを取ることなく代金をUSDに転換できる。その構造:

1. JPY固定クーポンで 5年のサムライ債を発行する。
2. CCBSを締結する:JPY固定(社債に対応)を支払い、USD変動 + ベーシス調整を受け取る。
3. USDの代金を意図したプロジェクトに用いる。

オールインのUSD調達コストは、現に存在するベーシスで調整された、USDにスワップされた円債のクーポンに等しい。JPY-USDベーシスが大きくマイナスである場合、このルートは、発行体にとって直接のUSD債の発行より安くなりうると同時に、ディーラーの相手方(通常は日本のメガバンク)に収益性のあるベーシス・エクスポージャーを与える。

## 生命保険会社のUSD債のヘッジコスト

日本側からのCCBSヘッジの支配的な構造的買い手は、日本の生命保険セクターである。生命保険会社は、国内のJGBに対する利回りの上乗せとして、大規模なUSD建ての債券ポートフォリオを保有している([[insurance/japan-life-insurance-alm-overview|Japan life-insurance ALM overview]] 参照)。それらの保有のFXリスクを排除するために、彼らは次の組み合わせを用いる:

- ローリングの 3か月のFXフォワード。
- 1年のFXフォワード。
- 安定した構造的ヘッジのための長期CCBS(3年、5年、10年)。

オールインのヘッジ後USD利回りの合計は、USD債のクーポン - ヘッジコスト(フォワードポイントとベーシス・スプレッド)に等しい。大きくマイナスのJPY-USDベーシスの期間(例:2010年代後半および 2022-2023年)には、10年の米国債のオールインのヘッジ後利回りは、JGBに対してほぼゼロに縮小するか、さらにマイナスに転じうる。これは、生命保険会社のJGBまたはヘッジなしのUSD債への再配分を促す。このフローのシフトは、世界の債券市場で最も注目されるダイナミクスの一つである。

### 数値による例示

メカニクスを具体的にするための様式化された例。4.5% 利回りの 10年の米国債と、1.0% の 10年のJGBを考える。素朴なキャリーは 350bp である。次にヘッジコストを重ねる:

| 構成要素 | おおよその値(例示) |
|---|---|
| 10年のUST利回り | 4.5% |
| 含意される 3か月ローリングのFXフォワードのヘッジコスト | 〜2.5-3.0%(短期金利差に駆動される) |
| 加えて 5年のJPY-USD CCBSベーシス・スプレッド | 〜30-50bp(マイナス) |
| オールインのヘッジ後USD利回り | 〜1.2-1.7% |
| 10年のJGB利回り | 〜1.0% |
| JGBに対する正味の上乗せ | 〜20-70bp |

正確な数値は時間とともに大幅に変動するが、パターンは一貫している:JPY-USDベーシスが広く短期金利差が大きい場合、オールインのヘッジ後USD利回りはJGB利回りに向けて収斂し、ヘッジ後USD債を保有することのオペレーション上の複雑さと信用エクスポージャーに対する追加の報酬はほとんど残らない。これは、生命保険会社のALM委員会において繰り返される意思決定のポイントである。

## 市場参加者とフローの分類

JPY-USD CCBS市場は構造的に偏っている。典型的なフローの分類:

| フローの方向 | 典型的な参加者 | ポジション |
|---|---|---|
| USD変動を支払い、JPY変動 + ベーシスを受け取る | ヘッジする日本に拠点を置くUSD資産の保有者 | 正味のUSDの借り手(合成) |
| JPY変動 + ベーシスを支払い、USD変動を受け取る | 合成の円を調達する非日本の企業 | 正味の円の借り手(合成) |
| ディーラー(メガバンク / グローバルバンク) | 両側を仲介する | リスク限度の下で残余のベーシス・エクスポージャーを保有 |
| 投機的(ヘッジファンド) | ベーシスの縮小または拡大の見方を取る | ストレス期には小さいシェア、平穏期にはより大きい |

構造的な不均衡 — 非日本の円調達の需要よりもはるかに多くの日本に拠点を置くUSD資産のヘッジ需要 — が、ベーシスがCIPのゼロに回帰するのではなく持続的にマイナスに位置する理由を説明する。

### 主要なディーラーのフランチャイズ

ディーラーの参加は集中している:

- [[megabanks/mufg-bank|MUFG Bank]]:相当なバランスシートと日本の法人フランチャイズ;JPY-USDベーシス市場における長年のPB / ディーラー。
- [[megabanks/sumitomo-mitsui-banking-corp|Sumitomo Mitsui Banking Corp]]:同様のプロファイル;SMFGのグループ全体の事業横断のシナジー。
- [[megabanks/mizuho-bank|Mizuho Bank]]:強い法人 / 金融機関のディーリングのフランチャイズ;深いJGB担保の統合。
- Citi、JPMorgan、Goldman Sachs、Morgan Stanley:支配的な外国ディーラー;とりわけメガバンクのバランスシートが制約されるストレス期に活発。
- Nomura、Daiwa:とりわけ生命保険会社関連のフローおよびより長期の構造において参加する。

この集中は価格のばらつきの機会を生む:複数ディーラーのパネルを稼働させる日本の企業または保険会社は、競争的に価格を調達することで、大きな想定元本の取引について数ベーシスポイントの価格改善を捉えうる。

## 規制とドキュメンテーションのフレームワーク

CCBSの取引は、CSAの担保化を伴うISDAマスター契約の下で文書化される。具体的な特徴:

- ISDA 2002 年のマスターが典型的な参照文書である;一部の日本の相手方は日本法の相当物も用いる。
- CSAの条件は、担保通貨(JPY対USD)、適格な担保の種類、しきい値、最低引渡し額、評価方法を規定する。
- 2008 年以降の改革は、標準化されたCCBSを、適格な場合に中央清算(CME、LCH SwapClear)へ押し進めたが、日本のCCBSの清算の採用は、USD IRSよりも遅い。
- 日銀およびFSAは、FIEAのデリバティブ報告ルールの下での取引情報蓄積機関への報告を通じて、システミックなエクスポージャーを監視する。
- バーゼルIIIの下での資本の取扱い(相手方信用リスク、CVAチャージ)は、とりわけ非清算の取引について、ディーラーの価格づけに影響する。

経済的な効果は、バイラテラルの非清算CCBSの価格づけが、完全に清算され完全に担保化された環境で支配的となるであろうものから、組み込まれた資本チャージがベーシスを拡大させる、というものである。
