# EBノックイン型仕組債（日本リテール）

## 要約

EB（*Equity Bond*）ノックインは、2022まで日本のリテール・ウェルスマネジメントで最も多く販売された仕組商品形式である。仕組み上、購入者は単一の日本株（またはワーストオブ・バスケット）を対象とする **ダウン・アンド・イン・プット・オプション** を売っているのと同等であり、その見返りとして高いクーポンと、早期償還型では早期償還の可能性を受け取る。表面クーポン（通常は年率 5-15 percent）は、利回りを求めるリテール層を引き付けた。**テールリスク** は二値的である。最も悪いパフォーマンスの参照資産がノックイン・バリア（通常は当初価格の 50-70 percent）を割り込むと、投資家は額面償還ではなく、当初行使価格で株式（または現金相当額）を受け取る。単一イベントで元本の 30-70 percent を失うことは、実際に発生した結果である。**2022-2023 の FSA による取締り** とその後の JSDA 自主規制強化は、EB ノックイン販売をリテール保護上の最優先課題として対象にした。[[JapanFG/nomura-hd|Nomura]]、[[JapanFG/smbc-nikko|SMBC Nikko]]、[[JapanFG/daiwa-sg|Daiwa]] の子会社に対する行政処分により、主要販売会社はリテール向け EB ノックイン販売の停止または制限を余儀なくされた。

## ウィキ上の位置づけ

この項目は [[derivatives/INDEX|derivatives index]] 配下にあり、日本のリテール向け仕組商品の中で最も影響が大きい商品形式を扱う、最も深い単一商品ページである。[[derivatives/structured-bond-japan-retail-issuance|structured bond Japan retail issuance]] とあわせて読むとより広い販売実態、[[derivatives/japan-cds-market-overview|Japan CDS market overview]] はクレジット・デリバティブの文脈、[[derivatives/cds-japan-corporate-spread-mechanics|Japan corporate CDS spread mechanics]] は発行体信用の側面、[[derivatives/japan-irs-market|Japan IRS market]] は EB 価格決定に使われる金利の下地、[[derivatives/yen-basis-swap-market|yen basis swap market]] は調達カーブの文脈が分かる。

より広い資本市場の位置づけは [[finance/INDEX|finance index]]、類似した数理を使う機関投資家向け株式リンク商品の対照は [[finance/japan-convertible-bond-mechanics|Japan convertible bond mechanics]]、関連する単一銘柄の流動性ダイナミクスは [[finance/japan-cross-shareholding-unwinding-economics|cross-shareholding unwinding economics]]、メガバンク系の販売文脈は [[banking/INDEX|banking index]]、機関投資家の仕組商品配分との対比は [[insurance/japan-life-insurance-alm-overview|Japan life insurance ALM]]、ディーラーのヘッジ側は [[securities/japan-prime-brokerage-and-institutional-financing|prime brokerage and institutional financing]] を相互参照する。

### 基本的な商品構造

| 構成要素 | 典型的なリテール販売 EB ノックイン |
|---|---|
| 形式 | 株式デリバティブを内包した、SPV または銀行発行の債券 |
| 通貨 | JPY（最も一般的）または USD（為替オーバーレイ型） |
| 満期 | 1-5 年（通常 3 年）。早期償還条項により短縮される |
| 原資産 | 単一の日本上場株、または 3-5 銘柄のワーストオブ・バスケット |
| 当初参照価格 | 条件決定日の終値（S₀） |
| ノックイン・バリア | S₀ の 50-70 percent（最も一般的には 60 percent） |
| 早期償還判定水準 | 観測日における S₀ の 100 percent |
| クーポン | 年率 5-15 percent。ノックインが発生しない場合のみ支払われる「条件付きクーポン」 |
| ノックイン観測 | 期間中の連続観測（日中接触を含む）、または満期時のみの欧州型観測 |
| 償還 | ノックインなしなら額面償還。ノックイン時は行使価格で株式交付（または現金相当額） |

### ワーストオブ・バスケット型

| 特徴 | 詳細 |
|---|---|
| 原資産数 | 通常 3-5 銘柄 |
| ワーストオブ・ルール | 最もパフォーマンスの悪い参照資産によりペイオフが決まる |
| 相関の影響 | 相関が低いほど、いずれかの資産がバリアを割り込む確率が高まり、クーポンも高くなる |
| 分散効果 | 直感に反し、原資産が増えるほど分散ではなくワーストオブ発生確率が高まる |

ワーストオブ・バスケットは、リテール向け商品の中でも特に批判が強い特徴の一つである。投資家はバスケットを「分散」と受け止めがちだが、実際には下方リスクの集中である。

### 早期償還条項

| 観測日 | 参照価格が早期償還判定水準以上の場合の処理 |
|---|---|
| 1 年目観測 | 参照価格 ≥ S₀ なら、額面 + 経過クーポンで自動早期償還 |
| 2 年目観測 | 同じトリガー |
| 3 年目観測（満期） | 同じトリガー、または最終償還ルールを適用 |

早期償還は発行体 / ディーラーにとってショート・ボラティリティのエクスポージャーを早期に終了できる利点があり、リテール購入者には早い資金回収と再投資機会を与える。

### 投資家ポジションの等価関係

```
EB ノックイン債のロング ≈ 債券ロング + 原資産のダウン・アンド・イン・プット・オプションのショート
```

このショート・プット・オプションが次の源泉である。

- 高いクーポン（受け取ったオプション・プレミアム）。
- テールリスク・エクスポージャー（バリア下での二値的な損失プロファイル）。

### 満期時のペイオフ・シナリオ

単一銘柄 EB で、S₀ = 1000, 、ノックイン・バリア = 600 （60 percent）、行使価格 = 1000, 、元本 = 1,000,000 JPY と仮定する。

| シナリオ | 最終株価 | 結果 |
|---|---|---|
| 株価が全期間を通じてバリアを上回った（例：850） | 850 （ノックイン事由なし） | 1,000,000 JPY の額面償還 + 全クーポン支払い |
| Stock breached barrier intraday (e.g., touched 580) but recovered to 800 | 800 (knock-in triggered) | Conditional final coupons forfeited; redemption in shares: 1,000,000 / 1000 = 1000 shares delivered, worth 1000 × 800 = 800,000 JPY (20 percent loss vs principal, plus lost coupons) |
| 株価がバリアを割り込み低位にとどまった | 500 （ノックイン発生、低位で終了） | 1000 株を 500 で交付 = 500,000 JPY（元本比 50 percent の損失、加えてクーポン喪失） |
| 株価が大幅にバリアを割り込んだ | 200 （ノックイン発生、大幅下落） | 1000 株を 200 で交付 = 200,000 JPY（元本比 80 percent の損失、加えてクーポン喪失） |

### ノックイン観測方式の重要性

| 観測方式 | 影響 |
|---|---|
| 連続（any-time）ノックイン | 最も厳しい方式。日中接触だけでも発動する。古い日本リテール EB で一般的 |
| 日次終値観測 | より緩やかで、終値だけが問題になる |
| 欧州型（満期時のみ）観測 | 最も緩やかで、最終価格だけが問題になる |

連続ノックインは欧州型より実質的にリスクが高いが、多くのリテール購入者はこの違いを理解していなかった。

## ディーラー側の価格決定要素

ディーラーは次の要素に基づいて組込デリバティブを価格決定する。

| 入力項目 | 債券クーポンへの影響 |
|---|---|
| 原資産のインプライド・ボラティリティ | ボラティリティが高いほど、オプション・プレミアムが高まり、クーポンも高くなる |
| 相関（ワーストオブ） | 相関が低いほど、ワーストオブのボラティリティが高まり、クーポンも高くなる |
| 原資産の配当利回り | 配当が高いほどコール価値は低下し、構造によってクーポンを下げる場合も上げる場合もある |
| 原資産の借株コスト | 借株コストが高いほどコール価値は低下し、早期償還確率に影響する |
| 参照発行体の信用スプレッド | 発行体信用が高いほど債券フロアは低くなり、クーポンが調整される |
| 無リスク金利カーブ | 金利上昇は方向により債券フロアを上げる / 下げる |
| バリア水準 | バリアが低いほどオプション価値は小さくなり、クーポンも低くなる |
| 早期償還判定水準 | 判定水準が低いほど早期償還確率が高まり、クーポンも高くなる |
| 債券満期 | 満期が長いほどオプション価値が大きくなり、クーポンも高くなる |

### ディーラー手数料負担

リテール購入価格は、公正価値にアレンジャー手数料と販売会社手数料を加えたものである。日本のリテール EB に関する業界集計分析では、発行時に想定元本の 3-10 percent 程度の総手数料負担が組み込まれていたことが示唆されており、最も複雑なワーストオブ構造ではさらに高い場合もあった。古い販売資料では、この手数料負担は透明に開示されていなかった。

### ヘッジ

ディーラーはバック・トゥ・バックでヘッジする。

| リスク | ヘッジ手段 |
|---|---|
| 株式デルタ | 原資産株式のショート |
| 株式ガンマ（バリア接近） | 株式または上場オプションによる能動的ヘッジ |
| 株式ベガ | 上場または OTC の株式オプション・ポジション |
| バリアリスク（ギャップダウン・シナリオ） | ヘッジが難しく、ディーラーが在庫として抱える |
| 相関（ワーストオブ） | OTC 相関取引。ディーラーが在庫として抱える |
| 発行体信用 | 発行体自身の債券 / CDS によるヘッジ |
| 金利 | スワップ・ヘッジ |

多くの債券が同じ参照銘柄を共有する場合、個別日本株に対する EB ヘッジのディーラー集中は意味のあるフローになり得る。

## リテール投資家の損失事例

EB ノックイン販売は、複数の注目度の高いリテール損失局面と重なった。公開報道で確認できる一般的なパターン（個別取引の詳細ではない）は次のとおり。

| 局面 | パターン |
|---|---|
| 2018-2019 日本金融株 | 日本の銀行 / 保険株を参照する EB 債が、2018 の売り局面でノックインを発生させた |
| 2020 Q1 COVID ショック | グローバル原資産を持つワーストオブ・バスケット EB で、複数参照資産が同時に下落し、広範なノックインが発生した |
| 2020-2021 個別株の急落 | 固有要因で下落した銘柄（Vision Fund 評価損局面の SoftBank Group、個別テック銘柄など）を対象にした単一銘柄 EB でノックインが発生した |
| 2022 H1 グローバル株式売り | 米国テック株と日本テック株を参照する EB 債で広範なノックインが発生した |
| 2022 H2 SoftBank Group のボラティリティ | 継続的なドローダウンの中で、単一銘柄 SoftBank EB 債がノックインを発生させた |

これらの局面で繰り返されたパターンは、リテール購入者が二値的な損失プロファイルを想定しておらず、ノックイン発生時に驚いたことである。JSDA / FSA の集計苦情データでは、各局面後に顕著な急増が示された。

### 執行強化の背景

FSA は **「Customer First」（顧客本位の業務運営）** の枠組みの下、2021からリテール向け仕組債販売への監督上の注目を高めた。2022 年10月の FSA 公表文は、仕組債の適合性と開示を高優先度の課題として明示的に指摘した。

### 2023

の行政処分FSA、JSDA、SESC は 2023に連携した執行措置を取った。

| 販売会社 | 措置区分 |
|---|---|
| [[JapanFG/nomura-hd|Nomura Securities]] | 仕組債（EB を含む）の販売プロセスと適合性に関する業務改善命令 |
| [[JapanFG/smbc-nikko|SMBC Nikko Securities]] | 業務改善命令。適合性プロセスをめぐる経営陣の責任 |
| [[JapanFG/daiwa-sg|Daiwa Securities]] | 業務改善命令。適合性と開示の改善 |
| 複数の地域銀行系証券会社 | 個別の行政対応および自主停止 |

一連の執行措置は、EB ノックイン販売を最も深刻な適合性問題として具体的に指摘した。手数料負担の開示、シナリオ分析（損失例示）、顧客適合性評価が主要な不備点だった。

### 販売会社の対応

主要販売会社は、執行措置の前後に自主的な対応を取った。

| 対応 | 販売会社 |
|---|---|
| リテール向け EB ノックインの新規販売を停止 | 2023 中の複数の主要販売会社 |
| 適格顧客または特別承認顧客に販売を制限 | 複数の販売会社が年齢 / 知識 / 経験の閾値を採用 |
| 開示・録音要件を強化 | JSDA 自主規制アップデートを通じて業界全体で対応 |
| リテール向け仕組債販売から全面撤退 | 一部の地域銀行系証券会社 |
| リテール商品構成をより単純な代替商品へ移行 | 多くの販売会社がプレーンバニラ債、バランスファンド、NISA 対象ファンドを重視 |

### 販売額への影響

JSDA / FSA の業界集計統計は、リテール EB ノックイン販売が 2022 から 2023にかけて大幅に縮小したことを示した。多くの販売会社は EB 販売を 80 percent 以上削減した。

### 適合性評価

| 要件 | 詳細 |
|---|---|
| 知識 / 経験テスト | 顧客がバリア・オプションの仕組みを理解していることを示す必要がある |
| リスクプロファイルの一致 | 顧客のリスク許容度が 50+ percent の損失可能性を明示的にカバーしている必要がある |
| 年齢閾値 | 定められた年齢閾値（通常 75 または 80）を超える顧客への特別手続 |
| 集中上限 | 顧客金融資産に占める仕組商品の割合に上限を設定 |
| 反復購入審査 | 短期間に複数の仕組商品を購入する顧客への特別レビュー |

### 開示要件

| 項目 | 必要な開示 |
|---|---|
| 総手数料負担 | アレンジャー手数料と販売会社手数料の明示的開示 |
| 損失シナリオ | 複数の不利なシナリオにおける具体的な数値損失例示 |
| ノックインの仕組み | バリア、観測方式、交付メカニズムの平易な説明 |
| ワーストオブ機能の説明 | ワーストオブは分散ではないという明示的警告 |
| 代替商品比較 | 類似目的を達成するより単純な商品への言及 |
| クーリングオフ期間 | 指定期間内の取消権 |

### 販売プロセス

| 手順 | 要件 |
|---|---|
| 録音 | 高リスク商品の販売会話の録音義務 |
| 確認 | リスク理解に関する顧客の書面確認 |
| レビュー | 閾値を超える販売適切性の第二線レビュー |
| 文書化 | 規制当局レビューに備えた適合性評価の保存 |

## 転換社債市場との比較

機関投資家向けの [[finance/japan-convertible-bond-mechanics|convertible bond market]] は、類似した基礎数理（組込株式デリバティブ + 債券フロア）を使うが、販売形態と保護プロファイルは異なる。

| 観点 | リテール EB ノックイン | 機関投資家向け転換社債 |
|---|---|---|
| 買い手 | 証券会社経由の個人投資家 | 機関投資家（転換社債アービトラージファンド、資産運用会社、保険会社） |
| 適合性枠組み | リテール適合性 + 高齢顧客保護 | 適格機関投資家の枠組み |
| 組込デリバティブ | ダウン・アンド・イン・プットのショート | コール・オプションのロング |
| 株式エクスポージャーの方向 | 株式下方リスクのショート（二値的） | 株式上方リスクのロング（非対称） |
| テールリスク | 50+ percent の損失があり得る | 債券価値がフロア（信用リスクを含む） |
| 開示基準 | 2023 後に強化されたリテール保護ルール | 機関投資家向け EDINET / TDnet 基準 |
| 規制当局の焦点 | Customer-First 執行 | 開示およびインサイダー取引規制の枠組み |

対比は明確である。機関投資家向け転換社債は買い手に非対称な上方余地を与える一方、リテール EB ノックインはクーポンと引き換えに非対称な下方リスクを買い手に負わせる。
