# 日本企業 CDS スプレッドの仕組み

## 要約

日本企業 CDS スプレッドは、プロテクションの買い手が、参照対象である日本の上場企業に対するデフォルト連動型支払いと引き換えに、プロテクションの売り手に支払うランニングベースの保証料(ベーシスポイント)である。スプレッドは、信用リスクプレミアム(予想デフォルト確率と想定回収率の関数)に、構造的プレミア(原債券に対するベーシス、ディーラーのキャリー、規制資本コスト)を加えたものに分解される。日本企業 CDS は **ソブリン対企業のスプレッド** に基づいて値付けされ、日本のソブリン CDS が暗黙の信用フリーアンカーとなり、個別企業のスプレッドがその上にレイヤー化される。2014  ISDA クレジットデリバティブ定義の下でのクレジットイベントトリガーは、**破産、支払不履行、リストラクチャリング**(Mod-R / Mod-Mod-R のバリアントが引渡可能債務を決定する)をカバーする。回収率は、クレジットイベント後の **ISDA が運営するオークション** によって決定される。生命保険会社と年金基金は ESR / 会計上の制約を踏まえ CDS を控えめに使うが、インデックス CDS を通じた信用スプレッドヘッジは [[insurance/japan-life-insurance-alm-overview|life ALM]] の議論で繰り返し登場するテーマである。

### 構成要素

期間 T にわたる日本のシングルネーム企業 CDS スプレッド *S(corp)* は、概念的に次のように分解できる:

```
S(corp, T) ≈ S(sovereign, T) + S(sector, T) + S(idiosyncratic, T) + basis_adjustment
```

| 構成要素 | ドライバー |
|---|---|
| ソブリンベースライン *S(sovereign, T)* | 期間 T における日本のソブリン CDS スプレッド;短いテナーではほぼゼロ、5Y / 10Y では緩やかにプラス |
| セクタープレミアム *S(sector, T)* | セクターレベルの信用ファクター(金融 vs 産業 vs 通信など) |
| 固有要素 *S(idiosyncratic, T)* | 企業固有のレバレッジ、キャッシュフローの安定性、ガバナンス、M&A プロファイル |
| ベーシス調整 | ディーラーのバランスシート、レポのスペシャル、規制コストによって駆動される CDS-債券ベーシス — [[derivatives/basis-trade-bond-cds-japan|bond-CDS basis trade]] 参照 |

この分解は概念的であって代数的ではない。市場の実務では、構成要素から積み上げるのではなく、市場クオートを通じて企業 CDS カーブを直接フィッティングする。

### リスク中立デフォルト確率

CDS スプレッドはリスク中立ハザードレート(またはハザードレートの期間構造)を含意する:

```
リスク中立デフォルト確率(期間あたり) ≈ S / (1 - R)
```

ここで S はランニングスプレッド、R は想定回収率である。想定 R = 40 パーセントの投資適格日本企業について:

| 5Y CDS スプレッド(bps) | 含意される 5Y リスク中立デフォルト確率 |
|---|---|
| 25 bps | ~2.0 パーセント |
| 50 bps | ~4.1 パーセント |
| 100 bps | ~8.0 パーセント |
| 200 bps | ~15.5 パーセント |
| 500 bps | ~34 パーセント |
| 1000 bps | ~57 パーセント |

リスク中立デフォルト確率は実際のデフォルト確率ではない;リスクプレミアムと流動性プレミアムを内包している。IG 日本企業の過去のデフォルト率は、通常の状況下ではこれらのリスク中立の含意レベルをはるかに下回る。

### 回収率の想定

| 参照債務のタイプ | 標準的な想定回収率 |
|---|---|
| シニア無担保企業債(日本 IG の大半) | 40 パーセント |
| 劣後企業債 | 25 パーセント |
| 銀行シニアプリファード | 40 パーセント |
| 銀行シニアノンプリファード / Tier 2 | 法域により 25-40 パーセント |
| ソブリン(先進国市場) | 40 パーセント(日本) |

40 パーセントの想定は慣行であって市場含意ではない。クレジットイベント後のオークションが実際の回収率を決定する(下記で説明)。

## 2014  ISDA 定義の下でのクレジットイベント

2014  ISDA クレジットデリバティブ定義は、CDS の決済をトリガーするクレジットイベントを規定する。日本企業 CDS については、三つのクレジットイベントが標準である:

### 破産

| トリガー | 説明 |
|---|---|
| 民事再生の申立て | 企業向けの日本の裁判所内再生手続 |
| 会社更生の申立て | より大規模な企業向けの日本の裁判所内更生手続 |
| 特別清算 | 裁判所監督下の清算 |
| 外国倒産手続 | 参照エンティティが外国法域の倒産イベントを有する場合 |
| 一般的な解散 / 清算 | 任意または強制の解散 |

破産は最もクリーンなトリガーである。ISDA 決定委員会(DC)が、申し立てられたイベントが該当するかを判断し、クレジットイベントの判定を公に公表する。

### 支払不履行

| トリガー | 説明 |
|---|---|
| 支払いデフォルト | 猶予期間後、しきい値(通常 USD 1 百万相当)を超える任意の債務における予定支払いの不履行 |
| 猶予期間 | 契約上の猶予期間が適用されない限り、標準的な猶予期間は通常 3 営業日 |
| キュア期間 | トリガー前の任意のキュア期間 |

日本企業については、猶予期間後に欠落した予定の債券またはローンの支払いが、DC の判定を条件として該当する。

### リストラクチャリング(およびバリアント)

| バリアント | カバレッジ |
|---|---|
| フルリストラクチャリング(R) | オリジナルの 1999 定義;債務リストラクチャリングイベントの広範なカバレッジ |
| モディファイドリストラクチャリング(Mod-R) | 米国慣行;引渡可能債務の満期がクレジットイベントから 30 か月に上限設定 |
| モディファイド・モディファイドリストラクチャリング(Mod-Mod-R) | 欧州 / 日本慣行;引渡可能債務の満期が 60 か月に上限設定 |
| ノーリストラクチャリング(No-R) | 米国 2009以降;リストラクチャリングをクレジットイベントから除外 |

| リストラクチャリングトリガー | 詳細 |
|---|---|
| クーポン減額 | 合意済みクーポンの減額 |
| 元本減額 | 元本額の減額 |
| 満期延長 | 支払日の延長 |
| 劣後化 | 支払優先順位の変更 |
| 通貨変更 | 許容されない通貨への変更 |

日本企業 CDS は慣例的に、欧州市場慣行に合わせて **Mod-Mod-R**(モディファイド・モディファイドリストラクチャリング)で取引される。引渡可能債務の満期に対する Mod-Mod-R の上限は、広範なリストラクチャリングが生み出す「最安受渡し(cheapest-to-deliver)」のオプショナリティを減らす。

### 政府介入(銀行)

日本の金融機関 CDS については、2014 定義がベイルインおよび破綻処理アクションをカバーする **政府介入** クレジットイベントを追加した:

| イベント | トリガー |
|---|---|
| ベイルイン | 債務商品の法定の元本減額または転換 |
| 破綻処理当局のアクション | 所管当局による債務の移転またはヘアカットの賦課 |
| 強制的な資産売却 | クレジットイベント相当を構成する強制的な処分 |

これは [[JapanFG/mufg|MUFG]]、[[JapanFG/sumitomo-mitsui-banking-corp|SMBC]](SMFG 経由)、[[JapanFG/mizuho-bank|Mizuho]](みずほ FG 経由)、その他の日本の金融機関 CDS に関連する。これらにおいては、預金保険法とより広範な破綻処理フレームワークの下での銀行破綻処理が、プロテクションのカバレッジにとって重要である。

## ISDA 決定委員会(DC)プロセス

| ステージ | 活動 |
|---|---|
| イベント報告 | 市場参加者が ISDA DC に質問を提出する |
| DC レビュー | DC が公表されたタイムライン内で、公表された基準に照らしてイベントをレビューする |
| 公的公表 | DC がクレジットイベントの有無について拘束力のある判定を発表する |
| オークションまたは決済の決定 | DC がオークションを招集するかを決定する |
| オークション運営 | オークションが招集された場合、Creditex / Markit が運営するプロセスで実施される |
| 最終価格 | オークションで決定された最終価格が公表される |

DC は、地域別(南北アメリカ、EMEA、アジア除く日本、日本、オーストラリア / ニュージーランド)に主要なディーラーおよびバイサイド企業で構成される常設委員会である。日本の参照エンティティについては、関連する DC に日本でアクティブな主要ディーラーが含まれる。

DC の判定は ISDA クレジットデリバティブ決定委員会のウェブサイトで公に公表され、標準的な ISDA 文書化された CDS 契約に対して拘束力を持つ。

## 回収率オークションの仕組み

クレジットイベント後、ISDA が運営するオークションが、現金決済 CDS に使われる回収率を設定する:

| ステージ | 活動 |
|---|---|
| オークション告知 | ISDA / Creditex がオークション日と引渡可能債務のリストを告知する |
| 初回入札(ステージ 1) | 参加入札者が両建てのマーケットを提出する;初回マーケットの中値が算定される |
| オープンインタレストの公表 | ネットオープンインタレストの方向(買いまたは売り)が公表される |
| 指値注文板(ステージ 2) | 入札者がオープンインタレストを埋めるための指値注文を提出する |
| 最終価格 | オークションのクリアリング価格が設定される;これが現金決済に使われる「回収率」である |

現金決済:プロテクションの売り手はプロテクションの買い手に(想定元本)×(1 - 最終価格)を支払う。例えば最終オークション価格が 35, なら回収率は 35 パーセントであり、プロテクションの売り手は想定元本の 65 パーセントを支払う。

現物決済(レガシーであり、現行のほとんどの契約では任意)は、額面の支払いと引き換えに適格債務を引き渡すことを伴う。最近のほとんどの CDS 契約はオークションを通じた現金決済をデフォルトとする。

## 日本固有のオークション史

ISDA オークションをトリガーする日本企業のクレジットイベントはまれである。過去のエピソードには次が含まれる:

| 期間 | 参照エンティティのタイプ | オークション結果(例示的パターン) |
|---|---|---|
| 2010 | 日本航空 | 破産申立てがクレジットイベントをトリガー;オークションが回収率を設定 |
| 2010 | 武富士 | 消費者金融の破産;オークション回収率は無担保債権者の見通しを反映 |
| 2012-2015 | 各種の小型株リストラクチャリング | 限定的な CDS カバレッジ;小規模な DC 活動 |
| 2017 | 東芝(正式なクレジットイベントなし) | 市場は CDS スプレッドを通じてクレジットイベントに近いリスクを織り込んだ;オークションはトリガーされず |
| 2020-2024 | COVID およびポスト COVID 期 | 主要な日本 IG のクレジットイベントなし;有意な CDS エクスポージャーを伴わない少数の小型株の困窮事例 |

日本 IG のクレジットイベントがまれであることは、低い原デフォルト率、強力な銀行債権者の支援、裁判外ワークアウトの規範の組み合わせを反映している。

## JGB ベンチマークに対するベーシス

CDS スプレッドは通常、一致するテナーでの **JGB ベンチマークカーブ** に対する企業の債券スプレッドと比較される:

| 指標 | 算定 |
|---|---|
| JGB に対する Z スプレッド | 割引キャッシュフローが債券価格と一致するように JGB カーブに加える一定スプレッド |
| アセットスワップスプレッド | スワップカーブに対するスプレッド;日本では通常 TONA スワップカーブに対して |
| CDS-債券ベーシス | CDS スプレッド - 対応する債券スプレッド |
| ネガティブベーシス | CDS スプレッド < 債券スプレッド;キャッシュロング・CDS ショートのトレードが魅力的 |
| ポジティブベーシス | CDS スプレッド > 債券スプレッド;キャッシュショート・CDS ロングのトレードが魅力的 |

[[derivatives/basis-trade-bond-cds-japan|Bond-CDS basis trade]] は、ファンディングコスト、レポの利用可能性、バランスシート制約を含め、ベーシストレードの経済性を詳細に展開する。

日本 IG 企業については、ベーシスは通常狭いが、ディーラーのバランスシート能力が縮小したり特定の JGB レポのダイナミクスが相対価値の値付けをシフトさせたりするストレスエピソード(COVID Q1 、2022-2023の BOJ の YCC 調整 2020, )の際に拡大しうる。

## セクタースプレッドのパターン

| セクター | スプレッドの性格 |
|---|---|
| メガバンク金融(MUFG、SMBC、みずほ) | 通常の状況下では緩やかな 5Y スプレッド;グローバルな銀行ストレス(2023 の SVB / Credit Suisse、2024 の地方銀行ストレス)の際に断続的に拡大 |
| 大手生保 / 損保 | 緩やかな 5Y スプレッド;BOJ の金利経路と ESR プロファイルへの感応度 |
| 商社(総合商社) | 緩やかな 5Y スプレッド;循環的な局面でのコモディティサイクルへの感応度 |
| 自動車 / 産業ブルーチップ | 安定した状況では低い 5Y スプレッド;EV 移行ストレスの際に断続的に拡大 |
| 通信 / 公益 | 広いレンジ;TEPCO は 2011 以降、長年にわたり高水準を維持 |
| 不動産 | 金利サイクルと J-REIT のファンディング状況に感応 |
| ソフトバンクグループ | 一貫して最も広い IG ないしクロスオーバーの日本のネーム;Vision Fund の時価評価が断続的なスプレッドのスパイクを駆動 |

ソフトバンクグループ CDS は、そのレバレッジプロファイル、M&A 史、株式ボラティリティとのリンクを反映し、取引高で大差をつけて日本の非金融シングルネーム CDS で最も取引されている。

### 生命保険会社と年金基金

| ユースケース | 詳細 |
|---|---|
| 信用スプレッドヘッジ | 信用スプレッドショックに対するポートフォリオレベルの CDS インデックスヘッジ |
| セクターヘッジ | セクタートランシェ CDS オーバーレイ(日本では流動性が限定的) |
| シングルネームのネガティブビューヘッジ | 懸念される保有に対する選択的なシングルネーム CDS プロテクション |
| シンセティック信用エクスポージャー | 債券購入なしで信用エクスポージャーを得るための CDS プロテクションの売却 |

実務上、日本の生命保険会社と年金基金は次の理由で CDS を控えめに使う:

- ヘッジ有効性の指定を複雑にする ESR 会計上の取扱い;
- JGAAP / IFRS 9 のヘッジ会計要件;
- 内部リスク管理ガバナンスの制約;
- Tier 1 のネームを下回るシングルネーム日本 CDS における流動性制約。

より広範なヘッジツールキットの文脈については [[insurance/japan-life-insurance-alm-overview|Japan life insurance ALM]] を参照。

### 銀行ディーラーとトレジャリー

| ユースケース | 詳細 |
|---|---|
| マーケットメイク | ディーラーブックのインベントリヘッジ |
| カウンターパーティ信用リスクヘッジ | ディーラーのカウンターパーティに対する CDS(CVA ヘッジ) |
| 自社ネームヘッジ | 限定的;自社信用ヘッジの会計上の問題 |
| ソブリンベーシストレード | JGB のクロスカレンシーベーシスエクスポージャーのヘッジ |

CVA(信用評価調整)ヘッジは、特に金融カウンターパーティのネームについて、ディーラーの CDS 需要を駆動する規制上・会計上のドライバーである。

### ヘッジファンド

| ユースケース | 詳細 |
|---|---|
| アウトライトの信用ビュー | 方向性のビューに基づくシングルネーム CDS の売買 |
| 相対価値 | CDS カーブ横断またはネーム横断のロング / ショート |
| 資本構成アービトラージ | CDS vs 株式、CDS vs CB、CDS vs プリファード |
| ベーシストレード | ネガティブベーシス向けのキャッシュロング・CDS ショート、またはその逆 — [[derivatives/basis-trade-bond-cds-japan|bond-CDS basis trade]] 参照 |
| インデックスアービトラージ | iTraxx Japan vs 構成シングルネーム CDS |

資本構成アービトラージは、アクティブな転換社債市場を踏まえ、日本で繰り返し用いられる戦略である — [[finance/japan-convertible-bond-mechanics|Japan convertible bond mechanics]] 参照。

### ストラクチャード商品の発行体

| ユースケース | 詳細 |
|---|---|
| クレジットリンク債(CLN)ヘッジ | 発行体は参照ネームに対する CDS プロテクションの購入でヘッジする |
