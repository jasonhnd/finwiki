# 仕組債の日本リテール発行

## 要約

「仕組債」(*shikumisai*) は、デリバティブのペイオフを内包するリテール向け債券商品の総称となる日本語であり — 典型的にはノックインバリアを持つ株価連動ノート (EB、*Equity Bond*)、通貨連動預金、ディスカウントでのプットストライクを伴うリバースコンバーチブル債、デジタル / レンジアクルアルノートである。これらの商品は、日本の長引く低利回り環境において、メガバンク系証券会社 ([[JapanFG/smbc-nikko|SMBC Nikko]]、[[JapanFG/mizuho-bank|Mizuho]] 証券、MUFG / MUMSS)、独立系リテール証券 ([[JapanFG/daiwa-sg|Daiwa]]、[[JapanFG/nomura-hd|Nomura]])、地方銀行により、**利回りを追うリテール投資家**へ大規模に配信された。適合性と開示に対する **2022-2023 の FSA 取締り**は、[[JapanFG/nomura-hd|Nomura]]、[[JapanFG/smbc-nikko|SMBC Nikko]]、[[JapanFG/daiwa-sg|Daiwa]] の子会社に対する行政命令につながり、主要な販売会社に対し高齢者 / 知識の乏しいリテールへの仕組債販売の停止または制限を強い、**適合性と開示**を構造的な規制上の優先事項へと引き上げた。新規のリテール発行は 2023 以降、急激に縮小した。

## ウィキ上の位置づけ

このエントリは、リテール仕組商品の配信ページとして [[derivatives/INDEX|derivatives index]] の配下にあり、最も深い個別銘柄 EB 分析の [[derivatives/structured-product-eb-knockin-japan-retail|EB knock-in structured product mechanics]] と対になっている。クレジットデリバティブの構成要素 (一部の仕組債はクレジットリンクトランシェを内包する) については [[derivatives/japan-cds-market-overview|Japan CDS market overview]]、クレジットスプレッドの構成ブロックについては [[derivatives/cds-japan-corporate-spread-mechanics|Japan corporate CDS spread mechanics]]、レートの下地については [[derivatives/japan-irs-market|Japan IRS market]]、FX 連動商品とのファンディングカーブの相互作用については [[derivatives/yen-basis-swap-market|yen basis swap market]] と併せて読むこと。

より広い資本市場の文脈については [[finance/INDEX|finance index]]、機関の株価連動カウンターパートについては [[finance/japan-convertible-bond-mechanics|Japan convertible bond mechanics]]、メガバンクの配信文脈については [[banking/INDEX|banking index]]、機関の外貨 / 仕組資産の利用については [[insurance/japan-life-insurance-alm-overview|Japan life insurance ALM]]、ディーラー側のヘッジ機構については [[securities/japan-prime-brokerage-and-institutional-financing|prime brokerage and institutional financing]] をクロスリファレンスすること。

### 仕組債のコアカテゴリ

| 商品 | 日本語の用語 | ペイオフの核 |
|---|---|---|
| ノックインバリア付き株価連動債 (EB) | EB債 / 株価連動債 | 原資産株がノックインバリアを突破しない限り額面 + クーポンを支払い、突破した場合は株式またはバリア後の金額で支払う |
| オートコーラブル EB / エクスプレス | オートコーラブル | 観察日に原資産がオートコール閾値を突破すれば加速クーポンと全額償還を支払い; そうでなければ継続 |
| リバースコンバーチブル | リバース・コンバーチブル | プットオプションのショートを内包した高クーポン債; 株価がストライク以下なら株式で償還 |
| 通貨連動預金 (二重通貨預金) | 二重通貨預金 / デュアルカレンシー | 預金は高い円利回りを支払う; 元本は所定のレートで円以外の通貨で償還される (通貨ショートを内包) |
| パワーリバースデュアルカレンシーノート (PRDC) | パワーリバース | 長期の FX 連動の円払い / ドル払いノート; 複雑なバミューダンコーラブル構造 |
| クレジットリンクノート (CLN) | クレジット・リンク債 | 参照クレジットがデフォルトしないことを条件に増額クーポンを支払う債券 |
| レンジアクルアルノート | レンジアクルアル | 参照レート / FX / 指数がレンジ内に留まる日にのみクーポンが累積する |
| デジタル / ワンタッチノート | デジタル | 参照が閾値にタッチ / 超過すれば大きなクーポンを支払い、そうでなければ無し |
| 株価指数連動ノート (バスケット) | バスケット型 | ペイオフはワースト・オブ・バスケットに連動 (典型的には Nikkei 225, S&P 500, EuroStoxx 50) |

EB ノックイン (特に単一の日本株またはワースト・オブ・バスケットのオートコーラブル) は、ピークの 2018-2022 期間において出来高ベースで支配的なリテール配信商品であった。

### 構造のバリエーション

| 特徴 | 説明 |
|---|---|
| 単一銘柄の原資産 | 配信されたリテール EB の大半は単一の日本の上場株を参照 (例: トヨタ、ソニー、ソフトバンク、NTT) |
| ワースト・オブ・バスケット | 3 から 5 の参照資産; ペイオフはワースト・オブで、リスクが増大 |
| オートコール閾値 | 典型的には初期参照の 100 パーセント (観察日に上昇していれば額面 + クーポンで償還) |
| ノックインバリア | 典型的には初期参照の 50-70 パーセント |
| クーポン | 条件付クーポン (ノックイントリガーがない場合のみ支払い) または無条件の固定クーポン |
| 満期 | 典型的には 3-5 年; オートコール機能が期待存続期間を短くする |
| 通貨 | 日本のリテール向けには JPY で発行; 通貨オーバーレイのバリアントには一部 USD または EUR で発行 |
| コーラブル機能 | 一部の構造ではバミューダンコーラブル |

### 発行体セット

| 発行体タイプ | 役割 |
|---|---|
| グローバル銀行 (特別目的事業体) | Goldman、JPMorgan、Morgan Stanley、BNP Paribas、Credit Suisse (歴史的に)、HSBC、Citi、Barclays が SPV プログラム経由で発行 |
| 日本拠点のメガバンクプログラム | [[JapanFG/mufg|MUFG]]、[[JapanFG/sumitomo-mitsui-banking-corp|SMBC]] (SMFG 経由)、[[JapanFG/mizuho-bank|Mizuho]] (Mizuho FG 経由) が専用プログラムを通じて発行 |
| 欧州銀行の発行体 | Société Générale、BNP、Credit Suisse、UBS が歴史的に仕組商品の組成で支配的 |
| 米国銀行の発行体 | Goldman、JPM、MS、Citi |
| 東京上場の特別目的発行体 | あまり一般的でない; 大半の仕組債は税務 / 規制効率のためオフショアで発行 |

### 配信チャネル

| 販売会社 | 配信モデル |
|---|---|
| [[JapanFG/smbc-nikko|SMBC Nikko]] | メガバンク系のフルサービス証券; 2022 を通じた主要な販売会社 |
| みずほ証券 ([[JapanFG/mizuho-bank|Mizuho]] FG 経由) | メガバンク系のフルサービス証券; 2022 を通じた主要な販売会社 |
| MUFG / MUMSS ([[JapanFG/mufg|MUFG]] 経由) | メガバンク / グローバル JV 証券; 主要な販売会社 |
| [[JapanFG/daiwa-sg|Daiwa Securities]] | 独立系リテール証券; 歴史的に仕組債の配信で非常に活発 |
| [[JapanFG/nomura-hd|Nomura Securities]] | 最大手のリテール証券; 相当な販売会社 |
| 地方銀行 (サブ配信) | メガバンクの証券パートナーからの仕組商品をサブ配信 |
| オンライン証券 | 複雑な仕組債にはあまり活発でない; 一部のプレーンバニラ仕組預金 |

### アレンジャー手数料

仕組債におけるアレンジャーおよび販売会社の手数料は、歴史的に商品経済性の相当な部分を占め、発行価格対公正価値のギャップに埋め込まれていた。**手数料負担** (典型的には想定元本の 3-10 パーセント、複雑な構造ではときにより高い) は FSA 取締りの重要な要素となった。

## ヘッジのメカニクス

発行体は通常、内包するデリバティブをディーラーとバックトゥバックでヘッジする:

| ポジション | ヘッジ |
|---|---|
| 発行体は債券をロング、内包デリバティブ (ノックインプット、オートコールコール、FX オプション等) をショート | 発行体は反対のポジションを取るディーラーとバックトゥバックのデリバティブ取引を結ぶ |
| ディーラーのウェアハウス | ディーラーはエクスポージャーを集約し、株式 / FX / クレジット市場でデルタ / ベガ / バリアガンマのヘッジを行う |
| リスクのリサイクル | ディーラーは指数取引、上場オプション、OTC ディーラー間取引を介してリスクを再配分しうる |

日本の単一銘柄 EB について、多くのノートが同じ参照銘柄を共有する場合 (例: ソフトバンク EB の集中的な発行が相当なソフトバンク株のヘッジフローを生む)、原資産株におけるディーラーのヘッジは有意義なフローとなりうる。

### メカニクス

| 特徴 | 説明 |
|---|---|
| 元本通貨 | JPY (預金) |
| 利回り | 既存の JPY 預金金利に対して増額された JPY クーポン (例: 年率 3-8 パーセント) |
| 償還オプション | 満期時、発行体は額面で JPY で償還、または所定の為替レートで円以外の通貨で償還 |
| 発行体の選択 | 発行体は発行体にとってより安い方を償還する (すなわち投資家はより価値の低い方を受け取る) |
| 内包デリバティブ | 投資家は暗黙裡に JPY プット / 円以外コールのオプションをショートしている |

一般的な参照通貨: USD、AUD、NZD、EUR、GBP、ZAR、TRY (歴史的により高利回りのテール)。TRY 連動のバリアントは、トルコリラのボラティリティを踏まえてエンフォースメントと消費者保護の注目を集めた。

### リスク

円以外の通貨がストライクに対して JPY に対し著しく減価すると、投資家は元の JPY 元本より価値の低い円以外の通貨を受け取る。損失は相当なものとなりうる。

### 配信

通貨連動預金は歴史的に、メガバンクのリテール窓口、地方銀行、郵便局チャネルを通じて配信された。ラッパーの単純さ (「預金」という枠組み) は、しばしばリテール投資家にとって内包される短いオプションのリスクを覆い隠した。

### メカニクス

| 特徴 | 説明 |
|---|---|
| クーポン | 存続期間を通じて支払われる固定の高クーポン |
| 満期ペイオフ | 参照株がストライク以上なら → 額面償還; 以下なら → ストライクでの株式の引渡し (すなわち投資家は額面より価値の低い株式を受け取る) |
| 内包デリバティブ | 投資家はストライクで参照株のプットオプションをショート |
| 参照 | 単一株またはワースト・オブ・バスケット |

メカニズム的には EB に類似するが、プット行使が通常、期中のノックインバリア経由ではなく満期時に行われる点で、より債券的な枠組みである。

### 背景

FSA は、2017, 以降の **「顧客本位」** (顧客本位の業務運営、*kokyaku honi no gyōmu un'ei*) のイニシアチブの下、複雑な商品のリテール配信をますます精査した。調査は、仕組債の複雑性とリテール投資家の知識 / 経験との間の広範なミスマッチを明らかにした。

### 2022 の FSA 所見

公的な FSA 資料 (2022年10月) と JSDA の自主規制アップデートは以下を記録した:

| 所見 | 懸念 |
|---|---|
| 知識のミスマッチ | リテールの買い手はバリアオプションのメカニクスとテールリスクの理解を欠くことが多かった |
| 手数料開示 | アレンジャー / 販売会社の手数料負担が十分に開示されていなかった |
| 適合性の失敗 | 高齢の退職者、知識の乏しい顧客、商品リスクにミスマッチな保守的投資家へ販売された |
| 集中 | 同一顧客への類似商品の反復販売が集中したテールエクスポージャーを生んだ |
| 代替案との比較 | 同様の利回り増強を達成するより安価 / 単純な代替案を提示しなかった |

### 行政命令

2022 年後半から 2023, 年にかけて、FSA は複数の主要な販売会社に対し業務改善命令と行政措置を発出した:

| 販売会社 | 措置のカテゴリ |
|---|---|
| [[JapanFG/nomura-hd|Nomura Securities]] | 仕組債の販売プロセスに関する業務改善命令 |
| [[JapanFG/smbc-nikko|SMBC Nikko]] | 業務改善命令; 上級経営陣の責任 |
| [[JapanFG/daiwa-sg|Daiwa Securities]] | 業務改善命令; 適合性プロセスのレビュー |
| 複数の地方銀行の証券部門 | 局所的な行政措置 |

販売会社は自主的に仕組債のリテール販売を停止または大幅に制限した。複数の会社がリテール仕組債事業から完全に撤退するか、適格投資家チャネルに制限した。

### JSDA の自主規制

[[securities/japan-prime-brokerage-and-institutional-financing|JSDA]] は仕組債の適合性に関する自主規制ガイダンスを厳格化した:

| 領域 | ガイダンスの変更 |
|---|---|
| 適合性評価 | より厳格な知識 / 経験テスト; 書面確認の要件 |
| 開示 | 標準化されたリスク開示テンプレート; 明示的な「損失シナリオ」の図示 |
| クーリングオフ | 強化されたクーリングオフと救済チャネル |
| 記録 | 販売会話の記録 / 文書化の義務付け |
| 高齢顧客保護 | 定められた年齢閾値を超える顧客のための特別手続 |

### 出来高への影響

公的な JSDA / FSA の集計データは、リテール仕組債の販売量が 2022 から 2023にかけて非常に大幅に縮小したことを示した。配信は「ノックイン EB」商品から、より単純な仕組預金および / またはより高格付けのプレーンバニラ債へとシフトした。

## グローバルの仕組商品リテール市場との比較

| 管轄 | リテール仕組商品市場の性格 |
|---|---|
| 日本 (2023前) | 非常に大きなリテール出来高; メガバンク系証券が支配的な販売会社 |
| 日本 (2023後) | 著しく縮小; 高齢者 / 知識の乏しいリテールへの制限 |
| EU (PRIIPs 2018後) | KID 開示が義務化; 出来高は回復したが開示負担が上昇 |
| 英国 (FCA 2014後) | 適合性規則が厳格化; 出来高が大幅に縮小 |
| 香港 (Lehman ミニボンド 2008後) | 厳格な適合性および集中規則が導入 |
| 米国 | 主に機関 / 適格投資家; 特定の配信チャネル経由のリテール仕組商品 |

日本の 2023 の取締りは、独自の規制アプローチというより、グローバルなリテール保護基準への著しいキャッチアップを表す。

## 機関の仕組債市場

リテールを超えて、機関の仕組債市場は以下のために継続している:

- 利回り増強型のクレジット商品を求める年金基金と生命保険会社 (ESR / 会計上の制約の対象、[[insurance/japan-life-insurance-alm-overview|life ALM]] を参照);
- 仕組負債または仕組資産のラッパーを用いる法人財務オペレーション;
- 特定のオーバーレイまたは利回り増強の目的のための資産運用会社のポートフォリオ。

機関の仕組債は異なる適合性基準 (FIEA の下の適格機関投資家の枠組み) の対象であり、限定的な規制上の摩擦で継続している。
