# 債券・CDS ベーシス取引 日本

## 要約

日本の債券・CDS ベーシスは、企業の CDS プロテクションコストと、同一のテナーにおけるスワップまたは JGB ベンチマークカーブに対するその現物債の信用スプレッドとの間のスプレッドである。**ネガティブベーシス**（CDS < 債券スプレッド）は、歴史的に、資金調達コストとバランスシートチャージを差し引いて「無料の」キャリーをロックインするロング現物・ショート CDS 取引を引き付けてきた。**ポジティブベーシス**（CDS > 債券スプレッド）は日本ではあまり一般的でないが、プロテクション買い需要がディーラーの能力を超えるストレス時に現れる。2010, 以降のバーゼル III レバレッジ比率、単一カウンターパーティ信用限度、JGB のレポスペシャル、そして BOJ の非伝統的政策（NIRP、QQE、YCC、YCC 調整）は、ベーシス取引をスケールしにくく、サイクルを通じてより変動的にしてきた。日本 IG 社債信用における主要なベーシスの乖離は、**COVID 第1 四半期2020**（現物債が CDS より速くディストレスト水準で取引され、ネガティブベーシスが大きく拡大した）、**2022-2023 の BOJ YCC 調整**（金利とスプレッドの再価格付けがベーシスを拡大した）、そして **2024 の散発的な日本金融信用ストレス**（選別された銘柄での小規模な乖離）に発生した。

### 定義

```
Bond-CDS basis = CDS spread - Bond Z-spread (or asset swap spread) at matched tenor
```

| 符号 | 解釈 |
|---|---|
| ネガティブベーシス | CDS スプレッド < 債券スプレッド → プロテクション買いが債券の信用スプレッドが示唆するより安い → ロング現物・ショート CDS が魅力的 |
| ゼロベーシス | 純粋な裁定均衡（理論的） |
| ポジティブベーシス | CDS スプレッド > 債券スプレッド → プロテクション売りが債券の信用スプレッドより多くを捕捉する → ショート現物・ロング CDS が魅力的 |

### なぜベーシスはゼロでないのか

摩擦のない世界ではベーシスはゼロになる。実世界の摩擦は、持続的かつ時間変動するベーシスを生む：

| 摩擦 | ベーシスへの影響 |
|---|---|
| 債券の資金調達コスト（レポレート、バランスシートチャージ） | ベーシスをネガティブに押す傾向（債券は合成より保有コストが高い） |
| クレジットイベントのカバレッジの差（引渡可能債務のユニバース、リストラクチャリングのスコープ） | CDS のカバレッジが債券のエクスポージャーより広くも狭くもなりうる |
| CDS の最割安引渡（CTD）オプション | 現物債に対する CDS プロテクションに価値を加え、ベーシスをポジティブに押す |
| CDS のカウンターパーティリスク | CDS プロテクションの価値を下げ、ベーシスをネガティブに押す |
| 流動性の非対称性 | 流動性の低いレッグがプレミアム / ディスカウントで取引される |
| 規制資本コスト | 銀行所有のロング現物・ショート CDS ポジションはリスクウェイト資産とレバレッジ能力を消費する |
| 税・会計 | 現物債収入と CDS プレミアムの認識タイミングの差 |

### 構造

| レッグ | アクション |
|---|---|
| ロング現物債 | 参照発行体の債券をパーまたはパー近辺で購入し、レポで資金調達する |
| ショート CDS プロテクション（すなわちプロテクション買い） | クレジットイベントに対するプロテクションのためにランニングスプレッドを支払う |
| ネットキャリー | 債券利回り - 資金調達レート - CDS プレミアム |

ネガティブベーシスが資金調達コストとバランスシートチャージを上回る限り、この取引は CDS レッグからのクレジットイベントプロテクションとともにポジティブなキャリーをロックインする。

### 損益分解

```
Daily P&L ≈ (Bond accrual - Repo funding cost) - (CDS premium accrual)
           + Mark-to-market change in (Bond price - CDS protection value)
```

ベーシスがゼロに向かって縮小すれば、ポジションは値上がりする（債券が CDS より速く上昇するか、CDS が債券より速く拡大する）。ベーシスがさらにネガティブに拡大すれば、ポジションは MTM を失う。

### 必要な保有期間

ロックインされたキャリーを実現するには、ポジションは通常、債券満期または既知のホライズンまで保有される必要がある。短期ホライズンの取引は、予測不能なベーシス収束に依存する。多くのファンドは歴史的にポジションをロールし、MTM 変動を受け入れてきた。

## 資金調達コストの考慮事項

債券レッグは資金調達を必要とする。資金調達コストは取引経済性の支配的な決定要因である：

| 資金調達源 | コストドライバー |
|---|---|
| レポ（バイラテラルまたはトライパーティ） | レポレート。JPY レポは BOJ 政策金利にヘアカットを加えた近辺 |
| JGB 担保の GC レポ | 一般担保 JGB レポレート。BOJ の JGB 供給オペに敏感 |
| 社債の特定担保レポ | しばしばより高価。債券の借入可能性と空売り者からの需要を反映 |
| 銀行のバランスシート資金調達 | 内部の資金原価（FTP）。ほとんどの銀行でレポより高い |
| プライムブローカーファイナンス | [[securities/japan-prime-brokerage-and-institutional-financing|prime brokerage]] 経由のヘッジファンドの標準ルート。GC レポにヘアカットを加えたスプレッド |

債券のレポが困難な場合、ヘアカットが急峻な場合、またはバランスシートチャージが高い場合、50 bps のネガティブベーシスは資金調達コストで完全に食い尽くされうる。ベーシス取引の取引経済性のしきい値は、資金調達スタックに依存する。

## レポ可用性の制約

レポの可用性は実務上のボトルネックである。日本の社債レポは JGB レポよりはるかに流動性が低い：

| 債券タイプ | レポ流動性 |
|---|---|
| JGB（オンザランおよびベンチマーク） | 深い GC および特定レポ。BOJ のオペに敏感 |
| JGB（オフザラン） | あまり深くない。散発的なスペシャル |
| 日本 IG 社債（大手発行体） | 限定的なバイラテラルレポ。主にプライムブローカーとのディーラーバイラテラル |
| 日本 IG 社債（中小発行体） | 非常に限定的なレポ。ポジションはアウトライトで保有 |
| 日本ハイイールド社債 | 事実上レポなし。バランスシート上にアウトライトで保有 |

Tier 1 発行体規模を下回るほとんどの日本社債ベーシス取引では、債券レッグは直接レポではなくプライムブローカーファイナンスを介して資金調達される。これはコストを追加し、米ドルまたはユーロ IG ベーシス取引に対するスケーラビリティを低下させる。

### バーゼル III レバレッジ比率

ロング現物・ショート CDS ポジションは、ネット信用リスクがヘッジされていても、銀行のバランスシート上のレバレッジエクスポージャー指標を膨張させる。2010 以降のレバレッジ比率の執行は、大規模なベーシスポジションを在庫として保有するディーラーの意欲を低下させ、ストレスが生じたときに構造的にベーシスを拡大させた。

### リスクウェイト資産（RWA）

| コンポーネント | 影響 |
|---|---|
| 債券レッグ | 発行体格付けに基づいて銀行勘定またはトレーディング勘定の RWA を生む |
| CDS レッグ | CDS カウンターパーティに対するカウンターパーティ信用リスク RWA を生む |
| ネットヘッジの認識 | 標準的手法では限定的。内部モデル手法（FRTB の対象）ではより良い認識 |

信用リスクがヘッジされていても、RWA の消費はディーラーの在庫保有を抑制する。

### 単一カウンターパーティ信用限度

米国と EU の規則は単一カウンターパーティへのエクスポージャーを制限する。現物債の発行体と同じカウンターパーティに対して買われた CDS プロテクションは、慎重にネッティングしなければならない二重エクスポージャーを生む。

### 流動性カバレッジ比率（LCR）

債券レッグは、レベル 2 HQLA として適格であれば一般に HQLA にカウントされる。CDS レッグはカウントされない。したがって一部のベーシス取引は、債券の特性に応じて、銀行にとって LCR 中立またはプラスである。

## 2010

以降の日本固有の市場構造

| 期間 | 市場構造の変化 |
|---|---|
| 2010-2012 | CDS の中央清算への移行（JSCC および ICE Clear Credit）。LIBOR-OIS ベーシスのスピルオーバー効果 |
| 2013-2015 | BOJ QQE 開始。大規模な JGB 買入プログラムがレポスペシャルと JGB カーブを再形成した |
| 2016 | BOJ NIRP とイールドカーブコントロール（YCC）の導入。10年 JGB がゼロ近辺にピン留めされた |
| 2017-2019 | 安定した YCC 期。タイトな日本 IG 社債スプレッドと狭いベーシス |
| 2020 第1 四半期 | COVID 信用ショック。日本を含むグローバルな IG 社債銘柄全体でのネガティブベーシスの広範な拡大 |
| 2020 第2四半期-第4 四半期 | BOJ 社債買入ファシリティとディーラーのバランスシート回復。ベーシスが部分的に正常化 |
| 2022-2023 | BOJ YCC 調整（許容バンドの拡大、その後 YCC 終了）。散発的な JGB カーブの変動と社債の再価格付け |
| 2024-2025 | ほとんどの日本 IG 銘柄で狭いベーシスが継続。グローバルな銀行ストレスイベント時の金融銘柄での散発的な拡大 |

構造的な構図：ディーラーの在庫保有能力は 2008, 以前より小さく、一方で散発的な信用リスクオフイベントは依然として発生する。したがってベーシス取引は、低頻度だが高振幅の乖離を示す。

## COVID 第1 四半期2020 ベーシス拡大

第1 四半期2020 の COVID エピソードは、2008-2009以来、日本 IG 社債信用で最大のネガティブベーシスの乖離を引き起こした：

| ステージ | パターン |
|---|---|
| 2020 年 2 月下旬 | リスクオフが始まる。CDS スプレッドが現物債よりやや速く拡大 |
| 2020 年 3 月上旬 | ETF、ディーラー、ALM ポートフォリオによる強制的な現物債売却。現物スプレッドが CDS より速く拡大 |
| 2020 年 3 月中旬 | 現物債のビッドアスクが急激に拡大。一部の債券はほぼ取引不能。CDS は気配が出続けた |
| 18 2020 年 3 月 | ベンチマーク日本 IG 社債のネガティブベーシスが複数年来のワイドに到達 |
| 2020 年 3 月下旬 | BOJ が社債買入ファシリティと ETF 買入を発表。ディーラーのバランスシートが安定化 |
| 2020 年 4 月-6 月 | ベーシスが正常化。3 月に開始したロング現物・ショート CDS 取引が強い損益を返した |
| 2020 年第3四半期-第4 四半期 | ベーシスが狭い通常レンジに戻って落ち着いた |

このエピソードは 2008-2009 のパターンを映したが、数カ月ではなく数週間続いた。ベーシス取引の実務家への教訓：極端な乖離は短い窓であり、コミットされた資本と短期的な MTM の痛みへの許容を必要とする。

## 2022-2023 の BOJ YCC 調整

| イベント | ベーシスへの影響 |
|---|---|
| 2022 年 12 月 BOJ が YCC 許容バンドを拡大（0.25 → 0.50 パーセント） | 10年 JGB 利回りのジャンプ。日本 IG 社債の現物債が再価格付け。CDS は遅れた。ベーシスが一時的に拡大 |
| 2023 の段階的な YCC 調整 | 散発的な再価格付け。より小規模な乖離 |
| 2023 年 7 月 BOJ が YCC を正式に緩和 | 10年 JGB の上限が事実上 ~1.00 パーセントに引き上げ。現物スプレッドが拡大。CDS は遅れて反応 |
| 2024 年 3 月 BOJ が YCC と NIRP を正式に終了 | JGB カーブが正常化。現物社債が再価格付け。ベーシスが数週間かけて正常化 |

パターン：現物債の再価格付けが CDS の再価格付けを先導する。なぜなら、現物債はイールドカーブに敏感なポートフォリオ（生命保険会社、銀行）によってより直接的に保有されるからである。CDS スプレッドは、トレーダーが合成ポジションを現物に合わせて押すときに、ディーラーのバランスシート能力によって決まる遅れを伴って調整される。

### 投資家プロファイル

| 投資家 | ベーシス取引プロファイル |
|---|---|
| コンバーチブルアービトラージ / クレジットファンド | アクティブなベーシス取引利用者。MTM 変動への許容 |
| マルチストラテジーヘッジファンド | 選別的な利用。しばしば資本構造アービトラージ取引と組み合わせる |
| 銀行ディーラーデスク | ボルカー / 同等規制後は限定的なプロップベーシス取引。主にマーケットメイクの在庫保有 |
| 保険 / 年金 | 一般にアクティブなベーシストレーダーではない。アウトライトの現物債保有に注力 — [[insurance/japan-life-insurance-alm-overview|life ALM]] を参照 |
| 政府系ファンド | 主要な乖離時の散発的なベーシス取引配分 |

### 取引サイジング

ベーシス取引は以下によってスケールされる：

| 制約 | 実務上の影響 |
|---|---|
| 債券レッグのレポ / ファイナンス能力 | ロング現物のサイズに上限 |
| ショートプロテクションレッグの CDS 市場流動性 | 同一銘柄・テナーでのショート CDS のサイズに上限 |
| カウンターパーティ CSA 条件（マージニング、適格担保） | CDS レッグの資金調達コストに影響 |
| 単一銘柄のクレジットイベントのテールリスク | ヘッジがあっても、クレジットイベント時のギャップリスク |
| 規制資本コスト | 銀行所有のポジションは RWA / レバレッジを消費 |

日本 IG 銘柄での 100 百万ドル相当のネガティブベーシスポジションは大きく、500 百万は非常に大きく、数十億は最も流動性の高い銘柄に集中するまれなものである。

### ロールダイナミクス

債券満期までの保有が実行可能でない場合、CDS プロテクションは各ロールでロールしなければならない（5年 CDS → 次の 5年シリーズなど）。ロールダウン損益にインデックスロールベーシスを加えたものが、ヘッドラインの取引経済性にノイズを加える。
