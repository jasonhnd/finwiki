# 国債市場特別参加者(日本のプライマリーディーラー制度)

## 要約

国債市場特別参加者制度は、日本のプライマリーディーラー体制である:財務省(MoF)が運営するフレームワークであり、これに基づき、一定の金融機関群が、JGB カーブ全体にわたる入札・マーケットメイク義務を引き受けることと引き換えに、MoF の JGB 入札における主要なカウンターパーティとして認定される。この制度は 2004 に導入され、複数回の改定を経て洗練されてきた;現行のリストは通常およそ 20 社を含み、日本のメガバンクの証券子会社、主要な日本の独立系証券会社、日本で営業する外国投資銀行の子会社で構成される。

特別参加者は次をコミットする:(i)各 JGB 入札において主要な年限にわたり有意な入札を提出すること;(ii)JGB セカンダリー市場で両建てのクオートを提供すること;(iii)取引活動とインベントリを報告すること;(iv)十分な財務資源を維持すること。引き換えに、彼らは次を受ける:(i)MoF の非価格入札(第 I 非価格競争入札および第 II 非価格競争入札)へのアクセスで、加重平均落札価格で JGB を取得できること;(ii)JGB 発行方針に関する MoF との定期的な対話;(iii)MoF の買入消却への参加;(iv)MoF、BoJ、FSA の目から見て認定された JGB マーケットメイカーとしての地位。

日本の制度は、米国のプライマリーディーラー制度(ニューヨーク連銀が運営)や英国のギルト・エッジ・マーケットメイカー(GEMM)制度と概念的に類似するが、日本の市場構造と MoF の入札設計のアプローチを反映した独特の運用上の特徴を持つ。

FinWiki にとって、このエントリは、特別参加者のリストとティアリング、義務、便益、非価格競争入札の仕組み、米国・英国のプライマリーディーラー制度との構造的比較を扱う。

## 制度概観

| 要素 | 詳細 |
|---|---|
| 規制者 / 運営者 | 財務省(MoF)、理財局、国債業務課 |
| 設立 | 2004 (従前の JGB プライマリーディーラー類似の取り決めの正式化) |
| 認定企業 | 随時およそ 20 社(企業の加入 / 離脱により年ごとに変動) |
| 認定の基準 | 申請 + MoF による自己資本充実度、マーケットメイク能力、JGB 活動、コンプライアンスの評価 |
| 更新 | 年次レビュー;企業の追加または削除が可能;行為基準が執行される |
| 会員カテゴリ | 単一ティア(JGB 制度に正式な A / B サブティアはない);実務上、一部の運用上の差別化は存在する |
| 公表 | MoF が特別参加者ページに現行のリストを公表する |

認定企業の数は、FSA のより広範な [[securities/financial-instruments-business-operators-japan-index|Financial Instruments Business Operators]] ユニバースより大幅に少なく、入札カウンターパーティを入札・マーケットメイク義務を満たせる企業に意図的に集中させる MoF の方針を反映している。

## リスト構成(典型)

正確なリストは MoF によって定期的に更新されるが、典型的な構成は次のとおりである:

| ディーラーカテゴリ | 代表的な企業 |
|---|---|
| 日本のメガバンクの証券子会社 | 三菱 UFJ モルガン・スタンレー証券(MUFG Securities)、SMBC 日興証券、みずほ証券(およびその証券エンティティ内の債券トレーディングデスク) |
| 独立系の日本の証券会社 | 野村證券、大和証券 |
| その他の日本の証券会社 | SBI 証券、東海東京証券、岡三証券(年により変動) |
| 信託その他のメガバンク系債券エンティティ | 該当する場合(具体的なエンティティは変動) |
| 日本における外国投資銀行の子会社 | JPMorgan 証券、Goldman Sachs 証券(Japan)、Morgan Stanley MUFG 証券、Citigroup グローバル・マーケッツ証券(Japan)、ドイツ証券、バークレイズ証券(Japan)、BNP パリバ証券(Japan)、HSBC 証券(Japan)、UBS 証券(Japan) |
| その他 | ソシエテ・ジェネラル証券(Japan)、クレディ・アグリコル証券(Japan)、および存在する場合の同様の外国企業の子会社 |

注目点:日本登録の証券会社として営業する外国企業の子会社(FIEA 免許の下 — [[securities/financial-instruments-business-operators-japan-index|FIBO registry]] 参照)は適格であり、リストの相当な割合を占める。外国企業へのこの開かれた会員資格は、日本の制度の構造的特徴の一つである。

上記のリストは例示的である;公式の現行リストは MoF によって維持され、公式ページで更新される(出典参照)。

## 特別参加者の義務

特別参加者は MoF 規則の下で次の義務を引き受ける:

| 義務 | 詳細 |
|---|---|
| 各入札での入札義務 | 主要な年限にわたる各 MoF JGB 入札で有意な入札を提出する;最低参加しきい値は MoF が入札規則で定義する |
| 落札シェア | 一定期間にわたり最低限の落札シェアを達成する(年次レビューで評価) |
| セカンダリー市場のクオート | JGB セカンダリー市場で両建てのクオートを提供する;義務は年限と時間帯により較正される |
| インベントリと活動の報告 | JGB のインベントリポジションと取引高を MoF に報告する |
| 自己資本充実度と規制コンプライアンス | FIEA が要求する資本と運用基準を維持する |
| MoF への協力 | 発行計画、市場機能、商品開発に関する MoF の対話に参加する |

これらの義務に対するパフォーマンスは年次でレビューされる;しきい値を満たさない企業は、リストから格下げまたは削除されうる。

## 特別参加者の地位の便益

義務と引き換えに、特別参加者は次を受ける:

| 便益 | 詳細 |
|---|---|
| 第 I 非価格競争入札(非価格 I) | JGB 入札の一部を加重平均落札価格で受けられる適格性;特別参加者のシェアに基づく入札前の配分 |
| 第 II 非価格競争入札(非価格 II) | 加重平均落札価格での追加的な入札後の配分;もう一つの非価格の権利 |
| MoF の対話と協議 | 発行計画、市場機能、JGB プログラム開発に関する MoF との定期的なフォーラム;MoF の計画への影響力 |
| 買入消却への参加 | MoF の JGB 買入消却 / 負債管理オペレーションに参加する適格性 |
| 認定 | JGB マーケットメイカーとしての MoF および BoJ による正式な認定;顧客向けのディストリビューションとレピュテーションに有用 |
| 運用施設へのアクセス | 標準的な JGB の清算・決済インフラへのアクセス(日銀ネット、[[securities/japan-securities-clearing-corp|JSCC]]) |

非価格競争入札(I および II)の仕組みは特に重要である:特別参加者に、入札の加重平均価格での保証された配分チャネルを提供し、競争入札単独の価格発見リスクの一部を取り除く。非価格 I は競争入札の前に配分される;非価格 II は後に、同じ加重平均価格で配分される。

## JGB 制度における入札の仕組み

JGB 入札プロセスは、他の入札者と並んで特別参加者を受け入れる:

| 入札の構成要素 | 適格性 / 仕組み |
|---|---|
| 価格競争入札 | すべての適格入札者(特別参加者 + その他)に開かれている;競争入札による配分 |
| 第 I 非価格競争入札(非価格 I) | 特別参加者のみ;入札前;予想加重平均価格での配分;過去の入札パフォーマンスに基づく権利が条件 |
| 第 II 非価格競争入札(非価格 II) | 特別参加者のみ;入札後;競争入札からの実際の加重平均落札価格での配分;追加的な権利 |
| 入札形式 | ほとんどの JGB はマルチプライス(コンベンショナル)入札;シリーズにより利回りベースまたは価格ベースのコンベンショナル |

非価格 I の権利は、特別参加者が強力な入札の実績を維持するインセンティブを生み出す。非価格 I のシェアが過去のパフォーマンスに応じてスケールするからである。

## 米国プライマリーディーラー制度との比較

ニューヨーク連邦準備銀行が運営する米国のプライマリーディーラー制度は、構造的な類似点と相違点を持つ:

| 側面 | 米国プライマリーディーラー | JGB 特別参加者 |
|---|---|---|
| 運営者 | ニューヨーク連邦準備銀行 | 財務省(日本) |
| 企業数 | ~24 社(変動);リストは NY 連銀が維持 | ~20 社;リストは MoF が維持 |
| 会員構成 | 米国の銀行系ディーラー + 非銀行ディーラー + 外国銀行の米国子会社 | 日本の証券会社 + 外国投資銀行の子会社 |
| 入札義務 | 各米国財務省入札で入札する;有意なシェア | 各 JGB 入札で入札する;有意なシェア |
| セカンダリー市場義務 | 合理的な市場活動;報告 | 両建てのクオート;報告 |
| 特別アクセス | TOMO(一時的公開市場操作)のカウンターパーティ;SOMA ポートフォリオオペレーションのカウンターパーティ | 第 I・第 II 非価格競争入札;MoF 買入消却への参加 |
| 落札の仕組み | 競争入札(財務省証券は 1992以降ユニフォームプライス) | マルチプライス競争入札 + 非価格 I・II |
| 規制者との関わり | 運用上は Fed(NY 連銀);実体的には OCC、SEC、銀行規制当局 | 運用上は MoF;プルーデンス上は FSA;金融オペレーションでは BoJ |
| 外国企業の参加 | あり(広範) | あり(広範) |

主要な構造的相違:米国財務省はユニフォームプライス入札を実施する(1992以降)ため、すべての落札者が同じ(最も高い受入)利回りを支払う;日本はほとんどの JGB 年限で引き続きマルチプライス(コンベンショナル)入札を使用するため、落札者は自らの入札利回りを支払う。これにより、日本の非価格 I・II の仕組みは、特別参加者に加重平均価格でのアクセスを提供するため、特に価値がある。

## 英国 GEMM 制度との比較

英国には、英国債務管理庁(DMO)に認定されたギルト・エッジ・マーケットメイカー(GEMM)がある:

| 側面 | 英国 GEMM | JGB 特別参加者 |
|---|---|---|
| 運営者 | 英国債務管理庁(DMO) | 財務省(日本) |
| 企業数 | およそ 15-20 社の GEMM | ~20 社の特別参加者 |
| 入札での入札義務 | 必須 | 必須 |
| セカンダリー市場義務 | 両建てのクオート | 両建てのクオート |
| インフレ連動の認定 | インフレ連動ギルト向けに別途のインデックスリンク GEMM(IL-GEMM)認定 | 単一ティア;JGBi 入札は同じ特別参加者のリストに開かれている |
| ティアリング | 一部の運用上の差別化 | 単一の認定リスト |
| ストリップ・スイッチ入札へのアクセス | 標準 | 標準 |

英国は歴史的にインフレ連動 GEMM の認定をサブティアに分離してきた;日本は JGBi についてその分割を採用していない。

## なぜこの制度が重要なのか

特別参加者制度は次にとって決定的に重要である:

1. **JGB 入札の成功**:信頼でき、資本を備えたカウンターパーティ基盤は、入札が秩序ある価格でクリアすることを MoF に保証し、入札不成立のリスクを減らす。
2. **セカンダリー市場の流動性**:両建てクオートの義務は、より広範な機関投資家と BoJ がアクセスできる連続的な JGB 市場を支える。
3. **MoF への情報フロー**:定期的な対話により、MoF は市場のフィードバックに基づいて発行計画を較正できる。
4. **ディーラーフランチャイズの経済性**:特別参加者の地位は証券会社にとって有意なフランチャイズ資産である;非特別参加者は JGB マーケットメイクにおいて構造的に不利である。
5. **外国企業の統合**:外国企業の日本子会社への開かれた会員資格は、グローバルな投資銀行を JGB 市場に統合し、クロスボーダーの JGB 投資家活動を支える。
