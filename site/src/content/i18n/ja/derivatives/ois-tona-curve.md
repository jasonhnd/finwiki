# OIS TONA カーブと円ディスカウンティング

## 要約

円のオーバーナイト・インデックス・スワップ(OIS)は、一方の当事者が定められたテナーにわたって固定金利を支払い、取引相手が同じ期間にわたる日次の東京ターム物平均金利(TONA)の幾何(複利)平均を支払う OTC 金利デリバティブである。TONA は日銀が公表する無担保コール翌日物取引の出来高加重平均であり、グローバルな IBOR 移行のもとで円通貨に指定されたリスクフリーレート(RFR)である。

LIBOR 後(円 LIBOR は 2021 に公表停止、合成円 LIBOR の段階的廃止は 2023に完了)において、TONA を参照する OIS は円デリバティブの標準的な割引カーブである ── OIS 自体の価格設定のためにも、また担保付き円 IRS([[derivatives/japan-irs-market]] を参照)、円-米ドルベーシススワップ([[derivatives/yen-basis-swap-market]] を参照)、円建て CCP 清算取引、円キャッシュで差し入れられる CSA 担保付き OTC ブッキングで用いられる割引係数としても。

FinWiki にとって、OIS-TONA カーブは日銀の短期金利政策期待の価格表現であり、あらゆる円建てデリバティブの基礎にある割引カーブであり、近い将来の日銀の金利決定に関する最も直接的で可視的な市場の読みである。

## ウィキ上の位置づけ

この項目は [[derivatives/INDEX|derivatives index]] の下に位置する。より長期のテナーカーブについては [[derivatives/japan-irs-market]] と、JGB が含意する金利カーブについては [[derivatives/jgb-futures-curve]] と、OIS カーブが通貨間調達とどう相互作用するかについては [[derivatives/yen-basis-swap-market]] と対照して参照のこと。基礎となる TONA の現物の仕組みは [[money-market/call-market-structure]]、日銀の政策実施は [[money-market/boj-open-market-operations]] である。

## TONA ── 基礎となる金利

TONA は無担保コール翌日物取引に基づいて日本銀行により日次で計算・公表される:

| 要素 | 詳細 |
|---|---|
| 正式名称 | 東京ターム物平均金利(無担保コールオーバーナイト物加重平均レート)。 |
| 計算・公表者 | 日本銀行。 |
| 基礎となる取引 | 短資会社を通じて仲介され日銀に報告される無担保コール市場翌日物取引。 |
| 手法 | 各営業日の適格な翌日物取引の出来高加重平均。 |
| 公表 | 日銀により翌営業日(T+1)に公表される;以前は TONAR / 東京ターム物平均金利としても配信された。 |
| リセット | 日次;金利自体に固有の期間構造はない。 |
| 通貨 / 慣行 | 日本円;円のマネーマーケットでは慣行により ACT/365 の日数計算。 |

TONA は、日銀と FSA の参加のもとで招集された官民の団体である日本円金利指標に関する検討委員会が調整した IBOR 移行のもとで、指定された RFR として円 LIBOR に置き換わった。この移行は、通貨ごとの RFR 採用に関する金融安定理事会(FSB)の勧告に従ったものである。

TONA は実際の無担保翌日物取引([[money-market/call-market-structure]] で詳述される同じ市場)を反映するため、LIBOR を終わらせた操作リスクに対して頑健である。ただし、日銀の政策実施には敏感である:日銀の潤沢な準備預金政策がコール取引を抑制すると、TONA の出来高が縮小し、一部の期間において金利がより薄くなることがある。

## LIBOR 後の移行

円 LIBOR の移行は調整されたタイムラインに沿って行われた:

| 年 | イベント |
|---|---|
| 2017  | 英 FCA が 2021末までの LIBOR 公表停止を発表。 |
| 2018  | 日銀 / FSA / 業界の参加のもとで日本円金利指標に関する検討委員会が設立。 |
| 2018–2021  | 新規の円スワップおよびローン業務の TONA 参照への移行;レガシーの円 LIBOR 契約は ISDA 2020 フォールバックプロトコルにより修正またはフォールバック適用。 |
| 2021 末 | 円 LIBOR が代表性のあるベースで公表停止。一部のレガシーの 1か月、3か月、6か月テナー向けに、ワインドダウンの橋渡しとして合成円 LIBOR(FCA 運営、非代表性)が導入。 |
| 2023  | 合成円 LIBOR が公表停止。すべての円金利デリバティブが TONA を参照する(あるいは一部のケースでは、ターム固定市場向けに TIBOR を参照)。 |
| 継続中 | TIBOR(東京銀行間取引金利)は、ターム固定を必要とする一部の円建てローンおよび債券商品のために存続する;D-TIBOR と Z-TIBOR は全銀協 TIBOR 運営機関(JBATA)が運営する。 |

移行の結果:円 IRS 市場、OIS 市場、CSA 担保付き OTC デリバティブは現在、基礎となる変動参照金利および割引金利として TONA を用いる。ISDA フォールバックを通じて移行された円 LIBOR デリバティブは現在、TONA の後決め複利に固定の信用調整スプレッド(CAS)を加えたものを参照する。

## 日銀の政策金利の波及

OIS-TONA カーブは、カーブのテナーにわたる期待される日銀翌日物金利の最も直接的な市場表現である:

| テナー | OIS レートが表現するもの |
|---|---|
| 1週、1か月、3か月 OIS | 次回の 1–2 回の金融政策決定会合(MPM)を通じた近い将来の日銀金利期待。 |
| 6か月、1年 OIS | 今後 12 か月にわたる、ありうる MPM での利上げ / 利下げを含む予想される日銀の金利パス。 |
| 2年、3年、5年 OIS | 中期の政策パス;MPM の軌道に小さなタームプレミアムを加えたものを反映する。 |
| 10年、20年、30年 OIS | 長期の均衡金利期待にタームプレミアムを加えたもの;同じテナーの IRS カーブより流動性は低い。 |

MPM の日付前後の 1年または 2年 OIS レートの動きは、政策のサプライズに関する最も注目されるシグナルである。マイナス金利と YCC を脱した 2024 年 3 月の日銀の体制転換後、OIS カーブはありうる金利正常化のパスを反映して急速にリプライシングされた;その後の各 MPM 前後の OIS の動きは綿密に注視されてきた。

OIS-TONA と政策金利の誘導目標の関係は次のとおりである:

- 政策金利の誘導目標(目標近辺の無担保コール翌日物金利)は、TONA が日次で追随するほぼゼロの下限を設定する。
- OIS フォワードレートは、各テナーにわたって日銀が政策金利をどこに設定するかの確率加重期待を織り込む。
- 観測される TONA と同じ期間の含意 OIS フォワードとの差は、MPM 間の政策期待のリプライシングを明らかにする。

政策実施層については [[financial-regulators/boj-monetary-policy]](存在する場合)および [[money-market/boj-open-market-operations]] を参照;この項目はデリバティブによる表現を扱う。

## OIS カーブの構築

市場の実務家は円 OIS-TONA カーブを次のように構築する:

| ステップ | インプット |
|---|---|
| 1. スポット | 日銀の政策誘導目標に固定された本日の TONA フィキシング(T-1 公表金利)。 |
| 2. 短期端 | インターディーラークオートからの 1週、2週、1か月、2か月、3か月テナーの OIS レート。 |
| 3. 中期テナー | 6か月、9か月、1年、18か月、2年、3年の OIS レート。 |
| 4. 長期端 | 流動性がクオートを支える範囲で、5年、7年、10年、15年、20年、30年の OIS レート。 |
| 5. ブートストラップ | 各インプットの OIS レートがパーで価格付けされるように割引係数カーブ D(t) を解く;前方に反復する。 |
| 6. スムージング | ディーラーの価格設定モデルと整合的なスムージング補間(三次スプライン、単調三次、または対数割引の区分線形)を適用する。 |

得られた割引カーブ D(t) は次に用いられる:

- 新規の OIS 取引をパーで価格付けする;
- 担保付き円スワップの将来キャッシュフローを割り引く(2010後に採用された「OIS ディスカウンティング」慣行);
- CSA 担保付き円エクスポージャーの現在価値(PV)を計算する;
- JSCC 清算の円 IRS ポジションを時価評価する(JSCC は円清算商品に TONA-OIS ディスカウンティングを用いる)。

非担保または異なる担保による取引については、割引カーブは資金調達評価調整(FVA)によって、あるいは異なる担保金利カーブへの切り替えによって調整される。これは 2010後に標準となった「マルチカーブ枠組み」である。

## 円 IRS のスワップレート割引カーブ

円 IRS 市場([[derivatives/japan-irs-market]] を参照)は価格設定において二重カーブである:

| カーブ | 目的 |
|---|---|
| TONA-OIS | 割引カーブ ── 将来キャッシュフローの PV に用いる。 |
| TONA フォワードカーブ | 予測カーブ ── 将来の TONA 変動レッグのリセットを予測するために用いる。 |
| (歴史的)TIBOR フォワードカーブ | TIBOR 参照のレガシー IRS について、1か月 / 3か月 / 6か月の TIBOR リセットを予測するために用いる;PV は依然として TONA-OIS で行う。 |

この二重カーブ枠組みは、TIBOR 変動の IRS であっても、取引が標準的な CSA 条件のもとで円キャッシュで担保付けされていれば、割引に TONA-OIS カーブを用いることを意味する。TIBOR-TONA ベーシスは別個のベーシススワップクオート(TIBOR 対 TONA OIS、bp 単位)に反映される。

## 市場の厚みと流動性

| テナー帯 | 流動性評価 |
|---|---|
| 1か月–6か月 | きわめて流動的;タイトなビッド・アスク(通常 1 bp 未満)。 |
| 1年–3年 | 流動的;最も取引される MPM 期待のテナー。 |
| 5年–10年 | 機関投資家フローにとって流動的;ビッド・アスクは通常 1–2 bp。 |
| 15年–30年 | OIS の形では TIBOR IRS や JGB 現物の形よりも流動性が低い;ビッド・アスクはより広い。 |

円 OIS の想定元本残高は、合算ベースで数兆円換算規模である(BIS の OTC デリバティブ統計は円 IRS / OIS 区分内にこれを集計している)。JSCC は清算された円 OIS の大部分を清算する;非標準のテナーや仕組み取引には非清算の相対 OIS が存在する。

## TONA 先物(隣接する上場商品)

OSE は TONA 3か月物先物を上場しており、短期テナーの TONA 期待に対する取引所取引・JSCC 清算のエクスポージャーを提供することで OTC OIS 市場を補完する。仕様:

| 要素 | 詳細 |
|---|---|
| 原資産 | 契約参照期間にわたる 3か月複利 TONA。 |
| 呼値 | 100.00 から先物レートを引いたもの;標準的な短期金利先物の慣行。 |
| ティック価値 | 契約ごとに定義;リテールおよび小規模機関向けの小さな想定元本サイズ。 |
| 決済 | 契約満期時に参照 3か月 TONA 複利に対して現金決済。 |
| 清算 | JSCC。 |

TONA 先物は流動性を増しつつあるが、依然として OTC OIS より小さい。OTC より取引所上場の構造を好むユーザーにとって、短期金利ヘッジに有用である。
