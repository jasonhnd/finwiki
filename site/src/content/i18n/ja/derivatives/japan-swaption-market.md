# 日本のスワップション(金利オプション)市場

## 要約

スワップションは、保有者に将来の日付かつ事前合意のストライク金利で定義された金利スワップを締結する権利(義務ではない)を付与する OTC 金利オプションである。JPY スワップションは [[derivatives/japan-irs-market|JPY interest-rate swap]] および [[derivatives/ois-tona-curve|OIS-TONA]] の市場に対するボラティリティ感応的な補完であり、日本の生命保険会社、メガバンクの財務デスク、ストラクチャードノートの発行体ヘッジデスクが将来の JPY 金利ボラティリティについての見方を表現する(またはヘッジする)主要な手段である。

JPY スワップションフローにおける2つの主要な構造形態は、ヨーロピアン型のペイヤー / レシーバースワップション(将来の日付に定義テナーの IRS で固定を支払うまたは受け取る権利)と、バミューダン型のコーラブルスワップション(複数の行使日、しばしばコーラブルな JPY 建て債券やストラクチャードノートに組み込まれる)である。後者は、金利オプション性を組み込んだ保険会社の変額年金型商品(GMxB 型の保証、ここで保険会社は実質的に契約者行動に対して金利ボラティリティをショートしている)のヘッジに特に関連する。

FinWiki にとって、本エントリはスワップションの仕組み、JPY のインプライドボラティリティグリッド、ペイヤー対レシーバーのフロープロファイル、バミューダンコーラブル構造、生命保険会社の GMxB ヘッジ事例、そして JPY スワップションのマーケットメイクのためのディーラーフランチャイズ構造をカバーする。

## ウィキ上の位置づけ

本エントリは [[derivatives/INDEX|derivatives index]] の配下、金利オプションのクラスターに位置する。スワップションが参照する根底の IRS 市場として [[derivatives/japan-irs-market]]、ディスカウントカーブと短期金利側として [[derivatives/ois-tona-curve]]、そして同じボラティリティグリッドのインプットを共有する CMS デリバティブの従兄弟として [[derivatives/japan-cms-constant-maturity-swap]] と併せて読むこと。生命保険会社のエンドユーザー需要側は [[insurance/japan-life-insurance-alm-overview|Japan life-insurance ALM overview]] および [[insurance/economic-value-based-solvency|economic-value-based solvency (ESR)]] にアンカーされている。

## 商品の仕組み

標準的な JPY スワップションは以下の構造を持つ:

| 要素 | 詳細 |
|---|---|
| 原資産 | 事前指定された JPY 金利スワップ(例:TONA 複利または TIBOR を参照する 10Y JPY IRS) |
| 権利 | オプション保有者がオプションの行使日にスワップを締結する権利 |
| 方向 | **ペイヤースワップション**:固定を支払う権利(= 変動を受け取る);金利がストライクを上回って上昇すれば利益。**レシーバースワップション**:固定を受け取る権利(= 変動を支払う);金利がストライクを下回って低下すれば利益。 |
| ストライク | 合意された固定金利のストライク(通常はアットザマネーフォワード、ATMF だが、ITM と OTM のストライクも取引される) |
| オプション満期 | オプション保有者が行使する日(「満期」) |
| スワップテナー | 原スワップのテナー、慣行的に「満期 × テナー」と書かれる(例:「1Y × 10Y」= 1年満期から 10年スワップへ) |
| スタイル | ヨーロピアン(単一行使)、バミューダン(複数行使日)、またはアメリカン(連続;スワップションではまれ) |
| 決済 | 定義された評価方法論への現金決済(インプライドスワップの PV)または現物決済(オプション保有者が行使時にライブのスワップへ引き渡される) |
| 文書 | ISDA マスター契約 + CSA |

経済的内容:スワップションはフォワードスワップ金利に対する Black 型オプションであり、原フォワードスワップ金利のインプライドボラティリティが鍵となる価格設定インプットである。

## JPY スワップションのインプライドボラティリティグリッド

市場は JPY スワップションのインプライドボラティリティ(通常は年率ベーシスポイントで表現され、ノーマルボラティリティの慣行、またはディーラーの慣行に応じて Black 対数正規ボラティリティで表現される)を2次元のグリッド上で気配する:

- **オプション満期**:1M、3M、6M、1Y、2Y、3Y、5Y、7Y、10Y、15Y、20Y
- **原スワップテナー**:1Y、2Y、3Y、5Y、7Y、10Y、15Y、20Y、30Y

最も流動性の高いポイントは通常:

| グリッドポイント | ユースケース |
|---|---|
| 1M × 10Y、3M × 10Y | 短期のイベントボラティリティの表現(BoJ MPM、市場ストレス周り) |
| 1Y × 10Y | ヘッドラインのベンチマークボラティリティポイント;広く気配される |
| 1Y × 5Y、1Y × 30Y | 異なるテナーでのカーブボラティリティの表現 |
| 5Y × 5Y、5Y × 10Y、10Y × 10Y | 長期の保険会社 / ストラクチャードノートのヘッジ |
| 10Y × 20Y、20Y × 30Y | 長テナーの保険会社 ALM と年金基金のヘッジ |

JPY のボラティリティの慣行は、IBOR 移行を経て Black 対数正規とノーマルボラティリティ(bp / 年)の間でいくらかシフトした;絶対的な金利水準が低くパーセントボラティリティが不安定になる場合には特に、ノーマルボラティリティの気配が今やより一般的である。

グリッドはディーラー間の気配、ブローカースクリーン(Tradition、ICAP、BGC、Tullett Prebon)、ディーラー銀行の IR 資料から構築される。フルのボラティリティサーフェスは [[derivatives/japan-cms-constant-maturity-swap|CMS]] と CMS スプレッドデリバティブの価格設定にとっても鍵となるインプットである。

## ペイヤー対レシーバーのフロー

JPY スワップションの構造的なフローの不均衡は、異なるテナーで異なる側に傾く:

| フローの源 | 方向 | テナーの集中 |
|---|---|---|
| 生命保険会社(変額年金ヘッジ) | **レシーバーを買う**(レシーバースワップションをロング) — 金利がさらに低下し契約者保証の価値が増すことに対するヘッジ | 長テナー(10Y × 20Y、20Y × 30Y) |
| 生命保険会社(一般勘定 ALM) | **レシーバーを売る**(長期債ポートフォリオでの利回り増強)または **レシーバーを買う**(下方ヘッジ) | 長テナー |
| メガバンク財務(IRRBB ヘッジ) | 混在;金利上昇を懸念するときは **ペイヤーを買う** | 中テナー(3Y、5Y、7Y) |
| ストラクチャードノート発行体(コーラブル債ヘッジ) | **バミューダンレシーバーを買う**(投資家に書いたコールオプションをヘッジするため) | 中から長;ノート構造に合致 |
| 外国マクロファンド | 両側;戦術的な見方ドリブン | 混在 |
| 事業会社財務 | 限定的な直接フロー;[[derivatives/japan-irs-market|outright IRS]] のヘッジへの依存がより大きい |  |

長期レシーバースワップションへの保険会社の需要は、長端のボラティリティサーフェスの構造的な駆動要因である。2024 以降の BoJ レジームのシフト(NIRP / YCC からの脱却)は、レシーバーボラティリティの価格設定ダイナミクスを実質的に変えた。なぜなら、それ以前の10年間のゼロ近傍金利の環境がレシーバーの価値を圧縮しペイヤースワップションの活動を前面に押し出していたからである;正常化する金利レジームはレシーバーヘッジの実務的価値を高める。

## バミューダンコーラブルスワップション

バミューダンスワップションは、(単一のヨーロピアンの日付や連続ではなく)事前定義された日付の集合での行使を許す。JPY における最も一般的なバミューダン構造は、JPY 建て債券やノートに組み込まれたコーラブルである:

| 構造 | 組み込まれたオプション |
|---|---|
| コーラブル債(発行体コーラブル) | 債券の発行体が指定日に債券をコールする権利を持つ;発行体が固定金利の負債に対するバミューダンレシーバースワップションをロングしていることに相当 |
| プッタブル債(投資家プッタブル) | 投資家のプット権利;あまり一般的でない;(発行体の視点から)投資家がバミューダンペイヤーをロングしていることに相当 |
| キャンセラブルスワップ | キャンセル権が組み込まれたスワップ;ストラクチャードな事業会社ヘッジで一般的 |
| コーラブルストラクチャードノート | 定期的な発行体コール権を持つクーポン支払いのノート;発行体はバミューダンレシーバースワップションを通じてヘッジ |

バミューダンスワップションの価格設定は、各日付での行使判断が金利水準とオプションの継続価値に依存するため、ヨーロピアンスワップションよりも実質的に複雑である。標準的な価格設定アプローチには、Hull-White 単一ファクターまたはマルチファクター短期金利モデル、LIBOR マーケットモデル、回帰ベースの行使ルールを伴うモンテカルロ(Longstaff-Schwartz)が含まれる。

JPY バミューダンスワップションの取引量は長テナー構造に集中しており、相当な規模の JPY コーラブル債 / ストラクチャードノートの発行市場([[derivatives/structured-bond-japan-retail-issuance|JPY structured bond retail issuance]] および事業会社の資金調達で触れられている)のヘッジ手段として機能する。

## 生命保険会社の変額年金(GMxB)ヘッジ

最低保証給付付きの変額年金(VA)商品(GMxB — GMDB、GMAB、GMIB、GMWB)を提供する生命保険会社は、バランスシート上に相当な金利ボラティリティリスクを組み込む。日本の VA 市場は米国市場より小さく 2000年代中盤のピークから縮小しているものの、残存する VA ブロック(および現代の変種)は能動的なヘッジを必要とする:

| 保証の種類 | 保険会社へのリスク | ヘッジ手段 |
|---|---|---|
| GMDB(最低死亡給付保証) | 死亡率連動の株式下方リスク;金利感応的な保証価値 | 株式プット + レシーバースワップションの組み合わせ |
| GMAB(最低積立金保証) | 株式下方リスク;保証 PV への金利感応度 | 株式プット + レシーバースワップション |
| GMIB(最低所得給付保証) | 長期の年金金利感応度;相当な rho エクスポージャー | レシーバースワップションのストリップ;長期スワップション |
| GMWB(最低引出給付保証) | 株式 / 金利 / 行動の複合リスク | レシーバースワップションを含むマルチアセットの動的ヘッジ |

一般的なパターン:NIRP 以前の 1990年代-2000年代に VA を書いた保険会社は、今や低金利環境で深く ITM な保証に直面し、相当なレシーバースワップションのヘッジポジションを必要とする。JPY 金利が正常化するにつれ、これらのヘッジは損失を生むが、原資産の保証価値が連動して低下する;動的ヘッジプログラムが値動きをネットアウトする。

日本の保険会社向けに段階的に導入されている [[insurance/economic-value-based-solvency|economic-value-based solvency (ESR)]] の枠組みは、金利ボラティリティのエクスポージャーをさらに明示的にし、特に長期レシーバースワップションのヘッジへの構造的需要を高める。より広い ALM の文脈については [[insurance/japan-life-insurance-alm-overview]] も参照。

日本の外資系生命保険会社の関連会社([[insurance/foreign-life-affiliate-japan-positioning]] でカバー)は、その親会社グループのリスク管理の枠組みが保証の明示的なデリバティブヘッジを要求するため、歴史的にスワップションヘッジで特に活発であった。

## ディーラーフランチャイズ

JPY スワップションのマーケットメイクは、主要な銀行系列の証券会社とグローバル投資銀行に集中している:

| ディーラーのカテゴリ | 代表的な会社 |
|---|---|
| 日本のメガバンク証券系列 | MUFG Securities、SMBC Nikko、Mizuho Securities(およびレーツトレーディング内のスワップデリバティブデスク) |
| 独立系日本証券会社 | Nomura、Daiwa Securities(メガバンクより小さいスワップションフランチャイズ) |
| グローバル投資銀行 | JPMorgan、Goldman Sachs、Citi、Morgan Stanley、Deutsche Bank、Barclays、BNP Paribas、HSBC、UBS(クロスボーダーフローと保険会社ヘッジプログラムのために JPY スワップションで活発) |
| ディーラー間ブローカー | Tradition、ICAP、BGC、Tullett Prebon |

JPY スワップションからのディーラー収益は、ディーラー銀行の IR におけるより広いレーツ / フィクストインカムのマーケットメイクラインの一部である;個別には開示されない。

## クリアリング状況

大半が [[securities/japan-securities-clearing-corp|JSCC]] でクリアされる標準的な JPY IRS と異なり、スワップションは歴史的に主として二者間で取引されてきた。JSCC と他の CCP は一部の通貨でスワップションのクリアリング能力を拡大した;JPY スワップションのクリアリングはバニラ IRS よりも新しく狭い範囲にとどまる。大半の保険会社 / ストラクチャードノートのヘッジスワップションフローは二者間であり、ISDA マスターと CSA のもとで文書化され、閾値を超える非クリア取引については UMR フェーズの IM 要件の対象となる。
