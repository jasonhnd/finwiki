# 日本の円金利スワップ(IRS)市場

## 要約

円金利スワップ(IRS)市場は、二者の取引相手が定められたテナーにわたって、円建ての固定金利支払いのストリームを変動金利の円支払いのストリームと交換する OTC デリバティブの場である。これは日本の銀行、生命保険会社、事業会社、および円エクスポージャーを管理する外国投資家にとって、最も主要な金利ヘッジ手段である。

変動参照金利は歴史的に 1か月、3か月、または 6か月の TIBOR であった。LIBOR 公表停止後は、カーブのうち TONA の後決め複利(OIS 市場と重なる ── [[derivatives/ois-tona-curve]] を参照)を参照する割合が増加している。テナーは 1年から 30年まで、さらにそれ以上の年限で取引される。最も流動性が高いポイントは 2年、3年、5年、7年、10年、20年、30年に集中している。

FinWiki にとって、この項目は固定対変動スワップの仕組み、TIBOR-IRS の継続とあわせた TIBOR から TONA への移行、想定元本残高、ディーラー銀行のフランチャイズ構造、JSCC 清算義務、エンドユーザー構成(事業会社対金融機関)を扱う。

## 商品の仕組み

標準的な円 IRS は、定められた想定元本にわたって交換される二つのレッグを持つ(元本の交換はない):

| 要素 | 詳細 |
|---|---|
| 固定レッグ | 定期的に支払われる固定金利(「スワップレート」)(通常、TIBOR 変動 IRS では半年ごと、TONA-OIS 型 IRS では年ごと)。 |
| 変動レッグ | 各期ごとに変動参照金利(1か月、3か月、または 6か月の TIBOR;あるいは後決め複利の TONA)にリセットされる。日数計算は円では通常 ACT/365 。 |
| 想定元本 | 交換されない。利息支払いの計算にのみ用いられる。 |
| テナー | 最も流動性が高いポイント:1年、2年、3年、5年、7年、10年、15年、20年、30年。生命保険会社のヘッジ向けに 40年までのテナーが取引される。 |
| 決済 | 各クーポン日のネット支払い(二つのレッグの差額のみが決済される)。 |
| 担保 | 担保付き取引については円キャッシュでの日次 VM を伴う標準的な CSA;非清算の相対取引については UMR の段階適用による IM。 |
| 清算 | 標準化されたテナーと参照指標は、FIEA の清算義務のもとで JSCC において清算される。 |

経済的な内容:固定金利支払者はテナーにわたって既知の調達コストを固定し、変動キャッシュフローを受け取る;変動金利支払者はその逆を行う。双方とも、金利変動に対するバランスシートエクスポージャーをヘッジできる。

## TIBOR 対 TONA への移行

円 IRS の参照金利は IBOR 移行を通じて進化してきた:

| 参照金利 | 状態 | 典型的な用途 |
|---|---|---|
| 1か月 TIBOR | アクティブ;JBATA が運営 | 一部のローン連動 IRS、仕組み商品。 |
| 3か月 TIBOR | アクティブ;JBATA が運営 | レガシーおよび新規の円 IRS の相当な割合、特に事業会社のローン連動ヘッジ。 |
| 6か月 TIBOR | アクティブ;JBATA が運営 | 長期テナーの IRS、特に保険連動。 |
| TONA(後決め複利) | リスクフリーレート(RFR);LIBOR 後に指定 | 新規の OIS、ならびに増加しつつある新規の短期〜中期テナーの IRS;割引カーブの参照金利([[derivatives/ois-tona-curve]] を参照)。 |
| 円 LIBOR(1か月、3か月、6か月) | 2021 末に公表停止(合成は 2023に段階的廃止) | レガシー契約は ISDA フォールバックプロトコルにより TONA + CAS に修正された。 |
| ユーロ円 TIBOR(Z-TIBOR) | 2024 年 12 月に廃止 | 歴史的な用途のみ。 |

この移行が重要なのは次の理由による:

1. 新規の円 IRS の組成は、清算され標準化されたスワップについて、TONA 複利を参照する割合がますます増えている。
2. TIBOR 参照の IRS は、ターム固定のローンヘッジ用途のために引き続き併存している。
3. TIBOR-TONA ベーシス(ベーシスポイント単位)は取引可能なクオートであり、TIBOR の信用・銀行調達コスト成分とリスクフリーの TONA との差を補償する。
4. ディーラーはアウトライトの IRS ポジションとあわせて、TIBOR-OIS および TIBOR-TONA のベーシスブックを運用する。

二重金利の世界は業務上複雑であるが、これは RFR ベースの価格設定へのグローバルな推進とあわせて、日本のローン市場の一部のセグメントにおけるターム固定参照金利への根強い需要を反映している。

## 想定元本残高

円金利デリバティブ(IRS + OIS 合算)は、BIS の OTC デリバティブ統計、および同調査の日銀による日本部分において半期ごとに報告される:

| 指標 | 規模(例示 ── 正確な数値は現行の BIS リリースを引用) |
|---|---|
| 想定元本残高総額、円単一通貨金利デリバティブ | BIS の円 IRS / OIS / FRA 区分の合算で、米ドル換算で数十兆ドル規模。 |
| 世界の IRS 想定元本に占める円のシェア | 4 大通貨(USD、EUR、GBP、JPY)のひとつ;円のシェアは有意であるが USD および EUR より小さい。 |
| 総市場価値 | 想定元本のわずかな割合(通常は一桁台前半のパーセント)であり、ディーラーブック全体での相殺ポジションを反映している。 |

標準的な注意点:想定元本残高は契約規模のストック指標である;総市場価値は経済的エクスポージャーにより近い代理指標である;ネッティング契約後のネットエクスポージャーはさらにずっと小さい。BIS と日銀はいずれも半期リリースでこの三つの指標すべてを公表している。現行の分析にあたっては、数値が年 2 回更新されるため、正確な調査の時点を引用すること。

新規の円 IRS の相当な割合が JSCC で清算される;清算シェアは FSA の清算義務が発効して以降、拡大している。

## ディーラー銀行の収益区分

円 IRS のディーラーフランチャイズは、日本のメガバンク系列の証券会社およびグローバル投資銀行によって支配されている:

| ディーラー区分 | 代表的な会社 |
|---|---|
| 日本のメガバンク系列証券 | MUFG モルガン・スタンレー証券、SMBC日興、みずほ証券(およびこれら親証券事業体内の円 IRS マーケットメイクデスク)。 |
| 独立系の日本証券会社 | 野村(メガバンク系以外で最大のフランチャイズ)、大和証券。 |
| 円で活動するグローバル投資銀行 | JPMorgan、Goldman Sachs、Citi、Morgan Stanley、Deutsche Bank、Barclays、BNP Paribas、HSBC、UBS。 |
| インターディーラーブローカー | ICAP / Tradition / BGC / Tullett Prebon ── 匿名の IDB 執行と参考価格の発見を提供する。 |

円 IRS のマーケットメイクからの収益は、ディーラー銀行の IR において、ホールセール / マーケット事業内の「フィクスト・インカム」または「レート」の一部として報告される。公開開示では円 IRS の損益をより広範なレート事業から個別に区分していないが、MUFG、SMFG、みずほ FG の IR 資料におけるフランチャイズの説明は、円レートがそれらのグローバルマーケット収益への有意な貢献者であることを示している。

フランチャイズの経済性は次に依存する:

- 顧客フローに対する両建てのビッド・アスク獲得。
- 在庫キャリーと、保有ポジションからの恒常的な収益。
- JGB 現物、レポ、CCBS、為替フォワード事業とのクロスプロダクトのシナジー。
- 規制上の足かせとしてのバランスシートコスト(RWA、LR、NSFR)。

親グループの開示層については [[banking/INDEX]] および JapanFG のアンカーページを参照;ディーラー活動を律する FIEA 登録については [[banking/japan-banking-license-tier-comparison-matrix]] を参照。

## 事業会社対金融機関のエンドユーザー

| エンドユーザー区分 | ユースケース | 方向性(典型的) |
|---|---|---|
| メガバンク(トレジャリー) | 円ローンのリプライシング、円債ポートフォリオのデュレーション、ALM ギャップ管理のヘッジ。 | 受け固定(資産が変動の場合);支払い固定(資産が固定の場合)。 |
| 地方銀行 | 円債ポートフォリオのデュレーションのヘッジ;IRRBB(銀行勘定の金利リスク)の管理。 | 混在;デュレーションを安価に延ばすため受け固定であることが多い。 |
| 信託銀行 / カストディ銀行 | ALM および年金関連の円レートヘッジ。 | 混在。 |
| 生命保険会社 | 長期テナーの円の責任準備金負債のヘッジ(実質的に短いデュレーションの資産対長デュレーションの負債 → 長期テナーの受け固定スワップへの需要)。 | 長期テナー(10年、20年、30年、40年)で受け固定。 |
| 損害保険会社 | より小規模な ALM ヘッジ。 | 混在。 |
| 事業会社(非金融) | 円建ての変動金利ローンエクスポージャーを固定にヘッジ;固定クーポンの円債発行を変動に転換;外貨建て債券発行で円へのスワップバックに従事。 | 変動金利ローンのヘッジ時は支払い固定;固定クーポン債を変動にスワップする時は受け固定。 |
| 外国投資家 | 日本の金利に対する見方を取る;日本対他主要通貨カーブのレラティブバリュー;JGB 現物のデュレーションのヘッジ。 | 高度に方向性があり戦術的。 |
| アセットマネージャー / 年金基金 | 円フィクスト・インカムのマンデートのデュレーション管理。 | 混在。 |

生命保険会社は、その長デュレーションの責任準備金が資産側に対してデュレーションギャップを生み出すため、構造的に長期テナーの円 IRS 受け固定需要の最大の単一区分である。このアンカー需要が 20年および 30年テナーの厚い流動性を説明する。

事業会社のエンドユーザーフローの詳細は [[finance/japan-corporate-fx-and-rate-hedge-policy]] で扱う。

## JSCC 清算義務

FSA は FIEA のもとで標準化された円 IRS について清算義務を実施し、対象となる取引相手間の適格取引を JSCC で清算することを要求した:

| 要素 | 詳細 |
|---|---|
| CCP | 日本証券クリアリング機構(JSCC)。 |
| 義務対象商品 | 主要テナーの標準化された円 IRS、清算対象の参照指標を伴う;複数のルールサイクルにわたって拡大された。 |
| 対象取引相手 | 日本の主要金融機関;時とともに拡大し、より多くの事業体を含むようになった。 |
| 証拠金 | JSCC のポートフォリオ証拠金手法に基づく IM および VM;円キャッシュで決済。 |
| デフォルト管理 | JSCC のウォーターフォール(破綻者証拠金 → 破綻者拠出 → JSCC 資本トランシェ → 非破綻会員基金 → さらなる原資)。 |

JSCC における円 IRS の清算シェアは大幅に拡大した。清算取引は次の恩恵を受ける:

- グロスエクスポージャーを削減する多者間ネッティング。
- 相対交渉の摩擦を取り除く標準化された証拠金手法。
- 相対の取引相手リスクの除去;CCP リスクは集中されストレステストされる。
- 非清算取引に対する(バーゼル枠組みのもとでの)資本軽減。

非清算の相対円 IRS は次のために継続する:非標準のテナー、非標準のリセット慣行、仕組み取引、義務の対象とならない取引相手。非清算取引は UMR(非清算証拠金規制)の段階適用による IM 要件の対象となる。

清算機構のインフラについては [[securities/japan-securities-clearing-corp]] を、より広範な市場インフラの文脈については [[securities/japan-market-infrastructure-map]] を参照。
