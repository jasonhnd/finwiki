# JCR / R&I 日本証券化格付方法論の運用プレイブック

## 要約

JCR（株式会社日本格付研究所）とR&I（格付投資情報センター）は、日本のストラクチャードファイナンスを支配する、FSA指定の国内信用格付会社2社である。ABS / RMBS / CMBS / 消費者ローン / カード / リース格付における両社の**運用方法論**は、4段階のプレイブックに従う。(1) 日本のプールデータから導いたビンテージ曲線を用い、格付カテゴリ別ストレス係数（AAA = 期待デフォルトの4–6倍、AA = 3–4倍、A = 2–3倍、BBB = 1.5–2倍）を掛ける**資産クラス別ストレスデフォルト率（SDR）計算**、(2) そのSDRを案件固有のウォーターフォール、期限前返済、回収、金利シナリオに流し込み、シニアトランシェがストレスに耐えるか検証する**キャッシュフローモデリング**、(3) 月次サービサーレポート、四半期パフォーマンスレビュー、格付アクション・トリガー（通常、SDR実績とSDRモデルのずれが閾値を超えると1-2 ノッチ格下げ）による**サーベイランス**、(4) オリジネーター信用悪化、プール集中違反、サービサー・デフォルト事由など、格付判断に織り込まれる**格下げトリガー**である。グローバル機関（S&P / Moody's / Fitch Japan）と比べ、JCR / R&Iは**日本に根ざしたデフォルトデータ**を使い（複数法域のプール化なし）、**低いベースケース・デフォルト頻度**を置き（日本の消費者信用行動の過去実績を反映）、**カントリーシーリング上限を置かない**（国内スケールで日本国債をAAA格付するため）。その結果、[[structured-finance/fitch-moody-sp-japan-criteria|global vs JCR / R&I criteria]]で記録されるように、シニア層で1–3 ノッチのスプリット格付差が生じる。この項目は、抽象的な方法論文書ではなく、アナリストのプレイブック水準で、SDR計算、キャッシュフローモデリング、サーベイランス頻度、格下げトリガーという運用メカニクスを体系化する。

## ウィキ上の位置づけ

この項目は、基礎的な[[structured-finance/credit-rating-methodology-jcr-r-and-i|JCR / R&I methodology entry]]と比較ページ[[structured-finance/fitch-moody-sp-japan-criteria|Fitch / Moody's / S&P Japan criteria]]を補完する**運用プレイブック型の方法論**ノードとして[[structured-finance/INDEX|structured-finance index]]の下に位置づけられる。資産クラス別運用ページである[[structured-finance/japan-auto-loan-abs-waterfall-mechanics|Japan auto-loan ABS waterfall mechanics]]、[[structured-finance/japan-consumer-loan-abs-structure|Japan consumer-loan ABS structure]]、[[structured-finance/japan-credit-card-receivable-abs|Japan credit-card receivable ABS]]、[[structured-finance/japan-equipment-lease-abs|Japan equipment lease ABS]]、[[structured-finance/japan-rmbs-issuance-structure|Japan RMBS issuance structure]]、[[structured-finance/japan-cmbs-issuance-structure|Japan CMBS issuance structure]]と照らして読む。規制上のアンカーは、基礎となる法人レイヤーとして[[structured-finance/spv-tk-gk-vehicle-japan-tax|TK / GK / TMK SPV vehicle]]である。

| 項目 | JCR | R&I |

| 設立 | 1985  | 1998 （前身合併） |
| 所有 | 独立、上場親会社 | 野村系 |
| FSA指定 | FIEA Article 66-27 に基づく | FIEA Article 66-27 に基づく |
| 国内ソブリン格付 | AAA | AA+ |
| ストラクチャードファイナンス人員（概算） | 60–80 名のアナリスト | 50–70 名のアナリスト |

両社はストラクチャードファイナンス商品について詳細なクライテリア文書とサーベイランスレポートを公表し、FIEAの下でFSA規制と行為規制を受ける。

## 2. 資産クラス別方法論アーキテクチャ

JCRとR&Iは、資産クラスごとに別個のクライテリア文書を維持している。

| 資産クラス | 方法論上の特徴 | 典型的なサーベイランス頻度 |
|---|---|---|
| **自動車ローンABS** | 新車 / 中古車別のビンテージ曲線、集中制限、バックアップサービサー要件 | 月次サービサーレポート、四半期格付レビュー、年次フルレビュー |
| **消費者ローンABS** | APR / ローンサイズ別のビンテージ曲線、利息制限法の規制ショックシナリオ、早期償還トリガー調整 | 月次サービサーレポート、四半期格付レビュー、半期フルレビュー |
| **クレジットカード債権ABS** | マスタートラストのプール動態、期間延長条件シナリオ、シリーズ間波及ストレス | 月次サービサーレポート、四半期格付レビュー |
| **設備リースABS** | 真正リースとファイナンスリースの区分、設備種別の残価曲線、業種 / 設備 / レッシー別集中 | 月次サービサーレポート、四半期格付レビュー |
| **RMBS** | LTV / DTI階層、地理的集中、期限前返済ビンテージ曲線、担保処分タイミング前提 | 月次サービサーレポート、四半期格付レビュー、半期フルレビュー |
| **CMBS** | 物件別NCFストレス、キャップレートストレス、リファイナンスリスク、スペシャルサービサー能力 | 月次サービサーレポート、四半期格付レビュー、年次フルレビュー |
| **NPL証券化** | NPLビンテージ / 資産種別ごとの回収率、サービサーのワークアウト実績、法的クリアランスリスク | 四半期サービサーレポート、半期格付レビュー |
| **プロジェクトファイナンス** | プロジェクトキャッシュフローストレス、カウンターパーティ信用、完工リスク、運営費ストレス | 年次格付レビュー（プロジェクト債）、マイルストーン事象時の格付アクション |

クライテリア文書は各社サイトで日本語と英語で公開され、主要な方法論改定では定期的（おおむね3–7 年ごと）に更新される。

## 3. ストレスデフォルト率（SDR）計算

中心的な運用ツールは**ストレスデフォルト率**である。これは、対象シニアトランシェ格付と整合するストレスシナリオ下でのプールの期待デフォルト頻度を指す。

### 3a. ベースケース・デフォルト頻度

| 資産クラス | 典型的なベースケース生涯デフォルト頻度 | 出所 |
|---|---|---|
| 自動車ローン（新車キャプティブ） | 1.5–3.0% | オリジネーター過去プールデータ、機関横断データベース |
| 自動車ローン（中古車 / マルチブランド） | 3.0–5.5% | 同上 |
| 消費者ローン（無担保） | 8–15% | オリジネーター + 機関データベース |
| クレジットカード債権 | 4–8% | オリジネーター + 機関データベース |
| 設備リース（混合プール） | 2.5–5.0% | 同上 |
| 住宅ローン（民間RMBS） | 1.0–2.5% | オリジネーター + 機関データベース |
| 商業用不動産ローン（CMBS） | 3.0–8.0% | 物件固有 + 機関過去データ |

### 3b. 格付カテゴリ別ストレス倍率

JCR / R&Iは、ベースケースに格付カテゴリ別のストレス倍率を適用する。

| 格付 | ストレス倍率（典型） | SDRに用いる累積デフォルト頻度 |
|---|---|---|
| AAA | 4–6倍のbase | 最も高いストレス。景気後退型シナリオ + テールイベントに耐える |
| AA | 3–4倍のbase | 景気後退シナリオ |
| A | 2–3倍のbase | 軽い景気後退 |
| BBB | 1.5–2倍のbase | 中程度のストレス |
| BB | 1.0–1.5倍のbase | ベース + 小さなストレス |

ストレス倍率は**格付定義**を反映する。AAAは「極端なストレスに耐える」ことを意味し、機関は資産クラス横断でこの整合性を実現するよう倍率を調整する。

**計算例**（自動車ローンABS）:
- ベースケース生涯デフォルト: 2.5%
- AAAストレス倍率: 5倍
- SDR（AAA）: 12.5%
- 回収率前提: 40%
- SDR ×（1  — 回収）= 12.5% × 60% = 7.5%の純損失
- AAAに必要な劣後 + リザーブ + OC: 7.5% + クッション = 約8.5%
- AAA達成には、案件の劣後構造が少なくとも8.5%を提供する必要がある

### 3c. プール固有要素の調整

SDRは次に応じて調整される。
- プール集中（単一債務者、地理、ビンテージ） — SDRを引き上げる。
- オリジネーター実績（長い履歴 + 低い過去分散） — SDRを引き下げる。
- サービサー能力とバックアップ手配 — 弱い場合はSDRを上方修正する。
- サービサーアドバンス慣行 — 直接SDRではなくキャッシュフローモデリングに影響する。
- トリガー調整 — 適切に調整されたトリガーは有利なSDR扱いを受ける。

## 4. キャッシュフローモデリング

SDRは、ストレス下のウォーターフォールをシミュレートする**キャッシュフローモデル**へ入力される。

| モデリング入力 | 内容 |
|---|---|
| プール回収スケジュール | 期間ごとの元利金期待回収 |
| デフォルト発生タイミング曲線 | デフォルトがいつ発生するか（通常は月12–36でピークを持つSカーブ） |
| 回収タイミング | 回収がいつ発生するか（通常はデフォルト後6–18 か月） |
| 期限前返済曲線 | 期間ごとの任意期限前返済 |
| 金利シナリオ | プールまたは債券の変動金利エクスポージャー |
| サービサーアドバンス行動 | サービシング契約ごとにモデル化 |
| トリガー発動 | トリガー到達時にウォーターフォール挙動が切り替わる |
| トランシェ支払い | 案件のウォーターフォールロジックに従う |

モデルは次を出力する。
- SDRシナリオ下でのシニアトランシェ全額返済確率
- シニアトランシェの支払不足シナリオ
- メザニントランシェの支払不足シナリオ
- 劣後の消費シナリオ
- リザーブの消費シナリオ

シニアトランシェがAAAを得るには、SDRとストレス上乗せの下で**元本不足がゼロ**であり、**利息不足をカバー**できることを示す必要がある。

## 5. サーベイランス — 継続監視レイヤー

格付付与後、JCR / R&Iは**継続サーベイランス**を行う。

### 5a. サービサーレポートレビュー（月次）

毎月、サービサーは受託者 + 格付会社へレポートを提出する。
- プール残高（当月 vs 前月）
- 延滞の経過日数別バケット（30  / 60  / 90  / 120+日）
- 償却（総額 + 回収）
- 期限前返済
- プール利回り（加重平均APR）
- リザーブ口座残高
- OC水準
- 累積純損失

格付会社は実績をモデル上の想定と比較する。

### 5b. 四半期格付レビュー

四半期ごとに、正式には、またストレス案件ではより頻繁に、次を行う。
- パフォーマンストレンド分析
- 同一資産クラスの他案件との比較
- オリジネーター信用レビュー
- サービサー運用レビュー
- トリガー状態確認
- 格付アクションの検討

### 5c. 年次 / 半期フルレビュー

6–12 か月ごとに次を行う。
- 更新された実績データによるキャッシュフローモデルの全面再実行
- ベースケースが変化した場合のSDR再計算
- プール構成の変化
- オリジネーター戦略の変化
- マクロ経済アップデート
- 格付アクション勧告

### 5d. イベントドリブンレビュー

特定事象で発動される。
- オリジネーター破綻 / 格下げ
- サービサー・デフォルト事由
- トリガー発動
- 資産クラスに影響する規制変更
- マクロショック（COVID、GFCなど）

## 6. 格下げトリガー — 格付が変わる時

| トリガー | 典型的な格下げ幅 |
|---|---|
| オリジネーター格付が必要閾値（例: BBB）を下回る | シニア1 ノッチ。バックアップサービサー発動により調整される場合あり |
| サービサー・デフォルト事由 → バックアップサービサー発動 | バックアップ運用能力が不確かな場合、1–2 ノッチ |
| 累積純損失 > SDR × 早期警戒閾値 | 初期は1 ノッチ。傾向が続くとさらに格下げ |
| 90+延滞 > モデルストレスの1.5倍 | 1–2 ノッチ |
| リザーブがフロアを下回り、Xか月以内に補充されない | 1 ノッチ |
| トリガー発動 → 早期償還 | 初期は1 ノッチ（仕組みが作動したと認識）。プール劣化が深刻ならさらに大きい可能性 |
| プール集中違反（レッシー、地理） | 1 ノッチ |
| マクロ経済 / 規制ショック | 持続する場合は複数ノッチ |
| 方法論改定 | 可変。対象ユニバースに大きな影響の可能性 |

格下げは自動ではない。格付会社のアナリストはサーベイランスレビューで判断を加える。**単一データ点よりパターンが重要**である。

## 7. グローバル機関（S&P / Moody's / Fitch Japan）との比較

| 観点 | JCR / R&I | S&P / Moody's / Fitch Japan |
|---|---|---|
| デフォルトデータ源 | 日本に根ざす | 複数法域のプールデータ |
| ベースケース・デフォルト頻度 | 低い（日本の過去の低デフォルトを反映） | 高い（高デフォルト市場を含むグローバルプールデータ） |
| ストレス倍率 | 日本のストレスシナリオに合わせて調整 | グローバルストレスに合わせて調整 |
| カントリーシーリング上限 | なし（国内スケールで日本をAAA / AA+格付） | あり。日本ソブリンに上限（S&P: A+、Moody's: A1、Fitch: A） |
| 回収前提 | 日本固有（住宅ではより高いことが多い） | 一部プール（地方住宅ローン）ではより保守的 |
| 方法論更新頻度 | 定期的（3–7 年） | 定期的（3–5 年） |
| 資産クラス横断の格付整合性 | 日本国内スケール向けに設計 | グローバルスケール向けに設計 |
| サーベイランス頻度 | 月次サービサー + 四半期レビュー | 類似 |
| 投資家基盤での認知 | 日本の機関投資家が中心 | グローバル機関投資家 |
| レポ適格 / 銀行資本 | 日本規制上認識 | 銀行資本、ECBレポなどでグローバルに認識 |

**構造的なスプリット格付の帰結**: 日本国内の自動車ローンABSシニアトランシェは、同一プールでも、7%の劣後でJCR / R&IからAAAを得る一方、S&P / Moody'sでは8.5–10%の劣後でAA / AA+となることが典型である。1–2 ノッチ差は機械的で、方法論の調整を反映する。これはJCR / R&Iの甘さではなく、日本基準とグローバルプール基準という異なる方法論である。

## 8. 格付プロセス — 運用タイムライン

| ステップ | 典型的な期間 | 内容 |
|---|---|---|
| 1. エンゲージメント / マンデート | 1 週間 | 発行体 / アレンジャーが格付会社に依頼し、格付会社が対応余力を確認 |
| 2. 初期ストラクチャリング協議 | 2–4 週間 | 格付会社が構造、劣後、トリガーについてフィードバック |
| 3. デューデリジェンス + プールデータレビュー | 4–6 週間 | プールデータを提出し、格付会社が分析 |
| 4. キャッシュフローモデリング | 2–4 週間 | 格付会社がモデルを回し、劣後についてアレンジャーと反復 |
| 5. 格付委員会 | 1–2 週間 | 内部委員会が格付を承認 |
| 6. 格付リリース | 価格決定の1–2 週間前 | プレセール / 価格決定前の格付リリース |
| 7. 発行後サーベイランス | 継続 | 月次 + 四半期 + 年次レビュー |

マンデートから格付までの総期間は通常8–14 週間である。グローバル機関と近いタイムラインだが、エンゲージメントモデルは異なる。JCR / R&Iでは発行体との関与がより細かく、グローバル機関の東京オフィスは人員が少ない。

## 9. 反論

- **「JCR / R&Iは甘すぎる」** — 擁護側は低いベースケース・デフォルトを正当化する日本固有データを挙げる。批判側は、日本のデフォルト率が上方収束した後で初めて方法論がグローバルに近づき、テールリスクを生むと論じる。
- **「ソブリン上限の機械的例外はJCR / R&Iに不公平な批判である」** — JCR / R&Iはカントリーシーリングに縛られない。問題は、それが方法論上妥当か、構造的優位かである。
- **「SDR倍率は恣意的である」** — 批判側は4–6倍のAAA倍率が経験的に導出されたものではなく調整値だと指摘する。擁護側は格付カテゴリの整合性には調整が必要だと述べる。
- **「サーベイランスは後追いである」** — 月次サービサーレポートレビューは反応的だが遅れる。オリジネーターのストレスなど一部事象は月次レビューより速く動く。
- **「方法論改定は崖効果を生む」** — SDR倍率や回収前提の改定時には、複数案件が同時に動きうる。これは避けにくいが混乱を生む。
- **「JCR / R&Iの分岐は実質的であり、どちらかを選ぶ必要がある」** — 発行体がJCRのみ、またはR&Iのみを起用することもある。共同格付は一般的だがコストが増える。選択は方法論選好より過去の関係に左右されることが多い。

## 10. 未解決の論点

- 日本の資本市場が国際化するにつれ、JCR / R&Iが方法論をグローバル機関へ収束させるかどうか。
- 気候リスクと人口減少シナリオがSDR計算に体系的に組み込まれるかどうか。
- FSAが格付会社へ、より厳格な方法論開示または独立第三者レビューを求めるかどうか。
- BNPL、EVバッテリー関連ABS、tokenized RWAなど新資産クラスで、伝統的資産クラスと同程度の速度で方法論が整備されるかどうか。
- [[JapanFG/sumitomo-mitsui-trust|trust banks]]が受託者として、フィデューシャリーリスクの増大に伴い、より細かなサーベイランス頻度など方法論変更を求めるかどうか。
- BOJの金利正常化がプールにストレスをかけ、モデリング差を露呈させた場合の、JCR / R&Iとグローバル機関の競争上の位置。
