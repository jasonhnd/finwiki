# 日本の CMBS 発行構造

## 要約

日本の CMBS は、単一借入人（単一資産または小規模プール、スポンサー一者）構造と、コンデュイット（相関の低い複数借入人、大きなプール）構造に分かれる。市場は 2006-2007,  にピークを迎え、2008-2010  の世界金融危機で満期ロールオーバー時のデフォルトが多発して壊滅的な打撃を受け、2010 年代を通じてほぼ休眠状態にあった。2020 年代には物流倉庫とオフィスの単一借入人案件を背景に再興しつつあるが、発行額は 2008  前のピークの一部にとどまる。このページは CMBS 構造メカニクスを理解するために使い、裏付け不動産キャッシュフローは [[real-estate-finance/INDEX]]、市場全体の文脈は [[structured-finance/japan-abs-market-overview]] と組み合わせる。

## 1. 構造タイプ

| タイプ | 説明 |
|---|---|
| 単一借入人 | スポンサー一者、物件一つまたは小規模プール、モーゲージローン一式。2020 年代の再興期に一般的。 |
| コンデュイット | 相関の低い複数借入人、大きなプール、分散効果。2008,  前は一般的だったが、現在はまれ。 |
| 単一資産 / 単一借入人（SASB） | 最も集中した形。一つの大型物件、一つのローンをトランシェ化する。 |
| 複数スポンサー・コンデュイット | 複数オリジネーターが共通プールにローンを拠出する。日本では 2008 後ほぼ消滅した。 |

現在の日本の CMBS 発行は、圧倒的に単一借入人、単一資産、または小規模プールである。コンデュイットモデルは意味のある形では戻っていない。

## 2. トランチング

| トランシェ | 典型的な買い手 |
|---|---|
| シニア（AAA / AA 目標） | 生命保険会社、資産運用会社、メガバンク ALM 勘定 |
| メザニン（A / BBB） | 専門投資家、ヘッジファンド |
| B ピース（BB / B / NR） | 専門 B ピース買い手（後述） |
| エクイティ / ファーストロス | スポンサー保有、時に専門投資家 |

トランチングは、劣後構造と利息キャッシュフローのウォーターフォールによって実現される。

## 3. B ピース買い手市場

- B ピース（支配権を持つ劣後クラス）の買い手は、2008  前の米国型 CMBS では重要な役割を果たしたが、日本では深く発達しなかった。
- 日本では、スポンサーまたは専門の不動産投資家が、専用の B ピース買い手市場へ売却するのではなく、B ピースを直接保有することが多い。
- これは、日本の CMBS にコンデュイット案件が少ない構造的理由の一つである。活発な B ピース買い手基盤がなければ、コンデュイットの経済性は難しい。

## 4. 回収シナリオ

| シナリオ | 起きること |
|---|---|
| 正常 | キャッシュフローがトランシェを支払い、シニアが先に支払われ、エクイティが残余を受け取る。 |
| パフォーマンス悪化 | トリガーにより余剰キャッシュがシニアへ振り向けられ、エクイティ / メザニンのキャッシュフローが止まることがある。 |
| 満期デフォルト | 予定満期にローンがリファイナンスできず、特別サービサーが引き継ぐ。 |
| 担保実行 / 強制売却 | 物件が売却され、回収金がトランシェの優先順位に従って分配される。 |
| 条件変更 / 延長 | ローン期間が延長され、トランシェのキャッシュフローがずれる。 |

日本での回収は、比較的透明な不動産鑑定制度と発達した不動産仲介市場の恩恵を受けるが、商業テナント保護や契約上の制約により処分は遅くなることがある。

## 5. 2008  後の発行減少

| 期間 | パターン |
|---|---|
| 2003-2006 | 急成長。コンデュイット案件が一般的で、構造化は積極的。 |
| 2006-2007 | 発行ピーク。クロスボーダー投資家の参加が大きい。 |
| 2008-2010 | 世界金融危機。リファイナンス凍結。多くの 5 年案件が満期ロールオーバーでデフォルト。 |
| 2008-2012 | 格下げ、損失実現、債券保有者によるワークアウト。 |
| 2012-2018 | 公募案件の市場は実質的に休眠。 |
| 2018-2020 | 選別的な単一借入人案件が復活。 |
| 2020-現在 | 物流倉庫ブームが単一借入人 CMBS を牽引。オフィスとホテル案件は選別的。 |

2008-2010  のデフォルト波は、日本の CMBS に深い痕跡を残した。投資家基盤、構造化の保守性、格付け機関の手法は、いずれもその経験を反映している。

## 6. 2020 年代の市場再興

| 要因 | 効果 |
|---|---|
| 物流倉庫ブーム | EC 需要 → 大型、単一テナント、長期賃貸倉庫 → 単一借入人 CMBS に適した資産。 |
| 外国人投資家の再参入 | クロスボーダー資本が日本不動産に再参入。直接 REIT 投資の代替として CMBS を利用。 |
| マイナス金利 / 低利回り環境（2024 前） | 利回りを求める投資家が、ストラクチャード不動産リスクを取る意欲を持った。 |
| 物流 J-REIT の代替 | 一部ポートフォリオでは、スポンサーが J-REIT IPO より CMBS を選好。 |

2020 年代の再興は実際にあるが、2008  前のピークと比べると控えめである。コンデュイット案件は不在のままで、単一借入人または単一資産案件が中心である。

## 7. ビークル選択

日本の CMBS 案件は通常、資産流動化法に基づく TMK（特定目的会社）または TK-GK SPV 構造を用いる。ビークル選択は [[structured-finance/spv-tk-gk-vehicle-japan-tax]] を参照。公募上場債発行では TMK が好まれ、私募では TK-GK がより一般的である。

## 8. 裏付け物件リスク

CMBS のリスクは、主に裏付け物件のキャッシュフローに依存する。オフィス、商業施設、ホテル、物流、住宅投資物件は、それぞれ異なるリスクプロファイルを持つ。裏付け不動産金融の経路は [[real-estate-finance/INDEX]] を参照。

J-REIT 債務との重なり: J-REIT（スポンサー系列は [[JapanFG/INDEX]]）はポートフォリオ物件を担保に借入を行うことが多いが、それらのローンは CMBS として証券化されるわけではない。直接銀行借入または J-REIT 発行の無担保債である。
