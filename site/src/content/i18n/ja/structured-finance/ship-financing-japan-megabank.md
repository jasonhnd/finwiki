# 日本の船舶ファイナンス — メガバンクのシップファイナンスデスクと商社リース子会社

## 要約

日本の船舶ファイナンスはいくつかの層で構成されます: (1) [[JapanFG/mufg|MUFG]]、[[JapanFG/smfg|SMBC]]、[[JapanFG/mizuho-fg|Mizuho FG]] の**メガバンク船舶ファイナンスデスク**は、歴史的に世界トップ級の船舶融資機関として、世界および日本の船主にシニア担保ローンを提供してきました。(2) **日本のリース会社**（[[JapanFG/mitsubishi-hc-capital|Mitsubishi HC Capital]]、[[JapanFG/smfl|SMFL]]、[[JapanFG/mizuho-lease|Mizuho Leasing]]、[[JapanFG/tokyo-century|Tokyo Century]]）は [[structured-finance/spv-tk-gk-vehicle-japan-tax|TK / GK SPV]] 構造を用いた JOL 型船舶リースを提供します。(3) **日本の Big-3  海運会社**（[[JapanFG/nyk-line|NYK Line]]、[[JapanFG/mitsui-osk-lines|MOL]]、[[JapanFG/k-line|K-Line]]）は、船主であると同時にオフテイク相手方でもあります。(4) 2008 以前に日本型クロスボーダー税務ストラクチャーと競合したドイツとノルウェーの **KG（Kommanditgesellschaft）/ KS（Kommanditselskab）** 構造も歴史的参照点です。市場は 2008  金融危機と 2010 年代の海運不況後に大きくデレバレッジし、HSH Nordbank、Commerzbank などの欧州船舶銀行が撤退した領域で日本のメガバンクがシェアを伸ばしました。NYK / MOL / K-Line がコンテナ部門を **Ocean Network Express（ONE）** に統合した 2018 の株主再編も、オフテイク環境を作り替えました。

## 1。 船舶金融市場 — 簡単なおさらい

世界の商業海運 (~100,000  商船、~200 億 DWT) は多額の借金で賄われています。
- タンカー、ばら積み貨物船、コンテナ船、LNG 船 - それぞれに異なるクレジット/サイクル ダイナミクスがあります
- 資産寿命は20–30 年ですが、融資期間は10–15 年以上です
- 残存価値の変動性が高い — 船舶の価値は用船料金、スクラップ価値、貿易サイクルに応じて変動します
- 循環型産業 - 5～10 年の山と谷が典型的

日本は 3 つの点で構造的に重要です。**主要な船舶所有管轄区域** (日本郵船、商船三井、K ライン、および数十の中規模船主)、**主要な船舶融資管轄区域** (メガバンク融資 + リース会社 JOL の船舶構造)、**主要な造船管轄区域** (歴史的および現在の日本の造船所) としてです。

## 2。 メガバンクのシップファイナンスデスク

メガバンクの船舶融資デスクは、何十年にもわたって世界トップクラスの金融業者であり続けています。

| 銀行 | ポジション | オフィス所在地 |
|---|---|---|
| [[JapanFG/mufg|MUFG Bank]] | ローンブックによる世界最大の船舶貸付会社の 1 つ | 東京、ロンドン、シンガポール、ニューヨーク |
| [[JapanFG/smfg|SMBC]] | トップクラスの世界的船舶貸付業者 | 東京、ロンドン、シンガポール、ニューヨーク |
| [[JapanFG/mizuho-fg|Mizuho Bank]] | トップクラスの世界的船舶貸付業者、MUFG / SMBC よりも若干規模が小さい | 東京、ロンドン、シンガポール |

彼らの融資帳簿は次のように構成されています。
- **船舶タイプ**: タンク / バルカー / コンテナ / LNG / 特殊船
- **オーナークレジット**: 国内大手企業 (NYK、MOL、K-Line) vs ギリシャの個人オーナー vs ノルウェーの一流オーナー vs アジアの複合企業
- **ローン・トゥ・バリュー (LTV)**: 通常、開始時に55–75%、ブローカーの評価に関連付けられた定期的なLTV維持約款付き
- **期間**: 7–12 年 (長期用船契約を結んだ LNG 船の場合は 15  となる場合もあります)
- **チャーター/雇用**: 定期チャーター (固定金利、固定期間) からスポット市場雇用 (契約収入なし) までの範囲 — 信用指標はどれに依存するかによって異なります。

## 3。 2008 後のレバレッジ解消と欧州銀行撤退

2008 危機とその後の2010の海運低迷（特にドライバルク運賃とコンテナ運賃）は、世界の船舶金融業界全体に巨額の損失をもたらしました。
- **HSH Nordbank** — かつて世界最大の船舶貸し手であった — は EU 国家援助規則に基づいて縮小/再編されました
- **コメルツ銀行**と**ブレマー州銀行**が実質的に撤退
- **RBS** は出荷を終了しました
- 欧州の小規模船舶銀行数社が統合、ポートフォリオの売却、または撤退を行った

この撤退分野においては、日本のメガバンク（シンガポールに本拠を置くDBS、OCBC、および少数のアジアの銀行とともに）が相対的に大きなプレーヤーとなった。 これは意図的な戦略的増強であり、単なるご都合主義ではなく、一部の欧州銀行が失った資本と資金コストのポジションによって可能になった。

## 4。 日系リース会社船舶部門

| 賃貸人 | 船舶部門の重点 |
|---|---|
| [[JapanFG/mitsubishi-hc-capital|Mitsubishi HC Capital]] | 大手船舶リースプラットフォーム。 JOL スタイルの構造 + 貸借対照表所有権 |
| [[JapanFG/smfl|SMFL]] | より広範な輸送資産部門の一環としての船舶リース |
| [[JapanFG/mizuho-lease|Mizuho Leasing]] | 船舶ファイナンスJOL手配 |
| [[JapanFG/tokyo-century|Tokyo Century]] | 厳選された船舶リースへの参加 |

日本のリース会社との船舶取引では、航空機の JOL と同様の [[structured-finance/spv-tk-gk-vehicle-japan-tax|TK / GK SPV]] 構造が使用されます。
- 日本の投資家は減価償却シールドを利用する
- SPV は通常、海運税が優遇されている管轄区域に居住します (シンガポール、香港、場合によっては船籍登録のためマーシャル諸島またはリベリア、別の所有権の SPV が別の場所にある)。
- 賃借人には国内および海外の運送会社が含まれます

## 5。 KG / KS 構造 — 歴史的なヨーロッパの類似物

2008, 以前、ヨーロッパ市場には独自の小売税構造の船舶融資がありました。
- **KG (Kommanditgesellschaft)** — JOL と同様の税減価償却保護を備えた、船舶所有のための個人投資家の株式をプールするドイツのリミテッド パートナーシップ構造
- **KS (Kommanditselskab)** — ノルウェー/スカンジナビア語のアナログ
- これらの構造は、ドイツの海運会社と広範なヨーロッパの艦隊の新造注文のかなりの部分に資金を提供しました。

2008  以降、KG 市場は崩壊しました。船舶価値の下落により KG の株式は消滅し、ドイツの税規則は強化され、個人投資家は多額の損失を被りました。 日本の JOL 船舶構造は継続しましたが（小売よりも機関投資家や法人投資家が多く、より回復力を提供しています）、世界の税制構造の船舶株式市場は縮小しました。

## 6。 大手-3 日本の海運会社 — 日本郵船、商船三井、K-Line

| 会社概要 | 在庫 | 所属グループ | メモ |
|---|---|---|---|
| [[JapanFG/nyk-line|NYK Line]] | 9101 | 三菱グループ | ある意味日本最大手の海運会社 |
| [[JapanFG/mitsui-osk-lines|Mitsui OSK Lines (MOL)]] | 9104 | 三井グループ | タンカー / LNG / 乾かさ強度 |
| [[JapanFG/k-line|Kawasaki Kisen Kaisha (K-Line)]] | 9107 | みずほ系（DKB合併後の系譜） | 自動車運搬船 / ドライバルク / コンテナ |

これらの企業は、船主（所有する船団を運航し、リースする用船船）であり、オフテイク・カウンターパーティ（長期契約に基づく用船会社）でもあります。

### 2018 コンテナとアームを ONE に統合

2018,  年 4 月、NYK、MOL、K-Line はコンテナ輸送部門を **Ocean Network Express（ONE）** に統合しました。ONE はシンガポールに本社を置く合弁会社です。これは慢性的なコンテナ輸送能力過剰と中国系キャリアの参入に対する防衛的再編でした。その後 ONE は世界トップ 10  級のコンテナキャリアとなり、2020  後の運賃ブームの恩恵を受けました。

この合併により、日本の船舶融資の取り分が再構築され、コンテナ船融資は 3 つの別々の親会社ではなく 1 つの親会社に向けられるようになりました。

## 7。 LNG 船 — 特殊なケース

LNG 船は他の船とは異なる方法で資金調達されます。
- 船舶コストが高い（最新の LNG 運搬船の場合、200m 以上）
- チャーターは通常長期 (15–25  年) であり、特定の LNG 輸出ターミナル契約に関連付けられています
- 貸し手はアセットファイナンス貸し手というよりはプロジェクトファイナンス貸し手のように見える
- 日本の関与は重要 — 日本郵船と商船三井は大規模な LNG 船団を運営しており、[[JapanFG/jbic|JBIC]] は日本の LNG 取量に関連した特定の LNG 船の資金調達を支援しており、[[policy-finance/japan-project-finance-stack-diagram|policy-finance stack]] が適用されている

## 8。 国境を越えた船舶所有に対する税制構造

典型的な国境を越えた船舶融資の構造:

```
国内投資家（TK匿名組合）
        |
v
+---------------------------------+
| 日本 TK / GK SPV |
+----------------+----------------+
| 資本
v
+---------------------------------+
| オフショア所有権 SPV |
| (シンガポール / 香港 / |
| マーシャル諸島) |
| - 船舶の所有権 |
| - 銀行シンジケートからの借入 |
| - 借主へのチャーター船 |
+----+----------------------+-----+
     |                      |
チャーターローン
     |                      |
v v
借手銀行シンジケート
(海運会社) (メガバンクリード)
```

シンガポール/香港が好まれる理由:
- 有利な船舶トン税またはトン税適格制度
- チャーター収入に対する源泉徴収税の効率化
- 活発な海洋クラスターのエコシステム (ブローカー、弁護士、保険会社)

リベリア諸島とマーシャル諸島の国旗は一般的ですが（便宜置籍）、所有権主体は通常、国旗とは別の税効果の高い管轄区域にあります。

## 9。 対位法

- **「船舶ファイナンスは景気循環リスクが集中している」** — 日本のメガバンクの船舶ローン帳簿は不景気の際に大幅な評価損を被った。 海運業への集中は長期にわたるバランスシートのリスクとなる
- **「JOL 船舶投資家は、個人の KG 投資家が対処できなかった残存価値リスクを負っている」** — 日本の JOL 構造は企業/機関投資家に有利であり、それが回復力を提供しますが、根本的な残存リスクは同様です
- **「コンテナ輸送のスーパーサイクルは繰り返されない」** — 2020–2022 コンテナ貨物ブームは、ONEやその他の日本に関係のある船主に多大な恩恵をもたらしたが、同規模で再発する可能性は低い。
- **「LNG 船の発注は政策主導である」** — 新規 LNG 船の発注における重要な割合は、純粋な商業サイクルの推進力ではなく、日本の戦略的な LNG 輸入コミットメントを反映しています。 これは資産クラスの経済学を歪める可能性があります
- **「アジア銀行の船舶金融の成長は逆転する可能性がある」** — 中国および他のアジアの銀行がさらに規模を拡大すれば、日本のメガバンクの船舶金融市場シェアは新たな圧力に直面する可能性がある

## 10。 未解決の質問

- 海運における脱炭素化のペース（IMO排出規則、二元燃料新造船、アンモニア/メタノール船）と日本の船舶金融がどのように適応するか
- ONE が安定した合弁会社であり続けるか、それとも最終的にはさらに統合されるか (日本の親会社 3 社のうちの 1 社が支配的な地位を占めるようになりつつある)
- [[JapanFG/japan-post-bank|Japan Post Bank]] または生命保険会社がファンド構造を通じて船舶金融に拡大するかどうか
- 船舶金融と[[structured-finance/synthetic-securitization-japan-bank-rwa-relief|synthetic securitisation]]との相互作用（大型船舶ローン帳簿に対するRWAの救済）
- [[JapanFG/boj|BOJ]]正常化下の金利上昇環境において、税金を控除した利回りを求める企業投資家の需要により、JOLの船舶構造が牽引力を獲得するかどうか
- 日本の船舶リース市場の成長を捉える上での[[JapanFG/mitsubishi-hc-capital|Mitsubishi HC Capital]]と[[JapanFG/smfl|SMFL]]の役割
