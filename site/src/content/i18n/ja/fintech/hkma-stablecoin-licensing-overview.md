# HKMA ステーブルコイン・ライセンス · 香港 Stablecoin Licensing Regime 概観

## ウィキ上の位置づけ

この項目は [[fintech/INDEX|fintech index]] の配下に位置する。隣接する文脈は [[fintech/japan-financial-regulation|日本金融規制 — トークン・暗号資産・決済に関する法体系]]、より広いシステム境界は [[fintech/japan-stablecoin-regulatory-landscape|日本 Stablecoin 法制度の三層構造（JPYC・USDC・Project Pax）]] とあわせて読む。

> [!info] 要約
> 香港 HKMA(Hong Kong Monetary Authority)ステーブルコイン立法は 2025-08 採択、アジア初の完成された SC ライセンス・フレームワーク。**初回審査結果は 2026-05-21 前後に公表予定**(メディアの推定、HKMA は日付を公式確認していない);9 社申請者のうち市場予測では実際 2 件のみ(HSBC + StanChart Anchorpoint)、その他は遅延。HK ライセンス = 三円 MRA 第三圏候補 + 中国本土で唯一取得可能なルート。

## 主要事実

- HKMA SC 立法:2025-08 採択 ^[extracted]
- 香港 HKD-stable + USD-stable の二軌ライセンス ^[extracted]
- 京東 JD-HKD = 中国本土系大手による香港での初の SC 発行申請 ^[extracted]
- 準備金 100%(HKD / USD 短期国債 + 中銀預金)+ 第三者カストディ + 月次監査 ^[extracted]
- HKD 100M+ 資本要件 ^[extracted]

## 仕組み

**HKMA SC ライセンスのコア要件**:100% 準備金(HKD / USD 短期国債 + 中銀預金)+ 準備金第三者カストディ + 月次監査 + 発行体が HK にエンティティ + HKD 100M+ 資本要件 + KYC + AML/CFT + Travel Rule + OFAC SDN との協調(間接的に USD 準備金経由)+ PBoC / e-CNY との境界画定(中国本土リテール顧客非受入)。香港 VASP / OTC 側のもう一つの規制軸は [[exchanges/hk-sfc-vasp-licensing-overview|HK SFC VASP ライセンス概観]] を参照。

**9 社申請者(2026-05-21 決定)**:
| 申請者 | 背景 | ライセンス決定 |
|---|---|---|
| HSBC | グローバル大手銀行 | ✓ 取得 |
| Standard Chartered Anchorpoint(StanChart + Animoca + HKT) | 連合体 | ✓ 取得 |
| 圓幣科技 | HK 本拠 | 保留 |
| 京東 JD-HKD | 中国本土系 | 保留(政治的に敏感) |
| Ant International | 中国本土系 | 保留(政治的に敏感) |
| 中銀(BOCHK) | 中国系 | 保留(政治的に敏感) |
| ZA Bank | デジタル銀行 | 保留 |
| Tether | グレー市場ドル・リーダー | 保留(重要) |
| Conflux | 中国系公開チェーン | 保留 |

**重要な発見**:HKMA 初回実際 2 件のみ発行、予想された 9 件ではない。他の 7 社は遅延または拒否。

## 起源と発展

HKMA は 2022-12 に SC 市中協議 を開始、2023-12 第二回 CP で サンドボックスリストを確立。2025-08 Stablecoin Ordinance 採択(立法に約 3 年)。2026-05-21 初回決定が重要なシグナルを発する:HKMA は「低政治リスク + 高コンプライアンス背景」を持つグローバル大手銀行(HSBC / StanChart)を優先発行、中国本土系大手 + グレー市場プレイヤーは遅延。準備金 FRTB 資本計量フレームワークは [[fintech/hk-frtb-stablecoin-reserve-overview|HK FRTB 準備金概観]]、アジア全景比較は [[exchanges/global-vasp-regulatory-comparison-matrix|グローバル VASP 規制比較マトリクス]] を参照。
