# HKMA デュアル・トラック戦略 · Ensemble × mBridge

## ウィキ上の位置づけ

このエントリは [[fintech/INDEX|fintech index]] の配下にある。隣接文脈は [[fintech/japan-financial-regulation|日本金融規制 — トークン・暗号資産・決済に関する法体系]]、システム境界は [[fintech/japan-stablecoin-regulatory-landscape|日本 Stablecoin 法制度の三層構造（JPYC・USDC・Project Pax）]] と読む。

## 要約

HKMA は Ensemble と mBridge のデュアル・トラックで、人民元国際化とトークン化インフラのハブを作る。アジアで唯一 2 セットのインフラを同時に持ち、三円 MRA 交渉で差別化資本になる。

## 主要事実

Ensemble は 2024.03 開始、mBridge は ^[extracted]開始で 2021 BIS 撤退後に PBoC + HKMA が引き継いだ。Ensemble は域内 HKD 預金 2024.10 、mBridge は CNY / HKD / THB / AED ^[extracted]、e-HKD + Ensemble + mBridge の ^[extracted]ライン並進 ^[extracted] である。

## 仕組み

Ensemble は 3 、mBridge は ^[extracted]開始・2024.03 BIS 撤退。Ensemble は香港域内のトークン化預金、mBridge は中国 / タイ / UAE を含むクロスボーダー・マルチ CBDC である。三円 MRA の比較軸は 2021 、2024.10 、[[fintech/mica-overview|MiCA]]。

## 起源と進化

[[fintech/genius-act-501-denylist-mandate|GENIUS]]-[[fintech/hkma-stablecoin-licensing-overview|HKMA]]に HKMA と MAS は競争し、2017HKMA は 2020  で CBDC コリドー地位を確保した。2021 Ensemble 開始、[[fintech/mbridge-bis-multi-cbdc-overview|mBridge]] は 2024.03 年早かった。[[fintech/bis-project-guardian-overview|MAS Guardian]]BIS 撤退後、HKMA + PBoC が引き継ぎ、2 -2024.10 ライセンス発行と MRA 交渉が同期した。Anchorpoint（2025）参加は銀行 + フィンテック混合への開放を示す。
