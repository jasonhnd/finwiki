# 日本における外国銀行代理業務ルート

## 概要

外国銀行代理業務は、外国銀行支店免許とは異なる。銀行または日本支店が、委託元の外国銀行の代理または媒介を行うルートである。クロスボーダー法人銀行業務、グローバル・キャッシュマネジメント、オフショア支店サービス、外国金融グループの顧客カバレッジで重要になる。

このページは [[financial-licenses/foreign-bank-branch-license-route|foreign-bank branch license route]]、[[foreign-financial-institutions/foreign-bank-branches-japan-index|foreign-bank branch registry index]]、[[banking/foreign-bank-japan-retreat|foreign-bank Japan retreat]]、[[financial-licenses/foreign-financial-group-adjacent-licenses|foreign financial group adjacent licenses]]、[[financial-licenses/INDEX|legal / financial licenses]] と合わせて使う。

## ルート比較

| ルート | 主体 | 確認できること | 確認できないこと |
|---|---|---|---|
| 外国銀行支店 | 外国銀行の免許を受けた日本支店 | 日本支店としての直接の銀行免許 | 証券、信託、決済、またはすべての個人向けサービス |
| 外国銀行代理銀行 | 銀行 / 支店が委託元外国銀行のために行為する | 代理・媒介関係が FSA 一覧で認識されている | 代理側がバランスシート上の貸し手または預金受入主体であること |
| 日本法人銀行子会社 | 外国グループが支配する日本法人銀行 | 国内法人としての日本の銀行免許 | 外国銀行支店としての地位 |
| 駐在員事務所 | 連絡・情報収集の拠点 | 拠点の存在のみ | 銀行、預金、貸出、代理、証券業務 |

## 利用場面

| 利用場面 | 代理業務ルートが重要になり得る理由 |
|---|---|
| グローバル・キャッシュマネジメント | 日本の顧客が、国内銀行インターフェースを通じて外国支店 / オフショア商品にアクセスする必要がある場合がある。 |
| 貿易金融 | 信用状、保証、クロスボーダー決済に外国の委託元銀行が関わる場合がある。 |
| 企業財務 | 多国籍企業の顧客は、日本と海外の銀行エンティティをまたぐカバレッジを必要とする。 |
| プライベート / ウェルスバンキング | クロスボーダー紹介と口座アクセスでは、免許境界の慎重な確認が必要になる。 |
| グループ支店連携 | 免許を受けた日本支店が、承認 / 届出ルートの下で本店または海外支店のために行為する場合がある。 |

## 確認すべき論点

| 問い | 公開情報上の意味 |
|---|---|
| 代理銀行と委託元外国銀行は誰か。 | 責任関係と顧客向け開示ルートは、この組み合わせに依存する。 |
| 承認ルートか届出ルートか。 | FSA 一覧は許可 / 申請日を区別している。 |
| どのサービスを媒介しているか。 | 預金、貸出、FX、貿易金融、証券、決済類似サービスは別制度を発動し得る。 |
| 顧客は日本支店、オフショア支店、日本子会社のどれと取引しているか。 | 法主体、預金保護、開示が異なる。 |
| 証券エンティティも存在するか。 | FIBO / JSDA の状態を別途確認する。 |

## JapanFG における位置づけ

- [[megabanks/mufg-bank|MUFG Bank]]、[[megabanks/sumitomo-mitsui-banking-corp|SMBC]]、[[megabanks/mizuho-bank|Mizuho Bank]] は、メガバンクが日本の顧客をグローバル支店網とコルレス網につなぐため重要である。
- [[foreign-financial-institutions/hsbc-japan|HSBC Japan]]、[[foreign-financial-institutions/standard-chartered-japan|Standard Chartered Japan]]、ANZ Bank Japan、[[foreign-financial-institutions/ing-bank-japan|ING Bank Japan]]、[[foreign-financial-institutions/deutsche-japan|Deutsche Japan]] は、サービス範囲がクロスボーダーの場合、支店 / 代理 / 証券の切り分け確認が必要である。
- [[payments/japan-bank-api-payment-agency-route|Electronic payment agency]] は「代理」という語が似ていても別制度である。

## 調査チェックリスト

1. FSA の外国銀行代理銀行一覧で、代理側 / 委託元の正確な組み合わせを確認する。
2. 当該主体が外国銀行支店、FIBO、信託、決済関連の登録にも掲載されているか確認する。
3. 銀行の日本公式サイトで顧客向け説明を確認する。
4. 代理銀行一覧だけから、預金保護、証券業務の許可、日本支店としての地位を推定しない。
5. 論点が市場での位置づけである場合、クロスボーダー銀行業務の問いを [[banking/foreign-bank-japan-retreat|foreign-bank Japan strategy]] に接続する。
