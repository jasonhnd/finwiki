# キャストグローバル信託

ウィキ上の位置づけ: [[trust-banks/INDEX]] -> [[trust-banks/trust-companies-japan-index]] -> [[trust-banks/castglobal-trust]]

## 概要

CastGlobal Trust は、不動産、エスクロー、クロスボーダー投資支援と結びついた管理型信託会社である。FinWiki では、[[trust-banks/yamada-escrow-trust]] と [[trust-banks/kyodo-trust]] の横に置く専門サービス系の信託経路として有用である。

## エンティティ境界

このページはキャストグローバル信託株式会社を扱い、CastGlobal の専門サービスネットワーク全体は扱わない。法務、税務、国際パートナーサービスに関する記述は、公開された会社資料に結びつけておく。

## 事業上の役割マップ

| 経路 | 読み方 |
|---|---|
| エスクロー | 管理信託口座を通じた支払・決済支援。 |
| 不動産投資 | 日本の不動産取引構造に対する信託支援。 |
| クロスボーダー文脈 | 信託業務を国際投資家支援と接続する。 |
| 登録簿管理 | [[trust-banks/trust-companies-japan-index]] の行。 |

## 公開情報上の位置づけ

CastGlobal Trust は、信託会社が専門サービス、エスクロー、不動産投資の間に位置し得ることを示すため重要である。この経路は、エスクロー類似の信託サービスを [[payments/account-to-account-payment-japan|payment rails]] や PSP 決済リスクページから切り分ける助けにもなる。
