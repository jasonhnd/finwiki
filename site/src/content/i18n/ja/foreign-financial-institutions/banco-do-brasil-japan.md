# ブラジル銀行日本 (Banco do Brasil Japan)

## ウィキ上の位置づけ

Banco do Brasil Japan は [[JapanFG/INDEX|JapanFG index]] 配下に位置づけ、[[foreign-financial-institutions/foreign-bank-branches-japan-index|the foreign-bank branch registry]] におけるブラジル・日本間の外国銀行支店ルートである。[[payments/funds-transfer-service-providers-japan-index|funds-transfer registry]]、[[financial-licenses/foreign-bank-branch-license-route|foreign-bank branch route]]、[[financial-licenses/foreign-bank-agency-business-route|foreign-bank agency route]]、[[finance/cross-border-m-a-japan|cross-border finance]] とあわせて読む。

BB Japão、日本・ブラジル間送金、外貨預金、支店銀行業務、法人国際業務を扱う場合にこのページを使う。

## 概要

Banco do Brasil の日本サイトは、単なる登録上の一行ではなく、顧客向けの完全な公開窓口である。口座開設、送金、投資 / 定期預金、為替、支店 / モバイルサービスポイント、法人国際業務を扱う。

ブラジル・日本回廊はディアスポラ金融と貿易金融の関連性が強いため、独立ページとして扱う価値がある。純粋なホールセール投資銀行支店というより、[[foreign-financial-institutions/philippine-national-bank-japan|PNB Japan]] や [[foreign-financial-institutions/metrobank-japan|Metrobank Japan]] に近い。

## 事業上の役割マップ

| 機能 | 日本での関連性 |
|---|---|
| 送金 | BB Japão は送金とポルトガル語 / 多言語サービスを重視する。 |
| 国内円受取ルート | 公開送金ページは、BB Japan 口座への指定国内銀行振込の取扱いを説明している。 |
| 外貨送金 | 公開ページは SWIFT による USD / EUR の国内外貨送金を説明している。 |
| 法人国際業務 | BB Japan は企業向けに国際業務 / 貿易金融関連ソリューションを提供する。 |
| FX / 預金 | 顧客向けの為替および預金 / 定期預金資料は日本サイトの一部である。 |

## 公開情報上の位置づけ

Banco do Brasil Japan は、ブラジル人居住者、ブラジル関連企業、日本・ブラジル貿易の金融ブリッジである。重要性は親銀行のブラジル国内での規模だけではない。日本支店の関連性は、主要な日本のメガバンクが必ずしも同じ顧客言語や送金ワークフローで提供しない回廊サービスにある。

分析では、送金とディアスポラ金融について [[foreign-financial-institutions/philippine-national-bank-japan|PNB Japan]] および [[foreign-financial-institutions/metrobank-japan|Metrobank Japan]] と、大企業向けブラジル金融について [[megabanks/mufg-bank|MUFG Bank]] / [[megabanks/sumitomo-mitsui-banking-corp|SMBC]] と比較する。
