# コメルツ銀行 東京支店


## Wiki ルート

本エントリは [[foreign-financial-institutions/INDEX|foreign-financial-institutions INDEX]] に属する。ピア / 対比の文脈については [[foreign-financial-institutions/credit-agricole-bank-japan|クレディ・アグリコル銀行 東京支店]] と対照し、より広いシステム / 規制の境界については [[banking/INDEX|banking index]] と併読されたい。

## TL;DR

コメルツ銀行 東京支店は、金融庁の外国銀行支店一覧に掲載される Germany / Commerzbank AG の在日銀行支店。日本では corporate / institutional clients、外国貿易、FX、トランザクションバンキング、輸出志向の日本企業向け支援を中心とした wholesale branch として扱う。

## 2. オペレーティングモデル

Commerzbank は、1961 に東京駐在員事務所を開設し、1977に支店へ転換したと述べている。日本ページは、東京支店を、日本の企業・機関投資家向け、および世界中の輸出志向の日本企業向けの、現地に設立されグローバルにネットワーク化されたプラットフォームとして説明している。

現行の Commerzbank worldwide PDF は、東京支店を Toranomon Hills Station Tower に掲載し、コーポレートとフィナンシャル・インスティテューションの両デスクを示しており、これは外国銀行支店のパターンに合致する：リテールとの関連は限定的だが、意味のある B2B、貿易、FX、機関投資家向けの接続性を持つ。
