# ソシエテ・ジェネラル日本

## ウィキ上の位置づけ

この項目は [[foreign-financial-institutions/INDEX|foreign-financial-institutions INDEX]] の配下に置く。比較・対照の文脈は [[foreign-financial-institutions/bank-of-america-japan|バンク・オブ・アメリカ日本 (Bank of America Japan)]] と照合し、より広い制度・規制上の境界は [[banking/INDEX|banking index]] とあわせて読む。

## 要約

ソシエテ・ジェネラル日本は、仏大手銀行 Societe Generale グループの在日拠点群。銀行東京支店と証券法人を通じて、ホールセール銀行、マーケッツ、デリバティブ、証券・投資銀行業務を行う。[[foreign-financial-institutions/deutsche-japan|deutsche-japan]]、[[foreign-financial-institutions/bnp-paribas-japan|bnp-paribas-japan]]、[[foreign-financial-institutions/barclays-japan|barclays-japan]] と並ぶ欧州系在日 IB / markets 参加者。

## 1. 会社概要

| 項目 | 内容 |
|---|---|
| 親会社 | Societe Generale S.A. |
| 主な在日銀行拠点 | ソシエテ・ジェネラル銀行 東京支店 |
| 主な在日証券拠点 | Societe Generale Securities Japan Limited |
| 東京支店設立 | 1973年6月 |
| 主要業務 | 銀行業務、登録金融機関業務、外国銀行代理業務、証券・デリバティブ |
| 監督 | 金融庁、関東財務局、本国フランス・EU 監督 |

金融庁の登録金融機関一覧では、ソシエテ・ジェネラル銀行東京支店は関東財務局長（登金）第90号の登録金融機関として確認できる。外国銀行代理銀行一覧では、ソシエテ・ジェネラル銀行東京支店に関する認可も確認できる。

## 2. 事業セグメント・マップ

| 事業 | 内容 | 日本での位置づけ |
|---|---|---|
| ホールセール銀行 | 法人向け与信、トレード、外為 | 多国籍企業・金融機関向け |
| マーケッツ | 株式、FX、債券、コモディティ、デリバティブ | 欧州系 structured products 強み |
| 証券 | セールス、トレーディング、上場デリバティブ、市場アクセス | 機関投資家向け |
| 投資銀行 | 資本市場、M&A、クロスボーダー案件 | 邦銀・欧米 IB との競合 |
| 外国銀行代理 | グループ銀行業務の日本代理 | 規制上の明確な枠組み |

## 3. 戦略・競争

- **欧州系 markets house**: SocGen はグローバルにデリバティブ・structured products で存在感があり、日本では機関投資家・金融機関向けが中心。
- **欧州勢との比較**: [[foreign-financial-institutions/deutsche-japan|Deutsche]] は FICC・欧州銀行、[[foreign-financial-institutions/bnp-paribas-japan|BNP Paribas]] は仏最大級総合金融、SocGen は markets / derivatives 色が強い。
- **米系 IB との差**: [[foreign-financial-institutions/jpmorgan-japan|JPMorgan]]、[[securities-firms/goldman-sachs-japan|Goldman Sachs]]、[[securities-firms/morgan-stanley-japan|Morgan Stanley]] より日本での大型 M&A プレゼンスは相対的に小さいが、商品設計と欧州顧客接点に強みがある。
- **規制接点**: 登録金融機関・外国銀行代理・証券会社という複数ライセンスを組み合わせるため、銀行法・金商法・デリバティブ規制の横断理解が必要。

## 4. 規制・政策

- ソシエテ・ジェネラル銀行東京支店は外国銀行支店として銀行法監督対象。
- 登録金融機関業務は金商法に基づく登録金融機関規制の対象。
- 証券法人は第一種金融商品取引業等の金商法規制対象。
- 加入協会として全国銀行協会、日本証券業協会、金融先物取引業協会等が公式ページで確認できる。
