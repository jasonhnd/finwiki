# クレディ・アグリコル銀行 東京支店


## ウィキ動線

本項目は [[foreign-financial-institutions/INDEX|foreign-financial-institutions INDEX]] の配下に位置する。同業 / 対比の文脈については [[foreign-financial-institutions/natixis-japan|ナティクシス 東京支店]]、より広範なシステム / 規制の境界については [[banking/INDEX|banking index]] と対比して読むこと。

## TL;DR

クレディ・アグリコル銀行は、金融庁の外国銀行支店一覧に掲載される、フランスの Crédit Agricole CIB の在日銀行支店である。[[life-insurers/credit-agricole-life|クレディ・アグリコル生命]] とは別個の銀行法上の支店として、ホールセールバンキング / CIB / グローバルマーケッツ / 貿易・輸出金融の日本拠点を扱う。

## 2. オペレーティングモデル

Crédit Agricole CIB の日本ページは、投資・法人銀行、グローバルマーケッツ、実物資産ファイナンス、国際貿易・トランザクションバンキング、コーポレート・レバレッジファイナンス、ディストリビューション、およびファイナンスサービスにわたる東京フランチャイズを記述している。公式の日本ネットワークページはさらに、プロジェクト、輸出、貿易、航空機、不動産、円建てファイナンス、デリバティブ、ヘッジ、デットキャピタルマーケッツ、JGB プライマリー・ディーラー、および JSCC の IRS 参加を記述している。

金融庁の一覧は、銀行法上の支店名および現在の登録上の収載に関する真正性の典拠である。2026  の Crédit Agricole CIB のオフィス告知はグループの麻布台ヒルズへの移転を確認している。支店の所在地の詳細は、住所の精度が重要となる場合には、現在の金融庁ワークブックと照合して確認すべきである。

## 3. このページの意義

- クレディ・アグリコルの日本の銀行支店を、その生命保険会社および証券支店から区別する。
- [[foreign-financial-institutions/bnp-paribas-japan]]、[[foreign-financial-institutions/socgen-japan]]、[[foreign-financial-institutions/natixis-japan]] に対して、フランス / CIB の支店という同業者を加える。
- 外国銀行をリテール銀行として扱うことなく、[[banking/foreign-bank-japan-retreat]] における外国銀行支店のマップを補完する。
