# クリアストリーム・バンキング日本 (Clearstream Banking Japan)

## ウィキ上の位置づけ

このページは、[[foreign-financial-institutions/foreign-bank-branches-japan-index|the foreign-bank branch registry]] 内の Clearstream 行を独立した ICSD ルートへ昇格させるものである。[[foreign-financial-institutions/euroclear-bank-japan|Euroclear Bank Japan]]、[[securities/japan-market-infrastructure-map|market infrastructure map]]、[[banking/japan-trust-bank-custody-map|trust-bank custody map]]、[[financial-licenses/foreign-financial-group-adjacent-licenses|foreign financial group adjacent-license route]] とあわせて読む。

この項目はまず [[foreign-financial-institutions/INDEX|foreign-financial-institutions INDEX]] を通じて位置づけ、その後にポストトレード・インフラについては [[securities/INDEX|securities domain]]、カストディ / 支店ステータスについては [[banking/INDEX|banking domain]] を使う。

## 概要

Clearstream Banking S.A. Japan Branch は、Deutsche Boerse / Clearstream のポストトレード・スタック内にあるルクセンブルク ICSD 銀行支店として読むのが最も適切である。日本での重要性は、消費者向け銀行業ではなく、証券決済、カストディ、担保管理、開示ルーティング、JASDEC / JGB アクセスにある。

登録は、確認済みの FSA 出典で 2026-05-13時点に確認される。このページは、引用した公開出典を超えて現在の免許状態について独自判断を行わない。

## 登録一覧だけでは足りない理由

Clearstream と [[foreign-financial-institutions/euroclear-bank-japan|Euroclear]] は、日本ウィキにおける自然な ICSD 比較ページである。登録一覧の一行だけでは不十分であり、Clearstream の公開資料は次のルート整理に使われる。

- 日本証券の決済とカストディ;
- 日本市場保有に関する開示要件;
- 日本の社債、地方債、国債への機関投資家アクセス;
- 担保およびグローバル流動性管理;
- 日本の CSD / CCP ルートとオフショア投資家口座を結ぶポストトレード・インフラリンク.

| ICSD 役割 | Clearstream Banking S.A. グローバルな決済 / カストディ・インフラに位置する. | 比較する [[foreign-financial-institutions/euroclear-bank-japan|Euroclear]]. |
| 日本決済 / カストディ | Clearstream の公開市場資料は、日本の決済・カストディルートを説明している。 | 具体的事項に依拠する前に、現在の市場カバレッジページを確認する。 |
| 開示ルート | Clearstream は日本市場の開示要件資料を公表している。 | 実質所有者と nominee-name の論点に有用。 |
| ローカル FMI インターフェース | 日本決済分析は [[securities/japan-securities-depository-center|JASDEC]] and [[securities/japan-securities-clearing-corp|JSCC]]. | ICSD、CSD、CCP の役割を混同しない。 |

## Euroclear との比較

| 問い | Clearstream | Euroclear |
|---|---|---|
| グループ context | Deutsche Boerse / Clearstream group context. | Euroclear group / Euroclear Bank context. |
| 日本支店の役割 | Local banking branch plus market-access and disclosure surface. | Local banking branch plus ICSD / JGB collateral surface. |
| ウィキでの使い方 | JASDEC アクセス、日本債券カストディ、開示要件、Deutsche グループ隣接性に有用。 | JGB 担保、クロスボーダー DVP、カストディ、ポストトレード・リスク削減に有用。 |
| 読み取り上の主なリスク | Treating a market coverage page as universal availability for every client. | Treating ICSD custody name as economic ownership. |

## JapanFG での関連性

Clearstream は同じ調査クラスターに属する: [[foreign-financial-institutions/deutsche-japan|Deutsche Japan]], [[foreign-financial-institutions/bny-mellon-japan|BNY Mellon Japan]], [[foreign-financial-institutions/state-street-japan|State Street Japan]], [[financial-regulators/japan-exchange-group|Japan Exchange Group]], and [[financial-regulators/japan-securities-finance|Japan Securities Finance]]. 外国金融グループがリテール銀行にならずに、銀行支店の足跡と隣接する証券 / ポストトレード上の関連性を同時に持ちうることを説明する助けになる。

グループが日本で複数の法人または活動登録を持つ場合は [[financial-licenses/foreign-financial-group-adjacent-licenses|foreign financial group adjacent licenses]] を使い、問いが特に支店ルートである場合は [[financial-licenses/foreign-bank-branch-license-route|foreign-bank branch license route]] を使う。

## 調査チェックリスト

1. 証券の種類を特定する。JGB、社債、地方債、株式、投資信託、またはその他の商品か。
2. 問いが決済、カストディ、担保、開示、税務 / 実質所有者書類のどれに関するものか確認する。
3. サービス水準に関する記述を使う前に、Clearstream の最新市場カバレッジページを確認する。
4. Cross-read local infrastructure with [[securities/japan-securities-depository-center|JASDEC]], [[securities/japan-securities-clearing-corp|JSCC]], and [[securities/japan-market-infrastructure-map|market infrastructure map]].
5. カストディ銀行比較では、[[banking/japan-trust-bank-custody-map|trust-bank custody map]] とグローバルカストディアンのページを開く。

## 注意点

- ICSD の存在は、すべての日本証券がすべての口座種別を通じて同じように利用可能であることを意味しない。
- 開示および源泉徴収関連書類は、投資家種別、口座チェーン、商品によって異なりうる。
- 現在の免許状況が重要な場合は、公開登録ステータスを FSA 出典から更新する。
