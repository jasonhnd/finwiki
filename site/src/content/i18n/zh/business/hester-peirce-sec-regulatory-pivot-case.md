## Key facts

- Peirce 因 2018年对 Winklevoss BTC ETF 的反对意见而获得「Crypto Mom」的别名 ^[extracted]
- 于 2020-02提出 Token Safe Harbor Proposal 2.0(向加密资产初创企业授予 3年宽限期) ^[extracted]
- 于 2025-01-21出任 SEC 加密工作组主席(Trump + Paul Atkins SEC 主席的授权) ^[extracted]
- 在任期间 Q1-Q2撤销对 Coinbase / Robinhood / Uniswap / Consensys 的诉讼 ^[extracted]
- 同期 SEC 加密资产执法部门削减约 50%,并就 staking / memecoin / PoR 发出 no-action 指引 ^[extracted]

## Mechanism / How it works

监管转向经由 3点同时触发机制成立。**1) 个人意识形态的长期一贯性**(Peirce 的 2018-2025年立场不变,就任前即已公开承诺)→ 就任后的行动可预测,声誉风险也较小。**2) 机构授权**(加密工作组主席就任)→ 一位委员的个人见解晋升为 SEC 的政策路线。**3) 三角共振**(行政 = Trump / Atkins、立法 = Lummis / GENIUS Act、业界 = Coinbase / Circle 的监管解除)→ 三者缺一则转向不会发生(与 [[business/jamie-dimon-anti-crypto-pivot-case|Jamie Dimonの反暗号資産から転換した事例]] 镜像对比更易理解)。

诊断模板:识别「下一个监管转向」的关键,是确定该机构内长期持续持有异议的官僚(在任 5年以上 + 公开的 dissent record + 不持有与大企业的经济利害),并等待行政侧的任命 + 立法侧的法案齐备。3点齐备后,该官僚任期内 6-12个月以内会出现政策的大幅转向。Peirce 的案例 = 完整的三点齐备,撤诉浪潮在 2025-Q1集中爆发(参照:[[business/INDEX|business事例ライブラリ]] 与 [[fintech/genius-act-501-denylist-mandate|GENIUS Act §501 denylist 強制条項]] 的立法侧落地)。

## Origin & evolution

起点是 2018-01的 Peirce SEC 委员就任(Trump 第一任期任命)。2018-07对 Winklevoss ETF 的反对意见确立了「Crypto Mom」的别名,此后 6年间维持加密资产友好的少数派立场。2020-02的 Token Safe Harbor Proposal 2.0 → 业界把握了该政策框架。2024-Q4的 Trump 当选 + Atkins SEC 主席指名 → 3点触发条件齐备。2025-01-21加密工作组发足,Peirce 出任主席 → 监管转向启动。2025-02SEC 撤销 Coinbase / Robinhood / Uniswap / Consensys(2-3周以内集中爆发)→ 转向落地。后段:Peirce 一贯主张「经由国会立法解决」(而非 SEC 的行政裁量)→ GENIUS Act / CLARITY Act 成为她的政治帰结。
