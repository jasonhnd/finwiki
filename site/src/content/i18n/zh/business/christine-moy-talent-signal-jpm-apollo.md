## Mechanism / How it works

将人才流动作为信号解读的机制:**早期建设者**比公开的高管更早接触到下一个产品周期的证据(数据 / 与客户的对话 / 内部路线图)。当他们以跳槽的形式下注时,会将**个人声誉 + 股票期权 + 时间成本**全额押注于新方向,从而构成比公开公告更强的信号。解读方法:1) 列出行业 top 20的早期建设者、2) 监控 LinkedIn / Twitter 的转职告知、3) 向同领域不同 vertical 的移动(例:TD → MMF / Credit / RWA)= 重心转移、4) 向完全不同领域的移动 = 个人选择,作为信号较弱。

应用 = 向新 vertical 提前 6-18个月投入研究资源。Christine Moy 转任 Apollo + 同期 Arc 经由私募收购 Apollo + ACRED on Arc 构想 → 三点交叉验证「私募信用的链上化」是机构稳定币战争的下一个战场(参照:[[fintech/apollo-acred-private-credit-tokenization|Apollo ACRED 私募信用トークン化]] 与 [[business/larry-fink-blackrock-digital-asset-template|BlackRock 三角形テンプレート]] 的资产运营 RWA 对照)。

## Origin & evolution

起源是 2020年年代初期对 fintech 招聘的观察(Stripe / Plaid / Adyen 之间的资深工程师流动),在加密资产时代扩大。2024-2025年期间出现多对类似流动:Christine Moy(JPM → Apollo)/ Sergey Nazarov 麾下团队(Chainlink → Goldman tokenization)/ ex-Coinbase 资深工程师 → Tempo 等。共同特征:转职告知的时间窗 = 6-12个月以内在同领域发生 3件起以上的转职 → 1年以内在该 vertical 出现公开战略发布。本调查采用 Christine Moy 作为锚点案例,因其在 JPM Onyx 的早期角色有明确记录。
