# 株式会社JCB（JCB）

## Wiki 路径

本条目位于 [[card-issuers/INDEX|card-issuers INDEX]] 之下。请对照 [[card-issuers/orico|株式会社オリエントコーポレーション (Orico)]] 以获取同业 / 对比背景，并参阅 [[banking/INDEX|banking index]] 以了解更广泛的体系 / 监管边界。

## TL;DR

**日本唯一的国际卡品牌**。于 1961-01 由旧三和银行 + 日本信贩 等出资，以「日本信用调查所（Japan Credit Bureau）」的名义成立，其后简称「JCB」作为公司名称与品牌名称固定下来。它构成国际 5 大卡品牌（**Visa / Mastercard / American Express / Discover / JCB**）之一，**在亚洲（日本·中国·韩国·台湾·东南亚）具有优势**。非上市 — 主要股东为 [[megabanks/mufg]] 系·[[megabanks/smfg]] 系的银行联盟。美国通过 Discover、中国通过银联（UnionPay）的相互通用提携，覆盖全球 200 以上的国家与地区。国内与 [[megabanks/smfg]] 系三井住友卡（Visa）/ 楽天卡等竞争。

## 1. 公司概况

**正式名称**：株式会社JCB
**英文名称**：JCB Co., Ltd.
**创业**：1961-01 （以「株式会社日本信用调查所」名义设立）
**上市**：**非上市**（主要股东为银行系）
**总部**：东京都港区南青山 5-1-22 青山 Rise Square
**业态**：信用卡公司 + 国际卡品牌（发行 / 商户拓展 / 品牌·授权的 3 机能）

### 简称的由来

- **JCB** = **J**apan **C**redit **B**ureau（日本信用调查所）
- 创业时的公司名「日本信用调查所」的英文首字母。其后英文简称作为公司名称与品牌名称固定下来。

### 国际品牌差异化

- **「在亚洲强势的国际品牌」定位**：在日本·中国·韩国·台湾·东南亚（泰国 / 越南等）的商户网具备优势
- **美国**：通过与 Discover Card 提携实现相互通用（JCB 卡可在美国 Discover 商户使用）
- **欧洲**：有限。以旅游地·主要酒店等为中心的商户网
- **5 大品牌（Visa / Mastercard / Amex / Discover / JCB）之一**：日本唯一·亚洲唯一的国际品牌

### 入境战略

- **访日中华圈旅客需求**：通过银联提携（2006〜）捕捉访日中国游客的支付需求
- **亚洲旅游地**：以泰国·韩国·台湾的商户网吸纳日本人海外旅行的支付
- **跨境支付**：在国际旅客流动的双向（in / out）上具备优势

### 国内竞争格局

- **Visa 提携卡**：三井住友卡（[[megabanks/smfg]] 系·最大份额）/ 楽天卡 / 永旺卡等
- **Mastercard 提携**：楽天卡 / dカード（[[megabanks/ndfg]] 系）/ au PAY 卡等
- **JCB 品牌**：国内份额仅次于 Visa，约 2 位前后（推定 / 公开数值有限）
- **数字支付竞争**：PayPay / 楽天 Pay / d 払い / au PAY 等码支付正在侵蚀「卡 + 移动支付」需求

### 数字化对应

- **JCB 非接触**（触碰支付 NFC 非接触）：2020〜扩大，捕捉疫情期间的非接触需求
- **Apple Pay / Google Pay / Garmin Pay / Fitbit Pay** 等移动支付对应
- **电子商务·3D Secure 2.0** 对应
- **协助日本央行数字日元试行**（推定 / 基于 Wikipedia 公开信息）

## 4. 监管·政策

- **主管**：经济产业省（割赋销售法所管）+ 金融厅（资金结算法·修订割赋销售法等）
- **主要依据法**：割赋销售法（信用卡公司监管）、资金结算法（预付式支付手段等）
- **个人信息保护**：个人信息保护法（信用使用记录·本人确认信息）
- **商户管理**：修订割赋销售法（2018〜）将商户调查义务化
- **近期政策议题**：
  - 2020 年年代 无现金推进（政府目标 2025 年 40%、2030 年年代 80%）
  - 发票制度（2023-10〜）使卡明细的处理发生变化
  - 对稳定币 / 数字日元试行的应对
