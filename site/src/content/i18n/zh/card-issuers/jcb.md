# JCB 株式会社 (JCB)

## Wiki route

本条目位于 [[card-issuers/INDEX|card-issuers INDEX]] 下。可与 [[card-issuers/orico|株式会社オリエントコーポレーション (Orico)]] 对照阅读，以理解同业和差异；更广泛的制度和监管边界见 [[banking/INDEX|banking index]]。

## TL;DR

**日本唯一的国际卡品牌。** 该公司在 1961-01  由旧三和银行、日本信贩等出资，以 Japan Credit Bureau 的形式成立；后来缩写 "JCB" 成为公司名和品牌名。它构成国际 5  大卡品牌（**Visa / Mastercard / American Express / Discover / JCB**）之一，并在**亚洲（日本、中国、韩国、台湾和东南亚）**具有优势。公司未上市，主要股东是以 [[megabanks/mufg]] 系、[[megabanks/smfg]] 系为中心的银行联盟。通过与美国 Discover、中国银联 (UnionPay) 的相互受理合作，JCB 覆盖全球超过 200  个国家和地区。在日本国内，它与 [[megabanks/smfg]] 系三井住友卡 (Visa)、乐天卡等竞争。

## 1. 公司概要

**正式名称**：JCB Co., Ltd.
**英文名称**：JCB Co., Ltd.
**创立**：1961-01 （作为 Japan Credit Bureau 成立）
**上市状态**：**未上市**（主要股东为银行系）
**总部**：东京都港区南青山 5-1-22  青山 Rise Square
**业务形态**：信用卡公司 + 国际卡品牌，兼具发行、加盟店开拓和品牌授权 3  项功能。

- 该缩写来自创立时公司英文名称 Japan Credit Bureau 的首字母。后来英文缩写固定为公司名和品牌名。

## 2. 业务分部地图

| 分部 | 内容 | 备注 |
|---|---|---|
| **Issuer（发行）** | 自社直接发行 JCB 卡 + 向合作发卡公司授权品牌 | 以日本国内为中心 |
| **Acquirer（加盟店开拓）** | 日本国内和海外加盟店合同 / 支付处理 | 海外由 JCB International 负责 |
| **国际品牌** | "JCB" 标识和授权业务 | 与 Visa / Mastercard / Amex / Discover 并列的 5  大品牌之一 |
| **合作卡** | 与银行、零售、航空、酒店等共同发行 JCB 品牌卡 | 三井住友卡、瑞穗银行、乐天、JAL 等众多伙伴 |
| **高端产品** | JCB Platinum、JCB The Class、JCB Gold 等 | 面向日本国内富裕层 |
| **Acceptance（相互受理）** | 与 Discover（美国）/ 银联（中国）/ 旧 Amex 的相互受理 | 补足海外加盟店覆盖 |
| **数字 / 触碰支付** | JCB Contactless / Apple Pay / Google Pay / 移动支付 | 自 2020 起扩大 |

### 国际品牌差异化

- **“亚洲优势国际品牌”的定位**：在日本、中国、韩国、台湾以及东南亚（泰国 / 越南等）具有相对强的加盟店网络。
- **美国**：通过 Discover Card 合作实现相互受理，JCB 卡可在美国 Discover 加盟店使用。
- **欧洲**：覆盖有限，主要集中在旅游地和主要酒店等。
- **5  大品牌（Visa / Mastercard / Amex / Discover / JCB）之一**：日本唯一、亚洲唯一的国际卡品牌。

### 入境战略

- **访日中华圈旅客需求**：通过银联合作（2006 起）捕捉访日中国游客支付需求。
- **亚洲观光地**：通过泰国、韩国、台湾的加盟店网络吸收日本人海外旅行支付。
- **跨境支付**：在国际旅客流动的两个方向（in / out）都具有优势。

### 日本国内竞争结构

- **Visa 合作卡**：三井住友卡（[[megabanks/smfg]] 系，最大份额）/ 乐天卡 / 永旺卡等。
- **Mastercard 合作**：乐天卡 / d 卡（[[megabanks/ndfg]] 系）/ au PAY 卡等。
- **JCB 品牌**：国内份额估计在 Visa 之后、约居第二（2 位；公开数值有限）。
- **数字支付竞争**：PayPay、乐天 Pay、d 払い、au PAY 等二维码支付正在侵蚀“卡 + 移动支付”需求。

### 数字化应对

- **JCB Contactless**（NFC 触碰式非接触支付）：自 2020 起扩大，捕捉疫情时期和之后的非接触需求。
- **支持 Apple Pay / Google Pay / Garmin Pay / Fitbit Pay** 等移动支付。
- **支持电商和 3D Secure 2.0**。
- **参与日本银行数字日元试行**（推定 / 基于 Wikipedia 公开信息）。

## 4. 监管和政策

- **主管**：经济产业省（《割赋销售法》主管）+ 金融厅（《资金结算法》、修订割赋销售法等）。
- **主要依据法**：《割赋销售法》（信用卡公司监管）、《资金结算法》（预付式支付工具等）。
- **个人信息保护**：《个人信息保护法》（信用使用历史、本人确认信息）。
- **加盟店管理**：修订割赋销售法自 2018 起强化加盟店调查义务。
- **国际标准**：遵循 PCI DSS (Payment Card Industry Data Security Standard)。
- **近期政策议题**：
  - 2020 年年代的无现金推进政策，包括政府目标 2025 年 40%、2030 年年代 80%。
  - 发票制度自 2023-10 起改变卡明细处理。
  - 稳定币 / 数字日元试行应对。
