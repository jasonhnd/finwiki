# 株式会社View卡 (Viewcard)


## Wiki route

本条目归属于 [[card-issuers/INDEX|card-issuers INDEX]]。请对照 [[card-issuers/jal-card|株式会社JALカード]] 阅读以获取同业 / 对比背景，并对照 [[payments/credit-purchase-card-operators-japan-index|credit purchase / card operators index]] 阅读以了解更广泛的监管 / 登记边界。

## TL;DR

株式会社View卡（Viewcard Co., Ltd.）是 **JR 东日本的完全子公司，负责发行和运营"View卡"品牌信用卡的公司**。2009年10月 设立（由原 JR 东日本信用卡事业部分拆而来），总店位于东京都品川区大崎，资本金 50 億円，员工 590 名（2024-04  时点）。作为 **"Suica"一体型卡的主力发行机构**，构成 JR 东日本集团无现金・出行经济的核心。其持有 **银行代理业者登记** 这一点与其他卡公司不同，拥有包括 Suica 充值・卡贷中继等在内的广泛金融职能。竞争对手为 [[card-issuers/jal-card|JAL カード]]、ANA 卡（航空系）、[[card-issuers/credit-saison|セゾン]]・[[card-issuers/epos-card|エポス]]（流通系）等。

## 1. 法人・股东

- **商号**：株式会社View卡（英文：Viewcard Co., Ltd.）
- **设立**：2009年10月1日（将东日本旅客铁道株式会社的信用卡事业部分拆）
- **总店所在地**：〒141-8540  东京都品川区大崎 1-5-1  大崎新城 1号 馆
- **资本金**：50 億円
- **法人编号**：2010701023263
- **母公司**：东日本旅客铁道株式会社（JR 东日本，东证 PRIME 9020）100% 出资的完全子公司
- **代表人**：代表取缔役社长
- **员工人数**：590 名（2024 年 4 月 1 日 时点）

## 2. 牌照・登记

- **分期付款销售法**：作为综合信用购买斡旋业者向 METI 登记
- **信用卡号码等处理合同缔结事业者**：METI 登记
- **银行代理业者**：承担与 JR 东日本集团相关的银行代理业务（特色牌照，其他卡公司所没有）
- **行业团体**：[[card-issuers/jcb|日本クレジット協会]] (JCA) 会员
- **国际品牌合作**：JCB・VISA・Mastercard
- **发行协作**：除自社发行外，还整合 [[card-issuers/jr-east-financial|JR東日本ファイナンシャル]] 系列的卡功能

## 3. 产品线

- **View・Suica卡（VIEW Suica）**：可在 JR East 全站充值 Suica，自动充值功能，JRE POINT 返还
- **「View・卡」**（基础款）
- **「Bic Camera Suica 卡」** — Bic Camera 合作卡
- **JRE CARD / JRE CARD 金卡** — JRE POINT 返还强化
- **JAL 卡 Suica**（与 [[card-issuers/jal-card|JAL カード]] 共同发行，航空 + 铁道里程统合）
- **大人的休日俱乐部 Jipangu** — 面向年长者的 JR 折扣卡
- **View 金卡 Plus 卡** — 金卡级
- **View卡 高级** — 高端层

## 4. 主要 KPI（公开披露基础）

- **会员数**：约 600 万 人以上（基于 JR East 集团综合报告书）
- **Suica 充值交易额**：自动充值 + JRE POINT 返还的主力渠道
- **JRE POINT 经济圈会员**：1,900 万 人以上（JR East 整体，Suica + Viewcard 统合 ID）
- **交易额**：非公开（统合于母公司 JR East 的"IT・Suica 事业"板块）

## 5. 行业地位

- 作为 **铁道系亲和卡**，与 **ICOCA（JR 西日本）・PASMO 合作卡（各私铁公司）・TOICA（JR 东海）** 并列为交通系卡经济圈的核心
- 作为 **JR East 的 Suica + JRE POINT 生态系统** 的"唯一专属信用卡发行机构"占据独特地位
- 在 **JR East 的 MaaS（出行即服务）战略** 中是重要的支付枢纽
- 竞争对手：[[card-issuers/jal-card|JAL カード]]（航空）、[[card-issuers/credit-saison|セゾン]]・[[card-issuers/epos-card|エポス]]・[[card-issuers/seven-card-service|セブンカード]]（流通）、[[card-issuers/aeon-financial-service|イオン]]（流通系）
- 随着 **移动 Suica** 的普及・扩大，其作为支付平台的重要性正在扩大

## 6. 沿革

| 年月 | 事项 |
|------|------|
| 1980 年 年代后半 | JR 东日本将信用卡事业内制化 |
| 2001-11  | Suica 服务开始 — View卡与 Suica 的一体型卡商品扩大 |
| 2007-03  | 移动 Suica 服务开始 |
| 2009-10-01 | 株式会社View卡设立（JR 东日本信用卡事业部的分拆） |
| 2010  | JRE POINT 制度的前身「View Thanks Points」运营扩充 |
| 2016-06  | JRE POINT 统合 — Suica + Viewcard + JRE POINT 统合为 1  个 ID 经济圈 |
| 2019-10  | 移动 Suica 功能扩展，Apple Pay / Google Pay 支持推进 |
| 2022  | View卡 高级 卡贷扩充（活用银行代理业者牌照） |
| 2024-04  | 员工人数 590 名，Suica + Viewcard + JRE POINT 连结至 JR East 统合 ID |

## 7. 关联企业・交易对象

- **母公司**：[[card-issuers/jr-east-financial|JR East Financial]] / 东日本旅客铁道
- **国际品牌**：JCB・VISA・Mastercard
- **共同发行卡**：[[card-issuers/jal-card|JAL カード Suica]]
- **加盟店伙伴**：Bic Camera（Bic Camera Suica 卡）、JR East 站内加盟店网络（NewDays・ecute 等）
- **Suica 充值・移动 Suica**：与 JR East 的系统基础联动

## 8. 为何需要作为独立页面

需要从 JR East 的本体 [[card-issuers/jr-east-financial|JR East Financial]] 页面中，将作为卡发行体的牌照边界・银行代理业者牌照・JRE POINT 生态系统・Suica 一体型卡战略分离出来，进行独立比较基准。为了准确记录 **"持有银行代理业者牌照的卡公司"** 这一罕见的复合牌照，也是必不可少的。
