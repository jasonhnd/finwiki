# 株式会社 Viewcard (Viewcard)

## Wiki 路径

本条目位于 [[card-issuers/INDEX|card-issuers INDEX]] 之下。可与 [[card-issuers/jal-card|株式会社JALカード]] 对照阅读以理解同业 / 对比语境，并参考 [[payments/credit-purchase-card-operators-japan-index|credit purchase / card operators index]] 把握更广泛的监管 / 注册边界。

## TL;DR

株式会社 Viewcard（Viewcard Co., Ltd.）是 **JR 东日本的全资子公司，发行并运营“Viewcard”品牌信用卡**。2009年10月 设立，由原 JR 东日本信用卡事业部分拆而来，总部位于东京都品川区大崎，资本金 50 億円，员工 590 名（截至 2024-04 ）。作为 **“Suica”一体型卡的主力 issuer**，它构成 JR 东日本集团无现金和 mobility economy 的核心。其持有 **银行代理业者注册**，不同于其他卡公司，拥有包括 Suica 充值、卡贷中介等在内的广泛金融功能。竞争对手包括 [[card-issuers/jal-card|JAL カード]]、ANA Card（航空系）、[[card-issuers/credit-saison|セゾン]] / [[card-issuers/epos-card|エポス]]（流通系）等。

## 1. 法人 / 股东

- **商号**：株式会社 Viewcard（英文：Viewcard Co., Ltd.）
- **设立**：2009年10月1日（东日本旅客铁道株式会社信用卡事业部分拆）
- **总部所在地**：〒141-8540  东京都品川区大崎 1-5-1  大崎 New City 1号馆
- **资本金**：50 億円
- **法人编号**：2010701023263
- **母公司**：东日本旅客铁道株式会社（JR 东日本，东证 PRIME 9020）100% 出资的全资子公司
- **代表者**：代表取缔役社长
- **员工数**：590 名（截至 2024 年 4 月 1 日）

## 2. 牌照 / 注册

- **割赋销售法**：作为综合信用购买中介业者在 METI 注册
- **信用卡号码等处理合同签约业者**：METI 注册
- **银行代理业者**：承担与 JR 东日本集团相关的银行代理业务（特色牌照，其他卡公司少见）
- **行业团体**：[[card-issuers/jcb|日本クレジット協会]] (JCA) 会员
- **国际品牌合作**：JCB、VISA、Mastercard
- **发行合作**：除自社发行外，也整合 [[card-issuers/jr-east-financial|JR東日本ファイナンシャル]] 系列卡功能

## 3. 产品线

- **View Suica Card**：可在 JR East 全站 Suica 充值，具备自动充值功能和 JRE POINT 回馈
- **View Card**（基础标准卡）
- **Bic Camera Suica Card** — Bic Camera 联名卡
- **JRE CARD / JRE CARD Gold** — 强化 JRE POINT 回馈
- **JAL Card Suica**（与 [[card-issuers/jal-card|JAL カード]] 共同发行，整合航空 + 铁路里程）
- **大人的休日俱乐部 Zipangu** — 面向老年人的 JR 折扣卡
- **View Gold Plus Card** — 金卡级
- **Viewcard Premium** — 高阶层

## 4. 主要 KPI（公开口径）

- **会员数**：约 600 万人以上（基于 JR East 集团综合报告书）
- **Suica 充值交易额**：自动充值 + JRE POINT 回馈的主力渠道
- **JRE POINT 经济圈会员**：1,900 万人以上（JR East 整体，Suica + Viewcard 统一 ID）
- **取扱高**：未公开（整合入母公司 JR East 的“IT / Suica 事业”分部）

## 5. 行业位置

- 作为 **铁路系 affinity card**，是与 **ICOCA（JR 西日本）、PASMO 提携卡（私铁各社）、TOICA（JR 东海）** 并列的交通系卡经济圈核心。
- 作为 **JR East 的 Suica + JRE POINT 生态系统** 的“唯一专属信用卡 issuer”，具有独特位置。
- 在 **JR East 的 MaaS（Mobility as a Service）战略** 中是重要支付枢纽。
- 竞争对手：[[card-issuers/jal-card|JAL カード]]（航空）、[[card-issuers/credit-saison|セゾン]] / [[card-issuers/epos-card|エポス]] / [[card-issuers/seven-card-service|セブンカード]]（流通）、[[card-issuers/aeon-financial-service|イオン]]（流通系）。
- 随着 **Mobile Suica** 普及扩大，其作为支付平台的重要性继续上升。

## 6. 沿革

| 年月 | 事项 |
|------|------|
| 1980 年代后半 | JR 东日本将信用卡业务内部化 |
| 2001-11 | Suica 服务开始，Viewcard 与 Suica 一体型卡商品扩大 |
| 2007-03 | Mobile Suica 服务开始 |
| 2009-10-01 | 株式会社 Viewcard 设立（JR 东日本信用卡事业部分拆） |
| 2010 | 扩充 JRE POINT 制度前身“View Thanks Point”运营 |
| 2016-06 | JRE POINT 整合，Suica + Viewcard + JRE POINT 统一为 1  个 ID 经济圈 |
| 2019-10 | Mobile Suica 功能扩展，Apple Pay / Google Pay 支持推进 |
| 2022 | Viewcard Premium 卡贷扩充（活用银行代理业者牌照） |
| 2024-04 | 员工数 590 名，Suica + Viewcard + JRE POINT 连接至 JR East 统一 ID |

## 7. 相关企业 / 交易方

- **母公司**：[[card-issuers/jr-east-financial|JR East Financial]] / 东日本旅客铁道
- **国际品牌**：JCB、VISA、Mastercard
- **共同发行卡**：[[card-issuers/jal-card|JAL カード Suica]]
- **加盟店伙伴**：Bic Camera（Bic Camera Suica Card）、JR East 站内商业网络（NewDays、ecute 等）
- **Suica 充值 / Mobile Suica**：与 JR East 系统基础设施联动

## 8. 为什么需要独立页面

需要将卡发行主体的牌照边界、银行代理业者牌照、JRE POINT 生态系统、Suica 一体型卡战略从 JR East 本体 [[card-issuers/jr-east-financial|JR East Financial]] 页面中分离出来，形成独立比较基准。为了准确记录 **“持有银行代理业者注册的卡公司”** 这一少见复合牌照，也必须设立独立页面。
