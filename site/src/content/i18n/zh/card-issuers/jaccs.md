# Jaccs股份有限公司 (Jaccs)

## Wiki route

本条目位于 [[card-issuers/INDEX|card-issuers INDEX]] 下。可结合 [[card-issuers/orico|株式会社オリエントコーポレーション (Orico)]] 阅读以进行同业/对比分析，并结合 [[banking/INDEX|banking index]] 理解更广泛的体系与监管边界。

## TL;DR

Jaccs是发源于北海道的大型信用销售/消费信贷公司。**1954-07 以北海道信用贩卖株式会社名义在函馆设立**，并于1968 更名为“ジャックス”。其业态以信用销售、个品分期、信用卡贷款、代收款、房租保证、汽车贷款为轴心。作为**[[megabanks/mufg]]集团的权益法适用关联公司**（三菱UFJ银行为主要股东），Jaccs与MUFG的信用卡/消费者金融轴（[[megabanks/mufg]]中的三菱UFJ NICOS、Acom）并列，承担“**第三条信用销售轴**”。公司在东京证券交易所PRIME市场8584 上市。竞争对手包括[[card-issuers/orico]]、[[card-issuers/credit-saison]]。海外方面，在越南、柬埔寨、印度尼西亚开展金融业务。

## 1. 公司概要

**正式名称**: Jaccs股份有限公司（Jaccs Co., Ltd.）
**证券代码**: 东证PRIME **8584**
**设立**: **1954-07**（原北海道信用贩卖株式会社，发源于函馆）
**改称**: 1968 年更名为“ジャックス”
**上市**: 1971 在东京证券交易所第一部上市
**总部**: 东京都港区（登记）/ 函馆、札幌设有主要据点
**母公司**: **[[megabanks/mufg]]（权益法适用关联公司，三菱UFJ银行为主要股东）**

### 业态类别

在信用销售/信贷行业中，是仅次于**Orico（[[card-issuers/orico]]）的2至3 位集团**。
- 大型3 强: [[card-issuers/orico]] / **Jaccs** / [[card-issuers/credit-saison]]
- 行业轴: 个品分期 + 信用卡 + 代收款 + 保证

### 资本系列的历史

```
1954-07 北海道信用贩卖（函馆）
   ↓ 1968 商号变更
Jaccs
   ↓ 1971 东证第一部上市
   ↓ 1990s 经营困难 → 三和银行支援
   ↓ 2005-10 MUFG成立（[[megabanks/mufg]]母公司）
   ↓ 2007-04 纳入三菱UFJ银行系列（[[megabanks/mufg]]权益法适用）
现在: [[megabanks/mufg]]集团、东证PRIME 8584

```

## 2. 业务板块地图

| 板块 | 主要业务 | 特征 |
|---|---|---|
| **个品分期（购物信贷）** | 汽车贷款 / 教育贷款 / 装修贷款 | **主力3 大贷款轴**，强项是经加盟店的销售金融 |
| **信用卡** | Jaccs卡（VISA / Mastercard / JCB合作）| 将卡业务作为消费者接点开展 |
| **卡贷款** | 面向个人的无担保贷款 | 自有渠道、银行保证合作 |
| **代收款** | 公共费用、持续收费的账户转账代收 | BtoB存量收益 |
| **房租保证** | 面向租赁住宅的保证服务 | 与不动产业界合作 |
| **法人金融** | 法人租赁、分期业务 | 补充性业务 |
| **海外**（ASEAN） | 越南、柬埔寨、印度尼西亚消费者金融 | 以两轮车贷款、耐用消费品贷款为中心 |

### 汽车贷款轴的优势

Jaccs拥有**汽车销售店（含二手车）的加盟店网络**，在店头“贷款提案”流程中处于信用销售行业上位。教育贷款、装修贷款也采用类似的加盟店经由模式。这是其与[[card-issuers/orico]]直接竞争的主战场。

### 海外战略（ASEAN）

- **越南**: 自2014 年左右正式进入，提供两轮车、家电贷款
- **柬埔寨**: 通过当地法人提供面向个人的分期业务
- **印度尼西亚**: 两轮车、耐用消费品贷款

ASEAN战略与MUFG整体海外战略（Krungsri / Bank Danamon / VietinBank，参见[[megabanks/mufg]]）保持一致。Jaccs承担补充母公司银行网络的**消费者金融层**。

| 年 | 事项 |

| 1968 | 商号变更为“ジャックス” |
| **1971** | **在东京证券交易所第一部上市** |

| **1990s** | **经营困难** → 获三和银行（后来的UFJ → [[megabanks/mufg]]）支援 |

| **2007-04** | **纳入三菱UFJ银行系列**（权益法适用关联公司） |
| 2014〜 | **海外扩张加速**（ASEAN：越南、柬埔寨、印度尼西亚） |
| 2022-04 | 东证市场区分调整 → **东证PRIME 8584** |
| 2024〜 | 中期经营计划（公开部分）启动 |

### 集团内定位

[[megabanks/mufg]]集团内的消费者金融、信用卡、信贷3 轴：

| 功能 | 负责公司 | 从[[megabanks/mufg]]角度的定位 |
|---|---|---|
| 兆级银行、信用卡 | 三菱UFJ NICOS | 100%合并子公司、三大信用卡公司 |
| 消费者金融（无担保） | Acom | 权益法39.6%、东证STD 8572 |
| **信用销售、个品分期** | **Jaccs** | **权益法适用关联公司、东证PRIME 8584** |

也就是说，Jaccs负责“[[megabanks/mufg]]难以由直接银行贷款覆盖的**销售金融，即经加盟店的分期业务**”。

### 竞争结构

| 竞争对手 | 优势 | 与Jaccs的差异 |
|---|---|---|
| [[card-issuers/orico]] | 行业最大公司、伊藤忠 + 瑞穗系列 | 规模上领先；Jaccs以北海道、汽车、教育领域对抗 |
| [[card-issuers/credit-saison]] | 信用卡 x 存量型、流通系 | 更偏重信用卡；Jaccs在分期业务上更厚 |
| Aeon Financial | 流通系、WAON | 依赖流通基础；Jaccs加盟店更加多样 |
| Rakuten Card | 数字化、EC | 数字轴领先；Jaccs以线下加盟店对抗 |

### 中期战略（公开方向）

- **深化3 大贷款（汽车、教育、装修）**
- **扩大加盟店网络**：经销商、教育机构、住宅装修公司
- **扩充数字化申请流程**：从纸质授信转向Web完结
- **ASEAN消费者金融**：通过当地法人开展两轮车、耐用消费品贷款
- **作为MUFG系列的补充功能**：与[[megabanks/mufg]]银行、NICOS、Acom分工

## 5. 监管与政策

- **主管**: 金融厅（FSA）
- **登记**: 基于《分期付款销售法》和《贷金业法》的登记业者
- **行业团体**: 加盟日本信用协会（JCA）
- **主要监管**:
  - 分期付款销售法：综合信用购买中介业者、个别信用购买中介业者
  - 贷金业法：受总量限制（年收入1/3 限制）约束
  - 个人信息保护法、修订资金结算法（加密资产相关不适用）
- **近期政策论点**:
  - 2024〜 在授信模型中引入AI / 机器学习
  - 无现金化政策（卡手续费、加盟店监管）
  - 2025〜 修订分期付款销售法、本人确认高度化
