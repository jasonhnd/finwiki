# 株式会社ジャックス (Jaccs)

## Wiki 路径

本条目位于 [[card-issuers/INDEX|card-issuers INDEX]] 之下。请与 [[card-issuers/orico|株式会社オリエントコーポレーション (Orico)]] 对照以获取同侪 / 对比语境，并参照 [[banking/INDEX|banking index]] 了解更广义的系统 / 监管边界。

## TL;DR

发祥于北海道的信用·信贷销售大手。**作为北海道信用贩卖株式会社在函馆设立 (1954-07 )**，于 1968 改称为「ジャックス」。业态以信贷销售·单品分期·消费贷·代收代缴·房租担保·汽车贷款为轴。定位为 **[[megabanks/mufg]] 集团的权益法适用关联公司**（三菱 UFJ 银行为主要股东），与 MUFG 的信用卡·消费金融轴（[[megabanks/mufg]] 内的三菱 UFJ 尼克斯·Acom）并列，承担「**第三信贷销售轴**」。东证 PRIME 8584 上市。竞争对手为 [[card-issuers/orico]]·[[card-issuers/credit-saison]]。海外在越南·柬埔寨·印度尼西亚开展金融业务。

## 1. 公司概要

**正式名称**: 株式会社ジャックス（Jaccs Co., Ltd.）
**证券代码**: 东证 PRIME **8584**
**设立**: **1954-07**（旧北海道信用贩卖株式会社，发祥于函馆）
**改称**: 1968 年改为「ジャックス」
**上市**: 1971 东京证券交易所一部上市
**总部**: 东京都港区（登记）/ 函馆·札幌设有主要据点
**母公司**: **[[megabanks/mufg]]（权益法适用关联公司，三菱 UFJ 银行为主要股东）**

### 业态类别

在信贷销售·信用业界，**仅次于 Orico（[[card-issuers/orico]]）的 2〜3 位集团**。
- 大手 3 : [[card-issuers/orico]] / **Jaccs** / [[card-issuers/credit-saison]]
- 业界轴: 单品分期 + 信用卡 + 代收代缴 + 担保

### 资本系列的历史

```
1954-07 北海道信用贩卖（函馆）
   ↓ 1968 商号变更
ジャックス
   ↓ 1971 东证一部上市
   ↓ 1990s 经营困难 → 三和银行支援
   ↓ 2005-10 MUFG 成立（[[megabanks/mufg]] 母公司）
   ↓ 2007-04 并入三菱 UFJ 银行系列（[[megabanks/mufg]] 权益法适用）
现在: [[megabanks/mufg]] 集团·东证 PRIME 8584
```

## 2. 业务分部图

| 分部 | 主要业务 | 特征 |
|---|---|---|
| **单品分期（购物信贷）** | 汽车贷款 / 教育贷款 / 装修贷款 | **主力 3 大贷款轴**，在经由加盟店的销售金融方面具优势 |
| **信用卡** | ジャックスカード（VISA / Mastercard / JCB 提携）| 将卡业务作为消费者接点展开 |
| **消费贷** | 面向个人的无担保贷款 | 自有渠道·银行担保提携 |
| **代收代缴** | 公共费用·持续计费的账户扣款代理 | BtoB 存量收益 |
| **房租担保** | 面向租赁住宅的担保服务 | 不动产业界联动 |
| **法人金融** | 法人的租赁·分期处理 | 补充业务 |
| **海外**（ASEAN） | 越南·柬埔寨·印度尼西亚消费金融 | 以两轮车贷款·耐用品贷款为中心 |

### 汽车贷款轴的优势

Jaccs 拥有 **汽车销售店（含二手车）的加盟店网络**，凭借店头「贷款提案」流程位居信贷销售业界前列。教育贷款·装修贷款同样采用经由加盟店的模式。这是与 [[card-issuers/orico]] 直接竞争的主战场。

### 海外战略（ASEAN）

- **越南**: 自 2014 年前后起正式进入，两轮车·家电贷款
- **柬埔寨**: 以当地法人开展面向个人的分期
- **印度尼西亚**: 两轮车·耐用消费品贷款

ASEAN 与 MUFG 整体的海外战略（Krungsri / Bank Danamon / VietinBank，参见 [[megabanks/mufg]]）相契合。Jaccs 承担补充母公司银行网络的**消费金融层**。

## 3. 沿革

| 年 | 事件 |
|---|---|
| **1954-07** | **北海道信用贩卖株式会社 设立（函馆）** |
| 1968 | 商号变更为「ジャックス」 |
| **1971** | **东京证券交易所一部上市** |
| 1980s | 全国展开·总部职能迁至东京 |
| **1990s** | **经营困难** → 三和银行（后来的 UFJ → [[megabanks/mufg]]）的支援 |
| 2001 | UFJ 控股成立，Jaccs 并入 UFJ 系列 |
| **2005-10-01** | **[[megabanks/mufg]] 成立**，Jaccs 走向 MUFG 关联公司化的进程 |
| **2007-04** | **并入三菱 UFJ 银行系列**（权益法适用关联公司） |
| 2014〜 | **加速海外进入**（ASEAN：越南·柬埔寨·印度尼西亚） |
| 2022-04 | 东证市场区分调整 → **东证 PRIME 8584** |
| 2024〜 | 中期经营计划（公开部分）启动 |

### 集团内定位

[[megabanks/mufg]] 集团内的消费金融·信用卡·信贷销售 3 轴：

| 职能 | 负责公司 | 从 [[megabanks/mufg]] 的定位 |
|---|---|---|
| 超大型银行·卡 | 三菱 UFJ 尼克斯 | 100% 合并子公司·三大信用卡 |
| 消费金融（无担保） | Acom | 权益法 39.6%·东证 STD 8572 |
| **信贷销售·单品分期** | **Jaccs** | **权益法适用关联公司·东证 PRIME 8584** |

也就是说，Jaccs 处于承担「[[megabanks/mufg]] 以直接银行融资难以覆盖的**销售金融＝经由加盟店的分期**」的位置。

### 竞争结构

| 竞争对手 | 优势 | 与 Jaccs 之别 |
|---|---|---|
| [[card-issuers/orico]] | 业界最大手·伊藤忠 + 瑞穗系列 | 规模上领先，Jaccs 在北海道·汽车·教育方面抗衡 |
| [[card-issuers/credit-saison]] | 卡 × 存量型·流通系 | 偏重卡，Jaccs 在分期方面更厚 |
| 永旺金融 | 流通系·WAON | 依赖流通基础，Jaccs 加盟店多样 |
| 乐天卡 | 数字·EC | 数字轴领先，Jaccs 以线下加盟店抗衡 |

### 中期战略（公开的方向性）

- **3 大贷款（汽车·教育·装修）的深耕**
- **加盟店网络扩大**: 经销商·教育机构·住宅装修公司
- **数字申请流程的扩充**: 从纸质授信走向 Web 完结
- **ASEAN 消费金融**: 经由当地法人的两轮·耐用品贷款
- **作为 MUFG 系列的补充职能**: 与 [[megabanks/mufg]] 银行·尼克斯·Acom 的角色分工

## 5. 规制·政策

- **主管**: 金融厅（FSA）
- **登记**: 依据分期销售法·贷金业法的登记业者
- **业界团体**: 加入日本信用协会（JCA）
- **主要规制**:
  - 分期销售法: 综合信用购买中介业者·个别信用购买中介业者
  - 贷金业法: 总量规制（年收入 1/3 规制）对象
  - 个人信息保护法·修正资金结算法（加密资产相关不在对象内）
- **近期政策议题**:
  - 2024〜 向授信模型导入 AI / 机器学习
  - 无现金化政策（卡手续费·加盟店规制）
  - 2025〜 修正分期销售法·本人确认的高度化
