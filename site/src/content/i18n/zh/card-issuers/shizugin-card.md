## TL;DR

株式会社しずぎんカード(Shizugin Card / Shizuoka Bank Card)是 [[regional-banks/shizuoka-fg|しずおか FG (東証 PRIME 5831)]] 集团的信用卡子公司,依托核心子公司 [[regional-banks/shizuoka-bank|静岡銀行]](静冈县份额第 1  位 / 制造业集聚地的 main bank)的个人 / 法人客户基础,提供**以静冈县为地盘的银行发行信用卡**(JCB / VISA / Master 品牌)、贷款、现金借贷(cashing)、代收、电子货币、QR 支付相关服务。作为地方银行系信用卡公司的典型 model,承担银行账户联动、区域加盟店网络、区域经济圈内结算的功能。[[regional-banks/shizuoka-fg]] 2022-10-03 改为控股公司化之后,被重新配置为集团并列子公司中的一员。^[extracted]

### 强项・弱项

- 强项:[[regional-banks/shizuoka-bank|静岡銀行]] 静冈县份额第 1  位 + 存款客户基础 + 与制造业员工工资代发账户联动 + 区域加盟店网络 + 制造业集聚地(铃木 / 雅马哈等)的业主 / 高管富裕阶层

### 监管框架

- **主管**:经济产业省(分期付款销售法主管)・金融厅(FSA,放贷业法・资金结算法主管)
- **法律依据**:
  - **分期付款销售法(割贩法)**:信用卡发行・购物信贷业务的主体监管(综合信用购买斡旋业者登记)
  - **放贷业法**:现金借贷・卡贷款业务(放贷业登记)
  - **资金结算法**:电子货币 / 预付卡 / QR 支付相关业务
  - **个人信息保护法 / My Number 法**:卡会员信息管理
- **母 FG 治理**:作为 [[regional-banks/shizuoka-fg]] 并表子公司,处于控股公司治理之下,按银行业以外子公司监管(主要业务 / 附随业务 / 业务范围监管)进行定位

### 监管议题

- **割贩法修订(2018-)**:强化加盟店签约方(acquirer)义务、安全措施义务(符合 PCI DSS)
- **修订割贩法(2020-)**:强化用户保护(授信审查・使用上限管理・投诉处理体制)
- **个人信息保护法(2022- 修订)**:信用卡会员信息的跨境转移・适当取得义务
- **无现金推进**:在经产省无现金愿景 2025・"2025 年无现金比率 40%"目标下,围绕加盟店手续费水平的争论
- **反洗钱对策・反社会势力核查**:与 [[regional-banks/shizuoka-fg]] 集团共同实施
- **EDINET / 有价证券报告书**:静银卡单体非上市、披露有限,经 [[regional-banks/shizuoka-fg]] 并表分部信息间接披露

### 会计上的处理

- **单体财务 KPI**:在母 [[regional-banks/shizuoka-fg]] 有价证券报告书附注 / 分部信息中部分披露(资本金 5,000 万円・员工 81 名 = 静银卡官方公司介绍)
