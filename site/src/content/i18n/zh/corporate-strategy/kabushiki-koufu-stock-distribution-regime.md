# 株式交付（kabushiki koufu）— 日本《公司法》2021  修订下的股份换股份收购制度

## Wiki 路由

本词条归属于 [[corporate-strategy/INDEX|corporate-strategy INDEX]]，并接入 [[finance/INDEX|finance INDEX]] 以提供交易金融背景。配合 [[corporate-strategy/japan-kaisha-bunkatsu-tax-regime|会社分割 tax regime]] 阅读以对比分割机制重组，[[corporate-strategy/partial-spinoff-tax-deferral|partial spinoff tax deferral]] 了解分配侧的平行机制，[[corporate-strategy/spinoff-decision-tree-japan|spinoff decision tree Japan]] 了解期权集背景，[[finance/japan-tender-offer-process|Japan tender offer process]] 了解公开要约的替代方案。

## TL;DR

**株式交付（kabushiki koufu）** 是日本《公司法》于 **2021  公司法修订**（2021-03-01 生效）中引入的机制，使一家日本股份公司能够**以自身股份作为对价**收购另一家日本股份公司的股份，同时将标的保留为子公司而非全资子公司。收购方直接向标的的出售股东发行新股，以换取标的股份。

它位于两项既有制度**之间**：

- **株式交換（kabushiki koukan，股份交换）** — 使标的成为**全资**子公司；要求取得 100%
- **株式移転（kabushiki iten，股份转移）** — 在既有实体之上创设一家**新控股公司**；仅用于控股公司化

株式交付 填补了空白：它使收购方在希望取得控股权（如 50-90%）而不强制 100% 所有权时，能够进行以股份为对价的收购。在规定条件下的税务处理允许**出售股东递延资本利得**（仅限对价中的股份部分），使其在精神上经济等同于美国的"B 类重组"。

## 1. 为何创设 2021  制度

2021  之前的问题：一家日本收购方若想用自身股份收购日本标的，会遇到一个程序陷阱：

- 株式交換 要求取得标的的 100% — 在偏好保留少数股东时过于激进
- 向出售股东直接发行股份（实物增资）很复杂；卖方的税务处理不确定
- TOB + 现金对价可行，但需要收购方资产负债表上的现金
- TOB + 股份交换（公开买付 + 株式交換）是一个 2 步骤的序列，存在时点与税务摩擦

结果：交叉持股的解除、友好的分立收购，以及希望以股份为对价的 PE 支持的整合（roll-up），都被迫采用别扭的结构。法务省的 2021  改革明确创设了 **株式交付** 来填补空白。

## 2. 机制

| 要素 | 细节 |
|---|---|
| 收购方 | 日本股份公司（上市股份公司或股份公司） |
| 标的 | 日本股份公司 |
| 对价 | 收购方自身股份（可混合现金 / 债券，但税务递延要求须含股份部分） |
| 交付后所有权 | 标的成为**子公司**（要求 >50%，可少于 100%） |
| 决议 | 收购方股东大会特别决议（2/3  门槛） |
| 出售股东 | 各自独立决定是否要约；**并非要求全体股东参与** |
| 税务 — 出售股东 | 对价中的股份部分**递延**资本利得（满足规定条件时） |
| 税务 — 收购方 | 处理上类似合格重组 |
| 申报 | 若适用 FIEA 披露则进行证券申报；普通 M&A 申报 |

## 3. 三种股份侧重组制度的比较

| 制度 | 结果 | 是否要求股份换股份 | 最低 % | 用例 |
|---|---|---|---|---|
| **株式交換（kabushiki koukan）** | 标的成为全资子公司 | 是（或允许现金 / 混合，但有不利） | 100% | 以股份为对价的完全收购 |
| **株式移転（kabushiki iten）** | 在既有实体之上形成新控股公司 | 是 | 100%（各被转移实体） | 纯控股公司化或对等合并 |
| **株式交付（kabushiki koufu）** | 标的成为（受控）子公司；非全资 | 是（税务递延需股份部分） | >50% | 以股份为对价的部分控股收购 |

决策树：

- 想要 100% 所有权 + 股份对价 → **株式交換**
- 想转换为控股公司结构 → **株式移転**
- 想要控股权（50-99%）+ 股份对价 → **株式交付**

资产层面（而非股份层面）的重组，见 [[corporate-strategy/japan-kaisha-bunkatsu-tax-regime|会社分割]]。

## 4. 税务处理机制

出售股东收到收购方股份外加可能的现金 / 债券。税务处理拆分如下：

| 对价部分 | 对卖方的税务处理 |
|---|---|
| 股份部分（收到的收购方股份） | 若满足条件则**递延**资本利得 |
| 现金 / 债券部分 | 按比例应税（适用视同股利或资本利得规则） |

股份部分递延的条件包括：

- 收购方发行其**自身股份**（非子公司股份）— 直接发行
- 标的成为收购方的子公司（交易后 >50%）
- 指定股东的持续性规则
- 视情况向 NTA / METI 申报的要求

该机制在精神上类似美国 IRC §368(a)(1)(B) 的"B 类重组"——股份换股份收购，股份部分的利得递延。

## 5. 实务用例

| 用例 | 为何 株式交付 适用 |
|---|---|
| **友好的部分控股收购** | 收购方想要 60-80% 所有权而不强制 100% 挤出 |
| **经由交叉持股转换的战略联盟** | 将相互的少数持股转换为以股份为对价的控股权 |
| **PE 整合（roll-up）合并** | 主投资者经股份对价取得标的控股权，保留少数股东参与 |
| **IPO 前整合** | 母公司在 IPO 前将关联实体整合入子公司结构 |
| **交叉持股解除** | 以更清晰的控股结构替换交叉持有的历史头寸 |
| **外国收购方适配** | 外国收购方的日本子公司用 株式交付 吸收额外的日本业务 |

注意：株式交付 **仅适用于日本股份公司之间**。跨境股份换股份收购通常仍使用 [[finance/cross-border-m-a-japan|cross-border M&A Japan]] 结构，而非直接使用 株式交付。

## 6. 程序要求

《公司法》第 774-2  条至 774-11 条下的 株式交付 程序：

1. **收购方编制 株式交付計画（股份交付计划）**，明确标的、对价比率、条款
2. **收购方董事会批准计划**
3. **在收购方总部披露计划**，股东大会前 2  周
4. **收购方股东特别决议** 批准计划（2/3  门槛）
5. **劝诱标的股东** 要约其股份（各自独立决定）
6. **要约的标的股东同意条款**
7. **收购方向要约的标的股东发行新股**
8. **收购方取得被要约标的股份的所有权**
9. **结果：若要约股份 >50% 则标的成为子公司；若 <50%，则 株式交付 失败**

若未达门槛，整个交易被撤销——存在类似 TOB 最低要约条件的"最低要约"风险。

## 7. 与 TOB 加挤出路径的比较

| 维度 | 株式交付 | TOB + 挤出 |
|---|---|---|
| 对价 | 收购方股份（带税务递延） | 现金（通常） |
| 所有权结果 | 部分控股（>50% 至 <100%） | 完全所有权（经挤出达 100%） |
| 对卖方税务 | 递延（股份部分） | 实现（现金的资本利得） |
| 批准门槛 | 收购方股东 + 各卖方个别决定 | 依 [[finance/japan-tender-offer-process|TOB process]] + [[finance/japan-mbo-and-squeeze-out-process|squeeze-out]] |
| 时间线 | 单一一体化程序 | 多步骤（TOB → 挤出） |
| 用例 | 以股份为对价的战略收购 | 现金出资的完全收购 |

TOB 侧机制见 [[finance/japan-tender-offer-process|Japan tender offer process]]，TOB 后私有化路径见 [[finance/japan-mbo-and-squeeze-out-process|Japan MBO / squeeze-out process]]。

## 8. 与 会社分割 + 株式分配（分拆路径）的比较

株式交付 是一种**收购**机制（买入标的）。会社分割 + 株式分配（[[corporate-strategy/japan-kaisha-bunkatsu-tax-regime|kaisha bunkatsu]] + [[corporate-strategy/partial-spinoff-tax-deferral|partial spinoff]] 路径）是一种**剥离**机制（分离既有业务）。它们占据公司重组图谱的相反两侧。

| 方向 | 机制 |
|---|---|
| 用自身股份取得对独立实体的控制 | 株式交付（本词条） |
| 用自身股份取得独立实体的 100% | 株式交換 |
| 在既有实体之上形成新控股公司 | 株式移転 |
| 通过拆入子公司剥离业务 | 会社分割（新设分割后分配） |
| 在保留少数股权下剥离子公司 | 部分分拆（带制度的 kabushiki bunpai） |
| 不保留任何股权剥离子公司 | 分拆（kabushiki bunpai 完全分配） |
| 经现金取得 100% | TOB + 株式併合 挤出 |

## 9. 反论点

- 株式交付 自 2021  引入以来相对替代路径采纳有限——部分因为现金出资的 TOB 仍是日本 M&A 的主导模式
- 税务递延条件具技术性；需顾问驱动的结构设计
- 即便程序上可行，收购方股份稀释对现有股东也可能不可接受
- 外国收购方一侧不能直接使用 株式交付（须经日本子公司或替代结构）
- "低于 50% 即失败"的触发条件造成了纯 TOB 所没有的执行风险（在某些结构中，TOB 仍可在低于目标要约量时执行）

## 10. 待解问题

- 随着越来越多日本上市公司将 株式交付 视为完全收购 TOB 的更温和替代，其采纳是否会增加？
- 株式交付 将如何与未来跨境 M&A 改革交互——能否为外国收购方创设一个平行机制？
- 对需要部分现金 + 股份对价的交易，最优混合结构是什么？
- 该制度如何与收购后的少数股东保护（压迫救济、异议股东评估权）交互？
- 该制度是否会在 PE 支持的日本收购中被用作保留创始人 / 管理层股权的方式？
