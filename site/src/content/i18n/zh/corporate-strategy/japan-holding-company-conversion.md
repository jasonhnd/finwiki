# 持株会社化 (日本控股公司化) — 株式移転 / 株式交換 / 会社分割 (抜け殻方式) 通往纯粹型与事业型控股公司结构的路径

## 维基路径

本条目位于 [[corporate-strategy/INDEX|corporate-strategy INDEX]] 之下，并为集团资本叠加层路由至 [[finance/INDEX|finance INDEX]]。它是一份**结构选择指南**，而非制度页面：它在 [[corporate-strategy/japan-kabushiki-koukan-iten-regime|株式交換 / 株式移転 regime]] 与 [[corporate-strategy/japan-kaisha-bunkatsu-tax-regime|会社分割 tax regime]] 所记载的各制度之间进行抉择。请与 [[corporate-strategy/spinoff-decision-tree-japan|the spinoff decision tree]] 一并阅读以了解其分拆侧的对应物，并与 [[finance/japan-listed-financial-groups-investable-universe|the listed-financial-groups universe]] 一并阅读以理解为何如此多的日本集团坐落于一家控股公司之下。

## TL;DR

**持株会社化 (mochikabu-gaisha-ka, holding-company conversion /「ホールディングス化」)** 是指对一个集团进行重组，使得一家 **控股公司 (持株会社)** 居于顶端，并将各经营业务作为子公司持有的行为。它本身并非《会社法》上的一种机制 — 它是通过三种底层机制之一所达成的*目标*。

两项结构性选择界定了任何一次转换:

1. **控股公司的类型** — **純粋持株会社 (纯粹型控股公司)**，仅持有子公司股份并依靠股息生存，对比 **事業持株会社 (经营型控股公司)**，其持有子公司*且*自营业务。
2. **转换方法** — **株式移転 (股份转移)**、**株式交換 (股份交换)**、或 **会社分割 (公司分割,「抜け殻方式」/ 空壳方式)**。

标准的映射关系:

- **株式移転** → 公司将*自身*置于一家新设立的纯粹型控股公司之下。是单一上市公司通往纯粹型控股公司最常见的路径。
- **株式交換** → 一家既存公司成为目标公司的 100% 母公司 (经营型或纯粹型) — 当未来的控股公司已经存在时使用。
- **会社分割 (抜け殻方式)** → 经营公司*将其业务剥离*至一家子公司，留下**原法律实体作为控股公司空壳**(「抜け殻」，空壳)。在顶端保全原实体的上市地位、牌照与历史。

由于日本在 **1997 年解除了战后对纯粹型控股公司的禁令**(独占禁止法修订)，持株会社化成为集团设计的主流工具，并构成日本上市集团中大量「○○ホールディングス」名称的底层基础。

## 1. 纯粹型 vs 经营型控股公司

| | 純粋持株会社 (纯粹型) | 事業持株会社 (经营型) |
|---|---|---|
| 自营业务 | 无 — 仅持有并管理子公司 | 经营业务*且*持有子公司 |
| 收入 | 子公司股息 + 集团管理费 | 营业收入 + 股息 |
| 典型名称 | 「○○ホールディングス」/「○○グループ本社」 | 原经营公司，如今亦为母公司 |
| 优势 | 清晰的集团治理、中立的资本配置者、灵活的附加式 M&A | 设立成本较低；无需新设顶端实体 |
| 权衡 | 新实体、集团减免 / 连结纳税方面的考量、仅股息的现金流 | 母公司自营业务与集团监督之间的利益冲突 |

纯粹型控股公司模式在集团希望拥有一个**中立顶点**、在相互竞争的经营子公司之间配置资本而不偏袒任何一方时受到青睐 — 这是 [[business/japan-listed-corp-strategic-restructuring-matrix|the listed-corp strategic-restructuring matrix]] 所汇集的战略重组案例中反复出现的主题。

## 2. 三种转换方法

### 方法 A — 株式移転 (股份转移): 搭建一个新屋顶

经营公司执行一次单一公司的 [[corporate-strategy/japan-kabushiki-koukan-iten-regime|株式移転]]：**新设立一家纯粹型控股公司**，经营公司成为其全资子公司，股东将经营公司股份换为控股公司股份 (通常为一对一)。

- **结果**: 通往纯粹型控股公司最清晰的路径。
- **上市**: *新*控股公司取代经营公司上市 (技术性再上市 / 上场维持依交易所规则处理)。
- **多公司变体**: 两家公司同时进行株式移転并入一家控股公司 — [[JapanFG/INDEX|JapanFG INDEX]] 中许多金融集团背后的共同控股 / 对等合并模式。

### 方法 B — 株式交換 (股份交换): 让一家既存公司朝上指向

当意图中的控股公司 (或收购方) **已经存在**时，一次 [[corporate-strategy/japan-kabushiki-koukan-iten-regime|株式交換]] 使其成为目标公司的 100% 母公司。若收购母公司继续经营自身业务，这通常产生一家 **事業持株会社**，或将一家既存控股公司延伸覆盖至一家新子公司。

### 方法 C — 会社分割 (抜け殻方式 / 空壳方式): 掏空经营公司

原公司执行一次 [[corporate-strategy/japan-kaisha-bunkatsu-tax-regime|新設分割]] (或吸収分割)，将其**全部**营业业务转移至一家新设立的 (或既存的) 子公司。**原法律实体留在顶端作为控股公司**，如今仅持有该经营子公司的股份 — 一个「空壳」(抜け殻)。

- **关键优势**: 顶端实体**保持不变**，因此其证券交易所上市地位、监管牌照、合同以及公司历史全部留在控股公司层级，无需经历再上市事件。
- **员工承继**: 由于使用会社分割，雇佣合同依据 労働契約承継法 **自动**承继 (伴随法定协商程序) — 此承继机制详见 [[corporate-strategy/japan-kaisha-bunkatsu-tax-regime|the 会社分割 regime]]。
- **权衡**: 分割必须满足 適格 测试方可获递延课税，且债权人保护 / 员工协商的时间表适用。

## 3. 方法选择矩阵

| 目标 | 最佳方法 | 原因 |
|---|---|---|
| 单一上市公司 → 纯粹型控股公司，清晰 | **株式移転** | 创建中立顶点；股东只需滚入新控股公司 |
| 在顶端保留*原*实体 (其上市地位 / 牌照) | **会社分割 (抜け殻方式)** | 顶点永不改变；只有业务下移 |
| 未来母公司已存在；将目标吸收至 100% | **株式交換** | 让既存公司朝上指向目标之上 |
| 两家公司在一个中立母公司之下结合 | **株式移転 (joint)** | 并行转移至单一新控股公司 (对等合并) |
| 仅部分控制 (非 100%) — *并非*完整的控股公司化 | **株式交付** | 参见 [[corporate-strategy/kabushiki-koufu-stock-distribution-regime|株式交付 regime]]；止步于 100% 之下 |

## 4. 税务层 — 適格 处理贯穿始终

控股公司化只有在**税务中性**时才具吸引力，这意味着底层机制必须合格 (適格)。各项测试为标准的 組織再編成 测试:

- **株式移転 / 株式交換**: 单一公司转换通常是一项 **100% 集团 (完全支配関係)** 交易 — 最轻的测试集 — 因此 適格 (递延课税) 处理是常规操作。测试细节参见 [[corporate-strategy/japan-kabushiki-koukan-iten-regime|the 株式交換 / 株式移転 regime]]。
- **会社分割 (抜け殻方式)**: 向子公司的分割必须满足 適格分割 测试 (全资集团分割再次是最轻的情形)，使资产维持结转基准。亏损结转的反流转规则 (法人税法第 57 条以下) 在所有权变更测试触发时可能咬合。

一旦控股公司存在，持续的集团课税 (**グループ通算制度** 连结集团制度、母子公司之间的受领股息处理) 便成为一项现行的设计因素 — 贯穿 [[corporate-strategy/INDEX|this domain]] 的税务对结构的反复张力。

## 5. 集团为何转换 — 战略驱动因素

- **集团治理与资本配置**: 一个中立顶点可依据组合逻辑而非内部政治，在经营子公司之间转移资本。
- **M&A 敏捷性**: 附加式收购作为控股公司之下的新子公司就位，不扰动既有经营实体；其融资位于 [[finance/japan-acquisition-finance|Japan acquisition finance]]。
- **事业承继**: 对于业主经营的公司，介入一家控股公司可集中控制权并在世代交接之前重塑股份基础 — 与承继税制框架的相互作用见 [[corporate-strategy/japan-business-succession-jigyou-shoukei|business succession (事業承継)]]。
- **风险围栏**: 分立的经营子公司隔离负债；其中一家的问题不会直接污染顶点或姊妹公司。
- **受监管行业结构**: 在金融领域，控股公司模式与 FSA 之下的银行 / 保险控股公司监督相契合 — 这是 [[finance/japan-listed-financial-groups-investable-universe|the listed-financial-groups universe]] 中的实体如此组织的首要原因。

## 6. 监管与反垄断层

- **独占禁止法**: 纯粹型控股公司**直至 1997 年修订前一直被禁止**；如今获得许可，但日本公正交易委员会仍监管那些会造成过度集中的集团结构 — 合并管控叠加层见 [[finance/jftc-merger-control-process|the JFTC merger-control process]]。
- **行业控股公司规则**: 银行 / 保险控股公司处于专门的 FSA 监督 (持株会社 認可) 之下，叠加于通用的《会社法》机制之上。
- **上市连续性**: 在株式移転下交易所处理新顶点的技术性再上市；在会社分割 (抜け殻方式) 下顶点的上市地位获得保全 — 这一区别与 [[securities/japan-ipo-listing-disclosure-route|the listing / disclosure route]] 相互作用。

## 7. 反论与注意事项

- **控股公司并非免费**: 一个新顶点增加了一层管理、董事会与审计成本，而纯粹型控股公司层级仅股息的现金流需要刻意的集团股息政策。
- **適格 必须确认**: 税务中性取决于满足所选机制的合格测试；现金对价或失败的连续性测试可使转换变为应税。
- **事業持株会社 中的冲突**: 一家经营型控股公司可能面临经营自身业务与监督子公司之间的冲突 — 许多集团最终转向纯粹型控股公司的一个原因。
- **方法是手段，而非目的**: 持株会社化是*结果*；现行的抉择是使用 [[corporate-strategy/japan-kabushiki-koukan-iten-regime|株式移転 / 株式交換]] 还是 [[corporate-strategy/japan-kaisha-bunkatsu-tax-regime|会社分割]] 来抵达彼处。
