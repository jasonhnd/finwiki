# SBI FX交易股份有限公司（BI FXTRADE）— 日本加密资产衍生品业者 overview

This entry sits under [[exchanges/INDEX|exchanges index]]. Read it with [[exchanges/fsa-vasp-registration-system|FSA 暗号資産交換業登録制度 — 番号体系・財務局管轄・登録要件]] for adjacent context and [[fintech/japan-financial-regulation|日本金融規制 — トークン・暗号資産・決済に関する法体系]] for the broader system boundary.

> FSA 关东财务局长（金商）第2635号 · JVCEA 第一种会员 #1026（**仅加密资产衍生品交易**） · 设立 2011年11月

## 1. 法人・股东

- **商号（日）**：SBI FXトレード株式会社（JVCEA 登记名：BI FXTRADE Co., Ltd.）
- **商号（英）**：SBI FXTRADE Co., Ltd.
- **设立**：2011年11月
- **总部**：〒106-6019 东京都港区六本木1-6-1 泉花园塔（与 SBI VC 交易・SBI 证券同地址）
- **资本金等**：9億6,000万円（含资本公积金）
- **股东**：SBI 流动性市场股份有限公司 100%（SBI HD 合并子公司）
- **上市状态**：非上市（SBI HD 集团全资子公司）

**注记**：JVCEA 会员一览中标注为 "BI FXTRADE Co., Ltd."，但其公司网站・金商登记（第2635号）・所在地（泉花园塔）一致，确认与 SBI FX交易股份有限公司为同一法人。

## 2. 牌照・登记状况

- **金融商品交易业**：关东财务局长（金商）第2635号（第一种金融商品交易业）
- **加密资产交换业（FSA）**：**无登记**（不持有现物交易・交换业牌照）
- **JVCEA 第一种会员 #1026**：仅加密资产衍生品交易（Cryptocurrency derivatives trading）
- **加入协会**：一般社团法人金融期货交易业协会（会员编号 1588）、一般社团法人日本加密资产等交易业协会（会员编号 1026）

## 3. 经办通货

- **经办币种数**：6币种（仅衍生品。无现物交易）
- **币种**：BTC/JPY, XRP/JPY, ETH/JPY, BTC/USD, XRP/USD, ETH/USD
- **特色**：同时提供对日元、对美元的加密资产 CFD（差价结算）。无现物持有・无钱包。定位为 FX 主业者附加的加密资产衍生品服务。

## 4. 业务范围

该公司是 FX（外汇保证金交易）专营公司，加密资产作为在 FX 同一账户基础上附加的衍生品服务（**SBI 比特币 FX**）提供。

| 服务 | 提供状况 |
|---------|---------|
| FX（场外外汇保证金交易） | ✓ 主业・34货币对 |
| 积存外币（积立 FX） | ✓ 外币积立服务 |
| SBI 比特币 FX（加密资产 CFD） | ✓ 6币种・杠杆最高2倍（个人） |
| 加密资产现物交易 | ✗ 无 |
| NEOBANK | ✓ 银行代理业（住信SBI网络银行） |
| 加密资产借贷・NFT・staking | ✗ 无 |
| 电子结算手段等交易业 | ✗ 无 |

**入金渠道（加密资产 CFD）**：快捷入金（住信SBI网络银行、乐天银行、三菱UFJ银行、三井住友银行、邮储银行、瑞穗银行、PayPay银行）、由 SBI FX交易 NEOBANK 即时入金。**无便利店付款・无虚拟货币入金**。

**加密资产 CFD 主要规格**：交易种类＝场外加密资产保证金交易，杠杆＝个人账户2倍（法人另议），强制平仓水准＝保证金维持率50%以下，交易手续费・强制平仓手续费・账户维持费 全部免费，资金费率浮动制。

## 5. 市占・成交量

- **成交量・账户数**：未公布（加密资产 CFD 单体无定期披露）
- **客户层**：主要为 FX 既有客户（以零售为主）。无面向机构投资者专用计划的记载。
- **定位**：加密资产 CFD 的成交量排名未公布。FX 部门主打「价差业界最窄水准」「Oricon 客户满意度 FX 初学者部门 第1位」。以 FX 专营及 SBI 集团知名度为背景，客户基础主要为小额 FX 交易者层。

## 7. 战略动向

- **2026年4月**：「外币宅配服务」开始（面向 FX 账户客户的外币现金配送）
- **2026年4月**：设立15周年特设页面公开，开展周年活动
- **2026年5月**：新货币对（土耳其里拉/日元）的基准价差缩小（1.28钱）
- 加密资产 CFD 自 2020年8月开始以来，币种扩充（BTC→XRP→ETH 的顺序）已完成，对美元币种也已追加。最近12个月内未确认加密资产部门的新扩张发布。
- SBI 集团整体的加密资产战略由集团统括公司（[[JapanFG/sbi-hd|SBI HD]]）及 [[exchanges/jp-exchange-sbi-vc-trade|SBI VCトレード]] 主导，该公司定位为零售 FX 基础上的补充。JP VASP 的衍生品・现物兼营结构参见 [[exchanges/jp-vasp-derivative-license-system|JP VASP デリバティブライセンス制度]]。
