# 株式会社 Digital Asset Markets — 日本加密资产交换业者 overview


## Wiki route

本条目位于 [[exchanges/INDEX|交易所索引]]。可与 [[exchanges/fsa-vasp-registration-system|FSA 加密资产交换业登记制度]] 和 [[fintech/japan-financial-regulation|日本金融监管]] 配套阅读。

> FSA 登记号 关东财务局 第00024号 · JVCEA 会员 #1029 · 登记日 2021-02-17

## 1. 法人・股东

- **商号**: 株式会社 Digital Asset Markets（英文：Digital Asset Markets, Inc.）
- **设立年月日**: 2018年8月1日（旧称：InterTrade Coin 株式会社）
- **总部所在地**: 〒102-0082 东京都千代田区一番町18番地 川喜多 Memorial Building8层
- **资本金**: 29億6,749万円（截至 2025年12月5日）
- **决算日**: 3月末
- **上市状态**: 非上市。多家上市企业・大型企业列名为股东

**主要股东**（已公开部分）：InterTrade（东证标准板）、三井物产、日本交易所集团（JPX）、Monex 集团、Japan Investment Adviser、bitFlyer Blockchain、Keyware Solutions、NEC Capital Solutions、大发、Seven Bank、爱媛银行、东京短资、光证券、赤木屋 HD、丰证券、日产证券、山和证券、广田证券、Astmax、WOW WORLD、MIRAITRONC 投资事业有限责任组合 等

来自三井物产（2019年8月）・日本交易所集团（2022年2月）的资本参与值得特别关注。

## 2. 牌照・登记状况

- **加密资产交换业**: 关东财务局长 第00024号（登记完成：2021年2月）
- **衍生品交易业**: 未取得（作为 JVCEA 第一种会员仅经办加密资产交换业）
- **电子结算手段等交易业**: 未取得
- **JVCEA 会员种类**: 第一种会员（会员编号 1029）
- **Fintech 协会**: 法人会员
- **日本数字分散型金融协会**: 正式会员（2026年4月入会）
- **ISMS 认证**: JIS Q 27001:2025（ISO/IEC 27001:2022）取得（2024年8月）
- **隐私标志**: 取得（2025年7月、登记编号 10670061）

## 3. 经办币种

经办币种为 3 种（截至 2026年5月），均为该公司独自发行・管理的 Zipangcoin 系列：

- **ZPG（Zipangcoin）**: 旨在与金（GOLD）价格联动的加密资产（2022年2月开始经办）
- **ZPGAG（Zipangcoin Silver）**: 与银（SILVER）价格联动（2023年7月开始经办）
- **ZPGPT（Zipangcoin Platinum）**: 与铂金（PLATINUM）价格联动（2023年7月开始经办）

不经办 BTC・ETH 等主要山寨币。在国内属于罕见的、专注于贵金属价格联动型加密资产的利基定位。

## 4. 业务范围

- **销售所形式**: 仅 ZPG・ZPGAG・ZPGPT 的现货交易（spot）。不提供交易所（板交易）形式
- **保证金交易（衍生品）**: 无
- **加密资产借贷 / staking**: 未公开
- **NFT**: 未公开
- **ST（证券型代币）事业**: 在公司网站上将「ST 事业」明示为事业领域。作为支持发行・流通・募资・管理的平台事业展开中（详情未公开）
- **网络安全事业**: 以「Digital Shelter」品牌另行提供数据保全服务
- **法定货币入金渠道**: 银行转账（Resona 银行・PayPay 银行・GMO Aozora 网络银行）、GMO Aozora 网络银行的「银行入金」、PayPay 银行的「Link 结算」
- **Passkey 认证**: 预定 2026年5月下旬引入（在重要通知中公开）

加密资产的管理为 100% 冷钱包，金钱通过向日证金信托银行的金钱信托进行分别管理。

## 5. 市占・交易量

- **交易量・账户数**: 未公开
- **客户层**: 以个人 retail（iPhone 应用 + 网页开户流程）为主体。面向法人・机构投资者的服务未明确确认提供
- **竞争内定位**: 在国内 26社 家加密资产交换业者中属于 niche 玩家。不经办 BTC・ETH 等主要币种，专注于贵金属联动币 3 种，面向不同于一般交易所的客户需求

## 6. 沿革・重大事件

| 年月 | 事项 |
|------|------|
| 2018-08 | 以「InterTrade Coin 株式会社」名义设立 |
| 2019-08 | 接受三井物产出资 |
| 2020-03 | 加入日本加密资产等交易业协会（JVCEA） |
| 2021-02 | 加密资产交换业者登记完成（关东财务局长 第00024号） |
| 2022-02 | 开始经办 Zipangcoin（ZPG）的销售所服务 |
| 2022-02 | 接受日本交易所集团（JPX）出资 |
| 2022-06 | 加入 Fintech 协会 |
| 2023-07 | 开始经办 ZPGAG（银）・ZPGPT（铂金） |
| 2024-08 | 取得国际规格 ISMS 认证（JIS Q 27001:2025） |
| 2025-03 | 协助 InterTrade 与 Fireblocks 的战略合作伙伴关系 |
| 2025-07 | 取得隐私标志 |
| 2026-04 | 加入日本数字分散型金融协会 |

未确认有行政处分・黑客攻击・重大系统故障的公开案例。

## 7. 战略动向

- **Fireblocks 联动**（2025年3月）：协助母公司级别的 InterTrade 与 Fireblocks 的战略合作伙伴关系。暗示托管技术的高度化。
- **ST（证券型代币）事业**：在公司网站上将其列为事业领域，标榜构建有形资产权利的数字化・发行・流通平台。具体项目・规模未公开。
- **引入 Passkey 认证**（预定 2026年5月下旬）：强化用户认证的安全性。
- 由于拥有 JPX・三井物产・Monex 集团等大型金融机构作为股东，具备机构联动的基础，但公开的 retail 扩张计划现阶段尚未确认。

## 8. 相关人物

- **代表董事・西本一也（Kazuya Nishimoto）**: 出身日本劝业角丸证券（现瑞穗证券）。1999年设立 InterTrade（2004年东证 Mothers 上市）。2018年设立本公司。证券用前台系统开发的专家。
- **常勤董事・加藤次男（Tsuguo Kato）**: 出身三井物产（历任美国・英国 Mitsui Bussan Commodities CEO/COO 等）。2023年12月从三井物产退休后任现职。商品交易专家。
- **社外董事・森中宽（Hiroshi Morinaka）**: 光证券 代表董事社长（现任）。2019年12月就任。
- **社外董事・白岩直人（Naoto Shiraiwa）**: Japan Investment Adviser 代表董事（现任）。2020年6月就任。
- **社外董事・和歌伸介（Shinsuke Waka）**: 三井物产企业发展本部参与（现任）。2024年1月就任。
- **常勤监查役・和木克己（Katsumi Waki）**: 出身东京银行等金融机构。2018年11月就任。
- **监查役（非常勤）・山本晃（Akira Yamamoto）**: 原大藏省・日本银行理事。2018年11月就任。
