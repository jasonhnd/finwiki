# BTCBOX 株式会社 — 日本加密资产交换业者 overview


## Wiki route

This entry sits under [[exchanges/INDEX|exchanges index]]. Read it with [[exchanges/fsa-vasp-registration-system|FSA 暗号資産交換業登録制度 — 番号体系・財務局管轄・登録要件]] for adjacent context and [[fintech/japan-financial-regulation|日本金融規制 — トークン・暗号資産・決済に関する法体系]] for the broader system boundary.

> FSA 登记号 关东财务局 第00008号 · JVCEA 会员 #1008 · 登记日 2017-09-29

## 1. 法人·股东

- **商号**:BTCボックス株式会社
- **英文名**:BtcBox Co.,Ltd.
- **设立**:2014年3月6日(David Zhang 以资本金500万円设立)
- **总店**:〒103-0025 東京都中央区日本橋茅場町2丁目8番1号 BRICK GATE 茅場町 5階(2018年1月17日从旧地址·京橋1丁目迁出)
- **资本金**:22億2704万円(含资本准备金)—— 2019年以后增资后的公布数值
- **母公司**:株式会社TTXHoldings(2019年3月David Zhang 转让全部股份使其成为完全母公司。TTXHoldings 自身为非上市的非公开公司)
- **上市状态**:非上市
- **法人编号**:7020001104824(国税厅法人编号公布网站。编号指定 2015年10月5日)

## 2. 牌照·登记状况

- **加密资产交换业**:关东财务局长 第00008号(FSA,登记日 2017年9月29日)
- **JVCEA 会员**:第一种会员 #1008(仅加密资产交换业。未持有衍生品交易业相关的第二种相当牌照)
- **日本资金结算业协会**:会员编号 第00745号

## 3. 经办货币

- 经办币种数:7种(2026-05-18 时点,据官方博客沿革)
- 主要币种:BTC · ETH · BCH · LTC · DOGE(2022年4月追加)· DOT(2022年6月追加)· TRX(2022年7月追加)
- **特色**:以 BTC 为中心的 spot 专营交易所。BTC 从2014年4月服务开始之初即经办。BCH 拥有在国内率先于2017年8月开始交易的实绩。山寨币的追加节奏相对缓慢,维持小规模 spot 特化的方针。
- 贩卖所(かんたん売買):从2020年10月起开始服务

## 4. 业务范围

- **Spot 现货交易所**:交易所形式(挂单交易)
- **贩卖所(かんたん売買)**:2020年10月开始
- **衍生品 / 信用交易**:无(JVCEA 第一种·仅加密资产交换业)
- **加密资产融资(贷款)**:2016年2月开始。可借入最高为总资产3倍的 BTC 的服务
- **あんしん定期購入**:2021年9月开始(定投服务)
- **法定货币入金**:银行汇款(从住信SBI网络银行汇入手续费免费)、即时入金(2020年2月开始)
- **eKYC**:スピーディー本人確認 2020年4月开始
- **支持语言**:日语·英语·中文(3国语言)

## 5. 市占·交易量

- **账户数·月度交易量**:无公布(2026-05-18 时点)
- **过去的市场地位**:2014年4月服务开始后,几乎独占国内 BTC/JPY 交易量,持续国内1位直至2015年11月(Wikipedia 记述,出处为 bitcoin 日语信息网站)
- **2019年4月时点**:在 CoinMarketCap"调整后交易量·虚拟货币交易所 Top100"中,在日本的交易所中记录为1位(Wikipedia 记述,出处为 CoinMarketCap)
- **当前地位**:无交易量·账户数的最新公布数据。与 [[exchanges/jp-exchange-bitflyer|bitFlyer]]·[[exchanges/jp-exchange-gmo-coin|GMOコイン]]·[[exchanges/jp-exchange-coincheck|Coincheck]] 等大型机构相比,作为定位于 niche / mid 的独立系老牌在业界内获认知(参见 [[exchanges/jvcea-spot-volume-statistics-analysis|JVCEA スポット取引量統計分析]])。
- **客户层**:国内个人(海外个人·法人账户亦可开设)

## 6. 沿革·重大事件

| 年月 | 事项 |
|------|------|
| 2014-03 | BTCボックス株式会社设立(David Zhang,资本金500万円) |
| 2014-04 | BTC/JPY 交易所"BTCBOX"服务开始(同时也开始经办 LTC)。现存日本加密资产交易所中持续服务历史最悠久者之一 |
| 2015-07 | BTC 月度交易量超过50万BTC |
| 2016-02 | 加密资产融资服务开始 |
| 2017-08 | 在国内率先开始 BCH 交易 |
| 2017-09 | 作为加密资产交换业者完成 FSA 登记(第00008号) |
| 2017-11 | ETH 的经办开始 |
| 2017-12 | 迁至茅場町的新办公室 |
| 2018-10 | 株式会社TTXHoldings 设立(代表取缔役 David Zhang,资本金500万円) |
| 2018-10 | 表明参加 Liquid Network(Blockstream 的比特币侧链) |
| 2019-03 | David Zhang 将全部股份转让给 TTXHoldings → TTXHoldings 成为完全母公司 |
| 2019-05 | 母公司承购增资 2億円。社长更替:David Zhang 就任会长,辻治俊 就任新社长 |
| 2019-06 | 追加增资 3億円(母公司承购) |
| 2020-02 | 即时入金服务开始 |
| 2020-04 | スピーディー本人確認(eKYC)开始 |
| 2020-10 | かんたん売買(贩卖所)服务开始 |
| 2021-09 | あんしん定期購入 服务开始 |
| 2022-04 | DOGE 经办开始 |
| 2022-06 | DOT 经办开始 |
| 2022-07 | TRX 经办开始 |
| 2025-04 | 迎来服务开始10周年 |

**重大事件**:自2014年4月服务开始以来,未公布黑客等安全事件(官方博客·Wikipedia 的记述,2019年时点)。在公布记录中未确认有 FSA 的行政处分(2026-05-18 时点)。

## 7. 战略动向

- 最近的公开发布限于 blog.btcbox.jp(以信息专栏为主)。新增币种·大型资本举措的公开发表自 2022年以后未确认(无公布)。
- 维持专营 spot 的小规模独立系路线。关于衍生品或 staking 等功能扩展无公布计划。
- 持续强调服务开始10年以上(2025年4月时点)的品牌诉求。

## 8. 关联人物

- **辻治俊(つじ はるとし)**:代表取缔役社长(2019年5月就任)
- **David Zhang(张大为)**:创业者、前代表取缔役。中国出身的工程师。2014年3月设立。2019年5月就任会长。2019年3月将全部股份移交给自己设立的 TTXHoldings。
- **顾问律师事务所**:アンダーソン・毛利・友常法律事務所
- **会计审计**:ゼロス有限責任監査法人
