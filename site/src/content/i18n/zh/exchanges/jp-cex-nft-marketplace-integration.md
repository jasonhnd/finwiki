# 国内 CEX × NFT 市场（marketplace）整合模型

## 概要

国内 CEX 利用其持有 VASP 牌照的优势，自营或合作运营 NFT 市场（MP）。在 2021-2022  的 NFT 泡沫期，各公司一齐进入，但 2023-2026  已转入明显的收缩整理阶段。由于结算层需要处理加密资产，已注册 VASP 的 CEX 在结构上占有有利地位。

## 主要服务

- **Coincheck NFT**（β · 2021-03  开始）— CEX 整合型 NFT MP 的先驱。支持 ETH / Polygon，与 CryptoSpells / Sorare / The Sandbox 等建立伙伴关系。主打免手续费（Gas-less）。
- **Adam byGMO**（2022-04  开始，GMO Internet 系）— 支持日元结算的 NFT MP。强化著作权检查，专注于创作者支持。
- **LINE NFT**（2022-04  开始，LINE Xenesis 系）— 与 LINE Friends 生态系统联动。基于 LINE Blockchain（Finschia → KAIA）。支持 LINE Pay 结算。
- **Mercari NFT 构想**（Mercoin 相关）— 在构想阶段讨论与 Mercari 销售额对接的可能性。正式上线未确认。
- **乐天 NFT**（2022-02  开始）— 与乐天积分联动。限于艺能・体育系经办，重视 IP 的封闭型模型。

## 商业模式

- **手续费收入**：交易额的 2-10%。
- **CEX 客户流入**：以 NFT 购买为入口向加密资产交易交叉销售。主要目的是改善 CAC。
- **监管定位**：NFT 本身在 VASP 范围之外，但结算货币（ETH/USDC/JPY）经由 VASP，因此 CEX 整合型在国内具结构优势。

## 退出 / 收缩阶段

- **2023-2024**：因 NFT 泡沫崩溃 + 活跃用户骤减，各公司收缩。
- **2026-06**：LINE NFT 服务预定全面终止（随 LINE Xenesis 整体服务终止）。
- **Adam byGMO**：有规模缩减报道（详情未公布）。
- **Coincheck NFT**：持续中但维持 β 标记，功能扩展有限。

## 国际比较

美国 OpenSea / Magic Eden / Blur 等作为独立 MP，以连接加密资产 wallet 为前提。国内由于 VASP 监管和日元结算需求，CEX 整合型独自模型得以成立。海外为 MP 主导，国内为 CEX 主导的非对称结构。
