# SBI VC交易株式会社 — 日本加密资产交易业者 overview

> FSA 登记编号 关东财务局 第00011号 · JVCEA 会员 #1011 · 登记日 2017-12-01

## 1. 法人・股东

- **商号**：SBI VC交易株式会社
- **英文表记**：SBI VC Trade Co., Ltd.
- **设立**：2017年5月26日
- **总店所在地**：〒106-6021 东京都港区六本木1-6-1 泉花园塔
- **资本金**：13億5,000万円（含资本准备金）
- **股东**：SBI 集团 100%（母公司：SBI 控股株式会社，东证 Prime 上市・证券代码 8473）

## 2. 牌照・登记状况

该公司是目前日本唯一的「3种牌照同时持有业者」，单独持有加密资产交易业・金融商品交易业（加密资产衍生品）・电子结算手段等交易业。

| 业种 | 登记编号 | 备注 |
|------|---------|------|
| 加密资产交易业 | 关东财务局长 第00011号 | 登记 2017-12-01 |
| 第一种金融商品交易业（加密资产衍生品） | 关东财务局长（金商）第3247号 | OTC 衍生品对应 |
| 电子结算手段等交易业 | 关东财务局长 第00001号 | 2025-03-04 登记完成，USDC 经办用 |

- **JVCEA 会员种类**：第一种（会员编号 1011）
- **加入协会**：日本加密资产等交易业协会（JVCEA）、日本加密资产商业协会（JCBA）、日本数字经济联盟、日本数字去中心化金融协会、JPCrypto-ISAC

### 电子结算手段 / USDC cross-link

在金融厅的电子结算手段等交易业者登记一览中，SBI VC交易被记载为关东财务局长 第00001号的登记业者，所经办电子结算手段为 USDC。因此该公司不仅是单纯的加密资产交易业者，在解读日本的电子结算手段 / stablecoin 实施时也是 primary route。

Related stablecoin routes: [[fintech/japan-ecisb-license]], [[fintech/japan-stablecoin-regulatory-landscape]], [[fintech/stablecoin-channel-japan-sbi-jpyc-ring]], [[payment-firms/jpyc]], [[payment-firms/progmat]].

## 3. 经办币种

2026年5月时点现货交易对象约40种。主要币种：BTC・ETH・XRP・LTC・BCH・SOL・ADA・DOGE・DOT・LINK・XLM・AVAX・MATIC・SHIB・TON・SUI 等。此外作为电子结算手段（EP 手段）经办 **USDC** 及 JPY 稳定性资产（ZPG・DAI）。

特色：
- **USDC**（USD Coin）于 2025年3月25日面向国内一般用户开始交易。利用电子结算手段等交易业登记（第00001号），提供日元⇔USDC 现货兑换及 **USDC 借贷服务**（2026年3月18日开始・国内首例）。
- 有**支持质押的币种**（ETH・SOL 等）。
- **NIDT**（Nippon Idol Token）・**FLR**（Flare）・**XDC** 等较国内其他公司略为广泛的上市阵容。

## 4. 业务范围

| 服务 | 概要 |
|---------|------|
| 现货销售所 | 对日元价差交易 |
| 现货交易所（盘口交易） | Maker/Taker 手续费制，提供面向 PC 的 VCTRADE Web |
| 杠杆交易（OTC 衍生品） | 最大 2 倍（个人）、JVCEA 规定的倍率（法人） |
| 借币（借贷） | 面向个人・法人的加密资产消费借贷 |
| 定投 | 定额自动定投 |
| 质押 | ETH 等 PoS 币种 |
| SBI Web3 钱包 | 自托管联动 |
| SBIVC for Prime | 面向大额・法人的 OTC 专用服务 |
| USDC 借贷 | 国内首例（2026-03-18 开始），按杂项所得处理 |

**法定货币入金渠道**：银行转账・快速入金（Pay-easy 等）。账户管理费・年费・日元出入金手续费・加密资产出入库手续费均免费。

## 5. 市占・交易额

公开指标：
- **自有资本规制比率** 354.0%（未固定化的自有资本 76億9,800万円）※截至2026年3月末时点・该公司 IR 披露
- 账户数・月度交易额的公开数值无法确认（未公布）

定位：从持有 3种牌照・承接 DMM Bitcoin 客户资产移交（推定数十万账户规模）・SBI 集团百分百子公司这几点来看，应处于国内大手 5–6 社，但交易额・账户数的公开比较排名中并未公布该公司独有的数字。客户层以个人零售为主，并附设面向大额・法人的 Prime 服务。

| 2026-01-30 | 发表 SBI VC交易与 BITPoint Japan 株式会社的合并决议（SBI VC交易为存续公司，生效日预定 2026年4月1日） |
| 2026-04-01 | BITPoint Japan 吸收合并生效（两品牌暂时并行维持） |

## 7. 战略动向

- **3 种牌照结构的战略意义**：将加密资产交易业（现货）・金融商品交易业（衍生品）・电子结算手段等交易业（稳定币）集约于单一法人，在国内属唯一。特别是电子结算手段等交易业（USDC 经办）于 2025-03 完成登记，正展开 Circle 公司 USDC 的日元计价现货兑换及借贷。
- **SBI 集团内协同**：具备可与 SBI 证券（金商 第44号、JVCEA #1041）・BI FXTRADE（金商 第2635号、JVCEA #1026）等同集团旗下金融商品交易业者联动的体制。SBI 控股与 Ripple Labs（XRP 相关）有出资・合作关系。
- **BITPoint Japan 合并**（2026-04 生效）以应对金融商品交易法对加密资产监管的强化、集中集团内经营资源为目的（据官方发表）。合并后 VCTRADE 与 BITPOINT 的 2 品牌暂时维持。
- 设有**面向 MTGOX 债权人的对应窗口**，参与 Mt.Gox 清算相关的资产返还（详情在公开信息范围内）。
