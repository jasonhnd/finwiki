# 株式会社 Mercury（CoinTrade） — 日本加密资产交换业者 overview

This entry sits under [[exchanges/INDEX|exchanges index]]. Read it with [[exchanges/fsa-vasp-registration-system|FSA 暗号資産交換業登録制度 — 番号体系・財務局管轄・登録要件]] for adjacent context and [[fintech/japan-financial-regulation|日本金融規制 — トークン・暗号資産・決済に関する法体系]] for the broader system boundary.

> FSA 登记编号 关东财务局长 第00025号 · JVCEA 会员 #1030 · 登记日 2021-02-17

## 1. 法人・股东

- **商号**: 株式会社 Mercury（英文: Mercury Inc.）
- **服务名**: CoinTrade（CoinTrade）
- **设立**: 2017年9月1日
- **总店所在地**: 东京都涩谷区樱丘町20番1号 涩谷 Infoss Tower 12层（〒150-0031）
- **资本金**: 3,000万円（截至 2025年12月末）
- **股东**: 株式会社 Ceres（Mothers 上市 IT 企业，积分・移动业务）100%出资（公开信息。未明示出资比例）
- **关系公司**: bitbank 株式会社（官网记载）

株式会社 Ceres 是都木聪先生于 2005年设立的 Mothers 上市企业。作为 Ceres 的加密资产事业部门，Mercury 于 2017年9月设立。

## 2. 牌照・登记状况

- **加密资产交换业**: 关东财务局长 第00025号
- **JVCEA 会员种类**: 第一种（仅加密资产交换业。无衍生品交易业登记）
- 加入协会: JVCEA・JCBA（日本加密资产商业协会）・JBA（日本区块链协会）・BCCC・JPCrypto-ISAC
- 金融商品交易业・电子结算手段等交易业・资金移动业的登记，从公开信息中无法确认

## 3. 取扱货币

- **取扱币种**: 官网首页显示了 BTC・ETH・XRP・LTC・BCH・ADA 的价格，可确认这些为主要取扱币种。取扱币种总数官网未明记（2026-05-18时点）。
- **质押对象**: 共 11种（官网记载）
- **特色**: 贩卖所形式（交易所 = 撮合盘交易不提供）。以质押・借贷・定投为特化的多功能运用型平台谋求差异化。

## 4. 业务范围

- **现货买卖**: 加密资产贩卖所（spot 贩卖所）
- **质押**: CoinTrade Stake（对应11币种）
- **借贷**: CoinTrade Lending
- **定投运用**: 定期买入（最低500円/次）
- **PC 版交易工具**: 有
- **最低购买额**: 100円
- **保证金・衍生品**: 无（FSA 登记上亦仅为加密资产交换业）
- **面向法人服务**: 开户个人・法人均支持（设有「开户流程（法人客户）」文档）
- **入金渠道**: 公开信息提及「日元入金方法・定期入金」。具体渠道（是否有快速入金）在公开页面浏览范围内无法确认
- **2026-05-15 获奖**: 获得 Oricon 顾客满意度调查「首次高评企业」（官方公告）

## 5. 市占・交易量

- 交易量・开户数・AUM 未公开（2026-05-18 时点）
- 从该公司企业网站及 JVCEA 披露资料中亦无法取得数值
- JVCEA 第一种会员 26 社家中，登记编号为第 00025 号（按登记顺序属后发组）
- 推测为面向零售个人的零细〜中等规模平台。可见其特化于质押・定投、面向利基群体的战略。bitbank 为集团关系公司，在基础设施层面存在联动可能性
- 竞争内定位: **niche**（无交易量依据。JVCEA 统计上无各社个别公开）

## 6. 沿革・重大事件

| 年月 | 事件 |
|------|--------|
| 2017年9月1日 | 株式会社 Mercury 设立（株式会社 Ceres 100%出资） |
| 2017年9月 | 都木聪先生就任董事 |
| 2018年8月 | 寺东宏城先生・多田斎先生就任董事/监事 |
| 2021年2月17日 | FSA 加密资产交换业者登记（关东财务局长 第00025号） |
| 2020年7月 | 都木聪先生就任代表董事社长 |
| 2021年（据 Copyright 记述推测） | CoinTrade 服务正式投入运营 |
| 2021年3月 | 藤原崇亮先生就任董事副社长 |
| 2023年3月 | 小宫贵仁先生就任董事 |
| 2026年5月 | Oricon 顾客满意度调查「高评企业」首次认定（官方公告 2026-05-15） |

行政处分・黑客攻击・重大故障等公开信息，截至2026-05-18 时点未确认。2026年4月〜5月有 POL・BCH 出入金停止措施的公告（推测为代币侧的技术性原因。与 FSA 行政处分无关）。

## 7. 战略动向

- **2026年5月**: POL・BCH 出入金临时停止应对（具体事由无法从公开信息判断）
- **2026年4月〜5月**: 展开包括福冈在内的自主企划活动系列（用户获取・启蒙类）
- 质押强化路线明确。通过专栏・媒体曝光，以个人投资者群体为中心扩大认知
- 与 bitbank 的资本・技术联动详情未公开。官网存在指向 bitbank MARKETS 的链接
- 关于 IPO・M&A 无公开信息

## 8. 相关人物

- **代表董事社长**: 都木聪（もとき さとし）— 上智大学经济学部毕业→野村证券→CyberAgent 社长室长。2005年设立 Ceres，任代表董事。自 2020年7月起兼任 Mercury 代表董事
- **董事副社长**: 藤原崇亮 — 出身 Recruit，历经 LastRoots 等后于 2021年3月就任
- **董事**: 寺东宏城（出身 Ceres 经营企划室长）、小宫贵仁（出身大和证券・OANDA・LastRoots）
- **监事**: 多田斎（原野村证券董事兼执行役会长・Daiko 证券 Business 社长，兼任 Ceres 董事）、神田隆
