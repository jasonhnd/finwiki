# 境内加密监计法人 全景 —— Big4 + 太阳 + BDO 三优 加密实务比较

## Wiki route

This entry sits under [[exchanges/INDEX|exchanges index]]. Read it against [[exchanges/jp-vasp-security-audit-certification|国内 VASP セキュリティ監査・認証取得状況]] for peer / contrast context and [[exchanges/fsa-vasp-registration-system|FSA 暗号資産交換業登録制度]] for the broader system / regulatory boundary.

## 概述

FSA 注册 VASP 依据资金结算法 + 内阁府令，负有每个事业年度从会计师或监计法人处接受**财务报表审计**和**分别管理审计**（对客户资产与自有资产分别保管的保证）的义务。**JICPA 行业别委员会实务指针第 61 号（加密资产相关事业者审计）** 作为指针运用。Big4 （DT Tohmatsu / EY ShinNihon / KPMG AZSA / PwC Japan）与准大型所（太阳 Grant Thornton / BDO 三优）分占市场，面向 Web3 初创亦有中小监计法人进入。自 **Coincheck NEM 流出事件（2018-01）** 以后，审计质量与分别管理治理趋于严格化。

## Big4 监计法人加密实务

### **有限责任监计法人 トーマツ（Deloitte Touche Tohmatsu）**

- **加密 / 数字资产专门组织**：设置「区块链・加密资产顾问」组
- **目标客户**：[[exchanges/jp-exchange-bitflyer|bitFlyer]] 系・大型 VASP 审计实绩众多（公开口径）
- **优势**：分别管理审计方法论的完备，参与制定 JICPA 实务指针第 61 号
- **附加服务**：Travel Rule 应对咨询（与 [[exchanges/jp-vasp-aml-travel-rule-implementation|JVCEA Travel Rule 実装]] 相关）、税务（Deloitte Tohmatsu 税理士法人）、风险咨询

### **EY 新日本有限责任监计法人（EY ShinNihon）**

- **加密实务**：「金融服务本部 加密资产团队」
- **优势**：通过全球 EY 网络转移海外 VASP・稳定币发行体审计 know-how
- **目标**：大型 VASP・稳定币发行体・代币发行体
- **附加**：IFRS 加密资产会计建议、Web3 IPO 准备支持

### **有限责任 あずさ监计法人（KPMG AZSA）**

- **加密实务**：「KPMG Japan 数字资产」组
- **目标**：大型 VASP・地银系加密资产相关企业
- **优势**：内部控制（J-SOX）+ 加密保管控制的联动审计、SOC2 / ISMS 准备支持
- **附加**：[[exchanges/jp-crypto-bank-credit-facilities|銀行系クリプト与信検討]] 的建议（金融机构 + 加密资产双方业务知见）

### **PwC あらた有限责任监计法人（PwC Aarata）**

- **加密实务**：「FinTech・加密资产」团队
- **优势**：PwC 全球 "Time for trust" 加密资产框架、Web3 战略咨询
- **目标**：Web3 初创 + 大型 VASP 的组合
- **附加**：NFT・DAO 法务税务建议、代币经济学（tokenomics）设计支持

## 准大型所（Tier 2）

- **太阳有限责任监计法人（Grant Thornton Japan）** —— 在中坚 VASP 的财务报表审计・分别管理审计上具优势。经由 Grant Thornton 全球网络应对跨境加密资产案件
- **BDO 三优监计法人** —— BDO 全球网络的日本据点。面向中坚 VASP・Web3 初创的审计
- **PwC 京都监计法人 / EY 等关西据点** —— 面向以关西为本据的 VASP

## Web3 / 代币发行体特有的论点

审计论点在 VASP 审计与 Web3 代币发行体之间大不相同：

1. **持有加密资产的估值** —— 法人税法 61 条 期末时价评估（2024 修订下排除自家发行 + 第三方持有代币）
2. **分别管理保证** —— 冷钱包 / 多签 / 托管方分离（[[exchanges/jp-custody-ginco|Ginco]] / [[exchanges/jp-custody-komainu|Komainu]] / [[exchanges/jp-custody-fireblocks-japan|Fireblocks Japan]] 的选择直接关系到审计论点）
3. **交易额 / 余额完整性验证** —— 区块链上交易与账簿的核对程序
4. **Travel Rule 应对** —— KYC / 客户确认证迹的存在验证
5. **gas 费 / 质押奖励的确认时点** —— IFRS / 日本基准的差异
6. **NFT / 代币的会计确认** —— 存货 / 无形固定资产 / 加密资产的区分判定

## VASP 审计 + 法务建议 + 取证的三大支柱

境内 VASP 的合规体制由以下三层支撑：

- **会计审计**：本页的监计法人
- **法务建议**：[[exchanges/japan-crypto-law-firm-landscape|国内クリプト法律事務所ランドスケープ]]
- **取证 / 交易监视**：[[exchanges/global-crypto-forensics-vendor-layer|グローバル暗号資産 forensics vendor 層]]

这些 3 层 × [[exchanges/jvcea-self-regulatory-overview|JVCEA 自主規制]] + [[exchanges/jcba-japan-crypto-business-association|JCBA 産業団体]] + [[exchanges/fsa-vasp-registration-system|FSA 登録]] 即为境内 VASP 合规的全貌。
