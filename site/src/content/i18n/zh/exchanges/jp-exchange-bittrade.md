# BitTrade 株式会社 — 日本加密资产交换业者 概览


## Wiki route

This entry sits under [[exchanges/INDEX|exchanges index]]. Read it with [[exchanges/fsa-vasp-registration-system|FSA 暗号資産交換業登録制度 — 番号体系・財務局管轄・登録要件]] for adjacent context and [[fintech/japan-financial-regulation|日本金融規制 — トークン・暗号資産・決済に関する法体系]] for the broader system boundary.

> FSA 登记号 关东财务局 第00007号 · JVCEA 会员 #1007 · 登记日 2017-09-29

## 1. 法人・股东

- **商号**: ビットトレード株式会社(英文: BitTrade Inc.)
  - 旧商号: ビットトレード株式会社 → フォビジャパン株式会社(2018-12-17 改称)→ ビットトレード株式会社(2023-04-21 改称)
- **设立**: 2016年9月
- **总店所在地**: 〒105-0003 东京都港区西新桥1-1-1
- **资本金**: 1億円(2023年2月末时点)
- **上市状态**: 非上市
- **主要股东(公表部分)**:
  - SINOHOPE SG PTE. LTD.(被视为最大股东)
  - Sinohope Technology Holdings Limited
  - 东海东京金融控股股份有限公司
  - 株式会社FPG

## 2. 牌照・登记状况

- **加密资产交换业**: 关东财务局长 第00007号
- **第一种金融商品交易业**: 关东财务局长(金商)第3295号
- **JVCEA 会员**: 第一种会员 #1007(加密资产交换业 + 加密资产衍生品交易 两者均涵盖)
- **JCBA 会员**: 一般社团法人日本加密资产业务协会 正会员
- **Fintech 协会**: 风险会员
- **日本STO协会 / 日本证券型代币协会**: 赞助会员

## 3. 交易币种

- **上架币种数**: 全49币种(2026年5月现在)
  - ※ 2026年5月8日有 BSV(Bitcoin SV)交易废止预定通知
- **主要币种(示例)**: BTC / ETH / XRP / SOL / ADA / DOGE / BNB / DOT / LTC / BCH / LINK / TON / APT / SUI / MATIC / ARB / OP 等
- **特色**:
  - 山寨币阵容为国内最多水平(交易开始当初标榜"币种数国内No.1")
  - 也经办多个独有・项目系币种(JOC, UPC, CRTS, ZPG 等)
  - 设置 IEO(Initial Exchange Offering)平台

## 4. 业务范围

| 服务 | 提供状况 |
|---------|---------|
| 销售所(现货) | ✓ |
| 交易所(盘口交易) | ✓ |
| 杠杆交易(CFD) | ✓(第一种金商业的依据) |
| 定投(定期定投) | ✓ |
| 借贷加密资产 | ✓ |
| IEO | ✓(有专用一览页面) |
| 法人服务 | ✓(有专用页面) |
| API | WebSocket 规格(毫秒级对应) |

- **法定货币入金渠道**: 快捷入金(除 Pay-easy 外免费)/ 转账入金 / Pay-easy 入金 / 便利店入金
- **交易银行**: PayPay 银行・乐天银行・住信SBI网络银行・理索那银行・邮储银行・GMO Aozora 网络银行
- **安全**: 自服务开始以来黑客攻击次数为零(官方发布),加密资产 100%冷钱包管理,24小时 365日系统监控

## 5. 市占・成交量

- **账户数・成交量**: 无公表(披露信息中无数值记载)
- **市场内定位**: 以币种数丰富为主要差异化轴的中坚交易所。JVCEA Type 1 会员 26社中,同时持有加密资产交换业和衍生品交易双方者仅限于 13社,该公司为其中之一(详情参见 [[exchanges/jvcea-self-regulatory-overview|JVCEA 自主規制体系]] / [[exchanges/jp-vasp-derivative-license-system|JP VASP デリバティブライセンス制度]])。山寨币轴的竞争对手为 [[exchanges/jp-exchange-bitbank|bitbank]],全范围行业比较可参考 [[exchanges/jp-exchange-binance-japan|Binance Japan]]。
- **客户层**: 主要为个人零售。也展开面向法人的服务。

## 6. 沿革・重大事件

| 年月 | 事项 |
|------|------|
| 2016年9月 | 作为 ビットトレード株式会社 设立 |
| 2017年9月29日 | 加密资产交换业 登记(关东财务局 第00007号) |
| 2018年12月17日 | 随与 Huobi 集团强化合作,将服务品牌变更为"Huobi Japan(フォビジャパン)" |
| 2019年10月 | "销售所"服务上线。向 FPG株式会社 第三方配售增资 |
| 2019年12月 | 与东海东京金融控股资本业务合作 |
| 2019年4月 | 据金融厅・路透报道实施了进入检查(似乎未至行政处分) |
| 2020年6月 | 上架 Huobi Token(HT) |
| 2023年2月15日 | 品牌名变更为"ビットトレード" |
| 2023年4月21日 | 法人名正式改称为"ビットトレード株式会社"(自フォビジャパン株式会社) |
| 2023年3月31日 | 网站域名变更(huobi.co.jp → bittrade.co.jp) |
| 2025年11月 | 熔断机制(circuit breaker)开始 |
| 2026年5月 | BSV・SXP 等部分币种的交易废止通知 |

**黑客・重大事件**: 据官方发布自设立以来无黑客攻击(无公表)。

## 7. 战略动向(最近 12 个月)

- **2025年7月**: 与 JTB・NTT TechnoCross 开始关于通过"应援活动(推し活)"实现地区活性化的共同研究(新闻稿)
- **2025年8月**: 与株式会社北纺缔结关于加密资产・Web3.0相关业务的基本合意书
- **2025年11月**: 与 ENI Foundation 就 Web3.0 领域协作推进达成合意(新闻稿)
- **2026年1月**: 与澳大利亚不动产开发公司 Avantaus Development(ADG)缔结活用 RWA(现实资产)的战略合作
- **IEO 展开**: 持续对国内独有币种(JOC 等)进行定期信息披露,新上架申请窗口亦公开中
- **第一种金商业活用**: 提供 CFD(加密资产衍生品)平台。每季度披露自有资本规制比率(至 2026年3月末披露分已公表)
