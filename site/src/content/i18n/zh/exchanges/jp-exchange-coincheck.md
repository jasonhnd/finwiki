# Coincheck 株式会社 — 日本加密资产交换业者 overview

> FSA 登记号 关东财务局 第00014号 · JVCEA 会员 #1017 · 登记日 2019-01-11

## 1. 法人·股东

- **商号**：コインチェック株式会社 / Coincheck, Inc.（日本法人）
- **设立**：2012年（作为 Wonder 株式会社设立。2014年 变更商号为 Coincheck 株式会社）
- **总店**：东京都涩谷区樱丘町 1 番 4号 涩谷 Sakura Stage SHIBUYA Side 27F
- **法人番号**：1010001148860

- **母公司**：Monex Group 株式会社（东证 Prime 8698）
  - 2018年4月 Monex Group 通过取得股份将其完全子公司化（取得价额 约 36億円）
- **上市控股公司**：Coincheck Group N.V.（荷兰公开有限责任公司，总部设于阿姆斯特丹）
  - Nasdaq 上市（代码 CNCK / CNCKW）。Coincheck 株式会社的直接母公司。
  - Monex Group 为 Coincheck Group N.V. 的第一大股东（合并完成后仍保持过半）

## 2. 牌照·登记状况

- **加密资产交换业**：关东财务局长 第00014号（登记日 2019-01-11）
- **JVCEA 会员**：第一种会员 #1017（仅加密资产交换业；衍生品交易业未获认可）
- 在 JVCEA 公开信息范围内，未确认电子结算手段等交易业·金融商品交易业·资金移动业的登记

## 3. 取扱货币

- **取扱币种数**：截至 2022年9月 为 18 个币种（+1  QoQ）。此后亦随时持续扩充
- **主要币种**：BTC、ETH、ETC、XRP、NEM（XEM）、LTC、XLM、MONA、BAT、QTUM、LSK、XEM、ENJ、SAND、AXS 等
- **特色**：在国内较早支持多币种。在国内首家导入 IEO（Initial Exchange Offering）功能。运营 NFT 市场（Coincheck NFT）。以现货为中心，不提供保证金·衍生品交易

## 4. 业务范围

- **现货交易**：同时提供交易所（盘口交易）+ 贩卖所（OTC）
- **借出加密资产**（Coincheck Lending）：从客户处借入加密资产，用于对冲交易
- **NFT 市场**：Coincheck NFT (β)。与 IP 持有者的 1 次销售（BtoC）+ 用户间 2 次买卖（CtoC）
- **IEO**：以「Coincheck IEO」展开国内首家 IEO 服务
- **法定货币入金**：银行转账·快捷入金（便利店·Pay-easy 等）
- **Web 3  / 元宇宙**：Coincheck Labs（创业支持·投资）、面向元宇宙的 NFT LAND 销售（The Sandbox、Otherside、Decentraland 等）
- 亦提供面向法人的服务（OTC 大额交易等）

## 5. 市占·交易额

- **App 下载量**：国内加密资产交易 App No.1（连续 5 年 2019–2023年，AppTweak 调查）
- **认证用户数**：175万 人（2022年9月 末）→ 183万 人（2023年6月 末）
- **托管资产（冷钱包）**：
  - 2022年9月 末：约 2,796億円
  - 2023年6月 末：约 3,465億円
  - 2024年12月 末：约 1兆 305億円（Coincheck Group N.V. 财务披露 2025-02）
- **交易额**：现货交易额 2Q FYE Mar.2022（3 个月）7,695百万円（当时）。市场低迷期（2023年 前后）大幅缩小。市况恢复后的 2024年 年度大幅扩大。个别月度数据参见 JVCEA 统计
- **竞争内定位**：以认证账户数计为国内最大规模。与 bitFlyer·GMO 币等并列上位 3社 之一角

## 6. 沿革·重大事件

| 年月 | 事项 |
|------|------|
| 2012-08 | 作为 Wonder 株式会社设立。启动加密资产钱包·交易服务 |
| 2014 前后 | 变更商号为 Coincheck 株式会社 |
| **2018-01-26** | **NEM（XEM）非法流出事件**：保管于热钱包的 XEM 约 5億 2,630万 NEM（当时市价约 580億円 相当）被外部黑客非法转移。当日深夜该公司正式发表。日本史上最大规模的加密资产流出事件（详情参见 [[exchanges/coincheck-nem-hack-detailed-analysis|Coincheck NEM ハック詳細分析]] 及 [[security/forensic-identity-anchor-chain|forensic identity anchor chain]]） |
| 2018-03 | 收到金融厅业务改善命令（多次）。应对 FSA 检查·改善要求 |
| 2018-04 | Monex Group 以约 36億円 取得全部股份并完全子公司化。经营体制刷新 |
| 2019-01-11 | 取得加密资产交换业登记（关东财务局 第00014号） |
| 2020 前后 | 启动 NFT 市场「Coincheck NFT（β）」 |
| 2022-03 | 公布与 Nasdaq 上市 SPAC Thunder Bridge Capital Partners IV, Inc.（THCP）缔结 Business Combination Agreement（Monex Group IR 披露） |
| 2024-10 | Business Combination Agreement 第 3 次修订 |
| **2024-12-05** | 在 Thunder Bridge IV 股东特别决议中批准合并 |
| **2024-12-10** | 合并完成（Closing）。Coincheck Group N.V. 作为 THCP 的后继法人成立 |
| **2024-12-11** | **Nasdaq 上市开始**：代码 CNCK（普通股）·CNCKW（认股权证）。总募集额 约 3,160万 美元（赎回后净额）。日本加密资产交换业者首次 Nasdaq 上市 |

## 7. 战略动向

- **接入全球资本市场**：Nasdaq 上市后，强化面向美国机构投资者的 IR 活动。作为 Coincheck Group N.V. 实施 F-1  / F-3  / 6-K 等 SEC 持续披露（CIK：0001913847）
- **Web 3  / 元宇宙拓展**：力图维持国内 NFT 平台的 No.1  定位，持续与 IP 持有者合作
- **IEO 扩大**：发挥国内首家 IEO 服务的优势，持续招揽项目
- **Coincheck Labs**：持续对区块链相关创业企业的投资·孵化
- **取扱币种扩充**：追加现货·新增币种。在 2024-2025年 的市场恢复期，开户·托管资产骤增

## 8. 相关人物

- **松本大（Oki Matsumoto）**：Monex Group 代表执行役社长 CEO、Coincheck Group N.V. Executive Chairman（经 SEC 披露确认。自 NEM 事件后的 2018年4月 收购时起参与 Coincheck 治理）
- **有定诚（Satoshi Hasuo）**：Coincheck 株式会社 代表取缔役社长（经 2022-2023年 时点的 SEC 披露确认为 "President: Satoshi Hasuo"）
- **Gary Simanson**：Thunder Bridge IV 原 President & CEO → 合并后 Coincheck CEO（经 SEC 披露确认）
- **创始人**：和田晃一良（Koichiro Wada）·大塚雄介（Yusuke Otsuka）共同设立（公知信息）。随 NEM 事件后的经营体制刷新，转入由 Monex 主导的经营层
