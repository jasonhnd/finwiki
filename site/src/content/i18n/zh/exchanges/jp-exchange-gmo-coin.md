# GMO Coin 株式会社 — 日本加密资产交换业者 overview


## Wiki route

> FSA 注册号 关东财务局 第00006号 · JVCEA 会员 #1006 · 注册日 2017-09-29

## 1. 法人・股东

- **商号**: GMO Coin 株式会社 / GMO Coin, Inc.
- **设立年月**: 2016年10月
- **总店所在地**: 〒150-0043 东京都涩谷区道玄坂1-2-3 涩谷 FUKURAS（SHIBUYA FUKURAS）
- **资本金**: 37.58億円（含资本准备金，2026年5月官方网站刊载值）
- **母公司**: GMO Financial Holdings 株式会社（该公司的100%子公司）
- GMO Financial Holdings 是位于 GMO Internet Group 株式会社（东证 Prime 9449）旗下的中间持股公司。统合加密资产・FX・外汇业务的金融子集团。
- 上市状态: 非上市（母公司 GMO Internet Group 已上市）
- **法人番号**: 7011001113188

## 2. 牌照・注册状况

- **加密资产交换业**: 关东财务局长 第00006号（注册日 2017-09-29）
- **金融商品交易业**: 关东财务局长（金商）第3188号
  - 为实施加密资产衍生品交易（加密资产 FX・交易所杠杆）而进行的金商注册
- **JVCEA 会员**: 第一种会员 #1006（加密资产交换业 + 加密资产衍生品交易双资格）
- **加入协会**: 一般社团法人 日本加密资产等交易业协会（JVCEA）、一般社团法人 金融期货交易业协会（FFAJ）
  - 加入 FFAJ 是为了兼营外汇 FX 业务（金融期货交易业）

## 3. 经办币种

- **销售所**: 22个币种（2026年5月时点 — BTC/ETH/BCH/LTC/XRP/XLM/XTZ/DOT/ATOM/ADA/DAI/LINK/DOGE/SOL/FIL/SAND/CHZ/AVAX/SUI/ZPG/ZPGAG/ZPGPT）
  - 2026年4月在国内首次经办 Zipangcoin（ZPG）系列（黄金・白银・铂金联动）
  - 2026年3月在国内首次经办 Wilder World（WILD）
- **交易所（现货）**: 19个币种（BTC/ETH/BCH/LTC/XRP/XLM/XTZ/DOT/ATOM/ADA/DAI/LINK/FCR/DOGE/SOL/ASTR/NAC/SUI/WILD）
- **质押支持**: ETH/XTZ/DOT/ATOM/ADA/SOL/ASTR 的7个币种（自动分配・分别管理）
- 特色: 自称交易所杠杆币种数量「国内最多」（2024年4月12日调查）。亦经办 NOT A HOTEL COIN（NAC）・FCR Coin 等自有 IEO 币种。

## 4. 业务范围

- **现货交易**: 销售所（2Way 价格）+ 交易所（盘口交易・Maker 负手续费 -0.01%）
- **衍生品**: 交易所杠杆（最大2倍・个人）+ 加密资产 FX（同）
- **外汇 FX**: 作为金融商品交易业经办。21个货币对（2026年5月时点），最大25倍杠杆。
- **加密资产借贷**: 基础型（定期型・最大10%年率）+ 高级型的2种类型
- **质押**: 7个币种・按持有量基准每月自动发放
- **定投加密资产**: 积立服务（500円〜）
- **IEO**: 过去曾以 FCR、NAC 等实施
- **法人服务**: 支持法人账户、API 交易（Public/Private）、WebTrader（高功能交易工具）
- **入金渠道**: 银行转账（推荐 GMO Aozora Net Bank）、即时入金（Pay-easy、网络银行等）、即时出金（2026年2月9日开始）

## 5. 市占・交易额

- 已公开的月间交易额・账户数: 非公开（无单独的 IR 披露）
- GMO Internet Group 的合并决算中以「加密资产事业」名义包含数字，但 GMO Coin 单体的交易额未定期公开
- 行业地位: FSA 注册号 第00006号（注册第6社号），在业历・合规遵守方面属上位层。现货交易 + 衍生品 + 外汇 FX 的兼营体制被认为与 [[exchanges/jp-exchange-coincheck|コインチェック]]・[[exchanges/jp-exchange-bitflyer|bitFlyer]]・[[exchanges/jp-exchange-sbi-vc-trade|SBI VC トレード]] 一同位于上位 tier（无数字依据・待公开数据）。质押・借贷业务的监管边界参见 [[exchanges/jp-cex-staking-lending-regulation|JP CEX ステーキング・レンディング規制]]。
- 客户层: 主要为个人零售投资者。通过法人账户・API 提供也对应交易员・机构需求

## 6. 沿革・重大事件

| 年月 | 事项 |
|------|------|
| 2016-10 | GMO Coin 株式会社 设立 |
| 2017-09-29 | FSA 加密资产交换业 注册（第00006号） |
| 2020-09 | 在交易所国内首次导入「Post-Only（SOK）」订单条件 |
| 2024-04 | 告知达成交易所杠杆币种数量「国内最多」（自社调查） |
| 2026-01 | 开始经办 Sui（SUI）（销售所・交易所・定投） |
| 2026-02-09 | 开始日元即时出金服务 |
| 2026-03 | 国内首次经办 Wilder World（WILD） |
| 2026-04 | 国内首次经办 Zipangcoin（ZPG）系列（黄金・白银・铂金联动） |
| 2026-04-30 | 发布废止经办 Dai（DAI） |
| 2026-05 | 外汇 FX 追加7个货币对（HUF/JPY 等） |

- 重大黑客攻击・行政处分・业务停止命令: 无公开（2026年5月时点）
- GMO 集团整体的安全障碍或系统维护导致的临时停止已发生多次，但未确认资产流出等大规模事件

## 7. 战略动向

- **积极追加新币种**: 仅在2026年就陆续上市 SUI・WILD・ZPG 系列等。战略性地主打国内首次上市
- **外汇 FX 的扩充**: 在2026年5月将货币对扩充至21对（追加 HUF/JPY・NOK/SEK 等高利率・新兴货币）
- **AI 联动**: 在2026年2月导入 AI 价格警报「SignalWise」，提供与自动交易服务「GPT-Trade」的外部联动
- **即时入出金基础设施**: 通过与 GMO Aozora Net Bank 的联动实现入金，自2026年2月起对应即时出金
- **质押手续费零费用活动**: 作为用户获取措施持续展开
- **集团内 DLT・SC 相关**: GMO Internet Group 通过另一法人 GMO Trust（美国）发行 USD 稳定币（旧 GYEN/ZUSD），但本公司不直接参与这些币的发行・运营。关于参入国内 stablecoin・电子支付手段等交易业，目前无公开信息
