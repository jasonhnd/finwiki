# Payward Asia（Kraken Japan）— 日本加密资产交换业者（休眠 / 撤退）概览


## Wiki route

This entry sits under [[exchanges/INDEX|exchanges index]]. Read it against [[exchanges/jp-exchange-coinbase-japan|Coinbase 株式会社 — 日本暗号資産交換業者]] for the closest parallel pattern (外资系 retail exit) and [[exchanges/fsa-vasp-registration-system|FSA 暗号資産交換業登録制度]] for the broader system / regulatory boundary.

> Kraken 是在日本市场 **2 度进入、2 度撤退** 的罕见外资系交易所。1 次 2014–2018，2 次 2020–2023。目前处于休眠状态。

**foreign-exchange-exit / 2 度日本市场撤退 / FSA 注册记录 → 已注销**

## 1. 法人·股东

- 商号（最终）：Payward Asia Inc.（日本分公司体制）/ 原 Kraken Japan 部门
- 母公司：Payward, Inc.（美国·旧金山，品牌名 Kraken）
  - 创立于 2011年，Jesse Powell 创始 CEO
  - 2025-04  NASDAQ 上市准备阶段报道（已公开公告）
- 日本事业形态：最初由 Payward 美国总公司直接提供服务（2014–2018），此后由 Payward Asia 以现地注册法人形式再度进入（2020–2023）

## 2. 牌照·注册状况（历史）

- 第 1 期（2014–2018）：FSA 加密资产交换业注册制度（2017-04 施行）前后停止服务。未取得注册即停止面向日本居住者的服务
- 第 2 期（2020–2023）：以 Payward Asia 名义在关东财务局以加密资产交换业者身份完成注册（2020 年期间）。加入 JVCEA 第一种会员
- 第 3 期（2023-01 以后）：宣布撤出日本。实施 FSA 注册注销手续。退出 JVCEA
- 2026-04  snapshot 时点：FSA 注册名单中无 Payward Asia 记载（注销确认）

## 3. 交易币种

- 第 1 期服务停止前：以 BTC、ETH、BCH、LTC、XRP 等为中心，采用与美国总公司一致的品种阵容
- 第 2 期再度进入时：面向日本精简上币，限定为符合 JVCEA 白名单的主要品种
- 撤退后：交易品种为零（休眠）

## 4. 业务范围

- 仅限现货 spot 交易（面向零售）。衍生品·保证金·质押的公开信息有限
- 法币入金：银行转账（JPY）
- 第 2 期 OTC 及法人服务扩展十分有限，随零售撤退一并全面停止

## 5. 市占·交易量

- 无公开数据。两个时期的账户数·交易量均未能在国内市场取得显著份额，作为外资系交易所落后于 bitFlyer / Coincheck / GMO Coin 等国内头部平台
- 客户层：以日本居住者零售为主（法人客户有限）
- 行业定位：作为外资系交易所的日本市场撤退案例，常与 [[exchanges/jp-exchange-coinbase-japan|Coinbase Japan]] 并列被引述

## 6. 沿革·重大事件

| 时期 | 事项 |
|------|------|
| 2014 | Kraken（美国总公司直营）开始向日本居住者提供服务 |
| 2017-04 | 修订版资金结算法施行（FSA 加密资产交换业注册制度成立）|
| 2018-04 | Kraken Global Blog 公告「将于 2018-06 前停止面向日本居住者的服务」，以成本过高、监管环境为由 |
| 2018-06 | 第 1 期服务完全停止。日本居住者账户关闭 |
| 2019–2020 | 重返日本准备期间。以 Payward Asia 名义提交 FSA 注册申请 |
| 2020 年期间 | 以 Payward Asia 名义在关东财务局完成加密资产交换业者注册。加入 JVCEA 第一种会员 |
| 2020–2022 | 第 2 期服务运营。提供 BTC/ETH 等主要品种零售 spot |
| 2023-01 | Kraken 总公司正式宣布撤出日本事业（背景：加密资产市场低迷、全球裁员 30%、监管环境负担）|
| 2023-04 | 第 2 期服务停止·客户资产返还完成。实施 FSA 注册注销手续 |
| 2026-04 | 确认 FSA 注册名单 snapshot 中 Payward Asia 不在其列（休眠状态持续）|

## 7. 战略动向

第 2 期撤退（2023-01 公告）源于母公司 Payward 的以下战略因素：

1. 2022 年加密资产市场低迷与全球削减成本（裁员 30%）
2. 日本市场外资系在上市品种方面的限制（须符合 JVCEA 白名单）及 KYC/AML 应对成本
3. 美国本土 SEC 监管强化及集中国内市场的战略选择

撤退后，Payward 总公司正推进 **2025–2026 年 NASDAQ 上市准备**，截至 2026-05 尚无公开公告显示将再度进入日本市场。在日本市场可访问的外资系 spot CEX 仅限于 [[exchanges/jp-exchange-binance-japan|Binance Japan]] 与 [[exchanges/jp-exchange-bittrade|BitTrade]]。

**行业定位**：与 [[exchanges/jp-exchange-coinbase-japan|Coinbase Japan]]（2023-02 撤退）并列为「外资系 retail CEX 撤退模式」的典型案例。两家公司的同期撤退表明，JVCEA 白名单·FSA 监管负担·实名账户要求（参照 [[exchanges/jvcea-self-regulatory-overview|JVCEA 自主規制]]）已成为外资系进入的结构性壁垒。比较对照参见 [[exchanges/jp-vasp-ma-consolidation-history|JP VASP M&A コンソリデーション史]]。

## 8. 相关人物

- **Jesse Powell**：Payward / Kraken 创始人·前 CEO（2023-09 卸任）。2018 年第 1 期撤退时多次就日本市场发声
- **Dave Ripley**：Payward 现任 CEO（2023-09 就任）。2023 年第 2 期撤退决策时的最高管理层
- 日本法人代表取締役：无公开信息（2026-05 时点官网已消失）
