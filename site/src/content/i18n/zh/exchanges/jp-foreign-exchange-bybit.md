# Bybit — 海外无登记加密资产交易所（日本 FSA 警告3次）


## Wiki route

This entry sits under [[exchanges/INDEX|exchanges index]]. Read it with [[exchanges/fsa-vasp-registration-system|FSA 暗号資産交換業登録制度 — 番号体系・財務局管轄・登録要件]] for adjacent context and [[fintech/japan-financial-regulation|日本金融規制 — トークン・暗号資産・決済に関する法体系]] for the broader system boundary.

> 在日本无加密资产交易业登记 · FSA 警告书发出：共3次（2021-05 / 2023-03 / 2024-11）· JVCEA 非会员 · 截至2026年3月末完全终止面向日本居住者的服务

## 1. 法人・股东

- **品牌名**: Bybit
- **正式法人名**: Bybit Fintech Limited
- **设立**: 2018年（新加坡）
- **总部所在地**: 迪拜（阿拉伯联合酋长国）。2022年将总部从新加坡迁至迪拜。
- **股东・资本**: 非上市的民营企业。出资结构的详情未公布。
- **创始人**: Ben Zhou（周·新加坡籍创业者）。现仍作为 CEO 主导经营。

## 2. 牌照・登记状况

- **日本加密资产交易业登记**: 无（无登记）
- **日本 FSA 警告书**: 共3次发出（详情参 §6 ）
- **JVCEA 会员**: 非会员
- **UAE（VARA）**: 在迪拜虚拟资产监管局（VARA）的管辖下运营。2022年迁移后，以 UAE 为主要管辖开展活动。
- **印度（FIU-IND）**: 2025年2月完成向印度金融情报机构（FIU-IND）的登记。同时作为对过去无登记运营的罚则，支付了 9.27 克若尔卢比（约约 1.1 億円相当）。
- **其他监管当局**: 因英国 FCA 于2021年发出加密资产衍生品禁令，暂停面向英国居住者的服务。在欧洲・加拿大等地亦有在多个管辖受到警告・限制的记录。

## 3. 经办币种

- 全球范围内经办数百种以上的加密资产。除 BTC・ETH 等主要币种外，将广泛覆盖中小山寨币作为竞争优势。
- 经办币种数的最新官方数值在公开资料中无法确认。
- **自有代币**: BIT（BitDAO / Mantle 相关代币）。2021年实施 BitDAO 代币 TGE，通过该生态系展开治理・DeFi。
- 面向日本居住者的经办指引，随该公司面向日本服务的终止，自2026年3月以降无法参照。

## 4. 业务范围

- **Spot 交易**: 各类加密资产的现货买卖
- **衍生品**: 永续掉期（Perpetual）・期货（Futures）・期权。Bybit 在衍生品领域具备优势，作为全球衍生品交易所快速成长。
- **跟单交易**: 自动复制其他交易者策略的服务
- **加密资产借贷（Earn）**: 存入加密资产以获取收益的服务
- **NFT 市场**: 以 Bybit NFT 展开
- **Web3 钱包**: 提供非托管钱包
- 法定货币入金渠道: 信用卡・银行转账（限对应地区）

## 5. 市占・交易额

- 全球范围内位居主要交易所前列，截至2024年6月时点 Bloomberg 报道其在衍生品交易额上居世界第2位（参 2024-06 Bloomberg 报道）。
- 面向日本居住者的服务已于2026年3月末终止，日本市场份额的具体数值未公布。
- 客户层: 以全球零售为主体。亦展开面向机构投资者的服务（OTC・托管等）。
- 自2021年起担任 Formula 1 Oracle Red Bull Racing 的赞助商（2022〜2024年赛季），提升全球品牌认知度。该赞助合作于2024年赛季末经双方合意终止。

## 6. 沿革・重大事件

| 时期 | 内容 |
|------|------|
| 2018 | Ben Zhou 在新加坡设立 Bybit Fintech Limited。作为专注衍生品的交易所开设 |
| 2021-05 | **FSA 第1次警告**：金融厅以「以日本居住者为相对方无登记经营加密资产交易业」为由，对 Bybit Fintech Limited（当时新加坡）发出警告书。日语服务提供・积极营销被视为问题 |
| 2021-08 | 与乌克兰电竞组织 Natus Vincere（NAVI）缔结赞助合约 |
| 2022 | 将总部从新加坡迁至迪拜（UAE） |
| 2022 | 与 Formula 1 Oracle Red Bull Racing 缔结赞助合约（3年间，报道值 1億5,000万美元） |
| 2023-03 | **FSA 第2次警告**：金融厅对 Bybit・MEXC・Bitget・Bitforex 4社发出警告书。Bybit 受到第2次警告 |
| 2023-11 | FTX 破产财团对 Bybit 的投资部门 Mirana 提起约10億美元的诉讼。主张 Mirana 在 FTX 破产前夜优先提走资产 |
| 2024-10 | Bybit 以支付 2億2,800万美元和解了与 FTX 的诉讼 |
| 2024-11-28 | **FSA 第3次警告**：金融厅对 KuCoin・bitcastle・Bybit Fintech Limited（所在地: 迪拜）・MEXC Global・Bitget 5社同时发出警告书。Bybit 累计第3次警告 |
| 2025-02 | **约15億美元规模的黑客攻击（史上最大）**：2025年2月21日，Bybit 的多签钱包遭攻击。约40万枚 ETH（当时约1.4〜1.5億美元相当）流出，创加密资产交易所史上最大被害额。攻击者滥用第三方提供的 Safe{Wallet} 基础设施。Arkham Intelligence・Elliptic・美国 FBI 结论认为系朝鲜系黑客集团「Lazarus Group / TraderTraitor」涉案。Bybit 在72小时以内从 Galaxy Digital・FalconX・Wintermute 等处确保了约44.7万枚 ETH，恢复了准备金 |
| 2025-02 | 印度 FIU-IND 登记完成・罚款支付（Rs 9.27 克若尔）|
| 2025-02 | **从日本的 App Store・Google Play 下架**：应金融厅要求，MEXC・Bybit・Bitfinex・KuCoin・Bitget 各 App 从日本的 Apple App Store 及 Google Play 下架・无法搜索 |
| 2025-10-31 | **停止日本居住者的新规注册**：Bybit 自2025年10月31日起停止受理日本居住者的新规账户开设 |
| 2025-12 | 正式发表面向日本居住者服务的分阶段终止 |
| 2026-01 | 将面向日本居住者的持仓转为「仅平仓」模式。发表2026年3月末为最终期限 |

## 7. 战略动向

- **从日本市场撤退（2026-Q1）**: 经历 FSA 警告3次后，面向日本居住者的服务于2026年3月末完全终止。与 [[exchanges/jp-exchange-binance-japan|Binance]] 于2022年设立日本法人（Sakura Exchange BitCoin，现 Binance Japan）并取得登记形成对照，Bybit 未采取面向日本市场的合规化途径，而选择切离现货（背景由 [[exchanges/fsa-foreign-exchange-warning-system|FSA 海外取引所警告制度]] 与 [[exchanges/uae-vara-licensing-overview|UAE VARA ライセンス]] 补充信息）。
- **专注 UAE・全球主要市场**: 以迪拜为大本营，持续扩大面向机构投资者的服务・衍生品。
- **2025年黑客攻击后的信任恢复**: 对约1.5億美元的被害，实现短期内的准备金恢复。发表以提升透明度为目的强化 POA（Proof of Assets）披露。
- **重新进入印度市场**: 于2025年2月完成 FIU-IND 登记・罚款支付，转为监管下的合法运营。目标确立在印度加密资产市场作为正规玩家的地位。

## 8. 相关人物

- **Ben Zhou（CEO・创始人）**: 新加坡籍创业者。自2018年创办 Bybit 以来，作为 CEO 主导经营。2025年2月黑客攻击后，在 X（旧 Twitter）上亲自说明状况、主导迅速应对而受到瞩目。
- 其他主要高管・出资者的详情因公开资料无法确认，故不记载。
