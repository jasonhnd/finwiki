# Bybit — 海外无注册加密资产交换所（日本FSA警告3次）


## Wiki route

This entry sits under [[exchanges/INDEX|exchanges index]]. Read it with [[exchanges/fsa-vasp-registration-system|FSA 暗号資産交換業登録制度 — 番号体系・財務局管轄・登録要件]] for adjacent context and [[fintech/japan-financial-regulation|日本金融規制 — トークン・暗号資産・決済に関する法体系]] for the broader system boundary.

> 日本无加密资产交换业注册 · FSA 警告书发出: 共3次（2021-05 / 2023-03 / 2024-11）· JVCEA 非会员 · 2026年3月末起面向日本居民的服务已完全终止

## 1. 法人·股东

- **品牌名**: Bybit
- **正式法人名称**: Bybit Fintech Limited
- **成立**: 2018年（新加坡）
- **总部所在地**: 迪拜（阿拉伯联合酋长国）。2022年从新加坡迁至迪拜。
- **股东·资本**: 非上市私人企业。出资结构详情未对外公布。
- **创始人**: Ben Zhou（周·新加坡籍创业者）。现仍以 CEO 身份主导经营。

## 2. 牌照·注册状况

- **日本加密资产交换业注册**: 无（未注册）
- **日本FSA警告书**: 共发出3次（详情见 §6 ）
- **JVCEA会员**: 非会员
- **UAE（VARA）**: 在迪拜虚拟资产监管局（VARA）管辖下运营。2022年迁址后，以UAE为主要管辖地开展业务。
- **印度（FIU-IND）**: 2025年2月，完成印度金融情报机构（FIU-IND）注册。同时就过去无证经营支付罚款 9.27 克罗尔卢比（约合约 1.1 億円）。
- **其他监管机构**: 英国 FCA 于2021年发出加密资产衍生品禁令后，暂停向英国居民提供服务。欧洲、加拿大等多个司法管辖区亦有警告或限制记录。

## 3. 交易货币

- 在全球范围内交易数百种以上的加密资产。以 BTC、ETH 等主要币种为基础，广泛覆盖中小山寨币，将此作为竞争优势。
- 交易币种数量的最新官方数据在公开资料中无法确认。
- **自有代币**: BIT（BitDAO / Mantle 相关代币）。2021年进行 BitDAO 代币 TGE，通过该生态系统开展治理·DeFi。
- 面向日本居民的交易信息，随该公司终止日本服务，2026年3月之后已无法查阅。

## 4. 业务范围

- **现货交易**: 各类加密资产的现货买卖
- **衍生品**: 永续合约（Perpetual）·期货（Futures）·期权。Bybit 在衍生品领域具有强项，作为全球衍生品交易所快速成长。
- **跟单交易**: 自动复制其他交易者策略的服务
- **加密资产借贷（Earn）**: 存入加密资产获取收益的服务
- **NFT 市场**: 以 Bybit NFT 形式运营
- **Web3 钱包**: 提供非托管钱包
- 法币入金渠道: 信用卡·银行转账（仅限支持地区）

## 5. 市场份额·交易量

- 在全球主要交易所中位居前列，2024年6月时点 Bloomberg 报道其衍生品交易量排全球第2位（参见2024-06 Bloomberg 文章）。
- 面向日本居民的服务已于2026年3月末终止，日本市场份额具体数值未公布。
- 客户群体: 以全球散户为主。机构投资者服务（场外交易·托管等）亦有开展。
- 自2021年起赞助 Formula 1 甲骨文红牛车队（2022〜2024年赛季），提升全球品牌知名度。该赞助协议于2024年赛季末经双方协议终止。

## 6. 沿革·重大事件

| 2021-08 | 与乌克兰电子竞技组织 Natus Vincere（NAVI）签订赞助合同 |

| 2023-11 | FTX 破产受托方对 Bybit 投资部门 Mirana 提起约10億美元诉讼。主张 Mirana 在 FTX 破产前夕优先提取资产 |

| 2025-02 | **约15億美元规模黑客攻击（史上最大）**：2025年2月21日，Bybit 多签钱包遭到攻击。约40万枚 ETH（当时约1.4〜1.5億美元）流出，创加密资产交易所史上最大损失额记录。攻击者利用第三方提供的 Safe{Wallet} 基础设施。Arkham Intelligence·Elliptic·美国 FBI 均认定朝鲜系黑客组织"Lazarus Group / TraderTraitor"参与其中。Bybit 在72小时内从 Galaxy Digital·FalconX·Wintermute 等处取得约44.7万枚 ETH，恢复准备金 |

| 2025-10-31 | **停止日本居民新注册**：Bybit 自2025年10月31日起停止受理日本居民新开账户 |
| 2025-12 | 正式宣布分阶段终止面向日本居民的服务 |
| 2026-01 | 将面向日本居民的仓位切换为"仅可平仓"模式。宣布2026年3月末为最终期限 |

## 7. 战略动向

- **退出日本市场（2026-Q1）**: 经历3次 FSA 警告后，于2026年3月末完全终止面向日本居民的服务。与[[exchanges/jp-exchange-binance-japan|Binance]]在2022年设立日本法人（Sakura Exchange BitCoin，现 Binance Japan）并取得注册的做法形成对比，Bybit 未采取合规进入日本市场的路径，选择切割现货业务（背景详见 [[exchanges/fsa-foreign-exchange-warning-system|FSA 海外取引所警告制度]] 和 [[exchanges/uae-vara-licensing-overview|UAE VARA ライセンス]]）。
- **专注UAE·全球主要市场**: 以迪拜为基地，持续扩大机构投资者服务及衍生品业务。
- **2025年黑客攻击后的信任重建**: 针对约1.5億美元损失，在短期内恢复准备金。宣布加强以提升透明度为目的的 POA（资产证明）披露。
- **重新进入印度市场**: 于2025年2月完成 FIU-IND 注册及罚款支付，转向受监管的合法运营。致力于在印度加密资产市场确立合规参与者地位。

## 8. 相关人物

- **Ben Zhou（CEO·创始人）**: 新加坡籍创业者。自2018年创立 Bybit 以来，一直以 CEO 身份主导经营。2025年2月黑客攻击后，在 X（原 Twitter）上亲自说明情况并主导迅速应对，广受关注。
- 其他主要高管·出资方详情因公开资料无法确认，故不予记载。
