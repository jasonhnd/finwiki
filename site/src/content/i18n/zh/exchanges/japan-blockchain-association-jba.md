# JBA — 日本区块链协会 (Japan Blockchain Association)

## Wiki route

This entry sits under [[exchanges/INDEX|exchanges index]]. Read it against [[exchanges/jcba-japan-crypto-business-association|JCBA — 日本暗号資産ビジネス協会]] for peer / contrast context and [[exchanges/jvcea-self-regulatory-overview|JVCEA — 認定自主規制協会]] for the broader system / regulatory boundary.

## 概要

**一般社团法人 日本区块链协会 (JBA / Japan Blockchain Association)** 是日本加密资产行业中**最老牌的行业团体**之一。**其前身是 2014-09 设立的"日本价值记录事业者协会 (JADA)"**,**于 2016-04 更名为现名称**,重组为统管区块链技术全般的协会。会员包括 VASP・区块链基础设施事业者・大学・研究机构。**交易所部会于 2018-04 与 [[exchanges/jcba-japan-crypto-business-association|JCBA]] 整合,成立 [[exchanges/jvcea-self-regulatory-overview|JVCEA]]**。此后 JBA 将重心转向技术・研究・教育・国际联动,作为区块链技术团体持续活动。

## 沿革

- **2014-09** — JADA(日本价值记录事业者协会)设立。Mt.Gox 崩溃后不久,作为行业自主监管核心,国内 VASP 发起成立
- **2016-04** — 更名为"日本区块链协会 (JBA)"。重组为统管区块链技术全般的协会
- **2017-04** — 因修订版资金结算法施行(虚拟货币交换业登记制度),交易所部会强化自主监管功能
- **2018-04** — JBA 交易所部会 + JCBA → 通过整合成立 **[[exchanges/jvcea-self-regulatory-overview|JVCEA]]**(FSA 认定自主监管协会)
- **2018 以后** — JBA 强化作为技术团体・研究团体的性格,JCBA 为产业团体,JVCEA 为认定自主监管,三分的角色分担确立

## 当前活动领域

- **技术验证 / R&D**:区块链技术的标准化・互操作性验证,用例 PoC 共享
- **教育 / 人才培养**:区块链教育研讨会,与大学・研究机构联动
- **国际联动**:GBBC・ISO/TC 307 (区块链国际标准化)等的日本侧窗口
- **政策建言**:从技术中立观点出发的政策评审(与 JCBA 切入点不同)
- **面向会员的学习会**:DeFi・L2・ZK・稳定币技术・节点运营等技术讨论

## 组织

- **法人形态**:一般社团法人
- **总部**:东京
- **代表理事**:从会员企业 / 有识之士中轮番选出(在官方网站随时公布)
- **会员区分**:正会员(法人)/ 个人会员 / 赞助会员 / 学术会员

## 与 JCBA / JVCEA 的分工(三层结构)

| 团体 | 角色 | 会员中心 | 法律定位 |
|---|---|---|---|
| **[[exchanges/jvcea-self-regulatory-overview|JVCEA]]** | 认定自主监管执行 | FSA 登记 VASP | 认定金融商品交易业协会 + 认定资金结算事业者协会 |
| **[[exchanges/jcba-japan-crypto-business-association|JCBA]]** | 产业政策・税制建言 | 加密资产业务全般 | 一般社团法人(任意团体) |
| **JBA** | 技术・研究・教育 | 区块链技术社区 | 一般社团法人(任意团体) |

许多主要 VASP 重复加盟 3 团体([[exchanges/jp-exchange-bitflyer|bitFlyer]]・[[exchanges/jp-exchange-coincheck|Coincheck]]・[[exchanges/jp-exchange-bitbank|bitbank]] 等)。JBA 作为技术团体的历史价值,在于 JVCEA 成立前的自主监管规则形成期,国内 VASP 自律制定反洗钱・冷钱包运营基准的遗产。这作为现行 [[exchanges/jp-vasp-aml-travel-rule-implementation|JVCEA Travel Rule 実装]] 和 [[exchanges/jp-vasp-security-audit-certification|VASP セキュリティ監査基準]] 的基础得到继承。

## 著名历史会员 / 代表者

- 初期 JADA 以 Mt.Gox 后的国内 VASP(bitFlyer・Coincheck・QUOINE・bitbank 等)为中心
- 更名后的 JBA 代表理事由 bitFlyer 加纳裕三等历任
