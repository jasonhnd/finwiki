# 国内加密资产 VASP 监管时间线 (2014-2026)

## 概要

日本的加密资产监管，以**国内大型事件** (Mt.Gox・Coincheck) 与**国际协调压力** (FATF Travel Rule・MiCA・GENIUS Act 等) 双重因素为触发点，逐步层叠累积而成。它是全球最早拥有全面 VASP 法律体系的司法管辖区之一，目前以**资金结算法 + 金商法 + 自主监管 (JVCEA)** 的三层结构运行。详细的登记制度参见 [[exchanges/fsa-vasp-registration-system]]，自主监管详情参见 [[exchanges/jvcea-self-regulatory-overview]]。

## 主要 监管里程碑 (按年代顺序)

- **2014-02** — Mt.Gox 破产 (详情 [[exchanges/jp-vasp-incident-history]])。形成监管立法的动机。
- **2016-05** — 修订资金结算法 成立。**全球首个** VASP 的法律定位。
- **2017-04** — 修订资金结算法 施行 → 「虚拟货币交换业」登记制度启动。
- **2018-01** — Coincheck NEM 580 億円流出事件 → 成为安全监管强化的契机。
- **2018-04** — JVCEA (日本虚拟货币交换业协会) 设立。
- **2018-10** — JVCEA → 升格为**认定资金结算事业者协会**。赋予自主监管权限。
- **2019** — 金商法 + 资金结算法 修订成立。名称由「虚拟货币」→ 统一为「**加密资产**」。
- **2020-05** — 修订金商法 施行 → 加密资产衍生品交易业编入金融商品交易业 ([[exchanges/jp-vasp-derivative-license-system]])。
- **2020-05** — 热钱包 **5% 限制** / 冷钱包 **95% 义务** / 客户资产分别管理义务。
- **2021** — [[fintech/fatf-travel-rule-overview]] 国内实装启动。
- **2022-06** — 修订资金结算法 成立 → 稳定币制度 ([[fintech/japan-epi-three-types-overview]]) 法制化。
- **2023-06** — 修订资金结算法 SC 监管 施行 ([[fintech/japan-stablecoin-regulatory-landscape]])。
- **2024-09** — JVCEA 会员数突破 50 社 (推定)。
- **2025+** — App Store 删除要求 ([[exchanges/fsa-foreign-exchange-warning-system]]) + JPYC 资金移动业 SC 发行启动。
- **2026** — Travel Rule 强化 + 旅行规则数据共享体制建设。

## 监管的三层结构 (截至2026 )

- **第一层 (法律)**: 资金结算法 (spot / SC) + 金商法 (衍生品) + 犯收法 (AML/CFT)
- **第二层 (省令・监督指针)**: FSA 监督指针 + 内阁府令
- **第三层 (自主监管)**: JVCEA 规则 + WhiteList 制度 (上市审查)

上位概念参见 [[fintech/japan-financial-regulation]]。
