# 国内加密资产 VASP 监管时间轴 (2014-2026)

## 概述

日本的加密资产监管，以**国内重大事件**（Mt.Gox·Coincheck）和**国际协调压力**（FATF 旅行规则·MiCA·GENIUS Act 等）为双重触发点，逐步累积演进。日本是全球最早建立综合性 VASP 法律体系的司法管辖区之一，目前以**资金结算法 + 金融商品交易法 + 自律规制（JVCEA）**的三层结构运营。注册制度详情参见 [[exchanges/fsa-vasp-registration-system]]，自律规制详情参见 [[exchanges/jvcea-self-regulatory-overview]]。

## 主要监管里程碑（按年代顺序）

- **2014-02** — Mt.Gox 破产（详情 [[exchanges/jp-vasp-incident-history]]）。构成监管立法的动因。
- **2016-05** — 修订版资金结算法成立。**全球首个** VASP 法律定义。
- **2017-04** — 修订版资金结算法施行 → "虚拟货币交换业"注册制度启动。
- **2018-01** — Coincheck NEM 580 億円 流出事件 → 安全监管强化的契机。
- **2018-04** — JVCEA（日本虚拟货币交换业协会）成立。
- **2018-10** — JVCEA → 升格为**认定资金结算事业者协会**，赋予自律规制权限。
- **2019** — 金融商品交易法 + 资金结算法修订成立。名称由"虚拟货币"统一改为"**加密资产**"。
- **2020-05** — 修订金融商品交易法施行 → 加密资产衍生品交易业纳入金融商品交易业（[[exchanges/jp-vasp-derivative-license-system]]）。
- **2020-05** — 热钱包 **5% 限制** / 冷钱包 **95% 义务** / 客户资产分别管理义务。
- **2021** — [[fintech/fatf-travel-rule-overview]] 国内实施启动。
- **2022-06** — 修订资金结算法成立 → 稳定币制度（[[fintech/japan-epi-three-types-overview]]）法制化。
- **2023-06** — 修订资金结算法稳定币监管施行（[[fintech/japan-stablecoin-regulatory-landscape]]）。
- **2024-09** — JVCEA 会员数突破 50 社（估算）。
- **2025+** — App Store 下架要求（[[exchanges/fsa-foreign-exchange-warning-system]]）+ JPYC 资金移动业 SC 发行启动。
- **2026** — 旅行规则强化 + 旅行规则数据共享体制整备。

## 监管三层结构（2026 时点）

- **第一层（法律）**：资金结算法（spot / SC）+ 金融商品交易法（衍生品）+ 犯罪收益转移防止法（AML/CFT）
- **第二层（省令·监督指针）**：FSA 监督指针 + 内阁府令
- **第三层（自律规制）**：JVCEA 规则 + 白名单制度（上市审查）

上位概念参见 [[fintech/japan-financial-regulation]]。
