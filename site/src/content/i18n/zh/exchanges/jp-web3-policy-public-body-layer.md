# 国内 Web3 / 加密资产 公共政策机构层（经产省 Web3 政策室 / 自民党 web3 PT / 内阁官房）

## Wiki route

This entry sits under [[exchanges/INDEX|exchanges index]]. Read it against [[exchanges/jcba-japan-crypto-business-association|JCBA — 日本暗号資産ビジネス協会]] for peer / contrast context and [[exchanges/fsa-vasp-registration-system|FSA 暗号資産交換業登録制度]] for the broader system / regulatory boundary.

## 概述

国内 Web3 / 加密资产政策由 **FSA（金融监管）+ 经产省（产业政策）+ 自民党 web3 PT（议员立法 + 党内政策）+ 内阁官房（数字战略横向协调）** 四极结构形成。FSA 负责 VASP 许可证、JVCEA 监督、资金结算法 / 金商法，而 **经产省 Web3 政策室负责产业培育、代币税制（法人课税）、初创企业支持、国际标准化**。自民党 web3 PT 在政治层面架桥两者，内阁官房 / 数字厅承担省厅横向协调。**2022 年"岸田 web3 宣言"**以来，政府整体确立了推进 web3 的姿态，每年"骨太方针""新资本主义"中持续明记 web3 / 加密资产政策。

## 主要公共政策机构

### **经济产业省 Web3 政策推进室（METI Web3 Policy Office）**

- **设置**：2022-07 （经济产业省商务信息政策局）
- **位置**：经济产业省 商务信息政策局 信息经济课 下属
- **职责**：Web3 产业培育、代币税制（法人课税）制度改革提案、Web3 初创企业支持、NFT 产业政策、海外拓展支持、国际标准化
- **官方网站**: https://www.meti.go.jp/policy/it_policy/web3/index.html
- **主要成果**：
  - 发表"Web3 白皮书"系列（年度）
  - 提案修订法人持有加密资产期末按市值课税制度（以2024 修正案落实为自家发行 + 第三方发行代币豁免形式）
  - 整备 DAO 指引（有限责任事业组合 LLC + DAO 结构）
  - NFT 指引（景品表示法 + 赌博罪的边界整理）
  - 向海外 web3 活动派遣省厅人员（丹佛 Eth Denver / ETHGlobal Tokyo 等）

### **自民党 web3 项目组（web3 PT）**

- **设置**：2022-01 （数字社会推进本部下属 PT）
- **位置**：自民党 政务调查会 数字社会推进本部 下属
- **历届座长 / 顾问**：以平将明 众议院议员（前数字副大臣）为核心，盐崎彰久 / 神田润一 / 山田太郎 / 川崎秀人等议员持续参与
- **主要成果**：
  - 年度发表"web3 政策提言 2022""web3 政策提言 2023""web3 政策提言 2024""web3 政策提言 2025"
  - 持续提案推进个人所得税分离课税（尚未实现）
  - 推动法人税法修正（2024 修正案的驱动力）
  - DAO 法制度化讨论
  - Web3 海外人才签证放宽
  - 利用监管沙盒
- **行业联动**：与 [[exchanges/jcba-japan-crypto-business-association|JCBA]] + [[exchanges/jvcea-self-regulatory-overview|JVCEA]] 密切协商，共同起草政策提言

### **内阁官房 / 数字厅（Digital Agency）**

- **数字厅**：2021-09 成立。负责我的号码 / 政府数字基础设施。**与 Web3 / 加密资产无直接管辖，但在数字 ID + 稳定币 + CBDC 等方面存在交叉**
- **内阁官房 新资本主义实现本部**：在"新资本主义"总体设计 + 实施计划中持续记载 web3 / 加密资产
- **内阁官房 数字田园城市国家构想实现会议**：地方创生 × web3 / NFT 应用（新潟县 山古志 NFT 等先例）

### **金融厅（FSA）— 监管端**

- **位置**：内阁府 外局
- **职责**：VASP 登录（[[exchanges/fsa-vasp-registration-system|FSA 登録制度]]）、JVCEA 认定自主规制监督、稳定币监管（修订资金结算法 2023-06）、金商法加密资产衍生品监管
- **金融审议会 工作组**：持续召开加密资产·稳定币相关 WG，审议修订法案前序
- **与经产省的关系**：监管 vs 产业培育的职能分工，两省局长级协商持续进行

### **日本银行（BOJ）**

- **职责**：CBDC（中央银行数字货币）实证实验（Stage 1: 2021-04  / Stage 2: 2022-04  / Stage 3  / 试运行 2024-）
- **与加密资产的关系**：官方立场为 CBDC ≠ 加密资产，但存在与民间稳定币（→ [[exchanges/jp-exchange-jpyc|JPYC]]）的竞合 / 互补论点

## 政策 4 极的职能分工与相互作用

| 极 | 主要职能 | 主要工具 |
|---|---|---|
| **FSA** | 监管执法·许可证 | 资金结算法 / 金商法 / 内阁府令 / [[exchanges/jvcea-self-regulatory-overview|JVCEA]] 认定 |
| **经产省 Web3 政策室** | 产业培育·税制提案 | 白皮书 / 补助金 / 监管沙盒 |
| **自民党 web3 PT** | 政治层·议员立法 | 年度政策提言 / 党内决定 / 预算编制压力 |
| **内阁官房 / 数字厅** | 省厅横向协调 | 骨太方针 / 新资本主义 / 数字田园城市 |

实际政策形成沿以下路径推进：行业（[[exchanges/jcba-japan-crypto-business-association|JCBA]] / [[exchanges/jvcea-self-regulatory-overview|JVCEA]]）→ 自民党 web3 PT → 经产省 / FSA → 内阁官房 / 执政党税调 → 税制改革大纲 → 提交国会。

## 主要政策议程（2024-2026）

1. **个人所得税分离课税** — 行业长期诉求，执政党税调持续审议。**2026 年年度税制改革大纲**的落实为焦点
2. **DAO 法制度化** — LLC + DAO 混合设计，公司章程 + 智能合约的法律定位
3. **NFT 二次流通法制** — 著作权 + 景品表示法 + 赌博罪的边界整理
4. **稳定币扩大** — 三菱 UFJ 信托 Progmat / [[exchanges/jp-exchange-jpyc|JPYC]] / 主要银行发行主体框架整理
5. **海外 web3 人才引进** — 签证放宽 + 高度专业人才居留资格
6. **AML / Travel Rule 强化** — → [[exchanges/jp-vasp-aml-travel-rule-implementation|犯収法 + FATF Travel Rule 国内実装]]
7. **代币税制法人化应对** — 2024 修正案的运用，尚未覆盖领域（Gas 费·空投等）的整理

## 国际比较 / 联动

- **美国** SEC / CFTC / 财务省 / FIT21  / CARF 1099-DA（→ [[fintech/carf-1099da-end-of-crypto-anonymity|CARF / 1099-DA]]）
- **EU** MiCA（2024-12 完全施行）
- **英国** FCA / HMT crypto regime
- **新加坡** MAS Payment Services Act
- **韩国** FSC + DAXA + Virtual Asset User Protection Act（2024-07）
- **香港** SFC VASP licensing

日本制度被评价为与 MiCA 相近的"综合·先行型"，在担任全球 G7 议长国时（2023），主导了 web3 / 稳定币国际一致性讨论。
