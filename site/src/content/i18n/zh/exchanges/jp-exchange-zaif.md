# 株式会社 Zaif — 日本加密资产交换业者 overview

## 1. 法人・股东

- **商号**：株式会社Ｚａｉｆ（英文：Zaif Inc.）
- **法人番号**：1120101054642
- **设立**：2016年（平成28年）4月12日（最初为株式会社 Fisco Coin）
- **资本金**：52,945,000円（令和7年10月8日时点）
- **登记上总店所在地**：大阪府岸和田市荒木町2丁目18番15号（FSA 注册地址）
- **事业所（实际运营总部）**：东京都港区南青山5-11-9
- **母公司**：株式会社ＪＮグループ（东证 Standard・证券代码 6634，旧称 Nexus Group → JN Group）
- **中间持股公司**：株式会社ＺＥＤホールディングス（大阪府堺市）— JN Group 的合并子公司。Zaif 的直接母公司
- **非上市**（Zaif 公司单体）；母公司 JN Group 于东证 Standard 上市

## 2. 牌照・注册状况

- **加密资产交换业**：近畿财务局长 第00001号（注册日 2017-09-29）
  - 注册局为近畿财务局这一点反映了其关西总部的根源。拥有关东以外注册局的 VASP 较为罕见
- **JVCEA 会员**：第一种（仅加密资产交换业），会员号 1014
- **加入协会**：一般社团法人 日本加密资产等交易业协会（JVCEA）

## 3. 经办币种

截至2026年5月，经办以下24个币种（官方支持页刊载）：

> BTC、ETH、SNPT、SKEB、ZPG、ZPGAG、ZPGPT、BORA、XYM、XEM、FSCC、CICC、NCXC、KAIA(KLAY)、POL(MATIC)、TRX、COT、BCH、MONA、ZAIF、CMS:XEM、CMS:ETH、XCP、JOC

- **特色**：MONA（MonaCoin）、XEM（NEM）、ZAIF 代币等国内社区发祥的币种丰富。FSCC（Fisco Coin）或 SKEB 等前所有者集团发行的代币也有残存。也经办 ZPG（ZeplinGo）、SNPT（SNPIT）等 GameFi・区块链游戏联动币种
- 此外于 2026年2月废止经办 MV・ROND・DEP 的3个币种。于 2026年4月完成同3个币种的日元兑换・返还作业

## 5. 市占・交易额

- 账户数・交易额的数字截至2026年5月该公司无公开
- 该公司的通知（2026年5月）中使用了「提供现货盘口交易的国内全部加密资产交换业者8社」这一表述，将8社认作国内现货盘口交易的竞争对手
- 据母公司 JN Group 的 IR，Zaif 在2026年11月期也预计计提亏损，正式扭亏为盈的前景为2027年11月期以后（crex-data.com 2026-04-23 刊载）。交易额规模被认为在国内中位以下（行业顶部为 [[exchanges/jp-exchange-bitflyer|bitFlyer]]・[[exchanges/jp-exchange-coincheck|Coincheck]]）
- 客户层：主要为国内 retail 投资者

## 6. 沿革・重大事件

### 前身法人：TechBureau 株式会社（法人番号 1120001184556）

| 时期 | 事件 |
|------|--------|
| 2014年前后 | TechBureau 株式会社（大阪府大阪市西区）设立。代表为朝山贵生。运营 Zaif 交易所 |
| 2015-16年 | Zaif 作为国内主要交易所，在 MONA・XEM 等山寨币交易中有存在感 |
| 2017-09-29 | 加密资产交换业注册（近畿财务局长 第00001号） |
| **2018-09-14** | **从热钱包流出约67億円相当的加密资产**（BTC・BCH・MONA 等。其中约45億円为从客户处保管的资产。据该公司新闻稿・FSA 参考资料）— 作为国内 VASP 重大事案，参见 [[exchanges/jp-vasp-incident-history|JP VASP インシデント史]]，对比参见 [[exchanges/coincheck-nem-hack-detailed-analysis|Coincheck NEM 流出事件詳細]] |
| 2018-09-25 | 金融厅对 TechBureau 株式会社发出业务改善命令（近畿财务局发出命令，金融厅网站刊载） |
| **2018-11前后** | **将 Zaif 交易所事业转让给 Fisco 虚拟货币交易所（现：Zaif Inc.）**。TechBureau 此后仍作为另一法人存续（2021年9月吸收合并子公司2社，至今仍以同一番号继续） |

### 现 Zaif Inc. 的法人：株式会社 Fisco Coin → 现在（法人番号 1120101054642）

| 时期 | 事件 |
|------|--------|
| 2016-04-12 | 株式会社 Fisco Coin 设立（Fisco 系） |
| 2016-07-01 | 无商号变更（据 NTA 变更履历）|
| 2018-11前后 | 从 TechBureau 取得 Zaif 交易所事业。作为「Fisco 虚拟货币交易所」继续运营 |
| 2020-11-02 | 商号变更：株式会社 Fisco 虚拟货币交易所 |
| 2021-11-02 | 商号变更：株式会社Ｚａｉｆ |
| 2023-11-15 | 商号变更：株式会社 CAICA Exchange（CAICA／CAICA 集团旗下时代） |
| 2023年前后〜 | 商号变更：恢复为株式会社Ｚａｉｆ（NTA 最新信息中现为「Zaif」） |
| **2025-02-03** | 从株式会社 Kushim 转移为株式会社 Nexus Group 的关联子公司（母公司变更。该公司新闻稿 2025-02-10 刊载） |
| **2025-07-15** | Nexus Group（现 JN Group）通过 ZED Holdings 完全取得 Zaif 股份（日本 M&A 中心刊载） |
| 2026年前后 | Nexus Group 商号变更为 JN Group（东证 Standard 6634） |

> **注**：TechBureau 与现 Zaif Inc. 是**不同法人**（法人番号不同）。Zaif 品牌・事业通过事业转让得以继承，但法人格无连续性。注册号（近畿财务局 第00001号）是在事业转让之际承继的。

## 7. 战略动向

- **2025-2026 年的主要动向**：
  - 从 2025年2月〜7月期间母公司由 Kushim → Nexus Group（后为 JN Group）转移，完全子公司化完成
  - 截至 2025年11月，JN Group 整体的集团合并99名（小规模持股公司体制）
  - 与 SNPIT（Snap to Earn 游戏）的协作活动（2026年4〜5月），GameFi 币种（SNPT）的上市
  - ETH 现货盘口交易手续费的返还活动（2026年5月）中标榜「国内8社中最大返还」
  - 展开 Zaif Card（带加密资产返现的卡）。于 2026年5月追加 ETH・XYM 返还对应
  - 2026年11月期预计持续亏损。据 JN Group IR，正式扭亏为盈的前景为2027年11月期以后
  - 株式会社 Fisco 发布 FSCC（Fisco Coin）的增值施策终止・销毁中止（2026年5月15日）

- 关于上市（IPO）的公开信息未确认

## 8. 相关人物

- **代表取缔役社长**：石原 直树（兼任 Zaif 公司及 JN Group 代表取缔役社长）
- **取缔役**：齐藤 洋介、大内 智则、安原 弘二（Zaif 公司）；齐藤亦兼任 JN Group 取缔役
- **监查役**：佐佐木 弘（Zaif 公司）；亦兼任 JN Group 社外监查役
- **JN Group 取缔役会长**：秋山 司
- **创业者经纬**：现法人（旧 Fisco Coin）的创业者为 Fisco 相关人物。原 Zaif 交易所的创业者・朝山贵生是 TechBureau（另一法人）的代表，与现 Zaif Inc. 无直接的人事连续性
