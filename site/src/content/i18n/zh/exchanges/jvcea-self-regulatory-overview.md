# JVCEA — 日本加密资产交易业协会 自律监管制度概览

> 认定金融商品交易业协会 · 认定资金结算业者协会（第二种）· 认定 2018-10-24

## 1. 概要

- **名称**：一般社团法人 日本加密资产交易业协会（旧称：日本虚拟货币交换业协会，2020-05 更名）
- **英文名**：Japan Virtual and Crypto assets Exchange Association
- **成立**：2018年4月（由前身2协会整合而成）
- **法律定位**：
  - 依据金融商品交易法的**认定金融商品交易业协会**
  - 依据资金结算法的**第二种认定资金结算业者协会**
- **认定日**：2018-10-24（获 FSA 双重认定）
- **目的**：完善加密资产交换业·衍生品交易业的自律监管规则，保护用户，促进行业健全发展
- **所在地**：东京都中央区

## 2. 组织·治理

- **会长**：从会员企业代表中选出（基于公开信息）
- **副会长**：多名（行业大型 VASP 代表轮流担任）
- **监事**：独立审计职能（含外部监事）
- **事务局**：常勤职员负责日常运营
- **专业委员会**：代币审查委员会、审计委员会、纠纷解决委员会、AML/CFT 委员会 等

## 3. 自律监管的5 项核心职能

1. **会员资格审查** — 审查入会申请 VASP 的体制·合规情况（FSA 注册前后）
2. **代币审查（白名单）** — 新上架币种的事前审查制度。会员在处理新币种前须取得 JVCEA 批准（分事前审查制和简化审查制两阶段，详情见 [[exchanges/jvcea-whitelist-token-listing|JVCEA whitelist token listing]]）
3. **审计·监控** — 对会员的业务·财务·资产分离管理进行持续核查（[[exchanges/jvcea-spot-volume-statistics-analysis|スポット取引量統計分析]] 中亦有体现）
4. **纠纷解决（ADR）** — 作为指定 ADR 机构，处理用户与会员之间的投诉·纠纷
5. **宣传·教育** — 行业报告、统计数据发布、金融素养普及活动

## 4. 会员类别

- **第一种会员（Type 1）**：加密资产交换业者·加密资产衍生品交易业者（已完成 FSA 注册的 VASP）
  - 例：[[exchanges/jp-exchange-bitflyer|bitFlyer #1002]]、[[exchanges/jp-exchange-coincheck|Coincheck #1017]]、[[exchanges/jp-exchange-bittrade|BitTrade #1003]] 等多家
  - 2024-09 时点 第一种会员数：30 社余家（公开信息以会员一览页面随时更新）
- **第二种会员（Type 2）**：赞助会员·相关业者（未完成 FSA 注册 / 申请准备中 / 基础设施业者·海外合作机构）
- **准确总会员数**：在 JVCEA 官方会员页面随时公布（如无时间点指定，则为"未公布"）

## 5. 与 FSA 的关系 — 双重监管结构

- 加密资产交换业者受到 **FSA 直接监管**（资金结算法 / 修订金融商品交易法）＋ **JVCEA 自律监管** 的双层监管
- FSA 将 JVCEA 定位为认定团体，将部分监管细节委托执行（例如：白名单、自律监管规则修订）
- 这是日本 VASP 监管被称为"行政＋自律监管混合型"的结构性特征的核心
- 相关：[[fintech/japan-financial-regulation|日本金融規制]] · FSA 为监管主管机关，JVCEA 为行业自律监管，两者共同构成双轮驱动
