# Coinone（코인원）— 韩国 CEX 中坚 / NH 农协银行实名账户 / 独立系 overview

## Wiki 路由

本条目位于 [[exchanges/INDEX|exchanges index]]。韩国同业 / 监管语境见 [[exchanges/korea-cex-five-pillars-comparison|韩国五大 CEX 制度比较]]；最接近的一线同业为 [[exchanges/cex-upbit-korea|Upbit]] / [[exchanges/cex-bithumb-korea|Bithumb]]；更广泛的系统和市场边界见 [[exchanges/global-cex-top10-comparison|global CEX top 10 comparison]]。

> 2014-08  设立 · 韩国 spot 市场 #3（推定 3% 前后）· **NH 농협은행 实名账户** · DAXA 共同设立 5 社 的 1  家之一 · **2022  上币 청탁 检察起诉事件**致经营层经历司法程序的历史

## 1. 法人・股东

- 商号：주식회사 코인원（Coinone Inc.）
- 设立：2014-08

- 总部所在地：서울특별시 영등포구
- 创业者：**차명훈（Daniel Lim、Cha Myung-hoon）**— 作为韩国早期加密货币业界主要人物之 1  而知名（Coinone 共同创业 / CEO）
- 主要股东：
  - **차명훈 创业者股份** 为核心
  - **데일리금융그룹**（Daily Financial Group、初期投资者）→ 后持股比例变动
  - **이상준** 等韩国系 IT / 金融投资家
  - **카카오뱅크系出资研议报道**（2018–2019）等，多次寻求战略性资本关系的历史
  - 详细股东构成因非上市而为部分披露
- IPO 研议：无直接公告，但若 Bithumb 的 KOSDAQ 上市推进成功则有跟进可能

## 2. 登记・牌照状况（韩国制度）

- **가상자산사업자 (VASP)** 登记：FIU 已登记（2021-09  韩国第 1  期登记类别）
- **ISMS / ISMS-P 认证**：持有 KISA ISMS-P
- **실명확인 입출금 계정**（实名账户）：与 **NH 농협은행（NH NongHyup Bank）** 签约
  - NH 농협은행 是韩国 5  大银行之 1 ，[[exchanges/cex-bithumb-korea|Bithumb]] 亦与同一 NH 농협은행签约 → 韩国 5  大 CEX 中 **NH 농협은행是与 2 社（Bithumb + Coinone）签约** 的唯一银行
  - 详细制度参见 [[exchanges/korea-cex-five-pillars-comparison|韓国 5 大 CEX 制度比較]]

## 3. 经办币种

- KRW spot 市场（作为韩国 5  大 CEX 之 1  而合法）
- 经办币种数：200+（与 [[exchanges/cex-upbit-korea|Upbit]] / [[exchanges/cex-bithumb-korea|Bithumb]] 并列的中等规模档次）
- 以 BTC / ETH / 主要山寨币为中心，Klaytn 系 / WEMIX / XPLA 等韩国系代币亦积极经办
- 上币审查：DAXA 共通基准 + Coinone 自有基准

## 4. 业务范围

- 现货 spot 交易（以 KRW pair 为中心）
- 衍生・margin 交易：**韩国 VASP 监管下不提供**
- staking：有限服务（**Cross**：staking / lending 相关产品的历史性提供，在监管约束下收缩）
- 法币入金：仅经由 NH 农协은행 实名账户
- **Coinone Pro**：面向专业交易者的 UI / API
- **Cross 海外汇款**：以加密货币为媒介的海外汇款服务（部分展开）

## 5. 市占・成交量

- **韩国 spot 市场份额**：约 3%（次于 [[exchanges/cex-upbit-korea|Upbit]] 70% / [[exchanges/cex-bithumb-korea|Bithumb]] 25% 的 #3  位，超过 [[exchanges/cex-korbit-korea|Korbit]] / [[exchanges/cex-gopax-korea|GOPAX]] 的中坚 #3）
- 24h 成交量：$20–200M（与市场波动率联动）
- 客户层：以韩国居民为中心（完整实施 KYC）

## 7. 战略动向

- **独立系经营**：与 [[exchanges/cex-upbit-korea|Upbit]]（카카오系・하나금융・한화等）/ [[exchanges/cex-korbit-korea|Korbit]]（SK Square）/ [[exchanges/cex-gopax-korea|GOPAX]]（Binance 部分介入）不同，延续 **创业者主导的独立系经营**。这既意味着独立性的优势，也意味着资本筹措力的局限
- **NH 농협은행 关系**：在与 [[exchanges/cex-bithumb-korea|Bithumb]] 共享同一银行的情形下，NH 농협은행侧 AML/CFT 责任分担协商为持续课题
- **DAXA 参与**：与 [[exchanges/cex-upbit-korea|Upbit]] / [[exchanges/cex-bithumb-korea|Bithumb]] / [[exchanges/cex-korbit-korea|Korbit]] / [[exchanges/cex-gopax-korea|GOPAX]] 一同在 DAXA 中统一上币基准。在 **2022  上币 청탁 事件后的治理重构** 中发挥主导作用
- **2024  利用者保护法 应对**：缴纳利用者保护基金、强化不公正交易监视、重构内部统制
- **韩国系代币主要上币场所**：作为 Klaytn / 后续 Kaia、WEMIX、XPLA 等韩国系 L1  / L2  代币的主要上币场所而延续
- **汇款服务 Cross**：以加密货币为媒介的海外汇款服务，应对监管环境变化

## 8. 业界中的定位

Coinone 作为 **韩国 spot 市场 #3  的中坚 CEX**，保持着 top 2（[[exchanges/cex-upbit-korea|Upbit]] / [[exchanges/cex-bithumb-korea|Bithumb]]）与下位（[[exchanges/cex-korbit-korea|Korbit]] / [[exchanges/cex-gopax-korea|GOPAX]]）之间的 **中间位置**。同时背负着独立系经营这一个性与 2022  上币 청탁 事件这一治理问题之重负。

在 [[exchanges/korea-cex-five-pillars-comparison|韓国 5 大 CEX 制度比較]] 之中，**创业者主导的独立系** 这一独特定位发挥着维持韩国 CEX 业界结构性多样性的作用。2022  事件以来的 DAXA 共通上币基准完善，也有 Coinone 主导的一面，其对全行业治理水平提升的贡献不容忽视。
