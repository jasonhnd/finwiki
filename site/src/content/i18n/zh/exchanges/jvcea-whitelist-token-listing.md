# JVCEA 经办币种白名单制度 — 代币上市流程 + Green/Yellow 列表

## 1. 制度概要

JVCEA（日本加密资产交易业协会）作为认定自律规制团体，承担对国内 VASP（加密资产交换业者）可经办币种进行审查及公开列表化的职能。依据修订后的《资金结算法》，新代币上市实际上必须经过 JVCEA 事前审查，FSA（金融厅）已将审查职能的大部分委任给协会。会员 VASP 依据同协会的经办方针规则，对新规及现有币种的经办判断作出决定。详情参见 [[exchanges/jvcea-self-regulatory-overview]]。

## 2. Green List（简易审查）vs Yellow List（个别审查）

自2022 年前后作为两级制度运行:

- **Green List** — 已有多家会员经办且流动性充足的主要代币（BTC / ETH / XRP / LTC / BCH / XLM / DOT 等）。新 VASP 仅需简易申报即可经办。该制度的导入使上市期间大致缩短至约 1  个月。
- **Yellow List（个别审查）** — 新代币或国内尚未经办的代币。须经个别审查 + 经办可否判定（期间 3-6  个月，视案件而定）。包含发行体尽职调查·市场流动性·合规确认。

## 3. 审查基准（8  大项）

依据 JVCEA 经办方针规则的审查项目:

1. 发行体·团队的公开性（谁在发行·所在地）
2. 价值支撑（经济合理性·实际使用情况）
3. AML/CFT 风险（匿名性·交易可追溯性）
4. 技术安全性（代码审计·黑客事件记录）
5. 法律定位（证券属性·相关法规）
6. 流动性（海外已上市·成交量）
7. 国内用户需求
8. 经办业者的风险管理体制

## 4. 国内 vs 海外 经办币种数差距

即便是国内现货领先的 [[exchanges/jp-exchange-coincheck]]，经办币种数也不过 30  个左右（截至2026-05 ）。而海外大型交易所 [[exchanges/jp-foreign-exchange-mexc]] 有 2,376  个，Binance 也有数百至 2,000  以上。Yellow List 个别审查负担和经办业者风险管理体制要求，构成制约国内币种数的结构性因素。

## 5. 主要案件（2025-2026）

- **ZPG / WILD** — 于 [[exchanges/jp-exchange-gmo-coin]] 实现 2026 年 国内首次上市（Wilder World 系）。专业平台 [[exchanges/jp-exchange-digital-asset-markets]] 经办 ZPG/ZPGAG/ZPGPT
- **KAIA** — Finschia 后继链，于 [[exchanges/jp-exchange-line-xenesis]] 开始经办
- **NIDT** — Nippon Idol Token，于 [[exchanges/jp-exchange-backseat-exchange]] 发行·流通
