# RWA（现实世界资产）代币化 × 中心化交易所上架

## 概述

RWA（Real World Asset）代币化 = 将实物资产（美国国债 / 房地产 / 大宗商品 / 私募信贷）作为链上代币发行的机制。以 BlackRock BUIDL（2024-03  上线 · 美国财政部货币市场基金）为催化剂，于 2024-2026  快速增长。RWA 总 TVL 约 $12-20B（2026  估计），代币化美国国债为核心细分领域。

## 主要参与方

- **BlackRock BUIDL**（以太坊）—— 通过 Securitize 发行 · 限美国机构投资者 · 5.4% 收益率 · AUM $1.5B（2025）
- **Ondo Finance OUSG/USDY** —— 以美国财政部 ETF（SHV）为担保 · OUSG（机构）+ USDY（零售）两级结构
- **Securitize** —— 代币化平台 · 承接 BUIDL / Apollo / Hamilton Lane 发行
- **Maple Finance** —— 链上私募信贷借贷
- **Centrifuge** —— 多资产 RWA 资金池
- **Provenance Blockchain（Figure）** —— 房地产 + 私募信贷

## 中心化交易所上架状况

- **国内（日本）**：无直接现货上架 · JVCEA 白名单未批准 · 三井物产（Zipangcoin）等 RWA 相关产品走另行渠道（证券化金融商品 / 信托）
- **海外（美国）**：Coinbase 为机构提供 OUSG 托管 · Binance 部分提供 RWA prime 服务
- **中心化交易所策略**：RWA 是机构 prime brokerage 的延伸 → 零售中心化交易所不如机构型中心化交易所具备优势

## 中心化交易所 × RWA 整合战略

- **托管**：BUIDL / OUSG 多由 Coinbase Custody / Anchorage 承接
- **交易**：仅限机构 OTC 场外市场（零售端有限）
- **结算**：通过 USDC / USDT 进行 24/7  结算
- **税制 + 监管**：美国 SEC 监管 + Reg D / Reg S 豁免 + 限合格机构投资者

## 国内 + 亚洲布局

- **Progmat**（三菱 UFJ 信托系）—— 信托型证券化金融商品国内最大（约 68% 市占率）
- **DCJPY** + **JPYC** —— RWA 结算补充候选
- **香港 SFC** + **MAS DPT** 正推进 RWA 整合实验（香港金管局 Project Ensemble 等）
- 日本零售 RWA 上架待法令修订（投资信托法 + 金融商品交易法）
