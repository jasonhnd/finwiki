# FSA 加密资产交换业登记制度 — 编号体系・财务局管辖・登记要件

## 制度概要

2017-04 施行的修订资金结算法，使日本领先世界引入了加密资产交换业的登记制。施行时点正在营业的事业者作为「视同业者」被准予暂时继续营业，并依次过渡到正式登记。2019 年的法律修订将以往的「虚拟货币」称谓统一为「加密资产」，业态名也改为「加密资产交换业」。所管为金融厅（FSA），实务窗口为各财务局。

## 登记编号体系

登记编号为「{财务局长} 第NNNNN号」格式。编号越小登记越早。主要管辖如下:

- **关东财务局长 第NNNNN号** — 占压倒性多数（首都圈总部）
  - 第00001号 = [[exchanges/jp-exchange-money-partners|マネーパートナーズ]]（国内首个登记）
  - 第00031号 = [[exchanges/jp-exchange-binance-japan|Binance Japan]]（截至 2026-05  的最新水平）
- **近畿财务局长 第NNNNN号** — 关西总部
  - 第00001号 = [[exchanges/jp-exchange-zaif|Zaif]]
  - 第00004号 = [[exchanges/jp-exchange-gaia|ガイア]]（BTM 专营）
- **北海道财务局长 第NNNNN号** — 道内总部事业者
- 其他，东海・九州等财务局也有管辖

## 登记要件 5 大支柱

1. **资本金 1,000 万円以上**（与净资产为正的要件并存）
2. **净资产不为负**
3. **业务执行体制的整备**（合规・内部审计・系统风险管理）
4. **用户财产的分别管理**（金钱以信托，加密资产以冷钱包为原则）
5. **AML/CFT 体制**（基于犯罪收益移转防止法的交易时确认・可疑交易申报）

## 从登记到加入 JVCEA 的流程

FSA 登记完成 → 向作为认定资金结算事业者协会的 [[exchanges/jvcea-self-regulatory-overview|JVCEA]] 申请入会 → 授予 JVCEA 会员编号（置于自律监管之下）。实务上，FSA 登记与 JVCEA 入会作为一套运作，相关法令体系构成 [[fintech/japan-financial-regulation|日本金融規制]] 的一部分。
