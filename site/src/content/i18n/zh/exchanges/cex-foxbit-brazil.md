# Foxbit —— 巴西 CEX #2 / 老牌 BRL 交易对流动性 overview

> 2014-09 设立 · 巴西老牌 CEX #2 · BRL 交易对长期流动性提供 · PIX 整合 · BCB / CVM 监管合规 · ABCripto 会员

## 1. 法人・股东

- 商号：**Foxbit Serviços Digitais Ltda.**
- 设立：2014-09（São Paulo, Brazil）
- 创始人：João Canhada（CEO）等共同创始人
- 主要股东：
  - 创始人股份维持核心
  - **ParaFi Capital** 等加密货币专业 VC 的战略出资（部分）
  - **Genoa Capital**（巴西本地 VC）
  - 因非上市，详细股东构成为部分披露
- IPO 考量：2021–2022 期有关于在巴西 B3 上市的考量报道，因加密市场下跌而搁置
- 融资规模：与 [[exchanges/cex-mercado-bitcoin-brazil|Mercado Bitcoin]] 的 $200M 软银轮相比相对小规模，持续独立系成长

## 2. 注册・牌照状况（巴西制度）

- **Banco Central do Brasil (BCB) 认可**：在 **Lei 14.478/2022（marco legal de criptoativos，2022-12 公布、2023-06 施行）** 下，受 BCB 的 PSAV（Prestadores de Serviços de Ativos Virtuais）认可制度对象
  - 2024–2025 年期：BCB 正式 PSAV 认可制度开始运行，Foxbit 也属注册流程对象
- **CVM**：属代币化证券的商品经办在 CVM 监管下（Foxbit 主要集中于 utility token / 主要加密货币，CVM 监督对象范围有限）
- **Receita Federal**：加密货币课税 / 交易报告义务
- **AML/CFT**：向 COAF（巴西版 FIU）的报告义务
- **ABCripto（Associação Brasileira de Criptoeconomia）**：巴西加密货币行业团体的主要会员，参与行业自主规制 / 监管对话

## 3. 经办币种

- BRL 现货市场（在巴西国内长期提供流动性）
- 经办币种数：100+（以主要加密货币为中心，币种数比 [[exchanges/cex-mercado-bitcoin-brazil|Mercado Bitcoin]] 更精简）
- 并设 USDT 市场
- 币种甄选为**保守 / 以主要币种为中心**（重视风险管理）

## 4. 业务范围

- 现物现货交易（以 BRL 交易对为中心）
- 衍生・margin 交易：有限 / 监管合规观察中
- staking：有限（托管式 staking）
- 法定货币入金：**PIX**（巴西中央银行即时结算系统）整合，经由 TED / DOC
- 面向机构：OTC desk、API trading 提供
- **Foxbit Pay**：加密货币结算相关服务（有限）

## 5. 市占・交易量

- **巴西现货市场份额**：#2（估计 10–20%，位于 [[exchanges/cex-mercado-bitcoin-brazil|Mercado Bitcoin]] 的 60–70% 之后）
- 24小时 volume：$10–100M（BRL 流动性换算，与市场波动率联动）
- 用户数：超 100 万人（官方公布基础）
- 全球 CEX 排名：top 100 圈内
- 因海外离岸 CEX（Binance、OKX 等）进军巴西，与 [[exchanges/cex-mercado-bitcoin-brazil|Mercado Bitcoin]] 一同经历国内 CEX 整体份额受压

## 6. 沿革・重大事件

| 时期 | 事项 |
|------|------|
| 2014-09 | Foxbit 设立（São Paulo） |
| 2015–2016 | 作为巴西早期 CEX 确立地位，形成与 [[exchanges/cex-mercado-bitcoin-brazil|Mercado Bitcoin]] 并列的两强结构 |
| 2017 | 巴西加密货币热潮期急速成长 |
| 2018–2019 | 银行账户关闭事案（多家巴西银行单方面关闭加密货币 CEX 账户，发展为诉讼系争） |
| 2020 | PIX 整合准备，应对巴西中央银行即时结算 |
| 2021 | 加密市场峰值，用户数急增，IPO 考量报道 |
| 2022-12 | **Lei 14.478/2022 公布**，巴西加密资产法成立 |
| 2023-06 | Lei 14.478/2022 施行，BCB 被指定为 PSAV 监督机关 |
| 2024 | BCB PSAV 认可制度正式运行，Foxbit 属注册对象 |
| 2024–2026 | 在全球 CEX（Binance、OKX 等）进军巴西的竞争下维持国内 #2 地位 |

## 7. 战略动向

- **保守币种战略**：不同于 [[exchanges/cex-mercado-bitcoin-brazil|Mercado Bitcoin]] 的 RWA tokenization / 多样商品展开，集中于**主要加密货币 + BRL 交易对流动性**的保守战略
- **PIX 整合**：通过巴西中央银行即时结算系统的完整整合，提供 24/7 出入金 UX
- **BCB / CVM 监管合规**：Lei 14.478/2022 施行后的 PSAV 注册 + CVM 代币化证券监管的双重应对，监管合规成本负担加重
- **ABCripto 行业团体活动**：积极参与巴西加密货币行业自主规制 / 监管对话，助力监管环境的稳定化
- **应对海外 CEX 竞争**：对 Binance、OKX、Bybit 等离岸 CEX 进军巴西，以**完全遵循本地监管 + PIX 即时结算 + 葡萄牙语支持**进行差异化
- **面向机构展开**：OTC desk、API trading、custody 相关服务的分阶段扩张

## 8. 在行业中的定位

Foxbit 作为**巴西加密货币行业的老牌 #2 CEX**，自 2014 年以来维持着与 [[exchanges/cex-mercado-bitcoin-brazil|Mercado Bitcoin]] 并列的两强结构。面对 Mercado Bitcoin 的软银融资 / RWA tokenization 攻势，Foxbit 以**保守币种甄选 + PIX 即时结算 UX + 独立系经营**持续差异化。

虽未及 [[exchanges/global-cex-top10-comparison|グローバル top 10]]，但在巴西国内作为与 Mercado Bitcoin 并列的两大平台之一角，构成**巴西加密货币市场的结构性两强体制**。Lei 14.478/2022 + BCB PSAV 认可制度的正式运行，一方面提高两家公司的监管合规成本，另一方面具有强化**国内 CEX 对海外离岸 CEX 竞争的制度性优势**的效果。
