# GOPAX（고팍스）— 韩国 CEX #5 / Binance 2023-02 过半数取得 → 因 FIU 抗议而部分后退 overview

> 2017-11 上线 · 旧 Streami（스트리미）· 韩国现货市场 #5（推定 1%）· **2023-02 Binance 收购 GOPAX 股份约 72%** → FIU 变更申报处理延迟 → 因韩国 FIU 的担忧而 **Binance 表决权稀释的妥协方案** 持续协商 · GoFi（GOPAX Earn）的 Genesis Global 破产敞口事件

## 1. 法人・股东

- 商号：주식회사 스트리미（Streami Inc.）— 控股 / 事业公司
- 设立：2015-04（Streami 设立）/ 2017-11（GOPAX 交易所推出）
- 总部所在地：서울특별시 강남구
- 创始人：이준행（Lee Joon-haeng）— Streami 创始人，从区块链汇款业务起步
- **股东结构（2026 年时点）**：
  - **Binance**：于 **2023-02-08** 宣布从 Streami 既有股东处取得 GOPAX 股份约 **72%**（经由 **Binance Asia Services Pte. Ltd.**）
  - 但韩国 FIU 的 **변경신고**（VASP 变更申报）处理 **长期持续协商中** → 正式的控股股东变更批准被延迟 / 部分化
  - 2024–2025 期：Binance 表决权稀释 / 强化韩国法人独立性等妥协方案持续讨论
  - 既有少数股东（初期投资者）也继续持有
- IPO 探讨：因 Binance 相关的监管不确定性，无具体公告

## 2. 登记・牌照状况（韩国制度）

- **가상자산사업자 (VASP)** 登记：FIU 已登记（2021-09 韩国第 1 期登记类别）
- **ISMS / ISMS-P 认证**：持有 KISA ISMS-P
- **실명확인 입출금 계정**（实名账户）：与 **전북은행（Jeonbuk Bank、JB금융그룹）** 签约
  - **전북은행是韩国 5 大银行（KB / Shinhan / NH / Hana / Woori）以外** 的 지방은행（地方银行）。GOPAX 是唯一提供地方银行系 KRW 现货的 VASP
  - 依赖 JB금융그룹（Jeonbuk Financial Group、KOSDAQ 上市）信用力的结构
- **2023-02 Binance 股份取得** 公告后：
  - 韩国 FIU 采取 **변경신고 처리지연**（实质性搁置）的姿态
  - **全球 Binance** 与美国 SEC / DOJ 达成和解（2023-11、$4.3B 罚款、CZ 退任）之后，韩国 FIU 立场更加审慎
- 详细制度参见 [[exchanges/korea-cex-five-pillars-comparison|韓国 5 大 CEX 制度比較]]

## 3. 经办币种

- KRW 现货市场（作为韩国 5 大 CEX 之 1 合法）
- 经办币种数：80+（韩国 5 大 CEX 中最保守）
- 以 BTC / ETH / 主要山寨币为中心
- 上市审查：DAXA 共通基准 + GOPAX 独自基准

## 4. 业务范围

- 现货 spot 交易（以 KRW 交易对为中心）
- 衍生・margin 交易：**在韩国 VASP 监管下不提供**
- **GoFi（GOPAX Earn）**：lending 产品，于 2022 末发生 **Genesis Global 破产敞口事故**（因 Genesis Capital 等全球 lending 业者破产的影响，发生 GoFi 提款冻结事态）
- staking：有限的服务
- 法定货币入金：仅经由 전북은행 实名账户
- B2B / 机构用：API 交易、OTC

## 5. 市占・交易量

- **韩国现货市场份额**：约 1%（韩国 5 大 CEX 中垫底 #5）
- 24h 成交量：$1–20M（与市场波动联动）
- 客户层：以韩国居民为中心（完全实施 KYC）
- GoFi 事态之后用户信任削弱，正在恢复中

## 7. 战略动向

- **Binance 收购后的治理谈判**：由于韩国 FIU 的审慎姿态，可能向 **Binance 表决权稀释 + 强化韩国法人独立性** 模式妥协。全球 Binance 与美国 SEC / DOJ 和解之后，韩国当局的担忧进一步升高
- **GoFi 偿付处理**：2022-09 GoFi 提款冻结之后的用户偿付 / restructuring 处理是 GOPAX 恢复信任的关键
- **전북은행 关系**：与非韩国 5 大银行的地方银行（전북은행）签订实名账户合约，体现了 GOPAX 的独特地位。银行方的 AML/CFT 责任分担问题直接影响 GOPAX 的业务可持续性
- **DAXA 参与**：与 [[exchanges/cex-upbit-korea|Upbit]] / [[exchanges/cex-bithumb-korea|Bithumb]] / [[exchanges/cex-coinone-korea|Coinone]] / [[exchanges/cex-korbit-korea|Korbit]] 一起在 DAXA 共通化上市基准
- **2024 用户保护法应对**：用户保护基金出资，强化不公正交易监视，重建内部控制
- **对 Binance 全球协同的期待**：存在利用 Binance 全球流动性 / 币种阵容 / 技术基础设施的可能性，但由于韩国 FIU 的审慎姿态，实现受限

## 8. 业界中的定位

GOPAX 虽是 **韩国 5 大 CEX 中垫底**，但作为 **全球 Binance 试图进入韩国分支的核心案例**，是监管当局 / 全球 CEX / 韩国国内 VASP 三角治理问题的代表性案例。

在 [[exchanges/korea-cex-five-pillars-comparison|韓国 5 大 CEX 制度比較]] 的语境下，它是 **5 大 KRW 现货 CEX 中唯一一个全球 CEX（Binance）试图成为控股股东的案例**，这显示了韩国监管当局对全球 CEX 进入国内有多审慎。2023-11 全球 Binance 与美国和解之后的谈判停滞，反映了韩国监管当局对信誉的担忧。

GoFi 事态在 [[exchanges/global-crypto-exchange-bankruptcy-comparison|글로벌 CEX 파산 비교]] 的语境下也很重要，是 **全球 lending 业者破产传染至韩国 CEX 子公司 lending 产品** 模式的案例。
