# 美国加密资产交易业务多层牌照制度

## 1. 制度概述

美国实行联邦 + 50 州的多层监管结构，运营全球最复杂的加密资产牌照制度。VASP（虚拟资产服务提供商）须组合取得以下牌照：

- **(a) 联邦 FinCEN MSB 注册** — AML/CFT 必要基础
- **(b) 各州 MTL**（Money Transmitter License，汇款业者牌照）**或 OCC 联邦信托特许** — 汇款及托管业务
- **(c) 纽约州经营时的 BitLicense** — 纽约州金融服务局单独设立的严格制度
- **(d) SEC / CFTC 个别监管** — 依证券性质、衍生性质分别适用

## 2. 4 牌照层级

- **FinCEN MSB**：联邦货币服务业注册。AML/CFT 申报义务。全部 VASP 必须具备的基础层
- **州 MTL**：50 州分别取得。大多数州均要求，全国铺开最多需 50 张牌照。各州净资产 / 保证金 / 申报标准不一
- **纽约州 BitLicense**：2015-06 纽约州 DFS 制度。被评为业界最严，取得者约 30 社家（Coinbase / Gemini / bitFlyer USA / Circle 等）。Kraken 已退出
- **OCC 联邦信托特许**：联邦信托银行认可。Anchorage Digital 于 2021-01 取得美国首张。可豁免 50 州 MTL，是机构托管的核心牌照

## 3. SEC vs CFTC 管辖权竞争

若代币被认定为"证券"则归 SEC 管辖（依 Howey 测试标准），若被认定为"大宗商品"则归 CFTC 管辖（BTC / ETH 为代表）。认定不明确的状态持续多年，但 **CLARITY Act / FIT21**（2024 年下院通过）正尝试明确二分法。若被认定为证券交易所，还需进行 ATS 登记或持有经纪自营商牌照。

## 4. 主要业者牌照组合

- **Coinbase**（纳斯达克：COIN）— BitLicense + 多州 MTL + SEC 注册经纪商
- **Kraken** — 州 MTL + FinCEN MSB · 未持 BitLicense（已退出纽约）
- **Gemini** — BitLicense + 纽约州信托特许
- **Anchorage Digital** — OCC 联邦信托特许（美国首家）
- **Circle** — 州 MTL + FinCEN MSB + 2025 年 OCC 信托特许（USDC 发行方）

## 5. 国际比较

与日本 FSA / JVCEA（单一联邦监管）、韩国 FSC / FIU（单一监管）不同，美国联邦 + 州双重监管结构为全球最复杂。全国铺开的监管合规成本达数千万美元规模，导致生态系统向少数大型机构集中。与香港 SFC 和欧盟 MiCA 等统一框架形成鲜明对比，是典型的"碎片化"模式。
