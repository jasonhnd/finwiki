# JVCEA Type 2 (第二种会员) 制度 — 赞助会员 4 社分析

## 1. 制度概要

JVCEA ([[exchanges/jvcea-self-regulatory-overview]]) 将会员区分设定为 2 种类:

- **第一种会员 (Type 1)** — 暗号资产交换业 + 暗号资产衍生品交易业者。FSA 登录必须。
- **第二种会员 (Type 2 / 赞助会员)** — FSA 登录 不要。 (a) FSA 登录准备中、(b) 关联事业者、(c) 面向机构事业者 等 允许多样的业态。

## 2. Type 2 加盟意义

- **对 JVCEA 自主规制的遵循表态** — 对行业标准的自发性对齐
- **行业信息共享 + 游说渠道** — 对 FSA / 金融厅 制度动向的早期把握
- **为将来 Type 1 迁移作准备** — 登录申请前的关系构建
- **交易所与机构客户的接续路径** — 与第一种会员 + 机构投资者的接续点

## 3. 主要 Type 2 会员 (2026-05 时点 4 社)

- **#2027 Saxo Bank 证券** (Saxo Bank A/S 丹麦母公司) — FSA 暗号资产交换业登录为零、加盟理由未公布、作为海外 prime broker 的准备?  → [[exchanges/jp-exchange-saxo-bank-japan]]
- **#2033 Gaudiy Financial Labs** (Gaudiy Group · 索尼G + 万代南梦宫 + 集英社 + 讲谈社 134 億累计融资) — Web3 IP 粉丝经济平台、以 NFT 为中心  → [[exchanges/jp-exchange-gaudiy]]
- **#2034 Trek Labs Japan** (Backpack 全球母公司 · 原 FTX 系 Armani Ferrante 创业) — FSA 登录申请准备中、进入日本初期阶段  → [[exchanges/jp-exchange-trek-labs-japan]]
- **#2037 Laser Digital Japan** (野村 HD 瑞士子公司 100%) — Komainu Board 并表 ([[exchanges/jp-custody-komainu]])、机构托管 + 交易基础设施  → [[exchanges/jp-exchange-laser-digital-japan]]

## 4. 行业含义

- **海外大手的准备据点化** — 美/欧/HK 系玩家以 Type 2 构建“日本市场前哨基地”(Trek Labs / Saxo / Laser Digital)
- **生态系统企业** — Web3 / IP / NFT 系 (Gaudiy) 亦可加盟，业态具灵活性
- **Type 1 迁移门槛** — FSA 登录需数亿日元成本 + 半年至数年审查。 以 Type 2 维持关系的同时作准备

## 5. 比较 — Type 1 vs Type 2

若 Type 1 为“事业者”，则 Type 2 为“行业生态系统”。 并非如韩国 KISA 认证或香港 SFC 牌照那样的阈值型 (取得/不取得 的二值)，而是允许行业参与灵活性的日本特有结构。 详细母公司关系参见 [[exchanges/jp-vasp-parent-company-map]]。
