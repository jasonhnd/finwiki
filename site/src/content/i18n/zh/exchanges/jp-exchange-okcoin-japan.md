# OKCoin Japan 株式会社（OKJ） — 日本加密资产交换业者 overview

> FSA 注册号 关东财务局 第00020号 · JVCEA 会员 #1023 · 注册日 2020-03-30

## 1. 法人・股东

- **商号**: OKCoin Japan 株式会社
- **英文名**: OKCoin Japan K.K.
- **法人番号**: 8030001122337
- **设立**: 2017年9月19日
- **总店**: 〒105-0001 东京都港区虎之门1-2-10 5层
- **母公司**: OKX（旧 OKEx / OKEx Holdings Limited）集团。作为全球法人 OKX（塞舌尔登记，旗下持有 Okcoin International Ltd. 等）的日本本地法人设立。出资比例・中间持股公司结构未公开。
- **上市状态**: 非上市（母公司 OKX 作为集团整体也非上市）
- **备注**: 服务品牌由旧称「OKCoinJapan」（2020年8月20日开始服务）于 2024年8月20日刷新为「OKJ」（与服务启动4周年纪念同时）。商号・FSA 注册名称仍为 OKCoin Japan 株式会社，无变更。

## 2. 牌照・注册状况

- **加密资产交换业**: 关东财务局长 第00020号（注册日 2020年3月30日）
- **JVCEA 会员种类**: 第一种会员 #1023（仅加密资产交换业）
- **金融商品交易业**: 未取得（无衍生品交易）
- **电子支付手段等交易业**: 未取得
- **资金移动业**: 未取得

## 3. 经办币种

- **经办币种数（2026年5月时点）**: 销售所 49个币种 / 交易所（盘口交易）规模大致相同
- **主要币种**: BTC、ETH、XRP、SOL、ADA、DOT、DOGE、AVAX、LTC、BCH、LINK、DAI、SHIB、PEPE、TON、TRX、TRUMP、SUI 等
- **值得关注的币种**: OKB（OKX 生态原生代币）、FNCT（Financie 代币）、OAS（Oasys）、PLT（Palette）、DEP（Digital Entertainment Asset）— 日本国产・亚洲系项目币种相对较多
- **特色**: 也积极采用山寨币・新兴链代币（KAIA、ASTR、IOTX 等）。与全球 OKX 在世界经办的数百个币种相比，面向日本的大幅受限（受 FSA 审查流程影响）。

## 4. 业务范围

- **Spot 交易所（盘口交易）**: 有。Maker/Taker 手续费体系（Lv1: Maker 0.07% / Taker 0.14%，月间交易量超过 30億円 的 Lv7 面议）
- **销售所（OTC dealer）**: 有。全币种手续费免费（含点差）
- **衍生品（期货・保证金交易）**: 无
- **质押**: 有。对象 12个币种（SOL、TRX、SEI、ADA、APT、ASTR、AVAX、ETH、IOST、QTUM、SUI、XTZ）。有灵活・定期方案。全资产冷钱包管理
- **加密资产借贷（Crypto Lending）**: 有
- **Flash Deals（限时抢购）**: 有
- **自动定投（定期买入）**: 有
- **大额 OTC 服务**: 有（面向法人・大额，BTC・ETH 等）。在2026年4月追加 ETH，在2026年5月追加卖出订单功能
- **法人尊享服务**: 有（「OKJ 法人 Premier」，2025年4月10日开始）
- **NFT**: 无
- **法定货币入金渠道**: 仅银行转账（经由 GMO Aozora Net Bank 24/365，手续费免费。便利店入金・快速入金不支持）
- **出金限制**: 加密资产出库上限 1日3億円相当

## 5. 市占・交易额

- **交易额・账户数**: 非公开
- **客户层**: 主要以零售（个人）为中心。通过面向法人的尊享服务也开始对应机构投资者・法人层
- **竞争内地位**: JVCEA 26社之中的中坚。与主要做市商（[[exchanges/jp-exchange-bitflyer|bitFlyer]]・[[exchanges/jp-exchange-coincheck|Coincheck]]・[[exchanges/jp-exchange-sbi-vc-trade|SBI VC Trade]] 等）相比规模被认为较小，但经办币种数量属国内上位级别。账户数・交易额未公开。VASP 母公司的地理分布参见 [[exchanges/jp-vasp-parent-company-map|JP VASP 親会社マップ]]。

## 6. 沿革・重大事件

| 年月 | 事项 |
|------|------|
| 2017年9月 | OKCoin Japan 株式会社 设立 |
| 2020年3月 | 取得 FSA 加密资产交换业注册（关东财务局 第00020号） |
| 2020年8月 | 开始提供加密资产交易服务「OKCoinJapan」 |
| 2022年7月 | 开始 AVAX（Avalanche）盘口交易。同年 SHIB（2023年2月）等山寨币相继上市 |
| 2023年5月 | 开始公布「OKJ BTC Index」（中间价的透明化） |
| 2023年5月 | 开始 eKYC（最短5分钟开户）服务 |
| 2024年8月 | 将服务品牌由「OKCoinJapan」刷新为「OKJ」（4周年纪念，商号无变更） |
| 2025年1月 | 与 Kaia（旧 Klaytn）缔结合作伙伴关系（促进 Line Mini Dapp 利用） |
| 2025年4月 | 开始提供「OKJ 法人 Premier」面向大额法人的服务 |
| 2026年4月 | 大额 OTC 追加 ETH，其后追加卖出订单功能 |
| 2026年5月 | 开始经办 BERA（Berachain） |
- **行政处分历史**: 在公开资料范围内未确认（需持续确认）
- **黑客・系统障碍**: 无重大事案公开（需持续确认）
- **母公司 OKX 的监管风险**: 全球 OKX 及 Aux Cayman Islands Ltd.（前名 Okcoin International Ltd.）于 2024年与美国 FinCEN・CFTC・DOJ 达成和解（超过 1億6,000万美元的制裁金）。虽非日本法人的直接处分，但作为集团信用风险需留意。

## 7. 战略动向

- **区块链直接委托**: 在2026年5月，随着与 Dawn Labs（Solana 相关）的合作实施对 Solana 验证者节点的委托。深化对 PoS 基础设施的参与
- **OKJ Academy（教育）**: 在2026年5月时点预告启动。零售客户培育路线
- **Kaia 战略联动**: 作为与 Line BITMAX 的差异化，扩充对 LINE 经济圈（Kaia 链）的接触点
- **多币种・新兴币种积极采用**: 从公开信息可读取出以领先同行的币种扩充（国产・Web3 游戏系代币等）作为增长驱动力的姿态

## 8. 相关人物

- **主要股东**: 非公开。集团母公司为 OKX（塞舌尔）系
- **创业者故事**: 非公开
