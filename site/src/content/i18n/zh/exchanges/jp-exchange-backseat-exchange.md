# BACKSEAT暗号資産交換業株式会社 — 日本加密资产交换业者 overview

> FSA 登记号 关东财务局 第00026号 · JVCEA 会员 #1031 · 登记日 2021-04-16

## 1. 法人・股东

- **商号**:BACKSEAT暗号資産交換業株式会社(旧称:株式会社coinbook)
- **英文名**:BACKSEAT Exchange Inc.
- **设立**:2018年5月18日
- **本店所在地**:〒102-0082 東京都千代田区一番町7番地1
- **母公司**:BACKSEAT株式会社(100% 出资)。BACKSEAT株式会社的详细资本构成未公开。
- **资本金**:2,327百万円(含资本准备金)(截至 2026年4月30日)— 于 2026年1月・4月相继实施增资。
- **结算期**:3

关于商号:以「BACKSEAT暗号資産交換業」的形式将业种名嵌入公司名中,这一点别具一格。公司网站首页标榜「Web3」。旧品牌「coinbook」作为交易平台(service.coinbook.co.jp)的服务名沿用至今,通过 URL 重定向引导至本公司官网。

## 2. 牌照・登记状况

- **加密资产交换业**:关东财务局长 第00026号(登记日 2021-04-16)
- **金融商品交易业(加密资产衍生品)**:未公开 — JVCEA 会员种别仅为「加密资产交换业」(JVCEA #1031 第一种会员)
- **电子结算手段等交易业 / 资金移动业**:未登记(未公开)
- **加入协会**:一般社团法人日本暗号资产等交易业协会(JVCEA)、一般社团法人日本数字分散式金融协会、一般社团法人日本暗号资产商业协会
- **加入团体**:一般社团法人JPCrypto-ISAC(2026年4月加盟)、一般社团法人日本CSIRT协议会(2026年1月加盟)

## 3. 经办币种

- 在交易所页面(service.coinbook.co.jp)可确认的公开价格对为 **ADA/JPY** 及 **NIDT/JPY** 共 2 对(2026年5月时点的显示)。
- 服务一览中存在销售所、交易所、OTC、IEO 各服务分类,但经办币种总数在官网上未明示。
- **NIDT(Nippon Idol Token)**:有 IEO 实施实绩,并持续进行季度定期信息披露(2026年4月披露确认)。

## 4. 业务范围

- **销售所**(加密资产便捷交易):面向个人・法人
- **交易所**(spot):面向个人・法人
- **OTC(相对交易)**:有记载
- **IEO**:有实绩(NIDT)
- **质押(staking)**:服务菜单中有记载,但已于 2026年2月25日宣布停止质押及外部出入库(详细条件已公开)
- **法定货币入金**:日元(银行转账)。于 2026年1月变更入金账户名义(旧 coinbook → BACKSEAT 名义)。
- **个人・法人均对应**:官网明确指引法人账户开设。

## 5. 市占・交易量

- 月度・年度交易量、账户数 **未公开**。
- 在 JVCEA 33 社中为第一种会员 #1031(按会员编号顺序从后发方算起第 3 位)。FSA 登记号 第00026号 为 2021年登记的后发组。
- 从交易所显示的币种仅限 ADA・NIDT 共 2 对,以及质押・外部出入库的停止通知(2026年2月)等情况推断,推测为 **偏利基 / 偏机构的定位**,但无公开的定量依据。主流币种数量比较参见 [[exchanges/jp-exchange-bitflyer|bitFlyer]] / [[exchanges/jp-exchange-coincheck|Coincheck]] / [[exchanges/jp-exchange-gmo-coin|GMOコイン]](IEO 制度整体见 [[exchanges/jp-ieo-system-and-cases|JP IEO 制度・事例]])。
- 客户层:官网明确以个人・法人双方为对象。亦提供面向机构的 OTC。

## 7. 战略动向

- **2026年相继增资**:通过 2026年1月・4月共 2 次的增资将资本金积累至 2,327百万円,可推测有强化业务基础的意图。但增资的目的・出资方未公开。
- **安全・合规体制的整备**:相继加盟 JPCrypto-ISAC(2026年4月)・日本CSIRT协议会(2026年1月)等网络安全领域的行业团体。
- **质押・外部出入库停止(2026年2月)**:是业务收缩还是体制重审未公开。
- **标榜 Web3 **:公司网站的宣传语明确标示「Web3」。将区块链技术的社会落地作为经营目标。
- IPO・海外拓展・具体合作的公开信息未获确认。
