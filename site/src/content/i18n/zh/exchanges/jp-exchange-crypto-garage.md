## 1. 法人・股东

- **商号**：株式会社 Crypto Garage（英文：Crypto Garage, Inc.）
- **设立**：2018年9月
- **总店所在地**：东京都涩谷区惠比寿西 2-19-9 弗朗塞斯大厦
- **法人编号**：6011001124112
- **出资结构**（公开部分）：
  - 株式会社数字车库（Digital Garage · 东证 Prime 4819）为最大出资方。创业时的合资伙伴为东京短资株式会社。2021年10月以野村控股株式会社为认购方实施了第三方配股增资，野村 HD 作为新出资方参与。
  - Blockstream 也刊载于其企业官网的 "Backed By" 栏目，资本・技术合作伙伴关系已公示。
  - 各公司的具体出资比例未公开。

数字车库将该公司定位为金融科技・区块链领域的战略子公司，派遣董事 2 名（林 郁・踊 契三）。东京短资也派遣董事 2 名（寺田 寿明・加藤 岬造）。

## 2. 牌照・登记状况

- **加密资产交易业**：关东财务局长 第00029号（登记日：2021年6月29日，依据资金结算法）
- **金融商品交易业**：未登记（JVCEA 第一种会员 #1035 仅为加密资产交易业，无衍生品登记）
- **电子结算手段等交易业 / 资金移动业**：未登记（公开部分）
- **JVCEA**：第一种会员 #1035（仅加密资产交易业）
- **Global Digital Finance**：加盟（行业自律团体）

## 3. 经办币种

经办币种数量及具体清单在官网・公开文件中未逐一列举，未公开。

从服务设计推断的币种特性：
- **Bitcoin (BTC)** 为核心资产（R&D 主题为 Bitcoin / Lightning / Liquid Network，以 BTC 计价的 OTC・结算为核心）
- SETTLENET PRO 提供「日元计价代币（JPY 计价）× 链上 DvP 结算」，使用日元计价代币（内部结算货币）
- 该公司的 R&D 涵盖 Discreet Log Contracts（DLC）以及 Liquid Network 侧链资产
- 由于并非面向零售的多币种交易所，而是机构特化型，故经办币种数量推测为少而精

## 4. 业务范围

该公司**并非面向零售的现货交易所**，而是专注于面向机构・法人的数字资产基础设施的业者（自称："regulated institutional digital assets infrastructure company"）。主要服务线如下：

| 服务 | 内容 |
|----------|------|
| **SETTLENET PRO** | 加密资产业者间（交易商间）市场中的中介・OTC 结算平台。提供非托管型 DvP 结算、日元计价代币（该公司发行）、时点净额结算功能 |
| **SETTLENET CUSTODY** | 符合国内监管的面向机构投资者・法人的数字资产保管服务（2023年4月启动）。具备保险化分别管理、质押（2024年3月新增）、OES（Off Exchange Settlement）功能 |
| **SETTLENET OTC** | 面向法人・事业法人的加密资产 OTC 销售所（2023年4月启动）。运用全球 LP 网络，提供邮件/电话对应的高级服务 |
| **R&D** | 关于 Bitcoin・Lightning・Liquid Network 的研究开发，对 Discreet Log Contracts（DLC）协议的开源贡献，P2P 衍生品交易的研究 |
| **Blockchain Dev** | 面向法人客户的区块链开发受托 |
| **mahola** | 区块链・后端服务（API），2024年10月正式发布 |

**法定货币入金渠道**：仅限法人（银行转账）。无便利店入金・快捷入金等面向零售的渠道。

## 5. 市占・交易额

- 交易额・账户数均**未公开**
- **客户群**：专注于机构投资者・事业法人・加密资产交易所・流动性提供商・高净值人群・家族办公室・资管公司。不提供面向一般零售投资者的服务
- 市场定位：未参与零售交易量竞争的**利基（机构基础设施）**玩家。推测以数十家法人客户为对象，但定量数据未公开

## 7. 战略动向

- 一贯保持**面向机构 × Bitcoin 原生**的路线。SETTLENET PRO 在加密资产交易商间市场这一国内独此一类的利基定位中确立地位
- **与 Blockstream / Liquid Network 的协作**：官网 "Backed By" 栏目刊载有 Blockstream。R&D 主题专注于 Bitcoin・Lightning・Liquid Network，暗示其作为 Liquid Network 侧链技术在日本市场主要普及伙伴的角色（与 Blockstream 的具体出资额・合同内容未公开）
- **野村集团 × Komainu 联动**（2023年11月）：在面向机构投资者的托管服务领域的协作。通过与 [[exchanges/jp-custody-komainu|Komainu]]（[[securities-firms/nomura-hd|野村 HD]] × Ledger × CoinShares 的合资托管方）的联动，推进全球层级的面向机构基础设施建设（该公司的定位在 [[exchanges/jp-institutional-custody-three-pillars|JP institutional custody 三本柱]] 和 [[exchanges/jp-crypto-market-maker-otc-layer|JP crypto market maker / OTC レイヤー]] 中详述）
- **web3 开发受托的扩张**：2024年以后，面向日产汽车・日本英语检定协会・SKY Perfect JSAT・岩手县紫波町等非 crypto 领域的区块链开发支援案件增加。通过 mahola API 使面向开发者的业务线多元化
- **最近 12 个月（以 2025年5月为基准）**：与 Pacific Meta 共同提供加密资产持有咨询（2025年8月），IEEE Blockchain 2025 论文采纳（2025年12月）。无关于 IPO・海外拓展的公开发布
