# 越南加密货币交易所格局 — SBV 监管模糊 / ONUS / Remitano / 灰色市场 overview

## Wiki route

本条目位于 [[exchanges/INDEX|exchanges index]] 下。可与亚洲和新兴市场 CEX 条目对照阅读。

> 越南在 Chainalysis Global Crypto Adoption Index 中是世界顶级的采用国家之一 · 但 **SBV（State Bank of Vietnam）不承认加密货币为法定货币・支付手段、且无 VASP 牌照制度** · 主要经营者为 **ONUS / Remitano** + offshore CEX（Binance, OKX, MEXC, Bybit 等）

## 1. 监管环境概要（截至 2026 年）

### SBV（State Bank of Vietnam）的官方立场

- **2017 年通函**：加密货币 **不被承认为法定货币・支付手段**（作为 payment instrument 使用属违法）
- **2018 年总理令 (Decision 1255/QĐ-TTg)**：指示着手研究加密资产相关法律的制定
- **2024–2026 年期**：**虚拟资产 / 数字资产法（Luật về tài sản số）** 的起草推进中（财政部 / MoF 负责起草,国会持续审议）
- **截至 2026 年**：**尚未确立正式的 VASP 牌照制度**,持有・交易本身并非明示禁止（**支付违法,投资无明示监管**的灰色地带）

### 课税

- 加密货币售出收益课税 **无明示制度**（PIT / CIT 适用范围仍在讨论）
- 个人营业所得课税 / VAT 课税的适用与否因经营者而不确定
- 2025 年期：MoF 发布 **加密资产课税制度** 的建设方针,实施在 2026 年 之后

### AML/CFT

- **金融行动特别工作组（FATF）** 成员,对 Travel Rule 等全球标准的应对滞后
- 越南加密资产反洗钱监督由 SBV / MoF / Public Security Ministry 多个当局并立,责任分担模糊

## 2. 主要经营者画像

### ONUS（旧称 VNDC）

- 商号：**ONUS Pro Co., Ltd.**（旧 VNDC Pte. Ltd.）
- 设立：2018（Vietnam）/ 2021 进行品牌重构（VNDC → ONUS）
- 用户数：**超 400 万人**（越南最大级,基于官方公布）
- 特征：
  - 发行・运营 **VNDC 稳定币**（VND 1:1 锚定）
  - 经由链上提供 VND-pair 流动性（越南国内最大级）
  - 作为新加坡法人采用 offshore 结构,面向越南国内用户以移动 App 为中心
  - 整合 Web3 wallet、DeFi 联动、NFT marketplace 等功能
- 监管应对：
  - 研究应对新加坡 MAS（Monetary Authority of Singapore）的 DPT 监管（参见 [[exchanges/sg-mas-dpt-licensing-overview|MAS DPT licensing overview]]）
  - 在越南国内以无须明示 license（无 VASP 制度）的方式运营

### Remitano

- 商号：**Babylons Solution Co., Ltd.** 或关联新加坡法人
- 设立：2014（Vietnam 起源,新加坡 据点）
- 特征：
  - 以 **P2P 交易平台** 为起源（在马来西亚 / 尼日利亚 / 越南等新兴市场展开）
  - 在越南国内 P2P 流动性上有长期实绩
  - 以 **P2P escrow** 模式实现经由银行账户的 VND 入出金
- 2024–2025 期：**安全事故报道**（hot wallet 外流疑虑）,运营透明性存忧

### offshore CEX 在越南市场的存在

- **Binance**：越南用户数众多（经由官方 P2P 功能）,为 SBV 警告对象（多次）
- **OKX / Bybit / MEXC**：类似的做法,完全实施 KYC + P2P / direct deposit 路径并存
- **Huobi（HTX）/ Gate.io / KuCoin**：在越南市场的存在持续

## 3. VND-pair 流动性的结构

- **直接 VND pair**：明示提供的 CEX 有限（以 ONUS / Remitano 为中心）
- **间接 VND**：offshore CEX 以 **USDT / USDC 对为基轴**,经由 **P2P escrow** 或 **OTC** 实现 VND ↔ USDT 兑换
- **USDT P2P 流动性**：越南市场流动性最高的对的 1  个,达全球 top 10 级
- **稳定币**：VNDC / VNST 等 VND 锚定稳定币部分流通,但无监管批准

## 4. 灰色市场动态

- **支付违法 + 投资合法（灰色）**：SBV 明示表态作为支付手段的使用属违法,但投资 / 持有本身无明示禁止
- **Chainalysis Global Crypto Adoption Index**：越南在 **2021–2023 年期为世界 #1**,其后亦持续位列 top 5 圈内。**尽管立法滞后,实需却占压倒性优势**
- **汇款需求**：来自海外居住越南人的 remittance 用途（尤其来自欧美・台湾・韩国・日本的汇款）广泛使用 USDT
- **OTC 经纪商**：河内 / 胡志明市的店头 OTC 经纪商（咖啡馆 / 手机店等）作为 VND-USDT 兑换的物理据点发挥功能
- **税务风险**：对交易利益的课税义务不明确,存在税务当局事后追缴的风险

## 5. 在业界中的定位

越南处于 **尽管监管滞后,却是世界顶级加密货币采用国** 这一特异定位。作为比较对象：

- **[[exchanges/cex-coindcx-india|インド]]**：1% TDS + 30% capital gains + FIU-IND PMLA 形成 **明示监管 + 重课税** → 国内 CEX 萎缩
- **[[exchanges/cex-mercado-bitcoin-brazil|ブラジル]]**：Lei 14.478/2022 + BCB PSAV 形成 **明示的 VASP 牌照制度** → 国内 CEX 成长
- **越南**：**监管几近未建 + 课税不明确** → 灰色市场实需爆发,但国内 CEX 的机构化有限

作为 [[exchanges/global-vasp-regulatory-comparison-matrix|グローバル VASP 規制比較]] 中 **监管空白地带的最大案例**,是新兴国家加密货币市场形成模式的代表例。今后 **虚拟资产 / 数字资产法** 的施行（预计 2026–2027 ）对市场结构的影响,将成为最大的论点。

## 6. 战略动向

- **ONUS 的国际拓展**：应对新加坡 MAS 监管 + 东南亚地区拓展（马来西亚 / 印尼 / 菲律宾等）
- **VNDC / VNST 等 VND 稳定币**：处于未批准状态下的流通,监管风险持续
- **offshore CEX 的监管风险**：SBV / Public Security Ministry 查处非法 P2P 经纪商的案例（2024–2025）,Binance / OKX 的 P2P 功能监管风险持续
- **MoF 虚拟资产法案**：2024–2026 国会持续审议,若施行将成为 **国内 VASP 牌照制度的首次成立**,预计带来市场结构的大规模重组
