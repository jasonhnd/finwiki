# Korbit（코빗）— 韩国首家 CEX / SK Square 子公司 / 4-pair-bank exclusivity overview

> 2013-07 设立 · **韩国首家正规 CEX** · SK Square 子公司 · 新韩银行（신한은행）实名账户 · 韩国 spot 市场 #4（估计 2% 前后）· **DAXA 共同设立 5 社家中的 1 家**

## 1. 法人・股东

- 商号：주식회사 코빗（Korbit Inc.）
- 设立：2013-07
- 本店所在地：首尔特别市江南区（서울특별시 강남구）
- **母公司**：**SK Square 株式会社**（SK 集团旗下 ICT 投资控股公司、KOSPI 上市）
  - 2017-09：NHN Entertainment（后来的 NHN）以约 1,200 亿韩元收购 Korbit（韩国 IT 大厂在加密货币行业的首笔大型 M&A）
  - 2021-08：**SK Square**（当时由 SK Telecom 人事分立新设）**从 NHN 处购入 Korbit 股权 35%**（规模约 900 亿韩元）→ 分阶段扩大持股比例
  - 现状：SK Square 为**持股 65% 以上的控股股东**，NHN 作为少数股东留存
- 创始人：Tony Lyu（Korbit Inc.）— 以韩国首家 CEX 设立者而为人所知
- IPO 探讨：无直接公告，但作为 SK Square 的上市子公司，可经由母公司接入资本市场

## 2. 注册・牌照状况（韩国制度）

- **가상자산사업자 (VASP)** 注册：FIU 已注册（2021-09  韩国第 1  期注册批次）
- **ISMS / ISMS-P 认证**：持有 KISA ISMS-P
- **实名确认存取款账户**（실명확인 입출금 계정，实名账户）：与**新韩银行（신한은행，Shinhan Bank）**签约
  - 新韩银行是韩国 5  大银行中的 1  家（KB / Shinhan / NH 农协 / Hana / Woori）
  - Korbit 是维持与新韩银行签约的 **韩国 KRW spot 5 社** 之 1  家（**4-pair-bank exclusivity**：[[exchanges/cex-upbit-korea|Upbit]]-K-Bank、[[exchanges/cex-bithumb-korea|Bithumb]]-NH 农协、Korbit-Shinhan、[[exchanges/cex-coinone-korea|Coinone]]-NH 农协、[[exchanges/cex-gopax-korea|GOPAX]]-全北银行（전북은행））
- 详细制度参见 [[exchanges/korea-cex-five-pillars-comparison|韓国 5 大 CEX 制度比較]]

## 3. 经办币种

- KRW spot 市场（作为韩国 5  大 CEX 之 1  家而合法）
- 经办币种数量：100+（与 [[exchanges/cex-upbit-korea|Upbit]] 的 180+、[[exchanges/cex-bithumb-korea|Bithumb]] 的 200+ 相比偏保守）
- 以 BTC / ETH / 主要山寨币为中心
- 上市审查：DAXA 共同基准 + Korbit 独自基准
- **2025–2026  期**：USDT market 扩张、追加全球币种

## 4. 业务范围

- 现货 spot 交易（以 KRW pair 为中心）
- 衍生品・margin 交易：**在韩国 VASP 监管下不提供**
- staking：有限服务
- 法定货币入金：仅经由新韩银行实名账户
- **NFT marketplace**：Korbit NFT（2021 launch），在 Klaytn 系 NFT 经办上领先
- **B2B / 面向机构**：Korbit Pro、API trading、custody 相关服务

## 5. 市占・交易量

- **韩国 spot 市场份额**：约 2%（继 [[exchanges/cex-upbit-korea|Upbit]] 70% / [[exchanges/cex-bithumb-korea|Bithumb]] 25% / [[exchanges/cex-coinone-korea|Coinone]] 3% 之后的 #4  位次）
- 24h volume：$10–100M（与市场波动率联动）
- 客户层：以韩国居民为中心（完全实施 KYC）
- 扭亏为盈：在 2024–2025  期实现多个季度盈利（SK Square 合并口径下的盈利化）

## 7. 战略动向

- **SK Square 战略子公司定位**：在 SK 集团的 ICT 投资组合中，被定位为加密货币 / Web3  相关业务的核心子公司。与 SK Telecom 的 **T-Wallet / SK Coin** 等集团 Web3  战略联动的可能性
- **新韩银行关系**：实名账户契约的长期稳定性，与新韩银行的关系符合**新韩金融集团（신한금융그룹）的数字资产战略**
- **DAXA 参与**：与 [[exchanges/cex-upbit-korea|Upbit]] / [[exchanges/cex-bithumb-korea|Bithumb]] / [[exchanges/cex-coinone-korea|Coinone]] / [[exchanges/cex-gopax-korea|GOPAX]] 一同在 DAXA 共同化上市基准，分散监管应对成本
- **NFT / Klaytn 系展开**：作为韩国系 L1 / L2  代币（Klaytn 后续 Kaia、WEMIX、XPLA 等）的主要上市场所而持续
- **2024  用户保护法应对**：缴纳用户保护基金、强化不公正交易监视、重建内部控制
- **保守经营**：相比 Upbit / Bithumb 那样的市场份额竞争，优先考虑 **SK Square 合并口径下的盈利化 + 监管应对成本最小化**

## 8. 行业中的定位

Korbit 作为**韩国首家正规 CEX** 拥有历史地位，并通过成为 SK Square 子公司，保持着**韩国大型 ICT 集团旗下唯一 VASP** 这一独特定位。虽然与 [[exchanges/cex-upbit-korea|Upbit]] / [[exchanges/cex-bithumb-korea|Bithumb]] 的头部 2 社 拉开了较大差距，但凭借 **5  大 VASP 之 1 社 这一制度性地位 + SK Square 后盾**，长期生存能力较高。

在 [[exchanges/korea-cex-five-pillars-comparison|韓国 5 大 CEX 制度比較]] 之中，位于继 Upbit-K-Bank / Bithumb-NH 农协的头部 2  对之后的**第 3  集团（Korbit-Shinhan + Coinone-NH 农协 + GOPAX-全北银行）**，以市场份额 1–3% 的水平稳定持续营业的结构。
