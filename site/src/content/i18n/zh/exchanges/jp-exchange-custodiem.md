# 株式会社 Custodiem —— 日本加密资产交换业者 overview

> FSA 登录号码 关东财务局 第00002号 · JVCEA 会员 #1003 · 登录日 2017-09-29

## 1. 法人・股东

- **商号**：株式会社 Custodiem（英文：Custodiem, Inc.）
- **法人号码**：7010401115356（国税厅法人号码公表网站已确认）
- **设立日**：2014年11月25日（作为 QUOINE株式会社设立）
- **总店所在地**：东京都港区赤坂 9-7-1  Midtown Tower 37F
- **母公司**：株式会社 bitFlyer Holdings（2024年收购完成）
- **代表取缔役**：名古路 博史

## 2. 牌照・登录状况

- **加密资产交换业**：关东财务局长 第00002号（登录 2017-09-29）
- **金融商品交易业（第一种）**：关东财务局长（金商）第3297号
- **JVCEA 会员**：第一种会员（会员号码 1003）· 加密资产交换业 + 加密资产衍生品交易业

## 3. 经办币种

官方网站未公表经办币种的现行阵容（自 2024年末以来，优先推进既有账户向 bitFlyer 的移管）。前身 FTX Japan 时代经办包含 BTC・ETH 在内的主要币种。详细的现行币种列表未公表。

## 4. 业务范围

- **加密托管（加密资产预托）**：自 2024年以后作为核心事业新规展开
- **加密资产交换业**：现物交易（面向既有客户的服务持续中）
- **衍生品交易**：依据金融商品交易业牌照应对（运行状况未公表）
- **未来事业（计划）**：日本国内法制整备后的加密资产现物 ETF 相关服务
- **账户移管**：以客户同意为基础推进向 bitFlyer 的移管
- **信息泄露应对**：2026年4月，公表并应对因云环境误配置导致的客户信息泄露可能性

## 5. 市占・交易量

- 交易量・账户数均未公表
- 自 2024年以后，优先进行既有客户的资产返还及向 bitFlyer 的移管，新规客户获取停止中
- 行业定位：持有牌照号码 第00002号、历史上第 2 个登录业者，但现事业规模处于 retail 服务缩减阶段

## 6. 沿革・重大事件

| 年月 | 事由 |
|------|------|
| 2014年11月 | QUOINE株式会社 设立（自设立当初即为东京据点）|
| 2014年 | 可用法定货币购买加密资产的 Quoinex（coin exchange）上线 |
| 2015年 | 追加保证金交易・开始提供移动应用 |
| 2016年 | 从 VC 筹集 2,000万 USD 资金 |
| 2017年 | 经金融厅登录为加密资产交换业者（**关东财务局 第00002号**）; Qryptos 上线 |
| 2018年 | 整合 Quoinex + Qryptos 的 **Liquid** 平台上线 |
| 2020年4月 | 总店所在地变更（中央区京桥 → 千代田区） |
| 2022年4月 | **FTX Trading 收购旧 Liquid 公司（FTX 日本控股）及 QUOINE** → 商号变更为「FTX Japan株式会社」（法人号码保持同一不变） |
| 2022年11月 | FTX Trading 破产（FTX Japan 在日本法的保护下持续业务・资产隔离） |
| 2023年 | 开始・实施向客户的资产返还服务 |
| 2024年7月 | 公表 **株式会社 bitFlyer Holdings 将 FTX Japan 完全子公司化**（缔结股份转让合同） |
| 2024年8月26日 | 商号变更为「**株式会社 Custodiem**」（法人号码 7010401115356 同一） |
| 2024年9月13日 | 总店迁移（千代田区 → 港区赤坂 Midtown Tower 37F） |
| 2026年4月 | 公表并致歉因云环境误配置导致的客户信息泄露可能性 |

**法人连续性的确定**：经国税厅法人号码公表网站，由一次源确认 QUOINE株式会社 → FTX Japan株式会社 → 株式会社 Custodiem 为同一法人号码 7010401115356 。FSA 第00002号（原本以 QUOINE 名义登录）亦原样承继。

## 7. 战略动向

- **核心事业转换（2024-2025）**：从加密资产交换业（retail 经纪）向**加密托管（面向机构投资者的加密资产预托）**的转向
- **[[exchanges/jp-exchange-bitflyer|bitFlyer]] 集团内角色分工**：bitFlyer = 销售所・交易所，Custodiem = 托管 + 未来的 ETF 相关服务（FTX Japan 的客户资产返还经纬参见 [[exchanges/ftx-japan-100pct-return-case-study|FTX Japan 100% 返還ケーススタディ]]）
- **加密资产现物 ETF 准备**：以日本国内法制度整备为条件，计划提供现物 ETF 相关服务（法制整备时期不确定，市场影响参见 [[exchanges/btc-spot-etf-japan-impact|BTC spot ETF 日本 impact]] 及 [[exchanges/jp-institutional-custody-three-pillars|JP institutional custody 三本柱]]）
- **向 bitFlyer 的账户移管**：持续推进既有 Custodiem 账户客户的同意制移管
- **自有资本规制比率**：于 2026年3月31日付披露（详细数值参见 PDF）

## 8. 关联人物

- **代表取缔役**：名古路 博史（官方网站记载）

- **收购方**：株式会社 bitFlyer Holdings（详细董事构成未公表）
