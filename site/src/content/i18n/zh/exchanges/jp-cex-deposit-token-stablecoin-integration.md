## 概要

2022-06 修订资金结算法使 EPI（电子结算手段）三型 — **信托型 / 银行型 / 资金移动业者型** — 的制度施行，确立了日元锚定 stablecoin 和存款代币的法律定位。既有 VASP 虽非 EPI 发行体，但在 2025-2026 期间经办整合正迅速推进。CEX × EPI 的联动模型正进入推动国内市场流动性 hub 化的局面。

## 主要 EPI / 存款代币 玩家

- **[[exchanges/jp-exchange-jpyc|JPYC 株式会社]]**（资金移动业者型 EPI · JVCEA Type 1 #1042）— 发行日元锚定 stablecoin **JPYC** + 运营自社 JPYC EX 市场。零售导向。
- **[[fintech/jp-stablecoin-progmat|株式会社 Progmat]]**（信托型 EPI platform · 三菱 UFJ 信托 49% 出资）— 多巨行 SC 发行平台 + Project Pax 跨境结算。机构导向。
- **[[fintech/jp-stablecoin-dcjpy|DCJPY]]**（株式会社 DeCurret DCP · 银行型 存款代币）— IIJ + SBI + 邮储 + KDDI + NTT + MUFG 出资。2024-08 商用开始。银行间结算导向。

## CEX 侧的应对

- **[[exchanges/jp-exchange-sbi-vc-trade|SBI VC トレード]]** — USDC / EP USDC 经办开始 + EPI 发行体预定（唯一持有三种牌照）
- **[[exchanges/jp-exchange-bitflyer|bitFlyer]]** / **[[exchanges/jp-exchange-gmo-coin|GMO コイン]]** / **[[exchanges/jp-exchange-bitbank|bitbank]]** — JPYC 经办研讨中
- **[[exchanges/jp-exchange-crypto-garage|Crypto Garage]]** — Progmat SETTLENET PRO 合作 + 提供面向机构的结算基础设施

## 业务含义

- **CEX → EPI 经办**: 通过追加客户 JPY ⇔ stablecoin 兑换功能，可扩大国内结算 / 海外汇款渠道。将 CEX 的业务范围扩展至 payment rails。
- **存款代币（DCJPY）**: 以银行直接联动 + 机构结算利用为中心。无需经由 CEX（Banking infra 直连型）。与 CEX 既竞争又互补。
- **竞争关系**: Progmat（机构）vs JPYC（零售）vs DCJPY（银行间结算）的 3 极竞争结构。各自针对不同的客户层・用途优化。
- **§501 denylist 相关**: 在 2026 [[fintech/genius-act-501-denylist-mandate|GENIUS Act §501]] 影响下，日本 EPI 作为 "[[fintech/three-circles-stablecoin-mra-framework|MRA 互認]]" 候选浮现。

## 战略路线图

2026 H2 以后，各 CEX 将依次解禁 EPI 系 stablecoin 经办。随着 CEX × EPI 整合推进，国内市场整体的流动性 hub 化预计将加速。存款代币（DCJPY）作为 B2B 机构结算 layer 并行，二层结构（零售 EPI + 机构存款代币）将走向完成。
