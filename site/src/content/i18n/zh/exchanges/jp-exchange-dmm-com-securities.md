# 株式会社DMM.com证券 — 日本金融商品交易业者（加密资产衍生品）overview


## Wiki 路径

本条目归属于 [[exchanges/INDEX|exchanges index]]。请结合 [[exchanges/fsa-vasp-registration-system|FSA 暗号資産交換業登録制度 — 番号体系・財務局管轄・登録要件]] 阅读以了解相邻背景，并结合 [[fintech/japan-financial-regulation|日本金融規制 — トークン・暗号資産・決済に関する法体系]] 了解更广泛的系统边界。

> FSA 关东财务局长（金商）第1629号 · JVCEA 第一种会员 #1043（仅加密资产衍生品交易业）· 第一种金融商品交易业登记日 2007-09-30
> **注意**：与 DMM Bitcoin（株式会社DMM Bitcoin · 已停业）为**不同法人**。该公司承担加密资产现货交换业，于 2025-03 停业。本法人以 FX/CFD/股票为主力，加密资产仅有**衍生品（CFD）**登记。→ [[exchanges/jp-exchange-dmm-bitcoin]]

## 1. 法人・股东

- **商号**：株式会社DMM.com证券（英文：DMM.com Securities Co.,Ltd.）
- **设立**：2006年12月6日
- **总部所在地**：〒103-6010 东京都中央区日本桥2-7-1 东京日本桥大厦
- **资本金**：98億円（资本准备金 73億9千万日元）
- **代表取缔役**：谷川 龙二
- **母公司**：合同会社 DMM.com（DMM.com 集团）— 无上市
- **法人编号**：未公布（官网未刊载）

## 2. 牌照・登记状况

- **第一种金融商品交易业**：关东财务局长（金商）第1629号（登记日 2007年9月30日）
- **第二种金融商品交易业**：含于同 第1629号（2017年6月27日登记）
- **商品期货交易业**：含于同 第1629号（2011年1月1日登记）
- **JVCEA 第一种会员 #1043**：仅加密资产衍生品交易业（无现货 spot / 加密资产交换业的登记）
- 电子支付手段等交易业・资金移动业：无登记（未公布）
- **加入协会**：
  - 日本证券业协会（协会员编号1105）
  - 一般社团法人 金融期货交易业协会（协会员编号1145）
  - 日本商品期货交易协会
  - 一般社团法人 第二种金融商品交易业协会（协会员编号480）
  - 一般社团法人 日本加密资产等交易业协会（JVCEA 协会员编号1043）
- **自有资本规制比率**：419.6%（截至 令和8年3月 = 2026-03）

## 3. 经办币种（加密资产）

- **无加密资产现货交易**。JVCEA 登记仅为「加密资产衍生品交易业」。
- 官方 CFD 服务（DMM CFD）所经办的 22个品种由 **股指 8个品种 ＋ 商品 CFD 14个品种** 构成。截至 2026-05-18，官网上未能确认有加密资产 CFD 品种的刊载。
- 公司概要页面在业务内容中明确记载了「加密资产相关场外衍生品交易」，与 JVCEA 登记一致。加密资产 CFD 的实际提供状况判断为「已登记・未公开服务（或停止中）」（未公布）。

## 4. 业务范围

该公司的主力业务为以下 4大支柱：

| 服务 | 内容 |
|---------|------|
| DMM FX | 场外外汇保证金交易 |
| DMM CFD | 股指 8个品种 + 商品 CFD 14个品种 = 共 22个品种（场外衍生品） |
| 股票交易（DMM株）| 对应国内股・美国股・IPO 的上市有价证券等交易 |
| DMM バヌーシー | 赛马基金（商品投资关联业务） |

- 加密资产现货（spot）：**无**（曾由不同法人 DMM Bitcoin 承担，但已于 2025-03 停业）
- 杠杆交易（加密资产）：有 JVCEA 登记，但官方服务页面上加密资产 CFD 品种未刊载（截至 2026-05-18）
- 法定货币入金渠道：银行汇款・快速入金（对应全国约 340家金融机构）
- 质押・借贷・NFT：不支持
- 法人服务：无公开信息

## 5. 市占・交易量

- 账户数・月度交易量：未公布（披露资料未公开）
- 自有资本规制比率 419.6%（2026-03末）— 财务健全性维持高水平
- 同业内地位：在 FX・CFD 领域与 GMO Click 证券・外汇 dot com 并列大手。作为加密资产衍生品交易业，在 JVCEA 第一种会员 33 社中，属于无现货・衍生品专营的业者类别（13社中的 1社）。加密资产 CFD 的实际经办量未公布。

## 6. 沿革・重大事件

| 年月 | 事项 |
|------|------|
| 2006-12 | 设立（作为证券公司开始业务） |
| 2007-09-30 | 作为第一种金融商品交易业在 FSA 登记（关东财务局长（金商）第1629号） |
| 2011-01-01 | 追加登记商品期货交易业 |
| 2017-06-27 | 追加登记第二种金融商品交易业 |
| 时期不明 | 作为 JVCEA 第一种会员登记「加密资产衍生品交易业」（JVCEA 会员编号1043） |
| 2024-05-31 | **关系集团公司 DMM Bitcoin（不同法人）发生约 482億円相当的加密资产非法流出** |
| 2024-12-02 | DMM Bitcoin 宣布停业・宣布向 SBI VC 交易迁移。DMM 集团的加密资产现货交换业消失 |
| 2025-03-08 | DMM Bitcoin 服务全面终止。因 DMM.com证券与 DMM Bitcoin 为不同法人，故对其有无直接业务影响未公布（无官方声明） |
| 2026-03 | 披露自有资本规制比率 419.6%（财务健全） |

**与 DMM Bitcoin（不同法人）停业的关系**：

DMM.com证券与 DMM Bitcoin 为不同法人（地址同为东京日本桥大厦，但法人格・登记号不同）。DMM Bitcoin 的停业（2024-12 宣布・2025-03 完成）是否直接影响了 DMM.com证券的 JVCEA 加密资产衍生品登记・业务存续，未能确认有官方公告（截至 2026-05-18）。但因作为 DMM 集团的加密资产现货交易业务已消失，集团整体的加密资产业务大幅缩减。

## 7. 战略动向

- 近期的公开公告以 FX・CFD・股票交易的功能增强为主。未能确认有加密资产领域的新增服务公布（截至 2026-05-18）。
- DMM CFD 已从 2024年以前的 14个品种扩充至 22个品种（新增商品 CFD 8个品种）。官网未记载加密资产 CFD 的追加计划。
- DMM バヌーシー（赛马基金）作为他社所无的特色服务继续提供。
