# 株式会社 DMM.com 证券 — 日本金融商品交易业者（加密资产衍生品）overview

> FSA 关东财务局长（金商）第1629号 · JVCEA 第一种会员 #1043（仅加密资产衍生品交易业）· 第一种金融商品交易业注册日 2007-09-30
> **注意**：与 DMM Bitcoin（株式会社 DMM Bitcoin · 已停业）为**另一法人**。该公司经营加密资产现货交换业，于 2025-03  停业。本法人以 FX/CFD/股票为主力，加密资产仅有**衍生品（CFD）**注册。→ [[exchanges/jp-exchange-dmm-bitcoin]]

## 1. 法人・股东

- **商号**：株式会社 DMM.com 证券（英文：DMM.com Securities Co.,Ltd.）
- **设立**：2006年12月6日
- **总店所在地**：〒103-6010  东京都中央区日本桥 2-7-1  东京日本桥大厦
- **资本金**：98億円（资本准备金 73億9千万 日元）
- **代表取缔役**：谷川 龙二
- **母公司**：合同会社 DMM.com（DMM.com 集团）— 未上市
- **法人番号**：未公示（官网未刊载）

## 2. 牌照・注册状况

- **第一种金融商品交易业**：关东财务局长（金商）第1629号（注册日 2007年9月30日）
- **第二种金融商品交易业**：同 第1629号 内含（2017年6月27日 注册）
- **商品期货交易业**：同 第1629号 内含（2011年1月1日 注册）
- **JVCEA 第一种会员 #1043**：仅加密资产衍生品交易业（无现货 spot / 加密资产交换业注册）
- 电子结算手段等交易业・资金移动业：无注册（未公示）
- **加入协会**：
  - 日本证券业协会（协会员番号 1105）
  - 一般社团法人 金融期货交易业协会（协会员番号 1145）
  - 日本商品期货交易协会
  - 一般社团法人 第二种金融商品交易业协会（协会员番号 480）
  - 一般社团法人 日本加密资产等交易业协会（JVCEA 协会员番号 1043）
- **自有资本规制比率**：419.6%（令和8年3月 末 = 2026-03）

## 3. 处理币种（加密资产）

- 加密资产**无现货交易**。JVCEA 注册仅为"加密资产衍生品交易业"。
- 官方 CFD 服务（DMM CFD）处理的 22 个币种由 **股价指数 8 个币种 + 商品 CFD 14 个币种** 构成。截至 2026-05-18，官网上未能确认加密资产 CFD 币种的刊载。
- 公司概要页面在事业内容中明记有"加密资产相关店头衍生品交易"，与 JVCEA 注册一致。加密资产 CFD 的实际提供状况判断为"已注册・非公开服务（或停止中）"（未公示）。

## 4. 业务范围

该公司的主力事业为以下 4 大支柱：

| 服务 | 内容 |
|---------|------|
| DMM FX | 店头外汇保证金交易 |
| DMM CFD | 股价指数 8 个币种 + 商品 CFD 14 个币种 = 共 22 个币种（店头衍生品） |
| 股票交易（DMM 股）| 国内股・美国股・IPO 对应的上市有价证券等交易 |
| DMM 巴努西（Banusy）| 赛马用马基金（商品投资相关业务） |

- 加密资产现货（spot）：**无**（曾由另一法人 DMM Bitcoin 承担，但于 2025-03  停业）
- 杠杆交易（加密资产）：有 JVCEA 注册，但官方服务页面未刊载加密资产 CFD 币种（截至 2026-05-18）
- 法定货币入金渠道：银行转账・快捷入金（对应全国约 340 家金融机构）
- 质押・借贷・NFT：不对应
- 法人服务：无公开信息

## 5. 市占・交易额

- 账户数・月度交易额：未公示（披露资料未公开）
- 自有资本规制比率 419.6%（2026-03 末）— 财务健全性维持在高水准
- 竞争内定位：在 FX・CFD 领域与 GMO Click 证券・外汇 dotcom 并列为大手。作为加密资产衍生品交易业者，在 JVCEA 第一种会员 33 社 中，属于无现货・衍生品专营的业者区分（13社 中的 1社）。加密资产 CFD 的实际处理量未公示。

## 6. 沿革・重大事件

| 年月 | 事项 |
|------|------|
| 2006-12  | 设立（作为证券公司开始事业） |
| 2007-09-30 | 作为第一种金融商品交易业向 FSA 注册（关东财务局长（金商）第1629号） |
| 2011-01-01 | 商品期货交易业 追加注册 |
| 2017-06-27 | 第二种金融商品交易业 追加注册 |
| 时期不明 | 作为 JVCEA 第一种会员注册"加密资产衍生品交易业"（JVCEA 会员番号 1043） |
| 2024-05-31 | **关联集团公司 DMM Bitcoin（另一法人）发生约 482億円 相当的加密资产非法流出** |
| 2024-12-02 | DMM Bitcoin 发布停业・发布向 SBI VC Trade 的移管。DMM 集团的加密资产现货交换业消灭 |
| 2025-03-08 | DMM Bitcoin 服务全面终止。DMM.com 证券因与 DMM Bitcoin 为另一法人，直接业务影响未公示（无官方声明） |
| 2026-03  | 披露自有资本规制比率 419.6%（财务健全） |

**与 DMM Bitcoin（另一法人）停业的关系**：

DMM.com 证券与 DMM Bitcoin 为另一法人（地址同为东京日本桥大厦，但法人格・注册号不同）。无法确认有关于 DMM Bitcoin 的停业（2024-12  发布・2025-03  完成）直接影响 DMM.com 证券的 JVCEA 加密资产衍生品注册・事业持续的官方发布（截至 2026-05-18）。不过由于作为 DMM 集团的加密资产现货交易业务消灭，集团整体的加密资产事业大幅缩小。

## 7. 战略动向

- 最近的公开公告以 FX・CFD・股票交易的功能扩充为主。未能确认加密资产领域的新服务公布（截至 2026-05-18）。
- DMM CFD 从 2024年 之前的 14 个币种扩充至 22 个币种（追加商品 CFD 8 个币种）。加密资产 CFD 的追加计划官网无记载。
- DMM 巴努西（赛马用马基金）作为他社无可比拟的特色服务继续。
