# 株式会社 DMM Bitcoin — 日本加密资产交换业者（former VASP）overview


## Wiki route

This entry sits under [[exchanges/INDEX|exchanges index]]. Read it with [[exchanges/fsa-vasp-registration-system|FSA 暗号資産交換業登録制度 — 番号体系・財務局管轄・登録要件]] for adjacent context and [[fintech/japan-financial-regulation|日本金融規制 — トークン・暗号資産・決済に関する法体系]] for the broader system boundary.

> **former VASP / 2024-12 发布停止服务 · 2025-03-08 全面终止**
> 旧 FSA 注册号 关东财务局长 第00010号（已注销） · 旧 JVCEA 第一种会员（已退会） · 加密资产交换业注册日 2018年1月

## 1. 法人・股东

- **商号**: 株式会社 DMM Bitcoin
- **设立**: 2016年11月7日（作为株式会社东京比特币交易所设立）
- **本店所在地**: 东京都中央区日本桥2-7-1 东京日本桥大厦10层
- **资本金**: 1億円（最终期）
- **母公司**: 株式会社 DMM FX 控股（DMM.com 集团）
- **上市**: 未上市
- **代表董事**: 松田升树（停业时点）
- **法人番号**: 5010401128129

## 2. 牌照・注册状况

- **加密资产交换业（FSA）**: 旧 关东财务局长 第00010号 — **已注销**（随 2025年停业）
- **第1种金融商品交易业**: 旧 关东财务局长（金商）第3189号 — **已注销**（2020年5月取得）
- **JVCEA 会员**: 旧 第一种会员 — **已退会**（经 2026-05-18现在 JVCEA 会员一览未刊载确认）
- 电子结算手段等交易业・资金移动业的注册实绩：无公布

## 3. 经办币种

截至 2023年时点支持现物交易的18个币种（主要币种）：

BTC · ETH · XRP · LTC · BCH · XLM · MATIC · LINK · MKR · AVAX · CHZ · TRX · ZPG · ETC · BAT · ENJ · OMG · MONA

亦支持杠杆交易（差金结算），提供 BTC・ETH 等主要币种。经办币种为以比特币为中心的中型阵容，山寨币有限。

## 4. 业务范围

- 现物交易：贩卖所（OTC）
- 杠杆交易（差金结算 / 作为第1种金融商品交易业提供）
- 法定货币入金渠道：银行转账・快速入金
- 质押・借贷：无官方说明
- NFT：无经办实绩
- 法人服务：一般性开户支持（详情无公布）

## 5. 市占・交易量

- 账户数・月度交易量：无公布（披露资料未公开）
- **营收**: 54億500万円（2024年3月期）
- **净利润**: △481億3,100万円（2025年3月期 — 计入流出损失）
- 竞争内定位：中等规模。相对 GMO 币・bitFlyer 等头部势力偏 niche。据称客户基础杠杆交易比例较高，但无第三方公开数据。

## 6. 沿革・重大事件

| 年月 | 事项 |
|------|------|
| 2016-11 | 作为株式会社东京比特币交易所设立 |
| 2017-03 | 开设虚拟货币交易所「东京比特币交易所」 |
| 2017-09 | 向关东财务局申请虚拟货币交换业者注册 |
| 2017-12 | 转让给株式会社 DMM FX 控股，商号变更为「株式会社 DMM Bitcoin」 |
| 2018-01 | 以品牌「DMM Bitcoin」开始受理开户 |
| 2020-05 | 作为第1种金融商品交易业者（关东财务局长 第3189号）注册 |
| **2024-05-31** | **发生约482億円相当（流出当时行情）的加密资产（BTC）非法流出。首次公布** |
| 2024-06 | 发布 550億円规模的资金筹措计划，表明买回相当于流出 BTC 的数量 |
| 2024-07 | 警察厅・金融厅公布注意提醒，暗示以朝鲜为背景的网络攻击集团「Trader Traitor（别名：Lazarus Group）」的参与 |
| **2024-12-02** | **发布全面停业。与 SBI VC 贸易株式会社就客户账户・寄存资产转移签订基本协议书** |
| 2025-03-08 | 服务全面终止。全部客户账户及寄存资产（日元・加密资产）向 SBI VC 贸易转移完成 |

**非法流出事件（2024-05-31）详情**：

流出额约 482 億円相当（按流出发生当时汇率换算）。流出的 BTC 数量约 4,502.9 BTC（基于多家报道 — 该公司一次新闻稿现已无法访问）。该公司为收拾事态发布了 550 億円规模的资金筹措并实施了 BTC 的市场买回，但最终放弃事业持续（详情参见 [[exchanges/dmm-bitcoin-lazarus-hack-detailed-analysis|DMM Bitcoin Lazarus ハック詳細分析]] 与 [[exchanges/jp-vasp-incident-history|JP VASP インシデント史]]，作为姊妹案件 [[exchanges/bybit-lazarus-hack-detailed-analysis|Bybit Lazarus ハック]] 与 [[exchanges/coincheck-nem-hack-detailed-analysis|Coincheck NEM 流出]] 可比较）。

日本警察厅・金融厅・美国 FBI 于 2024-12-24 发布，将本案特定为「Trader Traitor（朝鲜 Lazarus Group）」的攻击并予以注意提醒（记录于 FSA 页面上「令和6年12月24日『关于以朝鲜为背景的网络攻击集团 Trader Traitor 的网络攻击（注意提醒）』的公布」）。

本案是日本加密资产行业史上最大规模的非法流出案件之一。

## 7. 战略动向

停业确定后不存在战略动向。

- 2024-12-02 基本协议：制定向 SBI VC 贸易转移的框架
- 2024-12-25：SBI VC 贸易・DMM Bitcoin 双方公布关于转移的通知
- 2025-03-08 服务终止。作为 DMM 集团从加密资产交换业完全撤退。

## 8. 关联人物

- **松田升树**：代表董事（停业时点）— 无详细经历公布
- **株式会社 DMM FX 控股**：100%母公司（停业时点）
- DMM.com Securities Co.,Ltd.（DMM.com 证券）作为集团内另一法人，持续保有 JVCEA 第一种会员（会员号1043・加密资产衍生品交易），与株式会社 DMM Bitcoin 为各自独立的事业体
