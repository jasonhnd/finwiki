# 乐天钱包株式会社 — 日本加密资产交换业者 overview


## Wiki 路由

本条目位于 [[exchanges/INDEX|exchanges index]]。相邻制度语境见 [[exchanges/fsa-vasp-registration-system|FSA 加密资产交换业登记制度：编号体系、财务局管辖与登记要求]]；更广泛的制度边界见 [[fintech/japan-financial-regulation|日本金融监管：代币、加密资产与支付法体系]]。

> FSA 登记号 关东财务局 第00015号 · JVCEA 会员 #1018 · 登记日 2019-03-25

## 1. 法人・股东

- **商号**: 乐天钱包株式会社（英文: Rakuten Wallet, Inc.）
- **设立年月日**: 2016年12月15日（旧商号: 大家的比特币株式会社）
- **总店所在地**: 〒107-0062 东京都港区南青山二丁目6番21号 乐天Crimson House青山
- **资本金**: 1,850百万円（18.5億円）
- **母公司**: 乐天证券控股株式会社（持股 100%）。乐天证券控股是乐天集团株式会社（东证Prime 4755）旗下公司。
- **法人编号**: 7010401128837
- **代表董事社长**: 山田 达也

## 2. 牌照・登记状况

- **加密资产交换业**: 关东财务局长 第00015号（登记完成 2019年3月25日）
- **电子结算手段等交易业**: 关东财务局长（电代）第48号
- **金融商品交易业（第一种）**: 关东财务局长（金商）第3190号（2020年5月登记完成）
- **JVCEA 会员种类**: 第一种，会员编号 1018。同时持有加密资产交换业・加密资产衍生品交易两项资格。
- **ISMS认证**: ISO/IEC 27001:2013（认证登记号 IS 510581）

## 3. 经办币种

**经办品种数**: 16种（截至 2026-05-18，依据官网经办加密资产一览）

| 类别 | 品种 |
|---------|------|
| 有现货交易 | BTC, ETH, BCH, XRP, ADA, DOT, XLM, POL, SOL, TRX, AVAX, DOGE, SHIB, TON |
| 仅保证金交易 | LTC, XTZ |

保证金交易（加密资产衍生品）支持 BTC・ETH・BCH・LTC・XRP・ADA・DOT・XLM・XTZ 共9种。

**特色**: 以与乐天生态系统的联动为优势，以主流币种为中心的均衡型品种构成。SHIB・TON 等相对较新的Meme・实用型代币也陆续追加（截至 2026年显示「NEW」）。

## 4. 业务范围

- **现货交易（spot）**: 有（智能手机应用及网页）
- **加密资产保证金交易（derivative）**: 有。2020年3月启动服务。提供专用网站「保证金交易所」。
- **乐天积分兑换**: 可将乐天积分（普通积分）按1积分=1円等值兑换为现货交易品种（最低100积分）。2021年3月以「乐天PointClub的积分比特币」名义启动。
- **乐天Cash充值**: 可将持有的加密资产换算并充值为乐天Cash，在乐天Pay加盟店使用（2021年2月启动）。
- **web3钱包**: 另行提供可用乐天ID使用的 NFT/web3 钱包应用（与加密资产交易应用为不同服务）。
- **法人账户**: 有（提供面向法人的账户开设服务）。
- **API**: 面向外部开发者提供交易API（含「GPT-Trade」联动）。
- **入金渠道**: 支持与乐天银行账户联动的即时入金、银行转账、周末及节假日支持的存取款。

## 5. 市占・交易额

- 账户数・月度交易额未公布（截至 2026-05-18）。
- 以乐天集团（会员超1億人）的生态系统为基础的客户群是差异化要素。通过与乐天积分・乐天Cash的联动，致力于将无加密资产经验的乐天用户引导上线。
- 同业中的定位: 中坚（mid tier）。在 [[exchanges/jp-exchange-bitflyer|bitFlyer]]・[[exchanges/jp-exchange-coincheck|Coincheck]]・[[exchanges/jp-exchange-gmo-coin|GMOコイン]] 占据上位的格局中，在 [[JapanFG/rakuten-fg|楽天経済圏]] 确立独特定位（[[exchanges/jp-cex-points-economy-integration|JP CEX ポイント経済統合]] 模式）。相较面向机构投资者的服务，更偏向零售特化的色彩。

## 6. 沿革・重大事件

| 年月 | 事件 |
|------|--------|
| 2016年12月 | Traders Holdings 设立 **大家的比特币株式会社** |
| 2017年3月 | 启动虚拟货币交易所服务 |
| 2017年9月 | 向关东财务局提交虚拟货币交换业者登记申请书 |
| 2018年4月 | 自关东财务局收到业务改善命令（作为登记前的视同业者） |
| 2018年10月 | 乐天Card自Traders Holdings子公司取得大家的比特币全部股份并子公司化 |
| 2019年3月 | 商号变更为 **乐天钱包株式会社**。完成FSA加密资产交换业者登记（第00015号） |
| 2019年4月 | 因乐天集团内重组成为乐天Payment株式会社的100%子公司 |
| 2019年8月 | 启动加密资产现货交易服务 |
| 2020年3月 | 启动加密资产保证金交易服务 |
| 2020年5月 | 完成第一种金融商品交易业者登记（关东财务局长（金商）第3190号） |
| 2020年6月 | 总部从东京都世田谷区玉川迁至东京都港区南青山 |
| 2020年10月 | 将资本金减资至1億円（后增资，现为18.5億円） |
| 2021年2月 | 启动加密资产→乐天Cash充值服务 |
| 2021年3月 | 启动乐天积分→BTC的拟投资「乐天PointClub的积分比特币」 |
| 2022年7月 | 因乐天集团内重组成为乐天证券的100%子公司 |
| 2022年10月 | 重组为乐天证券控股的100%子公司 |

主要行政处分: 2018年4月的业务改善命令（登记前视同业者时期，由母公司Traders Holdings公布）。登记后的行政处分记录在公开信息中未予确认。

## 7. 战略动向

- **乐天经济圈整合的深化**: 通过与乐天积分・乐天Cash・乐天Pay的联动，持续推行将超2,000万人的乐天积分会员作为潜在客户群的战略。
- **web3钱包拓展**: 2025年以后，将NFT・web3支持的应用作为独立服务提供。通过乐天ID的单点登录设计，降低既有用户的进入门槛。
- **GPT-Trade联动**: 将AI驱动型交易界面作为官方服务提供（截至 2026年）。
- **SPU（超级积分提升计划）**: 将乐天钱包的使用纳入乐天市场购物积分返还率提升的条件，谋求与其他乐天服务的相互引流。
- **经办品种扩充**: 在 2026年前后追加SHIB・TON・DOGE等Meme系・高流动性品种，强化品类。

## 8. 相关人物

- **山田 达也**: 代表董事社长（公开信息）
- **旧有的出发点**: Traders Holdings（FX系企业集团）设立了原法人，但已于 2018年10月完全移交至乐天集团。创始人・设立当时董事的详情在公开资料中无记载。
