# 楽天ウォレット株式会社 — 日本加密资产交换业者 overview

> FSA 登记号 关东财务局 第00015号 · JVCEA 会员 #1018 · 登记日 2019-03-25

## 1. 法人・股东

- **商号**: 楽天ウォレット株式会社（英文: Rakuten Wallet, Inc.）
- **设立年月日**: 2016年12月15日（旧商号: みんなのビットコイン株式会社）
- **总部所在地**: 〒107-0062 东京都港区南青山二丁目6番21号 乐天 Crimson House 青山
- **资本金**: 1,850百万円（18.5億円）
- **母公司**: 乐天证券控股株式会社（持有 100%）。乐天证券控股隶属于乐天集团株式会社（东证 Prime 4755）。
- **法人番号**: 7010401128837
- **代表董事社长**: 山田 达也

## 2. 牌照・登记状况

- **加密资产交换业**: 关东财务局长 第00015号（登记完成 2019年3月25日）
- **电子决济手段等取引业**: 关东财务局长（电代）第48号
- **金融商品交易业（第一种）**: 关东财务局长（金商）第3190号（2020年5月登记完成）
- **JVCEA 会员种类**: 第一种，会员番号 1018。同时持有加密资产交换业・加密资产衍生品交易资格。
- **ISMS 认证**: ISO/IEC 27001:2013（认证登记号 IS 510581）

## 3. 取扱货币

**取扱币种数**: 16币种（2026-05-18 时点，据官方网站取扱加密资产一览）

| 类别 | 币种 |
|---------|------|
| 有现货交易 | BTC, ETH, BCH, XRP, ADA, DOT, XLM, POL, SOL, TRX, AVAX, DOGE, SHIB, TON |
| 仅保证金交易 | LTC, XTZ |

保证金交易（加密资产衍生品）对应 BTC・ETH・BCH・LTC・XRP・ADA・DOT・XLM・XTZ 这 9种币种。

**特色**: 以与乐天生态的联动为强项，以主流币为中心的均衡型币种构成。SHIB・TON 等较新的迷因・实用型币种也陆续追加（2026年目前显示「NEW」）。

## 4. 业务范围

- **现货交易（spot）**: 有（智能手机应用及网页）
- **加密资产保证金交易（derivative）**: 有。2020年3月开始服务。提供专用网站「保证金交易所」。
- **乐天积分兑换**: 可将乐天积分（普通积分）按 1积分=相当于 1円 兑换为现货交易币种（最低 100积分）。2021年3月以「Point Bitcoin by 乐天 PointClub」名义开始。
- **乐天 Cash 充值**: 可将持有的加密资产换算并充值为乐天 Cash，在乐天 Pay 加盟店使用（2021年2月开始）。
- **web3钱包**: 另行提供可用乐天 ID 使用的 NFT/web3钱包应用（与加密资产交易应用为不同服务）。
- **法人账户**: 有（提供面向法人的开户服务）。
- **API**: 向外部开发者提供交易 API（含「GPT-Trade」联动）。
- **入金渠道**: 支持与乐天银行账户联动的即时入金、银行转账，以及支持周末及节假日的出入金。

## 5. 市占・交易高

- 账户数・月度交易高未公布（2026-05-18 时点）。
- 运用乐天集团（会员超 1億人）生态的客户基础为差异化要因。通过与乐天积分・乐天 Cash 的联动，致力于将无加密资产经验的乐天用户引入。
- 竞争中的地位: 中坚（mid tier）。在 [[exchanges/jp-exchange-bitflyer|bitFlyer]]・[[exchanges/jp-exchange-coincheck|Coincheck]]・[[exchanges/jp-exchange-gmo-coin|GMOコイン]] 居于前列的格局中，于 [[payment-firms/rakuten-fg|楽天経済圏]] 确立了独特定位（[[exchanges/jp-cex-points-economy-integration|JP CEX ポイント経済統合]] 模式）。相较面向机构投资者的服务，零售特化的色彩更浓。

## 6. 沿革・重大事件

| 年月 | 事件 |
|------|--------|
| 2016年12月 | Traders Holdings 设立 **みんなのビットコイン株式会社** |
| 2017年3月 | 开始虚拟货币交换所服务 |
| 2017年9月 | 向关东财务局提交虚拟货币交换业者登记申请书 |
| 2018年4月 | 收到关东财务局的业务改善命令（作为登记前的视同业者） |
| 2018年10月 | 乐天卡从 Traders Holdings 子公司处取得みんなのビットコイン全部股份・使其子公司化 |
| 2019年3月 | 将商号变更为 **楽天ウォレット株式会社**。FSA 加密资产交换业者登记完成（第00015号） |
| 2019年4月 | 因乐天集团内重组成为乐天 Payment 株式会社的 100%子公司 |
| 2019年8月 | 开始加密资产现货交易服务 |
| 2020年3月 | 开始加密资产保证金交易服务 |
| 2020年5月 | 第一种金融商品交易业者登记完成（关东财务局长（金商）第3190号） |
| 2020年6月 | 总部从东京都世田谷区玉川迁至东京都港区南青山 |
| 2020年10月 | 将资本金减资至 1億円（后增资，现为 18.5億円） |
| 2021年2月 | 开始加密资产→乐天 Cash 充值服务 |
| 2021年3月 | 开始乐天积分→BTC 拟似投资「Point Bitcoin by 乐天 PointClub」 |
| 2022年7月 | 因乐天集团内重组成为乐天证券的 100%子公司 |
| 2022年10月 | 重组为乐天证券控股的 100%子公司 |

主要的行政处分: 2018年4月的业务改善命令（登记前视同业者时期，由母公司 Traders Holdings 公布）。登记后的行政处分记录在公开信息中未获确认。

## 7. 战略动向

- **乐天经济圈整合的深化**: 通过与乐天积分・乐天 Cash・乐天 Pay 的联动，持续以超 2,000万人的乐天积分会员为潜在客户基础的战略。
- **web3钱包展开**: 2025年以来，将 NFT・web3对应的应用作为独立服务提供。以乐天 ID 单点登录的设计降低既有用户的进入门槛。
- **GPT-Trade 联动**: 将 AI 活用型交易界面作为官方服务提供（2026年目前）。
- **SPU（超级积分提升计划）**: 将乐天钱包的使用纳入乐天市场购物积分还原率提升的条件，谋求与其他乐天服务的相互引流。
- **取扱币种扩充**: 在 2026年前后追加 SHIB・TON・DOGE 等迷因系・高流动性币种，强化品类。

## 8. 相关人物

- **山田 达也**: 代表董事社长（公布信息）
- **早期的起点**: Traders Holdings（FX 系企业集团）设立了原法人，但于 2018年10月完全转入乐天集团。创始人・设立当时高管的详情在公布资料中无记载。
