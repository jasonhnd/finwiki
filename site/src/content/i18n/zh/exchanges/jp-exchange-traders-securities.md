# Traders 证券 — 日本加密资产衍生品交易业者 overview


## Wiki route

This entry sits under [[exchanges/INDEX|exchanges index]]. Read it with [[exchanges/fsa-vasp-registration-system|FSA 暗号資産交換業登録制度 — 番号体系・財務局管轄・登録要件]] for adjacent context and [[fintech/japan-financial-regulation|日本金融規制 — トークン・暗号資産・決済に関する法体系]] for the broader system boundary.

> 金融商品交易业 关东财务局长（金商）第123号 · JVCEA 会员 #1037（第一种 · 仅限加密资产衍生品交易）

## 1. 法人・股东

- 商号：Traders 证券股份有限公司（英文名 TRADERS SECURITIES CO., LTD.）
- 设立：平成11年11月（1999年11月）在东京都港区南麻布以资本金1億5,000万円设立
- 总店：〒150-6028 东京都涩谷区惠比寿4丁目20-3 惠比寿花园广场大厦28层（令和5年4月迁移）
- 母公司：Traders Holdings 股份有限公司（东证 Standard 市场上市 · 证券代码 8704）
  - 于平成18年10月转为控股公司制。Traders 证券作为100%子公司承继证券・外汇交易事业
- 资本金：2,324,285,000円（截至平成30年2月）
- 员工人数：72名（截至令和7年3月31日）

## 2. 牌照・登记状况

- 金融商品交易业：关东财务局长（金商）第123号（第一种・第二种・投资顾问代理业）
- 加密资产相关：JVCEA 第一种会员 #1037 — **仅限加密资产衍生品交易**。不持有现货加密资产交换业（FSA 加密资产交换业登记）。
- 加入协会：日本证券业协会 / 一般社团法人 金融期货交易业协会 / 一般社团法人 第二种金融商品交易业协会 / 一般社团法人 资产运用业协会 / 一般社团法人 日本加密资产等交易业协会（JVCEA）/ 日本投资者保护基金

## 3. 经办币种

- 在加密资产衍生品交易（「大家的币（みんなのコイン）」「LIGHT FX 币」）中经办。其网站的服务页面（min-fx.jp）提及 BTC・ETH・XRP・LTC・BCH 等主要5币种。
- 无现货经办。全部交易为杠杆2倍上限的 CFD（加密资产保证金交易）。
- 无币种数的官方明示（未公布）。

## 4. 业务范围

- **主业（FX）**：大家的FX（みんなのFX）/ LIGHT FX — 接近国内 FX 专业的定位。FX 占销售额的大部分。
- **加密资产CFD**：大家的币（令和4年1月 2022-01 启动）/ LIGHT FX 币（同时启动）。无现货持有，全额信托保全对应，杠杆2倍。
- 其他：大家的系统交易（みんなのシストレ，自动交易）、大家的期权（みんなのオプション，场外外汇期权）
- 法定货币入金：银行转账（三菱UFJ银行・三井住友银行・瑞穗银行等）
- NFT / staking / 加密资产借贷：无

## 5. 市占・交易额

- 账户数（FX 2品牌合计）：667,900口座（截至2027年3月期 4月 = 2026-04 公布）
- 存管资产（同）：133,611,648千日元（≒1,336億円）（2026-04 公布）
- 加密资产 CFD 单独的账户数・交易额未公布。
- 竞争内定位：FX 中坚证券（推定大家的FX 在国内 FX 账户数排名前10之内，但无官方排名出处）。加密资产衍生品推定在同业13社中规模居中（未公布）。同业其他公司有 [[exchanges/jp-exchange-monex|マネックス証券]]、[[exchanges/jp-exchange-sbi-securities|SBI 証券]]、[[exchanges/jp-exchange-bi-fxtrade|SBI FXトレード]]，上市企业的对比参照 [[exchanges/jp-listed-cex-related-companies-matrix|JP 上場 CEX 関連企業マトリクス]]。

## 6. 沿革・重大事件

| 年月 | 事项 |
|------|------|
| 1999-11 | 在东京都港区南麻布设立，启动外汇交易业务 |
| 2001-02 | 与 MUFG（旧三和银行）集团达成业务协作合意 |
| 2001-05 | 商号变更为东京外汇 Traders 证券股份有限公司 |
| 2001-06 | 国内首家・启动外汇交易客户资产分别信托 |
| 2002-06 | 商号变更为 Traders 证券股份有限公司 |
| 2005-04 | 在大阪证券交易所 Hercules 市场上市（现东证 Standard） |
| 2006-10 | 转为控股公司制 → 分立为 Traders Holdings 股份有限公司（母公司）与现在的证券事业公司 |
| 2010-07 | 启动「大家的FX」服务（承继 EMCOM 证券事业） |
| 2021-02 | 加入 JVCEA（日本加密资产交易业协会） |
| 2022-01 | 启动加密资产保证金交易「大家的币」「LIGHT FX 币」 |
| 2023-04 | 总公司迁移至惠比寿花园广场大厦28层 |

- 行政处分・黑客攻击・大规模系统故障：未公布（取得日 2026-05-18 现在）
