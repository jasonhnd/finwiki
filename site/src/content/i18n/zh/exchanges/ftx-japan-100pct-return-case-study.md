# FTX Japan 客户资产 100% 返还案例 — 全球首例即时全额返还事例


## Wiki route

This entry sits under [[exchanges/INDEX|exchanges index]]. Read it against [[exchanges/jp-vasp-incident-history|国内 VASP 暗号資産流出事件史 (2014-2026)]] for peer / contrast context and [[exchanges/fsa-vasp-registration-system|FSA 暗号資産交換業登録制度 — 番号体系・財務局管轄・登録要件]] for the broader system / regulatory boundary.

## 1. 经纬

2022-11-11，FTX Trading Ltd. 等申请 Chapter 11 。在全球范围内，由于与 Alameda Research 的流动性危机 + 客户资金与自营运用的混同，全球 FTX 客户资产被冻结。日本法人 FTX Japan（旧 [[exchanges/jp-exchange-custodiem|QUOINE]]）因遵守了 [[exchanges/jp-vasp-cold-storage-segregation-rules|分別管理義務]]（向信托银行的信托保管），客户资产无损残存，这一点存在决定性差异。

## 2. 返还 timeline

- **2022-11-10** — FSA 向 FTX Japan 发出业务停止命令（部分）+ 业务改善命令
- **2022-11-30** — FSA 确认 FTX Japan 的客户资产分别管理（信托保管发挥作用）
- **2023-02-20** — 客户资产返还方案获批（提供基于 [[exchanges/jp-exchange-bitflyer|bitFlyer]] 合作的提款 UI）
- **2023-02-21** — 客户提款开始 — 实现全球首例即时 100% 返还

## 3. 制度性意义

- 日本的分别管理义务 + 信托保管要求在破产时实际发挥作用的实证案例
- [[exchanges/fsa-business-improvement-orders-history|FSA 業務改善命令制度]] + JVCEA 自主监管 + 信托保管三重结构的有效性
- 在全球 VASP 监督模型（美国 SEC・EU MiCA・HK SFC 等）中被引用的案例
- 反向映照出允许「自营托管 + 客户资金混同」的制度设计的危险性

## 4. 此后的经纬

2024-07，bitFlyer Holdings 将 FTX Japan（= QUOINE）完全子公司化。于 2024-08  改称为 [[exchanges/jp-exchange-custodiem|Custodiem]]，正在转型为加密托管业务。是 [[exchanges/jp-vasp-ma-consolidation-history|VASP M&A 集約]] 的象征性事例之一。

## 5. 海外比较

美国 FTX International 在 2025-06  实现对债权人 100% 偿付 — 耗时 3  年以上。另一方面，日本从破产到全额返还约用 3  个月实现。这是制度设计差异的戏剧性证明，印证了分别管理 + 信托保管模型的实效性。在 [[exchanges/jp-vasp-incident-history|JP VASP インシデント史]] 上，是最成功的破产处理案例。
