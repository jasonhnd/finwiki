# 株式会社ガイア — 日本加密资产交换业者 overview

> FSA 登记号 近畿财务局 第00004号 · JVCEA 会员 #1034 · 登记日 2021-06-18

## 1. 法人·股东

- **商号**:株式会社ガイア
- **设立**:2007年12月25日(法人组织变更)
- **总店所在地**:〒550-0002 大阪府大阪市西区江戸堀2-1-1 江戸堀センタービル9F
- **大阪办公室**:大阪市西区新町1-28-3 四ツ橋グランスクエア9F
- **东京办公室**:東京都千代田区岩本町3-3-3 秋葉原サザンビル5F
- **资本金**:2,990万円
- **员工人数**:15名(公布)
- **母公司**:株式会社ガイアホールディングス(2021年2月设立,移管加密资产交换业以外的业务)
- **上市状态**:非上市

## 2. 牌照·登记状况

- **加密资产交换业**:近畿财务局长 第00004号(登记日 2021-06-18)
- **JVCEA 第一种会员**:#1034(2021年1月以第二种加入 → 同年6月变更种别为第一种)
- 无金融商品交易业·电子结算手段等交易业·资金移动业的登记(在公布信息的范围内)
- 关西所在的加密资产交换业 VASP 在 26 社家中仅 2 社家([[exchanges/jp-exchange-zaif|Zaif]]:近畿财务局长 第00001号 与本公司,参考为 [[exchanges/fsa-vasp-registration-system|FSA 暗号資産交換業登録制度]])

## 3. 经办货币

- **经办币种数**:3种(2026年5月现在的公布信息)
- **对应币种**:BTC(比特币)、ETH(以太坊)、ADA(卡尔达诺)
- **特色**:币种种类较少且以 BTC 为中心,但对应 ADA 在国内 VASP 中也属少见。ETH 有 2022年12月一度停止→2025年12月恢复经办的经过。

## 4. 业务范围

- **主力服务**:加密资产自动兑换机 BTM(Bitcoin Teller Machine)— 作为日本国内唯一的加密资产 ATM 展开(2022年8月开始,对比为 [[exchanges/jp-exchange-coinhub|COINHUB]] 的 ATM 业态)
- **BTM**:日元 ↔ BTC / ETH 购买·卖出,仅对应 ADA 购买。1单笔交易最高10万円。
- **WEB 兑换服务**:无需 BTM 的在线兑换(24小时受理,对应 BTC / ETH / ADA 的购买·卖出)
- **入境游客对应**:无需用户登记·仅凭符合 ICAO 的 IC 护照即可办理 ADA 卖出以外的兑换(面向外国游客)
- **BTM 机体销售**:面向合作方设有 BTM 机体的销售制度
- **法定货币入金渠道**:现金纸币(ATM 投入)或银行账户汇款
- **margin / derivative / staking / NFT / 加密资产出借**:无公布信息

## 5. 市占·交易量

- 账户数·交易量为非公布
- 规模归类为小规模 niche(员工15名,资本金2,990万円)
- BTM 设置台数:2022年8月在国内设置第1台(大阪)→ 9月第2台(东京)→ 11月第3台。2026年5月1日时点暂时停止使用中(原因在公布信息范围内未详)
- 客户层:retail(BTM 利用者)＋入境游客

## 6. 沿革·重大事件

| 年月 | 事项 |
|------|------|
| 2007年6月 | 面向网络的信用卡结算系统导入事业的开始 |
| 2007年12月 | 变更组织为株式会社ガイア |
| 2008年6月 | 面向餐饮店·物贩店的信用卡结算事业、账户划拨服务开始 |
| 2008年10月 | 餐饮事业部「まるくら」设立 |
| 2010年6月 | 将餐饮事业部移管至另一公司 |
| 2011年1月 | 电子货币·积分卡发行业务开始 |
| 2013年4月 | 信贩·贷款经办开始 |
| 2018年1月 | 将加密资产交换业追加为事业目的 |
| 2021年1月 | JVCEA 第二种会员加入 |
| 2021年2月 | 株式会社ガイアホールディングス设立(移管加密资产交换业以外的业务) |
| 2021年6月 | 加密资产交换业登记完成(近畿财务局长 第00004号),变更种别为 JVCEA 第一种会员 |
| 2022年8月 | 基于加密资产交换业的服务开始(该公司发表);国内首台加密资产自动兑换机 BTM 在大阪启用 |
| 2022年9月 | 在东京设置国内第2台 BTM |
| 2022年11月 | 设置国内第3台 BTM |
| 2022年12月 | ETH 经办暂时停止 |
| 2025年12月 | ETH 经办恢复 |
| 2026年4月 | 因导入 passkey 变更登录规格 |
| 2026年5月 | BTM 暂时停止使用(原因不明,有公布) |

## 7. 战略动向

- 提供 BTM 机体的合作方销售制度,谋求向其他公司·其他业者的 BTM 设置展开
- 面向外国游客(入境)开始无需身份验证的兑换(护照核对方式),是吸纳入境需求的独特路线
- 2026年5月现在的 BTM 暂时停止使用在战略·监管上的含义不明(在公布信息的范围内未详)
- 加盟日本自动贩卖系统机械工业会,显示出将 BTM 作为"自动贩卖机"类别进行业界横向展开的意图
