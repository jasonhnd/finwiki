# Mt.Gox 破产处理 timeline (2014-2026)


## Wiki route

This entry sits under [[exchanges/INDEX|exchanges index]]. Read it against [[exchanges/jp-vasp-incident-history|国内 VASP 暗号資産流出事件史 (2014-2026)]] for peer / contrast context and [[exchanges/fsa-vasp-registration-system|FSA 暗号資産交換業登録制度 — 番号体系・財務局管轄・登録要件]] for the broader system / regulatory boundary.

## 1. 概要

2014-02-28,Mt.Gox (株式会社 MTGOX、东京・涩谷) 发布约 850,000 BTC + 约 28 億円的流出并申请民事再生。其后转入破产程序,2018-06 获得全球首例"加密资产事业者民事再生重启"决定。2024-06 终于启动现物清偿,至2026 年仍在持续 — 历时 12 年、世界最长级别的加密资产破产处理案例。它是 [[exchanges/jp-vasp-incident-history|JP VASP インシデント史]] 的起点,亦是日本 VASP 制度设计的最大触发器。

## 2. timeline 主要日期

- **2014-02-07** — 停止提现 (以 technical issue 为官方理由)
- **2014-02-28** — 申请民事再生 · 发布约 850,000 BTC + 28 億円流出
- **2014-03〜10** — 回收部分流出 BTC (约 200,000 BTC)
- **2014-04-24** — 民事再生程序中止 · 破产程序开始决定 (东京地方法院)
- **2018-06-22** — 民事再生程序重启决定 — 全球首例加密资产事业者再生案例
- **2019-04** — 提交民事再生计划案
- **2021-11-16** — 民事再生计划案认可决定
- **2024-06-24** — 启动清偿 (5-7 月 集中清偿期间)
- **2025-2026** — 剩余债权・持续清偿中

## 3. 清偿结构

- **CRC (Crypto Receiver Coupon)** vs **JPY 提前一次性清偿** 的选择制 — 各债权人可自行选择
- **BTC + BCH** 现物清偿 (按时价 · 反映 2014 当时的持有比例)
- 清偿平台合作: **Bitstamp / Kraken / [[exchanges/jp-exchange-bitflyer|bitFlyer]] / [[exchanges/jp-exchange-sbi-vc-trade|SBI VC トレード]]**
- 国内・海外债权人经各登记平台分别汇款・出库的分散模型

## 4. 制度性意义

- 2014 流出 → 2017 修订资金结算法 (VASP 登记制度) 立法的最大触发器 — [[exchanges/jp-vasp-regulatory-timeline|JP VASP 規制 timeline]] 的起点
- 民事再生 → 破产 → 民事再生重启的 3 阶段转换为全球首例 (在法律解释上,形成将比特币作为"物"而非"金钱债权"处理的判例)
- 多家交易所联合清偿平台的模型是 [[exchanges/ftx-japan-100pct-return-case-study|FTX Japan (2023)]] 中 bitFlyer 合作返还方案的先例
- [[exchanges/fsa-business-improvement-orders-history|FSA 業務改善命令制度]] · [[exchanges/jp-vasp-cold-storage-segregation-rules|分別管理義務]]皆为 Mt.Gox 教训的反射

## 5. 全球比较

美国 [[exchanges/ftx-japan-100pct-return-case-study|FTX International]] 于 2022-11 破产 → 2025-06 实现 100% 清偿 (约 3 年)。而 Mt.Gox 历时 12 年 (2014→2026 持续中) — 差异并非源于监管设计,而是诉讼复杂性 (流出 BTC 的所有权争讼 + Karpelès 个人责任诉讼 + 海外债权人协调) 的差异。这是逆向印证"破产处理的迅速性取决于事业者的分别管理体制"这一制度设计命题的案例。
