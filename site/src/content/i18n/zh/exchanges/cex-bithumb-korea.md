# Bithumb（빗썸）— 韩国 KRW spot CEX overview


## Wiki route

This entry sits under [[exchanges/INDEX|exchanges index]]. Read it against [[exchanges/korea-cex-five-pillars-comparison|韓国 5 大 CEX 制度比較]] for the Korea peer / regulatory context, [[exchanges/cex-upbit-korea|Upbit]] for the closest peer comparison, and [[exchanges/global-cex-top10-comparison|global CEX top 10 comparison]] for the broader system / market boundary.

> 2014-01 设立 · 韩国 spot 市场 #2（份额约 25%）· NH 농협은행 实名账户 · ISMS-P 认证 · KOSDAQ IPO 计划推进中（2026）

## 1. 法人・股东

- 商号：주식회사 빗썸코리아（Bithumb Korea Co., Ltd.）
- 设立：2014-01
- 总部所在地：서울특별시 강남구
- 母公司结构：Bithumb Holdings（持股公司）→ Bithumb Korea（事业公司）的 2 层结构
- 股东构成：多家韩国系金融・IT 投资者间接持有（过去有多次控制权变动历史,详情见 IR 披露）
  - 2018–2019 年期：经由 BTC Korea Holdings 的控制结构
  - 2020 年之后：Vidente 集团系（Vidente Co., Ltd.）参与的报道
  - 2024–2026 年期：控股股东依旧复杂,因 IPO 准备而重组推进中

## 2. 注册・牌照状况（韩国制度）

- **가상자산사업자 (VASP)** 注册：금융정보분석원（FIU）已注册（2021-09-24 韩国 VASP 注册制度首日同日期类别）
- **ISMS / ISMS-P 认证**：持有 KISA 认证（含强化版 ISMS-P）
- **실명확인 입출금 계정**（实名账户）：与 **NH 농협은행** 的合约（2018 年之后持续）
- 满足韩国 VASP 注册 2 要件（ISMS + 实名账户）从而可提供 KRW spot 市场的 **5 社家之中的 1 社家**
- 详细制度参见 [[exchanges/korea-cex-five-pillars-comparison|韓国 5 大 CEX 制度比較]]

## 3. 经办币种

- KRW spot 市场 + USDT 市场（有限）
- 经办币种数：约 200+（2024–2026 期,除 ETH / BTC / XRP / SOL / ADA 等主要币种外,另有 Klaytn 系 / 韩国 IP 项目系的独家上架）
- BTC / ETH / XRP 的交易量份额占压倒性优势（韩国市场整体结构）
- 上架审查：遵循 DAXA（한국 가상자산 거래소 자율규제 협회、Digital Asset eXchange Alliance）的共通上架标准

## 4. 业务范围

- 现货 spot 交易（以 KRW pair 为中心）
- 部分 USDT pair（有限）
- 衍生・margin 交易：**在韩国 VASP 监管下不提供**（韩国 VASP 受限于 spot only）
- staking / lending：有限服务（存在监管约束）
- 法定货币入金渠道：仅经由 NH 농협은행 实名账户

## 5. 市占・交易额

- **韩国 spot 市场份额**：约 25%（次于 [[exchanges/cex-upbit-korea|Upbit]] 的 70%,位列 #2）
- 24h volume：$0.5–2B（与市场波动联动）
- 客户群：以韩国居民为中心（完全实施 KYC）
- 全球 CEX 排名：top 20 圈内（CoinGecko / CoinMarketCap 汇总口径）

## 6. 沿革・重大事件

| 时期 | 事项 |
|------|------|
| 2014-01 | Bithumb 设立,作为韩国早期正规 CEX 开始营业 |
| 2017 | 韩国加密货币热潮期,与 Upbit 并列作为最大型机构成长 |
| 2017-12 | 隐私侵害事故,3 万件以上客户信息泄露 |
| 2018-06 | 因非法访问导致约 350 億韩元相当的加密资产外流 |
| 2019-03 | 因内部舞弊导致 200 億韩元规模外流事件 |
| 2021-03 | 韩国 특정금융정보법 施行,VASP 注册制度启动 |
| 2021-09-24 | 完成向 FIU 的 VASP 注册（韩国第 1 期注册类别） |
| 2022–2023 | 历经各类诉讼・所有权之争后经营体制重组 |
| 2023 | 加入 DAXA（自律监管协议体）,上架审查共通化 |
| 2024–2025 | IPO 准备正式化,选定 KOSDAQ 上市主承销商 |
| 2025 | ISMS-P 强化认证更新,重建合规体制 |
| 2026 | 持续推进 KOSDAQ 上市（基于官方公告口径） |

## 7. 战略动向

- **IPO 战略**：参照 [[exchanges/cex-upbit-korea|Upbit]] 母公司 Dunamu 的上市先例,以经由 KOSDAQ 上市实现资本市场准入为目标。若实现,将成为 **作为韩国 VASP 第 2 社家（或首家）的上市公司**
- **DAXA 参与**：与 Upbit / Coinone / Korbit / GOPAX 一同在 DAXA 共通化上架标准,分散监管应对成本
- **NH 농협은행 关系**：实名账户合约的持续是事业存续的生命线。NH 银行侧的 AML/CFT 责任分担协商仍是持续课题
- **2024-2026 监管环境**：构建韩国 가상자산이용자보호법（2024-07 施行）的遵守体制,出资利用者保护基金,应对不公正交易监视强化
- **DeFi 联动**：作为 Klaytn（카카오 系）・WEMIX・XPLA 等韩国系 L1 / L2 代币的主要上架场所发挥功能
