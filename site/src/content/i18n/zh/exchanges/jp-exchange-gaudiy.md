# 株式会社Gaudiy Financial Labs — JVCEA 第二种会员（Web3 IP 平台运营商）


## Wiki route

本条目位于 [[exchanges/INDEX|交易所索引]]。可与 [[exchanges/fsa-vasp-registration-system|FSA 加密资产交换业登记制度]] 和 [[fintech/japan-financial-regulation|日本金融监管]] 配套阅读。

> FSA 登记编号 **未公开（第二种会员 = 申请准备中或申请预定）** · JVCEA 会员 #2033（第二种会员） · 母集团: 株式会社 Gaudiy Group（原名 株式会社 Gaudiy）

株式会社 Gaudiy Financial Labs 是**并非加密资产交换业者，而是 Web3 × IP 粉丝经济领域创业公司**株式会社 Gaudiy Group 旗下的金融基础设施子公司。加入 JVCEA 第二种会员，被定位为该集团为整备连接 IP 持有方与粉丝的粉丝代币及加密资产相关服务基础设施而进行的牌照取得准备。其商业模式与现有加密资产交易所（第一种会员各社）有很大不同，主要目的被认为是扩展面向 B2B 的 IP 社区平台的金融功能。

## 1. 法人·股东

- **商号（JVCEA 登记名）**：株式会社 Gaudiy Financial Labs
- **母法人**：株式会社 Gaudiy Group（旧称: 株式会社 Gaudiy）— 位于东京都千代田区神田锻冶町，非上市创业公司
- **设立年月日（母公司）**：2018 年 5 月（Gaudiy Group 的设立。Gaudiy Financial Labs 单体的设立时期: 未公开。freee 公告页面刊载第1期·第2期决算报告书）
- **本店所在地（Gaudiy Financial Labs）**：东京都千代田区神田锻冶町三丁目 6-7 5F（JVCEA 登记地址）
- **母公司本店所在地**：东京都千代田区神田锻冶町（JVCEA 上为同一地址）
- **股东·出资比例**：Gaudiy Group 为主要股东，但详情未公开
- **Gaudiy Group 主要股东（截至2025年5月的公开信息）**：
  - 索尼集团（2025年Pre-Series C 新加入）
  - 万代南梦宫控股（2022年Series B 起参与）
  - STRIVE、JAFCO（Series A 起的跟进投资方）
  - SBI Investment、KDDI、索尼音乐娱乐
  - 三丽鸥、三菱 UFJ 创新合伙、瑞穗资本
  - Akatsuki、Anime Times、Coamix、讲谈社、集英社、小学馆、Bushiroad（2025年股份交换新加入）
- **上市状态**：非上市

## 2. 牌照·登记状况

- **加密资产交换业（FSA）**：未公开 — JVCEA 第二种会员身份意味着"登记申请中或申请预定"，但 FSA 正式登记编号在 JVCEA 页面及官方网站上均未记载
- **JVCEA 会员**：**第二种会员 #2033**（4社中的1社）
- 金融商品交易业·电子支付手段等交易业·资金转移业：均未公开
- 第二种会员的定位：根据 JVCEA 规则，对象为"正在申请或预定申请加密资产交换业者或加密资产相关衍生品交易业者登记的事业者"。2026-05-18 时点无法确认 FSA 正式登记

## 3. 经办货币

- 截至目前（2026-05-18），无法确认已正式启动加密资产交换服务的事实
- Gaudiy Group 运营的"Gaudiy Fanlink"平台具备 IP 持有方发行·授予独自**粉丝代币**的机制。这些代币与加密资产交换业语境下的"经办品种"不同，但在加密资产定位明确化阶段预计需要进行合规应对
- 经办品种数·主要品种：未公开

## 4. 业务范围

Gaudiy Financial Labs 在母集团 Web3 × IP 商业模式中承担金融基础设施的定位。母集团 Gaudiy Group 展开的主要事业如下：

**Gaudiy Fanlink（主力 SaaS）**
- 面向 IP 持有方（出版社·动画·游戏公司等），以 B2B 形式提供粉丝共创型社区平台
- 综合运用区块链技术，一站式实现代币发行·粉丝贡献评价·积分设计·NFT 发行等功能
- 结合独自区块链与生成式 AI，构建以各 IP 为单位的经济圈
- 已导入的 IP 示例（公开部分）：集英社"少年 Jump+"、三丽鸥、万代南梦宫娱乐（"World of 电音部"）、Aniplex、索尼音乐娱乐、KONAMI（"Qto"）等

**Gaudiy Financial Labs（JVCEA 登记法人）**
- 以创造 Web3 时代新金融基础设施为使命（官方网站）
- 整备将粉丝热情·贡献转化为"经济价值"的代币经济学基础
- 指向未来 IP 独自代币上市支援·提供交易基础设施（已在过往 Series B 公告中明示）

**MyAnimeList**
- 旗下持有拥有全球 240 个国家以上、12 億件以上观看·阅读大数据的世界最大级动漫·漫画社区网站（收购时期·收购金额未公开）

**Gaudiy US Inc.**（2023 年设立，纽约州）
- 作为北美据点设立。全球化布局的一环

法币入金渠道·保证金交易·衍生品：未公开（加密资产交换业正式启动前）

## 7. 战略动向

- **Pre-Series C 资金用途**（2025 年 5 月发布）：对现有服务的大规模投资及并购、通过海外据点扩充推进全球化、Web3·AI 相关研发及工程师招聘、社会·教育事业的启动。年内也预定进行 Second Close
- **与大型出版社·娱乐公司的资本合作**：集英社·讲谈社·小学馆·万代南梦宫 HD·索尼集团等成为股东，构建大幅降低与内容 IP 持有方合作门槛的结构
- **AI × 区块链双轮驱动**：除独自区块链外，亦将利用大型语言模型进行粉丝行为数据分析·奖励设计优化商业化。在公司内部设立"Gaudiy AI Lab"·"Gaudiy Web3 Lab"等内部实验室推进研发
- **优先全球化布局**：明示纽约据点（Gaudiy US）设立及以东南亚（2023 年以后）为首的全球 IP 市场进军
- **MyAnimeList 的活用**：凭借持有 12 億件以上动漫·漫画观看数据的全球平台协同效应，志向扩大 IP 粉丝经济
- **IP 独自代币上市支援**：自 Series B 公告阶段（2022 年）起已公开表示"中长期将支援各 IP 独自代币上市"，Gaudiy Financial Labs 的 JVCEA 登记与该方向一致

## 8. 相关人物

- **石川裕也**：株式会社 Gaudiy Group 代表取締役 CEO·创始人。自2018 年设立起担任代表。以"粉丝国家的创造"为目标，主导与庆应大学教授的学术合作及与娱乐巨头的资本合作
- **坂井丰贵**：庆应义塾大学教授，经济设计顾问（2019 年就任·已在官方 PR Times 确认）。作为制度设计·拍卖理论第一人，协助 Gaudiy 的代币经济学设计
- Gaudiy Financial Labs 代表取締役·役员：未公开（截至2026-05-18）
