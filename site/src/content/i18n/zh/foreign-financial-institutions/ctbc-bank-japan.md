# 中國信託商業銀行 东京支店

## Wiki 路径

本条目位于 [[foreign-financial-institutions/INDEX|foreign-financial-institutions INDEX]]。可对照 [[foreign-financial-institutions/mega-icbc-japan|兆豐國際商業銀行 东京支店]] 理解同业和对比语境，也可对照 [[banking/INDEX|banking index]] 理解更广的系统和监管边界。

## TL;DR

中國信託商業銀行东京支店，是 FSA 银行牌照 workbook 中列出的台湾 CTBC Bank 在日外国银行支店。^[source:fsa-ginkou-2026] CTBC 面向日本的申请书以「中國信託商業銀行東京支店」提供 Internet Banking、Fax Banking 和 Automated Message Notification 为前提，因此可以将该支店视为 Japan-facing corporate operations 的窗口。^[source:ctbc-application-2026] JBA 会员名单则将 CTBC Bank 列在 Associate Members / Foreign Banks 下。^[source:jba-members-2026]

## 1. License / branch boundary

| Item | Notes |
|---|---|
| FSA listed name | 中國信託商業銀行。^[source:fsa-ginkou-2026] |
| Legal / branch name | CTBC Bank Co., Ltd. Tokyo Branch |
| License route | 银行业法下的外国银行支店；FSA 银行牌照 workbook。^[source:fsa-ginkou-2026] |
| Home jurisdiction | 台湾 |
| Parent / group | CTBC Bank / CTBC Financial Holding。Tokyo Star Bank 的公开资料说明 CTBC Bank 是 CTBC Financial Holding 的核心公司。^[source:tokyo-star-ctbc-2024] |
| Wiki role | 台湾民营银行日本支店锚点 |
| Tokyo office window | Roppongi Hills Mori Tower 37F, 10-1 Roppongi 6-chome, Minato-ku, Tokyo 106-6137, Japan（CTBC Tokyo Branch agreement 的 designated notice window）。^[source:ctbc-agreement-2026] |

## 2. Operating model

CTBC 的日本支店表格和披露资料都指向 Tokyo Branch 路径。申请书覆盖查询用户、交易制单/复核人员、互联网银行扣款账户、传真银行认证，以及借贷记、汇款、定期存款和贷款等交易通知。^[source:ctbc-application-2026] 因此 wiki 应将 CTBC Japan 作为外国银行支店处理，而不是日本法人零售银行。

集团背景方面，Tokyo Star Bank 的 CTBC 股东页面称 CTBC Bank 是 CTBC Financial Holding 的核心公司，并说明 CTBC Bank 在台湾、美国、加拿大、日本等市场运营。^[source:tokyo-star-ctbc-2024] 这支持台湾-日本 corridor 的解读，但不等于存在单独的日本子公司。
