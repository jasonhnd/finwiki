# 巴西银行日本 (Banco do Brasil Japan)

## Wiki 路径

巴西银行日本归属于 [[foreign-financial-institutions/INDEX|foreign-financial-institutions INDEX]]，是 [[foreign-financial-institutions/foreign-bank-branches-japan-index|the foreign-bank branch registry]] 中的巴西-日本外国银行分行路径。请与 [[payments/funds-transfer-service-providers-japan-index|funds-transfer registry]]、[[financial-licenses/foreign-bank-branch-license-route|foreign-bank branch route]]、[[financial-licenses/foreign-bank-agency-business-route|foreign-bank agency route]] 与 [[finance/cross-border-m-a-japan|cross-border finance]] 一并阅读。

本页适用于 BB Japão、日本-巴西汇款、外币存款、分行银行业务以及企业国际业务。

## 概述

巴西银行的日本站点是面向客户的完整公开界面，而不仅是登记簿上的一行。它涵盖开户、汇款、投资 / 定期存款、兑换、分行 / 移动服务网点，以及企业国际业务。

本页之所以值得单独处理，是因为巴西-日本走廊具有很强的侨民与贸易融资相关性。它更接近 [[foreign-financial-institutions/philippine-national-bank-japan|PNB Japan]] 与 [[foreign-financial-institutions/metrobank-japan|Metrobank Japan]]，而非纯粹的批发投资银行分行。

## 公开背景

巴西银行日本是巴西居民、与巴西相关的企业以及日本-巴西贸易的金融桥梁。其意义并不仅在于母行在巴西的规模。该日本分行的相关性来自走廊服务——主流日本大型银行未必以相同的客户语言或汇款流程覆盖这些服务。

为进行分析，可在汇款与侨民金融方面将其与 [[foreign-financial-institutions/philippine-national-bank-japan|PNB Japan]] 和 [[foreign-financial-institutions/metrobank-japan|Metrobank Japan]] 对照，并在大型企业巴西融资方面与 [[megabanks/mufg-bank|MUFG Bank]] / [[megabanks/sumitomo-mitsui-banking-corp|SMBC]] 对照。

## 研究清单

1. 辨明产品属于账户、存款、汇款、外汇、企业国际业务，还是外国银行代理。
2. 使用 BB Japão 公开页面获取当前服务说明。
3. 使用 FSA 来源确认分行 / 代理牌照。
4. 将 2004 FSA 行政处分仅视为历史背景；在没有当前来源的情况下，不要暗示存在当前问题。
5. 将巴西银行的母行全球角色与该日本分行的服务范围相区分。
