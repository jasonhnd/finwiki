# 毕尔巴鄂比斯开阿根廷银行东京分行


## 维基路线

该条目位于[[foreign-financial-institutions/INDEX|foreign-financial-institutions INDEX]]。阅读它反对[[foreign-financial-institutions/credit-agricole-bank-japan|クレディ・アグリコル銀行 東京支店]]对于同行/对比上下文和[[banking/INDEX|banking index]]适用于更广泛的系统/监管边界。

## 长话短说

毕尔巴鄂比斯开银行东京分行是西班牙/BBVA 银行在日本的分行，被列入金融服务局的外国银行分行名单。作为 BBVA 亚洲分行网络的一部分，它专注于批发银行业务和机构投资者领域。

## 1。牌照/分支机构边界

| 物品 | 笔记 |
|---|---|
| FSA 列出的名称 | 毕尔巴鄂比斯开阿根廷银行 |
| 法定/分支机构名称 | 毕尔巴鄂比斯开银银银行东京分行 |
| 许可途径 | 银行法外国银行分行； FSA 银行牌照清单 |
| 所属司法管辖区 | 西班牙 |
| 家长/团体 | 毕尔巴鄂比斯开银行银牌有限公司 |
| 日本分公司边界 | 亚洲批发分公司；不是日本零售银行平台 |
| 维基角色 | 西班牙批发/机构银行在日本的分行主力 |

## 2。运营模式

BBVA 表示，其亚洲网络包括香港、上海、新加坡、台北和东京的分支机构。它将亚洲分行描述为服务批发银行和机构投资者客户，而代表处则履行联络职能，不提供银行服务。

BBVA 的联系页面将东京列为八重洲洋马东京的分支机构，符合 FSA 分支机构表面逻辑。因此，该分行应被视为 BBVA CIB 亚洲机构网络的一部分。

- 澄清 BBVA 在日本的角色是批发/机构而非零售。
