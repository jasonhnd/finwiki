# Clearstream 日本银行

## 维基路线

此页面将 [[foreign-financial-institutions/foreign-bank-branches-japan-index|the foreign-bank branch registry]] 中的 Clearstream 行提升为独立的 ICSD 路径。使用 [[foreign-financial-institutions/euroclear-bank-japan|Euroclear Bank Japan]]、[[securities/japan-market-infrastructure-map|market infrastructure map]]、[[banking/japan-trust-bank-custody-map|trust-bank custody map]] 和 [[financial-licenses/foreign-financial-group-adjacent-licenses|foreign financial group adjacent-license route]] 来阅读。

首先通过 [[foreign-financial-institutions/INDEX|foreign-financial-institutions INDEX]] 路由此条目，然后使用 [[securities/INDEX|securities domain]] 表示交易后基础设施，使用 [[banking/INDEX|banking domain]] 表示托管/分支机构状态。

＃＃概述

Clearstream Banking S.A. 日本分行最好理解为德意志交易所 / Clearstream 交易后堆栈内的卢森堡 ICSD 银行分行。其在日本的意义在于证券结算、托管、抵押品管理、披露路由和 JASDEC / JGB 访问，而不是消费银行业务。

自 2026-05-13 起，在已检查的 FSA 源中观察到注册情况；除引用的公共来源外，本页面不会对当前许可发表独立意见。

## 为什么它不是仅限注册表

Clearstream 和 [[foreign-financial-institutions/euroclear-bank-japan|Euroclear]] 是日本 wiki 中两个自然的 ICSD 比较页面。注册表行是不够的，因为 Clearstream 的公共材料用于路由：

- 日本证券结算和托管；
- 日本市场持仓的披露要求；
- 获得日本企业、市政和政府证券的机构准入；
- 抵押品和全球流动性管理；
- 日本 CSD / CCP 路线与离岸投资者账户之间的交易后基础设施链接。

## JapanFG 相关性

Clearstream 与 [[foreign-financial-institutions/deutsche-japan|Deutsche Japan]]、[[foreign-financial-institutions/bny-mellon-japan|BNY Mellon Japan]]、[[foreign-financial-institutions/state-street-japan|State Street Japan]]、[[financial-regulators/japan-exchange-group|Japan Exchange Group]] 和 [[financial-regulators/japan-securities-finance|Japan Securities Finance]] 属于同一研究集群。它有助于解释外国金融集团如何能够在不成为零售银行的情况下同时拥有银行分行足迹和相邻的证券/交易后相关性。

当团体拥有多个日本法人实体或活动注册时，请使用 [[financial-licenses/foreign-financial-group-adjacent-licenses|foreign financial group adjacent licenses]]；当问题专门针对分支路线时，请使用 [[financial-licenses/foreign-bank-branch-license-route|foreign-bank branch license route]]。

## 研究清单

1。确定证券类型：日本国债、公司债券、市政债券、股票、投资信托或其他工具。
2。确认问题是否涉及结算、托管、抵押品、披露或税务/受益所有人文件。
3。在使用任何服务级别声明之前，请检查 Clearstream 的当前市场覆盖页面。
4。使用 [[securities/japan-securities-depository-center|JASDEC]]、[[securities/japan-securities-clearing-corp|JSCC]] 和 [[securities/japan-market-infrastructure-map|market infrastructure map]] 交叉读取本地基础设施。
5。如需托管银行比较，请打开 [[banking/japan-trust-bank-custody-map|trust-bank custody map]] 和全球托管页面。

## 注意事项

- ICSD 的存在并不意味着每种日本证券都可以通过每种账户类型平等使用。
- 披露和预扣文件可能取决于投资者类型、账户链和工具。
- 当当前许可很重要时，可以从 FSA 源刷新公共注册表状态。
