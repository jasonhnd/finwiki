# 汇丰日本

## 维基路线

此条目位于 [[foreign-financial-institutions/INDEX|foreign-financial-institutions INDEX]] 下。针对同行/对比背景阅读 [[foreign-financial-institutions/citigroup-japan|シティグループ・ジャパン (Citigroup Japan)]]，针对更广泛的系统/监管边界阅读 [[banking/INDEX|banking index]]。

##TL;博士

英国汇丰控股有限公司（伦敦证券交易所/香港交易所上市，G-SIB Bucket 2）的日本基地。 **1866 横滨分行开业，成为历史最悠久的外资银行之一**，**2012-03 退出“汇丰卓越理财”零售→主要转向企业专用**是定义当前业务形式的转折点。以香港上海银行东京分行（商业/批发银行）+汇丰证券（IB）+汇丰环球资产管理为3 轴，我们专注于日本银行通过亚洲（香港）枢纽的全球扩张、跨国公司的交易银行业务以及外汇商品业务。比赛是[[citigroup-japan]] /渣打银行/德意志银行。 ^[extracted]

## 1。公司简介

**亲会社**：HSBC Holdings plc（LSE / HKEX、ADR: NYSE） ^[extracted]
**G-SIB**：Bucket 2（+1.5% CET1 追加资本要件）^[extracted]
**总部**：英国伦敦（原香港，迁至2010 London）
**业务类型**：在日本**外资银行（企业专业）+证券+资产管理**

|2024 |作为亚洲战略的一部分，维持日本基地并加强批发|

### 亚洲中心战略

- **经香港/新加坡开展日本业务**：日本是汇丰集团整体“重返亚洲”战略中的重要客户群，但产品中心是香港/新加坡。
- **支持日本银行的东盟业务**：随着日本公司向亚洲扩张，抓住交易银行和贸易融资需求
- **跨国公司的主要银行职能**：与日本跨国公司的区域财务中心（RTC）合作，全球现金管理

### 零售撤出的战略影响

- **2012-03 退出“汇丰卓越理财”的原因**：日本零售市场与利润率不匹配（公开声明）^[extracted]
- **客户转账**：主要转账给SBI信托银行（目前[[sbi-shinsei-bank]]集团）等。
- **矛盾效应**：通过退出降低固定成本→通过专业化公司提高资本效率
- **控制**：[[citigroup-japan]] 退出 2015  零售（出售给三井住友 [[smfg]]）。外资IB退出零售是结构性趋势

### 竞争关系

|竞赛|位置|与汇丰银行的区别|
|---|---|---|
|[[citigroup-japan]]|米最大外资、Citi Private Bank で富裕层残存|汇丰银行专注于企业，花旗则拥有私人银行|
|渣打银行（东京分行）|英国新兴市场专业化|汇丰银行拥有香港/中国柜台优势|
|德意志银行东京分行|德国IB，衍生品实力雄厚|汇丰银行专注于商业银行业务|
|摩根大通/高盛/摩根士丹利东京|米鼓支架 IB|汇丰银行比兴业银行更像是一家商业银行|

### 与日本银行的关系

- **MUFG / SMFG / Mizuho之间的互补关系**：为日本银行向亚洲（特别是香港和东南亚）扩张提供本地支持
- **竞争关系较弱**：通过退出零售业务避免与大型银行的正面竞争

## 4. 规制・政策

- **日方监管**：金融厅（FSA）外国银行分行监管
- **本国法规**：英国 PRA（审慎监管局）+ FCA
- **G-SIB Bucket 2**: FSB 认定（+1.5% CET1）^[extracted]
- **日本的监管问题**：
- 外资银行分行 → 基于“日本境内所有资产”的监管
- 第一类金融工具业务（汇丰证券公司）→普通证券公司规定
- 反洗钱（AML）→全球政策+日本金融情报机构合作
