# BNY 梅隆日本 (BNY Mellon Japan)

## Wiki 路线

本条目归于 [[foreign-financial-institutions/INDEX|foreign-financial-institutions INDEX]] 之下。请将其与 [[trust-banks/master-trust-bank|日本マスタートラスト信託銀行]] 对照阅读以获取同业 / 对比语境，并参阅 [[banking/INDEX|banking index]] 以了解更宏观的系统 / 监管边界。

## 1. 公司概要

**最终母公司**：The Bank of New York Mellon Corporation（美国特拉华州法人，NYSE: **BK**，总部纽约市）
**设立**（母公司）：2007（The Bank of New York + Mellon Financial Corporation 合并） ^[extracted]
**前身**：1784 **The Bank of New York**（美国最古老级别的商业银行，Alexander Hamilton 创办的银行之一）→ 与 2007 Mellon Financial（匹兹堡据点，1869  创办）合并 → BNY Mellon
**股票代码**：NYSE: BK

### 日本主要法人（基于公开信息）

```
The Bank of New York Mellon Corporation（美 NYSE: BK・FRB / OCC 监督）
  ├── BNY 梅隆信托银行株式会社 ── 日本法人，持有信托业牌照
  │     └── 在国内据点提供托管・信托・资产管理服务
  ├── 纽约梅隆银行 东京分行 ── 与美国总部直接连接的在日分行
  │     └── 全球托管・交易银行业务
  └── BNY 梅隆资产管理日本 ── 在日本的资产管理业务
        └── 面向机构投资者的全球运用产品的销售・运用
```

**特点**：
- **完全不经营零售银行・个人存款**。彻底专注于 B2B / B2 机构投资者

### 信托银行 route 的处理

BNY 的日本官方页面说明，其于 2009 年 收购了日本的信托银行业务，并连接到现在的 **纽约梅隆信托银行株式会社**。另一份官方 disclosure page 也将纽约梅隆信托银行说明为在日本设立的信托银行，是以面向法人客户的受托服务为主业的登记金融机构。

因此，FinWiki 不将 `bny-mellon-trust-bank` 作为独立页面大量产出，而是将本页作为 [[foreign-financial-institutions/foreign-bank-branches-japan-index|foreign-bank branch registry]] 与 [[trust-banks/trust-companies-japan-index|trust-company registry]] 的 umbrella route 处理。仅在必要时，从本节进入与 [[trust-banks/master-trust-bank|日本マスタートラスト信託銀行]] 或 [[trust-banks/custody-bank|日本カストディ銀行]] 的 custody / trust-bank 比较。

## 2. 沿革（年表节选）

| 年 | 事件 |
|---|---|
| **1968** | **The Bank of New York 开始日本业务**（在东京启动银行业务） ^[extracted] |
| 1970s〜80s | 在日资银行全球扩张期扩大托管・交易银行业务 |
| **1989** | **以股票投资代理人业务（ADR 代理人）支持众多日本企业的海外证券发行**（确立作为在美上市日本企业 ADR 存托机构的地位） ^[extracted] |
| 1990s〜2000s | 随日资银行・养老基金海外资产运用的扩大而全面开展托管业务 |
| **2007-07-01** | **The Bank of New York + Mellon Financial 合并 → BNY Mellon 成立**（诞生专注于托管・资产管理的全球银行） ^[extracted] |
| 2010s | **日资银行・养老基金（GPIF 等）的全球托管受托扩大**（日元贬值・海外投资转移的顺风） ^[extracted] |

## 3. 业务分部地图

| 分部 | 在日承担方 | 特点 |
|---|---|---|
| **全球托管** | 纽约梅隆银行 东京分行 + BNY 梅隆信托 | 美国 ADR・海外股票・海外债券・另类资产的保管・结算・公司行动处理 — 母公司主力的全球收益支柱 |
| **国内信托（托管周边）** | BNY 梅隆信托银行 | 信托业牌照下的国内资产管理・受托业务 |
| 机构投资者服务 | BNY 梅隆信托 + 东京分行 | 面向日资银行・养老基金・运用公司的后台・基金行政・过户代理 |
| 资产管理（AM）| BNY 梅隆资产管理日本 | 全球运用产品的日本销售・运用 — 经由 Insight / Newton / Walter Scott 等多精品店（multi-boutique）结构 |
| ADR / 存托凭证 | 东京分行 + 美国总部 | 支持日本企业的美国 ADR 发行（存托机构业务）/ 支持海外发行 |
| 个人零售 | **无** | — |

## 5. 监督・监管

- **主管**：金融厅（FSA）— 作为信托银行・在日分行
- **美方监督**：FRB（联邦储备理事会）/ OCC（货币监理署）/ FDIC，全球压力测试 DFAST 对象
- **近期议题**：
  - **GPIF・公共养老金的海外资产运用比例上升** → 托管需求结构性扩大
  - **日元贬值・海外投资转移** → 日资银行・运用公司增持海外资产带来的受托资产增长
  - 外汇法・对内直接投资审查 → 海外资产保管业务中的监管应对
