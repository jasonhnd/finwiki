# 道富 (State Street Japan)

## Wiki 路径

本条目位于 [[foreign-financial-institutions/INDEX|foreign-financial-institutions INDEX]] 之下。请将其与 [[foreign-financial-institutions/bny-mellon-japan|BNY メロン日本 (BNY Mellon Japan)]] 对照阅读以获取同类／对比语境，并与 [[banking/INDEX|banking index]] 对照阅读以了解更广泛的系统／监管边界。

## 1. 公司概要

**最终母公司**：State Street Corporation（美国马萨诸塞州法人，NYSE: **STT**，总部马萨诸塞州波士顿）
**母公司设立**：1792（作为 The Union Bank，是美国 2 古老的持续经营银行之一）^[extracted]
**现体制成立**：1969（State Street Boston Financial Corporation 设立控股公司）→ 1978 改称 State Street Corporation ^[extracted]
**股票代码**：NYSE: STT

### 全球财务（2025〜2026Q1 期）

| 项目 | 规模 |
|---|---|
| AUC/A（受托资产 + 管理资产）| **约 49 兆美元**（2025 2Q）→ **约 53.8 兆美元**（2025 4Q，历史最高）^[extracted] |
| 全球排名（托管）| **No.2**（居首的 [[foreign-financial-institutions/bny-mellon-japan]] 约 59.3 兆美元）^[extracted] |
| 近期新增受托净增 | 约 1.1 兆美元（2025 期内，主要经由 Asset Owners / Asset Managers）^[extracted] |

### 日本主要法人（基于公开信息）

```
State Street Corporation（美国 NYSE: STT・FRB / OCC 监管）
  ├── 道富信托银行股份公司 ── 日本法人，持有信托业牌照
  │     ├── 东京本店: 港区虎之门 1-23-1 虎之门 Hills 森塔 25F
  │     ├── 福冈营业所: 福冈市博多区纲场町 (2012 开设，截至 2016 已超 100 名)
  │     └── 横滨营业所 (2012 开设 → 2015 关闭)
  ├── 道富银行 东京支行 (State Street Bank and Trust Company 在日支行)
  │     └── 直连美国本店的全球托管・交易银行窗口
  ├── 道富环球投资管理股份公司 (SSGA)
  │     ├── 1998-02 设立，关东财务局长 (金商) 第 345 号
  │     └── SPDR ETF 系列的运营方 (含 SPY = 全球最大 ETF)
  └── 查尔斯河 开发 公司 (Charles River Development Japan)
        └── 2007 设立，面向运营公司的交易 / 投资组合管理系统
```

**特征**：
- **完全不经营零售银行・个人存款**。完全专注于 B2B / B2 机构投资者
- **SSGA 的存在是与 BNY Mellon 的差异点**：在日本亦展开 SPDR ETF（含全球最大的 SPY），兼具 ETF 运营 + 托管两轮 ^[extracted]
- **道富信托银行 注册资本 25 億 1000 万円，员工 426 名（截至 2024-04-30），R&I 评级 AA-** ^[extracted]
- **代表人**：信托银行社长 **寺田惠子**，东京支行长 **若林宪弘**，SSGA 社长 **越前谷通博** ^[extracted]

## 2. 沿革（年表摘录）

| 年 | 事件 |
|---|---|
| **1792** | **The Union Bank（State Street 前身）设立**（美国 2 古老的银行系谱之一）^[extracted] |
| 1969 | State Street Boston Financial Corporation 设立控股公司 ^[extracted] |
| 1978 | 改称为 State Street Corporation ^[extracted] |
| **1986-03** | **道富信托银行的前身 Manufacturers Hanover 信托银行股份公司 设立**（美国 Manufacturers Hanover 公司 100% 出资）^[extracted] |
| **1990** | **道富银行 东京支行 开设**（作为 State Street Bank and Trust Company 的在日支行）^[extracted] |
| **1991-12** | **Manufacturers Hanover 信托银行 → 更名为道富信托银行**（纳入 State Street 旗下）^[extracted] |
| **1998-02** | **道富环球投资管理股份公司 (SSGA) 设立**（资产运营业务正式化，SPDR 系列在日本展开的基盘）^[extracted] |
| **2006** | **对道富银行 东京支行的行政处分**（详情见公开信息，当时的业法违反）^[extracted] |
| **2007** | **Charles River Development 日本据点设立**（扩大面向运营公司的 OMS/PMS 供给）^[extracted] |
| **2012** | **道富信托银行 开设福冈营业所 + 开设横滨营业所**（据点扩张期）^[extracted] |
| 2015 | 关闭横滨营业所 ^[extracted] |
| 2016 | 福冈营业所人员扩大至 100 名超（集中后台职能）^[extracted] |
| **2025 2Q** | **达成 AUC/A 约 49 兆美元（历史最高）** ^[extracted] |
| **2026-01-16** | **2025 4Q 决算中刷新 AUC/A 53.8 兆美元的新纪录** ^[extracted] |

## 3. 业务板块・版图

| 板块 | 在日承担者 | 特征 |
|---|---|---|
| **全球托管** | 道富银行 东京支行 + 信托银行 | 美国 ADR・海外股票・海外债券・另类资产的保管・结算・公司行动处理 — 母公司的主力收益支柱 |
| **国内信托（托管周边）** | 道富信托银行 | 信托业牌照下的国内资产管理・受托业务，GPIF 运营受托机构的一角 ^[extracted] |
| 机构投资者服务 | 信托银行 + 东京支行 | 面向本国银行・养老金・运营公司的后台・基金行政・过户代理 |
| **资产运营（AM）** | **SSGA（道富环球投资管理）** | 面向公共・私人养老金、机构投资者、投资信托的运营 / **SPDR ETF 系列的运营方**（SPY = 全球最大 ETF）^[extracted] |
| **运营系统** | Charles River Development Japan | 面向运营公司的 OMS/PMS/EMS（买入・运营管理系统）SaaS — BNY Mellon 所没有的独特业态 |
| 个人零售 | **无** | — |

### 在日本的定位

- **GPIF・大型养老金・本国银行的海外投资托管主要承担者**：与日本 Master Trust [[trust-banks/master-trust-bank]] 并列的 GPIF 资产管理机构之一 ^[extracted]
- **[[megabanks/mufg]] 股东构成 4.51%（2025-03-31）的含义**：这是 **机构投资者经由海外保管账户的名义集中**（托管行名义集积），而非 State Street 自身的战略持有 ^[extracted]

- **SPDR 系列**：联动 S&P 500 的 SPY 是全球最大 ETF，在日本亦经由 SSGA 面向机构投资者展开 ^[extracted]

## 5. 监督・监管

- **主管**：金融厅（FSA）— 作为信托银行・在日支行・SSGA（金商业者）
- **美国侧监督**：FRB（联邦储备制度理事会）/ OCC（货币监理署）/ FDIC，纳入全球压力测试 DFAST 对象
- **近期论点**：
  - **GPIF・公共养老金的海外资产运营比率上升** → 托管需求结构性扩大
  - **日元贬值・海外投资转移** → 本国银行・运营公司增持海外资产带来受托资产增长
  - **2025〜2026 AUC/A 刷新历史最高**：达 53.8 兆美元，但伴随成本上升出现「收益悖论」担忧（市场冷静看待）^[extracted]
