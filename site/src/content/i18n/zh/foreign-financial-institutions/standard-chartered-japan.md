# 渣打银行东京分行 (Standard Chartered Bank Tokyo)

## Wiki route

本条目位于 [[foreign-financial-institutions/INDEX|foreign-financial-institutions INDEX]] 下。可与 [[foreign-financial-institutions/bny-mellon-japan|BNY メロン日本 (BNY Mellon Japan)]] 对照阅读，以理解同业和差异；更广泛的制度和监管边界见 [[banking/INDEX|banking index]]。

## TL;DR

这是英国 Standard Chartered PLC（总部伦敦，LSE: STAN / HKSE: 2888  / FTSE 100  成分股）的日本据点。由于**业务集中在亚洲、中东和非洲（AMEA）新兴市场**，其战略轴明显不同于 [[hsbc-japan]]、[[citigroup-japan]] 这类全球 universal bank。在日本，它拥有可追溯至 1880 年 年代的长期历史，但不开展零售业务，属于**法人专营**。其利基位置在于，为日本银行难以自建网点的新兴市场（南亚 / 东南亚 / 中东 / 非洲）提供 correspondent banking、transaction banking 和 trade finance 功能。^[extracted]

## 1. 公司概要

**正式名称（日本据点）**：Standard Chartered Bank, Tokyo Branch ^[extracted]
**英文名称**：Standard Chartered Bank, Tokyo Branch
**母公司**：Standard Chartered PLC（英国伦敦总部，1969  设立，LSE: STAN / HKSE: 2888）^[extracted]
**主要所在地**：东京（丸之内区域）^[extracted]
**业务范围（日本）**：以法人银行业务为中心，包括 corporate banking、transaction banking、trade finance、外汇和资金结算。**不开展个人零售业务** ^[extracted]

### 集团位置（母公司）

- **总部所在地**: 伦敦（实际业务中心为 AMEA = 亚洲、中东和非洲）。
- **上市**: 伦敦证券交易所（LSE: STAN，FTSE 100  成分股）/ 香港证券交易所（HKSE: 2888）。
- **网点**: 约 60  个国家 / 地区。
- **优势**: 新兴市场（中国、香港、新加坡、韩国、印度、巴基斯坦、UAE、撒哈拉以南非洲）correspondent / trade 网络。

### 主要法人（日本相关）

- **Standard Chartered Bank, Tokyo Branch**: 法人银行业务核心。
- **Standard Chartered Securities (Japan) Ltd.**: 在东京继续经营证券业务。它是第一种金融商品交易业者（关东财务局长（金商）第 2922 号），并加入日本证券业协会。覆盖 Markets / transaction banking / specialized finance 领域；截至 2025 年 仍持续公开披露和各类政策，表明现存并运营中。

## 2. 沿革

| 年月 | 事项 |
|---|---|
| 1853 | Chartered Bank of India, Australia and China 在伦敦设立，获英国王室特许 |
| **1880** | **Chartered Mercantile Bank of India, London and China 横滨分行开设**（进入日本）^[extracted] |
| 1862 | Standard Bank of British South Africa 在伦敦设立 |
| **1969** | **Chartered Bank + Standard Bank 合并为 Standard Chartered Bank**；日本据点也整合到“Standard Chartered”名下 ^[extracted] |
| 1990s-2000s | 集中于亚洲新兴市场（中国、印度、东南亚）+ 中东和非洲战略；日本则收窄为法人专营 |
| 2010s | 作为亚太地区战略枢纽；日本强化面向邦银的新兴市场 correspondent 功能 |
| 2020s | 强化 ESG / transition finance 和数字 transaction banking |

## 3. 业务分部图（日本据点）

| 分部 | 内容 | 客户层 |
|---|---|---|
| **Corporate banking** | 面向日本法人和在日外资的融资 / 授信 | 日本大型企业、在日外资 |
| **Transaction banking** | 国际结算、现金管理、托管 | 邦银（correspondent 对手）、跨国公司 |
| **Trade finance** | LC、进出口金融、供应链金融 | 与 AMEA 交易的商社和制造业 |
| **外汇 / 资金市场** | FX、利率、新兴市场货币套期保值 | 法人客户 |
| **Correspondent banking** | 为邦银无法直接设点的非洲、南亚、中东路径汇款提供中继 | **邦银（最重要分部）** |
| 零售 | **未开展**（日本不办理个人业务）| - |

### 母公司 PLC 的独特位置

- 与其他大型外资银行（[[hsbc-japan]] 母 HSBC / [[citigroup-japan]] 母 Citigroup / JPMorgan / BNP Paribas）不同，其战略轴是“**专注亚洲、中东和非洲的全球银行**”^[extracted]。
- 以旧大英帝国贸易网络（India / Hong Kong / Singapore / South Africa / UAE）为基础，拥有 170  年级别的当地存在。
- 不在北美和西欧大陆开展零售扩张，而将资本和人才集中于 AMEA。

### 日本的竞争与分工

- **竞争对手**: [[hsbc-japan]]（英国系、战略类似但 HSBC 更全球化）/ [[citigroup-japan]]（美国系，在日富裕层零售一部分）/ Deutsche Bank / BNP Paribas Tokyo / JPMorgan Tokyo 等外资法人银行。
- **差异化**: 集中 AMEA，尤其是**非洲和南亚存在感强于其他外资银行** ^[extracted]。
- **与邦银的关系**: 不是竞争而是**互补**，向邦银提供基础设施。

## 5. 监管 / 政策

- **母公司监管**: 英国 PRA（审慎监管机构）/ FCA（金融行为监管机构）+ 香港 HKMA（香港金融管理局）。
