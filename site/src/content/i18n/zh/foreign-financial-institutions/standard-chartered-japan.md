# 渣打银行 东京支店 (Standard Chartered Bank Tokyo)

## Wiki 路径

本条目位于 [[foreign-financial-institutions/INDEX|foreign-financial-institutions INDEX]] 之下。请与 [[foreign-financial-institutions/bny-mellon-japan|BNY メロン日本 (BNY Mellon Japan)]] 对照阅读以获取同业／对比语境，并与 [[banking/INDEX|banking index]] 对照阅读以了解更广泛的体系／监管边界。

## TL;DR

英国 Standard Chartered PLC（总部伦敦，LSE: STAN / HKSE: 2888 / FTSE 100  成分股）的日本据点。**业务集中于亚洲・中东・非洲（AMEA）新兴市场**这一点，使其与全球综合性银行型的 [[foreign-financial-institutions/hsbc-japan]] 或 [[foreign-financial-institutions/citigroup-japan]] 在战略轴线上划清界限。在日本拥有可追溯至 1880 年 年代的悠久历史，但不开展零售业务、**专营法人业务**。其定位为利基市场，代邦银承担面向“难以自设据点的新兴国”（南亚 / 东南亚 / 中东 / 非洲）的代理行、交易银行、贸易融资功能。^[extracted]

## 1. 公司概要

**正式名称（日本据点）**：渣打银行 东京支店 ^[extracted]
**英文名**：Standard Chartered Bank, Tokyo Branch
**母公司**：Standard Chartered PLC（英国・伦敦总部，1969  设立，LSE: STAN / HKSE: 2888）^[extracted]
**主要所在地**：东京（丸之内区域）^[extracted]
**业务范围（日本）**：以法人银行业务（公司银行、交易银行、贸易融资、外汇・资金结算）为中心。**个人零售业务未开展** ^[extracted]

### 集团定位（母公司）

- **总部所在**: 伦敦（实体业务在 AMEA = 亚洲・中东・非洲）
- **上市**: 伦敦证券交易所（LSE: STAN，FTSE 100  成分股）/ 香港证券交易所（HKSE: 2888）
- **据点**: 约 60  个国家 / 地区
- **优势**: 新兴国（中国・香港・新加坡・韩国・印度・巴基斯坦・UAE・撒哈拉以南非洲）代理行・贸易网络

### 主要法人（日本相关）

- **渣打银行 东京支店**: 法人银行业务的核心
- **渣打证券株式会社（Standard Chartered Securities (Japan) Ltd.）**: 在东京持续开展证券业务。第一种金融商品交易业者（关东财务局长（金商）第 2922 号），日本证券业协会加盟。办理 Markets / 交易银行 / 专业金融领域（截至 2025 年 持续公开披露及各类方针，现存・运营中）

## 2. 沿革

| 年月 | 事件 |
|---|---|
| 1853 | 特许银行（Chartered Bank of India, Australia and China）伦敦设立，获英国王室特许 |
| **1880** | **Chartered Mercantile Bank of India, London and China 横滨支店 开设**（进入日本）^[extracted] |
| 1862 | 标准银行（Standard Bank of British South Africa）伦敦设立 |
| **1969** | **特许银行 + 标准银行 合并 → Standard Chartered Bank**（日本据点亦整合为“渣打”）^[extracted] |
| 1990 年代〜2000 年代 | 集中于亚洲新兴国（中国・印度・东南亚）+ 中东・非洲战略。在日本收窄为法人专营 |
| 2010 年代 | 亚太地区的战略性枢纽功能。在日本强化面向邦银的新兴国代理行功能 |
| 2020 年代 | 强化 ESG・转型金融、数字交易银行 |

## 3. 业务分部图（日本据点）

| 分部 | 内容 | 客户层 |
|---|---|---|
| **公司银行** | 向邦银法人・在日外资提供融资・授信 | 日本大企业・在日外资 |
| **交易银行** | 国际结算・现金管理・托管 | 邦银（代理行客户）・跨国企业 |
| **贸易融资** | LC・进出口金融・供应链金融 | 商社・制造业的 AMEA 交易 |
| **外汇・资金市场** | FX・利率・新兴国货币对冲 | 法人客户 |
| **代理行业务** | 中转邦银无直接据点的非洲・南亚・中东汇款 | **邦银（最重要分部）** |
| 零售 | **未开展**（在日本不开展个人业务）| - |

### 母公司 PLC 的独特定位

- 不同于其他大型外资（[[foreign-financial-institutions/hsbc-japan]] 母公司 HSBC / [[foreign-financial-institutions/citigroup-japan]] 母公司 Citigroup / JPMorgan / BNP Paribas）的战略轴线，即“**专精亚洲・中东・非洲的全球银行**” ^[extracted]
- 以旧大英帝国贸易网络（India / Hong Kong / Singapore / South Africa / UAE）为母体的 170  年级别本地存在
- 在北美・西欧大陆不开展零售 → 将资本・人才集中于 AMEA

### 在日本的竞争与分工

- **竞争对手**: [[foreign-financial-institutions/hsbc-japan]]（英国系・类似战略，但 HSBC 更全球化）/ [[foreign-financial-institutions/citigroup-japan]]（美系・在日富裕层零售一部分）/ Deutsche Bank / BNP Paribas Tokyo / JPMorgan Tokyo 等外资法人银行
- **差异化**: AMEA 集中，尤其 **在非洲・南亚的存在凌驾于其他外资之上** ^[extracted]
- **与邦银的关系**: 并非竞争而是**互补**（向邦银提供基础设施）

## 5. 监管・政策

- **母公司监管**: 英国 PRA（审慎监管局）/ FCA（金融行为监管局）+ 香港 HKMA（香港金融管理局）
