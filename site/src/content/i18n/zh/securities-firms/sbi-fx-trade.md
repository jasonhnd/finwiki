# SBI FXTRADE


## Wiki 路径

本条目位于 [[JapanFG/INDEX|JapanFG index]] 之下。可与 [[securities-firms/traders-securities|トレイダーズ証券]] 对读以理解同业和对比关系，并与 [[securities/INDEX|securities index]] 对读以把握更广的制度 / 监管边界。

## TL;DR

SBI FXTRADE 是 [[megabanks/sbi-hd|SBI group]] 的 FX 专门 FIEA operator。在金融厅金融商品交易业者清单中，它以“SBI FXトレード株式会社”、关东财务局长（金商）第2635号 列示。它与 SBI 证券是不同法人，负责 FX / 定投 FX / option FX / crypto CFD adjacency。

## 1. 法人 / 牌照边界

| 项目 | 解读 |
|---|---|
| 法定名称 | SBI FXトレード株式会社 |
| 金融厅注册 | 关东财务局长（金商）第2635号 |
| FIEA 范围 | 第一类金融商品交易业 |
| 母公司边界 | SBI Liquidity Market 100% subsidiary |
| 核心产品 | FX、定投 FX、option FX、crypto CFD-adjacent services |

## 2. 业务角色

- 作为 FX 专门公司，主打小额、透明性和 SBI 集团联动。
- 独立于 [[securities-firms/sbi-securities|SBI証券]] 的综合 broker function，负责 FX execution / retail FX product。
- 在 crypto / CFD / EPI 周边与 [[exchanges/jp-exchange-sbi-vc-trade|SBI VC Trade]] 相邻。

## 3. 为什么需要独立页面

SBI group 的证券、银行、加密资产和 FX 密切交织。将 SBI FXTRADE 独立出来，可避免混同 FX 专业公司、FIEA 第一类、crypto CFD adjacency 和 NEOBANK 联动。
