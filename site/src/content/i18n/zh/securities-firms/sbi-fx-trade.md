# SBI FX交易 (SBI FXトレード)

## Wiki 路径

本条目位于 [[securities-firms/INDEX|securities-firms INDEX]]（其领域路径）之下。请将其与 [[securities-firms/traders-securities|トレイダーズ証券]] 对照阅读，作为最接近的 FX 专业同类／对照，并与 [[financial-regulators/fsa|the FSA]] 及 [[financial-regulators/jsda|JSDA]] 对照，以厘清 FIEA（金商法）的监管／自律边界。综合券商的兄弟公司为 [[securities-firms/sbi-securities|SBI証券]]。

## TL;DR

SBI FXトレード株式会社是 [[megabanks/sbi-hd|SBI group]] 的 FX 专门 FIEA operator。作为金融商品交易法上的 **第一种金融商品交易业者**，以「SBI FXトレード株式会社」（关东财务局长(金商)第2635号）列入金融厅金融商品交易业者一览。据公开会社概要，其为 SBIリクイディティ・マーケット株式会社的 100% 子公司，与 [[securities-firms/sbi-securities|SBI証券]] 作为不同法人，承担 店頭FX / 積立FX / option FX / crypto CFD adjacency。

## 2. 运营模式

SBI FXトレード作为 FX 专门公司，主打小额、透明性与 SBI 集团联动。与 [[securities-firms/sbi-securities|SBI証券]] 的综合 broker function 不同，其承担 FX execution / retail FX product。在 crypto / CFD 周边与 [[exchanges/jp-exchange-sbi-vc-trade|SBI VC Trade]] 相邻。在 retail 店頭FX 市场，与 [[securities-firms/dmm-com-securities|DMM.com証券]]、[[securities-firms/gmo-click-securities|GMOクリック証券]]、[[securities-firms/traders-securities|トレイダーズ証券]] 并列比较。

## 3. 本页面为何重要

- SBI group 的证券、银行、加密资产、FX 紧密交织。将 SBI FXトレード独立出来，便可不混淆地追踪 FX 专营公司、FIEA 第一种、crypto CFD adjacency 与 NEOBANK 联动。
- 确定 [[financial-regulators/fsa|FSA]] 监管下的 FX 专门第一种 operator anchor，并记录母公司（SBIリクイディティ・マーケット）边界。
