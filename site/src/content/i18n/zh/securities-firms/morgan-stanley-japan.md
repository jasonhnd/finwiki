# 摩根士丹利日本 (Morgan Stanley Japan)

## Wiki route

This entry sits under [[securities-firms/INDEX|securities-firms INDEX]]. Read it against [[securities-firms/goldman-sachs-japan|ゴールドマン・サックス・ジャパン (Goldman Sachs Japan)]] for peer / contrast context and [[securities/INDEX|securities index]] for the broader system / regulatory boundary.

## TL;DR

美国 Morgan Stanley (NYSE: MS) 的日本据点。通过 **与 MUFG [[megabanks/mufg]] 的战略合作 (2008-09)**，构建了日本银行最深度的全球 IB 合作关系。2010-05 年整合日本证券业务，以 **2 社体制**(三菱 UFJ 摩根士丹利证券 = MUMSS / 摩根士丹利 MUFG 证券 = MSMS) 运营。MUMSS 负责面对面零售 + ECM/DCM，MSMS 负责批发 (M&A·机构投资者)。占据 5 大证券之一席，在全球投资银行业务中与野村 [[securities-firms/nomura-hd]] 并列具有存在感。

## 1. 公司概要

**母公司**: Morgan Stanley (NYSE: MS) — 美国纽约总部，全球投资银行·WM·IM
**日本统辖**: 摩根士丹利日本控股株式会社 (Morgan Stanley Japan Holdings Co., Ltd.)
**业态**: 外资系投资银行 (通过与 MUFG 的战略合作在日本拓展)
**与 MUFG 的关系**: MUFG 持续持有 Morgan Stanley 普通股 ~20% (2008-09 取得后，经稀释调整后亦曾有 ~24% 时期)

### 主要法人 (4 社体制)

| 法人 | 出资比例 | 业态 | 设立 |
|---|---|---|---|
| **三菱 UFJ 摩根士丹利证券 (MUMSS)** | **MUFG 60% / MS 40%** | 面对面零售 + ECM/DCM | 2010-05-01 整合 |
| **摩根士丹利 MUFG 证券 (MSMS)** | **MUFG 49% / MS 51%** | 批发 (M&A / 机构投资者) | 2010-05-01 整合 |
| 摩根士丹利·MUFG 研究与培训 | MS / MUFG JV | 研究·培训 | - |
| 摩根士丹利·资本株式会社 | MS 100% | 资本业务 | - |

> 表决权分配的非对称性: MUMSS 由 MUFG 过半 (日本银行主导)，MSMS 由 MS 过半 (外资主导)。以 **证券牌照的双重结构** 将面对面零售 (日本银行品牌) 与批发 (外资品牌) 分离 → 监管应对 + 客户细分优化。

### 战前〜整合前 (1970s〜2007)

- **1970 年年代** Morgan Stanley 进入东京 (驻在员事务所 → 分行)
- **1980 年年代** 日本股票热潮期，在东京大规模扩张
- **1990 年年代** 与三菱信托银行 (现 MUFG 信托) 合作 (投信·养老金领域)
- **2007〜2008** 次贷危机显现 → 全球金融危机

### MUFG 资本合作与整合 (2008〜2010) ★ 核心

- **2008-09-15** 雷曼兄弟破产 → Morgan Stanley 股价暴跌，流动性不安
- **2008-09-22** **MUFG 以约 90 億美元 (约 9000 億円超) 取得 Morgan Stanley 普通股 21%** ★
  - 最初预定为带表决权优先股 → 在交易完成前夕变更为普通股 + 优先股的混合
  - 与美国财政部 TARP 的并行协定，雷曼之后的标志性交易
  - 对 MUFG 而言是日本银行最大的对外投资
- **2009-06** 优先股 → 普通股转换完成
- **2010-05-01** **MUFG × Morgan Stanley 日本证券业务整合** ★
  - 三菱 UFJ 证券 + Morgan Stanley Japan Securities → **MUMSS (MUFG 60% / MS 40%)** + **MSMS (MUFG 49% / MS 51%)** 2 社体制
  - MUMSS 负责面对面零售 + 国内 IB，MSMS 负责批发
  - 作为日本银行的全球 IB 合作而言最为深度 (与野村收购雷曼并列为标志)

### 整合后 (2011〜现在)

- **2011-06** MUFG 开始领取分红 (来自 Morgan Stanley 普通股)
- **2013** MUMSS 系统故障·治理强化阶段
- **2016〜** MUMSS 零售整合效果显现，账户数扩大
- **2020〜** 新冠疫情下 M&A·ECM 案件扩大，MSMS 业绩稳健
- **2023〜2024** 日本股票上涨·大型 IPO 增加，MUMSS / MSMS 双双取得良好业绩
- **2024** 战略合作持续 16 年，Morgan Stanley 持续将 MUFG 定位为「Strategic Alliance Partner」

### 与 MUFG 的 JV 体制 (核心战略)

- **双重牌照结构**: MUMSS (日本银行过半) + MSMS (外资过半)，实现监管·客户优化
- **跨职能**: MUFG 的银行客户基础 × MS 的全球 IB 产品
- **资本效率**: MUFG 以权益法投资利润纳入 Morgan Stanley 业绩 (年间数千亿日元规模的利润贡献)
- **竞争差异化**: 日本银行大型机构 + 外资 IB 的组合独一无二 ([[megabanks/smfg]] = 日兴独立，[[megabanks/mizuho-fg]] = 瑞穗证券独立)

### 批发战略 (MSMS)

- **跨境 M&A**: 日本企业海外收购·海外企业日本收购两方面
- **机构投资者交易**: 搭配全球流动性
- **股票研究**: 以全球整合覆盖获得机构投资者信赖

### 零售战略 (MUMSS)

- **集中于个人富裕层**: 面对面营销 + 产品多样性
- **MUFG 银行介绍渠道**: 银行客户 → 引导至证券
- **海外产品**: 搭配 MS 的全球基金

### 海外联动

- 与亚洲 (香港·新加坡)·美国·欧洲据点联动
- 作为 Morgan Stanley 全球组织的一部分运营

## 5. 监管·政策

- **主管**: 金融厅 (FSA) / 证券交易等监视委员会 (SESC)
- **第一种金融商品交易业**: MUMSS / MSMS 两社均登记
- **金融商品交易法**: 应对业务行为监管
- **美国方面**: Morgan Stanley 受 FRB·SEC·FINRA 监管 (Bank Holding Company)
- **国际**: G-SIB Bucket 1 (FSB 认定，作为 Morgan Stanley)
