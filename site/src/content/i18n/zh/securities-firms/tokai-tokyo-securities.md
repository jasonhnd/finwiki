# 東海東京証券 (Tokai Tokyo Securities)

## TL;DR

東海東京証券株式会社 是 **名古屋本店的中坚独立系证券公司**，东证 PRIME 上市 [[securities-firms/tokai-tokyo-fh|東海東京フィナンシャル・ホールディングス (8616)]] 的核心 100% 子公司。**2008-10  通过旧 东海东京证券（东海银行系）的事业分公司化** 形成现体制，其源流为 1929 年（昭和 4 年）创业的丸万證券。**金融商品交易业者 东海财务局长(金商)第140号**（FIEA Type 1  第一种金融商品交易业）。本店 名古屋市中村区名駅 Midland Square，资本金 60 億円，员工 1,869 名。作为 **以东海地方（爱知・岐阜・三重・静冈）为基础、以面对面零售为中心的中坚独立证券**，展开地银合作模式（与东海地方地银的地域共同网点）、IPO 承销、咨询型营业。与地银合资 / 共同网点品牌「东海东京 〇〇 证券」是其特征。竞争对手为 [[securities-firms/iwai-cosmo-securities|岩井コスモ証券]]・[[securities-firms/okasan-sg|岡三証券グループ]]・[[securities-firms/aizawa-sg|藍澤証券]]・[[securities-firms/matsui-sec|松井証券]]。

## 1. 法人・股东

- **商号**：東海東京証券株式会社（英文：Tokai Tokyo Securities Co., Ltd.）
- **设立**：2008 年 10 月 8 日（向 [[securities-firms/tokai-tokyo-fh|東海東京FH]] 转为控股公司时分公司化）
- **创业源流**：1929 年（昭和 4 年）以丸万證券创业
- **本店所在地**：〒450-6212 名 名古屋市中村区名駅 4-7-1  Midland Square
- **资本金**：60 億円
- **母公司**：[[securities-firms/tokai-tokyo-fh|東海東京フィナンシャル・ホールディングス]]（东证 PRIME 8616，总部 名古屋）100% 出资
- **代表者**：代表取缔役社长
- **员工数**：1,869 名（HD 合并基础，2024 年 时点）

## 2. 牌照・登记

- **第一种金融商品交易业**：东海财务局长（金商）第140号
- **行业团体加入**：
  - 日本证券业协会 (JSDA) 正会员
  - 一般社团法人 金融期货交易业协会
  - 一般社团法人 第二种金融商品交易业协会
- **交易参加权**：[[securities/tokyo-stock-exchange|TSE]]・[[securities/osaka-exchange|OSE]] 交易参加者・[[securities/japan-securities-clearing-corp|JSCC]] 清算参加者

## 3. 营业基础

- **面对面营业**：本店 + 全国超 130  个据点的网点 / 营业所
- **地银合作模式**：拥有与以下地银合资设立的地域证券公司
  - 「**ほくほく TT 証券**」（与 [[regional-banks/hokuriku-bank|北陸銀行]] / [[regional-banks/hokkaido-bank|北海道銀行]] 系 [[regional-banks/hokuhoku-fg|ほくほくFG]] 合资）
  - 「**東海東京＆シティ証券**」（与 [[regional-banks/oita-bank|大分銀行]] / FFG 系 合资）
  - 「**浜銀 TT 証券**」（与 [[regional-banks/yokohama-bank|横浜銀行]] / [[regional-banks/concordia-fg|コンコルディアFG]] 系 合资）
  - 「**西日本シティ TT 証券**」（与 [[regional-banks/nishi-nippon-city-bank|西日本シティ銀行]] 合资）
  - 与其他多家地银的合作
- **网络交易服务**：「あんしん総合サービス」「東海東京ネットフィッシュ」
- **呼叫中心**：多个据点

## 4. 经办产品

- **国内股票**：现货・信用，作为 TSE / OSE / 名古屋证券交易所 (NSE) 交易参加者的 3  大市场线
- **外国股票**：美国股、欧洲股、亚洲股
- **投资信托**：自社运用 + 各社基金
- **债券**：国债・地方债・公司债・外国债券
- **NISA**：成长投资额度・定额投资额度
- **iDeCo**：个人型确定缴费年金计划
- **衍生品**：期货・期权・FX
- **IPO 承销**：以东海地方为中心的中坚企业 IPO 的主承销 / 承销负责
- **M&A 顾问**：面向中坚企业的事业承继 / 出售 / 收购咨询

## 5. 行业站位

- 作为 **名古屋・东海地方基础的中坚独立证券**，展开大型 FG 系证券与地银的「合资证券」模式的独特站位
- 也被称为 **「东海地方的野村」** 的面对面 + 地银联动混合战略
- 竞争：[[securities-firms/iwai-cosmo-securities|岩井コスモ証券]]（关西）、[[securities-firms/okasan-sg|岡三証券]]（名古屋 peer）、[[securities-firms/aizawa-sg|藍澤証券]]、[[securities-firms/matsui-sec|松井証券]]、[[securities-firms/marusan-securities|丸三証券]]
- **优势**：与东海地方中坚 / 大型企业的深厚人脉、经由地银渠道的零售客户扩大、地银证券合资模式的先驱者
- **特殊事业**：以与 [[securities/japan-online-brokerage-competition|オンライン証券競争]] 不同的面对面 + 地银联动模式差异化
- **战略**：深化地银合作・东南亚 （越南 / 印度尼西亚）海外证券事业展开

[[securities-firms/tokai-tokyo-fh|東海東京FH]] 的页面偏向控股公司视角的财务・股东结构。**牌照边界（东海财务局长 第140号 FIEA Type 1）・营业基础（130  个据点 + 地银合资品牌）・经办产品・IPO 承销战略・地银联动模式** 不分拆到操作公司（证券公司）的独立页面就无法准确分析。在与中坚独立证券各 peer（[[securities-firms/iwai-cosmo-securities|岩井コスモ証券]]・[[securities-firms/okasan-sg|岡三]]・[[securities-firms/aizawa-sg|藍澤]]）比较时，本页面也成为 anchor。作为地银合作模式的先驱者，是独特的比较对象。
