# 丸三证券（丸三証券 / Marusan Securities）

## Wiki 路线

本条目位于 [[securities-firms/INDEX|securities-firms INDEX]] 之下。请对照 [[securities-firms/aizawa-sg|アイザワ証券グループ]]（同业 / 对比语境）、[[securities/financial-instruments-business-operators-japan-index|FSA 金融商品取引業者 index]]（监管牌照层）以及 [[financial-licenses/securities-license-stack|securities license stack]]（FIEA 牌照边界）一并阅读。

## TL;DR

丸三证券是 **1910 年创业的老牌中坚独立证券**（东证 PRIME **8613**），创业逾 110 年，专注于 **以面对面零售为主轴 + 咨询型营业** 的独立系证券公司。在 FSA 金融商品交易业者一览中列为「丸三証券株式会社」，关东财务局长（金商）第167号。**JSDA 正会员**。不隶属于巨型银行或电信系，以股票・债券・投资信托的面对面销售为核心，维持传统的中坚证券模式，竞争对手为 [[securities-firms/aizawa-sg|アイザワ証券]] / [[securities-firms/okasan-sg|岡三証券グループ]] / [[securities-firms/iwai-cosmo-hd|岩井コスモ HD]] / [[securities-firms/tokai-tokyo-fh|東海東京 FH]] 等中坚独立证券档次。

## 2。公司结构

**正式名称**：丸三証券株式会社
**创业**：1910 年（明治期在东京作为个人商店创业）^[extracted]
**上市**：战后股票市场重组后于东证上市，现为东证 PRIME（8613）^[extracted]
**总部**：东京都千代田区麹町
**业态**：金融商品交易业（第一种金融商品交易业者）— 中坚独立系证券

### 业态特征

丸三证券是未经控股公司化的**单一事业公司型**老牌中坚独立证券。集团内不持有银行・保险・加密资产业务，纯粹专注于证券业（特别是面对面零售的股票・债券・投资信托），其简洁的事业结构是一大特征。这与 [[securities-firms/aizawa-sg|アイザワ証券グループ]]（已控股公司化）或 [[securities-firms/okasan-sg|岡三証券グループ]]（已控股公司化）的组织形态不同。

### 作为老牌中坚独立证券的定位

丸三证券是 1910 年创业的老牌企业，拥有逾 110 年历史的中坚独立证券。与巨型银行系证券（[[securities-firms/smbc-nikko|SMBC 日興]] / [[securities-firms/mufg-mums|MUMSS]] / [[securities-firms/mizuho-securities|みずほ証券]]）、大型面对面（[[securities-firms/nomura-hd|野村]] / [[securities-firms/daiwa-sg|大和]]）、网络专营（[[securities-firms/sbi-securities|SBI 証券]] / [[securities-firms/rakuten-securities|楽天証券]] / [[securities-firms/monex-group|マネックス]]）处于不同的立场，维持 **面对面零售 × 中坚独立** 的利基。与中京・关西的 [[securities-firms/okasan-sg|岡三]] / [[securities-firms/iwai-cosmo-hd|岩井コスモ]] / [[securities-firms/tokai-tokyo-fh|東海東京]] 在地域基础（东京 vs 中京・关西）上有所不同。

### 竞争环境（[[securities/japan-online-brokerage-competition|japan-online-brokerage-competition]] 语境）

- 针对网络证券手续费免费化趋势，以面对面咨询型营业实现差异化
- 维持面向富裕层・年长层的「教育型营业」「面对面咨询」利基
- 与大型面对面证券的竞争因规模差距处于劣势，但在中坚独立证券档次维持独特品牌

## 5。监管与政策

- **主管**：金融厅（FSA）・关东财务局
- **业态**：第一种金融商品交易业（证券业）
- **监管**：金融商品交易法・日本证券业协会（[[financial-regulators/jsda|JSDA]]）自主监管
- **近期政策要点**：
  - 2025〜 强化以客户为本的业务运营（FD 原则）：面对面咨询型营业的正当化要素
