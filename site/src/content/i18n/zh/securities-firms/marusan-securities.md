# 丸三证券（Marusan Securities）

## Wiki 路径

本条目位于 [[JapanFG/INDEX|JapanFG index]] 之下。可与 [[securities-firms/aizawa-sg|アイザワ証券グループ]] 对读以理解同业和对比关系，与 [[securities/financial-instruments-business-operators-japan-index|FSA 金融商品取引業者 index]] 对读以理解监管牌照层，并与 [[financial-licenses/securities-license-stack|securities license stack]] 对读以理解金商法牌照边界。

## TL;DR

丸三证券是 **1910 年 创业的老牌中坚独立证券公司**（东证 PRIME **8613**），创业已超过 110  年，专注于 **面对面零售 + 咨询型销售**。在 FSA 金融商品交易业者名单中，公司以「丸三证券株式会社」列示，登记为关东财务局长（金商）第167号。公司是 **JSDA 正会员**。它不属于 megabank 或通信系集团，维持以股票、债券、投资信托面对面销售为核心的传统中坚证券模式；竞争对手包括 [[securities-firms/aizawa-sg|アイザワ証券]] / [[securities-firms/okasan-sg|岡三証券グループ]] / [[securities-firms/iwai-cosmo-hd|岩井コスモ HD]] / [[securities-firms/tokai-tokyo-fh|東海東京 FH]] 等中坚独立证券。

## 2. 公司结构

**正式名称**：丸三证券株式会社  
**创业**：1910 年，明治时期在东京以个人商店形式创业 ^[extracted]  
**上市**：战后股票市场重组后在东证上市，目前为东证 PRIME（8613）^[extracted]  
**总部**：东京都千代田区麹町  
**业态**：金融商品交易业（第一种金融商品交易业者）— 中坚独立证券

### 业态特征

丸三证券是未经过控股公司化的**单一经营公司型**老牌中坚独立证券。集团内不持有银行、保险或加密资产业务，业务结构简洁，纯粹专注于证券业，尤其是面对面零售股票、债券、投资信托销售。这与已经控股公司化的 [[securities-firms/aizawa-sg|アイザワ証券グループ]] 和 [[securities-firms/okasan-sg|岡三証券グループ]] 不同。

### 老牌中坚独立证券的位置

丸三证券创立于 1910 年，拥有超过 110  年历史。它与 megabank 系证券（[[securities-firms/smbc-nikko|SMBC 日興]] / [[securities-firms/mufg-mums|MUMSS]] / [[securities-firms/mizuho-securities|みずほ証券]]）、大型面对面证券（[[securities-firms/nomura-hd|野村]] / [[securities-firms/daiwa-sg|大和]]）、网络专营证券（[[securities-firms/sbi-securities|SBI 証券]] / [[securities-firms/rakuten-securities|楽天証券]] / [[securities-firms/monex-group|マネックス]]）不同，维持 **面对面零售 × 中坚独立** 的 niche。与中京 / 关西的 [[securities-firms/okasan-sg|岡三]] / [[securities-firms/iwai-cosmo-hd|岩井コスモ]] / [[securities-firms/tokai-tokyo-fh|東海東京]] 相比，其地域基础不同，是东京 vs 中京 / 关西。

### 竞争环境（[[securities/japan-online-brokerage-competition|japan-online-brokerage-competition]] 语境）

- 面对网络证券手续费免费化趋势，通过面对面咨询销售差异化。
- 维持面向富裕层、老年客户的「教育型销售」「面对面咨询」niche。
- 与大型面对面证券竞争时规模处于劣势，但在中坚独立证券类别中维持独自品牌。

## 5. 监管与政策

- **主管**：金融厅（FSA）与关东财务局。
- **业态**：第一种金融商品交易业（证券业）。
- **监管**：金融商品交易法、日本证券业协会（[[financial-regulators/jsda|JSDA]]）自律监管。
- **近期政策论点**：
  - 2025〜 顾客本位业务运营（FD 原则）强化：这是面对面咨询型销售正当性的关键要素。
