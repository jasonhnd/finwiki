# Mebuki 证券 (Mebuki Securities)

## Wiki route

This entry sits under the [[securities-firms/INDEX|securities-firms INDEX]] as the FG-subsidiary securities operating-company anchor for [[regional-banks/mebuki-fg|めぶき FG]]. ## TL;DR

株式会社 Mebuki 证券是 [[regional-banks/mebuki-fg|めぶき FG]]（东证 PRIME 7167）100% 子公司的地银系证券公司，以旧 **常阳证券**（[[regional-banks/joyo-bank|常陽銀行]] 系）为母体，在 [[regional-banks/mebuki-fg]] 设立后（2016-10）经集团整合·重组而来。为以茨城·栃木·首都圈为地盘的中坚企业 main bank 客户提供资产运用·公募股承销·私募债中介·事业承继 advisory。持有第一种金融商品交易业（证券业）牌照，加入 [[financial-regulators/jsda|日本証券業協会]]。^[extracted]

### 旧 常阳证券谱系

- **2007-11-30**：常阳证券株式会社 设立（茨城县水户市，[[regional-banks/joyo-bank|常陽銀行]] 系证券子公司）
- 以茨城县内 retail brokerage 和面向中坚企业的公募 / 私募债中介为主业
- 2010 年年代〜：作为地银系证券，在银证联动下扩大资产运用·私募债中介的角色
- **2017-04-03**：商号变更为 Mebuki 证券株式会社

### Mebuki FG 设立与集团整合

- **2016-10-01**：[[regional-banks/mebuki-fg|めぶき FG]] 设立（[[regional-banks/joyo-bank|常陽銀行]] + [[regional-banks/ashikaga-bank|足利銀行]] 共同股份转移）

## 5. 事业分部 map

| 分部 | 特征 |
|---|---|
| Retail brokerage | 面向茨城·栃木地盘个人投资者的股票·投信中介 |
| Wealth management | 基金包装·SMA 等资产运用产品 |
| 公募股承销 | 参与本地上市企业的公募增资（地银系承销团）|
| 私募债中介 | 与中坚企业 main bank 联动的私募债组建 |
| IPO 转介 | 北关东 IPO 案件转介（主承销为大型机构）|
| M&A 中介 | 事业承继·地区 M&A advisory |
| 银证联动 | 来自 [[regional-banks/joyo-bank]] / [[regional-banks/ashikaga-bank]] 的介绍交易 |

### FG 内定位

- **银证联动的核心**：[[regional-banks/mebuki-fg]] 客户（个人·中坚企业）的证券窗口
- **资产运用 cross-sell**：客户从银行存款 → 投信 / 基金包装的迁移
- **事业承继·M&A**：面向北关东中坚企业业主的 advisory 服务

### 竞争格局

- **大型证券**（[[securities-firms/nomura-hd|野村]] / [[securities-firms/daiwa-sg|大和]] / [[securities-firms/smbc-nikko]] / [[securities-firms/mizuho-securities]]）：在都内大型企业 / 富裕层方面大型机构占优，在本地客户方面以银证联动对抗
- **网络证券**（[[securities-firms/sbi-securities]] / [[securities-firms/rakuten-securities]] / [[securities-firms/matsui-sec]]）：在个人数字交易方面处于劣势，以面对面 advisory 差异化
- **地银系证券**：[[securities-firms/ffg-securities|FFG 証券]] / [[securities-firms/smbc-nikko|SMBC日興]] 等同型 model 的银证联动相手
- **县内竞争**：[[regional-banks/77-bank|七十七銀行]]（宫城）·[[regional-banks/chiba-bank|千葉銀行]] 系证券

### 优势与劣势

- 优势：[[regional-banks/joyo-bank]] / [[regional-banks/ashikaga-bank]] 客户基础 + 北关东中坚企业 main bank 网络

## 7. 监管·政策

- **主管**：金融厅（FSA）/ 关东财务局（第一种金融商品交易业者登记）
- **行业团体**：[[financial-regulators/jsda|日本証券業協会]] 正式会员
- **监管议题**：
  - 银证防火墙监管放宽（2022-）使银证联动范围扩大
  - 顾客本位的业务运营原则（FD principle）遵守
  - 高龄者投资劝诱规则强化
  - 投信销售手续费的透明化·低水平化压力
