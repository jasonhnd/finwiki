# 爱泽证券集团 (Aizawa Securities Group)

## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[securities-firms/okasan-sg|岡三証券グループ (Okasan Securities Group)]] for peer / contrast context and [[securities/INDEX|securities index]] for the broader system / regulatory boundary.

## TL;DR

中坚独立证券控股公司（东证 Standard **8708**）。由 1918-02  蓝泽弥八在东京日本桥兜町创立“藍澤證券店” → 1948  公司化 → 1989  在 JASDAQ 上市 → 2017-04  转为控股公司。其作为**“亚洲新兴国家投资专业证券公司”**确立了独特定位。在日本证券公司中，其对亚洲新兴市场的覆盖面最广之一（越南、印尼、泰国、柬埔寨、缅甸等）。以零售为中心，法人 IB 有限。竞争对手包括同属独立中坚梯队的 [[okasan-sg]] / [[tokai-tokyo-fh]]，以及网络系 [[matsui-sec]] / [[monex-group]]。^[extracted]

### 差异化轴：“亚洲新兴国家投资专业证券”

- 在日本证券公司中，**对亚洲新兴市场的覆盖面最广之一**（包括前沿市场）^[extracted]
- 拥有越南、印尼、泰国、柬埔寨、缅甸的当地网络
- 以包含面对面服务的方式，为个人投资者提供原本难以接触的前沿市场股票中介服务
- 通过聚焦大型券商（[[ndfg]] / 大和 / SMBC 日兴）不重点经营的地区，获得缝隙市场

### 日本国内定位

- 属于中坚独立证券梯队（与 [[okasan-sg]] / [[tokai-tokyo-fh]] / [[matsui-sec]] / [[monex-group]] / [[iwai-cosmo-hd]] 并列）
- 专注零售（个人客户），不主打法人 IB
- 采用面对面营业 + 部分线上交易的混合模式（并非纯网证券）

### 海外战略

- **以 5  个亚洲新兴国家为重点市场**：越南、印尼、泰国、柬埔寨、缅甸
- 美国被定位为补充市场（因与大型面对面券商及网络券商竞争，较为同质化）

## 4. 监管 / 政策

- **主管**：金融厅（FSA）与关东财务局
- **业态**：第一种金融商品交易业（证券业）
- **监管规则**：金融商品交易法 + 日本证券业协会（JSDA）自律规则
- **海外监管**：
  - 越南：State Securities Commission of Vietnam (SSC)
  - 印尼：OJK（Otoritas Jasa Keuangan）
  - 泰国：SEC Thailand
  - 柬埔寨 / 缅甸：当地证券监管机构
- **近期政策议题**：
  - 2025 起，客户本位业务运营（FD 原则）：教育型销售模式的持续性
