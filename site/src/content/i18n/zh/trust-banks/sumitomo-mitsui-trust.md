# 三井住友信托集团 (SuMi TRUST)

## Wiki route

This entry sits under [[trust-banks/INDEX|trust-banks INDEX]]. Read it against [[trust-banks/master-trust-bank|日本マスタートラスト信託銀行]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

从大型银行独立出来的唯一的信托专营持股公司，**3 大信托之一角**（三菱 UFJ 信托 / 三井住友信托 / 瑞穗信托）。2002-02 年设立中央三井信托 HD → 2011-04 年与住友信托银行整合 → 三井住友信托·控股成立 → **2024-10-01 更名为三井住友信托集团**。专注于养老金·不动产·证券代办·遗嘱信托。FY2024 净利润 **2,576 億円**（同比 3.3 倍，历史最高），总资产 **78.2 兆円**，**信托财产余额 263.3 兆円（业界首位）**。**注意：与 [[megabanks/smfg]] (三井住友 FG) 为不同法人** — 虽共用「三井住友」名称，但 SuMi TRUST 是旧三井家 + 旧住友家系的 **信托** 根源，SMFG 是旧太阳神户 + 旧住友银行系的 **银行** 根源，二者无资本关系。与 SBI HD 合资运营 [[regional-banks/ssnb]] (SBI 住信网络银行)。

### 合并历史·前身

- **旧 三井信托银行**：1924 设立（旧三井财阀系信托，日本最早的信托公司系列之一）
- **旧 住友信托银行**：1925 设立（旧住友财阀系信托）
- **旧 中央信托银行**：战后设立，后与三井信托合并
- **中央三井信托银行**：2000 三井信托银行 + 中央信托银行 合并
- **中央三井信托 HD**：2002-02 设立（中央三井信托 + 中央三井资产信托的持股公司）
- **三井住友信托 HD**：2011-04 中央三井信托 HD + 住友信托银行 整合

> **整合背景**：住友信托银行原本志在单独路线，但出于雷曼冲击后的行业重组压力与维持信托专营独立这一战略的一致，于 2009 年年宣布与中央三井信托 HD 的经营整合方针。2011-04 以股份交换方式整合，确立了作为信托专营 FG 的最大规模。^[likely]

### 信托专营独立路线

- **从大型银行独立**：3 大信托中，三菱 UFJ 信托在 [[megabanks/mufg]] 旗下，瑞穗信托在 [[megabanks/mizuho-fg]] 旗下。**唯有 SuMi TRUST 是从大型银行独立的持股公司，信托专营**。这是其身份认同也是差异化轴。
- **「三井住友」名称的混淆风险**：与 [[megabanks/smfg]]（三井住友 FG = SMBC 系）名称相同，但为不同法人·不同资本。SMFG 是旧太阳神户银行 + 旧住友银行系的银行系列，SuMi TRUST 是旧三井信托 + 旧住友信托的信托系列。

### 向富裕层·法人运用转移

- **个人富裕层**：以遗嘱信托·资产承继·继承对策，活用旧三井家·旧住友家以来的客户基础
- **法人**：以养老金资产受托（DB / DC）·不动产信托·证券代办拥有大型企业客户基础
- **机构投资者**：以日兴资产 + 三井住友信托 AM 形成 2 品牌体制的投信运用

### 合作·收购战略

- **与 SBI HD 的 JV** ([[regional-banks/ssnb]] / SBI 住信网络银行)：50% 出资，与 SBI HD 合资，进入 BaaS / 网络银行领域的主要渠道
- **与松下的 JV**（三井住友信托·松下金融）：租赁·金融
- **取得日兴资产管理**（2009 从花旗分阶段）：强化投信运用领域

### 中期经营计划「Trust 2030」（2024）

- 以信托功能为轴，扩大养老金·不动产·资产承继·可持续性相关受托
- BaaS 领域经由 SBI 住信网络银行间接展开 ^[likely]

## 4. 监管·政策

- **主管**：金融厅（FSA）
- **持股公司监管**：银行法 第 52 条之 17（也准用于信托银行的持股公司）
- **信托业法**：信托银行业务全般
- **G-SIB**：与 MUFG 不同，未被指定为 G-SIB（规模·全球关联性相对较小）^[likely]
- **近期政策议题**：
  - 2024〜 日银政策利率正常化 → 信托报酬·运用收益率环境的变化
  - 2025〜 BaaS / 网络银行领域中 住信 SBI 网络银行（[[regional-banks/ssnb]]）的动向
  - 2025〜2026 资产承继·继承相关服务的监管环境变化
