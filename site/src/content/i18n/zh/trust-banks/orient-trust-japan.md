# 北方信托日本 (Northern Trust Japan)

> [!note] 命名 note
> 本页最初以 "orient-trust-japan" 这个 slot 被创建请求；但日本未能确认存在名为“Orient Trust”的独立法人（“Orient Corporation”[[card-issuers/orico]] 是信贩公司，属于另一类别；“东洋信托银行”是旧 UFJ 系，现已并入 [[trust-banks/mitsubishi-ufj-trust-bank|三菱 UFJ 信託銀行]]），因此作为外资系信托子公司的替代，本 slot 充当 **Northern Trust Japan** 页面。

## Wiki 路径

本条目位于 [[trust-banks/INDEX|trust-banks INDEX]] 之下，是**外资系信托 / asset-servicing 子公司**锚点。可与 [[foreign-financial-institutions/state-street-japan|State Street Japan]] / [[foreign-financial-institutions/bny-mellon-japan|BNY Mellon Japan]] / [[foreign-financial-institutions/jpmorgan-japan|JPMorgan Japan]] 作为外资 custodian / asset-servicing 同业集群阅读，与 [[trust-banks/master-trust-bank|マスタートラスト信託銀行]] / [[trust-banks/mitsubishi-ufj-trust-bank|三菱 UFJ 信託銀行]] 作为国内信托同业集群阅读，与 [[trust-banks/trust-companies-japan-index|the Japan trust companies registry index]] 作为登记语境阅读，并与 [[banking/japan-trust-bank-custody-map|the Japan trust-bank custody map]] 一并理解系统层定位。

## 3. 业务线

| 分部 | 在日承担方 | 内容 |
|---|---|---|
| **全球 custody** | 在日支店 + Global Services | 海外股票、债券和另类资产的保管、结算、公司行动处理 |
| **国内信托（custody 周边）** | Global Services（信托业牌照） | 面向 GPIF、公的年金和运用公司的后台功能 ^[likely] |
| Fund administration | Global Services | 投信净资产价值计算、基准价计算、运用报告书制作支持 |
| Transfer agent | Global Services | 机构投资者账户管理、受益权转移管理 |
| 证券中介 / trading | Securities Japan | 面向机构投资者的外国证券交易中介 |
| 富裕层私人银行 | 日本未全面展开 | 美国本土主力业务，但日本专注 wholesale |

### 主要客户类型

- **GPIF（年金积立金管理运用独立行政法人）**：公的年金海外资产 custody 受托机构之一 ^[likely]
- **大型寿险 / 损保**：[[life-insurers/dai-ichi-life|第一生命]] / [[life-insurers/nippon-life|日本生命]] / [[non-life-insurers/tokio-marine|東京海上]] 等海外运用资产 custody
- **大型运用公司**：[[asset-managers/nomura-am|野村アセットマネジメント]] / [[asset-managers/daiwa-am|大和アセットマネジメント]] / [[asset-managers/mitsubishi-ufj-am|三菱 UFJ アセットマネジメント]] 等 fund admin / custody
- **大型企业年金基金**：海外资产部分的 custody

### 作为美国 3 大 custodian 末尾的相对位置

- [[foreign-financial-institutions/bny-mellon-japan|BNY Mellon]]：全球 No.1，规模最大。
- [[foreign-financial-institutions/state-street-japan|State Street]]：全球 No.2，拥有 SSGA / SPDR ETF。
- **Northern Trust**：全球 No.3，强项是美国本土富裕层私人银行；机构投资者服务聚焦经筛选的大型客户。

### 6.1 . 作为美国 3 大 custodian 末尾的差异化

- 规模上小于 [[foreign-financial-institutions/bny-mellon-japan|BNY Mellon]] / [[foreign-financial-institutions/state-street-japan|State Street]]。
- 差异化轴：**与富裕层私人银行功能的整合**（美国本土主轴）+ **对经筛选大型机构投资者的深度服务**。
- 日本不全面开展富裕层私人银行，而是聚焦 GPIF、大型寿险 / 损保、大型运用公司等机构投资者服务。^[likely]

### 6.2 . 作为 GPIF / 公的年金受托机构的位置

- GPIF 的海外资产 custody 分散委托给多个外资 custodian（[[foreign-financial-institutions/state-street-japan|State Street]] / [[foreign-financial-institutions/bny-mellon-japan|BNY Mellon]] / Northern Trust 等）。
- 公的年金和企业年金海外运用比率上升 → 对外资 custodian 整体构成顺风。

### 6.3 . 与邦银信托的分工

- [[trust-banks/master-trust-bank|マスタートラスト信託]]、[[trust-banks/mitsubishi-ufj-trust-bank|三菱 UFJ 信託銀行]]、[[trust-banks/sumitomo-mitsui-trust|住信トラスト]] 等**邦银系信托**在国内资产 custody 和运用受托上占压倒性份额。
- Northern Trust 等外资 custodian 则在**海外资产专业受托**方面分工。

### 6.4 . 富裕层私人银行（美国本土）

- Northern Trust 的**创立以来核心业务**是面向芝加哥富裕层的私人银行（自 1889 开业以来）。
- 在面向美国超高净值客户的 wealth management、信托和遗产管理方面拥有深厚客户关系。
- 日本不直接与 UBS [[foreign-financial-institutions/ubs-japan]] / [[trust-banks/lgt-wealth-management-trust|LGT]] / [[trust-banks/smbc-trust-bank|SMBC 信託（PRESTIA）]] 等私人银行势力竞争，而是集中于机构投资者渠道。

## 7. 监管 / 监督

| 项目 | 内容 |
|---|---|
| 母公司监管 | 美国 FRB（联邦储备制度理事会）+ OCC（货币监理署）监管 |
| 日本据点监管 | 金融厅（信托业 + 银行业在日支店 + FIEA 业者各牌照） |
| Global Services Japan | 信托业牌照（金融厅公开金监编号） |
| 在日支店 | 银行业牌照（在日外国银行支店） |
| Securities Japan | 第一种金融商品交易业（FIEA） |
| 行业团体 | 信托协会、全国银行协会、日本证券业协会 |
