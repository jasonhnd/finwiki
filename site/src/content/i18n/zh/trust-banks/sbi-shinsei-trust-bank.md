# SBI新生信托银行 (SBI Shinsei Trust Bank)

## Wiki 路径

SBI Shinsei Trust Bank 是 [[JapanFG/INDEX|JapanFG]] 之下的持牌信托银行运营公司页面。可与 [[regional-banks/sbi-shinsei-bank|SBI Shinsei Bank]]、[[megabanks/sbi-hd|SBI Holdings]]、[[trust-banks/mitsubishi-ufj-trust-bank|Mitsubishi UFJ Trust Bank]]、[[payment-firms/progmat|Progmat]]、[[fintech/japan-stablecoin-regulatory-landscape|Japan stablecoin regulation]]、[[banking/japan-trust-bank-custody-map|Japan trust-bank custody map]] 一并阅读。

当问题涉及信托银行法人、信托方案、证券化、证券代币，或由信托银行支撑的数字金融路径时，使用本页。普通银行战略和公的资金偿还历史请使用 [[regional-banks/sbi-shinsei-bank|SBI Shinsei Bank]]。

## 概览

SBI Shinsei Trust Bank 是 SBI Shinsei / SBI Holdings 集团内的信托银行部门。公开公司概要显示，现名称为 **SBI新生信託銀行株式会社**，自 2026-04-01 起使用；股权由 [[regional-banks/sbi-shinsei-bank|SBI Shinsei Bank]] 持有 51%、[[megabanks/sbi-hd|SBI Holdings]] 持有 49%。

本页重要之处在于，该实体不再只是安静的集团信托子公司。公开发布和公司网站将其与不动产信托、应收款证券化、证券代币结构和受监管数字金融举措连接起来，使其处在传统信托银行功能与 [[fintech/jp-stablecoin-progmat|stablecoin / tokenized-money]] 研究面之间。

分析重点在于信托 wrapper。信托银行可作为资产的法律所有权和管理层，之后这些资产可能以证券、信托受益权或数字工具形式呈现在投资者面前。

## 公开语境

对 [[megabanks/sbi-hd|SBI Holdings]] 而言，信托银行功能为集团在证券、银行、crypto / EPI 分销和信托账户资产管理之间提供受监管桥梁。这有三点重要性。

第一，SBI 已通过 [[securities-firms/sbi-securities|SBI Securities]] 和 [[exchanges/jp-exchange-sbi-vc-trade|SBI VC Trade]] 拥有分销和交易所表面。只要公法允许，信托银行功能可以位于发行、托管、准备金或资产持有结构背后。

第二，[[regional-banks/sbi-shinsei-bank|SBI Shinsei Bank]] 提供银行资产负债表和企业金融路径，而本信托银行可以构造破产隔离、资产分离或行政管理机制。

第三，本信托银行与 [[trust-banks/mitsubishi-ufj-trust-bank|Mitsubishi UFJ Trust Bank]]、[[payment-firms/progmat|Progmat]] 形成同业 / 对比关系，后两者也连接信托、数字证券和稳定币基础设施。

## 监管 / 研究备注

- 当前牌照状态重要时，检查 FSA 信托业务金融机构列表。
- 检查公司公告中的当前名称和股东结构，因为 FSA 表格可能滞后于公司名称变更。
- 不要因为存在信托银行牌照就推断存在消费者银行产品。
- 分析证券代币时，区分发行人、受托人、平台、券商、托管人和结算轨道。
- 分析稳定币 / EPI 时，应连接本页与 [[fintech/japan-stablecoin-regulatory-landscape|Japan stablecoin regulation]]、[[exchanges/jp-exchange-sbi-vc-trade|SBI VC Trade]]，而不是把所有数字金融活动视为一个牌照。
