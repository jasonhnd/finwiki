# パソナ知财信託 (Pasona IP Trust)

## Wiki route

Pasona IP Trust 是 [[JapanFG/INDEX|JapanFG]] 下的独立信托公司路径，并由 [[trust-banks/trust-companies-japan-index|the Japan trust companies registry index]] 提升出来。可与 [[finance/japan-acquisition-finance|Japan acquisition finance]]、[[business/INDEX|business cases]]、[[banking/japan-trust-bank-custody-map|Japan trust-bank custody map]] 一起阅读。

本页用于知识产权信托、专利 / 工业产权管理、IP 变现以及非金融资产信托相关问题。

## Overview

Pasona IP Trust 将自身定位为知识产权信托服务，支持工业产权的申请、权利维持、使用和管理。其公开网站说明该公司是知识产权信托服务的专业机构。

本页重要之处在于，它把 FinWiki 的信托公司覆盖范围扩展到货币、证券、不动产和数字证券之外。IP 信托是无形资产管理和潜在 IP 金融分析的独立路径。

## Public Context

Pasona IP Trust 很有用，因为无形资产通常位于普通资产负债表金融地图之外。信托结构可以分离管理、法律所有权和经济使用，这对企业重组、许可和 IP 管理很重要。

除非 Pasona IP Trust 或交易方公开披露，否则不要推断其存在当前证券化、担保或融资产品。
