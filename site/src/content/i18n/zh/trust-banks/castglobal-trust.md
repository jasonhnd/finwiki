# CastGlobal Trust / キャストグローバル信託

Wiki 路径：[[trust-banks/INDEX]] -> [[trust-banks/trust-companies-japan-index]] -> [[trust-banks/castglobal-trust]]

## 概览

CastGlobal Trust 是一家与房地产、托管（escrow）及跨境投资支持相关的管理型信托公司。在 FinWiki 中，它属于专业服务型信托路径，适合与 [[trust-banks/yamada-escrow-trust]] 和 [[trust-banks/kyodo-trust]] 并列参考。

## 实体边界

本页涵盖キャストグローバル信託株式会社，而非整个 CastGlobal 专业服务网络。关于法律、税务或国际合作伙伴服务的表述应始终以公开的公司资料为依据。

## 业务角色图

| 路径 | 解读 |
|---|---|
| 托管（Escrow） | 通过受管理的信托账户提供支付/结算支持。 |
| 房地产投资 | 为日本房地产交易结构提供信托支持。 |
| 跨境语境 | 将信托业务与国际投资者支持相连接。 |
| 登记管控 | [[trust-banks/trust-companies-japan-index]] 中的一行。 |

## 公开语境

CastGlobal Trust 之所以重要，是因为它展示了信托公司如何介于专业服务、托管与房地产投资之间。该路径也有助于将类托管的信托服务与 [[payments/account-to-account-payment-japan|payment rails]] 以及 PSP 结算风险页面区分开来。
