# 瑞穗信托银行 事业详情 (Mizuho Trust & Banking Operating Profile)

## TL;DR

[[trust-banks/mizuho-trust-bank|みずほ信託銀行株式会社]] 是 **[[megabanks/mizuho-fg|みずほフィナンシャルグループ]] 旗下的信托银行 operating company**。在 [[megabanks/mizuho-bank|みずほ銀行]] 承担商业银行核心的同时，瑞穗信托银行承担 **信托 + 养老金 + 不动产 + 证券代理 + 受托财产** 一侧，是 FG 内事业分工的一翼。源流为 1925-05 设立的 **「中央信托」** 以及 「**安田信托」**（后来的富士信托银行系统）— 经过 2000-2003 旧 3 行整合（第一劝业银行·富士银行·日本兴业银行）所伴随的信托功能整合而成为现法人形态。本条目处理事业分部·养老金信托·不动产信托·证券代理·国际业务·FG 内 banking-trust 联动的 **operating-level depth**。在 3 大信托银行（瑞穗信托 / [[trust-banks/mitsubishi-ufj-trust-bank|三菱UFJ信託]] / [[trust-banks/sumitomo-mitsui-trust|三井住友信託]]）中属于中位规模。

## 1. 母集团关系（瑞穗 FG）

- **持股公司**：[[megabanks/mizuho-fg|みずほフィナンシャルグループ]]（8411 / 东证 PRIME）的 100% 子公司
- **FG 内事业分工**：
  - [[megabanks/mizuho-bank|みずほ銀行]] — 商业银行·国内法人·零售
  - **瑞穗信托银行** — 信托功能·养老金·不动产·证券代理·受托财产
  - [[securities-firms/mizuho-securities|みずほ証券]] — 投资银行·证券业
- **One Mizuho 战略**：3 社（银行 + 信托 + 证券）的 cross-sell·客户介绍强化的 FG 横向模式
- **董事会**：FG 派遣高管 + 信托专门高管

## 2. 法人沿革

- **源流 1 （1925-05）**：中央信托株式会社 设立
- **源流 2**：安田信托银行（旧 安田财阀系信托功能，后来的富士信托银行）
- **整合**：伴随 2000-2003 瑞穗 FG 设立的信托功能的集约·整合
- **现体制**：瑞穗信托银行株式会社（瑞穗 FG 100% 子公司）

## 3. 牌照·监管边界

- **普通银行**：基于银行法的普通银行牌照（关东财务局长牌照）
- **信托业务**：基于兼营法的信托业务认可（信托银行特有）
- **行业团体加入**：
  - 全国银行协会
  - **信托协会**（信托银行特有的行业自主监管团体）
- **存款保险机构**：加入

## 4. 事业分部详情

### 4.1 养老金信托·养老金运用

- **对象**：企业年金（确定给付年金 / 确定缴费年金）·公共养老金
- **业务**：养老金资产受托·运用·养老金支付事务
- **定位**：3 大信托的主力业务 — 瑞穗信托在 **企业年金领域拥有长期客户基础**
- **监管**：确定给付企业年金法·确定缴费年金法·厚生年金保险法

### 4.2 不动产信托·不动产中介

- **不动产信托受益权**：不动产流动化方案·REIT 相关的信托受益权组建
- **不动产中介**：面向法人 / 富裕层的不动产买卖中介·咨询
- **不动产基金**：私募不动产基金组建
- **竞争对手**：[[trust-banks/mitsubishi-ufj-trust-bank|MUTB]] / [[trust-banks/sumitomo-mitsui-trust|三井住友信託]] / 大型不动产公司（[[trust-banks/INDEX|trust-banks INDEX]]·三菱地所等）

### 4.3 证券代理（股东名簿管理）

- **业务**：上市公司的股东名簿管理·股东大会事务·分红支付事务
- **客户**：上市公司（以旧 瑞穗 FG 系银行 main bank 关系方为中心）
- **市场份额**：3 大信托 + [[trust-banks/INDEX|trust-banks INDEX]] 等几乎占据日本市场

### 4.4 受托财产（资产运用受托·托管）

- **投资信托受托**：投信的信托财产受托·计理
- **特定金钱信托**：面向法人 / 个人富裕层的运用信托
- **托管服务**：面向机构投资者的托管业务

### 4.5 个人信托·继承相关

- **遗嘱信托**：遗嘱书制作支援·遗嘱执行
- **继承相关业务**：遗产整理业务·继承税申报联动
- **教育资金赠与信托·结婚育儿支援信托**：利用赠与税免税额度的家族间资产转移信托
- **不动产管理处分信托**：应对继承 / 高龄化的家族信托

### 4.6 国际业务

- **亚洲·美洲**：通过驻在员事务所·现地法人对日资企业的支援
- **海外不动产信托**：日资投资者的海外不动产投资受托

## 6. 战略·近期动向

- **One Mizuho 战略**：与 [[megabanks/mizuho-bank|みずほ銀行]] + [[securities-firms/mizuho-securities|みずほ証券]] 的 3 社联动带来的客户介绍·cross-sell（特别是法人 IPO / 事业承继·富裕层资产形成·养老金）
- **不动产事业强化**：法人 CRE（企业不动产）战略 + REIT 发起人支持
- **数字继承 / 家族信托**：应对高龄化社会的信托产品开发
- **资产管理（Wealth Management）**：面向富裕层的整合资产管理服务强化
- **ESG 信托 / 影响力投资**：在养老金运用中植入可持续性要素
- **DX 投资**：信托事务的数字化·电子合同·区块链应用探讨

## 8. 风险·监管议题

- **养老金受托费压缩**：低利率环境下运用收益率下降所伴随的养老金受托手续费压迫
- **不动产行情依赖**：国内不动产价格·REIT 市场波动影响不动产信托 / 中介收益
- **FG 内母子上市治理不适用**（因瑞穗信托是 100% 子公司）。但 FG 整体的 One Mizuho 战略的实效性是持续议题
- **TCFD / 可持续性**：养老金运用·不动产投资中的气候相关风险披露
- **DX 投资的回收**：信托事务数字化大规模投资的盈利化
- **个人信托的说明责任**：遗嘱信托·家族信托等面向高龄客户产品的说明义务·适合性原则对应
- **反洗钱（AML）**：对利用信托方案的资金洗白风险的持续监视
