# Kyodo Trust / 共同信托

Wiki 路径：[[trust-banks/INDEX]] -> [[trust-banks/trust-companies-japan-index]] -> [[trust-banks/kyodo-trust]]

## 概览

Kyodo Trust 是一家管理型信托公司，在公开资料中与不动产、权利调整、保全和信托功能使用相关。在 FinWiki 中，它是 [[trust-banks/castglobal-trust]] 和 [[trust-banks/kiriu-real-estate-trust]] 旁边的小型 specialist trust 路径。

## 实体边界

本页覆盖 共同信託株式会社 这一信托公司条目。除非公开资料明确将司法书士 / 咨询事务所语境连接到公司画像，否则不应吸收整个法律事务所或咨询办公室语境。

## 业务角色地图

| 路径 | 解读 |
|---|---|
| 权利调整 | 面向复杂财产 / 权利协调的信托公司路径。 |
| 不动产 | 不动产信托与保全语境。 |
| 类 escrow 逻辑 | 与结算和资产隔离逻辑相关，但不是 PSP。 |
| 登记控制 | [[trust-banks/trust-companies-japan-index]] 中的条目。 |

## 公开语境

Kyodo Trust 重要，是因为它展示了专业服务和小公司信托路径。它也适合作为上市集团信托公司和住宅集团信托公司的对照。

## 研究清单

1. 优先使用官方 PDF 或公开机关资料，而非二级数据库。
2. 将法律事务所语境保持在狭窄且有来源支持的范围内。
3. 只有公开资料点名 Kyodo Trust 时，才交叉链接权利调整案例。
