# JValue Trust / JValue Trust Co., Ltd.

Wiki route: [[trust-banks/INDEX]] -> [[trust-banks/trust-companies-japan-index]] -> [[trust-banks/jvalue-trust]]

## Overview

JValue Trust 是列入 FSA 信托公司登记名单的管理型信托公司。在 FinWiki 地图中，它是一条有来源支撑的小型专业信托公司路线，可用于同 [[trust-banks/capital-trust]]、[[trust-banks/kiriu-real-estate-trust]] 等房地产和结构化信托经营者进行比较。

## Entity Boundary

本页覆盖作为登记资料支撑的信托公司经营者 JValue Trust Co., Ltd.。在公开披露提供更多产品类别、集团所有权或交易案例前，本页应保持简洁。

## Public Context

JValue Trust 作为独立路线保留，是因为登记名单对应官方公司页面，并且有助于补全信托公司长尾，而不把不同法律实体合并进通用信托银行页面。
