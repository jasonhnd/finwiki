## 2. 业务分部地图

| Segment | 内容 |
|---|---|
| **CMS (cash management services)** | 丸红集团各公司的集中结算和资金池 |
| **一揽子 factoring** | 集团各公司的应收债权收购 / 支付代理 |
| **集团金融** | 向集团内公司提供 short-term / mid-term loan |
| **外汇 / 利率风险 hedge 中介** | 面向集团各公司的 derivatives intermediation |
| **海外汇款服务** | 集团各公司的海外汇款和回收 |
| **贸易金融辅助** | 与丸红本体电力、plant、食品、农药、化学品交易相关的授信辅助 |

## 4. 监管与政策

- **主管**：金融厅（贷金业 / 外汇业务相关）
- **监管论点**：
  - 集团内金融交易的**转让定价税制**（尤其是海外 IPP / plant 项目）
  - 集团 factoring 与**分期销售法 / 贷金业法**的边界
  - 合并会计上的**内部交易抵消**
- **近期政策论点**:
  - 2024起 [[financial-regulators/boj-monetary-policy|日銀政策金利正常化]] -> 重新评估集团内利息归属
  - 2024起扩大与瑞穗租赁的资本关系 -> 强化外向型 asset finance 功能
