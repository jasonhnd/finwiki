# 三菱商事 Financial Services（Mitsubishi Corporation Financial Services）

| 层级 | 主体 | 角色 |

## 2. 业务板块地图

| 板块 | 内容 |
|---|---|
| **CMS（现金管理服务）** | 三菱商事集团各公司的集中结算、资金池、集团内转账 |
| **一揽子 factoring** | 三菱商事集团各公司的应收账款购买、支付代理 |
| **集团 finance** | 向集团内公司提供 short-term / mid-term 贷款 |
| **汇率 / 利率风险 hedge 中介** | 面向集团各公司的衍生品交易中介（代行总部 treasury） |
| **海外汇款服务** | 集团各公司的海外汇款和回收 |
| **信用补完 / 保证** | 为集团各公司的交易对手提供信用补完 |

### 商社系 group-treasury 的典型形态

- **提高资金效率**：将集团内富余资金调配给资金不足公司（intra-group netting）
- **降低交易成本**：集中银行手续费 / 汇率手续费 / 汇款成本
- **集中风险管理**：由总部 treasury 集中管理汇率、利率风险
- **集中信用**：通过单一信用额度（三菱商事本体）确保与银行的谈判能力

通过将这一功能独立子公司化，集团确保内部金融服务的专业性、治理、会计分离以及部分监管应对。

## 4. 监管 / 政策

- **主管**：金融厅（部分贷款业 / 外汇业务相关登记范围）
- **监管论点**：
  - 集团内金融交易的 **转让定价税制**（与海外子公司的交易价格）
  - 合并会计中 **内部交易抵销** 的范围
- **近期政策论点**:
  - 2024〜 [[financial-regulators/boj-monetary-policy|日銀政策金利正常化]] → 集团内利息赋予 / 转让定价再评估
  - 2025〜 海外汇款 / 跨境资金移动的 AML/CFT 强化
