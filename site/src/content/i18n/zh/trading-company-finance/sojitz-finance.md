## 2. 业务板块地图

| Segment | 内容 |
|---|---|
| **CMS（现金管理服务）** | 双日集团各公司的结算集中和资金池 |
| **一揽子保理** | 集团各公司的应收账款收购 / 支付代理 |
| **集团金融** | 向集团内公司提供 short-term / mid-term loans |
| **汇率和利率风险套保中介** | 面向集团各公司的衍生品中介 |
| **海外汇款服务** | 集团各公司的海外汇款和回收 |
| **贸易金融辅助** | 与汽车 / 航空产业 / 基础设施和医疗健康 / 金属、资源、回收 / 化学品 / 食品和农业业务 / 生活产业 / 零售交易相关的授信辅助 |

### 双日业务结构和金融功能

双日金融支撑这些板块中的 supplier finance、海外货币套保、跨境汇款等。由于双日是中坚商社，金融功能规模小于 5  大商社，但功能组合相似。

## 4. 监管和政策

- **主管**：金融厅（贷金业 / 外汇业务相关）
- **监管论点**：
  - 集团内金融交易的 **转让定价税制**
  - 集团保理与 **分期付款销售法 / 贷金业法** 的边界
  - 合并会计中的 **内部交易抵消**
- **近期政策论点**：
  - 2024 起 [[financial-regulators/boj-monetary-policy|日銀政策金利正常化]] → 重新评估集团内计息安排
