# 楽天集团金融业务 (Rakuten FG)

## Wiki route

本条目位于 [[JapanFG/INDEX|JapanFG index]] 之下。可与 [[JapanFG/gmo-payment-gateway|GMO ペイメントゲートウェイ]] 对照阅读同业 / 对比语境，并与 [[fintech/japan-stablecoin-regulatory-landscape|日本 Stablecoin 法制度の三層構造（JPYC・USDC・Project Pax）]] 一起理解更广泛的制度 / 监管边界。

## TL;DR

楽天集团株式会社 (Rakuten Group, Inc., 东证 PRIME 4755) 的金融科技板块。其 **EC 轴独自模型** 以楽天市场 + 楽天积分为 core，将银行 / 证券 / 卡 / 保险 / 支付作为“楽天经济圈”捆绑，通过交叉销售展开。不同于巨型银行系 ([[mufg]] / [[smfg]] / [[mizuho-fg]]) 和通信系 FG ([[ndfg]] / au-FH / [[paypay-fg]])，它的业务组装方式是 **从 commerce 起点走向金融**。2023-04 楽天银行 IPO，2024 楽天证券接受みずほ证券 49% 出资。**2024 期 FinTech 板块 Non-GAAP 营业利润 1,534 億円 (+37.9%)，集团 IFRS 营业利润转正 (530 億円)**。楽天 Mobile 亏损 (Non-GAAP 营业损失 2,163 億円) 仍高于 FinTech 利润，但处于改善趋势。创始人三木谷浩史 (1965-) 自 1997 创立以来一直担任代表者。

### “楽天经济圈”模型

- **楽天积分 = OS**: 在 EC 购物、通信、卡、银行、证券、保险、旅行中全部可获得 / 使用积分
- **SPU (Super Point Up Program)**: 通过“楽天 Mobile 合约 / 楽天 Card 使用 / 楽天银行工资收取 / 楽天证券投信定投”等，分阶段提高楽天市场购物积分倍率 -> 金融商品交叉销售的强力驱动
- **统一会员 ID**: 以 1 楽天 ID 覆盖全部服务 -> CRM 数据整合优势

### EC 轴独特性（vs 通信系 / 巨型银行系）

| 竞争类别 | 轴 | 代表 |
|---|---|---|
| **EC 轴** | **Commerce -> finance** | **Rakuten FG（本条目）** |
| 通信轴 | 通信线路 -> 金融 | [[ndfg]] (docomo) / au-FH (KDDI) / [[paypay-fg]] (Yahoo/Z) |
| 银行轴 | 商业银行 -> 数字化 | [[mufg]] / [[smfg]] / [[mizuho-fg]] |
| 专业网络轴 | 网络证券 -> 扩张 | [[sbi-hd]] |

### Mobile 亏损 x 金融补贴结构

- 2019-10起，楽天 Mobile 进入 MNO -> 基站投资使 **数年级大额亏损** 常态化
- 2024 期: Mobile Non-GAAP 营业损失 2,163 億円 vs FinTech Non-GAAP 营业利润 1,534 億円 -> **FinTech 仍无法直接抵消 Mobile 亏损**，但改善幅度 (Mobile +850 億・FinTech +37.9%) 推动 2024 期 IFRS 营业利润转正 (530 億円)
- FinTech 板块（尤其是楽天银行 / 楽天 Card）的稳定收益支撑集团合并 EBITDA
- 2022-2024 期间: 楽天证券 IPO 计划 -> 接受みずほ证券 49% 出资，重组为 **筹资 + IPO 撤回**
- 2023-04 实施楽天银行 IPO -> 集团流动性 + 上市账面价值显性化

### 与みずほ FG 的资本业务合作

- 2023-10 みずほ证券 19.99% 出资 -> 2024 分阶段提高至 **49%**
- 楽天证券 IPO 实质上被撤回
- 对みずほ侧而言，是获得零售和网络客户基础；对楽天侧而言，是集团融资 + 深化与巨型银行系证券的联动

## 4. 监管 / 政策

- **主管**: 金融厅（FSA）
- **按业态监管**:
  - 楽天银行: 银行法（网络银行）
  - 楽天证券: 金融商品交易法（第一种金融商品交易业）
  - 楽天 Card: 割赋销售法 + 贷金业法
  - 楽天生命 / 楽天损保: 保险业法（寿险・财险各自）
  - 楽天 Wallet: 资金结算法（暗号资产交换业者登记 = JVCEA 会员）
  - 楽天 Pay / 楽天 Edy: 资金结算法（预付式 / 资金移动业）
- **控股公司监管**: 楽天集团本体是事业控股公司（不是纯粹控股公司）。各业态子公司分别受各自业法监管。
- **近期政策论点**:
  - 楽天 Mobile 频率分配 / 设备投资 -> 总务省管辖，但会间接影响金融板块
  - 2024起，新 NISA 制度 -> 楽天证券账户增长
  - 稳定币 / 暗号资产监管修订 -> 影响楽天 Wallet
