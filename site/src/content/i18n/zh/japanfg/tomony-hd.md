# Tomony控股

## Wiki route

本条目位于 [[JapanFG/INDEX|JapanFG index]] 之下。可与 [[JapanFG/fukuoka-fg|ふくおか FG (FFG)]] 对照阅读以理解同业 / 对比语境，也可与 [[banking/INDEX|banking index]] 一起阅读以把握更大的系统和监管边界。

## TL;DR

Tomony Holdings 是在四国（德岛、香川）+ 大阪（大正银行系）跨区域经营的地方银行 FG。它通过 **德岛银行 + 香川银行经营整合** 于 2010-04-01 成立，随后将 2017  大正银行（旧 Hokuto Shoji 系、大阪）救济式子公司化，并在 2020-01  合并德岛与大正，诞生 **[[tokushima-taisho-bank|徳島大正銀行]]**。现在的主要子公司是 [[tokushima-taisho-bank|徳島大正銀行]] + [[kagawa-bank|香川銀行]] 的 2 行 体制。^[extracted]

### 合并历史和前身 ^[extracted]

- 旧 **德岛银行**：1944  设立（由旧德岛合同银行等战时整合而来）
- 旧 **香川无尽**：1943  设立 → 1951  转换为相互银行 → **1989  转换为普通银行并成为香川银行**
- 旧 **大正银行**：1922  设立（大阪地盘）→ 旧 Hokuto Shoji 系 → 经营困难
- **Tomony HD**：2010-04-01 通过德岛银行 + 香川银行共同股份转移成立
- **大正银行子公司化**：2017  救济型 M&A（获得大阪地盘）
- **德岛大正银行**：2020-01  德岛银行 + 大正银行合并而成立

### 交叉地盘战略

- **四国（德岛 + 香川）** + **大阪（旧大正银行门店网）**，形成其他四国地方银行少见的广域展开
- 阿波舞经济（德岛）+ 乌冬经济（香川）+ 关西经济圈接入（大正银行系大阪）
- 四国八十八所巡礼旅游金融 + 濑户内造船、四国造纸制造业金融
- 农业金融（蜜柑、乌冬粉、甘薯等地方农产品）

### 提携和收购战略

- **德岛 + 香川经营整合**（2010-04-01）：相邻四国 2  县地银的对等合并型 HD 化（类似案例：山口 FG、西日本 FH 等）
- **大正银行救济子公司化**（2017）：获得大阪地盘的战略 M&A，吸收经营困难的大正银行
- **德岛大正银行合并**（2020-01）：在 HD 内由 2 行 转向 1 行 体制，削减重复成本并整合大阪 + 德岛门店网

### 人口减少地区的生存战略

- 四国 4  县是日本人口减少率较高地区之一 → 通过大正银行系接入关西经济圈来补充
- 处于 megabanks 与 shinkin banks 之间的中间 niche（地方中小企业 main bank）
- 专注旅游、农业、制造业等地方产业金融

### 地银重组语境中的位置

- 四国地银: Tomony HD ([[tokushima-taisho-bank|徳島大正]] + [[kagawa-bank|香川]]) / [[iyogin-hd]] ([[iyo-bank|伊予銀行]]) / [[awa-bank|阿波銀行]] (德岛) / [[hyakujushi-bank|百十四銀行]] (香川) / [[shikoku-bank|四国銀行]] (高知) 等
- 德岛县内: [[tokushima-taisho-bank|徳島大正銀行]] vs [[awa-bank|阿波銀行]] 的 2 行 体制
- 香川县内: [[kagawa-bank|香川銀行]] vs [[hyakujushi-bank|百十四銀行]] 的 2 行 体制
- 大阪府内: [[tokushima-taisho-bank|徳島大正銀行]]（旧大正银行门店）vs megabanks + [[smfg]] 系 + Kansai Mirai FG 等

## 4. 监管和政策

- **主管**: 金融厅（FSA）+ 四国财务局
- **控股公司监管**：银行法第 52  条之 17

- **近期政策论点**：
  - 2024 起，日本银行政策利率正常化 → 国内存贷款利差改善红利
  - 2025 起，地银重组压力（金融厅引导地银整合政策）
