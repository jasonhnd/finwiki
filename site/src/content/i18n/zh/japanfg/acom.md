# Acom株式会社（Acom）

## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[JapanFG/jaccs|株式会社ジャックス (Jaccs)]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

[[mufg]] 旗下大型消费金融公司。**1936  木下政雄在神户以“丸糸吴服店”创立，1978  将商号改为“Acom”**，并于 1996-10  在东证一部上市。业务结构以个人无担保贷款为主轴（卡贷产品“AC MasterCard”“无人君”自动签约机）+ 信用保证 + 海外业务（泰国 Easy Buy PCL）。它与 Acom（[[mufg]] 系列）、Promise（[[smfg]] 内的 SMBCCF）和 Aiful（独立系 [[aiful]]）并列，是“3  大消费金融公司”中的 MUFG 系列。以 2006-2008  灰色地带利率最高裁判决为契机，超额利息返还请求激增，**在 2008-10  MUFG TOB 后成为权益法适用公司（现为权益法 37.6% + 信托 2.0% = 39.6%）**。^[extracted]

## 1. 公司概要

**正式名称**: アコム株式会社（Acom Co., Ltd.）
**证券代码**: 东证 STANDARD **8572**
**创立**: **1936**（木下政雄在神户创办“丸糸吴服店”）
**改名**: **1978**（转向消费金融业并更名为“Acom”）
**总部**: 东京都千代田区丸之内
**上市**: 1993 店 店头注册 → **1996-10  东证一部上市** → 因 2022-04  市场区分调整转为东证 STANDARD ^[extracted]
**母公司**: 三菱 UFJ フィナンシャル・グループ（[[mufg]]）

### 业态类别 / 3  大消费金融公司

| 公司 | 系列 | 主要股东 | 上市区分 |
|---|---|---|---|
| **Acom** | **[[mufg]] 集团** ★ | **三菱 UFJ 权益法 37.6% + 信托 2.0% = 39.6%** | 东证 STD 8572 |
| Promise（SMBC Consumer Finance）| [[smfg]] 集团 | SMFG 100% 全资子公司 | 非上市（2012  退市）|
| [[aiful]] | 独立系（非大型银行系）| 以创始福田家为核心的独立资本 | 东证 PRIME 8515 |

### 主要集团公司

```
Acom（上市 8572，MUFG 权益法 39.6%）
  ├── Acom 本体 ── 以“Acom”品牌经营个人无担保贷款
  │     ├── AC MasterCard ── 带有信用卡功能的卡贷
  │     └── Mujinkun ── 自动签约机（行业象征性渠道）
  ├── Easy Buy PCL（泰国，71% 合并子公司）
  │     ── 泰国最大级别消费金融公司之一，“Umay+”摩托车贷款为主力
  ├── IR债权回收（100% 子公司）
  │     ── 资产服务商业务（债权回收）
  ├── Acom Guarantee（100% 子公司）
  │     ── 银行卡贷信用保证受托
  └── 三菱UFJ银行“Bankique”卡贷保证 ── 与 MUFG 联动的核心
```

### 与 MUFG 的资本关系

**2008-10-22 MUFG 通过 TOB 取得约 40% → 成为权益法适用公司**。此后，Acom 通过银行卡贷保证业务（Bankique）被纳入 MUFG 零售战略核心。截至 2025-03-31，权益法 37.6% + 信托 2.0% = **39.6%**（基于 MUFG 第 20  期有价证券报告书）^[extracted]

## 2. 业务板块地图

| 板块 | 主要业务 | 特征 |
|---|---|---|
| **个人无担保贷款** | Acom / AC MasterCard | **主力业务**，以卡贷为中心，Web 完结 + Mujinkun 自动签约机 |
| **信用保证业务** | 三菱 UFJ 银行“Bankique”等保证受托 | **MUFG 零售战略核心**，也为多家地方银行卡贷提供保证 |
| **海外（泰国）** | Easy Buy PCL “Umay+”（71% 合并子公司）| **泰国最大级消费金融公司之一**，以摩托车贷款为主，是行业先行模式 |
| **债权管理回收（servicer）** | IR债权回收 | 自有债权 + 第三方债权回收 |
| **信用卡** | AC MasterCard | 自主发行，卡贷一体化，独立品牌 |

### 自动签约机“无人君”

据称于 1993  首次在行业内导入，并作为**消费金融的象征性渠道**在全国铺开。“无人君”电视广告在 2000 年 时代成为行业图腾。超额利息返还问题后，随着业务转向 Web，该渠道趋于缩小，但作为即时审查与即时发卡渠道仍保留一定作用。^[extracted]

### Easy Buy PCL（泰国 Easy Buy）★

**泰国消费金融的行业先行模式。**以“Umay+”品牌围绕摩托车贷款展开，Acom 持有 71%，为其合并子公司。它与泰国景气、泰铢和监管风险联动，但在 ASEAN 小额个人金融领域先于其他公司布局。也与 MUFG 的 ASEAN 战略（[[mufg]] Krungsri / Bank Danamon）一致。^[extracted]

### 信用保证业务（MUFG 联动）

Acom Guarantee（100% 子公司）承接三菱 UFJ 银行“Bankique”卡贷保证。由此，Acom 通过直营个人贷款 + 银行后台信用保证的 2  轴获得收益；MUFG 则能在银行卡贷中提供上限利率约为 14.6% 的低利率产品。它也为许多地银承接卡贷保证。^[extracted]

### 超额利息问题与 MUFG TOB（2006-2008）★

- **最高裁判决（2006-01-13）**：几乎否定贷金业法中的“视同弁济”规定 → 超过利息限制法上限（15-20%）的部分须作为不当得利返还
- **2006  修订贷金业法**：上限利率由 29.2% 下调至 20%，并导入 **总量规制**（年收入 1/3）
- **超额利息返还**：全行业规模达到数万亿日元，Acom 也因巨额计提而承压
- **2008-10-22 MUFG TOB**：三菱 UFJ 银行（当时为东京三菱 UFJ 银行）取得约 40%，将其纳入权益法适用，实现注资与银行系重组 ^[extracted]

### 作为 MUFG 系列的位置 ★

- **银行卡贷保证**：通过 Bankique 保证成为 MUFG 零售战略核心
- **资金调达**：依托 MUFG 集团内融资，成本低于 [[aiful]] 等独立系
- **客户导流**：与三菱 UFJ 银行 ATM、网点联动，提升获客效率
- **海外**：Easy Buy（泰国）与 MUFG 的 ASEAN 战略（[[mufg]] Krungsri）互补

### 竞争结构

| 竞争者 | 系列 | 与 Acom 的差异 |
|---|---|---|
| Promise（SMBCCF）| [[smfg]] 100% 全资子公司 | 与母行一体化运营，作为 SMFG 100% 子公司导流效率高，已退市 |
| [[aiful]] | 独立系 | 以创始家族为核心、京都起源、脱离大型银行体系、依赖自有信贷数据 |

| [[orico]] / [[credit-saison]] / [[jaccs]] | 信贩系 | 以个品分期为主轴，与消费金融相邻但不同 |

### 中期战略（公开方向）

- **深化 MUFG 零售协同**：Bankique 保证 + 个人无担保贷款双轮驱动
- **AI 授信评分**：提升 Web 完结与即时审查精度
- **扩张海外（泰国 Easy Buy）**：利用在 ASEAN 小额个人金融中的先行地位
- **AC MasterCard**：通过卡贷 + 信用卡一体化扩大使用场景
- **债权回收（IR债权回收）**：将自有债权 + 外部债权回收培育为收益轴

### B2C 品牌

- **“无人君”广告** 在 1990-2000 年 时代成为行业图腾，象征即时审查与即时发卡
- “Acom”品牌 = 消费金融，“AC MasterCard”“Umay+”（泰国）构成不同品牌层
- 以绿色（企业色）为中心的高识别度品牌设计

## 5. 监管 / 政策

- **主管**: 金融厅（FSA）
- **登记**: 依据贷金业法注册的贷金业者
- **行业团体**: 日本贷金业协会（JFSA）会员
- **主要监管**:
  - **贷金业法**: 上限利率 20%（10 万円 以下为 20%，10〜100 万円 为 18%，超过 100 万円 为 15%）
  - **总量规制**: 个人借款总额为年收入的 1  / 3  以内（2010-06  全面施行）
  - **分期销售法**: AC MasterCard 部分属于综合信用购买斡旋业者
  - **个人信息保护法**: 授信信息处理
- **海外监管**: 泰国央行（BoT）对 Easy Buy PCL 的监管
- **近期政策议题**:
  - 2024 起 AI 授信的可解释性与反歧视
  - 超额利息返还请求虽在下降，但并未完全消失
  - 银行卡贷“事实性超出总量规制”问题（行业自律）
