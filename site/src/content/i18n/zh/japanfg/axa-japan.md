# 日本安盛

## 维基路线

该条目位于[[JapanFG/INDEX|JapanFG index]]。阅读它反对[[JapanFG/prudential-japan|プルデンシャル・ジブラルタ・ファイナンシャル (Prudential Japan)]]对于同行/对比上下文和[[insurance/INDEX|insurance index]]适用于更广泛的系统/监管边界。

## 长话短说

日本最大的外资人寿和非人寿保险集团之一。母公司为法国**AXA SA**（全球最大的保险及投资集团之一，全球顶尖的保费收入集团）3）。前身是1932 年成立于**日本团体人寿保险**，1994 年原UAP集团（后来的AXA集团）投资1999 年名称更改为“AXA Life”。同年，成立**安盛财产保险**，并推出非寿险行业首个成熟的线上直销模式。人寿和非人寿保险+网上直销人寿保险（AXA Direct Life）**3 具有通道混合**。

## 1。集团架构

**最终母公司**：AXA SA（法国，在巴黎泛欧交易所上市，CAC40 成分股）
**日本控股公司**：AXA Holdings Japan Co., Ltd.（原名 AXA Japan Holding）
**日本主要总部**：东京都港区白金1-17-3 NBF 白金塔（AXA 人寿保险）

### 主要日本法人

```
AXA SA（法国巴黎）
  └── 日本安盛控股（控股公司/非上市）
        ├── 安盛人寿保险有限公司── 以销售人员渠道为主的传统寿险
        │ └── 原日本集团人寿保险（1932 创业）→1999 改称
        ├── AXA Direct Life Insurance Co., Ltd. ── 网上直销人寿保险（专营医疗及收入保障）
        ├── 安盛财产保险有限公司── 网上车险/境外旅游保险
        ├── 安盛投资管理公司（AXA IM）日本基地── 资产管理
        └── 日本安盛援助 ── 海外旅行附带服务
```

### 重要历史（年表）

| 年月 | 现象 |
|---|---|
| 1932 | **日本团体人寿保险公司** 成立（战前作为团体人寿保险公司开始）|
| 1994 | 原UAP集团（现AXA集团）对日本集团人寿保险株式会社进行资本投资 |
| 1996 | 法国UAP与AXA合并，创建全球最大的保险集团之一AXA-UAP（后来的AXA） |
| **1999** | **更名为日本集团人寿 → AXA 人寿保险**，同年 **成立 AXA General Insurance**（进入在线汽车保险）|
| 2000 | 团体寿险及非寿险重组、非寿险业务专营网上直销（电话营销+网络） |
| 2000s 中盘 | 继**Nissei.com Life**等之后并入**AXA Direct Life** — 建立线上直销寿险品牌 |
| 2010 | **Winterthur Swiss Life** 整合日本子公司 — 合并现有合同区块 |
| 2010s 前半 | 销售人员渠道+银行柜台销售+网上直销3 建立渠道体系 |
| 2014〜2016 | 全球 AXA 集团战略 “雄心 AXA” “雄心2020”，日本也在转向防护产品。 |
| 2020〜2022 | 冠状病毒-19 期间，面对面的渠道限制→强化数字化接触点 |
| 2023〜2024 | 日本在集团“解锁未来”战略下也进军健康与预防（医疗保健）领域 |

## 2。业务线

| 部分 | 主要事业者 | 特征 |
|---|---|---|
| 个人人寿保险（销售人员）| 安盛人寿 | 终身、医疗、癌症、保证收入、可变金额。销售人员“财务计划顾问” |
| 团体人寿保险 | 安盛人寿 | 成立以来的中流砥柱（前身为日本集团人寿）。企业员工保障/退休金 |
| 银行销售 | 安盛人寿 | 终身一次性给付、外币计价、年金。与区域银行和大型银行建立广泛的场外销售联盟 |
| 网上直销人寿保险 | 安盛直接人寿保险 | 专注于医疗、期限和收入保障。价格低廉，每月几百日元起 |
| 网上直销非寿险 | 安盛财产保险公司 | 汽车保险、摩托车保险、海外旅行保险。电信+网络 |
| 资产管理 | AXA IM 日本基地 | 机构投资管理，全球 AXA IM 当地分支机构 |
| 海外援助 | 安盛援助 | 提供旅行保险的当地救援服务 |

### 定位

- **混合渠道策略**：日本主要公司（[[dai-ichi-life]]/[[nippon-life]]/ 住友生命保险公司 / 明治安田生命保险公司）以销售人员为主，专门从事互联网服务（[[JapanFG/lifenet|lifenet-insurance]]）只有一个在线业务，而 AXA 有**销售人员+银行柜台销售+网上直销**。3 渠道共存
- **非人寿保险直接在线销售**：[[JapanFG/tokio-marine|tokio-marine-hd]]/[[JapanFG/sompo|sompo-hd]]/[[JapanFG/msad|ms-and-ad-insurance]]国内经销商渠道3 与大型非人寿保险不同，“直接非人寿保险”部分与 Sony Assurance、Secom Assurance 和 SBI Assurance 相同

### 全球杠杆

- **安盛集团再保险平台**：通过集团内再保险吸收大额合同和巨灾风险
- **健康和预防领域**：在日本扩展 AXA 集团的“付款人到合作伙伴”战略（从简单的保险付款人到健康支持合作伙伴）

## 4。法规政策

- **监管**：由金融服务局 (FSA) 监管。人寿保险和非人寿保险均以《保险业法》为基础
- **偿付能力**：单一日本子公司受日本偿付能力充足率法规的约束，集团合并基于法国监管机构 ACPR
- **当前政策问题**：
  - **2025~引入基于经济价值的偿付能力法规（ESR）**：与日本主要公司类似，内部模型开发是一个问题
  - **代理商/销售人员渠道合规性**：加强老年人外币单次支付、终身支付的责任追究
  - **进军医疗健康领域**：打通金融服务中介业务+保险相关服务的界限
