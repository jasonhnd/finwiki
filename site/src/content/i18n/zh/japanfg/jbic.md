# 国际协力银行 (JBIC)

## Wiki route

本条目位于 [[JapanFG/INDEX|JapanFG index]] 之下。可与 [[JapanFG/jfc|日本政策金融公庫 (JFC)]] 对照阅读同业 / 对比语境，并与 [[banking/INDEX|banking index]] 阅读更广泛的系统 / 监管边界。

## TL;DR

日本面向海外的 **政策金融机构**（财务大臣为 100% 股东、特殊公司）。通过 **出口金融 + 海外投资金融 + 非约束性贷款 + 出资** 这 4  项业务，支持日本企业全球展开、资源能源确保和 GX。2012-04-01 从 [[jfc]]（日本政策金融公库）再次独立并特殊公司化。前身是 1950  设立的日本输出银行（后为日本输出入银行）和 1961  设立的海外经济协力基金（OECF）。1999-10  两者整合成立旧 JBIC，2008-10  一度并入 JFC，随后在 2012-04  再独立。与民间大型银行（[[mufg]] / [[smfg]] / [[mizuho-fg]]）的 **协同融资是核心模式**，与 [[policy-finance/jica|jica]] 的分工是“JBIC = 出口 / 投资，JICA = ODA”。^[extracted]

## 1. 公司概要

**正式名称**：株式会社国際協力銀行 ^[extracted]
**英文名称**：Japan Bank for International Cooperation (JBIC)
**法律依据**：株式会社国际协力银行法（平成 23 年 法律 第 39 号）^[extracted]
**设立**：**2012-04-01**（日本政策金融公库的国际金融部门分离并特殊公司化）^[extracted]
**股东**：**财务大臣 100%** ^[extracted]
**总部**：东京都千代田区大手町 1-4-1 ^[extracted]
**业态**：特殊公司（株式会社形态的政策金融机构，非上市）

### 沿革和前身

```
1950-12   日本输出银行设立（战后复兴的出口金融）
1953-04   改称日本输出入银行（追加进口金融）
1961-03   海外经济协力基金 (OECF) 设立（日元贷款 / ODA 核心）
1999-10   日本输出入银行 + OECF -> “国际协力银行 (旧 JBIC)”成立
2008-10   并入日本政策金融公库 (JFC) -> JBIC 成为 JFC 的国际金融部门
2012-04   从 JFC 再独立 -> “株式会社国际协力银行 (新 JBIC)”成立并特殊公司化
```

## 2. 业务分部地图

| 业务 | 内容 | 主要对象 |
|---|---|---|
| **出口金融** | 面向日本企业出口项目的融资（成套设备、船舶、基础设施、防卫装备等） | 日本出口者 / 海外买方 |
| **进口金融** | 确保能源和资源（LNG / 石油 / 稀有金属等）进口的融资 | 日本商社、电力、燃气 |
| **海外投资金融** | 日本企业海外 M&A、当地法人融资、JV 出资资金 | 日本企业海外子公司 |
| **非约束性贷款** | 产业发展支持融资（不与特定日本企业案件绑定，面向中等收入国家） | 中等收入国家政府 / 公共机构 |
| **出资** | 对战略案件的少数股权投资（风险资金供给） | 基础设施 / 资源 / GX 案件 |
| **担保** | 对民间金融机构贷款和债券提供担保 | 民间银行 / 机构投资者 |

### 业务模式

- **协同融资（cofinancing）是核心**：大型银行（[[mufg]] / [[smfg]] / [[mizuho-fg]]）+ 地方银行 + 海外金融机构与 JBIC 的组合是典型形式。
- **特别业务账户**（2016 起）：面向高风险案件和 GX 案件的风险资金供给框架。
- **与日元贷款的分工**：ODA 日元贷款由 [[policy-finance/jica|jica]] 负责（2008-10  移交）-> JBIC 专注于具有商业性的海外投融资。

### 3  战略轴

1. **支持日本企业全球展开**：M&A 资金 / 成套设备出口 / 基础设施出口
2. **确保资源和能源**：LNG / 石油 / 稀有金属 / 蓄电池材料
3. **GX（绿色转型）**：可再生能源 / 氢 / 氨 / CCS / EV 供应链 ^[extracted]

### 合作和收购战略

- **大型银行协同融资**：与 [[mufg]] / [[smfg]] / [[mizuho-fg]] 经常协同（成套设备出口案件等）。
- **与 JICA 分工**：ODA = JICA，商业性海外投融资 = JBIC（2008-10 円 贷款部门移交以后）。
- **NEXI 联动**：日本贸易保险（NEXI）的出口保险与 JBIC 融资组合。
- **海外政策金融机构联动**：与美国 EXIM Bank、韩国 K-EXIM、法国 Bpifrance、德国 KfW IPEX 等协同融资。
- **国际能源框架**：与 JOGMEC、经济产业省联动，支持资源外交。

### 与民间金融机构的关系

- “**民业补完**”原则：补充仅靠民间金融机构难以承担风险的案件，例如发展中国家 sovereign 风险、长期项目融资、有政治风险的案件等。

### 经济安全保障转向（2022 起）

- **美中对立下的经济安全保障**：半导体 / 重要矿物 / 蓄电池 / 生物等供应链韧性案件。
- **friend-shoring 支持**：从金融层面支持日本企业迁移到盟国和友好国。

## 4. 监管和政策

- **主管**：财务省（股东和主管大臣）
- **法律依据**：株式会社国际协力银行法（平成 23 年 法律 第 39 号）^[extracted]
- **业务范围**：该法列举出口金融 / 进口金融 / 海外投资金融 / 非约束性贷款 / 出资 / 担保等。
- **环境社会配虑指引**（JBIC 指引）：符合 OECD Common Approaches 和赤道原则的环境社会影响评估框架。
- **OECD 出口信用安排**：符合官方出口信用的国际规则，包括最低利率和最长偿还期限。
  - 2022 起 与经济安全保障推进法联动
  - 2023 起 与 GX 推进法 / GX 经济转型债联动（transition finance）
  - 2024 起 应对美国 IRA / 欧洲 CBAM，并支持对美欧投资案件

### 相关机构生态系统

```
财务省 ---- 主管 ---- JBIC（海外投融资 / 出口金融）
                  |
外务省 ---- 主管 ---- JICA（ODA / 日元贷款 / 技术协力）
                  |
经济产业省 ---- 主管 ---- NEXI（贸易保险）
                  |
              JOGMEC（资源开发 / 出资）
```
