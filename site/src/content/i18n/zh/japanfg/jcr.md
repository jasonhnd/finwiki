# 日本信用评级机构 JCR

## Wiki 路径

本条目位于 [[JapanFG/INDEX|JapanFG index]] 之下。可结合 [[JapanFG/rating-and-investment|格付投資情報センター (R&I)]] 阅读，以获得同业 / 对比语境；也可结合 [[JapanFG/fsa|金融庁]] 阅读，以理解更大的监管边界（信用评级机构注册）。

## TL;DR

日本资本市场的 **2 社 家本土信用评级机构之一（其中的 1 社 家）**（另一个 1 社 是 [[JapanFG/rating-and-investment|R&I]]）。成立于 1985-04 ，标榜独立系（没有特定金融集团母公司）。在 **主权 / 金融机构 / 事业公司 / 证券化产品 / 项目融资 / 可持续金融** 全领域提供评级。已取得**基于《金融商品交易法》的信用评级机构注册**（金融厅长官注册），并作为 ECAI（外部信用评估机构）在日本标准和巴塞尔标准下合格。与全球 3  大机构（S&P / Moody's / Fitch）相比，长期以来被指出**对日本企业的评级存在 1 至 2  个 notch 的偏宽倾向**。

## 1. 法人结构与沿革

**正式名称**: 株式会社日本格付研究所
**英文名**: Japan Credit Rating Agency, Ltd. (JCR)
**设立**: **1985-04-01**
**总部**: 东京都千代田区神田锦町
**业态**: 信用评级机构（《金融商品交易法》第 66  条之 27  注册）
**主要股东**: 由国内金融机构和事业公司分散持有（独立系）
**主管**: 金融厅（信用评级机构监管）

### 重要年表

| 年月 | 事项 |
|---|---|
| **1985-04** | **设立**（为应对泡沫期前公司债发行急增带来的评级需求） |
| 1991 | 国内公司债评级正式展开 |
| 1998 | 《金融商品交易法》修订（信用评级机构注册制度于 2010 施行） |
| 2007起 | 次贷危机 -> 国际上围绕评级行业监管的讨论活跃化 |
| **2010-04** | **信用评级机构注册制度施行** -> JCR 作为注册 第 1 号 级机构注册（与 [[JapanFG/rating-and-investment|R&I]] 并列） |
| 2013起 | 强化结构化金融（RMBS / ABS）评级透明度 |
| 2017起 | 开始 **绿色债券 / 社会债券 / 可持续发展挂钩债券** 等 ESG 相关评价业务 |
| 2020起 | 新冠后公司债发行急增阶段，评级件数增加 |

## 2. 评级业务 map

| 领域 | 内容 | 主要客户 |
|---|---|---|
| **主权评级** | 日本国债、地方债、政府相关机构债 | 国家 / 地方公共团体 / 政府相关机构 |
| **金融机构评级** | 银行、证券、保险、非银机构的发行体评级 | [[JapanFG/mufg|MUFG]] / [[JapanFG/smfg|SMFG]] / [[JapanFG/mizuho-fg|みずほ FG]] 等 |
| **事业公司评级** | 一般事业公司的公司债和长期发行体评级 | 以上市大企业为中心 |
| **结构化金融** | RMBS / ABS / CMBS / ABCP 等 | SPV / 安排人 |
| **项目融资** | 基础设施、可再生能源、PPP 项目 | 项目 SPV |
| **保险财务实力评级** | 寿险、非寿险公司的赔付能力评价 | 寿险 / 非寿险 |
| **ESG / 可持续评价** | 绿色 / 社会 / 可持续 / 转型债评价 | 各类发行体 |
| **第三方意见书** | 关于框架合规性的 SPO | 发行体 |

### 评级尺度

- **长期评级**: AAA -> AA+ / AA / AA- -> A+ / A / A- -> BBB+ / BBB / BBB- -> BB 以下（投机级）-> D（违约）
- **短期评级**: J-1+ / J-1 / J-2 / J-3 / NJ / D
- 也另行提供面向外币发行体的国际尺度。

### 日本国内 4  大评级机构

| 评级机构 | 特征 | 母公司 / 系列 |
|---|---|---|
| **JCR** | 独立系、本土系 | 独立系（无特定母公司） |
| **R&I** | 瑞穗系（旧 IBJ 系 + 日经系）| 瑞穗系（与 [[JapanFG/mizuho-fg|みずほ FG]] 相关） |
| **S&P Global Ratings Japan** | 美国 S&P 子公司 | S&P Global |
| **Moody's Japan / Moody's SF Japan** | 美国 Moody's 子公司 | Moody's Corp |

### 国内 vs 国际评级差距问题

- 实证研究确认，日本企业的 **JCR / R&I 评级高于 S&P / Moody's** 的倾向（典型情况下高 1 至 2  个 notch，部分案件也报告 4  个 notch 以上的差距） — 日本银行 working paper《Credit Rating Gaps in Japan》(2007) 及其他同行评审论文
- 背景：相对评价方法将日本企业的主银行制度和集团救助惯行纳入考虑（实证研究将其解释为信用风险评价方法的系统性差异，而非简单的“上方偏差”）
- 国际投资者持续批评“日本评级存在乐观偏差”，另一方面也有反驳认为其反映了国内市场实际。
- **许多日本企业会从 2 社 家以上机构取得评级**（国内和国际并用是常规做法）。

### 竞争特征

- 标榜**独立系**（相对于 [[JapanFG/rating-and-investment|R&I]] 被视为“瑞穗系”，其差异点在于没有特定金融集团母公司）
- 在中坚、中小企业发行体评级中的份额相对较高
- 在可持续评价业务上先行（提供符合 GBP / SBP / SLBP 各原则的 SPO）

## 4. 监管与规制

- **主管**: [[JapanFG/fsa|金融庁]]（信用评级机构监管）
- **依据法律**: 《金融商品交易法》第 66  条之 27  以下
- **注册要求**: 业务管理体制、防止利益冲突、评级流程文件化、投诉处理体制等
- **ECAI 认定**: 基于巴塞尔银行监管委员会标准的外部信用评估机构认定（金融厅告示） -> 可用于银行风险资产计算
- **国际协调**: 遵循 IOSCO 信用评级机构行为准则
