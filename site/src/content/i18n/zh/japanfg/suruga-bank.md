# Suruga Bank（スルガ銀行）

## Wiki 路径

本条目位于 [[JapanFG/INDEX|JapanFG index]] 之下。可结合 [[JapanFG/fukuoka-fg|ふくおか FG (FFG)]] 作为同业 / 对照背景，并结合 [[banking/INDEX|banking index]] 理解更广泛的制度 / 监管边界。

## 摘要

スルガ銀行是以静冈县东部（沼津）为地盘的中坚地方银行。它**于 1895 年 作为根方银行创业，拥有 130  年历史**。1990 年 年代至 2010 年 年代，它曾以“个人贷款特化型地银的优等生”身份拥有高 ROE，但 **2018 年 面向共享住宅“Kabocha no Basha”的违规融资事件**使评价完全逆转。面向个人投资者的贷款超过 1 兆円，其中文件篡改和审查舞弊以组织形式进行，最终导致金融厅业务改善命令和经营层集体辞职。2020 年 以后，银行进入以**与 [[JapanFG/au-fh|ノジマ]] 的资本业务合作**为轴的信赖恢复阶段。它与县内 No.1  的 [[shizuoka-fg]] 构成结构性竞争。^[extracted]

| 分部 | 内容 | 特征 |
|---|---|---|
| 个人贷款（旧主力）| 住房贷款、投资用房地产、卡贷款 | 事件前的高 ROE 引擎 -> 大幅缩小 |
| 个人存款 | 静冈县东部、神奈川县西部 | 本地密着，退休世代较厚 |
| 法人贷款 | 中坚 / 中小企业 | 过去较弱的领域，事件后增长轴 |
| 投资信托 / 保险销售 | 面向个人的资产形成 | 零售手续费业务 |
| ノジマ合作门店内银行 | ノジマ门店内 ATM / 账户窗口 | 2020 起合作核心 |
| 区域合作 | 静冈县东部 / 神奈川西部的自治体、产学合作 | 信赖恢复阶段的公共领域 |

### 信赖恢复阶段战略

- **摆脱个人贷款特化**：从投资用房地产贷款转向法人和区域合作。
- **与ノジマ资本业务合作**（2020-05）：在家电量贩店门店内设置银行窗口，扩大年轻层接点。
- **强化合规和治理**：反映第三方委员会指出的问题，扩充社外董事和审计功能。
- **数字投资**：网络银行功能和移动账户。

### 竞争关系

- **县内**：[[shizuoka-fg]] / [[shizuoka-bank]] 在全县范围为 No.1。[[shimizu-bank]] and [[shizuoka-chuo-bank]] 是另外的本地银行同业；スルガ偏重县东部。
- **县外扩张**：在神奈川县西部和东京西部也有网点，直接与大型地银、大型银行和 [[concordia-fg]] 竞争。
- **个人贷款残留**：在房地产贷款缩小阶段，年轻层流向 [[paypay-fg]] / [[au-fh]] / [[sony-fg]] 等网络银行。

### 合作 / 收购战略

- **ノジマ资本业务合作**（2020-05）：ノジマ 出资约 18%，推进门店内银行 + 数字融合。
- **区域合作**：与静冈县、沼津市、神奈川县西部自治体签署地方创生协定。
- **与大型银行关系**：传统上与都市银行有人才交流，事件后强化外部人才引入。

## 4. 监管 / 政策

- **主管**：金融厅（FSA）+ 关东财务局
- **2018-10-05 业务改善命令** ^[extracted]：
  - 共享住宅相关投资用房地产贷款停止 6  个月
  - 明确经营责任，根本性复核治理
  - 客户保护（构建受害业主应对机制）
- **第三方委员会报告书（2018-09-07）**：认定组织性实施了文件篡改、源泉征收票伪造、租金台账篡改、存折“展示资金”、跳过审查等行为。
- **G-SIB**：不适用（地银规模）
- **近期政策论点**：
  - 日本央行政策利率正常化 -> 存贷利差改善（对地银整体有利）
  - 地银重组压力（金融厅主导的地银整合）— スルガ走独立路线、以ノジマ为轴
  - 持续监控信赖恢复进展

## 5. 重要事件：共享住宅“Kabocha no Basha”违规融资 ★

震动行业的组织性违规事件。它后来成为地银治理讨论和投资用房地产贷款监管强化的触发点。

### 事件结构 ^[extracted]

- **舞台**：共享住宅运营商 Smart Days（女性专用共享住宅“Kabocha no Basha”品牌）
- **机制**：Smart Days 向个人投资者出售物业 -> スルガ银行提供高额贷款（1  物件、超过 1 億円）-> Smart Days 以转租方式提供租金保证
- **2018-01 至 04**：Smart Days 停止支付租金 -> 经营破产 -> 约 700 名 名业主（个人投资者）无法偿还
- **违规的组织性**：
  - 篡改存款余额证明书和源泉征收票（伪装年收入）
  - 篡改租金台账
  - 通过“展示资金”伪装自有资金
  - 为达成销售指标跳过审查
  - 被认定并非部分网点 / 业务人员单独行为，而是**网点整体参与、总部默许**
- **贷款总额**：投资用房地产整体超过约 1 兆円，共享住宅只是其中一部分
- **损害**：业主约 700 名，平均贷款超过 1 億円

### 解决经过 ^[extracted]

- 2018-09：第三方委员会报告 -> 认定组织性违规
- 2018-10：金融厅业务改善命令
- 2018-2019：经营层集体辞职（社长、专务以下），大量引入外部董事
- 2019-2020：与受害业主进行债务减免谈判（债权转让 + 本金减免）
- 诉讼：部分刑事告发和民事诉讼持续

### 对行业的影响

- **对投资用房地产贷款整体加强监管**：金融厅也全面检查其他地银和大型银行的房地产贷款。
- **治理讨论**：重新思考地银社外董事和审计功能的应有状态。
- **个人贷款特化模式的极限**：高 ROE 背后的风险显现，引发地银商业模式复核讨论。
