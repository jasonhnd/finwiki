# 冲绳振兴开发金融公库（ODFC）

## Wiki 路径

本条目位于 [[JapanFG/INDEX|JapanFG index]] 之下。可与 [[JapanFG/jfc|日本政策金融公庫 (JFC)]] 对读以理解同业 / 对比语境，并与 [[banking/INDEX|banking index]] 对读以把握更广的制度 / 监管边界。

## TL;DR

仅限冲绳县的政策金融特殊法人。它在 **1972-05-15冲绳回归日本本土的同时设立**，是县内唯一的综合政策金融机构，独特之处在于把本土的 [[jfc]]（国民、中小企业、农林渔业）+ [[dbj]]（中坚 / 大企业）+ 住宅金融支援机构功能 **在冲绳县内由一个机构整合提供**。它是由国家 100%出资的特殊法人，由内阁府冲绳担当部门主管。依据 **冲绳振兴特别措施法** 的特别利率和特别保证，支持县内企业、购房者和离岛居民。在 2008 政策金融重组、本土公库整合为 JFC 时，也因 **“冲绳的特殊情况”而决定单独存续**，是少见机构。^[extracted]

## 1. 公司概要

**正式名称**：冲绳振兴开发金融公库 ^[extracted]
**英文名**：Okinawa Development Finance Corporation (ODFC)
**依据法律**：冲绳振兴开发金融公库法（昭和 47 年法律第 31 号）^[extracted]
**设立**：1972-05-15（与冲绳回归本土同时，承继琉球政府金融机构）^[extracted]
**形态**：特殊法人（不是株式会社形态，也不同于本土 JFC 的独立组织体）^[extracted]
**总部**：冲绳县那霸市 Omoromachi 1-2-26（那霸新都心地区）^[likely]
**出资者**：日本政府 100%（一般会计出资 + 财投借款）^[extracted]
**主管**：内阁府（冲绳担当部门）+ 财务省（共管）^[extracted]

### 设立背景 ^[extracted]

1972-05-15 冲绳回归本土时，整合承继了琉球政府下列 4 家金融机构：

| 整合前机构（琉球政府下） | 承继功能 |
|---|---|
| 琉球开发金融公社 | 面向产业基础和中坚企业的开发融资 |
| 民政府工业开发金融公库 | 面向工业振兴的融资 |
| 大众金融公库 | 面向中小和小规模事业者的融资 |
| 琉球住宅公库 | 购房和住房建设融资 |

-> 与回归本土同时，重组整合为 **冲绳振兴开发金融公库** 并发足

### 治理 / 机构设计

- **理事长**：经内阁总理大臣认可后由主管大臣任命 ^[likely]
- 役员：理事长 + 副理事长 + 理事 + 监事（特殊法人治理标准）
- 通过经营委员会 / 业务委员会运营业务
- **国会报告义务**：每年度向国会提交业务概况和决算 ^[likely]
- **会计检查院**：因政府全额出资而经常接受检查

### 业务部门（县内完结型・一机构整合）

```
ODFC（冲绳振兴开发金融公库・100% 国出资）
  ├── 中小企业事业
  │     └── 面向县内中小企业的运营・设备资金（本土 JFC 中小企业事业 + DBJ 功能的冲绳版）
  ├── 产业开发事业
  │     └── 面向旅游、制造、农水产加工、物流的开发融资（本土 DBJ 功能的冲绳版）
  ├── 住宅事业
  │     └── 个人住房贷款・租赁住宅建设（本土住宅金融支援机构功能的冲绳版）
  ├── 农林水产业事业
  │     └── 面向县内农家、畜产、渔业的融资（本土 JFC 农林水产事业的冲绳版）
  ├── 国民生活事业
  │     └── 面向县内个人经营者和小规模事业者的融资（本土 JFC 国民生活事业的冲绳版）
  ├── 灾害复兴事业
  │     └── 台风、地震、海啸等灾害复兴特别贷款
  └── 教育贷款事业
        └── 面向冲绳县内大学生、专修学校生的奖学融资・教育贷款
```

### 县内完结型的特殊性

- **由 1 个机构提供本土 3 个公库功能**：JFC 在本土将“国民生活 + 中小企业 + 农林水产”分部门，但 ODFC 仅在冲绳县内把这些功能再加 DBJ 功能 + 住宅金融支援机构功能 **全部整合** 提供。
- **基于冲绳振兴特别措施法的特别利率**：可向县内事业者适用比本土 JFC 更低一档的政策利率。
- **岛屿地域（离岛）対応**：面向宫古、八重山、大东诸岛等县内离岛事业者和居民的独自融资菜单。

### 民间补完原则（冲绳县内版）

ODFC 的存在意义与本土 JFC 相同，在于 **覆盖民间金融机构难以放贷的风险**，但它与冲绳县内 3 行寡占结构的补完关系很独特：

- **[[okinawa-fg]] 冲绳银行**（县内存贷份额约 30%，首位）
- **[[ryukyu-bank]] 琉球银行**（同 No.2，与冲绳银行隔年承接县指定金融机构）
- **冲绳海邦银行**（同 No. 3）

-> 这 3 行家银行的自有贷款 + ODFC 政策贷款协同融资，是县内中小企业金融的标准形态。

### 当前战略的 4 轴 ^[likely]

1. **旅游业复兴・扩张支持**：那霸机场第 2 跑道 + JUNGLIA 主题公园（2025〜开业）+ MICE 据点建设等，为新冠后旅游业 V 字复苏提供融资供给
2. **美军基地旧址再开发**：伴随返还预定地（西普天间、Camp Kinser 等）的都市再开发和产业用地化提供项目融资
3. **离岛振兴・岛屿地域支持**：支持宫古、八重山、大东诸岛等具有不同于本土经济结构的离岛事业者和居民
4. **灾害复兴・防灾**：反映台风、地震、海啸等自然灾害较多的县情，常设复兴融资体制

### 与第 6 次冲绳振兴计划（2022〜2031）联动 ^[extracted]

在 2022-2031 年年度的第 6 次冲绳振兴计划中，以下被列为重点领域，ODFC 被定位为核心资金供给机构：

- **旅游相关产业高度化**
- **信息通信相关产业・IT 据点强化**
- **物流・国际枢纽功能（那霸机场・那霸港）**
- **健康医疗・生物相关产业**
- **农林水产业高附加值化（甘蔗、芒果、水产加工等）**
- **离岛振兴・条件不利地区支持**

### 与民间金融机构的关系

- **协同融资主角**：与 [[okinawa-fg]] / [[ryukyu-bank]] / 冲绳海邦银行的自有贷款并用，是县内中小企业和旅游业融资的标准形态。
- **与县内地方银行的补完原则**：为避免压迫民业，优先覆盖创业期、无担保无保证人、离岛、灾害复兴等民间难以放贷的层。
- **与本土公库分工**：与 [[jfc]] / [[dbj]] / 住宅金融支援机构之间，按地区分工为“冲绳县内案件由 ODFC，本土案件由本土公库”。

## 4. 监管 / 政策

- **主管**：内阁府（冲绳担当部门）+ 财务省共管 ^[extracted]
- **依据法**：冲绳振兴开发金融公库法（昭和 47 年法律第 31 号）
- **基本法**：冲绳振兴特别措施法（平成 14 年法律第 14 号，每 5 年修订）
- **会计检查院**：因政府全额出资而经常接受检查
- **国会报告**：每年度向国会报告业务概况和决算
- **资金筹措**：财投借款 + 政府保证债 + 自有资金 ^[likely]

### 2008 政策金融重组中的单独存续决定 ^[extracted]

2008-10-01 本土 4 家政策金融机构（国民生活金融公库 + 中小企业金融公库 + 农林渔业金融公库 + JBIC 国内部门）整合为 JFC 时，也曾讨论整合 ODFC，但基于以下理由决定 **单独存续**：

- **冲绳县固有的历史经纬**（回归本土特别措施的继承）
- 维持 **基于冲绳振兴特别措施法的特别利率体系** 的必要性
- 面向 **岛屿地域和离岛対応** 的独自 know-how
- **美军基地和基地旧址再开发** 这一其他县没有的政策课题

-> 在政策金融改革中，ODFC 作为例外措施被单独保留下来

### 近期政策议题

- 2024〜 随着新冠特别贷款 **还款期到来**，县内旅游业者和中小企业违约风险引发关注
- 2025〜 应对旅游业 V 字复苏带来的设备投资资金需求扩大
- 2025〜 美军基地旧址再开发（西普天间、Camp Kinser 等）的长期项目融资
- 2031 第 6 次冲绳振兴计划结束后，围绕机构和特别措施法能否继续的讨论
