# Japan Post Bank（株式会社ゆうちょ銀行 / 7182）运营画像

| 最终国家股东 | 财务大臣根据《邮政民营化法》框架持有 Japan Post Holdings 的政府持股 |

| 上市路径 | 2015-11-04 年 Japan Post Holdings + Japan Post Bank + Japan Post Insurance 三家公司同步 IPO |
| 银行类别 | 《银行法》下的普通银行；尽管有邮政储蓄源流，并非特殊金融机构 |

### 三银行母公司链（IPO 后）

```
财务大臣（国家股东）
        │  Japan Post Holdings 约 1/3+ 为国家持有（截至 2025-09-30，财务大臣 37.23%；《邮政民营化法》规定三分之一下限）
        ▼
日本邮政株式会社（Japan Post Holdings，东证 6178）
        ├── Japan Post Bank（东证 7182；母公司议决权从 IPO 时的 100% 稀释至截至 2025-03-31 的 50.04%，并在 2025 低于 50%）
        ├── Japan Post Insurance（东证 7181）
        ├── Japan Post（100%）— 运营约 24,000  个邮局网络
        └── 未上市不动产 / 物流相邻业务
```

### 资产侧：由存款融资，但不是贷款主导

| 侧面 | Megabank 常态 | Japan Post Bank |
|---|---|---|
| 贷款 | 资产中 50-60%，包括企业、房贷和海外组合 | 资产中占比较小；直接零售贷款有限，且没有大型企业贷款账簿 |
| 存贷比 | 约 70-90% 区间 | 结构性偏低；JPB 是证券组合银行，不是贷款银行（大多数资产在 JGB / 证券而非贷款） |

"貯金" 一词是邮政储蓄命名遗产；在《银行法》下这些负债是银行存款（"預金"），但 JPB 产品品牌保留了历史用语。^[extracted]

### 为什么没有大规模贷款账簿

三个原因叠加：

1. **民间补完原则。** 《邮政民营化法》框架预期 JPB 不应排挤民间银行。新增贷款业务线需要邮政民营化委员会批准，并已逐步放行（例如个人无担保贷款许可、有限的房贷能力），但该银行没有 megabank 型企业贷款能力。  
2. **没有企业银行 franchise。** JPB 从未拥有 megabank 面向大型企业的关系型银行基础设施。没有该等获客 / 发放引擎，从零构建贷款账簿并不经济。  
3. **围绕邮局柜台设计的运营模式。** 约 24,000  个邮局分销网络优化于高频小额存款、汇款和基础金融产品销售，而非信用承销和关系型银行业务。

### 证券组合：政策利率敞口问题

JPB 的证券账簿长期是主要盈利驱动和主要风险集中。在长期负利率 / ZIRP 时期，JGB 占比较高的组合造成收益率压缩并限制再平衡空间。2024-2026  日本银行利率正常化周期在两个相反方向上具有重大影响：

- 利率上升时，存量长久期 JGB 账簿面临**盯市逆风**。
- 新投资可按更高再投资收益率进入、政策利率联动的存款侧利差解压，形成 **NIM 顺风**。

综合净影响取决于久期结构、套保和投资组合轮换速度。JPB IR 披露证券构成和未实现损益的概要；细颗粒跟踪属于 IR 披露周期，不属于 wiki 页面。

### 邮局柜台实际如何运作

JPB 并不拥有大多数面向客户的柜台。交易流程为：

```
客户在邮局柜台
      ↓  （存款 / 取款 / 汇款 / 投资产品下单）
Japan Post — 柜台运营者、ATM 运营者
      ↓  （根据合同作为 JPB 的银行代理）
Japan Post Bank — 账户持有人、资产负债表承担者
```

| 表层 | 运营者 | 客户体验 |
|---|---|---|
| 全国约 24,000  个邮局柜台 | Japan Post 作为银行代理 | 同一柜台处理邮件、包裹和 JPB 交易 |
| 约 30,000+ 台 Japan Post Bank ATM | 邮局、车站、零售场所的 JPB 品牌 ATM | 直接 JPB 品牌；通过 [[banking/INDEX|banking index]] 银行间 rails 与其他银行 ATM 网络互联 |
| 直营分店 | 235  个直营店（营业所，截至 2025-03-31） | 集中于大城市，提供更高接触度服务 |
| Japan Post Bank 存折 app + Web | JPB 自有数字渠道 | 账户余额、明细、转账；功能集比 [[banking/rakuten-bank|Rakuten Bank]] · [[banking/paypay-bank|PayPay Bank]] · [[banking/au-jibun-bank|au Jibun Bank]] 更薄 |

### 为什么代理模式既锚定又约束 franchise

- 24,000 个柜台覆盖在结构上**没有任何日本银行可比**。它给 JPB 带来最深的地理覆盖，包括 megabank 网点无法服务的农村和高龄客户基础。
- 但成本结构与 Japan Post（邮件 / 包裹）共享，JPB 向姊妹公司支付内部手续费，在低利率时期会吸收存款利差经济性。
- 客户人口结构比 megabank 或网络银行更偏高龄，这既稳定存款基础（较少追逐利率），也限制了这些客户向更高利润产品组合（房贷、投资产品、卡）转化。

## 4. KPI 与盈利画像

JPB 公开的核心财务数据显示出结构性模式：

| 维度 | 模式 |
|---|---|
| 总资产 | ¥233.6tn（2025-03-31） |
| 存款基础 | ¥190.5tn（2025-03-31）——零售组合占主导 |
| NIM | 受 JGB 占比较高的组合和历史 ZIRP/NIRP 环境结构性压缩 |
| 成本收入比 | 通过向 [[JapanFG/japan-post|Japan Post]] 支付集团内费用承担邮局柜台代理成本 |
| 资本 | 资本比率持续高于《银行法》最低要求；未被指定为 G-SIB——JPB 是大型国内机构，不是全球跨境系统性银行 |

关于时点数值，请参照最新 JPB 决算短信 / 有价证券报告书周期；本页跟踪的是 franchise 的**形态**，而非滚动 headline 数字。

### JPB 的竞争领域

- **存款黏性**是 JPB 的结构性护城河。自 1875 至今仍留存的零售储蓄，即便利率不利，也异常不容易迁移。
- **柜台覆盖**以 megabank 无法匹配的方式主导农村日本。
- 在 2024-2026  正常化周期中，**债券组合收益率上行**带来一次性的多年再投资顺风。

### JPB 不竞争的领域

- **企业银行、投行和信托。** Megabank 主导该领域。
- **数字原生主账户银行。** [[banking/rakuten-bank|Rakuten Bank]]、[[banking/paypay-bank|PayPay Bank]]、[[banking/au-jibun-bank|au Jibun Bank]]、[[JapanFG/ssnb|住信SBI / NEOBANK]] 在 app-native UX 和生态联动获客上更强。JPB 的数字渠道（Japan Post Bank 存折 app、mijika）明显落后。
- **规模化房贷发放。** 在民间补完框架下，JPB 在此能力有限。

### 管理层战略杠杆

1. **证券组合多元化。** 持续从 JGB 轮换至外债、另类资产和私募基金；这是中期计划下的多年努力。  
2. **邮局渠道成本纪律。** 在邮件量下降背景下，与 Japan Post 协商内部费用架构。  
3. **数字渠道投资。** Japan Post Bank 存折 app 和 mijika 项目旨在减缓客户基础老龄化。  
4. **适度扩展手续费业务相邻项。** 投资信托销售、发卡合作（JP BANK Card——Visa/Mastercard 由三井住友卡担保，JCB 版本与 JCB 共同发行）、与 Japan Post Insurance 的保险分销重叠。  
5. **民营化步骤合规。** 随着 Japan Post Holdings 降低 JPB 持股，邮政民营化委员会框架下更多业务线批准成为可能。

### 叠加监管

| 层级 | 法规 / 监管者 | 对 JPB 的含义 |
|---|---|---|
| 银行业 | 《银行法》、金融厅监管指引 | 普通银行标准监管：资本、流动性、治理、IT、AML |
| 邮政民营化 | 《邮政民营化法》、邮政民营化委员会 | 新业务线许可框架；母公司持股降低顺序 |
| 邮政系统政策 | 总务省（邮政行政） | 柜台网络政策、影响 Japan Post 的普遍服务义务（不直接作用于 JPB） |
| 存款保险 | 存款保险机构 | JPB 作为成员参加；标准制度下适用存款保障 |
| 证券监管 | 《金融商品交易法》——适用于投资信托销售和证券业务 | JPB 投信销售行为按标准规则受监管 |
| 消费者保护 | 《消费者契约法》+ 行业指引 | 在 Japan Post Insurance 问题后，对 JP Group 整体重要性提高 |

### 民间补完原则——运营约束

《邮政民营化法》写入“民间补完”预期：JPB 的业务扩张不应排挤既有民间金融提供者。实践中，这通过邮政民营化委员会对新产品许可逐案审批来执行，例如无担保个人贷款扩张、房贷能力和企业贷款试点。该原则是 JPB 未能构建与其存款规模相匹配贷款账簿的结构性原因。随着国家持股下降，是否应放宽该原则，是反复出现的政策争论。

### Japan Post Insurance 问题后的治理阴影

2019  Japan Post Insurance 误售丑闻（通过邮局柜台不当销售保险产品）引发集团范围合规和治理改革，包括更严格监督邮局交叉销售模式。JPB 的投资信托销售实践也同步收紧，新产品交叉销售扩张速度因此受到制约。
