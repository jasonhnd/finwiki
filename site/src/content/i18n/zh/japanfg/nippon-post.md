# 日本邮政（Japan Post Holdings）

## Wiki 路线

本条目位于 [[JapanFG/INDEX|JapanFG index]] 之下。可对照 [[JapanFG/kampo-life|かんぽ生命保険 (Japan Post Insurance)]] 阅读同业 / 对比背景，并结合 [[banking/INDEX|banking index]] 理解更广泛的系统与监管边界。

## TL;DR

日本邮政是由旧日本邮政公社改组而来的“邮政 + 金融 [银行 + 保险]”3  本柱上市控股公司。它是**日本国内最大级金融集团**之一（合并基础下集团运用资产规模超过 250 兆円）。2015-11-04，控股公司 / [[yucho]] / [[kampo-life]] 完成 **3 社 同时 IPO**。第一大股东为财务大臣（根据邮政民营化法，**当面有义务持有超过 1/3 **）。2019-2020  [[kampo-life]] 不当销售事件后，公司经历经营层更替并收到业务改善命令，恢复信任与改善运用收益率成为中期主题。^[extracted]

## 1. 公司概要

**正式名**：日本邮政株式会社 ^[extracted]
**英文名**：Japan Post Holdings Co., Ltd.
**证券代码**：东证 PRIME 6178（2015-11-04 上市）^[extracted]
**设立**：2006-01-23（作为邮政民营化准备公司“日本邮政株式会社”）^[extracted]
**民营化启动**：2007-10-01（邮政民营化法施行，4 社 体制启动）

### 股东结构（民营化法上的特例）^[extracted]

| 股东 | 比率 | 备注 |
|---|---|---|
| 财务大臣（日本政府） | **超过 1/3 **（当面维持义务） | 邮政民营化法 / 日本邮政株式会社法 |
| 一般股东 / 机构投资者 | 剩余 | 阶段性出售（2015-11  / 2017-09  / 2021-10  等） |

> 根据日本邮政株式会社法与邮政民营化法，政府负有**当面持有 3  分之 1  以上**的义务。此外，日本邮政自身也被要求分阶段处置 [[yucho]] / [[kampo-life]] 各公司股份，因此政府对金融 2 社 的参与以双重结构残留。^[extracted]

### 主要子公司 / 持分

```
日本邮政株式会社（控股公司・上市 6178）
  ├── 日本邮便株式会社（100%）── 邮政、物流、邮局网络运营
  │     ├── Yu-Pack / Yu-Mail / 国际邮政 EMS
  │     ├── 全国约 24,000  个邮局网络（民间最大店铺网络）

  ├── Japan Post Bank Co., Ltd.（上市 7182）── 按存款规模为国内最大级 [[yucho]]
  ├── Japan Post Insurance Co., Ltd.（上市 7181）── 基于简易生命保险 [[kampo-life]]
  ├── Japan Post Capital Co., Ltd.（100%）── 集团 CVC / 出资功能
  ├── JP Biz Mail Co., Ltd. ── DM・mailing BPO
  └── （旧）Rakuten Group 出资（2021-03  约 1,500 億円 → 2024-03  基本出售）
```

## 2. 业务板块图

| 板块 | 主要运营者 | 特征 |
|---|---|---|
| 邮政 / 物流 | Japan Post | 国内 universal service 义务，约 24,000  邮局网络 |
| EC / 配送 | Japan Post（Yu-Pack） | 承接 Amazon、Rakuten、Mercari 等宅配 BPO |
| 零售银行 | [[yucho]]（上市 7182） | 存款规模国内最大级，通过邮局窗口进行账户销售 |
| 生命保险 | [[kampo-life]]（上市 7181） | 基于简易生命保险，按保费收入为大型 |
| 不动产 | Japan Post Real Estate（含 JP Tower） | 大手町、邮局旧址开发 |
| 集团 CVC | Japan Post Capital | startup / logistics tech 出资 |
| senior / 地域 | Japan Post “Mimamori Service”等 | 将邮局网络平台化为护理 / 自治体协作基础 |

### 集团统合协同

- **邮局网络 × 金融**：将全国约 24,000 局 个网点作为 [[yucho]] / [[kampo-life]] 的销售窗口。窗口手续费（金融 2 社 → Japan Post）是集团内部转移的主轴 ^[extracted]
- **物流 × EC**：Japan Post 的 last-mile 网络与 EC 平台联动。旧 Rakuten Group 合作即位于这一语境 ^[extracted]

### 合作与收购战略

- **Rakuten Group 资本业务合作**（2021-03-12）：日本邮政认购 Rakuten Group（4755）第三方增资，出资约 1,500 億円（出资比例 8.32%，第 4 位 大股东），在物流、移动、金融、EC 领域合作。之后因 Rakuten Mobile 业务低迷导致 Rakuten G 股价下跌，日本邮政在 2023 年 4〜6  月期计入约 800〜850 億円 的特别损失（减值）
- **金融 2 社 股份的阶段处置**：根据民营化法，日本邮政有阶段性处置 [[yucho]] / [[kampo-life]] 股份的目标。另一方面，由于邮局网络与金融销售不可分，完全分离存在政治阻力 ^[extracted]
- **政府持股追加出售**：作为复兴财源和国债偿还财源，政府持股出售已多次实施

### 海外战略

- 旧 Toll Holdings（澳洲大型物流公司，2015-05  约 6,200 億円 收购）→ **2021-04  出售**（计入巨额 goodwill 减值后）。海外物流 M&A 实质进入撤退模式 ^[extracted]
- [[yucho]] / [[kampo-life]] 在运用面通过海外债券和 alternatives 扩大全球 exposure

## 4. 监管与政策

- **主管**：总务省（邮政行政 / universal service）+ 金融厅（[[yucho]] 银行业 / [[kampo-life]] 保险业）
- **法律依据**：邮政民营化法 / 日本邮政株式会社法 / 日本邮便株式会社法 / Japan Post Bank 与 Japan Post Insurance 各自依据法
- **政府持有义务**：日本邮政股份由财务大臣**当面持有超过 1/3 ** ^[extracted]
- **金融 2 社 的处置目标**：日本邮政分阶段处置 [[yucho]] / [[kampo-life]] 股份（时间和比例通过政令 / 内阁决定运用）
- **Universal service 义务**：Japan Post 有义务提供全国统一资费的邮政服务
- **近期政策议题**：
  - 2019-2020  [[kampo-life]] 不当销售事件 → governance / 销售体制改革
  - 2024 〜 邮政资费上调（25 g 84 円 → 110 円 等）与邮政事业赤字化
  - 2025 〜 数字邮政、电子私书箱、MyPost 构想
