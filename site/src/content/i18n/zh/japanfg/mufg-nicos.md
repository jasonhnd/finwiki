# 三菱UFJ NICOS (Mitsubishi UFJ NICOS)


## Wiki 路线

本条目位于 [[JapanFG/INDEX|JapanFG index]] 之下。可对照 [[JapanFG/smbc-card|三井住友カード (SMCC)]] 理解同业 / 对比背景，并参照 [[payments/INDEX|payments index]] 理解更广泛的系统 / 监管边界。

## TL;DR

三菱UFJ NICOS 是 [[JapanFG/mufg|MUFG]] 的卡片与信贩 operating company。其官方公司概要明示包括信用购买中介业务者登记编号，以及信用卡号码等处理合同缔结业务者登记编号，因此在理解 issuer / acquirer / consumer-credit 边界时很重要。

## 1. 法人 / 牌照边界

| 项目 | 解读 |
|---|---|
| 法人实体 | 三菱UFJニコス株式会社 |
| 集团 | [[JapanFG/mufg|MUFG]] |
| 主要业务通道 | 信用卡、卡贷款、加盟店服务、分期信用 |
| 监管路径 | 《分期付款销售法》登记，以及卡号加盟店合同缔结业务登记。 |

## 2. 业务角色

- MUFG 面向零售客户和加盟店的支付触点，包括发卡和加盟店收单。
- 日本卡品牌和分期信用业务的历史整合点。
- 可与 [[JapanFG/smbc-card|SMBC Card]]、[[JapanFG/rakuten-card|Rakuten Card]]、[[JapanFG/jcb|JCB]] 对照比较。

## 3. 为什么需要独立页面

该公司不只是 MUFG 旗下的一个品牌。它是受监管的运营公司，拥有自己的登记编号、客户合同、加盟店规则，以及信用风险 / 欺诈控制流程。
