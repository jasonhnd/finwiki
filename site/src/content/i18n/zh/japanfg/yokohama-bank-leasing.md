# 横浜银 Lease (Yokohama Bank Leasing)

## Wiki 路径

本条目位于 [[JapanFG/INDEX|JapanFG index]] 之下，是 [[concordia-fg|横浜フィナンシャルグループ (旧コンコルディア FG)]] 的 FG 子公司租赁运营公司锚点。应与母运营银行 [[yokohama-bank|横浜銀行]] 和兄弟子公司 [[higashi-nippon-bank|東日本銀行]] 对照阅读。同业比较包括 [[mebuki-lease|めぶきリース]]（北关东地银系）以及大型租赁公司 [[mitsubishi-hc-capital|三菱 HC キャピタル]] / [[mizuho-lease|みずほリース]] / [[tokyo-century|東京センチュリー]]。更广泛制度路径参见 [[banking/regional-bank-consolidation-pattern|regional-bank consolidation pattern]] 和 [[banking/INDEX|banking index]]。

## 摘要

株式会社横浜銀リース是 [[yokohama-bank|横浜銀行]]（现 [[concordia-fg|横浜フィナンシャルグループ]] 子公司）的合并子公司，是以神奈川县和首都圈为地盘的地银系租赁公司，提供设备租赁、融资租赁、经营租赁、自治体租赁、医疗租赁等服务。在地银系租赁中属于上位规模，背靠 [[concordia-fg]]（旧 CFG，2025-10  改称）这一最大级地银 FG 的存款客户基础。^[extracted]

### 横浜银 Lease 设立

- 作为 [[yokohama-bank|横浜銀行]] 系列租赁公司，主营面向神奈川县内中坚企业的设备租赁
- 背靠横浜银行在县内 No.1  的压倒性份额，在租赁业务中也位于神奈川县内份额上位
- 战后地银系租赁的典型：以设备租赁形态支持中小企业设备投资，而不是银行贷款

### CFG -> YFG 商号变更 (2025-10-01)

- 母控股公司 [[concordia-fg]] 通过 2016-04  [[yokohama-bank]] + [[higashi-nippon-bank]] 共同股份转移成立
- **2025-10-01**：旧 Concordia FG 改称为 **Yokohama Financial Group**
- 横浜銀リース 本身不受商号变更影响，继续作为 [[yokohama-bank]] 合并子公司运营

### FG 内定位

- **银行租赁协作核心**：承接 [[yokohama-bank]] 的神奈川县内中坚企业主办行客户的租赁需求
- **设备投资支援**：补完银行贷款 + 应对表外处理需求
- **县内自治体 / 公共部门协作**：神奈川县、横滨市、川崎市等自治体租赁案件
- **横浜银行品牌下的租赁展开**：与 [[concordia-fg|YFG]] “横浜”品牌回归（2025-10）保持一致

### 竞争结构

- **大型租赁**：[[mitsubishi-hc-capital|三菱 HC キャピタル]] / [[smfl|SMFL]] / [[mizuho-lease|みずほリース]] / [[fuyo-lease|芙蓉リース]] / [[tokyo-century|東京センチュリー]] / [[ricoh-leasing|リコーリース]] — 在大型、全国、复杂案件中占优
- **地银系租赁**：[[mebuki-lease|めぶきリース]]（北关东）/ [[hokkaido-lease|北海道リース]] / 千叶兴业租赁等 — 同型模式
- **制造商系租赁**：[[toyota-finance|トヨタファイナンス]] / [[nec-leasing|NEC キャピタルソリューション]] — 产品联动
- **县内竞争**：横浜银行系、神奈川县内信用金库租赁、大型租赁支店

### 强项和弱项

- 强项：[[yokohama-bank]] 存款超过 15 兆円（单体）的中坚企业 main bank 网络 + 首都圈 GDP 规模 + 神奈川县内 No.1  知名度

## 7. 监管 / 政策

- **主管**：金融厅（FSA）（租赁业单独不需要银行业牌照，贷款业登记则依对应领域而定）
- **监管论点**：
  - 支持租赁会计基准变更（IFRS16  / 日本基准）
  - 活用中小企业租赁税制（中小企业投资促进税制）
  - 环境租赁（可再生能源设备租赁）的脱碳税制应对
  - 反社会势力检查 / 反洗钱对策（[[concordia-fg]] 集团共同实施，并与 2018-08  东日本银行业务改善命令后的内部控制强化保持一致）
