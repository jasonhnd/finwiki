# MS&AD 保险集团控股

## Wiki route

本条目位于 [[JapanFG/INDEX|JapanFG index]] 之下。可与 [[JapanFG/sompo|SOMPO ホールディングス (Sompo Holdings)]] 对照阅读同业 / 对比语境，并与 [[banking/INDEX|banking index]] 阅读更广泛的系统 / 监管边界。

## TL;DR

日本国内 3  大损保之一（与 [[JapanFG/tokio-marine|東京海上 HD]] / [[JapanFG/sompo|SOMPO HD]] 并列）。通过 **2008-04  [[JapanFG/mitsui-sumitomo-insurance|三井住友海上]] + Aioi 损保 + Nissay Dowa 损保经营统合**而成立。公司名来自构成 3 社 家公司的首字母“**M**itsui Sumitomo + **A**ioi + **D**owa”。国内曾采取维持 **3  品牌体制**（三井住友海上、Aioi Nissay Dowa、三井 Direct）的特殊战略，但随着 **2027-04  三井住友海上 + Aioi Nissay Dowa 损保计划合并**，3  品牌体制将走向终结。海外方面，**2016 年 收购 MS Amlin（约 3600 億円）**使其正式进入英国 Lloyd's 市场。通过“Aioi”谱系，丰田与日本生命的资本系列仍然保留，在汽车保险和银行窗口销售寿险方面较强。**FY2024  纯利润 6,916 億円（+87% YoY，创历史新高）**。证券代码 **东证 PRIME 8725**。

## 1. 公司概要

**正式名称**：MS&AD Insurance Group Holdings, Inc.
**英文名**：MS&AD Insurance Group Holdings, Inc.
**证券代码**：东证 PRIME 8725

**设立**：2008-04（从控股公司“Millea Holdings”的改组发展而来）
**成立经过**：2010-04-01 三井住友海上集团控股 + Aioi 损害保险 + Nissay Dowa 损害保险经营统合，确立当前体制
**总部**：东京都千代田区
**机构设计**：设置监事会的公司

- **Aioi Nissay Dowa 损保**：2010-10-01 Aioi + Nissay Dowa 合并

### 国内 3  品牌战略的特殊性 -> 2027  终结

MS&AD 在国内损保中采取了 **3  品牌维持**（三井住友海上 / Aioi Nissay Dowa / 三井 Direct）的体制，与其他大手不同。一般合并后通常会整合品牌，但 MS&AD 优先避免损害**渠道基础**（旧三井住友 = 大企业代理店，旧 Aioi = 丰田经销商，旧 Nissay Dowa = 日本生命系）。

不过，**2027-04  三井住友海上 + Aioi Nissay Dowa 损保计划合并**已经公布，长期维持的 3  品牌战略将结束。背景包括：
- Bigmotor 案件后，代理店治理强化成本上升
- 企业保险 cartel 问题 -> 2025-03  三井住友海上收到金融厅业务改善命令
- 追求整合协同的压力（压缩重复成本、统一 IT 基础设施）
- 重新评估丰田 / 日本生命系渠道维持成本与效率化之间的取舍

### 与丰田、日本生命的资本系列

- **经 Aioi 延续的丰田系**：旧大东京火灾（含千代田火灾）属于丰田系列，保持通过丰田经销商销售汽车保险的渠道
- **经 Nissay Dowa 延续的日本生命系**：旧 Nissay 损害保险属于日本生命系，形成银行窗口销售和营业职员渠道基础
- **Telematics 保险**：与 Toyota Connected Car 数据联动的汽车保险（由 Aioi 主导）

### 海外战略

- **2016  收购 MS Amlin**（约 3600 億円）是转折点。集团正式持有英国 Lloyd's syndicate，取得特殊保险和再保险组合
- **集中亚洲**：通过 Asia Capital Re / Bohai / Newa / BNP Cardif Vietnam 多面展开再保险、损保、寿险
- **美洲**：MS Reinsurance（旧 Trans-Re）

### 可持续性 / 气候

- **碳中和举措**：在投资和承保两方面推进 transition 战略
- **TCFD / TNFD**：披露气候、自然资本风险
- **灾害保险**：国内自然灾害频发 -> 火灾保险费率修订（行业共通课题）

## 4. 监管 / 政策

- **主管**：金融厅（FSA）
- **保险业法**：损保原保险、再保险、寿险的偿付能力充足率监管
- **海外监管**：
  - **PRA / FCA（英国）**：MS Amlin（Lloyd's syndicate）
  - **NAIC / 州监管（美国）**：MS Reinsurance
  - **MAS（新加坡）**：Asia Capital Re
- **近期政策论点**：
  - 汽车保险费率修订（事故率、修理费上涨）
  - 火灾保险费率修订（自然灾害频发）
  - **Bigmotor 问题**（2023 起）后的损保代理店治理强化（行业横断，含损保 4 社 cartel 疑义）^[公開情報ベース・複数社案件]
  - **2025-03  三井住友海上金融厅业务改善命令**（客户信息管理问题，含企业保险 cartel 相关事项）-- 触发整合判断的重要因素
  - **政策保有股缩减**（行业共通课题、损保 4 社 申合）-- 含益释放和出售收益计入时点
