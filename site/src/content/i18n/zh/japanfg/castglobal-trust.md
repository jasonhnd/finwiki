# CastGlobal 信托 / CastGlobal 信托

维基路线：[[JapanFG/trust-companies-japan-index]] -> [[JapanFG/castglobal-trust]] -> QZX003QZX

## 概述

CastGlobal Trust是一家与房地产、托管、跨境投资支持相关的管理型信托公司。在 FinWiki 中，这是一条专业服务信任路径，除了 [[JapanFG/kyodo-trust]] 和 QZX002QZX 之外也很有用。

## 实体边界

本页面涵盖 Cast Global Trust Co., Ltd.，而非整个 CastGlobal 专业服务网络。有关法律、税务或国际合作伙伴服务的声明应与上市公司材料相关。

## 公共环境

CastGlobal Trust 很重要，因为它展示了信托公司如何在专业服务、托管和房地产投资之间发挥作用。该路线还有助于将类似托管的信托服务与 QZX001QZX 和 PSP 结算风险页面分开。
