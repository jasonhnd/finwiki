# 三井物产 (Mitsui & Co., Ltd.)

## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]] as a **sōgō shōsha parent holding-company**. Read it against [[JapanFG/mitsubishi-corp|三菱商事]]（争夺净利润首位的最大竞争对手）and [[JapanFG/itochu-corp|伊藤忠商事]]（在 5  大商社中偏非资源型的对照）for peer / contrast context. For the financial-services adjacency that anchors this entry on JapanFG, see [[JapanFG/mitsui-bussan-financial-services|三井物産フィナンシャルサービス]]（面向集团内部的 CMS）。[[JapanFG/smfl|SMFL (住友三井オートサービス系の総合リース)]] is **不准确** — 三井物产的主要租赁子公司是 [[JapanFG/ja-mitsui-leasing|JA三井リース]]（JA 全农 + 三井物产 50/50  系列；旧三井租赁 + JA 租赁的合并体），这与其持有 [[JapanFG/smfl|SMFL]]（SMFG / 住友商事系，SMFL = 50% SMFG + 50% 住友商事）的权益不同；此外还有对 [[JapanFG/sumitomo-mitsui-auto-service|住友三井オートサービス]] 的一定持股。System frame: [[finance/INDEX|finance index]] 和 [[finance/japan-listed-financial-groups-investable-universe|Japan 上場金融 G 投資ユニバース]]。

### 沿革亮点 ^[extracted]

| 年 | 事项 |
|---|---|
| 1876 | 旧三井物产会社 (Mitsui Bussan Kaisha) 设立 — 日本第一家综合商社 |
| 1947 | GHQ 财阀解体导致旧三井物产解体（分拆为超过 200 社 家小公司） |
| 1959 | 通过再整合设立现三井物产 |
| 1970s〜 | 进入 LNG 上游（文莱、澳大利亚 NWS） |
| 1990s〜 | 通过 Valepar、IBVN 持股参与巴西 Vale 铁矿石项目 |
| 2010s | 参与美国 Cameron LNG、莫桑比克 Area1  LNG |
| 2018 | 成为 IHH Healthcare 主要股东（亚洲医院连锁） |
| 2020s | 净利润达到 1 兆円 级，股东回报加速 |
| 2024 | Berkshire Hathaway 增持商社股报道后估值重估 |

此外，**三井集团的 financial-services 生态系统** 包括：
- [[JapanFG/smfg|SMFG]] / 三井住友银行（三井集团主要大型银行，cross-shareholdings）

三井物产本身没有 banking license，但通过集团系列锚定在 **“商社 ↔ 大型银行 ↔ 信托 ↔ 保险 ↔ 租赁” 的全线金融生态系统** 中。

### 5  大商社中的身份

- **非资源扩张**：2010s 以后，积极在 **医疗健康（IHH）、移动出行（Penske Automotive）、基础设施（海水淡化）** 等非资源领域推进 global M&A
- **股东回报**：累进分红 + 大规模回购，关注 PBR 超过 1.0x

### 竞争

- **日本国内**：5  大商社（[[JapanFG/mitsubishi-corp|三菱]] / [[JapanFG/itochu-corp|伊藤忠]] / [[JapanFG/sumitomo-corp|住友]] / [[JapanFG/marubeni-corp|丸紅]]）
- **海外资源**：BHP、Vale（合作伙伴兼竞争者）、Glencore
- **海外移动出行**：各国汽车流通公司、美国 dealership groups
