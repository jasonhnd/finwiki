# 山口金融集团（YMFG）

## Wiki route

本条目位于 [[JapanFG/INDEX|JapanFG index]] 之下。可与 [[JapanFG/fukuoka-fg|ふくおか FG (FFG)]] 对照阅读以理解同业 / 对比语境，也可与 [[banking/INDEX|banking index]] 一起阅读以把握更大的系统和监管边界。

## TL;DR

YMFG 是以中国地方西部 + 北九州为地盘的广域地方银行集团。2006-10-02 山口银行通过单独股份转移控股公司化而成立，之后加入 [[momiji-bank|もみじ銀行]]、[[kitakyushu-bank|北九州銀行]]，形成 **3 行体制**（山口银行 / 红叶银行 / 北九州银行），这是单一地银起源 FG 中少见的结构。它代表了山阳・关门海峡经济圈（山口 + 广岛 + 北九州）的广域联合，以及与横滨银行（Concordia）的域外合作 2  轴。也是跨县地银整合和“3 行 3 品牌”模式的代表例。

## 1. 公司概要

**正式名**：株式会社山口フィナンシャルグループ
**英文名**：Yamaguchi Financial Group, Inc.
**证券代码**：东证 PRIME 8418 / 福证 8418
**设立**：2006-10-02（山口银行通过单独股份转移控股公司化）
**总部**：山口县下关市竹崎町 4-2-36 / 名古屋市中区荣（在东京、大阪、名古屋设有据点）

### 主要子公司和持股

```

  ├── 山口银行（100%）── 山口县地盘、集团母体、总部下关
  │     └── 山口县内份额压倒性，也在关门、福冈北部设有门店
  ├── 红叶银行（100%）── [[momiji-bank]] / 广岛县地盘，2001 进入山口银行系列 → 2006 进入 FG
  │     └── 旧广岛综合银行 + 濑户内银行整合形成的第二地银
  ├── 北九州银行（100%）── [[kitakyushu-bank]] / 2011-10 设立，北九州市地盘（山口银行北九州支店分离独立）
  │     └── 以独立品牌经营福冈县北九州地区
  ├── YM Career（100%）── 人才介绍和派遣

```

### 合并历史和前身

- 旧 **山口银行**：1944  设立。前身包括旧山口无尽 / 大岛银行等，通过战时管制下县内银行整合成立。总部位于下关，覆盖山口县全域
- 旧 **广岛综合银行**：旧广岛相互银行（无尽系）转为普通银行后，作为第二地银立足广岛县
- 旧 **濑户内银行**：旧吴相互银行系，濑户内沿岸地盘的第二地银
- **红叶银行**：2001  由广岛综合银行 + 濑户内银行合并，随后获得山口银行经营支援并进入其系列
- **北九州银行**：2011-10  将山口银行北九州支店分离，通过新设分割诞生（当时是 47  年来首次新设地银）

> 整合背景：中国地方西部的地银面临人口减少和低利率利差压力 → 推动跨县广域联合战略。集团将广岛第二地银救济（红叶银行）与进入福冈县北九州地区的独立品牌（北九州银行）结合，构建了单一地银起源 FG 中罕见的“3 行 3 品牌”结构。

### “YMFG ZONE Planning”3 行体制

- 维持山口银行 / 红叶银行 / 北九州银行的 **3 品牌并立**。战略不是合并集约，而是保留各地区客户接点品牌
- 是跨县地银整合的代表例，在福冈县内与 [[fukuoka-fg]]（[[fukuoka-bank|福岡銀行]]・[[kumamoto-bank|熊本銀行]]・[[18-shinwa-bank|十八親和銀行]]・Minna Bank）形成地域竞争关系
- 由 1 个 FG 单独持有 3 行 的结构，与其他许多 2 行体制或县内单一型地银 FG 相比十分少见

### 山阳・关门海峡经济圈

- 将山口县（下关）+ 广岛县 + 福冈县北九州视为 1 个经济圈的地理战略
- 将隔关门海峡相望的下关 / 北九州定位为“同一经济圈”，总部设在下关，同时以北九州品牌进入福冈县
- 在中国 / 九州边界地区两侧都拥有桥头堡，是独特的位置

### 与横滨银行（Concordia FG）的业务合作提携（2021起）

- 域外合作的代表例。地银之间不是经营整合，而是停留在 **“业务合作”**，维持彼此独立性，同时在法人客户介绍、ATM 相互开放等方面合作
- 作为地银重组的选择之一，展示了“不合并、通过提携生存”的模式

### B2C 品牌

- “山口银行”“红叶银行”“北九州银行”3 个品牌都采用地区名 + 银行的直接地域密着型身份
- 核心战略不是统一集团品牌，而是维持 3 行 3 品牌

## 4. 监管和政策

- **主管**：金融厅（FSA）/ 中国财务局
- **地银监管**：银行法
- **地银重组论**：属于金融厅地银重组引导政策（2017起）的对象，但 YMFG 已经整合完成，与第 4  megabank 构想（SBI 系）等保持距离
- **近期政策论点**：
  - 2024 起，日本银行政策利率正常化 → 地银贷款利差改善红利（YMFG 也预计受益）
  - 地方人口减少 → 持续跨县广域联合战略
  - 2021 起，与横滨银行的域外业务合作 → 同型模式可能扩散
