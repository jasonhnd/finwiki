# 三井物产 Financial Services (Mitsui & Co. Financial Services)

## 2. 业务分部地图

| 分部 | 内容 |
|---|---|
| **CMS（现金管理服务）** | 三井物产集团各公司的集中结算、资金池和集团内转账 |
| **一揽子保理** | 集团各公司的应收债权购买 / 支付代理 |
| **集团金融** | 向集团内公司提供 short-term / mid-term 贷款 |
| **外汇和利率风险对冲中介** | 面向集团各公司的衍生品中介 |
| **海外汇款服务** | 集团各公司的海外汇款和回收 |
| **商品交易附带金融** | 与三井物产本体铁矿石、原油、LNG、铜等商品交易相关的授信和对冲中介 |

### 三井物产的业务结构和金融功能

尤其是 **资源和能源** 分部，以 LNG、原油、铁矿石等长期合同 + 大型授信案件为中心，对集团内金融功能的使用度较高。

## 4. 监管和政策

- **主管**：金融厅（贷金业 / 外汇业务相关）
- **监管论点**：
  - 集团内金融交易的 **转让定价税制**（与海外子公司的交易价格）
  - 集团保理与 **分期销售法 / 贷金业法** 的边界整理
  - 合并会计上的 **内部交易抵销**
- **近期政策论点**:
  - 2024 起 [[JapanFG/boj-monetary-policy|日銀政策金利正常化]] -> 重新评估集团内利率赋予
  - 2025 起 加强海外汇款 / 跨境资金移动的 AML/CFT
