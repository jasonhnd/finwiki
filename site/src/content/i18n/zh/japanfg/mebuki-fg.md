# Mebuki FG（めぶき金融集团）

## Wiki 路径

本条目位于 [[JapanFG/INDEX|JapanFG index]] 之下。可结合 [[JapanFG/fukuoka-fg|ふくおか FG (FFG)]] 阅读，以获得同业 / 对比语境；也可结合 [[banking/INDEX|banking index]] 阅读，以理解更大的系统 / 监管边界。

## TL;DR

Mebuki FG 是以北关东 2  县（茨城 + 栃木）为地盘的大型地方银行 FG，按存款规模约位于地方银行 FG 第 2-3 位 档。它由 2016-10-01 **常阳银行**（茨城地盘）+ **足利控股**（栃木地盘，旗下有足利银行）共同股份转移而成立。其 3  条主轴是 **“mega-regional bank group” 志向 + 首都圈扩张 + 数字化 / 地方创生**。足利银行曾经历 **2003 年 破产 -> 临时国有化 -> 2008 年 民间转让** 的剧烈历史，之后才走向整合，这一点不同于其他地方银行 FG。在相邻首都圈市场，它与 [[concordia-fg]]（横滨银行 + 东日本银行）/ [[JapanFG/chiba-bank|chiba-bank-fg]] 竞争并探索合作。^[extracted]

## 1. 公司概要

**正式名称**：株式会社めぶきフィナンシャルグループ ^[extracted]
**英文名称**：Mebuki Financial Group, Inc.
**证券代码**：东证 PRIME 7167 ^[extracted]
**设立**：2016-10-01（常阳银行 + 足利控股通过共同股份转移设立控股公司）^[extracted]
**总部**：东京都中央区八重洲 / 茨城县水户市・栃木县宇都宫市（2  总部制）^[extracted]

### 主要子公司 / 持股

```
Mebuki FG（控股公司・上市 7167）
  ├── 常阳银行（100%）── 茨城县地盘核心
  │     └── 由 1935  常磐银行 + 五十银行合并设立，在茨城县内份额压倒性
  ├── 足利银行（100%）── 栃木县地盘核心
  │     └── 1895  创业，2003  破产・临时国有化 -> 2008  民间转让经历
  ├── Mebuki 证券（100%）── 集团证券（原常阳证券）
  ├── Mebuki Lease（100%）── 集团租赁
  ├── Mebuki Card（100%）── 集团信用卡
  └── 常阳 Capital / 足利 Capital 等 ── 地域投资・咨询业务
```

### 合并经过与前身

- 原 **常阳银行**：1935-07-30 由 **常磐银行 + 五十银行** 合并设立。总部位于茨城县水户市，是县内份额压倒性的第一地方银行。
- 原 **足利银行**：1895（明治 28）创业，总部位于栃木县宇都宫市。它是栃木县份额压倒性的第一地方银行，并在战后高速增长期扩张为北关东有力地方银行。
- **足利银行破产**（2003-11-29）：适用金融再生法 102  条 -> **临时国有化**。这是地方银行破产中的最大规模案例，存款保险机构取得全部股份并将其纳入特别危机管理银行。
- **足利 HD 成立**（2008-07）：野村控股系投资基金（Nomura Principal Finance）+ Next Capital Partners 等参与的 **再生 SPC** 以 1200 億円 规模取得股份，并重组至足利控股旗下。
- **足利 HD 上市**（2013-12-19）：东证一部上市，作为地方银行再上市案例。
- **Mebuki FG 成立**（2016-10-01）：常阳 + 足利 HD 共同股份转移，形成北关东广域 FG。

### “Mega-regional bank group” 志向

- 自整合成立时起即将 **“mega-regional bank group”** 作为口号，志在超越传统地方银行框架，追求广域 + 规模。^[extracted]
- 按存款规模位于地方银行 FG 第 2-3 位 档，是与 [[concordia-fg]] / [[fukuoka-fg]] / [[JapanFG/chiba-bank|chiba-bank-fg]] 并列的最上位集团之一。
- 规模扩张依托 2  条轴线：茨城 + 栃木县内 main bank 地位，以及向首都圈扩张。

### 首都圈扩张战略

- **县外网点**：向东京、埼玉、千叶、神奈川等地扩张。重点是开拓中坚企业 main bank 关系与富裕层 retail。
- 地理依据是经济圈连接：茨城 -> 东京东北部，栃木 -> 埼玉北部。

### 数字化与地方创生

- **DX 推进**：中期经营计划强调强化数字渠道、优化门店、提高总部业务效率。
- **地方创生**：包括自治体合作、事业承继、地域企业 M&A 中介、观光振兴等。

### 在竞争与地方银行重组中的位置

- **竞争对手**：[[concordia-fg]]（横滨 + 东日本）/ [[JapanFG/chiba-bank|chiba-bank-fg]]（作为首都圈邻接广域 FG 的直接竞争者）/ [[fukuoka-fg]]（广域 FG 规模竞争）。
- 相对于 **megabank**（[[mufg]] / [[smfg]] / [[mizuho-fg]]），它通过地方银行整合 + 县内份额进行差异化。
- 与 [[sbi-hd]] 主导的 **SBI 第 4  mega 构想** 保持距离，走自身广域路线。

## 4. 监管与政策

- **主管**：金融厅（FSA）+ 关东财务局。
- **控股公司监管**：银行法第 52  条之 17
- **足利银行国有化处理历史**：2003-11  适用金融再生法 102  条 -> 存款保险机构取得全部股份 -> 2008-07  民间转让。作为地方银行破产处理的典型且最大案例，是金融行政上的参照例。
- **近期政策论点**：
  - 2024 起：日本银行政策利率正常化 -> 地方银行利差改善，受益程度大于大型银行。
  - 2025 起：地方银行重组加速（人口减少 + 数字投资负担）-> 探索北关东广域联盟的下一阶段。
  - 与自治体合作、地方创生政策连接，体现地方银行的公共性角色。
