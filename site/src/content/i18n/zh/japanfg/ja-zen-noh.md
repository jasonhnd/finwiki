# JA全农（全国农业协同组合联合会 / National Federation of Agricultural Cooperative Associations）

## Wiki 路线

本条目位于 [[JapanFG/INDEX|JapanFG index]] 之下，是 **JA Group 的经济事业联合会**，与 [[JapanFG/norinchukin|農林中央金庫 (信用)]] / [[JapanFG/ja-kyosairen|JA 共済連 (共済)]] / [[JapanFG/ja-zenchu|JA 全中 (代表)]] 并列。可与 [[banking/ja-bank-system-japan|JA Bank system]] 一并阅读以理解信用侧邻接关系，并参照 [[banking/cooperative-banking-japan|cooperative banking domain]] 理解系统边界。

## TL;DR

**JA 集团 4  大全国联合会中的经济事业核心 = 全国规模的采购 + 销售事业联合会**。1972  设立（全販連 + 全購連 合并），是以《农业协同组合法》为依据的 **联合会**。业务内容包括 **(1) 采购 = 饲料、肥料、农药、农业机械、燃料、生活物资的采购与供应**，**(2) 销售 = 大米、小麦、园艺产品、畜产品的集货与流通**，以及 **(3) 加工、物流、直营店（A-Coop 等）**。**年处理额为数万亿日元规模**，是日本最大级别的农产品流通组织。[[JapanFG/norinchukin|農林中金]] 负责金融，JA 全农负责**“物”的流动**，两者是互补关系。全国 JA -> JA 全农 -> 市场 / 大型流通 -> 消费者 的流向，以及**反向采购流（制造商 -> JA 全农 -> 全国 JA -> 农业者）**，是两大业务线。**自身是不经营存款或保险的非金融事业体**，但与 JA Bank 共享同一套全国 JA 497  组合客户基础。^[extracted]

| 2017  | 促进对美国农产品出口（设立大米出口公司等） |

#### 采购事业（Buying / Supply）^[extracted]

| 领域 | 内容 |
|---|---|
| **生产资材** | 饲料、肥料、农药、农业机械、种苗 |
| **燃料** | 通过 A-Coop 系列 SS、JA-SS 供应汽油、轻油、LP gas |
| **生活物资** | 通过 A-Coop（类似生协的店铺）供应食品和日用品 |
| **其他** | 汽车、住宅资材、农业温室资材等 |

#### 销售事业（Selling / Marketing）^[extracted]

| 领域 | 内容 |
|---|---|
| **米谷** | 从全国 JA 集货大米，政府投标 + 民间流通 |
| **园艺** | 蔬菜、水果的集出货与市场流通 |
| **畜产** | 牛、猪、鸡的集出货 + 食肉加工（全农チキンフーズ等） |
| **海外** | 通过美国子公司（CGB 等）开展 cargo / shipping 业务 |

### 金融互补关系

- **物的流动** = JA 全农，**钱的流动** = [[JapanFG/norinchukin|農林中央金庫]] + [[JapanFG/ja-shinnoren|信連]] + JA Bank
- 农业者的采购款、销售款通过 **JA 存款账户** 结算 -> 加厚 JA Bank 的存款基础 ^[extracted]
- 大型商社（三井物産、住友商事、伊藤忠商事等）既是 JA 全农的竞争对手，也是交易对象

### 自我改革与外部压力

- **2014  规制改革会议**：强烈要求全农“股份公司化”“事业统废合”“降低生产资材成本”^[extracted]
- **2015〜 自我改革年度计划**：降低饲料 / 肥料成本 10%、流通效率化、强化海外出口 ^[extracted]
- 股份公司化最终**搁置** — 维持联合会形态，但持续推进事业效率化

### 海外展开

- **美国子公司 CGB**（Consolidated Grain and Barge）：美国大型谷物集货与物流公司
- **促进海外出口**：大米、牛肉、青果的海外销售（美国、亚洲、EU 市场）

### 竞争结构

- **大型商社**：与三井物産、住友商事、伊藤忠商事的生产资材 + 食品流通业务竞争
- **大型食品制造商**：在米谷流通中与大型大米批发商（Pearl Rice 系、神明等）竞争
- **大型流通企业**：直营 A-Coop 与地方超市、便利店（Seven、Aeon）竞争

## 4. 监督 / 监管

- **主管**：农林水产省（联合会监督）+ 公正交易委员会（反垄断法 — 处理集中度论点）
- **法律依据**：《农业协同组合法》（联合会条款）+ 对联合会成员加入的规制
- **近期政策论点**：
  - **降低生产资材成本**（改革会议持续论点）
  - **促进农产品出口**
  - 在 **粮食安全保障计划（2026〜）** 中重新定义 JA 全农的角色
  - 应对 **数字农业（smart agriculture）**
