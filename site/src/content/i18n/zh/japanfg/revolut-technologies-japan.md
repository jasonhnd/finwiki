# REVOLUT TECHNOLOGIES JAPAN

## Wiki 路径

本条目位于 [[JapanFG/INDEX|JapanFG index]] 之下。可与 [[JapanFG/paypal-pte-ltd-japan|PayPal Pte. Ltd. Japan service boundary]] 对照阅读，用于同业 / 对比语境；也可与 [[payments/INDEX|payments index]] 一并理解更广义的系统 / 监管边界。

## TL;DR

REVOLUT TECHNOLOGIES JAPAN株式会社是 Revolut 的日本运营实体。官方日本网站明示其作为第二种资金移动业者及电子支付等代理业者的登记，并提供卡、外汇兑换、国内外汇款、家计管理等服务。

## 2. 业务角色

- 在资金移动业牌照下，将全球 neobank 风格 app 带入日本。
- 在跨境 UX 上最直接与 [[JapanFG/wise-payments-japan|Wise]]、[[JapanFG/airwallex-japan|Airwallex]] 竞争，同时也与 [[JapanFG/paypay|PayPay]] 等预付 / 钱包产品重叠。
- 对追踪全球 fintech 模型如何适应日本监管很重要。

## 3. 独立页面的意义

Revolut 的全球品牌不等同于其受日本监管的实体。本页记录日本牌照边界和产品限制。
