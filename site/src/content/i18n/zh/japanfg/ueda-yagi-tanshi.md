# 上田八木短资株式会社 (Ueda Yagi Tanshi)

## Wiki 路线

本条目位于 [[JapanFG/INDEX|JapanFG index]] 之下。可与 [[JapanFG/central-tanshi|セントラル短資株式会社 (Central Tanshi)]] 对照阅读，以把握同业 / 对比语境；也可与 [[banking/INDEX|banking index]] 一起阅读，以确认更广的制度 / 监管边界。

## TL;DR

上田八木短资是日本 3  大短资公司之一的独立系短资公司。它由 **2001-04  上田短资 + 八木短资合并**而成。核心业务是短期金融市场（call market、票据买卖、CD/CP）的中介，以及作为日本银行市场操作对手方。短资行业在 2009  雷曼危机后的超低利率环境中长期低迷，但随着 **2024-03  负利率解除**，行业 margin 改善预期开始上升。公司非上市、独立系，商业模式集中于与 megabank / regional bank 的交易。^[extracted]

## 1. 公司概要

**正式名**：上田八木短资株式会社
**英文名**：Ueda Yagi Tanshi Co., Ltd.
**设立**：2001-04（上田短资 + 八木短资合并成立）^[extracted]
**业态**：短资公司（日本独有的短期金融市场专门业态）
**上市**：非上市（独立系）
**行业位置**：日本 3  大短资公司之一（[[central-tanshi]] / 上田八木短资 / [[tokyo-tanshi]]）

### 前身 / 合并历史

- **1909** 上田短资创立（战前以来的老牌短资商）
- **1928** 八木商店设立（八木短资前身）
- **2001-04** 上田短资 + 八木短资 → **上田八木短资合并成立** ^[extracted]
- 行业重组的象征：短资行业在 2000 年 年代中逐步集约为 3  大体制

## 2. 业务 segment map

| 业务 | 内容 | 客户 |
|---|---|---|
| **Call market 中介** | 银行间无担保 / 有担保 call funds 中介 | Megabanks、regional banks、trusts、securities firms |
| **票据买卖市场** | 短期票据（商业票据、银行承兑票据）买卖中介 | 银行、大企业 |
| **CD / CP 买卖** | 可转让存款（CD）和 commercial paper（CP）买卖 | 银行、事业法人、机构投资者 |
| **日本银行对手方业务** | 日本银行市场操作（公开市场操作）的对手方 / 中介 | 日本银行 |
| **国债 / 地方债买卖** | 短期国债（T-Bill）、中长期国债、地方债买卖 | 银行、机构投资者 |
| **海外债券（美元 call 等）** | 美元计价短期资金市场（Eurodollar call 等）中介 | 日本银行海外据点、外国银行东京支店 |

### 商业模式

- **薄利多销型**：短期资金市场 bid-offer spread 和手续费是收益来源。
- **依赖日本银行政策利率**：政策利率水平和操作量会带动行业 margin。
- **Megabank / regional bank 集中**：客户基础以日本银行 sector 为中心。
- **面向 non-bank**：也向大型证券、信托、保险、农林系金融机构提供服务。

### 3  大短资公司

| 短资公司 | 沿革 | 特征 |
|---|---|---|
| **Central Tanshi** | Yamane Tanshi + Central Tanshi + Tanshi 合并 | 行业最大 |
| **Ueda Yagi Tanshi** | Ueda Tanshi + Yagi Tanshi（2001-04  合并） | 独立系，3  大之一 |
| **Tokyo Tanshi** | 战前以来的东京据点短资公司 | 3  大之一 |

### 行业缩小 cycle

- **〜2008**：有政策利率、市场活跃，即便在 3  大体制下也能确保收益
- **2009-2024**：雷曼后超低利率 → 零利率 / 负利率环境导致薄利化，行业缩小 ^[extracted]
- **2024-03〜**：负利率解除 → 行业 margin 改善预期

### 中期战略（2024  中计）

- **以日本银行政策利率正常化为机会**：应对 2024-03  负利率解除后的短期金融市场活性化
- **提供短期金融市场流动性**：强化核心业务
- **集中与 megabank / regional bank 的交易**：深耕核心客户

### 竞争关系

- **直接竞争**：[[central-tanshi]] / [[tokyo-tanshi]]（3  大短资公司之间相互竞争）
- **间接竞争**：大型证券公司（短期国债 OTC 市场）、银行间直接交易（call direct）
- **行业保护因素**：作为日本银行市场操作对手方的制度性位置

### 市场趋势

- **2024-03  负利率解除**：行业复活预期的最大因素
- **日本银行 QT（quantitative tightening）**：随着国债市场流动性恢复，短资业务有扩大空间

## 5. 监管 / 政策

- **主管**：金融厅（FSA）+ 日本银行（作为市场操作对手方的监督）
- **业法**：短资公司是银行法外的独立业态，作为短资业者定位
- **日本银行操作对手方要求**：需要满足一定的资本和体制标准
- **近期政策议题**：
  - 2024-03〜 日本银行政策利率正常化（负利率解除）
  - 2024-2026  日本银行 QT（缩减持有国债）
  - 2025-2026  短期金融市场数字化 / 分布式账本应用讨论
