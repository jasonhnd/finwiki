# 东京都职员信用组合 (Tokyo Metropolitan Government Employees Credit Cooperative)

## Wiki route

本条目位于 [[JapanFG/INDEX|JapanFG index]] 之下，是日本**最大的自治体职员（地方政府职员）shinkumi**，并在 [[banking/credit-cooperative-registry-japan|Japan credit cooperative registry]] 收录的 143-cooperative population 中，补充 [[JapanFG/keishicho-shinkumi|警視庁職員信用組合]]（警察）和 [[JapanFG/tokyo-shobo-shinkumi|東京消防信用組合]]（消防）等职业域 shinkumi 同业。

### 组合员资格 ^[extracted]

由于是职业域信组，组合员资格严格**限定于东京都职员**：

1. **基本要求**：东京都知事部局・公营企业（交通、水道、下水道、医院等）的职员（一般职）
2. **退休者**：从东京都退休的 OB / OG（通常可在一定期间内继续加入）
3. **配偶者 / 亲属**：组合员的配偶者、同一世带家属（依据章程）
4. **员外利用**：严格受限

### 在全信组连内的位置

- **[[JapanFG/zenshin-kumi|全信組連]]** 的关东 block 会员行
- 关东 49  信组中东京 19  信组的一翼
- 系统 ATM 联携和余资运用经由全信组连

### 主要业务 ^[extracted]

| 业务 | 内容 |
|---|---|
| **住宅贷款** | 面向组合员世带的住宅取得资金（主要业务） |
| **教育贷款** | 面向组合员子女的教育资金 |
| **汽车贷款** | 面向组合员的汽车购入资金 |
| **卡贷款** | 面向组合员的自由资金 |
| **冠婚葬祭融资** | 组合员家庭的庆吊费用 |
| **定期 / 流动性存款** | 面向组合员世带的存款 |
| **退休金运用** | 面向达到定年退休的组合员的资产运用咨询 |

## 6. 监督 / 监管

- **主管**：金融厅・关东财务局
- **法律依据**：中小企业等协同组合法 + 关于协同组合金融事业的法律（协金法）
- **存款保险**：[[JapanFG/dic|預金保険機構]] 一般金融机构框架
- **近期政策论点**：
  - **DX 对应**（智能手机 app、电子合同、WEB 完结贷款）
  - **与共济组合的功能协调**
  - **OB / OG 组合员管理**（高龄化对应）
