# 三菱UFJ银行（MUFG Bank）


## Wiki 路径

本条目位于 [[JapanFG/INDEX|JapanFG index]] 之下。可对照 [[JapanFG/sumitomo-mitsui-banking-corp|三井住友銀行 (SMBC)]] 阅读，以获得同业 / 对比语境；也可对照 [[banking/INDEX|banking index]] 阅读，以把握更广的体系 / 监管边界。

## TL;DR

三菱UFJ银行是 [[JapanFG/mufg|MUFG]] 的商业银行核心法人。在 FSA 的银行牌照列表中，它以“株式会社三菱UFJ銀行”列示，并拥有不同于 [[JapanFG/mitsubishi-ufj-trust-bank|三菱UFJ信託銀行]]、证券 HD、银行卡 / 消费金融子公司的银行牌照层。

## 2. 业务角色

- 承担 MUFG 集团的存款、贷款、支付结算、外汇和海外商业银行网络的核心法人。
- 与银行单体绑定的业务很多，包括亚洲 commercial bank 投资、法人结算、外国银行代理业务、信托契约代理等。
- [[JapanFG/mitsubishi-ufj-trust-bank|三菱UFJ信託銀行]] 承担 trust / asset administration / real estate / pension 侧业务，与 MUFG Bank 功能分离。

## 3. 为什么需要独立页面

[[JapanFG/mufg|MUFG]] 是上市控股公司，是观察决算、资本政策和 Morgan Stanley alliance 的单位。三菱UFJ银行是追踪银行法牌照、存款基础、法人结算、海外分行、外国银行代理、信托代理的单位。特别是在 stablecoin / Progmat / EPI 周边，需要区分信托银行与商业银行的边界。
