# 东京世纪 业务详情 (Tokyo Century Operating Profile)

## Wiki route

本条目位于 [[JapanFG/INDEX|JapanFG index]] 下，并补充 [[JapanFG/tokyo-century|東京センチュリー]] 的公司层面摘要。可与 [[JapanFG/orix-corp|オリックス]] / [[JapanFG/mitsubishi-hc-capital|三菱HCキャピタル]] / [[JapanFG/smfl|三井住友ファイナンス&リース]] / [[JapanFG/mizuho-lease|みずほリース]] 对照阅读，以理解综合租赁行业同业背景；[[JapanFG/mizuho-fg|みずほFG]] 用于一个母集团路线；[[banking/INDEX|banking index]] 用于更广泛的系统 / 监管边界。

## TL;DR

[[JapanFG/tokyo-century|東京センチュリー株式会社]]（8439  / 东证 PRIME）是日本 3  大综合租赁 / 金融公司之一（ORIX、Mitsubishi HC Capital、Tokyo Century），并拥有独特的**“Mizuho FG x ITOCHU Corporation” 2  轴主要股东结构**。本条目处理业务分部、收益结构、航空机金融、海外扩张和 SDGs 战略等 **operating-level depth**。其路径是：1969-12  原“Tokyo Leasing”设立 -> 2009-04 与“Century Leasing System”合并成立“Tokyo Century Leasing” -> 2016-10 更名为“Tokyo Century”。通过 **2019  将美国 Aviation Capital Group (ACG) 完全子公司化**，进入全球航空机租赁 Top 10 。**5  业务分部**：(a) 国内法人租赁 / 金融，(b) 国际业务（美洲・亚洲・欧洲），(c) **航空机**，(d) **环境基础设施**，(e) 汽车（原 Nippon Rent-A-Car Service / 汽车租赁）。

## 1. 母集团关系 - 2  轴股东结构

- **[[JapanFG/mizuho-fg|みずほフィナンシャルグループ]]**：原第一劝业银行系租赁公司（Tokyo Leasing）的谱系
- **ITOCHU Corporation**：商社系（Century Leasing System）的谱系
- 两家公司出资比例合计约 50% 级别。**Mizuho FG 侧提供金融支持和银行账户访问，ITOCHU 侧提供商社网络（汽车、产业设备、海外项目）和案件供给**。
- 这是“银行系租赁”和“商社系租赁”经营整合形成的少见资本结构。

## 2. 业务分部详情

### 2.1  国内法人租赁 / 金融

- **目标资产**：IT 设备、产业设备、OA 设备、医疗设备、建设设备、印刷机等
- **客户层**：大企业、中坚企业、中小企业、地方政府
- **渠道**：直销 + 厂商 / 供应商合作 + 地方银行合作
- **产品**：融资租赁、经营租赁、分期销售

### 2.2  国际业务（美洲・亚洲・欧洲）

- **美洲业务**：CSI Leasing 子公司（美国 / 北美 / 中南美最大级别独立 IT 设备租赁公司）
- **亚洲业务**：在中国、韩国、ASEAN（泰国、越南、印度尼西亚、菲律宾）设有合资和当地法人
- **欧洲**：与欧洲金融机构合作和参股

### 2.3  航空机金融（战略差异化）

- **Aviation Capital Group (ACG)**：2019 年 完全子公司化，美国总部大型航空机租赁公司（Top 10）
- **航空机持有数量**：数百架规模
- **客户**：全球各国航空公司
- **意义**：少数拥有世界顶级航空机租赁平台的日本综合租赁 1 社
- **GTF / reverse lease / 出售收益**：根据航空机供需产生的 resale / 二手机销售收益也很重要

### 2.4  环境基础设施业务

- **可再生能源**：对太阳能发电 (PV)、风电、地热、生物质能的 SPC 出资和项目金融
- **氢能 / 蓄电池 / EV 充电**：进入下一代能源领域
- **国际协同**：扩大“可持续租赁”业务

### 2.5  汽车 / specialty

- **汽车租赁 / car sharing**：面向法人的汽车租赁、面向个人的订阅
- **NRC（原 Nippon Rent-A-Car Service）**：曾将大型租车公司并表为子公司，之后合并 / 整合战略发生变化
- **移动出行相关投资**：MaaS、car sharing、EV 平台

## 5. 战略 / 近期动向

- **以 ACG 为轴进一步扩大航空机租赁**：捕捉航空需求恢复和 LCC 扩张带来的租赁机需求。
- **扩大环境基础设施**：加强对再生能源、蓄电、氢能和 EV 相关领域的出资（与 ITOCHU 协同）。
- **强化美洲 CSI Leasing**：扩大在北美 IT 设备租赁市场的寡占地位。
- **亚洲移动出行**：在东南亚投资汽车租赁、car sharing 和 EV 相关业务。
- **DX 投资**：AI 化授信模型、数字化合同管理。
- **SDGs / TCFD 对应**：积极化气候相关财务信息披露，扩充可持续金融产品。
- **分红性向**：通过股东回报（分红性向超过 30%）维持机构投资者评价。

## 7. 风险 / 监管论点

- **航空机业务波动性**：航空需求周期性 + 机型别供需和残值风险。
- **对利率上升的耐性**：租赁资产资金成本上升阶段的息差管理，尤其是大规模航空机采购。
- **2  轴股东结构下的治理**：在 Mizuho FG + ITOCHU 这一 2  轴母公司结构下的经营独立性和利益冲突管理。
- **美洲 CSI / ACG 的美元计价风险**：汇率波动和美国经济情景影响。
- **环境相关投资的长期风险**：再生能源项目长期 FIT 价格和固定成本结构风险。
- **ESG / 气候相关披露**：TCFD 和可持续性报告要求扩大。
