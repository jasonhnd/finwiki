# 东海东京证券


## Wiki 路线

本条目位于 [[JapanFG/INDEX|JapanFG index]] 之下。可与 [[JapanFG/tokai-tokyo-fh|東海東京フィナンシャル・ホールディングス]] 对照阅读，以把握母公司 holding 语境；与 [[JapanFG/okasan-sg|岡三証券グループ]] / [[JapanFG/iwai-cosmo-securities|岩井コスモ証券]] 一起阅读，以把握中坚独立证券 peer；与 [[securities/INDEX|securities index]] 一起阅读，以确认更广的市场结构边界。

## TL;DR

东海东京证券株式会社是**总店位于名古屋的中坚独立系证券公司**，也是东证 PRIME 上市 [[JapanFG/tokai-tokyo-fh|東海東京フィナンシャル・ホールディングス (8616)]] 的核心 100% 子公司。现行体制是在 2008-10  对旧东海东京证券（东海银行系）进行业务分拆后形成，其源流可追溯到 1929 年（昭和 4 年）创立的丸万证券。它是**金融商品交易业者 东海财务局长（金商）第140号**，即 FIEA Type 1  第一种金融商品交易业者。总店位于名古屋市中村区名站 Midland Square，资本金 60 億円，员工 1,869 名。作为**以东海地方（爱知、岐阜、三重、静冈）为基础、以面对面零售为中心的中坚独立证券公司**，它开展地银合作模式（与东海地方地银的区域共同门店）、IPO 承销、咨询型销售。其特征是与地银合资 / 共同门店品牌 “Tokai Tokyo XX Securities”。竞争对手为 [[JapanFG/iwai-cosmo-securities|岩井コスモ証券]]、[[JapanFG/okasan-sg|岡三証券グループ]]、[[JapanFG/aizawa-sg|藍澤証券]]、[[JapanFG/matsui-sec|松井証券]]。

## 1. 法人 / 股东

- **商号**：东海东京证券株式会社（英文：Tokai Tokyo Securities Co., Ltd.）
- **设立**：2008 年 10 月 8 日（在向 [[JapanFG/tokai-tokyo-fh|東海東京FH]] 持股公司体制转移时分拆）
- **创业务源**：1929 年（昭和 4 年）作为丸万证券创立
- **总店所在地**：邮编 450-6212 名，名古屋市中村区名站 4-7-1  Midland Square
- **资本金**：60 億円
- **母公司**：[[JapanFG/tokai-tokyo-fh|東海東京フィナンシャル・ホールディングス]]（东证 PRIME 8616，总部名古屋）100% 出资
- **代表者**：代表取缔役社长
- **员工数**：1,869 名（HD consolidated basis，2024 年 时点）

## 2. 牌照 / 注册

- **第一种金融商品交易业**：东海财务局长（金商）第140号
- **加入行业团体**：
  - 日本证券业协会 (JSDA) 正会员
  - 一般社团法人 金融期货交易业协会
  - 一般社团法人 第二种金融商品交易业协会
- **交易参加权**：[[securities/tokyo-stock-exchange|TSE]]、[[securities/osaka-exchange|OSE]] 交易参加者，[[securities/japan-securities-clearing-corp|JSCC]] 清算参加者

## 3. 营业基础

- **面对面营业**：总店 + 全国超过 130  个网点 / 营业所
- **地银合作模式**：与以下地银合资设立区域证券公司
  - “**Hokuhoku TT Securities**”（与 [[JapanFG/hokuriku-bank|北陸銀行]] / [[JapanFG/hokkaido-bank|北海道銀行]] 系 [[JapanFG/hokuhoku-fg|ほくほくFG]] 合资）
  - “**Tokai Tokyo & City Securities**”（与 [[JapanFG/oita-bank|大分銀行]] / FFG 系合资）
  - “**Hamagin TT Securities**”（与 [[JapanFG/yokohama-bank|横浜銀行]] / [[JapanFG/concordia-fg|コンコルディアFG]] 系合资）
  - “**Nishi-Nippon City TT Securities**”（与 [[JapanFG/nishi-nippon-city-bank|西日本シティ銀行]] 合资）
  - 与其他多家地银合作
- **网上交易服务**：“Anshin Sogo Service”“Tokai Tokyo Net Fish”
- **Call center**：多个据点

## 4. 经办产品

- **国内股票**：现货、信用交易；作为 TSE / OSE / 名古屋证券交易所 (NSE) 交易参加者，覆盖 3  大市场线
- **外国股票**：美国股、欧洲股、亚洲股
- **投资信托**：自社运用 + 各社 fund
- **债券**：国债、地方债、公司债、外国债
- **NISA**：成长投资枠、tsumitate 投资枠
- **iDeCo**：个人型确定缴费养老金计划
- **衍生品**：期货、期权、FX
- **IPO 承销**：以东海地方中坚企业 IPO 为中心的主承销 / 承销担当
- **M&A advisory**：面向中坚企业的事业承继 / 出售 / 收购咨询

## 5. 行业位置

- 作为**以名古屋 / 东海地方为基础的中坚独立证券**，它在 megabank FG 系证券与地银之间开展“合资证券”模式，形成独特位置。
- 也被称为**“东海地方的野村”**，采取面对面 + 地银合作的 hybrid strategy。
- 竞争：[[JapanFG/iwai-cosmo-securities|岩井コスモ証券]]（关西）、[[JapanFG/okasan-sg|岡三証券]]（名古屋 peer）、[[JapanFG/aizawa-sg|藍澤証券]]、[[JapanFG/matsui-sec|松井証券]]、[[JapanFG/marusan-securities|丸三証券]]
- **优势**：与东海地方中坚 / 大型企业的人脉深厚，经由地银渠道扩大零售客户，是地银证券合资模式的先行者
- **特殊业务**：通过不同于 [[securities/japan-online-brokerage-competition|オンライン証券競争]] 的面对面 + 地银合作模式实现差异化
- **战略**：深化地银合作，在东南亚（越南 / 印度尼西亚）开展海外证券业务

## 6. 沿革

| 年月 | 事项 |
|------|------|
| 1929  | 作为丸万证券在名古屋创立 |
| 战后 | 以名古屋为据点成长为区域证券公司 |
| 1997-12  | 旧山一证券破产；承接部分营业权 / 人员 |
| 2000-06  | **东海东京证券**（旧法人）成立：承接东海银行（现 [[JapanFG/mufg-bank|三菱UFJ銀行]]）系证券子公司 + 丸万证券 |
| 2003  | 扩大与 Mizuho / UFJ / Tokyo-Mitsubishi 的合作 |
| 2008-10-08 | **转为持股公司体制**：设立 Tokai Tokyo Financial Holdings，新生东海东京证券启动 |
| 2012  | 与 Nishi-Nippon City Bank 设立 Nishi-Nippon City TT Securities |
| 2014  | 与 Hokuhoku FG 设立 Hokuhoku TT Securities |
| 2017  | 与 Bank of Yokohama 设立 Hamagin TT Securities |
| 2019  | 设立越南子公司（胡志明） |
| 2022-04  | 东证市场区分调整；母公司 HD 转入东证 PRIME 市场 |
| 2024  | 对应新 NISA 制度 |

- **母公司**：[[JapanFG/tokai-tokyo-fh|東海東京フィナンシャル・ホールディングス]]（东证 PRIME 8616）
- **合资证券（地银合作）**：
  - Hokuhoku TT Securities（与 [[JapanFG/hokuhoku-fg|ほくほくFG]] 合资）
  - Hamagin TT Securities（与 [[JapanFG/concordia-fg|コンコルディアFG]] 合资）
  - Nishi-Nippon City TT Securities（与 [[JapanFG/nishi-nippon-city-bank|西日本シティ銀行]] 合资）
  - Tokai Tokyo & City Securities（大分银行系）

## 8. 为何需要独立页面

[[JapanFG/tokai-tokyo-fh|東海東京FH]] 页面偏向 holding company 角度的财务与股东结构。**牌照边界（东海财务局长 第140号 FIEA Type 1）、营业基础（130  据点 + 地银合资品牌）、经办产品、IPO 承销战略、地银合作模式** 如果不拆成 operating company（证券公司）的独立页面，就无法准确分析。与中坚独立证券 peer（[[JapanFG/iwai-cosmo-securities|岩井コスモ証券]]、[[JapanFG/okasan-sg|岡三]]、[[JapanFG/aizawa-sg|藍澤]]）比较时，本页也是 anchor。作为地银合作模式的先行者，它是独特的比较对象。
