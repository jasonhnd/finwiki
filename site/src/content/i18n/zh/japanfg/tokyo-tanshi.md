# 东京短资（Tokyo Tanshi）

## Wiki 路线

该条目位于 [[JapanFG/INDEX|JapanFG index]] 之下。可结合 [[JapanFG/central-tanshi|セントラル短資株式会社 (Central Tanshi)]] 阅读同业和对比背景，并结合 [[banking/INDEX|banking index]] 理解更广泛的制度和监管边界。

## 要点

东京短资是日本 3  大短资公司之一（非上市，1908-09  创业，明治末期以来的老牌公司）。它围绕 **call 市场中介 + 票据买卖 + 日本银行操作对手方 + 法人 FX** 这 4  条轴线，承担短期金融市场的 inter-dealer 功能。在 2016-01 至 2024-03  负利率政策下，call 市场功能停滞，行业整体进入收缩局面，因此一直通过外汇和法人 FX 经纪等方式推进收益来源多元化。2024-03-19 日本银行解除负利率、2024-07-31 将政策利率提高至 0.25% 后，短期利率市场时隔 8  年正常化，短资行业复苏预期升温。

## 1. 公司概要

**正式名称**：东京短资株式会社
**英文名**：Tokyo Tanshi Co., Ltd.
**设立**：1908-09（明治 41  年，创业超过 117  年）
**形态**：非上市（株式会社）
**总部**：东京都中央区日本桥（兜町、茅场町一带的传统金融街）
**业态**：短资业者（money market broker / inter-dealer broker）

### 3  大短资的定位

3  大短资是战后短资行业重组后存续下来的 3 社，被认定为日本银行公开市场操作（操作）对手方，主要业务是 call 市场的 inter-dealer 中介：

| 公司名 | 简称 | 创业 | 特征 |
|---|---|---|---|
| [[central-tanshi]] | Central Tanshi | 1909 | 旧山根短资 + 上田短资系统，被认为是行业最大型 |
| [[ueda-yagi-tanshi]] | Ueda Yagi Tanshi | 1956（合并） | 旧上田短资 + 八木短资合并，关西系资本 |
| **东京短资** | TT | **1908-09** | **最老牌，强化法人 FX 经纪** |

## 2. 业务板块地图

| 板块 | 内容 | 客户 |
|---|---|---|
| **Call 市场中介** | 从无担保 call、有担保 call O/N（隔夜）到短期品种的出借方和借入方匹配 | 日系银行、地银、信托、生财险、系统金融机构、外资银行东京支店 |
| **票据买卖中介** | CP（商业票据）、银行承兑票据等流通中介 | 同上 + 事业公司（发行体） |
| **日本银行操作对手方** | 作为日本银行公开市场操作（短期国债买入、共同担保资金供给、国债卖现先等）的对手方参加 | 日本银行 |
| **面向法人的 FX 经纪** | 大额外汇的银行间中介、法人套期保值 | 日系银行、外资银行、大企业财务部门 |
| **短期金融商品相关服务** | repo 交易中介、债券借贷交易中介等 | 同上 |

短资业者通常不大量持有自营交易头寸，**中介手续费（brokerage）是主要收入来源**。负利率下，利差消失、call 市场余额大幅减少，call 中介手续费枯竭；因此通过法人 FX 等非 call 业务推进收益来源多元化，成为行业共同课题。

### 负利率时期的困境（2016-01 至 2024-03）

- 银行间余资运用需求被日本银行当座存款宏观加算余额 / 政策利率余额 3  层结构吸收，call 市场中介需求本身结构性下降
- 3  大短资均通过**人员削减、费用削减、强化非 call 业务（FX 中介和债券相关）**来应对

### 负利率解除和复苏预期（2024-03 以后）

- **2024-03-19 日本银行解除负利率**（从▲0.1% 至 0-0.1%）并废止 YCC → 时隔 17  年加息
- **2024-07-31 政策利率提高至 0.25%**
- 短期利率正常化 → call 市场的利率形成功能恢复 → call 余额扩大预期 → 3  大短资核心业务复苏情景

### 竞争结构

- **3 社 寡占**：战后重组后集约为 3 社，没有新进入，也没有退出
- **与海外 IDB 竞争**：TP ICAP、BGC 等全球 inter-dealer broker 通过日本支店处理部分短期金融商品

## 4. 监管和政策

- **主管**：金融厅（FSA）和日本银行（作为操作对手方）
- **行业团体**：短资协会（3  大短资 + 短资业务相关机构）
- **与日本银行政策联动性极高**：政策利率、YCC、公开市场操作制度变化直接影响行业收益
- **近期政策论点**：
  - 2024-03 以后日本银行政策利率正常化（负利率解除 → 0.25% → 0.50%）→ call 市场再活性化预期
  - 日本银行当座存款超额准备金逐步缩小（QT，量化紧缩）→ call 市场余额恢复的前提条件
  - **无担保 call O/N 利率** 是政策利率的市场基准 → 短资业者的中介功能是货币政策传导路径的一部分
