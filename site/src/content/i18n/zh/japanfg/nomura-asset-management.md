# 野村资产管理

## Wiki 路径

本条目位于 [[JapanFG/INDEX|JapanFG index]] 之下。可与 [[JapanFG/asset-management-one|アセットマネジメントOne]] 对照阅读，用于同业 / 对比语境；也可与 [[securities/INDEX|securities index]] 一并理解更广义的系统 / 监管边界。

## TL;DR

野村资产管理是 [[JapanFG/nomura-hd|野村 HD]] 的资产管理运营公司。在 FSA 金融商品交易业者名单中，其以“野村アセットマネジメント株式会社”、关东财务局长（金商）第373号 登载。它是野村集团投信、ETF 和面向机构投资者运用业务的主体。

## 1. 法人 / 牌照边界

| 项目 | 解读 |
|---|---|
| 法定名称 | 野村アセットマネジメント株式会社 |
| FSA 注册 | 关东财务局长（金商）第373号 |
| FIEA 范围 | 第二种金融商品交易业、投资顾问 / 代理业、投资运用业 |
| 母公司边界 | [[JapanFG/nomura-hd|野村 HD]] 的资产管理部门 |

## 2. 业务角色

- 承担投资信托、ETF、年金和机构投资者运用。
- 是区别于 [[JapanFG/nomura-hd|野村 HD]] 证券经纪 / 投资银行的 fee-based 资产管理支柱。
- 与 [[JapanFG/nomura-trust-bank|野村信託銀行]] 一起构成证券集团的财富 / 资产形成栈。

## 3. 独立页面的意义

若只从证券经纪视角看野村集团，作为运用公司的 ETF / 指数 / 主动管理 / 机构委托功能不容易显现。将其独立成页，有助于连接资产运用立国、NISA、ETF 市场和信托银行 custody。
