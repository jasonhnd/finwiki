# 法国兴业银行日本 (Societe Generale Japan)

## Wiki route

本条目位于 [[JapanFG/INDEX|JapanFG index]] 之下。可与 [[JapanFG/bank-of-america-japan|バンク・オブ・アメリカ日本 (Bank of America Japan)]] 对照阅读，以理解同业 / 对比背景；也可与 [[banking/INDEX|banking index]] 一起阅读，以理解更广义的系统和监管边界。

## TL;DR

法国兴业银行日本是法国大型银行 Societe Generale 集团在日本的据点群。其通过东京银行分行和证券法人开展批发银行、markets、衍生品、证券及投资银行业务。它是与 [[JapanFG/deutsche-japan|deutsche-japan]]、[[JapanFG/bnp-paribas-japan|bnp-paribas-japan]]、[[JapanFG/barclays-japan|barclays-japan]] 并列的欧洲系在日 IB / markets 参与者。

## 1. 公司概要

| 项目 | 内容 |
|---|---|
| Parent company | Societe Generale S.A. |
| Main Japan banking presence | 法国兴业银行东京分行 |
| Main Japan securities presence | Societe Generale Securities Japan Limited |
| Tokyo branch establishment | 1973年6月 |
| Main businesses | 银行业务、登记金融机构业务、外国银行代理业务、证券和衍生品 |
| Supervision | 金融厅、关东财务局、本国法国 / EU 监管 |

在金融厅登记金融机构名单中，可以确认法国兴业银行东京分行为关东财务局长（登金）第90号 的登记金融机构。外国银行代理银行名单也可确认与法国兴业银行东京分行相关的许可。

## 2. 业务分部地图

| 业务 | 内容 | 在日本的定位 |
|---|---|---|
| Wholesale banking | 企业授信、贸易、外汇 | 面向跨国企业和金融机构 |
| Markets | 股票、FX、债券、大宗商品、衍生品 | 欧洲系 structured products 强项 |
| Securities | Sales、trading、上市衍生品、市场接入 | 面向机构投资者 |
| Investment banking | 资本市场、M&A、跨境交易 | 与日系银行和欧美 IB 竞争 |
| Foreign-bank agency | 集团银行业务的日本代理 | 监管上边界清晰 |

## 3. 战略 / 竞争

- **欧洲系 markets house**：SocGen 在全球衍生品和 structured products 领域具有存在感；在日本则以机构投资者和金融机构为中心。
- **与欧洲同业比较**：[[JapanFG/deutsche-japan|Deutsche]] 偏 FICC / 欧洲银行，[[JapanFG/bnp-paribas-japan|BNP Paribas]] 是法国最大级别综合金融集团，SocGen 则更具 markets / derivatives 色彩。
- **与美国 IB 的差异**：与 [[JapanFG/jpmorgan-japan|JPMorgan]]、[[JapanFG/goldman-sachs-japan|Goldman Sachs]]、[[JapanFG/morgan-stanley-japan|Morgan Stanley]] 相比，其在日本大型 M&A 中的存在感相对较小，但在产品设计和欧洲客户接点上有强项。
- **监管接点**：由于组合了登记金融机构、外国银行代理、证券公司等多项牌照，需要横向理解银行法、金商法和衍生品监管。

## 4. 监管 / 政策

- 法国兴业银行东京分行作为外国银行分行，受银行法监管。
- 登记金融机构业务受金商法下的登记金融机构规制。
- 证券法人受第一种金融商品交易业等金商法规制。
- 官方页面可确认其加入全国银行协会、日本证券业协会、金融期货交易业协会等协会。
