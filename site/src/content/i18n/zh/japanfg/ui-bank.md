# UI银行（UI Bank）

## Wiki 路线

该条目位于 [[JapanFG/INDEX|JapanFG index]] 之下。可结合 [[JapanFG/saitama-resona-bank|埼玉りそな銀行 (Saitama Resona Bank)]] 阅读同业和对比背景，并结合 [[banking/INDEX|banking index]] 理解更广泛的制度和监管边界。

## 要点

UI银行是 [[JapanFG/kiraboshi-fg|東京きらぼし FG]] 100% 出资的数字银行运营公司。它于 2022 年 开业，是区域金融集团推出的手机银行，主打面对面与非面对面混合、生活伙伴型金融和 API 联动。

## 2. 业务角色

- 作为 Kiraboshi 集团 DX 的核心角色而开业的手机银行。
- 以“面对面 / 非面对面混合”“数字生活伙伴”“外部 API 联动”为概念。
- 可作为地方银行集团在既有银行之外持有 digital bank 的模型，与 [[banking/minna-bank-baas-model|みんなの銀行]] 和 [[JapanFG/01-bank|01銀行]] 比较。

## 3. 为什么需要独立页面

UI银行不是东京 Kiraboshi FG 的单纯数字渠道，而是持有独立银行牌照的 operating company。为分析地银 DX、手机银行和区域金融集团全国扩展模型，需要单独保留。
