# 劳动金库联合会（Rokinren Bank）

## Wiki route

本条目位于 [[JapanFG/INDEX|JapanFG index]]。可与 [[JapanFG/mufg|三菱 UFJ FG (MUFG)]] 对照阅读，以理解同业和差异；更广泛的制度和监管边界见 [[banking/INDEX|banking index]]。

## TL;DR

劳动金库联合会是以全国 13  家劳动金库为会员的「ろうきん」业态中央金融机构。它于 1955年 设立，以《劳动金库法》为依据，承担会员劳金的资金供需调节、富余资金运用、全国汇兑和财形储蓄集中处理、业态统一在线系统运营等职能。它与 [[JapanFG/zenshin-kumi|全信組連]]、[[JapanFG/shinkin-central|信金中央金庫]] 并列，是协同组织金融体系中的中央机构。

## 2. 业务分部地图

| 业务 | 内容 | 角色 |
|---|---|---|
| 资金供需调节 | 与会员劳金之间的存款和贷款交易 | 业态中央银行 |
| 富余资金运用 | 将会员劳金的富余资金进行市场运用 | 机构投资者功能 |
| 金融功能补充 | 国内汇兑、国债和投信柜台服务等 | 会员劳金共同服务基础 |
| 事务受托 | 账户转账、财形储蓄处理等 | 提升业务效率 |
| 系统运营 | R-ONE 系统开发和运营 | 共同 IT 基础设施 |

官方说明称，劳金连把会员劳金富余资金的 58% 作为存款吸收；劳金连吸收的存款和可转让存单中，80% 来自会员劳金。

## 3. 战略 / 竞争

- **协同组织金融的中央机构**：劳金连不是营利银行，而是支持面向劳动者福利金融的劳动金库的中央机构。
- **与信金 / 信组比较**：[[JapanFG/shinkin-central|shinkin-central]] 支持信用金库，[[JapanFG/zenshin-kumi|zenshin-kumi]] 支持信用组合，劳金连则支持劳动金库。
- **系统共同化**：在地区金融机构成本上升的环境下，共同系统和事务受托成为竞争力来源。
- **政策性和市场运用并立**：一方面支持劳动者福利金融，另一方面在金融市场运用富余资金，因此同时具有 ALM 和社会使命。

## 4. 监管 / 政策

- 以《劳动金库法》为依据的协同组织金融机构。
- 适用包括《银行法》准用在内的信息披露、风险管理和监督规则。
- 在金融厅和财务局监督下，AML/CFT、网络安全、系统风险管理均是重点。
