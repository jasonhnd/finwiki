# Nichizei Trust / 日税信託

Wiki route: [[JapanFG/INDEX]] -> [[JapanFG/trust-companies-japan-index]] -> [[JapanFG/nichizei-trust]]

## Overview

Nichizei Trust 是与 Nichizei 集团路径相关的管理型信托公司。公开资料将其定位在葬礼费用保管、高龄者资产管理、继承、不动产信托及相关遗产规划支持等领域。

## Entity Boundary

本页覆盖作为信托公司运营主体的株式会社日税信託。它不应并入一般税理士网络页面；税务咨询角色与信托公司角色需要保持分离。

## Public Context

Nichizei Trust 的重要性在于，它把税务专业服务网络与受监管的信托公司服务连接起来。当 FinWiki 区分专业服务分销与银行分销时，本页很有用。
