# 日本外国银行分行牌照路径

## 概览

在日本的外国银行分行，是外国银行在日本银行监管路径下取得牌照的分行，不是日本银行子公司，也不是单纯的代表处。FSA 的监测框架强调日本分行的业务模式、治理、合规、风险管理，以及与外国总行 / 母国监管机构的协调。

本页可与 [[JapanFG/foreign-bank-branches-japan-index|foreign-bank branch registry index]]、[[banking/foreign-bank-japan-retreat|foreign-bank Japan retreat]]、[[JapanFG/jpmorgan-japan|JPMorgan Japan]]、[[JapanFG/bank-of-america-japan|Bank of America Japan]]、[[JapanFG/hsbc-japan|HSBC Japan]]、[[JapanFG/bny-mellon-japan|BNY Mellon Japan]]、[[JapanFG/euroclear-bank-japan|Euroclear Bank Japan]]、[[JapanFG/clearstream-banking-japan|Clearstream Banking Japan]] 和 [[JapanFG/legal-financial-licenses/INDEX|legal / financial licenses]] 一起使用。

| 形态 | 含义 | FinWiki 路径 |

| 外国银行分行 | 外国银行在日本的持牌分行。 | [[JapanFG/foreign-bank-branches-japan-index|foreign-bank branch registry index]]。 |
| 外国银行代理银行 | 日本银行 / 分行在代理路径下为外国银行开展业务。 | FSA 外国银行代理银行名单。 |

| 代表处 | 不开展银行牌照活动的市场 / 联络存在。 | 除非在 FSA 名单中，否则不要作为银行分行处理。 |
| 证券公司 / FIBO | 经纪、承销、衍生品或证券业务。 | [[JapanFG/legal-financial-licenses/securities-license-stack|securities license stack]]。 |

## 监管重点

| 重点 | 公开相关性 |
|---|---|
| 总行依赖 | 分行依赖外国总行治理、资本、系统和风险偏好。 |
| 日本业务模式 | 批发银行、市场、托管、贸易金融、项目融资或亚洲走廊银行业务会产生不同风险。 |
| 合规 / 行为 | AML、制裁、市场行为、客户适合性和跨境 booking controls 很重要。 |
| 风险管理 | 流动性、信用、市场、运营、外包和 booking-model controls 必须适配分行。 |
| 母国 / 东道国协调 | FSA 可能与外国总行和母国监管机构协调。 |
| 披露清晰度 | 不应混淆分行、子公司、代表处和证券公司。 |

## 战略性外国银行 P2 Exceptions

大多数长尾外国银行分行条目可以保持 registry-only，但当分行具备以下特征时，应提升为独立页或路径页：

- 具有大型 global-market / investment-bank 相关性：[[JapanFG/jpmorgan-japan|JPMorgan]]、[[JapanFG/goldman-sachs-japan|Goldman Sachs]]、[[JapanFG/bank-of-america-japan|Bank of America]]、[[JapanFG/citigroup-japan|Citigroup]]；
- 具有 custody / asset-servicing / ICSD 相关性：[[JapanFG/bny-mellon-japan|BNY Mellon]]、[[JapanFG/state-street-japan|State Street]]、[[JapanFG/euroclear-bank-japan|Euroclear]]、[[JapanFG/clearstream-banking-japan|Clearstream]]；
- 具有亚洲走廊政策 / 贸易相关性：[[JapanFG/dbs-bank-japan|DBS]]、[[JapanFG/ocbc-bank-japan|OCBC]]、[[JapanFG/uob-bank-japan|UOB]]、[[JapanFG/bangkok-bank-japan|Bangkok Bank]]、[[JapanFG/icbc-japan|ICBC]]、[[JapanFG/bank-of-china-japan|Bank of China]]、[[JapanFG/bank-of-communications-japan|Bank of Communications]]、[[JapanFG/state-bank-of-india-japan|State Bank of India]]、[[JapanFG/bank-of-india-japan|Bank of India]]；
- 具有多牌照银行 / 证券 stack 相关性：[[JapanFG/anz-bank-japan|ANZ Japan]]；
- 具有事件驱动的牌照 / 监测相关性，例如新取得牌照的分行业务或重大业务模式转变；
- 与 [[securities/INDEX|securities]]、[[banking/INDEX|banking]]、托管、支付或政策金融地图存在明确链接。

## 边界案例

| 案例 | 正确处理 |
|---|---|
| “外国银行退出日本” | 检查分行是否仍在 FSA 名单中、是否转为仅证券业务，或是否已关闭银行活动。 |
| 外国银行拥有证券部门 | 将 FIEA / JSDA 路径与银行分行路径分开链接。 |
| 外国银行拥有信托 / 托管实体 | 当托管是核心功能时链接到 [[banking/japan-trust-bank-custody-map|trust-bank custody map]]。 |
| 日本子公司银行 | 作为日本持牌银行实体处理，而不是外国银行分行。 |
| 代理关系 | 检查 FSA 外国银行代理银行名单，并与直接分行牌照分开记录。 |

## 研究清单

1. 检查 FSA 持牌 / 注册机构 portal 以及银行牌照 PDF / Excel，确认当前分行状态。
2. 检查 FSA 外国银行分行监管指南和监测公开材料，确认当前监管重点。
3. 从官方披露确认精确法定名称、分行名称、母国和业务模式。
4. 如果证券业务重要，另行交叉核对 FIEA / JSDA 会员资格。
5. 当战略问题是零售退出还是批发聚焦时，链接到 [[banking/foreign-bank-japan-retreat|foreign-bank retreat pattern]]。
