# 日本外国银行代理业务路径

## 概览

外国银行代理业务不同于外国银行支店牌照。它是由银行或日本支店作为某一主外国银行的代理人或中介方开展业务的路径。该路径对跨境企业银行、全球现金管理、离岸分行业务服务和外国集团覆盖范围都很重要。

本页可与 [[JapanFG/legal-financial-licenses/foreign-bank-branch-license-route|foreign-bank branch license route]]、[[JapanFG/foreign-bank-branches-japan-index|foreign-bank branch registry index]]、[[banking/foreign-bank-japan-retreat|foreign-bank Japan retreat]]、[[JapanFG/legal-financial-licenses/foreign-financial-group-adjacent-licenses|foreign financial group adjacent licenses]]、[[JapanFG/legal-financial-licenses/INDEX|legal / financial licenses]] 一起阅读。

## 路径对比

| 路径 | 行为主体 | 能证明什么 | 不能证明什么 |
|---|---|---|---|
| 外国银行支店 | 持有牌照的外国银行日本支店。 | 直接的日本支店银行牌照。 | 证券、信托、支付或所有零售服务。 |
| 外国银行代理银行 | 银行 / 支店为主外国银行开展代理。 | FSA 名单认可代理或中介关系。 | 代理方就是资产负债表上的贷款人或存款吸收方。 |
| 日本银行子公司 | 由外国集团控制的日本法人银行。 | 国内法人实体下的日本银行牌照。 | 外国银行支店地位。 |
| 代表处 | 联络 / 信息收集据点。 | 仅证明存在据点。 | 银行业务、存款、贷款、代理或证券业务。 |

## 使用场景

| 使用场景 | 为什么代理路径可能重要 |
|---|---|
| 全球现金管理 | 日本客户可能需要通过本地银行界面访问外国分行 / 离岸产品。 |
| 贸易金融 | 信用证、担保和跨境结算可能涉及外国主银行。 |
| 企业财资 | 跨国企业客户需要覆盖日本及海外银行实体。 |
| 私人 / 财富银行 | 跨境转介和账户访问需要谨慎处理牌照边界。 |
| 集团分行协调 | 持牌日本支店可在批准 / 备案路径下为总行或其他海外分行行事。 |

## 控制问题

| 问题 | 公开资料中的意义 |
|---|---|
| 谁是代理银行，谁是主外国银行？ | 责任和客户披露路径取决于这组关系。 |
| 该路径是批准还是备案？ | FSA 名单会区分许可 / 申请日期。 |
| 被中介的是哪类服务？ | 存款、贷款、外汇、贸易金融、证券或类似支付的服务可能触发其他制度。 |
| 客户面对的是日本支店、离岸分行，还是日本子公司？ | 法律实体、存款保护和披露要求各不相同。 |
| 是否还存在证券实体？ | 需另行检查 FIBO / JSDA 状态。 |

## JapanFG 相关性

- [[JapanFG/mufg-bank|MUFG Bank]]、[[JapanFG/sumitomo-mitsui-banking-corp|SMBC]]、[[JapanFG/mizuho-bank|Mizuho Bank]] 很重要，因为大型银行连接日本客户与全球分行及代理行网络。
- 当服务范围涉及跨境时，[[JapanFG/hsbc-japan|HSBC Japan]]、[[JapanFG/standard-chartered-japan|Standard Chartered Japan]]、ANZ Bank Japan、[[JapanFG/ing-bank-japan|ING Bank Japan]]、[[JapanFG/deutsche-japan|Deutsche Japan]] 都需要拆分检查分行 / 代理 / 证券边界。
- 尽管同样使用 "agency" 一词，[[payments/japan-bank-api-payment-agency-route|Electronic payment agency]] 属于不同制度。

## 研究清单

1. 检查 FSA 外国银行代理银行名单，确认准确的代理方 / 主银行组合。
2. 检查该实体是否也出现在外国银行支店、FIBO、信托或支付登记名单中。
3. 核对银行日本官网面向客户的说明。
4. 不要仅凭代理银行名单推断存款保护、证券许可或日本支店地位。
5. 当问题属于市场定位时，将跨境银行问题链接到 [[banking/foreign-bank-japan-retreat|foreign-bank Japan strategy]]。
