# Sumire Regional Trust / すみれ地域信托

Wiki route: [[JapanFG/INDEX]] -> [[JapanFG/trust-companies-japan-index]] -> [[JapanFG/sumire-regional-trust]]

## Overview

Sumire Regional Trust 是东海地区的管理型信托公司，公开资料将其与地区资产利用、社区 / 不动产项目联系起来。在 FinWiki 中，它是 [[JapanFG/kiriu-real-estate-trust]] 旁边的地区振兴信托路线。

## Entity Boundary

本页覆盖すみれ地域信託株式会社。它不应吸收整个 Sumire group 或高山地区开发背景；这些是相邻的公开语境，不是信托公司实体本身。

## Public Context

Sumire Regional Trust 之所以重要，是因为它展示了东京 / 大阪金融中心之外，在地方和地区项目中使用信托公司的方式。它有助于描绘信托功能如何支持社区资产连续性。
