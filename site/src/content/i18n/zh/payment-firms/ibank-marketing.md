# iBank Marketing（iBank マーケティング）

## Wiki 路由

本条目归于 [[payment-firms/INDEX|payment-firms INDEX]] 之下，作为 [[regional-banks/fukuoka-fg|ふくおか FG (FFG)]] 的 FG 子公司金融科技 / 钱包 App 经营公司锚点。请结合母经营银行 [[regional-banks/fukuoka-bank|福岡銀行]] / [[regional-banks/18-shinwa-bank|十八親和銀行]] / [[regional-banks/kumamoto-bank|熊本銀行]] 以及钱包 App 同业 [[megabanks/paypay-fg|PayPay FG]] / [[megabanks/au-fh|au フィナンシャル HD]] / [[payment-firms/rakuten-fg|楽天 FG]] 的背景阅读。更广泛的体系路由参见 [[banking/regional-bank-api-digital-partnership-route|regional-bank API / digital partnership route]] 与 [[banking/INDEX|banking index]]。

## TL;DR

株式会社 iBank Marketing 是 [[regional-banks/fukuoka-fg|ふくおか FG (FFG)]] 100% 子公司的地银起源金融科技子公司，负责 **Wallet+**（Wallet Plus）智能手机 App 的企划・运营。作为地银起源的智能手机优先个人金融服务，向 [[regional-banks/fukuoka-bank|福岡銀行]] / [[regional-banks/18-shinwa-bank|十八親和銀行]] / [[regional-banks/kumamoto-bank|熊本銀行]] 客户提供 **目的别储蓄（myCoin）**・**家计管理**・**预付卡 Wallet+ Card**・**地区联动优惠券** 等。作为地银金融科技的代表案例，其轴线为「银行账户联动 + 目的别储蓄」，与 [[megabanks/paypay-fg|PayPay]] / [[megabanks/au-fh|au PAY]] / [[payment-firms/rakuten-fg|楽天]] 等通信系 / EC 系钱包不同。^[extracted]

### iBank Marketing 设立

- **2016**：[[regional-banks/fukuoka-fg|FFG]] 设立 iBank Marketing（地银起源金融科技子公司的先驱案例）
- 以「重新发明银行」为理念，企划智能手机完结・面向年轻层的金融服务
- 以 [[regional-banks/fukuoka-bank|福岡銀行]] 账户联动为基础，将家计管理 + 目的别储蓄功能置于核心

### Wallet+ App 展开

- **2016**：Wallet+ App 发布
- **myCoin 功能**：目的别储蓄（设定旅行・结婚・住房首付等目的，按月定额储蓄）
- **Wallet+ Card**：JCB 品牌的预付卡（充值式）
- **地区联动**：福冈市内・九州各地的店铺・观光地联动优惠券

### 集团扩展

- **2018-2020**：新增 [[regional-banks/18-shinwa-bank|十八親和銀行]] / [[regional-banks/kumamoto-bank|熊本銀行]] 联动 → 覆盖九州 3 县

### FG 内定位

- **年轻层获取渠道**：[[regional-banks/fukuoka-fg]] 银行 3 行将年轻层 main bank 化的入口
- **数字战略的入口**：与 Minna Bank（数字银行）・FFG 银行 App 并列的数字接触点
- **地域经济活性化**：通过福冈市内・九州各地的店铺联动优惠券实现地域内消费循环

### 竞争格局

- **通信系钱包**（[[megabanks/paypay-fg|PayPay]] / [[megabanks/au-fh|au PAY]] / d 払い / [[payment-firms/rakuten-fg|楽天ペイ]]）：全国规模・以支付为中心，Wallet+ 以银行联动 + 目的别储蓄轴差异化
- **银行 App**（大型银行・其他地银）：以 UI/UX 差异化，myCoin 等独有功能
- **家计簿 App**（[[payment-firms/money-forward|マネーフォワード]] / freee 系个人）：银行联动范围广，但地区联动 / 预付为 Wallet+ 独有
- **其他地银金融科技**：与北国银行（CCI 集团）或山阴合同银行系的地银起源金融科技类似 model

### 强项与弱项

- 强项：[[regional-banks/fukuoka-fg]] 银行 3 行账户客户基础 + 九州地盘的地区联动深度 + 目的别储蓄 myCoin 的独有性

## 7. 监管・政策

- **主管**：金融厅（FSA）（作为 App 事业公司单独无需银行业牌照，经联动方银行 [[regional-banks/fukuoka-bank]] / 预付发行者）
- **规制议题**：
  - 预付发行业务受资金结算法上的预付式支付手段发行者规制
  - 银行 API 联动受银行法上的电子结算等代行业（由联动方银行登记）
  - 个人信息保护法 + My Number 法（家计数据处理）
  - 反社会势力核查・反洗钱对策（与联动方银行共同实施）
