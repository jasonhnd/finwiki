# JPYC株式会社


## Wiki 路线

本条目位于 [[payment-firms/INDEX|payment-firms INDEX]] 之下。可对照 [[payment-firms/wise-payments-japan|Wise Payments Japan]] 理解同业 / 对比背景，并参照 [[payments/INDEX|payments index]] 理解更广泛的系统 / 监管边界。

## TL;DR

JPYC株式会社是日本日元计价 stablecoin / electronic payment instrument 的资金转移型发行人。JPYC EX 是日元与 JPYC 之间的发行和赎回窗口，该公司在 FSA 资金转移业者登记一览中以关东财务局长 第00099号 登载。

| 项目 | 说明 |
|---|---|
| 法人实体 | JPYC株式会社 |
| 主轴 | 通过 JPYC EX 发行和赎回日元稳定币 |
| 登记路径 | FSA 资金转移服务提供者名单；不是 crypto-asset exchange operator 页面。 |
| 既有详情页 | [[exchanges/jp-exchange-jpyc|JPYC exchange-namespace legacy / detailed page]] |

## 2. 业务角色

- JPYC EX 允许用户通过银行转账以日元发行和赎回 JPYC。
- JPYC EX 官方材料将 JPYC 描述为可赎回为日元的工具，并与 JPYC Prepaid 区分。
- 该公司是观察 [[payment-firms/progmat|Progmat]] 信托型基础设施和 [[exchanges/jp-exchange-sbi-vc-trade|SBI VC Trade]] USDC 分销的重要对照对象。

## 3. 为什么需要这个独立页面

JPYC 过去主要通过 `exchanges/` 命名空间进入，但它并不是传统 CEX。JapanFG 页面可以把已登记资金转移 / 稳定币发行人的边界放在金融机构地图中可见的位置。
