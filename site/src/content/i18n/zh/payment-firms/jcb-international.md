# JCB International（ジェーシービー・インターナショナル）

## Wiki 路径

本条目位于 [[payment-firms/INDEX|payment-firms INDEX]] 之下。可对照 [[card-issuers/jcb|JCB]] 阅读以了解同类/对比语境（母公司 / 体系运营者关系），对照 [[payments/japan-card-issuer-acquirer-processor-split|Japan card issuer / acquirer / processor split]] 了解更广泛的系统 / 监管边界。

## TL;DR

JCB International Co., Ltd. (JCBI) 是 [[card-issuers/jcb|JCB]] 的 **承担海外业务的 100% 子公司**。它一手承担国际品牌「JCB」的 **海外发行 (overseas issuance) / 海外商户拓展 (acquiring) / 品牌许可** 的 3 职能。与日本国内发行由母公司 JCB 直接承担不同，与海外被许可银行（亚洲・美洲・欧洲）的合作、海外商户网络（含与 Discover / UnionPay 的互通使用）的运营全部经由 JCBI。

## 2. 在日本的业务线

JCBI 本身 **拥有面向海外的职能，但法人总部设于东京（港区南青山）**，人员・总部职能集中在日本。其在日本国内的活动主要如下：

- **海外商户网络运营**: 亚洲（中国・韩国・台湾・东南亚）、美洲（经由 Discover 合作）、欧洲主要旅游地的商户合同・支持。
- **面向访日外国人（入境游）商户的支持**: 负责国内商户受理海外发行 JCB 卡持有者访日支付时的处理。
- **与海外被许可银行的品牌合同**: 向亚洲的合作银行（Bangkok Bank / Bank of Communications 等）授予 JCB 品牌许可。
- **与 Discover / UnionPay 等其他品牌的互通使用合同**: 管理 JCB 卡在海外可在 Discover 商户・UnionPay 商户使用 / 反之的业务合同。

由于面向国内消费者的发行由母公司 [[card-issuers/jcb|JCB]] 负责，JCBI 与国内普通消费者几乎没有直接接触。

## 3. 战略・相对 JCB / 国内体系的竞争定位

JCB 的结构接近 **3-party scheme**（[[payment-firms/american-express-international-japan|American Express]] 型・自身兼任发行 + 商户签约 + 品牌），但在海外采用 **经由被许可银行的 4-party scheme** 混合方式 —— 这一海外展开模式的执行主体就是 JCBI。与 Visa / Mastercard 作为完全 4-party 的纯粹体系运营者不同，JCB 同时灵活运用「自身直接发行 + 许可授予」两种方式。海外商户覆盖大幅逊于 Visa / Mastercard，但采取以 **与 Discover（美）/ UnionPay（中）的互通使用合作** 来填补缺口的战略，而 JCBI 成为该合作合同・运营的窗口。从国内 issuer 视角看，与 [[card-issuers/smbc-card|三井住友カード]] 或 [[card-issuers/mufg-nicos|三菱UFJニコス]] 这类以 Visa / Mastercard 为主体的 issuer 不同，JCB 品牌 issuer（乐天卡等的 JCB 版・JCB 自有）经由 JCBI 共享海外商户网的机制。

## 4. 本页为何重要

- 用于在 **法人层面分离梳理** JCB 的「海外品牌业务」与「国内发行业务」的边界页面
- 明示 Discover / UnionPay 互通使用合同的 **签约主体**（是 JCBI 而非 JCB）
- 在国际品牌比较中梳理「JCB 是 3-party 还是 4-party」这一含糊之处
