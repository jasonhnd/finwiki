# Paidy 株式会社

## Wiki 路径

## TL;DR

日本首个正式规模化的 **BNPL（Buy Now Pay Later）** 提供商。2008-10 以原 **株式会社 Exchange Corporation** 名义成立，2018-09 社更名为 Paidy。创始人为 **Russell Cummer**（美国出身）。**2021-09-07 PayPal Holdings 宣布以约 27 億美元（约 3000 億円）收购**，2022 完成，成为 PayPal 日本子公司。用户仅凭邮箱 + 电话号码即可付款，之后通过便利店、银行转账或账户扣款后付。加盟店约 100 万家（Amazon Japan / SHEIN / Apple Store / Mercari 等）。处于美国 Affirm / Klarna / Afterpay 的日本版位置。^[extracted]

### 主要服务线

| 服务 | 内容 |
|---|---|
| **Paidy（后付）** | 用邮箱 + 电话号码付款，次月通过便利店 / 银行转账 / 账户扣款一次性支付 |
| **Paidy 后付计划** | 3/6/12 期 分期 / 循环支付（含免手续费计划） |
| **Paidy Card** | Visa 虚拟卡，支持 Apple Pay 绑定，也可用于实体店和海外电商 |
| **Paidy Card Plus** | 高阶等级（发行实体卡、扩大使用额度） |
| **Amazon Japan 后付** | Amazon 官方的“后付”支付提供商 |

### 重要时间线 ^[extracted]

| 年月 | 事项 |
|---|---|

> 收购背景：PayPal 在日本市场的消费者存在感较弱，希望通过吸收 Paidy 的 **基于电话号码认证的即时授信模型** 和 **无需账户的 BNPL UX** 攻略日本电商。27 億美元的收购额在当时属于日本 fintech 史上最大级别。^[extracted]

## 2. 业务板块图

| 板块 | 主要业务 | 特征 |
|---|---|---|
| BNPL 一次性支付 | Paidy（后付） | 邮箱 + 电话号码认证、授信 AI、加盟店手续费模型 |

| 卡 | Paidy Card / Card Plus | Visa 虚拟卡、支持 Apple Pay，高阶等级有实体卡 |

### 授信模型

- **电话号码 + 邮箱认证**：无需银行账户或信用卡注册，最快数秒完成授信判断
- **AI 评分**：根据交易历史 / 设备 / 行为数据动态计算信用额度
- **后付回收渠道**：便利店 / 银行转账 / 账户扣款 / Pay-easy
- **手续费模型**：加盟店侧手续费（高于信用卡）+ 分期时消费者侧手续费（一次性支付免费）

### 全球比较

- **美国 Affirm**：上市 BNPL，与 Amazon US 合作
- **瑞典 Klarna**：欧洲最大 BNPL，全球展开
- **澳大利亚 Afterpay**：Block（原 Square）于 2022 收购（约 290 億美元）

### 加盟店战略

- **Amazon Japan 官方 BNPL**：最大获客渠道，作为 Amazon 的“后付”选项常设
- **SHEIN / 跨境电商**：面向 Z 世代的时尚电商
- **Apple Store**：作为 iPhone / Mac 分期购买方式渗透年轻群体
- **Mercari**：跳蚤市场 + BNPL 交叉（注：Mercari 自身也在 [[payment-firms/mercari-hd]] 内运营 Merpay 后付，既是竞争方也是加盟店）

## 4. 监管 / 政策

- **主管**：经济产业省（分期销售法・作为少额包括信用购买中介业者登记）+ 金融厅（资金决算法相关）
- **分期销售法修订对应**（2020-06 公布，2021-04 施行）：“少额包括信用购买中介业者”区分新设 -> 包括 Paidy 在内的 BNPL 各公司成为登记对象。AI 授信模型被正式认可 ^[extracted]
- **个人信息保护法**：围绕基于电话号码 + 邮箱认证的授信所涉及的个人数据处理
