# GMO 币 (GMO Coin)

## Wiki 路径

本条目位于 [[payment-firms/INDEX|payment-firms INDEX]] 之下。请对照 [[securities-firms/monex-group|マネックスグループ (Monex Group)]] 阅读以获取同业 / 对比背景，并对照 [[banking/INDEX|banking index]] 了解更广泛的系统 / 监管边界。

## TL;DR

**GMO 互联网集团** 旗下的加密资产交易所。**关东财务局长 第 00006 号**（业界最早期登记的 1 家之一），JVCEA 第一种会员。以 **销售所 + 交易所 + 杠杆交易** 3 轴 + **20 种以上的可交易币种** 成为国内大型机构。与母公司 [[payment-firms/gmo-payment-gateway|GMO ペイメントゲートウェイ]] / GMO Click 证券 / [[payment-firms/gmo-aozora-net|GMO あおぞらネット銀行]] 并列的 GMO 金融生态系统的 **加密资产担当**。竞争对手为 [[payment-firms/bitflyer]] / Coincheck（[[securities-firms/monex-group]] 内）/ SBI VC Trade（[[megabanks/sbi-hd]] 内）/ DMM Bitcoin / bitbank / [[payment-firms/bittrade]]。

## 1. 公司概要

**正式名称**：株式会社 GMO 币
**英文名称**：GMO Coin, Inc.
**设立**：2016-10-21
**总部**：东京都涩谷区樱丘町 26-1 Cerulean Tower（GMO 互联网集团总部内）
**业态**：加密资产交换业者
**登记编号**：**关东财务局长 第 00006 号**（加密资产交换业）
**行业团体**：日本加密资产交易业协会（JVCEA）第一种会员
**母公司**：[[payment-firms/gmo-financial-hd|GMO フィナンシャルホールディングス]]（东证 PRIME 7177，表决权 100%）→ GMO 互联网集团（东证 PRIME 9449）

### 集团结构

```
GMO 互联网集团（9449 / 东证 PRIME）
  └── GMO 金融控股（7177 / 东证 PRIME）
        ├── GMO Click 证券 ── 网络证券・FX 业界最大级
        ├── GMO 币 ★ ── 本条目
        ├── GMO 外汇 ── 旧 FXプライム by GMO
        └── 海外 FX 据点（泰国・香港・英国等）
  └── [[payment-firms/gmo-payment-gateway|GMO ペイメントゲートウェイ]]（3769 / 东证 PRIME）── 支付代理业界最大
  └── [[payment-firms/gmo-aozora-net|GMO あおぞらネット銀行]] ── 与青空银行的 JV
```

## 2. 业务分部地图

| 分部 | 内容 | 特征 |
|---|---|---|
| 销售所 | 客户 vs GMO 币 的相对交易 | 点差方式・面向初学者 |
| 交易所（现货）| 撮合交易（客户间匹配）| 以 BTC / ETH / XRP / LTC / BCH 为中心 |
| **杠杆交易** | 加密资产 FX（最大 2 倍，个人）| 自业界初期起提供・主力收益来源 |
| 面向法人的 OTC | 大额相对交易 | 应对机构投资者・上市企业的加密资产处理 |
| 质押 | 部分 PoS 币种（ETH 等）| 对存入的加密资产给予奖励 |
| 借出加密资产 | 借贷 | 面向个人 |

**可交易币种**：BTC / ETH / XRP / LTC / BCH / XEM / XLM / BAT / OMG / XTZ / QTUM / ENJ / DOT / ATOM / ADA / MKR / DAI / LINK / DOGE / SOL / FCR / 等，约 20-30 种（视时点变动）。**以交易所形式提供的可交易币种数为国内最大级**。

**海外拓展**：有限。GMO 金融集团整体设有海外 FX 据点（泰国・香港・英国），但 GMO 币本体为 **基于国内牌照**。

## 3. 沿革

| 年月 | 事件 |
|---|---|
| 2016-10-21 | 作为株式会社 Z.com Coin 设立（GMO 互联网 子公司）|
| 2017-04-01 | 修订《资金决济法》施行（加密资产交换业 登记制开始）|
| 2017-05-26 | **加密资产交换业者登记（关东财务局长 第 00006 号）** ── 业界最早期登记的 1 家之一 |
| 2017-09-13 | 更改商号为「GMO 币」|
| **2018-01-26** | **Coincheck NEM 外流事件（约 580 億円）** ── 金融厅对全行业的集中检查・监管强化 |
| 2018-03 | 金融厅 业务改善命令（内部管理体制不健全，全行业统一处分）|
| 2018 后半 | 内部管理体制整备・合规强化 |
| 2019-05 | 《资金决济法》修订・《金商法》修订（加密资产衍生品 → 金商法，杠杆监管强化）|
| 2020-04 | 杠杆倍率 个人 4 倍 → 2 倍（伴随《金商法》修订）|
| 2022起 | 在 JVCEA 自主监管下扩大可交易币种（DOT / ATOM / ADA / SOL 等）|
| 2023～2024 | 探讨进入 NFT / Web3 领域（GMO 集团整体的方向性）|
| 2024 | 母公司 GMO FHD 中期经营计划明文写入「加密资产・Web3 强化」|

## 5. 监管・政策

- **主管**：金融厅（FSA）/ 关东财务局
- **登记**：加密资产交换业者（《资金决济法》第 63 条之 2）/ 关东财务局长 第 00006 号
- **自主监管**：JVCEA（日本加密资产交易业协会）第一种会员 ── 可交易币种审查・AML/KYC 统一基准
- **衍生品**：金商法 第一种金融商品交易业（加密资产衍生品交易・2019-05 修订以后）
- **2024起 政策议题**：
  - 稳定币制度（《资金决济法》修订 2023-06 施行）→ GMO 经由美国子公司具有先行经验
  - **税制**：加密资产损益 杂所得 累进课税 → 要求改为申报分离课税（业界共同）
  - 法人持有加密资产 期末时价评估 课税 → 修订进行中
  - JVCEA 可交易币种审查的提速（业界共同课题）
