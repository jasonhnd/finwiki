# SBI Remit

## Wiki 路径

本条目位于 [[JapanFG/INDEX|JapanFG index]] 之下。可与 [[payment-firms/wise-payments-japan|Wise Payments Japan]] 对读以理解同业 / 对比语境，并与 [[payments/INDEX|payments index]] 对读以把握更广的制度 / 监管边界。

## TL;DR

SBI Remit 是 [[megabanks/sbi-hd|SBI group]] 的国际汇款运营商。官方企业信息显示其第二种资金转移业者登记编号，并且它通过与外国人才、区域金融机构和银行代理业务的连接而具有较高重要性。

## 1. 法人 / 牌照边界

| 项目 | 内容 |
|---|---|
| 法定实体 | SBI Remit 株式会社 |
| 集团 | [[megabanks/sbi-hd|SBI Holdings]] / [[regional-banks/sbi-shinsei-bank|SBI Shinsei adjacency]] |
| 主要业务线 | 国际汇款、外国居民汇款、银行代理 / 地方银行合作 |
| 金融厅登记路径 | 金融厅资金转移名单：关东财务局长 第00008号，SBI Remit 株式会社。 |

## 2. 业务角色

- 提供从日本发起的国际汇款，覆盖外籍劳动者和在日外国居民使用场景。
- 通过银行和区域金融机构合作扩展。
- 对跟踪 ISO 20022 和 AML/CFT 压力之后的银行-fintech 汇款合作很重要。

## 3. 本独立页面为何重要

SBI 的银行、证券、加密资产和地方银行页面无法说明汇款运营商。SBI Remit 是其自身持牌的资金转移实体。
