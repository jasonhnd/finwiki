# 因菲库里翁 (Infcurion)

## Wiki 路径

本词条归属于 [[payment-firms/INDEX|payment-firms INDEX]]，是一家位于面向消费者的钱包下一层的**白标 fintech（white-label fintech）**供应商。请将其与 [[payment-firms/bnpl-landscape|Japan BNPL landscape]] 一同阅读以了解与 BNPL 相邻的支付背景，与 [[payment-firms/dg-financial-technology|DG Financial Technology]] 和 [[payment-firms/famima-digital-one|FamiMa Digital One]] 同业一同阅读以作为白标 / 商户侧 fintech 的对照，并与 [[payments/japan-bnpl-credit-purchase-boundary|BNPL and credit-purchase boundary]] 一同阅读以了解发卡机构 / 收单机构牌照边界的落点。[[payments/credit-purchase-card-operators-japan-index|METI credit-purchase registry]] 和 [[payments/japan-payment-scheme-economics-matrix|Japan payment scheme economics matrix]] 是注册与经济性的锚点。

## TL;DR

因菲库里翁是一家**白标 fintech 基础平台**供应商。通过**面向企业以 OEM 形式提供**咨询 + 处理（processing）+ 卡 / 钱包产品，使银行 / 流通 / 通信 / 自治体能够以自有品牌推出支付服务。与面向消费者的 BNPL（[[payment-firms/paidy|Paidy]] / [[payment-firms/net-protections-hd|Net Protections HD]]）或钱包（[[payment-firms/merpay|メルペイ]] / [[payment-firms/kyash|Kyash]] / [[payment-firms/smartbank|SmartBank]]）不同，它作为不出现在台前的**幕后平台运营方**，被定位于 fintech 存量之中。

## 2. 业务角色

- 作为银行 / 流通 / 通信 / 自治体以自有品牌推出"**Pay**""**卡**""**钱包**"的幕后方，提供处理、咨询及产品。^[extracted]
- 它不是直接的消费者品牌，而是作为 **B2B2C 支付平台运营方**，位于 fintech 的基础设施层。

## 3. 为何需要本独立页面

与 [[payment-firms/freee|freee]] / [[payment-firms/money-forward|Money Forward]] 这类 SaaS 不同，Infcurion 是一家**不拥有面向终端用户产品的 fintech 基础平台事业者**，是 JapanFG 上的独特类别。需要将其作为横跨 BNPL / 钱包 / 支付 PSP 各生态系统所提及的共同依赖（common dependency）独立出来处理。

## 4. 监管・政策

- **个人信息保护法**：由于采用 B2B2C 模式，终端用户数据的"数据主体是谁"在合同上至关重要。
- **分期付款销售法 / 资金结算法**：采用遵循提供对象合作伙伴登记要件的架构（自身直接登记的情形有限）。
