# Mercari HD（株式会社メルカリ）

## Wiki 路径

本词条位于 [[payment-firms/INDEX|payment-firms INDEX]] 之下。请对照 [[securities-firms/monex-group|マネックスグループ (Monex Group)]] 阅读以了解同业／对比背景，对照 [[banking/INDEX|banking index]] 了解更广的系统／监管边界。

## 摘要

日本最大级别的 **C2C 二手交易 App + 支付 + 加密资产** 垂直整合生态。2013-02-01 由山田进太郎创业，2018-06-19 东证 Mothers IPO（2022 迁移至 PRIME，东证 PRIME **4385**）。**Mercari（二手交易）+ Merpay（支付・后付）+ Mercoin（比特币）+ Mercard（JCB 合作）** 的 4 轴。国内 MAU **2,304 万**（FY2025.6 4Q）、Marketplace GMV **1.12 兆円**（FY2025.6 全年）、Mercard 发行 **300 万张超**、Mercoin 账户 **200 万超**。FY2025.6 全年营业收入 **1,926 億円**（YoY +3%）、本期利润 **261 億円**（YoY +94%、3 期连续最高利润），**美国业务实现首次全年盈利**（US GMV 728 百万美元 ≒ 1,091 億円）。未持有银行牌照（资金移动业者）。与巨型银行系 [[megabanks/mufg]] / [[megabanks/smfg]] 不同，特征在于「以市场（marketplace）为起点」的金融平台化。

## 1. 公司概要

**正式名**：株式会社メルカリ（英: Mercari, Inc.）
**证券代码**：东证 PRIME **4385**（2018-06-19 Mothers 上市 → 2022-04 迁移至 PRIME）
**设立**：2013-02-01（旧称: 株式会社コウゾウ → 2013-11 改称 Mercari）
**总部**：东京都港区六本木 6-10-1 六本木 Hills 森塔
**创业者・代表**：山田进太郎（代表董事 CEO，原 Unoh 创业者）
**业态**：C2C 市场 + 支付 + 后付 + 信用卡 + 加密资产 + 零工（spot work）

> 注: Mercari 不持有银行牌照，是通过 Merpay（资金移动业者）・Mercoin（加密资产交换业者）开展金融业务的、以电商为起点的集团。在本 wiki/JapanFG 内分类上的处理（是否作为「EC / Bank 一体型」垂直整合范例）属编辑判断，请参阅 `.opinions/JapanFG/mercari-hd.md`。

### 服务联动的特征

- **Mercari 销售额 → Merpay 余额 → 购买 Mercoin / Mercard 支付**: 在单一钱包内完成电商・支付・加密资产・授信的垂直整合（支付・后付的监管执行主体为资金移动业者 [[payment-firms/merpay|メルペイ (Merpay)]]）
- **Merpay 智能付**: 利用 Mercari 销售额作为「拟似收入证明」的授信模型
- **Mercard**: 虽为 JCB 品牌，但审核・授信采用 Merpay 自有标准
- **Fintech 债权余额**: 约 2,000 億円规模（FY2025.6 4Q 说明资料时点）── 后付 + Mercard 的授信累积

### 垂直整合战略

- **市场 + 支付 + 授信 + 加密资产**: 在单一 App 内完成的金融平台化
- **Mercoin**: 与乐天 Wallet / SBI VC Trade / bitFlyer 等不同，优势在于「二手交易销售额 → 加密资产」的导流
- **Mercard**: 经 JCB 合作 **实体 / 虚拟双支持**，可即时计提 Mercari 销售额

### 海外战略

- **美国业务（Mercari US）于 FY2025.6 全年首次盈利**: GMV 728 百万美元，分部利润 737 百万円
- 与裁员・成本削减措施并行，转入力求盈利稳定的阶段
- 回避与全球电商（Amazon / eBay）的直接竞争，转向国内深耕＋维持美国盈利

### 银行牌照（现状）

- 现状: **未持有自有银行**，由 Merpay 作为资金移动业者（关东财务局长 第 00058 号）+ 预付式支付手段发行者运营
- 未来取得银行牌照的情景・BaaS 合作 vs 自行取得的战略观测分离至 `.opinions/JapanFG/mercari-hd.md`（公开资料未确定）

## 4. 监管・政策

- **主管**:
  - Mercari 本体: 经济产业省（特定商交易法）
  - Merpay: **资金移动业者**（关东财务局长 第 00058 号，资金结算法）+ 预付式支付手段发行者
  - Mercoin: **加密资产交换业者**（关东财务局长 第 00029 号，资金结算法）
- **加入 JVCEA（日本加密资产交易业协会）**: Mercoin 为正式会员
- **加入 JADA（日本资金结算业协会）**: Merpay
- **近期政策议题**:
  - 2023〜 加密资产质押（staking）监管完善 → 关注对 Mercoin 的影响
  - 2024〜 BNPL（后付）监管讨论 → 对 Merpay 智能付的影响
  - 个人信息保护法・旧物营业法 → Mercari 本体的上架管理
