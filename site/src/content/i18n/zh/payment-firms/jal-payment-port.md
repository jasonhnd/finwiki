# JAL Payment Port

## Overview

JAL Payment Port Co., Ltd. 是 JAL Pay / 旅行钱包支付服务背后的运营主体。JAL 官方公司简介列出第三方型预付式支付工具发行、资金转移、银行代理、金融工具中介等业务；FSA 资金转移服务提供者名单将其登记为关东财务局长 第00062号，登记日期为 2018-08-15。

对 FinWiki 而言，这是连接航空会员、预付价值、转账能力、银行代理和金融产品中介的旅行 / 忠诚度钱包路径。

## 2. 业务角色

- JAL Pay 将航空忠诚度 / 客户 App 入口转化为钱包和支付路径。
- 公司简介明确把预付、资金转移、银行代理和金融工具中介活动组合在一起。
- 战略解读是围绕旅行的嵌入式金融：支付、账户联动、汇款 / 转账以及潜在投资产品中介，可以和里程及旅行预订流程并存。

## 3. 为什么需要这个独立页面

多数支付页面聚焦银行、电信、零售或平台生态。JAL Payment Port 展示的是旅行生态路径，更接近忠诚度驱动的嵌入式金融，而不是纯 PSP 或银行 App 竞争。应与 [[loyalty/japan-points-landscape|Japan points landscape]] 和 [[payments/japan-code-payment-competitive-map|code-payment competition]] 一起阅读。
