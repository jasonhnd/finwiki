# 财富导航株式会社 (WealthNavi)

## Wiki route

本条目位于 [[payment-firms/INDEX|payment-firms INDEX]] 之下。请将其与 [[payment-firms/rakuten-fg|楽天グループ金融事業 (Rakuten FG)]] 对照阅读以获取同业／对比背景，并与 [[fintech/japan-stablecoin-regulatory-landscape|日本 Stablecoin 法制度の三層構造（JPYC・USDC・Project Pax）]] 对照阅读以了解更广泛的系统／监管边界。

## TL;DR

国内机器人投顾（robo-advisory）型资产管理服务最大手。**受托资产突破 2 兆円（2026-05-12 公布）**，运用者数约 41.6 万人（2024-09 末）。由 2015-04-28 柴山和久（前麦肯锡、财务省出身）创立，2016-07 服务开始。2020-12-22 东证 Mothers 上市（证券代码 **7342**，现 Growth）。**2024-12-12 三菱 UFJ 银行 [[megabanks/mufg]] 通过公开收购（TOB）宣布完全子公司化 ★** ── 国内机器人投顾的先驱被整合进 megabank AI 金融生态系统的转折点。^[extracted]

## 2. 事业分部地图

| 分部 | 服务 | 特征 |
|---|---|---|
| 个人资产管理 | WealthNavi（自有）| 全自动・长期分散投资 / 以美国 ETF 为中心 |
| 合作（B2B2C）| WealthNavi for SBI / マネックス / 索尼银行 / 横滨银行 等 | 白标供应，活用合作方客户基础 |
| NISA 支持 | おまかせ NISA | つみたて投资额度 + 成长投资额度 一体化运用 |
| 功能选项 | 自动定投 / 再平衡 / DeTAX（税务优化）| 遵循美国机器人投顾的功能 |
| 投顾 | 生活规划诊断 / 目标金额模拟 | 面向投资无经验者的入门引导 |

### 运用方针

- 基于**现代投资组合理论（MPT）**的分散投资
- **以美国上市 ETF 为中心**（VTI / VEA / VWO / AGG / GLD / IYR / TIP 系的组合）
- 根据客户的风险承受度的 **5 级投资组合**
- **自动再平衡**（半年 1 次 + 偏离触发时）

### 商业模式特性

- **手续费统一 1.1%（含税）**：与投信信托报酬分开，与受托资产联动的存量型收益
- **客户获取成本（CAC）**：通过白标合作得以抑制，自有直销以数字广告为中心

## 4. 监管・政策

- **主管**：金融厅（FSA）
- **业务登记**：第一种金融商品交易业者・投资运用业者
- **加入协会**：日本证券业协会，一般社团法人 日本投资顾问业协会
- **客户资产保全**：信托保全（在三井住友信托银行 等分别管理）^[extracted]
- **近期政策论点**：
  - **2024-01～ 新 NISA 制度**：对整个机器人投顾行业构成顺风（つみたて + 成长投资额度扩充）
  - **2025～ 客户本位的业务运营原则**：对手续费水平・运用业绩披露透明性的要求强化
  - **2024～ 金融服务中介业 制度**：白标合作的监管整理
