# 乐天集团金融业务（Rakuten FG）

## Wiki route

本条目位于 [[payment-firms/INDEX|payment-firms INDEX]] 之下。请与 [[payment-firms/gmo-payment-gateway|GMO ペイメントゲートウェイ]] 对照阅读以获取同业 / 对比背景，并参见 [[fintech/japan-stablecoin-regulatory-landscape|日本 Stablecoin 法制度の三層構造（JPYC・USDC・Project Pax）]] 了解更广义的系统 / 监管边界。

## TL;DR

楽天グループ株式会社（Rakuten Group, Inc., 东证 PRIME 4755）的金融科技板块。**以 EC 为轴的独特模式**：以 乐天市场 + 乐天积分 为核心，将 银行 / 证券 / 卡 / 保险 / 支付 捆绑为「乐天经济圈」的交叉销售战略。与大型银行系（[[megabanks/mufg]] / [[megabanks/smfg]] / [[megabanks/mizuho-fg]]）或通信系 FG（[[megabanks/ndfg]] / au-FH / [[megabanks/paypay-fg]]）不同，采取 **以商务为起点 → 金融** 的独特业务构建。2023-04 乐天银行 IPO，2024 乐天证券接受瑞穗证券 49% 出资。**2024 期 FinTech 板块 Non-GAAP 营业利润 1,534 億円（+37.9%）、集团 IFRS 营业利润 扭亏为盈（530 億円）**。乐天移动亏损（Non-GAAP 营业亏损 2,163 億円）仍高于 FinTech 利润，但呈改善趋势。创始人・三木谷浩史（1965-）会长兼社长是自 1997 创业以来的代表者。

## 1. 公司概要

**正式名称**：楽天グループ株式会社（控股公司・综合企业集团）

**代表者**：三木谷浩史（会长兼社长 CEO，1997 创业）

### 「乐天经济圈」模式

- **乐天积分 = OS**：EC 购买・通信・卡・银行・证券・保险・旅游 全部给予/使用积分
- **SPU (Super Point Up Program)**：以「乐天移动签约 / 乐天卡使用 / 乐天银行工资入账 / 乐天证券投信定投」等使乐天市场购物的积分倍率分阶段提升 → 金融产品交叉销售的强力驱动器
- **会员 ID 统一**：以 1 乐天 ID 贯通全部服务 → CRM 数据整合的优势

### EC 轴的独特性（vs 通信系 / 大型银行系）

| 竞争类别 | 轴 | 代表 |
|---|---|---|
| **EC 轴** | **商务 → 金融** | **Rakuten FG（本项）** |
| 通信轴 | 通信线路 → 金融 | [[megabanks/ndfg]]（docomo）/ au-FH（KDDI）/ [[megabanks/paypay-fg]]（Yahoo/Z）|
| 银行轴 | 商业银行 → 数字化 | [[megabanks/mufg]] / [[megabanks/smfg]] / [[megabanks/mizuho-fg]] |
| 专业网络轴 | 网络证券 → 扩展 | [[megabanks/sbi-hd]] |

### 移动亏损 × 金融补贴结构

- 2019-10〜 乐天移动 MNO 进入 → 因基站投资使 **数年规模的大幅亏损** 常态化
- 2024 期: Mobile Non-GAAP 营业亏损 2,163 億円 vs FinTech Non-GAAP 营业利润 1,534 億円 → **FinTech 仍无法直接抵消 Mobile 亏损**，但凭改善幅度（Mobile +850 億・FinTech +37.9%）使 2024 期 IFRS 营业利润 扭亏为盈（530 億円）
- FinTech 板块（尤其 乐天银行 / 乐天卡）的稳定收益，支撑集团合并 EBITDA
- 2022-2024 期间: 乐天证券 IPO 计划 → 以接受瑞穗证券 49% 出资进行 **募资 + IPO 撤回** 的重组
- 2023-04 乐天银行 IPO 实施 → 集团流动性 + 上市账面价值显现化

### 与瑞穗 FG 的资本业务合作

- 2023-10 瑞穗证券 19.99% 出资 → 以 2024 分阶段提高至 **49%**
- 乐天证券 IPO 实际撤回
- 对瑞穗侧而言获得零售・网络客户基础，对乐天侧而言获得集团募资 + 与大型银行系证券的合作深化

## 4. 监管・政策

- **主管**: 金融厅（FSA）
- **按业态监管**:
  - 乐天银行: 银行法（网络银行）
  - 乐天证券: 金融商品交易法（第一种金融商品交易业）
  - 乐天卡: 分期销售法 + 放贷业法
  - 乐天生命 / 乐天损保: 保险业法（寿险・损保各自）
  - 乐天钱包: 资金结算法（加密资产交换业者登记 = JVCEA 加盟）
  - 乐天 Pay / 乐天 Edy: 资金结算法（预付式 / 资金转移业）
- **控股公司监管**: 乐天集团本体为事业控股公司（非纯粹控股公司）。各业态子公司分别受业法监管
- **近期政策议题**:
  - 乐天移动 频率分配 / 设备投资 → 总务省管辖，但对金融板块有间接影响
  - 2024〜 新 NISA 制度 → 乐天证券账户增势
  - 稳定币 / 加密资产监管修订 → 波及 乐天钱包
