# Aflac 生命保险（Aflac Japan）

## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[life-insurers/prudential-japan|プルデンシャル・ジブラルタ・ファイナンシャル (Prudential Japan)]] for peer / contrast context and [[insurance/INDEX|insurance index]] for the broader system / regulatory boundary.

## TL;DR

这是美国 Aflac Incorporated（NYSE: AFL，总部位于美国乔治亚州哥伦布）的日本子公司。其于 1974-11  推出日本首个癌症保险，并在第三领域保险（癌症 / 医疗）中成长为国内 No.1  级的保有契约规模。**2018-04-02 年从旧日本分公司转换为日本法人“アフラック生命保険株式会社”**并子公司化，同年 **2018-07-12 又与第一生命 HD（[[dai-ichi-life]]）达成战略业务与资本合作**，第一生命以长期持有为目的取得约 7% 的议决权基础持股。通过营业职员 + 代理店 + 银行柜台销售 + 网络的 4  渠道，它持续维持第三领域市场份额。^[extracted]

## 1. 公司概况

**正式名称**：アフラック生命保険株式会社 ^[extracted]  
**英文名**：Aflac Life Insurance Japan Ltd.  
**母公司**：Aflac Incorporated（NYSE: AFL，总部位于美国乔治亚州哥伦布，Fortune 500 企业、S&P 500  成分股）^[extracted]  
**日本分公司设立**：1974-11 （旧 American Family Life Insurance Company 日本分公司；推出日本首个癌症保险）^[extracted]  
**日本法人化**：2018-04-02（旧分公司业务吸收转换至新设“アフラック生命保険株式会社”）^[extracted]  
**总部**：东京都新宿区西新宿（Aflac Square）^[extracted]  
**代表人**：代表取締役社長 古出眞敏（Masatoshi Koide，2018-01〜）  
**员工数**：约 4,800 名（按近年 Aflac 生命保险公开披露口径）  
**保有契约**：约 2,300 万 件规模（按 Aflac 集团日本分部披露口径）

### 主要股东与资本结构 ^[extracted]

- **由 Aflac Incorporated（美国母公司）在议决权基础上全资持有**

- 美国母公司 Aflac Incorporated 自身的股东以机构投资者为主（主要是 Vanguard / BlackRock / State Street 等被动持仓；Wikipedia / 10-K）

## 2. 业务分部地图

| 分部 | 产品 / 特征 |
|---|---|
| **癌症保险** | 以日本首个癌症保险（1974-11  上市）为起点的核心产品，包括“生きるためのがん保険”系列、终身癌症险、青年向癌症险、女性向产品等 |
| **医疗保险** | “ちゃんと応える医療保険 EVER”、收入支持 / 就业不能类产品等。第三领域中，“癌症 + 医疗”套装销售是核心 |
| **护理保险** | “アフラックの介護保険”（与公共长期护理保险联动） |
| **生死两全 / 终身** | “未来の自分が決める保険 WAYS”“アフラックの終身保険どなたでも”（免告知型）等 |
| **学资 / 储蓄** | 如“アフラックの夢みるこどもの学資保険”等储蓄型产品 |

### 沿革年表 ^[extracted]

| 年月 | 事件 |
|---|---|
| 1955  | American Family Life Insurance Company 在乔治亚州哥伦布成立 |
| 1958  | 在美国开始销售癌症保险（美国首个癌症保险产品） |
| **1974-11** | **American Family Life Insurance 日本分公司设立，并推出日本首个癌症保险（当时在日本拥有癌症保险的独占性批准）** ^[extracted] |
| 1980s〜90s | 在日本第三领域形成独占地位（1980-90  年代，日本主要寿险公司因审批限制难以进入癌症 / 医疗险） |
| 2001  | 第三领域监管放松（日美保险协商结果）→ 国内大型寿险与损保系寿险开始进入，竞争加剧 |
| 2008  | 全面采用“Aflac”企业品牌（从旧“American Family”称呼逐步过渡） |
| 2009  | 在日本正式大规模启用 Aflac Duck 作为广告角色 |
| 2013  | Aflac × 日本邮便业务合作（在邮局柜台销售 Aflac 癌症保险）—— 第三领域跨渠道扩张 |
| **2018-04-02** | **从旧日本分公司转换为“アフラック生命保険株式会社”（通过新设公司方式承继全部业务；美国母公司 100% 子公司）** ★ ^[extracted] |
| **2018-07-12** | **宣布与第一生命 HD（[[dai-ichi-life]]）达成战略业务与资本合作。第一生命 HD 以长期持有为目的取得美国 Aflac Incorporated 约 7% 的议决权基础持股（分两阶段）。** ★ ^[extracted] |
| 2019〜 | 与第一生命合作深化（营业职员交叉销售、共同开发产品、联合营销） |
| 2020〜2022  | 疫情期间医疗险 / 就业不能险需求扩大，营业职员转向远程销售 |
| 2024〜 | Aflac Japan 中期经营战略：强化数字直销、深挖第一生命渠道、推出应对老龄化的产品 |

### 数字战略

- **数字直销**：强化“Aflac Direct”（全线上完成型），争取年轻客群与第二张保单
- **代理平台**：面向代理店的平板提案工具、电子签约基础设施
- **联络中心 AI**：AI 化咨询处理、自动化合同查询
- **美国母公司数字投资**：Aflac Incorporated 整体扩大 AI / 云预算（10-K 披露）

### 合作与并购战略

- **与第一生命 HD（[[dai-ichi-life]]）的战略业务与资本合作（2018-07-12）** ★ ^[extracted]
  - 第一生命 HD 以长期持有为目的取得约相当于 7% 议决权基础的 Aflac Incorporated 股份（公告时为分两阶段取得计划）
  - 通过营业职员渠道进行交叉销售（第一生命营业职员可销售 Aflac 癌症保险）
  - 共同开发产品、联合营销、信息系统协作
  - 战略目的：对第一生命而言是强化第三领域（癌症 / 医疗）产品线；对 Aflac 而言是获得约 9 万 人规模销售网络的接入
- **与日本邮便的销售合作（自 2013〜）**：在全国邮局柜台销售癌症保险，取得地方零售网络入口 ^[extracted]
- **大和证券 / 瑞穗证券等银行柜台销售**：通过银行柜台渠道提供第三领域保险（具体代销机构会随时期变化）
- **代理网络**：与损保系代理店、乘合代理店保持合作

### 海外战略

- Aflac 生命保险（日本法人）基本上**专注于日本国内业务**
- 全球扩张由美国母公司 Aflac Incorporated 负责，美国分部（职域销售的补充医疗保险）对日本分部形成补充
- 在美国母公司层面，业务集中于日美 2  国，新兴市场扩张有限 ^[extracted]

### B2C 品牌化

- **Aflac Duck**：会叫“Aflac!”的鸭子角色。美国自 2000 年〜、日本自 2009 年〜展开全面投放。日本广告中使用过西田敏行、宫崎葵、川荣李奈、松田翔太、大野智等多位艺人。
- **“生きるための保険”**：癌症保险的品牌口号（从癌症幸存者视角出发的信息表达）
- **企业色**：Aflac 蓝（信赖）+ Aflac Duck 白

## 4. 监管与政策

- **主管机关**：金融厅（FSA）—— 基于日本《保险业法》的注册与监管
- **美国母公司**：SEC / NAIC / 乔治亚州保险局
- **《保险业法》**：第三领域保险商品许可、偿付能力边际监管、契约者保护
- **国际资本规则**：应对 ICS（Insurance Capital Standard，IAIS 主导）
- **近期政策议题**：
  - **2024〜 IFRS 17  / 国际会计准则衔接**（美国母公司采用 US GAAP，日本子公司采用日本准则，但会影响集团报告）
  - **新偿付能力监管（经济价值基础）** 于 2025-04  适用 → 评估以第三领域为中心的负债结构
  - **全数字合同与身份核验强化**（eKYC、犯收法应对）
  - **客户本位业务运作原则**（FSA）应对 —— 加强代理店监控

### 竞争对手（国内第三领域 / 医疗险）

- **第一生命 [[dai-ichi-life]]**：既是合作方，也是第三领域竞争者（癌症险 / 医疗险产品线）
- **日本生命 [[nippon-life]]**：日本最大的寿险公司，正在扩大第三领域份额
- **三井住友海上あいおい生命 [[msad]]**：MS&AD 集团，以医疗险为主力
- **MetLife 生命 [[metlife-japan]]**：外资系，聚焦医疗险与终身险
- **Zurich 生命**：医疗险网络直销能力强
- **ORIX 生命**：专注医疗与终身，兼做直销 + 代理
- **損保ジャパンひまわり生命**（[[non-life-insurers/sompo|sompo-hd]] 系）
- **東京海上日動あんしん生命**（[[non-life-insurers/tokio-marine|tokio-marine-hd]] 系）

> 从 1974〜2001  的近独占地位出发，在 2001 年 第三领域自由化之后市场进入群雄割据状态。Aflac 依靠“癌症保险品牌”与销售渠道多样性，维持了 No.1  级位置。^[extracted]
