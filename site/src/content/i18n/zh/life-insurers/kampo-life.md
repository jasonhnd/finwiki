# 简保生命保险（Japan Post Insurance）

## Wiki 路径

本条目归属于 [[life-insurers/INDEX|life-insurers INDEX]]。请对照 [[life-insurers/dai-ichi-life|第一ライフグループ / 旧 第一生命 HD]] 了解同类 / 对比背景，并对照 [[insurance/INDEX|insurance index]] 了解更广泛的系统 / 监管边界。

## TL;DR

**旧简易保险（1916 开始）民营化后的承继子公司**。隶属日本邮政集团旗下，东证 PRIME 7181（2015-11-04 母子 3 社同时上市）。并表总资产约 59.6 兆円（59兆5,556億円，2025-03 期末），以全国邮局网络（约 2 万所）为零售基础的国内大型寿险公司。因 **2019-06〜2020 不当销售事件**（旧合同与新合同的双重收费、面向高龄者的不当销售等约 18 万件以上）而被处以 2019-12 业务改善命令、2020-01 经营层集体辞职、2020-04 业务停止命令（暂停营业 3 个月），这一历史性事件的遗留影响犹存。在中期经营计划下力求恢复信任与再增长。随着民营化推进，日本邮政分阶段出售，政府持股比率正逐步下降。^[extracted]

## 1. 公司概况

**正式名称**：株式会社简保生命保险 ^[extracted]
**英文名**：Japan Post Insurance Co., Ltd.
**证券代码**：东证 PRIME 7181（2015-11-04 上市）^[extracted]
**成立**：2006-09-01（作为邮政民营化准备公司）^[extracted]
**开业**：2007-10-01（与邮政民营化同时开始营业）^[extracted]
**总部**：东京都千代田区大手町 2-3-1
**业态**：国内大型寿险公司（旧简易保险民营化后的承继）
**母公司**：日本邮政株式会社（并表子公司，持股比率分阶段出售中）^[extracted]

### 集团结构

```
日本邮政（控股公司・上市 6178）
  ├── 日本邮便株式会社（100%）── 邮政・物流・邮局运营
  ├── 邮储银行（东证 PRIME 7182，分阶段出售使持股比率逐步下降）
  └── 简保生命保险（东证 PRIME 7181，分阶段出售使持股比率逐步下降）
        ├── 面向个人的寿险产品（终身・养老・学资・医疗特约 等）
        └── 以日本邮便作为主要销售代理店签约
```

> 在民营化方案下，日本邮政的方针是分阶段处置简保生命与邮储银行的股份 → 政府持股比率（经由日本邮政）持续下降。^[extracted]

## 2. 业务板块・图谱

| 板块 | 主要产品・渠道 | 特征 |
|---|---|---|
| 面向个人的寿险 | 终身保险・养老保险・学资保险 等 | 基于旧简保的传统产品 + 民营化后的新产品 |
| 医疗・第三领域 | 医疗特约・癌症特约 等 | 民营化后的扩充领域 |
| 销售渠道（本体）| 自营直营店 | 有限 |
| 销售渠道（合作）| 日本邮便 = 约 2 万所邮局（销售代理店合同） | **最大的差异化要素**・覆盖全国 |
| 客户层 | 高龄者・地方居民・中小规模个人 | 持有大量自旧简保时代以来的长期投保人 |

### 旧简易保险品牌与邮局网络

- **1916 创设的简易保险品牌**对地方・高龄者层的渗透极深，民营化后仍以**全国约 2 万所邮局网络**作为零售基础加以运用 → 形成其他寿险公司所无的物理可及性优势
- 银行柜面销售・网络销售相较其他公司（[[life-insurers/dai-ichi-life]] Frontier 生命等）起步较晚

### 产品多样化与民营化后的制约

- 旧简保时代因属国营事业，故在**保险金额上限・产品设计上存在制约** → 民营化后分阶段实现产品多样化（医疗・癌症・终身高额合同等）
- 但在民营化方案下被要求「**顾及其他寿险公司**」，产品审批是监管机关（金融厅・总务省）审慎审查的对象
- 旧合同（简保）与新合同（简保生命）并存成为**双重收费问题的温床** ^[extracted]

### 不当销售事件与信任恢复

- **2019-06〜2020 不当销售事件**（约 18 万件以上的违规调查）★ — 旧合同与新合同同时投保（双重收费）、产生无保险空窗期、面向高龄者的不当销售、合同转换时的不利益告知遗漏等
- 2019-12 业务改善命令、2020-01 经营层集体辞职、2020-04 业务停止命令（暂停营业 3 个月）
- 在新经营层下**强化合规・重新审视营业指标・严格化面向高龄者的劝诱规则**
- 实施顾客回馈・合同复原计划
- 治理重建：强化内部举报・合规体制・监控

### 政府持股比率的下降

- 日本邮政（母公司）**分阶段出售简保生命股份**被纳入民营化方案

### 数字化・以顾客为中心的战略

- 在 2024 中期经营计划下，将**数字化・提升以顾客为中心的服务**置于主轴
- 推进无纸化・在线手续・引入聊天机器人等
- 通过与全国邮局网络的联动，志在**数字 + 实体双轴**

## 4. 监管・政策

- **控股公司规制**：日本邮政属于金融控股公司规制对象（银行・保险双方）
- **民营化法**：日本邮政股份・简保生命股份的分阶段处置义务
- **业务改善命令历史**：2019-12 / 2020-04（**3 个月业务停止**）— 对国内大型寿险公司而言属罕见的重度行政处分 ^[extracted]
- **普遍服务义务**：经由邮局网络在全国提供金融服务（民营化法上的要求）→ 也影响简保生命销售渠道的维持
- 近期政策论点：
  - 完全民营化（解除政府持股）的时间表
  - 旧投保人保护（自简保时代以来的长期合同的处理）
  - 营业指标管理・面向高龄者的销售规制（事件后规制强化）
