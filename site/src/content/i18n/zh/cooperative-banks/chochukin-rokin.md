# 中央劳动金库 / Central Labour Bank

## Wiki 路径

本条目位于 [[cooperative-banks/INDEX|cooperative-banks INDEX]] 之下，是**日本 13 家 rokin（劳动金库）运营主体中规模最大的一家**，在 rokin 体系中对应于 [[cooperative-banks/kyoto-chuo-shinkin|京都中央信用金库]]（shinkin）与 [[cooperative-banks/osaka-shogin-shinkumi|大阪商银信用组合]]（shinkumi）那类运营主体锚点。可结合 [[cooperative-banks/rokin-renraku|劳动金库连合会 (Rokinren)]] 理解 cooperative-system central institution，结合 [[banking/rokin-bank-registry-japan|rokin registry]] 把握 13 家 rokin 全貌，并结合 [[banking/cooperative-banking-japan|cooperative banking domain]] 理解更广泛的监管边界。

## TL;DR

中央劳动金库（Chuo Rokin）是覆盖关东地区的劳动金库，在 13 家 rokin 中拥有最大的存款与资产规模之一。依据《劳动金库法》，其营业基础覆盖东京、神奈川、埼玉、千叶、茨城、栃木、群马、山梨。其成员以**工会、消费合作社、NPO 等劳动相关组织，以及个人劳动者**为主，核心业务是劳动者福利金融，例如住房贷款、教育贷款与生活相关融资。

它既是 13 家 rokin 中的最大成员，也是全国 rokin 体系中具有代表性的运营主体之一。其模式是**“合作组织型 + 面向劳动者的限定用途”**，既不是股份制普通银行，也不是面向中小企业的信用组合，更不是 JA Bank 那样的农业协同体系，而是专门聚焦劳动者福利金融的金融机构。^[extracted]

### 主要业务 ^[extracted]

| 业务 | 内容 | 行业位置 |
|---|---|---|
| **个人存款** | 面向劳动者家庭的流动性存款与定期存款 | 与银行、信用金库同类 |
| **住房贷款** | 面向劳动者的住房取得资金 | 主力产品 |
| **教育贷款** | 大学入学与教育支出融资 | 主力产品 |
| **汽车贷款** | 汽车购置资金 | 主力产品 |
| **用途型贷款** | 婚礼、丧葬、医疗等支出 | 面向劳动者福利 |
| **财形储蓄** | 工资代扣、住房资产形成 | 劳动者专属产品 |
| **团体交易** | 工会、消费合作社、NPO 的结算与账户服务 | 成员组织特色业务 |
| **国债 / 投资信托销售** | 通过系统内渠道办理（经由 Rokinren） | 范围有限 |

### ALM / 资金流 ^[extracted]

```text
成员基础（劳动者家庭 + 成员工会、消费合作社、NPO）
  -> 存款 -> 中央劳动金库
              -> 面向地区劳动者的住房贷款、教育贷款、资产形成等
              -> 存款剩余资金 -> 系统内运用
                                  -> 机构投资 + 回馈
                                  -> 中央劳动金库 -> 向成员返还红利与服务
```

### 在 13 家 rokin 中的代表性

- 是 [[cooperative-banks/rokin-renraku|Rokinren]] 体系内的**最大成员**，对系统方向具有很强影响力。
- 参与由 Rokinren 统筹的 **R-ONE 系统** 与 IT / JASTEM 类基础设施协同。^[extracted]

### 住房贷款主力模式

- 中央劳动金库的主力产品是**面向劳动者的住房贷款**。
- 与 Flat 35、合作组织属性、成员优待利率等组合使用。
- 与 megabank、区域银行、shinkin 等形成竞争，但其通过成员资格带来的优惠利率是关键差异点。

### 数字化战略

- **Rokin App**：余额查询、转账、住房贷款余额管理等。^[extracted]
- **R-ONE 共同系统**：由 Rokinren 统筹的 13 家 rokin 共同基础设施。^[extracted]

## 5. 监管 / 论点

- **监管方**：金融厅 / 关东财务局（金融稳健性）+ 厚生劳动省（合作组织监管）。
- **法律依据**：劳动金库法（第 227 号法律）。
- **存款保险**：与一般金融机构相同的 1,000 万円上限 + 利息。
- **当前议题**：
  - **Rokin App + R-ONE 系统** 的数字化推进
  - 利率正常化时期的**住房贷款利率策略**
  - 在少子老龄化下开发面向年轻劳动者的新产品
  - **ESG 投资 / 社会金融** 的强化（如 NPO、团体融资）
