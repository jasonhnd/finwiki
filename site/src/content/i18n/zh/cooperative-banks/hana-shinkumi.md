### 组合员资格 ^[extracted]

由于是民族系信组，组合员资格为**业域 + 民族社区**的双重结构（依据中小企业等协同组合法）：

1. **业域要件**：以工商业者・个体经营者为中心（严格限制员外利用）
2. **民族社区**：在日韩国・朝鲜人 + 其配偶・亲属・关联事业者
3. **地区要件**：在以关东圈为中心的章程指定地区居住・在职・设有事业所者

### 朝银系信用组合的历史经纬 ^[extracted/inferred]

| 年代 | 事件 |
|---|---|
| 1950〜60 年代 | 面向在日朝鲜人（主要为支持北朝鲜系）工商人的互助金融**「朝银」**系信用组合在各地设立 |
| 1970〜80 年代 | 在全国主要城市确立朝银信组网络 |
| 1990 年年代 | 泡沫破裂使**面向不动产・建筑业的融资**不良债权化严重加剧 |
| 2001〜2002  | **朝银信组连锁破产问题** — 全国数家朝银信组经营破产，由存款保险机构进行大规模资金援助（花信组的承继对象包含朝银东京・朝银千叶・朝银新潟・朝银长野・朝银关东共 5 家组合，事业重整投入了约 4,000 億円的公共资金） |
| 2002-03-20 | **花信用组合设立**（同年 12-30 开业）。商号「花（ハナ）」在朝鲜语中意为「一个・统合」，象征多家原朝银信组的事业整合 |
| 2010 年年代〜 | 应对在日社区的世代交替 + 加强 AML/CFT 应对 |

### 主要业务 ^[extracted]

| 业务 | 内容 |
|---|---|
| **面向中小企业融资** | 在日韩侨 SME 的周转资金・设备资金（餐饮・零售・弹珠机・不动产・建筑等） |
| **个人存款** | 面向组合员家庭的定期・流动性存款 |
| **住房贷款** | 面向组合员的住房取得资金 |
| **国际汇款** | 面向韩国・社区的跨境汇款（民族系信组的固有需求） |
| **社区金融** | 面向组合员的教育・婚丧嫁娶等生活资金 |
| **资产运用咨询** | 国债・投信代销（经系统） |

## 6. 监督・监管

- **主管**：金融厅・关东财务局
- **法律依据**：中小企业等协同组合法 + 关于协同组合金融事业的法律（协金法）
- **存款保险**：[[financial-regulators/dic|預金保険機構]] 一般金融机构额度
- **近期政策议题**：
  - **加强 AML/CFT**（跨境汇款监控）— 民族系信组固有的监管应对成本 + **对北朝鲜制裁相关**的制裁监控要件
  - **组合员世代交替**（在日 2 世・3 世以后疏离信组的风险）
  - 与**韩国系银行（新韩银行东京支店等）**的竞争
