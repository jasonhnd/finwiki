# JF 共水联 (全国共济水产业协同组合联合会 / Kyosuiren)

## Wiki 路径

本条目归属于 [[cooperative-banks/INDEX|cooperative-banks INDEX]]，作为 **JF 集团合作型互助保险全国联合会**，是农业合作系统中 [[non-life-insurers/ja-kyosairen|JA 共済連]] 的 JF 侧对应方。它弥补了一处审计缺口：此前 JA 共济联得到了深入记录，而 JF 侧的对应方（JF 共水联）却没有锚点。请结合 [[cooperative-banks/jf-zengyoren|JF 全漁連 (JF グループ代表 + 経済事業)]] 阅读以了解 JF 集团代表联合会，结合 [[cooperative-banks/jf-shingyoren|JF 信漁連]] 了解 JF 信用联合会层，结合 [[cooperative-banks/jf-marine-bank-hokkaido|JF Marine Bank 北海道]] 了解都道府县信用锚点，结合 [[cooperative-banks/norinchukin|農林中央金庫]] 了解 JF Marine Bank 系统中央机构，并结合 [[insurance/INDEX|insurance domain]] 了解合作型与股份制保险形态的对比。

## TL;DR

**JF 共水联 (全国共济水产业协同组合联合会) = JF (渔业协同组合) 集团互助事业的核心**。以水产业协同组合法为根据的**全国联合会**，是 JA 集团中相当于 [[non-life-insurers/ja-kyosairen|JA 共済連]] 的渔协系互助事业者。**会员 = 全国 JF (渔业协同组合) + 都道府县渔联**，向组合员 (渔业者・地区居民) 提供**生命共济・年金共济・建筑物更生共济・汽车共济・渔船・渔业设备共济**等。**与 JA 共济联 (200 兆円规模) 相比，其规模约为数 %〜10 % 左右** (因渔业产值相对农业较小)，但凭借**面向渔业者的固有共济 (渔船共济・渔业者灾害共济等)** 而具有独特性。在 JF 集团的三大支柱 ([[cooperative-banks/jf-zengyoren|JF 全漁連 = 代表 + 経済]] / **JF 共水联 = 互助 ★本页** / [[cooperative-banks/jf-shingyoren|JF 信漁連 = 信用]]) 中，承担**专营互助事业**的全国联合会职能。^[extracted]

### 与 JA 共济联的规模对比

- **JA 共济联 = 200 兆円规模 vs JF 共水联 = 数千亿〜数万亿日元规模** = 反映渔业与农业的经济规模差
- 但凭借**渔船共济等固有商品**维持与民营财险的差异化

### 渔业者组合员特性

- **海上职业风险** (海难・失踪・天候) → 共济需求的特殊性
- **季节集中型收入** (随渔期变动的所得波动) → 共济掛金支付时机的调整
- **高龄化与后继者问题** → 共济参加者的世代变化应对

## 5. 监督・监管

- **主管**：**农林水产省** (主) + **金融厅** (协助监督) + **水产厅** (渔业政策)
- **法律根据**：水产业协同组合法 (共济事业条款) + 共济事业指南
- **近期政策论点**：
  - **共济 = 保险的监管缺口**议论 (与 JA 共济联同一论点)
  - 对共济类似适用**经济价值基准・偿付能力 (ESR)** 监管之论
  - **渔船共济法制度**的运用与 JF 共水联的角色
  - **食品安全保障计划 (2026〜)** 中渔业共济的功能再定义
  - **2018 渔业法修订** + 对渔业者结构变化的应对
