# 劳动金库连合会 (The Rokinren Bank)

## Wiki route

This entry sits under [[cooperative-banks/INDEX|cooperative-banks INDEX]]. Read it against [[megabanks/mufg|三菱 UFJ FG (MUFG)]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

劳动金库连合会是以全国 13  家劳动金库为会员的「ろうきん」业态中央金融机构。1955年设立，以劳动金库法为根据法，承担会员劳金的资金供需调整、剩余资金运用、内国汇兑·财形储蓄等的集中处理，以及业态统一在线系统的运营。是与 [[cooperative-banks/zenshin-kumi|全信組連]]·[[cooperative-banks/shinkin-central|信金中央金庫]] 并列的协同组织金融的中央机构。

官方简介显示，截至 2025-03-31，资金量为 6兆2,093億円、贷出金余额为 9,925億円、自有资本比率为 15.83%。出资金为 1,600億円，常勤董监事及员工人数为 417 人。

## 2. 业务板块图

| 业务 | 内容 | 作用 |
|---|---|---|
| 资金供需调整 | 与会员劳金的存款·贷出交易 | 业态内中央银行 |
| 剩余资金运用 | 将会员劳金的剩余资金进行市场运用 | 机构投资者职能 |
| 金融职能补充 | 内国汇兑、国债·投信窗口销售等 | 会员劳金的共同服务基础 |
| 事务集中 | 账户划转、财形储蓄等 | 业务效率化 |
| 系统运营 | R・ONE 系统的开发·运营 | IT 共通基础 |

官方说明指出，劳金连吸收会员劳金剩余资金的 58% 作为存款，而劳金连所吸收的存款·可转让存款中 80% 来自会员劳金。

## 3. 战略·竞争

- **协同组织金融的中央机构**：劳金连并非营利银行，而是承担勤劳者福祉金融的劳动金库的中央机构。
- **与信金·信组的比较**：[[cooperative-banks/shinkin-central|shinkin-central]] 支撑信用金库，[[cooperative-banks/zenshin-kumi|zenshin-kumi]] 支撑信用组合，劳金连支撑劳动金库。
- **系统共同化**：在地区金融机构成本上升的局面下，共同系统·事务集中成为竞争力的源泉。
- **政策性与市场运用兼顾**：在支撑勤劳者福祉金融的同时，将剩余资金于金融市场运用，因此兼具 ALM 与社会使命两个层面。

## 4. 监管·政策

- 以劳动金库法为根据的协同组织金融机构。
- 包含银行法准用在内的披露·风险管理·监督规则的适用对象。
- 在金融厅·财务局的监管之下，AML/CFT、网络安全、系统风险管理至关重要。
