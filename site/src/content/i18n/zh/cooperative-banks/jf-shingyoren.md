# JF 信渔连 (信用渔业协同组合连合会 / Prefectural Credit Federations of Fishery Cooperatives)

## Wiki 路径

本条目位于 [[cooperative-banks/INDEX|cooperative-banks INDEX]] 之下，作为 [[banking/jf-marine-bank-system-japan|JF Marine Bank system]] 的**都道府县信用联合会层**，位于地方 JF（71 渔协）与全国中央机关 [[cooperative-banks/norinchukin|農林中央金庫]] 之间。请与 [[cooperative-banks/ja-shinnoren|JA 信連]]（农业侧的对应机构）、[[cooperative-banks/jf-zengyoren|JF 全漁連]]（代表 / 经济事业联合会）以及 [[banking/cooperative-banking-japan|cooperative banking domain]]（更广泛的语境）一并阅读。

## TL;DR

**JF Marine Bank system 的都道府县中间层 = 信用渔业协同组合连合会（信渔连）**。以水产业协同组合法为依据的**都道府县或地域单位的信用事业联合会**。据 [[banking/jf-marine-bank-system-japan|JF Marine Bank system]] 官方资料（截至 2025-04-01），**10 家信渔连**在运营（按邻接县整合的地域来组织，而非按 47 个都道府县）。其作用为 **（1）汇集域内 JF（71 渔协）的余资**、**（2）将余资向 [[cooperative-banks/norinchukin|農林中央金庫]] 缴存 + 自主运用**、**（3）向域内 JF 提供融资・流动性供给**、**（4）提供 JF Marine Bank system 共通基础设施**。与 [[cooperative-banks/ja-shinnoren|JA 信連]]（31 机关）属同型结构，但因渔协业界规模较小 + 都道府县整合推进，**信渔连数约为 1/3**。以**「71 JF → 10 信渔连 → 1 农林中金」的三段结构**，承担地域渔业的安全网 + 系统金融职能。^[extracted]

## 1. 组织结构

| 项目 | 内容 |
|---|---|
| 通称 | JF 信渔连 / 信渔连 |
| 英文名 | Prefectural Credit Federation of Fishery Cooperatives (Shingyoren) |
| 法律形态 | **依据水协法的地域・都道府县联合会**（信用事业联合会） |
| 设立 | 1948 以后，按各地域分别设立 |
| 信渔连数 | **10 家信渔连**（截至 2025-04-01）^[extracted] |
| 监督官厅 | 金融厅（信用事业）+ 农林水产省・水产厅（协同组合监督） |

### JF Marine Bank system 的三段结构 ^[extracted]

```
JF Marine Bank system（3 层结构）
  Layer 1: 全国 71 家JF（个别渔业协同组合・信用事业实施组合）
                ↓ 余资缴存
  Layer 2: 10 家信渔连（地域・都道府县中间层）★本页面
                ↓ 余资缴存
  Layer 3: 1 农林中央金库（中央）→ [[cooperative-banks/norinchukin]]
                ↓ 机构投资・系统返还
```

### 与 JA 信连的对比 ^[extracted]

| 轴 | [[cooperative-banks/ja-shinnoren|JA 信連]] | JF 信渔连 |
|---|---|---|
| 依据法 | 农业协同组合法 | 水产业协同组合法 |
| 中间层数（2025-04） | **31 家机构** | **10 家机构** |
| 个别组合数 | 497 家JA | 71 家JF |
| 系统规模 | 大（JA Bank 存款超 100 兆円） | 小（数万亿日元规模 — JF 系统储金于 平成22年年度末为 2兆2,544億円） |
| 都道府县构成 | 部分都道府县已转向直连 | 以地域整合（邻接县合并）为主体 |
| 上位机关 | 农林中央金库（共通） | 农林中央金库（共通） |
| 监督 | 金融厅 + 农水省 | 金融厅 + 农水省・水产厅 |

### 4 大职能 ^[extracted]

| 职能 | 内容 |
|---|---|
| **余资汇集** | 汇集域内 JF 的余资（组合员存款的超过部分） |
| **向中央缴存** | 将汇集的余资作为系统存款上缴 [[cooperative-banks/norinchukin|農林中央金庫]] |
| **自主运用** | 一部分由信渔连自身进行机构投资 |
| **域内 JF 支援** | 经营指导・流动性供给・系统共同化 |

### 规模感

- **储金规模（JF 系统整体）**：数万亿日元规模 — JF（渔协）系统储金于 平成22年年度末为 **2兆2,544億円**（相对于 [[banking/ja-bank-system-japan|JA Bank]] 超 100 兆円约为 1/50  规模）。据农林中金披露（2024-03 末），JF 系统贷款约为 0.5 兆円。
- **个别信渔连的总资产**：从数百亿到 8千亿日元规模 — 最大的 [[cooperative-banks/jf-marine-bank-hokkaido|北海道信漁連]] 总资产约 8,099 億円・储金等余额约 7,654 億円（令和7年3月末），其下并列规模更小的地域信渔连。

> 注：各信渔连的详细财务请参照各信渔连披露（例：北海道信渔连 https://marinebank.jp/disclosure/ ）・农林中金披露。

### 在 JF Marine Bank system 内的定位

- 以 **JF Marine Bank Basic Policy**（2015〜）加强 **JF + 信渔连 + 农林中金的一体化运营** ^[extracted]

### 整合・重组动向

- 由都道府县分立的 47 家信渔连 → 经邻接县整合缩减为 **10 家信渔连** ^[extracted]

## 4. 监督・监管

- **主管**：金融厅（信用事业的稳健性）+ 农林水产省・水产厅（协同组合监督）
- **法律依据**：水产业协同组合法（信用事业联合会条款）
- **近期政策议题**：
  - **信渔连的进一步整合・转向直连方式**
  - **渔业法修订（2018）**对信渔连影响的评估
  - **粮食安全保障计划（2026〜）**中对信渔连职能的重新定义
  - **2024〜农林中金美国债浮亏**对信渔连的分红影响
