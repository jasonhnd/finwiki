# 信用组合爱知商银（Aichi Shogin Credit Cooperative）

## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]] as the **Chubu (中京) representative of the 商銀系 (in-Japan Korean business) shinkumi network**, complementing [[cooperative-banks/osaka-shogin-shinkumi|大阪商銀信用組合]] (Kansai) and routing through [[banking/credit-cooperative-registry-japan|Japan credit cooperative registry]] for the broader 143-cooperative population.

### 组合员资格 ^[extracted]

作为民族系信组，组合员资格具有 **业域 + 民族社区** 的双重结构（依据《中小企业等协同组合法》）：

1. **业域要求**：工商业者（以在日韩国 / 朝鲜人经营者为中心，严格限制员外利用）
2. **民族社区**：在日韩国 / 朝鲜人，以及其配偶、亲属、相关事业者
3. **地区要求**：居住、工作或在以爱知县为中心的章程指定地区设有营业场所者

### 在全信组连中的位置

- **[[cooperative-banks/zenshin-kumi|全信組連]]** 东海板块会员机构
- 东海 15  家信组中、爱知 8  家信组之一翼
- ATM 联动与余资运用主要经由全信组连

### 商银系信组的历史背景 ^[extracted]

| 年代 | 事项 |
|---|---|
| 1950〜60  年代 | 作为面向在日韩国 / 朝鲜人工商业者的互助金融，“商银”系信组在各地设立 |
| 1970〜80  年代 | 在名古屋、东京、大阪等主要城市形成商银信组网络 |
| 1990 年 时代 | 泡沫崩溃后，**不动产 / 建设业贷款**不良化 |
| 2002〜2003 | **关西商银信组连锁破产问题** + 朝银系信组破产问题 |
| 2003〜 | 向承接信组进行业务转让 + **商银系信组重组整合**推进 |
| 2010 年 时代起 | 在日社区代际更替 + AML/CFT 强化 |

### 主要业务 ^[extracted]

| 业务 | 内容 |
|---|---|
| **中小企业贷款** | 面向在日韩国 / 朝鲜人 SME 的营运资金与设备资金（餐饮、零售、柏青哥、不动产、建设等） |
| **个人存款** | 面向组合员家庭的定期 / 活期存款 |
| **住房贷款** | 面向组合员的购房资金 |
| **国际汇款** | 向韩国的跨境汇款（民族系信组特有需求） |
| **社区金融** | 面向组合员的教育、婚丧等生活资金 |
| **资产运用咨询** | 通过系统销售国债与投信 |

## 6. 监管 / 规则

- **主管**：金融厅・东海财务局
- **法律依据**：《中小企业等协同组合法》 + 《协同组合金融事业法》
- **存款保险**：[[financial-regulators/dic|預金保険機構]] 一般金融机构框架
- **近期政策议题**：
  - **AML/CFT 强化**（跨境汇款监控）—— 民族系信组特有的监管成本问题
  - **组合员代际更替**（在日 2  世、3  世以后远离信组的风险）
  - **韩国系银行进军中京** —— 与新韩银行名古屋支店等竞争
