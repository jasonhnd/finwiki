# 信用组合爱知商银 (Aichi Shogin Credit Cooperative)

## Wiki 路径

本条目位于 [[cooperative-banks/INDEX|cooperative-banks INDEX]] 之下，作为 **商银系 (在日韩国人经商) 信组网络的中部 (中京) 代表**，补充 [[cooperative-banks/osaka-shogin-shinkumi|大阪商銀信用組合]] (关西)，并经 [[banking/credit-cooperative-registry-japan|Japan credit cooperative registry]] 通往更广义的 143-合作社母集团。

### 组合员资格 ^[extracted]

因属民族系信组，组合员资格为 **业域 + 民族社区** 的双重结构 (依据中小企业等协同组合法):

1. **业域要件**: 工商业者 (以在日韩国·朝鲜人事业者为中心，严格限制员外利用)
2. **民族社区**: 在日韩国·朝鲜人 + 其配偶·亲属·关联事业者
3. **地区要件**: 在以爱知县为中心的章程指定地区内居住·在职·设有事业所者

> 注: 爱知 8 家信组体现 **业域 / 民族系 / 地域型** 的多层结构，可从中读出信组业态的多样性。

### 在全信组连内的定位

- **[[cooperative-banks/zenshin-kumi|全信組連]]** 的东海区块会员行
- 东海 15 家信组中爱知 8 家信组的一翼
- 系统 ATM 联动·余资运用经全信组连进行

### 商银系信组的历史背景 ^[extracted]

| 年代 | 事件 |
|---|---|
| 1950〜60 年代 | 作为面向在日韩国·朝鲜人工商业者的互助金融，各地设立了 **「商银」** 系信组 |
| 1970〜80 年代 | 在名古屋·东京·大阪等主要城市确立了商银信组网络 |
| 1990 年年代 | 泡沫破裂导致 **面向不动产·建筑业的融资** 不良债权化 |
| 2002〜2003  | **关西商银信组连锁破产问题** + 朝银系信组破产问题 |
| 2003〜 | 向承接信组的业务转让 + **商银系信组的重组·整合** 推进 |
| 2010 年年代〜 | 应对在日社区的世代交替 + AML/CFT 强化 |

### 主要业务 ^[extracted]

| 业务 | 内容 |
|---|---|
| **中小企业融资** | 在日韩国·朝鲜人 SME 的运转资金·设备资金 (餐饮·零售·弹珠机·不动产·建筑等) |
| **个人存款** | 面向组合员家庭的定期·流动性存款 |
| **住房贷款** | 面向组合员的住房取得资金 |

| **社区金融** | 教育·婚丧嫁娶等面向组合员的生活资金 |
| **资产运用咨询** | 国债·投信窗口销售 (经系统) |

## 6. 监督·规制

- **主管**: 金融厅·东海财务局
- **法律依据**: 中小企业等协同组合法 + 协同组合金融事业相关法律 (协金法)
- **存款保险**: [[financial-regulators/dic|預金保険機構]] 一般金融机构框架
- **近期政策议题**:
  - **AML/CFT 强化** (跨境汇款监视) — 民族系信组固有的规制应对成本
  - **组合员世代交替** (在日 2 世·3 世以降脱离信组的风险)
  - **韩国系银行进入中京** — 与新韩银行名古屋支店等的竞争
