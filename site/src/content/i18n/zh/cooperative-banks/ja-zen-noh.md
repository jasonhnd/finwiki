# JA 全农 (全国农业协同组合连合会 / National Federation of Agricultural Cooperative Associations)

## Wiki route

This entry sits under [[cooperative-banks/INDEX|cooperative-banks INDEX]] as the **JA Group economic-business federation**, parallel to [[cooperative-banks/norinchukin|農林中央金庫 (信用)]] / [[non-life-insurers/ja-kyosairen|JA 共済連 (共済)]] / [[cooperative-banks/ja-zenchu|JA 全中 (代表)]]. Read with [[banking/ja-bank-system-japan|JA Bank system]] for the credit-side adjacency and [[banking/cooperative-banking-japan|cooperative banking domain]] for the system boundary.

## TL;DR

**JA 集团 4  大全国连的经济事业核心 = 全国规模的采购 + 销售的事业连合会**。1972  设立（全贩连 + 全购连 合并），以农业协同组合法为根据的**连合会**。事业内容为 **(1) 采购 = 饲料·肥料·农药·农业机械·燃料·生活物资的调配与供给**、**(2) 销售 = 米·麦·园艺·畜产品的集货与流通**、**(3) 加工·物流·直营店铺（A-Coop 等）**。**取扱额为年间数万亿日元规模**，是日本最大级的农产品流通组织。[[cooperative-banks/norinchukin|農林中金]] 负责金融，JA 全农负责 **「物」的流动**，呈互补关系。全国 JA → JA 全农 → 市场·大型流通 → 消费者 的流动，以及 **逆向的采购流动（厂商 → JA 全农 → 全国 JA → 农业者）**，是两大事业线。**其自身是不经营存款或保险的非金融事业体**，但与 JA Bank 同样以全国 JA 全部 497  组合为客户基础。^[extracted]

### 整合史 ^[extracted]

| 年 | 事项 |
|---|---|
| 1948 | **全贩连**（全国销售农业协同组合连合会）设立 |
| 1948 | **全购连**（全国采购农业协同组合连合会）设立 |
| 1949 | 按行业分别发足各种连合会（米麦·园艺·畜产等） |
| **1972-03** | **全贩连 + 全购连 合并发足「全国农业协同组合连合会（JA 全农）」** |
| 1992 | 随「JA」品牌统一，**JA 全农** 通称化 |
| 2002〜 | [[banking/ja-bank-system-japan|JA Bank]] 整合启用，JA 全农与金融作为事业维持分离 |
| 2014 | **规制改革会议上 JA 全农的「股份公司化」成为政治论点** |
| 2015〜 | **JA 全农 自我改革** 推进（流通效率化·成本削减） |
| 2017 | 促进面向美国的农产品出口（设立大米出口公司等） |

| **畜产** | 牛·猪·鸡的集出货 + 食肉加工（全农 Chicken Foods 等） |
| **海外** | 通过美国子公司（CGB 等）开展货运·装船事业 |

### 金融互补关系

- **物的流动** = JA 全农，**钱的流动** = [[cooperative-banks/norinchukin|農林中央金庫]] + [[cooperative-banks/ja-shinnoren|信連]] + JA Bank
- 农业者的采购款·销售款 **经由 JA 存款账户** 结算 → 形成增厚 JA Bank 存款基础的结构 ^[extracted]
- 大型商社（三井物产·住友商事·伊藤忠商事等）既是 JA 全农的竞争对手，又是交易对象

### 自我改革与外压

- **2014  规制改革会议**：强烈要求全农「股份公司化」「事业整合存废」「下调生产资材成本」 ^[extracted]
- **2015〜 年度自我改革计划**：饲料·肥料成本下调 10%、流通效率化、强化海外出口 ^[extracted]
- 股份公司化最终被 **搁置** — 维持连合会形态，但事业效率化持续

### 海外拓展

- **美国子公司 CGB**（Consolidated Grain and Barge）：美国谷物集货·物流大手
- **促进海外出口**：大米·牛肉·果蔬的海外销售（美·亚·EU 市场）

### 竞争结构

- **大型商社**：与三井物产·住友商事·伊藤忠商事的生产资材 + 食品流通事业竞争
- **大型食品厂商**：在米谷流通上与大型大米批发商（Pearl Rice 系·神明等）竞争
- **流通大手**：直营 A-Coop 与地方超市·便利店（Seven·AEON）竞争

## 4. 监管·规制

- **主管**：农林水产省（连合会监督）+ 公正交易委员会（反垄断法 — 取扱集中度的论点）
- **法律根据**：农业协同组合法（连合会条款）+ 对连合会的组合员加入规制
- **近期政策论点**：
  - **下调生产资材成本**（改革会议持续论点）
  - **促进农产品出口**
  - **粮食安全保障计划（2026〜）** 中 JA 全农角色的再定义
  - 对 **数字农业（智慧农业）** 的应对
