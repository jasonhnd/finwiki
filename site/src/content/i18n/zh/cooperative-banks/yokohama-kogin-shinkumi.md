# 横滨幸银信用组合（Yokohama Kogin Credit Cooperative）

## Wiki 路线

该条目位于 [[JapanFG/INDEX|JapanFG index]] 之下，是 [[banking/credit-cooperative-registry-japan|Japan credit cooperative registry]] 所收录 143 cooperative 群体中的**首都圈（神奈川）民族系商业 shinkumi 锚点**。可结合 [[cooperative-banks/osaka-shogin-shinkumi|大阪商銀信用組合]] 阅读 Korean-business shinkumi 网络中的关西对应方，并结合 [[cooperative-banks/zenshin-kumi|全信組連]] 理解合作组织体系母体。

## 要点

横滨幸银信用组合是总部位于神奈川县横滨市中区尾上町、以**在日韩国 / 朝鲜人系事业者为母体的民族系信用组合**。FSA registry No. 58 （关东财务局管辖），法人编号 3020005003571。其成员基础包括神奈川县内的**在日商工人 SME** 以及**Koreatown（鹤见、川崎、横滨中华街周边）**社群，是关东圈 **幸银系 / 商银系** 民族信组网络的一翼。它是 [[cooperative-banks/zenshin-kumi|全国信用協同組合連合会 (全信組連)]] 会员，并受 [[financial-regulators/dic|預金保険機構]] 覆盖。该机构以**地域 + 业域 + 民族社区**三层成员资格运营，跨境汇款、社区金融和 SME 融资是业务核心。^[extracted]

### 成员资格 ^[extracted]

由于是民族系信组，成员资格由**地区 + 业域 + 民族社区**三层构成（依据中小企业等协同组合法）：

1. **地区要求**：居住、工作或拥有事业所于以神奈川县为中心的章程指定地区者
2. **业域要求**：商工业者（以在日韩国 / 朝鲜人事业者为中心，严格限制成员外利用）
3. **民族社区**：事实上的成员基础是在日韩国 / 朝鲜人 + 其配偶、亲属和相关事业者

### 关东神奈川信组区块（registry 比较）

神奈川县在 FSA registry 上有 6  个信用组合（registry No. 55-60）：

| 信组 | 总部 | 类型 |
|---|---|---|
| 小田原第一信用组合 | 小田原市 | 县西部地域型 |
| 神奈川县医师信用组合 | 横滨市中区 | 业域（医师） |
| 神奈川县牙科医师信用组合 | 横滨市中区 | 业域（牙科医师） |
| **横滨幸银信用组合** ★本页 | **横滨市中区** | **民族系（在日韩国 / 朝鲜人）** |
| 相爱信用组合 | 爱甲郡爱川町 | 县央地域型 |
| 信用组合横滨华银 | 横滨市中区山下町 | 民族系（华侨系） |

> 注：**横滨幸银**（在日韩国 / 朝鲜人系）和 **横滨华银**（华侨系）共存于同一地址地区（中区），形成神奈川县代表性的民族系信组 2 行 体制。

### 在全信组连中的位置

- **[[cooperative-banks/zenshin-kumi|全信組連]]** 的关东区块会员行
- 关东 49  信组中神奈川 6  信组的一翼
- 系统 ATM 联动和余资运用经由全信组连

### 主要业务 ^[extracted]

| 业务 | 内容 |
|---|---|
| **中小企业融资** | 面向在日韩国 / 朝鲜人 SME 的营运资金和设备资金（餐饮、零售、不动产、建设等） |
| **个人存款** | 面向成员家庭的定期和流动性存款 |
| **住房贷款** | 面向成员的住房取得资金 |
| **国际汇款** | 面向韩国的跨境汇款（民族系信组固有需求） |
| **社区金融** | 婚嫁、教育、冠婚葬祭等面向成员的生活资金 |
| **资产运用咨询** | 通过系统进行国债和投信柜台销售 |

## 6. 监督和监管

- **主管**：金融厅、关东财务局
- **法律依据**：中小企业等协同组合法 + 关于协同组合金融事业的法律（协金法）
- **存款保险**：[[financial-regulators/dic|預金保険機構]] 一般金融机构框架
- **近期政策论点**：
  - **AML/CFT 强化**（跨境汇款监控）— 民族系信组固有监管应对成本
  - **成员代际更替**（在日 2  世、3  世以后离开信组的风险）
  - **与韩国系银行（新韩银行东京支店等）竞争**
