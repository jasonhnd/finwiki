# JF 全渔联（全国渔业协同组合联合会 / National Federation of Fisheries Cooperative Associations）

## Wiki 路径

本条目位于 [[JapanFG/INDEX|JapanFG index]] 之下，是 **JF Group 的全国代表联合会**，与农业协同组织体系中的 [[cooperative-banks/ja-zenchu|JA 全中]] 平行。可结合 [[cooperative-banks/norinchukin|農林中央金庫]] 理解中央金融机构，结合 [[banking/jf-marine-bank-system-japan|JF Marine Bank system]] 理解信用体系边界，并结合 [[banking/cooperative-banking-japan|cooperative banking domain]] 理解更广泛的协同组织金融范围。

## 摘要

**JF（渔业协同组合）Group 的全国代表组织。** 于 1952  设立，以《水产业协同组合法》（水协法）为依据，是**全国联合会**。不同于 JA Group 的 4  大全国联（[[cooperative-banks/ja-zenchu|全中]] + [[cooperative-banks/ja-zen-noh|全農]] + [[non-life-insurers/ja-kyosairen|共済連]] + [[cooperative-banks/norinchukin|農林中金]]），在渔协集团中，**JF 全渔联兼具代表 + 经济事业（销售、采购、系统水产加工等）**；另有 **JF 共水连（全国共济水产业协同组合联合会）**承担共济事业，信用事业则由都道府县的 [[cooperative-banks/jf-shingyoren|信漁連]] + 中央 [[cooperative-banks/norinchukin|農林中央金庫]] 负责。**会员 = 全国渔协（JF）71  个组合 + 都道府县渔联 + 相关法人**（与 [[banking/jf-marine-bank-system-japan|JF Marine Bank system]] 相同母集団）。业务包括 **(1) 水产品销售（鱼市场 / 流通）**、**(2) 采购（渔具 / 燃料）**、**(3) 渔业政策运动 / 代表功能**、**(4) 系统水产加工 / 出口促进**。规模为 JA 全农的数 % 水平（渔业生产额小于农业）。^[extracted]

### 重要时间线 ^[extracted]

| 年 | 事项 |
|---|---|
| 1948  | 制定《水产业协同组合法》，战后渔协重组开始 |
| **1952** | **全国渔业协同组合联合会（JF 全渔联）设立** |
| 1960-70  年代 | 渔协数量达到峰值（约 2000  个组合），以沿岸渔业村落渔协为主体 |
| 1990-2000 年 年代 | 渔业生产额下降 + 渔业从业者老龄化 -> 渔协数量大幅减少 |
| 2000 年 年代 | **“JF”（Japan Fishery cooperatives）品牌统一化** |
| 2015 起 | **渔协改革**——规制改革会议讨论“增强渔协竞争力” |
| **2018 起** | **《渔业法》修正**（70  年来根本性修正）——调整渔业权分配，重新定义渔协角色 ^[extracted] |
| 2020 起 | **JF Marine Bank system** 一体运营强化（[[banking/jf-marine-bank-system-japan|JF Marine Bank]] 的 boundary） |
| 2024  | **截至 2025-04-01，71  个 JF + 10  个信渔联 + 1  农林中央金库**（据 [[cooperative-banks/norinchukin|農林中金]] 公开资料）^[extracted] |

### 主要业务 ^[extracted]

| 业务 | 内容 |
|---|---|
| **水产品销售** | 全国 JF 集货水产品通过批发市场销售和出口 |
| **采购事业** | 提供渔具、渔网、燃料、冰、养殖饲料 |

| **系统水产加工** | 加工流通的系统交易 |
| **教育 / 信息提供** | JF 役员培训、水产信息杂志等 |

### 与 JF Marine Bank system 的关系 ^[extracted]

- **JF 全渔联 = 代表 + 经济事业**。
- **[[banking/jf-marine-bank-system-japan|JF Marine Bank system]] = 信用事业框架**，由 [[cooperative-banks/jf-shingyoren|JF 信漁連]]（都道府县）+ [[cooperative-banks/norinchukin|農林中央金庫]]（中央）构成。
- JF 全渔联**不是 JF Marine Bank 的直接运营者**，但作为集团整体代表参与。

### 渔业改革与渔协存在意义

- **2018  渔业法修正**：70  年来的根本性改革，**渔业权分配自由化** -> 渔协以外经营者进入空间扩大 ^[extracted]。
- JF 全渔联主张**“渔协是地方渔业基础设施”**，以食品安全保障 + 地方创生框架应对。
- 但**年轻渔业者离开渔协**以及**大型水产公司（Maruha Nichiro、日本水产等）业务扩大**，对渔协基础形成缩小压力。

### 水产品出口促进

- 作为**“水产品出口 5 兆円 目标”**的一翼，强化经由 JF 系统的出口。
- 因面向中国的禁输问题（2023 起），替代市场开拓成为课题。

### 渔业资源管理

- 协作运营 **TAC（可捕量）制度**。
- 应对**资源恢复计划**和渔业 IT 化。

### 强项 / 弱项

**强项**
- 唯一**汇总全国渔协**的组织，在渔业政策中具有代表性。
- 与水产厅关系**密切**。
- 在食品安全保障框架下具有政治分量。

**弱项**
- **渔业生产额本身缩小**（1980 年 年代峰值 1200 万 吨 -> 2020 年 年代 400 万 吨级）。
- 年轻渔业者**脱离 JF** 的压力（倾向与大型水产公司直接交易）。
- 与大型水产公司（上市企业）竞争时存在**规模劣势**。

## 4. 监督 / 监管

- **主管**：水产厅（农林水产省内）+ 内阁府规制改革推进室（改革讨论）
- **法律依据**：《水产业协同组合法》（水协法）+ 《渔业法》（2018  修正）
- **近期政策论点**：
  - **渔业法修正的运营**（渔业权分配 / 新进入者）
  - **水产品出口促进**（5 兆円 目标）
  - **资源管理（TAC）与渔协角色**
  - **食品安全保障计划（2026 起）中的 JF 全渔联功能再定义**
  - **面向中国出口停止**（2023 起）的应对
