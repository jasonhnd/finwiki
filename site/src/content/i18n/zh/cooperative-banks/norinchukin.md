# 农林中央金库 (Norinchukin Bank)

## Wiki 路径

本词条位于 [[cooperative-banks/INDEX|cooperative-banks INDEX]] 之下。请对照 [[megabanks/mufg|三菱 UFJ FG (MUFG)]] 阅读以了解同业／对比背景，对照 [[banking/INDEX|banking index]] 了解更广的系统／监管边界。

## 摘要

**世界最大级别的合作社系金融机构**。1923 作为产业组合中央金库设立，1943改称「农林中央金库」，经 1986/2001 农林中央金库法修订转入现代银行业务。是**并非股份公司的特殊法人**，出资者为 JA（农协）・JF（渔协）・森林组合等系统团体。承担 [[banking/ja-bank-system-japan|JA Bank system]] 与 [[banking/jf-marine-bank-system-japan|JF Marine Bank system]] 的 national central institution layer，兼具系统信用事业之顶点 + **国内屈指可数的机构投资者** 的双重性格。合并总资产为 **91 兆 7,243 億円**（2024-09-30）。**2008 雷曼冲击中因美国次贷相关亏损约 1.5 兆円**，**2024 年针对美国国债等浮亏实施资本增强**（2024-05-23 公开研议约 1.2 兆円规模 → 2024-08-01 以后配出资 7,360 億円 + 期限劣后贷款约 6,000 億円合计确定为 **约 1.3 兆円**规模）^[extracted] — 2 年度的大规模机构投资亏损浮现为治理议题。

## 1. 公司概要

**正式名**：农林中央金库 ^[extracted]
**英文名**：The Norinchukin Bank
**简称**：农林中金 / 农中 / Nochu
**设立**：1923（作为产业组合中央金库）^[extracted]
**现法人成立**：1943-03（战时整合中改称「农林中央金库」）
**根据法**：农林中央金库法（昭和 61 年法律第 93 号号，1986 全面修订，2001 大幅修订）^[extracted]
**形态**：**合作社系金融机构的特殊法人**（并非股份公司，也未上市）^[extracted]
**总部**：东京都千代田区有乐町 1-13-2

### 出资结构（系统团体出资）^[extracted]

- **主要出资者**：JA（农业协同组合・联合会）・JF（渔业协同组合・联合会）・森林组合等系统团体
- **相当于股份公司「股东」的是「出资者」**：经 JA 全中・JA 全农・JA 共济连等上级团体出资
- **派息对象**：向系统团体派息是本业之一（系统回馈）
- 未上市（**并非东证/JPX 上市标的**）

### 主要据点

```
农林中央金库（总部：东京・有乐町）
  ├── 国内分行：札幌・仙台・名古屋・大阪・福冈 等
  ├── 海外据点：
  │     ├── 纽约分行
  │     ├── 伦敦分行
  │     └── 新加坡分行
  ├── 系统团体：
  │     ├── JA Bank（含农业协同组合・全国联合会的系统）
  │     ├── JF Marine Bank（渔业协同组合系统）
  │     └── JForest（森林组合系统）
  └── 子公司・关联公司：
        ├── 农中信托银行
        ├── 农林中金全共连资产管理（农林中央金库与全国共济农业协同组合联合会＝JA 共济连共同出资的资产运用公司）
        └── 其他金融服务子公司
```

## 2. 业务分部・地图

| 分部 | 主要经营者 | 特征 |
|---|---|---|
| **系统信用事业（中央银行职能）**| 农林中央金库 本体 | JA Bank・JF Marine Bank・JForest 系统的中央银行・最后贷款人 |
| **机构投资业务** | 总部 + NY/London/Singapore | **与本业相当的收益来源** — 国内外债券・股票・另类资产・证券化产品 |
| 面向系统团体的融资 | 本体 | 向农业协同组合・渔业协同组合・森林组合供应资金 |
| 面向农林水产业的融资 | 本体 + 经由系统 | 向农业法人・渔业者・林业者的直接 + 经由系统融资 |
| 国际业务 | NY/London/Singapore | 外币资金筹措・海外投资・国际结算 |
| 信托・年金 | 农中信托银行 | 系统团体年金・运用受托 |
| 资产管理 | 系统共同出资 AM | 系统运用产品・公募投信 |
| 信用卡等 | （过去）持有三菱 UFJ Nicos 的历史 | 2017-10 出售给 MUFG |

### 双重性格：系统中央银行 + 国内屈指可数的机构投资者

- **「上游」：JA Bank 系统的中央银行** — 将全国 JA 汇集的**充裕存款（系统存款）一手汇总**并运用的机制
- **「下游」：作为机构投资者进行国际运用** — 将仅靠国内农业资金需求无法消化的巨额资产，以国内外证券・债券・另类资产运用
- → **号称「农业金融」而实质为世界最大级别机构投资者** 的结构性特征

### 国际分散投资战略

- 以 **美国国债・公司债・MBS** 为中心的外币运用
- 在纽约・伦敦・新加坡 3 个据点运用・筹措
- 在 2008 次贷 / 2024 美利率急涨 **两个年度均在美国市场遭受大规模亏损** 2 

### 2024-2025 投资组合重构 ^[extracted]

- 2024 年公开方针：实现美国国债等浮亏的出售（10 兆円规模的低收益资产出售计划）+ 资本增强（后配出资 7,360 億円 等）+ 风险资产再配置
- 压缩对出资 JA 等的派息 → 系统内治理议题

### 系统金融重组

- JA Bank 全国系统（2002〜）的中央参与者
- 与都道府县信联（[[cooperative-banks/ja-shinnoren|信用農業協同組合連合会]]）的角色分工重组
- 渔协系统（JF Marine Bank）・森林组合系统（JForest）也具有同样的中央银行职能

### 竞争定位

- **与巨型银行（[[megabanks/mufg]] / [[megabanks/smfg]] / [[megabanks/mizuho-fg]]）的比较**：商业银行业务有限，本业为系统金融 + 机构投资
- **与寿险（[[life-insurers/nippon-life]] / [[life-insurers/dai-ichi-life]]）的比较**：作为机构投资者处于类似定位，运用余额属同一序列
- **与政策金融机构（[[financial-regulators/jfc]] / [[financial-regulators/dbj]]）的比较**：在面向农林水产业的融资上部分竞争 / 互补

## 4. 监管・政策

- **主管**：金融厅（FSA）+ **农林水产省**（系统金融部分）— 双重监督为特征 ^[extracted]
- **根据法**：农林中央金库法（昭和 61 年法律第 93 号号）
- **近期政策议题**：
  - 对出资 JA 的派息・系统回馈水平的妥当性
  - 系统金融重组（JA Bank 整合・与信联的角色分工）
  - 作为机构投资者的 ESG / 气候风险披露

### 代表职务（理事长）

- 2024-2025 当时的理事长是 **奥和登**。因 FY2025-03 期出售外债预计计提约 1.9 兆円规模的净亏损，奥理事长表明辞任意向（Bloomberg 2025-02-20 报道）
- 最新的高管体制・任期请参阅农林中央金库 官方 IR / 信息披露刊物
