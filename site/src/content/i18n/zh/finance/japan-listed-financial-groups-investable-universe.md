# Japan listed financial groups investable universe

## Wiki route

It is a public-source reference universe for Japan-listed financial groups, read against [[finance/japan-ib-league-table]] for adviser-side peer mapping, [[finance/japan-activist-investor-playbook]] for engagement / control reading, [[finance/japan-large-shareholding-disclosure]] for stake disclosure mechanics, [[finance/japan-shareholder-proposal-and-agm-voting-route]] for AGM / stewardship route, and [[JapanFG/INDEX|JapanFG]] for entity-level pages. ## TL;DR

本页是日本上市金融集团的参考投资范围图谱，按子板块组织。本页**不是**买入/卖出建议，**不是**投资建议，**不是**推荐，**也不是**模型投资组合。所有 KPI 区间（市值类别、ROE 类别、股息率类别、外资持股类别、指数纳入、资本行动类别）均为基于公开信息的描述性指标，在任何读者将其用于实际决策之前，必须根据各发行方最新 IR 材料、JPX 及时披露及 EDINET 申报进行核实。数据核实责任由读者自行承担；任何过时或已变更的条目应与原发行方来源进行核对。

## Stewardship / Engagement Anchor Layer

上方许多实体是日本上市金融集团投资范围内的尽职管理参与目标。参与路径记录于 [[finance/japan-shareholder-proposal-and-agm-voting-route]] 及 [[finance/japan-activist-investor-playbook]]。本范围内的常见参与主题包括：

- 资本成本、PBR < 1.0 压力以及资本政策披露（东证资本成本要求）；
- 交叉持股（政策保有株）解除节奏与披露；
- 分红比率、回购政策和股东总回报政策；
- 董事会独立性、提名、薪酬及技能矩阵；
- 集团折价与分部披露（针对 SBI HD、ORIX、KDDI、软银、Mercari）；
- 治理事件应对（瑞穗 FG 系统事件、损保控股代理渠道治理事件、地方 FG 特定事件）。

保险公司和主要银行的交叉持股解除在大宗移动时出现在 [[finance/japan-large-shareholding-disclosure]] 层面；激进投资者或战略仓位在地方 FG 及消费金融发行方中也会出现于同一层面。

## Reader Verification Checklist

在将本页任何内容用于任何实际决策之前，读者应：

1. 通过 [JPX TDnet](https://www.jpx.co.jp/equities/listing/disclosure/tdnet/index.html) 查阅发行方的 TDnet 近期披露。
2. 从 [EDINET](https://disclosure.edinet-fsa.go.jp/) 获取最新证券报告和临时报告。
3. 根据 [JPX TOPIX 方法论页面](https://www.jpx.co.jp/markets/indices/topix/) 及相关指数提供商，核实指数纳入情况（日经 225  / TOPIX / TOPIX Core 30  / 富时罗素）。
4. 在发行方 IR 页面上核实最新决算短信和 IR 材料。
5. 对于尽职管理 / 参与阅读，与 [[finance/japan-shareholder-proposal-and-agm-voting-route]] 和 [[finance/japan-activist-investor-playbook]] 进行交叉参考。
6. 对于大额持股，按照 [[finance/japan-large-shareholding-disclosure]] 查阅 EDINET 的大量保有报告书 / 变更报告书档案。
7. 在读者自己的笔记中记录核实步骤的日期；本页矩阵描述符仅为公开信息层面，会随时间过时。
