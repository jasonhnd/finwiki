# 日本上市金融集团 investable universe

## Wiki 路径

本页位于 [[finance/INDEX|finance index]] 之下。它是一份关于日本上市金融集团的 public-source reference universe，请将其与作为顾问侧同业映射的 [[finance/japan-ib-league-table]]、用于参与 / 控制解读的 [[finance/japan-activist-investor-playbook]]、持股披露机制的 [[finance/japan-large-shareholding-disclosure]]、AGM / 受托管理路径的 [[finance/japan-shareholder-proposal-and-agm-voting-route]]、以及实体层面页面的 [[finance/INDEX|finance INDEX]] 对照阅读。系统侧语境位于 [[securities/tokyo-stock-exchange]] 与 [[securities/japan-market-infrastructure-map]]。

## TL;DR

这是一份按子板块组织的日本上市金融集团参考宇宙图。它不是买入 / 卖出清单，不是投资建议，不是推荐，也不是模型组合。所有 KPI 区间 (市值类别、ROE 类别、收益率类别、外资持股类别、指数纳入、资本行动类别) 均为 public-surface descriptors，读者在用于任何现实世界决策之前，必须对照各发行体最新的 IR 资料、JPX 及时披露、EDINET 提交文件重新确认。数据核验是读者的责任; 任何陈旧或轮换的条目都应与原始发行体来源核对调和。

## Stewardship / Engagement Anchor Layer

上面的许多实体是日本上市金融集团宇宙中的受托管理参与对象。参与路径记录于 [[finance/japan-shareholder-proposal-and-agm-voting-route]] 与 [[finance/japan-activist-investor-playbook]]。横贯本宇宙的反复出现的参与主题包括:

- 资本成本、PBR < 1.0 的压力，以及资本政策披露 (TSE 资本成本要求);
- 交叉持股 (政策保有株) 的解除节奏与披露;
- 派息率、回购政策与总股东回报政策;
- 董事会独立性、提名、薪酬与技能矩阵;
- 综合企业折价与分部披露 (针对 SBI HD、ORIX、KDDI、SoftBank Corp、Mercari);
- 治理事件应对 (瑞穗 FG 系统事件、Sompo HD 代理渠道治理事件、地区 FG 特有事件)。

保险公司与巨型银行的交叉持股解除会在区块移动时出现在 [[finance/japan-large-shareholding-disclosure]] 面上; 地区 FG 与消费者金融发行体中的激进或战略性持仓亦会在那里浮现。

## 读者核验清单

在将本页任何一行用于任何现实世界决策之前，读者应:

1. 经由 [JPX TDnet](https://www.jpx.co.jp/equities/listing/disclosure/tdnet/index.html) 打开发行体的 TDnet 最新披露源。
2. 从 [EDINET](https://disclosure.edinet-fsa.go.jp/) 调取最新的有价证券报告书与临时报告书。
3. 将指数纳入 (Nikkei 225 / TOPIX / TOPIX Core 30 / FTSE Russell) 对照 [JPX TOPIX 方法论页](https://www.jpx.co.jp/markets/indices/topix/) 与相关指数提供商交叉核对。
4. 在发行体 IR 页面重新确认最新的决算短信与 IR 演示资料。
5. 对于受托管理 / 参与解读，交叉参考 [[finance/japan-shareholder-proposal-and-agm-voting-route]] 与 [[finance/japan-activist-investor-playbook]]。
6. 对于大量持股，按照 [[finance/japan-large-shareholding-disclosure]] 调取 EDINET 大量保有報告書 / 変更報告書 栈。
7. 在读者自己的笔记中为核验步骤标注日期; 本页矩阵描述子仅为 public-surface，且会过时。

## 注意事项

- 这是一个 public-surface reference universe 页面，不是模型组合，不是推荐，不是投资建议，不是税务建议，不是法律建议。
- Ticker、子板块分类与主要股东形态会因合并、联盟、TOB、重组与公司行动而变化。
- ROE、股息率、外资持股与资本政策描述子仅为区间 / 类别标签; 它们不是 point estimates，且会很快过时。
- 指数纳入与 FTSE Russell 资格会定期轮换; 指数方法论页优先于此处所述的任何内容。
- 一些 named entities 位于上市母公司之内而非作为独立上市 (例如 SoftBank Corp 之内的 PayPay Bank; 历史上位于 SBI HD 之内的 SBI 住信网络银行); parent-of 映射请参见 [[finance/INDEX|finance INDEX]]。
- 私有金融集团 (例如 JA 三井租赁、相互制寿险公司) 因未上市，被明确排除在「investable universe」页面范围之外。
- 数据核验是读者的责任。FinWiki 的 public-surface 约定是 route-and-link，而非信号生成。
