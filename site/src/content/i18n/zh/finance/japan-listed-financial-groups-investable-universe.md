# 日本上市金融集团可投资范围

## Wiki route

本页位于 [[finance/INDEX|finance index]]。这是一个基于公开来源整理的日本上市金融集团参考范围，建议结合 [[finance/japan-ib-league-table]] 理解顾问侧同业映射，结合 [[finance/japan-activist-investor-playbook]] 理解参与 / 控制逻辑，结合 [[finance/japan-large-shareholding-disclosure]] 理解持股披露机制，结合 [[finance/japan-shareholder-proposal-and-agm-voting-route]] 理解股东大会 / stewardship 路线，并与 [[JapanFG/INDEX|JapanFG]] 的实体页面配套阅读。系统侧背景见 [[securities/tokyo-stock-exchange]] 与 [[securities/japan-market-infrastructure-map]]。

## TL;DR

这是一个按子行业组织的日本上市金融集团参考范围图。它**不是**买卖清单，**不是**投资建议，**不是**推荐名单，也**不是**模型投资组合。文中所有 KPI 区间标签（市值等级、ROE 等级、股息率等级、外资持股等级、指数纳入情况、资本动作等级）都只是公开信息层面的描述；在任何读者将其用于现实世界决策之前，都必须对照各发行人**最新** IR 材料、JPX 适时披露和 EDINET 文件重新确认。数据核验责任在读者本人；任何过时或已轮换的条目，都应以发行人原始来源为准。

## Stewardship / Engagement Anchor Layer

日本上市金融集团范围中的许多实体，都是 stewardship engagement 的目标。对应路径见 [[finance/japan-shareholder-proposal-and-agm-voting-route]] 与 [[finance/japan-activist-investor-playbook]]。这一范围里反复出现的 engagement 主题包括：

- 资本成本、PBR < 1.0 压力，以及资本政策披露（东证资本成本要求）；
- 交叉持股压缩节奏与披露；
- 分红率、回购政策、总股东回报政策；
- 董事会独立性、提名、薪酬与技能矩阵；
- 综合体折价与分部披露（如 SBI HD、ORIX、KDDI、SoftBank Corp、Mercari）；
- 治理事件应对（Mizuho FG 系统事件、Sompo HD 代理渠道治理事件、区域 FG 特有事件）。

保险公司与巨型银行的交叉持股压缩在大宗持股移动时会出现在 [[finance/japan-large-shareholding-disclosure]]；区域 FG 与消费金融发行人中的激进 / 战略持仓也会在该披露面浮现。

## Capital-Action Layer

本范围中近期资本动作（仅作存在性描述，使用前需核验）：

| 动作类别 | 在本范围中的表现 |
|---|---|
| 回购计划 | 巨型银行 FG、Tokio Marine、MS&AD、ORIX、Nomura、SBI HD 等历史上都曾披露 |
| 交叉持股压缩 | 巨型银行 FG 与三大非寿险集团都曾将其作为计划性承诺披露 |
| 资本与业务联盟 | SBI HD - Shinsei Bank 案、Aeon 集团消费金融重组、区域 FG 资本联盟案例 |
| TOB 目标 | 一些区域 FG 以及 SBI HD 对 Shinsei Bank 的收购中都出现过；应通过 TDnet / EDINET 核对具体案例 |
| 激进股东或大股东目标 | 若干区域 FG 与消费金融发行人在大股东披露中曾出现；应对照 EDINET 记录核对申报人及持股目的 |
| 可转债 / 股权挂钩发行 | 少数消费金融、租赁和金融科技发行人会使用；不能假定存在，必须查看最新 TDnet |

对任何具体资本动作，第一手记录都应是发行人的 TDnet 披露、EDINET 证券报告或临时报告，以及发行人 IR 页面。以上内容都不是投资建议。

## Reader Verification Checklist

在将本页任何一行用于现实世界决策之前，读者应当：

1. 通过 [JPX TDnet](https://www.jpx.co.jp/equities/listing/disclosure/tdnet/index.html) 打开发行人的近期披露流。
2. 从 [EDINET](https://disclosure.edinet-fsa.go.jp/) 调取最新证券报告和临时报告。
3. 结合 [JPX TOPIX methodology page](https://www.jpx.co.jp/markets/indices/topix/) 及相关指数提供商，交叉核对指数纳入情况（Nikkei 225 / TOPIX / TOPIX Core 30 / FTSE Russell）。
4. 在发行人 IR 页面重新确认最新决算短信与 IR deck。
5. 对 stewardship / engagement 阅读，交叉参考 [[finance/japan-shareholder-proposal-and-agm-voting-route]] 与 [[finance/japan-activist-investor-playbook]]。
6. 对大股东持股，按照 [[finance/japan-large-shareholding-disclosure]] 的路线调取 EDINET 的大股东报告 / 变更报告。
7. 在自己的笔记中为核验步骤打日期戳；本页矩阵描述仅是公开信息层内容，失效很快。

## Caveats

- 本页是公开信息层参考范围页，不是模型投资组合，不是推荐，不是投资建议，不是税务建议，也不是法律建议。
- Ticker、子行业分类和主要股东形态会因并购、联盟、TOB、重组及公司行动而变化。
- ROE、股息率、外资持股和资本政策描述都只是区间 / 类别标签，不是点估值，而且过时很快。
- 指数纳入与 FTSE Russell 资格会定期轮换；若与本页表述冲突，应以指数方法论文档为准。
- 某些点名实体是置于上市母公司内部，而非独立上市（例如 PayPay Bank 在 SoftBank Corp 内，SBI Sumishin Net Bank 历史上在 SBI HD 内）；母子映射见 [[JapanFG/INDEX|JapanFG]]。
- 私有金融集团（例如 JA Mitsui Leasing、相互制寿险公司）明确不属于“可投资范围”页面的范围，因为它们未上市。
- 数据核验责任在读者本人。FinWiki 的公开信息约定是 route-and-link，而不是 signal generation。
