# 日本公平披露与内幕交易管控

## 概述

日本公平披露与内幕交易记录涵盖选择性披露、及时披露、未公开重要事实、发行人信息管理、内幕交易限制及交易所披露系统等内容。公开途径包括 FSA 公平披露资料、《金融商品交易法》条文、JPX / TSE 及时披露页面，以及 TDnet。

本页归属于 [[finance/INDEX|finance domain]]，并链接至 [[securities/INDEX|securities domain]]、[[securities/tokyo-stock-exchange|Tokyo Stock Exchange]]、[[financial-regulators/japan-exchange-group|Japan Exchange Group]]、[[finance/japan-large-shareholding-disclosure|large-shareholding disclosure]]、[[finance/japan-tender-offer-process|Japan tender offer process]]、[[securities/japan-ipo-listing-disclosure-route|IPO listing disclosure route]]、[[securities/japan-market-infrastructure-map|Japan market infrastructure map]] 及 [[securities/japan-underwriting-market-structure|Japan underwriting market structure]]。

## 公平披露规则字段

| 字段 | 公开事实途径 |
|---|---|
| 法律依据 | 金商法第 27-36 条。 |
| FSA 说明 | 该规则旨在确保向投资者公平披露信息。 |
| 触发途径 | 上市公司向特定业务相关方提供未公开重要信息。 |
| 同步披露途径 | 在信息传达的同时披露，但适用例外条件的情况除外。 |
| 保密/不交易例外 | 接收者依法律、法规或合同承担保密义务和不交易义务。 |
| 及时披露途径 | 在获悉特定非故意或非同步传递后，进行及时披露。 |
| 泄露/交易途径 | 如保密/不交易义务接收者在披露前泄露或交易，须及时披露。 |
| 重要信息字段 | 未公开的、可能对证券价值产生重大影响的精确信息。 |
| 拼凑信息字段 | FSA 指引指出，拼凑信息本身通常不属于应披露信息。 |
| 内控字段 | FSA 指引将内部规则和员工培训列为切实可行的管控措施。 |

## 内幕交易公开字段

金商法第 166  条是公司内部人员内幕交易限制的公开法定途径。日本法律翻译资料通过内阁令规定的公开措施，或在事实记载于第 25  条规定披露文件中并可公开查阅时，对"公开化"作出定义。

JPX 的 TDnet 页面载明，通过"公司公告披露服务"（适时开示情报阅览服务）进行披露，可完成重要事实及特定收购要约事实的法定公开措施。同一页面还载明，TDnet 公开发布的内容在包含披露日期的 31  天内可在公共披露服务上查阅，而上市公司查询则覆盖过去 10  年的历史披露资料。

## TDnet 公开界面

| 界面 | 公开字段 |
|---|---|
| TDnet | 向 TSE 提交披露文件、向 TSE 事前说明、公开发布、媒体分发及数据库归档的电子工作流程。 |
| 公司公告披露服务 | TDnet 发布的及时披露公开查阅网站；31 天公开展示字段。 |
| 上市公司查询 | 历史及时披露资料；JPX TDnet 页面记录的 10 年字段。 |
| TDnet 数据库服务 | 历史披露数据的付费数据库途径。 |

## 法定公开与实际信息落差

法定公开途径与金商法/内阁令公开措施框架及 TDnet 公开措施字段挂钩。JPX 内幕交易公开资料还就正式公开公告后的即时时段记录了实际信息落差说明。该字段作为行为/公平说明记录，与法定公开措施字段分开。

## 与日本金融集团的关联性

上市金融集团及上市金融科技/证券/保险公司使用 TDnet 和 EDINET 披露途径。典型案例包括 [[megabanks/mufg]]、[[megabanks/smfg]]、[[megabanks/mizuho-fg]]、[[securities-firms/nomura-hd]]、[[securities-firms/daiwa-sg]]、[[megabanks/sbi-hd]]、[[megabanks/sony-fg]] 及 [[payment-firms/wealthnavi]]。
