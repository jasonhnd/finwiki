# 日本并购交易流程比较矩阵

## TL;DR

日本并购并非单一流程，而是至少七种不同的交易类型流程，这些流程共享法定基础架构，但在买方类型、控制机制、监管审批层次、融资形式和时间轴方面存在分歧。本矩阵将**境外收购**（外国买方收购日本目标）、**境外扩张**（日本买方收购外国目标）、**国内战略收购**、**国内 PE 杠杆收购**、**MBO/强制挤出**、**合资企业成立**及**要约收购强制交易**并排列示，涵盖法定控制机制（合并/股份交换/要约收购/资产收购/三角合并）、监管审批（[[finance/japan-fair-disclosure-and-insider-trading-controls|FSA disclosure]]/[[finance/japan-tender-offer-process|MOF FEFTA]]/[[finance/cross-border-m-a-japan|JFTC antitrust]]）、融资结构、公平意见实践、少数股东评估权及法定持股门槛（5%/30%/50.1%/66.67%/90%）。本页为路线图，**不构成**法律、税务或投资建议。

## 为何需要本矩阵

大多数外部评论将交易类型、法定机制和监管审批混为一谈。实际上，外国战略买方收购东京证券交易所上市目标时，面临的审批门槛与日本保荐方对同一目标进行私有化 MBO 完全不同，尽管两者最终都收购了同样的股份。前者须经历强制要约收购+外汇及对外贸易法（FEFTA）事前申报+公正交易委员会（JFTC）合并前申报+上市目标公平意见+评估权等完整门槛；而实操者需要一份对比视图，具体包括：

- 哪些**买方类型**触发哪些**监管审批**；
- 法律允许哪些**控制机制**可用，哪些在 [[corporate-strategy/japan-kaisha-bunkatsu-tax-regime|kaisha bunkatsu tax]] 下具有税务效率；
- **要约收购何时为强制性**，何时为可选；
- **公平意见何时为实质性要求**，何时为市场惯例；
- **评估权何时为少数股东提供真实的现金退出手段**；
- 哪些**法定持股门槛**可解锁哪些机制。

本矩阵是上述问题的比较对照平台。详细机制请参见相关各流程页面。

## 交易类型分类

| 交易类型 | 简要定义 |
|---|---|
| **境外收购** | 外国（非日本）买方收购日本目标。 |
| **境外扩张** | 日本买方收购非日本目标（仅限日本披露/税务范围）。 |
| **国内战略收购** | 日本买方出于产业/战略目的收购日本目标。 |
| **国内 PE 杠杆收购** | 国内 PE 保荐方以杠杆方式收购日本目标。 |
| **MBO/强制挤出** | 现任管理层+保荐方将日本上市目标私有化。 |
| **合资企业成立** | 两方或多方向新合资主体注入资产/现金。 |
| **要约收购强制交易** | 金融商品交易法（FIEA）强制要求进行公开要约收购的任何交易（如：场外收购超过1/3 ）。 |

本分类请与 [[finance/cross-border-m-a-japan|cross-border M&A Japan]] 配合使用，参阅境外收购/境外扩张对应图。

## 法定控制机制（"如何"）

| 机制 | 法规依据 | 典型用途 |
|---|---|---|
| 法定合并（合併） | 公司法 §§748-756 | 两实体合并；存续/新设实体发行股份；须经股东特别决议。 |
| 股份交换（株式交換） | 公司法 §§767-771 | 收购方成为100%母公司；目标公司作为全资子公司存续；以股换股。 |
| 股份转让（株式移転） | 公司法 §§772-774 | 两家现有公司在其上设立新控股公司。 |
| 要约收购（TOB/公開買付け） | 金融商品交易法 §§27-2 等 | 对上市目标股份的现金要约；在特定门槛以上为强制性。 |
| 资产收购（事業譲渡） | 公司法 §§467-470 | 买方取得特定业务单元；不涉及股份。 |
| 三角合并（三角合併） | 公司法 §§749 +768 | 存续实体发行母公司（通常为外国公司）股份作为对价。 |
| 现金强制挤出（キャッシュ・アウト） | 公司法 §§179 /179-2 （特别支配股东请求） | 持股≥90%者可在无需进一步表决的情况下挤出剩余少数股东。 |
| 股份出售请求（株式等売渡請求） | 公司法 §179 （2014 改革后） | 特别支配股东在持股≥90%时进行现金挤出。 |

保荐方主导机制请参见运营页面 [[finance/japan-mbo-and-squeeze-out-process]]；公开要约机制请参见运营页面 [[finance/japan-tender-offer-process]]。

## 监管审批层次

| 审批 | 主管机构 | 触发条件 |
|---|---|---|
| **要约收购监管** | [[finance/japan-tender-offer-process|FSA / Local Finance Bureau]] | 场外收购跨越5%/1/3 /50%门槛，或其他金融商品交易法触发条件。 |
| **大量持股报告** | [[finance/japan-large-shareholding-disclosure|EDINET]] | 对上市发行人持股≥5%；持股变动≥1%时提交变动报告。 |
| **内幕交易管控** | [[finance/japan-fair-disclosure-and-insider-trading-controls|FSA / Stock Exchange Surveillance]] | 公告前窗口期内持有重大非公开信息。 |
| **FEFTA 事前申报** | 财务省/各行业监管机构 | 外国投资者收购指定行业上市日本发行人≥1%，或达到一定门槛的非上市发行人。 |
| **反垄断合并前申报** | 公正交易委员会 | 合并后国内销售额超过门槛（收购方集团¥20亿且目标方¥5亿，适用于股份收购；不同机制门槛各异）。 |
| **行业监管机构** | 日本央行、FSA、经产省、总务省、厚生劳动省、国土交通省等 | 银行、保险、广播、电信、医疗等领域需获得行业审批。 |
| **外国反垄断** | 欧盟委员会、美国司法部/联邦贸易委员会、中国市场监管总局、韩国公正交易委员会、竞争与市场管理局等 | 境外扩张或跨境交易跨越外国门槛时适用。 |
| **CFIUS/国家安全同等机制** | 交易对手方司法管辖区 | 敏感行业外商投资（主要为日本境外投资赴美等情形）。 |

审批层次对顺序敏感。外汇及对外贸易法（FEFTA）事前申报须在外国投资者完成交割**前**取得；反垄断申报可并行进行；要约收购期间顺序由金融商品交易法规定。在依赖流程时间表前请先打开 [[finance/japan-tender-offer-process]]。

## 各交易类型流程

### 境外收购（外国战略方/保荐方收购日本目标）

| 维度 | 典型情形 |
|---|---|
| 买方类型 | 外国企业战略方、外国 PE 保荐方（如贝恩、KKR、CVC、凯雷），或外国主权财富基金/养老金。 |
| 控制机制 | 上市目标采用现金要约收购；非上市目标采用股份/资产收购；因偏好现金，三角合并少见。 |
| 监管审批 | 要约收购监管+FEFTA 事前申报（通常为关键门槛）+JFTC+行业审批+外国方审批。 |
| 融资结构 | 通常为现金要约；SPC/新设公司层面借债；杠杆通常联合[[megabanks/mufg-bank]]、[[megabanks/mizuho-bank]]、[[megabanks/sumitomo-mitsui-banking-corp|SMBC]]及外资银行。 |
| 典型时间轴 | FEFTA/反垄断审批顺利时，从公告到结算4-9 个月；有行业审批时更长。 |
| 公平意见 | 上市目标董事会通常为市场惯例；存在利益冲突委员会时通常需双份公平意见。 |
| 少数股东评估权 | 如结合后续股份整合/现金强制挤出，反对股东可根据公司法申请。 |
| 法定门槛 | 5% 大量持股报告通知；1/3 强制要约收购；2/3 章程修订/强制挤出特别决议；90%特别支配股东现金挤出。 |
| 顾问图谱 | 外资投行通常并行担任卖方/买方顾问，同时配合日本本土机构（[[securities-firms/nomura-hd|Nomura]]、[[securities-firms/daiwa-sg|Daiwa]]、[[securities-firms/smbc-nikko|SMBC Nikko]]、[[securities-firms/mizuho-securities|Mizuho Securities]]）；详见 [[finance/japan-ib-league-table]]。 |

本节请结合 [[finance/cross-border-m-a-japan]]（跨境法律层次）和 [[finance/japan-activist-investor-playbook]]（境外收购为敌意或未经请求的保荐方时的接触角度）阅读。

### 境外扩张（日本买方收购外国目标）

| 维度 | 典型情形 |
|---|---|
| 买方类型 | 日本战略方（如综合商社、大型银行金融集团子公司、大型工业企业）、日本保荐方（如 JIC、J-STAR、Advantage），或日本上市金融集团。 |
| 控制机制 | 受外国法律规范的股份收购（英国股份购买协议、特拉华州股份购买协议等）；视目标方司法管辖区，可能采用法定合并/方案合并。 |
| 监管审批 | 外国方主导：外国反垄断、外国外商直接投资（CFIUS、英国国家安全与投资法、欧盟外国补贴条例等）；如属重大交易，日本方通常仅需完成交割后 EDINET/治理披露。 |
| 融资结构 | 跨货币：日元借款+跨货币互换，或直接以美元/欧元借款；以日元债券或美元债券置换桥接融资；武士债与离岸债券的选择。 |
| 典型时间轴 | 签约至交割6-12 个月；外国反垄断和外商直接投资审查主导节奏。 |
| 公平意见 | 日本收购方通常进行买方公平意见，用于董事会/治理辩护。 |
| 少数股东评估权 | 外国法律概念（如特拉华州评估、英国方案异议）替代公司法机制。 |
| 法定门槛 | 目标方适用外国法律门槛；仅当收购方发行新股融资时，日本披露范围才在 [[finance/japan-large-shareholding-disclosure]] 下触发。 |
| 顾问图谱 | 外资投行通常主导目标方流程；日本本土机构作为买方顾问配对；详见 [[finance/japan-ib-league-table]]、[[securities-firms/goldman-sachs-japan]]、[[securities-firms/morgan-stanley-japan]]。 |

境外扩张交易受**目标方司法管辖区**规范，不适用日本法定并购；下方矩阵仅展示日本范围内的接触点。

### 国内战略收购（日本买方+日本目标）

| 维度 | 典型情形 |
|---|---|
| 买方类型 | 日本上市企业、工业集团、大型银行金融集团战略收购方。 |
| 控制机制 | 以股换股常用股份交换（株式交換）；全面整合常用法定合并（合併）；上市目标则进行要约收购；业务剥离则采用资产收购。 |
| 监管审批 | 上市目标则适用要约收购监管+JFTC+行业审批。不适用 FEFTA（无外国投资者）。 |
| 融资结构 | 以股换股常见；现金通过资产负债表、银团贷款或债券发行；高杠杆少见。 |
| 典型时间轴 | 4-7 个月；纯粹股份交换且无外国审批时可缩短。 |
| 公平意见 | 上市目标董事会的市场惯例标准，尤其是设立特别委员会时。 |
| 少数股东评估权 | 公司法 §§785, 797, 806 赋予异议股东要求公平价格购买的权利。 |
| 法定门槛 | 2/3 股份交换/合并特别决议；1/3 强制要约收购；90%强制挤出。 |
| 顾问图谱 | 大型银行关联证券公司主导（[[securities-firms/smbc-nikko]]、[[securities-firms/mizuho-securities]]、[[securities-firms/mufg-mums|MUFG / MUMS]]）；详见 [[finance/japan-ib-league-table]]。 |

### 国内 PE 杠杆收购

| 维度 | 典型情形 |
|---|---|
| 买方类型 | 日本注册 PE 保荐方（如 Advantage、J-STAR、JIC Capital、Polaris）或全球保荐方日本团队（KKR Japan、Bain Japan、Carlyle Japan）。 |
| 控制机制 | 上市目标采用 SPC/新设公司要约收购；非上市目标采用私下股份购买协议；有时在保荐方 SPC 与目标控股公司之间进行股份交换。 |
| 监管审批 | 上市目标则适用要约收购监管+JFTC+行业审批；若保荐方为管理境外有限合伙基金的日本注册普通合伙人，FEFTA 通常不触发（视解释而定）。 |
| 融资结构 | 优先杠杆收购贷款（通常联合[[megabanks/mufg-bank]]、[[megabanks/mizuho-bank]]、[[megabanks/sumitomo-mitsui-banking-corp|SMBC]]、[[trust-banks/smtb|SMTB]]）；夹层/次级债偶有出现；日本债券置换桥接融资不如美国常见。 |
| 典型时间轴 | 保荐方承诺至交割4-8 个月；竞价中常附带融资安排。 |
| 公平意见 | 上市目标董事会的实践惯例要求；来自独立投行的公平意见通常双轨并行。 |
| 少数股东评估权 | 强制挤出后可依公司法 §172 申请公平价格确定。 |
| 法定门槛 | 1/3 要约收购触发；2/3 后端挤出表决；90%现金挤出请求。 |
| 运营模式 | 参见 [[finance/japan-private-equity-operating-model]] 和 [[finance/japan-leveraged-buyout-economics]] 了解内部收益率数学。 |

### MBO/强制挤出

| 维度 | 典型情形 |
|---|---|
| 买方类型 | 现任管理层+保荐方（纯管理层收购罕见）。 |
| 控制机制 | 两步走：（1）现金要约收购以最大化应约；（2）通过 §179 出售请求（≥90%）或 §180 股份整合（≥2/3 表决）完成后端强制挤出。 |
| 监管审批 | 要约收购监管+[[finance/japan-mbo-and-squeeze-out-process|METI Fair M&A Guidelines]] 下的加强公平性审查；若达到合并营业额门槛则适用 JFTC。 |
| 融资结构 | 保荐方权益+向新设公司提供的杠杆收购贷款；有时向管理层提供优先股/可转换工具。 |
| 典型时间轴 | 6-10 个月；如有独立委员会或股东诉讼则更长。 |
| 公平意见 | 依据经产省《公正 M&A 指针》实为强制要求；通常为双份公平意见（一份给特别委员会，一份给要约方）。 |
| 少数股东评估权 | 在 MBO 场景最为活跃；法院多次重新校准公平价格（如 Tecmo/Rex/Cybird/JCOM 系列经典判例）。 |
| 法定门槛 | 2/3 强制挤出/整合表决；90%特别支配股东现金挤出。 |
| 关键保护 | 独立特别委员会、市场检验、多数少数股东条件日益成为市场惯例。 |

请结合 [[finance/japan-mbo-and-squeeze-out-process]]（程序排序）和 [[finance/japan-activist-investor-playbook]]（MBO 场景下的积极投资者回应）阅读。

### 合资企业成立

| 维度 | 典型情形 |
|---|---|
| 买方类型 | 两方或多方战略方注入资产、现金、知识产权或业务单元。 |
| 控制机制 | 成立新设公司+实物出资（現物出資）或资产转让（事業譲渡）；通常通过 [[corporate-strategy/japan-kaisha-bunkatsu-tax-regime|kaisha bunkatsu]] 实现税务递延。 |
| 监管审批 | 若达到合并销售额门槛则适用 JFTC；行业审批；若有外国合资方则适用 FEFTA。 |
| 融资结构 | 各母公司资本出资+合资企业层面融资；有时由母公司担保。 |
| 典型时间轴 | 6-12 个月，包含治理/股东协议谈判。 |
| 公平意见 | 罕见，除非一方为上市公司且出资金额重大。 |
| 少数股东评估权 | 若合资企业成立涉及上市目标剥离，异议股东可依公司法 §785 /§806行使权利。 |
| 法定门槛 | 重大资产转让时需2/3 表决；普通合资审批需1/2 +1 。 |
| 税务结构 | 通常构建为税务合格的[[corporate-strategy/partial-spinoff-tax-deferral|partial spinoff]]/以股换股，以递延日本资本利得税。 |

### 要约收购强制交易

| 维度 | 典型情形 |
|---|---|
| 触发条件 | 场外收购跨越5%（快速收购）或1/3 （强制）门槛，或其他金融商品交易法规定的触发条件（如大量小额收购）。 |
| 买方类型 | 任何方：战略方、保荐方、个人、母公司收购子公司、积极投资者。 |
| 控制机制 | 依金融商品交易法 §§27-2 等进行现金要约收购；证券对价要约收购罕见。 |
| 监管审批 | 金融商品交易法要约收购章节具有约束力；如适用则叠加 FEFTA+JFTC+行业监管。 |
| 融资结构 | 金融商品交易法要求"资金确定性"；权益、债务或两者均须提供证明。 |
| 典型时间轴 | 要约期间30-60 个交易日；如出现竞争要约或重大变更可延期。 |
| 公平意见 | 目标董事会须在10 个交易日内出具意见（赞同/反对/中立）；公平意见为市场惯例。 |

## 交叉持股接口

涉及目标方持有活跃交叉持股（如大型银行金融集团、保险公司、大型工业企业）超过5%的交易流程与 [[finance/japan-cross-shareholding-unwinding-economics]] 相互关联，原因在于现有交叉持股方通常会：（a）参与要约收购提供关键股份；（b）拒绝参与要约收购迫使后端强制挤出；或（c）成为对抗敌意境外收购的防御性障碍。交叉持股行为决定时间轴和价格。

## 积极投资者/接触接口

对于上市目标，积极投资者仓位可能阻碍或加速交易流程。积极投资者路径参见 [[finance/japan-activist-investor-playbook]]，股东大会表决接口参见 [[finance/japan-shareholder-proposal-and-agm-voting-route]]。"价格推高套利"模式（积极投资者在要约价格以上建仓以迫使提价）在日本上市目标 MBO 和境外收购案件中日益常见。

## 多司法管辖区税务面

跨境收购的税务结构涉及多个司法管辖区。多司法管辖区税务杠杆框架参见 [[finance/multi-jurisdiction-identity-tax-leverage]]；合资/剥离案件中使用的会社分割（kaisha bunkatsu）税务递延机制参见 [[corporate-strategy/japan-kaisha-bunkatsu-tax-regime]]。以上均不构成税务建议；请核对法规文本及经产省/国税厅指导意见。

## 边界案例

上述七种交易类型分类无法对所有实际交易进行整洁分类。常见边界案例：

- **外国保荐方通过日本注册 SPC**：外国保荐方（如贝恩、KKR）通常通过由境外有限合伙基金出资的日本注册 SPC 进行收购。FEFTA 事前申报的解释取决于 SPC 是否被视为外国投资者；如境外基金最终由外国有限合伙人和外国普通合伙人控制，通常被如此视为。使用前请先查阅财务省/经产省 FEFTA 解释指导。

- **境外扩张交易中日本上市收购方增发新股**：以新发行日本上市权益融资的境外收购，将交易拉回日本披露范围（EDINET 证券登记说明书、如有必要则召开股东大会、权益配售触发[[finance/japan-large-shareholding-disclosure|large shareholding]]）。

- **反向合并/SPAC 式**：日本罕见；东京证券交易所目前不支持美式 SPAC。通过标准市场/成长市场挂牌的反向合并案件可能触发东证退市审查和空壳公司规则。

- **通过场外大宗交易出售交叉持股**：大型银行金融集团/保险公司通过场外大宗交易解除上市工业企业5%，若依金融商品交易法要约收购豁免规则合理构建，则不构成要约收购；参见 [[finance/japan-cross-shareholding-unwinding-economics]] 了解机制，参见 [[finance/japan-fair-disclosure-and-insider-trading-controls]] 了解内幕交易审查。

- **母公司两步要约收购**：母公司已持有>50%，对子公司进行"退市要约收购"以完成私有化。因存在固有利益冲突，依经产省《公正 M&A 指针》按 MBO 式处理；实践中独立委员会为强制要求。

- **敌意要约收购**：在日本罕见但日益增多。触发积极投资者/委托书争夺叠加效应；防御措施（毒丸计划、白衣骑士、员工持股计划忠诚度设计）须符合经产省《收购防御措施指针》。

- **以会社分割+股份出售构建剥离**：两步结构：（1）母公司通过[[corporate-strategy/japan-kaisha-bunkatsu-tax-regime|kaisha bunkatsu]]（税务合格）将目标业务分割入新设公司；（2）将新设公司股份出售给买方。税务效率高，但时间轴延长。

- **以外国母公司股票为对价的三角合并**：2007 公司法修订后允许以外国母公司股票作为合并对价；因税务合格性和股东批准复杂性，实际使用极少。

- **上市母公司间通过剥离成立合资企业**：如合资企业由上市母公司剥离业务单元而成，剥离方触发[[finance/japan-large-shareholding-disclosure]]、异议股东评估权，且如合资伙伴取得>1/3可能触发要约收购。

- **联合/俱乐部交易**：联合体中每位保荐方/战略方各自承担 FEFTA 申报义务（如适用），并以"共同持有人"方式承担各自的[[finance/japan-large-shareholding-disclosure|LSR]]义务。

- **交易前抢先建仓**：公告前在市场上收购至4.99%可避免大量持股报告；在5 个交易日内跨越5%将启动大量持股报告计时，且视速度可能推向要约收购监管。

3. 从 JFTC 英文页面获取最新 JFTC 申报门槛。
4. 从财务省外商直接投资页面获取财务省/经产省 FEFTA 指定行业名单及门槛规则。
5. 对照 [JPX TDnet](https://www.jpx.co.jp/equities/listing/disclosure/tdnet/index.html) 和 [EDINET](https://disclosure.edinet-fsa.go.jp/) 上的实时申报示例，核查披露路径。
6. 对照 [[finance/japan-ib-league-table]] 核查顾问体系解读。
7. 在核查步骤上注明日期；法定门槛和经产省指针版本定期更新。

## 注意事项

- 本页为路线图，**不构成**法律、税务或投资建议。
- 格级别描述仅为分类性描述；请核对法规、指针及最新机构通知。
- 共同持有人的处理、FEFTA 下的外国投资者身份认定及股份交换/会社分割的税务合格性因案而异；请咨询律师。
- 经产省《公正 M&A 指针》和《收购防御措施指针》已多次修订；请核对适用于特定交易的有效版本。
- 东证有关退市、控制权转移和强制挤出的上市规则可能变化；请查阅最新东证规则修订。
