# 日本大额持股披露

## 概述

日本大额持股披露制度是《金融商品交易法》中用于揭示大宗持股、控制影响及市场供求信息的"5% 规则"。持股比例一旦超过 5%，通常须在五个营业日内提交大额持股报告书。此后发生重大变化，包括持股比例变动 1% 以上的情况，通常须在五个营业日内提交变更报告书。申报通过 EDINET 进行。

本页面基于 1-2026  年 5 月修订后的制度。FSA 的申报页面于 2026-05-01 更新，并注明修订后的大额持股制度于当日生效，内容涉及持股比例计算方式、共同持股人范围及申报表格的变更。使用资料时请注意信息来源日期。

## 制度概览

| 主题 | 实操要点 |
|---|---|
| 触发条件 | 上市股票等持股比例超过 5%。 |
| 初始报告 | 大额持股报告书，通常在五个营业日内提交。 |
| 变更报告 | 通常在持股比例变动 1% 以上或发生其他重大变化后五个营业日内提交。 |
| 申报渠道 | 通过 EDINET 电子申报；纸质申报非常规方式。 |
| 目的 | 提高管理层影响力及市场供求相关信息的透明度。 |
| 2026  后注意事项 | 2026-05-01 生效的改革改变了计算方式、共同持股人范围及申报表格。 |

## 本制度的适用边界

大额持股披露并非公开收购文件。该制度披露持有人、持股比例、持股目的、共同持股人及相关大宗持股信息，但本身不披露收购价格、要约期间、收购条件、资金证明或目标公司董事会意见。

| 申报文件 | 公开记录内容 |
|---|---|
| 大额持股报告书 | 哪些人超过了 5%，以何种持股结构、以何种目的持股？ |
| 变更报告书 | 自最近一次报告书以来发生了哪些变化？ |
| 公开收购申报书 | 收购条款、资金来源、条件及收购后计划是什么？ |
| TDnet 披露 | 通过适时披露发布的上市公司事件信息。 |

[[finance/japan-tender-offer-process|Japan tender offer process]] 与本页面属于相互关联但各自独立的路径。

## 持股人分析

在 2026  修订后的制度下，公开申报文件包含以下持股人分析字段：

- 直接持股；
- 投资决策权；
- 投票权或指示权；
- 共同持股人；
- 经修订规则纳入的衍生品或股权联动敞口（如适用）；
- 持股目的；
- 重要提案行为；
- 已披露的质押/借贷/融资情况；
- 快速累积或出售的模式。

汇总字段可包括基金、顾问主体、关联方、衍生品交易对手方或共同行动人。申报人、共同持股人、目的及变更历史（如已披露）一并记录。

## 积极股东/控制层面解读

大额持股申报文件是 [[finance/japan-activist-investor-playbook|activist investor playbook]] 页面的组成部分。其可显示入场时机、持股增加情况、共同持股人变动、目的修订，以及申报人是否发出重要提案行为信号。相关公开记录还包括：

- 股东提案；
- 股东大会表决结果；
- 公开信/维权网站；
- TDnet 公司回应；
- 公开收购申报书（如有）；
- 发行人公司治理改革；
- 市场价格及成交量情况。

对于银行、券商、保险及金融科技机构，仅在公开信息来源支持主体层面相关性的情况下，将相关控制案例与 [[JapanFG/INDEX|JapanFG]] 页面关联。

## EDINET 操作流程

1. 在 EDINET 中检索发行人和申报人。
2. 调取大额持股报告书及最新变更报告书。
3. 确认报告类型：普通报告书、简式报告书、补充申报或更正申报。
4. 提取持股比例、共同持股人、持股目的、重要提案行为，以及已披露的质押/融资情况。
5. 构建附日期的持股时间线。
6. 交叉核对 TDnet/上市公司披露，确认公司回应或相关公司行为。
7. 如存在公开收购，单独阅读公开收购申报书。

## 阅读核查清单

| 字段 | 原因 |
|---|---|
| 申报人/持股人 | 识别法定申报主体，不一定是最终经济利益集团。 |
| 共同持股人 | 确定汇总范围及可能存在的协调影响力。 |
| 持股比例 | 核心阈值及变更报告依据。 |
| 持股目的 | 区分纯投资、政策性持股、重要提案、控制或其他目的。 |
| 取得资金/借贷情况 | 可揭示杠杆、融资及交易意图。 |
| 近期交易情况 | 反映持股累积/处置节奏。 |
| 衍生品/合同权利 | 改革后及隐性敞口分析的重要字段。 |
| 修订历史 | 更正早期申报内容，可能改变解读结论。 |
