## 概述

本页为日本股东维权行动及股东参与的公开信息路线图，并非选股页面。有效的控制链条为：东京证券交易所资本成本压力 → 公司治理守则 → 尽职管理守则 → EDINET 大额持股报告 → 股东提案权 → 股东大会表决 → TDnet 披露 → 要约收购规则。

请将本页与 [[finance/japan-tender-offer-process|Japan tender offer process]]、[[finance/cross-border-m-a-japan|cross-border M&A Japan]]、[[securities/tokyo-stock-exchange|Tokyo Stock Exchange]]、[[financial-regulators/japan-exchange-group|Japan Exchange Group]]、[[financial-licenses/securities-license-stack|securities license stack]] 配合使用。

## 公开行动各阶段

| 阶段 | 公开文件 | 核心问题 |
|---|---|---|
| 筛选 | 东证资本成本要求、估值、资产负债表、治理报告 | 是否存在积极股东可能介入的公开理由？ |
| 披露审查 | 公司治理报告、有价证券报告书、TDnet公告 | 公司已公开承诺了什么？ |
| 初步参与 | 信函、会议、投资者说明材料、尽职管理披露（如已公开） | 此次参与是普通尽职管理还是控制权争夺？ |
| 大额持股 | EDINET大额持股报告 / 变更报告 | 谁持有该股份，声明目的是什么？ |
| 股东提案 | 股东大会通知、提案内容、董事会意见 | 使用了何种法律权利及治理问题？ |
| 股东大会表决 | 表决结果及代理顾问评论（如已公开） | 该行动是否获得机构投资者支持？ |
| 控制权交易 | 要约收购申报、目标方意见、竞争要约、强制收购 | 参与行动是否已演变为收购 / 控制权争夺？ |
| 退出 / 解决 | 和解、回购、分红、董事会变更、出售或未变更 | 发生了哪些变化，哪些未变？ |

## 治理 / 披露节点

| 节点 | 使用方式 |
|---|---|
| 公司治理守则 | 董事会独立性、资本政策、披露及股东权利的基准 |
| 尽职管理守则 | 投资者参与及表决原则 |
| 东证资本成本 / 股价行动 | 对资本效率低于预期公司形成的公开压力面 |
| 金融商品交易法第5%条大额持股规则 | 识别主要持股人、共同持有人、持股目的及变动情况（以已申报为限） |
| 公司法 | 股东大会、提案及公司法路径 |
| 要约收购规则 | 当行动演变为公开收购 / 控制权进程时适用 |
| TDnet / EDINET | 及时披露及法定申报层面 |

## 参与类型

| 类型 | 参考资料 |
|---|---|
| 普通尽职管理参与 | 资产所有人 / 管理人就治理及资本政策开展的对话 |
| 协作参与 | 多名投资者可能就共同主题开展参与；需核查共同持有人风险 |
| 大额持股人行动 | EDINET 申报成为核心证据 |
| 股东提案 | 通过公司法 / 股东大会材料的正式提案路径 |
| 董事会争夺 | 董事候选名单 / 治理控制权问题；公开资料需严格区分来源 |
| 要约收购 / 控制权收购 | 转入 [[finance/japan-tender-offer-process|tender offer process]] 及收购指引路径 |

## 参与方地图

| 参与方 | 角色 |
|---|---|
| 积极投资基金 / 投资者 | 建立持仓、依规披露、参与、提案、发起行动或退出 |
| 机构投资者 | 行使表决权、参与、遵循尽职管理政策，可能影响行动结果 |
| 目标公司董事会 | 回应、设立委员会、更新资本政策、支持 / 反对提案或要约 |
| 独立董事 / 特别委员会 | 重大交易中的利益冲突控制及少数股东保护路径 |
| 代理顾问 | 提供表决建议（如已公开） |
| 东证 / JPX | 上市、治理、披露及市场结构压力面 |
| 金融厅 / EDINET | 披露及法定申报路径 |

## 风险提示

| 风险 | 公开相关性 |
|---|---|
| 共同持有人定性模糊 | 多名持有人协同行动可能改变披露义务及控制权分析 |
| "重要建议"措辞 | 大额持股报告中的目的描述需仔细解读 |
| 内幕信息 | 股东参与不得与合法持有 / 使用非公开信息相混淆 |
| 市场操纵 | 公开行动与交易活动需分开举证，在法律层面须谨慎处理 |
| 恶意收购过度认定 | 并非所有参与行动都是控制权交易 |
| 顾问利益冲突 | 投资银行、经纪商、贷款方及顾问可能在同一事项中处于多方立场 |

## 研究清单

1. 从公司公开文件入手：治理报告、有价证券报告书、TDnet、EDINET、股东大会通知及表决结果。
2. 在判断持股规模、目的或共同持有人状态之前，先调取大额持股报告及变更报告。
3. 将治理参与与公开要约收购 / 控制权交易区分开来。
4. 核查投资者的身份：资产管理人、受益所有人、顾问还是集团关联公司。
5. 使用精确日期，因为持仓、提案、目标方回应及要约条款均可能迅速变动。
