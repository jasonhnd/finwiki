# 日本私募股权基金结构矩阵

## 摘要

日本私募资本市场并非单一资产类别，而是至少六种截然不同的子策略（并购收购、成长、二级市场、接续基金、基础设施、私募信贷）以及三种截然不同的 GP 原型（全球大型 PE 日本团队、国内独立 GP、政策导向/国家关联 GP），各自采用不同的基金结构组合。本矩阵是对以下各维度进行并排比较的参考面：**基金类型、目标公司特征（规模/行业）、杠杆假设、持有期、附带权益和管理费经济结构（通常为 1/20, 1.5/20,  或 2/20）、LP 构成（主权、养老金、保险公司、家族办公室），以及工具司法管辖区（日本投资事业有限责任组合、开曼豁免 LP、特拉华州 LP）**。本文是路线图，不构成投资建议、不构成推荐、不构成募资邀约、不构成对任何 GP 的背书。

## 为何需要此矩阵

大多数外部评论将"日本 PE"一词混用于全球大型 PE 日本团队（KKR Japan、Bain Japan、Carlyle Japan、EQT Japan、CVC Japan）、国内独立机构（Advantage Partners、J-STAR、Integral、Polaris）和国家关联/主权邻近载体（JIC Capital、INCJ / JIC、DBJ / DBJC）。这些并非同一商业模式。

从业者需要并排可见性：

- **基金类型**（并购收购/成长/二级市场/接续/基础设施/私募信贷）；
- **目标特征**，按企业价值区间和行业；
- **杠杆假设**（低杠杆成长股权 vs 5-7x EBITDA 收购 vs 无杠杆基础设施）；
- **持有期**（3-5 年成长，4-7 年收购，7-10+年基础设施）；
- **费用/附带权益分成**（信贷 1/20 ，新兴管理人 1.5/20 ，收购标准 2/20 ）；
- **LP 构成**（主权/养老金/保险公司/家族办公室/母基金）；
- **工具司法管辖区**（日本投资事业有限责任组合/投資事業有限責任組合、开曼豁免 LP、特拉华州 LP，有时采用平行结构）。

本矩阵是上述问题的比较参考面。任何单一基金的机制详情见 GP 公开网站披露和 LP 年度报告。

## 基金类型分类

| 子策略 | 简要定义 |
|---|---|
| **并购收购** | 收购成熟企业的多数/100%控制权；使用杠杆。 |
| **成长股权** | 收购成长阶段企业的少数/共同控制权；杠杆有限。 |
| **二级市场** | 从前期 LP 处购入现有 LP 权益/投资组合。 |
| **接续基金** | GP 主导的二级交易；现有 GP 将一个或多个组合公司转入新载体。 |
| **基础设施** | 长久期、产生现金流的资产（公用事业、可再生能源、交通、数字基础设施）。 |
| **私募信贷** | 直接贷款、夹层融资、困境债、资产担保融资。 |

## 各基金类型结构

### 并购收购

| 维度 | 典型情况 |
|---|---|
| 目标公司特征 | 成熟企业，EBITDA 典型区间¥1-50亿；EV 典型区间¥10-500亿；行业无特定偏好，但医疗、工业、软件、消费集中。 |
| 行业 | 工业、医疗、软件/IT 服务、消费、金融服务。 |
| 杠杆假设 | 典型 5-7x EBITDA；稳定现金流业务有时更高；LBO 贷款在 [[JapanFG/mufg-bank]]、[[JapanFG/mizuho-bank]]、[[JapanFG/sumitomo-mitsui-banking-corp|SMBC]]、[[JapanFG/smtb|SMTB]] 之间联贷。 |
| 持有期 | 典型 4-7 年（全球大型 PE 日本团队部分为 3-5 年；国内独立机构部分为 5-8 年）。 |
| 费用/附带权益 | 2% / 20%（全球大型 PE 日本团队标准）；1.5% / 20%（部分新兴/国内）；优先于 8% 优先回报/首选回报；国内常见欧洲式分配瀑布，全球常见逐项分配。 |
| LP 构成 | 全球养老金（CalPERS、CPP、OTPP、GPIF（有限参与））、主权财富（ADIA、GIC、KIA）、日本大型银行自营、日本寿险公司、基金会、家族办公室、母基金。 |
| 工具司法管辖区 | 全球大型 PE 日本团队：开曼 ELP 主载体 + 日本平行载体（面向免税 LP）。国内独立：日本投资事业有限责任组合为主载体，有时辅以开曼平行载体。 |
| 退出路径 | 向战略买家出售、二级转让（赞助商对赞助商）、在东交所主板/成长板 IPO、股息重组。 |
| 交易来源 | 东交所私有化、上市母公司剥离、传承/家族企业、赞助商对赞助商二级转让。 |

### 成长股权

| 维度 | 典型情况 |
|---|---|
| 目标公司特征 | 高成长，通常未盈利或处于早期盈利，EV 典型区间¥5-50亿；科技、医疗、消费品牌。 |
| 行业 | 软件/SaaS、金融科技、医疗、D2C 消费品、气候/能源转型。 |
| 杠杆假设 | 低/零；成长股权以股权融资为主。 |
| 持有期 | 典型 3-5 年（真正的成长型投资有时更长）。 |
| 费用/附带权益 | 2% / 20% 标准；规模较小/较早期基金有时为 2.5% / 20%；欧洲式分配瀑布。 |
| LP 构成 | 主权财富、养老金、保险公司、母基金、家族办公室；因认购规模较小，大型养老金参与较少。 |
| 工具司法管辖区 | 国内主要为日本投资事业有限责任组合；跨境为开曼 ELP。 |
| 退出路径 | 在东交所成长板/主板 IPO、向战略买家出售、赞助商对赞助商转让。 |
| 交易来源 | 创始人/管理层关系、VC 毕业项目、早期 VC 二级转让、创始人主导剥离。 |

国内成长型 GP 包括 Polaris Capital Group（原 Polaris Holdings）、Integral Corporation、NSSK 及 JIC Capital（日本投资公司旗下）。各有公开网站披露。

### 二级市场

| 维度 | 典型情况 |
|---|---|
| 目标 | PE/VC 基金现有 LP 权益、二级仓位组合、GP 主导重组。 |
| 行业 | 行业无关；二级买方对底层投资组合构成进行尽调。 |
| 杠杆假设 | 基金层面信贷额度（基于 NAV 或认缴额）；组合公司杠杆继承底层基金。 |
| 持有期 | 典型 2-5 年（因剩余基金期限较短，短于初级）。 |
| 费用/附带权益 | 1% / 10%（部分），1.25% / 12.5%，或 1.5% / 15% — 通常低于收购，因为预期回报较低、期限较短。 |
| LP 构成 | 养老金、保险公司、主权、母基金 — 具有二级市场专业知识的投资者。 |
| 工具司法管辖区 | 开曼 ELP 标准；有时为卢森堡 SCSp 或特拉华州 LP（面向西方 LP 群体）。 |
| 路径 | LP 主导（LP 出售权益）、GP 主导（接续基金）、结构化/优先二级。 |

在日本活跃的全球二级 GP 包括 Coller Capital、Ardian、Strategic Partners（黑石）、Lexington Partners（富兰克林邓普顿）、HarbourVest、Pantheon 和高盛 Vintage。日本特定二级市场活动仍较美欧薄弱，但呈增长态势。

### 接续基金

| 维度 | 典型情况 |
|---|---|
| 目标 | 同一 GP 的一个或多个现有组合公司，从现有基金转移至新载体。 |
| 行业 | 继承底层组合公司行业。 |
| 杠杆假设 | 继承组合公司杠杆；转移时有时进行再融资。 |
| 持有期 | 转移后典型 3-5 年。 |
| 费用/附带权益 | 转移时重置；通常 1% / 10% 或 1.5% / 15%；单资产时有时按"交易层面"收费。 |
| LP 构成 | 现有 LP（可选择滚存）、新 LP（注入资本）；冲突和流程遵循 ILPA 接续基金指引。 |
| 工具司法管辖区 | 开曼 ELP 标准；面向免税 LP 群体存在日本注册变体。 |
| 利益冲突管理 | LPAC 同意、公平意见、GP 承诺、向现有 LP 提供维持现状选项。 |

接续基金在日本是较新的结构；利益冲突管理程序遵循 ILPA 指引，并越来越多地参考国内 LP 预期。

### 基础设施

| 维度 | 典型情况 |
|---|---|
| 目标 | 长久期产生现金流的资产：可再生能源（太阳能/风能/水电）、数字基础设施（数据中心、铁塔、光纤）、交通（收费公路、机场）、公用事业、受监管网络。 |
| 行业 | 能源转型、数字基础设施、交通、社会基础设施。 |
| 杠杆假设 | 资产层面融资（项目融资、基础设施贷款、基础设施债券）；基金层面适度；典型总杠杆 50-70% 资产价值。 |
| 持有期 | 典型 7-10+年（部分核心增值 7-10 年；核心 10-15+年；超核心 15-25 年）。 |
| 费用/附带权益 | 1% / 10%（核心），1.25% / 15%（核心增值），1.5% / 20%（价值增值）；优先回报 6-8%以上。 |
| LP 构成 | 养老金、保险公司、主权、基础设施专项母基金；长久期 LP 群体。 |
| 工具司法管辖区 | 全球为开曼 ELP；面向欧盟 LP 群体为卢森堡 SCSp；国内基础设施为日本特定 TK-GK 结构。 |
| 收益特征 | 以收益为导向：4-8% 现金收益率 + 资本增值。 |

在日本/亚太基础设施活跃的全球基础设施 GP 包括麦格理资产管理、Global Infrastructure Partners（贝莱德）、KKR Infrastructure、Brookfield Infrastructure、I Squared Capital、Stonepeak。日本国内基础设施包括 [[JapanFG/dbj|DBJ]] 关联的 DBJ Capital、JOIN（日本海外基础设施投资公司）及多只日本上市可再生能源/基础设施基金。

### 私募信贷

| 维度 | 典型情况 |
|---|---|
| 目标 | 向赞助商支持的企业进行直接贷款、夹层融资、困境/特殊情况、资产担保融资、房地产信贷。 |
| 行业 | 直接贷款行业无关；困境/资产担保专业化。 |
| 杠杆假设 | 基金层面信贷额度（基于 NAV）典型 0.5-1.5x 杠杆；底层贷款本身非"加杠杆"，但借款人杠杆 4-6x。 |
| 持有期 | 典型每笔贷款 3-5 年；基金存续期 5-8 年。 |
| 费用/附带权益 | 1% / 10% 或 1.25% / 12.5%；低于股权策略；优先回报 5-7%以上。 |
| LP 构成 | 保险公司（大比例）、养老金、主权、家族办公室；保险 LP 尤其受收益吸引。 |
| 工具司法管辖区 | 开曼 ELP 标准；面向美国 LP 为特拉华州 LP；有时为卢森堡 SCSp/RAIF（面向欧盟）。 |
| 收益特征 | 收益率 + 利差；高级直接贷款以浮动利率为主。 |

在日本活跃的全球私募信贷机构包括 Apollo、Blackstone Credit（BXCI）、Ares Japan、Oaktree、Sixth Street、Bain Capital Credit、KKR Credit。日本国内私募信贷发展较不成熟；大型银行贷款组合和信托银行贷款参与占据相邻空间。

## 边界情形

上述六种基金类型分类无法对所有现实策略进行清晰归类。常见边界情形：

- **收购加成长混合型**：部分基金明确执行混合授权（如日本国内 GP 同时涉及控制权和少数权益交易）；LPA 附带权益经济结构可能因交易分类而异。

- **单资产接续基金**：仅持有一个组合公司的 GP 主导二级交易。ILPA 接续基金指引将其视为需要更高利益冲突审查（公平意见、维持现状选项、GP 滚存）。

- **联合投资载体**：许多大型 LP（尤其是主权）与 GP 谈判获得并肩联合投资权利，费用/附带权益降低或为零；这些载体不在主基金 LPA 范围内。

- **独立管理账户（SMA）**：大型 LP 谈判专属授权，条款定制；通常为单一 LP 载体，附带独立的附带权益/费用结构。

- **开放式常绿基金**：尤其见于私募信贷和基础设施；永久资本，设有定期赎回窗口，而非有年份的基金存续期。

- **跨资产平台**：部分全球机构（如 Apollo、Blackstone、Brookfield、KKR）运营整合的 PE + 信贷 + 基础设施 + 房地产平台；跨平台的 LP 承诺配置与基金层面经济结构同样重要。

- **政府关联/政策附带权益上限**：部分日本政策导向载体（JIC Capital、部分 DBJ 基金）以有上限的附带权益、政策导向投资标准或联合投资义务运营，与市场标准 LPA 经济结构有所区别。

- **持有期类 PE 的激进对冲基金**：部分日本专注激进基金（Effissimo、Strategic Capital、村上集团、Oasis）在上市公司持有长期 PE 式仓位。它们在正式意义上非 PE GP，但功能上竞争部分日本上市目标交易流。见 [[finance/japan-activist-investor-playbook]]。

- **房地产/基础设施 TK-GK 结构**：日本特有的 TK（匿名組合）和 GK（合同会社）结构用于房地产和基础设施投资，具有税务穿透性；附带权益和分配瀑布机制与标准 LP/LPS 结构不同。

- **日本投资事业有限责任组合 vs 开曼 ELP 税务处理**：日本投资事业有限责任组合在 LPS 层面具有税务透明性；开曼 ELP 同样税务透明，但日本居民 LP 的处理取决于视同常设机构/管理控制测试。见 [[finance/multi-jurisdiction-identity-tax-leverage]] 了解多司法管辖区问题。

- **地方银行贷款 vs PE 交易流**：随着地方银行整合持续（见 [[banking/regional-bank-consolidation-pattern]]），地方银行已成为国内 PE GP 面向中小企业传承目标的 LP 投资者（有限）和交易来源合作伙伴。

- **接续基金 vs 初级基金张力**：承诺参与基金三期的 LP 可能被要求同意将基金三期资产转入接续基金。ILPA 接续基金指引规定了流程；LPAC 同意和多数 LP 批准为典型要求。
