# 日本MBO与强制收购流程

## 概述

日本上市公司MBO或控股股东全面收购是一种上市公司控制权交易。公开流程包括：董事会及特别委员会程序、估值/公平性意见工作、要约收购启动与信息披露、目标方意见/TDnet披露、结算、要约收购后强制收购，以及适用时的退市。

本页位于 [[finance/INDEX|finance domain]] 下，将MBO / 强制收购视为上市公司控制权路径。请与 [[finance/japan-tender-offer-process|Japan tender offer process]]、[[finance/japan-acquisition-finance|Japan acquisition finance]]、[[finance/japan-activist-investor-playbook|Japan activist investor playbook]]、[[finance/cross-border-m-a-japan|cross-border M&A Japan]]、[[securities/tokyo-stock-exchange|Tokyo Stock Exchange]]、[[JapanFG/japan-exchange-group|Japan Exchange Group]]、[[JapanFG/sbi-shinsei-bank|SBI Shinsei Bank]]、[[JapanFG/sony-fg|Sony FG]] 和 [[JapanFG/wealthnavi|WealthNavi]] 配合使用。

## 流程图

| 阶段 | 发生事项 | 文件 / 证据 |
|---|---|---|
| 初始提案 | 管理层、赞助人、母公司或控股股东提出私有化 / 收购方案 | 提案函、已披露的董事会会议纪要、公告背景说明 |
| 利益冲突设置 | 目标公司识别利益冲突并设置程序保护措施 | 特别委员会、外部董事、独立顾问、法律顾问 |
| 估值 / 公平性 | 财务顾问评估价格及公平性 | 估值报告、公平性意见（如已取得）、特别委员会意见 |
| 要约收购启动 | 要约方启动要约收购 | 公告通知、要约收购说明书、要约方新闻稿 |
| 目标方回应 | 目标公司董事会表明支持 / 董事会立场 / 中立 / 反对 | 立场意见书、TDnet披露、特别委员会理由说明 |

| 退市 | 满足交易所标准 / 申请路径时，上市状态终止 | JPX / 交易所通知、退市日程安排 |

## 强制收购路径

| 路径 | 适用情形 | 解读规则 |
|---|---|---|
| 特别控制股东现金退出 | 收购方达到公司法路径所需的高额法定表决权门槛 | 满足持股门槛时的快速路径；核查通知、批准、收购日期及股东查阅权 |
| 股份合并 | 要约收购后持股比例较高但不直接采用现金退出门槛路径时的常见第二步 | 需要股东决议程序，赋予异议股东股份估值权 |
| 类别股份全面取得 | 资本结构支持取得附赎回权股份的情形 | 结构特定性更强；核查章程、类别股份设计及信息披露 |
| 合并 / 股份交换 | 集团重组背景 | 公司法、税务及M&A控制权领域 |

强制收购与退市并非同一概念。强制收购是消除剩余少数股东持股的公司法手段；退市是成功私有化交易后通常随之而来的交易所地位变更结果。

## 公平性 / 少数股东保护重点

经济产业省公平M&A相关材料涵盖MBO及控股股东交易，此类交易中内部人员或控股股东可能同时处于经济决策的两方。公开程序字段包括：

- 特别委员会的构成及独立性；
- 目标方是否就价格及条款进行了谈判；
- 估值方法及假设；
- 公平性意见（如有）；
- 市场核查 / 竞争报价窗口；
- 对少数股东的处置方式；
- 融资确定性；
- 交易后政策；
- 程序是否存在强制性或非强制性表现。

FinWiki 记录公开程序保护措施及其来源文件。

## 融资层面

MBO通常需要收购债务、赞助人股权、滚存股权、过桥贷款或银行融资。融资分析请路由至 [[finance/japan-acquisition-finance|Japan acquisition finance]]。在要约收购说明书中，关注融资证明、已披露的贷款方名称、先决条件、最高购买金额，以及要约方在全数股份应募情形下能否完成结算。

## 来源字段

1. 调取要约方公告、目标方意见、EDINET要约收购说明书、修订文件及TDnet披露。
2. 识别利益冲突类型：管理层收购（MBO）、母子公司收购、控股股东交易、赞助人主导交易或战略买方。
3. 参阅经济产业省公平M&A框架了解程序保护措施。
4. 要约收购条款：价格、期限、最低 / 最高数量、条件、融资安排、结算日及要约收购后政策。
5. 识别强制收购路径及所需门槛 / 股东批准要求。
6. 将法律层面的强制收购与交易所退市区分开来。
7. 记录精确日期，因为条款和意见均可能被修订。
