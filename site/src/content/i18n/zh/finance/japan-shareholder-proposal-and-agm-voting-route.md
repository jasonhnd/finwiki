# 日本股东提案与股东大会表决路径

## 概述

日本股东提案与股东大会表决记录，涵盖公司法股东权利、召集通知、参考文件、书面表决、电子表决、公司治理披露、代理表决、尽职管理披露及会后表决结果。本页记录上市公司治理事件的公开记录字段。

本页属于 [[finance/INDEX|finance domain]]，与 [[finance/japan-activist-investor-playbook|Japan activist investor playbook]]、[[finance/japan-large-shareholding-disclosure|large-shareholding disclosure]]、[[finance/japan-fair-disclosure-and-insider-trading-controls|fair disclosure and insider trading controls]]、[[finance/japan-mbo-and-squeeze-out-process|Japan MBO and squeeze-out process]]、[[finance/japan-tender-offer-process|Japan tender offer process]]、[[securities/tokyo-stock-exchange|Tokyo Stock Exchange]] 及 [[financial-regulators/japan-exchange-group|Japan Exchange Group]] 相关联。

## 路径图

| 层次 | 公开来源路径 | 公开记录字段 |
|---|---|---|
| 股东大会决议 | 公司法第 298 条 | 日期、时间、地点、会议目的、书面表决、电子表决 |
| 召集通知 | 公司法第 299 条 | 通知期限及书面通知路径 |
| 书面表决文件 | 公司法第 301 条 | 参考文件及表决用纸 |
| 电子表决文件 | 公司法第 302 条 | 参考文件及电子表决方式信息 |
| 股东提案权 | 公司法第 303-305 条 | 议题纳入、会议期间提案、通知中提案摘要 |
| 电子表决效力 | 公司法第 312 条 | 以电子方式行使的表决票视同出席股东的表决票 |
| 上市公司治理 | JPX 公司治理守则 / 报告 | 股东大会便利化及治理披露字段 |
| 表决结果 | 发行人披露 / TDnet / 治理材料 | 赞成比率、提案结果、董事选举结果 |

## 公司法字段

| 条文 | 公开字段 |
|---|---|
| 第 298 条 | 董事决定会议日期 / 时间 / 地点、会议目的事项，以及是否允许书面或电子表决 |
| 第 299 条 | 股东大会通知通常须在会议前不迟于两周送达 |
| 第 299 条 | 特定情形下非公众公司适用一周通知路径 |
| 第 299 条 | 采用书面 / 电子表决或设有董事会的公司须以书面方式发出通知 |
| 第 301 条 | 采用书面表决时，随通知一并发送参考文件及表决用纸 |
| 第 302 条 | 采用电子表决时，随通知一并发送参考文件，并以电子方式提供表决方式信息 |
| 第 303 条 | 设有董事会的公司，持有 1% 表决权或 300  票且连续持股六个月以上的股东，可要求将事项纳入会议目的 |
| 第 303 条 | 设有董事会公司的要求时限为会议前不迟于八周 |
| 第 304 条 | 股东可就已列入会议目的的事项在会议上提交议案，但须符合法定限制 |
| 第 305 条 | 符合条件的股东可要求将提案摘要纳入召集材料 |
| 第 305 条 | 设有董事会公司提案摘要要求的 1% / 300 票、持股六个月及八周期限条件同样适用 |
| 第 312 条 | 经公司批准并依内阁命令 / 法务省令路径可进行电子表决；以电子方式行使的表决票视同出席股东的表决票 |

## 股东大会材料路径

| 公开材料 | 记录字段 |
|---|---|
| 召集通知 | 会议日期、地点 / 形式、议程、表决截止日期、表决方式 |
| 参考文件 | 公司提案、股东提案、董事会意见、董事 / 监察人候选人信息 |
| 表决用纸 | 书面表决路径及逐提案选项 |
| 电子表决页面 | 电子表决方式、认证路径、表决截止日期 |
| 公司治理报告 | 董事会结构、委员会结构、独立董事、治理守则说明 |
| 表决结果披露 | 赞成比率、逐提案表决结果、董事选举结果、股东提案结果 |

## JPX 治理报告字段

JPX 及时披露手册材料确认了公司治理报告路径中上市公司股东大会 / 表决便利化字段。来源包汇总了以下公开披露事项：

| JPX 治理报告事项 | 公开字段 |
|---|---|
| 早期发布通知 | 召集通知发布时间及方式 |
| 非集中日程安排 | 股东大会日期集中度 / 日程安排字段 |
| 电子行使表决权 | 电子表决采用情况字段 |
| 机构表决平台 | 参与 ICJ 等机构平台的情况 |
| 英文通知 / 摘要 | 英文通知或摘要字段 |
| 其他便利化措施 | 发行人额外披露的股东大会便利化事项 |
