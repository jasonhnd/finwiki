# 日本股东提案与股东大会表决路径

## 概述

日本的股东提案与股东大会（AGM）表决记录，串联起公司法上的股东权利、召集通知、参考资料、书面表决、电子表决、公司治理披露、委托表决、尽责管理披露以及会后表决结果。本页记录上市公司治理事件的公开记录字段。

本页归属于 [[finance/INDEX|finance domain]]，并链接至 [[finance/japan-activist-investor-playbook|Japan activist investor playbook]]、[[finance/japan-large-shareholding-disclosure|large-shareholding disclosure]]、[[finance/japan-fair-disclosure-and-insider-trading-controls|fair disclosure and insider trading controls]]、[[finance/japan-mbo-and-squeeze-out-process|Japan MBO and squeeze-out process]]、[[finance/japan-tender-offer-process|Japan tender offer process]]、[[securities/tokyo-stock-exchange|Tokyo Stock Exchange]] 与 [[financial-regulators/japan-exchange-group|Japan Exchange Group]]。

## 路径图

| 层级 | 公开来源路径 | 公开记录字段 |
|---|---|---|
| 股东大会决议 | 公司法第 298 条| 日期、时间、地点、会议目的、书面表决、电子表决。 |
| 召集通知 | 公司法第 299 条| 通知期限与书面通知路径。 |
| 书面表决文件 | 公司法第 301 条| 参考资料与表决票。 |
| 电子表决文件 | 公司法第 302 条| 参考资料与电子表决票信息。 |
| 股东提案权 | 公司法第 303-305 条| 议程纳入、会议阶段提案、通知中的提案摘要。 |
| 电子表决效力 | 公司法第 312 条| 以电子方式行使的表决权计为出席股东的表决权。 |
| 上市公司治理 | JPX 公司治理准则 / 报告 | 股东大会便利化与治理披露字段。 |
| 表决结果 | 发行人披露 / TDnet / 治理资料 | 赞成比例、提案结果、董事选任结果。 |

## 公司法字段

| 条文 | 公开字段 |
|---|---|
| 第 298 条| 董事决定会议日期 / 时间 / 地点、作为会议目的的事项，以及是否允许书面或电子表决。 |
| 第 299 条| 股东大会通知原则上须不迟于会议前两周发出。 |
| 第 299 条| 在特定情形下，非公开公司适用一周通知路径。 |
| 第 299 条| 在采用书面 / 电子表决或公司设有董事会的情形下，须发出书面通知。 |
| 第 301 条| 若采用书面表决，参考资料与表决票随通知一并寄送。 |
| 第 302 条| 若采用电子表决，参考资料随通知一并寄送，表决票信息以电子方式提供。 |
| 第 303 条| 对设有董事会的公司，持有 1% 表决权或 300 表决权且持有六个月以上的股东，可要求将事项纳入会议目的。 |
| 第 303 条| 董事会公司的请求时点为不迟于会议前八周。 |
| 第 304 条| 股东可在会议上就已属会议目的范围内的事项提出提案，但受法定限制约束。 |
| 第 305 条| 符合条件的股东可要求将提案摘要纳入召集资料。 |
| 第 305 条| 对设有董事会的公司提出提案摘要请求时，适用相同的 1% / 300 表决权、六个月持有及八周时点字段。 |
| 第 312 条| 经公司批准并依内阁令 / 法务省令路径，可进行电子表决；以电子方式行使的表决权计为出席股东的表决权。 |

## 股东大会资料路径

| 公开资料 | 记录字段 |
|---|---|
| 召集通知 | 会议日期、地点 / 形式、议程、表决截止时间、表决方式。 |
| 参考资料 | 公司提案、股东提案、董事会立场、董事 / 监事候选人信息。 |
| 表决票 | 书面表决路径与逐项提案选择。 |
| 电子表决页面 | 电子表决方式、认证路径、表决截止时间。 |
| 公司治理报告 | 董事会结构、委员会结构、独立董事、治理准则说明。 |
| 表决结果披露 | 赞成比例、各项提案表决结果、董事选任结果、股东提案结果。 |

## JPX 治理报告字段

JPX 的及时披露指引资料在公司治理报告路径中标明了上市公司股东大会 / 表决便利化字段。来源资料包记录了以下公开披露项：

| JPX 治理报告项 | 公开字段 |
|---|---|
| 提前公布通知 | 召集通知公布的时点与方式。 |
| 避开高峰日安排 | 股东大会日期集中 / 安排字段。 |
| 表决权的电子行使 | 电子表决采用字段。 |
| 机构表决平台 | 参与 ICJ 等机构平台。 |
| 英文通知 / 摘要 | 英文通知或摘要字段。 |
| 其他便利化措施 | 发行人披露的其他股东大会便利化事项。 |
