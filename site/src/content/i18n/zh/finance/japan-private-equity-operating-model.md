# 日本私募股权运营模式

## Wiki route

This page sits under [[finance/INDEX|finance domain]]. Read it with [[finance/japan-acquisition-finance|Japan acquisition finance]], [[finance/japan-mbo-and-squeeze-out-process|Japan MBO and squeeze-out process]], [[finance/japan-tender-offer-process|Japan tender offer process]], and [[finance/japan-ib-league-table|Japan IB league table]] when a buyout fund route touches tender offer, acquisition debt, advisory mandate, or post-deal exit.

## TL;DR

日本私募股权由可识别的全球大型基金日本团队（相邻 [[JapanFG/goldman-sachs-japan|Goldman Sachs Japan]] 的 KKR Japan、Bain Capital Japan、Carlyle Japan、CVC Japan、MBK Partners）和国内 GP（Advantage Partners、J-STAR、Polaris Capital、Integral、JIC Capital）共同构成。其运营模式是 GP-LP 合伙制、管理费与附带收益经济结构、多期基金、通过 SPC 实施董事会控制、价值创造计划，以及通过 IPO、二次出售或策略性出售退出。2020, 后的延续基金和 GP 主导的二级交易已成为第三条退出渠道。收购债务使用 [[finance/japan-acquisition-finance|acquisition finance]]，公众公司私有化路径使用 [[finance/japan-tender-offer-process|tender offer process]]。

## GP 格局

日本私募股权 GP 格局呈多层次分布。公开来源包括 GP 网站、新闻稿、JPX TDnet 目标企业披露文件、EDINET 公开要约收购申报及 METI 公允 M&A 指引引用资料。

### 全球大型基金日本团队

| GP | 日本布局 | 典型交易规模 |
|---|---|---|
| KKR Japan | 东京办公室；泛亚基金配置于日本；分拆、私有化、基础设施 | 大型及上中型 |
| Bain Capital Japan | 东京办公室；行业中立，侧重科技、医疗、消费 | 大型及上中型 |
| Carlyle Japan | 东京办公室，设有专属日本并购基金各期 | 中型至上中型 |
| CVC Japan | 东京办公室；泛亚基金 | 中型至上中型 |
| MBK Partners | 专注东北亚，包括日本分拆业务 | 大型 |

全球大型基金通常与大型银行贷款方联合主导分拆、[[finance/japan-mbo-and-squeeze-out-process|MBO]] 及私有化进程。财务顾问资质归属 [[JapanFG/nomura-hd|Nomura]]、[[JapanFG/daiwa-sg|Daiwa]]、[[JapanFG/smbc-nikko|SMBC Nikko]]、[[JapanFG/mizuho-securities]]、[[JapanFG/morgan-stanley-japan|Morgan Stanley Japan]] 和 [[finance/japan-ib-league-table|league tables]] 中的 [[JapanFG/goldman-sachs-japan|Goldman Sachs Japan]]。

### 国内日本 GP

| GP | 基金概况 | 典型交易特征 |
|---|---|---|
| Advantage Partners | 多期 AP 基金 | 中型市场、传承、分拆 |
| J-STAR | 多期 J-STAR 基金 | 小型至中型市场传承 |
| Polaris Capital | 多期 Polaris 基金 | 中型市场、成长、传承 |
| Integral | 多期 Integral 基金，配合实操型运营模式 | 中型市场、扭亏、成长 |
| JIC Capital | JIC 旗下政府关联资本基础 | 战略性、大型、与公共政策一致 |

国内 GP 深耕传承型并购、创始人主导的分拆及中型市场机会，在这些领域中与创始人 CEO、地方银行及本地 M&A 中介机构的关系深度比全球品牌认知更为关键。

## 基金期别与资本结构

私募基金通常以日本有限合伙（LPS）或开曼 LP/特拉华 LP 形式构建。公开来源的期别信息来自 GP 宣布基金完成募集的新闻稿，而非法定申报文件。

| 层级 | 公开来源字段 |
|---|---|
| GP 主体 | 管理人/普通合伙人 |
| 基金载体 | LP / LPS，包括平行馈线基金 |
| 期别 | 最终完成募集年份 |
| 基金规模 | 最终完成募集时的总认缴额 |
| 干粉 | 尚未划款的认缴额（来自新闻稿和历史划款记录的估算） |
| 投资期 | 通常自最终完成募集起 4-6  年 |
| 基金期限 | 通常 10  年加延期 |
| 后续基金 | 后续期别表明特许经营延续性 |

当日本私募股权标的为上市公司时，法定信息披露通过 EDINET 公开要约收购申报及 JPX TDnet 目标方意见进行。融资来源在 SPC 层面披露，不在基金层面披露。SPC 层见 [[finance/japan-acquisition-finance|Japan acquisition finance]]，申报路径见 [[finance/japan-tender-offer-process|tender offer process]]。

## 费用与附带收益经济结构

经典的 2-和-20  模式仍是日本私募股权的基础，但实际条款因 GP、基金规模、LP 构成和期别而有所不同。FinWiki 记录结构性字段而非基金特定数字，因为精确条款表属于 LP 机密。

| 组成部分 | 典型结构 |
|---|---|
| 管理费 | 投资期内通常为认缴额的 1.5-2.0  百分比，投资期后递减至已投入或净资产价值的一定百分比 |
| 附带收益 | 通常为超过优先回报率的基金利润的 20  百分比 |
| 优先回报率/门槛 | 通常为附带收益归属前 8  百分比 IRR |
| 追赶条款 | GP 追赶条款各异（完全追赶至 80/20  vs 部分追赶） |
| 分配瀑布 | 逐项目 vs 欧式全基金瀑布 |
| GP 认缴 | 通常为基金规模的 1-3  百分比，对于独立出来的 GP 偶尔更高 |
| 追回权 | LP 保护机制，若后期损失使总基金 IRR 降至门槛以下时适用 |

大型基金日本团队通常采用欧式全基金分配瀑布。中型和新兴日本 GP 可能使用逐项目瀑布并配合美式机制，以提升对处于成熟阶段市场中 LP 的吸引力。

## 投资流程

典型的日本私募股权投资遵循标准机构流程。一旦目标企业为上市公司，公开来源的可见度即提升。

| 阶段 | 内容 | 公开来源 |
|---|---|---|
| 项目来源 | 财务顾问、投资银行、中介或主动联系 | 独家协议后发布新闻稿（私有目标不公开） |
| 初步报价/意向报价 | 无约束力报价函 | 私有目标无 |
| 尽职调查 | 商业、财务、税务、法律、IT、ESG、运营 | 不公开 |
| 最终报价 | 最终价格、融资函、SPA 条款 | 不公开 |
| 签署 | 签署 SPA；对于上市目标，随后发起 TOB | EDINET TOB 申报，JPX TDnet 公告 |
| 监管审批 | 反垄断（JFTC）、外商投资（FEFTA）、行业监管机构 | 新闻稿，财务省外资通知 |
| 交割 | 注入权益、提取债务、股份转让 | 结算公告 |
| 强制收购 | 会社法现金退出、股份合并 | EDINET，JPX TDnet，参见 [[finance/japan-mbo-and-squeeze-out-process|MBO and squeeze-out]] |
| 退市 | JPX 退市通知 | JPX 上市状态更新 |

目标企业为上市公司时，TOB 流程受 FIEA 公开要约收购规则和 METI 公允 M&A 指引约束。披露主干参见 [[finance/japan-tender-offer-process|Japan tender offer process]]。

## 价值创造手册

私募股权价值创造通常结合运营改善、资本效率提升和战略重定位。

| 杠杆 | 日本典型应用 |
|---|---|
| 收入增长 | 新产品推出、地理扩张（尤其是亚洲内部和美国）、价格纪律 |
| 成本削减 | 采购、制造基地、销售管理费用整合 |
| 营运资本 | 存货周转、应收账款、应付账款、不动产变现 |
| 分拆 | 将非核心部门出售给私募股权以实现专注管理 |
| 补强并购 | 在分散行业的整合策略 |
| 治理 | 新增董事会、CEO、CFO、KPI 仪表盘、月度经营审查 |
| 资本结构 | 再融资、股息资本再融资、优化杠杆率 |
| ESG/可持续发展 | 碳足迹、治理准则采纳、多元化指标 |

在日本，传承型并购（创始人 CEO 年迈退休）是有别于西方市场的结构性机遇。国内中型 GP（Advantage、J-STAR、Polaris）更强调关系延续性、创始人家族协调及渐进式专业化，而非激进重组。

## 退出渠道

日本私募股权退出通过三条主要渠道进行。近期各期别已将 GP 主导的二级交易和延续基金作为第四条路径。

### IPO 退出

上市 IPO 退出通过 [[securities/japan-ipo-listing-disclosure-route|Japan IPO listing disclosure route]] 进行，利用 [[securities/japan-underwriting-market-structure|underwriting market structure]] 的主承销商和路演执行能力。私募股权方通常在 IPO 后保留残余股权，须遵守锁定期。

| IPO 字段 | 细节 |
|---|---|
| 上市场所 | Prime、Standard、Growth、TOKYO PRO 市场 |
| 承销商 | 主承销商 + 联合管理人（参见 [[finance/japan-ib-league-table|league table]]） |
| 锁定期 | 私募股权赞助方通常为 180  天 |
| 绿鞋 | 超额配售期权支持价格稳定 |
| IPO 后持股 | 私募股权可能保留董事会席位和影响力 |

### 策略性出售

向企业买方出售，包括 [[JapanFG/mitsubishi-corp|Mitsubishi Corp]]、[[JapanFG/mitsui-co|Mitsui & Co]]、[[JapanFG/itochu-corp|Itochu Corp]]、跨境买方或行业战略买方。通常通过财务顾问主导的竞争性竞标程序进行。

### 向另一私募股权的二次出售

赞助商间交易，通常在目标公司从中型成长为大型后，由具备增量价值创造能力的不同 GP 接手。

### 延续基金 / GP 主导的二级交易

单一资产或多资产延续载体，GP 将资产转入由二级市场投资者支持的新基金，使原有 LP 获得流动性，同时 GP 保留运营控制权。定价通常通过正式净资产价值流程、公允意见及 LP 咨询委员会批准确定。

## 延续基金与二级交易

GP 主导的二级交易已成为日本可认可的退出/延期渠道，与全球趋势一致。

| 路径 | 结构 |
|---|---|
| 单一资产延续载体 | 新基金持有一个核心资产；二级市场投资者为主要资本提供方 |
| 多资产延续载体 | 新基金持有剩余投资组合公司的组合 |
| 条带出售 | 二级市场买方在基金层面跨多项资产收购条带 |
| 向 LP 的要约收购 | 以商定净资产价值向 LP 提供现金退出；GP 滚动延续 |
| LP 二级交易 | LP 之间的基金权益转让，须经 GP 同意 |

与美国/欧洲相比，日本国内二级交易规模仍较小，但随着 GP 特许经营在 3-4 期别后成熟，规模正在增长。

## 大型银行/夹层贷款方地图

日本并购债务通常由大型银行的杠杆收购部门安排。完整结构见 [[finance/japan-acquisition-finance|Japan acquisition finance]]。

| 贷款方 | 角色 |
|---|---|
| [[JapanFG/mufg|MUFG]] | MUFG 银行并购融资/LBO 贷款，频繁担任牵头安排行 |
| [[JapanFG/smfg|SMFG]] | SMBC LBO 部门，频繁参与 |
| [[JapanFG/mizuho-fg|Mizuho FG]] | 瑞穗银行 LBO 部门 |
| [[JapanFG/dbj|DBJ]] | 日本政策投资银行作为战略性/夹层参与方 |
| 夹层提供方 | 专业基金、地方银行联合体、保险公司私人债务 |

贷款方角色归属可通过 [[finance/japan-ib-league-table|league tables]] 验证（贷款/LBO 类别中已公开部分）。

## JIC Capital 具体案例

[[JapanFG/dbj|DBJ]] 与政策相关但有别于 JIC Capital（日本投资公司 Capital），后者属于 METI 战略投资伞形结构。JIC Capital 投资于仅靠私人资本不足的战略行业和重组领域。公开披露仅限于 JIC 整体新闻稿。

## 监管与信息披露渠道

| 渠道 | 相关性 |
|---|---|
| FIEA 公开要约收购规则 | 上市目标收购信息披露（参见 [[finance/japan-tender-offer-process|tender offer process]]） |
| METI 公允 M&A 指引 | 对少数股东的程序保护 |
| METI 收购指引 | 董事会在非邀约/维权投资者情境下的职责（参见 [[finance/japan-activist-investor-playbook|activist playbook]]） |
| FSA 大额持股信息披露 | 大宗股权持有报告（参见 [[finance/japan-large-shareholding-disclosure|large shareholding disclosure]]） |
| FEFTA 外商投资通知 | 触发行业/国籍条件时须事前通知 |
| JFTC 合并审查 | 反垄断审批 |
| TSE / JPX 上市规则 | 退市标准，私有化后注销登记 |

私募基金层面的监管渠道包括：根据 FIEA 登记为第二类金融商品交易业者或投资管理业者，以及针对基金营销的合格机构投资者/专业投资者豁免路径。

## LP 群体

日本私募股权基金 LP 通常包括：

| LP 类型 | 概况 |
|---|---|
| 日本机构 | GPIF、日本邮政银行/保险、农林中央金库、大型银行私募基金组合基金、地方银行、主要保险公司 |
| 日本企业 | 上市企业养老金和财务分配 |
| 亚洲主权/养老基金 | 配置于日本的亚洲主权财富基金和养老基金 |
| 全球 LP | 美国/欧洲捐赠基金、基金会、家族办公室、组合基金 |
| GP 认缴 | 内部 GP 合伙人资本 |

LP 构成影响基金治理、补充协议条款、ESG 报告和报告语言。中型日本国内基金通常日本 LP 占比较高；全球大型基金日本载体对全球 LP 池的依赖程度更高。
