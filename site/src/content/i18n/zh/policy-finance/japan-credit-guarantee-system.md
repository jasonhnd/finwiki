# 日本信用保证制度

## Wiki route

本条目位于 [[policy-finance/INDEX|policy-finance index]] 之下，是中小企业金融机制页面。可与 [[policy-finance/national-federation-credit-guarantee-corporations|Japan Federation of Credit Guarantee Corporations (JFG)]] 对照以理解协调机构视角，与 [[policy-finance/japan-policy-finance-system|Japan policy finance system]] 对照以理解更广边界，并与行业特定对应项 [[policy-finance/agriculture-credit-guarantee-system|agriculture]] / [[policy-finance/fisheries-credit-guarantee-system|fisheries]] 对照。地方示例包括 [[policy-finance/hokkaido-credit-guarantee-corp|Hokkaido CGC]]、[[policy-finance/tokyo-credit-guarantee-corp|Tokyo CGC]]、[[policy-finance/osaka-credit-guarantee-corp|Osaka CGC]]、[[policy-finance/aichi-credit-guarantee-corp|Aichi CGC]]、[[policy-finance/hyogo-credit-guarantee-corp|Hyogo CGC]]。

## TL;DR

日本信用保证制度（菫｡逕ｨ菫晁ｨｼ蛻ｶ蠎ｦ）是支持日本中小企业和小型企业融资的公共信用基础设施。它是分层体系：（1）51  个都道府县 / 市级 **菫｡逕ｨ菫晁ｨｼ蜊比ｼ・**（Credit Guarantee Corporations, CGCs）为符合条件的中小企业从金融机构取得的借款提供保证；（2）**蜈ｨ蝗ｽ菫｡逕ｨ菫晁ｨｼ蜊比ｼ夐｣蜷井ｼ・**（JFG）协调 51  家 CGC，并汇集责任共享负担；（3）**菫｡逕ｨ菫晞匱**（信用保险）层由 [[JapanFG/jfc|Japan Finance Corporation (JFC)]] 运作，对 CGC 保证进行再保险，使最终剩余损失落在公共信用基础设施上，而不是压垮 CGC 层。合起来，这就是 **菫｡逕ｨ陬懷ｮ悟宛蠎ｦ**（信用补充制度）。它不是单一贷款人，而是一套公共信用架构。2020-2022  ZeroZero 贷款（螳溯ｳｪ辟｡蛻ｩ蟄舌・辟｡諡・ｿ晁檮雉・）特别项目通过该体系运行，并持续在 2024-2026  的代位清偿周期中消化。

## 1. System architecture（菫｡逕ｨ陬懷ｮ悟宛蠎ｦ stack）

| Layer | Role | FinWiki route |
|---|---|---|
| 中小企业 / 小型企业借款人 | 通过金融机构和保证路径申请经营资金。 | 本系统路线 |
| 金融机构（贷款人） | 发放贷款，并在适用制度下分担责任。 | 相关时见 [[banking/regional-bank-consolidation-pattern]] / 信用金库 / 信用合作社页面 |
| 地方信用保证协会（51） | 保证符合条件的中小企业债务；处理代位清偿 / 回收。 | 本系统 + 具战略意义的地方 CGC 页面（如 [[policy-finance/hokkaido-credit-guarantee-corp]]） |
| 蜈ｨ蝗ｽ菫｡逕ｨ菫晁ｨｼ蜊比ｼ夐｣蜷井ｼ・（JFG） | 协调 51  家 CGC；管理责任共享制度的资金流。 | [[policy-finance/national-federation-credit-guarantee-corporations]] |
| 菫｡逕ｨ菫晞匱（信用保险）：JFC | 对 CGC 保证进行再保险；吸收 CGC 代位清偿后的剩余损失。 | [[JapanFG/jfc]] |
| METI / SME Agency（荳ｭ蟆丈ｼ∵･ｭ蠎・） | 政策监督方；设定项目设计、适格性、危机模式授权。 | [[policy-finance/INDEX]] |
| 驛ｽ驕灘ｺ懃恁 / 蟶ら伴譚・ | 地方政府：共同出资地方项目，指定地方紧急事件，提供补贴。 | 地方政府路线 |

因此，该体系是一条**垂直公共信用管线**：借款人 竊・贷款人 竊・地方 CGC（第一损失）竊・JFG 协调 竊・JFC 信用保险（再保险）竊・METI 政策后盾。

## 4. Guarantee ratio mechanics（雋ｬ莉ｻ蜈ｱ譛牙宛蠎ｦ）

**雋ｬ莉ｻ蜈ｱ譛牙宛蠎ｦ**（责任共享制度）于 2007-10  引入，它把明确定义的一部分风险留给贷款金融机构，而不是 100% 保证：

- **标准（荳闊ｬ菫晁ｨｼ，騾壼ｸｸ）**：CGC 保证贷款的 **80%**；贷款人保留 **20%** 剩余风险（或 pari-passu 损失共享变体）。这是 2007  之后的默认安排。
- **全额保证（100%）例外**：某些政策项目（繧ｻ繝ｼ繝輔ユ繧｣繝阪ャ繝・1号繝ｻ2号繝ｻ3号、蜊ｱ讖滄未騾｣菫晁ｨｼ、满足条件的 蜑ｵ讌ｭ、满足条件的 繝槭Ν邨・、危机设计中的 ZeroZero）保留 100% 保证，无贷款人保留。
- **Pari-passu 损失共享变体**：在责任共享制度下，贷款人可选择（a）80% 保证结构，或（b）CGC 与贷款人在既定公式下按 pari-passu 分享损失的结构。

2007  引入 雋ｬ莉ｻ蜈ｱ譛・ 是一次有意的政策转向，旨在前 2007  100% 保证制度引发道德风险担忧后，恢复贷款人筛选纪律。

## 5. Safety-Net Guarantee programs（繧ｻ繝ｼ繝輔ユ繧｣繝阪ャ繝井ｿ晁ｨｼ）

**繧ｻ繝ｼ繝輔ユ繧｣繝阪ャ繝井ｿ晁ｨｼ** 框架有 8  个编号指定项，每项覆盖不同困境情景，并提供强化条件（通常为 100% 保证 + 借款人通常保证上限之外的单独额度）：

- **1号**：大型企业重组影响（由 METI 基于大型企业倒闭 / 重组影响下游中小企业认定事件）。
- **2号**：大型企业运营冲击（例如工厂关闭并影响供应链）。
- **3号**：银行倒闭 / 金融机构清算对借款中小企业的影响。
- **4号**：自然灾害 / 大范围紧急事件（经常因台风、地震、洪水启动）。
- **5号**：行业范围结构性困境；METI 按宏观 / 金融条件每季度指定特定行业。（在通胀 / 疫情 / 能源冲击期大量使用。）
- **6号**：金融机构倒闭（具体范围来自 3号）。
- **7号**：伴随合作金融影响的大型企业重组。
- **8号**：RCC（Resolution and Collection Corporation）请求影响。

注意，**蜊ｱ讖滄未騾｣菫晁ｨｼ**（与 繧ｻ繝ｼ繝輔ユ繧｣繝阪ャ繝・ 分开）是更广的国家紧急框架，提供 100% 保证，只在罕见全国危机指定时启动（COVID-19, 、2008  GFC 等同阈值等）。

## 6. ZeroZero loan unwind status（2020  竊・2026）

**ZeroZero loan**（螳溯ｳｪ辟｡蛻ｩ蟄舌・辟｡諡・ｿ晁檮雉・）项目是应对 COVID-19 的核心危机金融措施：

| Phase | Period | Status |
|---|---|---|
| 发放放量 | 2020-03  竊・2020-07  | METI 于 2020-05 通过信用保证体系推出民间部门 ZeroZero；数月内快速增长至数万亿日元。JFC 同时直接运行平行 ZeroZero 项目。 |
| 持续发放 | 2020-08  竊・2021-03  | 通过全国 51  家 CGC 继续大规模保证发放。 |
| 发放关闭 | 2021-03  竊・2022  | 民间部门 ZeroZero 停止接受新申请（少数类别有限延期）。JFC ZeroZero 也通过 2022 类似关闭。 |
| 宽限期 | 2020  发放 + 3 年宽限 | 借款人通常有本金宽限期（本金常见最长 5  年，初期期间有利息补贴）。2020 年度大部分贷款在 2023-2024 进入还款开始。 |
| 代位清偿波 | 2023-2026  | 随还款开始触发，CGC 代位清偿（莉｣菴榊ｼ∵ｸ・）上升。多家商业行业媒体报道显示，自 2023  后期起，全国 莉｣菴榊ｼ∵ｸ・ 量明显增加。 |
| 后 COVID 退出框架 | 2022-2025  | METI / SME Agency 与 JFG 协调用于 ZeroZero 借款人的重整 / 再融资统一指引（蛟滓鋤菫晁ｨｼ / 邨悟霧謾ｹ蝟・ 产品提供；“莠区･ｭ閠・＃縺ｨ縺ｮ邊倥ｊ蠑ｷ縺・髪謠ｴ”框架）。 |
| 持续尾部 | 2026+ | 随剩余 2020 年度贷款完成还款，代位清偿尾部延续；破产统计在 2024-2026 期间继续将 ZeroZero 相关破产作为跟踪子类别。 |

ZeroZero 消化是信用保证体系**近期定义性压力事件**，在全国 CGC 代位清偿统计中可见。该消化在地方层面的覆盖出现在 [[policy-finance/hokkaido-credit-guarantee-corp|Hokkaido CGC]] 以及其他具有战略意义的地方页面中。

## 7. Funding source

各地方 CGC 的资本来源混合包括：

- **国家（蝗ｽ）出资**：METI / SME Agency 资本和补贴。
- **都道府县 / 市（驛ｽ驕灘ｺ懃恁繝ｻ蟶・）出资**：地方政府资本，反映地方中小企业政策优先级。
- **贷款人出资（驥題檮讖滄未諡蜃ｺ）**：合作银行 / 信用金库 / 信用合作社向其地方 CGC 出资。
- **保证费收入（菫晁ｨｼ譁・）**：对保证本金收取的年费率，通常处于低个位数百分比区间（政策项目有补贴）。
- **联合会 pooled fund**：JFG 管理的储备，支持系统层面协调。
- **JFC 信用保险**：通过 [[JapanFG/jfc|JFC 信用保険業務]] 账户在代位清偿事件上提供再保险赔付。

**公共信用风险承担瀑布**为：借款人回收 竊・CGC 储备 竊・JFC 再保险 竊・国家财政后盾。

## 8. Default / subrogation flow

1. 借款人在保证贷款上违约。
2. 贷款人根据保证提出请求；CGC 向贷款人就保证部分支付 **莉｣菴榊ｼ∵ｸ・（代位清偿）**（在 雋ｬ莉ｻ蜈ｱ譛・ 下通常为 80%，在危机方案下为 100%）。
3. CGC 对借款人 / 抵押品 / 个人保证人进行**豎ょ─（回收）**。
4. CGC 根据信用保险合同向 JFC 提出保险索赔；JFC 向 CGC 支付保险金。
5. 剩余损失（扣除回收 + JFC 保险后）由 CGC 储备吸收；如果储备耗尽构成威胁，联合会 / 国家资本注入提供后盾。

回收率因地区、行业和抵押品类型而异：城市 CGC（东京、大阪）通常比农村 CGC（北海道、Tﾅ紘oku、Kyﾅｫshﾅｫ 农村）回收率更高，因为商业地产和设备的二级市场更深。

## 10. Boundary cases

- **制度 vs 联合会**：本页解释机制；[[policy-finance/national-federation-credit-guarantee-corporations]] 解释全国协调机构。
- **保证 vs 贷款**：CGC 保证信用；贷款由金融机构发放。CGC 不是贷款人。
- **制度 vs 单个 CGC**：本页覆盖制度；地方 CGC 页面覆盖特定都道府县 / 市法人。
- **中小企业 vs 农业 / 渔业**：农业使用 [[policy-finance/agriculture-credit-guarantee-system]]；渔业使用 [[policy-finance/fisheries-credit-guarantee-system]]。
- **CGC vs JFC 直接贷款**：JFC 也直接向中小企业贷款；CGC 体系是对民间贷款提供保证的机制，而不是直接贷款项目。
