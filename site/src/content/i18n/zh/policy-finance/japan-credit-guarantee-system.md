# 日本信用保证制度

## Wiki 路线

本条目位于 [[policy-finance/INDEX|政策金融索引]] 之下，是理解日本中小企业融资支持机制的制度页面。可与 [[policy-finance/national-federation-credit-guarantee-corporations|全国信用保证协会联合会（JFG）]] 对照阅读，以把握全国协调机构的角色；也可与 [[policy-finance/japan-policy-finance-system|日本政策金融体系]] 对照阅读，以理解更广义的公共信用边界。行业特定对应项包括 [[policy-finance/agriculture-credit-guarantee-system|农业信用保证制度]] 和 [[policy-finance/fisheries-credit-guarantee-system|渔业信用保证制度]]。地方示例包括 [[policy-finance/hokkaido-credit-guarantee-corp|北海道信用保证协会]]、[[policy-finance/tokyo-credit-guarantee-corp|东京信用保证协会]]、[[policy-finance/osaka-credit-guarantee-corp|大阪信用保证协会]]、[[policy-finance/aichi-credit-guarantee-corp|爱知信用保证协会]]、[[policy-finance/hyogo-credit-guarantee-corp|兵库信用保证协会]]。

## TL;DR

日本信用保证制度是支持中小企业和小规模经营者融资的公共信用基础设施。体系由地方信用保证协会、全国信用保证协会联合会（JFG）和 JFC 信用保险层构成。2020-2022 年 Zero-Zero 贷款特别项目通过该体系运行，并在 2024-2026 年的代位清偿周期中继续消化。

## 1. 制度架构（信用补完制度栈）

| 层级 | 职能 | FinWiki 路线 |
|---|---|---|
| 中小企业 / 小规模经营者 | 通过金融机构和保证路径申请经营资金。 | 本系统路线 |
| 金融机构（贷款人） | 发放贷款，并在适用制度下分担一部分信用风险。 | 相关时参照 [[banking/regional-bank-consolidation-pattern]] / 信用金库 / 信用合作社页面 |
| 地方信用保证协会（51 家） | 保证符合条件的中小企业债务；处理代位清偿和回收。 | 本系统 + 具战略意义的地方 CGC 页面（如 [[policy-finance/hokkaido-credit-guarantee-corp]]） |
| 全国信用保证协会联合会（JFG） | 协调 51 家 CGC；管理责任共享制度的资金流。 | [[policy-finance/national-federation-credit-guarantee-corporations]] |
| 信用保险：JFC | 对 CGC 保证进行再保险；吸收 CGC 代位清偿后的剩余损失。 | [[JapanFG/jfc]] |
| METI / SME Agency（经济产业省 / 中小企业厅） | 政策监督方；设定项目设计、适格性和危机模式授权。 | [[policy-finance/INDEX]] |
| 都道府县 / 市町村 | 共同出资地方项目，指定地方紧急事件，提供补贴。 | 地方政府路线 |

因此，该体系是一条**垂直公共信用管线**：借款人 → 贷款人 → 地方 CGC（第一损失）→ JFG 协调 → JFC 信用保险（再保险）→ METI 政策后盾。

## 2. 机构概览：地方信用保证协会

| 项目 | 内容 |
|---|---|
| 法令 | 《信用保证协会法》（1953） |
| 法律形态 | 特别法授权的认可信用机构；具有公共信用基础设施性质 |
| 数量 | 51 家（47 个都道府县 + 4 个市，包括横滨市、川崎市、名古屋市、岐阜市） |
| 所管 | 经济产业省（METI）/ 中小企业厅（SME Agency）/ 都道府县 |
| 全国锚点 | 全国信用保证协会联合会（JFG） |
| 再保险 | JFC 信用保险业务 |

## 3. 主要保证菜单（制度层）

- **一般保证**：标准中小企业银行贷款保证，位于**责任共享制度**下；普通项目通常为 80% 保证 / 20% 由贷款人保留。
- **小规模企业经营改善贷款保证**：面向商工会、商工会议所等推荐案件的小规模经营改善贷款保证。
- **创业相关保证**：创业 / 新业务保证。
- **事业承继特别保证**：事业承继场景下的信用保证。
- **经营创新等支援保证**：经营创新 / 业务转型保证。
- **借换保证**：再融资 / 债务整合保证。
- **流动资产担保融资保证（ABL）**：以应收账款 / 存货为担保的保证。
- **Safety-Net 保证（1 号至 8 号）**：针对指定行业或特定冲击事件、带有强化条件的保证。（见 §5。）
- **危机相关保证**：国家紧急框架；在宣布危机时最高可达 100% 保证。
- **灾害相关保证**：灾害恢复保证。
- **新型冠状病毒感染症特别保证 / Zero-Zero 贷款**：COVID 时期的实质无息、无担保特别保证（2020-05 放量；2021-03 停止新发放；服务和消化持续到 2026+）。（见 §6。）

## 4. 保证比例机制（责任共享制度）

**责任共享制度**于 2007-10 引入，把明确定义的一部分风险留给贷款金融机构，而不是继续普遍提供 100% 保证：

- **标准（一般保证，通常情形）**：CGC 保证贷款的 **80%**；贷款人保留 **20%** 剩余风险（或 pari-passu 损失共享变体）。这是 2007 年之后的默认安排。
- **全额保证（100%）例外**：某些政策项目（Safety-Net 保证 1 号、2 号、3 号，危机相关保证，满足条件的创业保证，满足条件的小规模企业经营改善贷款保证，以及危机设计中的 Zero-Zero 贷款）保留 100% 保证，无贷款人保留。
- **Pari-passu 损失共享变体**：在责任共享制度下，贷款人可选择（a）80% 保证结构，或（b）CGC 与贷款人在既定公式下按 pari-passu 分享损失的结构。

2007 年引入责任共享制度，是一次有意的政策转向：在 2007 年以前 100% 保证制度引发道德风险担忧后，恢复贷款人的筛选纪律。

## 5. Safety-Net 保证项目

**Safety-Net 保证**框架有 8 个编号指定项，每项覆盖不同困境情景，并提供强化条件（通常为 100% 保证 + 借款人通常保证上限之外的单独额度）：

- **1 号**：大型企业重组影响（由 METI 基于大型企业倒闭 / 重组对下游中小企业的影响认定事件）。
- **2 号**：大型企业运营冲击（例如工厂关闭并影响供应链）。
- **3 号**：银行倒闭 / 金融机构清算对借款中小企业的影响。
- **4 号**：自然灾害 / 大范围紧急事件（经常因台风、地震、洪水启动）。
- **5 号**：行业范围结构性困境；METI 按宏观 / 金融条件每季度指定特定行业。（在通胀 / 疫情 / 能源冲击期大量使用。）
- **6 号**：金融机构倒闭（具体范围来自 3 号）。
- **7 号**：伴随合作金融影响的大型企业重组。
- **8 号**：RCC（Resolution and Collection Corporation）请求影响。

注意，**危机相关保证**与 Safety-Net 保证分开，是更广的国家紧急框架，提供 100% 保证，只在罕见全国危机指定时启动（COVID-19、2008 年全球金融危机等同阈值事件）。

## 6. Zero-Zero 贷款消化状态（2020 → 2026）

**Zero-Zero 贷款**（实质无息、无担保贷款）项目是应对 COVID-19 的核心危机金融措施：

| 阶段 | 期间 | 状态 |
|---|---|---|
| 发放放量 | 2020-03 → 2020-07 | METI 于 2020-05 通过信用保证体系推出民间部门 Zero-Zero；数月内快速增长至数万亿日元。JFC 同时直接运行平行 Zero-Zero 项目。 |
| 持续发放 | 2020-08 → 2021-03 | 通过全国 51 家 CGC 继续大规模保证发放。 |
| 发放关闭 | 2021-03 → 2022 | 民间部门 Zero-Zero 停止接受新申请（少数类别有限延期）。JFC Zero-Zero 也在 2022 年前后类似关闭。 |
| 宽限期 | 2020 年发放 + 3 年宽限 | 借款人通常有本金宽限期（本金宽限常见最长 5 年，初期期间有利息补贴）。2020 年度大部分贷款在 2023-2024 年进入还款开始阶段。 |
| 代位清偿波 | 2023-2026 | 随还款开始触发，CGC 代位清偿上升。多家商业行业媒体报道显示，自 2023 年后期起，全国代位清偿量明显增加。 |
| 后 COVID 退出框架 | 2022-2025 | METI / SME Agency 与 JFG 协调用于 Zero-Zero 借款人的重整 / 再融资统一指引（借换保证 / 经营改善类产品；“伴走支援”框架）。 |
| 持续尾部 | 2026+ | 随剩余 2020 年度贷款完成还款，代位清偿尾部延续；破产统计在 2024-2026 年期间继续将 Zero-Zero 相关破产作为跟踪子类别。 |

Zero-Zero 消化是信用保证体系**近期定义性压力事件**，在全国 CGC 代位清偿统计中可见。该消化在地方层面的覆盖出现在 [[policy-finance/hokkaido-credit-guarantee-corp|北海道信用保证协会]] 以及其他具有战略意义的地方页面中。

## 7. 资金来源

各地方 CGC 的资本来源混合包括：

- **国家出资**：METI / SME Agency 资本和补贴。
- **都道府县 / 市出资**：地方政府资本，反映地方中小企业政策优先级。
- **贷款人出资**：合作银行 / 信用金库 / 信用合作社向其地方 CGC 出资。
- **保证费收入**：对保证本金收取的年费率，通常处于低个位数百分比区间（政策项目有补贴）。
- **联合会 pooled fund**：JFG 管理的储备，支持系统层面协调。
- **JFC 信用保险**：通过 [[JapanFG/jfc|JFC 信用保险业务]] 账户在代位清偿事件上提供再保险赔付。

**公共信用风险承担瀑布**为：借款人回收 → CGC 储备 → JFC 再保险 → 国家财政后盾。

## 8. 违约 / 代位清偿流程

1. 借款人在保证贷款上违约。
2. 贷款人根据保证提出请求；CGC 向贷款人就保证部分支付**代位清偿**（在责任共享制度下通常为 80%，在危机方案下为 100%）。
3. CGC 对借款人 / 抵押品 / 个人保证人进行**求偿 / 回收**。
4. CGC 根据信用保险合同向 JFC 提出保险索赔；JFC 向 CGC 支付保险金。
5. 剩余损失（扣除回收 + JFC 保险后）由 CGC 储备吸收；如果储备耗尽构成威胁，联合会 / 国家资本注入提供后盾。

回收率因地区、行业和抵押品类型而异：城市 CGC（东京、大阪）通常比农村 CGC（北海道、东北、九州农村）回收率更高，因为商业地产和设备的二级市场更深。

## 9. 年度演进

| 年份 | 事件 |
|---|---|
| 1937 | 东京信用保证协会作为首家信用保证协会成立。 |
| 1953 | 《信用保证协会法》颁布；形成全国 CGC 框架。 |
| 1958 | 中小企业信用保险公库成立（后整合入 JFC）。 |
| 2007-10 | 引入责任共享制度：默认从 100% 保证转为 80% / 20% 贷款人保留。 |
| 2008-2009 | 全球金融危机紧急应对：Safety-Net 保证 5 号广泛启动；紧急保证项目采用后来式 100% 框架。 |
| 2011 | 东北大地震：灾害相关保证 + 4 号广泛启动。 |
| 2020-05 | 通过 CGC 体系启动民间部门 Zero-Zero（实质无息、无担保）项目。 |
| 2021-03 | 民间部门 Zero-Zero 发放关闭（少数延期）。 |
| 2023+ | 随 2020 年度宽限期结束，Zero-Zero 代位清偿波上升。 |
| 2024-2025 | 重整 / 借换产品组合扩大；“伴走支援”框架持续使用。 |
| 2025 | 持续消化；破产统计显示 Zero-Zero 相关子类别尾部。 |

## 10. 边界情形

- **制度 vs 联合会**：本页解释机制；[[policy-finance/national-federation-credit-guarantee-corporations]] 解释全国协调机构。
- **保证 vs 贷款**：CGC 保证信用；贷款由金融机构发放。CGC 不是贷款人。
- **制度 vs 单个 CGC**：本页覆盖制度；地方 CGC 页面覆盖特定都道府县 / 市法人。
- **中小企业 vs 农业 / 渔业**：农业使用 [[policy-finance/agriculture-credit-guarantee-system]]；渔业使用 [[policy-finance/fisheries-credit-guarantee-system]]。
- **CGC vs JFC 直接贷款**：JFC 也直接向中小企业贷款；CGC 体系是对民间贷款提供保证的机制，而不是直接贷款项目。
