# Japan Export-Import Bank (historical predecessor of JBIC)

## TL;DR

日本输出入银行（日本輸出入銀行 / JEXIM）在 1950 至 1999期间是日本主要的国家出口信贷与海外投资贷款机构。1999 ，它与海外经济协力基金（海外経済協力基金 / OECF）合并，组成初代国际协力银行（JBIC）。理解 JEXIM 对于任何关于日本战后出口主导型工业化、ODA 架构，以及 [[financial-regulators/jbic|JBIC]] 与 [[policy-finance/jica|JICA]] 之间现代分割的历史分析都很重要。

本页作为当代 JBIC / JICA / NEXI / DBJ 公共金融图谱的历史锚点，归属于 [[policy-finance/INDEX|policy-finance index]]。

## 1. Institutional boundary

| Item | Reading |
|---|---|
| Japanese name | 日本輸出入銀行（最初：日本輸出銀行 1950–1953） |
| English name | Export-Import Bank of Japan / Japan Export-Import Bank |
| Established | 1950-12 ，作为日本输出银行 |
| Renamed | 1953-04 ，更名为日本输出入银行（增加进口融资） |
| Dissolved | 1999-10-01 — 与 OECF 合并组成初代 JBIC |
| Legal form | 特殊政府法人，100% 政府出资 |
| Supervisory route | 大藏省 |
| Succession lane | [[financial-regulators/jbic|JBIC]] 承继出口与海外投资贷款；OECF 的 ODA 贷款后于 2008 移交至 [[policy-finance/jica|JICA]] |

## 2. Why this page exists separately

1999 的合并与 2008 的重组，给当代日本政策金融图谱留下了清晰的机构分割 — [[financial-regulators/jbic|JBIC]] 负责海外业务融资，[[policy-finance/jica|JICA]] 负责 ODA — 但 JEXIM 的遗产在以下方面仍然重要：

- 阅读使用「Eximbank Japan」或「JEXIM」术语的较旧项目融资文档。
- 理解为何日本当代出口信贷架构在结构上不同于 US EXIM、Korea EXIM、China EXIM 或 KfW IPEX — 1999 的合并整合了其他国家保留在独立机构中的职能。
- 追溯当代 [[financial-regulators/jbic|JBIC]] 业务条线与 [[policy-finance/japan-project-finance-stack-diagram|Japanese project-finance stack]] 的谱系。
- 区分符合 OECD 安排资格的出口信贷（JBIC 继承者）与 ODA 优惠贷款（JICA 继承者）。

## 3. Function map (historical)

| Function | Description |
|---|---|
| Export buyer credit | 向日本资本货物（成套设备、船舶、运输设备）的外国买家提供长期贷款。 |
| Export supplier credit | 向日本出口企业提供贷款，为延期付款的出口合同融资。 |
| Overseas investment lending | 为日本企业的海外股权投资提供融资。 |
| Untied loan（アンタイドローン） | 向不与特定日本出口挂钩的外国主权方及实体提供长期贷款，通常支持资源进口。 |
| Import financing | 向日本进口企业提供长期贷款，尤其是能源与原材料方面。 |

这些职能如今全部在 [[financial-regulators/jbic|JBIC]] 之下延续，并在当代 [[policy-finance/oecd-export-credit-arrangement|OECD Arrangement]] 纪律的框架内加以框定。

## 4. Historical trajectory

| Year | Event |
|---|---|
| 1950-12 | 设立日本输出银行（日本輸出銀行）以支持战后出口主导型复苏。 |
| 1953-04 | 更名为日本输出入银行，增加进口融资授权。 |
| 1957–1970年代 | 战后工业化时期；为成套设备出口、造船及重工业出口融资。 |
| 1980年代 | 日元国际化与日本企业海外投资扩张；JEXIM 扩展海外投资贷款。 |
| 1990年代 | 泡沫后时期；资源安全保障与新兴市场敞口扩大。 |
| 1999-10-01 | 与 OECF 合并组成初代国际协力银行（旧 JBIC）。 |
| 2003-10 | 改革下 OECF 的 ODA 贷款组合移交至 [[policy-finance/jica|JICA]]。 |
| 2008-10-01 | 初代 JBIC 作为其国际金融部门被吸收进 [[financial-regulators/jfc|JFC]]。 |
| 2012-04-01 | 国际金融部门作为现行 [[financial-regulators/jbic|JBIC]]（特殊持股公司）重新分拆出来。 |

## 5. Institutional significance

JEXIM 的存在及其 1999 的合并解释了当代日本政策金融三个持续的特征，这些都体现在 [[policy-finance/japan-policy-finance-system|Japan policy finance system]] 中：

1. **整合的海外金融载体**：与美国 EXIM Bank 和 OPIC / DFC 之间的分离不同，日本将出口信贷与海外投资金融整合进单一机构，简化了与超大型银行（[[megabanks/mufg|MUFG]]、[[megabanks/smfg|SMFG]]、[[megabanks/mizuho-fg|Mizuho FG]]）在 [[policy-finance/japan-project-finance-stack-diagram|project-finance stack]] 上的协同融资。
2. **较晚的 ODA 分离**：ODA 贷款管理直到机构历史开始 2008, 年之后才完全移交至 [[policy-finance/jica|JICA]]。这就是为何尽管当代 JBIC 并不与 ODA 相邻，旧教科书仍将 JEXIM 描述为 ODA 相邻。
3. **WTO 之前的出口信贷规范**：JEXIM 在现代 [[policy-finance/oecd-export-credit-arrangement|OECD Arrangement]] 部门的理解收紧 ECA 纪律之前便已运营；JBIC 继承的遗留账簿包含在更宽松规则下达成的交易。

## 6. Boundary cases

- **并非当代 JBIC**：本页为历史性内容。当前海外政策金融路由至 [[financial-regulators/jbic|JBIC]]。
- **并非 JICA**：ODA / 发展合作历史主要属于 OECF，并从 2008 至 [[policy-finance/jica|JICA]]，而非直接属于 JEXIM。
- **并非 US EXIM**：该机构与 US EXIM 共享英文「Eximbank」标签，但拥有包括海外投资贷款及无挂钩贷款在内的更广泛职能。
- **并非 [[financial-regulators/dbj|DBJ]]**：DBJ 的前身（日本开发银行 / 日本開発銀行）是 **国内** 开发银行的对应方；JEXIM 是其 **海外** 对应方。两者是同级的特殊目的国家银行。

## 7. Open questions

- JEXIM 在 1990年代后期的海外投资账簿中，有多少延续至初代 JBIC 的资产负债表，进而到当代 JBIC？
