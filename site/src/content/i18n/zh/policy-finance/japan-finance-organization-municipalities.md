# 日本市町村金融机构 (JFM)

## 长话短说

日本市政金融组织是一家联合资助的公共信贷机构，向所有 47  都道府县和市政府提供长期低成本贷款，以满足供水、污水处理、交通、医院、学校、灾难恢复和一般地方融资需求。它不是**国家政府实体：资本由地方政府共同出资，总务省和财政部负责监督。

本页通过 [[policy-finance/INDEX|policy-finance index]] 镜头来看待 JFM：将 JFM 作为公共信用地图中的一条车道，与 [[financial-regulators/jfc|JFC]]（中小企业/农业/渔业）、[[financial-regulators/jbic|JBIC]]（海外）、[[financial-regulators/dbj|DBJ]]（开发银行/GX）和 [[policy-finance/japan-housing-finance-agency|JHF]]（住房）并列。匹配的 JapanFG 域公司历史页面是 [[financial-regulators/local-govt-finance]]。

## 2。功能图

| 功能 | 为什么这很重要 |
|---|---|
| 地方政府债券承销和直接贷款 | 与财政贷款基金和民间金融机构并列的地方政府三大长期融资支柱之一。 |
| 向地方公营企业提供长期低成本贷款 | 供水和污水处理、公立医院、城市交通、天然气和类似的非商业公共服务难以吸引纯粹的私人信贷。 |
| 灾后恢复贷款 | 在地震、台风和洪水发生后提供紧急长期贷款（东日本2011, 熊本2016, 西日本2018, 能登2024）。 |
| JFM债券发行（无政府担保） | 在国内和国际市场上发行高级无抵押债券，没有明确的政府担保——信用状况取决于地方政府的联合支持和法定地位。 |
| 向地方政府提供财政咨询/培训 | 为资源贫乏的城市提供债务管理、信贷分析和资本规划支持。 |

## 3。系统位置

JFM 是解决市场缺口的结构性答案：规模较小的农村地方政府无法以有效成本利用公开发行的地方债券市场，而私人银行本身也无法延长供水、污水处理和交通基础设施所需的 20 至 30 年期限。通过将所有地方政府集中到一个共同出资的信贷工具中，JFM 实现了任何单个地区都缺乏的发行规模、AAA 级市场准入和风险分散化。

在 [[policy-finance/japan-policy-finance-system|Japan policy finance system]] 映射的更广泛的日本公共信贷堆栈中，它是 [[financial-regulators/jfc|JFC]]、[[financial-regulators/jbic|JBIC]] 和 [[financial-regulators/dbj|DBJ]] 的同等机构，但其借款人基础是政府，而不是企业。 [[policy-finance/japan-project-finance-stack-diagram|Japan project finance stack diagram]] 不包括 JFM，因为 JFM 不共同资助海外资源项目——其通道是国内公共基础设施。

在[[money-market/INDEX|money-market]]图中，JFM债券和联合地方政府债券提供了最大的非JGB高等级日元计价公共信贷类别之一，被银行、人寿保险公司、地区银行和公共养老基金用作抵押品和持有资产。

## 4。历史轨迹

| 年 | 事件 |
|---|---|
| 1957  | 前身（100%国家政府股权）为水利、交通、电力市政企业提供资金支持。 |
| 2001–2007  | 小泉时代的特殊法人改革决定废除国有股权模式。 |
| 2008-08-01 | 地方公営企业等金融机构 established under joint local-government ownership; predecessor wound up 2008-09-30. |
| 2009-06-01 | 更名为日本市町村金融机构；授权范围从市政企业扩大到一般地方政府贷款。 |
| 2011  | 日本东部地震； JFM 成为受影响城市最大的灾难恢复贷款机构之一。 |
| 2024-01-01 | 能登半岛地震； JFM 为受影响的石川县各市启动了灾难恢复贷款计划。 |

## 5。为什么这很重要

- 在任何以 JFC / JBIC / DBJ 为终点的日本公共财政地图中，它都是**缺失的地方政府信贷锚**。
- 其联合所有权模式在国际上并不常见——它更像斯堪的纳维亚市政融资机构（Kommuninvest、KommuneKredit），而不是美国的市政债券发行。
- 其灾难恢复作用使其成为区域危机期间的结构性反周期贷款人。
- JFM 债券是主要的非 JGB 公共信贷资产类别，对地区银行和人寿保险资产负债表构成产生影响。

## 6。边界情况

- **不是国家政府银行**：与 JFC 和 DBJ 不同，JFM 没有国家政府股权。国家控制是通过法定监督，而不是所有权。
- **不是区域银行**：区域银行也向地方政府提供贷款，但根据《银行法》，它们是私营部门存款银行。 JFM 是一家法定联合地方政府公司。
- **不是财政贷款基金（FILP）**：财政融资资金是财政部管理的国家政府直接贷款池。 JFM 是一个由资本市场发行融资的独立并行渠道。
- **不是 [[policy-finance/japan-housing-finance-agency|JHF]]**：JHF 通过证券化和保险支持私人住房贷款市场； JFM 直接向政府借款人提供贷款。

## 7。开放式问题

- JFM 贷款份额如何与县、市、镇和村借款人之间的私营区域银行和 FILP 分配？
- 人口下降将如何改变 JFM 对人口萎缩城市的长期信用状况？
- 气候转型市政投资（公共交通脱碳、区域供暖、水资源弹性）是否开辟了新的贷款通道。
- 在没有明确的国家政府担保的情况下，JFM 债券利差在压力条件下的表现如何。
