# 全国信用保证协会连合会

## Wiki route

本条目归属于 [[policy-finance/INDEX|policy-finance index]]，作为日本信用保证制度的**联合会／协调机构**页面。请将其与 [[policy-finance/japan-credit-guarantee-system|Japan credit guarantee system]]（机制页面）和 [[policy-finance/japan-policy-finance-system|Japan policy finance system]]（更广泛的边界）配对。将其角色与覆盖 [[policy-finance/agriculture-credit-guarantee-system|agriculture]] 和 [[policy-finance/fisheries-credit-guarantee-system|fisheries]] 保证的行业专属联合会进行比较。本地成员 CGC 包括 [[policy-finance/hokkaido-credit-guarantee-corp|Hokkaido CGC]]、[[policy-finance/tokyo-credit-guarantee-corp|Tokyo CGC]]、[[policy-finance/osaka-credit-guarantee-corp|Osaka CGC]]、[[policy-finance/aichi-credit-guarantee-corp|Aichi CGC]] 和 [[policy-finance/hyogo-credit-guarantee-corp|Hyogo CGC]]。

## TL;DR

全国信用保证协会连合会（一般社团法人全国信用保证协会连合会，**JFG**／**全信保连**）是日本 51 家本地信用保证协会（47 家都道府县 + 4 家市级 CGC — 横滨／川崎／名古屋／岐阜）的全国联合会。它**本身并非保证制度** — 它是位于 51 家本地 CGC 之上的协调、研究、培训、系统管理及财务支持机构，并与 [[financial-regulators/jfc|JFC]] 信用保险及 METI／中小企业厅政策对接。其在运营上最重要的职能是在全国范围内管理 **责任共有制度** 的财务流动。在 2020–2022 年 ZeroZero 贷款推行期间，JFG 是联合会层级的协调者，在 51 家 CGC 之间设定统一的运营标准，现在则随着代位清偿浪潮贯穿 2023–2026年，协调后疫情退出框架。

## 2. 51 会员 CGC 名单

51 家成员 CGC 覆盖全部 47 个都道府县以及 4 个特别指定的城市：

- **47都道府県**: 北海道, 青森, 岩手, 宮城, 秋田, 山形, 福島, 茨城, 栃木, 群馬, 埼玉, 千葉, 東京, 神奈川 (excluding Yokohama/Kawasaki city CGCs which are separate members), 新潟, 富山, 石川, 福井, 山梨, 長野, 岐阜 (excluding Gifu city CGC), 静岡, 愛知 (excluding Nagoya city CGC), 三重, 滋賀, 京都, 大阪, 兵庫, 奈良, 和歌山, 鳥取, 島根, 岡山, 広島, 山口, 徳島, 香川, 愛媛, 高知, 福岡, 佐賀, 長崎, 熊本, 大分, 宮崎, 鹿児島, 沖縄.
- **4 市**: 横浜市信用保証協会, 川崎市信用保証協会, 名古屋市信用保証協会, 岐阜市信用保証協会.

4 家市级 CGC 之所以存在，是因为在该制度的历史性创立之际，这四座城市拥有规模足以独立于都道府县层级 CGC 而支撑自身信用保证协会的独特产业基础。与都道府县 CGC（神奈川 CGC、爱知 CGC、岐阜 CGC）共存在运营上属正常 — 市级 CGC 与都道府县 CGC 存在地理／行政边界上的区分。

## 3. JFG 角色图

| Role | Detail |
|---|---|
| 成员支持 | 向 51 家本地 CGC 提供财务／运营／IT／培训支持；标准操作程序；风险管理协调。 |
| 責任共有制度 管理 | 在全国范围内管理 CGC 与放贷方之间 80% / 20% 责任共有的财务流动；中央对账。 |
| 研究／出版物／统计 | 汇总的 CGC 统计（保证余额、代位弁済、求償、求償権回収率、行业细分）；行业研究；中小企业金融出版物。 |
| 联合会汇集资金 | 管理支撑该制度的联合会层级汇集财务资源（资本注入、紧急支持、损失汇集）。 |
| 放贷方协调 | 就系统范围的产品／流程变更与全国银行协会、信金中央金库、NCB（信组）及其他放贷方机构协调。 |
| JFC 接口 | 与 [[financial-regulators/jfc\|JFC]] 进行信用保险合同管理及运营协调的主要联合会层级接口。 |
| METI／中小企业厅 接口 | 政策实施接口；在全国中小企业金融政策中作为 51 家 CGC 的中央代言。 |
| 危机模式协调 | 危机时期项目的联合会层级协调者（1998 安定化特別保証、2008 緊急保証、2011 災害関係、2020 ZeroZero）。 |
| 标准化产品设计 | 设计用于全国部署的通用模板项目（创设、事业承继、ABL 等）。 |
| 信息系统／連合会システム | 运营支撑 CGC 运营的联合会层级 IT 系统。 |

## 4. ZeroZero 全国推行 2020 → 退出框架 2025

JFG 是 **2020–2022 ZeroZero 贷款** 推行（实质无息・无担保融资）的运营层面联合会层级协调者 — 系统层级的机制参见 [[policy-finance/japan-credit-guarantee-system]]。JFG 的角色：

| Phase | JFG action |
|---|---|
| 项目设计（2020-04）| 联合会层级标准化 ZeroZero 申请表、放贷方-CGC 运营接口、资格检查、文件标准。 |
| 快速推行（2020-05 → 2020-12）| 联合会层级监控 51家 CGC 的处理能力；系统范围瓶颈的协调；指标标准化。 |
| 持续运营（2021）| 联合会层级统计汇总；附属项目的协调（延期、地区分层项目）。 |
| 发放结束（2021-03 → 2022）| 联合会层级协调项目结束机制；清退的过渡规划。 |
| 处置／清退框架（2022–2025）| 联合会层级关于逐借款人处置（借換／経営改善／事業承継／廃業 菜单）的指引；"粘り強い支援"（耐心支持）原则。 |
| 代位清偿浪潮协调（2023–2026）| 联合会层级汇总代位弁済规模的统计；就信用保险理赔处理与 JFC 协调；就残余政策支持与 METI 协调。 |
| 破产数据接口 | 联合会层级就 ZeroZero 相关破产分类与破产／重组数据提供商协调。 |

后疫情退出框架（自 ~2022年起）强调**逐借款人的耐心处置**而非以执行驱动的催收，反映了 (a) 通过清退期保全可存续中小企业的政策方向，以及 (b) 认识到大规模违约情景将压垮 CGC 和 JFC 的承载能力。该框架包括 JFG 在 51 家 CGC 之间协调的特定 借換保証／経営改善／事業承継／廃業 产品菜单。

## 5. 按保证余额对成员 CGC 排名

在 51 家成员 CGC 中，按保证余额（保証残高）计最大的通常包括：

| Rank tier | CGCs (illustrative) | Why |
|---|---|---|
| Top 5 （城市-工业重量级）| [[policy-finance/tokyo-credit-guarantee-corp\|Tokyo CGC]], [[policy-finance/osaka-credit-guarantee-corp\|Osaka CGC]], [[policy-finance/aichi-credit-guarantee-corp\|Aichi CGC]] (+ Nagoya City CGC overlap), Kanagawa CGC (+ Yokohama / Kawasaki City CGCs), Saitama CGC | 最大的城市中小企业群体；高产业集中度；ZeroZero 时期大量发放。 |
| Mid tier（大型地区／地区-工业带）| Hyogo, Chiba, Fukuoka, Shizuoka, [[policy-finance/hokkaido-credit-guarantee-corp\|Hokkaido]] | 具有中等密度工业／商业／第一产业组合的大型中小企业基础。 |
| Smaller-prefecture tier | Tōhoku rural (Akita, Iwate, etc.), Shikoku, Sea-of-Japan-side prefectures | 较小的中小企业群体；绝对保证余额较低但人均政策权重高。 |

具体排名因年份和项目组合而异；前 5 名在身份上保持稳定（东京／大阪／爱知／神奈川／埼玉），兵库／千叶／福冈在相邻名次中轮换。ZeroZero 时期的发放显著影响了 2020–2022, 年期间的相对余额排序，而贯穿 2023–2026 年的代位清偿浪潮现已在滚动余额轨迹中显现。

## 6. 与 JFC 信用保险的互动

**JFC 信用保険業務**（信用保险业务）是吸收 CGC 残余代位清偿损失的再保险层。JFG 与 JFC 的接口：

- 联合会层级协调适用于 51 家 CGC 的信用保险合同条款。
- 协调保险理赔处理标准及时间安排。
- 联合会层级统计汇总，输入到 JFC 的精算／准备金评估。
- 在信用保险经济性需要联合会层级对齐的情况下，就危机模式产品设计进行协调（例如，危机相关保证 危機関連保証 100% 保证设计，配以配套的信用保险条款）。
- 就 CGC + JFC 组合系统经济性与 METI 进行联合政策接口。

JFC 从前身 中小企業信用保険公庫（1958）继承了信用保险业务，该机构作为政策金融重组的一部分于 2008 年并入 JFC。信用保险业务现于 JFC 内部运营，与 JFC 的直接中小企业／小规模事业／农林水产／国际放贷并行 — 参见 [[financial-regulators/jfc]]。

## 7. 年份演变

| Year | Event |
|---|---|
| 1937 | 東京信用保証協会 — 首个信用保证协会。 |
| 1953 | 信用保証協会法 制定；全国 CGC 框架。 |
| 1951 → | 全国联合会前身组织。 |
| 1958 | 中小企業信用保険公庫 设立（JFC 信用保险的前身）。 |
| 1998–1999 | 安定化特別保証 — 亚洲危机紧急 100% 保证项目。JFG 协调全国推行。 |
| 2007-10 | 責任共有制度 引入 — JFG 管理财务流动。 |
| 2008–2009 | 緊急保証 — GFC 紧急项目。 |
| 2008-10 | JFC 设立；信用保险并入 JFC；JFG-JFC 接口重组。 |
| 2011 | 东北地震；災害関係保証 广泛启动；JFG 协调联合会层级应对。 |
| 2020-05 | ZeroZero 项目启动；JFG 协调全国快速推行。 |
| 2021-03 | ZeroZero 发放结束。 |
| 2022–2025 | 后疫情退出框架 — 粘り強い支援 原则。 |
| 2023–2026 | 代位清偿浪潮；JFG 协调联合会层级处置／重组。 |
| 2024–2025 | 持续聚焦于符合 経営者保証ガイドライン 的保证、事业承继保证、初创保证。 |

## 8. 边界情形

- **并非 JFC** — JFC 运营信用保险及直接放贷；JFG 是联合会。两者对接但为独立实体。
- **并非本地 CGC** — 本地 CGC 执行面向借款人的保证；JFG 支持／协调它们。
- **并非农业／渔业** — 农业／渔业信用保证制度有其自身的行业联合会及 JAFFIC 再保险，与 JFG／JFC 分离。
- **并非 METI／中小企业厅** — METI 是政策监管者；JFG 是代表 51 家 CGC 管理该制度的联合会。
- **并非放贷方** — JFG 不直接为借款人提供保证；本地 CGC 才是。
