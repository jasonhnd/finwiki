# 日本信用担保公司联合会

## Wiki 路径

此条目位于 [[policy-finance/INDEX|policy-finance index]] 下，作为日本信用担保系统的 **联邦/协调机构** 页面。将其与 [[policy-finance/japan-credit-guarantee-system|Japan credit guarantee system]]（机制页）和 [[policy-finance/japan-policy-finance-system|Japan policy finance system]]（更宽的周界）配对。将其作用与涵盖 [[policy-finance/agriculture-credit-guarantee-system|agriculture]] 和 [[policy-finance/fisheries-credit-guarantee-system|fisheries]] 担保的特定部门联盟进行比较。本地成员 CGC 包括 [[policy-finance/hokkaido-credit-guarantee-corp|Hokkaido CGC]]、[[policy-finance/tokyo-credit-guarantee-corp|Tokyo CGC]]、[[policy-finance/osaka-credit-guarantee-corp|Osaka CGC]]、[[policy-finance/aichi-credit-guarantee-corp|Aichi CGC]] 和 [[policy-finance/hyogo-credit-guarantee-corp|Hyogo CGC]]。

## 要点摘要

日本信用担保公司联合会（一般社団法人全国信用保证协会连合会，**JFG** / **全信保连**）是日本 51  地方信用担保公司（47 prefectural + 4 city CGCs — 横滨/川崎/名古屋/岐阜）的全国联合会。它本身并不是担保系统——它是位于 51  本地 CGC 之上的协调、研究、培训、系统管理和财务支持机构，并与 [[financial-regulators/jfc|JFC]] 信用保险和 METI / SME 机构政策接口。其最重要的运作职能是管理全国范围内的**责任共享制度**（责任共享制度）资金流动。在 2020–2022 ZeroZero 贷款推出期间，JFG 是联邦级协调员，在 51 CGC 中制定统一的操作标准，现在随着代位权浪潮贯穿 2023–2026，正在协调新冠疫情后的退出框架。

## 2。 51  CGC 成员名单

51 会员CGC覆盖所有47 都道府县以及4 特别指定城市：

- **47 都道府县**：北海道、青森、岩手、宫城、秋田、山形、福岛、茨城、栃木、群马、埼玉、千叶、东京、神奈川（不包括作为独立成员的横滨/川崎市 CGC）、新泻、富山、石川、福井、山梨、长野、岐阜（不包括岐阜市） CGC）、静冈县、爱知县（名古屋市 CGC 除外）、三重县、滋贺县、京都、大阪、兵库县、奈良县、和歌山县、鸟取县、岛根县、冈山县、广岛县、山口县、德岛县、香川县、爱媛县、高知县、福冈县、佐贺县、长崎县、熊本县、大分县、宫崎县、鹿儿岛县、冲绳县。
- **4  市**：横滨市信用保证协会、川崎市信用保证协会、名古屋市信用保证协会、岐阜市信用保证协会。

4 城市信用担保公司的存在是因为在该体系的历史基础上，这四个城市拥有足够大的独特工业基础，足以支持独立于地级信用担保公司的信用担保公司。与县级 CGC（神奈川县 CGC、爱知县 CGC、岐阜县 CGC）共存在运营上是正常的——市级 CGC 和县级 CGC 有地理/行政边界区别。

## 4。 ZeroZero 全国推广 2020 → 退出框架 2025

JFG 是 **2020–2022 ZeroZeroZero 贷款** 推广（実质无利子・无资助融资）的联邦级运营协调员 — 请参阅 [[policy-finance/japan-credit-guarantee-system]] 了解系统级机制。 JFG的作用：

|相| JFG行动|
|---|---|
|程序设计（2020-04）| ZeroZero 申请表、贷方-CGC 操作界面、资格检查、文件标准的联邦级标准化。 |
|快速推出（2020-05 → 2020-12）| 51-CGC处理能力联邦级监控；协调全系统瓶颈；公制标准化。 |
|持续运行（2021）|联邦级统计汇总；协调附属计划（扩展、区域分层计划）。 |
|始发关闭（2021-03 → 2022）|程序封闭机制的联盟级协调；放松的过渡计划。 |
|锻炼/放松框架 (2022–2025) |联邦层面对逐个借款人锻炼的指导（借换/経営改善/事业承継/廃业菜单）； “粘り强い支持”（患者支持）主义。 |
|代位波协调（2023–2026）|联邦级代位弁済交易量统计汇总；与 JFC 协调信用保险索赔处理；与经济产业省就剩余政策支持进行协调。 |
|破产数据接口|与破产/重组数据提供商就 ZeroZero 相关破产分类进行联邦级协调。 |

新冠疫情后的退出框架（自~2022起）强调**耐心的逐个借款人的解决**，而不是强制执行驱动的催收，反映了（a）在放松期保护有生存能力的中小企业的政策方向，以及（b）认识到大规模违约情况将压垮CGC和JFC的能力。该框架包括 JFG 在 51 CGC 上协调的特定借换保证/业务改善/事业承业/业务产品菜单。

## 5。会员CGC排名靠担保杰出

在51 成员CGC中，最大的保证残高通常包括：

|排名等级 | CGC（说明性）|为什么 |
|---|---|---|
|顶级5 (都市工业重量级人物) | [[policy-finance/tokyo-credit-guarantee-corp\|Tokyo CGC]]、[[policy-finance/osaka-credit-guarantee-corp\|Osaka CGC]]、[[policy-finance/aichi-credit-guarantee-corp\|Aichi CGC]]（+名古屋市 CGC 重叠）、神奈川 CGC（+横滨/川崎市 CGC）、埼玉 CGC |最大的城市中小企业群体；产业集中度高； ZeroZero时代重磅起源。 |
|中层（大型区域/区域工业带）|兵库县、千叶县、福冈县、静冈县、[[policy-finance/hokkaido-credit-guarantee-corp\|Hokkaido]] |具有中等密度工业/商业/第一产业组合的大型中小企业基地。 |
|较小的地级市 |东北乡村（秋田县、岩手县等）、四国县、日本海沿岸县 |中小企业数量较少；绝对保障余额较低，但人均政策权重较高。 |

具体排名因年份和项目组合而异；排名第一的5 身份稳定（东京/大阪/爱知/神奈川/埼玉），兵库/千叶/福冈在相邻排名中轮换。 ZeroZero时代的起源显着影响了2020–2022, 期间的相对平衡排序，通过2023–2026 的代位波现在在滚动平衡轨迹中可见。

## 6。与JFC信用保険互动

**JFC信用保険业务**（信用保险业务）是吸收剩余CGC代位损失的再保险层。 JFG与JFC的接口：

- 适用于 51 CGC 的信用保险合同条款的联邦级协调。
- 协调保险索赔处理标准和时间安排。
- 联邦级统计汇总纳入 JFC 的精算/准备金评估。
- 信用保险经济学需要联邦级协调的危机模式产品设计协调（例如，危机相关担保“危机关连保证”100% 担保设计与配对信用保险条款）。
- 与 METI 就 CGC + JFC 系统经济组合进行联合政策接口。

JFC继承了前身中小企业信用保険公库（1958）的信用保险业务，并于2008 作为政策金融重组的一部分并入JFC。信用保险业务现在在 JFC 内部与 JFC 的直接中小企业/小型企业/农林渔业/国际贷款一起运营 - 参见 [[financial-regulators/jfc]]。

## 7。年份演变

|年份|活动 |
|---|---|
| 1937 |东京信用担保协会——第一家信用担保公司。 |
| 1953 |颁布信用担保协会法；全国 CGC 框架。 |
| 1951 → |全国联合会的前身组织。 |
| 1958 |中小企业信用保险公司成立（JFC信保前身）。 |
| 1998–1999 |特别稳定保障——亚洲危机应急100%保障计划。 JFG 协调全国推广。 |
| 2007-10 |引入共同责任体系——JFG 管理资金流。 |
| 2008–2009 |应急保障——GFC应急方案。 |
| 2008-10 |联合部队司令部成立；信用保险纳入 JFC； JFG-JFC 接口重新组织。 |
| 2011 |东北地震；广泛启动与灾害相关的保障； JFG 协调联邦级别的响应。 |
| 2020-05 | ZeroZero计划启动； JFG 协调在全国范围内的快速推广。 |
| 2021-03 | ZeroZero 起源已关闭。 |
| 2022–2025 |新冠疫情后的退出框架——顽强的支持主义。 |
| 2023–2026 |代位权浪潮； JFG 协调联邦级别的锻炼/重组。 |
| 2024–2025 |继续关注符合企业担保准则的担保、企业继承担保、启动担保。 |

## 8。边界情况

- **不是 JFC** — JFC 经营信用保险和直接贷款； JFG是联合会。它们相互连接，但又是独立的实体。
- **不是本地 CGC** — 本地 CGC 向借款人提供担保； JFG 支持/协调他们。
- **不是农业/渔业** — 农业/渔业信用担保系统有自己的部门联合会和 JAFFIC 再保险，与 JFG / JFC 分开。
- **不是 METI / SME Agency** — METI 是政策监管者； JFG 是代表 51 CGC 管理系统的联盟。
- **不是贷方** — JFG 不直接提供担保；当地 CGC 这样做。
