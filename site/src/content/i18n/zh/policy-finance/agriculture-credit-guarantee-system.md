# 日本农业信用保证制度

## Wiki 路径

本条目位于[[policy-finance/INDEX|policy-finance index]]下。请与[[policy-finance/japan-credit-guarantee-system|Japan credit guarantee system]]进行 SME 侧对照，与[[policy-finance/fisheries-credit-guarantee-system|fisheries credit guarantee system]]进行姊妹行业制度对照，并与[[policy-finance/japan-policy-finance-system|Japan policy finance system]]一起理解更广的公共信用边界。农户的存款与贷款对应方是[[banking/ja-bank-system-japan|JA Bank system]]，合作体系顶点是[[cooperative-banks/norinchukin|Norinchukin]]。

## TL;DR

日本农业信用保证制度（農業信用保証保険制度）通过两层结构补充农户和农业相关借款人的信用：都道府县层面的**农业信用基金协会**（47 个地方机构）向贷款机构保证农户 / 农业法人债务，再由国家层面的**独立行政法人农林渔业信用基金**（JAFFIC）农业信用保险业务进行再保险。它与[[banking/ja-bank-system-japan|JA Bank system]]相邻，但这是保证-保险路径，不是存款 / 贷款路径。与[[policy-finance/japan-credit-guarantee-system|the general SME credit guarantee system]]相比，农业制度是行业特定、由 MAFF 监督，并与农业合作体系（系統金融）整合，而不是与一般区域银行体系整合。

## 1. 与系統金融的定位关系

日本农业金融流经**系統金融**合作体系金融栈：

| 层级 | 功能 | FinWiki 路径 |
|---|---|---|
| 农户 / 农业法人 | 借款人（农户、农业法人、农业从业者、农业业务）。 | 本系统页 |
| 单位农协 (JA / 单协) | 基层合作社；吸收农户存款并提供农业与家庭信贷。 | [[banking/ja-bank-system-japan]] |
| 信连 | 都道府县农业协同组合信用联合会；中介 JA 存款 / 贷款。 | [[banking/ja-bank-system-japan]] / prefectural-信连层 |
| 农林中央金库 (Norinchukin) | JA / 信连 / 渔业 / 林业合作社的全国顶点银行；作为全球投资者管理合作体系剩余存款。 | [[cooperative-banks/norinchukin]] |
| 农业信用基金协会 | 农业贷款的都道府县保证机构；违约后代位清偿。 | 本页 |
| JAFFIC | 对都道府县农业和渔业保证协会提供全国信用保险 / 再保险，也提供直接融资支持。 | 本页 |
| MAFF | 合作体系与信用保证保险制度的政策监督方。 | [[policy-finance/INDEX]] |

因此，保证制度是**叠加在合作银行栈上的平行公共信用基础设施**：JA / 信连 / Norinchukin 处理存款和贷款，农业信用基金协会 + JAFFIC 处理支持该贷款的保证和再保险层。

## 2. 机构概览 - 农业信用基金协会

| 项目 | 内容 |
|---|---|
| 设置 | 47都道府县（部分都府县以区域单位运营） |
| 法令 | 农业信用保证保险法（1961） |
| 法律形态 | 特别法上的法人，而非一般民间法人 |
| 监督 | MAFF 和都道府县 |
| 主要业务 | 对农户 / 农业法人 / 相关业务者贷款提供保证 |
| 资金来源 | 国家 / 都道府县 / 合作体系金融机构出资和补助 |
| 联合会 | 全国农业信用基金协会联合会 |
| 再保险 | 由 JAFFIC 农业信用保险业务承担 |

## 4. 主要保证产品

- **农业近代化资金保证**：农业设备投资和设施现代化的保证。
- **农业改良资金保证**：创新、新作物、转有机等改善贷款。
- **农业经营基盘强化资金（スーパーL资金）保证**：经 JFC 农林水产业务流入的大规模农业经营强化贷款。
- **就农支持资金保证**：新进入农户支持。
- **担い手 / 农业法人向け保证**：大规模农户 / 农业法人保证。
- **灾害复旧保证**：台风、水害、雪害、畜产疾病等灾害恢复。
- **6次产业化 / 加工流通相关保证**：农户主导食品加工和直销，即“6次产业化”政策通道。
- **借换 / 经营改善资金保证**：再融资和经营改善。

## 5. 运营逻辑：保证、代位、再保险

1. **申请** - 农户或农业法人从贷款机构借款，通常是 JA / 信连 / Norinchukin / 区域银行 / JFC 农业部门。贷款方需要保证信用并向都道府县农业信用基金协会申请。
2. **保证** - 协会审查贷款、接受保证并收取保证费。
3. **贷款发放** - 贷款机构在保证成立后发放贷款。
4. **违约** - 借款人违约时，贷款方按保证索赔；协会向贷款方代位清偿，再向借款人 / 担保物追偿。
5. **再保险** - JAFFIC 农业信用保险业务为协会保证敞口再保险，按保险合同向协会支付保险金。
6. **剩余损失** - 回收和 JAFFIC 保险后仍有的损失，由协会准备金吸收，并由国家 / 都道府县 / 合作体系出资补充。

制度的三层损失吸收为：借款人回收、都道府县协会准备金 + JAFFIC 再保险、公共 / 合作资本补充。

## 6. 保证费经济性

保证费表由各都道府县农业信用基金协会在 MAFF 指引内设定，通常包括：

- 对未偿本金按年率计费，优先政策贷款（如スーパーL资金）为低个位数基点区间，高风险产品更高。
- 按产品（现代化、改善、新进入等）和借款人类别分层。
- 对新进入农户、灾害恢复、6次产业化等优先政策类别提供补贴。
- 费率低于商业保证的市场等价定价，差额由公共 / 合作补贴吸收。

## 7. 年度演进

| 年份 | 事件 |
|---|---|
| 1961 | 农业信用保证保险法制定；都道府县农业信用基金协会建立。 |
| 1972 | 农业信用保险公库设立，后并入 JAFFIC。 |
| 1986 | 林业信用基金设立。 |
| 2003-10-01 | JAFFIC 通过整合农业、林业和渔业信用保险机构而成立。 |
| 2007 | 面向核心经营主体的优先保证项目在农业政策改革下扩大。 |
| 2010s | 6次产业化保证加入，农户主导加工 / 分销获得明确支持。 |
| 2014-2016 | TPP / FTA 对农业形成压力，保证制度支持农场现代化适应。 |
| 2018 | 农业政策重组时期。 |
| 2020-2022 | COVID 期间对农业供应链扰动启用灾害保证。 |
| 2022-2023 | 能源 / 肥料 / 饲料价格冲击，制度支持畜产和设施园艺流动性。 |
| 2024 | 智慧农业贷款产品获得保证支持。 |
| 2024-2025 | 农业改革和粮食安全投资支持定位强化。 |
| 2025 | 核心经营主体承继 + 法人化优先，制度与结构改革对齐。 |

## 8. 与一般 SME 信用保证制度比较

| 维度 | 农业信用保证制度 | [[policy-finance/japan-credit-guarantee-system\|General SME credit guarantee system]] |
|---|---|---|
| 借款人 | 农户 / 农业法人 / 农业相关业务者 | SME / 小企业（广泛行业） |
| 地方保证方 | 农业信用基金协会（47 都道府县） | 信用保证协会（51 都道府县 / 市） |
| 全国联合会 | 全国农业信用基金协会联合会 | [[policy-finance/national-federation-credit-guarantee-corporations\|全国信用保証協会連合会 (JFG)]] |
| 全国再保险方 | JAFFIC | [[financial-regulators/jfc\|JFC]]信用保险 |
| 监督 | MAFF | METI / 中小企业厅 |
| 贷款伙伴 | JA / 信连 / Norinchukin / 银行 / JFC 农业部门 | 区域银行 / 信金 / 信组 / JFC |
| 危机旗舰 | 灾害、畜产疾病、食品安全保证 | 安全网 / 危机相关（ZeroZero 2020-2022） |
| 资金经济 | 公共 / 合作体系补贴 | 公共补助 + 信用保险再保险 |

两套制度是平行公共信用保证基础设施，目标行业、监督方和合作银行接口不同。农户贷款走农业制度，小制造商贷款走 SME 制度。

## 9. 边界案例

- **不是 JA Bank**：JA Bank 是贷款路径；本页是其上的保证保险机制。
- **不是 SME 保证制度**：SME 保证路由至[[policy-finance/japan-credit-guarantee-system]]。
- **不是渔业**：渔业保证保险属于[[policy-finance/fisheries-credit-guarantee-system]]，是同一 JAFFIC 母体下的姊妹行业。
- **不是林业**：林业信用支持也是 JAFFIC 业务，但在 MAFF 林业政策下另行路由。
- **不是 JFC 农业贷款**：JFC 的[[financial-regulators/jfc|agriculture-forestry-fisheries arm]]直接向农户贷款；本页关注农户借款的保证保险，二者在 JFC 发放被保证贷款时可能重叠。
