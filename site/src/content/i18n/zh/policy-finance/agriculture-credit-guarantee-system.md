# Agriculture credit guarantee system in Japan

## Wiki route

This entry sits under [[policy-finance/INDEX|policy-finance index]]. Read it against [[policy-finance/japan-credit-guarantee-system|Japan credit guarantee system]] for SME-side contrast, [[policy-finance/fisheries-credit-guarantee-system|fisheries credit guarantee system]] for the sister sector system, and [[policy-finance/japan-policy-finance-system|Japan policy finance system]] for the broader public-credit perimeter. The deposit-and-lending counterpart for farmers is [[banking/ja-bank-system-japan|JA Bank system]] and the cooperative apex is [[cooperative-banks/norinchukin|Norinchukin]].

## TL;DR

日本的农业信用保证保险制度（農業信用保証保険制度）通过两层式分级结构补充农业者及农业相关借款人的信用：(1) 都道府县层级的 **农业信用基金协会**（Agriculture Credit Fund Associations，47 个都道府县主体），为农业者／农业法人对放贷机构的债务提供保证，并由 (2) 全国层级的 **独立行政法人农林渔业信用基金**（Agriculture, Forestry and Fisheries Credit Foundations，JAFFIC）的农业信用保险业务进行再保险。它与 [[banking/ja-bank-system-japan|JA Bank system]] 相邻，但属于保证保险路径，而非存款／放贷路径。与 [[policy-finance/japan-credit-guarantee-system|the general SME credit guarantee system]] 相比，农业制度具有行业专属性、由农林水产省（MAFF）监管，并与农业合作社系统（系統金融）相整合，而非与一般地方银行相整合。

## 1. 系統金融との位置関係

日本的农业金融通过 **系統金融**（cooperative system finance）栈流转：

| Layer | Function | FinWiki route |
|---|---|---|
| 農家 / 農業法人 | 借款人（农业者、农业法人、农业从业者、农业企业）。 | System route here |
| 単位農協 (JA / 単協) | 基层合作社；吸收农业者存款；提供农业及家庭信用。 | [[banking/ja-bank-system-japan]] |
| 信連 (JA信連 / 都道府県信用農業協同組合連合会) | 农业合作社的都道府县信用联合会；居间 JA 的存款／放贷。 | [[banking/ja-bank-system-japan]] / prefectural-信連 layer |
| 農林中央金庫 (Norinchukin) | JA／信连／渔业、林业合作社的全国顶级银行；作为全球投资者管理过剩的合作社存款。 | [[cooperative-banks/norinchukin]] |
| 農業信用基金協会 | 农业贷款的都道府县保证主体；违约时进行代位清偿。 | This page |
| 独立行政法人農林漁業信用基金 (JAFFIC) | 面向都道府县农业及渔业保证协会的全国层级信用保险／再保险主体；亦提供直接融资支持。 | This page |
| MAFF (農林水産省) | 合作社系统 + 信用保证保险制度的政策监管者。 | [[policy-finance/INDEX]] |

因此，保证制度是 **叠加在合作社银行栈之上的并行公共信用基础设施** ── JA／信连／农林中金处理存款与放贷，而农业信用基金协会 + JAFFIC 处理为该放贷提供支撑的保证与再保险层。

## 4. 主要保证产品

- **农业现代化资金保证（農業近代化資金保証）** — 面向农机投资、设施现代化（温室、挤奶厅、仓储）的现代化贷款保证。
- **农业改良资金保证（農業改良資金保証）** — 面向创新、新作物引进、有机转型的改良贷款保证。
- **农业经营基础强化资金（超级 L 资金 / スーパーL資金）保证** — 面向经由 JFC 农林渔业部门发放的大规模农业经营整合贷款的保证。
- **就农支援资金保证（就農支援資金保証）** — 新进入农户支援保证。
- **担い手・农业法人向保证** — 大规模农业者／法人农场保证。
- **灾害复旧保证（災害復旧保証）** — 灾害复原农业保证（台风损害、洪水损害、雪害、牲畜疫病）。
- **第 6 次产业化 / 加工流通相关保证** — 面向农业者主导的食品加工与直接流通的保证（「第 6 次产业化」政策通道）。
- **借换 / 经营改善资金保证** — 再融资与经营改善保证。

## 5. Operating logic — guarantee, subrogation, reinsurance

1. **申请** — 农业者或农业法人向放贷机构（通常为 JA／信连／农林中金／地方银行／JFC 农业部门）借款。放贷方需要保证信用，并向都道府县的农业信用基金协会提出申请。
2. **保证** — 都道府县协会评估贷款、接受保证，并收取保证费（保证料 —— 通常为残余本金上的低个位数年率，因产品而异）。
3. **放贷** — 放贷方在保证就位的情况下发放贷款。
4. **违约** — 若借款人违约，放贷方依据保证主张权利；都道府县协会向放贷方进行代位清偿（代位弁済），随后对借款人／担保物追索（求償）。
5. **再保险** — JAFFIC 的农业信用保险业务对都道府县协会的保证敞口进行再保险；依据适用的保险合同，保险赔付从 JAFFIC 流向都道府县协会。
6. **残余损失** — 在追偿与 JAFFIC 保险之后，残余损失由都道府县协会的准备金吸收，并由国家／都道府县／合作社的出资予以补充。

因此该制度具有 **三个损失吸收层**：(a) 借款人追偿，(b) 都道府县协会准备金 + 全国 JAFFIC 再保险，(c) 为制度托底的公共／合作社资本出资。

## 6. 保証料 economics

保证费体系由各都道府县农业信用基金协会在 MAFF 指引范围内设定，通常构成如下：

- 残余本金上的年率，对于高级别政策贷款（如超级 L 资金）处于低个位数基点区间，对于风险较高的产品则更高。
- 按产品（现代化资金／改良资金／就农支援 等）与借款人类别（担い手优先折扣）分层。
- 对优先政策类别（新进入农户、灾害复原、第 6 次产业化）予以补贴。
- 保证费经济性被刻意设定为低于市场等值的商业保证定价 —— 其利差由公共／合作社补贴吸收。

## 7. Year evolution

| Year | Event |
|---|---|
| 1961 | 制定农业信用保证保险法；设立都道府县农业信用基金协会。 |
| 1972 | 设立农业信用保险公库（后整合入 JAFFIC）。 |
| 1986 | 设立林业信用基金。 |
| 2003-10-01 | 通过整合农业、林业、渔业信用保险主体，创设独立行政法人农林渔业信用基金 (JAFFIC)。 |
| 2007 | 在农业政策改革下扩充担い手优先保证项目。 |
| 2010s | 增设第 6 次产业化保证；农业者主导的加工／流通获得明确的保证支持。 |
| 2014–2016 | TPP／FTA 对农业的压力；保证制度被用于支持农业现代化的适应。 |
| 2018 | 批发市场法修订／种子法废止／渔业法修订 等 —— 农业政策重构时代。 |
| 2020–2022 | COVID 时期灾害保证为农业供应链中断（尤其是乳制品过剩、餐饮业崩溃对和牛／园艺的冲击）而启动。 |
| 2022–2023 | 能源／化肥／饲料价格冲击 —— 保证制度支持畜牧及温室园艺行业的流动性。 |
| 2024 | 智慧农业（スマート農業 / smart-agriculture）贷款产品获得保证支持。 |
| 2024–2025 | 农业改革（基本法修订路径、食料安全保障重点）—— 保证制度被定位用于食料安全保障投资支持。 |
| 2025 | 担い手承继 + 法人化优先 —— 保证制度与法人农场／承继驱动的结构改革相一致。 |

## 8. Comparison vs general SME credit guarantee system

| Dimension | Agriculture credit guarantee system | [[policy-finance/japan-credit-guarantee-system\|General SME credit guarantee system]] |
|---|---|---|
| Borrower target | 農家 / 農業法人 / 農業関連事業者 | SME / small business (broad sectors) |
| Local guarantor | 農業信用基金協会 (47 prefectural) | 信用保証協会 (51 prefecture / city) |
| National federation | 全国農業信用基金協会連合会 | [[policy-finance/national-federation-credit-guarantee-corporations\|全国信用保証協会連合会 (JFG)]] |
| National reinsurer | 農林漁業信用基金 (JAFFIC) | [[financial-regulators/jfc\|JFC]] credit-insurance |
| Supervisor | MAFF (農林水産省) | METI / SME Agency (中小企業庁) |
| Lender partners | JA / 信連 / Norinchukin / banks / JFC ag-arm | Regional banks / shinkin / credit cooperatives / JFC |
| Crisis-era flagship | Disaster / livestock-disease / 食料安全保障 | セーフティネット / 危機関連 (ZeroZero 2020–2022) |
| Funding economics | 公的 / 系統金融 subsidies | 公的補助 + 信用保険 reinsurance |

这两个制度是具有不同行业目标、不同监管者、不同合作社银行接口的 **并行公共信用保证基础设施**。两者并不争夺借款人 —— 向农业者的贷款走农业制度，向小型制造商的贷款走中小企业制度。

## 9. Boundary cases

- **并非 JA Bank** — JA Bank 是放贷路径，本页是叠加于其上的保证保险机制。
- **并非中小企业保证制度** — 中小企业保证路由至 [[policy-finance/japan-credit-guarantee-system]]。
- **并非渔业** — 渔业的保证保险属于 [[policy-finance/fisheries-credit-guarantee-system]]（同一 JAFFIC 母体下的姊妹行业）。
- **并非林业（林業）** — 林业信用支持亦为 JAFFIC 业务，但在 MAFF 林业政策下单独路由。
- **并非 JFC 的农业放贷** — JFC 的 [[financial-regulators/jfc|agriculture-forestry-fisheries arm]] 直接向农业者放贷；本页关乎对农业者借款的保证保险，当保证贷款由 JFC 发起时二者可能重叠。
