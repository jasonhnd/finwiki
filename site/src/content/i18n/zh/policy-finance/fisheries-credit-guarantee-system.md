# 日本的渔业信用担保体系

## 维基路线

该条目位于 [[policy-finance/INDEX|policy-finance index]] 下。 Read it against [[policy-finance/japan-credit-guarantee-system|Japan credit guarantee system]] for SME-side contrast, [[policy-finance/agriculture-credit-guarantee-system|agriculture credit guarantee system]] for the sister sector (same JAFFIC parent), and [[policy-finance/japan-policy-finance-system|Japan policy finance system]] for the broader public-credit boundary.渔业存贷款对应机构为[[banking/jf-marine-bank-system-japan|JF Marine Bank system]]，合作社顶端为[[JapanFG/norinchukin|Norinchukin]]（共同服务于农林渔业合作社）。

## 长话短说

日本的渔业信用担保体系（漁業信用保証保険制度）通过两级结构补充中小渔业经营者的信用：（1）都道府县和广域一级**漁業信用基金協会**（渔业信贷基金协会）向贷款机构担保渔业借款人债务， **全国漁業信用基金協会連合会** 作为国家联合会，由 (2) 国家级 **独立行政法人農林漁業信用基金** (JAFFIC) 渔业信用保险业务再保险。该系统与 [[policy-finance/agriculture-credit-guarantee-system|agriculture credit guarantee system]] 平行，但具有特定部门的借款人目标 — 渔船金融、水产养殖、海鲜加工 —，地理上集中在日本沿海地区，其中北海道、九州（长崎/鹿儿岛）、Tōhoku 太平洋海岸和三陆海岸占保证体积大。

## 3。 機関概要 — 杰菲克 漁業信用保険業務

与农业相同的上级机构（独立行政机构 農林漁業信用基金 / JAFIC）。 JAFFIC 内有单独的**渔业信用保险**账户，具有单独的再保险池和单独的县协会接口。有关农业账户对应项，请参阅 [[policy-finance/agriculture-credit-guarantee-system]]。

## 4。 主要保証商品

- **漁業近代化資金保証** — 现代化贷款担保，用于渔船建造、发动机更换、齿轮升级、制冷、船上加工。
- **漁業経営改善支援資金保証** — 渔业管理改进贷款担保。
- **沿岸漁業改善資金保証** — 沿海渔业改良贷款担保。
- **新規就業者支援資金保証** — 新进入渔民/水产养殖经营者支持保证。
- **養殖関連融資保証** — 水产养殖设施/饲料/鱼种营运资金保证（大量用于扇贝/牡蛎/鰤鱼/昆布/珍珠段）。
- **水産加工 / 流通融資保証** — 海鲜加工和分销营运资金保证。
- **災害復旧保証** — 灾后恢复保障（台风损坏、海洋污染、赤潮、船舶灭失、港口损坏）。
- **燃油価格高騰対策保証** — 燃油价格冲击流动性保证。
- **借換保証** — 再融资担保。

## 5。 製品ミックス特点

渔业保证书与农业书相比，具有独特的产品组合概况：

- **渔船融资**需要大量抵押，但资产价值不断贬值；担保条款反映了特种船舶的船舶生命周期和转售市场的稀缺性。
- **水产养殖金融**在地理上高度集中（北海道和三陆的扇贝/昆布；广岛和三重的牡蛎/珍珠；鹿儿岛和爱媛的黄尾鱼；和歌山和长崎的金枪鱼养殖），并容易受到环境冲击（赤潮、海洋热浪、疾病爆发）。
- **处理融资**与一般中小企业的营运资金需求重叠，但季节性与出口市场的捕捞周期和库存建设相关。

## 6。地理集中度

与农业在地理上分布在几乎所有县不同，渔业保障体系**主要集中在沿海县**：

| 地区 | 渔业特色 | 保证书重点 |
|---|---|---|
| 北海道 | 太平洋鲑鱼、扇贝、海带、海胆、螃蟹 — 日本产量和价值最大的渔业县。 | 船舶金融+扇贝/海带养殖+加工。 |
| 三陆海岸（岩手县、宫城县） | 沿海渔业；牡蛎、扇贝、海带养殖；三文鱼;秋刀鱼。 | 2011-震后重建保障+养殖业。 |
| 九州（长崎、鹿儿岛、福冈、佐贺） | 金枪鱼养殖、黄尾鱼养殖、沿海渔业、远洋捕捞船队。 | 水产养殖+船舶融资+出口加工。 |
| 日本海（鸟取、岛根、富山） | 雪蟹、发光鱿鱼、黄尾鱼。 | 船舶+季节性加工保证。 |
| 太平洋中部（千叶、静冈、高知） | 鲣鱼、金枪鱼、鲭鱼；远洋舰队。 | 远洋船队的船舶融资。 |
| 内海（广岛、爱媛、三重） | 牡蛎、珍珠、鰤鱼。 | 水产养殖重保证书。 |

地理集中意味着渔业保障体系的投资组合风险面临**区域海洋环境冲击**，这可能同时影响单个沿海地区的多项保障。

## 7。年份演变

| 年份 | 活动 |
|---|---|
| 1952 | 中小漁業融資保証法颁布；成立都道府县渔业信用基金协会。 |
| 1972 | 漁業信用基金 国家级成立。 |
| 1990s | 200海里EEZ制度/远洋渔业重组；担保重组支持船舶减量政策。 |
| 2003-10-01 | 独立行政法人農林漁業信用基金（JAFFIC）由农林渔业信用保险机构整合而成。 |
| 2011-03-11 | Tōhoku 地震/海啸摧毁了三陆沿海渔业 — 渔业保障体系是重建贷款的核心。 |
| 2018 | 漁業法 70 年基本修订 — TAC/IQ 资源管理现代化；水产养殖许可证改革；保障体系适应新的资源架构。 |
| 2020–2022 | 新冠病毒时代的餐饮服务崩溃打击了金枪鱼/黄尾鱼/扇贝/牡蛎出口市场 — 担保系统支持流动性。 |
| 2022 | 能源/燃料价格冲击； 燃油価格高騰 保证已激活；沿海渔业经营利润大幅下降。 |
| 2023 | 福岛ALPS处理水释放 → 中国海鲜进口禁令 → 扇贝/海参出口冲击 — 保障系统为受影响的运营商（尤其是北海道/青森县）提供流动资金桥梁。 |
| 2024 | 与 ALPS 相关的出口市场持续复苏；担保书吸收了正在进行的重建。 |
| 2024–2025 | 影响海带/扇贝/贝类的海洋热浪冲击；担保体系涵盖经营亏损过桥融资。 |
| 2025 | 继续将政策重点放在水产养殖扩张上（国産養殖振興）作为粮食安全战略；保证体系一致。 |

## 8。与农业信用担保体系的比较

| 尺寸 | 渔业信用担保体系 | [[policy-finance/agriculture-credit-guarantee-system\|Agriculture credit guarantee system]] |
|---|---|---|
| 借款人目标 | 漁業者 / 漁業法人 / 養殖業者 / 水産加工業者 | 農家 / 農業法人 / 関連事業者 |
| 当地担保人 | 漁業信用基金協会（县/広域） | 農業信用基金協会 (47 县) |
| 全国联合会 | 全国漁業信用基金協会連合会 | 全国農業信用基金協会連合会 |
| 国家再保险公司 | 杰菲克漁業信用保険業務 | 杰菲克農業信用保険業務 |
| 主管 | 水産庁 / 农林水产省 | 农林水产省 |
| 贷方合作伙伴 | JF / 信漁連 / 农林中金 / 銀行 / JFC农林渔业臂 | JA / 信連 / Norinchukin / 銀行 / JFC农林渔业部门 |
| 地理范围 | 沿海县集中 | 全国范围 |
| 资产特征 | 船舶（折旧、专用）+水产养殖（暴露于环境中） | 土地+农具+牲畜 |
| 危机载体 | 资源崩溃/赤潮/台风/漏油/出口市场冲击（ALPS水事件） | 天气/疾病/TPP-FTA/食品价格 |
| 典型规模 | 个人贷款规模较小；集中投资组合 | 更广泛的借款人基础；贷款规模分布更加多样化 |

两个系统共享相同的 JAFFIC 再保险母公司，但以不同的精算假设和不同的县网络覆盖范围运营**独立的保险账户**。

## 9。边界情况

- **不是 JF Marine Bank** — JF Marine Bank 是贷方途径；此页面是顶层的保证保险机制。
- **不是农业** — 农业保证保险属于 [[policy-finance/agriculture-credit-guarantee-system]]（姐妹部门，同一 JAFIC 母公司）。
- **非普通中小企业担保** — 普通中小企业担保属于[[policy-finance/japan-credit-guarantee-system]]。
- **不是 森林（林业）** — 林业信贷支持也在 JAFIC 下，但是一个单独的账户。
- **不是JFC渔业贷款** — JFC的[[JapanFG/jfc|agriculture-forestry-fisheries arm]]直接贷款；本页介绍的是渔业借款的保证保险。
