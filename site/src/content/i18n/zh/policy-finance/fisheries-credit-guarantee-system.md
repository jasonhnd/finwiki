# 日本的渔业信用保证制度

## Wiki 路径

本条目位于 [[policy-finance/INDEX|policy-finance index]] 之下。请将其与 [[policy-finance/japan-credit-guarantee-system|Japan credit guarantee system]]（中小企业侧对比）、[[policy-finance/agriculture-credit-guarantee-system|agriculture credit guarantee system]]（姊妹板块，同一 JAFFIC 母体）以及 [[policy-finance/japan-policy-finance-system|Japan policy finance system]]（更广泛的公共信用边界）对照阅读。渔业的存贷对应物为 [[banking/jf-marine-bank-system-japan|JF Marine Bank system]]，合作社顶端为 [[cooperative-banks/norinchukin|Norinchukin]]（共同服务农业、林业和渔业合作社）。

## TL;DR

日本的渔业信用保证制度（漁業信用保証保険制度）通过两层结构补充中小渔业经营者的信用：(1) 在都道府县及广域层面、向贷款机构保证渔业借款人债务的**渔业信用基金协会**（Fisheries Credit Fund Associations），并以**全国渔业信用基金协会联合会**作为全国联合体，再由 (2) 国家层面的**独立行政法人农林渔业信用基金**（JAFFIC）渔业信用保险业务予以再保险。该制度与 [[policy-finance/agriculture-credit-guarantee-system|agriculture credit guarantee system]] 平行，但具有板块特定的借款人目标——渔船融资、养殖、水产加工——并在地理上集中于日本的沿海都道府县，以北海道、九州（长崎 / 鹿儿岛）、东北太平洋岸、三陆海岸占据保证量的大部分。

## 1. 与系统金融的位置关系（fisheries variant）

日本渔业金融通过与农业栈平行的**渔业系统金融**栈流转：

| 层 | 功能 | FinWiki 路径 |
|---|---|---|
| 渔业者 / 渔业法人 / 加工业者 | 借款人（渔民、渔业法人、养殖经营者、水产加工业者、合资格合作社）。 | 体系路径在此 |
| 单位渔协（JF / 单渔协） | 一级渔业合作社；接受社员存款；提供渔业相关信用；运营卸鱼港物流与鱼市场功能。 | [[banking/jf-marine-bank-system-japan]] |
| 信渔连（都道府县信用渔业协同组合联合会） | 渔业合作社的都道府县信用联合会。 | [[banking/jf-marine-bank-system-japan]] |

| 水产厅 / MAFF | 渔业合作社系统 + 保证保险系统的政策监督者。 | [[policy-finance/INDEX]] |

## 2. 机构概要 — 渔业信用基金协会

| 项目 | 内容 |
|---|---|
| 设置 | 都道府县 / 广域 — 全国渔业信用基金协会数量与 agriculture 不同，含整合多个都道府县的广域协会 |
| 法令 | 中小渔业融资保证法（1952） |
| 法律形态 | 特别法上的法人 |
| 所管 | 水产厅（Fisheries Agency）/ 都道府县 |
| 主要业务 | 向渔业者 / 渔业法人 / 关联事业者 的融资保证 |
| 联合会 | 全国渔业信用基金协会联合会（zengyoki.or.jp） |
| 再保险 | 农林渔业信用基金（JAFFIC）渔业信用保险业务 |

## 3. 机构概要 — JAFFIC 渔业信用保险业务

与农业同一母机构（独立行政法人农林渔业信用基金 / JAFFIC）。JAFFIC 内设有独立的**渔业信用保险**账户，拥有独立的再保险池和独立的都道府县协会接口。农业账户对应物请参见 [[policy-finance/agriculture-credit-guarantee-system]]。

## 4. 主要保证产品

- **渔业现代化资金保证** — 用于渔船建造、发动机更换、渔具升级、冷藏、船上加工的现代化贷款保证。
- **渔业经营改善支援资金保证** — 渔业经营改善贷款保证。
- **沿岸渔业改善资金保证** — 沿岸渔业改善贷款保证。
- **新规就业者支援资金保证** — 新入行渔民 / 养殖经营者支援保证。
- **养殖关联融资保证** — 养殖设施 / 饲料 / 鱼苗的营运资金保证（在扇贝 / 牡蛎 / 鰤鱼 / 昆布 / 珍珠板块中大量使用）。
- **水产加工 / 流通融资保证** — 水产加工及流通的营运资金保证。
- **灾害复旧保证** — 灾害恢复保证（台风损害、海洋污染、赤潮、船舶损失、渔港损害）。
- **燃油价格高涨对策保证** — 燃油价格冲击的流动性保证。
- **借换保证** — 再融资保证。

## 5. 产品组合特征

渔业保证簿相对农业簿具有独特的产品组合状况：

- **渔船融资**重度抵押，但伴随贬值的资产价值；保证条件反映船舶生命周期以及专业化船舶再售市场的稀薄。
- **养殖融资**地理上高度集中（北海道与三陆的扇贝 / 昆布；广岛与三重的牡蛎 / 珍珠；鹿儿岛与爱媛的鰤鱼；和歌山与长崎的金枪鱼养殖），并暴露于环境冲击（赤潮、海洋热浪、疾病暴发）。
- **加工融资**与一般中小企业营运资金需求重叠，但伴随与渔获周期及面向出口市场库存累积相关联的季节性。

## 6. 地理集中

与几乎分布于所有都道府县的农业不同，渔业保证制度**重度集中于沿海都道府县**：

| 地区 | 渔业性格 | 保证簿重点 |
|---|---|---|
| 北海道 | 太平洋鲑鱼、扇贝、海带、海胆、螃蟹 — 按量与价值计为日本最大渔业都道府县。 | 渔船融资 + 扇贝 / 海带养殖 + 加工。 |
| 三陆海岸（岩手、宫城） | 沿岸渔业；牡蛎、扇贝、海带养殖；鲑鱼；秋刀鱼。 | 2011后的地震重建保证 + 养殖。 |
| 九州（长崎、鹿儿岛、福冈、佐贺） | 金枪鱼养殖、鰤鱼养殖、沿岸渔业、远洋渔船队。 | 养殖 + 渔船融资 + 出口加工。 |
| 日本海（鸟取、岛根、富山） | 雪蟹、萤火鱿、鰤鱼。 | 渔船 + 季节加工保证。 |
| 太平洋中部（千叶、静冈、高知） | 鲣鱼、金枪鱼、鲭鱼；远洋渔船队。 | 用于远洋渔船队的渔船融资。 |
| 内海（广岛、爱媛、三重） | 牡蛎、珍珠、鰤鱼。 | 偏重养殖的保证簿。 |

地理集中意味着渔业保证制度的组合风险暴露于可能同时影响单一沿海区域多份保证的**区域海洋环境冲击**。

## 7. 年度演进

| 年份 | 事件 |
|---|---|
| 1952 | 中小渔业融资保证法 制定；都道府县渔业信用基金协会 设立。 |
| 1972 | 渔业信用基金 在国家层面设立。 |
| 1990年代 | 200海里 EEZ 制度 / 远洋渔业重构；由保证重构支持的船舶削减政策。 |
| 2003-10-01 | 独立行政法人农林渔业信用基金（JAFFIC） 通过整合农业、林业和渔业信用保险主体而创设。 |
| 2011-03-11 | 东北地震 / 海啸摧毁三陆沿岸渔业 — 渔业保证制度成为重建贷款的核心。 |
| 2018 | 渔业法 70年来的根本性修订 — TAC / IQ 资源管理现代化；养殖许可改革；保证制度适应新的资源框架。 |
| 2020–2022 | COVID 时代餐饮崩溃冲击金枪鱼 / 鰤鱼 / 扇贝 / 牡蛎出口市场 — 保证制度支援流动性。 |
| 2022 | 能源 / 燃油价格冲击；燃油价格高涨保证启动；沿岸渔业经营利润率崩溃。 |
| 2023 | 福岛 ALPS 处理水排放 → 中国海产品进口禁令 → 扇贝 / 海参出口冲击 — 保证制度支援受影响经营者（尤其北海道 / 青森）的营运资金过桥。 |
| 2024 | 持续的 ALPS 相关出口市场恢复；保证簿吸收正在进行的重建。 |
| 2024–2025 | 影响海带 / 扇贝 / 贝类的海洋热浪冲击；保证制度覆盖营业亏损的过桥融资。 |
| 2025 | 作为粮食安全战略对养殖扩张（国产养殖振兴）的持续政策聚焦；保证制度予以协调。 |

## 8. 与农业信用保证制度的比较

| 维度 | 渔业信用保证制度 | [[policy-finance/agriculture-credit-guarantee-system\|Agriculture credit guarantee system]] |
|---|---|---|
| 借款人目标 | 渔业者 / 渔业法人 / 养殖业者 / 水产加工业者 | 农户 / 农业法人 / 关联事业者 |
| 本地保证人 | 渔业信用基金协会（都道府县 / 广域） | 农业信用基金协会（47 都道府县） |
| 全国联合体 | 全国渔业信用基金协会联合会 | 全国农业信用基金协会联合会 |
| 全国再保险人 | JAFFIC 渔业信用保险业务 | JAFFIC 农业信用保险业务 |
| 监督者 | 水产厅 / MAFF | MAFF |
| 贷款伙伴 | JF / 信渔连 / Norinchukin / 银行 / JFC 农林渔业部门 | JA / 信连 / Norinchukin / 银行 / JFC 农林渔业部门 |
| 地理范围 | 沿海都道府县集中 | 全国 |
| 资产特征 | 船舶（贬值、专业化）+ 养殖（暴露于环境） | 土地 + 农机具 + 牲畜 |
| 危机向量 | 资源崩溃 / 赤潮 / 台风 / 漏油 / 出口市场冲击（ALPS 水事件） | 天气 / 疾病 / TPP-FTA / 粮价 |
| 典型规模 | 较小的单笔贷款规模；集中的组合 | 更广的借款人基础；更多元的贷款规模分布 |

两系统共享同一 JAFFIC 再保险母体，但运营具有不同精算假设和不同都道府县网络覆盖的**独立保险账户**。

## 9. 边界情形

- **并非 JF Marine Bank** — JF Marine Bank 是贷款途径；本页是叠加于其上的保证保险机制。
- **并非农业** — 农业保证保险归属于 [[policy-finance/agriculture-credit-guarantee-system]]（姊妹板块，同一 JAFFIC 母体）。
- **并非普通中小企业保证** — 普通中小企业保证归属于 [[policy-finance/japan-credit-guarantee-system]]。
- **并非森林（forestry）** — 林业信用支援亦在 JAFFIC 之下，但为独立账户。
- **并非 JFC 渔业贷款** — JFC 的 [[financial-regulators/jfc|agriculture-forestry-fisheries arm]] 直接放贷；本页关乎渔业借款的保证保险。
