# 福岡県信用保証協会 (Fukuoka Credit Guarantee Corporation)

## Wiki route

本条目位于 [[policy-finance/INDEX|policy-finance index]] 之下，是日本信用保证体系中 51  个都道府县 / 市级成员之一。可与 [[policy-finance/japan-credit-guarantee-system|Japan credit guarantee system]] 的体系概览和 [[policy-finance/national-federation-credit-guarantee-corporations|Japan Federation of Credit Guarantee Corporations]] 的协调机构一起阅读。与 [[policy-finance/tokyo-credit-guarantee-corp|Tokyo CGC]] 和 [[policy-finance/osaka-credit-guarantee-corp|Osaka CGC]]（二者保证余额更大）进行同业比较时，Fukuoka CGC 的特点在于它是九州地区中小企业金融的引力中心，锚定了横跨九州七县经济的 hub-and-spoke 模式。

## TL;DR

Fukuoka Credit Guarantee Corporation（福岡県信用保証協会）是日本 51 个协会地方信用保证体系中的福冈县成员。其保证账簿在结构上是九州中小企业金融的锚点：福冈县集中九州最大人口、金融服务中心（Hakata / Tenjin）、由 TSMC 带动的半导体集群（Kitakyushu 工业带 + Kumamoto 外溢）、Hakata Port 物流、面向东亚的入境旅游门户，以及重要的农业 / 食品加工长尾。主要共同贷款伙伴包括 [[JapanFG/fukuoka-bank|Fukuoka Bank ([[JapanFG/fukuoka-fg|Fukuoka FG]])]]、[[JapanFG/nishi-nippon-city-bank|Nishi-Nippon City Bank ([[JapanFG/nishi-nippon-fhd|Nishi-Nippon FHD]])]]、[[JapanFG/fukuoka-chuo-bank|Fukuoka Chuo Bank]]，以及密集的九州 shinkin / 信用组合网络。

## 2. 主要保证产品

- **一般保証** — 责任共享制度（責任共有制度）下的标准中小企业银行贷款保证。
- **マル経** — 小企业经营改善贷款保证。
- **セーフティネット保証** — 行业困境指定事件保证（如半导体周期反转、入境旅游冲击、港口货流下滑）。
- **危機関連保証** — 危机相关保证（国家紧急框架，100% 保证）。
- **創業保証** — 创业 / 新业务保证，在福冈市成长中的 startup 生态中使用活跃。
- **事業承継保証** — 事业传承保证。
- **経営革新保証** — 经营革新 / 业务转型保证。
- **借換保証** — 再融资 / 整合保证。
- **半導体関連サプライチェーン特別保証** — 福冈县政府协调、面向 TSMC 锚定半导体集群供应商基础的项目。

## 3. 与县内产业结构的关系

福冈县是九州经济的结构性中心，Fukuoka CGC 的保证账簿反映出多集群画像：

1. **半導体 (Semiconductor cluster)** — Kitakyushu / Kurate / Munakata 工业带在邻县 Kumamoto 的 TSMC-JASM (Japan Advanced Semiconductor Manufacturing) 投资于 2021–2024  催化区域供应链建设后，成为主要半导体供应商基地。为化学品、气体、封装、精密加工、设备维护和工程服务供货的福冈中小企业，使用保证支持信贷进行 capex ramp。
2. **物流 / 港湾 (Logistics and ports)** — Hakata Port（集装箱）和 Kitakyushu Port（散货 + roll-on / roll-off ferry）锚定大量货代、仓储、报关中小企业长尾。Hakata 也是日本通往韩国的主要 ferry / cruise 门户，以及重要入境入口。
3. **観光 (Tourism)** — 福冈是九州国内旅游（Hakata / Fukuoka City 城市旅游、Dazaifu、Yanagawa、Mojiko）和入境旅游（韩国 / 台湾 / 中国直航与 ferry）的枢纽。酒店、餐饮、零售、交通中小企业长尾密集，并在 2020–2022 边境关闭期间遭受重击。
4. **食品 / 農業 (Food and agriculture)** — 福冈拥有大量食品加工中小企业集中（Hakata ramen / motsunabe / mentaiko 品牌、清酒 / 烧酎酿造、农产品 — 草莓、柿子、柑橘），并有活跃的对东亚食品出口管线。
5. **金融 / サービス (Financial and services)** — Tenjin 和 Hakata 是九州金融服务和专业服务中心，承载许多金融机构、专业公司和 IT / startup 运营商的九州总部。創業保証 产品被福冈市成长中的 startup 生态大量使用。
6. **重工業 (Heavy industry)** — Kitakyushu 保留历史性钢铁（Nippon Steel Yawata works）、化学和机械基础，虽较战后高峰缩小，但 Tier-2/3  中小企业供应商长尾仍活跃。

Fukuoka CGC 在功能上也作为**九州地区中小企业金融重心**发挥作用。虽然九州各县都有自己的 CGC，但福冈集群的银行和企业集中度意味着，许多跨九州的中小企业运营会通过福冈银行分支和 Fukuoka CGC 保证来路由信贷。

## 4. 共同贷款和 JFC overlay

Fukuoka CGC 保证通常与 [[JapanFG/fukuoka-bank|Fukuoka Bank]]（主导区域银行，[[JapanFG/fukuoka-fg|Fukuoka FG]] 控股公司还包括 Kumamoto Bank、Nagasaki 的 [[JapanFG/18-shinwa-bank|Eighteenth-Shinwa Bank]] 和 Minato Bank operations）、[[JapanFG/nishi-nippon-city-bank|Nishi-Nippon City Bank]]（在 [[JapanFG/nishi-nippon-fhd|Nishi-Nippon FHD]] / NNFH 之下）以及 [[JapanFG/fukuoka-chuo-bank|Fukuoka Chuo Bank]]（较小区域银行）的贷款并列。九州 shinkin 和信用组合网络服务更长尾的中小企业。大型银行参与也具有实质性 — [[JapanFG/mufg|MUFG]]、[[JapanFG/smfg|SMFG]] 和 [[JapanFG/mizuho-fg|Mizuho]] 都在 Hakata / Tenjin 设有分行，服务中大型中小企业和企业客户。

[[banking/regional-bank-consolidation-pattern|regional-bank consolidation pattern]] 在九州非常可见，其中 Fukuoka FG 的跨县 roll-up（Fukuoka + Kumamoto + Nagasaki）是日本国内最大的区域银行控股公司构造之一。在福冈县内部，强大的 Hakata 经济锚点缓和了整合压力。

公共金融共同贷款通过 [[JapanFG/jfc|JFC]]（政府系金融機関）路由 — JFC 的中小企业和小企业部门与有保证支持的区域银行 tranche 共同贷款，同时 JFC 信用保险对保证层再保险。[[JapanFG/shoko-chukin|商工中金 (Shoko Chukin)]] 参与合作组织化的中小企业群体，尤其围绕半导体供应商集群和食品出口 consortium。

## 5. 代位弁済与求償

当获得保证的福冈中小企业违约时，Fukuoka CGC 根据保证合同向贷款金融机构支付代位弁済，然后追索求償。回收后的剩余损失流入由 [[JapanFG/jfc|JFC]] 再保险的联合会 pooled 信用保险层，并由 [[policy-finance/national-federation-credit-guarantee-corporations|全国信用保証協会連合会]] 协调 51  个协会之间的损失分担。

福冈的代位弁済量具有特征性冲击向量：(1) 入境旅游崩塌（边境关闭、与邻近国家的地缘政治紧张），(2) 全球半导体周期反转（供应商基础 capex 压力），(3) Kitakyushu 重工业需求周期，以及 (4) Hakata Port 货流冲击。按日本地方县标准，福冈市城区的回收价值合理，支持健康的求償回收率。

## 6. 危机时期回应

在 COVID-19  紧急时期（2020–2021），Fukuoka CGC 深度参与全国 実質無利子・無担保（ゼロゼロ融資 / ZeroZero loan）特别保证项目。冲击对入境旅游中小企业尤其严重，因为与韩国 / 台湾 / 中国的边境关闭完全切断主导入境流量，也重击了密集的 Hakata / Tenjin 餐饮 / 酒店基础。福冈县政府叠加了通过 Fukuoka CGC 渠道实施的县内紧急项目。

2023–2025  期间 COVID 后正常化，随着 2020 年代 ZeroZero 本金偿还宽限结束，出现预期中的代位弁済浪潮；同时入境旅游板块强劲恢复（韩国和台湾访客流在 2023–2024 期间快速反弹）。危機関連保証 框架因能源价格冲击（2022–2023）重新启动。2024–2025  期间，Fukuoka CGC 账簿正在向 一般保証 / 借換保証 / 事業承継保証 的平衡正常化，同时由区域 TSMC-JASM 生态建设推动的半导体供应商 capex tranche 上升。

## 7. 联合会体系锚点

在 [[policy-finance/japan-credit-guarantee-system|Japan credit guarantee system]] 架构内，Fukuoka CGC 按保证余额处于前五层级，并作为 [[policy-finance/national-federation-credit-guarantee-corporations|全国信用保証協会連合会]] 协调层的九州地区中小企业金融枢纽运作。当联合会评估区域锚点运营模板时，Fukuoka CGC 的多集群账簿（半导体 / 物流 / 入境旅游 / 食品 / 重工业）为同时具有产业和旅游周期性、并承担区域枢纽责任的县提供了有用参考。

[[JapanFG/jfc|JFC]] 的联合会 pooled 信用保险层吸收 Fukuoka CGC first-loss 代位弁済后的剩余损失。半导体供应商 capex tranche 在运营上值得关注，因为 TSMC-JASM Kumamoto 投资外溢至福冈，创造了一个风险周期画像不同于传统 Kitakyushu 重工业长尾的保证板块 — 半导体 capex 周期可能在单一财政年度内产生大幅波动，使联合会再保险和 JFC 信用保险平滑功能承担更大权重。[[JapanFG/fukuoka-fg|Fukuoka FG]] 多县控股公司形态中可见的 [[banking/regional-bank-consolidation-pattern|regional-bank consolidation pattern]]，也增加了干净归属保证相关风险到县级账簿的复杂性。

## 8. 与同业都道府县 CGC 的比较

| 维度 | Fukuoka CGC | [[policy-finance/tokyo-credit-guarantee-corp|Tokyo CGC]] | [[policy-finance/hokkaido-credit-guarantee-corp|Hokkaido CGC]] |
|---|---|---|---|
| 主导产业组合 | 半导体 + 物流 + 入境旅游 + 食品 + 重工业 | 服务 / 零售 / 金融 / IT / 酒店 | 农业 / 渔业 / 旅游 / 食品 |
| 区域角色 | 7  个县的九州中小企业金融重心 | 全国枢纽 | 北方第一产业锚点 |
| 入境旅游地缘敏感度 | 很高（韩国 / 台湾 / 中国门户） | 中等（全球入境） | 中等（东亚滑雪 / 自然入境） |
| 半导体暴露 | 新增且增长中（TSMC-JASM 外溢） | 有限 | 有限 |
| 主导区域银行 | [[JapanFG/fukuoka-bank|Fukuoka Bank]] / [[JapanFG/fukuoka-fg|Fukuoka FG]] 主导；[[JapanFG/nishi-nippon-city-bank|NNCB]] 次要 | 无区域银行主导 | [[JapanFG/hokkaido-bank|Hokkaido Bank]] / [[JapanFG/hokuyo-bank|Hokuyo Bank]] 双头 |
| 多县 FG 外溢 | [[JapanFG/fukuoka-fg|Fukuoka FG]] 控股公司横跨 3  个县（Fukuoka + Kumamoto + Nagasaki） | n/a | [[JapanFG/hokuhoku-fg|Hokuhoku FG]] 横跨 Hokkaido + Hokuriku |

该比较说明 Fukuoka CGC 在 [[policy-finance/japan-credit-guarantee-system|Japan credit guarantee system]] 框架下的独特位置：它是区域金融枢纽，而不是单一县账簿。
