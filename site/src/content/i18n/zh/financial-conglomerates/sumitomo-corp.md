# 住友商事 (Sumitomo Corporation)

## Wiki 路径

本条目作为 **综合商社母控股公司** 位于 [[financial-conglomerates/INDEX|financial-conglomerates INDEX]] 之下。请与 [[financial-conglomerates/mitsubishi-corp|三菱商事]]（旧财阀商社的两雄）及 [[financial-conglomerates/mitsui-co|三井物産]]（财阀商社的三巨头）对照阅读以获取同业／对比语境。关于赋予本条目 JapanFG 锚点的金融服务邻接性 —— 参见 [[trading-company-finance/sumitomo-corp-financial-management|住友商事フィナンシャルマネジメント]]（对内 CMS），尤其是 **[[leasing-firms/smfl|SMFL (三井住友ファイナンス&リース)]]**（住友商事 50% + [[megabanks/smfg|SMFG]] 50%、3 大租赁之一角）及 **[[leasing-firms/sumitomo-mitsui-auto-service|住友三井オートサービス]]**（汽车租赁国内首位级）。体系框架: [[finance/INDEX|finance index]] 与 [[finance/japan-listed-financial-groups-investable-universe|Japan 上場金融 G 投資ユニバース]]。

### 沿革要点 ^[extracted]

| 年 | 事项 |
|---|---|
| 1919 | 大阪北港株式会社 设立 (源于住友财阀的城市开发公司) |
| 1944 | 改名为 住友土地工务株式会社 |
| 1945 | 因 GHQ 财阀解体令改名为 日本建设产业 |
| 1952 | 因媾和条约生效改名为 住友商事，商社业务全面化 |
| 1970s〜 | 在钢管・运输机・建机 (主要为美国市场) 实现 global 拓展 |
| 1990s | 铜锭违规交易事件 (浜中事件、1996) — 计提巨额损失，成为治理改革契机 |
| 2010s | J:COM 持股强化 (与 KDDI 共同)、SCSK 整合 (旧 住商信息 + 旧 CSK) |
| 2020s | 净利润 3,000〜5,000 億円级，维持 5 大商社中位 |

此外，**住友集团的 financial-services 生态**:

「**SMFL = 住友商事的对外租赁・asset finance 之臂** (与 SMFG 的 50/50 JV)」「**住友三井汽车服务 = 汽车租赁** (与 3 社共同)」「住友商事金融管理 = 对内 CMS」的三层结构。与 [[financial-conglomerates/mitsubishi-corp|三菱商事]] (HC Capital 权益法 + MCFS) 及 [[financial-conglomerates/mitsui-co|三井物産]] (JA 三井租赁 + MBFS) 相比，**与大型银行 (SMFG) 的 50/50 JV** 这一共同控制体制是住友商事的特征。

### 5 大商社内的 identity

- **偏多元化商社**: 既非如 [[financial-conglomerates/mitsubishi-corp|三菱]] / [[financial-conglomerates/mitsui-co|三井]] 般偏资源、亦非如 [[financial-conglomerates/itochu-corp|伊藤忠]] 般偏非资源的多元化型定位
- **媒体・IT 商社**: 凭借 J:COM (有线电视) + SCSK (IT 服务)，在 5 大商社中是唯一媒体・IT 比重大者

### 竞争对手

- **国内**: 5 大商社 ([[financial-conglomerates/mitsubishi-corp|三菱]] / [[financial-conglomerates/mitsui-co|三井]] / [[financial-conglomerates/itochu-corp|伊藤忠]] / [[financial-conglomerates/marubeni-corp|丸紅]])
- **媒体 (J:COM 领域)**: NTT 集团、乐天移动、网络视频 (Netflix、Amazon Prime)
- **IT 服务 (SCSK 领域)**: NRI、富士通、NEC、TIS、伊藤忠 Techno-Solutions
