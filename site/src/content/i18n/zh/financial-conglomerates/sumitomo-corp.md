### 沿革亮点 ^[extracted]

| 年 | 事项 |
|---|---|
| 1919 | 大阪北港株式会社设立（起源于住友财阀的城市开发公司） |
| 1944 | 改称住友土地工务株式会社 |
| 1945 | 受 GHQ 财阀解体令影响，改称日本建设产业 |
| 1952 | 和平条约生效后改称住友商事，商社业务正式展开 |
| 1970年代起 | 在钢管、运输机、建机（主要为美国市场）展开 global 业务 |
| 1990年代 | 铜地金违规交易事件（滨中事件，1996）-- 巨额损失并成为治理改革契机 |
| 2010年代 | 强化 J:COM 持股（与 KDDI 共同），整合 SCSK（旧住商信息 + 旧 CSK） |
| 2020年代 | 纯利润 3,000～5,000 億円级，维持在 5 大商社中的中位 |

### 5 大商社中的 identity

- **偏多元商社**：既不像 [[financial-conglomerates/mitsubishi-corp|三菱]] / [[financial-conglomerates/mitsui-co|三井]] 那样偏资源，也不像 [[financial-conglomerates/itochu-corp|伊藤忠]] 那样偏非资源，而是多元型中位定位
- **媒体 / IT 商社**：J:COM（有线电视）+ SCSK（IT 服务）使其在 5 大商社中拥有唯一的高媒体 / IT 比重

### 竞争

- **国内**：5 大商社（[[financial-conglomerates/mitsubishi-corp|三菱]] / [[financial-conglomerates/mitsui-co|三井]] / [[financial-conglomerates/itochu-corp|伊藤忠]] / [[financial-conglomerates/marubeni-corp|丸紅]]）
- **媒体（J:COM 领域）**：NTT group、Rakuten Mobile、视频流媒体（Netflix、Amazon Prime）
- **IT 服务（SCSK 领域）**：NRI、Fujitsu、NEC、TIS、ITOCHU Techno-Solutions
