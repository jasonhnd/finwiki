# 伊藤忠商事 (ITOCHU Corporation)

## Wiki 路径

本条目归属于 [[financial-conglomerates/INDEX|financial-conglomerates INDEX]]，作为**综合商社母控股公司**（非金融服务业的商社母公司，但通过其金融服务子公司及战略持股而与 JapanFG 相关）。请将其与 [[financial-conglomerates/mitsubishi-corp|三菱商事]]（5 大首位 / 偏资源型的对照）以及 [[financial-conglomerates/marubeni-corp|丸紅]]（作为纺织·谷物商社的对照）对照阅读，以获取同业 / 对比语境。关于赋予本条目 JapanFG 锚点的金融服务邻接性——参见 [[trading-company-finance/itochu-finance|伊藤忠ファイナンス]]（对内 CMS），尤其是 **[[leasing-firms/tokyo-century|東京センチュリー]]**（伊藤忠的战略性持股 **约 25% 级** = 与瑞穗 FG 并列的 2 大股东，东证 PRIME **8439**）。系统框架：[[finance/INDEX|finance index]] 与 [[finance/japan-listed-financial-groups-investable-universe|Japan 上場金融 G 投資ユニバース]]。

**「东京 Century = 伊藤忠的对外租赁·融资之臂」「FamilyMart = 伊藤忠的零售金融平台」「伊藤忠 Finance = 对内 CMS」** 的三层结构是伊藤忠集团财务运营的定式。

### 5 大商社内的 identity

- **非资源 5 大商社**: 伊藤忠的资源比率在 5 大之中最低，靠**纺织·食品·ICT·便利店**赚取收益的「**生活产业商社**」
- **CITIC 联动**: 以中国市场准入为战略轴，通过 CITIC + 正大集团（泰国财阀）的 3 社方联动覆盖东亚·ASEAN

### 竞争

- **国内**: 5 大商社（[[financial-conglomerates/mitsubishi-corp|三菱]] / [[financial-conglomerates/mitsui-co|三井]] / [[financial-conglomerates/sumitomo-corp|住友]] / [[financial-conglomerates/marubeni-corp|丸紅]]）
- **便利店（FamilyMart 领域）**: 7&i HD（7-Eleven）、[[financial-conglomerates/mitsubishi-corp|三菱商事]] / KDDI（Lawson）
- **纺织商社**: [[financial-conglomerates/marubeni-corp|丸紅]]（纺织也强）、其他纺织专业商社
