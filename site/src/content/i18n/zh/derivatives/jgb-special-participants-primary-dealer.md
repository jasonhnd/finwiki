# JGB 特别参加者（日本一级交易商制度）

## TL;DR

JGB 特别参加者制度（国債市場特別参加者制度）是日本的一级交易商（primary dealer）体制：一个由财务省（MoF）管理的框架，在该框架下，一组明确界定的金融机构被认定为财务省 JGB 招标的主要交易对手，以换取承担在 JGB 曲线全段进行投标和做市的义务。该制度于 2004  引入，并经多次迭代加以完善；当前名单通常包含约 20  家机构，由日本巨型银行的证券关联机构、主要的日本独立证券公司，以及在日本运营的外国投资银行子公司组成。

特别参加者承诺：(i) 在每次 JGB 招标的各主要期限上提交有意义的投标；(ii) 在 JGB 二级市场提供双向报价；(iii) 报告交易活动和库存；以及 (iv) 维持充足的财务资源。作为交换，他们获得：(i) 进入财务省非价格招标（非价格竞争性招标 I 和 II）的资格，使其能够以加权平均中标价获取 JGB；(ii) 就 JGB 发行政策与财务省进行定期对话；(iii) 参与财务省回购；以及 (iv) 在财务省、日本银行（BoJ）和金融厅（FSA）眼中作为受认可的 JGB 做市商的地位。

日本的制度在概念上与美国一级交易商制度（由纽约联储管理）和英国金边债券做市商（GEMM）制度相似，但具有反映日本市场结构和财务省招标设计方法的独特运营特征。

对 FinWiki 而言，本条目涵盖特别参加者名单与分层、义务、利益、非价格竞争性招标机制，以及与美国和英国一级交易商制度的结构性比较。

## System Overview

| Element | Detail |
|---|---|
| 监管 / 管理方 | 财务省（MoF），理财局（Financial Bureau），国债管理课（Government Debt Management Division） |
| 设立 | 2004 （对先前类一级交易商安排的正式化） |
| 受认可机构 | 任一时点约 20  家（逐年随机构加入 / 退出而变化） |
| 认定依据 | 申请 + 财务省对资本充足性、做市能力、JGB 活动和合规的评估 |
| 续期 | 年度审查；机构可被增列或除名；执行行为标准 |
| 会员类别 | 单一层级（JGB 制度中无正式 A / B 子层级）；实践中存在一些运营上的区分 |
| 公布 | 财务省在其特别参加者页面公布当前名单 |

受认可机构的数量明显小于 FSA 更广泛的 [[securities/financial-instruments-business-operators-japan-index|Financial Instruments Business Operators]] 范围，反映了财务省有意将招标交易对手集中于有能力履行投标和做市义务的机构。

## List Composition (Typical)

虽然确切名单由财务省定期更新，但典型构成如下：

| Dealer category | Representative firms |
|---|---|
| 日本巨型银行证券关联机构 | MUFG Securities、SMBC Nikko Securities、Mizuho Securities（及其证券实体内的债券交易台） |
| 独立日本证券公司 | Nomura Securities、Daiwa Securities |
| 其他日本证券公司 | SBI Securities、Tokai Tokyo Securities、OKASAN Securities（逐年变化） |
| 信托及其他巨型银行关联债券实体 | 适用时（具体实体不一） |
| 在日外国投资银行子公司 | JPMorgan Securities Japan、Goldman Sachs Japan、Morgan Stanley MUFG Securities、Citigroup Global Markets Japan、Deutsche Securities、Barclays Securities Japan、BNP Paribas Securities (Japan)、HSBC Securities (Japan)、UBS Securities Japan |
| 其他 | Société Générale Securities (Japan)、Crédit Agricole Securities (Japan)，以及存在时的类似外国机构子公司 |

值得注意：作为日本注册证券公司运营的外国机构子公司（持 FIEA 牌照——见 [[securities/financial-instruments-business-operators-japan-index|FIBO registry]]）具有资格，并占名单相当大的份额。这种对外国机构开放的会员资格是日本制度的结构性特征之一。

上述名单为示例性；官方当前名单由财务省维护并在官方页面更新（见 Sources）。

## Obligations of Special Participants

特别参加者根据财务省规则接受以下义务：

| Obligation | Detail |
|---|---|
| 每次招标的投标义务 | 在每次财务省 JGB 招标的各主要期限上提交有意义的投标；最低参与门槛由财务省在招标规则中界定 |
| 中标份额 | 随时间达到最低中标份额（在年度审查中评估） |
| 二级市场报价 | 在 JGB 二级市场提供双向报价；义务按期限和当日时间校准 |
| 库存与活动报告 | 向财务省报告 JGB 库存头寸和交易量 |
| 资本充足与监管合规 | 维持 FIEA 要求的资本和运营标准 |
| 与财务省合作 | 参与财务省关于发行计划、市场运作和产品开发的对话 |

针对这些义务的履行情况每年进行审查；未达门槛的机构可被降级或从名单中除名。

## Benefits of Special Participant Status

作为对这些义务的交换，特别参加者获得：

| Benefit | Detail |
|---|---|
| 非价格竞争性招标 I（Non-Price I） | 有资格以加权平均中标价获取 JGB 招标的一部分；基于特别参加者份额的招标前分配 |
| 非价格竞争性招标 II（Non-Price II） | 以加权平均中标价进行的额外招标后分配；另一项非价格权益 |
| 与财务省的对话和咨询 | 与财务省就发行计划、市场运作和 JGB 计划发展进行的定期论坛；对财务省规划的影响 |
| 回购参与 | 参与财务省 JGB 回购 / 负债管理操作的资格 |
| 认可 | 财务省和日本银行对其作为 JGB 做市商的正式认可；对客户分销和声誉有用 |
| 接入运营设施 | 标准 JGB 清算和结算基础设施接入（BoJ-NET、[[securities/japan-securities-clearing-corp|JSCC]]） |

非价格竞争性招标（I 和 II）机制尤为重要：它为特别参加者提供了以招标加权平均价进行的有保障分配渠道，消除了仅凭竞争性投标所带来的部分价格发现风险。Non-Price I 在竞争性招标之前分配；Non-Price II 在之后以相同的加权平均价分配。

## Auction Mechanics in the JGB System

JGB 招标流程容纳特别参加者与其他投标者并存：

| Auction component | Eligibility / mechanism |
|---|---|
| 价格竞争性招标 | 向所有合格投标者开放（特别参加者 + 其他）；按竞争性投标分配 |
| 非价格竞争性招标 I（Non-Price I） | 仅限特别参加者；招标前；以预期加权平均价分配；按先前招标表现确定权益 |
| 非价格竞争性招标 II（Non-Price II） | 仅限特别参加者；招标后；以竞争性招标的实际加权平均中标价分配；额外权益 |
| 招标形式 | 多数 JGB 采用多价格（conventional）招标；视序列而定为收益率基准或价格基准的 conventional |

Non-Price I 权益为特别参加者创造了维持良好招标投标记录的激励，因为 Non-Price I 份额随历史表现而按比例增减。

## Comparison with US Primary Dealer System

由纽约联邦储备银行管理的美国一级交易商制度，存在结构性的相似与差异：

| Aspect | US Primary Dealers | JGB Special Participants |
|---|---|---|
| 管理方 | 纽约联邦储备银行 | 财务省（日本） |
| 机构数量 | ~24  家（不一）；名单由纽约联储维护 | ~20  家；名单由财务省维护 |
| 会员构成 | 美国银行关联交易商 + 非银行交易商 + 外国银行的美国子公司 | 日本证券公司 + 外国投资银行子公司 |
| 招标义务 | 在每次美国国债招标投标；有意义的份额 | 在每次 JGB 招标投标；有意义的份额 |
| 二级市场义务 | 合理的市场活动；报告 | 双向报价；报告 |
| 特别接入 | TOMO（临时公开市场操作）交易对手；SOMA 组合操作交易对手 | 非价格竞争性招标 I 和 II；财务省回购参与 |
| 中标分配机制 | 竞争性（自 1992 起国债采用统一价格） | 多价格竞争性 + Non-Price I 和 II |
| 监管互动 | 运营上为美联储（纽约联储）；实质上为 OCC、SEC、银行监管机构 | 运营上为财务省；审慎监管为 FSA；货币操作为日本银行 |
| 外国机构参与 | 是（广泛） | 是（广泛） |

关键结构性差异：美国财政部采用统一价格招标（自 1992 起），所有中标者支付相同的（最高接受）收益率；日本在多数 JGB 期限上继续采用多价格（conventional）招标，因此中标者支付各自的投标收益率。这使得日本的 Non-Price I 和 II 机制尤为有价值，因为它为特别参加者提供了以加权平均价进行的接入。

## Comparison with UK GEMM System

英国拥有由英国债务管理办公室（DMO）认可的金边债券做市商（GEMM）：

| Aspect | UK GEMMs | JGB Special Participants |
|---|---|---|
| 管理方 | 英国债务管理办公室（DMO） | 财务省（日本） |
| 机构数量 | 约 15-20  家 GEMM | ~20  家特别参加者 |
| 招标投标义务 | 必需 | 必需 |
| 二级市场义务 | 双向报价 | 双向报价 |
| 通胀挂钩认可 | 对通胀挂钩金边债券单独的指数挂钩 GEMM（IL-GEMM）认可 | 单一层级；JGBi 招标向同一份特别参加者名单开放 |
| 分层 | 一些运营上的区分 | 单一受认可名单 |
| 拆分（strip）与转换（switch）招标接入 | 标准 | 标准 |

英国历来将通胀挂钩 GEMM 认可分离为一个子层级；日本未对 JGBi 采用该细分。

## Why the System Matters

特别参加者制度对以下方面至关重要：

1. **JGB 招标成功**：一个可靠、有资本支撑的交易对手基础，向财务省保证招标将以有序价格成交，降低招标失败的风险。
2. **二级市场流动性**：双向报价义务支撑一个连续的 JGB 市场，供更广泛的机构投资者和日本银行接入。
3. **向财务省的信息流**：定期对话使财务省能够根据市场反馈校准发行计划。
4. **交易商特许经营（franchise）经济学**：特别参加者地位对证券公司是一项有意义的特许经营资产；非特别参加者在 JGB 做市中面临结构性劣势。
5. **外国机构整合**：对外国机构日本子公司开放的会员资格，将全球投资银行整合进 JGB 市场，并支撑跨境 JGB 投资者活动。
