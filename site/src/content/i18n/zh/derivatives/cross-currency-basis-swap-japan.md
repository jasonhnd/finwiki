# 交叉货币基差互换(日本焦点)

## TL;DR

交叉货币基差互换(CCBS)是一种场外(OTC)衍生品,其中两个交易对手以两种不同货币交换浮动利率利息支付,本金也在期初交换并在到期时再次交换。"基差"是在某一边(按惯例为非美元边)报出的利差,用于调整偏离抛补利率平价的部分。日元-美元基差是全球市场中最受关注的外汇融资信号之一,因为它反映了日本投资者(保险公司、银行、资产管理公司)将美元资产对冲回日元所面临的成本,以及全球借款人获取日元融资的成本。

本条目是 [[derivatives/INDEX|derivatives index]] 中 [[derivatives/corporate-fx-hedge-policy-japan-listed|corporate FX hedge policy]] 的长期、融资侧的"表亲",并与 [[derivatives/yen-basis-swap-market|yen basis swap market]] 紧密关联以获得更广义的基差曲线视角。

## 机制

标准的日元-美元 5Y CCBS 运作方式如下:

| 步骤 | 操作 |
|---|---|
| t=0 | 交易对手 A 向交易对手 B 交付日元名义本金;B 按当时即期汇率向 A 交付美元名义本金。 |
| t=0 → T | A 向 B 支付美元浮动利率(通常为 3M 美元 SOFR + 利差);B 向 A 支付日元浮动利率(通常为 3M 日元 TONA / LIBOR 等价 + 基差利差)。 |
| t=T | 本金按原始即期汇率(而非到期汇率)再次交换。 |

基差利差是在非美元浮动指数之上施加的调整,使互换在期初的 NPV 为零。按惯例,日元-美元基差报为日元边支付方所付的利差:-50bp 的"负基差"意味着日元边支付方支付日元浮动利率减去 50bp,这相当于日元融资方需要放弃 50bp 的收益才能取得美元。基差越宽(越负),日元充裕的实体在对冲基础上将日元换成美元的代价就越高。

[[derivatives/yen-basis-swap-market|yen basis swap market]] 页面更详细地涵盖了跨各期限的期限结构。

### 名义本金、抵押品与信用交换

在典型的双边 ISDA 文档化日元-美元 CCBS 中,现金流时间线包括:

- 在 t=0 (即期汇率)进行初始本金交换。
- 每条边的定期利息支付(通常按季,与相关浮动参考利率挂钩)。
- 在 t=T 按原始初始交换汇率(而非到期时的当时即期汇率)进行最终本金再交换。
- 在交易存续期内由 CSA 驱动的变动保证金移动,以对市值(MTM)进行抵押。

由于本金交换使用期初即期汇率,如果交易对手在合约存续期内违约,该交易具有可观的本金重置成本。CSA 抵押化缓解了此风险,但带来了运营和流动性需求。抵押品货币(美元对日元)的选择和再抵押权是具有重大商业意义的条款。

## 为何基差偏离零

抛补利率平价(CIP)指出,在无套利条件下,基差应为零。实践中,在全球金融危机之后,CIP 的成立非常不完美。持续的日元-美元基差(过去十年大部分时间为负)反映了:

1. 来自日本注册的保险公司、银行和资产管理公司对外汇对冲美元资产的结构性美元需求。
2. 巴塞尔 III / LCR / NSFR 之后美元放贷方的资产负债表约束。
3. 季末和年末的资产负债表窗口效应。
4. 由于美国银行交易商的杠杆率上限,套利渠道的美元供给有限。
5. 2008 之后差异化的货币政策体制(日本央行接近零利率而美联储正常化;息差与对冲成本的错配)。

BIS 季度评论系列广泛追踪了 CIP 偏离。该模式在欧元-美元、英镑-美元、澳元-美元基差中重复出现,但日元-美元在结构上是持续最宽的基差之一,因为日本有需要寻找美元回报的庞大日元计价储蓄池。

## 日元-美元与欧元-美元基差动态

两种基差都持续为负(美元需求超过 CIP 隐含的供给),但日元-美元往往比欧元-美元更持久、更宽。这一差异反映了:

- 相对于欧元区的等价物,日本有更大的绝对日元储蓄池在寻求美元资产。
- ECB / 欧元体系操作与日本央行操作在范围上存在差异;参见 [[money-market/boj-open-market-operations|BoJ open market operations]]。
- 各司法管辖区在寿险公司 ALM 框架中对外汇对冲套息的监管处理不同。

两种基差在全球美元融资压力事件期间同向变动(例如 2020 年 3 月新冠),但日元-美元基差是扩大最为剧烈、并需要最大规模央行互换额度提取的那一个。

### 基差的期限结构

日元-美元基差期限结构很少是平的。典型模式:

| 期限 | 典型相对水平 |
|---|---|
| 1W-1M | 最为波动;在本地融资事件上剧烈波动。 |
| 3M | 基准;在大多数市场数据源中报告。 |
| 1Y | 反映滚动远期对冲成本;受寿险公司资金部门关注。 |
| 3Y-5Y | 反映来自企业/主权发行人的结构性融资需求。 |
| 7Y-10Y | 流动性较低;由长期保险/养老金对冲驱动。 |
| 20Y-30Y | 流动性稀薄;多为定制交易。 |

在压力期形态可能倒挂:短端急剧扩大,而长端保持较紧。该模式对寿险公司决定是采用短期滚动对冲(通常便宜,压力时昂贵)还是长期锁定对冲(通常昂贵,压力时稳定)具有意义。

## 年末/季末扩大

CCBS 基差通常在季末、尤其是年末扩大,因为交易商银行在临近报告日时减少资产负债表的投入。该模式为:

| 时段 | 典型的日元-美元 3M 基差表现 |
|---|---|
| 季中 | 围绕当时结构性水平区间交易。 |
| 季度最后 2 周 | 随着交易商撤出而急剧扩大(更负)。 |
| 新季度第一周 | 回弹至结构性水平。 |
| 年末(12 月) | 全年最大的扩大。 |

该模式在 BIS 季度评论研究中有充分记载,并反映在即期与远期隐含基差曲线中。日本银行(尤其是 [[JapanFG/mufg-bank|MUFG Bank]]、[[JapanFG/sumitomo-mitsui-banking-corp|Sumitomo Mitsui Banking Corp]] 和 [[JapanFG/mizuho-bank|Mizuho Bank]])是日元-美元基差市场的主导交易对手,并同时承受这些动态的两侧。

## 央行互换额度

美联储与一小群央行(日本央行、ECB、英格兰银行、瑞士国家银行、加拿大银行)之间的永久/常设外汇互换额度,是美元融资压力的最后手段后盾。机制上:

- 美联储以日元抵押品按与 OIS 挂钩的利率向日本央行借出美元。
- 日本央行通过拍卖(以汇集抵押品进行的美元资金供给操作)向日本银行转贷美元。
- 该机制限制了 CCBS 基差在压力时实际能扩大到多宽,因为日本银行有替代的美元来源。

日本央行-美联储互换额度提取在 2008-2009 危机期间激增,并在 2020 年 3 月至 4 月新冠融资冲击期间再次激增。每次事件之后,提取额都正常化为零,常设机制恢复为后盾状态。

[[money-market/INDEX|money-market index]] 是包含美元资金供给操作的更广义日本央行流动性管理框架的入口。

### 日本央行美元资金供给操作的运营机制

启用时,该操作通常运作如下:

| 步骤 | 操作 |
|---|---|
| 1 | 日本央行通过新闻稿宣布拍卖参数(规模、期限、频率)。 |
| 2 | 合格的日本交易对手(主要是巨型银行和主要地方/信托银行)以汇集的日本国债(JGB)抵押品提交投标。 |
| 3 | 按投标利率分配(或在压力模式下采用固定利率、全额配售)。 |
| 4 | 美元通过代理行安排交付至投标方账户。 |
| 5 | 美元在到期时偿还(在压力期通常为 7天、84天或 3M 期限)。 |

该机制在 2020年 Q1 / Q2 被大量使用,采用 7天、14天、84天操作;在正常化之前总余额峰值超过 2000 亿美元。该机制本身的存在,是日元-美元基差被"锚定"而非在压力时被允许无限扩大的部分原因。

## 企业为美元项目进行的日元融资

一家拥有日元计价债券发行的非日本跨国公司,可以使用 CCBS 将募集资金转换为美元,而不在本金上承担外汇风险。结构:

1. 以日元固定票息发行 5Y 武士债券。
2. 进入 CCBS:支付日元固定(匹配债券),收取美元浮动 + 基差调整。
3. 将美元募集资金用于预定项目。

全包美元融资成本等于日元债券票息互换为美元后、并按当时基差调整的结果。当日元-美元基差大幅为负时,对发行人而言这一路径可能比直接发行美元债券更便宜,同时给予交易商交易对手(通常是日本巨型银行)有利可图的基差敞口。

## 寿险公司美元债券对冲成本

从日本一侧 CCBS 对冲的主导结构性买方是日本寿险业。寿险公司持有大量美元计价债券组合,作为相对于国内日本国债(JGB)的收益率增益(见 [[insurance/japan-life-insurance-alm-overview|Japan life-insurance ALM overview]])。为消除这些持仓的外汇风险,它们使用以下组合:

- 滚动 3M 外汇远期。
- 1Y 外汇远期。
- 长期 CCBS(3Y、5Y、10Y)作为稳定的结构性对冲。

全包对冲后美元收益率总额等于美元债券票息减去对冲成本(远期点数和基差利差)。在日元-美元基差大幅为负的时期(例如 20100 年代后期和 2022-2023),一只 10Y 美国国债的全包对冲后收益率相对于 JGB 可能缩减至接近零甚至转负,这促使寿险公司重新配置回 JGB 或未对冲的美元债券。这一资金流转移是全球固定收益市场中最受关注的动态之一。

### 数值示例

一个程式化示例使机制更具体。考虑一只收益率为 4.5% 的 10Y 美国国债和一只收益率为 1.0% 的 10Y JGB。朴素的套息为 350bp。现在叠加对冲成本:

| 组成部分 | 近似值(示例性) |
|---|---|
| 10Y 美国国债收益率 | 4.5% |
| 隐含 3M 滚动外汇远期对冲成本 | ~2.5-3.0%(由短端利率差驱动) |
| 加 5Y 日元-美元 CCBS 基差利差 | ~30-50bp(负) |
| 全包对冲后美元收益率 | ~1.2-1.7% |
| 10Y JGB 收益率 | ~1.0% |
| 相对 JGB 的净增益 | ~20-70bp |

确切数字随时间大幅波动,但模式是一致的:当日元-美元基差宽且短端利率差大时,全包对冲后美元收益率向 JGB 收益率收敛,为持有对冲美元债券的运营复杂性和信用敞口留下的额外回报很少。这是寿险公司 ALM 委员会反复出现的决策点。

## 市场参与者与资金流分类

日元-美元 CCBS 市场在结构上是失衡的。典型的资金流分类:

| 资金流方向 | 典型参与者 | 头寸 |
|---|---|---|
| 支付美元浮动,收取日元浮动 + 基差 | 进行对冲的日本注册美元资产持有者 | 净美元借款人(合成) |
| 支付日元浮动 + 基差,收取美元浮动 | 筹集合成日元的非日本企业 | 净日元借款人(合成) |
| 交易商(巨型银行/全球银行) | 在两侧之间做中介 | 在风险限额下持有残余基差敞口 |
| 投机者(对冲基金) | 持有基差收窄或扩大的观点 | 压力期占比小,平静期占比较大 |

结构性失衡——日本注册的美元资产对冲需求远多于非日本的日元借款需求——解释了为何基差持续为负,而非回归 CIP 零值。

### 主要交易商特许经营

交易商参与高度集中:

- [[JapanFG/mufg-bank|MUFG Bank]]:可观的资产负债表和日本企业特许经营;在日元-美元基差市场中长期担任 PB / 交易商。
- [[JapanFG/sumitomo-mitsui-banking-corp|Sumitomo Mitsui Banking Corp]]:类似的概况;SMFG 集团范围内的跨业务协同。
- [[JapanFG/mizuho-bank|Mizuho Bank]]:强大的企业/金融机构交易特许经营;深度的 JGB 抵押品整合。
- 花旗、摩根大通、高盛、摩根士丹利:主导的外资交易商;在巨型银行资产负债表受约束的压力期尤为活跃。
- 野村、大和:也参与,尤其是在寿险相关资金流和较长期限结构中。

这种集中创造了定价分散的机会:运营多交易商面板的日本企业或保险公司,可通过竞争性地获取报价,在大额名义交易上捕获数个基点的定价改善。

## 监管与文档框架

CCBS 交易在 ISDA 主协议下以 CSA 抵押化进行文档化。具体特征:

- ISDA 2002 主协议是典型的参考文件;一些日本交易对手也使用日本法的等价版本。
- CSA 条款规定抵押品货币(日元对美元)、合格抵押品类型、阈值、最低转移金额和估值方法。
- 2008 之后的改革将合格的标准化 CCBS 推向中央清算(CME、LCH SwapClear),不过日本 CCBS 清算的采纳比美元 IRS 慢。
- 日本央行和 FSA 通过 FIEA 衍生品报告规则下的交易报告库报告来监测系统性敞口。
- 巴塞尔 III 下的资本处理(交易对手信用风险、CVA 费用)影响交易商定价,尤其是对非清算交易。

经济效应是,双边非清算 CCBS 定价带有内嵌的资本费用,使基差比在完全清算、完全抵押化环境下的水平更宽。
