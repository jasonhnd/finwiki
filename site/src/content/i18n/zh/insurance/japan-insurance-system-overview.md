# 日本保险体系概览

## TL;DR

日本的**保险体系**横跨五大板块：**寿险**、**非寿险**、**共济 / kyosai**、**再保险**，以及**公私合营地震保险机制**。保险公司依据《保险业法》运营，由金融厅（FSA）监管，资本充足性则通过与 IAIS Insurance Capital Standard（ICS）接轨、分阶段引入的经济价值偿付能力比率（ESR）制度来衡量。

市场高度集中。四大寿险公司（日本生命、第一生命、住友生命、明治安田生命）主导寿险保费，三大非寿险集团（东京海上、MS&AD、Sompo）主导非寿险保费，而 JA Kyosai 是最大的合作社保险机构。地震保险机制则是由日本地震再保险（JER）运营、并由政府兜底的独特公私合营安排。

本条目是连接 [[insurance/INDEX|保险领域]] 的跨板块生态锚点，并衔接到实体页面锚点 [[JapanFG/INDEX|JapanFG]]。

| Sub-segment | Examples | Anchor |
|---|---|---|

### 5. 地震保险公私合营机制

地震保险公私合营机制是日本保险体系中的独特结构特征：

| 层级 | 功能 |
|---|---|
| 家庭地震保险 | 必须与住宅火险捆绑销售。 |
| 原保险公司 | 非寿险公司承保保单。 |
| 日本地震再保险（JER） | 家庭地震险的法定垄断再保险人；承接所有原保险公司的风险。 |
| 政府兜底 | 政府依据法律在既定损失阈值之上为 JER 提供支持。 |

详见 [[insurance/earthquake-insurance-public-private-scheme]] 与 [[non-life-insurers/japan-earthquake-reinsurance]]。

## 监管框架

| 层级 | 细节 |
|---|---|
| 《保险业法》 | 保险公司牌照、分销、资本、治理与行为规范的法定框架。 |
| FSA 监管 | 牌照审批、监管检查、市场行为审查。 |
| 经济价值偿付能力比率（ESR） | 与 IAIS ICS 原则接轨的资本充足制度。见 [[insurance/economic-value-based-solvency]], [[insurance/esr-economic-value-solvency]]。 |
| 全球偿付能力比较 | FSA ESR / IAIS ICS / EU Solvency II / US NAIC RBC。见 [[insurance/global-solvency-framework-comparison-matrix]]。 |
| IAIS / IAIG | 国际活跃保险集团认定；ICS 2.0  报告。见 [[insurance/japan-iaig-ics-mapping]]。 |
| 分销许可 | 保险代理人 / 经纪人在 FSA 下注册，并由 JIIA、JLIA 自律。见 [[insurance/insurance-agency-and-brokerage-japan]]。 |
| 银保 | 银行柜面销售保险，受 FSA 监管。见 [[insurance/bancassurance-economics-japan]], [[insurance/japan-bancassurance-distribution-overlay-matrix]]。 |
| 共济边界 | 在 MAFF / MHLW 下按合作社法系法规运行，并有单独监管制度。见 [[insurance/japan-kyosai-vs-fsa-insurance-perimeter-matrix]]。 |
| 保单持有人保护 | 日本生命保险保单持有人保护机构（LIPPC）+ 一般保险保单持有人保护机构（GIPPC）。 |

## 市场参与者

| 类别 | 示例 |
|---|---|
| 寿险公司（FSA 持牌） | 约 40  家，覆盖四大、邮政、线上、集团及外资玩家。 |
| 非寿险公司（FSA 持牌） | 约 30+ 家，覆盖三大、直销、专业、外资、海运 / P&I。 |
| 共济运营方 | JA-Kyosai、Kokumin Kyosai Coop、Coop Kyosai、公务员共济、职业共济。 |
| 再保险人 | Toa Re、Munich Re、Swiss Re、SCOR、Hannover Re、RGA、Lloyd's Japan、captives。 |
| 分销渠道 | 代理销售队伍、银行、经纪、线上直销、嵌入式 / 团体渠道。 |
| 行业协会 | 生命保险协会、一般保险协会。 |
| 自律组织 | JIIA、JLIA。 |
| 监管者 | 《保险业法》下的 FSA。 |
| 保单持有人保护 | LIPPC（寿险）、GIPPC（一般保险）。 |

| 连接 | 机制 |

| 寿险利率衍生品对冲 | 大型寿险公司是日元 IRS receive-fixed 头寸的最大使用者之一。见 [[derivatives/japan-interest-rate-derivatives-overview]], [[insurance/japan-life-insurance-alm-overview]]。 |

| 银保 | 银行通过 FSA 监管渠道分销寿险 / 非寿险。见 [[insurance/bancassurance-economics-japan]]。 |
