# 日本保险体系概览

## TL;DR

日本的 **保险体系** 横跨五大板块：**寿险** (生命保険)、**财险 / 非寿险** (損害保険)、**共济 (mutual aid / kyosai)** (共済)、**再保险** (再保険)，以及 **官民地震保险方案** (地震保険)。保险公司在《保险业法》(保険業法) 之下经营，受金融厅 (FSA) 监管，资本充足率则在经济价值型偿付能力比率 (ESR) 制度下衡量——该制度已开始朝着与 IAIS 保险资本标准 (ICS) 对齐的方向分阶段引入。

市场高度集中。四家寿险公司（日本生命、第一、住友、明治安田）主导寿险保费，三大财险集团（东京海上、MS&AD、Sompo）主导财险保费，JA 共济则是最大的合作社保险者。地震保险方案是由日本地震再保险 (JER) 在政府支持下运营的独特官民合作。

本词条是将 [[insurance/INDEX|insurance domain]] 联结在一起、并桥接至各实体页面 [[insurance/INDEX|insurance INDEX]] 的跨板块生态系统锚点。

### 3. 共济 (Mutual aid / Kyosai) (共済)

| 子板块 | 示例 | 锚点 |
|---|---|---|
| JA 共济（农业合作社） | 全共连（全国联合会） | [[insurance/ja-kyosai-zenkyoren-overview]], [[non-life-insurers/ja-kyosairen]] |
| 生协系共济（消费生活合作社） | 国民共济 coop（旧 全劳济）, COOP 共济 | [[insurance/cooperative-insurance-system-japan]] |
| 公务员共济 | 各类国家 / 地方公务员共济 | [[insurance/cooperative-insurance-system-japan]] |
| 职业别共济 | 各类职业团体共济（如教职员、医疗从业者） | [[insurance/cooperative-insurance-system-japan]] |

共济在合作社法的法令而非《保险业法》之下运营，并接受单独监管（通常 JA 共济为 MAFF，生协系共济为 MHLW）。境界叠加见 [[insurance/japan-kyosai-vs-fsa-insurance-perimeter-matrix]]。

再保险使原保险者得以转移集中的风险（尤其是地震、台风、洪水）。大额自然灾害承保能力大多来自外国再保险者与劳合社。

### 5. 地震保险官民方案

地震保险官民方案是一项独特的结构性特征：

| 层级 | 功能 |
|---|---|
| 家庭地震保险 | 住宅财产须与火灾保险捆绑。 |
| 原保险者 | 由财险公司承保保单。 |
| 日本地震再保险 (JER) | 家庭地震的法定垄断再保险者；为原保险者的全部敞口再保险。 |
| 政府兜底 | 政府依法令在既定损失阈值之上为 JER 兜底。 |

详情见 [[insurance/earthquake-insurance-public-private-scheme]] 与 [[non-life-insurers/japan-earthquake-reinsurance]]。

## 监管框架

| 层级 | 详情 |
|---|---|
| 保险业法 (Insurance Business Act) | 关于保险者牌照、分销、资本、治理与行为的法定框架。 |
| FSA 监管 | 牌照、监管检查、市场行为审查。 |
| 经济价值型偿付能力比率 (ESR) | 与 IAIS ICS 原则对齐的资本充足制度。见 [[insurance/economic-value-based-solvency]], [[insurance/esr-economic-value-solvency]]。 |
| 全球偿付能力比较 | FSA ESR / IAIS ICS / 欧盟 Solvency II / 美国 NAIC RBC。见 [[insurance/global-solvency-framework-comparison-matrix]]。 |
| IAIS / IAIG | 国际活跃保险集团 (IAIG) 指定；ICS 2.0 报告。见 [[insurance/japan-iaig-ics-mapping]]。 |
| 分销牌照 | FSA 之下的保险代理店 / 经纪人注册，辅以 JIIA（财险）与 JLIA（寿险）自律。见 [[insurance/insurance-agency-and-brokerage-japan]]。 |
| 银保 | FSA 监管下的保险银行柜面销售。见 [[insurance/bancassurance-economics-japan]], [[insurance/japan-bancassurance-distribution-overlay-matrix]]。 |
| 共济境界 | MAFF / MHLW 之下的合作社法法令，伴有单独的监管制度。见 [[insurance/japan-kyosai-vs-fsa-insurance-perimeter-matrix]]。 |
| 投保人保护 | 日本生命保险投保人保护机构 (LIPPC) + 损害保险投保人保护机构 (GIPPC)。 |

## 市场参与者

| 类别 | 示例 |
|---|---|
| 寿险公司（FSA 持牌） | 约 40 家持牌寿险公司，横跨四大、邮政、在线、团体、外资关联。 |
| 财险公司（FSA 持牌） | 约 30家以上持牌财险公司，横跨三大、直销、专项、外资关联、海上 / P&I。 |
| 共济经营者 | JA 共济, 国民共济 coop, COOP 共济, 公务员共济, 职业别共济。 |
| 再保险者 | 东亚再保险（国内）, 慕尼黑再保险, 瑞士再保险, SCOR, 汉诺威再保险, RGA, 劳合社日本, 自保。 |
| 分销渠道 | 代理店制营销队伍、银行（银保）、经纪人、在线直销、嵌入式 / 团体渠道。 |
| 行业协会 | 生命保险协会（生保协会）, 日本损害保险协会（损保协会）。 |
| 自律机构 | JIIA（财险）, JLIA（寿险）。 |
| 监管者 | 《保险业法》之下的 FSA。 |
| 投保人保护 | LIPPC（寿险）, GIPPC（财险）。 |

## 跨板块联系

| 联系 | 机制 |
|---|---|
| 寿险者利率衍生品对冲 | 大型寿险者是日元 IRS 最大级的固定收取（receive-fixed）使用者之一。见 [[derivatives/japan-interest-rate-derivatives-overview]], [[insurance/japan-life-insurance-alm-overview]]。 |
| 财险自然灾害再保险 | 财险三大在自然灾害合约方案下向外国再保险者分出。见 [[insurance/natcat-reinsurance-japan]]。 |
| 银保 | 银行在 FSA 监管渠道下分销寿险 / 财险。见 [[insurance/bancassurance-economics-japan]]。 |
| FG 系寿险子公司 | 大型银行 / FG 集团经营自有寿险子公司（如第一 Frontier、三井住友海上爱和谊生命）。 |
| 保险 / 资产管理 | 大型寿险者经营大规模资产管理业务；财险三大经营全球资产管理部门（尤其 Sompo 经由投资子公司）。 |
| 保险 / 全球 FG | 财险三大集团经营全球保险子公司（Tokio Marine HCC, MSI USA, Sompo Internazionale）。 |
