# ESR（经济偿付能力比率）


## Wiki route

This entry sits under [[insurance/INDEX|insurance index]]. Read it against [[insurance/economic-value-based-solvency|Economic value-based solvency regulation]] for peer / contrast context and [[JapanFG/INDEX|JapanFG index]] for the broader system / regulatory boundary.

## TL;DR

ESR 是用于表示在保险公司层面上，可用经济资本是否足以覆盖所需风险资本的经济偿付能力比率。其具体公式和命名会因公司和监管阶段而异，因此 FinWiki 应将 ESR 视为资本充足性信号，而不是单一通用的会计数字。

关于制度本身请看 [[insurance/economic-value-based-solvency|economic value-based solvency regulation]]，本页用于公司间比较。

## Reading Rule

| Question | How to read ESR |
|---|---|
| 保险公司在经济价值基础上资本是否充足？ | ESR 越高通常意味着经济资本缓冲越充足，但前提是先确认公司的定义和过渡假设。 |
| 管理层是否受约束？ | ESR 下滑会对分红、回购、产品定价、ALM、再保险和并购意愿形成压力。 |
| 不同公司之间可直接比较吗？ | 只能部分比较。比较标题百分比前，先核对方法说明。 |
| ESR 与偿付能力边际比率相同吗？ | 不同。偿付能力边际比率是旧监管指标；ESR 是经济价值视角。两者在过渡期可能并存。 |

## JapanFG Relevance

- [[life-insurers/dai-ichi-life]]：围绕集团监管 ESR 披露的资本充足性与治理变化，与上市股权估值直接相关。
- [[life-insurers/nippon-life]]：日本生命在业绩资料中使用经济价值偿付能力 / ESR；由于其为相互会社，投资者关注点更偏向资本配置与保单持有人回报，而不只是股份回购。
- [[non-life-insurers/tokio-marine]]、[[non-life-insurers/msad]]、[[non-life-insurers/sompo]]：财险 ESR 需要结合巨灾风险、再保险定价、海外 specialty exposure 和保单持有人保护一起解读。

## Analytical Uses

1. **资本政策**：ESR 是分红、回购、次级债和并购的门槛。
2. **ALM**：拥有长期负债的寿险公司对利率、保证收益产品和久期错配高度敏感。
3. **产品战略**：保障型产品、储蓄型产品、外币产品和年金占用资本的方式不同。
4. **再保险**：再保险可降低风险资本，但会带来交易对手、成本与可得性约束。

## Cautions

- 标题 ESR 的改善，可能来自市场变动、去风险化、新业务经济性或方法学变化；关键在于驱动因素。
- 公司披露的 ESR 未必与最终监管 ESR 完全一致。
- 比率很高也未必自动是好事，如果这意味着资本未被有效使用、增长疲弱或过度去风险化。
