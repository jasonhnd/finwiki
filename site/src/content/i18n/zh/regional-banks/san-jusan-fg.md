# 三十三 FG（San ju San Financial Group）

## Wiki 路由

本条目位于 [[regional-banks/INDEX|regional-banks INDEX]] 之下。请与 [[regional-banks/fukuoka-fg|ふくおか FG (FFG)]] 对照阅读以了解同业 / 对比背景，与 [[banking/INDEX|banking index]] 对照阅读以了解更广泛的系统 / 监管边界。

## TL;DR

三重县地盘的地方银行 FG。**2018-04-02 三重银行 + 第三银行 经营整合**而成立，**2021-05-01 两行合并 → 三十三银行**诞生。社名"三十三"取自"**三（mie / 三重）+ 三（third / 第三）**"两行的"三"这一双关含义（英文名：San ju San = 33）。位于中京经济圈靠近关西的边界，与县内 No.1  的 [[regional-banks/hyakugo-bank]]（百五银行）竞争。^[extracted]

## 1. 公司概要

**正式名称**：株式会社三十三金融集团
**英文名**：San ju San Financial Group, Inc.
**证券代码**：东证 PRIME 7322
**成立**：2018-04-02（三重银行 + 第三银行 经营整合，以共同持股公司方式成立）
**总部**：三重县四日市市
**主要子公司**：三十三银行（2021-05-01 三重银 + 第三银 合并成立）^[extracted]

### 主要子公司・关联公司

```
三十三 FG（持股公司・东证 PRIME 7322）
  ├── 三十三银行（100%）── 核心子银行，2021-05-01 合并成立
  │     ├── 三重县内营业基础（四日市・津・松阪・伊势）
  │     └── 名古屋圈・中京经济圈贷款
  └── 集团公司（租赁・卡・咨询等）
```

### 旧 第三银行系

- **1895**：第百十二国立银行 设立（旧 第三银行的源流）
- 历经战后重组而成为 **第三银行** 这一地区金融机构
- 以三重县南部（松阪・伊势）为地盘

### 旧 三重银行系

- **1928**：三重银行 设立（旧 四日市银行 + 桑名银行 等 整合）
- 以三重县北部（四日市・桑名）工业地带为地盘
- 与四日市联合企业（コンビナート）相关企业的交易基础

### 整合

- **2018-04-02**：三重银 + 第三银 → 三十三 FG 成立（共同持股公司方式）^[extracted]
- **2021-05-01**：三重银 + 第三银 → **三十三银行** 合并（子银行整合完成）
- **2024**：中期经营计划制定

### 地理定位

- **三重县地盘**：位于中京经济圈**靠近关西边界**的独特定位
- **二面性**：名古屋圈（中京）的经济圈波及 + 与关西圈的文化邻近性
- **边界地方银行定位**：中京（爱知县地方银行势力）vs 关西（近畿系）之间的中间地带

### 产业基础

- **观光**：伊势神宫（年参拜客 ~1000 万 人）+ 鸟羽・志摩度假区
- **水产**：伊势龙虾・珍珠养殖等本地产业
- **工业**：四日市联合企业（石油化学・电子部件）

### 整合效应

- **店铺网重叠整理**：三重银 + 第三银的重叠店铺整合
- **系统整合**：核心系统・营业店系统的共通化

## 5. 监管・政策

- **主管**：金融厅（FSA）+ 东海财务局
- **持股公司监管**：银行法 第 52  条之 17
- **近期政策议题**：
  - 2024〜 日本央行政策利率正常化 → 地方银行收益利差改善红利
  - 地方创生・人口减少下的地方银行重组压力
  - 中京圈 vs 关西圈的边界定位的战略选择
