# 千叶银行 (Chiba Bank)

## 维基导览

本词条归属于 [[regional-banks/INDEX|regional-banks INDEX]]。请对照 [[regional-banks/gunma-bank|群馬銀行 (The Gunma Bank)]] 阅读以获取同业／对比背景，并对照 [[banking/INDEX|banking index]] 了解更宏观的体系／监管边界。

## TL;DR

以千叶县为地盘的地方银行单独最大级别（合并存款 **16.3 兆円规模**·合并总资产 **约 21.6 兆円**（2025-03 期）/ 千叶县份额 No.1）。1943-03 由千叶合同银行·小见川农商银行·第九十八银行合并成立，同月改名千叶银行。1949 于东证上市。**未实施控股公司化的单独银行**，在确保经营灵活性的同时，以与地方银行金融集团（[[regional-banks/concordia-fg]]·[[regional-banks/fukuoka-fg]]·[[regional-banks/hokuhoku-fg]] 等）并列的规模占据重要地位。**2015 主导设立 TSUBASA 联盟** ★，通过 11 行协作（系统共同化·海外支持）在回避控股化的同时获得规模效益。2023-12 开设新加坡分行，正式推进海外战略。^[extracted]

### 前身·整合历史

- **1943-03-31** 千叶合同银行·小见川农商银行·第九十八银行合并，新设 **千叶银行**（战时整合·千叶县内地方银行整合）
- 1944 吸收千叶储蓄银行，受让野田商诱银行的营业
- **1949** 在东京证券交易所上市
- 战后高速增长期〜80 年代：在千叶县·首都圈东部确立营业基础
- 泡沫期：不良债权处理·店铺网络优化

### TSUBASA 联盟主导战略 ★

**2015 设立**，由千叶银行主导的 11 行联盟（**在回避地方银行控股化的同时获得规模效益**的替代模式）：

| 银行 | 地盘 |
|---|---|
| **千叶银行**（主导）| 千叶县 |
| 北洋银行 | 北海道 |
| 武藏野银行 | 埼玉县 |
| 滋贺银行 | 滋贺县 |
| 第四北越银行 | 新潟县（[[regional-banks/dai-shi-hokuetsu-fg|hokuetsu-fg]] 旗下）|
| 中国银行 | 冈山县 |
| 伊予银行 | 爱媛县 |
| 东邦银行 | 福岛县 |
| 北国银行 | 石川县 |
| 京都银行 | 京都府 |
| 群马银行 | 群马县 |

**TSUBASA 的战略意义**：
- **系统共同化**：通过核心系统·周边系统的共同化削减成本
- **海外支持**：相互利用加盟行的当地据点（新加坡等）
- **产品共同开发**：投信·保险等共同产品
