# 大垣共立銀行 (OKB)

## Wiki route

This entry sits under [[regional-banks/INDEX|regional-banks INDEX]]. Read it against [[regional-banks/fukuoka-fg|ふくおか FG (FFG)]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

总店设于岐阜县大垣市的东海地方地方银行。1896-03  创业，东证 PRIME 8361  上市，未实施控股公司化的 **单独银行体制**。以岐阜县西浓地方为地盘，将爱知县西北部・三重县北部的 **中京经济圈北缘** 作为主要营业圈。VI 简称「**OKB**」（1991 年 引入）。以「**快乐的银行**」为理念，以得来速（drive-through）网点・粉色基调的店内装饰・移动网点车等独特举措著称，并标榜「**日本第一的女性活跃银行**」。在县内份额上与 [[regional-banks/juroku-fg]] 并列为两大银行之一。^[extracted]

### 集团结构

```
大垣共立银行 (单独银行・上市 8361)
  ├── OKB 信用保证 (住房贷款・消费贷款保证)
  ├── OKB ビジネスサービス (事务代办・人才派遣)
  ├── OKB 卡 (信用卡・JCB 合作)
  ├── OKB リサーチセンター (区域智库)
  ├── OKB リース (租赁)
  └── OKB ベンチャーキャピタル / 投资基金系子公司
```

> 未实施控股公司化是其特征。在多数大型地银 FG 化的背景下，OKB 作为 **单独银行** 维持 PRIME 上市。^[extracted]

### 「快乐的银行」品牌塑造

- **得来速支店**: 可坐在车上完成存款・取款・转账。在地方银行业界是先驱性案例 ^[extracted]
- **粉色基调的店内**: 有意解构「银行很死板」的形象。吸纳女性客户战略
- **移动网点车**: 面向过疏地区・老年人的巡回服务
- **OKB レンガ通り (大垣总店周边)**: 配设文化设施・礼堂，使其成为区域社区据点

### 「日本第一的女性活跃银行」

- 提升女性管理职比率・育儿支援制度・面向女性的金融产品 ^[extracted]

### 竞争关系

- **[[regional-banks/juroku-fg]] (十六 FG・岐阜总店)** ★最大竞争对手: 在岐阜县全域 + 爱知北部重叠。岐阜县内十六偏向县北 / 岐阜市，OKB 偏向西浓，存在分工，但存款・融资均为直接竞争 ^[extracted]
- **[[regional-banks/san-jusan-fg]] (三十三 FG・三重)**: 在三重县北部 (桑名・四日市) 重叠
- **名古屋银行 (爱知)**: 在爱知县西北部重叠
- **大型银行 ([[megabanks/mufg]] / [[megabanks/smfg]] / [[megabanks/mizuho-fg]])**: 在名古屋市内大企业融资上竞争
- **信金・JA**: 西浓地方的小规模事业者・农业金融

### 地理定位

- 以 **岐阜县西浓地方** (大垣市・养老町・关原町・揖斐川町・海津市 等) 为地盘 ^[extracted]
- 向 **爱知县西北部 (一宫・稻沢・津岛 等)** 跨境扩张
- 也向 **三重县北部 (桑名・员弁 等)** 展开
- 「**中京经济圈北缘**」定位 — 名古屋经济圈通达 + 观光・农业 + 汽车零部件供应链融资

### 观光融资

- 来自中京圈的观光客流入 + 入境游复苏带来观光相关融资需求

### 中期经营计划 (2024〜)

- 强化可持续发展・ESG
- DX・网点数字化
- 女性活跃・人力资本投资

## 4. 监管・政策

- **主管**: 金融厅 (FSA) ・东海财务局
- **加盟协会**: 全国地方银行协会 (地银协)
- **近期政策论点**:
  - 2024〜 日银政策利率正常化 → 地银利率利差改善期待
  - 2025〜 地银重组压力 (地银之间的经营整合・FG 化) — OKB 维持单独银行
  - 中京经济圈汽车产业结构转型 (EV 转向) 影响本地中小企业融资

### 行长・经营

- **土屋嶢 (つちや たかし)**: 长期担任经营高层，推进「OKB」VI 引入 (1991)・「快乐的银行」理念的中兴之祖般人物 ^[extracted]
