# 青空银行 (Aozora Bank)

## Wiki route

本条目归属于 [[regional-banks/INDEX|regional-banks INDEX]]。请对照 [[regional-banks/sbi-shinsei-bank|SBI 新生銀行]] 了解同业／对比背景，对照 [[banking/INDEX|banking index]] 了解更广泛的系统／监管边界。

## TL;DR

旧 **日本债券信用银行（日债银, NCB）** 系谱的中坚独立银行。是经历过 **1998-12 经营破产 → 特别公的管理（暂时国有化）** 的邦银重生史代表案例。2000-09 出售给软银 + 欧力士 + 东京海上联合 → 2001-01改名「青空银行」 → 2003 美国 PE **瑟伯勒斯（Cerberus Capital Management）** 成为主要股东 → 2006-11 东证一部再上市 → 瑟伯勒斯分阶段出售完成。近期最大论点是 **2023-2024 期 美国 CRE（商业用不动产）融资巨额损失** → 2024-05 全年度赤字公布 → 业绩下方修正・经营战略重构阶段。既不属于大型系列也不属于地方银行系列的「中坚独立银行」模式 ^[extracted]。

### 「中坚独立银行」模式

- **既不属于大型系列也不属于地方银行系列的独立定位** ^[extracted]
- 个人零售以 BANK 网络支店为中心，店铺网络最小化

### 美国 CRE 损失问题（近期最大论点）

- **2018〜2023 面向美国办公 CRE 的融资扩大**（国内低利率环境下的收益率追逐）^[extracted]
- 2022〜2024 美国办公需求结构变化（远程定型）+ Fed 加息 → 担保价值急跌
- **2024-05 全年度赤字 约 320 億円公布 → 计提史上最大级别损失** ^[extracted]

### 过去的整合探讨历程

- **2010 与旧 新生银行（现 SBI 新生）的整合探讨 → 撤回** ^[extracted]
  - 两行均共有公的资金注入历程 + 海外 PE 退出问题 → 曾被期待协同但告吹
  - 旧 新生银行后来由 [[megabanks/sbi-hd]] 进行 TOB（2021-12〜2022-02）→ 完全子公司化（2023）

### 战略定位

- 与 [[megabanks/sbi-hd]] 的 SBI 新生银行模式（PE 退出后的战略伙伴化）的对比轴

## 4. 监管・政策

- **主管**：金融厅（FSA）
- **业态的历史**：旧长期信用银行 3 行（日本兴业银行 → 瑞穗系 [[megabanks/mizuho-fg]]／日本长期信用银行 → 旧新生银行 → [[megabanks/sbi-hd]]／日本债券信用银行 → 青空）中唯一的独立存续行
- **存款保险机构**：1998-12 破产时特别公的管理的执行主体
- **近期政策论点**：
  - 日银政策利率正常化（2024〜）→ 国内利率边际改善红利
  - 瑟伯勒斯退出后的中坚独立银行治理框架
