# 京都金融集团

## Wiki 路径

本条目位于 [[JapanFG/INDEX|JapanFG index]] 之下。可与 [[regional-banks/ikeda-senshu-hd|池田泉州ホールディングス]] 对读以理解同业和对比关系，并与 [[banking/INDEX|banking index]] 对读以把握更广的制度 / 监管边界。

## TL;DR

京都金融集团是以京都府为地盘的 **关西名门地方银行 FG**。证券代码为 **东证 PRIME 5844**。集团于 **2023-10-02** 通过京都银行 **单独股份转移** 实现控股公司化而成立，是地方银行中较晚控股公司化的例子。**京都银行维持京都府内份额 No.1**，并因与总部位于京都的任天堂、京瓷、村田制作所、日本电产（现 Nidec）等 **全球优质企业** 的交易和 **政策持有股票** 而知名。在关西，它与 [[resona-hd]]（关西未来银行）、[[ikeda-senshu-hd]]、滋贺银行、南都银行、纪阳银行等直接竞争。相对于 megabank 3 行（[[mufg]] / [[smfg]] / [[mizuho-fg]]），京都 FG 依靠京都地盘和上市公司交叉持股带来的 **资金实力** 形成独自存在感。^[extracted]

## 1. 公司概要

**正式名称**：株式会社京都金融集团 ^[extracted]  
**英文名**：Kyoto Financial Group, Inc.  
**证券代码**：东证 PRIME **5844**（2023-10-02 新上市，取代旧京都银行 8369  上市）^[extracted]  
**设立**：**2023-10-02**（京都银行通过单独股份转移进行控股公司化）^[extracted]  
**总部**：京都市下京区乌丸通松原上る药师前町 700（与京都银行总店并设）^[extracted]

### 合并历程与前身

- 旧 **京都银行**（京都府地盘）
  - **1941-10-01** 京都银行设立（战时统制下京都府地方银行整合，来源包括 **京都中央信用组合 + 西阵银行 + 京北银行 + 京都银行（旧）** 等整合）^[extracted]
  - 战后从京都市中心部扩展至京都府全域及关西广域。
  - **1969** 于大阪证券交易所、东京证券交易所第二部上市。
  - **1971** 于东证一部上市（旧证券代码 8369）^[extracted]。
  - 与任天堂、京瓷、村田制作所、日本电产 / Nidec 等京都总部全球企业建立长期交易与 **股票交叉持有** 关系，巨额 **政策持有股票未实现收益** 成为同行代名词。
- **京都 FG 发足**：**2023-10-02**（京都银行单独股份转移、HD 化；为地方银行中控股公司化较晚的 1  个例）^[extracted]。

### 战略轴

- **京都府份额 No.1  地方银行**：在京都府内存款、贷款份额均压倒性 No.1；利用本地密着与京都品牌 ^[extracted]。
- **政策持有股票未实现收益战略** ★：通过与任天堂、京瓷、村田制作所、日本电产（Nidec）等京都总部世界企业的 **长期股票交叉持有** 积累了 **数千亿日元规模未实现收益**，这是同行最大财务特征，也是资本充实度和信用力来源 ^[extracted]。
- **关西广域展开**：以京都府为地盘，同时向大阪、滋贺、兵库、奈良、爱知部分地区外溢，形成关西广域中坚地方银行位置 ^[extracted]。
- **「京银 Direct Finance」DX**：数字投资、面向法人的网上银行、fintech 合作 ^[extracted]。
- **京都品牌利用**：将观光、传统产业、京都总部企业关系作为差异化资产 ^[extracted]。
- **中期经营计划**：第1次中计（2023-03期〜2026-03期，净利润 300 億円目标）提前达成，并在 2024-11-14 公布新战略目标。下一期第2次中计（2026-04〜2029-03）设定归母利润 600 億円（2029-03）、ROE 8% 以上（2030 年代前半）等大幅更具挑战性的目标。

### 与京都总部世界企业的关系 ★

- **任天堂**（京都市南区）：京都银行作为稳定股东长期持有，股价上升时产生巨额未实现收益 ^[extracted]。
- **京瓷**（京都市伏见区）：同样的长期交叉持股 ^[extracted]。
- **村田制作所**（京都府长冈京市）：同样的长期交叉持股 ^[extracted]。
- **日本电产 / Nidec**（京都市南区）：同样的长期交叉持股 ^[extracted]。
- 这些 **政策持有股票未实现收益** 支撑京都银行的 **资本充实度** 和 **信用力**，形成地方银行中独特的资产负债表结构 ★ ^[extracted]。

### 整合协同与课题

- **收益多元化**：贷款存款利息收入 + 股票分红 / 出售收益两根支柱；股票出售收益年度波动大是课题 ^[extracted]。
- **政策持有股票缩减压力**：如何回应金融厅、东证的 **政策持有股票缩减要求** 是课题，包括说明「交叉持股」合理性与调整缩减节奏 ^[extracted]。

### B2C 品牌

- **京都银行 =「京银」**：在京都府民中的认知度压倒性，在京阪神广域也有一定认知 ^[extracted]。
- **「京都的银行」** 这一品牌资产：与观光、传统产业、世界企业的连接使地方银行品牌差异化 ^[extracted]。

## 4. 监管与政策

- **主管**：金融厅（FSA）
- **控股公司监管**：银行法第 52 条之 17（银行控股公司）^[extracted]
- **近期政策论点**：
  - **2024〜 日银政策利率正常化**：国内贷款利差改善 bonus（地方银行受益大，升息带来收益改善预期）。
  - **政策持有股票缩减要求**：金融厅、东证 stewardship code / corporate governance code 合规，以及政策持有股票缩减节奏，是市场关注点 ★ ^[extracted]。
  - **地银重组**：京都 FG 是 2023-10-02 由京都银行单独股份转移成立的控股公司，尚未与其他银行经营整合，以单独集团运营。
  - **2025〜 营业网点削减**：地方银行标准重组路线，瘦身网点网络并集中功能。
  - **面向中小企业的事业性评价融资**：推动不依赖担保和保证的融资。
  - **大阪・关西世博（2025-04〜10）**：关西经济圈活化预期 → 对京都观光、服务、建设业产生 follow-tail 效果。
