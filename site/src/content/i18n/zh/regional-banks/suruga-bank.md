# 骏河银行

## Wiki 路径

本条目位于 [[regional-banks/INDEX|regional-banks INDEX]] 之下。应将其与 [[regional-banks/fukuoka-fg|ふくおか FG (FFG)]] 对照阅读以了解同行 / 对比背景，并与 [[banking/INDEX|banking index]] 对照以了解更广泛的体系 / 监管边界。

## TL;DR

以静冈县东部（沼津）为地盘的中坚地银。**1895 年作为根方银行创业，有 130 年的历史**。1990 年年代～2010 年年代曾作为"个人贷款特化型地银的优等生"以高 ROE 自豪，但 **2018 年面向合租公寓『南瓜马车』的不正当融资事件** 使评价骤变。在 1 兆円超的面向个人投资者融资中系统性地进行了文件篡改・审查不正，导致金融厅业务改善命令・经营层总辞职。2020 年以降以 **与 [[megabanks/au-fh|ノジマ]] 的资本业务合作** 为轴处于信赖恢复阶段。与县内 No.1 的 [[regional-banks/shizuoka-fg]] 在结构上竞争。^[extracted]

### 信赖恢复阶段的战略

- **摆脱个人贷款特化**：从投资用不动产融资 → 向法人・地域联携 转移重心
- **与 Nojima 的资本业务合作**（2020-05）：在家电量贩店店内设银行窗口，扩大年轻层接触点
- **强化合规・治理**：反映第三方委员会的指出，扩充社外董事・监查职能
- **数字投资**：网络银行功能・移动账户

### 竞争关系

- **县内**：[[regional-banks/shizuoka-fg]] / [[regional-banks/shizuoka-bank]] 在县全域 No.1。[[regional-banks/shimizu-bank]] 和 [[regional-banks/shizuoka-chuo-bank]] 是各自独立的本地银行同行；骏河偏在于县东部
- **县外进军**：在神奈川县西部・东京西部也有店铺 → 与大手地银・超大型银行・[[regional-banks/concordia-fg]] 直接竞争
- **个人贷款残余**：在不动产向融资缩小局面，年轻层流出至 [[megabanks/paypay-fg]] / [[megabanks/au-fh]] / [[megabanks/sony-fg]] 等网络银行

### 合作・收购战略

- **Nojima 资本业务合作**（2020-05）：Nojima 出资 18% 左右，店内银行 + 数字融合
- **地域联携**：与静冈县・沼津市・神奈川县西部自治体的地方创生协定
- **与超大型银行的关系**：旧来与都市银行的人才交流，事件后强化人才接收

## 4. 监管・政策

- **主管**：金融厅（FSA）+ 关东财务局
- **2018-10-05 业务改善命令** ^[extracted]：
  - 合租公寓相关投资用不动产融资 6 个月停止
  - 明确经营责任，根本性重审治理
  - 顾客保护（构建受害业主应对方案）
- **第三方委员会报告书（2018-09-07）**：认定系统性地进行了文件篡改、源泉征收票伪造、租金台账篡改、存款存折的虚假资金、跳过审查
- **G-SIB**：不适用（地银规模）
- **近期政策论点**：
  - 日银政策利率正常化 → 存贷金利差改善（地银整体的顺风）
  - 地银重组压力（金融厅主导的地银整合）— 骏河走独立路线・Nojima 轴
  - 信赖恢复进展的持续监控

## 5. 重要事件：合租公寓『南瓜马车』不正当融资 ★

震撼业界的系统性不正事件。后来地银治理讨论・投资用不动产融资监管强化的触发点。

### 事件的结构 ^[extracted]

- **舞台**：合租公寓运营方 Smart Days 公司（女性专用合租公寓『南瓜马车』品牌）
- **方案**：Smart Days 向个人投资者销售物件 → 骏河银行高额融资（1 物件 1 億円超）→ Smart Days 以转租进行租金保证
- **2018-01〜04**：Smart Days 停止支付租金 → 经营破产 → 约 700 名的业主（个人投资者）无法偿还
- **不正的系统性**：
  - 篡改存款余额证明书・源泉征收票（伪造年收入）
  - 篡改租金台账（租赁台账）
  - 以"虚假资金"伪装自有资金
  - 为达成营业指标而跳过审查
  - 被认定并非部分支店・营业负责人的单独行为，而是 **支店整体・本部默许**
- **融资总额**：约 1 兆円超（投资用不动产整体，合租公寓为其中一部分）
- **受害**：业主约 700 名，平均融资 1 億円超

### 解决经纬 ^[extracted]

- 2018-09：第三方委员会报告书 → 认定系统性不正
- 2018-10：金融厅业务改善命令
- 2018-2019：经营层总辞职（社长・专务以下），大量起用外部董事
- 2019-2020：与受害业主的债务减免谈判（债权转让 + 本金减免）
- 诉讼：部分刑事告发・民事诉讼持续

### 对业界的影响

- **对投资用不动产融资整体的监管强化**：金融厅总点检其他地银・超大型银行的不动产融资
- **治理讨论**：重新思考地银社外董事・监查职能的应有方式
- **个人贷款特化模式的局限**：高 ROE 背后的风险显现，地银商业模式重审讨论
