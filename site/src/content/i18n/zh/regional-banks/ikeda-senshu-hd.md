# 池田泉州控股

## Wiki route

This entry sits under [[regional-banks/INDEX|regional-banks INDEX]]. Read it against [[regional-banks/kyoto-fg|京都フィナンシャルグループ]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

以大阪府为根据地的 **关西中坚地方银行 FG**。证券代码 **东证 PRIME 8714**。由 2009-10-01 **池田银行 + 泉州银行** 经营整合而成立，翌年 2010-05-01 两家子银行合并成立 **池田泉州银行**。以大阪府 **北摄·池田区域（旧池田银）** + **泉州区域 = 大阪府南部（旧泉州银）** + **兵库县东南部** 为 3 个主轴根据地，专注于京阪神巨型经济圈的中坚·中小企业金融。在关西与 [[megabanks/resona-hd]]（关西未来银）·京都 FG·滋贺银行·南都银行 等直接竞争。对于巨型 3 行（[[megabanks/mufg]] / [[megabanks/smfg]] / [[megabanks/mizuho-fg]]）则在中坚·中小层面错位竞争。^[extracted]

## 1. 公司概要

**正式名**：株式会社池田泉州控股 ^[extracted]
**英文名**：Ikeda Senshu Holdings, Inc.
**证券代码**：东证 PRIME **8714**（与旧 池田银行 8370 + 旧 泉州银行 8407 的退市相交换，2009-10-01 上市）^[extracted]
**设立**：**2009-10-01**（池田银行 + 泉州银行 通过共同股份转移进行经营整合）^[extracted]
**总部**：大阪市北区茶屋町 18-14（与池田泉州银行本店合设）^[extracted]

### 主要子公司·组织

```
池田泉州 HD（控股公司·东证 PRIME 8714）
  ├── 株式会社 池田泉州银行（100%）── 大阪府北摄 + 泉州 + 兵库东南部 根据地
  │     ├── 营业网点 约 130 店规模（以大阪府为中心，兵库·京都·奈良·和歌山一部分）
  │     ├── 面向中坚·中小企业的融资（制造业·物流·服务·不动产）
  │     ├── 住房贷款·个人存款（北摄区域中产阶层）
  │     └── 法人 IB / 银团贷款 参与（在巨型主导案件中作为 subordinate 参加）
  ├── 池田泉州资本 ── 风险投资 / 事业承继支援
  ├── 池田泉州 JCB ── 信用卡（JCB 合作）
  ├── 池田泉州商务服务 ── BPO / 事务代办
  └── 关联智库·调查机能（地域经济分析）
```

### 合并经纬·前身

- 旧 **池田无尽 → 池田银行**（北摄根据地）
  - **1949** 池田无尽 设立（无尽公司·大阪府池田市）
  - **1951** 更名池田相互银行（伴随相互银行法施行的转换）
  - **1989** 更名池田银行（向普通银行转换，旧证券代码 8370）
- 旧 **泉州银行**（大阪府泉州地区根据地）
  - **1942** 泉州银行 设立（战时统制下的地方银行整合·大阪府泉州地区）
  - 战后，在堺市·岸和田市·泉佐野市等 泉州区域专注中小企业金融（旧证券代码 8407）
- **池田泉州 HD 成立**: 2009-10-01
- **池田泉州银行 合并**: 2010-05-01（池田银行 + 泉州银行 → 池田泉州银行）

### 战略轴

- **地域贴近 + 中坚·中小专注**: 确立作为大阪府中坚·中小企业 main bank 的位置。对巨型则以小额·本地案件错位竞争 ^[extracted]
- **北摄（旧池田银）+ 泉州（旧泉州银）的地理互补**: 以 1 行覆盖大阪府的北部与南部，扩大对府内 GDP 的获取 ^[extracted]
- **2024 中计**: 在第 5 次中期经营计划 Plus（2024-05 公布）中以 **地域贴近 + 数字银行（01 银行）+ 潜在 B/S 解决方案** 为主轴，目标为 2025 年年度 HD 合并本期净利润 110 億円以上

### 整合协同·课题

- **池田银（北摄）+ 泉州银（南部）的地理互补性 高**: 在大阪府北部·南部两端把控的地方银行别无他家，单一行整合的协同大 ^[extracted]
- **京阪神巨型经济圈中的「中坚地方银行」生存**: 与巨型 3 行 + [[megabanks/resona-hd]] + 京都银 等争夺客户 ^[extracted]

## 4. 监管·政策

- **主管**: 金融厅（FSA）
- **控股公司监管**: 银行法 第 52 条之 17（银行控股公司）^[extracted]
- **近期政策论点**:
  - **2024〜 日银政策利率正常化**: 国内贷款利率利差改善红利（地方银行好处大，加息利好地方银行收益改善预期）
  - **2024〜 地方银行重组预期**: 金融厅对地方银行重组的立场是「不否定整合本身，但当局不主导」
  - **2025〜 营业网点削减**: 地方银行标准的重组路线，网点网精简与机能集约
  - **面向中小企业的事业性评价融资**: 推进不依赖担保·保证的融资
  - **ESG / 地域脱碳 融资**: 面向中小企业的脱碳转型支援
  - **大阪·关西万博（2025-04〜10）**: 关西经济圈活性化预期 → 对地方银行的交易对象（建设·物流·观光·服务）的 follow-tail 效应
