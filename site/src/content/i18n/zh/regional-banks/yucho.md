# 邮储银行（ゆうちょ銀行 / Japan Post Bank）

## Wiki 路由

本条目归属于[[regional-banks/INDEX|regional-banks INDEX]]。请将其与[[megabanks/resona-hd|りそなホールディングス (Resona Holdings)]]（同业 / 对比语境）一并阅读，并与[[banking/INDEX|banking index]]（更广泛的系统 / 监管边界）一并阅读。

## TL;DR

一家拥有日本最大级存款规模（**约 190 兆円**规模，存款余额 190.5 兆円〔2025-03-31〕，在国内银行中位居最大级别）的商业银行。它在邮政民营化（2007-10-01）的同日开业，以全国 **约 24,000 局**的邮局网络（委托给日本邮便经办）作为零售基础。其融资业务有限，运用资产的大部分长期依赖**以国债为中心的有价证券运用**，但近年将**向海外债券、另类投资、私募基金的多元化**作为中期经营计划的支柱。母公司是日本邮政（表决权持有比率因分阶段出售而 **2025-03-31 时点为 50.04%**，到2025 年降至 50% 以下）。2015-11-04 与日本邮政、简保生命 **3 社同时 IPO**。^[extracted]

## 1. 公司概要

**正式名称**：株式会社ゆうちょ銀行 ^[extracted]
**英文名**：Japan Post Bank Co., Ltd.
**证券代码**：东证 PRIME 7182（2015-11-04 上市）^[extracted]
**设立**：2006-09-01（作为邮政民营化准备公司）^[extracted]
**开业**：2007-10-01（与邮政民营化同日，从日本邮政公社承继事业）^[extracted]
**总部**：东京都千代田区大手町
**母公司**：日本邮政株式会社（股票代码 6178，有政府持股）

## 2. 事业板块图谱

| 板块 | 主要事业 | 特征 |
|---|---|---|
| 个人存款（储蓄）| 普通储蓄・定额储蓄・定期储蓄 | **国内最大级的存款余额**，全国邮局网络为渠道 |
| 汇兑・汇款 | 划拨・汇款・公共费用代收 | 邮政划拨的历史基础 |
| 贷款 | 有限（住房贷款等以介绍他行为主）| 持有商业银行牌照但**融资几乎集中于国债等运用** |
| 有价证券运用 | 国债・外债・另类投资・私募基金 | 作为存款运用去向的核心板块 |
| 卡片 | JP BANK 卡（提携卡）| JP BANK 卡 Visa/Mastercard 由三井住友卡担保，JCB 版为邮储银行 × JCB 的提携发行。面向法人的「邮储银行商务卡」也与三井住友卡提携（2024） |
| 数字 | **邮储存折应用** / **mijika**（面向年轻层的应用）| 谋求摆脱对邮局柜台的依赖 |
| 法人 | 有限（以结算・代收为主）| 面向大企业的融资・投行业务非主力 |

### 数字战略

- **邮储存折应用**：存折数字化・余额查询・交易记录・汇款 ^[extracted]
- 摆脱对邮局柜台的依赖・强化柜台以外渠道是中期经营计划的主题

### 合作・收购战略

- **与日本邮便的业务委托**：将全国 约 24,000 局 的柜台业务・现金处理委托给日本邮便（民营化方案的根本）
- **与简保生命的销售渠道协同**：在同一邮局网络内销售寿险（在2019 年 简保生命不当销售问题之后管理得到强化）
- 与提携卡・投信运用公司・金融科技的个别提携

### B2C 品牌建设

- **「ゆうちょ」品牌**：自邮政储蓄（1875〜）以来 150  年规模的认知度
- **全国 24,000 局 网络**：超过城区银行网点网的压倒性零售网点数
- 对地方・高龄客户的深度渗透（含大型银行未覆盖区域）

## 4. 监管・政策

- **主管**：金融厅（FSA）／总务省（邮政行政・日本邮便部分）
- **银行法**：持有商业银行牌照（普通银行）
- **邮政民营化法**：政府对日本邮政股份的持有义务・分阶段出售时间表
- **民间补充原则**：作为邮政民营化法的理念，对不挤压民间金融机构的业务范围加以约束（新业务为邮政民营化委员会的审查对象）
- **存款保险机构**：加盟（2007-10 因民营化，邮储银行成为存款保险制度的对象金融机构，所存储蓄受存款保险保护）

### 近期政策论点

- 政府持有义务的分阶段出售 → 完成民营化的路径
- 民间补充原则的边界（融资・服务扩张的极限）
- 2024〜 日本央行政策利率正常化 → **国债持有品种的估值损益・运用收益率改善**两面
- 简保销售问题后的治理强化（集团整体）
