# 第四北越 FG

## Wiki 路径

本条目位于 [[regional-banks/INDEX|regional-banks INDEX]] 之下。应将其与 [[regional-banks/fukuoka-fg|ふくおか FG (FFG)]] 对照阅读以了解同行 / 对比背景，并与 [[banking/INDEX|banking index]] 对照以了解更广泛的体系 / 监管边界。

## TL;DR

以新潟县为地盘的地银金融集团（在同县内份额压倒性领先・人口 220 万县的一县一行型）。由 2018-10-01 第四银行 + 北越银行 共同控股公司化而成立，于 2021-01-01 将子银行相互合并，过渡到 **第四北越银行** 单一行体制。**以县内地盘整合（全能银行业务 + 中坚企业 main bank）+ 县外进军（首都圈营业店）为 2 轴**。前身第四银行于 1873 年设立，是"日本最早的国立银行之一"（第四国立银行）这一渊源。观光（佐渡・滑雪）・农业（稻米产地）・制造业（燕三条等）金融为地盘特性。是地银重组中的中坚广域联合事例之一。

## 1. 公司概要

**正式名**：株式会社第四北越金融集团
**英文名**：Dai-shi Hokuetsu Financial Group, Inc.
**证券代码**：东证 PRIME 7327
**设立**：2018-10-01（第四银行 + 北越银行 共同控股公司化）
**总部**：新潟县新潟市中央区（第四北越银行总行所在地）

### 主要子公司・持股

```
第四北越 FG（控股公司・上市 7327）
  └── 第四北越银行（100%）── 以新潟县为地盘的单一银行子公司
        ├── 旧 第四银行（1873〜2020）── 新潟县内最大手地银
        ├── 旧 北越银行（1942〜2020）── 县内第二地银格
        ├── 县内营业基础：新潟市・长冈市・上越市・佐渡 等
        └── 县外营业店：首都圈（东京）等
```

### 合并历史・前身

- 旧 **第四银行**：作为 **1873 年 第四国立银行** 开业（日本最早的国立银行之一，编号银行 4 号）→ 1917 年 株式会社化 → **第四银行**。新潟县内最大手地银
- 旧 **第六十九国立银行**：1878 年设立，后来北越银行前身之一（编号银行 69 号）
- 旧 **长冈六十九银行**：第六十九银行系列，以长冈为地盘
- 旧 **高田银行**：以上越（高田）为地盘
- 旧 **柏崎银行**：以柏崎为地盘
- 旧 **北越银行**：**1942 年** 由长冈六十九银行 + 高田银行 + 柏崎银行 合并成立，新潟县内第二地银格

> 整合背景：新潟县的人口减少 + 低利率挤压利差 + 超大型银行法人融资竞争 → 以接近一县一行的广域地盘整合应对。共同控股公司化（2018-10）后进行银行合并（2021-01）的 2 阶段方式。

## 2. 业务分部・地图

| 分部 | 主要经营者 | 特征 |
|---|---|---|
| 地盘核心（新潟县内）| 第四北越银行 | 县内份额压倒性领先，全能银行业务 |
| 中坚企业 main bank | 第四北越银行 | 制造业（燕三条金属加工等）・农业（稻米产地）|
| 观光・地方产业金融 | 第四北越银行 | 佐渡・滑雪度假区・新潟港物流 |
| 县外展开 | 第四北越银行 首都圈营业店 | 进军东京等（随客户随行 + 新规开拓）|
| 个人 retail | 第四北越银行 | 县内人口 220 万，存款 / 住房贷款 |

### 地盘整合战略（县内一行型）

- **新潟县内 接近一县一行的地盘**：县内几乎不存在竞争对手的地银结构（已整合第二地银格的北越）
- 独占性地覆盖县内人口 220 万 → 稳定的存款基础和中坚企业 main bank 地位
- 2021-01 银行合并使 **店铺・人员・系统的整合协同** 进入显现阶段
- 县内竞争对手为信用金库 / JA 银行 / 邮储（[[regional-banks/yucho]]）/ 超大型银行法人融资

### 县外进军战略

- **首都圈营业店**：县内客户的客户随行（制造业 + 商社）+ 新规法人开拓
- **与超大型银行的法人融资竞争**：[[megabanks/mufg]] / [[megabanks/smfg]] / [[megabanks/mizuho-fg]] 直接接洽县内大手法人 → main bank 防卫战
- 县外展开与其说是为扩大规模，不如说更具有扩充县内客户服务的意味

### 地方产业特性

- **观光**：佐渡（金山・世界遗产候选）・汤泽 / 妙高滑雪 / 新潟市・长冈 观光振兴
- **农业**：稻米（越光 品牌产地）・酒坊（清酒酒厂数量 全国前列）
- **制造业**：燕三条金属加工集群（刀具・西式餐具・工具）/ 长冈 机械
- **物流**：新潟港（日本海一侧据点港）・新潟机场

### 在地银重组中的位置

- 作为单独 FG 是中坚广域联合的典型事例（县内一行型）
- 对超大型 3 行（[[megabanks/mufg]] / [[megabanks/smfg]] / [[megabanks/mizuho-fg]]）以县内 main bank 地位对抗
- 与 [[megabanks/ndfg]] 等其他广域 FG / 邻县（东北・北陆）地银未来联合的可能性尚未确定

## 4. 监管・政策

- **主管**：金融厅（FSA）+ 关东财务局新潟财务事务所
- **控股公司监管**：银行法 第 52 条之 17
- **银行合并监管**：2021-01 第四 + 北越 合并因县内份额超过半数而成为公取委审查对象（已获许可认可）
- **近期政策论点**：
  - 2024〜 日银政策利率正常化 → 地银利差改善（比超大型银行受益更大）
  - 2025〜 地银重组加速（人口减 + 数字投资负担）
  - 新潟县的人口减少（县人口 220 万 → 长期减少趋势）→ 挤压存贷量
