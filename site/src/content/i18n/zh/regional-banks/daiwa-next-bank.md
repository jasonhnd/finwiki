# 大和Next银行 (Daiwa Next Bank)


## Wiki 路径

本条目位于 [[regional-banks/INDEX|regional-banks INDEX]]（其领域路径——列在数字 / 网络 / 零售平台银行分组下）之下。请与 [[regional-banks/sony-bank|ソニー銀行 (Sony Bank)]] 对照阅读以了解网络银行同业 / 对比，并与 [[financial-regulators/fsa|the FSA]] 及 [[banking/INDEX|banking]] 对照以了解监管 / 系统边界。母集团背景见 [[securities-firms/daiwa-sg|大和証券グループ]]。

## TL;DR

大和Next银行是持有《银行法》项下银行牌照、并列入金融厅银行牌照一览的 [[securities-firms/daiwa-sg|大和証券グループ]] 旗下银行 operating company（网络银行 / 其他银行）。它是以与大和证券账户联动为前提的存款、外汇、资产形成支持银行；由于母公司（大和证券集团总公司）于 2026-04-27 公布以约 3,700 億円将 [[trust-banks/orix-bank|オリックス銀行]] 收为全资子公司、并计划未来将两行合并，其在证券集团银行职能扩张中的重要性正在上升。

## 2. 业务角色

- 作为与大和证券账户联动的网络银行，提供日元存款、外币存款、资金转移、资产形成支持。
- 由于接入全国大和证券门店与营业员网络，是不同于完全在线专营银行的 hybrid securities-bank model。
- 若Orix银行的子公司化与未来合并推进，房地产贷款、存款基础的扩张将成为焦点（按公布口径，两行合计意图大幅扩大总资产规模）。
- 可视为与既有网络银行 [[regional-banks/sony-bank|ソニー銀行]] 并列的「源自事业集团的网络银行」类型。

## 3. 监管・系统定位

- **监管**: 在《银行法》之下由金融厅 ([[financial-regulators/fsa|FSA]]) 监管。并非属于地方银行协会、第二地方银行协会，而是在 FSA 分类上属于「其他银行」的网络银行。通过自有资本比率规制等对健全性加以约束。
- **行业 / 系统**: 参加全国银行资金结算网络（全银系统），在行业团体方面连属于 [[financial-regulators/zenginkyo|全国銀行協会（全銀協）]] 的框架。
- **存款人保护**: 存款属于存款保险机构（DIC）的承保对象。

## 4. 为何独立页面重要

仅凭大和证券集团的证券公司页面，无法追踪银行牌照、存款基础、证券账户联动、Orix银行整合。大和Next银行是观察「证券集团在多大程度上拥有银行职能」的代表性案例。
