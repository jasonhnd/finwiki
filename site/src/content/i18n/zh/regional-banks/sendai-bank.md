# 仙台银行

## Wiki 路径

本条目位于 [[regional-banks/INDEX|regional-banks INDEX]] 之下。可与 [[regional-banks/kirayaka-bank|きらやか銀行]] 对读以理解同业 / 对比语境，并与 [[banking/INDEX|banking index]] 对读以把握更广的制度 / 监管边界。

## TL;DR

株式会社仙台银行是总部位于宫城县、列入金融厅银行牌照名单的第二地方银行，也是 [[regional-banks/jimoto-hd]] 旗下的运营银行。在宫城县市场，[[regional-banks/77-bank]] 是县内头部地方银行，仙台银行则是第二地方银行侧的 SME / 零售 / 复兴金融 anchor。

## 2. 运营模式

仙台银行是承担宫城县内零售、SME、复兴相关金融、企业贷款和本地咨询的第二地方银行。[[regional-banks/jimoto-hd]] 页面覆盖公的资金和控股公司角度；本页面分离出银行牌照与运营足迹。东北第二地方银行整合语境见 [[banking/regional-bank-consolidation-pattern]]。

## 3. 本页面为何重要

- 仅有母公司 [[regional-banks/jimoto-hd]] 覆盖无法显示持牌银行边界。
- 与 [[regional-banks/kirayaka-bank]] 互补，构成 Jimoto 的另一家运营银行。
- 可作为 [[regional-banks/77-bank]] 以及东北地方银行重组笔记的有用同业。
