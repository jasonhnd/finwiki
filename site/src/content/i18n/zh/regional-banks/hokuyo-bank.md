# 北洋银行

## Wiki route

This entry sits under [[regional-banks/INDEX|regional-banks INDEX]]. Read it against [[regional-banks/jimoto-hd|じもとホールディングス (JIMOTO HD)]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

北海道最大的地区性金融机构（总资产 **13 兆 4467 億円** / 存款 11 兆 1039 億円 / 网点 171 / FY2024 = 2025-03 期实绩，合并口径）。以 1917 设立的 **北海道无尽** 为起源，1951 转换为相互银行成为 **北洋相互银行**，1989 转换为普通银行成为 **北洋银行**。加入第二地方银行协会。**1998-11 承接旧北海道拓殖银行的道内业务**使其道内份额跃升至 No.1 的经历具有决定性意义。总行：札幌市中央区大通西 3-7（北洋大通中心）。

## 1. 公司概要

**正式名称**：株式会社北洋银行 / 英文名：North Pacific Bank, Ltd.
**设立**：大正 6 年（1917 年）8 月 20 日（作为北海道无尽株式会社）
**总行**：北海道札幌市中央区大通西 3 丁目 7 番地（北洋大通中心内）
**业态**：第二地方银行协会 加盟行
**母公司沿革**：旧上市持股公司 **札幌北洋控股（东证·札证 8328）→ 2012-10-01 北洋银行以逆向吸收合并方式解散并退市**（日本第 3 例由子银行吸收合并持股公司的案例，继红叶 HD、Kirayaka HD 之后）。**现为非上市的独立银行**

### 主要子公司·集团公司

- **北洋证券**（100%）— 零售证券，旧 大道证券系
- **札幌北洋租赁** — 租赁业
- **札幌北洋卡** — 信用卡
- **北洋商务服务** — 事务代理
- 旧：札幌北洋 HD（2012-10-01 解散）

### 合并·前身沿革

- **1917-08-20**：北海道无尽 设立（札幌，无尽业）
- **1951**：相互银行法施行 → **北洋相互银行** 商号变更
- **1989-02**：转换为普通银行 → **北洋银行** 商号变更（转为第二地银）
- **1998-11-13**：旧 **北海道拓殖银行** 道内业务转让合同签署 ★
- **1998-11-16**：从拓银承接 **109 店处** 网点，总网点 231 店，存款量约 4.7 兆円
- **2001-04**：设立持股公司「**札幌北洋控股**」，东证·札证 8328 上市
- **2008-10-14**：与 **札幌银行**（旧 札幌相互银行）合并，札银的 58 年历史终结
- **2012-10-01**：北洋银行逆向吸收合并札幌北洋 HD → 持股公司解散·退市

> 补正注：质询提示中称「2017-05 合并使持股公司解散」，但据 Wikipedia 公开信息，**2012-10-01** 才是准确的解散日期。本笔记遵从公开来源。

## 2. 业务分部·地图

| 分部 | 主要业务 | 特征 |
|---|---|---|
| 个人 | 存款·住房贷款·信用卡·投信 | 道内份额 No.1，第二地银最大级零售网络 |
| 法人 | 面向北海道中坚·中小企业的融资 | 承接旧拓银法人交易，道内 main bank 众多 |
| 公共 | 北海道厅·札幌市等 指定金融机构业务（部分）| 与道银（[[regional-banks/hokuhoku-fg]] 旗下）分担 |
| 证券 | 北洋证券（100%）| 零售证券 |
| 租赁 | 札幌北洋租赁 | 集团补充 |

### 竞争环境

- **道内 No.2**：**北海道银行**（[[regional-banks/hokuhoku-fg]] = HOKUHOKU FG 旗下，地银协加盟）— 道内份额之争长年是「**道银 vs 北洋**」的两极结构
- **KITASHIN 品牌**：北海道信金中金系，地区信金联合
- **大型银行道内据点**：[[megabanks/mufg]] / [[megabanks/smfg]] / [[megabanks/mizuho-fg]] — 在城市地区·大额法人上有重叠，但零售在道内的渗透度北洋占优
- **邮储·JA 银行**：存款获取竞争对手

### 承接拓银的历史意义 ★

1997-11-17 北海道拓殖银行 破产（城市银行首例经营破产）→ 1998-11 将道内 109 店转让给北洋银行。当时的北洋只有拓银的 **1/3 行员工·1/5 总资产·1/10 资本金**，是道内 3 位的第二地银。将其吞下而跃升道内 No.1 被形容为 **「蛇吞鲸」**。本州业务转让给中央信托银行（→ 三井住友信托）。

### B2C 品牌建设

- 标志商标 1985-10 导入（北洋相互时代）
- 扎根北海道的「**道民的银行**」定位
- 总行「北洋大通中心」是札幌大通公园沿线的地标

## 4. 监管·政策

- **主管**：金融厅（FSA）/ 财务省 北海道财务局
- **行业团体**：第二地方银行协会
- **指定金融机构**：北海道厅等部分公款收纳
- **近期政策议题**：
  - 2024〜 日银政策利率正常化 → 存贷利差改善红利
  - 道内人口减少·与道银竞争加剧
  - 承接拓银 25 周年（2023-11）的总结阶段
