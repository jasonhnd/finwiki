# ProCrea Holdings

## Wiki 路径

本条目位于 [[JapanFG/INDEX|JapanFG index]] 之下。可与 [[regional-banks/fidea-hd|フィデア HD (Fidea Holdings)]] 对读以理解同业和对比关系，并与 [[banking/INDEX|banking index]] 对读以把握更广的制度 / 监管边界。

## TL;DR

以青森县为地盘的地方银行 2 行（**青森银行 + みちのく银行**）通过 2022-04-01 经营整合成立了本控股公司（东京证券交易所 PRIME **7384**）。作为**《反垄断法》特别例外法（地域基础企业特别例外法）首次适用案件**，它在国家“背书”下实现县内 2 行 统合，是**人口减少地区地方银行生存模式案例**。**2025-01-01 两家子银行合并并成立“青森みちのく银行”**（总存款余额超过 5 兆円，在东北为 3 位，青森县内贷款份额超过 7  成）。公司名“ProCrea”由“Progress（前进）”和“Creation（创造）”合成，提出覆盖北东北（青森、岩手、秋田）的广域战略。

## 1. 公司概要

**正式名称**：株式会社プロクレアホールディングス
**英文名**：ProCrea Holdings, Inc.
**证券代码**：东京证券交易所 PRIME **7384**
**设立**：**2022-04-01**（青森银行 + みちのく银行通过共同股份转移进行经营整合）
**总部**：青森县青森市（以青森银行总店所在地为基础）

### 主要子公司（整合刚完成后的结构）

```
ProCrea Holdings（控股公司、上市 7384）
  ├── 株式会社青森银行（100%）── 县内份额最大、北东北核心
  │     └── 关联公司（租赁、卡、咨询等）
  └── 株式会社みちのく银行（100%）── 县内 2 位，在北海道道南也有网点
        └── 关联公司（租赁、卡、与道银的合作等）

→ 2025-01-01 合并（已实施）:
  ProCrea Holdings
  └── 株式会社青森みちのく银行（100%）
```

### 整合经过（重要）

- **2021-05** 青森银行和みちのく银行公布经营整合
- **2022-04-01** ProCrea HD 成立（共同股份转移）
- 因县内 2 行 统合，产生反垄断法上的“地理市场”问题 → 依据**反垄断法特别例外法**获得公正交易委员会合并许可
- 本案是该特别例外法的**首次适用案件**，成为地方银行统合的模式案例
- **2025-01-01** 两家子银行合并，商号“**青森みちのく银行**”成立（青森银行石川启太郎任行长，みちのく银行藤泽贵之任会长，员工约 2,300  人、网点 148 个）

### 青森银行谱系

| 年 | 事件 |
|---|---|
| **1879**（明治 12）| **第五十九国立银行** 创立（弘前，青森县第一家银行，青森银行最古老前身）|
| **1897**（明治 30）| 依据国立银行营业期满前特别处分法改组为**株式会社第五十九银行** |
| **1943-10-01**（昭和 18）| 在一县一行政策下，**第五十九、八户、津轻、板柳、青森的 5 行 合并，在青森市新设青森银行** |
| 战后〜2022  | 作为青森县内份额最大的地方银行营业 |
| 2022-04-01 | 成为 ProCrea HD 子公司 |
| 2025-01-01 | 与みちのく银行合并 → 青森みちのく银行 |

### みちのく银行谱系

| 年 | 事件 |
|---|---|
| **1921** | **青森商业银行** 成立（与青森银行不同系统）|
| 战中〜战后 | 弘前无尽 → 弘前相互银行等谱系 |
| **1976** | 弘前相互银行 + 青和银行合并，**みちのく银行**成立（转为普通银行）|
| 1990 年年代 | 进入北海道道南（函馆等）|
| 2000 年年代 | 探索与北洋银行、北海道银行合作 |
| 2022-04-01 | 成为 ProCrea HD 子公司 |
| 2025-01-01 | 与青森银行合并 → 青森みちのく银行 |

## 3. 业务板块地图

| 板块 | 主要运营者 | 特征 |
|---|---|---|
| 县内法人 | 青森银行、みちのく银行 | 面向青森县内企业的 main bank，合并后份额突出 |
| 县内个人 | 两行 | 存款、住房贷款、养老金账户 |
| 北东北广域 | 青森银行（岩手、秋田也有网点）| 北东北 3  县广域战略 |
| 北海道道南 | みちのく银行 | 函馆等，与北海道金融机构的接点 |
| 第一产业金融 | 两行 | **苹果、大米、水产（青森是日本第一苹果产地，渔业也是主要产业）** |
| 旅游金融 | 两行 | 睡魔祭、弘前樱花、新青森站、奥入濑、十和田湖 |

### 整合协同

- **县内份额集中**：合并后的青森みちのく银行在青森县内贷款和存款份额上几乎处于垄断地位 → 由反垄断法特别例外法允许
- **系统整合**：避免大规模 IT 投资重复
- **人事费用合理化**：整合重复管理部门

### 地域战略

- **北东北战略**：把青森、岩手、秋田三个北东北 3  县视为一体市场
- **道南连接**：みちのく银行的函馆据点因新干线（新青森〜新函馆北斗）开通而价值化
- **观光、农业、渔业**：苹果出口（面向台湾、香港）、扇贝、樱鳟、新青森站周边再开发等

### 课题（人口减少地区模型）

- **青森县人口减少**：减少率在全国也居前，年轻层流出至东京、仙台
- **通缩与老龄化**：存贷缺口扩大（存款聚集但贷款对象不足）
- **本地企业缩小**：停业和后继无人 → 事业承继支援成为主要业务
- 整合的本质不是“增长战略”，而是“**通过整理统合收益基础确保可持续性**”

## 5. 监管 / 政策

- **主管**：金融厅（FSA）东北财务局
- **控股公司监管**：银行法第 52  条之 17

- **反垄断法特别例外**：**反垄断法特别例外法**（地域基础企业特别例外法，令和 2 年 法律 第 32 号，2020-11  施行，10  年限时立法）
  - **2021 年 公取委认可** → **该特别例外法首次适用案件** ★
  - 将县内份额集中带来的竞争限制担忧，以“可持续性”为优先予以容许的政策转向
  - 成为后续地方银行统合（ふくおか FG 系等）的模型

## 7. Open questions

- 2025-01-01 合并后 **青森みちのく银行整合协同实现速度**（网点削减、IT 整合、人员再配置进展）
- **反垄断法特别例外法 10  年限时**（至 2030）后的制度延长和永久化讨论
- 北东北广域战略的实质进展（岩手、秋田的贷款份额扩大能否实现）
- 与 **TSUBASA Alliance 等广域合作**的关系：跨县进一步经营整合情景
- 北海道道南据点（みちのく银行函馆等）的战略定位（与北洋银行、北海道银行的关系）
- **地方银行重组的下一张多米诺**：其他县地方银行（特别是九州、四国、北陆）是否会跟随 ProCrea HD 模型
