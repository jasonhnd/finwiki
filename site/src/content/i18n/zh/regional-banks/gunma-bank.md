# 群马银行 (The Gunma Bank, Ltd.)

## Wiki route

This entry sits under [[regional-banks/INDEX|regional-banks INDEX]]. Read it against [[regional-banks/chiba-bank|千葉銀行 (Chiba Bank)]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

以群马县为根据地的地方银行（东证 PRIME 8334）。**1932-09 作为群马大同银行设立**（大同银+群马商工银+黑岩银 整合）→ **1955 更名为群马银行**。是维持**未控股公司化的单一银行体制**的首都圈 PRIME 上市地方银行的代表。以 **SUBARU 太田工厂·上信越自动车道经济圈** 为背景，汽车相关供应链融资是重要的客户基础。2015 参加 TSUBASA 联盟实现系统共同化·海外支援协同。^[extracted]

**正式名**：株式会社群马银行 ^[extracted]
**英文名**：The Gunma Bank, Ltd.
**证券代码**：东证 PRIME 8334（1949 上市·因市场分区重审而继续 PRIME）^[extracted]

**业态区分**：地方银行（全国地方银行协会 加盟）/ 单一银行（未控股公司化）^[extracted]

### 未控股公司化的论点

在巨型银行 3 行·有力地方银行（[[regional-banks/mebuki-fg]] / [[regional-banks/concordia-fg]] / [[regional-banks/fukuoka-fg]] 等）推进控股公司化的过程中，**群马银行维持单一银行体制**。这是因为：
- 地域专营·单一县根据地，控股公司化的事业多元化好处小
- 子公司（租赁·信用卡·咨询）在银行直属子公司即可充分管理
- 缺少整合·经营整合对象（县内唯一的 PRIME 地方银行）

## 2. 事业分部·地图

| 分部 | 特征 |
|---|---|
| 县内法人 | 汽车相关（SUBARU 太田工厂系·零部件供应商）·制造业集聚 |
| 首都圈扩大 | 埼玉北部（熊谷·本庄·深谷）·东京北部 营业拓展 |
| 个人 | 县内份额最大，以住房贷款·养老金账户为中心 |

| 租赁·信用卡 | 群银租赁 / 群银卡服务 |
| 海外支援 | 驻在员事务所（中国·NY·香港 等）→ 经由 TSUBASA 补充 |
| 数字化 | TSUBASA 共同基础设施 + 自行 App |

### 营业区域地理

- **本部**: 群马县全域（前桥·高崎·太田·桐生·伊势崎·馆林 等）
- **邻接县进出**: 埼玉县（熊谷·本庄·深谷·川越）/ 栃木县 / 东京都北部 / 神奈川 / 千叶 / 大阪 / 名古屋
- **特征**: **上信越自动车道经济圈**（关越道 + 上信越道）沿线的物流·制造业 → 汽车·零部件·电机 供应链融资

### 产业基础特性

- **SUBARU（富士重工业）太田工厂**: 群马县太田市 主力组装工厂 → 数百家零部件供应商在县内集聚 → 群马银行 法人融资的核心客户群
- **上信越自动车道经济圈**: 群马·长野·新潟 物流动脉 → 仓库·物流 / EC 相关法人融资
- **观光**: 草津·伊香保 等温泉地，富冈制丝场（世界遗产）→ 观光业贷款
- **农业**: 蒟蒻·卷心菜·养蚕（历史）→ 农业法人融资

### 参加 TSUBASA 联盟

**TSUBASA 联盟** = 千叶银行 主导的地方银行广域联动框架。参加行: 千叶银·第四北越银·中国银·伊予银·东邦银·北洋银·武藏野银·滋贺银·琉球银·群马银 等 ^[extracted]。

群马银行参加好处:
- **核心系统共同化**（成本分摊）
- **海外支援**（活用参加行的海外据点 → 无需自行持有）
- **商品·服务共同开发**（投信·基金包裹·网络安全对策）
- **人才交流·研修**

## 4. 监管·政策

- **主管**: 金融厅（FSA）·关东财务局
- **行业**: 全国地方银行协会（地银协）加盟·第二地方银行协会 非加盟（第 1 地方银行）
- **资本监管**: 巴塞尔 III 国内基准行 ^[likely]
- **存款保险**: 存款保险机构 加盟
- **近期政策论点**:
  - 2024〜 日银政策利率正常化 → 国内存贷利差改善 → 地方银行收益红利
  - 地方银行重组压力（金融厅·公共资金 / 推荐控股公司化）vs 维持单一银行的判断
  - 汽车产业 EV 转型 → 零部件供应链结构变化风险（对 SUBARU 系融资对象的依赖度）
