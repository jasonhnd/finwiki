# 东和银行 (The Towa Bank, Ltd.)

## Wiki route

This entry sits under [[regional-banks/INDEX|regional-banks INDEX]]. Read it against [[regional-banks/jimoto-hd|じもとホールディングス (JIMOTO HD)]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

总部位于群马县前桥市的**第二地方银行**（东证 PRIME 8558）。**1917-11-01 作为群马无尽株式会社创业**，经相互银行转换（1951）后，**配合 1989 普通银行转换更名为东和银行**。以**未控股公司化的单一银行体制**，在群马县内是仅次于第 1 地方银行 [[regional-banks/gunma-bank]] 的县内 No.2 地位。中小企业专注型营业基础与 **SUBARU 太田工厂供应链金融** 是法人业务的核心。通过**2003 与旧第一劝业银行（现 [[megabanks/mizuho-fg]]）系的战略合作**构建广域联动体制。简称「とうわ」。^[extracted]

### 未控股公司化的论点

在群马县内竞争的 [[regional-banks/gunma-bank]]（第 1 地方银行）也维持单一银行体制，群马县的上市 2 行均为 **单一银行 PRIME 上市** 这一罕见结构 ^[extracted]。就第二地方银行整体而言，在 **[[regional-banks/mebuki-fg]] / [[regional-banks/tomony-hd]] / [[regional-banks/fidea-hd]] / [[regional-banks/jimoto-hd]] / [[regional-banks/proclair-hd]]** 等推进广域控股公司化的过程中，东和银行维持单一体制。背景是:

- 县内集中·单一根据地，控股公司化的事业多元化好处小
- 关联公司（租赁·保证等）可作为银行直属子公司管理
- 通过与旧第一劝业银行系（现 [[megabanks/mizuho-fg]]）的战略合作进行广域补充

### 业态变迁的谱系

```
群马无尽（1917）
  → 群马殖产无尽（1942 更名）
  → 群马相互银行（1951 业态转换 / 相互银行法）
  → 东和银行（1989 普通银行转换 + 更名）
```

是一家遵循无尽 → 相互银行 → 普通银行（第二地方银行）这一日本中小企业金融机构典型业态演化的银行 ^[extracted]。

| 分部 | 特征 |
|---|---|

### 营业区域地理

- **本部**: 群马县全域（前桥·高崎·太田·桐生·伊势崎·馆林 等）
- **邻接县进出**: 埼玉县北部（熊谷·本庄 等）/ 栃木县 / 东京北部 一部分
- **特征**: **群马县中小企业专注** + **SUBARU 太田工厂经济圈**

### 旧第一劝银系（みずほ）战略合作

**2003 与旧第一劝业银行系的战略合作** ^[extracted]:
- 广域系统 / 商品共同开发
- 大企业向案件的上位行回转·银证协同
- 通过活用みずほ系基础设施补充单一银行资源
- 不进入大型系统而维持单一体制、同时以战略联动进行广域补充的典型案例

## 5. 监管·政策

- **主管**: 金融厅（FSA）·关东财务局
- **行业**: **第二地方银行协会（第二地银协）加盟**（全国地方银行协会 = 第 1 地银协 非加盟）^[extracted]
- **资本监管**: 巴塞尔 III 国内基准行 ^[likely]
- **存款保险**: 存款保险机构 加盟
- **近期政策论点**:
  - 2024〜 日银政策利率正常化 → 存贷利差改善 → 地方银行收益红利
  - 地方银行重组压力（金融厅·活用公共资金 / 推荐控股公司化）vs 维持单一银行的判断
  - SUBARU·汽车产业 EV 转型 → 零部件供应链结构变化风险（对中小供应商融资对象的依赖）
  - 中小企业僵尸化问题（新冠零零融资的出口）→ 第二地方银行的授信压力
