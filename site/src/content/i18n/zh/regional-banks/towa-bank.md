# 东和银行 (The Towa Bank, Ltd.)

## Wiki route

本条目位于 [[JapanFG/INDEX|JapanFG index]] 之下。可与 [[regional-banks/jimoto-hd|じもとホールディングス (JIMOTO HD)]] 对读，以理解同业 / 对照背景；也可与 [[banking/INDEX|banking index]] 对读，以理解更广的制度 / 监管边界。

## TL;DR

东和银行是一家总部位于群马县前桥市的 **第二地方银行**，在东证 Prime 上市 8558。它以 **1917-11-01 群马无尽株式会社** 起家，经历相互银行转型路线 1951，并配合 1989  普通银行转型改名为东和银行。它仍是 **非控股公司化的独立银行结构**；在群马县内，位置在第一 1  地方银行 [[gunma-bank]] 之后，并处于县内 No.2  位置。核心业务是中小企业特化型本地银行，包括 SUBARU 太田工厂供应链金融，并与 2003  旧第一劝业银行系、现 [[mizuho-fg]] 系建立了广泛战略合作。“东和”是当前法定身份 ^[extracted]。

### 为什么“非控股公司化”重要

在群马县内，与其竞争的 [[gunma-bank]]（第一 1  地方银行）也维持独立银行形态；因此群马县的上市 2 行 家银行都具有 **独立银行 / Prime 上市** 的特别结构 ^[extracted]。放眼第二地方银行整体，[[mebuki-fg]] / [[tomony-hd]] / [[fidea-hd]] / [[jimoto-hd]] / [[proclair-hd]] 等已推进控股公司化，而东和银行仍维持独立形态。背景包括：

- 县内市场集中，控股公司化带来的多元化收益有限；
- 租赁、保证等关联公司可在银行下方管理；
- 与旧第一劝业银行系、现 [[mizuho-fg]] 系的广泛战略合作可补足覆盖能力。

### 业务形态演变系谱

```
群马无尽 1917
  -> 群马相互银行 1942 
  -> 群马相互银行 / 相互银行法体系 1951 
  -> 东和银行 1989 ，普通银行转型 + 更名
```

无尽 -> 相互银行 -> 普通银行 / 第二地方银行，是日本中小企业金融机构的典型业务形态演进路线 ^[extracted]。

### 营业区域地理

- **核心区**：群马县内，尤其是前桥、高崎、太田、桐生、伊势崎、馆林等。
- **邻接县拓展**：埼玉北部、栃木西部、东京都北部一部分。
- **特征**：**群马县中小企业特化** + **SUBARU 太田工厂生态圈**。

### 产业基础特性

- **群马县中小企业集群**：制造业、汽车零部件、电机、食品加工、建设和零售，适合第二地方银行定位。
- **观光**：草津、伊香保等温泉地、富冈制丝厂等观光相关贷款。

### 与旧第一劝业银行系 / Mizuho 的战略合作

**与旧第一劝业银行系的战略合作 2003 ** ^[extracted]：

- 广域系统 / 产品共同开发；
- 大企业向上层交易的银证合作；
- 通过 Mizuho 系基础设施补足独立银行资源；
- 在保持独立形态的同时，通过大型银行联盟补足覆盖能力的典型例。

## 5. 监管 / 政策

- **主管**：金融厅与关东财务局。
- **行业**：第二地方银行协会加盟行；全国地方银行协会框架下的第二 1  地方银行会员 ^[extracted]。
- **资本监管**：Basel III 国内基准行 ^[likely]。
- **存款保险**：存款保险机构加盟。
- **近期政策问题**：
  - 2024 以后：日本银行政策利率正常化、贷款利差改善和地方银行收益红利。
  - 地方银行重组压力：公的资金利用、控股公司化鼓励，vs 维持独立银行判断。
  - SUBARU / 汽车产业 EV 转型：零部件供应链金融结构变化风险。
  - 中小企业继承问题、零零贷款退出和第二地方银行信用压力。
