# 九州 FG (Kyushu FG)

## Wiki route

This entry sits under [[regional-banks/INDEX|regional-banks INDEX]]. Read it against [[regional-banks/fukuoka-fg|ふくおか FG (FFG)]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

以九州南部为根据地的单一地方银行集团（以熊本 + 鹿儿岛 2 县为核心）。由 2015-10-01 [[regional-banks/higo-bank|肥後銀行]] + [[regional-banks/kagoshima-bank|鹿児島銀行]] 的经营整合而成立。**跨县「2 强联合」型整合（与 [[regional-banks/fukuoka-fg]] 的广域根据地整合·[[megabanks/mufg]] 等巨型广域合并不同的模式）**。**伴随 TSMC 熊本进出（2022〜菊阳町）的半导体集群相关融资需求激增** 是近期最大话题。在九州内是与 [[regional-banks/fukuoka-fg]]·[[regional-banks/nishi-nippon-fhd]] 并列的据点 FG，但根据地聚焦于九州**南部**的错位结构。

## 1. 公司概要

**正式名**：株式会社九州金融集团
**英文名**：Kyushu Financial Group, Inc.
**证券代码**：东证 PRIME 7180 / 福证 7180
**设立**：2015-10-01（肥后银行 + 鹿儿岛银行 共同控股公司化）
**总部**：熊本市中央区 + 鹿儿岛市（2 总部制）

### 主要子公司·持股

```
九州 FG（控股公司·上市 7180）
  ├── 肥后银行（100%）── [[regional-banks/higo-bank]] / 熊本县根据地，县内份额压倒性
  │     └── 熊本县 main bank 地位，TSMC 菊阳町相关融资的核心
  ├── 鹿儿岛银行（100%）── [[regional-banks/kagoshima-bank]] / 鹿儿岛县根据地，县内 2 强之一角
  │     └── 鹿儿岛县 main bank，农林渔业·观光融资
  ├── 九州 FG 证券（100%）── 集团证券
  ├── 九州 FG 租赁（100%）── 租赁业务
  └── 关联: 肥后卡 / 鹿儿岛卡 等 ─ 各银行系信用卡
```

### 合并经纬·前身

- 旧 **肥后银行**：
  - 1879 第一百三十五国立银行 设立（熊本，国立银行条例下）
  - 1925 普通银行转换 → 更名 **肥后银行**
  - 战后 作为以熊本县为根据地的地方银行重组
- 旧 **鹿儿岛银行**：
  - 1879 第一百四十七国立银行 设立（鹿儿岛，国立银行条例下）
  - 经普通银行转换，1944 战时统制下县内银行合并 → **鹿儿岛银行** 成立
  - 战后 以鹿儿岛县为根据地的地方银行
- **九州 FG**：2015-10-01 共同控股公司化而成立

> 整合背景：九州南部的人口减少·低利率利差压缩 + 邻县的根据地整合需求。肥后银行（熊本）+ 鹿儿岛银行（鹿儿岛）根据地邻接且重叠小，属「相互互补型」整合。与长崎县那样的县内竞争压缩型（[[regional-banks/fukuoka-fg]] 的十八亲和银行整合 = 公取委审查 3 年延迟）不同，反垄断法门槛相对较低。

### TSMC 相关特需（2022〜2024 的最大话题）★

- **TSMC（台积电）熊本菊阳町进出**：第 1 工厂（JASM，2024-02 开所）+ 第 2 工厂（2027 预定投产）
- [[regional-banks/higo-bank|肥後銀行]]作为**当地 main bank 的相关融资核心**：
  - 半导体供应链企业的进出融资（用地取得 / 设备投资）
  - 本地分包·相关业者的设备更新融资
  - 不动产·宿舍·商业设施相关融资（菊阳町·大津町周边的急速膨胀）

### 根据地战略

- 集中于**九州南部 2 县（熊本 + 鹿儿岛）+ 宫崎一部分**
- 与 [[regional-banks/fukuoka-fg]]（福冈 + 熊本 + 长崎）在熊本县重叠 → 肥后银行根据地占优，[[regional-banks/kumamoto-bank|熊本銀行]]（FFG）为第二地方银行式地位
- 鹿儿岛县为**[[regional-banks/kagoshima-bank|鹿児島銀行]] + [[regional-banks/minami-nippon-bank|南日本銀行]] 的 2 强结构**（南日本银行为第二地方银行，规模上鹿儿岛银行压倒）
- 观光·农业·渔业金融：南九州固有的本地产业（烧酒·农产物·渔业·观光）的融资诀窍

### 竞争·地方银行重组

- **九州内竞争**：与 [[regional-banks/fukuoka-fg]]（FFG）·西日本 FHD（西日本城市银行）并列的九州 3 大 FG，但因根据地南北分工，直接竞争有限
- **巨型银行**（[[megabanks/mufg]] / [[megabanks/smfg]] / [[megabanks/mizuho-fg]]）：大企业·全国展开让步，但本地中坚则地方银行占优

## 4. 监管·政策

- **主管**：金融厅（FSA）+ 九州财务局
- **控股公司监管**：银行法 第 52 条之 17
- **熊本地震应对**（2016-04）：作为受灾地金融机构的复原融资·延期偿付应对的运用知识
- **近期政策论点**：
  - 2022〜 与伴随 TSMC 进出的**经济产业省 / 熊本县的半导体集群政策**协同（补助金方案 / 用地造成 / 人才确保）
  - 2024〜 日银政策利率正常化 → 地方银行利差改善（比巨型受益更大）
  - 2025〜 地方银行重组加速（人口减少 + 数字投资负担）之中南九州 FG 的定位
