# 东京 Kiraboshi FG (Tokyo Kiraboshi FG)

## Wiki route

This entry sits under [[regional-banks/INDEX|regional-banks INDEX]]. Read it against [[regional-banks/fukuoka-fg|ふくおか FG (FFG)]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

罕见的**专注东京都内**的「东京专营地银」集团。2014-10-01 以 **东京都民银行 + 八千代银行** 的经营整合成立「东京 TY 金融集团」→ 2016-04 整合 **新银行东京**（处理旧石原都政的负面遗产）→ 2018-05-01 **3 行合并诞生「Kiraboshi 银行」+ FG 也更名为「东京 Kiraboshi FG」**。地盘收窄至东京都内（神奈川、千叶部分地区），是罕见的地银模式。中小企业金融 + 都心地段的富裕层 + 「Kiraboshi 数字银行」「UI 银行」等数字战略 + BaaS / iBank 合作，采取攻取「东京地区的中小·初创·富裕层」利基市场的战略 ^[extracted]。

### 整合背景：3 行合并的经纬

- **东京都民银行**（1951 设立）：战后专注中小企业金融的地银，东京都出资的遗产
- **八千代银行**（1924 无尽 → 1951 相互 → 1990 普银）：相互银行 → 普通银行转换组，第二地银
- **新银行东京**（2003 设立）：石原都政的象征性政策银行 → 因不盈利而经营问题化 → 作为都政负面遗产的处理于 2016-04 整合
- 整合方式：2014-10 以持股公司方式（TY FG）将都民 + 八千代经营整合 → 2016-04 吸收合并新银行东京 → 2018-05 将3 行完全合并为「Kiraboshi 银行」，FG 名称也统一为「Kiraboshi」^[extracted]

### 名称由来

- **「TY」（旧名）**：东京（**T**okyo）+ 八千代（**Y**achiyo）的首字母，合并时的临时名称

|---|---|---|
| 中小企业金融 | 事业性融资、银团、私募债 | **核心业务** — 旧 都民银行以来的传统领域 |
| 富裕层 | 私人银行、继承、不动产 | 都心地段众多 — 港区·千代田·中央 |

| 证券 | Kiraboshi 证券·Life Design 证券 | 零售证券 |

### 「东京专营地银」模式

- **将地盘收窄至东京都内** — 在全国地银中以「东京都为主战场」的罕见定位 ^[extracted]
- 融合了「相对大型银行的灵活性 + 地银的地缘网络」的利基

### 数字战略

- **UI 银行**（2022-01 服务开始）★：Kiraboshi 银行运营的面向个人的手机专营银行业务，获取 Z 世代·年轻层的渠道 ^[extracted]
- 「Kiraboshi 数字银行」品牌：面向法人的数字分行

### 旧 新银行东京遗产处理

- **新银行东京 整合（2016-04）不是单纯的业务整合，而是接手东京都的政策性负担** ^[extracted]

### 合作·收购战略

- **初创企业联动**：经由 Kiraboshi 资本的 CVC + 东京的 VC 网络

## 4. 监管·政策

- **主管**：金融厅（FSA）+ 关东财务局
- **持股公司监管**：银行法 第 52 条之 17
- **近期政策议题**：
  - 2024〜 日银政策利率正常化 → 国内利率利差改善（地银整体的红利）
  - 2025〜 中小企业的事业承继 / M&A 支援 — 政府对地银角色的期待
