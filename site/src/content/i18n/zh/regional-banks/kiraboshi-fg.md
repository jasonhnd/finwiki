# 东京 Kiraboshi FG（Tokyo Kiraboshi FG）

## Wiki route

本条目位于 [[regional-banks/INDEX|regional-banks INDEX]] 之下。可与 [[regional-banks/fukuoka-fg|ふくおか FG (FFG)]] 对照阅读，以理解同业 / 反差语境；也可与 [[banking/INDEX|banking index]] 一起阅读，以理解更大的系统 / 监管边界。

## TL;DR

罕见的**专注东京都内**的“东京专营地方银行”集团。2014-10-01 年，**东京都民银行 + 八千代银行**经营整合，成立“Tokyo TY Financial Group” -> 2016-04 整合**新银行东京**（处理旧石原都政的负面遗产）-> 2018-05-01 **通过 3 行合并诞生“Kiraboshi Bank”，FG 也改名为“Tokyo Kiraboshi FG”**。地盘聚焦东京都内（以及神奈川、千叶部分地区），是稀有的地方银行模式。其战略是通过中小企业金融 + 都心地段富裕层 + “Kiraboshi Digital Bank”“UI Bank”等数字战略 + BaaS / iBank 合作，争取“东京地区中小企业、初创企业和富裕层”利基 ^[extracted]。

### 整合背景：3 行合并的经过

- **东京都民银行**（1951 设立）：战后专注中小企业金融的地方银行，具有东京都出资 legacy
- **八千代银行**（1924 无尽 -> 1951 相互 -> 1990 普通银行）：相互银行转普通银行组，第二地方银行
- **新银行东京**（2003 设立）：石原都政的象征性政策银行 -> 因亏损产生经营问题 -> 2016-04 整合作为都政负面遗产处理
- 整合方式：2014-10 以控股公司方式（TY FG）整合都民 + 八千代 -> 2016-04 吸收合并新银行东京 -> 2018-05 将 3 行完全合并为“Kiraboshi Bank”，FG 名称也统一为“Kiraboshi” ^[extracted]

### 名称由来

- **“TY”（旧名）**：东京（**T**okyo）+ 八千代（**Y**achiyo）的首字母，是合并时的暂定名

## 2. 事业分部地图

| 分部 | 主力产品 | 特征 |
|---|---|---|
| 中小企业金融 | 事业性融资、银团、私募债 | **核心业务** -- 旧都民银行以来的传统领域 |
| 富裕层 | 私人银行、继承、不动产 | 都心地段多 -- 港区、千代田、中央 |
| 个人贷款 | 住房贷款、卡贷款 | 以都内通勤群体为中心 |
| **数字（UI Bank）** | **智能手机专营银行** | **2022-01 开始** -- 获取 Z 世代和年轻层 ★ |
| 初创企业金融 | 创业融资、CVC（Kiraboshi Capital） | 吸收东京创业生态系统 |
| 不动产 | 都内不动产融资、租赁不动产 | 旧八千代银行系优势 |
| 证券 | Kiraboshi Securities / Life Design Securities | 零售证券 |

### “东京特化地方银行”模式

- **将地盘聚焦在东京都内** -- 在日本全国地方银行中，属于“以东京都为主战场”的稀有定位 ^[extracted]
- 融合“相对 megabanks 的灵活性 + 地方银行地缘网络”的利基

### 数字战略

- **UI Bank**（2022-01 服务开始）★：由 Kiraboshi Bank 运营的面向个人智能手机专营银行，是获取 Z 世代和年轻层的渠道 ^[extracted]
- “Kiraboshi Digital Bank”品牌：面向法人的数字支店

### 旧新银行东京 legacy 处理

- **新银行东京整合（2016-04）并非单纯业务整合，而是接收东京都政策性负担** ^[extracted]

### 合作与收购战略

- **初创企业合作**：通过 Kiraboshi Capital 开展 CVC + 东京 VC 网络

## 4. 监管与政策

- **主管**：金融厅（FSA）+ 关东财务局
- **控股公司监管**：银行法第 52 条之 17

- **近期政策论点**：
  - 2024起，日本银行政策利率正常化 -> 国内息差改善（地方银行整体红利）
  - 2025起，中小企业事业传承 / M&A 支援 -- 政府对地方银行角色的期待
