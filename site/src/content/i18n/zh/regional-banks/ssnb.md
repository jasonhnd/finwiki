# 住信 SBI 网络银行 (SBI Sumishin Net Bank) → 多科莫 SMTB 网络银行

## Wiki 路由

本条目归属于[[regional-banks/INDEX|regional-banks INDEX]]。请将其与[[payment-firms/gmo-aozora-net|GMO あおぞらネット銀行]]（同业 / 对比语境）一并阅读，并与[[banking/INDEX|banking index]]（更广泛的系统 / 监管边界）一并阅读。

## TL;DR

**2025-09-25 东证标准市场退市 → 2025-10-01 NTT 多科莫并表子公司化 → 2025-10-02 SBI HD 资本关系完全解除 → 2026-08-03 拟更名为「多科莫 SMTB 网络银行」**——这家原网络银行经历了戏剧性的母公司更替。它作为三井住友信托 + SBI HD 50:50 JV 于 2007-09-24 开业，在 2023-03-29 PRIME 上市后，仅 2  年半，SBI HD 持股便被 NTT 多科莫收购。**FY2024（2025-03 期）合并净利润 281 億円、经常收益 1,465 億円、存款余额 9 兆 8,216 億円、账户数 825 万 户**，规模仅次于网络银行行业的 [[banking/rakuten-bank|rakuten-bank]]。**住房贷款经办量网络银行 No.1  级** + **日本最大级的 BaaS（NEOBANK）提供商**这 2  大轴心为核心。作为「docomo NEOBANK / d 智能银行」后端的该行，被合作方[[megabanks/ndfg]]本身所吸收，是一个罕见的事例。

## 1. 公司概要

**正式名称**：株式会社住信 SBI ネット銀行 → **株式会社ドコモ SMTB ネット銀行**（2026-08-03 拟更名）
**英文名**：SBI Sumishin Net Bank, Ltd.
**证券代码**：原 东证 STANDARD 7163（**2025-09-25 退市**，原 PRIME 因 2024  的标准市场重组而移行）
**设立**：2007-09-24（同日开业）
**现母公司**（2025-10-02〜）：
  - **NTT 多科莫**：持股比率 **55.37%**（[[megabanks/ndfg]] 旗下，并表子公司）
  - **三井住友信托银行**：持股比率 **44.63%**（表决权为 50:50）
**业态**：网络银行（普通银行牌照）+ BaaS 提供商
**总部**：东京都港区
**机关设计**：设监事会公司

### NEOBANK 服务（BaaS）一览

住信 SBI 网络银行以「NEOBANK」品牌通过 API 向他公司提供银行功能。主要合作伙伴:

| 品牌 | 合作伙伴 | 启动时期 |
|---|---|---|
| **JAL NEOBANK** | 日本航空 | 2020-04 |
| **T NEOBANK** | T 积分・日本（Tポイント・ジャパン） | 2021 |
| **山田 NEOBANK** | 山田 HD | 2021 |
| **おうちダイレクト NEOBANK** | 野村不动产 | 2021 |
| **第一生命 NEOBANK** | 第一生命 | 2022 |
| **docomo NEOBANK / d 智能银行** ★ | [[megabanks/ndfg]] / NTT 多科莫 | 2022 → 2025-10 母公司化 |
| **高岛屋 NEOBANK** | 高岛屋 | 2022 |
| **Open House NEOBANK** | Open House 集团（オープンハウスグループ） | 2023 |
| **京王 NEOBANK** | 京王集团 | - |
| **松井证券（MATSUI Bank）** | 松井证券 | - |
| **三井住友信托 NEOBANK** | 三井住友信托银行（共同母公司） | - |

> NEOBANK 是一种 API，使经营者（航空 / 通信 / 零售 / 不动产 / 寿险）能够以自有品牌向客户提供银行功能（普通存款・转账・卡片贷款等）。住信 SBI 提供银行牌照 + 后端，表层以经营者的应用 / 品牌完成闭环的设计。

### 母公司更替的战略含义（2025）

- **NTT 多科莫一方**：「为追回在银行业上相对于其他通信运营商的落后」是收购的动机。确立对抗 au じぶん银行（KDDI 系）/ PayPay 银行（SoftBank 系）的自有金融基础
- **SBI HD 一方**：通过出售住信 SBI 持股回收资金，将零售战略集中于[[regional-banks/sbi-shinsei-bank|SBI 新生銀行]]（旧新生银行）
- **三井住友信托一方**：合作伙伴从 SBI HD 转为 NTT 多科莫。表决权维持 50:50 ，经营影响力不变

### 差异化的核心

- **住房贷款专精 → BaaS 扩张的 2  大轴心**：开业初期以住房贷款（Flat 35 / 浮动）获取实需，后年将 BaaS 培育为第 2  个轴心
- **BaaS 是与其他网络银行的最大差异化要因**：[[banking/paypay-bank|paypay-bank]] / [[banking/rakuten-bank|rakuten-bank]] / au じぶん银行以自有经济圈内交叉销售为主，住信 SBI 则为外销模式
- **新母公司 NTT 多科莫经济圈**：退市后预计将深化与 d 积分・多科莫线路客户 9,000 万 的联动
- **三井住友信托系列骨干**：对信托・年金・不动产信托功能的访问，是其他网络银行所不具备的强项

### 合作・收购战略（2025-10  以降）

- **NTT 多科莫（新第一大股东）**：2025-10-01〜 并表子公司，[[megabanks/ndfg]] 旗下的核心金融功能。docomo NEOBANK 从「BaaS 合作」升格为「自有银行服务」
- **三井住友信托银行（共同母公司）**：自设立时起的 JV 伙伴，信托・年金基础设施的提供得以延续
- **SBI HD（资本关系解除）**：2025-10-02 完全解除。与 SBI 证券的「混合存款」/ 与 SBI VC Trade 的加密资产联动 API 今后未定
- **与过去母公司 SBI 系列的联动**：与 SBI 证券 / SBI VC Trade 既有的 API 联动如何延续 / 缩减为论点

### 竞争位置（网络银行行业，2025-10  以降）

| 银行 | 母公司 | 强项 | 弱项 |
|---|---|---|---|
| [[banking/rakuten-bank|rakuten-bank]] | 乐天 G | 经济圈积分・账户数最多 | 缺 BaaS |
| **住信 SBI → 多科莫 SMTB** | **NTT 多科莫 + 三井住友信托** | **住房贷款 + BaaS + 多科莫经济圈** | 无自有单独经济圈 |
| [[banking/paypay-bank|paypay-bank]] | LY Corp + SBI | PayPay 联动 | 住房贷款弱 |
| au じぶん银行 | KDDI（旧 MUFG 共同） | au 经济圈 | BaaS 弱 |
| 索尼银行 | 索尼 G | 外币存款 | 规模小 |
| 新生银行系（aozora / SBI 新生）| SBI G | 法人融资 | 零售数字化滞后追赶 |

## 4. 监管・政策

- **主管**：金融厅（FSA）
- **牌照**：普通银行牌照（银行法）
- **2025 TOB**：经银行法第 52  条的 9（银行主要股东认可）・反垄断法事前申报，NTT 多科莫成为第一大股东
- **加密资产联动**：与 SBI VC Trade 的 API 在 SBI 资本关系解除后的延续性为论点
- **BaaS 监管**：2022〜 银行代理业 / 电子结算等代行业 / 认可金融商品中介业 等的制度整备为持续论点
- **近期政策论点**:
  - 2024〜 日本央行政策利率正常化 → 存贷利差改善
  - 2024〜 NISA 扩充 → 借助网络证券联动加速资金流入
  - 2025〜 网络银行对稳定币制度（以大型银行 Progmat 为中心）的应对
