# 住信 SBI 网络银行 -> Docomo SMTB 网络银行

## Wiki route

本条目位于 [[regional-banks/INDEX|regional-banks INDEX]] 之下。可与 [[payment-firms/gmo-aozora-net|GMO あおぞらネット銀行]] 对读，以理解同业 / 对照关系；也可与 [[banking/INDEX|banking index]] 对读，以理解更广的系统和监管边界。

## TL;DR

这是经历了剧烈母公司更替的前网络银行：**2025-09-25 东证 Standard 退市 -> 2025-10-01 成为 NTT Docomo 连结子公司 -> 2025-10-02 与 SBI HD 的资本关系完全解除 -> 计划于 2026-08-03 更名为「Docomo SMTB 网络银行」**。它作为三井住友信托 + SBI HD 50:50  JV 于 2007-09-24 开业，并于 2023-03-29 在 PRIME 上市；仅过 2  年半后，NTT Docomo 收购了 SBI HD 持股。以 **FY2024（2025-03 期）连结纯利润 281 億円、经常收益 1,465 億円、存款余额 9 兆 8,216 億円、账户数 825 万 件** 计，其规模在网络银行业中仅次于 [[banking/rakuten-bank|rakuten-bank]]。核心是 **住房贷款办理额达到网络银行 No.1  级** + **日本最大级 BaaS（NEOBANK）提供方** 这 2  条轴线。原本作为「docomo NEOBANK / d Smart Bank」后台的该行，被合作方 [[ndfg]] 本身吸收，是较少见的案例。

## 1. 公司概要

**正式名称**：株式会社住信 SBI ネット銀行 -> **株式会社ドコモ SMTB ネット銀行**（计划于 2026-08-03 更名）
**英文名称**：SBI Sumishin Net Bank, Ltd.
**证券代码**：原 东证 STANDARD 7163（**2025-09-25 退市**；原 PRIME 因 2024  的 Standard 市场再编而迁移）
**设立**：2007-09-24（同日开业）
**现母公司**（2025-10-02起）：
  - **NTT Docomo**：持股比例 **55.37%**（[[ndfg]] 旗下，连结子公司）
  - **三井住友信托银行**：持股比例 **44.63%**（议决权为 50:50）
**业态**：网络银行（普通银行牌照）+ BaaS 提供方
**总部**：东京都港区
**机构设计**：设有监事会的公司

## 2. 业务分部地图

| 分部 | 主要业务 | 特征 |
|---|---|---|
| **住房贷款** | Flat 35 / 浮动 / 固定 | **网络银行 No.1  级**办理额。长期成长引擎 |
| **个人存款** | 普通存款、定期、外币存款 | 账户数 825 万件 / 存款余额 9.8 兆円（2025-03 期） |
| **证券账户联动** | 与 SBI 证券的 API 混合存款 | **2025-10  SBI 资本关系解除** 后关系能否持续是论点 |
| **加密资产联动** | 与 SBI VC Trade 的 API | 同上，脱离 SBI 系列后的持续性是论点 |
| **个人贷款** | 卡贷、目的型贷款 | 数字化完结审查 |
| **借记 / 卡** | Visa 借记 | 无现金支付 |
| **BaaS（NEOBANK）** ★| 向其他公司提供银行功能（详见下文） | **日本最大级 BaaS 提供方** |

### NEOBANK 服务（BaaS）一览

住信 SBI 网络银行以「NEOBANK」品牌通过 API 向其他公司提供银行功能。主要合作方如下：

| 品牌 | 合作方 | 开始时期 |
|---|---|---|
| **JAL NEOBANK** | 日本航空 | 2020-04 |
| **T NEOBANK** | T Point Japan | 2021 |
| **Yamada NEOBANK** | Yamada HD | 2021 |
| **Ouchi Direct NEOBANK** | 野村不动产 | 2021 |
| **Dai-ichi Life NEOBANK** | 第一生命 | 2022 |
| **docomo NEOBANK / d Smart Bank** ★ | [[ndfg]] / NTT Docomo | 2022  -> 2025-10  母公司化 |
| **Takashimaya NEOBANK** | 高岛屋 | 2022 |
| **Open House NEOBANK** | Open House Group | 2023 |
| **Keio NEOBANK** | 京王集团 | - |
| **松井证券（MATSUI Bank）** | 松井证券 | - |
| **三井住友信托 NEOBANK** | 三井住友信托银行（共同母公司） | - |

> NEOBANK 是让经营者（航空 / 通信 / 零售 / 不动产 / 寿险）以自有品牌向客户提供银行功能（普通存款、转账、卡贷等）的 API。住信 SBI 提供银行牌照和后台，前端则在经营者的应用 / 品牌中完成。

### 母公司更替的战略含义（2025）

- **NTT Docomo 侧**：收购动机是「挽回在银行业相对其他通信运营商的落后」。这是为了对抗 au Jibun 银行（KDDI 系）/ PayPay 银行（SoftBank 系），建立自有金融基础。
- **SBI HD 侧**：通过出售住信 SBI 持股回收资金，并将零售战略集中到 SBI 新生银行（旧新生银行）。
- **三井住友信托侧**：合作方从 SBI HD 变为 NTT Docomo。议决权维持 50:50 ，经营影响力不变。

### 差异化核心

- **住房贷款特化 -> BaaS 扩大的 2  条轴线**：开业初期通过住房贷款（Flat 35 / 浮动）获取真实需求，后期将 BaaS 培育为第 2  条轴线。
- **BaaS 是相对其他网络银行的最大差异化因素**：[[banking/paypay-bank|paypay-bank]] / [[banking/rakuten-bank|rakuten-bank]] / au Jibun 银行以自有经济圈内交叉销售为中心，住信 SBI 则是外部销售模式。
- **新母公司 NTT Docomo 经济圈**：退市后，预计将深化与 d Point 和 Docomo 线路客户 9,000 万 的联动。
- **三井住友信托系后台**：能够接入信托、养老金、不动产信托功能，是其他网络银行不具备的强项。

### 合作 / 收购战略（2025-10 以后）

- **NTT Docomo（新第一大股东）**：2025-10-01起成为连结子公司，是 [[ndfg]] 旗下核心金融功能。docomo NEOBANK 从「BaaS 合作」升级为「自有银行服务」。
- **三井住友信托银行（共同母公司）**：自设立时起的 JV 合作方，继续提供信托和养老金基础设施。
- **SBI HD（资本关系解除）**：2025-10-02 完全解除。与 SBI 证券的「混合存款」/ 与 SBI VC Trade 的加密资产联动 API 今后如何安排尚未确定。
- **与过去母公司 SBI 系列的联动**：与 SBI 证券 / SBI VC Trade 的既有 API 联动如何继续或缩小，是后续论点。

### 竞争位置（网络银行业，2025-10 以后）

| 银行 | 母公司 | 强项 | 弱项 |
|---|---|---|---|
| [[banking/rakuten-bank|rakuten-bank]] | Rakuten G | 经济圈积分、账户数最大 | 没有 BaaS |
| **住信 SBI -> Docomo SMTB** | **NTT Docomo + 三井住友信托** | **住房贷款 + BaaS + Docomo 经济圈** | 没有自身单独经济圈 |
| [[banking/paypay-bank|paypay-bank]] | LY Corp + SBI | PayPay 联动 | 住房贷款弱 |
| au Jibun 银行 | KDDI（原 MUFG 共同） | au 经济圈 | BaaS 弱 |
| Sony Bank | Sony G | 外币存款 | 规模小 |
| 新生银行系（Aozora / SBI 新生）| SBI G | 法人融资 | 零售数字化追赶 |

## 4. 监管 / 政策

- **主管**：金融厅（FSA）
- **牌照**：普通银行牌照（银行法）
- **2025  TOB**：经过银行法第 52  条的 9（银行主要股东认可）和反垄断法事前申报后，NTT Docomo 成为第一大股东。
- **加密资产联动**：与 SBI VC Trade 的 API 在 SBI 资本关系解除后的持续性是论点。
- **BaaS 监管**：2022起，围绕银行代理业 / 电子支付等代行业 / 获认可金融商品中介业等的制度建设仍是持续论点。
- **近期政策论点**：
  - 2024起：日本银行政策利率正常化 -> 存贷利差改善
  - 2024起：NISA 扩充 -> 通过网络证券联动加速资金流入
  - 2025起：网络银行如何应对以 megabank Progmat 为中心的稳定币制度
