# めぶき FG (Mebuki FG)

## Wiki 路径

本词条归属于 [[regional-banks/INDEX|regional-banks INDEX]]。请将其与 [[regional-banks/fukuoka-fg|ふくおか FG (FFG)]] 对照阅读以获取同类/对比语境，并参照 [[banking/INDEX|banking index]] 了解更宏观的系统/监管边界。

## TL;DR

以北关东 2  县（茨城 + 栃木）为地盘的地方银行 FG 大手（以存款规模计为地方银行 FG 第 2-3 位 级）。2016-10-01 由 **常阳银行**（茨城地盘）+ **足利控股**（栃木地盘·将足利银行纳入旗下）共同股份转移而成立。以**「超级区域银行集团」志向 + 进军首都圈 + 数字化/地方创生**为 3  轴。其经由足利银行 **2003 年 破产 → 临时国有化 → 2008 年 民间转让**这一戏剧性历史而走向整合的一点，与其他地方银行 FG 不同。在相邻的首都圈与 [[regional-banks/concordia-fg]]（横滨银行 + 东日本银行）/ [[regional-banks/chiba-bank|chiba-bank-fg]] 形成竞争·摸索联动。^[extracted]

## 1. 公司概要

**正式名称**：株式会社めぶきフィナンシャルグループ ^[extracted]
**英文名**：Mebuki Financial Group, Inc.
**证券代码**：东证 PRIME 7167 ^[extracted]
**设立**：2016-10-01（常阳银行 + 足利控股共同股份转移设立持股公司）^[extracted]
**总部**：东京都中央区八重洲 / 茨城县水户市·栃木县宇都宫市（2  本社制）^[extracted]

### 主要子公司·持股

```
めぶき FG（持股公司·上市 7167）
  ├── 常阳银行（100%）── 茨城县地盘核心
  │     └── 1935  常磐银行 + 五十银行合并设立，茨城县内份额压倒性
  ├── 足利银行（100%）── 栃木县地盘核心
  │     └── 1895  创业，2003  破产·临时国有化 → 2008  民间转让经历
  ├── めぶき证券（100%）── 集团证券（旧常阳证券）
  ├── めぶきリース（100%）── 集团租赁
  ├── めぶきカード（100%）── 集团信用卡
  └── 常阳 Capital / 足利 Capital 等 ── 地域投资·咨询系
```

### 合并经过·前身

- 旧 **常阳银行**：1935-07-30 **常磐银行 + 五十银行**合并设立。茨城县水户市总店。县内份额压倒性的第一地方银行
- 旧 **足利银行**：1895（明治 28）创业，栃木县宇都宫市总店。栃木县份额压倒性的第一地方银行。在战后高速增长期作为北关东有力地方银行扩张
- **足利银行破产**（2003-11-29）：适用《金融再生法》第 102  条 → **临时国有化**。作为地方银行破产中规模最大者，存款保险机构取得全部股份并转为特别危机管理银行
- **足利 HD 成立**（2008-07）：野村控股系投资基金（野村本金融资）+ Next Capital Partners 等 **再生 SPC** 以 1200 億円 规模取得，重组纳入足利控股旗下
- **足利 HD 上市**（2013-12-19）：东证一部·地方银行再上市
- **めぶき FG 成立**（2016-10-01）：常阳 + 足利 HD 共同股份转移 → 诞生北关东广域 FG

### 「超级区域银行集团」志向

- 自整合成立之时起便将 **「超级区域银行集团」** 作为口号：志在超越地方银行框架的广域 + 规模 ^[extracted]
- 以存款规模计为地方银行 FG 第 2-3 位 级（与 [[regional-banks/concordia-fg]] / [[regional-banks/fukuoka-fg]] / [[regional-banks/chiba-bank|chiba-bank-fg]] 并列的最上位集团）
- 以县内 main bank 地位（茨城 + 栃木）+ 进军首都圈的 2  轴实现规模扩大

### 进军首都圈战略

- **县外营业店网**：进军东京·埼玉·千叶·神奈川等。开拓中坚企业 main bank 与富裕层 retail
- 茨城 → 东京东北部 / 栃木 → 埼玉北部的经济圈连接是进军首都圈的地理依据

### 数字化·地方创生

- **推进 DX**：在中期经营计划中强化数字渠道·优化店铺·提高本部业务效率
- **地方创生**：自治体联动·事业承继·地域企业的 M&A 中介·观光振兴等

### 在竞争·地方银行重组中的位置

- **竞争对手**：[[regional-banks/concordia-fg]]（横滨 + 东日本）/ [[regional-banks/chiba-bank|chiba-bank-fg]]（作为首都圈相邻的广域 FG 直接竞争）/ [[regional-banks/fukuoka-fg]]（广域 FG 规模竞争）
- 对于**巨型银行**（[[megabanks/mufg]] / [[megabanks/smfg]] / [[megabanks/mizuho-fg]]），以地方银行整合 + 县内份额实现差别化
- 与 **SBI 第 4  巨型构想**（由 [[megabanks/sbi-hd]] 主导）保持距离，走独自广域路线

## 4. 监管·政策

- **主管**：金融厅（FSA）+ 关东财务局
- **持股公司监管**：银行法第 52  条之 17

- **足利银行国有化处理历史**：2003-11  适用《金融再生法》第 102  条 → 存款保险机构取得全部股份 → 2008-07  民间转让。作为地方银行破产处理的典型·最大案例，是金融行政上的参照例
- **近期政策论点**：
  - 2024〜 日银政策利率正常化 → 地方银行利差改善（较巨型银行受益更大）
  - 2025〜 地方银行重组加速（人口减少 + 数字投资负担）→ 摸索北关东广域联合的下一阶段
  - 与自治体联动·地方创生政策的衔接（地方银行的公共角色）
