# 南都银行（The Nanto Bank）

## Wiki route

本条目位于 [[JapanFG/INDEX|JapanFG index]] 之下。可与 [[regional-banks/ikeda-senshu-hd|池田泉州ホールディングス]] 对照阅读同业 / 对比语境，并与 [[banking/INDEX|banking index]] 阅读更广泛的系统 / 监管边界。

## TL;DR

以奈良县为地盘的单独银行（**未银行持股公司化的地方银行**之一＝奈良县唯一的本店银行）。1934-06-01 由六十八银行 + 八木合同银行 + 御所银行 + 吉野银行 + 大和实业银行 + 第三十四国立银行系等战时整合而成立。简称“**なんと / Nanto**”。**奈良县内份额压倒性（县内 No.1）+ 向大阪府东部、京都府南部的关西 3 县边界区域展开 + 观光业（奈良公园、东大寺、药师寺、春日大社、法隆寺）融资 + 不动产融资**是收益核心。关西经济圈内的竞争对手为 [[resona-hd]]（原关西 Urban 银＝现关西未来银行，曾有 2007 整合谈判不顺的经过）/ [[kyoto-fg]] / [[ikeda-senshu-hd]]；相邻县界则与纪阳银行（和歌山）/ 滋贺银行 / [[tomony-hd]]（原大正银行）竞争。

## 1. 公司概要

**正式名称**：株式会社南都银行
**英文名称**：The Nanto Bank, Ltd.
**证券代码**：东证 PRIME 8367
**创立**：1934-06-01（六十八银行 + 八木合同银行 + 御所银行 + 吉野银行 + 大和实业银行等战时整合）
**本店**：奈良县奈良市桥本町 16
**业态**：单独上市银行（**没有以银行持股公司为顶点进行 FG 化**，这一点与邻近 [[kyoto-fg|京都 FG]]・[[ikeda-senshu-hd|池田泉州 HD]]・[[tomony-hd|トモニ HD]] 等关西地银 FG 化趋势形成对照。虽在 2019 设立了统括非银行子公司的**中间持股公司“南都 Management Service”**，但上市与银行业主体仍为南都银行本体）

### 合并经过 / 前身

- 1878 **第六十八国立银行设立**（奈良，明治期国立银行条例下，于 第 68 号设立）— 南都银行最古老的源流
- 战前期，奈良县内并立六十八银行、八木合同银行、御所银行、吉野银行、大和实业银行等多家地方银行
- **1934-06-01 南都银行成立**：作为战时“一县一行主义”政策的先驱，通过 6 行整合实现奈良县内地银统一
- 1949 东证上市
- 战后作为县内核心银行长期上市，维持奈良县内份额
- **2022-04 东证市场区分调整** → 东证 PRIME 8367（由原东证一部迁移）

### 县内地盘战略

- **奈良县 = 观光大县**：县内拥有奈良公园（东大寺、春日大社）、药师寺、唐招提寺、法隆寺、吉野（樱花、修验道）等众多世界遗产级观光资源。面向观光酒店 / 寺社相关业务者 / 土产商的融资是地银组合的重要分部。
- **不动产融资（大阪府东部）**：仅在奈良县内开展业务时体量有限，因此强化向大阪府东部（东大阪、八尾、柏原、大东等中小企业集聚地）的展开，取得制造业中小企业 main bank 地位。
- 虽然**县内份额压倒性**，但奈良县人口减少（尤其年轻层向大阪流出）较全国平均更明显，是中长期结构性课题。

### 关西经济圈内的竞争结构

- **关西 3 县边界区域的特殊性**：奈良县西接大阪府、北接京都府、南接和歌山县、东接三重县 / 滋贺县，位于 5 府县边界。县内完结型地银经营困难，结构上必然卷入 **关西经济圈整体竞争**。
- **与 megabank（[[mufg]] / [[smfg]] / [[mizuho-fg]]）竞争**：大阪都市部的大企业案件流向 megabank，南都银行以奈良县内 + 大阪府东部的中坚 / 中小企业 main bank 地位应对。
- **与关西地银群竞争**：
  - [[resona-hd|りそな HD]]（原大和银 + 朝日银，含关西未来银行）：关西最大的地银集团，在大阪府全域与南都银直接竞争
  - [[kyoto-fg|京都 FG]]（京都银行）：向京都府全域 + 大阪府北部 + 滋贺县扩张，在京都府南部与南都银接触
  - [[ikeda-senshu-hd|池田泉州 HD]]：大阪府北部（原池田银行）+ 大阪府南部（原泉州银行），在大阪府全域竞争
  - 纪阳银行（和歌山县）：南侧县界接触，在和歌山北部 / 五条区域竞争
  - 滋贺银行：东侧县界接触，在东大寺 / 笠置区域竞争
  - [[tomony-hd|トモニ HD]] 旗下原大正银行：在大阪府直接竞争

### 单独银行的战略意义

- **未设立银行持股公司**这一点，与邻近 [[kyoto-fg|京都 FG]]（2009 单独 FG 化）、[[ikeda-senshu-hd|池田泉州 HD]]（2009 整合）等关西地银 FG 化趋势形成对照（南都银行仍是上市和银行业主体。2019 设立的中间持股公司“南都 Management Service”是非银行子公司统括，并非向银行持股公司体制迁移）
- 单独银行的优点：(1) governance 简化，(2) 经营资源集中于银行本体，(3) 不需要银行持股公司化的追加成本

### 在地银重组中的位置

- **作为奈良县单独地银 leader 维持独立路线**（与 megabank 提携、SBI 第 4 家 mega 构想保持距离）
- 观光业 + 不动产融资这一收益核心在关西中也具有独特性，可能有助于重组时的谈判力。

## 4. 监管 / 政策

- **主管**：金融厅（FSA）+ 近畿财务局
- **银行业监管**：银行法（因是单独银行，银行持股公司监管不适用）
- **近期政策论点**：
  - 2024〜 日本银行政策利率正常化 → 地银利差改善（相对 megabank 受益更大）
  - 2025〜 地银重组加速（人口减少 + 数字投资负担）→ 与关西地银群的合作 / 重组判断
  - 继续维持单独银行，还是转向持股公司化（FG 化）的判断
  - 对观光业组合（入境恢复的顺风 vs 外部冲击脆弱性）的应对
  - 继续向大阪府东部、京都府南部扩展营业区域
