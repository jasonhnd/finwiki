# 株式会社南都银行 (The Nanto Bank)

## Wiki 路径

本条目归属于 [[regional-banks/INDEX|regional-banks INDEX]]。请对照 [[regional-banks/ikeda-senshu-hd|池田泉州ホールディングス]] 阅读以获取同业 / 对比背景，并对照 [[banking/INDEX|banking index]] 了解更宏观的体系 / 监管边界。

## TL;DR

以奈良县为地盘的独立银行（**未实行银行控股公司化的地方银行**之一＝奈良县唯一的总行设于本县的银行）。由 1934-06-01 六十八银行 + 八木合同银行 + 御所银行 + 吉野银行 + 大和实业银行 + 第三十国立银行系等的战时整合而成立。简称「**南都（なんと）**」。收益核心是 **奈良县内份额压倒性领先（县内 No.1）+ 向大阪府东部・京都府南部的关西 3 县交界区域拓展 + 旅游业（奈良公园・东大寺・药师寺・春日大社・法隆寺）融资 + 不动产融资**。在关西经济圈的竞争对手是 [[megabanks/resona-hd]]（旧关西 Urban 银行＝现关西未来银行，与其 2007 整合谈判以破裂告终）/ [[regional-banks/kyoto-fg]] / [[regional-banks/ikeda-senshu-hd]]，在邻接县境则与纪阳银行（和歌山）/ 滋贺银行 / [[regional-banks/tomony-hd]]（旧大正银行）竞争。

## 1. 公司概要

**正式名**：株式会社南都银行
**英名**：The Nanto Bank, Ltd.
**证券代码**：东证 PRIME 8367
**创业**：1934-06-01（六十八银行 + 八木合同银行 + 御所银行 + 吉野银行 + 大和实业银行 等 战时整合）
**总行**：奈良县奈良市桥本町 16
**业态**：独立上市银行（**未形成以银行控股公司为顶点的 FG 化**，这一点与邻近的 [[regional-banks/kyoto-fg|京都 FG]]・[[regional-banks/ikeda-senshu-hd|池田泉州 HD]]・[[regional-banks/tomony-hd|トモニ HD]] 等关西地方银行 FG 化趋势形成对照。此外于 2019 设立了统辖非银行子公司的**中间控股公司「南都管理服务（南都マネージメントサービス）」**，但上市・银行业的主体仍为南都银行本体）

### 合并经纬・前身

- 1878 **第六十八国立银行 设立**（奈良，于明治期国立银行条例下，于 第 68 号设立）— 南都银行最古老的源流
- 战前时期，奈良县内有六十八银行・八木合同银行・御所银行・吉野银行・大和实业银行 等多家地方银行并立
- **1934-06-01 南都银行 成立**：作为战时「一县一行主义」政策的先驱，通过 6 行整合实现奈良县内地方银行的统一
- 1949 东证上市
- 作为战后县内的核心银行长期上市，维持奈良县内份额
- **2022-04 东证市场区分调整** → 东证 PRIME 8367（自旧东证一部迁移）

### 县内地盘战略

- **奈良县 = 旅游业大县**：县内拥有众多世界遗产级旅游资源，如奈良公园（东大寺・春日大社）・药师寺・唐招提寺・法隆寺・吉野（樱花・修验道）等。面向旅游酒店 / 寺社相关业者 / 土产业者的融资是地方银行投资组合的重要分部
- **不动产融资（大阪府东部）**：由于仅靠奈良县内规模有限，因此强化向大阪府东部（东大阪・八尾・柏原・大东 等中小企业集聚地）的拓展。获取制造业中小企业的 main bank 地位
- **县内份额压倒性领先**，但奈良县人口减少（尤其年轻群体外流至大阪）较全国平均更为显著，是中长期的结构性课题

### 关西经济圈的竞争结构

- **关西 3 县交界区域的特殊性**：奈良县西接大阪府・北接京都府・南接和歌山县・东接三重县・滋贺县，位于 5 府县交界处。县内自足型的地方银行经营颇为困难，必然被卷入 **关西经济圈整体竞争** 的结构中
- **与大型银行（[[megabanks/mufg]] / [[megabanks/smfg]] / [[megabanks/mizuho-fg]]）的竞争**：大阪都市区的大企业业务呈现流向大型银行的结构，南都银行以奈良县内 + 大阪府东部中坚・中小企业 main bank 地位与之抗衡
- **与关西地方银行群的竞争**：
  - [[megabanks/resona-hd|りそな HD]]（旧大和银 + 朝日银，含关西未来银）：关西最大的地方银行集团，在大阪府全域与南都银直接竞争
  - [[regional-banks/kyoto-fg|京都 FG]]（京都银行）：向京都府全域 + 大阪府北部 + 滋贺县扩张，在京都府南部与南都银接触
  - [[regional-banks/ikeda-senshu-hd|池田泉州 HD]]：大阪府北部（旧池田银行）+ 大阪府南部（旧泉州银行），在大阪府全域竞争
  - 纪阳银行（和歌山县）：南面县境接触，在和歌山北部 / 五条区域竞争
  - 滋贺银行：东面县境接触，在东大寺 / 笠置区域竞争
  - [[regional-banks/tomony-hd|トモニ HD]] 旗下的旧大正银行：在大阪府直接竞争

### 独立银行的战略意义

- **未设立银行控股公司**这一点，与邻近的 [[regional-banks/kyoto-fg|京都 FG]]（2009 单独 FG 化）・[[regional-banks/ikeda-senshu-hd|池田泉州 HD]]（2009 整合）等关西地方银行 FG 化趋势形成对照（南都银行为上市・银行业的主体。于 2019 设立的中间控股公司「南都管理服务」是对非银行子公司的统辖，并非向银行控股公司体制的过渡）
- 独立银行的优点：(1) 治理简洁，(2) 经营资源集中于银行本体，(3) 无需银行控股公司化的额外成本

### 在地方银行重组中的位置

- **作为奈良县独立地方银行的领头者维持独立路线**（与大型银行的合作・SBI 第 4 大行构想保持距离）
- 旅游业 + 不动产融资这一收益核心在关西也具有较高的独特性，可能在重组时增益其谈判能力

## 4. 监管・政策

- **主管**：金融厅（FSA）+ 近畿财务局
- **银行业监管**：银行法（因系独立银行，不在银行控股公司监管对象之列）
- **近期政策论点**：
  - 2024〜 日银政策利率正常化 → 地方银行利差改善（相对大型银行受益更大）
  - 2025〜 地方银行重组加速（人口减少 + 数字投资负担）→ 与关西地方银行群的合作 / 重组判断
  - 维持独立银行现状，还是转向控股公司化（FG 化）的判断
  - 应对旅游业投资组合（入境游复苏的顺风 vs 对外部冲击的脆弱性）
  - 持续向大阪府东部・京都府南部 扩张营业区域
