# 爱知FG (爱知FG)

## 维基路线

该条目位于 [[JapanFG/INDEX|JapanFG index]] 下。针对同行/对比上下文阅读 [[regional-banks/hachijuni-nagano-bank|八十二長野銀行]]，针对更广泛的系统/监管边界阅读 [[banking/INDEX|banking index]]。

##TL;博士

以爱知县为地盘的地方银行金融集团（东证 PRIME 7389）。通过 **2022-10-03 爱知银行 + 中京银行经营整合**成立，银行子公司已整合至 [[aichi-bank|あいち銀行]]。这是以中京经济圈（名古屋・丰田供应链）为基础的地方银行重组模式。县内竞争对手为 [[nagoya-bank]]（未控股公司化，县内 No.2），邻近同业为 [[juroku-fg]]（岐阜）/[[san-jusan-fg]]（三重）/[[ogaki-kyoritsu-bank]]（岐阜）。^[extracted]

**总公司**：爱知县名古屋市中区 ^[extracted]

### 主要子公司/集团构成

```
あいち FG（持株会社・上場 7389）
  └── [[aichi-bank|あいち銀行]]（100%）── 旧愛知銀行 + 旧中京銀行の統合後 operating bank
        └── 県内支店ネットワーク・中堅中小企業融資
```

#### 爱知银行谱系

- **1910** 爱知银行前身的设立（与原名古屋银行等合并，名古屋地区银行的起源） ^[extracted]
- **1943** 因战时整合而成立的“爱知银行” ^[extracted]
- 作为战后地区金融机构，我们主要为爱知县的中小企业提供贷款。

#### 中京银行谱系

- **1953** 中京银行成立（原第三崇光无神 + 第十七崇光无神合并 → 崇光银行 → 第二地方银行 Fugin 转换） ^[extracted]
- 名古屋市、中小企业、崇光银行所属的典型二线区域银行

#### 管理整合流程

- **2022-10-03 爱知FG成立**（股份转让，两家银行100%成为子公司，新FG在TSE PRIME上市） ^[extracted]
- **2025-01-01 爱知银行 + 中京银行合并**，当前运营银行为 [[aichi-bank|あいち銀行]]。

### 中京经济区集中战略

- **爱知县地盘**：名古屋经济圈 = 日本第 3  大经济圈（仅次于东京・大阪），人口集聚和制造业 GDP 密度高。
- **县内区域银行重组模式**：同县内2 行整合→减少重复分支机构，改善成本结构，通过规模效应增强放贷能力

### 合并协同效应（2022-10  整合 → 2025-01  银行合并）

- 重复门店与总部功能整合→降低成本
- 系统集成→IT投资效率
- 优化人员配置→强化销售能力

### 中京经济区地方银行重组模式

爱知FG是**2 行同县内整合→控股公司→银行合并**的3 阶梯整合模式。
- 阶段1（2022-10）：通过成为控股公司进行业务整合（保留两家银行的品牌）
- 阶段2（计划2025-01 ）：通过银行合并实现名实结合

附近地区银行参考示例：
- [[san-jusan-fg]]（2018  三重银行+大山银行合并 → 2021  三十三银行合并）= 爱知FG的前身型号
- [[juroku-fg]]（2021  Juroku银行成为控股公司）
