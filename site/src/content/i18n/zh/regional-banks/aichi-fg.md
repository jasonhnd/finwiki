# 爱知 FG (Aichi FG)

## Wiki route

本条目归属于 [[regional-banks/INDEX|regional-banks INDEX]]。请将其与 [[regional-banks/hachijuni-nagano-bank|八十二長野銀行]] 对照阅读以获取同业/对比背景，并参照 [[banking/INDEX|banking index]] 了解更广泛的系统/监管边界。

## TL;DR

以爱知县为地盘的地银金融集团（东证 PRIME 7389）。**2022-10-03 爱知银行 + 中京银行 经营整合**而成立，银行子公司已整合至 [[regional-banks/aichi-bank|あいち銀行]]。以中京经济圈（名古屋・丰田供应链）为地盘的地银重组模型。县内竞争对手为 [[regional-banks/nagoya-bank]]（未持股会社化，县内 No.2），周边为 [[regional-banks/juroku-fg]]（岐阜）/[[regional-banks/san-jusan-fg]]（三重）/[[regional-banks/ogaki-kyoritsu-bank]]（岐阜）。^[extracted]

## 1. 会社概要

**正式名**：株式会社爱知金融集团 ^[extracted]
**英名**：Aichi Financial Group, Inc.
**证券代码**：东证 PRIME 7389（2022-10-03 上市）^[extracted]
**设立**：2022-10-03（通过共同股份转移进行经营整合）^[extracted]
**总部**：爱知县名古屋市中区 ^[extracted]

### 主要子公司・集团构成

```
爱知 FG（持股会社・上市 7389）
  └── [[regional-banks/aichi-bank|あいち銀行]]（100%）── 旧爱知银行 + 旧中京银行整合后的 operating bank
        └── 县内分行网络・中坚中小企业融资
```

#### 爱知银行 谱系

- **1910** 爱知银行 前身行设立（旧 名古屋银行 等的合并・名古屋系地银的源流）^[extracted]
- **1943** 通过战时整合成立「爱知银行」^[extracted]
- 战后・作为地区金融机构以爱知县内中坚中小企业融资为主轴

#### 中京银行 谱系

- **1953** 中京银行 设立（旧 第三相互无尽 + 第十七相互无尽 整合 → 相互银行 → 第二地银普银转换）^[extracted]
- 名古屋市内・中小零细地盘，相互银行系第二地银的典型

#### 经营整合过程

- **2022-10-03 爱知 FG 设立**（共同股份转移，两行 100% 子公司化，新 FG 在东证 PRIME 上市）^[extracted]
- **2025-01-01 爱知银行 + 中京银行 合并**，current operating bank 为 [[regional-banks/aichi-bank|あいち銀行]]。

### 中京经济圈集中战略

- **爱知县地盘**: 名古屋经济圈 = 日本第 3 大经济圈（仅次于东京・大阪），人口集聚与制造业 GDP 高密度
- **县内地银重组模型**: 同一县内 2 行整合 → 削减重复店铺・改善成本结构・通过规模效应强化融资能力

### 合并协同效应（2022-10 整合 → 2025-01 银行合并）

- 重复店铺・总部职能整合 → 成本削减
- 系统整合 → IT 投资效率化
- 人员配置优化 → 营业力强化

### 中京经济圈地银重组模型

爱知 FG 是 **同一县内 2 行整合 → 持股会社化 → 银行合并** 的 3 步整合模型。
- 阶段 1（2022-10）: 通过持股会社化进行经营整合（保留两行品牌）
- 阶段 2（2025-01 预定）: 通过银行合并名实合一

近邻地银的参考案例:
- [[regional-banks/san-jusan-fg]]（2018 三重银 + 第三银 整合 → 2021 三十三银行合并）= 爱知 FG 的先行模型
- [[regional-banks/juroku-fg]]（2021 十六银行 持股会社化）
