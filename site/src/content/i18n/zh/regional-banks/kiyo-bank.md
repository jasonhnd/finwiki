# 纪阳银行 (The Kiyo Bank)

## Wiki route

This entry sits under [[JapanFG/INDEX|JapanFG index]]. Read it against [[regional-banks/ikeda-senshu-hd|池田泉州ホールディングス]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

纪阳银行是以和歌山县为地盘的单体银行，是一家具有罕见重组历史的地银：**经历控股公司化后又回到单体银行体制**。1895（明治 28）创业，现法人承接 1985-05-01 纪阳银行 + 和歌山银行等谱系，是县内核心银行。简称 “**きよう**”。证券代码为 **东证 PRIME 8370**（经旧池田银行 8370  路径后由纪阳银行继承）。**控股公司历程：2006-02-20 纪阳 Holdings 设立 → 2006-10-01 和歌山银行经营整合 → 2013-10-01 控股公司解散并并入银行（回归单体银行体制）**。其收益核心是 **和歌山县内压倒性份额（县内 No.1）+ 向大阪府南部（泉州区域）展开 + 旅游业融资（高野山、熊野古道、南纪白滨）+ 第一产业融资（蜜柑、梅、林业）**。

## 1. 公司概要

**正式名称**：株式会社纪阳银行 ^[extracted]
**英文名**：The Kiyo Bank, Ltd.
**证券代码**：**东证 PRIME 8370**（2013-10-01 纪阳 HD 解散时银行重新上市，代码继承旧纪阳 HD 8370 ）^[extracted]
**创业**：1895（明治 28）^[extracted]
**现法人设立**：1985-05-01（通过金融重组形成现纪阳银行）^[extracted]
**总店**：和歌山县和歌山市本町 1  丁目 35  番地 ^[extracted]
**业态**：单体银行（**2013-10-01 纪阳 HD 解散，解除控股公司体制并回归单体银行**）^[extracted]

### 合并历程 / 前身

- **1895（明治 28）创业**：作为明治时期地方银行在和歌山发足，是和歌山县内地银最古老源流之一
- 战前至战后：和歌山县内多家地银并立 → 战时统制经济下地银整合趋势
- **1985-05-01**：现纪阳银行发足（通过金融重组）
- 2006 年 以前：和歌山县内存在 **纪阳银行** + **和歌山银行** 的 2 行 体制 → 县内地银重叠结构
- **2006-02-20 纪阳 Holdings 设立**：控股公司化，将纪阳银行置于旗下，并在东证 8370  上市 ^[extracted]
- **2006-10-01 和歌山银行经营整合**：纪阳 HD 将和歌山银行全资子公司化，并由纪阳银行吸收合并和歌山银行 → 实现县内 1 行 体制 ^[extracted]
- **2013-10-01 纪阳 HD 解散并并入银行**：解散控股公司，由纪阳银行吸收合并纪阳 HD，**回归单体银行体制**，上市代码 8370  由纪阳银行继承 ^[extracted]
- **2022-04  东证市场区分调整** → 东证 PRIME 8370（从旧东证一部移行）

> 整合背景：2006  的和歌山银行整合是 “县内 2 行 → 1 行” 的典型地银重组。**值得注意的是 2013  的控股公司解散** — 在许多地银走向控股公司化（FG 化）的情况下，纪阳反而 **解散控股公司并回到单体银行**。

## 2. 业务板块图

| 板块 | 主要运营者 | 特征 |
|---|---|---|
| 地盘核心 | 纪阳银行本体 | 和歌山县地盘 No.1，县内份额压倒性 |
| 县内零售 | 纪阳银行 | 和歌山县人口约 90 万，个人存款厚 |
| 旅游业融资 | 纪阳银行 | 围绕 **高野山**（真言宗总本山、世界遗产）、**熊野古道**（世界遗产 / 熊野三山）、**南纪白滨**（温泉 / 熊猫）的旅游酒店 / 寺社相关业者 |
| 第一产业融资 | 纪阳银行 | **蜜柑**（和歌山 = 全国 1 位）、**梅**（南高梅 / 纪州梅，全国 1 位）、**林业**（吉野熊野系）、**渔业**（纪州胜浦金枪鱼等） |
| 大阪府南部展开 | 纪阳银行 | 向大阪府南部（泉州区域 = 岸和田、贝冢、泉佐野、泉南）扩张营业区域，覆盖关西国际机场周边物流和旅游相关业者 |
| 关西 3 县边界 | 纪阳银行 | 和歌山县全域 + 大阪府南部 + 三重县南部（纪伊半岛东岸）= 纪伊半岛地银位置 |
| 县外法人 | 纪阳银行（大阪、东京等县外据点） | 跟进和歌山总部企业的县外交易 |

### 县内地盘战略

- **和歌山县 = 旅游业 + 第一产业大县**：
  - **旅游**：县内拥有众多旅游资源，包括高野山（真言宗总本山，空海于 1200 年 开山；世界遗产 “纪伊山地的灵场和参诣道”）、熊野古道（熊野三山、世界遗产）、南纪白滨温泉、Adventure World（熊猫）等
  - **第一产业**：蜜柑全国 1 位、梅（南高梅）全国 1 位、林业（吉野熊野系的和歌山部分）、渔业（纪州胜浦金枪鱼等）
- **第一产业融资**：与 JA Bank（[[norinchukin]] 系统）的竞争 / 分工是结构性论点。支持蜜柑、梅农户的事业承继，并向加工食品业者（梅干、果汁等）融资
- **大阪府南部（泉州区域）扩张**：仅靠和歌山县内体量有限，因此 **强化向大阪府南部（岸和田、贝冢、泉佐野、泉南）的展开**。目标是获得关西国际机场（泉佐野市临空城）周边物流 / 旅游相关企业，以及泉州地区中小制造业的 main bank 位置

### 关西经济圈中的竞争结构

- **纪伊半岛特殊性**：和歌山县占据纪伊半岛西半部，北接大阪府，东接奈良县和三重县，南临太平洋。人口规模使县内完结型地银运营困难，因此结构上必然卷入 **关西经济圈 + 纪伊半岛的竞争**
- **与大型银行（[[mufg]] / [[smfg]] / [[mizuho-fg]]）竞争**：大阪都市部大企业案件流向大型银行，纪阳银行则以 **和歌山县内 + 大阪府南部（泉州区域）的中坚 / 中小企业 main bank 位置** 对抗
- **与关西地银群竞争**：
  - [[ikeda-senshu-hd|池田泉州 HD]]（池田泉州银行）：在大阪府全域（北摄 + 泉州）直接竞争，特别是在大阪府南部泉州区域，旧泉州银行 “泉州” 名称来源与纪阳大阪府南部展开发生碰撞
  - [[nanto-bank|南都銀行]]（奈良县地盘）：在县境（和歌山县东北部 / 五条区域）接触
  - [[kyoto-fg|京都 FG]]：直接接触较少，但作为关西经济圈整体同区间中坚地银并列
  - [[resona-hd|りそな HD]]（含关西未来银行）：关西最大地银集团，在大阪府全域与纪阳直接竞争
- **与三重县南部 / 纪伊半岛东岸接触**：三重县侧由 [[san-jusan-fg|三十三 FG]]（旧三重银行 + 第三银行）作为地盘，在纪伊半岛东岸（熊野市、尾鹫市）存在县境接触

### 控股公司解散的战略意义

- **2013-10-01 纪阳 HD 解散** 是逆地银重组趋势的罕见案例 ^[extracted]
- 单体银行优点：（1）治理简化，（2）经营资源集中于银行本体，（3）不需要控股公司化的额外成本

### 在地银重组中的位置

- 由于纪伊半岛的地理特殊性（人口减少 + 旅游业依赖 + 第一产业比重高），**旅游业 + 第一产业组合** 在关西中独特性较高，可能有助于重组时的谈判力

## 4. 监管 / 政策

- **主管**：金融厅（FSA）+ 近畿财务局
- **银行业监管**：银行法（2013-10-01 以后为单体银行，因此不适用银行控股公司监管）
- **近期政策论点**：
  - 2024 以后，日银政策利率正常化 → 地银利差改善（相对大型银行受益更大）
  - 2025 以后，地银重组加速（人口减少 + 数字投资负担）→ 与关西地银群合作 / 重组判断
  - 继续维持单体银行，还是再次转向控股公司化（FG 化）的判断
  - 应对旅游业组合（入境恢复的顺风 vs 外部冲击脆弱性）
  - 对第一产业（蜜柑、梅、林业）高龄化和后继者不足问题的金融应对（事业承继、农地集约融资）
  - 继续向大阪府南部（泉州区域）扩张营业区域
