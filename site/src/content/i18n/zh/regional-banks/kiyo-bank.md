# 株式会社纪阳银行（The Kiyo Bank）

## Wiki 路径

本条目位于 [[regional-banks/INDEX|regional-banks INDEX]] 之下。应将其与 [[regional-banks/ikeda-senshu-hd|池田泉州ホールディングス]] 对照阅读以了解同行 / 对比背景，并与 [[banking/INDEX|banking index]] 对照以了解更广泛的体系 / 监管边界。

## TL;DR

以和歌山县为地盘的单一银行（具有 **经过控股公司化后又回归单一银行体制** 这一罕见重组历的地银）。1895（明治 28）创业，现法人继承 1985-05-01 纪阳银行 + 和歌山银行 等谱系，为县内核心银行。简称"**きよう**"。证券代码 **东证 PRIME 8370**（经由旧 池田银行 8370 后由纪阳银行继承）。**控股公司经纬：2006-02-20 纪阳控股 设立 → 2006-10-01 和歌山银行 经营整合 → 2013-10-01 控股公司解散・整合进银行（回归单一银行体制）**。**和歌山县内份额压倒性领先（县内 No.1）+ 向大阪府南部（泉州地区）展开 + 观光业（高野山・熊野古道・南纪白滨）融资 + 第一产业（蜜柑・梅・林业）融资** 为收益核心。

## 1. 公司概要

**正式名**：株式会社纪阳银行 ^[extracted]
**英文名**：The Kiyo Bank, Ltd.
**证券代码**：**东证 PRIME 8370**（2013-10-01 纪阳 HD 解散时银行重新上市，代码继承旧 纪阳 HD 8370 ）^[extracted]
**创业**：1895（明治 28）^[extracted]
**现法人设立**：1985-05-01（因金融重组而成立现纪阳银行）^[extracted]
**总行**：和歌山县和歌山市本町 1 丁目 35 番地 ^[extracted]
**业态**：单一银行（**2013-10-01 纪阳 HD 解散使控股公司体制解除，回归单一银行体制**）^[extracted]

### 合并经纬・前身

- **1895（明治 28）创业**：作为明治期的地方银行在和歌山成立，是和歌山县内地银最古老的源流之一
- 战前〜战后：和歌山县内多家地银并立 → 战时统制经济下地银整合的潮流
- **1985-05-01**：现纪阳银行 成立（因金融重组）
- 2006 年以前：和歌山县内有 **纪阳银行** + **和歌山银行** 的 2 行体制 → 县内地银的重复结构
- **2006-02-20 纪阳控股 设立**：将纪阳银行 纳入旗下的控股公司化，在东证 8370 上市 ^[extracted]
- **2006-10-01 和歌山银行 经营整合**：纪阳 HD 将 和歌山银行 完全子公司化，并将 和歌山银行 吸收合并进 纪阳银行 → 实现县内 1 行体制 ^[extracted]
- **2013-10-01 纪阳 HD 解散・整合进银行**：解散控股公司，以纪阳银行吸收合并纪阳 HD 的形式 **回归单一银行体制**，上市代码 8370 由纪阳银行继承 ^[extracted]
- **2022-04 东证市场区分重审** → 东证 PRIME 8370（从旧东证一部迁移）

> 整合背景：2006 的和歌山银行整合是"县内 2 行 → 1 行"的地银重组的典型例。**值得注意的是 2013 的控股公司解散** — 在众多地银走向控股公司化（FG 化）之中，纪阳逆向地 **解散控股公司回归单一银行**，这是罕见的事例。

## 2. 业务分部・地图

| 分部 | 主要经营者 | 特征 |
|---|---|---|
| 地盘核心 | 纪阳银行本体 | 和歌山县地盘 No.1，县内份额压倒性领先 |
| 县内 retail | 纪阳银行 | 和歌山县人口约 90 万，个人存款深厚 |
| 观光业融资 | 纪阳银行 | **高野山**（真言宗总本山，世界遗产）・**熊野古道**（世界遗产・熊野三山）・**南纪白滨**（温泉・熊猫）周边的观光酒店 / 寺社相关经营者 |
| 第一产业融资 | 纪阳银行 | **蜜柑**（和歌山＝全国 1 位）・**梅**（南高梅・纪州梅，全国 1 位）・**林业**（吉野熊野系）・**渔业**（纪州胜浦的金枪鱼等）|
| 大阪府南部展开 | 纪阳银行 | 向大阪府南部（泉州地区 = 岸和田・贝塚・泉佐野・泉南）的营业区域扩张，关西国际机场周边的物流・观光相关经营者 |
| 关西 3 县边界 | 纪阳银行 | 和歌山县全域 + 大阪府南部 + 三重县南部（纪伊半岛东岸）= 纪伊半岛地银地位 |
| 县外法人 | 纪阳银行（大阪・东京等县外据点）| 跟进和歌山总部企业的县外交易 |

### 县内地盘战略

- **和歌山县 = 观光业 + 第一产业大县**：
  - **观光**：高野山（真言宗总本山，空海开山 1200 年・世界遗产"纪伊山地的灵场与参诣道"）+ 熊野古道（熊野三山・世界遗产）+ 南纪白滨温泉 + 冒险大世界（熊猫）等众多观光资源遍布县内
  - **第一产业**：蜜柑全国 1 位・梅（南高梅）全国 1 位・林业（吉野熊野系的和歌山部分）・渔业（纪州胜浦的金枪鱼等）
- **第一产业融资**：与 JA 银行（[[cooperative-banks/norinchukin]] 系统）的竞争・分工是结构性论点。蜜柑・梅农户的事业承继支援、对加工食品业者（梅干・果汁等）的融资
- **大阪府南部（泉州地区）扩张**：因仅靠和歌山县内规模有限，**强化向大阪府南部（岸和田・贝塚・泉佐野・泉南）的展开**。瞄准关西国际机场（泉佐野市临空城）的物流・观光相关经营者、泉州地区中小制造业的 main bank 地位获取

### 关西经济圈的竞争结构

- **纪伊半岛的特殊性**：和歌山县占据纪伊半岛的西半部，北邻大阪府・东邻奈良县与三重县・南临太平洋。县内完结型的地银运营在人口规模上困难，必然被卷入 **关西经济圈 + 纪伊半岛的竞争** 结构
- **与超大型银行（[[megabanks/mufg]] / [[megabanks/smfg]] / [[megabanks/mizuho-fg]]）的竞争**：大阪都市部的大企业案件向超大型银行流出的结构，纪阳银行以 **和歌山县内 + 大阪府南部（泉州地区）的中坚・中小企业 main bank 地位** 对抗
- **与关西地银群的竞争**：
  - [[regional-banks/ikeda-senshu-hd|池田泉州 HD]]（池田泉州银行）：在大阪府全域（北摄 + 泉州）直接竞争，尤其在大阪府南部的泉州地区，同名"泉州"由来的旧泉州银行与纪阳的大阪府南部展开发生冲突
  - [[regional-banks/nanto-bank|南都銀行]]（以奈良县为地盘）：在县境（和歌山县东北部 / 五条地区）接触
  - [[regional-banks/kyoto-fg|京都 FG]]：直接接触较少，但在整个关西经济圈作为同档次的中坚地银并列
  - [[megabanks/resona-hd|りそな HD]]（含关西未来银行）：关西最大的地银集团，在大阪府全域与纪阳直接竞争
- 与 **三重县南部・纪伊半岛东岸** 的接触：三重县一侧由 [[regional-banks/san-jusan-fg|三十三 FG]]（旧 三重银行 + 第三银行）为地盘，在纪伊半岛东岸（熊野市・尾鹫市）有县境接触

### 控股公司解散的战略意义

- **2013-10-01 纪阳 HD 解散** 这一逆地银重组趋势的罕见事例 ^[extracted]
- 单一银行的优点：(1) 治理简化，(2) 经营资源集中于银行本体，(3) 不需要控股公司化的额外成本

### 在地银重组中的位置

- 由于纪伊半岛这一地理特殊性（人口减 + 观光业依赖 + 第一产业比率高），其 **观光业 + 第一产业组合** 在关西之中独特性较高，可能对重组时的谈判力有所贡献

## 4. 监管・政策

- **主管**：金融厅（FSA）+ 近畿财务局
- **银行业监管**：银行法（2013-10-01 以降因系单一银行，故银行控股公司监管不适用）
- **近期政策论点**：
  - 2024〜 日银政策利率正常化 → 地银利差改善（相对而言比超大型银行受益更大）
  - 2025〜 地银重组加速（人口减 + 数字投资负担）→ 与关西地银群的联合 / 重组判断
  - 是维持单一银行，还是再度向控股公司化（FG 化）迁移的判断
  - 对观光业组合（入境恢复的顺风 vs 外部冲击脆弱性）的应对
  - 对第一产业（蜜柑・梅・林业）老龄化・后继者不足问题的金融应对（事业承继・农地集约融资）
  - 持续向大阪府南部（泉州地区）扩张营业区域
