# 伊予银 HD (Iyogin Holdings)

## Wiki 路径

This entry sits under [[regional-banks/INDEX|regional-banks INDEX]]. Read it against [[regional-banks/iyo-bank|伊予銀行]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

以爱媛县为根据地的四国最大级地方银行集团。2022-10-03 由伊予银行单独股份转移实现控股公司化，于东证 PRIME 5830 上市。核心子公司**[[regional-banks/iyo-bank|伊予銀行]]**是于1878 年作为「第二十九国立银行」(松山) 设立的四国最古老级别银行，爱媛县内份额 No.1。以今治造船为中心的**面向海运・造船集群的融资**为其特色。在四国 4 县与 [[regional-banks/tomony-hd]] (德岛・香川) / [[regional-banks/hyakujushi-bank|百十四銀]] (香川) / [[regional-banks/shikoku-bank|四国銀]] (高知) 竞争。

## 1. 公司概要

**正式名称**:株式会社伊予银控股
**英文名**:Iyogin Holdings, Inc.
**证券代码**:东证 PRIME 5830 (2022-10-03 上市)

**总部**:爱媛县松山市南堀端町 1 番地
**机关设计**:监查等委员会设置公司

## 2. 沿革

| 年月 | 事件 |
|---|---|
| 1878 | **第二十九国立银行** (松山) 设立 — 国立银行条例下，四国最古老级别 |
| 1895 | 改组为私立银行 → 「第二十九银行」 |
| 1941 | 因战时统合改称为**伊予银行** (爱媛县内地方银行统合) |
| 1949 | 东京证券交易所 上市 |
| 2022-10-03 | **伊予银 HD 设立** (单独股份转移，将伊予银行完全子公司化) |
| 2024〜 | 中期经营计划启动 |

### 地域基础

- **爱媛县根据地**:人口约 130 万，四国 4 县中最大县。以承担县内 GDP 核心的海运・造船・农业 (柑橘) ・造纸 (大王制纸) 等地方产业为客户基础。
- **海事集群**:今治市是世界屈指可数的造船・海运聚集地。对今治造船集团等的融资是伊予银行的特征性投资组合。
- **四国广域展开**:在爱媛县外的香川・德岛・高知・大阪・东京・大阪・广岛等地亦展开分行。

### 数字战略

- 「**伊予银卡**」「**伊予银网络**」「伊予银 App」等零售数字化
- 面向法人的 API 银行・地域 DX 支援

## 5. 监管・政策

- **主管**:金融厅 (FSA) ・四国财务局
- **控股公司规制**:银行法 第 52 条之 17
- **G-SIB 等**:不适用 (地方银行)
- **近期政策论点**:
  - 2024〜 日银政策利率正常化 → 国内存贷利差改善红利
  - 2024〜 地方银行重组・统合政策 (金融厅的地域金融机构政策)
