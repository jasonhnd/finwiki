# Tomony 控股 (Tomony HD)

## Wiki route

This entry sits under [[regional-banks/INDEX|regional-banks INDEX]]. Read it against [[regional-banks/fukuoka-fg|ふくおか FG (FFG)]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

以四国 (德岛·香川) + 大阪 (大正银行系) 的跨地盘广域展开的地银 FG。通过 **德岛银行 + 香川银行 经营整合** 于 2010-04-01 成立，2017 将大正银行 (旧 北斗商事系·大阪) 救济性子公司化，2020-01 合并德岛 + 大正，诞生了 **[[regional-banks/tokushima-taisho-bank|徳島大正銀行]]**。当前的主要子公司是 [[regional-banks/tokushima-taisho-bank|徳島大正銀行]] + [[regional-banks/kagawa-bank|香川銀行]] 的 2 行体制。^[extracted]

## 1. 公司概要

**正式名**：株式会社 Tomony 控股 ^[extracted]
**英文名**：Tomony Holdings, Inc.
**证券代码**：东证 PRIME 8600 ^[extracted]
**设立**：2010-04-01（德岛银行 + 香川银行 通过共同股份转移经营整合）^[extracted]
**总部**：香川县高松市·德岛县德岛市（2 总部体制）^[extracted]

### 合并历史·前身 ^[extracted]

- 旧 **德岛银行**：1944 设立（旧 德岛合同银行 等的战时统合）
- 旧 **香川无尽**：1943 设立 → 1951 转换为相互银行 → **1989 转换为普通银行 → 香川银行**
- 旧 **大正银行**：1922 设立（大阪地盘）→ 旧 北斗商事系 → 经营困难
- **Tomony HD**：2010-04-01 德岛银行 + 香川银行 通过共同股份转移成立
- **大正银行 子公司化**：2017 救济型 M&A（获得大阪地盘）
- **德岛大正银行**：2020-01 德岛银 + 大正银 合并诞生

### 跨地盘战略

- **四国 (德岛 + 香川)** + **大阪 (旧 大正银行店铺网)** 这种其他四国地银所没有的广域展开
- 阿波舞经济 (德岛) + 乌冬经济 (香川) + 关西经济圈接入 (大正银行系大阪)
- 四国八十八所巡礼旅游金融 + 濑户内造船·四国造纸的制造业金融
- 农业金融 (蜜柑·乌冬面粉·甘薯等本地农产品)

### 合作·收购战略

- **德岛 + 香川 经营整合**（2010-04-01）: 相邻四国 2 县地银的对等合并型 HD 化（同类案例: 山口 FG·西日本 FH 等）
- **大正银行救济子公司化**（2017）: 获得大阪地盘的战略 M&A，吸收经营困难的大正银行
- **德岛大正银行 合并**（2020-01）: HD 内由 2 行 → 1 行体制削减重复成本·整合大阪 + 德岛的店铺网

### 人口减少地区的生存战略

- 四国 4 县在日本也属人口减少率较高 → 以关西经济圈 (大正银行系) 接入加以补充
- 大型银行与信用金库之间的中间利基（本地中小企业 main bank）
- 专注于旅游·农业·制造业的本地产业金融

### 在地银重组脉络中的位置

- 四国地银: Tomony HD ([[regional-banks/tokushima-taisho-bank|徳島大正]] + [[regional-banks/kagawa-bank|香川]]) / [[regional-banks/iyogin-hd]] ([[regional-banks/iyo-bank|伊予銀行]]) / [[regional-banks/awa-bank|阿波銀行]] (德岛) / [[regional-banks/hyakujushi-bank|百十四銀行]] (香川) / [[regional-banks/shikoku-bank|四国銀行]] (高知) 等
- 德岛县内: [[regional-banks/tokushima-taisho-bank|徳島大正銀行]] vs [[regional-banks/awa-bank|阿波銀行]] 的 2 行体制
- 香川县内: [[regional-banks/kagawa-bank|香川銀行]] vs [[regional-banks/hyakujushi-bank|百十四銀行]] 的 2 行体制
- 大阪府内: [[regional-banks/tokushima-taisho-bank|徳島大正銀行]] (旧 大正银行店铺) vs 大型银行 + [[megabanks/smfg]] 系 + 关西 mirai FG 等

## 4. 监管·政策

- **主管**: 金融厅（FSA）+ 四国财务局
- **持股公司监管**：银行法 第 52 条之 17
- **近期政策议题**:
  - 2024〜 日银政策利率正常化 → 国内存贷利差改善红利
  - 2025〜 地银重组压力（金融厅的地银整合诱导政策）
