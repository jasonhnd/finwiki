# 株式会社琉球银行

## Wiki route

This entry sits under [[regional-banks/INDEX|regional-banks INDEX]]. Read it against [[regional-banks/okinawa-fg|おきなわ FG (Okinawa FG)]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

冲绳县内 No.2 地方银行（仅次于 [[regional-banks/okinawa-fg]] 旗下冲绳银行的规模）。**1948-05-01 由美国军政府主导设立，当初兼任琉球列岛的中央银行 + 商业银行 机能**，拥有极为罕见的出身。1972-05 冲绳本土复归后普通银行化，同年于东证上市（现 PRIME 8399）。简称「**りゅうぎん**」。冲绳县全域 + 本土主要城市 + 海外驻在事务所体制。贴近观光业·美军基地经济·离岛金融这一冲绳特有的经营环境。与其他巨型银行·地方银行不同，维持**未控股公司化的单一银行**结构。

## 1. 公司概要

**正式名**：株式会社琉球银行
**英文名**：The Bank of The Ryukyus, Limited
**证券代码**：东证 PRIME 8399
**设立**：1948-05-01（依琉球列岛美国民政府布令第 1 号「关于设立琉球银行的布令」设立）
**本店**：冲绳县那霸市久茂地 1-11-1
**形态**：单一上市银行（未控股公司化，未采用巨型银行·大型地方银行多数采用的 FG 结构）

### 设立的特殊性

琉球银行与通常的民间银行有很大不同的出身：

- **依美国军政府（USCAR / 琉球列岛美国民政府）的布令设立**
- **当初的机能**：商业银行业务 + **中央银行业务（货币发行·银行监督）** 兼任
- 1948 当时，琉球列岛处于 B 圆 → 美军发行军票 → 美元 的货币转换过渡期，琉球银行事实上承担了「琉球的日银」角色
- 1972-05-15 冲绳本土复归后，中央银行机能移交日本银行那霸支店，琉球银行成为普通银行

由于这一历史经纬，琉球银行在冲绳县内的存在感·象征性具有无法与本土地方银行相比的独特分量。

### 主要子公司·关联公司（概要）

```
琉球银行（单一上市 8399）
  ├── りゅうぎんビジネスサービス（事务代办·本店内）
  ├── りゅうぎんジェーシービー（信用卡，JCB 合作）
  ├── りゅうぎんディーシー（信用卡，三菱 UFJ 尼科斯 系 DC 合作）
  ├── りゅうぎん总合研究所（冲绳经济智库，县内屈指）
  ├── りゅうぎんリース（租赁）
  ├── りゅうぎん保证（信用保证）
  └── 关联: 冲绳县内风险投资出资·地域 PE 基金出资 等
```

智库子公司「**りゅうぎん总合研究所**」是冲绳县内经济统计·观光统计的主要发布机构之一。

### 中期经营计划（近期）

- 正在执行 2024 开始的中计
- 重点：**观光业 V 字复苏应对**（新冠后观光复活）、**县内 DX 支援**、**离岛金融维持**、**可持续性（脱碳·海上风电等冲绳案件）**
- 人力资本：县内雇佣维持与本土·亚洲人才的双向流动

### 竞争结构（冲绳县内）

| 行 | 母公司 | 定位 |
|---|---|---|
| **冲绳银行**（8369）| [[regional-banks/okinawa-fg]] 旗下 | 县内 No.1 地方银行，2021-10 控股公司化（冲绳金融集团）|
| **琉球银行**（8399）| 单一上市 | 县内 No.2，维持单一银行 |
| **[[regional-banks/okinawa-kaiho-bank|沖縄海邦銀行]]**（8390）| 单一上市 | 县内 No.3 第二地方银行，规模相当小 |
| **胡差信用金库** / **冲绳县劳动金库** / **JA 银行冲绳** 等 | - | 地域协同组织金融 |

→ **「冲银·琉银的县内二强体制」**长年持续。两行的本店均在那霸市内近距离。在地方公共团体指定金或观光大型案件上错位 / 竞争。第二地方银行海邦银行规模差距大，对二强的挑战有限。

### 与冲绳经济的联动

琉球银行的业绩与冲绳县经济的以下要素强烈联动：

1. **观光客数**（入境 + 国内）— 酒店·度假融资·相关事业者存贷
2. **美军相关支出**（基地交付金 + 在日美军人 个人消费）— 县内 GDP 的重要构成
3. **公共投资**（冲绳振兴预算）— 总承包商·建设向融资
4. **离岛人口·经济**— 县内离岛的人口减少·高龄化是结构风险
5. **冲绳县人口**（与本土不同，长期为自然增长，但近年呈出生率下降倾向）

### 合作·他行联动

- **广域地方银行网络**: 加盟全国地方银行协会，与本土地方银行相互 ATM 合作
- **对亚洲**: 香港驻在员事务所，与台湾·东南亚交易银行的代理行
- **与巨型银行的关系**: 与巨型 3 行（[[megabanks/mufg]]·[[megabanks/smfg]]·[[megabanks/mizuho-fg]]）为协调·授信补充关系
- **数字化**: 地方银行共同中心系核心系统（使用 NTT 数据的地方银行共同中心，※详情须确认 IR）

## 4. 监管·政策

- **主管**: 金融厅（FSA）
- **地域监督**: 福冈财务支局 冲绳财务事务所
- **特殊监管**:
  - 与冲绳振兴特别措施法挂钩的金融优惠制度（冲绳型特定免税店·经济特区相关）
  - 与冲绳县信用保证协会的协同
- **国际**: 与美军相关交易·美元计价账户持有相关的美国制裁法令（OFAC）应对
- **近期政策论点**:
  - 2024〜 日银加息 → 存款利差回升是地方银行整体的顺风
  - 冲绳县观光人口 V 字复苏 → 观光融资资产质量改善
  - 离岛人口减少 → 网点合并裁撤·ATM 维持成本
  - 冲绳振兴预算的中长期动向（本土复归 50 周年后振兴策的重新审视）
