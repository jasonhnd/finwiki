# 福冈 FG (FFG)

## Wiki route

This entry sits under [[regional-banks/INDEX|regional-banks INDEX]]. Read it against [[regional-banks/chiba-bank|千葉銀行 (Chiba Bank)]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

国内最大级的单独地银集团（九州 3 县地盘 + 全国数字化展开）。由 2007-04-02 [[regional-banks/fukuoka-bank|福岡銀行]] + 熊本 Family 银行（现·[[regional-banks/kumamoto-bank|熊本銀行]]）共同持股公司化而成立。**地盘整合（福冈 / 熊本 / 长崎）+ 数字化全国展开（みんなの银行 BaaS）2 轴**。2021-05 开业的 **みんなの银行** 是日本首家全云原生·手机闭环数字银行，对母公司 FFG 而言是数字战略的核心。是地银重组中的主导者之一（与 SBI 第 4 大型构想 / 千叶·横滨·Concordia 等并列的据点）。

## 1. 公司概要

**正式名**：株式会社福冈金融集团
**英文名**：Fukuoka Financial Group, Inc.
**证券代码**：东证 PRIME 8354 / 福证 8354
**设立**：2007-04-02（福冈银行 + 熊本 Family 银行 共同持股公司化）
**总部**：福冈县福冈市中央区天神 2-13-1（福冈银行总行大楼内）
**合并子公司·关联公司**：27 社 + 1 社（权益法）
**合并员工数**：7,928 名（2025-03 期，不含嘱托·临时 2,803 人·执行役员 43 人）

### 主要子公司·持股

```
FFG（持股公司·上市 8354）
  ├── 福冈银行（100%）── [[regional-banks/fukuoka-bank]] / 九州最大地银·地盘核心
  │     └── 以福冈县为中心，大都市 retail + 中坚企业 main bank
  ├── [[regional-banks/kumamoto-bank|熊本銀行]]（100%）── 旧 熊本 Family 银行（旧 肥后 Family 银行）
  │     └── 熊本县地盘
  ├── 十八亲和银行（100%）── [[regional-banks/18-shinwa-bank]] / 2020-10-01 十八银行 + 亲和银行 合并
  │     └── 长崎县地盘（县内份额压倒性）
  ├── みんなの银行（100%）── 日本首家全云原生·数字银行（2021-05-28 开业）
  │     └── 向他社提供 BaaS 功能（支付 API / 开户 API / 合作支店模式）
  ├── FFG 证券（100%）── 集团证券
  └── iBank 营销（100%）── Wallet+ App 运营（地区 Fintech）
```

### 合并经纬·前身

- 旧 **福冈银行**：1945 设立，九州最大地银（含旧 富士银行系·旧三和银行系等遗产）
- 旧 **熊本 Family 银行**：前身为旧 肥后 Family 银行（地方银行），后改名为 [[regional-banks/kumamoto-bank|熊本銀行]]
- 旧 **十八银行**：1877 创立，长崎县地盘（旧 第十八国立银行）
- 旧 **亲和银行**：1939 设立，长崎县地盘（含旧 三和银行系）
- 旧 **熊本银行**（第二地银）：FFG 整合前即存在，需与熊本 Family 改名后的现法人 [[regional-banks/kumamoto-bank|熊本銀行]] 区分

> 整合背景：地银因人口减少 / 低利率挤压利差 → 跨县广域联合与全国数字化展开的 2 轴。十八银行 + 亲和银行 合并因反垄断法应对而延迟 3 年（因长崎县内份额超过半数而受公取委审查）。

### 数字战略（みんなの银行为核心）

- **みんなの银行**（2021-05-28 开业）：
  - 日本首家「云原生·手机闭环」数字银行
  - **以 Google Cloud + Accenture 合作实现全云设计**
  - **2025-05：突破 130 万账户**（开业 4 周年）。以经由 SNS 获取年轻群体为中心
  - **2025-03 末 存款余额 331 億円·贷款余额 228 億円**（同比 +90%）
  - **BaaS 合作伙伴突破 30 社（2025-12 时点）**：
    - 仅合作支店模式：3 社
    - 仅 API 提供模式：8 社
    - 两种模式并用：7 社
    - 已签基本协议书（准备发布中）：12 社
  - **代表性合作方**：Mercari Pay（2025-08 业务合作）等
  - **盈亏平衡目标**：当初 2025 年年度 → 因先行投资 + 贷款余额增长乏力而推迟，**现以 2027 年年度盈亏平衡为新里程碑**
  - **2026 年年度预定**：扩展至以 1 人最多可至 5 支店开户
- **「地银发起的数字基础」的领导志向**：以 BaaS 向其他地银 / 异业提供核心系统的野心
- **iBank 营销**：以 Wallet+ App 展开地区 Fintech

### 地盘整合战略

- **九州 3 县（福冈 / 熊本 / 长崎）地盘整合 + 数字化全国展开** 的 2 轴
- 通过十八亲和银行 整合（2020-10）使长崎县内份额压倒性 → 确立县内 main bank 地位
- 通过 [[regional-banks/kumamoto-bank|熊本銀行]] 子公司化（2007〜）进军熊本县

### 在地银重组中的位置

- 单独地银集团中国内最大级（净利润 854 億円·总资产 33.56 兆円）
- 地银重组的主导者（与 SBI 第 4 大型构想 / [[regional-banks/chiba-bank|chiba-bank-fg]] / [[regional-banks/concordia-fg]] / [[regional-banks/mebuki-fg]] 等并列的广域联合据点）
- 对大型银行（[[megabanks/mufg]] / [[megabanks/smfg]] / [[megabanks/mizuho-fg]]）以地银整合 + BaaS 对抗

## 4. 监管·政策

- **主管**：金融厅（FSA）+ 九州财务局
- **持股公司监管**：银行法 第 52 条之 17
- **十八银行整合 反垄断法审查**：2016 申请 → 2018-12 整合决定（延迟 3 年）。地银整合中公取委审查的典型案例
- **みんなの银行 监管**：银行法 普通银行牌照（全功能银行，与网络专营银行不同的完整功能）
- **近期政策议题**：
  - 2024〜 日银政策利率正常化 → 地银利差改善（恩惠大于大型银行）。FY2025 经常收益 +36.3% 是利率上升的直接反映
  - 2025〜 地银重组加速（人口减少 + 数字投资负担）
  - みんなの银行 BaaS 扩大 → 向其他地银提供核心系统的制度整备
