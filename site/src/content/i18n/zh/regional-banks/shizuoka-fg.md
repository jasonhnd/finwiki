# しずおか FG (Shizuoka FG)

## Wiki route

This entry sits under [[regional-banks/INDEX|regional-banks INDEX]]. Read it against [[regional-banks/fukuoka-fg|ふくおか FG (FFG)]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

以静冈县为地盘的单独地银控股公司（作为地银单体属国内最大级之一角）。2022-10-03 静冈银行通过单独股份转移实现控股公司化而成立。**静冈县内 No.1  地盘（县内份额压倒性・制造业聚集地）+ 蒙古等海外进出 + 与县内同业 [[regional-banks/suruga-bank|スルガ銀行]]・[[regional-banks/shimizu-bank|清水銀行]] 并存**。通过单独地银的控股公司化，确保资本效率改善与 M&A 余地的战略。

## 1. 公司概要

**正式名称**：株式会社しずおかフィナンシャルグループ
**英文名**：Shizuoka Financial Group, Inc.
**证券代码**：东证 PRIME 5831
**设立**：2022-10-03（静冈银行通过单独股份转移实现控股公司化）
**总部**：静冈县静冈市葵区吴服町 1-10（静冈银行本店大楼内）

### 主要子公司・持股

```
しずおか FG（控股公司・上市 5831）
  ├── 静冈银行（100%）── 静冈县地盘核心・县内份额 No.1
  │     ├── 静冈县内营业网络（伊豆〜远州，覆盖制造业聚集地）
  │     ├── 东京・大阪・名古屋 等县外据点
  │     └── 海外驻在员事务所（纽约 / 洛杉矶 / 上海 / 新加坡 等)
  ├── 静银 TM 证券 / 集团证券（旧 静银证券系统）── 县内零售证券
  ├── しずぎん租赁 / 静银 Capital / 静银信用保证 等 ── 集团金融子公司群
  ├── 海外现地法人・关联公司（蒙古 MFG 银行 等）── 旧 Hi-Mongol 银行系的出资
  └── 系统・咨询子公司（しずぎん经营咨询 等)
```

### 合并历史・前身

- 旧 **静冈三十五银行**（明治时期，第三十五国立银行系谱）
- 旧 **浜松银行**（远州地盘）
- 旧 **静冈七十四银行**（第七十四国立银行系谱）
- **1943  静冈银行 设立**：战时整合合并县内主要 3 行（静冈三十五 + 浜松 + 静冈七十四 等）
- **静冈银行 东证一部上市**（作为战后地银核心长期上市，PRIME 迁移为 2022-04  市场区分重新评估）
- **2022-10-03 しずおか FG 设立**：静冈银行通过单独股份转移实现控股公司化（地银单独控股公司化的典型案例）

> 整合背景：通过地银单独控股公司化，旨在 (1) 将子公司化的证券・租赁・信用保证 等集团金融职能整理至控股公司直下以改善资本效率，(2) 确保不局限于银行本体的 M&A・新业态进出余地，(3) 提升县外进出・海外展开的机动性 — 这 3  点。与 [[regional-banks/fukuoka-fg|FFG]] 等先行案例不同，同样的单独 FG 化以「不是合并而是单独股份转移」为特征。

### 重要年表（节选）

| 年月 | 事件 |
|---|---|
| 1943 | **静冈银行 设立**（合并县内主要 3 行）|
| 战后 | 静冈银行 东证一部上市（地银核心）|
| 1980〜 | 确立县内份额 No.1 ，制造业（汽车・乐器・食品）main bank 化 |

| 2024〜 | 中期经营计划（地银单独 FG 化后的战略阶段）|

## 2. 业务分部・地图

| 分部 | 主要事业者 | 特征 |
|---|---|---|
| 地盘核心 | 静冈银行 | 静冈县地盘 No.1，制造业 main bank |
| 县内零售 | 静冈银行・集团证券 | 静冈县人口 360 万，富裕层 / 个人存款厚 |
| 县外法人 | 静冈银行东京・大阪・名古屋据点 | 跟进总部在静冈企业的县外交易 + 关东圈中坚企业 |
| 集团金融 | 静银租赁 / 信用保证 / Capital 等 | 控股公司化后并列再配置 |

### 县内地盘战略

- **静冈县 = 制造业聚集地**：人口约 360 万（全国 10 位 前后），汽车（铃木・雅马哈发动机・丰田系）、乐器（雅马哈・罗兰・河合乐器）、食品・茶 等有力中坚企业聚集。这些 main bank 地位是收益基础
- **与县内同业并存**：
  - [[regional-banks/suruga-bank|スルガ銀行]]（静冈县沼津本店，第二地银）── 县东部的竞争对手，因过去的南瓜马车投资问题正经营重建中
  - [[regional-banks/shimizu-bank|清水銀行 等]] ── 与县内中小地银的竞争・分工
- **县内份额压倒**，但人口减少与年轻层流出是中长期结构课题

### 海外战略

- **海外驻在员事务所** 纽约 / 洛杉矶 / 上海 / 新加坡 / 曼谷 等（静冈银行时代以来的传统全球据点网）
- 支援县内制造业海外进出（汽车供应链・OEM 交易）是其本来的动机

### 控股公司化的战略意义

- **资本效率改善**：将滞留于银行本体的证券・租赁・信用保证 等并列配置于控股公司直下 → 集团整体资源配置灵活化
- **确保 M&A 余地**：以银行本体取得新业态受银行法 约束大，经由控股公司则更灵活
- **数字 / 异业种联动**：在控股公司直下设立・出资 Fintech 子公司更容易
- **单独 FG 化的意义**：选择「不是合并而是单独股份转移」一点，体现了对维持独立性 + 治理简化的志向

### 在地银重组中的位置

- **作为单独地银集团属国内最大级之一角**（与 [[regional-banks/fukuoka-fg|FFG]]・[[regional-banks/chiba-bank|千葉銀行 FG]]・[[regional-banks/concordia-fg|コンコルディア FG]] 等并列的据点）
- **与 SBI 第 4  大型银行构想保持距离** 的独立路线（拥有地盘的强势与制造业 main bank 地位）
- **对大型银行（[[megabanks/mufg]] / [[megabanks/smfg]] / [[megabanks/mizuho-fg]]）以区域密着 + 制造业聚集对抗**

## 4. 监管・政策

- **主管**：金融厅（FSA）+ 东海财务局
- **控股公司监管**：银行法 第 52  条之 17（银行控股公司）
- **海外子公司监管**：蒙古中央银行（Bank of Mongolia）等的现地监督
- **近期政策论点**：
  - 2024〜 日银政策利率正常化 → 地银利差改善（相对大型银行受益更大）
  - 2025〜 地银重组加速（人口减少 + 数字化投资负担）→ しずおか FG 面临维持独立或与邻近地银联合的判断局面
  - 单独 FG 化后的 M&A・新业态进出节奏
