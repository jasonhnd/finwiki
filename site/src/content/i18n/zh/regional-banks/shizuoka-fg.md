# 静冈FG (Shizuoka FG)

## Wiki route

本条目位于 [[regional-banks/INDEX|regional-banks INDEX]] 下。可与 [[regional-banks/fukuoka-fg|ふくおか FG (FFG)]] 对照阅读，以理解同业和差异；更广泛的制度和监管边界见 [[banking/INDEX|banking index]]。

## TL;DR

Shizuoka Financial Group 是以静冈县为地盘的单独地方银行控股公司，也是日本最大级别单体地方银行集团之一。2022-10-03 静冈银行通过单独股份转移转为控股公司后成立。其特征是**静冈县内 No. 1  地盘（县内份额压倒性、制造业集聚地）+ 蒙古等海外布局 + 与县内同业 [[regional-banks/suruga-bank|スルガ銀行]]、[[regional-banks/shimizu-bank|清水銀行]] 并存**。通过单独地方银行控股公司化，集团试图改善资本效率并确保 M&A 空间。

## 1. 公司概要

**正式名称**：株式会社Shizuoka Financial Group
**英文名称**：Shizuoka Financial Group, Inc.
**证券代码**：东证 PRIME 5831

**设立**：2022-10-03（静冈银行通过单独股份转移转为控股公司）
**总部**：静冈县静冈市葵区呉服町 1-10（静冈银行总行大楼内）

### 主要子公司 / 持股

```
Shizuoka FG（控股公司・上市 5831）
  ├── 静冈银行（100%）-- 静冈县地盘核心・县内份额 No. 1
  
  │     ├── 静冈县内营业网络（伊豆至远州，覆盖制造业集聚地）
  │     ├── 东京、大阪、名古屋等县外网点
  │     └── 海外驻在员事务所（纽约 / 洛杉矶 / 上海 / 新加坡 等）
  ├── Shizugin TM Securities / 集团证券（原静银证券系统）-- 县内零售证券
  ├── Shizugin Lease / Shizugin Capital / Shizugin Credit Guarantee 等 -- 集团金融子公司群
  ├── 海外当地法人 / 关联公司（Mongolia MFG Bank 等）-- 原 High Mongolia Bank 系出资
  └── 系统 / 咨询子公司（Shizugin Management Consulting 等）
```

### 合并历史 / 前身

- 原 **静冈三十五银行**（明治期，第三十五国立银行谱系）
- 原 **滨松银行**（远州地盘）
- 原 **静冈七十四银行**（第七十四国立银行谱系）
- **1943  静冈银行设立**：战时整合县内主要 3 行（静冈三十五 + 滨松 + 静冈七十四等）合并
- **静冈银行东证一部上市**：作为战后地方银行核心长期上市；PRIME 迁移发生在 2022-04  市场区分重组
- **2022-10-03 Shizuoka FG 设立**：静冈银行通过单独股份转移转为控股公司（地方银行单独控股公司化的典型案例）

> 整合背景：地方银行单独控股公司化旨在实现以下 3  点：1 将已子公司化的证券、租赁、信用保证等集团金融功能整理至控股公司直属以改善资本效率，2 确保不局限于银行本体的 M&A 和新业态进入空间，3 提高县外和海外布局机动性。与 [[regional-banks/fukuoka-fg|FFG]] 等先行单独 FG 化不同，其特征是“不是合并，而是单独股份转移”。

## 2. 业务分部图

| 分部 | 主要经营者 | 特点 |
|---|---|---|
| 地盘核心 | 静冈银行 | 静冈县地盘 No. 1、制造业 main bank |
| 县内零售 | 静冈银行 / 集团证券 | 静冈县人口 360 万，富裕层 / 个人存款基础深厚 |
| 县外法人 | 静冈银行东京・大阪・名古屋网点 | 跟进总部在静冈企业的县外交易 + 关东圈中坚企业 |
| 集团金融 | Shizugin Lease / 信用保证 / capital 等 | 控股公司化后并列再配置 |

### 县内地盘战略

- **静冈县 = 制造业集聚地**：人口约 360 万（全国 10 位 前后），聚集汽车（Suzuki、Yamaha Motor、Toyota 系）、乐器（Yamaha、Roland、Kawai）、食品、茶等有实力的中坚企业。与这些企业的 main bank 关系是收益基础。
- **与县内同业并存**：
  - [[regional-banks/suruga-bank|スルガ銀行]]（静冈县沼津总部、第二地方银行）-- 县东部竞争者，因过去“南瓜马车”投资问题处于经营重建中。
  - [[regional-banks/shimizu-bank|清水銀行 等]] -- 与县内中小地方银行竞争并分工。
- **县内份额压倒性**，但人口减少和年轻层流出是中长期结构性课题。

### 海外战略

- **海外驻在员事务所** 纽约 / 洛杉矶 / 上海 / 新加坡 / 曼谷等，是静冈银行时代以来的传统全球网点网络。
- 支持县内制造业海外扩张（汽车供应链、OEM 交易）是原本动机。

### 控股公司化的战略意义

- **改善资本效率**：将留在银行本体内的证券、租赁、信用保证等并列配置到控股公司直属 -> 集团整体资源分配更灵活。
- **确保 M&A 空间**：在银行本体内取得新业态受银行法约束较大，经由控股公司则更灵活。
- **数字 / 异业种合作**：更容易在控股公司直属设立或投资 Fintech 子公司。
- **单独 FG 化的意义**：选择“单独股份转移”而非合并，体现维持独立性 + 简化治理的取向。

### 在地方银行重组中的位置

- **作为单独地方银行集团，是日本国内最大级别之一**（与 [[regional-banks/fukuoka-fg|FFG]]、[[regional-banks/chiba-bank|千葉銀行 FG]]、[[regional-banks/concordia-fg|コンコルディア FG]] 等据点并列）。
- 与 SBI 第 4  megabank 构想保持距离，走独立路线（拥有强地盘和制造业 main bank 位置）。
- 面对 megabank（[[megabanks/mufg]] / [[megabanks/smfg]] / [[megabanks/mizuho-fg]]），以地域密着 + 制造业集聚进行对抗。

## 4. 监管 / 政策

- **主管**：金融厅（FSA）+ 东海财务局
- **控股公司监管**：银行法第 52  条之 17（银行控股公司）
- **海外子公司监管**：蒙古中央银行（Bank of Mongolia）等当地监管
- **近期政策论点**：
  - 2024- 日银政策利率正常化 -> 地方银行息差改善（相对 megabank 受益更大）
  - 2025- 地方银行重组加速（人口减少 + 数字投资负担）-> Shizuoka FG 面临维持独立还是与邻近地方银行联合的判断局面
  - 单独 FG 化后的 M&A 和新业态进入速度
