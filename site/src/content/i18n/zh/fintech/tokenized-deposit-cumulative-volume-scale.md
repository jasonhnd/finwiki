# 代币化存款累计交易规模模式 · Kinexys $1.5T 作为银行存款代币领域的锚

## 维基路线

该条目位于 [[fintech/INDEX|fintech index]] 下。使用 [[fintech/japan-financial-regulation|日本金融規制 — トークン・暗号資産・決済に関する法体系]] 来读取相邻上下文，使用 [[fintech/japan-stablecoin-regulatory-landscape|日本 Stablecoin 法制度の三層構造（JPYC・USDC・Project Pax）]] 来读取更广泛的系统边界。

> [!info] TL;DR
> 银行存款代币（deposit token）板块与稳定币板块是**彼此不重叠的 2  个市场**。Kinexys（JPMD）累计交易额为 $1.5T（2026-05），日均 $5B+，拥有 **100% 家机构客户** —— 这一规模不能与 USDC / USDT 的累计流量直接比较，因为客户层、监管层和清算层完全不同。$1.5T 是衡量“代币化存款板块渗透率”的核心锚点 —— 它是第 2  个数字 USD 基础设施，但服务对象是 80  家全球机构（包括 6  家中央银行），而不是零售 / DeFi 用户。

## 关键事实

- Kinexys累计交易金额$1.5T（自2019 启动以来），日均$5B+，月均$150B+ ^[extracted]
- 客户 100% 机构 · ~80 社 （包括 Visa / Mastercard / Ant / 6  央行）^[extracted]
- JPMD = JPM 存款代币（欧元/美元/英镑） 遵循 OCC 美国国家银行章程，**§501  不适用于稳定币** ^[extracted]
- 全球数字美元基础设施排名（累计）：USDT $20T+（零售/灰色地带）> Kinexys $1.5T（机构）> USDC $1.2T（混合）^[extracted]
- Coinbase Base 公链上的 2026-04 JPMD = 银行存款代币 ↔ 稳定币领域第一桥 ^[extracted]

## 机制/如何运作

存款代币与稳定币 = 2  一个受监管部门，2  一个客户群：

```
                  ステーブルコイン(§501)        銀行預金トークン
                  ─────────────                ─────────────────────
発行体            Circle / Stripe / Tether    JPMorgan / Citi / BNY
準備金            国債 / 現金                  銀行預金（FDIC 保護）
規制              GENIUS §501 / MiCA           OCC bank charter / Reg E
顧客              リテール + 機関 + DeFi       機関のみ
利息              保有者への支払い不可         可（銀行預金利息）
代表              USDC / USDT / USDB           JPMD (Kinexys) / Citi TS
```

摩根大通利用存款代币来规避§501 中的“100%政府债券+无利息支付”约束：JPMD持有者本质上是JPM储户，享受银行存款利息+FDIC保护。 Kinexys $1.5T 这些数字证明了纯机构行业的真实规模，该行业将在 5  年内继续由银行（摩根大通/花旗/纽约银行）而非加密原生公司组成的联盟领导。

## 起源与演变

2019 JPM Coin启动（Onyx Coin System私有链/Quorum衍生品）·更名为2024 Kinexys·2025-12 客户数量达到80 社· 2026-04 [[fintech/jpmorgan-jpmd-coin|JPMD]] 登陆 Coinbase Base 公链（通往公链的第一桥） [[fintech/blackrock-buidl-tokenized-mmf-overview|BlackRock BUIDL]] 形成**代币化资产基础设施的双核**——贝莱德走中立军火商路线（向所有 SC 发行人出售 BUIDL），摩根大通走闭环银行路线（JPMD 向客户出售 JPM 服务）。
