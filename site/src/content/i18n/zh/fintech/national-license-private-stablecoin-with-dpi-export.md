# 国家许可型民间稳定币 + DPI 全球输出战略


## Wiki route

This entry sits under [[fintech/INDEX|fintech index]]. Read it with [[fintech/japan-financial-regulation|日本金融規制 — トークン・暗号資産・決済に関する法体系]] for adjacent context and [[fintech/japan-stablecoin-regulatory-landscape|日本 Stablecoin 法制度の三層構造（JPYC・USDC・Project Pax）]] for the broader system boundary.

> [!info] TL;DR
> 印度 ARC token × Polygon 于 2026 Q1  上线 = 全球首个 **国家许可型民间稳定币 + 数字公共基础设施（DPI）全球输出** 模式：政府许可单一民间发行人 + 与 UPI / MOSIP / Aadhaar 集成 + 通过 G20  / 双边协议向新兴市场输出整套技术栈 → 形成 **反美元 DPI 联盟** 叙事。这是新兴市场首次拥有可执行路径，以对抗 USDC / USDT 主导结构。

**3  个构成要素**：

1. **国家许可的单一发行人**：政府许可 1  家民营企业（印度 ARC token 由特定财团运营）
2. **DPI 集成**：稳定币从 day-1  起就接入 UPI（支付）+ MOSIP（身份）+ Aadhaar（KYC）
3. **全球输出 stack**：通过 G20  双边协议向非洲 / 拉美输出完整技术栈（印度已与 30+ 个国家签署 DPI 合作 MOU）

**核心差异（vs USDC / USDT）**：

| 维度 | USDC / USDT | 国家许可型 + DPI |
|---|---|---|
| 锚定货币 | USD | 本国法币 + 区域结算货币 |
| 发行人 | 民营（受美国监管） | 民营，但在本国政府许可下 |
| KYC | 由各平台分别处理 | 直接连接国家身份系统 |
| 跨境清算 | 经由美元离岸体系 | 经由 DPI 双边协议 |
| 制裁敏感度 | 高（OFAC 可冻结） | 低（本国主权） |
| 全球输出 | 无国家战略 | 具国家战略层 |

**印度 3  步（部分已公开）**：

```
Step 1 (2026 Q1): ARC token × Polygon 主网上线
  - 由政府许可的财团发行
  - 与 UPI / Aadhaar 集成
Step 2 (2026 H2): 在 G20  框架下与新兴市场签署 DPI 双边协议
  - 向非洲（尼日利亚、肯尼亚）输出技术栈
  - 向拉美（与巴西 DREX 互操作）输出技术栈
Step 3 (2027+): 反美元 DPI 联盟联合声明
  - 与中国 mBridge / 巴西 DREX 互操作
  - 形成覆盖约 35% 全球 GDP 的非美元结算环
```

**通用性 / 适用性**：

- 巴西 DREX（已部分实施：CBDC + 民间稳定币双轨）
- 尼日利亚 eNaira → 转向民间稳定币（eNaira 失败后的被动选择）
- 沙特 / 阿联酋（地区 AED / SAR 稳定币 + Aramco 推动）
- 中国（PBoC G20  信号 → 离岸 CNH 稳定币于 Q4 2026  试行）
- 广泛适用于“政府强、民间弱、需要将本国法币上链并进行国家战略输出”的新兴市场

**反例 / 边界**：

- 政府 / 民间信任度低的国家（阿根廷、土耳其）→ 民众仍偏好 USDT
- 在美国制裁压力下，DPI 全球输出可能遭到反制（DPI 标准可能被 IMF / 世界银行重新定义）
- 若民间发行人与政府关系不稳定，模型会崩溃（埃及、巴基斯坦的风险）
- 对外部 L1（Polygon / Solana）存在技术依赖，主权独立性会被打折
- 

**估值 / 决策含义**：

- 投资判断：及早参与国家许可型发行人 = 锁定 5–10  年政策租
- 战略买家信号：像 Polygon 这样的公链与印度合作 = 与国家战略形成长期绑定
- 地缘政治风险：国家许可型稳定币 + DPI 输出 = 中美争夺新兴市场的新战场
- 监管对话：印度 ARC token 能否被 [[fintech/aml-cft-fatf-grey-list-overview|IMF / FATF]] 接受 = 判断该模式能否普及的关键 KPI

**与中国 mBridge 的差异**：

- [[fintech/mbridge-bis-multi-cbdc-overview|mBridge]] = CBDC（中央银行直接发行）的多边互操作结算环
- 印度 ARC = 国家许可的民间稳定币（民营效率 + 国家背书）
- 两者目标相同（非美元结算），但路径不同 → 未来可能融合，也可能竞争
- DPI 联盟的完整图景见 [[fintech/india-anti-dollar-dpi-alliance|インド UPI / MOSIP DPI 連盟]]
