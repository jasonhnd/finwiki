# MAS Project Guardian 概览

## Wiki 路径

本文归入 [[fintech/INDEX|fintech index]]。可结合 [[fintech/japan-financial-regulation|日本金融监管]] 阅读其相邻监管背景，也可结合 [[fintech/japan-stablecoin-regulatory-landscape|日本稳定币监管格局]] 理解更大的制度边界。

> [!info] TL;DR
> Project Guardian 是新加坡金融管理局（MAS）于 2022-05 启动的代币化资产概念验证项目。DBS / Citi / JPM / Standard Chartered / HSBC / UBS / Franklin Templeton 等 25+ 家全球金融机构，正在 MAS 监管的 sandbox 内试点代币化债券、基金、外汇和碳信用。它已成为亚洲主导的代币化协同枢纽，并与 Agorá、Ensemble 一起构成“亚洲三极”之一。

## 关键事实

- 2022-05 启动，首批机构为 DBS / JPM / Marketnode ^[extracted]
- 2022-11 的 Pilot 1 中，JPM 在 Aave Arc permissioned pool 上进行了代币化 JPY / SGD 外汇试点 ^[extracted]
- 到 2024-03，参与机构扩展到 25+，并新增基金 / 外汇 / 碳信用领域 ^[extracted]
- 2024-11，Guardian Wholesale Network 启用，成为机构间结算层 ^[extracted]
- 五大试点方向为: 债券 / 代币化基金 / 外汇 / 贸易金融 / 碳信用 ^[extracted]
- MAS 同时参与 BIS Agorá ^[extracted]
- DBS / JPM / Standard Chartered 参与最深，覆盖 3+ 个试点 ^[extracted]
- 全部试点均位于 MAS 监管的 sandbox 内，不面向零售 ^[extracted]

## 机制 / 如何运作

Guardian 可以理解为“代币化资产实验室”。由于它由 MAS 这一单一监管者主导，因此不需要承担多国央行联盟那样的协调成本。25+ 家机构在监管 sandbox 内按资产类别推进**分领域试点**:

1. **债券**: UBS / Citi 进行代币化债券二级交易试点
2. **代币化基金**: Franklin Templeton / Schroders / Fidelity International
3. **外汇**: JPM / DBS / SBI 进行跨货币原子结算试点
4. **贸易金融**: Standard Chartered / Ant International
5. **碳信用**: 与新加坡 Climate Impact X (CIX) 联动

**Guardian Wholesale Network** 于 2024-11 启用，是 MAS 主导的机构级结算层，不依赖主权 CBDC。实质上，它提供的是“代币化资产对代币化资产”的机构结算基础设施。由于 MAS 是单一监管者，决策速度更快。与 [[fintech/bis-project-agora-overview|BIS Project Agorá]] 中 7 家央行协同的成本相比，Guardian 大致以每年一个重要里程碑的节奏推进。MAS 面向零售 / 机构的许可背景，可参见 [[fintech/singapore-mas-payment-services-act-overview|MAS PS Act 概览]] 与 [[exchanges/sg-mas-dpt-licensing-overview|SG MAS DPT 许可概览]]。

## 起源与演进

到 2020 年，MAS 已建立起其 fintech sandbox 框架。到 2022 年，它同时推进两条相邻路线: Project Orchid 侧重零售 CBDC 研究，Project Ubin 侧重早期批发 CBDC 试验。Project Guardian 于 2022-05 启动，明确聚焦“代币化资产”，与更强调“代币化货币”的 mBridge / Ensemble 形成分工。2022 到 2024 年间，试点领域从 3 个扩展至 5 个，参与机构也从 3 家增至 25+ 家。2024-11，Guardian Wholesale Network 上线，使其从“实验项目”进一步走向事实上的结算层。到 2025 年，MAS 通过同时参与 BIS Agorá，推动 Guardian 的代币化资产标准朝向与 Agorá 货币基础设施的可结算兼容性发展。
