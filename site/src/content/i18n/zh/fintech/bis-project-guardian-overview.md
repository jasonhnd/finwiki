# MAS Project Guardian 概述

## Wiki route

> [!info] TL;DR
> Project Guardian 是新加坡金融管理局（MAS）于 2022.05  启动的代币化资产产业实证项目 — 25+ 家全球金融机构（DBS / Citi / JPM / Standard Chartered / HSBC / UBS / Franklin Templeton 等）在 MAS 监管的沙盒环境中，试运行代币化债券、基金、外汇及碳信用额度。该项目是亚洲主导的代币化协作中心，与 Agorá / Ensemble 共同构成"亚洲三极"代币化基础设施分工格局。

## 关键事实

- 2022.05  启动，首批参与方为 DBS / JPM / Marketnode ^[extracted]
- 2022.11  试点 1：JPM 在 Aave Arc 许可型资金池上对日元/新加坡元外汇进行代币化 ^[extracted]
- 2024.03  扩展至 25+ 家机构，新增基金/外汇/碳信用额度领域 ^[extracted]
- 2024.11  Guardian Wholesale Network 启动（机构间结算层）^[extracted]
- 5  主要试点方向：债券 / 代币化基金 / 外汇 / 贸易金融 / 碳信用额度 ^[extracted]
- MAS 同时参与 BIS Agorá ^[extracted]
- DBS / JPM / Standard Chartered 参与最深（覆盖 3+ 个试点）^[extracted]
- 全部试点均在 MAS 监管的沙盒内，无零售敞口 ^[extracted]

## 机制 / 运作方式

Guardian 是"代币化资产实验室"，由 MAS 这一单一监管机构主导，不产生中央银行联盟的协调成本。25+ 家机构在监管沙盒内**按资产类别分线试行**：

1. **债券**：UBS / Citi 代币化债券二级流通
2. **代币化基金**：Franklin Templeton / Schroders / Fidelity International
3. **外汇**：JPM / DBS / SBI 跨币种原子结算
4. **贸易金融**：Standard Chartered / Ant International
5. **碳信用额度**：与新加坡 Climate Impact X（CIX）合作

**Guardian Wholesale Network**（2024.11  启动）是 MAS 主导的机构结算层，不依赖主权 CBDC，实质上提供"代币化资产对代币化资产"的机构结算基础设施。**MAS 单一监管机构 = 决策效率高**：与 [[fintech/bis-project-agora-overview|Agorá]] 中 7  个中央银行的协调成本相比，Guardian 以每年 1  个里程碑的节奏推进。MAS 零售/机构稳定币牌照的配套完善参见 [[fintech/singapore-mas-payment-services-act-overview|MAS PS Act 概要]] 和 [[exchanges/sg-mas-dpt-licensing-overview|SG MAS DPT ライセンス概要]]。

## 起源与演变

2020  MAS 发布金融科技沙盒框架，2022  推进 Project Orchid（零售 CBDC 研究）+ Project Ubin（批发 CBDC 初期实验）的 2  条线。2022.05  启动 Project Guardian，以"代币化资产"为核心差异化定位（与 mBridge / Ensemble 的"代币化货币"形成分工）。2022-2024  试点数量从 3  扩大至 5  个领域，参与机构从 3 行 扩展至 25+ 家。2024.11  Guardian Wholesale Network 启动 → 从"实验"升级为"事实上的结算层"。2025  启动与 BIS Agorá 的互联实验（MAS 同时参与），Guardian 的代币化资产标准可在 Agorá 的货币基础设施上完成结算。
