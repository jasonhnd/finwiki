# HKMA Project Ensemble 概要

## Wiki route

This entry sits under [[fintech/INDEX|fintech index]]. Read it with [[fintech/japan-financial-regulation|日本金融規制 — トークン・暗号資産・決済に関する法体系]] for adjacent context and [[fintech/japan-stablecoin-regulatory-landscape|日本 Stablecoin 法制度の三層構造（JPYC・USDC・Project Pax）]] for the broader system boundary.

> [!info] TL;DR
> Project Ensemble 是香港金融管理局(HKMA)于 2024.03 启动的代币化货币沙盒,以测试香港境内的代币化银行存款 + wholesale CBDC 应用为中心。第 1 批为 HSBC / Standard Chartered / BoCHK / Hang Seng;2024.08 加入 Anchorpoint(非传统银行)。凭借 Ensemble + mBridge 的双轨,香港成为亚洲唯一同时拥有本地代币化结算 + 跨境 CBDC 走廊的金融中心。

## Key facts

- 2024.03 启动,第 1 批为 4 行(HSBC / Standard Chartered / BoCHK / Hang Seng) + HKEX + PolyU ^[extracted]
- 2024.08 Anchorpoint 加入,扩展至非传统银行参加者 ^[extracted]
- 沙盒不向 retail 开放,纯属机构间实验 ^[extracted]
- HKMA 同时推进 e-HKD 零售 CBDC(独立项目) ^[extracted]
- 4 大主题: 代币化资产结算 / PvP / DvP / 绿色金融 ^[extracted]

## Mechanism / How it works

Ensemble 的架构以"代币化港元存款"为中心,商业银行在 HKMA 监督下的沙盒中发行代币化存款代币,机构间通过该代币完成结算。**4 大主题**:
1. 代币化资产结算 — 债券 / 基金的二级市场代币化交易
2. PvP (Payment-vs-Payment) — 港元・代币化存款 vs 外币的原子化 swap
3. DvP (Delivery-vs-Payment) — 代币化资产 vs 代币化港元结算
4. 绿色可持续金融 — 碳信用代币化与气候融资

**Anchorpoint 的战略意义**: 由前 HSBC / Goldman 高管创立,被定位为代币化基础设施服务提供商。入选 Ensemble 表明 HKMA 接纳"金融科技 + 传统银行"混合参与的姿态 → 沙盒不仅是大型银行的实验场,也是新型基础设施提供商的入场券。与 JPM Kinexys / Citi Token Services 之间形成"独立 vs 银行内置"的代币化路径对比(deposit token 的命题见 [[fintech/institutional-stablecoin-deposit-token-thesis|機関 SC と deposit token のテーゼ]])。

## Origin & evolution

2021 HKMA 与 BIS 共同启动 [[fintech/mbridge-bis-multi-cbdc-overview|mBridge]] → 2024.10 BIS 撤出后,由 HKMA + PBoC 接手。2024.03 Ensemble 启动 → HKMA 实现"境内 + 跨境"的双轨战略: 以 Ensemble 让本地金融机构习得代币化存款的 know-how,以 mBridge 使港元成为跨境 CBDC 走廊的"必需货币"。2024.08 Anchorpoint 加入扩展至非银行参加者。2025-2026 [[fintech/hkma-stablecoin-licensing-overview|HKMA Stablecoin Ordinance]] 牌照发放后,Ensemble 沙盒成为事前热身场 — 已取得牌照的 stablecoin 发行体可通过 Ensemble 测试与代币化港元存款的互操作性。
