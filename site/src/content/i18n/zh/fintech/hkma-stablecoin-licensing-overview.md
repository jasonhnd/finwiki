# 香港金管局稳定币牌照 · 香港 Stablecoin Licensing Regime 概览

## Wiki route

This entry sits under [[fintech/INDEX|fintech index]]. Read it with [[fintech/japan-financial-regulation|日本金融規制 — トークン・暗号資産・決済に関する法体系]] for adjacent context and [[fintech/japan-stablecoin-regulatory-landscape|日本 Stablecoin 法制度の三層構造（JPYC・USDC・Project Pax）]] for the broader system boundary.

> [!info] TL;DR
> 香港金融管理局（HKMA）稳定币立法于 2025-08  通过，是亚洲首个完整的稳定币牌照框架。**首批审批结果预计于 2026-05-21 前后公布**（媒体推测，HKMA 未官方确认日期）；9 社 家申请者中，市场预测实际仅 2 件 家获批（汇丰 + 渣打 Anchorpoint），其余均推迟。香港牌照 = 三圈互认协议第三圈候选 + 中国内地唯一可取得的路线。

## 关键事实

- HKMA 稳定币立法：2025-08  通过 ^[extracted]
- 香港港元稳定币 + 美元稳定币双轨牌照 ^[extracted]
- 京东 JD-HKD = 中国内地系大型机构在香港首次申请稳定币发行 ^[extracted]
- 准备金 100%（港元 / 美元短期国债 + 中央银行存款）+ 第三方托管 + 月度审计 ^[extracted]
- 港元 100M+ 资本要求 ^[extracted]

## 运作机制

**HKMA 稳定币牌照核心要求**：100% 准备金（港元 / 美元短期国债 + 中央银行存款）+ 准备金第三方托管 + 月度审计 + 发行方须在香港设立实体 + 港元 100M+ 资本要求 + KYC + AML/CFT + 旅行规则 + 与 OFAC SDN 的协调（通过美元准备金间接实现）+ 与 PBoC / 数字人民币的边界划定（不受理中国内地零售客户）。香港 VASP / OTC 的另一监管轴，参见 [[exchanges/hk-sfc-vasp-licensing-overview|HK SFC VASP ライセンス概観]]。

**9 社 家申请者（2026-05-21 决定）**：
| 申请方 | 背景 | 牌照决定 |
|---|---|---|
| 汇丰 | 全球大型银行 | ✓ 获批 |
| 渣打 Anchorpoint（渣打 + Animoca + HKT） | 联合体 | ✓ 获批 |
| 圆币科技 | 香港本地 | 暂缓 |
| 京东 JD-HKD | 中国内地系 | 暂缓（政治敏感） |
| 蚂蚁国际 | 中国内地系 | 暂缓（政治敏感） |
| 中国银行（BOCHK） | 中资系 | 暂缓（政治敏感） |
| ZA Bank | 数字银行 | 暂缓 |
| Tether | 灰色市场美元领导者 | 暂缓（值得关注） |
| Conflux | 中资公有链 | 暂缓 |

**重要发现**：HKMA 首批实际仅发放 2 件 张牌照，低于预期的 9 件 张。其余 7 社 家推迟或被拒。

## 起源与演变

HKMA 于 2022-12  启动稳定币磋商，2023-12  第二轮征询意见文件确立沙盒名单。2025-08 《稳定币条例》通过（立法历时约 3  年）。2026-05-21 首批决定释放重要信号：HKMA 优先向"政治风险低 + 合规背景强"的全球大型银行（汇丰 / 渣打）发放牌照，中国内地系大型机构 + 灰色市场参与者均推迟。准备金的 FRTB 资本计量框架参见 [[fintech/hk-frtb-stablecoin-reserve-overview|HK FRTB 準備金概観]]，亚洲全景比较参见 [[exchanges/global-vasp-regulatory-comparison-matrix|グローバル VASP 規制比較マトリクス]]。
