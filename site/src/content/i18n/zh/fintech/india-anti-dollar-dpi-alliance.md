# 印度反美元 DPI 联盟——UPI / MOSIP / ARC 三轴叙事

## Wiki route

This entry sits under [[fintech/INDEX|fintech index]]. Read it with [[fintech/japan-financial-regulation|日本金融規制 — トークン・暗号資産・決済に関する法体系]] for adjacent context and [[fintech/japan-stablecoin-regulatory-landscape|日本 Stablecoin 法制度の三層構造（JPYC・USDC・Project Pax）]] for the broader system boundary.

> [!info] TL;DR
> 印度对美元稳定币的官方反对立场并非孤立姿态，而是**完整反美元 DPI 联盟叙事**的组成部分：**UPI 跨境化**（已扩展至 8  个国家）+ **MOSIP 全球输出**（已部署至 20  个国家）+ **ARC 卢比稳定币**（Polygon 链上，政府背景）= 为新兴市场提供 USDT/USDC 的完整替代方案。这是一条与美国 GENIUS Act §501  / 三圈 MRA 框架直接对冲的"数字不结盟"（Digital Non-Alignment）路线。

## 关键事实

- **UPI 跨境**：印度-新加坡走廊年交易额 >$10 億；UPI 接受国 8  个（阿联酋 / 新加坡 / 不丹 / 尼泊尔 / 马尔代夫 / 毛里求斯 / 法国 / 斯里兰卡）
- **UPI 国内交易量**：2026-03  单月 226.4 億 笔 · 29.53 兆 卢比（IMF 认定 = 全球实时支付的 49%）
- **MOSIP 输出**：已部署 20  个国家（埃塞俄比亚 / 菲律宾 / 摩洛哥 / 塞拉利昂 / 多哥 / 斯里兰卡……）· 活跃用户 1.21 億
- **ARC 稳定币**：Polygon × Anq 联合开发 · 2026-Q1  上线 · 仅企业账户可 mint · 与 RBI LRS 兼容
- **政策姿态**：2023  印度 G20  主席国期间推动 IMF-FSB 综合论文，但拒绝简单模仿 GENIUS Act
- **Sandeep Nailwal**：Polygon CEO + ARC 运营方，X 粉丝超 250 万，是该联盟的人格化代言人
- **Nithin Kamath（Zerodha）**：2026-05-12 官方表态"以美元为背景的稳定币对印度是坏主意"——印度顶级金融科技 KOL 形成共鸣
- **JioCoin**：Reliance Jio 4.5 億 用户的 Polygon 链上忠诚度代币，以"非交易性"分类回避 30% 资本利得税

## 机制 / 运作方式

```
        印度反美元 DPI 联盟（三轴）
        │
   ┌────┼──────┬────────────────┐
   ▼    ▼      ▼                ▼
  UPI   MOSIP  ARC          政策框架
  │     │      │            │
  支付  ID     稳定币        反 GENIUS
  │     │      │            │
 跨境  8  20  个国家 Polygon × Anq  RBI / 财政部协调
  国家   输出   2026-Q1        DPI 优先
```

**三轴联锁逻辑**：

1. **UPI（支付层）**：跨境直连消除"需要美元中介"的场景。印度-阿联酋 / 新加坡走廊已绕过美元
2. **MOSIP（身份层）**：将 Aadhaar 技术开源输出至全球南方 20  个国家，与中国"数字丝绸之路"形成对照——掌握新兴市场身份基础设施
3. **ARC（货币层）**：Polygon 链上 INR 稳定币，**仅企业账户可 mint + 白名单互换 + LRS 兼容**——向 RBI 提供 USDT/USDC 的合规替代答案，削弱财政部推动美元稳定币合法化的政治压力

**反美元逻辑链**：USDT/USDC 在印度成为"事实上的美元化"渠道 → 削弱卢比购买力 → 影响货币主权 → 须在国内（ARC）和跨境（UPI）两个方向同时设防。具体的国家许可型稳定币 + DPI 输出模型参见 [[fintech/national-license-private-stablecoin-with-dpi-export|国家許可型民間ステーブルコイン + DPI 輸出]]。
