# T&D 资产管理 (T&D AM)

## Wiki 路径

本条目位于 [[asset-managers/INDEX|asset-managers INDEX]] 之下。可对照 [[asset-managers/nissay-asset-management|ニッセイアセットマネジメント]] 阅读以获取同类/对比语境，对照 [[financial-licenses/securities-license-stack|securities license stack]] 了解 FIEA 牌照边界，对照 [[securities/financial-instruments-business-operators-japan-index|FSA 金商業者 index]] 了解登记层。

## TL;DR

T&Dアセットマネジメント (T&D AM) 是 [[life-insurers/td-hd|T&D ホールディングス (8795)]] 的 asset-management operating company。在 FSA 金融商品交易业者一览中，登载为「T&Dアセットマネジメント株式会社」，关东财务局长(金商)第363号。它承担 **仅次于第一生命 / 日本生命 / 明治安田生命的中坚寿险 FG** [[life-insurers/td-hd|T&D HD]]（太阳生命 + 大同生命 + T&D 金融生命）的运用职能，处理集团寿险责任准备金运用 + 公募投信 + 养老金运用。作为 **中坚寿险系运用公司类别**，处于最大型 [[asset-managers/nissay-asset-management|ニッセイ AM]] 与 [[asset-managers/asset-management-one|アセマネOne]]（第一生命 + 瑞穗 JV）的互补位置。

## 2. 业务角色

- **寿险责任准备金运用**（面向 [[life-insurers/taiyo-life|太陽生命]] / [[life-insurers/daido-life|大同生命]] / T&D 金融生命）
- **公募投资信托运用**（面向个人投资者）
- **养老金运用**（DB / DC / iDeCo）：面向机构投资者・企业年金的运用一任
- **投资一任・投资咨询**（面向机构投资者）

## 3. 出资・治理结构

```
T&D ホールディングス (上場 8795)
  ├── 太陽生命保険 (中堅生保コア)
  ├── 大同生命保険 (中小企業向け生保)
  ├── T&D フィナンシャル生命 (販売チャネル特化生保)
  └── T&D アセットマネジメント (100%)
        ├── 生保責任準備金運用 (グループ生保向け)
        ├── 公募投信運用
        ├── 年金運用
        └── 投資一任 (機関投資家向け)
```

- [[life-insurers/td-hd|T&D HD]] 100% 子公司，集约 3 寿险（太阳 / 大同 / T&D 金融）的运用职能

## 4. 为何需要独立页面

[[life-insurers/td-hd|T&D HD]] 是仅次于 4 大寿险（日本生命 / 第一生命 / 住友生命 / 明治安田生命）的中坚寿险 FG，但承担其运用职能的 **T&D AM** 作为独立分析单位很重要：

1. **中坚寿险系运用公司类别**：填补 [[asset-managers/nissay-asset-management|ニッセイ AM]]（最大型）与 [[asset-managers/asset-management-one|アセマネOne]]（瑞穗 + 第一生命 JV）之间的位置
2. 整合 **太阳生命（专注高龄层）+ 大同生命（面向中小企业主）+ T&D 金融生命（专注销售渠道）** 的 3 异质寿险集团的运用需求
3. 在 **NISA 扩充 + 公募投信竞争** 中的中坚寿险系参与者（[[securities/nisa-2024-flow|nisa-2024-flow]] 语境）
4. 由于是 **上市母公司 (T&D HD)**，处于资本市场披露义务之下的运用业绩透明度（与相互会社系 [[life-insurers/nippon-life|日本生命]] / [[asset-managers/nissay-asset-management|ニッセイ AM]] 不同）
