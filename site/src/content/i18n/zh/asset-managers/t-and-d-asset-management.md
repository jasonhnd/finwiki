# T&D资产管理（T&D AM）

## Wiki 路线

该条目位于 [[asset-managers/INDEX|asset-managers INDEX]] 之下。可结合 [[asset-managers/nissay-asset-management|ニッセイアセットマネジメント]] 阅读同业和对比背景，结合 [[financial-licenses/securities-license-stack|securities license stack]] 理解 FIEA 牌照边界，并结合 [[securities/financial-instruments-business-operators-japan-index|FSA 金商業者 index]] 查看登记层。

## 要点

T&D资产管理（T&D AM）是 [[life-insurers/td-hd|T&D ホールディングス (8795)]] 的 asset-management operating company。在 FSA 金融商品交易业者名单中，它以“T&Dアセットマネジメント株式会社”、关东财务局长（金商）第363号 列示。它承担 **第一生命 / 日本生命 / 明治安田生命 之后的中坚寿险 FG** [[life-insurers/td-hd|T&D HD]]（太阳生命 + 大同生命 + T&D Financial Life）的运用功能，处理集团寿险责任准备金运用、公募投信和养老金运用。作为**中坚寿险系运用公司**，它补充了最大型的 [[asset-managers/nissay-asset-management|ニッセイ AM]] 以及 [[asset-managers/asset-management-one|アセマネOne]]（第一生命 + 瑞穗 JV）等玩家。

## 2. 业务角色

- **寿险责任准备金运用**（面向 [[life-insurers/taiyo-life|太陽生命]] / [[life-insurers/daido-life|大同生命]] / T&D Financial Life）
- **公募投资信托运用**（面向个人投资者）
- **养老金运用**（DB / DC / iDeCo）：面向机构投资者和企业养老金的投资一任
- **投资一任和投资顾问**（面向机构投资者）

## 3. 出资和治理结构

```
T&D Holdings（上市 8795）
  ├── 太阳生命保险（中坚寿险核心）
  ├── 大同生命保险（面向中小企业的寿险）
  ├── T&D Financial Life（销售渠道特化寿险）
  └── T&D Asset Management（100%）
        ├── 寿险责任准备金运用（面向集团寿险）
        ├── 公募投信运用
        ├── 养老金运用
        └── 投资一任（面向机构投资者）
```

- [[life-insurers/td-hd|T&D HD]] 100% 子公司，集中 3  寿险（太阳 / 大同 / T&D Financial Life）的运用功能

## 4. 为什么需要独立页面

[[life-insurers/td-hd|T&D HD]] 是仅次于 4  大型寿险（日本生命 / 第一生命 / 住友生命 / 明治安田生命）的中坚寿险 FG，但承担其运用功能的 **T&D AM** 作为独立分析单元很重要：

1. **中坚寿险系运用公司类别**：位于 [[asset-managers/nissay-asset-management|ニッセイ AM]]（最大型）和 [[asset-managers/asset-management-one|アセマネOne]]（瑞穗 + 第一生命 JV）之间
2. 整合 **太阳生命（面向高龄层）+ 大同生命（面向中小企业主）+ T&D Financial Life（销售渠道特化）** 这一 3  异质寿险集团的运用需求
3. **NISA 扩充 + 公募投信竞争** 中的中坚寿险系玩家（[[securities/nisa-2024-flow|nisa-2024-flow]] 语境）
4. 因母公司为**上市公司（T&D HD）**，其运用表现透明度受资本市场披露义务约束，这一点不同于相互会社系 [[life-insurers/nippon-life|日本生命]] / [[asset-managers/nissay-asset-management|ニッセイ AM]]
