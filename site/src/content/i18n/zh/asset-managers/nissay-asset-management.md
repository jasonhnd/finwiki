# 日生资产管理 (Nissay AM)

## Wiki 路径

本条目位于 [[asset-managers/INDEX|asset-managers INDEX]] 之下。请结合 [[asset-managers/asset-management-one|アセットマネジメントOne]] 阅读以获取同业/对照背景，结合 [[financial-licenses/securities-license-stack|securities license stack]] 了解 FIEA 牌照边界，结合 [[securities/financial-instruments-business-operators-japan-index|FSA 金商業者 index]] 了解登记层。

## TL;DR

日生资产管理 (Nissay AM) 是 [[life-insurers/nippon-life|日本生命保険 (相互会社)]] 的资产管理运营公司。在 FSA 金融商品交易业者一览中以「日生资产管理株式会社」、关东财务局长(金商)第369号 的身份列示。承担**国内最大的人寿保险公司** [[life-insurers/nippon-life|日本生命]] (总资产逾 80 兆円，相互会社) 的运用职能，以寿险责任准备金运用 (规模约逾 75 兆円) + 公募投信 + 养老金运用为核心。是**寿险系运用公司类别中的最大者**，与大型银行系 ([[asset-managers/mufg-asset-management|MUAM]] / [[asset-managers/asset-management-one|アセマネOne]]) 及财险系 ([[asset-managers/tokio-marine-asset-management|東京海上 AM]]) 不同，其差异化轴在于**寿险责任准备金运用基础的规模 + 相互会社母公司的独立性**。

## 2. 业务角色

- **公募投资信托运用** (面向个人投资者): 以「日生指数系列」「日生外国股票指数基金」等低成本投信扩大面向个人的市场
- **养老金运用** (DB / DC / iDeCo): 面向机构投资者・企业年金的全权委托运用
- **投资全权委托・投资咨询** (面向机构投资者)

## 3. 出资・治理结构

```
日本生命保险相互会社 (相互会社、非上市)
  ├── 日生资产管理 (100% 出资)
  ├── 大树生命保险 (子公司、旧三井生命)
  ├── 日生・财富生命 (子公司)
  └── 关联事业公司 (信托・代理店・运用相关)
```

- 并非大型银行系运用公司那样的合资 (JV) 结构，而是寿险单独 100% 模式

## 4. 为何独立页面重要

[[life-insurers/nippon-life|日本生命]] 是国内最大的相互会社人寿保险公司，承担其运用职能的 **Nissay AM** 作为独立分析单位十分重要：

1. **寿险系运用公司的最大者**：与 [[asset-managers/asset-management-one|アセマネOne]] (瑞穗 + 第一生命 JV) 及 [[asset-managers/tokio-marine-asset-management|東京海上 AM]] (财险) 并列，但以寿险单独 100% 出资，规模突出
2. **低成本・指数投信竞争**：「日生外国股票指数基金」等的信托报酬处于业界最低水平，与 [[asset-managers/sbi-asset-management|SBI AM]] / [[asset-managers/mufg-asset-management|MUAM]] (eMAXIS Slim) 等直接竞争
3. **NISA 扩充 + 个人投资者的低成本取向** 的主要受益方 ([[securities/nisa-2024-flow|nisa-2024-flow]] 语境)
4. **寿险责任准备金运用 (逾 75 兆円)** 对国内债券市场・股票市场・另类投资运用的影响
