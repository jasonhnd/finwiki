# めぶきリース (Mebuki Lease)

## Wiki route

本条目位于 [[leasing-firms/INDEX|leasing-firms INDEX]] 之下，作为 [[regional-banks/mebuki-fg|めぶき FG]] 的 FG 子公司租赁经营公司锚点。请将其与母体经营银行 [[regional-banks/joyo-bank|常陽銀行]] / [[regional-banks/ashikaga-bank|足利銀行]] 以及同属子公司 [[securities-firms/mebuki-securities|めぶき証券]] 对照阅读。同业比较: [[leasing-firms/hokkaido-lease|北海道リース]]（北洋・北海道银行系）与大型出租人 [[leasing-firms/mitsubishi-hc-capital|三菱 HC キャピタル]] / [[leasing-firms/mizuho-lease|みずほリース]]。更广泛的系统路径参见 [[banking/regional-bank-consolidation-pattern|regional-bank consolidation pattern]] 与 [[banking/INDEX|banking index]]。

## TL;DR

株式会社めぶきリースは、[[regional-banks/mebuki-fg|めぶき FG]]（东证 PRIME 7167）100% 子公司的地银系租赁公司，**以 1974-09  由 [[regional-banks/joyo-bank|常陽銀行]] 设立的「常陽リース」为母体，于 2017-04-03 由 めぶき FG 取得全部股份并更名为「めぶきリース」**而成立（经官方沿革・法人登记确认）。同日，原 [[regional-banks/ashikaga-bank|足利銀行]] 子公司「足利信用保証」所经营的租赁业务被一体化（业务承继）至 めぶきリース，并开设宇都宫・足利营业部向栃木一侧拓展（经日经 2017-03-07・官方沿革确认）。**就法人格而言这是常陽リース的单独更名，而非「常陽リース + 足利リース 的对等合并」（并不存在名为「足利リース」的独立公司，足利一侧的租赁业务由足利信用保証运营）。** 面向茨城・栃木・首都圈中坚企业 main bank 客户提供设备租赁・融资租赁・经营租赁・厂商租赁等。地银系租赁上位的一角。^[extracted]

### 旧 常陽リース系谱（母体）

- **1974-09**: [[regional-banks/joyo-bank|常陽銀行]] 作为租赁・融资部门的补充业务设立「常陽リース株式会社」（经官方沿革确认）
- 作为 [[regional-banks/joyo-bank|常陽銀行]] 系列租赁公司，以面向茨城县内中坚企业的设备租赁为主业
- 战后地银系租赁的典型: 以设备租赁形态而非银行贷款支持中小企业设备投资
- **2017-04-03**: めぶき FG 取得全部股份并更名为「めぶきリース株式会社」（经法人登记确认）

### 与足利银行系租赁职能的关系 — 已确定（2026-05-30 验证）

- **并不存在名为「足利リース」的独立公司。** [[regional-banks/ashikaga-bank|足利銀行]] 系的租赁公司历史上为「北関東リース株式会社」（金融厅 2003-03-11 事业重构计划认定，资本金 145 百万円），后年则由 **「足利信用保証株式会社」以行员出向的形式经营租赁业**（日经 2017-03-07）。旧版的「常陽リース + 足利リース 的对等合并」这一 framing，在公司名与整合形态两方面均为误。
- **足利一侧的租赁业务承继确实发生过。** 于 2017-04-03 **将足利信用保証的租赁业务一体化（业务承继）至 常陽リース（同日更名为「めぶきリース」）**（经日经 2017-03-07・めぶきリース官方沿革确认）。随之 めぶきリース 开设宇都宫・足利营业部向栃木一侧拓展，足利银行继续派出行员出向。亦即「承继了足利信用保証的租赁业务」一点**可以确认**（但承继对象・形态并非『与足利リース的对等合并』，而是向 常陽リース＝めぶきリース 的业务一体化）。
- 足利信用保証其后于 2020-10-01 由 めぶき FG 取得全部股份并更名为「めぶき信用保証株式会社」，现为信用保证专营（租赁业务早已于 2017-04  移交至 めぶきリース）（日经 2020-07・集团公司信息）。
- [[regional-banks/ashikaga-bank|足利銀行]] 经历过 2003-11  一时国有化局面，后于 2016-10  整合进 めぶき FG，上述 2017-04  的租赁一体化是该整合之后集团职能重组的一环。

### めぶき FG 设立与集团整合

- **2016-10-01**: [[regional-banks/mebuki-fg|めぶき FG]] 设立 ([[regional-banks/joyo-bank]] + [[regional-banks/ashikaga-bank]] 共同股份转移)
- **2017-04-03**: めぶき FG 取得常陽リース全部股份并更名为「めぶきリース」 — 作为面向银行子公司 2 行 客户基础的集团共通租赁职能加以重组

### FG 内定位

- **银租赁联动的核心**: 受托 [[regional-banks/mebuki-fg]] 银行 2 行 的中坚企业 main bank 客户的租赁需求
- **设备投资支援**: 银行贷款补充 + 对表外处理需求的应对
- **县内自治体・公共部门联动**: 茨城县・栃木县面向市町村的租赁项目

### 竞争格局

- **大型租赁**: [[leasing-firms/mitsubishi-hc-capital|三菱 HC キャピタル]] / [[leasing-firms/smfl|SMFL]] / [[leasing-firms/mizuho-lease|みずほリース]] / [[leasing-firms/fuyo-lease|芙蓉リース]] / [[leasing-firms/tokyo-century|東京センチュリー]] / [[leasing-firms/ricoh-leasing|リコーリース]] - 在大型・全国・复杂项目上占优
- **地银系租赁**: [[leasing-firms/hokkaido-lease|北海道リース]] / 千葉興業リース / 群馬リース 等 - 同型 model
- **厂商系租赁**: [[card-issuers/toyota-finance|トヨタファイナンス]] / [[leasing-firms/nec-leasing|NEC キャピタルソリューション]] - 产品联动
- **县内竞争**: [[regional-banks/tsukuba-bank|筑波銀行]] 系・[[regional-banks/tochigi-bank|栃木銀行]] 系租赁

### 优势与劣势

- 优势: [[regional-banks/joyo-bank]] / [[regional-banks/ashikaga-bank]] 客户基础 (茨城・栃木的中坚企业 main bank) + 地区紧密的现场应对

## 7. 规制・政策

- **主管**: 金融厅 (FSA) (租赁业单独而言不需银行业牌照，贷金业登记视对应领域而定)
- **规制论点**:
  - 租赁会计准则变更 (IFRS16 / 日本准则) 应对支援
  - 面向中小企业的租赁税制运用 (中小企业投资促进税制)
  - 环境租赁 (再生能源设备租赁) 的脱碳税制应对
  - 反社会势力核查・反洗钱对策 ([[regional-banks/mebuki-fg]] 集团共同实施)
