# 丰田金融服务 (TFS)

## Wiki 路线

本条目位于 [[leasing-firms/INDEX|leasing-firms INDEX]] 之下。可与 [[megabanks/mufg|三菱 UFJ FG (MUFG)]] 对照阅读，以把握同业 / 对比语境；也可与 [[banking/INDEX|banking index]] 一起阅读，以确认更广的制度 / 监管边界。

## TL;DR

**丰田汽车 100% 子公司、全球汽车金融统括公司**（2000-07-03 设立）。在全球 **超过 30  个国家**开展汽车贷款、租赁、保险和信用卡业务。**美国 Toyota Financial Services Americas（旧 TMCC）是美国大型汽车金融公司**，规模可与 Ford Motor Credit、Ally Financial 比肩。日本国内核心是 **Toyota Finance Corporation**（TS CUBIC Card、汽车贷款）。它通过新车销售 + 贷款 + 租赁 + 保险 + 信用卡的**综合 package**与 dealer network 一体化，成为 Toyota 品牌的收益源，并自 2024 年 以后向 EV / connected-car linked finance 转移。非上市（丰田汽车 consolidated subsidiary）。

## 1. 公司概要

**正式名**：株式会社 Toyota Financial Services
**英文名**：Toyota Financial Services Corporation
**设立**：2000-07-03
**总部**：爱知县名古屋市西区（临近丰田汽车集团据点）
**上市**：非上市 — 丰田汽车（东证 PRIME 7203）100% consolidated subsidiary
**母公司**：丰田汽车株式会社（[[manufacturer-finance/toyota-motor|toyota-motor]]）
**定位**：Toyota group 的**金融事业统括持股公司**

### 业态类别

- **汽车贷款**：面向新车购买 / 二手车购买的 credit
- **租赁**：operating lease、残值设定型租赁（残クレ）
- **保险**：汽车保险、寿险的代理销售 / 持股
- **信用卡**：TS CUBIC Card（面向 Toyota dealer members）
- **法人 fleet finance**：大口法人车辆管理租赁

## 2. 沿革 / 年表

| 年月 | 事项 |
|---|---|
| 1988  | **美国 Toyota Motor Credit Corporation (TMCC) 设立** — 进入美国汽车金融的起点 |
| 1989  | Toyota Credit Canada 设立（扩大北美业务） |
| 1990s | 欧洲 Toyota Kreditbank（德国）设立，展开欧洲汽车金融 |
| **2000-07-03** | **Toyota Financial Services Corporation 设立** — 作为丰田汽车的金融事业统括公司 |
| 2000s 前半 | 将全球各地汽车金融子公司集约 / 整合至 TFS 旗下 |
| 2010-2020  | 海外汽车贷款扩大，新兴市场（东南亚、拉美、东欧）展开加速 |
| 2010s 后半 | connected car × finance 试验（利用 telematics 的贷款） |
| 2020-2023  | 疫情期间加强残值风险管理；二手车市场上涨阶段残值收益扩大 |
| 2024〜 | 转向 **EV 普及 + connected-car linked finance**（充电基础设施联动、残值模型刷新） |

> 注：TFS 为非上市、丰田汽车 consolidated subsidiary，因此独立 IR 披露汇总在丰田汽车有价证券报告书的“金融事业 segment”中。本页基于公开 corporate site 与 Wikipedia 公开信息。

### 综合 package 战略

丰田的**dealer-financed sales**模型，是在购买新车时一站式提案 **车辆 + 贷款 + 租赁 + 汽车保险 + TS CUBIC Card**。凭借与 dealer network 一体化的送客模式，它相对银行系汽车贷款和独立系金融维持**销售渠道优势**。残值设定型贷款（残クレ）与二手车市场联动，形成 TFS 侧承担 resale value risk 的结构。

### 全球分散

- **美洲（TMCC）**：规模最大，在美国汽车金融市场可与 Ford Motor Credit、Ally Financial 比肩
- **欧洲（Toyota Kreditbank）**：以德国为中心，覆盖整个 EU
- **亚洲 / 澳洲 / 拉美**：与 Toyota 销售公司海外扩张联动，在新兴市场先行展开
- 通过货币分散和地区景气分散，降低单一市场依赖风险

### EV / connected linked finance（2024〜）

随着 EV 普及，**残值评估模型与传统 ICE 车辆出现背离**（电池劣化、快速变化的技术世代）→ 残值设定精细化成为课题。TFS 也在试验使用 connected-car 行驶数据的**使用量联动贷款 / 保险**（telematics UBI）。同时也在探索与充电基础设施事业者联动的金融商品。

### 竞争对手

| 竞争对手 | 母公司 | 特征 |
|---|---|---|
| Nissan NMA（NMAC） | 日产汽车 | 以北美为中心，与 Nissan dealer 联动 |
| Honda Finance | Honda | 日本国内 + 北美，Honda Cars 系 |
| [[orix-corp]] | 独立系（持股分散） | 法人租赁 / 个人汽车租赁综合 |
| Mitsubishi HC Capital | [[mufg]] group | 旧 Mitsubishi UFJ Lease + Hitachi Capital 合并，大型法人 |
| [[tokyo-century]] | Mizuho + Itochu | 租赁 / mobility，Avis 相关等 |
| 银行系汽车贷款 | Megabanks / regional banks | 低利率，但 dealer 送客力较弱 |

凭借 Toyota 销售公司渠道 × 残クレ产品设计的组合，TFS 相对独立系和银行系维持**销售现场优势**。

## 5. 监管 / 政策

- **日本国内主管**：金融厅（FSA）— 贷金业法、分期销售法、保险业法
- **美国**：Consumer Financial Protection Bureau（CFPB）及各州金融监管
- **欧洲**：各国金融监管（德国 BaFin 等）+ ECB / 欧洲银行监管
- **亚洲**：各国金融监管（泰国 BoT、印度尼西亚 OJK 等）
- **近期政策议题**：
  - 2024〜 EV 残值评估模型透明化（消费者保护议题）
  - 2025〜 connected-car data × finance 的个人信息保护整理
  - 2025〜 美国 CFPB 强化汽车金融 APR 披露趋势
