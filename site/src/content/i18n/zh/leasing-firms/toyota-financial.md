# 丰田金融服务 (TFS)

## Wiki route

This entry sits under [[leasing-firms/INDEX|leasing-firms INDEX]]. Read it against [[megabanks/mufg|三菱 UFJ FG (MUFG)]] for peer / contrast context and [[banking/INDEX|banking index]] for the broader system / regulatory boundary.

## TL;DR

**丰田汽车 100% 子公司，全球汽车金融统括公司**（2000-07-03 设立）。在全球 **30 多个国家** 展开汽车贷款・租赁・保险・信用卡。**美国 Toyota Financial Services Americas（旧 TMCC）作为美国大型汽车金融**，规模与 Ford Motor Credit・Ally Financial 比肩。国内核心是 **丰田金融株式会社**（TS 立方卡・汽车贷款）。以新车销售 + 贷款 + 租赁 + 保险 + 信用卡的**综合套餐**与经销商网一体化，作为丰田品牌的收益来源，自 2024 年以后正向 EV / 互联汽车联动金融转型。非上市（丰田汽车 合并子公司）。

## 1. 公司概要

**正式名称**：株式会社丰田金融服务
**英文名称**：Toyota Financial Services Corporation
**设立**：2000-07-03
**总部**：爱知县名古屋市西区（邻近丰田汽车 集团据点）
**上市**：非上市 — 丰田汽车（东证 PRIME 7203）100% 合并子公司
**母公司**：丰田汽车株式会社（[[manufacturer-finance/toyota-motor|toyota-motor]]）
**定位**：丰田集团的**金融事业统括控股公司**

### 主要子公司・集团公司

```
丰田汽车（7203，母公司）
  └── 丰田金融服务 (TFS, 100%)
        ├── 国内
        │   ├── 丰田金融株式会社 (TF) ── 国内核心
        │   │     ├── 汽车贷款（新车・二手车）
        │   │     ├── TS 立方卡（丰田经销商会员信用卡)
        │   │     └── 租赁（个人・法人）
        │   └── 丰田金融服务・保险 ── 保险控股公司
        │         (旧 Aioi Life 系统合，承办寿险/财产险)
        ├── 美洲
        │   └── Toyota Financial Services Americas
        │         └── Toyota Motor Credit Corporation (TMCC, 1988 设立)
        │               ── 美国大型汽车金融，含 Lexus Financial Services
        ├── 欧洲
        │   └── Toyota Kreditbank GmbH (德国据点)
        │         ── 统括欧洲全域的汽车贷款・租赁
        ├── 亚洲・澳洲・中南美
        │   └── 各国 Toyota Financial Services 法人 (30 多个国家)
        │         ── 泰国・印度尼西亚・澳洲・巴西・南非等
        └── 丰田移动零部件等关联金融职能
```

### 业态类别

- **汽车贷款**: 面向新车购买・二手车购买的信贷
- **租赁**: 经营性租赁・残值设定型租赁（残值贷）
- **保险**: 汽车保险・人寿保险的代理销售・控股
- **信用卡**: TS 立方卡（面向丰田经销商会员）
- **面向法人的车队金融**: 大额法人车辆管理租赁

## 2. 沿革・年表

| 年月 | 事件 |
|---|---|
| 1988 | **美国 Toyota Motor Credit Corporation (TMCC) 设立** — 进军美国汽车金融的起点 |
| 1989 | 设立 Toyota Credit Canada（北美展开扩大）|
| 1990s | 设立欧洲 Toyota Kreditbank（德国），展开欧洲汽车金融 |
| **2000-07-03** | **株式会社丰田金融服务 设立** — 作为丰田汽车的金融事业统括公司 |
| 2000s 前半 | 将全球各地的汽车金融子公司汇集・整合于 TFS 旗下 |
| 2010-2020 | 扩大海外汽车贷款，加速新兴国（东南亚・中南美・东欧）展开 |
| 2010s 后半 | 互联汽车 × 金融的试行（运用车载信息技术的贷款）|
| 2020-2023 | 疫情下强化残值风险管理，在二手车市场高涨局面下残值收益扩大 |
| 2024〜 | 向 **EV 普及 + 互联汽车联动金融** 转型（充电基础设施联动・残值模型刷新）|

> 注：TFS 因属非上市・丰田汽车合并子公司，独立的 IR 公开汇集于母公司（丰田汽车）的有价证券报告书「金融事业分部」中。本页基于公开企业网站与 Wikipedia 公开信息。

|---|---|---|

### 综合套餐战略

丰田的**经销商金融销售（dealer-financed sales）**模式 — 在购买新车时将 **车辆 + 贷款 + 租赁 + 汽车保险 + TS 立方卡** 一站式提案。以与经销商网一体化的引流模式，维持相对银行系汽车贷款及独立系金融的**销售渠道优势**。残值设定型贷款（残值贷）与二手车市场联动，形成由 TFS 一方承担转售价值风险的结构。

### 全球分散

- **美洲（TMCC）**: 规模最大，在美国汽车金融市场与 Ford Motor Credit・Ally Financial 比肩
- **欧洲（Toyota Kreditbank）**: 以德国为中心，覆盖欧盟全域
- **亚洲・澳洲・中南美**: 与丰田销售公司的进军联动，在新兴国市场先行展开
- 通过货币分散・区域景气分散，降低单一市场依赖风险

### EV / 互联联动金融（2024〜）

随 EV 普及，**残值评估模型与传统 ICE 车出现背离**（电池劣化・急剧变化的技术世代）→ 残值设定的精细化成为课题。运用互联汽车行驶数据的**用量联动贷款 / 保险** 的试行（车载信息 UBI）。也在探索与充电基础设施经营者的联动金融产品。

## 5. 监管・政策

- **国内主管**: 金融厅（FSA）— 贷金业法・分期付款销售法・保险业法
- **美国**: 联邦消费者金融保护局（CFPB）・各州金融监督
- **欧洲**: 各国金融监督（德国 BaFin 等）+ ECB / 欧洲银行监督
- **亚洲**: 各国金融监督（泰国 BoT・印度尼西亚 OJK 等）
- **近期政策议题**:
  - 2024〜 EV 残值评估模型的透明化（消费者保护议题）
  - 2025〜 互联汽车数据 × 金融的个人信息保护梳理
  - 2025〜 美国 CFPB 加强汽车金融 APR 披露的动向
