# 横浜銀リース (Yokohama Bank Leasing)

## Wiki route

本条目位于 [[leasing-firms/INDEX|leasing-firms INDEX]] 之下，作为 [[regional-banks/concordia-fg|横浜フィナンシャルグループ (旧コンコルディア FG)]] 的 FG 子公司租赁经营公司锚点。请将其与母体经营银行 [[regional-banks/yokohama-bank|横浜銀行]] 以及同属子公司 [[regional-banks/higashi-nippon-bank|東日本銀行]] 对照阅读。同业比较: [[leasing-firms/mebuki-lease|めぶきリース]]（北关东地银系）与大型出租人 [[leasing-firms/mitsubishi-hc-capital|三菱 HC キャピタル]] / [[leasing-firms/mizuho-lease|みずほリース]] / [[leasing-firms/tokyo-century|東京センチュリー]]。更广泛的系统路径参见 [[banking/regional-bank-consolidation-pattern|regional-bank consolidation pattern]] 与 [[banking/INDEX|banking index]]。

## TL;DR

株式会社横浜銀リースは、[[regional-banks/yokohama-bank|横浜銀行]]（现 [[regional-banks/concordia-fg|横浜フィナンシャルグループ]] 子公司）的并表子公司、地银系租赁公司，以神奈川县・首都圈为地盘提供设备租赁・融资租赁・经营租赁・自治体租赁・医疗租赁等。在地银系租赁中属于上位规模，背靠 [[regional-banks/concordia-fg]]（旧 CFG，2025-10 更名）这一地银 FG 最大级别的存款客户基础。^[extracted]

### 横浜銀リース 设立

- 作为 [[regional-banks/yokohama-bank|横浜銀行]] 系列租赁公司，以面向神奈川县内中坚企业的设备租赁为主业
- 背靠横浜银行的县内 No.1 压倒性份额，在租赁业务上的神奈川县内份额亦居上位
- 战后地银系租赁的典型: 以设备租赁形态而非银行贷款支持中小企业设备投资

### CFG → YFG 更名 (2025-10-01)

- 母持股公司 [[regional-banks/concordia-fg]] 于 2016-04  由 [[regional-banks/yokohama-bank]] + [[regional-banks/higashi-nippon-bank]] 共同股份转移成立
- **2025-10-01**: 旧 コンコルディア FG → **横浜フィナンシャルグループ** 更名
- 横浜銀リース 本身未受更名影响，继续作为 [[regional-banks/yokohama-bank]] 并表子公司运营

### FG 内定位

- **银租赁联动的核心**: 受托 [[regional-banks/yokohama-bank]] 的神奈川县内中坚企业 main bank 客户的租赁需求
- **设备投资支援**: 银行贷款补充 + 对表外处理需求的应对
- **县内自治体・公共部门联动**: 神奈川县・横浜市・川崎市等 自治体租赁项目
- **横浜银行品牌的租赁展开**: 与 [[regional-banks/concordia-fg|YFG]] 「横浜」品牌回归 (2025-10) 的一致性

### 竞争格局

- **大型租赁**: [[leasing-firms/mitsubishi-hc-capital|三菱 HC キャピタル]] / [[leasing-firms/smfl|SMFL]] / [[leasing-firms/mizuho-lease|みずほリース]] / [[leasing-firms/fuyo-lease|芙蓉リース]] / [[leasing-firms/tokyo-century|東京センチュリー]] / [[leasing-firms/ricoh-leasing|リコーリース]] - 在大型・全国・复杂项目上占优
- **地银系租赁**: [[leasing-firms/mebuki-lease|めぶきリース]] (北关东) / [[leasing-firms/hokkaido-lease|北海道リース]] / 千葉興業リース 等 - 同型 model
- **厂商系租赁**: [[card-issuers/toyota-finance|トヨタファイナンス]] / [[leasing-firms/nec-leasing|NEC キャピタルソリューション]] - 产品联动
- **县内竞争**: 横浜银行系・神奈川县内信金租赁・大型租赁分店

### 优势与劣势

- 优势: [[regional-banks/yokohama-bank]] 存款 15 兆円超 (单体) 的中坚企业 main bank 网络 + 首都圈 GDP 规模 + 神奈川县内 No.1 认知度

## 7. 规制・政策

- **主管**: 金融厅 (FSA) (租赁业单独而言不需银行业牌照，贷金业登记视对应领域而定)
- **规制论点**:
  - 租赁会计准则变更 (IFRS16 / 日本准则) 应对支援
  - 面向中小企业的租赁税制运用 (中小企业投资促进税制)
  - 环境租赁 (再生能源设备租赁) 的脱碳税制应对
  - 反社会势力核查・反洗钱对策 ([[regional-banks/concordia-fg]] 集团共同实施，与 2018-08 东日本银行 业务改善命令后的内部控制强化相一致)
