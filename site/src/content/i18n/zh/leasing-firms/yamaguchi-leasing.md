# 山口租赁 (Yamaguchi Leasing)

## Wiki 路径

本条目位于 [[leasing-firms/INDEX|leasing-firms INDEX]] 之下，是 [[regional-banks/yamaguchi-fg|山口 FG (YMFG)]] 的 FG 子公司 leasing 运营公司锚点。阅读时可对照母体运营银行 [[regional-banks/yamaguchi-fg|山口 FG]] 3 行体制（[[regional-banks/momiji-bank|もみじ銀行]] / [[regional-banks/kitakyushu-bank|北九州銀行]] / 山口银行）以及兄弟子公司 YM 证券。同业比较：[[leasing-firms/mebuki-lease|めぶきリース]]（北关东）/ [[leasing-firms/yokohama-bank-leasing|横浜銀リース]]（神奈川）/ [[leasing-firms/chibagin-leasing|ちばぎんリース]]（千叶）/ [[leasing-firms/hokkaido-lease|北海道リース]] / [[leasing-firms/iyogin-leasing|いよぎんリース]]（四国）以及大型租赁公司 [[leasing-firms/mitsubishi-hc-capital|三菱 HC キャピタル]] / [[leasing-firms/mizuho-lease|みずほリース]] / [[leasing-firms/tokyo-century|東京センチュリー]]。更广泛的系统路径参见 [[banking/regional-bank-consolidation-pattern|regional-bank consolidation pattern]]、[[banking/regional-bank-api-digital-partnership-route|regional-bank API/digital partnership route]]、[[banking/INDEX|banking index]]。可投资范围语境见 [[finance/japan-listed-financial-groups-investable-universe|Japan listed financial groups investable universe]]。

## TL;DR

株式会社山口租赁（YM Lease）是 [[regional-banks/yamaguchi-fg|山口 FG (YMFG, 東証 PRIME 8418)]] 集团的地方银行系租赁公司。以山口银行、[[regional-banks/momiji-bank|もみじ銀行]]（广岛县）、[[regional-banks/kitakyushu-bank|北九州銀行]]（福冈县北九州）的 3 行体制为背景，在 **中国地方西部 + 广岛 + 北九州（关门海峡经济圈）** 的广域地盘提供设备租赁、融资租赁、经营租赁、自治体租赁、医疗租赁、汽车租赁等服务。作为地方银行系租赁公司，其特点是以“3 行 3 品牌”为背景，形成跨县域广域 model。^[extracted]

### 主要业务线

| 分部 | 特征 |
|---|---|
| 融资租赁 | 面向中小至中坚企业的机械、车辆、OA 设备、通信设备租赁 |
| 经营租赁 | 二手设备 / 可再租赁设备的短期租赁 |
| 厂商租赁 | 与制造商、经销商合作的租赁（销售支持） |
| 自治体租赁 | 山口县、广岛县、福冈县北九州市等的公务车、设备租赁 |
| 医疗租赁 | 面向医院、诊所的医疗设备租赁 |
| 工业设备租赁 | 山阳工业地带（周南联合企业、宇部 / 德山化学、防府汽车）制造设备租赁 |
| 农业机械租赁 | 面向山口县、广岛县农业地带 |

### 强项 / 弱项

- 强项：3 行 3 品牌（山口 + 广岛 + 北九州）的广域地盘 + 山阳 / 北九州工业地带制造业集聚 + 与 [[regional-banks/concordia-fg|横浜 FG]] 的域外合作

### 监管框架

- **主管**：金融厅（FSA）・经济产业省
- **法律依据**：租赁业（银行法上不需银行业牌照），贷金业注册（根据保理领域需要取得）
- **母 FG 治理**：作为 [[regional-banks/yamaguchi-fg]] 连结子公司置于控股公司治理之下，并受银行业以外子公司监管（主要业务 / 附随业务 / 业务范围限制）定位

### 监管论点

- **租赁会计准则变化（IFRS16  / 日本准则修订）**：支持承租方上表处理（使用权资产、租赁负债）
- **中小企业租赁税制**：支持活用中小企业投资促进税制 / 经营强化税制
- **环境租赁（可再生能源设备 / EV）**：脱碳税制应对、绿色租赁商品开发
- **割赋销售法 / 贷金业法**：保理领域的适用整理
- **反社会势力检查 / 反洗钱措施**：[[regional-banks/yamaguchi-fg]] 集团共同实施
- **EDINET / 有价证券报告书**：山口租赁单体为非上市，披露有限；通过 [[regional-banks/yamaguchi-fg]] 连结分部信息间接披露

### 会计处理
