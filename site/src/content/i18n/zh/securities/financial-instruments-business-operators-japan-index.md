# 日本金融商品交易业者登记一览索引

## 摘要

金融厅的金融商品交易业者名单中，截至 2026-04-30 共有 **1,945  家业者**。本页面是 FIEA（金商法）/ FIBO 群体的登记管控路径。它并不复现全部 1,945  行；官方金融厅工作簿仍是行级别的完整权威来源。

使用本页面来区分经纪业务、第一种／第二种金融商品交易业、投资咨询、投资管理、PTS、代币化证券处理以及证券相关的自律监管路径。仅对重要的经纪商、资产管理公司、PTS／市场基础设施运营者、上市公司集团或战略上重要的外国业者创建独立页面。

## 边界

| 边界 | 处理方式 |
|---|---|
| 金融商品交易业者 | 由金融厅 `kinyushohin.xlsx` 管控的全部群体; 此处按类别和管辖加以汇总。 |
| 证券公司／在线经纪商 | 当市场份额、集团战略或零售分销重要时设独立页面。 |
| 资产管理公司 | 当 AUM、集团战略、ETF／基金平台或公开市场相关性重要时设独立页面。 |
| PTS／市场基础设施 | 相关时在 [[securities/INDEX]] 或 [[securities/INDEX]] 之下设独立页面。 |
| 长尾咨询业者／第二种业者 | 除非具战略意义，否则仅登记。 |

## 晋升规则

仅当某业者满足以下条件之一时，才将其从仅登记晋升为独立页面:

- 主要的经纪商、资产管理公司、PTS、市场基础设施运营者或上市公司集团;
- 对零售经纪、NISA、证券型代币、FX／衍生品或资本市场基础设施具有战略重要性;
- 在日本拥有重要分行／子公司存在的外国业者;
- 公开信息足够丰富，能够支撑独立分析。
