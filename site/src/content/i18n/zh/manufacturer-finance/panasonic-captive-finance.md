# Panasonic 自有金融（パナソニック顧客金融）

## Wiki 路由

本词条归属于 [[manufacturer-finance/INDEX|manufacturing index]]。请对照 [[manufacturer-finance/hitachi-industrial-finance-platform|Hitachi 産業金融]]、[[manufacturer-finance/sony-group-finance-arm|Sony FG]]、[[manufacturer-finance/mitsubishi-heavy-export-finance|MHI 輸出金融]]、[[manufacturer-finance/toyota-financial-services|Toyota Financial Services]] 阅读以作同业产业综合企业金融部门的比较。与 [[leasing-firms/smfl|SMFL (三井住友ファイナンス&リース)]] 和 [[megabanks/mufg|MUFG]] 配对了解 Panasonic 的 B2B vendor-finance 流的银行侧对应方，[[JapanFG/INDEX|JapanFG index]] 了解更广的监管边界。

## TL;DR

松下控股（Panasonic Holdings，6752）是拥有家电、住宅、车载电池、B2B 解决方案的综合电机 / 电池厂商。**与 Toyota 或 Sony 不同，它本体不持有直系的 captive 银行 / 保险 / 大型卡**。取而代之的是 **(1) 面向 B2B 客户的 vendor finance 经与三井住友融资租赁（SMFL）或银行系租赁的合作组建，(2) 在与 Tesla、Mazda 等的 EV 电池合资企业中以 JV 为单位安排设备投资金融，(3) 旧 松下金融（国内法人卡 / 分期）在 2017 年 前后的重组中大幅缩减** 这样的三层结构。从 manufacturing 视角看，这是「巨型制造业 **不持有 captive，而经银行系 / JV 来运转客户金融** 模式」的样本。该方针在 2022  控股公司制转型 + 2024  中期计划（Operational Excellence）下更趋鲜明。

### 金融部门的位置（2022  控股公司制转型后）

```
松下控股（6752）
  ├── 松下株式会社（生活电器、空气质量空调、冷链等）
  ├── 松下汽车系统
  ├── 松下产业（6786,  单独上市）
  ├── 松下 Connect（B2B 解决方案、Blue Yonder）
  ├── 松下能源（车载、产业用电池）
  ├── 松下住宅解决方案
  ├── 松下娱乐与通信
  └── 公司财务（本体财务）
        ├── 供应链金融、应收账款保理
        ├── 客户租赁组建与 SMFL / 三菱 HC 资本等合作
        ├── EV 电池 JV（Tesla、Mazda 等）按 JV 为单位安排设备投资金融
        └── 旧松下金融（国内法人卡 / 分期）功能缩减
```

旧「松下金融株式会社」在 2017 年 前后将一部分法人卡功能移交给 OBC / 银行系，现今的 Panasonic Holdings 合并内 captive finance 比重已缩小。

## 2. 产品 / 事业线 × 金融组建模式

| 事业线 | 主要客户金融组建 | 合作金融 |
|---|---|---|
| 生活电器（白色家电） | 经量贩店信用（Orico、Jaccs 等） | 量贩店合作，本体介入有限 |
| 空气质量空调（空调 B2B） | 租赁、订阅 | [[leasing-firms/smfl|SMFL]] / [[leasing-firms/mitsubishi-hc-capital|三菱HCキャピタル]] |
| 汽车系统 | 面向汽车 OEM 的零部件（vendor financing） | 与 OEM 系 captive（Toyota Industries 等）对接 |
| 松下能源（车载电池） | EV 电池 JV（Tesla、Mazda、Subaru） | 按 JV 为单位的设备投资金融、银团 + JBIC |
| 松下 Connect（Blue Yonder、产业用） | SaaS 订阅、IT 设备租赁 | 银行系租赁 + SaaS 计费 |
| 住宅解决方案 | 住房贷款斡旋、装修贷款 | 银行、信贷合作 |
| 产业（6786） | B2B 零部件 vendor financing | 单独上市，独立 IR |

EV 电池方面，面向 **Tesla Gigafactory 1 （内华达）** 自 2014  起持续供应 1810 / 21700  电芯，并推进面向 Tesla 4680  电芯的产能扩张投资（含美国堪萨斯工厂）。合资企业为单位的金融由银团 + [[financial-regulators/jbic|JBIC]]（海外设备投资）+ 美国 IRA 补贴（Section 45X 制造税额抵免）的组合组建。

## 4. 战略（Operational Excellence + Energy）

- **Operational Excellence（2024〜 中期计划）**：最重视集团整体的利润率改善与资本效率提升。借助控股公司制与各事业公司责任的明确化，加速低收益事业整理与对高增长领域（能源、Blue Yonder）的集中投资。
- **对车载电池的集中投资**：Tesla 4680  电芯增产、面向 Mazda / Subaru 的供应合同、美国堪萨斯州 De Soto 新工厂（约 4 GWh → 30 GWh 扩张计划）等。投资规模为数千亿日元〜万亿日元，由 IRA 补贴 + 银团融资 + JBIC 组建。
- **Blue Yonder 的 B2B SaaS 化**：将 2021  以约 7,100 億円 收购的美国供应链 SaaS Blue Yonder 置于 Panasonic Connect 的核心，提高订阅营收占比。
- **不持有 captive 的路线**：缩减旧松下金融功能，客户金融与 SMFL / 三菱 HC 资本等合作。减轻本体 B/S，作用于 ROE 改善。
- **2024〜2026  重点**：中国 EV 市场竞争加剧与 Tesla 依赖风险的平衡、车载电池的收益化、住宅事业（松下 Homes）的结构改革。

## 5. 监管 / 政策

- 作为上市公司，受东证、金商法、公司法、IFRS 披露约束。
- 车载电池强烈受美国 IRA（Inflation Reduction Act）的 Section 45X 制造税额抵免、欧洲 Battery Regulation、日本 GX（脱碳）投资促进策的影响。
- 海外大型设备投资中，有使用 [[financial-regulators/jbic|JBIC]] 协调融资、[[policy-finance/nexi|NEXI]] 覆盖的案件。
- 住宅事业曾有与丰田住宅的整合（Prime Life Technologies），但其后体制变更持续。
- 旧松下金融的缩减，是相较于贷金业法、分期销售法应对成本而作出的「不拘泥于直接金融的选择」之事例。
