# Toyota Motor Corporation (丰田汽车)

## Wiki route

This entry sits under [[manufacturer-finance/INDEX|manufacturing index]]. Read it against [[manufacturer-finance/seiko-epson|Seiko Epson Corporation (セイコーエプソン)]] for peer / contrast context and [[manufacturer-finance/INDEX|manufacturer-finance INDEX]] for the broader system / regulatory boundary.

## TL;DR

丰田汽车是 1937-08-28 成立的日本最大级别汽车制造商，是 [[leasing-firms/toyota-financial|トヨタファイナンシャルサービス]] 的母公司。在 FinWiki 中，用作观察「制造业本体如何支撑金融子公司、销售金融、残值风险、EV/混合动力战略」的 manufacturing anchor。

## 1. 公司结构

| 项目 | 内容 |
|---|---|
| 正式名称 | Toyota Motor Corporation / 丰田汽车株式会社 |
| 成立 | 1937年8月28日 |
| 总部 | 1 Toyota-Cho, Toyota City, Aichi Prefecture 471-8571, Japan |
| 资本金 | 6,350億円（2025-03-31） |
| 员工 | 单体 71,515人，合并 383,853人（2025-03-31） |
| 主营业务 | Motor vehicle production and sales |
| 金融子公司 | [[leasing-firms/toyota-financial|Toyota Financial Services]] |

丰田以汽车制造与销售为核心，形成涵盖金融、出行、零部件、物流、销售网络的庞大集团。金融子公司承担销售金融、租赁、保险、信用卡业务，将车辆销售与残值管理一体化。

## 2. 业务线 map

| 业务线 | 内容 | 与金融・区域金融的接点 |
|---|---|---|
| 汽车制造 | Toyota / Lexus / 相关品牌车辆 | 设备投资、零部件供应商金融 |
| 销售 | 国内外销售网络、经销商 | 经销商金融、库存金融 |
| 混合动力 / EV | HEV, PHEV, BEV, FCEV 的多路径战略 | 残值设定・充电基础设施金融 |
| 金融 | [[leasing-firms/toyota-financial|TFS]] | 贷款、租赁、保险、卡 |
| 全球生产 | 北美、欧洲、亚洲等 | 汇率、贸易金融、供应链 |

在 FY2025 Integrated Report 中，FY2025 的总销量为 9,362,410 台，销售收入为 48,036.7 billion yen。Toyota Financial Services 覆盖 35 以上的国家和地区。

## 3. 战略与竞争

- **多路径电动化**: Toyota 并非只押注 BEV，而是组合 HEV / PHEV / BEV / FCEV。在金融层面，残值设定与二手车价格展望至关重要。
- **与销售金融的一体化**: [[leasing-firms/toyota-financial|toyota-financial]] 连接车辆销售、租赁、保险、经销商网络，支撑制造业本体的销售周期。
- **供应链金融**: 丰田系零部件公司、区域供应商、设备投资与 [[megabanks/mufg|mufg]]、[[megabanks/smfg|smfg]]、[[megabanks/mizuho-fg|mizuho-fg]]、地方银行的法人金融相连接。
- **汇率・通商风险**: 作为全球生产・出口企业，受日元贬值/升值、关税、美中・美日通商政策的影响很大。

## 4. 监管与政策

- 作为上市公司，属于东证・金商法・公司法的披露对象。
- 受汽车安全・环境法规、CO2 排放法规、各国 EV 补贴・关税政策的影响。
- 金融子公司是各国金融监管、消费者信贷、租赁、保险销售法规的对象。
