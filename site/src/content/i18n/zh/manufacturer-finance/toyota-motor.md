# 丰田汽车公司（トヨタ自動車）

## Wiki 路由

本词条归属于 [[manufacturer-finance/INDEX|manufacturing index]]。请对照 [[manufacturer-finance/seiko-epson|Seiko Epson Corporation (セイコーエプソン)]] 阅读以了解同业 / 对比背景，[[JapanFG/INDEX|JapanFG index]] 了解更广的系统 / 监管边界。

## TL;DR

丰田汽车是 1937-08-28 设立的日本最大级汽车厂商，是 [[JapanFG/toyota-financial|トヨタファイナンシャルサービス]] 的母公司。在 FinWiki 中，将其用作 manufacturing anchor，以观察「制造业本体如何支撑金融子公司、销售金融、残值风险、EV/ 混动战略」。

## 1. 公司结构

| 项目 | 内容 |
|---|---|
| 正式名 | Toyota Motor Corporation / 丰田汽车株式会社 |
| 设立 | 1937年8月28日 |
| 总部 | 1 Toyota-Cho, Toyota City, Aichi Prefecture 471-8571, Japan |
| 资本金 | 6,350億円（2025-03-31） |
| 员工 | 单体 71,515 人、合并 383,853 人（2025-03-31） |
| 主业 | Motor vehicle production and sales |
| 金融子公司 | [[JapanFG/toyota-financial|Toyota Financial Services]] |

Toyota 以汽车制造 / 销售为核心，形成了包含金融、移动出行、零部件、物流、销售网的巨大集团。金融子公司承担销售金融、租赁、保险、信用卡，将车辆销售与残值管理一体化。

## 2. 事业线 map

| 事业线 | 内容 | 与金融 / 区域金融的接点 |
|---|---|---|
| 汽车制造 | Toyota / Lexus / 相关品牌车辆 | 设备投资、零部件供应商金融 |
| 销售 | 国内外销售网、经销商 | 经销商金融、库存金融 |
| 混动 / EV | HEV, PHEV, BEV, FCEV 的多路径战略 | 残值设定、充电基础设施金融 |
| 金融 | [[JapanFG/toyota-financial|TFS]] | 贷款、租赁、保险、卡 |
| 全球生产 | 北美、欧洲、亚洲等 | 汇率、贸易金融、供应链 |

在 FY2025  Integrated Report 中，FY2025  的总销量为 9,362,410  辆，营业收益为 48,036.7 billion yen。Toyota Financial Services 覆盖 35  以上的国家 / 地区。

## 3. 战略与竞争

- **多路径电动化**：Toyota 并非 BEV 一条腿，而是组合 HEV / PHEV / BEV / FCEV。金融面上残值设定与二手车价格预测重要。
- **与销售金融的一体化**：[[JapanFG/toyota-financial|toyota-financial]] 连接车辆销售、租赁、保险、经销商网，支撑制造业本体的销售周期。
- **供应链金融**：丰田系零部件公司、区域供应商、设备投资与 [[JapanFG/mufg|mufg]]、[[JapanFG/smfg|smfg]]、[[JapanFG/mizuho-fg|mizuho-fg]]、地方银行的法人金融相连。
- **汇率 / 通商风险**：作为全球生产 / 出口企业，日元贬值/升值、关税、美中 / 美日通商政策的影响很大。

## 4. 监管与政策

- 作为上市公司，受东证、金商法、公司法披露约束。
- 受汽车安全 / 环境监管、CO2  排放监管、各国 EV 补贴 / 关税政策的影响。
- 金融子公司受各国金融监督、消费者信用、租赁、保险销售监管约束。
