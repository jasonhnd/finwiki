# 三井住友银行 (SMBC)

## Wiki 路径

本条目位于 [[megabanks/INDEX|megabanks INDEX]]（其领域路径）之下。可对照 [[megabanks/mufg-bank|三菱UFJ銀行 (MUFG Bank)]] 阅读以作同业 / 对比参考（另一家都市银行megabank运营公司），并对照 [[financial-regulators/fsa|the FSA]] 了解监管 / 规制边界。集团 / 母公司视角见 [[megabanks/smfg|SMFG]]；信托与证券的兄弟公司见 [[trust-banks/smbc-trust-bank|SMBC信託銀行]] 和 [[securities-firms/smbc-nikko|SMBC日興証券]]。

## TL;DR

三井住友银行 (Sumitomo Mitsui Banking Corporation) 是 [[megabanks/smfg|SMFG / SMBC グループ]] 的商业银行核心法人。在 FSA 的银行牌照清单上以「株式会社三井住友銀行」(都市银行) 列出，银行代码为 0009。其拥有与母公司 SMFG、[[trust-banks/smbc-trust-bank|SMBC信託銀行]]、[[securities-firms/smbc-nikko|SMBC日興証券]] 相区别的 licensed bank layer。

## 2. 运营模式

- 承担 SMBC 集团的存款吸收、放贷、国内/外汇汇兑、结算、全球法人金融的银行法人。
- bank-license layer 在功能上与 [[trust-banks/smbc-trust-bank|SMBC信託銀行]] (信托 / PRESTIA / 外币・不动产・信托功能) 及 [[securities-firms/smbc-nikko|SMBC日興証券]] 的 brokerage / underwriting layer 相分离。
- 通过与三井住友卡、SMBC 消费金融的联动，构建出超越银行本体的 retail / wealth / payment funnel。

## 3. 为何独立成页重要

[[megabanks/smfg|SMFG]] 是上市控股公司，但银行法上存款吸收、放贷、汇兑交易的主体是三井住友银行。要分析 Olive、法人结算、海外分行、与 SMBC Trust 之间的银行代理・信托代理结构，仅凭 SMFG 页面粒度过粗。
