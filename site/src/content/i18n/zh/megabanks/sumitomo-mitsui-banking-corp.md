# 三井住友银行 (SMBC)

## Wiki route

本条目位于 [[JapanFG/INDEX|JapanFG index]] 下。可与 [[megabanks/mufg-bank|三菱UFJ銀行 (MUFG Bank)]] 对照阅读，以理解同业和差异；更广泛的制度和监管边界见 [[banking/INDEX|banking index]]。

## TL;DR

三井住友银行是 [[megabanks/smfg|SMFG / SMBC グループ]] 的商业银行核心法人。在 FSA 银行许可名单上，它以“株式会社三井住友银行”列示，并拥有与母公司 SMFG、[[trust-banks/smbc-trust-bank|SMBC信託銀行]]、[[securities-firms/smbc-nikko|SMBC日興証券]] 分开的 licensed bank layer。

## 1. 法人 / 许可边界

| 项目 | 说明 |
|---|---|
| 法定名称 | 株式会社三井住友银行 |
| 英文名称 | Sumitomo Mitsui Banking Corporation |
| FSA 分类 | 都市银行 |
| 母公司边界 | [[megabanks/smfg|SMFG]] 100% bank operating company |
| 核心功能 | 存款、贷款、国内 / 外国汇兑、法人金融、海外网点 |

## 2. 业务角色

- 负责 SMBC 集团存款、贷款、结算和全球法人金融的银行法人。
- [[trust-banks/smbc-trust-bank|SMBC信託銀行]] 作为 SMBC 的 100% 子公司，负责信托 / PRESTIA / 外币、不动产和信托功能。
- 通过与 [[securities-firms/smbc-nikko|SMBC日興証券]]、三井住友卡、SMBC Consumer Finance 的协同，构建超出单体银行的 retail / wealth / payment funnel。

## 3. 为何需要独立页面

SMFG 是上市控股公司，但在银行法下，吸收存款、贷款和汇兑交易的主体是三井住友银行。要分析 Olive、法人结算、海外分行以及与 SMBC Trust 的银行代理 / 信托代理结构，仅看 SMFG 页面粒度过粗。
