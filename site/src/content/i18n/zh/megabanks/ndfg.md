# NTT DOCOMO Financial Group（NDFG）

## Wiki route

本条目位于 [[JapanFG/INDEX|JapanFG index]]。可与 [[non-life-insurers/rakuten-insurance-hd|楽天インシュアランスホールディングス]] 对照阅读，以理解同业和差异；更广泛的制度和监管边界见 [[banking/INDEX|banking index]]。

## TL;DR

NTT DOCOMO 通过吸收分割承继金融领域业务和相关股份，设立并启动 **NTT DOCOMO Financial Group** 这一金融业务统括中间控股公司，业务开始日为 2026-07-01。它把证券、银行、卡、支付和相关金融业务放到 1  一个屋檐下重组，包括 Monex Group、住信 SBI 网络银行 / docomo Neobank、d Card、d Barai、d Smart Bank 等。其定位是与 [[megabanks/au-fh|au-fg]]、[[paypay-fg]]、[[rakuten-fg]] 并列的**通信系 FG 第 4  极**。母公司是 NTT DOCOMO，属于 NTT 100% 集团。^[extracted]

## 1. 公司概要

**正式名称**：株式会社 NTT DOCOMO Financial Group ^[extracted]  
**英文名称**：NTT DOCOMO Financial Holdings，因正式英文名在来源注记中未完全公表，作为使用表记处理。  
**业务开始**：2026-07-01，吸收分割效力发生日，基于官方发布。^[extracted]  
**母公司**：株式会社 NTT DOCOMO，隶属于日本电信电话株式会社 100%。^[extracted]  
**总部**：东京都千代田区永田町 2-11-1 ，Sanno Park Tower。^[extracted]

### 设立方案 / 公开发布基础 ^[extracted]

NTT DOCOMO 公布了把既有金融相关子公司和持股移动到中间控股公司 NDFG 的方案。NDFG 被定位为 NTT DOCOMO 的 100% 子公司，未公表上市计划。集团整体最上位母公司仍为日本电信电话株式会社 NTT，东证 Prime 9432。

### 通信系 FG「第 4  极」定位

- **既有 3  极**：[[megabanks/au-fh|au-fg]]（KDDI / au，au Jibun Bank 出售后再编期）、[[paypay-fg]]（SoftBank / Z Holdings 系，PayPay Bank + PayPay Securities + PayPay Card）、[[rakuten-fg]]（Rakuten Bank / Rakuten Securities / Rakuten Card / Rakuten Pay）。

### 金融服务整合的目标 / 公开发布旨趣 ^[extracted]

1. 将证券、银行、卡、积分从分散运营管理变为统括管理，明确集团 KPI 和 ROE 管理。  
2. 将金融业务经营责任与 NTT DOCOMO 通信本体分离，以便监管应对和治理。  
3. 在中间控股公司层面统一设计 d Point 经济圈与金融产品的交叉销售。

### 合作 / 收购战略

- **Monex Group**：2023-10-04 签订资本业务合作，2024-01-04 通过 DOCOMO Monex HD 完成连结子公司化，DOCOMO 议决权约 49%；是证券分部核心。^[extracted]
- **住信 SBI 网络银行**：从 docomo Neobank 品牌提供发展为 2025-05-30 开始 TOB，并在 2025-09  成立后，于 2025-10-01 成为 **NTT DOCOMO 连结子公司**。资本重组后 DOCOMO 持有 55.37%，三井住友信托持有 44.63%，议决权 50% / 50% 形成共同经营；2026-08-03 计划更名为「DOCOMO SMTB Net Bank」。^[extracted]
- **Coincheck Group**：作为 Monex 旗下加密资产业务承继。^[extracted]

### B2C 品牌

- **d 品牌统一**：持续把 d Card、d Smart Bank、d Point、d Barai 的「d」reference 共同化。^[extracted]

## 4. 监管 / 政策

- **主管**：金融厅。
- Monex Securities：第一种金融商品交易业。
- 住信 SBI 网络银行 / DOCOMO SMTB Net Bank：银行业。
- d Card：包括信用购买斡旋业务。
- d Barai / d Point：资金移动业 / 预付式支付手段。
- Coincheck：加密资产交易业。
