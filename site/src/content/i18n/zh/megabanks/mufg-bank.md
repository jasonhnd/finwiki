# 三菱UFJ銀行 (MUFG Bank)

## Wiki 路径

本条目位于 [[megabanks/INDEX|megabanks INDEX]]（其领域路径）之下。就同业／对比（另一家 都市銀行 超大型银行运营公司），请与 [[megabanks/sumitomo-mitsui-banking-corp|三井住友銀行 (SMBC)]] 对照阅读；就监管／监理边界，请与 [[financial-regulators/fsa|the FSA]] 对照阅读。集团／母公司视角在 [[megabanks/mufg|MUFG]]; 信托兄弟公司在 [[trust-banks/mitsubishi-ufj-trust-bank|三菱UFJ信託銀行]]。

## TL;DR

三菱UFJ銀行 (MUFG Bank, Ltd.) 是 [[megabanks/mufg|MUFG]] 的商业银行核心法人。在 FSA 的银行牌照列表中以「株式会社三菱UFJ銀行」(都市銀行) 登载，银行代码为 0005。它持有与 [[trust-banks/mitsubishi-ufj-trust-bank|三菱UFJ信託銀行]]、证券 HD、卡 / 消费金融子公司不同的 bank-license layer。

## 2. Operating model

- 承担 MUFG 集团的存款受理・放贷・国内/外汇・结算・海外商业银行网络的核心法人。
- 亚洲商业银行投资、法人结算、外国银行代理业务、信托契约代理等，许多业务系于银行单体牌照。
- bank-license layer 在功能上与 [[trust-banks/mitsubishi-ufj-trust-bank|三菱UFJ信託銀行]] (trust / asset administration / real estate / pension) 以及证券子公司的 investment-banking / brokerage layer 相分离。

## 3. Why standalone page matters

[[megabanks/mufg|MUFG]] 是上市控股公司，是观察合并决算・资本政策・Morgan Stanley alliance 的单位。三菱UFJ銀行是追踪银行法上的牌照、存款基础、法人结算、海外分行、外国银行代理、信托代理的单位。尤其在 stablecoin / Progmat / EPI 周边，需要将商业银行与信托银行的边界分开分析。
