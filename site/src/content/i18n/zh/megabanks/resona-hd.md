# Resona Holdings（りそな控股）

## Wiki route

本条目位于 [[megabanks/INDEX|megabanks INDEX]] 之下。可与 [[foreign-financial-institutions/citigroup-japan|シティグループ・ジャパン (Citigroup Japan)]] 对照阅读同业 / 对比语境，并与 [[banking/INDEX|banking index]] 阅读更广泛的系统 / 监管边界。

## TL;DR

定位为继 3  大 megabank（[[megabanks/mufg]] / [[megabanks/smfg]] / [[megabanks/mizuho-fg]]）之后的“**第 4 家 megabank**”。作为**拥有信托功能的商业银行 G**，其不分离信托银行，而是在银行兼业中运营信托功能，是日本首个正式的“**商银 + 信托**”模式。其重整史为 2003-05 公共资金 **1 兆 9600 億円**注入（实质国有化）→ 2015-06 全额偿还。地盘为**首都圈（埼玉）+ 关西**。2021-04 关西未来 FG 完全子公司化 → **2024-04 KMFG 吸收合并・关西未来银 + みなと银直接子公司化**，完成关西整合阶段。FY2024（2025-03 期）净利润 **2,133 億円** / 总资产 **77.4 兆円**。^[extracted]

### 主要子公司 / 持分

```
Resona HD（持股公司・东证 PRIME 8308）
  ├── Resona 银行（100%）── 原大和银行 + 朝日银行（一部分）整合・大阪 / 东京双本社
  │     └── “拥有信托功能的商业银行”模式本体
  ├── 埼玉 Resona 银行（100%）── 继承原朝日银行埼玉县内业务（2003-03 分拆设立）
  ├── 关西未来银行（100%，2024-04 直接子公司化）★ ── 原关西 Urban 银 + 近畿大阪银整合（2019-04）
  ├── [[regional-banks/minato-bank|みなと銀行]]（100%，2024-04 直接子公司化）★ ── 原阪神银行系
  │     ※ 2021-04 关西未来 FG 完全子公司化 → 2024-04 KMFG 吸收合并，消除中间持股
  ├── Resona Asset Management ── 资产管理
  └── （信托功能在子银行内兼业 = 不拥有独立信托银行）
```

> **结构特征**：3 大 megabank 将“信托银行另设法人”，而 Resona **在银行本体兼业信托功能**。这是“日本首个正式商业银行”模式的依据。^[extracted]

### 合并历史 / 前身

- 原 **大和银行**：1918 创立（野村财阀系）
- 原 **朝日银行**：1991-04 协和银行 + 埼玉银行合并成立
- 原 **近畿大阪银行**：1999-04 近畿银行 + 大阪银行合并
- 原 **奈良银行**：地银系
- 1999 大和银 + 朝日银开始研究整合
- 2001-12-12 **大和银 Holdings**（持股公司）设立
- 2003-03 改称“Resona HD”，集团重组

### “日本首个正式商业银行 + 信托”模式

- 3 大 megabank（[[megabanks/mufg]] / [[megabanks/smfg]] / [[megabanks/mizuho-fg]]）**将信托银行另设法人**（三菱 UFJ 信托・三井住友信托・Mizuho 信托）
- Resona **在银行本体兼业信托功能** → 对中坚客户以“银行窗口一站式信托提供”形成差异化

### 个人 / 中小企业特化

- Wholesale IB（大型 M&A、全球 DCM/ECM）有限
- 两大地盘为首都圈（埼玉）和关西；不是全国展开，而是区域特化

### 关西整合阶段完成（2024-04）

- 2017-11 关西未来 FG 设立 → 关西 Urban + 近畿大阪 + みなと 的 3 行整合平台
- 2019-04 关西 Urban + 近畿大阪 → 关西未来银行
- **2021-04 关西未来 FG 完全子公司化**（解除亲子上市）^[extracted]
- **2024-04 KMFG 吸收合并・关西未来银 + みなと银直接子公司化** → 消除中间持股阶段 ^[extracted]
- 关西区域整合完成 → 下一阶段是“实现整合后 synergy”

### 公共资金全额偿还后的稳定经营 / 重视分红

- 2003-05 注入 → 2015-06 全额偿还（**12 年内 1.96 兆円全额偿还**）
- 偿还后重视核心收益、扩大股东回报

### 竞争 / 分工

- vs 3 大 megabank：规模差大（[[megabanks/mufg]] 总资产 413 兆 vs Resona 数十万亿）→ 在大型 IB / 全球法人融资上不竞争
- vs 地银（[[megabanks/ndfg]] 等）：以信托功能和首都圈 / 关西城市部地盘形成差异化

## 4. 监管 / 政策

- **主管**：金融厅（FSA）
- **持股公司监管**：银行法第 52 条之 17

- **信托兼业许可**：银行本体兼营信托业务基于特别批准
- **近期政策论点**：
  - 2024〜 日本银行政策利率正常化 → 利差改善 bonus
  - 2024〜 KMFG 吸收合并后实现关西整合 synergy
  - 2025〜 面向个人的 digital strategy（与 retail 特化的一致性）
