# 瑞穗 FG (Mizuho FG)

## Wiki route

本条目归属于 [[megabanks/INDEX|megabanks INDEX]]。请对照 [[megabanks/mufg|三菱 UFJ FG (MUFG)]] 阅读以获取同业／对比背景，并对照 [[banking/INDEX|banking index]] 阅读以了解更广泛的体系／监管边界。

## TL;DR

日本第 3 大金融集团・3 大超大型银行之一角（总资产 **283.3 兆円**、净利润 **8854 億円**（历史最高）/ 2025-03 期实绩）^[extracted]。是以 2003-01-08 的**第一劝业银行 + 富士银行 + 日本兴业银行 3 行整合**为起源的控股公司。3 行合并所致的结构性复杂性与**2002 / 2011 / 2021 接连发生的大规模系统故障**长期成为品牌受损因素。以**「One MIZUHO」**战略转向集团一体化运营。通过**与乐天证券的资本业务合作**（2022-11-〜）・**对 PayPay 银行的战略出资**・**与 CyberAgent 的合资（CA Mobile / 其他）**，以合作方式补足 B2C 数字短板。G-SIB Bucket 1。^[extracted]

### 合并经纬・前身

- 旧 **第一劝业银行 (DKB)**: 1971 第一银行 + 日本劝业银行 合并。旧第一银行为 1873 涩泽荣一创设的日本最古老银行
- 旧 **富士银行**: 1948〜（旧 安田银行 / 安田财阀系）战后 GHQ 财阀解体而改名
- 旧 **日本兴业银行 (IBJ)**: 1902 设立，作为特殊银行担当长期信用金融 → 战后普通银行化，其后作为**长期信用银行**专注法人金融
- **2000-09-29** 瑞穗控股设立（3 行完全子公司化，股份转移）^[extracted]
- **2002-04-01** 瑞穗银行 + 瑞穗 Corporate 银行 成立（个人 / 中小 vs 大企业 / 国际的功能性分割）^[extracted]
- **2003-01-08** 瑞穗 FG 成立（HD 上位的纯控股公司化）^[extracted]
- **2013-07-01** 瑞穗银行 + 瑞穗 Corporate 银行 → 新「瑞穗银行」**再整合**（功能性分割于 11 年重新审视）^[extracted]

### 数字战略

- **「One MIZUHO」战略**: 3 行文化整合・集团一体化运营的长期主题 ^[extracted]
- **e-Mizuho**: 面向个人的数字银行 ^[extracted]
- **MINORI**（瑞穗下一代核心系统）: 2019 年完全运行，超 4500 億円投资的超大型系统翻新。但也成为 2021 年接连故障的震源地 ^[extracted]
- **AI / ChatGPT 业务应用**: 于 2023-06 面向国内几乎全体员工（约 3.5 万人）开始对话型 AI 的使用。与日本微软共同开发，应用于资料制作・摘要・翻译・编程等。**超大型银行中首个**全体员工业务导入（于 2023-04 公布研讨）

### 合作・收购战略

- **乐天证券 资本业务合作**（2022-10〜）★: 瑞穗证券取得乐天证券股 约 49%（后续阶段增资至约 50% 弱），约 800 億円规模。**超大型银行首个对外部网络证券的深度出资**。零售客户渠道与产品平台共通化 ^[extracted]
- **美国 Greenhill 收购**（2023-09 完成，约 770 億円 / 5.5 億美元）: 将全球 M&A 顾问独立系纳入旗下，日资银行首个独立系投资银行收购 ^[extracted]
- **与第一生命的 JV**: Asset Management One（51:49） ^[extracted]
- **PayPay 证券 出资**（旧 One Tap BUY）: 瑞穗证券与 SoftBank 共同于 2020-10 合资化，瑞穗证券出资约 24.8%。2025-02  PayPay 将该公司并表为子公司后，瑞穗证券仍作为协业伙伴持续出资
- **与 CyberAgent 等的业务合作**: 瑞穗银行・CyberAgent・FreeWheel 于 2024-01-26 缔结营销平台事业的战略性业务合作，于 2024-10 启动数字标牌广告等广告事业
- **与丸红的 JV**: 瑞穗租赁（权益法 23.5%，丸红 23%）^[extracted]
- **与 Bain Capital 组建 PE 基金**（2025-09 发表）^[extracted]

### 海外战略

- **美洲**: 以 NY 为本部的 Mizuho Americas（旧瑞穗 Corporate 银行 NY 分行）+ Greenhill（独立系 M&A）
- **亚洲**: 中国（瑞穗银行中国）/ 泰国 / 印度尼西亚 / 新加坡 等
- **欧洲**: 伦敦 IBD（源自旧兴银伦敦）

### B2C 品牌

- **企业色**: 瑞穗蓝（信赖 / 智性）+ 稻穗主题 logo ^[extracted]

## 4. 规制・政策

- **主管**: 金融厅（FSA）
- **控股公司规制**: 银行法 第 52 条之 17
- **海外规制**: 美联储（FRB / OCC）、ECB / BoE、中国 CBIRC、泰国 BoT 等
- **业务改善命令**: **以 2002 / 2011 / 2021 与 3 次大规模系统故障多次收到业务改善命令**。2021-11 命令为史上第 2 次的银行业务改善命令（含过去）^[extracted]
- **近期政策论点**:
  - 2024〜 日银政策利率正常化 → 国内利率利差改善红利（超大型银行 3 行共通）
  - 2025〜 中计 2024-2026 进展・ROE 8% 目标达成
  - 强化可持续金融・脱碳融资政策
