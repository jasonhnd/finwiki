# 三井住友银行 (SMBC)

## Wiki route

本条目位于 [[megabanks/INDEX|megabanks INDEX]] 之下。可与 [[megabanks/smfg|SMFG (親会社)]]、[[megabanks/mufg-bank|MUFG 銀行]]、[[megabanks/mizuho-bank|みずほ銀行]] 对读，以理解 3  大型都市银行之间的同业 / 对照关系；也可与 [[financial-regulators/fsa|金融庁]] / [[financial-regulators/boj|日本銀行]] 对读，以理解更广的监管边界。

## TL;DR

SMBC 是 [[megabanks/smfg|SMFG（三井住友フィナンシャルグループ 8316）]] 的核心子公司和 100% 持有的商业银行，总资产约 250 兆円，是日本 3  大型都市银行之一。它于 2001-04-01 由 Sakura Bank（其本身由三井银行和太阳神户银行整合而来）与住友银行合并成立。本店位于东京千代田区丸之内。业务覆盖零售、中堅 / 大企业、国际业务以及投行业务相邻领域。集团内，它与证券业务 [[securities-firms/smbc-nikko|SMBC 日興証券]]、信托 / 私人银行 [[trust-banks/smbc-trust-bank|SMBC 信託銀行]]、卡业务 [[card-issuers/smbc-card|三井住友カード]]、消费者金融 [[consumer-finance/smbc-consumer-finance|SMBC コンシューマーファイナンス（プロミス）]] 分工。全球展开以美洲和亚洲为中心。

## 1. 法人结构

**法定名称**：株式会社三井住友银行  
**英文名称**：Sumitomo Mitsui Banking Corporation (SMBC)  
**设立**：**2001-04-01**，由 Sakura Bank + Sumitomo Bank 合并而来  
**本店**：东京都千代田区丸之内 1-1-2；原住友银行大阪本店也作为大阪据点保留  
**业务类型**：商业银行 / 普通银行  
**主要股东**：**[[megabanks/smfg|三井住友フィナンシャルグループ]] 100%**  
**主要监管**：[[financial-regulators/fsa|金融庁]] 与 [[financial-regulators/boj|日本銀行]]

### 沿革 / 前身路线

```
1876    三井银行设立，日本最古老银行之一
1895    住友银行设立
1990    三井银行 + 太阳神户银行 -> Sakura Bank
2001-04-01 Sakura Bank + Sumitomo Bank -> **三井住友银行 (SMBC)**
2002-12    控股公司 [[megabanks/smfg|SMFG]] 设立 -> SMBC 成为 SMFG 全资子公司
```

## 3. 业务分部

| 分部 | 主要内容 |
|---|---|
| **零售** | 个人存款、住房贷款、投信销售和个人资产管理 |
| **Wholesale** | 面向中堅 / 大企业的融资、银团贷款、DCM / ECM 和 M&A 咨询 |
| **国际业务** | 通过北美、欧洲、亚洲、拉美据点提供当地融资和 transaction banking |
| **市场业务** | 外汇、债券、衍生品和交易 |
| **Transaction banking** | 法人结算、现金管理和贸易金融 |

### 主要据点

- **国内**：全国约 442 店 网点规模，随时期变化。
- **国际**：超过 50  个国家，主要据点包括纽约、伦敦、香港、新加坡、上海、雅加达等。

## 4. 竞争定位

### 与其他大型都市银行比较

| 项目 | SMBC | [[megabanks/mufg-bank|MUFG]] | [[megabanks/mizuho-bank|みずほ]] |
|---|---|---|---|
| 母公司 | [[megabanks/smfg|SMFG (8316)]] | [[megabanks/mufg|MUFG (8306)]] | [[megabanks/mizuho-fg|MHFG (8411)]] |
| 总资产 | 约 250 兆円 | 约 400 兆円 | 约 250 兆円 |
| 国际展开 | 以美国 + 亚洲为中心 | 美国 Morgan Stanley 出资 + 亚洲 / 泰国等 | 美国 / 欧洲 + 亚洲 |
| 国内份额 | 旧住友 + 旧三井 + 旧太阳神户 | 旧三菱 + 旧 UFJ + 旧东京 + 旧三和 + 旧东海 | 旧 IBJ + 旧第一劝业 + 旧富士 |
| 强项领域 | 项目融资、航空器租赁和 wholesale 机动性 | 信托与 Morgan Stanley 联动投行 | 信托与国际 ECM |

### 战略主题

- **资本效率重视**：ROE / RORA 经营纪律。
- **GX / 可持续金融**：强化 SLB / 绿色贷款。
- **美国市场深耕**：以 2023  Jefferies 战略联盟为轴，强化美国 IB 和 leveraged finance。
- **亚洲 / 新兴市场**：越南、印尼的个人业务，以及泰国、印度等 wholesale。

## 5. 监管 / 控制

- **主要监管**：[[financial-regulators/fsa|金融庁]]（银行业法）。
- **宏观审慎**：[[financial-regulators/boj|日本銀行]] 的考查与金融系统监测。
- **资本监管**：Basel III + G-SIB buffer。
- **G-SIB 指定**：SMFG 被指定为 G-SIB bucket 1  2024。
