# 株式会社三井住友银行 (SMBC)

## Wiki 路径

本条目位于 [[megabanks/INDEX|megabanks INDEX]] 之下。请将其与 [[megabanks/smfg|SMFG (親会社)]]、[[megabanks/mufg-bank|MUFG 銀行]]、[[megabanks/mizuho-bank|みずほ銀行]] 对照阅读以了解同业 / 对比（3 大巨型银行），并与 [[financial-regulators/fsa|金融庁]] / [[financial-regulators/boj|日本銀行]] 对照以了解更广泛的监管边界。

## TL;DR

**[[megabanks/smfg|SMFG（三井住友フィナンシャルグループ 8316）]] 的核心子公司·100% 持有的商业银行**。总资产 250 兆円规模，是**日本 3 大巨型银行之一**。**由 2001-04-01 樱花银行（旧三井 + 旧太阳神户）与住友银行合并而成立**。总行位于东京都千代田区丸之内（旧住友银行·旧三井银行系统整合本部）。承担商业银行业务（零售 + 中坚·大企业 + 国际 + 投资银行），在集团内与 [[securities-firms/smbc-nikko|SMBC 日興証券]]（证券）·[[trust-banks/smbc-trust-bank|SMBC 信託銀行]]（信托）·[[card-issuers/smbc-card|三井住友カード]]（信用卡）·[[consumer-finance/smbc-consumer-finance|SMBC コンシューマーファイナンス（プロミス）]]（消费金融）分工。全球以美洲·亚洲为中心展开。

## 1. 法人结构

**正式名称**：株式会社三井住友银行
**英文名**：Sumitomo Mitsui Banking Corporation (SMBC)
**设立**：**2001-04-01**（樱花银行 + 住友银行 合并）
**总行**：东京都千代田区丸之内 1-1-2（东京日本桥总行，大阪总行并设）
**业态**：商业银行（普通银行）
**主要股东**：**[[megabanks/smfg|三井住友フィナンシャルグループ]] 100%**
**主管**：[[financial-regulators/fsa|金融庁]]·[[financial-regulators/boj|日本銀行]]

### 沿革·前身

```
1876   三井银行 设立（日本最古老的银行之一）
1895   住友银行 设立
1990   三井银行 + 太阳神户银行 → 樱花银行
2001-04-01   樱花银行 + 住友银行 → **三井住友银行 (SMBC)**
2002-12   持株公司 [[megabanks/smfg|SMFG]] 设立 → SMBC 成为 SMFG 全资子公司
```

## 3. 业务分部

| 分部 | 主要内容 |
|---|---|
| **零售** | 个人存款 + 住房贷款 + 投资信托销售 + 面向个人的资产管理 |
| **批发** | 面向中坚·大企业的贷款 + 银团贷款 + DCM/ECM + M&A 顾问 |
| **国际业务** | 通过北美·欧洲·亚洲·中南美据点的当地贷款·交易银行 |
| **市场业务** | 外汇·债券·衍生品·交易 |
| **交易银行** | 面向法人的结算·现金管理·贸易融资 |

### 主要据点

- **国内**：全国 442 店店铺规模（随时期变动）
- **国际**：以纽约·伦敦·香港·新加坡·上海·雅加达等为主要据点的全球 50 多个国家展开

## 4. 竞争地位

### vs 其他巨型银行

| 项目 | SMBC | [[megabanks/mufg-bank|MUFG]] | [[megabanks/mizuho-bank|みずほ]] |
|---|---|---|---|
| 母公司 | [[megabanks/smfg|SMFG (8316)]] | [[megabanks/mufg|MUFG (8306)]] | [[megabanks/mizuho-fg|MHFG (8411)]] |
| 总资产 | 约 250 兆円规模 | 约 400 兆円规模 | 约 250 兆円规模 |
| 国际展开 | 美洲 + 亚洲中心 | 美国摩根士丹利出资 + 亚洲·泰国等 | 美欧 + 亚洲 |
| 国内份额 | 旧住友 + 旧三井 + 旧太阳神户 | 旧三菱 + 旧 UFJ + 旧东京 + 旧三和 + 旧东海 | 旧 IBJ + 旧第一劝银 + 旧富士 |
| 优势领域 | 项目融资 + 飞机租赁 + 批发机动性 | 信托·经由美国摩根的投资银行 | 信托·国际 ECM |

### 战略主题

- **重视资本效率**：彻底推行 ROE / RORA 经营
- **GX / 可持续金融**：强化 SLB / 绿色贷款
- **深耕美国市场**：以与 2023 Jefferies（Jefferies）的战略资本业务合作为轴，强化美国 IB·杠杆融资
- **亚洲·新兴国**：面向个人（越南·印度尼西亚）+ 批发（泰国·印度）

## 5. 监督·监管

- **主管**：[[financial-regulators/fsa|金融庁]]（银行法）
- **宏观审慎**：[[financial-regulators/boj|日本銀行]] 考察·金融机构局监测
- **资本监管**：巴塞尔 III + G-SIB（global systemically important bank）缓冲
- **G-SIB 指定**：SMFG 被指定为 G-SIB 桶 1 （2024）
