# 日本邮政 (Japan Post Holdings)

## Wiki route

本条目位于 [[megabanks/INDEX|megabanks INDEX]] 之下。请与 [[life-insurers/kampo-life|かんぽ生命保険 (Japan Post Insurance)]] 对照阅读以了解同业／对照背景，并与 [[banking/INDEX|banking index]] 对照以了解更广泛的系统／监管边界。

## TL;DR

由旧 日本邮政公社 改组而来的「邮政 + 金融 [银行 + 保险]」3 大支柱上市控股公司。**国内最大级别的金融集团**（合并口径下集团运用资产规模处于 250 兆円超 的区间）。于 2015-11-04 将控股公司 / [[regional-banks/yucho]] / [[life-insurers/kampo-life]] **3 社同时 IPO** ★。第一大股东为财务大臣（依郵政民营化法**当下负有持有 1/3 超** 的义务）。2019-2020 因 [[life-insurers/kampo-life]] 不当销售事件而经历管理层更替・业务改善命令，恢复信任与改善运用收益率为中期主题。^[extracted]

## 1. 公司概要

**正式名称**：日本邮政株式会社 ^[extracted]
**英文名**：Japan Post Holdings Co., Ltd.
**证券代码**：东证 PRIME 6178（2015-11-04 上市）^[extracted]
**设立**：2006-01-23（作为邮政民营化筹备公司「日本邮政株式会社」）^[extracted]
**民营化发足**：2007-10-01（郵政民营化法施行・4 社体制启动）

### 股东结构（民营化法上的特例）^[extracted]

| 股东 | 比例 | 备注 |
|---|---|---|
| 财务大臣（日本国政府）| **1/3 超**（当下维持义务）| 郵政民营化法・日本邮政株式会社法 |
| 一般股东・机构投资者 | 余额 | 分阶段出售（2015-11 / 2017-09 / 2021-10 等）|

> 依据日本邮政株式会社法及郵政民营化法，政府**当下负有持有 3 分之 1 超的义务**。此外，日本邮政 自身亦就 [[regional-banks/yucho]] / [[life-insurers/kampo-life]] 各公司股份的处置被课以分阶段目标，金融 2 社的政府参与以双重结构留存。^[extracted]

> 作为上市控股公司的沿革・市场区分・政府持股方案的细节，已分离整理至 companion 页面 [[megabanks/japan-post-holdings|日本郵政株式会社 (operating profile)]]。

```
  │     └── 商品销售・便利店运营（罗森・全家提携等）
  ├── 邮储银行株式会社 (ゆうちょ銀行)（上市 7182）── 存款规模国内最大级 [[regional-banks/yucho]]
  ├── 简保生命保险株式会社 (かんぽ生命保険)（上市 7181）── 简易生命保险为基础 [[life-insurers/kampo-life]]
  ├── 日本邮政资本株式会社 (日本郵政キャピタル)（100%）── 集团 CVC・出资职能
  ├── JP 商务邮件株式会社 (JP ビズメール) ── DM・邮寄 BPO
  └── （旧）乐天集团 出资（2021-03 约 1,500 億円 → 2024-03 几乎已售出）
```

## 2. 业务分部・地图

| 分部 | 主要经营者 | 特征 |
|---|---|---|
| 邮政・物流 | 日本邮便 | 国内普遍服务义务，~24,000 邮局网 |
| EC・配送 | 日本邮便（邮包）| 承接 Amazon・乐天・Mercari 等的快递 BPO |
| 零售银行 | [[regional-banks/yucho]]（上市 7182）| 存款规模 国内最大级，于邮局窗口开展账户业务 |
| 生命保险 | [[life-insurers/kampo-life]]（上市 7181）| 以简易生命保险为基础，保费收入居大型之列 |
| 不动产 | 日本邮政不动产（含 JP 大厦）| 大手町・邮局旧址开发 |
| 集团 CVC | 日本邮政资本 | 对初创・物流科技出资 |
| 老年・地区 | 日本邮便「看护服务 (みまもりサービス)」等 | 将邮局网打造为护理・地方政府联动平台 |

### 集团整合协同

- **邮局网 × 金融**: 将全国 ~24,000 局用作 [[regional-banks/yucho]] / [[life-insurers/kampo-life]] 的销售窗口。窗口手续费（金融 2 社 → 日本邮便）是集团内部转移的主轴 ^[extracted]
- **物流 × EC**: 日本邮便的最后一公里网络与 EC 平台的联动。旧 乐天 G 提携曾在此语境中加以定位 ^[extracted]

### 提携・收购战略

- **乐天集团资本业务提携**（2021-03-12）: 日本邮政认购乐天集团（4755）的第三方定向增资，出资约 1,500 億円（出资比例 8.32%、第 4 位大股东）。在物流・移动通信・金融・EC 领域联动。其后，因乐天移动业务不振导致乐天 G 股价下跌，日本邮政在 2023 年 4〜6 月期计提约 800〜850 億円的特别损失（减值）
- **金融 2 社股份的分阶段处置**: 依民营化法，日本邮政以分阶段处置 [[regional-banks/yucho]] / [[life-insurers/kampo-life]] 股为目标。另一方面，由于邮局网与金融销售的不可分性，完全分离面临政治阻力 ^[extracted]
- **政府持股的追加出售**: 作为重建财源・国债偿还财源，政府持股出售已多次实施

### 海外战略

- 旧 Toll Holdings（澳洲物流巨头，2015-05 ~6,200 億円收购）→ **2021-04 出售**（计提巨额商誉减值后）。海外物流 M&A 实质上处于撤退模式 ^[extracted]
- [[regional-banks/yucho]] / [[life-insurers/kampo-life]] 在运用层面通过海外债券・另类资产扩大全球敞口

## 4. 监管・政策

- **主管**：总务省（邮政行政・普遍服务）+ 金融厅（[[regional-banks/yucho]] 银行业 / [[life-insurers/kampo-life]] 保险业）
- **根据法**：郵政民营化法 / 日本邮政株式会社法 / 日本邮便株式会社法 / 邮储银行株式会社・简保生命保险株式会社的各项根据法
- **政府持股义务**：日本邮政股由财务大臣**当下持有 1/3 超** ^[extracted]
- **金融 2 社的处置目标**：日本邮政分阶段处置 [[regional-banks/yucho]] / [[life-insurers/kampo-life]] 股（时期与比例以政令・内阁决议运作）
- **普遍服务义务**：日本邮便负有提供全国统一资费邮政服务的义务
- **近期政策议题**:
  - 2019-2020 [[life-insurers/kampo-life]] 不当销售事件 → 治理・销售体制改革
  - 2024 〜 邮政资费上调（25 g 84 円 → 110 円 等）与邮政事业转为亏损
  - 2025 〜 数字邮政・电子信箱・MyPost 构想
