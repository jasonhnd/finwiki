# 日本建筑基金（NBF、J-REIT 8951）

## 长话短说

**Nippon Building Fund Inc. (NBF、TSE J-REIT 8951)** 是日本上市 REIT 市场 — 的 **旗舰办公室 J-REIT**，由 [[real-estate-finance/mitsui-fudosan-financing-model|Mitsui Fudosan]] (TSE Prime 8801) 赞助，历史上是大多数资产管理规模最大的 J-REIT** J-REIT 时代，以及每个全球 REIT 指数日本袖子中的基准名称。 NBF 专注于**东京中央商务区办公楼** — 大手町、丸之内、日本桥、涩谷和新宿 —，拥有主要从三井不动产赞助商渠道收购的长期稳定的大型多租户办公楼投资组合。 NBF 是**首次上市的日本 REIT**（9 月 2001），并与 [[real-estate-finance/japan-real-estate-j-reit-8952|Japan Real Estate (JRE, 8952)]] — 三菱地产赞助的写字楼 J-REIT — 配对，作为上市写字楼 REIT 市场中三井与三菱的结构性竞争代理。

NBF 的投资者概况结合了 (i) 40% 中低范围内的**保守 LTV**，(ii) **AA 国内评级区信贷**，支持严格的投资公司债券定价，(iii) **J-REIT 分布低端的 DPU 收益率**（反映赞助商 + 东京 CBD 办公室 + 规模 + 流动性溢价），以及 (iv) ** 较高的外国投资者相关性**作为全球房地产投资信托基金指令中的基准日本办事处名称。顶级租户的风险在结构上是适中的（通常没有一个租户占总租金的中个位数百分比以上），因为该投资组合由大型多租户办公楼组成，而不是由单租户定制资产组成。

## 2。投资组合构成

| 轴 | NBF模式 |
|---|---|
| 资产类别 | 办公室（绝大多数）；轻微的非办公室接触历来都是偶然的 |
| 地理集中度 | 东京23 wards（重量级）；规模较小的二级城市（横滨、大阪、名古屋、福冈、札幌） |
| 东京子市场焦点 | 大手町/丸之内/日本桥（三井不动产核心地区）、涩谷、新宿、虎之门、霞关 |
| 物业规模 | 大多为大型多租户办公楼 — 大楼层，每项资产多个租户 |
| 采集管道源 | 主要来自三井不动产赞助商管道（资产回收—赞助商将稳定的财产出售给NBF并将资本重新部署到下一个开发周期） |
| 房产年龄 | 由赞助商提供的较旧的稳定核心建筑和较新的重建竣工资产的组合 |

## 3。资本和杠杆

| 项目 | NBF模式 |
|---|---|
| LTV 政策区间 | 红外材料中公开的保守 — 通常为低至中 40% 区域 |
| 债务组合 | 银行贷款（大型银行+信托银行银团）和**投资公司债券**（公募投資法人債）的组合以AA级国内评级公开发行，利差较小 |
| 债券曲线 | NBF 投资公司债券是办公 J-REIT 领域的基准发行；期限 5Y、7Y、10Y，较长期机会主义 |
| 赞助商支持股权 | 三井不动产保留少量赞助商支持单位持有人股权（个位数百分比），与 [[real-estate-finance/j-reit-sponsor-structure-conflict|sponsor-support pattern]] 一致 |
| 分销政策 | 半年度DPU； J-REIT 90% 传递结构符合 [[real-estate-finance/j-reit-market-overview|J-REIT market overview]] |
| 外资股 | 全球 REIT 委托中的高 — 基准名称（参见 [[real-estate-finance/j-reit-foreign-investor-ownership|J-REIT foreign investor ownership]]） |

## 4。顶级租户曝光度

NBF 的大型多租户办公楼组合从结构上限制了单租户集中度。公共来源的观察：

| 租户集中度指标 | NBF模式 |
|---|---|
| 赞助商（三井不动产）作为租户 | 最小 — Mitsui Fudosan 是赞助商和开发商，不是 NBF 投资组合中的主要租户 |
| 租户行业组合 | 业务多元化，涵盖金融服务、专业服务、IT/电信、制造业、政府 |
| 租户租赁结构 | 标准日本定期租赁 (普通借家契約 / 定期借家契約) 具有定期租金重置机制 |

多租户结构是 NBF 的 DPU 比单租户物流 J-REIT（每项资产集中于一到两个租户）波动性更小的关键原因。

## 5。 NBF vs JRE — MEC vs Mitsui 竞争代理

| 轴 | NBF（三井不动产） | JRE（三菱地所） |
|---|---|---|
| 股票行情指示器 | 8951 | 8952 |
| 赞助商 | [[real-estate-finance/mitsui-fudosan-financing-model|Mitsui Fudosan]] | [[real-estate-finance/mitsubishi-estate-financing-model|Mitsubishi Estate]] |
| 上市日期 | 九月 2001 （第一） | 9月 2001 (并列第一) |
| 东京子市场主力 | 大手町/日本桥/涩谷/新宿 | 丸之内/大手町/有乐町 |
| 管道 | 三井不动产资产回收 | 三菱地所资产回收 |
| 资产管理规模规模 | 顶级；与 JRE 争夺最大写字楼 J-REIT 地位 | 顶级；与NBF竞争 |
| 生命周期价值 | 低至中 40% | 低至中 40% |
| DPU屈服区 | 溢价（J-REIT 分布的低端） | 高级（低端） |
| 外国投资者概况 | 基准日本办事处名称 | 基准日本办事处名称 |

NBF-JRE 配对是更广泛的三井不动产与三菱地产竞争 — 的**上市代理**，这两家顶级上市开发商各自拥有顶级办公 J-REIT，并将 CBD 办公楼回收到各自的车辆中。

## 6。对位法

- **“NBF 只是三井不动产的稳定资产桶”** — 部分正确。赞助商-管道依赖性是[[real-estate-finance/j-reit-sponsor-structure-conflict|external-management J-REIT]]的结构特征； NBF的关联交易治理是保护层。
- **“随着远程工作的继续，Office J-REIT 溢价将压缩”** — 争论道。疫情过后，东京 CBD 写字楼需求比美国 CBD 更具弹性；东京市中心的多租户大型建筑模式比单租户郊区办公室更能保留租户需求。
- **“NBF DPU 收益率太低，没有吸引力”** — 取决于基准。收益率相对于 JGB 较高（适用 [[real-estate-finance/j-reit-dividend-yield-vs-jgb-spread|J-REIT dividend yield vs JGB spread]] 框架）；但在对冲美元的基础上，收益率可能没有吸引力 — 参见 [[real-estate-finance/j-reit-foreign-investor-ownership|J-REIT foreign investor ownership]]。
- **“保守的 LTV 过于谨慎”** — 的反驳是，AA 区评级和紧缩的债券定价经济使得较低的 LTV 在整个周期的基础上是值得的。
- **“考虑到成熟的三井不动产管道，NBF 将不会进一步增长”** — 三井不动产继续大规模 CBD 重建，因此赞助商管道供应继续存在；增长速度随重建周期而变化。
