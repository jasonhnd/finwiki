# 日本国库短期证券(TDB / 国庫短期証券)

## Wiki 路线

此条目位于 [[money-market/INDEX|money-market index]] 下。请与 [[money-market/japan-cp-commercial-paper-market|Japan CP market]] 对照阅读以获得同类/对比背景,与 [[money-market/jgb-repo-market-japan|JGB repo market]] 对照阅读以获得更广泛的系统/监管边界。

## TL;DR

国库短期证券(TDB / 国庫短期証券)是由日本财务省(MOF)以 3 月、6 月和 1 年期限发行的短期零息政府证券。TDB 是短期日元货币市场的主权构建模块,也是 [[money-market/japan-cp-commercial-paper-market|CP]]、[[money-market/japan-ncd-negotiable-cd-market|NCD]] 和短期日元回购定价的自然无风险基准。

TDB 位于 MOF 的常规 JGB 拍卖日历上,通过日本央行 JGB 账簿转账系统以非实物形式持有,并被接受为日本央行公开市场操作和双边回购最纯净的日元抵押品。它们也是外国投资者进入日元短期利率敞口的关键切入点。

## 发行机制

| 要素 | 阅读 |
|---|---|
| 期限 | 3 月、6 月和 1 年。现金管理票据也可能为财政周期平滑而发行。 |
| 形式 | 零息折扣;投资者在到期时收到面值。 |
| 发行方式 | 常规收益率竞争性拍卖;结果由 MOF/日本央行披露。 |
| 频率 | 取决于期限的常规每周/每月节奏;参见 MOF JGB 拍卖日历。 |
| 登记 | 日本央行 JGB 账簿转账系统;完全非实物化。 |
| 税收 | 在指定外国投资者豁免制度下,非居民持有人零预扣。 |

## 拍卖节奏

MOF 发布涵盖息票 JGB 和 TDB 的 JGB 拍卖日历。TDB 拍卖周期是日本政府债券市场中最可靠的发行节奏之一:短期限需要近每周的 3月发行,以及周期性的 6月和 1年操作。当财政周期需要时,可以增加现金管理票据发行。

拍卖是统一价格或常规的,投标人提交收益率投标。JGB 市场的主要交易商是主要投标人,但通过交易商中介存在非交易商投标路线。结果当日发布,提供清晰的短期利率标记。

## 投资者组成

对于日元货币市场工具而言,TDB 持有异常多样化:

- **外国投资者**:TDB 在历史上吸引了大量外国投资者份额,用于短期日元敞口、日元融资管理和货币对冲基差交易。外国持有份额随外汇对冲收益率差异而显著波动。
- **日本央行**:日本央行在宽松阶段通过直接购买操作持有 TDB;持有量随政策制度而变化。
- **国内银行**:银行在其银行账簿中持有 TDB,用于高质量流动资产、日本央行抵押品和短期日元收益。
- **MMF/MRF/短期日元基金**:TDB 是这些工具的规范持有。
- **保险和信托账户**:TDB 提供短久期主权敞口。

MOF 发布按投资者类型的 JGB 和 TDB 持有细分,应查阅以了解当前数字。

## 日本央行与 TDB

日本央行在多个操作表面使用 TDB:

| 日本央行功能 | TDB 联系 |
|---|---|
| 直接购买 | 日本央行在过去框架中购买 TDB 作为供给资金操作的一部分。 |
| 回购/抵押品 | TDB 被接受为日本央行联合抵押品贷款和其他有抵押品操作的纯净日元抵押品。 |
| 日本央行 NET | TDB 结算通过日本央行的 JGB 账簿转账通道运行。 |
| 政策传导 | TDB 收益提供清晰的短期利率标记,与 [[money-market/boj-post-2024-floor-system-complementary-deposit-facility|post-2024 floor system]] 政策利率互动。 |

## 阅读 TDB 收益

| 阅读视角 | 注意事项 |
|---|---|
| 主权短期利率 | 无风险短期日元利率的最接近市场阅读。 |
| OIS 利差 | TDB 收益相对于 TONA OIS 显示抵押品/稀缺性溢价。 |
| CP/NCD 利差 | TDB 在 [[money-market/japan-cp-commercial-paper-market|CP]] 和 [[money-market/japan-ncd-negotiable-cd-market|NCD]] 中作为信用利差的基准。 |
| 外汇对冲收益 | 外国投资者在 USD/EUR 对冲后的回报;驱动边际外国需求。 |
| 日本央行利率利差 | TDB 收益与补完当座预金制度利率之间的差距指示抵押品稀缺性。 |

## 后 2024  背景

在日本央行 2024 年 3 月结束负利率的政权改变之后,TDB 收益转正,拍卖日历继续提供常规的短期利率标记。阅读规则:

- 将政策利率决议与实际 TDB 成交收益分开;
- 观察外国投资者份额以了解由对冲收益算术驱动的变化;
- 检查日本央行是在购买、持有还是让到期滚动;
- 将 TDB-OIS 基差视为有用的抵押品和融资信号,而非单一原因指标。

## 边界情况

- **TDB 与短期息票 JGB**:TDB 是零息的,仅以 3M/6M/1Y 发行;具有相同范围剩余期限的短期息票 JGB 是不同的工具,具有单独的流动性特征交易。
- **TDB 与 CP/NCD**:TDB 是主权和无风险的;CP/NCD 承担发行人信用风险并以利差交易。
- **TDB 与现金管理票据**:现金管理票据为财政周期平滑而临时发行,不属于常规拍卖日程。
