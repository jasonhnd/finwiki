# 日本保险牌照与偿付能力路径

## 概览

日本保险监管将 **承担风险的保险公司** 与 **分销中介** 分开。寿险公司、非寿险公司、保险控股公司、少额短期保险公司、保险代理人和保险经纪人，是不同的公开监管类别。承保实体承担保险负债和偿付能力监管；代理人 / 经纪人路径则规制招揽、媒介、客户说明和行为规范。

本页作为 [[insurance/INDEX|insurance domain]] 页面使用的法律控制层。请与 [[insurance/economic-value-based-solvency|economic value-based solvency regulation]]、[[insurance/esr-economic-value-solvency|ESR]]、[[insurance/insurance-agency-and-brokerage-japan|insurance agency and brokerage Japan]]、[[insurance/japan-life-insurance-big-four|Japan life insurance big four]]、[[insurance/japan-nonlife-big-three|Japan non-life big three]]，以及 [[life-insurers/nippon-life|Nippon Life]]、[[life-insurers/dai-ichi-life|Dai-ichi Life]]、[[non-life-insurers/tokio-marine|Tokio Marine]]、[[non-life-insurers/msad|MS&AD]]、[[non-life-insurers/sompo|Sompo]] 等保险公司实体页面一起阅读。

## 承保与分销

在日本，银行柜台、线上券商、零售商、电信集团、支付应用或旅行平台都可能分销保险，而保险风险仍留在持牌保险公司的资产负债表上。

承担风险的保险公司拥有保单持有人负债、承保风险、资产负债管理、赔付、准备金、偿付能力边际 / ESR 监管，以及产品审批 / 精算控制。分销方可以拥有客户触点和品牌体验，但法律合同和资本风险仍可能归属于另一实体。

同样的资产负债表 / 分销分离也出现在 [[banking/japan-baas-operating-models|BaaS operating models]]。嵌入式金融和嵌入式保险可以把客户界面放在平台上，而受监管的资产负债表归属于银行或保险公司。公开产品映射在有披露时记录缔约实体、保费接收方、承担赔付实体、牌照类别和招揽实体。

## 偿付能力路径

日本保险公司资本框架现在需要两种读法：

| 主题 | 解读规则 |
|---|---|
| 偿付能力边际 | 仍出现在披露和监管材料中的传统 / 法定资本充足性词汇。 |
| 经济价值基础偿付能力 | 金融厅围绕经济价值评估、保单持有人保护、风险管理和披露建立的框架。 |
| ESR | 保险公司按自身报告日期和定义披露的公司层面经济偿付能力比率。 |
| 支柱结构 | 金融厅围绕偿付能力规制、内部管理 / 监督审查和披露来框定经济价值制度。 |
| 来源日期 | 金融厅通知、现场测试材料、公司披露和实施材料都具有日期特定性。 |

公司层面的资本说明必须绑定到命名指标和来源日期。[[life-insurers/dai-ichi-life|Dai-ichi Life]]、[[non-life-insurers/tokio-marine|Tokio Marine]] 等上市集团发布综合报告 / 业绩披露材料。[[life-insurers/nippon-life|Nippon Life]]、[[life-insurers/meiji-yasuda|Meiji Yasuda]]、[[life-insurers/sumitomo-life|Sumitomo Life]] 等相互会社通过自身披露路径发布偿付能力、治理形式和盈余返还信息。

## 实体对照

| 问题 | 不应使用 | 更好的路径 |
|---|---|---|
| “这是保险公司吗？” | 只看品牌名 | 检查金融厅寿险 / 非寿险 / 少额短期名单和实体披露。 |
| “这是保险集团吗？” | 只看产品网站 | 检查集团是否有保险控股公司层，还是只有保险子公司。 |
| “这个平台能销售保险吗？” | 只看应用界面 | 检查代理 / 经纪 / 金融服务中介 / 合作保险公司结构。 |
| “ESR 高还是低？” | 不带日期复制单一比率 | 比较来源日期、产品组合、市场敏感性和披露目标区间。 |
| “这是 broker 吗？” | 英语口语中的 “broker” 用法 | 检查日本法律状态：代理人 / 招揽人 / 保险经纪人 / 中介。 |
