# Acom 业务详情（Acom Operating Profile）


## Wiki route

Read it against [[consumer-finance/smbc-consumer-finance|SMBCコンシューマーファイナンス (プロミス)]] for MUFG-group competitor SMFG-side context, [[consumer-finance/aiful|アイフル]] for independent-pure-play consumer-finance peer, and [[megabanks/mufg|MUFG]] / [[megabanks/mufg-bank|三菱UFJ銀行]] for parent-group context.

## TL;DR

[[consumer-finance/acom|アコム株式会社]]（8572  / 东证 STANDARD）是大型消费金融公司，同时是 **[[megabanks/mufg|MUFG]] 持分法适用公司（表决权基础上高于 39.6%）**。本条目覆盖业务板块、收益结构、保证业务、海外扩张与监管应对等 **operating-level depth**。其于 1936-04  在神户由木下政雄以“丸糸吴服店”创立，后于 1978  转型为消费金融并改名“Acom”，在 1996-10  于东证一部上市，并于 2008-10  通过 MUFG TOB 成为持分法适用公司。Acom 的**4 大战略核心板块**为：(a) 贷款（个人卡贷款，实体渠道为“むじんくん”自动合约机），(b) 信用卡（AC Mastercard），(c) **信用保证**（面向三菱UFJ银行及地银的银行贷款保证），(d) **海外金融**（如泰国 Easy Buy PCL）。

## 1. 与母集团的关系

- **MUFG 持股**：[[megabanks/mufg|MUFG]] 以表决权基础持有高于 39.6%（基于 2024  期报告，含表决权与信托合计）
- **并表 vs 持分法**：在 MUFG 合并口径下按持分法处理；Acom 自身仍为独立上市公司，在东证 STANDARD 8572  向股东进行披露
- **公司治理**：来自 MUFG 的派驻董事与创始人木下家系董事并存
- **MUFG 集团内定位**：消费金融业务归入 MUFG 的 **Retail & Commercial Banking Business Group**
- **主交易银行**：[[megabanks/mufg-bank|三菱UFJ銀行]] 是主交易银行，也是银行贷款保证业务的主要合作方

## 2. 业务板块详情

### 2.1  贷款业务（个人无担保贷款）

- **主力产品**：“Acom AC Mastercard”“AC Card Loan”
- **实体渠道**：**“むじんくん”自动合约机**长期以来是 Acom 的标志性渠道。从曾在全国布设超过 1,000  台的时代起，如今因转向 web 完结而持续收缩，但在地方城市仍有需求
- **线上渠道**：通过“Acom 官方网站”“Acom 官方 App”进行即时审查与即时放款
- **客户层**：个人（以 20 代至 50 代为主）、主妇、自营业者
- **利率水平**：年化 3.0%-18.0%（低于修订贷金业法上限 20%）
- **授信模型**：历史数据 + 行为数据 + 信用信息机构（CIC / JICC）数据的组合
- **监管边界**：与 [[consumer-finance/aiful|アイフル]] 相同，受《贷金业法》+ 修订贷金业法约束（总量规制：不超过年收入的 1/3 ）

### 2.2  信用卡业务

- **产品**：“AC Mastercard”（Master 品牌）
- **特点**：主打“可当日发卡”——仅限循环付款，内置现金预借功能
- **商户网络**：通过 Master 品牌接入全球
- **贷金业法 + 分期销售法**：卡贷部分适用贷金业法，购物部分适用分期销售法（CIA）

### 2.3  信用保证业务（战略核心）

- **业务内容**：银行向个人客户发放卡贷款，由 Acom 提供信用保证（银行面向客户放贷 → Acom 向银行提供保证）
- **主要保证对象**：
  - **[[megabanks/mufg-bank|三菱UFJ銀行]] Bank Quick**（最大保证对象，是银行贷款保证业务的核心）
  - 地方银行与第二地方银行的卡贷款
- **战略意义**：银行与消费金融之间的 **表内 / 表外分工模式** —— 银行利用存款资金进行低利率放贷，Acom 代行授信判断并承担风险
- **保证余额**：Acom 合并保证余额达数千亿至万亿日元级（参见官方 IR）

### 2.4  海外金融业务

- **泰国 Easy Buy PCL**：Acom 子公司，泰国领先的汽车 / 家电分期金融公司
- **移动 / 零售展业**：在泰国开展个人无担保贷款与分期金融
- **东南亚拓展**：在印尼、菲律宾等地仍有进一步扩张空间

## 3. 按法人板块划分的收益结构

| セグメント | 収益タイプ | 業界規模感での位置 |
|---|---|---|
| 贷款业务 | 利息收入 | 个人无担保 No.1〜2  |

| 信用保证 | 保证费 | **行业顶级** |
| 海外金融 | 利息 + 分期手续费 | 泰国最大 |

## 4. 监管应对与合规

- **修订贷金业法（2010 全面施行）**：上限利率 20% / 总量规制（年收入的 1/3）/ 催收限制 —— Acom 已完全应对
- **多收利息返还请求应对**：受 2006-2010  灰色利率最高法院判决影响，返还请求激增，公司持续计提拨备并实际支付。近期请求件数已进入下降阶段
- **反洗钱（AML）**：持续强化贷金业法项下的身份确认与交易时确认
- **以客户为本的业务运营原则**：应对 FSA 受托责任原则向零售金融领域的延伸
- **个人信息保护法**：管理通过 CIC / JICC 信用信息机构使用及向第三方提供个人信用信息

## 6. 战略与近期动向

- **向移动端 / web 完结转型**：将原本投向“むじんくん”自动合约机的投资，转向 web 申请与 App 审查
- **扩大 MUFG 银行贷款保证**：即使在银行低利率环境下，保证费收入仍较稳定，是战略扩张领域
- **扩大 Thailand Easy Buy**：吸收泰国中间层对汽车和家电分期的需求，提高海外收益占比
- **活用信用评分**：将行为数据、移动数据纳入授信模型
- **与 MUFG 零售战略联动**：通过保证贷款渠道，与 MUFG 零售客户基础进行 cross-sell

## 7. 风险与监管议题

- **国内贷款市场缩小**：随着日本向无现金支付与预付卡转移，现金贷需求结构性下滑
- **总量规制下新增授信空间受限**：在年收入 1/3  限制下，优质客户的额外授信空间可能见顶
- **利率上升情景**：融资成本上升导致息差承压风险
- **泰国业务的国家风险**：泰国政局、货币与监管变化风险
- **多收利息返还的长期尾部风险**：即使请求件数减少，拨备重估事件仍可能持续发生
- **AI 授信模型的公平性 / 可解释性**：随着授信判断 AI 化，歧视性处理与可解释性成为议题
