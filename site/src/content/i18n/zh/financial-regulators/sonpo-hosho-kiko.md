# 损害保险契约者保护机构

## Wiki 路径

本条目位于 [[JapanFG/INDEX|JapanFG index]] 之下。可结合 [[financial-regulators/seimei-hokensha-hogo-kiko|生命保険契約者保護機構]] 作为平行安全网同业，并结合 [[financial-regulators/dic|預金保険機構]] / [[financial-regulators/fsa|金融庁]] 理解更广泛的制度 / 监管边界。

## 摘要

这是一个**由日本国内经营的所有损害保险公司作为会员、用于保护投保人的认可法人**。它在 1998-12  修订《保险业法》施行时成立。它是负责**破产损保公司合同延续 + 保险金支付 + 合同承接**的最后安全网。资金由会员缴纳的“负担金”和政府“补助金（特别措施时）”构成。它应对了 **1990 年 年代后半至 2000 年 年代初的损保破产潮（[[financial-regulators/dai-ichi-mutual|第一火災海上]] 2000-05  / [[non-life-insurers/taisei-kasai|大成火災海上]] 2001-11  / [[non-life-insurers/taisho-kasai|大正海上]] 1996  等）**。当前会员约为国内损保 24 社 家规模。它与寿险版（[[financial-regulators/seimei-hokensha-hogo-kiko|生命保険契約者保護機構]]）并列，是两大保险安全网。

## 1. 法律依据 / 设立

**正式名称**：損害保険契約者保護機構
**英文名称**：Non-life Insurance Policyholders Protection Corporation of Japan
**法律依据**：**《保险业法》第 259  条以下**（1995 年 修订，1998-12  全面施行）
**设立**：**1998-12-01**
**总部**：东京都千代田区神田淡路町
**业态**：认可法人
**主管**：[[financial-regulators/fsa|金融庁]]、财务省（法令所管）

### 加入义务

- 日本国内经营的**所有损害保险公司**（不含寿险专营公司，包含外国损保日本分公司）均有加入义务。
- 大型 4 社（[[non-life-insurers/tokio-marine-hd|東京海上日動]] / [[non-life-insurers/sompo-japan|損保ジャパン]] / [[non-life-insurers/mitsui-sumitomo|三井住友海上]] / [[non-life-insurers/aioi-nissay-dowa-insurance|あいおいニッセイ同和]]）+ 中坚 + 外资系 + 共济系 = **约 24 社**（随时间变动）。

### 重要年表

| 年月 | 事件 |
|---|---|
| 1995-06  | 《保险业法》全面修订成立（旧法为 1939） |
| **1998-12** | **《保险业法》新法施行，损害保险契约者保护机构成立** |
| 2000-05  | **[[financial-regulators/dai-ichi-mutual|第一火災海上]] 破产** -> 机构首个正式救济案件 |
| 2001-11  | **[[non-life-insurers/taisei-kasai|大成火災海上]] 破产** -> 机构财政基础强化议论 |
| 2002-06  | 《保险业法》修订 -> 政府补助金特别措施的延长和扩充 |
| 2010-04  | 自然灾害频发阶段的负担金体系复核 |
| 2011-03  | 东日本大震灾 -> 地震保险再保险方案与相互补完讨论 |
| 2024 起 | 中坚损保经营整合讨论与机构财政稳定性 |

## 2. 业务机制

| 业务 | 内容 |
|---|---|
| **承接破产保险公司的合同** | 若没有救济保险公司出现，由机构自身（或通过承继公司）承接合同 |
| **向救济保险公司提供资金援助** | 救济保险公司承接破产保险公司时提供**金钱赠与** |
| **保险金支付补偿** | 对破产时点已发生的保险金请求权按**一定比例补偿** |
| **退保返还金等补偿** | 对退保返还金、满期返还金等按**一定比例补偿** |
| **征收负担金** | 每个事业年度向会员损保征收（按收入保费分摊等） |
| **资金借入** | 必要时从金融机构借款（附政府担保） |

### 补偿水平（主要险种）

| 保险种类 | 补偿水平 |
|---|---|
| **自赔责保险** | **100%**（因强制保险而最优先） |
| 地震保险 | **100%**（政府再保险机制下） |
| 汽车保险（任意） | 破产后 3  个月内发生事故为 100%，之后为 **80%** |
| 火灾保险（个人） | **80%**（破产后 3  个月内为 100%） |
| 赔偿责任保险 | 80% |
| 海上保险、运输保险、信用保险等（企业向） | **80%** |

补偿水平会随合同种类和破产时点变动。最新制度详情参见机构官方网站。

## 3. 资金结构

```
收入
  ├── 负担金（会员 24 社 缴纳，按收入保费分摊）
  ├── 借入金（必要时，附政府担保）
  └── 政府补助金（特别措施时，《保险业法》附则）

支出
  ├── 向救济保险公司提供资金援助
  ├── 承继保险公司运营费
  ├── 向投保人的直接补偿支付
  └── 机构运营费
```

### 负担金计算

- 以各损保的**净收入保费**为基准分摊。
- 大型 4  集团（[[non-life-insurers/tokio-marine-hd|東京海上]] / [[non-life-insurers/sompo-japan|損保ジャパン]] / [[non-life-insurers/mitsui-sumitomo|三井住友海上]] / [[non-life-insurers/aioi-nissay-dowa-insurance|あいおいニッセイ同和]]）承担过半。
- 机构原则上不持有很厚的常设积立资金，而是在破产时通过事后负担应对，结构与寿险版类似。

## 4. 过去救济案例

| 年月 | 破产损保 | 处理机制 |
|---|---|---|
| 1996  | 大正海上火灾 | 救济合并（由住友海上承继） |
| **2000-05** | **[[financial-regulators/dai-ichi-mutual|第一火災海上]]** | 机构首个正式救济（资金援助） |
| **2001-11** | **[[non-life-insurers/taisei-kasai|大成火災海上]]** | 机构参与；由損保ジャパン系（旧安田火灾）承继 |
| 2008  | 大和生命（因属寿险，故为 [[financial-regulators/seimei-hokensha-hogo-kiko|生保機構]] 案件） | — |

> 2000 年代中期以后，随着中坚损保经营整合（[[non-life-insurers/sompo-japan|損保ジャパン]] / [[non-life-insurers/aioi-nissay-dowa-insurance|あいおいニッセイ同和]] 形成过程等），破产案件大幅减少。

## 5. 国际比较 / 论点

- 接近美国 **NAIC** 指南下的 **State Guaranty Fund** 制度，但日本是单一全国机构（美国按州设置）。
- 英国 **FSCS（Financial Services Compensation Scheme）** 将银行和保险一体化，而日本则由存保、寿险机构、损保机构分立。
  - 气候相关风险（自然灾害频发和激烈化）下的负担金体系复核。
  - 随着网络保险、宠物保险等新型保险扩大，完善补偿机制。
  - 损保公司经营整合推进导致会员数减少及财政基础讨论。
