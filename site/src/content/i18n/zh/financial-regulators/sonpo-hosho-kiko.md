# 损害保险契约者保护机构

## Wiki route

本条目归属于 [[financial-regulators/INDEX|financial-regulators INDEX]]。请将其与 [[financial-regulators/seimei-hokensha-hogo-kiko|生命保険契約者保護機構]]（并行的安全网同业）以及 [[financial-regulators/dic|預金保険機構]] / [[financial-regulators/fsa|金融庁]]（更广阔的体系 / 监管边界）对照阅读。

## TL;DR

**在国内经营的全部损害保险公司均为会员、以保护契约者为目的的认可法人**。随 1998-12 修订保险业法施行而成立。承担**破产损保的契约延续 + 保险金支付 + 契约承接**的最后安全网。资金由会员出资的「负担金」与政府的「补助金（特例措施时）」构成。**应对了 1990 年年代后半至 2000 年年代初的损保破产潮（[[financial-regulators/dai-ichi-mutual|第一火災海上]] 2000-05 / [[non-life-insurers/taisei-kasai|大成火災海上]] 2001-11 / [[non-life-insurers/taisho-kasai|大正海上]] 1996 等）**。现有会员为国内损保 24 社规模。与寿险版（[[financial-regulators/seimei-hokensha-hogo-kiko|生命保険契約者保護機構]]）并列为两大保险安全网。

## 1. 法律依据·设立

**正式名称**: 损害保险契约者保护机构
**英文名**: Non-life Insurance Policyholders Protection Corporation of Japan
**法律依据**: **保险业法 第 259 条以下**（1995 年修订·1998-12 全面施行）
**设立**: **1998-12-01**
**总部**: 东京都千代田区神田淡路町
**业态**: 认可法人
**主管**: [[financial-regulators/fsa|金融庁]]·财务省（法令所辖）

### 加入义务

- 在国内经营的**全部损害保险公司（寿险专营除外、含外国损保的日本分支）负有加入义务**
- 大型 4 社（[[non-life-insurers/tokio-marine-hd|東京海上日動]] / [[non-life-insurers/sompo-japan|損保ジャパン]] / [[non-life-insurers/mitsui-sumitomo|三井住友海上]] / [[non-life-insurers/aioi-nissay-dowa-insurance|あいおいニッセイ同和]]）+ 中坚 + 外资系 + 共济系 = **约 24 社**（随时期变动）

### 重要年表

| 年月 | 事象 |
|---|---|
| 1995-06 | 保险业法全面修订成立（旧法为 1939）|
| **1998-12** | **保险业法新法施行·损害保险契约者保护机构成立** |
| 2000-05 | **[[financial-regulators/dai-ichi-mutual|第一火災海上]] 破产** → 机构首例正式救济案件 |
| 2001-11 | **[[non-life-insurers/taisei-kasai|大成火災海上]] 破产** → 机构财政基础强化议论 |
| 2002-06 | 保险业法修订 → 政府补助金特例措施的延长·扩充 |
| 2010-04 | 自然灾害多发局面下负担金体系的重新审视 |
| 2011-03 | 东日本大地震 → 地震保险再保险机制与相互补充的论议 |
| 2024〜 | 中坚损保的经营整合议论与机构财政的稳定性 |

## 2. 业务机制

| 业务 | 内容 |
|---|---|
| **破产保险公司的契约承接** | 在无救济保险公司出现的情况下，机构自身（或经由承继公司）承接契约 |
| **向救济保险公司提供资金援助** | 在救济保险公司承接破产保险公司时进行**金钱赠与** |
| **保险金支付的补偿** | 对破产时点已发生的保险金请求权**按一定比例补偿** |
| **解约返还金等的补偿** | 对解约返还金·满期返还金等**按一定比例补偿** |
| **征收负担金** | 按事业年度向会员损保征收（按收入保险料分摊等）|
| **借入资金** | 必要时向金融机构借入（附政府担保）|

### 补偿水平（主要险种）

| 保险险种 | 补偿水平 |
|---|---|
| **强制汽车责任险（自赔责）** | **100%**（因属强制保险故最优先）|
| 地震保险 | **100%**（在政府再保险机制下）|
| 汽车保险（任意）| 破产后 3 个月以内发生的事故为 100%，其后为 **80%** |
| 火灾保险（个人）| **80%**（破产后 3 个月以内为 100%）|
| 责任保险 | 80% |
| 海上保险·运送保险·信用保险等（企业向）| **80%** |

※ 补偿水平随契约种类·破产时期而变动。最新的制度详情请参阅机构官方网站。

## 3. 资金结构

```
收入
  ├── 负担金（会员 24 社出资、按收入保险料分摊）
  ├── 借入金（必要时、附政府担保）
  └── 政府补助金（特例措施时、保险业法附则）

支出
  ├── 向救济保险公司的资金援助
  ├── 承继保险公司运营费
  ├── 向契约者的直接补偿支付
  └── 机构运营费
```

### 负担金计算

- 以各损保的**正味收入保险料为基准**分摊
- 由大型 4 集团（[[non-life-insurers/tokio-marine-hd|東京海上]] / [[non-life-insurers/sompo-japan|損保ジャパン]] / [[non-life-insurers/mitsui-sumitomo|三井住友海上]] / [[non-life-insurers/aioi-nissay-dowa-insurance|あいおいニッセイ同和]]）出资过半数
- 机构基本上不厚备常设的积累资金，而以破产时的事后负担来应对的结构（与寿险版相同）

## 4. 过往救济案例

| 年月 | 破产损保 | 处理机制 |
|---|---|---|
| 1996 | 大正海上火灾 | 救济合并（住友海上承继）|
| **2000-05** | **[[financial-regulators/dai-ichi-mutual|第一火災海上]]** | 机构首例正式救济（资金援助）|
| **2001-11** | **[[non-life-insurers/taisei-kasai|大成火災海上]]** | 机构介入·损保 Japan 系（旧安田火灾）承继 |
| 2008 | 大和生命（因属寿险故为 [[financial-regulators/seimei-hokensha-hogo-kiko|生保機構]] 案件）| — |

> 2000年代中期以后，因中坚损保的经营整合（[[non-life-insurers/sompo-japan|損保ジャパン]] / [[non-life-insurers/aioi-nissay-dowa-insurance|あいおいニッセイ同和]] 形成过程等）破产案件大幅减少。

## 5. 国际比较·论点

- 与美国 **NAIC** 指引下的 **State Guaranty Fund** 制度相近，但日本为单一的全国性机构（美国为各州分设）
- 相对于英国 **FSCS（Financial Services Compensation Scheme）** 将银行与保险一体化，日本则是存保·寿险机构·损保机构分立
  - 气候相关风险（自然灾害频发·剧烈化）下负担金体系的重新审视
  - 应对网络保险·宠物保险等新型保险扩张的补偿机制建设
  - 因损保公司经营整合推进导致会员数减少与财政基础的论议
