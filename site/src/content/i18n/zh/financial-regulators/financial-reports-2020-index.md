# JapanFG 财务报告 2020+ 索引

## TL;DR

本页将 [[financial-regulators/INDEX|financial-regulators INDEX]] 中金融公司 / 金融机构的 **2020 年以降的官方财务报告、统合报告书、披露杂志（ディスクロージャー誌）、有价证券报告书、母公司年次报告书入口** 汇总成一个索引。

方针为 **archive-first**：优先记录能够覆盖 2020 年以降全部报告的官方 archive / backnumber 页面；仅当官方页面没有完整 archive 时，才记录直接 PDF 样本、母公司年次报告书、监管披露，或「公开年次报告书缺口」。

## Scope

- 对象：JapanFG 下的银行、FG、政策金融、信托、证券、保险、信贩、租赁、支付、fintech、加密资产、短资公司、外资在日法人 / 分行。
- 排除：跨公司页面 [[payment-firms/bnpl-landscape|bnpl-landscape]]、[[financial-regulators/boj-monetary-policy|boj-monetary-policy]]、[[card-issuers/installment-sales-act-2020-amendment|installment-sales-act-2020-amendment]]、[[financial-regulators/japan-policy-finance-map|japan-policy-finance-map]]、[[financial-regulators/jsda|jsda]]、[[financial-regulators/local-bond-market|local-bond-market]]，以及 deprecated 误链 [[regional-banks/ssjbank|ssjbank]]。
- 日期基准：2026-05-20 JST。2026-03 期 annual / integrated report 未公开时不强行补全。
- 「2020+」指 2020 年以降可追踪的年次报告书链接；对于 2020 年以降设立的公司，在表中明确标注前身公司 / 母公司边界。
