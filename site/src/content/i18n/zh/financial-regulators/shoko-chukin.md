# 商工中金 (Shoko Chukin Bank)

## Wiki 路径

本词条归属于 [[financial-regulators/INDEX|financial-regulators INDEX]]。请对照 [[megabanks/mufg|三菱 UFJ FG (MUFG)]] 阅读以获取同业 / 对比背景，并对照 [[banking/INDEX|banking index]] 了解更广泛的体系 / 监管边界。

## TL;DR

旧政策金融机构的 **改制为民间商业银行版**。1936 (昭和 11 年) 依《商工组合中央金库法》作为特殊公司设立，于 2008-10 改制为股份公司，**2023-10 政府持有股份出售完成 → 完全民营化**。专注于面向中小企业组合、组合员的融资 + 一般中小企业融资。因 2016-2017 **危机应对融资舞弊事件**（约 4,800 件规模）受到金融厅 2 次业务改善命令，经历经营层全面更替、治理改革后达至民营化完成。在与大型银行、地方银行、信用金库于中小企业市场的三方鏖战中，于规模处于下风之际摸索「贴近组合员 + 事业承继/M&A」差异化的民营化后阶段。^[extracted]

## 1. 公司概要

**正式名称**：株式会社商工組合中央金庫 ^[extracted]
**英文名**：The Shoko Chukin Bank, Ltd. ^[extracted]
**简称**：商工中金 / SCB
**设立**：1936-11-30（依《商工组合中央金库法》的特殊公司）^[extracted]
**改制为股份公司**：2008-10-01（《商工组合中央金库法》废止 + 《株式会社商工组合中央金库法》施行）^[extracted]
**完全民营化**：**2023-10 政府持有股份出售完成** ^[extracted]
**总店**：东京都中央区八重洲 2-10-17 ^[extracted]
**上市**：非上市（股东为中小企业组合等 + 民间）

### 主要股东（民营化后）

- 中小企业组合 + 组合员企业（历史上商工中金的出资母体）
- 民间金融机构 / 事业公司（2023-10 政府出售后的承接方）
- ※ 截至 2023-10 政府曾持有一定比例，但随出售完成转为 **政府持有 0%** ^[extracted]

### 重要年表 ^[extracted]

| 年月 | 事件 |
|---|---|
| 1936-11-30 | 《商工组合中央金库法》制定 → 作为特殊公司设立 |
| 战后〜1990s | 作为中小企业金融三大公共金融机构（商工中金、中小公库、国民公库）之一定位 |
| 2002-04 | 政策金融改革讨论全面展开 |
| **2008-10-01** | **改制为股份公司** — 《商工组合中央金库法》废止 + 新法施行、政策金融重组（与政策投资银行、日本政策金融公库等并行）|
| 2008-2016 | 受政府委托执行危机应对融资（雷曼后、震灾后、金融危机等）|
| **2016-10** | **危机应对融资舞弊事件曝光** — 以鹿儿岛分行为端绪的内部举报 |
| **2017-05-09** | **金融厅 第 1 次业务改善命令** — 舞弊处理 760 件确认时点 |
| 2017-10-25 | **金融厅 第 2 次业务改善命令** — 全件调查结果 **约 4,800 件、职员 444 名涉及** 查明 |
| 2017-10 | 安达健祐 社长（前经产事务次官）引咎辞职 |
| 2018-06 | 关根正裕 新社长就任（民间出身）、治理改革全面展开 |
| 2020-04 | 中期经营计划「价值共创阶段 I」 |
| 2022-04 | 中期经营计划「价值共创阶段 II」 — 着眼于民营化的事业模式转型 |
| **2023-10** | **政府持有股份出售完成 → 完全民营化** ^[extracted] |
| 2024-04 | 民营化后的新中期经营计划 — 扩大一般融资比例、强化事业承继/M&A |

## 2. 业务分部地图

| 分部 | 内容 | 定位 |
|---|---|---|
| **组合员融资** | 面向中小企业组合 + 组合员企业的融资 | 自创业以来的核心、历史性 identity |
| **一般中小企业融资** | 面向组合员以外中小企业的融资 | 民营化后扩大方针 |
| **事业承继 / M&A 支持** | 中小企业 M&A 中介、后继者支援 | 民营化后的差异化领域 |
| **存款 / 结算** | 组合员存款 + 一般法人结算 | 补充业务 |

### 民营化后的事业模式转型

- 由旧政策金融机构时代的 **政府委托模式** → 向 **自立的商业银行模式** 转移
- 在中小企业市场与大型银行（法人交易基础）/ 地方银行（地域贴近）/ 信用金库信用组合（微小企业贴近）三方鏖战

### 治理改革 (2017-2018)

- 经营层全面更替（社长更换 + 高管刷新）
- 内部控制、合规体制重建
- 危机应对融资处理程序严格化
- 业务改善计划 → 金融厅定期报告

### 与政府的关系

- 随 2023-10 政府持有股份出售，资本关系终止
- 完全民间化与「准公共角色」的双重结构

### 竞争格局

| 竞争对手 | 重叠领域 |
|---|---|
| 大型银行 ([[megabanks/mufg]] / [[megabanks/smfg]] / [[megabanks/mizuho-fg]]) | 中坚企业融资、海外进军支援 |
| 地方银行 | 地域中小企业融资、事业承继 |
| 信金中央金库、信用金库 | 中小企业组合、微小企业融资 |
| [[financial-regulators/jfc|日本政策金融公庫 (JFC)]] | 危机应对融资、政策金融领域 |

## 4. 监管 / 政策

- **主管**：金融厅 (FSA) — 作为一般银行依《银行法》进行监督
- **特殊性**：旧《商工组合中央金库法》废止后，作为危机应对融资的指定金融机构，**《株式会社商工组合中央金库法》** 仍继续适用 ^[extracted]
- **业务改善命令的经过**：
  - 2017-05-09 第 1 次业务改善命令 (760 件)
  - 2017-10-25 第 2 次业务改善命令（约 4,800 件）
  - 业务改善计划的持续报告义务
- **G-SIB / D-SIB 指定**：无（按规模在大型银行、大型地方银行以下）
