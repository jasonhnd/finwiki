# 缺失的日本金融机构扩展待办清单

## 概述

本页是将 [[financial-regulators/INDEX|financial-regulators INDEX]] 从一个主要机构 wiki 扩展为具备注册登记意识的日本金融机构地图的执行清单。

当前 wiki 已具备坚实的核心：超大型银行、主要 FG、地方银行集团、政策性金融、保险公司、证券集团、租赁、支付、金融科技、外资在日子公司，以及加密货币交易所页面。但相对于官方牌照 / 注册登记名单而言，它尚不详尽。

执行规则：

1. 为具有系统重要性的机构、上市金融公司、主要运营子公司、市场基础设施，以及拥有公开来源的金融科技 / 支付运营商创建独立页面。
2. 对于诸如信用金库、信用合作社、预付式发行人以及长尾 FIEA 运营商等规模庞大的群体，使用注册登记 / 索引页面。
3. 不得依据非公开语境创建公司页面。每个页面都必须可从公开注册登记、IR、披露、公司简介、年度报告、协会名单，或法律 / 政策文件中溯源。

## 推荐执行顺序

1. 批次 A：市场基础设施，因为它解锁证券、IPO、TDnet、结算、清算及 PTS 链接。
2. 批次 F：两条缺失的加密运营商及注册运营商完整性表，因为它规模小且能让加密面变得整洁。
3. 批次 B1：面向超大型银行 / 信托 / 互联网银行边界的银行运营公司。
4. 批次 J：注册登记 QA 快照，因为 A/F + B1  已新增超过 20  个页面，后续批次需要一份覆盖控制表。
5. 批次 C：证券与资产管理运营商；FIEA 注册登记、证券牌照栈、在线经纪、最优执行 / SOR / PTS、PTS 流动性数据、保证金交易 / 证券融资、承销及市场基础设施控制地图均已完成。
6. 批次 E：支付 / 卡 / 资金移动运营商。E1-E3  及批次 K 扩展已在重要页面 / 注册登记控制层面完成；仅为战略例外继续。
7. 批次 D：保险运营公司。D1-D9  已为寿险公司、主要 / 直营 / 专业财产险公司、宠物 / 户外 / 电信联动 / 旅行 / 租赁住房 / 外资关联专业保险公司及保险控股公司锚点完成；FSA 41  寿险公司面已在独立 / 有意保留集团锚点的层面收尾。请继续注册登记 / 索引或精度工作，而非重复寿险公司页面。
8. 批次 B2  + H：地方银行页面及合作社金融注册登记索引。B2.1-B2.8  及 H 信用金库 / 信用合作社 / 劳动金库注册登记路径已完成。
9. 批次 G + H + I：外资分行、合作社体系缺口及政策性金融相邻领域。批次 G 外资银行分行面已在全部 57 行路径层面通过 [[foreign-financial-institutions/foreign-bank-branches-japan-index]]、P2  战略例外页面、P3  走廊例外页面及 P4  仅注册登记收尾页面收尾。批次 H 合作社体系缺口经由 [[banking/ja-bank-system-japan]] 与 [[banking/jf-marine-bank-system-japan]] 路由，批次 I 公共信贷相邻领域经由 [[policy-finance/japan-housing-finance-agency]]、[[policy-finance/japan-credit-guarantee-system]]、[[policy-finance/national-federation-credit-guarantee-corporations]]、[[policy-finance/agriculture-credit-guarantee-system]] 及 [[policy-finance/fisheries-credit-guarantee-system]] 路由。未来的外资银行工作应为刷新 / 精度工作，而非原始名单完成。
