# 日本政策金融公库 (JFC)

## Wiki 动线

本项目位于 [[financial-regulators/INDEX|financial-regulators INDEX]] 之下。请与 [[financial-regulators/jbic|国際協力銀行 (JBIC)]] 一并阅读以了解同业 / 对比背景，与 [[banking/INDEX|banking index]] 一并了解更广泛的系统 / 监管边界。

## TL;DR

日本最大的政策性金融机构。**2008-10-01 由国民生活金融公库 + 中小企业金融公库 + 农林渔业金融公库 + 国际协力银行国际金融部门 4  家机构整合**而设立的特殊会社。财务 / 厚劳 / 农水 / 经产 4  大臣 100% 出资。以**4  个业务部门（国民生活・中小企业・农林水产・危机应对顺畅化）**，覆盖民营金融机构无法放贷之风险的「民营补充原则」为基轴。新冠特别贷款实行 20 兆円 以上（2020-2022）是近期最大的出场。创业支援 / 事业承继 / 海外拓展 / 安全网的 4  轴为现行战略。^[extracted]

## 1. 公司概要

**正式名称**：株式会社日本政策金融公库 ^[extracted]
**英文名**：Japan Finance Corporation (JFC)
**依据法**：株式会社日本政策金融公库法（平成 19 年 法律 第 57 号）^[extracted]
**设立**：2008-10-01（小泉政权下政策金融重组而进行的 4  家机构整合）^[extracted]
**形态**：特殊会社（虽为株式会社形态但非上市・全部股份由国家持有）^[extracted]
**总店**：东京都千代田区大手町 1-9-4（大手町金融城 北塔）^[extracted]
**出资者**：财务大臣 + 厚生劳动大臣 + 农林水产大臣 + 经济产业大臣（4  大臣共管・100% 政府出资）^[extracted]

### 4  家机构整合的经纬 ^[extracted]

| 整合前机构 | 设立 | JFC 内现业务部门 |
|---|---|---|
| 国民生活金融公库（国金）| 1949  → 1999  改组 | **国民生活事业** |
| 中小企业金融公库（中小公库）| 1953  | **中小企业事业** |
| 农林渔业金融公库（农林公库）| 1953  | **农林水产事业** |
| 国际协力银行（JBIC）国际金融部门 | 1999  整合 | 后于 2012-04  分离・再独立 |

### 治理・机构设计

- **代表董事总裁**：田中一穗（原财务事务次官，2018〜现任）^[likely]
- 设置董事会・监事会的会社
- 经营委员会（4  个业务部门代表 + 公司部门）
- 国会报告义务（决算・业务概况）


## 2. 业务分部・地图

| 业务部门 | 旧前身 | 主要对象 | 每 1 件 融资规模 | 主要菜单 |
|---|---|---|---|---|
| **国民生活事业** | 国民生活金融公库 | 个人工商户・小规模事业者・创业者 | 数百万〜数千万日元 | 普通贷款 / 新创业融资 / 马经融资（商工会议所推荐）/ 教育贷款 |
| **中小企业事业** | 中小企业金融公库 | 中坚・中小企业 | 数千万〜数十亿日元 | 普通贷款 / 新事业培育资金 / 海外拓展资金 / 事业承继・集约・活性化资金 |
| **农林水产事业** | 农林渔业金融公库 | 农业者・林业者・渔业者・食品产业 | 数百万〜数十亿日元 | 超级 L 资金 / 经营体培育强化资金 / 食品产业向资金 |

### 教育贷款（国民生活事业的独有菜单）

国民生活事业是**唯一**经办**面向个人的教育贷款**的业务部门，作为子女升学资金，每 1  人最高至 350 万円（在一定要件下 450 万円）的低息融资。与民营教育贷款・奖学金（[[policy-finance/japan-student-services-organization|japan-student-services-organization]]）形成补充关系。

### 民营补充原则（基本姿态）

JFC 的存在意义在于「**覆盖民营金融机构无法放贷之风险**」，为避免压迫民营事业，按以下规则运营：

- 优先创业期・无担保无保证人等民营踟蹰不前的群体
- 以与民营银行・信金的**协调融资**（专项 + JFC）为中心
- 原则上不以大企业为对象（按资本金・员工数设上限）
- 利率为顾及民营市场利率的政策利率体系

### 现行战略的 4  轴 ^[extracted]

1. **创业支援**：新创业融资制度（无担保无保证人・上限 3000 万円）、创业前后的事业计划支援
2. **事业承继**：遵循经营者保证指引的承继资金、M&A 资金
3. **海外拓展**：中小企业的海外进出资金、面向海外子公司的转贷资金
4. **安全网**：危机应对顺畅化业务、震灾・物价高涨・大流行应对

### 危机应对的出场 ^[extracted]

- **雷曼冲击**（2008-10〜）：紧急保证 + JFC 特别贷款
- **东日本大震灾**（2011-03〜）：复兴特别贷款（无息・无担保）
- **新冠**（2020-03〜2022）：**实质无息・无担保特别贷款实行 20 兆円 以上**（与民营零零融资并行）
- **能登半岛地震**（2024-01〜）：受灾事业者向特别贷款
- **物价高涨**（2023〜）：物价高涨应对资金

### 与民营金融机构的关系

- **协调融资的主角**：与地银・信金・信组的专项并用是国民生活事业・中小企业事业的标准形态
- **指定金融机构方案**：危机应对业务以民营银行为窗口，JFC 提供损害担保・两步贷款
- **非竞争而是补充**：与 [[megabanks/mufg]] / [[megabanks/smfg]] / [[megabanks/mizuho-fg]] 等大型银行亦各有分工（大型银行面向中坚以上，JFC 面向小规模・创业・农林渔业）

### 与 JBIC 的分离（2012-04）

2012-04-01，国际协力银行（JBIC）自 JFC 分离・再独立。理由是基于应将**海外基础设施出口・资源确保的战略性**与国内中小企业金融切离的政策判断。现在的 JFC 纯化为国内 4  个业务部门，海外金融职能由 JBIC（特殊会社・财务省所管）承担。

## 4. 监管・政策

- **主管 4  大臣**：财务大臣（公司整体・危机应对）/ 厚生劳动大臣（教育贷款・国民生活事业的一部分）/ 农林水产大臣（农林水产事业）/ 经济产业大臣（中小企业事业）^[extracted]
- **依据法**：株式会社日本政策金融公库法（平成 19 年 法律 第 57 号）
- **会计检查院的检查对象**：因政府全额出资而为会计检查院的常时检查对象
- **国会报告**：每年度向国会报告业务概况・决算
- **政府保证**：可通过发行政府保证债进行资金筹措（市场筹措 + 财投借入 + 政府保证债）

### 资金筹措结构

- **财政投融资**（财投资金）为主柱：来自财务省财政融资资金的借入
- **政府保证外债 / 内债**：面向机构投资者发行
- **财投机构债**（无政府保证）：以自身信用进行市场筹措

### 近期政策论点

- 2024〜 伴随新冠特别贷款**还款期到来**的中小企业违约・代位清偿担忧
- 2024〜 经营者保证指引的贯彻（无保证人融资的扩大）
- 2025〜 初创支援・GX（绿色）投融资作为政策金融的定位
- 2025〜 物价高涨・涨薪与中小企业资金周转的兼顾
