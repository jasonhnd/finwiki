# 上田八木短资株式会社 (Ueda Yagi Tanshi)

## Wiki 路线

本条目归于 [[financial-regulators/INDEX|financial-regulators INDEX]] 之下。请将其与 [[financial-regulators/central-tanshi|セントラル短資株式会社 (Central Tanshi)]] 对照阅读以获取同业 / 对比语境，并参阅 [[banking/INDEX|banking index]] 以了解更宏观的系统 / 监管边界。

## TL;DR

日本 3  大短资公司之一的独立系短资公司。由**2001-04  上田短资 + 八木短资 合并**而成立。以短期金融市场（同业拆借市场・票据买卖・CD/CP）的中介，以及作为日本银行操作（公开市场操作）的交易对手为核心业务。短资行业在 2009  雷曼危机后的超低利率环境中长期低迷，但随着**2024-03  负利率解除**，行业利润率改善的预期开始抬头。非上市・独立系，采用与大型银行・地方银行交易集中型的商业模式。^[extracted]

## 1. 公司概要

**正式名称**：上田八木短资株式会社
**英文名**：Ueda Yagi Tanshi Co., Ltd.
**设立**：2001-04（上田短资 + 八木短资 合并成立）^[extracted]
**业态**：短资公司（日本独有的短期金融市场专营业态）
**上市**：非上市（独立系）
**行业地位**：日本 3  大短资公司之一（[[financial-regulators/central-tanshi]] / 上田八木短资 / [[financial-regulators/tokyo-tanshi]]）

### 前身・合并历史

- **1909** 上田短资 创业（自战前延续至今的老字号短资商）
- **1928** 八木商店 设立（八木短资 前身）
- **2001-04** 上田短资 + 八木短资 → **上田八木短资 合并成立**^[extracted]
- 行业重组的象征：短资行业贯穿整个 2000 年 年代，持续向 3  大格局集中

## 2. 业务分部地图

| 业务 | 内容 | 客户 |
|---|---|---|
| **同业拆借市场中介** | 银行间无担保・有担保拆借资金的中介 | 大型银行・地方银行・信托・券商 |
| **票据买卖市场** | 短期票据（商业票据・银行承兑票据）买卖中介 | 银行・大企业 |
| **CD / CP 买卖** | 可转让存单（CD）・商业票据（CP）买卖 | 银行・事业法人・机构投资者 |
| **日银交易对手业务** | 日本银行市场操作（公开市场操作）的交易对手・中介 | 日本银行 |
| **国债・地方债买卖** | 短期国债（T-Bill）・中长期国债・地方债买卖 | 银行・机构投资者 |
| **海外债券（美元拆借等）** | 美元短期资金市场（欧洲美元拆借等）的中介 | 日资银行海外据点・外资银行东京分行 |

### 商业模式

- **薄利多销型**: 短期资金市场的买卖价差和手续费为收益来源
- **依赖日银政策利率**: 行业利润率随政策利率水平和操作量联动
- **大型银行・地方银行集中**: 客户基础以日本银行板块为中心
- **面向非银行**: 也向大型券商・信托・保险・农林系金融机构提供服务

### 3  大短资公司

| 短资公司 | 沿革 | 特点 |
|---|---|---|
| **Central 短资** | 山根短资 + Central 短资 + 短资 的合并 | 行业最大 |
| **上田八木短资** | 上田短资 + 八木短资（2001-04  合并）| 独立系・3  大之一 |
| **东京短资** | 自战前延续的东京据点短资 | 3  大之一 |

### 行业收缩周期

- **〜2008**: 有政策利率・市场活跃・即便在 3  大格局下也能确保收益
- **2009-2024**: 雷曼后超低利率 → 在零利率・负利率环境下利润变薄，行业收缩^[extracted]
- **2024-03〜**: 负利率解除 → 行业利润率改善预期

### 中期战略（2024  中计）

- **以日银政策利率正常化为契机**: 应对 2024-03  负利率解除后短期金融市场的活跃化
- **提供短期金融市场流动性**: 强化核心业务
- **与大型银行・地方银行交易集中**: 深耕核心客户

### 竞争关系

- **直接竞争**: [[financial-regulators/central-tanshi]] / [[financial-regulators/tokyo-tanshi]]（3  大短资相互竞争）
- **间接竞争**: 大型券商（短期国债 OTC 市场）・银行间直接交易（拆借直接成交）
- **行业保护因素**: 作为日银操作交易对手的制度性地位

### 市场趋势

- **2024-03  负利率解除**: 行业复苏预期的最大因素
- **日银 QT（量化紧缩）**: 国债市场流动性恢复，短资业务有扩张空间

## 5. 监管・政策

- **主管**: 金融厅（FSA）+ 日本银行（作为操作交易对手的监督）
- **业法**: 短资公司是银行法之外的独有业态（定位为短资业者）
- **日银操作交易对手要件**: 须满足一定的资本・体制标准
- **近期政策议题**:
  - 2024-03〜 日银政策利率正常化（负利率解除）
  - 2024-2026  日银 QT（缩减持有国债）
  - 2025-2026  短期金融市场的数字化・分布式账本应用的讨论
