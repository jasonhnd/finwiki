# 日本格付研究所 (JCR)

## Wiki 路由

本条目归属于 [[financial-regulators/INDEX|financial-regulators INDEX]]。请对照 [[financial-regulators/rating-and-investment|格付投資情報センター (R&I)]] 以了解同侪／对比脉络，并结合 [[financial-regulators/fsa|金融庁]] 了解更广泛的监管边界（信用评级机构登记）。

## 要点

日本资本市场 **2 社家本土系信用评级机构中的 1 社家**（另 1 社家为 [[financial-regulators/rating-and-investment|R&I]]）。1985-04 设立，标榜独立系（不拥有特定金融集团母公司）。在**主权／金融机构／事业公司／证券化产品／项目融资／可持续金融**全领域提供评级。已取得**依据《金融商品交易法》的信用评级机构登记**（金融厅长官登记），并作为 ECAI（外部信用评估机构）在日本标准与巴塞尔标准下合格。与全球 3 大（S&P／ Moody's／ Fitch）相比，**对日本企业的评级偏宽 1〜2 个子级**的倾向多年来一直被指出。

## 1. 法人结构・沿革

**正式名称**: 株式会社日本格付研究所
**英文名**: Japan Credit Rating Agency, Ltd. (JCR)
**设立**: **1985-04-01**
**总部**: 东京都千代田区神田锦町
**业态**: 信用评级机构（《金融商品交易法》第 66 条之 27 登记）
**主要股东**: 由国内金融机构・事业公司分散持有（独立系）
**主管**: 金融厅（信用评级机构监督）

### 重要年表

| 年月 | 事项 |
|---|---|
| **1985-04** | **设立**（应对泡沫期前公司债发行激增所带来的评级需求） |
| 1991  | 国内公司债评级全面展开 |
| 1998  | 《金融商品交易法》修订（信用评级机构登记制度施行于 2010） |
| 2007〜 | 次贷危机 → 国际上评级行业监管议论趋于活跃 |
| **2010-04** | **信用评级机构登记制度施行** → JCR 作为登记 第 1 号级登记（与 [[financial-regulators/rating-and-investment|R&I]] 并列） |
| 2013〜 | 结构化金融（RMBS／ ABS）评级透明度强化 |
| 2017〜 | 开展**绿色债券／社会债券／可持续发展挂钩债券**等 ESG 相关评估业务 |
| 2020〜 | 在新冠后公司债发行激增局面中评级件数增加 |

## 2. 评级业务 map

| 领域 | 内容 | 主要客户 |
|---|---|---|
| **主权评级** | 日本国债・地方债・政府关联机构债 | 国家／地方公共团体／政府关联机构 |
| **金融机构评级** | 银行・证券・保险・非银行的发行体评级 | [[megabanks/mufg|MUFG]] ／ [[megabanks/smfg|SMFG]] ／ [[megabanks/mizuho-fg|みずほ FG]] 等 |
| **事业公司评级** | 一般事业公司的公司债・长期发行体评级 | 以上市大企业为主 |
| **结构化金融** | RMBS／ ABS／ CMBS／ ABCP 等 | SPV／安排人 |
| **项目融资** | 基础设施・可再生能源・PPP 案件 | 项目 SPV |
| **保险财务实力评级** | 寿险・财险的偿付能力评估 | 寿险／财险 |
| **ESG／可持续性评估** | 绿色／社会／可持续性／转型债评估 | 发行体全体 |
| **第三方意见书** | 关于框架合规性的 SPO | 发行体 |

### 评级量表

- **长期评级**: AAA → AA+／ AA／ AA- → A+／ A／ A- → BBB+／ BBB／ BBB- → BB 以下（投机级）→ D（违约）
- **短期评级**: J-1+／ J-1 ／ J-2 ／ J-3 ／ NJ／ D
- 国际量表（面向外币计价发行体）亦另行提供

### 国内 4 大评级机构

| 评级机构 | 特征 | 母公司・系列 |
|---|---|---|
| **JCR** | 独立系・本土系 | 独立系（无特定母公司） |
| **R&I** | 瑞穗系（原 IBJ 系＋日经系） | 瑞穗系（[[megabanks/mizuho-fg|みずほ FG]] 关联） |
| **S&P Global Ratings Japan** | 美国 S&P 子公司 | S&P Global |
| **Moody's Japan／ Moody's SF Japan** | 美国 Moody's 子公司 | Moody's Corp |

### 国内 vs 国际的评级差距问题

- 实证研究确认，对日本企业的 **JCR／ R&I 评级高于 S&P／ Moody's** 的倾向（典型情形下高 1〜2 个子级，部分案件中亦有报告称差距达 4 个子级以上）—— 日本央行工作论文「Credit Rating Gaps in Japan」(2007) 等同行评审论文
- 背景: 将日本企业的主银行制度・集团救济惯例纳入考量的相对评估方法（实证研究将其解释为信用风险评估方法的体系性差异，而非「上方偏差」）
- 国际投资者持续批评「日本评级存在乐观偏差」，另一方面也有反驳称其反映了国内市场的实际情况
- **多数日本企业从 2 社级以上取得评级**（国内・国际并用为常规做法）

### 竞争上的特征

- 标榜**独立系**（相对于 [[financial-regulators/rating-and-investment|R&I]] 被视为「瑞穗系」，本机构以不拥有特定金融集团母公司这一点形成差异化）
- 中坚・中小企业的发行体评级份额相对较高
- 在可持续性评估业务方面领先（提供遵循 GBP／ SBP／ SLBP 各原则的 SPO）

## 4. 监督・监管

- **主管**: [[financial-regulators/fsa|金融庁]]（信用评级机构监督）
- **依据法**: 《金融商品交易法》第 66 条之 27 及以下
- **登记要件**: 业务管理体制・防止利益冲突・评级流程文档化・投诉处理体制等
- **ECAI 认定**: 基于巴塞尔银行监督委员会标准的外部信用评估机构认定（金融厅告示）→ 可用于银行的风险资产计算
- **国际协调**: 遵循 IOSCO 信用评级机构行为准则
