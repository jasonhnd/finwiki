# 格付投資情報センター（R&I）

## Wiki route

本条目位于 [[financial-regulators/INDEX|financial-regulators INDEX]]。可与 [[financial-regulators/jcr|日本格付研究所 (JCR)]] 对照阅读，以理解同业和差异；更广泛的监管边界见 [[financial-regulators/fsa|金融庁]]。

## TL;DR

株式会社格付投資情報センター（R&I）是日本国内信用评级机构的 2  大玩家之一；另一个 1 社 是 [[financial-regulators/jcr|JCR]]。R&I 于 1998-04  由日本公社债研究所（[[megabanks/mizuho-fg|旧 IBJ・日本興業銀行系]] 系谱）和日本投资者服务（日本经济新闻社系）合并成立。它被定位为与瑞穗金融集团有关联的公司，[[megabanks/mizuho-fg|みずほ FG]] 是主要股东。业务覆盖主权、金融机构、事业公司、结构化融资、投信评价等。并行业务 **R&I Fund Information** 是其与 JCR 的重要差异点。它也是可持续金融第三方意见书（SPO）市场中的主要参与者。

## 1. 法人结构 / 沿革

**正式名称**：株式会社格付投資情報センター  
**英文名称**：Rating and Investment Information, Inc. (R&I)  
**设立**：1998-04-01，由日本公社债研究所和日本投资者服务合并而成  
**总部**：东京都千代田区大手町  
**业务类型**：信用评级业者，依据《金融商品交易法》第 66  条之 27  登记  
**主要股东**：[[megabanks/mizuho-fg|みずほフィナンシャルグループ]] 相关股东为核心，具有瑞穗系谱  
**主管**：[[financial-regulators/fsa|金融庁]]，负责信用评级业者监督

### 前身

```
1979       日本公社债研究所设立（日本兴业银行系，即后来的 [[megabanks/mizuho-fg|みずほ FG]] 源流）
1985       日本投资者服务设立（日本经济新闻社系）
1998-04    两者合并，株式会社格付投資情報センター (R&I) 成立
```

## 2. 业务地图

| 领域 | 内容 |
|---|---|
| **发行体评级** | 主权、金融机构、事业公司、政府相关机构 |
| **结构化融资** | RMBS、ABS、CMBS、ABCP、项目融资 |
| **保险财务力评级** | 寿险 / 损保 |
| **投信评价 / R&I Fund Information** | 日本国内最大级别的投信评价服务之一 |
| **ESG / 可持续性评价** | 绿色、社会、可持续挂钩债券 SPO |
| **第三方意见书** | 对框架一致性的意见 |
| **评级相关信息服务** | 数据分发和分析报告 |

### 评级尺度

- **长期评级**：AAA 到 AA+ / AA / AA-，A+ / A / A-，BBB+ / BBB / BBB-，BB 以下至 D。
- **短期评级**：a-1+ / a-1  / a-2  / a-3  / b / c / d。
- 保险财务力评价使用单独尺度，从 AA 到 D。

### vs 全球 3  大评级机构：S&P / Moody's / Fitch

- 在日本国内市场，R&I + JCR 的 2 社 评级组合常见；国际发行体通常还会取得 S&P 或 Moody's。
- R&I 被认定为 ECAI（外部信用评价机构），可在 Basel 框架下用于银行风险资产计算。

### 投信评价业务

- **R&I Fund Information** 是日本国内投信评价最大级别的参与者之一。
- 提供基金定量和定性评价、星级评级、benchmark 评价等。
- 金融机构销售现场会经常参考 R&I 投信评价。
- 竞争者包括 Morningstar、Mitsubishi Asset Brains 等评价机构。

## 4. 监督 / 监管

- **主管**：[[financial-regulators/fsa|金融庁]]。
- **法律依据**：《金融商品交易法》第 66  条之 27  以下。
- **登记**：信用评级业者登记；由于具有瑞穗系谱，利益冲突管理是重要监督点。
- **ECAI 认定**：在 Basel 框架下合格。
- **符合 IOSCO 行为规范**。

### 利益冲突管理

- 由于 R&I 属于瑞穗系谱，在对瑞穗 FG 集团公司评级时如何确保独立性，是持续的监督重点。
- R&I 通过内部规则规定人员、信息、收益和信用判断流程的防火墙与分离。
