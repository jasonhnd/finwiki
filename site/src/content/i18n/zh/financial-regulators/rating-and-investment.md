# 格付投资信息中心 (R&I)

## Wiki 路径

本条目归属于 [[financial-regulators/INDEX|financial-regulators INDEX]]。请与 [[financial-regulators/jcr|日本格付研究所 (JCR)]] 对照阅读以获取同侪 / 对比脉络，并参阅 [[financial-regulators/fsa|金融庁]] 了解更广泛的监管边界。

## TL;DR

日本本土系信用评级机构 **2 大主要参与者之一**（另 1 社 家为 [[financial-regulators/jcr|JCR]]）。于 1998-04  由 **日本公社债研究所**（[[megabanks/mizuho-fg|旧 IBJ・日本興業銀行系]]）与 **日本投资者服务**（日经系）合并成立。作为 **瑞穗金融集团关联公司** 定位（[[megabanks/mizuho-fg|みずほ FG]] 为主要股东）。开展主权 / 金融机构 / 事业公司 / 结构化金融 / 投资信托评价 等业务。**兼营投信评价业务（R&I Fund Information）** 是与 JCR 的重大差异化因素。在可持续金融第三方意见书（SPO）市场也是主要参与者。

## 1. 法人结构・沿革

**正式名称**：株式会社格付投资信息中心
**英文名**：Rating and Investment Information, Inc. (R&I)
**设立**：**1998-04-01**（日本公社债研究所 + 日本投资者服务 合并）
**总部**：东京都千代田区大手町
**业态**：信用评级业者（金融商品交易法 第 66 条之 27 登记）
**主要股东**：[[megabanks/mizuho-fg|みずほフィナンシャルグループ]] 关联为核心（瑞穗系）
**主管**：[[financial-regulators/fsa|金融庁]]（信用评级业者监督）

### 沿革・前身

```
1979      日本公社债研究所 设立（日本兴业银行系 = 后来的 [[megabanks/mizuho-fg|みずほ FG]] 源流）
1985      日本投资者服务 设立（日本经济新闻社系）
1998-04   两者合并 → 株式会社格付投资信息中心 (R&I) 成立
```

### 重要年表

| 年月 | 事件 |
|---|---|
| 1979 | 日本公社债研究所 设立（IBJ 系 = 瑞穗系源流） |
| 1985 | 日本投资者服务 设立（日经系） |
| **1998-04** | **两者合并 → R&I 成立** |
| 2000-04 | 瑞穗控股成立 → R&I 的系列关系明确化 |
| **2010-04** | **信用评级业者登记**（伴随金融商品交易法登记制度施行） |
| 2013〜 | 强化结构化金融评级的透明度 |
| 2017〜 | **R&I 绿色债券评估** 等 ESG 评价业务正式化 |
| 2020〜 | 疫情局面下公司债发行激增 → 评级件数增加 |

## 2. 业务 map

| 领域 | 内容 |
|---|---|
| **发行体评级** | 主权 / 金融机构 / 事业公司 / 政府关联机构 |
| **结构化金融** | RMBS / ABS / CMBS / ABCP / PF |
| **保险财务实力评级** | 寿险 / 财险 |
| **投信评价（R&I Fund Information）** | **国内最大级的投信评价服务** |
| **ESG / 可持续发展评价** | 绿色 / 社会 / 可持续发展挂钩 债券 SPO |
| **第三方意见书** | 框架合规性意见书 |
| **评级相关信息服务** | 数据分发・分析报告 |

### 评级量表

- **长期评级**：AAA → AA+ / AA / AA- → A+ / A / A- → BBB+ / BBB / BBB- → BB 以下 → D
- **短期评级**：a-1+ / a-1 / a-2 / a-3 / b / c / d
- 保险财务实力评价为另一量表（AAA → ... → D）

### vs JCR

| 项目 | R&I | [[financial-regulators/jcr|JCR]] |
|---|---|---|
| 设立年 | 1998（前身为 1979）| 1985 |
| 系列 | **瑞穗系** | 独立系 |
| 投信评价 | **兼营 R&I Fund Information** | 无 |
| 发行体评级件数 | 数千件规模 | 数千件规模 |
| 结构化金融 | 强 | 强 |
| ESG 评价 | 绿色债券评估 | 可持续发展评价业务 |

### vs 全球 3 大（S&P / Moody's / Fitch）

- 在国内市场，R&I + JCR 的 2 社评级为标准，国际发行体则另行取得 S&P 或 Moody's 为惯例
- 已获 ECAI（外部信用评价机构）认定 → 可在巴塞尔标准下用于银行风险资产计算

### 投信评价业务

- **R&I Fund Information** 是国内投资信托评价中最大级的参与者
- 基金的定量・定性评价、星级评级、基准评价等
- 在金融机构的销售现场，R&I 投信评价被广泛参照
- 竞争对手：晨星・三菱资产・Brains 等

## 4. 监督・监管

- **主管**：[[financial-regulators/fsa|金融庁]]
- **依据法**：金融商品交易法 第 66 条之 27 以下
- **登记**：信用评级业者登记（围绕属于瑞穗系的利益冲突管理是重要监督要点）
- **ECAI 认定**：在巴塞尔标准下适格
- **遵循 IOSCO 行为准则**

### 利益冲突管理的论点

- 既然属于瑞穗系，**在对瑞穗 FG 集团公司的评级中确保独立性** 是持续的监督要点
- R&I 在内部规章中规定了防火墙（人事・信息・收益・授信判断的分离）
