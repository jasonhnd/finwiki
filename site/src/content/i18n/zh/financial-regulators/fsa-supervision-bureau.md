# 監督局 (FSA Supervision Bureau)

## Wiki 路径
本条目位于 [[financial-regulators/INDEX|financial-regulators INDEX]] 之下。请与 [[financial-regulators/fsa-planning-coordination-bureau|FSA 企画市場局]]（FSA 内部的规则制定对应部门）、[[financial-regulators/fsa-strategy-bureau|FSA 総合政策局]]（战略 / 跨局层面）以及 [[financial-regulators/boj-financial-system-dept|BoJ 金融機構局]]（中央银行侧的审慎监管锚点）对照阅读。关于牌照范围，请通过 [[financial-licenses/INDEX|legal / financial licenses]] 与 [[banking/INDEX|banking landscape]] 锚定。

## TL;DR

監督局（Supervision Bureau）是金融厅（FSA）内部负责对日本持牌金融服务经营者进行持续监督与检查的部门——涵盖银行、证券公司、保险公司、FIEA 经营者、资产管理公司以及金融科技企业。自 2018 年 FSA 机构改组以来，原先由独立的検査局承担的现场检查职能，已被并入本局，成为一体化的监督与检查职能。

对 JapanFG 而言，每当某个页面提及 FSA 的监管行动、业务改善命令、现场检查，或针对整个持牌领域的分行业监管政策时，本局就是恰当的锚点。

## 1. 职能 / 范围

FSA 的英文组织架构图将監督局确定为负责以下事项的部门：

- 对依据银行法、金融商品交易法（FIEA）、保险业法、信托业法、贷金业法、资金结算法及相邻制度而注册或持牌的金融服务经营者进行持续监督。
- 对上述同一批机构进行现场检查（検査）——该职能在 2018 年改组中并入本局，当时原先独立的 [[financial-regulators/fsa-inspection-bureau|検査局]] 被吸收。
- 发布行政处分：业务改善命令、业务停止命令、注册撤销以及劝告。关于 VASP 领域分行业的处分轨迹，参见 [[exchanges/fsa-business-improvement-orders-history|FSA business-improvement orders history]]。
- 设有面向主要银行集团、地方银行、合作型机构（信用金库 / 信用合作社）、保险、证券与资产管理公司，以及创新 / 金融科技企业的分行业处室。
- 与地方财务局（其对规模较小的注册经营者执行受委托的监督）以及针对跨境集团的外国监管机构进行协调。

本局是 FSA 执法的主要公开门面，也是大多数已公布的、针对具名机构的行政处分的来源。

## 2. 交易对手 / 互动

由于持牌领域范围广泛，交易对手的互动也十分广泛：

- **银行**: 从巨型银行（[[megabanks/smfg|SMFG]]、[[securities-firms/mufg-mums|MUFG / MUMS]]）和信托银行，到地方银行（如 [[regional-banks/chiba-bank|Chiba Bank]]、[[regional-banks/77-bank|77 Bank]]）、信用金库与信用合作社（[[banking/shinkin-bank-registry-japan|shinkin registry]]、[[banking/credit-cooperative-registry-japan|credit cooperative registry]]），以及外国银行分行。
- **证券与资产管理**: 第一类与第二类金融商品交易业者（FIBO）、投资顾问 / 投资信托经营者、包括 [[asset-managers/asset-management-one|Asset Management One]] 在内的资产管理公司，以及 PTS 经营者。
- **保险**: 寿险与非寿险公司、外国保险分支机构，以及少额短期保险业者；参见诸如 [[life-insurers/aflac-japan|Aflac Japan]] 与 [[non-life-insurers/aioi-nissay-dowa-insurance|Aioi Nissay Dowa]] 等条目。
- **支付 / 金融科技**: 资金移动服务提供者、预付式发行者，以及在资金结算法（注册侧）下日益增多的加密资产 / 稳定币领域；参见 [[exchanges/fsa-vasp-registration-system|FSA VASP registration system]]。
- **其他监管机构**: 日本银行（[[financial-regulators/boj-financial-system-dept|金融機構局]]）用于审慎监管的重叠；[[financial-regulators/jsda|JSDA]] 用于证券公司的自律协调；[[financial-regulators/fsa-strategy-bureau|総合政策局]] 用于跨局战略；SESC 用于市场监察执法。

## 3. 为何本页重要

- 它是关于「如今由谁来执掌 FSA 监督」最清晰的公开锚点——每当某个 JapanFG 页面提及 FSA 的行动、现场检查或分行业监管者时都会用到。
- 它记录了那次吸收检查职能的 2018 年改组，从而避免将对検査局的历史性引用与当前的安排相混淆。
- 它把监督（本局）与规则制定（[[financial-regulators/fsa-planning-coordination-bureau|企画市場局]]）以及 FSA 战略（[[financial-regulators/fsa-strategy-bureau|総合政策局]]）区分开来——这是 JapanFG 页面经常需要厘清的三条不同脉络。
- 它为诸如 [[exchanges/fsa-business-improvement-orders-history|FSA business-improvement orders history]] 这样的业务改善命令页面以及 [[exchanges/fsa-foreign-exchange-warning-system|FSA foreign-exchange warning system]] 提供恰当的锚点。
