# CCI 集团 (CCI Group)

## Wiki 路径

CCI Group 是对应旧 [[regional-banks/hokkoku-fhd|Hokkoku Financial Holdings]] 路径的、采用现行名称的银行控股公司页面。可与 [[regional-banks/hokkoku-bank|Hokkoku Bank]]、[[banking/regional-bank-api-digital-partnership-route|regional bank API / digital partnership]]、[[fintech/jp-stablecoin-dcjpy|deposit-token / digital-money routes]] 和 [[banking/INDEX|banking domain]] 一同阅读。

本页用于了解控股公司战略、现行法人名称、集团扩张、数字 / 跨境 / 咨询业务，以及 FSA 银行控股公司边界。运营银行本身请用 [[regional-banks/hokkoku-bank|Hokkoku Bank]]。

## 概览

CCI Group 是围绕北国银行构建的现行上市银行控股公司。公开公司简介称，该集团于 2021-10-01 设立为 北國フィナンシャルホールディングス，并于 2025-10-01 更名为 株式会社CCIグループ。

本页之所以重要，是因为这次更名不仅仅是表面性的。官方简介描述了一个银行控股公司的宗旨，而集团本身则将自己定位于金融、咨询、BPO、系统开发、资产管理、地区振兴和海外业务之间。这使 CCI Group 成为一个采用现行名称的地区金融平台，而不仅仅是一个「旧北国 FHD」页面。

## 公开语境

CCI Group 是地方银行控股公司试图超越传统存贷款而拓宽的较为明确的例子之一。公开简介将集团价值与银行、咨询、投资、系统、地区振兴和海外职能联系起来。

这符合 [[banking/regional-bank-api-digital-partnership-route|regional bank API / digital partnership]] 中所捕捉的更广泛的地方银行打法。地方银行可以在保持本地根基的同时，尝试将技术、咨询、支付和跨境支持货币化。

围绕 Microsoft / 开放系统 / 数字地方银行领导力的较旧的 [[regional-banks/hokkoku-fhd|Hokkoku FHD]] 叙事仍然相关，但现行的 CCI 名称更能捕捉集团重塑后的平台抱负。

## 监管 / 研究备注

- 当现行法人名称边界重要时，使用 FSA 银行控股公司一览。
- 对于 2025  更名和集团业务类别，使用公司简介 / IR 页面。
- 将 [[regional-banks/hokkoku-bank|Hokkoku Bank]] 与控股公司分开；运营银行位于 [[financial-regulators/zenginkyo|全国銀行協会 (Zenginkyō)]] 所映射的行业团体层之内。
- 在分析存款代币或数字货币主张时，路由至 [[fintech/jp-stablecoin-dcjpy|tokenized deposit / DCJPY context]] 和公开公司披露，而非从品牌推断产品范围。
- 在分析地区金融转型时，与 [[regional-banks/fukuoka-fg|Fukuoka FG]]、[[regional-banks/shizuoka-fg|Shizuoka FG]] 和 [[regional-banks/kyushu-fg|Kyushu FG]] 比较。

## 注意事项

- 公开的集团类别并不意味着每项业务都具有实质性的盈利规模。
- 更宽泛的集团品牌可能使银行、咨询、资产管理和地区开发的边界变得模糊；在作出主张之前先确定运营实体。
- 本页不构成投资建议。
