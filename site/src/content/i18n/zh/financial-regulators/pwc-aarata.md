# PwC Japan 有限责任审计法人 (旧 PwC あらた)

## TL;DR

**PwC Japan 有限责任审计法人**（旧称：**PwC あらた有限责任审计法人**）是 **Big 4 系列** 的一角，为全球 **PricewaterhouseCoopers（PwC）** 的日本成员所。其出身构成了日本审计业重组史的核心——**2006-07 中央青山审计法人 业务停止处分（钟纺粉饰事件）→ あらた审计法人 设立** ^[extracted]。在 Big 4 之中，其上市审计客户数较其他 3 家法人为小，但凭借 **FinTech / Web3 / NFT 战略咨询联动** 与 **PwC 全球 "Time for trust" 加密资产框架** 实现差异化 ^[extracted]。于 **2023-04** 整合至 PwC Japan 品牌（与旧 京都审计法人合流）^[extracted]。

## 2. 沿革

| 年月 | 事件 |
|---|---|
| **2005-09** | **钟纺粉饰事件 — 中央青山审计法人 涉案公认会计士的刑事责任问题** ★ ^[extracted] |
| **2006-07** | **金融厅对中央青山审计法人作出业务停止 2 个月处分** → 审计客户大量流失 ^[extracted] |
| **2006-09** | **あらた审计法人 设立**（作为 PwC 全球日本成员新设，自中央青山分离）^[extracted] |
| 2006-09 | 旧 中央青山改称为 **みすず审计法人** → 于 2007 走向解散 ^[extracted] |
| **2008-07** | **公认会计士法修订实现有限责任化** → PwC あらた有限责任审计法人 ^[extracted] |
| 2010s | 在 FinTech / Web3 领域咨询先行 ^[extracted] |
| **2023-04** | **整合至 PwC Japan 品牌**（あらた + 京都审计法人合流的经过）^[extracted] |

### 过往处分记录（含中央青山时代）★ ^[extracted]

- **2006-07 中央青山审计法人 业务停止 2 个月处分**（以钟纺事件为起点）
