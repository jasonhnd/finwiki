# 国际协力银行 (JBIC)

## Wiki 路径

本条目归属于 [[financial-regulators/INDEX|financial-regulators INDEX]]。请与 [[financial-regulators/jfc|日本政策金融公庫 (JFC)]] 对照阅读以获取同侪 / 对比脉络，并参阅 [[banking/INDEX|banking index]] 了解更广泛的体系 / 监管边界。

## TL;DR

日本面向海外的 **政策性金融机构**（财务大臣 100% 股东，特殊公司）。以 **出口金融 + 海外投资金融 + 非束缚性贷款（untied loan）+ 出资** 的 4  项业务，支援日本企业的全球拓展、资源能源确保与 GX。2012-04-01 从 [[financial-regulators/jfc]]（日本政策金融公库）再度独立并转为特殊公司。前身为 1950  设立的日本输出银行（→ 日本输出入银行）与 1961  设立的海外经济协力基金（OECF）。1999-10  两者合并成立旧 JBIC，2008-10  一度并入 JFC → 2012-04  再度独立。与民间大型银行（[[megabanks/mufg]] / [[megabanks/smfg]] / [[megabanks/mizuho-fg]]）的 **协调融资为核心方案**，与 [[policy-finance/jica|jica]] 以「JBIC = 出口/投资、JICA = ODA」分工。^[extracted]

## 1. 公司概要

**正式名称**：株式会社国际协力银行 ^[extracted]
**英文名**：Japan Bank for International Cooperation (JBIC)
**法律依据**：株式会社国际协力银行法（平成 23 年法律第 第 39 号号）^[extracted]
**设立**：**2012-04-01**（日本政策金融公库的国际金融部门分离・转为特殊公司）^[extracted]
**股东**：**财务大臣 100%** ^[extracted]
**总部**：东京都千代田区大手町 1-4-1 ^[extracted]
**业态**：特殊公司（股份公司形态的政策性金融机构，非上市）

### 沿革・前身

```
1950-12  日本输出银行 设立（战后复兴的出口融资）
1953-04  日本输出入银行 改称（增加进口金融）
1961-03  海外经济协力基金 (OECF) 设立（日元贷款・ODA 核心）
1999-10  日本输出入银行 + OECF → 「国际协力银行 (旧 JBIC)」成立
2008-10  日本政策金融公库 (JFC) 合并 → JBIC 成为 JFC 的国际金融部门
2012-04  从 JFC 再度独立 → 「株式会社国际协力银行 (新 JBIC)」成立・转为特殊公司
```

## 2. 业务板块・地图

| 业务 | 内容 | 主要对象 |
|---|---|---|
| **出口金融** | 面向日本企业出口案件的融资（成套设备・船舶・基建・防卫装备等）| 日本出口商 / 海外买家 |
| **进口金融** | 能源・资源（LNG / 石油 / 稀有金属 等）进口确保融资 | 日本商社・电力・燃气 |
| **海外投资金融** | 日本企业海外 M&A・当地法人融资・合资出资资金 | 日本企业的海外子公司 |
| **非束缚性贷款** | 产业发展支援融资（不与特定日本企业案件挂钩，面向中等收入国）| 中等收入国政府 / 公共机构 |
| **出资** | 对战略案件的少数出资（风险资金供给）| 基建 / 资源 / GX 案件 |
| **担保** | 对民间金融机构融资・债券提供担保 | 民间银行 / 机构投资者 |

### 业务方案

- **协调融资（cofinancing）为核心**：大型银行（[[megabanks/mufg]] / [[megabanks/smfg]] / [[megabanks/mizuho-fg]]）+ 地方银行 + 海外金融机构 与 JBIC 的组合为典型
- **特别业务账户**（2016〜）：面向高风险案件・GX 案件的风险资金供给额度
- **与日元贷款的分工**：ODA 日元贷款由 [[policy-finance/jica|jica]] 负责（2008-10 移交）→ JBIC 专注于具商业性的海外投融资

### 3 战略轴

1. **支援日本企业全球拓展**：M&A 资金 / 成套设备出口 / 基建出口
2. **资源・能源确保**：LNG / 石油 / 稀有金属 / 蓄电池材料
3. **GX（绿色转型）**：可再生能源 / 氢 / 氨 / CCS / EV 供应链 ^[extracted]

### 合作・收购战略

- **大型银行协调融资**：与 [[megabanks/mufg]] / [[megabanks/smfg]] / [[megabanks/mizuho-fg]] 的恒常性协调（成套设备出口案件等）
- **与 JICA 的分工**：ODA = JICA，具商业性的海外投融资 = JBIC（2008-10 円贷款部门移交以后）
- **NEXI 联动**：日本贸易保险（NEXI）的出口保险与 JBIC 融资的组合
- **海外政策性金融机构联动**：与美 EXIM Bank / 韩 K-EXIM / 法 Bpifrance / 德 KfW IPEX 等的协调融资
- **国际能源框架**：与 JOGMEC・经产省联动的资源外交支援

### 与民间金融机构的关系

- 「**补充民间业务**」原则：补充仅靠民间金融机构难以承担风险的案件（发展中国 sovereign 风险 / 长期项目融资 / 具政治风险的案件 等）

### 经济安全保障转向（2022〜）

- **美中对立下的经济安全保障**：半导体 / 重要矿物 / 蓄电池 / 生物 等供应链强韧化案件
- **friend-shoring 支援**：在金融层面支援日本企业向同盟国・友好国转移

## 4. 监管・政策

- **主管**：财务省（股东・主管大臣）
- **法律依据**：株式会社国际协力银行法（平成 23 年法律第 第 39 号号）^[extracted]
- **业务范围**：该法列举了出口金融 / 进口金融 / 海外投资金融 / 非束缚性贷款 / 出资 / 担保 等
- **环境社会考量指引**（JBIC 指引）：依据 OECD 共同方针・赤道原则的环境社会影响评估框架
- **OECD 出口信贷安排**：遵循公共出口信贷的国际规则（利率下限・偿还期限上限）
  - 2022〜 与经济安全保障推进法联动
  - 2023〜 与 GX 推进法 / GX 经济转型债 联动（转型融资）
  - 2024〜 应对美国 IRA / 欧洲 CBAM・支援对美欧投资案件

### 相关机构生态

```
财务省 ──── 主管 ──── JBIC（海外投融资 / 出口金融）
                  │
外务省 ──── 主管 ──── JICA（ODA / 日元贷款 / 技术合作）
                  │
经产省 ──── 主管 ──── NEXI（贸易保险）
                  │
              JOGMEC（资源开发・出资）
```
