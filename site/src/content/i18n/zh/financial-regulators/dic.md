# 存款保险机构 (DIC)

## Wiki 路径

本条目位于 [[financial-regulators/INDEX|financial-regulators INDEX]] 之下。请将其与 [[financial-regulators/sonpo-hosho-kiko|損害保険契約者保護機構]] 及 [[financial-regulators/seimei-hokensha-hogo-kiko|生命保険契約者保護機構]] 对照阅读以了解并行的安全网同侪，并与 [[financial-regulators/fsa|金融庁]] / [[financial-regulators/boj|日本銀行]] 对照阅读以了解更广泛的系统／监管边界。

## TL;DR

日本**承担存款保险与破产处理的唯一特殊法人**。依据 1971-07 《存款保险法》设立。**保险金支付（pay-off）** + **资金援助（financial assistance）** + **过桥银行** + **特定回收困难债权收购** + **破产金融机构的经营管理与资产收购**，一手包办。出资方为**政府 + 日本银行 + 民间金融机构三方出资**（各占 1/3 相当）。理事长由财务大臣、金融厅长官、日本银行总裁协议任命。在 2003 [[megabanks/resona-hd|りそな]]、2003 [[regional-banks/ashikaga-hd|足利銀行]] 的公共资金注入与临时国有化等案例中，于平成时期的银行危机中实质上充当金融安全网的最后防线。

## 1. 法律依据・设立

**正式名称**：存款保险机构
**英文名**：Deposit Insurance Corporation of Japan (DIC)
**法律依据**：**存款保险法**（昭和 46 年法律第 34 号）
**设立**：**1971-07-01**
**总部**：东京都千代田区有乐町 1-12-1
**业态**：认可法人（类似特殊法人）
**主管**：金融厅・财务省・日本银行（三位一体治理）

### 出资结构

```
政府        ── 1/3 相当 ── 存款保险机构
日本银行    ── 1/3 相当 ──
民间金融机构 ── 1/3 相当 ── （megabank・地方银行・信金・信组等）
```

### 重要年表

| 年月 | 事件 |
|---|---|
| **1971-07** | 存款保险法施行・**存款保险机构 设立**（参考美国 FDIC 而构建） |
| 1986 | 引入**资金援助方式**（在破产处理中追加救济合并等方案） |
| 1996 | 住专处理法成立 → 住宅金融债权管理机构（住管机构）设立 |
| **1998-10** | 金融再生法／早期健全化法成立 → 确立公共资金注入框架 |
| **2000-04** | 整理回收机构（[[financial-regulators/rcc|RCC]]）成立（住管机构 + 整理回收银行整合・成为 DIC 子公司） |
| 2001-04 | **解除存款保险冻结** 第 1 阶段（定期性存款）|
| **2002-04** | 全面解禁存款保险（pay-off）→ 次年延期 |
| **2003-05** | **[[megabanks/resona-hd|りそな]] 公共资金注入** 约 2 兆円（事实上的临时国有化） |
| **2003-11** | **[[regional-banks/ashikaga-hd|足利銀行]] 临时国有化**（特别危机管理银行第 1 号）|
| 2005-04 | **全面实施存款保险**（除结算用存款外，本金 1,000 万円 + 利息予以保护） |
| 2010-09 | [[financial-regulators/japan-shinko-bank|日本振興銀行]] 破产 → **日本首次启动存款保险（pay-off）** |
| 2014 | 存款保险法修订 → 有序处理（对金融体系具有重要性的金融机构等的处理） |
| 2020～ | 应对新冠危机・持续运用金融机能强化法 |

## 2. 业务领域

| 业务 | 内容 | 启动案例 |
|---|---|---|
| **保险金支付（pay-off）** | 向破产金融机构的存款人直接支付本金 1,000 万円 + 利息 | 2010-09 [[financial-regulators/japan-shinko-bank|日本振興銀行]] |
| **资金援助** | 向救济金融机构实施金钱赠与・资金贷付・资产收购等 | 1990s 后半多数 |
| **特别危机管理银行** | 将破产金融机构临时国有化，存保作为股东进行经营管理 | 2003 足利银行 |
| **过桥银行** | 在找到接盘候选之前设立临时承继银行 | 第二日本承继银行 等 |
| **公共资金注入** | 依金融机能强化法等以优先股・次级债注入资本 | 2003 りそな、平成金融危机各案 |
| **资产收购** | 通过 RCC 收购不良债权・特定回收困难债权 | 经由 [[financial-regulators/rcc|RCC]] |
| **金融机能强化** | 支持中小金融机构自主性资本增强（金融机能强化法）| 2008-09 雷曼之后 |
| **金融商品交易业者特例** | 与投资者保护基金的联动领域 | 有限 |

### 主要机构生态系统

```
                 存款保险机构（DIC）
                       │
            ┌──────────┼──────────┐
            │          │          │
        [[financial-regulators/rcc|RCC]]   特别危机管理银行  过桥银行
   （整理回收机构）   （临时国有化接盘）  （临时承继银行）
            │
    不良债权收购・回收
```

### 2003 りそな救济（最大级公共资金注入）

- [[megabanks/resona-hd|りそなホールディングス]] 的自有资本比率濒临跌破 4% 国内基准
- 启动存款保险法第 102 条第 1 项第 1 号措施（资本注入）
- 约 **1.96 兆円** 的公共资金注入 → 实质国有化
- 至 2015 已偿清公共资金（计入收益后对纳税人确定为净收益）

### 2003 足利银行 临时国有化

- 以栃木县为根据地的地方银行被认定为资不抵债
- 启动存款保险法第 102 条第 1 项第 3 号措施（**特别危机管理**）
- **存保取得全部股份 → 经营管理 → 2008 出售给野村系接盘方**
- 日本唯一一例「银行国有化 → 完全民营化」走完全程的案例

### 2010 日本振兴银行 首次启动存款保险

- 由前日银审议委员木村刚设立的振兴银行破产
- **日本首次启动存款保险（pay-off）**（战后首次）
- 1,000 万円超出部分的存款人被部分削减 → 「除结算用存款外，本金 + 利息 1,000 万円」原则的现实适用案例

## 4. 国际定位

- **美国 FDIC（Federal Deposit Insurance Corporation）** 于 1970 年年代被参考引入，但与美国不同，其特点在于**将保险职能 + 破产处理职能 + 不良债权处理职能一元化**
- **IADI（国际存款保险协会）** 成员
- 危机时与 [[financial-regulators/boj|日本銀行]] 特别融资（最后贷款人职能）互补运作
