# 地方公共团体金融机构 (JFM)

## Wiki 路径

本条目位于 [[financial-regulators/INDEX|financial-regulators INDEX]] 下。可与 [[financial-regulators/jfc|日本政策金融公庫 (JFC)]] 对读，作为同业 / 对照语境；也可与 [[banking/INDEX|banking index]] 对读，理解更广的系统与监管边界。

## TL;DR

地方公共団体金融機構是专门向地方自治体（都道府县、市区町村）提供长期融资的**地方共同法人**。2008-08-01 以地方公营企业等金融机构形式设立，2009-06-01 改组为现名。它继承了旧 **公営企業金融公庫**（1957-06 设立，2008-09-30 废止）的业务。出资者为全部 47 都道府县 + 全部市区町村的地方共同出资（无国家出资）。贷款余额约 **22.7 兆円**（FY令和6年末 / 2025-03末，JFM IR），是地方财政长期资金供给的基础设施。它与**民间 mega bank / 地银既竞争又互补**，并在灾害复兴融资、上下水道等地方公营企业融资中发挥独特作用。

## 1. 法人结构

**正式名称**：地方公共団体金融機構 ^[extracted]
**英文名**：Japan Finance Organization for Municipalities (JFM) ^[extracted]
**法人形态**：地方共同法人（特殊法人，基于地方公共団体金融機構法）^[extracted]
**设立**：2008-08-01（作为地方公营企业等金融机构）^[extracted]
**现名改组**：2009-06-01（更名为地方公共团体金融机构，并扩大业务范围）^[extracted]
**总部**：东京都千代田区麹町 ^[extracted]
**出资者**：全部 47 都道府县 + 全部市区町村（地方共同出资，无国家出资）^[extracted]

### 机构设计 ^[extracted]

- **理事长**：从地方公共团体侧选任，历任多人拥有地方自治体经验
- **代表者会议**：由全国知事会、全国市长会、全国町村会代表监督
- **经营审议委员会**：由外部有识者进行经营检查

### 业务范围 ^[extracted]

| 业务 | 内容 |
|---|---|
| 地方债承接 | 以长期低利方式承接并运用都道府县、市区町村地方债 |
| 地方公营企业融资 | 向上下水道、电力、交通、医院等地方公营企业提供长期融资 |
| 灾害复兴融资 | 为大地震、台风、地震等灾害复兴提供地方自治体融资 |
| 资金筹措 | 在无政府担保下于国内外资本市场发行债券（地方金融机构债） |
| 经营支持 | 为地方自治体和地方公营企业提供财政分析与经营建议 |

### 贷款余额

- FY令和6年末（2025-03末）约 **22.7 兆円**（JFM IR / 地方金融机构债 IR 资料）
- 作为地方债承接方，是三大资金供给来源之一（与民间银行、财政融资资金并列）

### 公营企业金融公库时代 (1957-2008)

| 年月 | 事件 |
|---|---|
| 1957-06 | **公营企业金融公库设立**（中央政府面向地方自治体融资、国家 100% 出资的特殊法人）^[extracted] |
| 1957-2008 | 向上下水道、交通、电力等地方公营企业提供低利长期融资 |
| 1980s-90s | 地方财政膨胀期贷款余额扩大，成为地方基础设施建设的主要资金供给来源 |
| 2001-12 | **小泉政权特殊法人改革** -> 决定废止公营企业金融公库 ^[extracted] |
| 2008-08-01 | **地方公营企业等金融机构设立**（前身组织，转向地方共同出资）^[extracted] |
| 2008-09-30 | **公营企业金融公库废止** -> 业务和资产由新机构承继 ^[extracted] |

### 地方公共团体金融机构时代 (2009-现在)

| 年月 | 事件 |
|---|---|
| **2009-06-01** | **改组为地方公共团体金融机构**（业务范围从地方公营企业扩大至一般地方自治体）^[extracted] |
| 2011-03-11 | 东日本大震灾 -> 灾害复兴融资的主要承担者之一 |
| 2016-04-14 | 熊本地震 -> 应对灾害复兴融资 |
| 2018-07 | 西日本豪雨 -> 应对灾害复兴融资 |
| 2019-10 | 令和元年东日本台风 -> 应对灾害复兴融资 |
| **2024-01-01** | **能登半岛地震** -> 应对灾害复兴融资 ^[extracted] |

## 3. 业务分部地图

| 分部 | 主要对象 | 特征 |
|---|---|---|
| 一般地方债承接 | 都道府县、市区町村 | 与民间银行、财政融资资金竞争 + 互补 |
| 上下水道融资 | 地方公营企业 | 长期低利（低于民间） |
| 交通融资 | 地方公营交通（公交、地铁） | 维持盈利性较低的公共交通 |
| 医院融资 | 公立医院、地方独立行政法人 | 维持区域医疗的长期资金 |
| 电力 / 燃气融资 | 地方公营电力、燃气事业 | 部分地区的独立公营事业 |
| 灾害复兴融资 | 受灾自治体、公营企业 | 应对大地震、台风、地震的特别融资 |
| 资金筹措 | 地方金融机构债（JFM 债） | 无政府担保。发行体评级 Moody's A1 / S&P A+ / R&I AA+（均稳定，JFM IR）

### 竞争与互补关系

```
地方自治体、地方公营企业的资金筹措
  ├── 地方公共团体金融机构 (JFM) -- 长期低利、政策金融
  ├── 财政融资资金（财务省）-- 国家直接融资、长期低利
  ├── 民间金融机构
  │     ├── [[mufg]] -- 大都市自治体、大规模地方公营企业
  │     ├── [[smfg]] -- 大都市自治体、关西圈自治体
  │     ├── [[mizuho-fg]] -- 大都市自治体、传统地方自治体融资
  │     └── 地银、第二地银 -- 当地自治体 main bank
  └── 资本市场（地方债发行）-- 大都市市场融资增加趋势
```

### 公营企业金融公库废止 -> 地方共同方式转移的背景

- **小泉政权特殊法人改革**（2001-）使国家出资型政策金融机构成为整理对象
- **2008-09-30 公营企业金融公库废止 -> 2008-08-01 新机构设立**，实质上延续业务

### 与民间金融机构的关系

- **互补**：面向盈利性较低的小规模自治体、公营企业的长期融资（民间不积极的领域）
- **利率优势**：地方共同出资 + 高发行体评级（Moody's A1 / S&P A+ / R&I AA+）带来的低筹资成本，使其能够提供低于民间的利率
- **mega bank 侧战略**：[[mufg]] [[smfg]] [[mizuho-fg]] 通过银团贷款、市场性地方债承接服务大都市自治体

### 灾害复兴融资的特殊角色

- 大地震、台风、地震时，地方自治体和公营企业资金需求急增
- JFM 提供仅靠民间金融机构难以覆盖的长期低利融资
- 在东日本大震灾（2011）、熊本地震（2016）、能登半岛地震（2024）等事件中发挥重要作用

### 近期政策论点

- **2024  能登半岛地震复兴**：正式应对地震复兴融资 ^[extracted]

## 5. 监督与监管

- **依据法**：**地方公共団体金融機構法**（2007 年制定，2008 施行）^[extracted]
- **监督**：地方公共团体金融机构代表者会议（全国知事会、市长会、町村会代表）+ 经营审议委员会
- **会计检查**：作为地方共同法人接受会计检查院检查
- **评级**：Moody's A1 / S&P A+ / R&I AA+（均稳定，2024-2025，JFM IR）。虽无政府担保，但凭借地方共同出资的信用力，在日本公共债发行机构中处于最高水准之一

### 与公共金融机构的关系

- **日本政策金融公库 (JFC)**：面向中小企业、农林渔业；融资对象不同于 JFM
- **日本政策投资银行 (DBJ)**：面向民间大企业、基础设施；融资对象不同于 JFM
- **国际协力银行 (JBIC)**：面向海外；JFM 明确面向日本国内地方自治体
