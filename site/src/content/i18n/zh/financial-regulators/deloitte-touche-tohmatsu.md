# 有限责任监查法人德勤 (Deloitte Touche Tohmatsu)

## Wiki 路径

本条目位于 [[financial-regulators/INDEX|financial-regulators INDEX]] 之下。请与 [[financial-regulators/ey-shinnihon|EY 新日本]] 及 [[financial-regulators/pwc-aarata|PwC あらた]] 一并阅读以获取同业 Big 4  审计语境，并与 [[financial-licenses/INDEX|金融ライセンス INDEX]] 一并阅读以了解《公认会计士法》／审计质量管理的监管边界。CPAAOB 监督语境由 [[financial-regulators/fsa-supervision-bureau|金融庁 監督局]] 与 [[financial-regulators/fsa-strategy-bureau|金融庁 総合政策局]] 所管。

## 1. 法人概要

- **正式名**：有限責任監査法人トーマツ ^[extracted]
- **英文名**：Deloitte Touche Tohmatsu LLC ^[extracted]
- **法人形态**：有限责任监查法人 ^[extracted]
- **总部**：东京都千代田区丸之内 ^[extracted]
- **加盟网络**：**Deloitte Touche Tohmatsu Limited（DTTL）**（伦敦总部）^[extracted]
- **集团法人**：
  - 德勤咨询合同会社（战略 / 咨询）
  - 德勤财务咨询合同会社（FAS / M&A）
  - 德勤税理士法人（税务）
  - 德勤风险咨询合同会社（风险 / 网络安全）^[extracted]

## 2. 沿革

| 年月 | 事项 |
|---|---|
| **1968** | **等松・青木监查法人 设立**（トーマツ 的直接前身，トーマツ＝等松青木的略称）^[extracted] |
| 1975 | Touche Ross 国际合作 ^[extracted] |
| **1989** | **Touche Ross + Deloitte Haskins & Sells 全球合并 → 形成 Deloitte Touche Tohmatsu**（社名中纳入了トーマツ，体现日本系的存在感）^[extracted] |
| **2008-07** | **依据《公认会计士法》修订转为有限责任监查法人** ^[extracted] |
| 2003 | DTT 的咨询部门（旧 德勤咨询）解除与监查法人トーマツ的资本关系并退出 DTT → 布拉克斯顿（Braxton，2003-01）→ 作为 **ABeam Consulting（艾意凯咨询）** 独立（2003-11）。现在的“德勤咨询”属另一脉络（旧 トーマツ咨询于 2008-07 改名），与 ABeam 无直接关系 |
| 2020 前后〜 | **数字资产审计 / Travel Rule 咨询率先进入** ^[extracted] |

### 银行 / 保险 / 证券 ^[extracted]

- トーマツ 参与大型证券・资管・寿险及非寿险・地方银行的审计，但个别公司的最新审计人须以 EDINET 一次资料核对，本条目不逐一列举具体客户名称

### 加密资产 / 数字资产 ^[extracted]

- **区块链・加密资产咨询** 组
- 如 **[[exchanges/japan-crypto-audit-firm-landscape|国内クリプト監査ランドスケープ]]** 中所述，在 FSA 注册 VASP 的 **财务报表审计 + 分别管理审计** 方面率先布局
- 参与 **JICPA 行业别委员会实务指引第 61 号(加密资产相关事业者的审计)** 的制定 ^[extracted]
- 国内大型加密资产交易所（含 [[exchanges/jp-exchange-bitflyer|bitFlyer]] 系的公开相关案例）

### Travel Rule / AML 咨询 ^[extracted]

- 关于 [[exchanges/jp-vasp-aml-travel-rule-implementation|JVCEA Travel Rule 実装]] 的咨询
- AML / 制裁筛查建议
- 与风险咨询部门（风险咨询合同会社）协同

### M&A / FAS（与审计分离）^[extracted]

- 德勤财务咨询（DTFA）负责 **M&A 项目咨询**

## 4. 主要 listed FG 客户关系（公开基础）

トーマツ 广泛参与大型金融关联方的审计，但因个别公司的审计人随时点变动，本条目仅限定于可验证的范围：

- **大型 VASP** — [[exchanges/jp-exchange-bitflyer|bitFlyer]] 系等的相关审计实绩（有公开基础的提及）^[extracted]

注：个别公司的审计人随时点变动。最新情况以 **EDINET 各公司最新有价证券报告书「审计人状况」** 或 **JICPA 检索** 作为一次来源确认。

## 5. 监管・监督

- **主管法令**：公认会计士法 ^[extracted]
- **注册**：日本公认会计士协会（JICPA）^[extracted]
- **监督**：**金融厅 公认会计士・监查审查会（CPAAOB）** 进行质量管理复核 / 检查 ^[extracted]
- **CPAAOB 检查结果**：在年度「公认会计士・监查审查会的活动状况」报告书中公布大型监查法人检查结果概要 ^[extracted]

### 过去的处分历史 ^[extracted]

- 个别行政处分历史应以 CPAAOB 检查结果・金融厅公布为基础随时确认
