# 日本政策金融版图

## TL;DR

日本的政策金融体系是一个**专门性公共金融机构的堆栈**，而非单一的政府银行。就 JapanFG 分析而言，有用的路由方式为：

- **国内 SME／初创／农业／危机信贷** -> [[financial-regulators/jfc|JFC]]。
- **海外出口、投资、资源、基础设施与战略金融** -> [[financial-regulators/jbic|JBIC]]。
- **大型企业的长期国内开发、基础设施、GX、危机应对与风险资本** -> [[financial-regulators/dbj|DBJ]]。
- **ODA 与开发合作** -> [[policy-finance/jica|JICA]]。
- **贸易／投资保险** -> [[policy-finance/nexi|NEXI]]。
- **能源与矿产资源安全** -> [[policy-finance/jogmec|JOGMEC]]。
- **冲绳专属的整合性政策金融** -> [[policy-finance/okinawa-development-finance-corp|ODFC]] / [[financial-regulators/okinawa-shinko|ODFC]]。

核心的分析性区分是**商业银行牌照 vs 政策金融授权**。这些主体可以放贷、承保、投资、担保或提供 ODA，但其权限源自专门法规／政府授权，而非普通私人银行的市场定位。请以 [[policy-finance/INDEX|policy-finance INDEX]] 作为该领域的入口，以 [[financial-licenses/INDEX|JapanFG legal / financial licenses domain]] 作为法律牌照层。

| [[financial-regulators/jbic|JBIC]] | Japan Bank for International Cooperation Act；官方简介列出围绕资源、竞争力、环境与金融秩序稳定的四大使命领域 | 海外金融；常与 megabank 合作，但并非 ODA 机构 |

| [[policy-finance/nexi|NEXI]] | Trade and Investment Insurance Act；METI／NEXI 将其定位为出口与投资保险 | 保险／担保式风险吸收，而非直接放贷 |

| 一家日本公司正在向海外出口成套设备、基础设施或装备 | [[financial-regulators/jbic|JBIC]] + [[policy-finance/nexi|NEXI]] | JBIC 可提供融资；NEXI 可承保交易、国别、付款或投资风险。 |
| 一个电力、LNG、关键矿产、氢能、CCS 或上游资源项目需要公共支持 | [[policy-finance/jogmec|JOGMEC]] + [[financial-regulators/jbic|JBIC]] + [[policy-finance/nexi|NEXI]] | JOGMEC 是资源安全节点；JBIC 与 NEXI 是融资／保险两条腿。 |
| 一个发展中国家项目被框定为 ODA 或优惠性开发支持 | [[policy-finance/jica|JICA]] | JICA 承载 ODA 实施通道，包括技术合作、ODA 贷款与无偿援助。 |
| 一个国内基础设施、GX、重组或大型企业风险资本案例需要长期资金 | [[financial-regulators/dbj|DBJ]] | DBJ 更接近开发银行／成长资本／危机应对金融，而非 SME 政策信贷。 |
| 一个冲绳借款人或地区开发项目需要政策金融 | [[policy-finance/okinawa-development-finance-corp|ODFC]] | ODFC 是冲绳专属的整合性机构，不应机械地路由至本土 JFC。 |
| 问题在于「何种法律权限允许此项业务？」 | [[financial-licenses/INDEX]] | 法律／牌照页面区分了银行、证券、保险、支付、加密资产与特殊法人各项制度。 |

## JapanFG 相关性

对 JapanFG 而言，政策金融之所以重要，是因为公共机构常常处于可见的私人银行交易背后。

- **Megabank 联合融资**：在大型基础设施、出口、能源与海外投资交易中，[[megabanks/mufg|MUFG]]、[[megabanks/smfg|SMFG]]、[[megabanks/mizuho-fg|Mizuho FG]] 可能与 [[financial-regulators/jbic|JBIC]]、[[financial-regulators/dbj|DBJ]]、[[policy-finance/nexi|NEXI]] 并列出现。
- **地方银行近接性**：国内 SME 政策金融常涉及 JFC／本地银行联合融资；冲绳案例可能涉及 ODFC 与 [[regional-banks/okinawa-fg|Okinawa FG]]、[[regional-banks/ryukyu-bank|Ryukyu Bank]]、[[regional-banks/okinawa-kaiho-bank|Okinawa Kaiho Bank]]。
- **政策授权 vs 牌照**：JFC／JBIC／DBJ／ODFC 的分析不应被归并为普通银行牌照分析。它们的资产负债表、风险偏好与授权，都与专门法规和公共政策相系。
- **项目堆栈逻辑**：海外资源或基础设施金融可能组合 JOGMEC、JBIC、NEXI、私人银行、商社、公用事业公司，以及在开发合作层确实存在时偶尔加入的 JICA。

## 边界案例

- **JICA vs JBIC**：JICA 是 ODA／开发合作通道；JBIC 是面向日本企业、资源、基础设施与金融稳定的海外政策金融通道。
- **NEXI vs JBIC**：NEXI 承保贸易与投资风险；JBIC 放贷、投资、担保并提供其他融资产品。
- **JOGMEC vs JBIC**：JOGMEC 锚定资源安全政策与风险分担；JBIC 是可能为项目融资的更广泛的海外金融机构。
- **JFC vs DBJ**：JFC 聚焦国内 SME、微型企业、农业／渔业、食品与危机应对；DBJ 是面向更大型、更长期、基础设施、风险资本与危机应对职能的开发银行通道。
- **ODFC vs 本土机构**：ODFC 并非仅仅是「JFC 冲绳分行」。官方来源将其框定为集中化的冲绳政策金融；内阁府指出它承担相当于多个本土机构的职能。
- **特殊法人 vs 独立行政法人**：JFC／JBIC／DBJ／ODFC 与 JICA／JOGMEC／NEXI 并非全部具有相同的法律形态。应将「政策金融」视为一张功能性版图，再逐案核实确切的法规与法人形态。
