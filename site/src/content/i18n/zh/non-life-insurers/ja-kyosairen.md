# JA 共济连（全国共济农业协同组合连合会 / National Mutual Insurance Federation of Agricultural Cooperatives）

## Wiki 路径

本条目归属于 [[non-life-insurers/INDEX|non-life-insurers INDEX]]，作为 **JA 集团的合作社保险联合会**，与 [[cooperative-banks/norinchukin|農林中央金庫 (信用)]] / [[cooperative-banks/ja-zen-noh|JA 全農 (経済)]] / [[cooperative-banks/ja-zenchu|JA 全中 (代表)]] 平行。请与 [[insurance/INDEX|insurance domain]] 一并阅读以了解合作社 vs 股份制保险形态，与 [[non-life-insurers/kyoritsu-fire-marine|共栄火災海上保険]]（JA 共济连为第一大股东）一并阅读以了解非寿险补充，并与 [[banking/ja-bank-system-japan|JA Bank system]] 一并阅读以了解 JA 集团的系统边界。

## TL;DR

**JA 集团 4 大全国联中的共济事业核心 = 横跨寿险 + 非寿险两域的合作社「共济」事业者**。于 1951 成立，以农业协同组合法为依据的 **联合会**。**生命系保有合同额约 200 兆円规模，汽车・建物等损害系亦大规模展开**，面向组合员（农户・地区居民）一揽子提供生命・年金・汽车・建物・伤害等共济，是**国内屈指可数的保险事业组织**。**它并非保险业法下的股份制「保险公司」，而是依据农协法的「共济」**，受到不同法域的规制。它在拥有可与 **[[insurance/japan-life-insurance-big-four|生命保険ビッグ 4]]（日本生命 / 第一生命 / 明治安田 / 住友生命）比肩的规模**的同时，是不公开上市的合作组织型独特板块。作为 [[non-life-insurers/kyoritsu-fire-marine|共栄火災海上保険]] 的 **74.20% 股东**，它也是非寿险公司的母公司（通过共荣火灾开展合作社之外的非寿险事业）。^[extracted]

### 共济事业 vs 保险业法（区别）^[extracted]

| 维度 | 保险公司（生・非寿险） | JA 共济连（共济） |
|---|---|---|
| 依据法律 | 保险业法 | 农业协同组合法 |
| 主管 | 金融厅 | 农林水产省（主）+ 金融厅协力 |
| 形态 | 股份公司 / 相互公司 | 联合会（合作组织） |
| 客户 | 公众（open） | **组合员 + 员外利用额度** |
| 产品名称 | 保险 | **共济**（年金共济 / 终身共济 / 建物更生共济 / 汽车共济 等） |
| 偿付能力规制 | ESR（经济价值基准） | 共济掛金基础率规制 + 自主健全性 |

### 共济 vs 保险的渠道

- **销售渠道**：全国 497 个 JA 组合 + 都道府县连 + 全共连直营
- 以面向组合员家庭的 **面对面・上门销售** 为主 — 类似于寿险行业的「销售女士」型
- **保有合同额（生命综合・建物更生共济 合计）为 216 兆 9,652 億円**（令和 5 年年度末）— 与寿险行业 Big 4  比肩的规模感

### 非寿险补充的母公司职能

- **[[non-life-insurers/kyoritsu-fire-marine|共栄火災海上保険]] 的 74.20% 股东** ^[extracted]
- 共荣火灾是 **符合保险业法的非寿险公司** = 可向组合员以外销售非寿险 + 可进行一般法人交易
- JA 共济连自身的共济无法覆盖的领域（海上保险・赔偿责任的高额案件等）由共荣火灾对应
- **合作社与股份制非寿险公司的混合模式**

### 国内最大级的保险事业者定位

- **总资产规模**：58 兆 4,751 億円（令和 5 年年度末，较上年度 101.4%）— 与 [[life-insurers/nippon-life|日本生命]]（逾 90 兆円）比肩的规模感
- 不过，共济基于 **农协法上的组合员基础**，故通常不进入基于保险业法的规模排名（监管报表不同）

### 产品竞争

- **建物更生共济（建更）**：火灾 + 地震 + 死亡保障的 **一体型产品** = 在民营保险业界无类例，是共济独有的强项
- **农协品牌・地区贴近** 直接关系到掛金（保费）竞争力

### 数字战略

- 通过 **JA 共济 App** 进行合同查询・保险金申请・预定利率确认 ^[extracted]

### 竞争格局

- **寿险 Big 4**：[[life-insurers/nippon-life|日本生命]] / [[life-insurers/dai-ichi-life|第一生命]] / [[life-insurers/meiji-yasuda|明治安田生命]] / [[life-insurers/sumitomo-life|住友生命]] — 以同型生命系产品竞争
- **3 大非寿险**：[[non-life-insurers/tokio-marine|東京海上]] / [[non-life-insurers/msad|MS&AD]] / [[non-life-insurers/sompo|SOMPO]] — 在汽车・建物等方面竞争
- **其他合作社系**：[[non-life-insurers/zenrosai|全労済 / こくみん共済 coop]]、都道府县民共济 — 以组合员基础的互助进行部分竞争
- **对 JA 内客户的圈层化** 是与其他金融机构的根本性差异（组合员贴近 + JA Bank 账户联动）

## 4. 监管・规制

- **主管**：**农林水产省**（合作社监管）+ **金融厅**（在金融健全性・消费者保护方面协力监管）
- **法律依据**：农业协同组合法（共济事业条款）+ 共济事业指南
- **近期政策论点**：
  - **共济 = 保险的规制差距** 讨论（保险业法 vs 共济法的竞争公平性）
  - 对共济适用类似 **经济价值基准・偿付能力（ESR）** 规制的讨论
  - **数字销售渠道** 中的消费者保护
  - **粮食安全保障计划（2026〜）** 中的组合员保护职能
