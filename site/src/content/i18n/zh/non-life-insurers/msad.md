# MS&AD 保险集团控股

## Wiki 路由

本条目归于 [[non-life-insurers/INDEX|non-life-insurers INDEX]] 之下。请结合 [[non-life-insurers/sompo|SOMPO ホールディングス (Sompo Holdings)]] 以获取同业／对比背景，并结合 [[banking/INDEX|banking index]] 以了解更广泛的体系／监管边界。

## TL;DR

国内 3 大财险之一（与 [[non-life-insurers/tokio-marine|東京海上 HD]] / [[non-life-insurers/sompo|SOMPO HD]] 并列）。经 **2008-04 [[non-life-insurers/mitsui-sumitomo-insurance|三井住友海上]] + Aioi 财险 + 日生同和财险 经营整合** 而成立。社名源自构成 3 社的首字母「**M**itsui Sumitomo + **A**ioi + **D**owa」。国内曾采取维持 **3 品牌体制**（三井住友海上・Aioi 日生同和・三井 Direct）的特异战略，但随 **2027-04 三井住友海上 + Aioi 日生同和财险 拟合并**，3 品牌体制走向终结。海外则以 **2016 年 MS Amlin 收购（约 3600 億円）** 正式进入英国劳合社市场。与丰田・日本生命的资本系列经由「Aioi」存续，在汽车保险・银行窗口销售寿险方面强势。**FY2024 净利润 6,916 億円（+87% YoY，历史最高）**。证券代码 **东证 PRIME 8725**。

## 1. 公司概要

**正式名称**：MS&AD 保险集团控股株式会社（MS&AD インシュアランス グループ ホールディングス株式会社）
**英文名**：MS&AD Insurance Group Holdings, Inc.
**证券代码**：东证 PRIME 8725
**设立**：2008-04（由控股公司「Millea Holdings」改组发展而来）
**成立经过**：经 2010-04-01 三井住友海上集团控股 + Aioi 损害保险 + 日生同和损害保险 经营整合确立现体制
**总部**：东京都千代田区
**机关设计**：设监查役会公司

### 合并经过・前身

- 旧 **三井海上火灾保险**：1918 设立（三井系）
- 旧 **住友海上火灾保险**：1893 设立（住友系・大阪起源）
- **三井住友海上火灾保险**：**2001-10-01 三井海上 + 住友海上 合并**
- 旧 **大东京火灾海上保险** + 旧 **千代田火灾海上保险** → **Aioi 财险**（2001-04 合并，丰田系）
- 旧 **同和火灾海上保险** + 旧 **日生损害保险** → **日生同和财险**（2001-04 合并，日本生命系）
- **MS&AD HD**：2010-04-01 经营整合确立
- **Aioi 日生同和财险**：2010-10-01 Aioi + 日生同和 合并

### 国内 3 品牌战略的特异性 → 2027 终结

MS&AD 在国内财险采取了与其他大型不同的体制，即 **维持 3 品牌**（三井住友海上 / Aioi 日生同和 / 三井 Direct）。一般合并中统合品牌为常态，但 MS&AD 一直优先避免对其**渠道基础**（旧三井住友＝大企业代理店，旧 Aioi＝丰田经销店，旧日生同和＝日本生命系）的损害。

★ 但 **2027-04 三井住友海上 + Aioi 日生同和财险 拟合并**已公布，长年维持的 3 品牌战略将终结。背景包括：
- Big Motor 案件以来的代理店治理强化成本
- 企业向保险卡特尔问题 → 2025-03 三井住友海上 金融厅业务改善命令
- 追求整合协同的压力（压缩重复成本，统一 IT 基础）
- 丰田・日本生命系渠道的维持成本 vs 效率化的取舍再评估

### 与丰田・日本生命的资本系列

- **经由 Aioi 的丰田系**：旧大东京火灾（含千代田火灾）为丰田系列，保有经丰田经销店的汽车保险销售渠道
- **经由日生同和的日本生命系**：旧日生损害保险为日本生命系，银行窗口销售・营业职员渠道基础
- **车联网保险**：与丰田 Connected Car 数据联动的汽车保险（Aioi 主导）

### 海外战略

- **2016 MS Amlin 收购**（约 3600 億円）为转折点。正式持有英国劳合社辛迪加，获得特殊保险・再保险组合
- **亚洲集中**：经 Asia Capital Re / Bohai / Newa / BNP Cardif Vietnam 多面展开再保险・财险・寿险
- **美洲**：MS Reinsurance（旧 Trans-Re）

### 可持续性・气候

- **碳中和举措**：投资・承保两方面的转型战略
- **TCFD / TNFD**：气候・自然资本风险披露
- **灾害保险**：国内自然灾害频发化 → 火灾保险费率改定（业界共通课题）

## 4. 监管・政策

- **主管**：金融厅（FSA）
- **保险业法**：财险原保险・再保险・寿险的偿付能力边际规制
- **海外规制**：
  - **PRA / FCA（英国）**：MS Amlin（劳合社辛迪加）
  - **NAIC / 州规制（美国）**：MS Reinsurance
  - **MAS（新加坡）**：Asia Capital Re
- **近期政策议题**：
  - 汽车保险费率改定（事故率・修理费高涨）
  - 火灾保险费率改定（自然灾害频发化）
  - **Big Motor 问题**（2023〜）后的财险代理店治理强化（业界横向，含财险 4 社卡特尔嫌疑）^[公開情報ベース・複数社案件]
  - ★ **2025-03 三井住友海上 金融厅业务改善命令**（客户信息管理问题，含企业向保险卡特尔相关）— 整合判断的催化剂
  - **政策性持股缩减**（业界共通课题，财险 4 社协商一致）— 浮盈兑现与计入处置收益的时机
