# 共荣火灾海上保险

## Wiki 路径

本条目位于 [[non-life-insurers/INDEX|non-life-insurers INDEX]] 之下。请对照 [[non-life-insurers/sompo|SOMPO ホールディングス (Sompo Holdings)]] 阅读以获取同业 / 对比背景，并对照 [[banking/INDEX|banking index]] 了解更广泛的系统 / 监管边界。

## TL;DR

以 JA 共济连为第一大股东（**74.20%**）的中坚财产保险公司。于 1942-07-01 战时统合（大东海上 + 大福海上）创业，于 1946 年 **在财产险行业罕见地相互会社化**，于 2003-04 转换为股份公司。不属于 3 大财产险（[[non-life-insurers/tokio-marine]] / [[non-life-insurers/msad]] / [[non-life-insurers/sompo]]），走 **合作社・合作组织系列** 的独立路线。在北海道・东北・北陆・九州拥有相对较强的份额，定位为补充 JA 共济无法承办领域（汽车・火灾等）。自 2025-04 起启动新中期计划「**SHIFT〜面向可持续增长〜**」。^[extracted]

## 2. 业务分部地图

| 分部 | 主要产品 | 特征 |
|---|---|---|
| 汽车保险 | 任意保险・交强险 | 主力。经由经销商・维修厂代理店 |
| 火灾保险 | 住宅火灾・地震保险 | JA 共济补充领域 |
| 伤害保险 | 休闲・运动・交通伤害 | - |
| 海上保险 | 货物・船舶 | 创业领域 |
| 赔偿责任保险 | 个人・法人 PL・设施赔 | - |
| 新种・其他 | 动产综合・利润保险等 | - |
| JA 联动产品 | 补充 JA 共济无法承办领域的财产险 | **面向农协客户的独有渠道** |

### 销售渠道 ^[extracted]

- **合作社・合作组织系代理店**（JA、信金、渔协等）
- **企业・经销商・维修厂代理店**
- **共荣专业俱乐部**（专业代理店网络）
- **直销员工**

### 区域特性 ^[extracted]

**在北海道・东北・北陆・九州的投保份额相对较高**。与农协基础较强的区域一致，以 JA 共济连为第一大股东的结构反映在销售网络中。

#### GO TO THE NEXT STAGE（2022～2024 年年度）^[extracted]
- 「成为更受客户选择的公司」
- 2024 年年度为最终年度

#### **SHIFT〜面向可持续增长〜（2025〜）**★ ^[extracted]
- 「SHIFT」= 不只是单纯的变化，而是在适应环境变化的同时 **移动重心**
- 重点主题：
  1. 与合作社・合作组织诸团体及交情深厚的企业・经销商・维修厂的 **信赖关系强化**
  2. 强化共荣专业俱乐部・直销员工渠道
  3. 确立能够持续实现 **人才投资・系统投资** 等增长投资的事业结构
  4. 打造在高度不确定时代生存下去的组织

## 4. 监管・政策

- **主管**：金融厅（FSA）
- **行业团体**：日本损害保险协会
- **监管**：保险业法、面向财产保险公司的监督指针
- **近期政策议题**：
  - 汽车保险费率（损害率上升 → 费率上调局面）
  - 自然灾害（台风・暴雨・地震）与再保险成本
  - JA 共济连系列合作社金融制度的讨论
  - 财产险行业卡特尔问题（2023～2024 大型 4 社）向中坚财产险的波及
