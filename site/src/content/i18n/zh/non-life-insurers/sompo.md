# SOMPO 控股 (Sompo Holdings)

## Wiki 路径

本条目位于 [[non-life-insurers/INDEX|non-life-insurers INDEX]] 之下。请将其与 [[non-life-insurers/msad|MS&AD インシュアランス グループ ホールディングス]] 对照阅读以了解同业 / 对比背景，并与 [[banking/INDEX|banking index]] 对照以了解更广泛的体系 / 监管边界。

## TL;DR

国内 3 大损保之一（[[non-life-insurers/tokio-marine|東京海上 HD]]·[[non-life-insurers/msad|MS&AD HD]]·SOMPO HD）。**2010-04 由旧 [[non-life-insurers/sompo-japan-insurance|損保ジャパン]] + 日本兴亚损保 经营整合**而成立 NKSJ HD，2014-09 改称为 Sompo HD → 2016-10 SOMPO HD。以**"主题公园战略"**将 损保 + 寿险 + **护理（SOMPO Care = 国内护理最大企业）** + 数字 + 海外保险（**Sompo International**，2017 以约 6300 億円收购旧 Endurance）作为 5 大支柱。因**2023-2024 Big Motor 不当修理问题**，损保 Japan 受到金融厅业务改善命令（2024）。东证 PRIME 8630。^[extracted]

**业态**：保险持株公司（损保 + 寿险 + 护理 + 海外）

### 主题公园战略

- SOMPO 经营的代名词。将**损保 + 寿险 + 护理 + 数字 + 海外保险**定位为"超越保险的安心·安全·健康的主题公园" ^[extracted]
- 樱田谦悟（会长）长年主导的概念
- 将护理（SOMPO Care）置于非保险分部战略核心这一点，是国内 3 大损保中最为明确的

### 护理事业

- **SOMPO Care**：旧和民护理（2015 从和民以 210 億円收购）+ 旧 Message（2016 以约 270 億円进行 TOB）整合 ^[extracted]
- 2019 统一为 SOMPO Care → 以居室数计国内护理最大企业规模

### 海外战略

- **Sompo International**（旧 Endurance Specialty）：2017-03 以约 6300 億円收购 → 百慕大系·特殊保险 + 再保险枢纽 ^[extracted]
- **以北美为中心**：美国损保 + 百慕大再保险
- **新兴国**：以 Sompo Sigorta（土耳其）/ Sompo Brasil / 亚洲据点进行小额分散
- **DC Capital**：英国劳合社·辛迪加运营

### 数字

- **SOMPO Digital Lab**（东京·硅谷·特拉维夫）
- **Palantir 战略合作**（2020）：SOMPO HD 向 Palantir Technologies 出资约 5 億美元（约 540 億円）。SOMPO HD·Palantir·Palantir Japan 3 方以 Palantir Foundry 为基础设立"安心·安全·健康的真实数据平台 (RDP)"，以护理·健康领域为起点整合分析集团内外数据
- 护理数据 + 损保数据的整合平台构想

### Big Motor 事件之后的治理改善 (FY2024 进展) ^[extracted]

- **2024-01-26**：损保 Japan 白川仪一 社长 辞任
- **2024-03**：SOMPO HD **樱田谦悟 CEO 卸任**（转任会长职）→ 经营高层换新
- **后任**：石川耕治 就任损保 Japan 社长 / 奥村干夫 SOMPO HD 社长体制
- **2024 业务改善命令**（金融厅·致损保 Japan）→ 内部控制报告·明确经营责任
- **FY2024 内部控制评价："有效"**（重建后首次综合评价正式确认改善）
- 董事会改革·提高社外董事比例
- 重新审视依赖二手车维修厂型营业模式（强化特定维修厂风险管理）
- 与 DRS Group（车辆维修平台）/ Zego（英国 insurtech）等的 2026 战略合作 → 转向治理重建后的海外·数字攻势 ^[extracted]
- 3 大损保卡特尔嫌疑（企业向保险费事前协调，违反反垄断法）亦另由公平交易委员会调查 → 全行业结构改革压力持续 ^[extracted]

## 4. 监管·政策

- **主管**：金融厅（FSA）
- **持株公司监管**：保险业法（保险持株公司）
- **国内损保行业监管**：保险业法、反垄断法（卡特尔嫌疑相关）
- **海外监管**：
  - 百慕大金融厅（BMA, Sompo International）
  - 美洲 各州保险监督厅
  - 英国 PRA / FCA（DC Capital / 劳合社相关）
  - 土耳其 SEDDK（Sompo Sigorta）
- **近期政策论点**:
  - **2024 业务改善命令**（致损保 Japan，Big Motor 案件）→ 内部控制报告·明确经营责任
  - **违反反垄断法嫌疑**（3 大损保卡特尔嫌疑，企业向保险费协调）→ 公平交易委员会调查
  - 政策持有股票削减（金融厅 解除交叉持股方针）
  - 自然灾害多发化 → 火灾保险费率修订压力（经由损害保险费率算定机构）
