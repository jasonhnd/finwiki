# 大同火灾海上保险

## 维基导览

本词条归属于 [[non-life-insurers/INDEX|non-life-insurers INDEX]]。请对照 [[non-life-insurers/sompo|SOMPO ホールディングス (Sompo Holdings)]] 阅读以获取同业／对比背景，并对照 [[banking/INDEX|banking index]] 了解更宏观的体系／监管边界。

## TL;DR

**冲绳县唯一的本地财险公司**。1951-04 在美军政下的冲绳以「冲绳海上火灾保险」之名设立，是战后冲绳唯一一家日本籍财险公司。1972 年随着冲绳本土回归，转为日本国内财险经营，1985 年改名为现社名「大同火灾海上保险」。2000 年年代以降，通过与损保 Japan（现 [[non-life-insurers/sompo]]）的业务合作覆盖全国网络。**冲绳县内财险份额 No.1**，专注于旅游保险（旅行意外·租车·民宿赔偿）与台风·离岛再保险方案。在与大型 3 财险（[[non-life-insurers/tokio-marine]] / [[non-life-insurers/msad]] / [[non-life-insurers/sompo]]）冲绳据点的竞争之下维持本地优势。非上市。^[extracted]

## 1. 公司概要

**正式名称**：大同火灾海上保险株式会社 ^[extracted]
**英文名**：Daido Fire and Marine Insurance Co., Ltd.
**设立**：1951-04（以冲绳海上火灾保险株式会社之名）^[extracted]
**现社名变更**：1985（改名为大同火灾海上保险）^[extracted]
**总部**：冲绳县那霸市 ^[extracted]
**上市**：**非上市**（由冲绳县内财界构成持股）^[extracted]
**业态**：财产保险业（金融厅认可，损保协会加盟）

### 主要业务·分支网络

```
大同火灾海上保险（总部：那霸市）
  ├── 冲绳县内网络
  │     ├── 那霸总店·宫古支店·八重山支店·名护营业所等
  │     ├── 离岛覆盖（先岛诸岛·大东诸岛）
  │     └── 县内代理店网络（汽车经销商·房地产·本地金融）
  ├── 本土据点（东京·大阪·福冈）
  │     └── 冲绳关联法人·面向游客的接触点
  └── 损保 Japan（[[non-life-insurers/sompo]]）业务合作
        └── 全国网络·再保险·产品许可
```

## 2. 业务板块·地图

| 板块 | 主要产品 | 特征 |
|---|---|---|
| 汽车保险 | 任意汽车（县内份额 No.1）| 面向租车·游客的短期险也很强 |
| 火灾·地震保险 | 住宅·店铺·营业场所 | 台风·风暴潮·盐害覆盖为本地化设计 |
| 旅行意外保险 | 国内旅行·面向游客 | 与旅游产业的代理店网络深厚 |
| 租车保险 | 冲绳旅游必需 | 与大型租车公司的整体合同 |
| 民宿·住宿赔偿 | 小规模民宿·度假屋 | 覆盖离岛·本地住宿事业者 |
| 海上保险 | 渔船·小型船舶·海运 | 创业以来的主业（社名由来）|
| 责任赔偿保险 | 小规模事业者 PL·设施赔偿 | 面向县内中小企业 |

### 本地优势的来源

- **冲绳县内财险份额 No.1**（本地代理店网络·财界网络）^[extracted]
- **离岛覆盖**：宫古·八重山·大东诸岛等本土大型机构薄弱的地区也予以覆盖
- **旅游保险的专业性**：租车·民宿·潜水等冲绳特化产品
- **本地品牌信赖**：县民意识·本地雇佣·与代理店的长期关系

### 与大型 3 财险的竞争

| 竞争对手 | 在冲绳的定位 |
|---|---|
| [[non-life-insurers/tokio-marine]]（东京海上日动）| 全国大型·面向法人强势·大规模风险 |
| [[non-life-insurers/msad]]（MS&AD）| 通过爱和谊日生同和等在县内展开 |
| [[non-life-insurers/sompo]]（损保 Japan）| **与大同火灾业务合作** → 并非竞争而是互补关系 |

### 灾害风险管理

- **台风**：应对每年多次直击的火灾·地震·海上再保险方案
- **地震**：西南诸岛地震带的长期风险
- **离岛风险**：交通中断·物资供应断绝等业务连续性风险
- **盐害·风暴潮**：沿岸地区特有的覆盖设计

### 旅游保险·入境游

- 伴随冲绳游客数量（疫情前年间超 1000 万人，2024年起呈回升趋势）的旅行意外保险需求
- 租车保险（冲绳旅游的标准出行方式）
- 潜水·海洋运动赔偿
- 民宿·青年旅舍赔偿（含 Airbnb 等小规模事业者）

### 美军相关风险

- 美军基地周边的事故·噪音·业务连续性风险为冲绳特有
- 伴随基地归还·旧址开发的长期风险评估

## 4. 监管·政策

- **主管**：金融厅（FSA）
- **行业团体**：日本损害保险协会（损保协会）加盟
- **近期政策议题**：
  - 伴随台风大型化·气候变化的保险费率改定
  - 伴随游客增加的入境游保险产品扩充
  - 伴随美军基地重整·旧址利用的风险重新评估
  - 财险行业代理店手续费制度的重新审视（在冲绳亦有影响）
