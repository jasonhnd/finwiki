# 日本地方银行并购整合家谱矩阵

## TL;DR

1990  后日本的地方银行整合，产生了一张分层的银行控股公司地图，每家控股公司合并了一家或多家第一地方银行（「地銀」）和 / 或第二地方银行（「第二地銀」）前身，某些情况下还包含一家信用合作社 / 新银行前身。本矩阵收集了截至 2026 日本每一家实质活跃的地方银行 FG 的存续 FG 名称、FG 成立年份、前身银行、都道府县覆盖、资本规模及控股公司结构。它是 [[banking/regional-bank-consolidation-pattern|Regional bank consolidation pattern]]（为何）与 [[banking/japan-banking-license-tier-comparison-matrix|Japan banking-license tier comparison matrix]]（何种牌照）的家谱对应版本。

## Why this family-tree matters

地方银行整合是分层的。许多引用只说「X銀行」，而不指明说话者所指的是：

- **原始前身银行**（例如 第四銀行 1873-2020）；
- **合并后的存续银行**（例如 [[JapanFG/daishi-hokuetsu-bank|第四北越銀行]]）；
- 运营银行之上的**控股公司**（[[JapanFG/dai-shi-hokuetsu-fg|第四北越 FG]] 7327）；
- 控股公司所属的**联盟 / 财团**（例如 TSUBASA、[[JapanFG/sbi-regional-bank-hd|SBI 地銀 HD]]）。

没有明确的家谱地图，同一个品牌名称会被用于四个不同的法律实体。本矩阵将这四个层级强制纳入一个一致的视图，以便在做出任何资产负债表或战略主张之前，能够回答「我实际谈论的是哪个实体」。

具体而言，本矩阵呈现：

- 哪些 FG 是**单一都道府县单一银行**（例如 [[JapanFG/iyogin-hd|いよぎん HD]]、[[JapanFG/shizuoka-fg|しずおか FG]]、[[JapanFG/hirogin-hd|ひろぎん HD]]、[[JapanFG/juroku-fg|十六 FG]]）——那些是 单独株式移転 模式，其中控股公司本质上是单银行外壳；
- 哪些 FG 是**跨都道府县两家或更多银行**（[[JapanFG/mebuki-fg|めぶき FG]]、[[JapanFG/concordia-fg|横浜 FG (旧コンコルディア)]]、[[JapanFG/kyushu-fg|九州 FG]]、[[JapanFG/fukuoka-fg|FFG]]、[[JapanFG/hokuhoku-fg|ほくほく FG]]）——那些是 共同株式移転 模式，跨都道府县界线合并银行；
- 哪些 FG 由**救济性并购**形成（[[JapanFG/jimoto-hd|じもと HD]] 在 2024-06 取得 RCC 多数；[[JapanFG/tomony-hd|トモニ HD]] 在 2017 吸收 大正銀行；[[JapanFG/mebuki-fg|めぶき FG]] 加入由 Resona / DICJ 处置的 足利銀行 谱系）；
- 哪些 FG 合并了**第一地方 + 第二地方**混合，而非一对同层银行；
- 哪些 FG 属于**更广泛的财团**（[[JapanFG/sbi-regional-bank-hd|SBI 地銀 HD]] 股权联盟环；围绕 [[JapanFG/kansai-mirai-bank|関西みらい銀行]] 与 [[JapanFG/minato-bank|みなと銀行]] 的 Resona 集团簇）。

这一家谱直接馈入牌照分层矩阵 [[banking/japan-banking-license-tier-comparison-matrix|Japan banking-license tier comparison matrix]]（此处每一行都在 Tier 2 第一地銀 或 Tier 3 第二地銀）以及合作社银行矩阵 [[banking/japan-cooperative-banking-unified-federation-matrix|Japan cooperative-banking unified federation matrix]]（合作社银行整合位于与此处所述控股公司路线不同的轴上）。

## How to read this matrix

1. **先选定 FG（控股公司）行。** FG 是发行底层银行控股、签署并购文件，并在 FSA 银行控股公司监管下报告合并资本充足率的法律实体。这是战略得以执行的单位。
2. **阅读前身银行一列。** 它告诉你该 FG 是单银行外壳（例如 [[JapanFG/iyogin-hd|いよぎん HD]]、[[JapanFG/shizuoka-fg|しずおか FG]]）还是真正的多银行整合（[[JapanFG/fukuoka-fg|FFG]]、[[JapanFG/concordia-fg|横浜 FG]]）。
3. **检查运营银行模式一列。** 2-bank umbrella 在一个 FG 下保持两家运营银行存活；1-operating-bank 模式意味着各组成银行已在法律上合并（[[JapanFG/daishi-hokuetsu-bank|第四北越銀行]] 于 [[JapanFG/dai-shi-hokuetsu-fg|DHFG]] 之下）。
4. **检查都道府县覆盖一列。** 单一都道府县 FG（[[JapanFG/shizuoka-fg|しずおか]]、[[JapanFG/hirogin-hd|ひろぎん]]、[[JapanFG/juroku-fg|十六]]）是本地平台打法；跨都道府县 FG（[[JapanFG/mebuki-fg|めぶき]]、[[JapanFG/hokuhoku-fg|ほくほく]]、[[JapanFG/kyushu-fg|九州]]）是区域平台打法；全国覆盖 FG（[[JapanFG/sbi-regional-bank-hd|SBI 地銀 HD]] 财团）是数字平台打法。
5. **与 [[banking/regional-bank-api-digital-partnership-route|regional-bank API and digital partnership route]] 交叉核对**，看相同的 FG 如何位于数字轨道轴上（[[JapanFG/fukuoka-fg|FFG]] 的 みんなの銀行 BaaS、[[JapanFG/hokkoku-fhd|CCI / Hokkoku]] 的 Tonari、[[JapanFG/kiraboshi-fg|きらぼし]] 的 UI 銀行）。

## Key consolidation events as turning points

- **2003-09 → 2004-09 [[JapanFG/hokuhoku-fg|ほくほく FG]] (北陸 + 北海道).** 日本银行史上首个「離れた地域の地銀どうしの広域連合」模式。
- **2003-11-29 足利銀行 破綻 → 一時国有化 → 2008-07 民間譲渡 (野村ファンド + ネクスト・キャピタル等) → 2008-12 足利 HD → 2016-10 [[JapanFG/mebuki-fg|めぶき FG]] 合流.** 后破产复健弧线，后来在 FG 伞下加入了一个健康的伙伴。
- **2007-04 [[JapanFG/fukuoka-fg|FFG]] 発足 (福岡 + 熊本ファミリー).** 为 1地銀 主导的跨都道府县 FG 设定了模板。
- **2009-10 [[JapanFG/fidea-hd|フィデア HD]] (荘内 + 北都).** 東日本 首个跨都道府县 FG。
- **2010-04 [[JapanFG/tomony-hd|トモニ HD]] (徳島 + 香川).** 第二地方跨都道府县 FG，后经 2017 大正銀 救济扩展 → 2020-01 徳島大正銀行 誕生。
- **2014-10 → 2016-04 → 2018-05 [[JapanFG/kiraboshi-fg|東京きらぼし FG]].** 都民 + 八千代 → +新銀行東京 → 3行合併（合并 都政の新銀行東京 结果的东京都市圈特例）。
- **2015-10 [[JapanFG/kyushu-fg|九州 FG]] (肥後 + 鹿児島).** 九州南部的跨都道府县 2强连合。
- **2016-04 [[JapanFG/concordia-fg|横浜 FG (旧コンコルディア)]] (横浜 + 東日本).** 最大单一 FG 类别的跨层（1地銀 + 2地銀）跨都道府县伞。
- **2016-10 [[JapanFG/mebuki-fg|めぶき FG]] (常陽 + 足利 HD).** 与后破产处置伙伴的跨都道府县组合。
- **2018-04 → 2021-05 [[JapanFG/san-jusan-fg|三十三 FG]] (三重 + 第三 → 三十三銀行).** 带完整法律合并的 2地銀 + 2地銀。
- **2018-10 → 2021-01 [[JapanFG/dai-shi-hokuetsu-fg|第四北越 FG]] (第四 + 北越 → 第四北越銀行).** 新潟 的新 一県一行 模式。
- **2020-12 [[JapanFG/sbi-regional-bank-hd|SBI 地銀 HD]] 設立.** 「第4のメガバンク構想」非 FG 联盟模式。
- **2022-04 → 2025-01 [[JapanFG/proclair-hd|プロクレア HD]] (青森 + みちのく).** 为都道府县垄断清关**首次适用 独禁特例法（令和2年法律第32号）**。
- **2024-06-20 [[JapanFG/jimoto-hd|じもと HD]] RCC 議決権 63.52% 取得.** 为 2地銀 FG 强化了公共资金监管。
- **2025-10 [[JapanFG/concordia-fg|横浜 FG]] 商号変更**（旧 コンコルディア → 横浜 FG）。
- **2025-10 [[JapanFG/hokkoku-fhd|北國 FHD]] → [[JapanFG/cci-group|CCI グループ]] 商号変更.** 数字平台转向反映在公司名称上。

## Holding-company FSA permission process

银行法下的 FSA 银行控股公司许可涉及：

1. 设立新银行控股公司的**银行法事前批准**（无自动权利；FSA 审查）。
2. 在合并基础上所需的**资本充足率**（巴塞尔 III 合并报告；视是否国际活跃而采用 BIS / 国内标准）。
3. **治理审查**，包括控股公司董事会结构、内部审计框架、集团风险管理。
4. **集团业务战略**披露，包括每家子银行的角色。
5. **完成期限**：从公告到 FG 设立通常 6-12  个月；在跨都道府县或需竞争清关的案例中有时更长（[[JapanFG/18-shinwa-bank|十八親和 銀行合併]] 因 公取委 审查延迟了 3  年）。
6. **FG 设立后的持续监管**：FSA 对控股公司与运营银行子公司一并检查；合并资本充足率报告在控股公司层面。

2021  银行法 改正 使地方银行投资于子公司（区域振兴、数字银行、咨询）略微容易了——这降低了使用控股公司结构进行非银行投资的运营摩擦。多个 FG（[[JapanFG/concordia-fg|横浜]]、[[JapanFG/fukuoka-fg|FFG]]、[[JapanFG/hokkoku-fhd|北國 / CCI]]、[[JapanFG/kiraboshi-fg|きらぼし]]）积极利用扩大的范围设立数字 / 咨询 / 风投子公司。

## Why consolidation accelerated post-2015

Multiple structural drivers compressed the regional-bank universe through holding-company consolidation:

- 东京-大阪走廊以外都道府县的**人口下降**减少了存款 / 贷款基础。
- **负利率环境**（BoJ 2016-）压缩了地方银行的净息差。
- 小型地方银行无法独立承担的**不断上升的合规 / AML / 网络安全 / 数字成本**。
- 扩展至集团级共享平台的**系统投资要求**。
- **公共政策推动**：FSA 监管姿态；2020 独禁特例法 减少同都道府县合并的反垄断障碍；2021 银行法 扩大子公司投资范围。
- **银行业劳动力市场的人口压力**：独立地方银行难以雇用专业人才。
- **中小企业客户的事业承继需求**：需要在集团层面更易扩展的咨询能力。
- 像 [[JapanFG/hokuhoku-fg|ほくほく FG]] 这样的**跨都道府县合作先例**证明了地理非相邻的可行性。

这些驱动因素的组合产生了本矩阵所记录的 FG 浪潮——多集中于 2014-2022。该浪潮仍在继续，[[JapanFG/fidea-hd|フィデア HD]] 计划 2027-01 銀行合併 作为下一个里程碑之一。

## Boundary cases and caveats

- **T&D HD (8795) ≠ TD HD / Daishi-Hokuetsu。** [[JapanFG/td-hd|T&D Holdings]] 是一家寿险控股公司（太陽 + 大同）。按用户规格，本矩阵使用「TD HD」作为指向 [[JapanFG/dai-shi-hokuetsu-fg|第四北越 FG]] 的标签以表示地方银行背景。除非另有明确说明，否则现实中对「TD HD」的引用几乎总是指寿险实体。
- **[[JapanFG/sapporo-hokuyo-bank|札幌北洋 (Sapporo Hokuyo)]] 现为 [[JapanFG/north-pacific-bank-hd|北洋 HD]] / [[JapanFG/ndfg|札幌北洋 HD lineage]] 下的 [[JapanFG/north-pacific-bank|北洋銀行]]**；历史上的 札幌 + 北海道 城市银行整合属于比本矩阵主焦点略早的一波，但该实体位于同一地方银行全集之中。
- **[[JapanFG/resona-hd|Resona HD]] vs 地方银行 FG。** [[JapanFG/resona-hd|Resona]] 在牌照分层矩阵中是大型银行边缘 / D-SIB 指定实体，但它将 [[JapanFG/kansai-mirai-bank|関西みらい銀行]] 与 [[JapanFG/minato-bank|みなと銀行]] 作为地方银行运营子公司经营。这意味着同一集团横跨大型银行层（母公司 [[JapanFG/resona-hd|Resona]] / [[JapanFG/resona-bank|Resona Bank]] / [[JapanFG/saitama-resona-bank|Saitama Resona Bank]]）与地方银行层（关西运营银行）。
- **[[JapanFG/sbi-regional-bank-hd|SBI 地銀 HD]] 并非对所有伙伴而言传统 FSA 意义上的银行控股公司。** 它是一个锚定于 [[JapanFG/sbi-shinsei-bank|SBI 新生銀行]]（自 2023-09 起为全资 普通銀行）的战略联盟与股权投资平台。合作的地方银行（筑邦 / 清水 / 大東 / 福島 / [[JapanFG/shimane-bank|島根]] / 仙台 / 筑波 / 東和 / きらやか）保留其独立的 FSA 持牌法律实体，并未在正常的银行控股公司基础上并入 SBI 地銀 HD。据公开披露（Wikipedia「SBI地銀ホールディングス」），SBI 持股比例因伙伴而差异极大——大致从低端的 1%（東和銀）到约 34%（福島銀）及顶级股东地位（島根銀 ~21%、じもと HD ~12%），还有数家伙伴（清水銀 ~2.5%、大東銀 ~2.8%、東北銀 ~2.9%、筑邦銀 ~2.9%）处于低个位数区间。
- **运营银行 vs 控股公司：名称重用。** [[JapanFG/concordia-fg|横浜 FG]] 是控股公司；[[JapanFG/yokohama-bank|横浜銀行]] 是其下的运营银行。[[JapanFG/dai-shi-hokuetsu-fg|第四北越 FG]] 是控股公司；[[JapanFG/daishi-hokuetsu-bank|第四北越銀行]] 是运营银行。称「横浜銀行」或「第四北越銀行」的引用指的是运营银行，而非 FG。
- **一県一行 vs 多银行伞。** 单独株式移転 单银行外壳与 共同株式移転 多银行伞不是同一商业模式。单银行外壳保留并购可选性与资本配置灵活性，但本身并不降低分行 / 系统成本；多银行伞在各组成银行日后法律合并时可带来系统 / 分行整合潜力。
- **悬而未决或计划中的合并步骤。** 若干 FG 正处弧线中段：[[JapanFG/fidea-hd|フィデア HD]] 计划于 2027-01 将 荘内 + 北都 → フィデア銀行；[[JapanFG/proclair-hd|プロクレア HD]] 已于 2025-01 完成 青森 + みちのく → 青森みちのく銀行。这些计划事件会随时间改变运营银行模式一列。
- **银行控股公司官方名单。** FSA 发布的完整 銀行持株会社一覧 PDF 是任一时点 FG 实体名单的权威来源。在引用本矩阵前，使用它来核实实体名称的规范性。
