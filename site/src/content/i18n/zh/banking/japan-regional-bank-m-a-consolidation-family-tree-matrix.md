# 日本地域银行 M&A 整合家谱矩阵

## TL;DR

1990  后日本的地域银行整合，产生了一张分层的银行控股公司地图，每家均合并了一家或多家第一地银（"地銀"）和／或第二地银（"第二地銀"）前身，部分情形下还包含一家信用合作组织／新銀行前身。本矩阵汇集了截至 2026 日本每一家实质活跃的地域银行 FG 的存续 FG 名称、FG 成立年份、前身银行、覆盖都道府县、资本规模及控股公司结构。它是 [[banking/regional-bank-consolidation-pattern|Regional bank consolidation pattern]]（缘由）与 [[banking/japan-banking-license-tier-comparison-matrix|Japan banking-license tier comparison matrix]]（属何种牌照）的家谱对应物。

## 为何这一家谱重要

地域银行整合是分层的。许多引用说「X銀行」时，并未指明说话者所指为：

- **原始前身银行**（如 第四銀行 1873-2020）；
- **合并后的存续银行**（如 [[regional-banks/daishi-hokuetsu-bank|第四北越銀行]]）；
- **位于运营银行之上的控股公司**（[[regional-banks/dai-shi-hokuetsu-fg|第四北越 FG]] 7327）；
- **控股公司所属的联盟／consortium**（如 TSUBASA、[[megabanks/sbi-regional-bank-hd|SBI 地銀 HD]]）。

没有明确的家谱地图，同一品牌名会被用于四个不同的法律实体。本矩阵将这四个层级强制纳入一个一致的视图，以便在做出任何资产负债表或战略主张之前，先回答「我实际在谈论哪个实体」这一问题。

具体而言，该矩阵呈现：

- 哪个 FG 是**单一都道府县单一银行**（如 [[regional-banks/iyogin-hd|いよぎん HD]]、[[regional-banks/shizuoka-fg|しずおか FG]]、[[regional-banks/hirogin-hd|ひろぎん HD]]、[[regional-banks/juroku-fg|十六 FG]]）——这些是 単独株式移転 模式，控股公司本质上是单一银行的外壳；
- 哪个 FG 是**跨都道府县两家或更多银行**（[[regional-banks/mebuki-fg|めぶき FG]]、[[regional-banks/concordia-fg|横浜 FG (旧コンコルディア)]]、[[regional-banks/kyushu-fg|九州 FG]]、[[regional-banks/fukuoka-fg|FFG]]、[[regional-banks/hokuhoku-fg|ほくほく FG]]）——这些是 共同株式移転 模式，跨都道府县界线合并银行；
- 哪个 FG 由**救援式 M&A**形成（[[regional-banks/jimoto-hd|じもと HD]] 于 2024-06 获得 RCC 多数；[[regional-banks/tomony-hd|トモニ HD]] 吸收 大正銀行 2017；[[regional-banks/mebuki-fg|めぶき FG]] 加入经 Resona / DICJ 处置的 足利銀行 谱系）；
- 哪个 FG 是**第一地银 + 第二地银**的混合，相对于一对同层级银行；
- 哪个 FG 属于**更广泛的 consortium**（[[megabanks/sbi-regional-bank-hd|SBI 地銀 HD]] 股权联盟环；围绕 [[regional-banks/kansai-mirai-bank|関西みらい銀行]] 与 [[regional-banks/minato-bank|みなと銀行]] 的 Resona 集团群）。

这一家谱直接馈入牌照层级矩阵 [[banking/japan-banking-license-tier-comparison-matrix|Japan banking-license tier comparison matrix]]（此处每一行均处于 Tier 2 第一地銀 或 Tier 3 第二地銀）以及合作银行矩阵 [[banking/japan-cooperative-banking-unified-federation-matrix|Japan cooperative-banking unified federation matrix]]（合作银行的整合与此处描述的控股公司路径处于不同的轴上）。

## 如何阅读本矩阵

1. **先选取 FG（控股公司）行。** FG 是发行底层银行持股、签署 M&A 文件、并在 FSA 银行控股公司监督下报告合并资本充足率的法律实体。这是战略得以执行的单位。
2. **阅读前身银行列。** 这会告诉你 FG 是单一银行外壳（如 [[regional-banks/iyogin-hd|いよぎん HD]]、[[regional-banks/shizuoka-fg|しずおか FG]]）还是真正的多银行整合（[[regional-banks/fukuoka-fg|FFG]]、[[regional-banks/concordia-fg|横浜 FG]]）。
3. **检查运营银行模式列。** 一个 2-bank umbrella 在一家 FG 下保留两家运营银行存续；一个 1-operating-bank 模式意味着构成银行已在法律上合并（[[regional-banks/daishi-hokuetsu-bank|第四北越銀行]] 在 [[regional-banks/dai-shi-hokuetsu-fg|DHFG]] 之下）。
4. **检查覆盖都道府县列。** 单一都道府县 FG（[[regional-banks/shizuoka-fg|しずおか]]、[[regional-banks/hirogin-hd|ひろぎん]]、[[regional-banks/juroku-fg|十六]]）是本地平台打法；跨都道府县 FG（[[regional-banks/mebuki-fg|めぶき]]、[[regional-banks/hokuhoku-fg|ほくほく]]、[[regional-banks/kyushu-fg|九州]]）是区域平台打法；全国覆盖 FG（[[megabanks/sbi-regional-bank-hd|SBI 地銀 HD]] consortium）是数字平台打法。
5. **与 [[banking/regional-bank-api-digital-partnership-route|regional-bank API and digital partnership route]] 交叉核对**，以了解同样这些 FG 在数字轨道轴上的位置（[[regional-banks/fukuoka-fg|FFG]] 的 みんなの銀行 BaaS、[[regional-banks/hokkoku-fhd|CCI / Hokkoku]] 的 Tonari、[[regional-banks/kiraboshi-fg|きらぼし]] 的 UI 銀行）。

## 作为转折点的关键整合事件

- **2003-09 → 2004-09 [[regional-banks/hokuhoku-fg|ほくほく FG]] (北陸 + 北海道).** 日本银行业史上首个「離れた地域の地銀どうしの広域連合」模式。
- **2003-11-29 足利銀行 破綻 → 一時国有化 → 2008-07 民間譲渡 (野村ファンド + ネクスト・キャピタル等) → 2008-12 足利 HD → 2016-10 [[regional-banks/mebuki-fg|めぶき FG]] 合流.** 破产后的重整弧线，后在 FG 伞下与一家健康伙伴合并。
- **2007-04 [[regional-banks/fukuoka-fg|FFG]] 発足 (福岡 + 熊本ファミリー).** 为 1地銀 主导的跨都道府县 FG 树立了模板。
- **2009-10 [[regional-banks/fidea-hd|フィデア HD]] (荘内 + 北都).** 東日本 首个跨都道府县 FG。
- **2010-04 [[regional-banks/tomony-hd|トモニ HD]] (徳島 + 香川).** 第二地银跨都道府县 FG，后经 2017 大正銀 救援扩大 → 2020-01 徳島大正銀行 誕生。
- **2014-10 → 2016-04 → 2018-05 [[regional-banks/kiraboshi-fg|東京きらぼし FG]].** 都民 + 八千代 → +新銀行東京 → 3行合併（合并了 都政の新銀行東京 结局的东京都市圈特例）。
- **2015-10 [[regional-banks/kyushu-fg|九州 FG]] (肥後 + 鹿児島).** 九州南部的跨都道府县 2強連合。
- **2016-04 [[regional-banks/concordia-fg|横浜 FG (旧コンコルディア)]] (横浜 + 東日本).** 为最大单一 FG 类别而设的跨层级（1地銀 + 2地銀）跨都道府县伞。
- **2016-10 [[regional-banks/mebuki-fg|めぶき FG]] (常陽 + 足利 HD).** 与一家破产处置后伙伴的跨都道府县组合。
- **2018-04 → 2021-05 [[regional-banks/san-jusan-fg|三十三 FG]] (三重 + 第三 → 三十三銀行).** 2地銀 + 2地銀 的完全法律合并。
- **2018-10 → 2021-01 [[regional-banks/dai-shi-hokuetsu-fg|第四北越 FG]] (第四 + 北越 → 第四北越銀行).** 新潟 的新「一県一行」模式。
- **2020-12 [[megabanks/sbi-regional-bank-hd|SBI 地銀 HD]] 設立.** 「第4のメガバンク構想」非 FG 联盟模式。
- **2022-04 → 2025-01 [[regional-banks/proclair-hd|プロクレア HD]] (青森 + みちのく).** 为清理都道府县内垄断而**独禁特例法 (令和2年法律第32号) 初適用**。
- **2024-06-20 [[regional-banks/jimoto-hd|じもと HD]] RCC 議決権 63.52% 取得.** 对该 2地銀 FG 的公共资金监督强化。
- **2025-10 [[regional-banks/concordia-fg|横浜 FG]] 商号変更**（旧 コンコルディア → 横浜 FG）。
- **2025-10 [[regional-banks/hokkoku-fhd|北國 FHD]] → [[financial-regulators/cci-group|CCI グループ]] 商号変更.** 数字平台转向在公司名称中得到体现。

这些联盟使家谱图景复杂化：一家地域银行可同时身处一个 FG 中以及一个或多个跨 FG 联盟中。TSUBASA Alliance 尤其横跨多家 FG（千葉銀 保持独立、[[regional-banks/dai-shi-hokuetsu-fg|第四北越]] FG、北洋 集团等），是叠加在 FG 层之上的重要系统成本层。

## 控股公司的 FSA 许可流程

依《银行法》的 FSA 银行控股公司许可涉及：

1. **《银行法》事前批准**，用于设立新的银行控股公司（无自动权利；FSA 审查）。
2. **资本充足率**，要求按合并基础（Basel III 合并报告；视是否国际活跃而采用 BIS／国内标准）。
3. **治理审查**，包括控股公司董事会结构、内部审计框架、集团风险管理。
4. **集团业务战略**披露，包括各子公司银行的角色。
5. **完成期限**：通常从公告到 FG 设立为 6-12  个月；在跨都道府县或竞争审查通过的案例中有时更长（[[regional-banks/18-shinwa-bank|十八親和 銀行合併]] 因 公取委 审查延迟了 3  年）。
6. **FG 设立后的持续监督**：FSA 一并检查控股公司与运营银行子公司；合并资本充足率报告在控股公司层面进行。

2021 年《银行法》改正 略微降低了地域银行投资子公司（区域振兴、数字银行、咨询）的难度——这降低了利用控股公司结构进行非银行投资的运营摩擦。多家 FG（[[regional-banks/concordia-fg|横浜]]、[[regional-banks/fukuoka-fg|FFG]]、[[regional-banks/hokkoku-fhd|北國 / CCI]]、[[regional-banks/kiraboshi-fg|きらぼし]]）已积极利用扩大后的范围设立数字／咨询／风险投资子公司。

## 为何整合在 2015

 后加速 多重结构性驱动因素通过控股公司整合压缩了地域银行群：

- 东京—大阪走廊以外都道府县的**人口下降**减少了存款／贷款基础。
- **负利率环境**（BoJ 2016-）压缩了地域银行的净息差。
- 小型地域银行无法独自承担的**不断上升的合规／AML／网络／数字化成本**。
- 需要扩展至集团级共享平台的**系统投资要求**。
- **公共政策推动**：FSA 监督姿态；2020 独禁特例法 降低同都道府县合并的反垄断障碍；2021 《银行法》扩大子公司投资范围。
- **银行劳动力市场的人口压力**：独立地域银行难以招聘专业人才。
- **SME 客户的业务承继需求**：需要在集团层面更易扩展的咨询能力。
- [[regional-banks/hokuhoku-fg|ほくほく FG]] 等**跨都道府县合作先例**证明了地理非相邻的可行性。

这些驱动因素的组合产生了本矩阵所记录的 FG 浪潮——最集中于 2014-2022。该浪潮仍在持续，[[regional-banks/fidea-hd|フィデア HD]] 计划 2027-01 銀行合併 作为下一个里程碑之一。

## 边界案例与注意事项

- **T&D HD (8795) ≠ TD HD / Daishi-Hokuetsu.** [[life-insurers/td-hd|T&D Holdings]] 是一家人寿保险控股公司（太陽 + 大同）。依用户规范，本矩阵以「TD HD」作为指向 [[regional-banks/dai-shi-hokuetsu-fg|第四北越 FG]] 的地域银行语境标签。现实世界对「TD HD」的引用，除非另有明确说明，几乎总是指人寿保险实体。
- **[[regional-banks/sapporo-hokuyo-bank|札幌北洋 (Sapporo Hokuyo)]] 现为 [[regional-banks/north-pacific-bank-hd|北洋 HD]] / [[megabanks/ndfg|札幌北洋 HD lineage]] 旗下的 [[regional-banks/north-pacific-bank|北洋銀行]]**；历史上的札幌 + 北海道城市银行整合属于略早于本矩阵主要关注的浪潮，但该实体处于同一地域银行群。
- **[[megabanks/resona-hd|Resona HD]] vs 地域银行 FG.** [[megabanks/resona-hd|Resona]] 在牌照层级矩阵中是大型银行边缘／D-SIB 指定实体，但它运营 [[regional-banks/kansai-mirai-bank|関西みらい銀行]] 与 [[regional-banks/minato-bank|みなと銀行]] 作为地域银行运营子公司。这意味着同一集团横跨大型银行层（母公司 [[megabanks/resona-hd|Resona]] / [[megabanks/resona-bank|Resona Bank]] / [[megabanks/saitama-resona-bank|Saitama Resona Bank]]）与地域银行层（关西运营银行）。
- **[[megabanks/sbi-regional-bank-hd|SBI 地銀 HD]] 对所有伙伴而言并非传统 FSA 意义上的银行控股公司。** 它是一个锚定于 [[regional-banks/sbi-shinsei-bank|SBI 新生銀行]]（在法律上自 2023-09 起为全资的 普通銀行）的战略联盟与股权投资平台。伙伴地域银行（筑邦 / 清水 / 大東 / 福島 / [[regional-banks/shimane-bank|島根]] / 仙台 / 筑波 / 東和 / きらやか）保留其独立的 FSA 持牌法律实体，并未以正常的银行控股公司基础并入 SBI 地銀 HD。据公开披露（Wikipedia「SBI地銀ホールディングス」），SBI 的持股比例因伙伴而异——大致从低端的 1%（東和銀）到高至 ~34%（福島銀）及第一大股东地位（島根銀 ~21%、じもと HD ~12%），数家伙伴（清水銀 ~2.5%、大東銀 ~2.8%、東北銀 ~2.9%、筑邦銀 ~2.9%）处于低个位数区间。
- **运营银行 vs 控股公司：名称重用。** [[regional-banks/concordia-fg|横浜 FG]] 是控股公司；[[regional-banks/yokohama-bank|横浜銀行]] 是其旗下的运营银行。[[regional-banks/dai-shi-hokuetsu-fg|第四北越 FG]] 是控股公司；[[regional-banks/daishi-hokuetsu-bank|第四北越銀行]] 是运营银行。说「横浜銀行」或「第四北越銀行」的引用指的是运营银行，而非 FG。
- **一县一行 vs 多银行伞。** 一个 単独株式移転 单一银行外壳与一个 共同株式移転 多银行伞并非同一种业务模式。单一银行外壳保留 M&A 选择权与资本配置灵活性，但其本身并不降低网点／系统成本；多银行伞若构成银行日后法律合并，则能带来系统／网点整合潜力。
- **待定或计划中的合并步骤。** 数家 FG 正处于弧线中段：[[regional-banks/fidea-hd|フィデア HD]] 计划于 2027-01 进行 荘内 + 北都 → フィデア銀行；[[regional-banks/proclair-hd|プロクレア HD]] 已于 2025-01 完成 青森 + みちのく → 青森みちのく銀行。这些计划事件会随时间改变运营银行模式列。
- **银行控股公司官方清单。** 完整的 FSA 发布的 銀行持株会社一覧 PDF 是任何时点 FG 实体清单的权威来源。引用本矩阵前，请用它核实实体名称的规范性。
