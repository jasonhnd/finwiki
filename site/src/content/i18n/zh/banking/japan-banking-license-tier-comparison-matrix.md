# 日本银行牌照层级比较矩阵

## TL;DR

日本的吸收存款体系并非单一牌照。银行法、信托业法、信用金库法、中小企业等协同组合法、劳动金库法、农林中央金库法各自划分出不同的牌照层级，具有不同的法令、监管当局、资本下限、存款保险制度、许可业务范围与治理形态。本矩阵汇集了横跨 **大型银行 / 地方 / 第二地方 / 信托 / 网络 / 外国银行支店 / 信用金库 / 信用组合 / 劳金 / 农林中央金库-JA-JF** 的可公开验证轴线，以便在品牌或资产负债表问题之前先回答牌照形态问题。

## Wiki route

本条目归属于 [[banking/INDEX|banking index]]，与协同组合金融通道 [[banking/cooperative-banking-japan|Cooperative banking in Japan]]、数字银行通道 [[banking/japan-net-bank-competition-map|Japan net bank competition map]]、外国银行支店通道 [[banking/foreign-bank-japan-retreat|Foreign-bank Japan retreat]]、地方银行通道 [[banking/regional-bank-consolidation-pattern|Regional bank consolidation pattern]] 配对。监管边界为 [[financial-licenses/INDEX|JapanFG legal / financial licenses]] 内的 [[financial-licenses/bank-license-and-baas-boundary|bank license and BaaS boundary]]。存款保险体系为 [[financial-regulators/yokin-hoken-kiko|預金保険機構 (DICJ)]]。协同组织中央机关为 [[cooperative-banks/norinchukin|農林中央金庫]] 与 [[cooperative-banks/shinkin-central|信金中央金庫]]。

## 为何层级矩阵很重要

"日本的银行"这样一个单一短语，掩盖了牌照层级控制着对分析而言几乎所有重要公开表层字段这一事实：

- 该机构归属于哪部法令之下；
- 哪个监管当局检查它（FSA，劳金由厚劳省共同、JA / JF / 农林中央金库由农水省共同）；
- 存款保险对手方是 [[financial-regulators/yokin-hoken-kiko|DICJ]]，还是 JA共济 / 协同组织保护制度；
- 该机构能否持有证券牌照、信托牌照，还是仅普通银行业务；
- 能否在银行控股公司内部法人化；
- 作为对手方是否拥有对日银的直接 [[banking/INDEX|banking-domain]] 接入；
- 以及会员资格 / 地理是否为严格的法律约束。

没有牌照层级，[[banking/japan-net-bank-competition-map|net bank]] 看起来就像又一家零售银行，[[banking/shinkin-bank-registry-japan|shinkin]] 看起来就像又一家小型地方银行，[[banking/foreign-bank-japan-retreat|foreign branch]] 看起来就像又一个批发玩家。

## 层级 1  — 大型银行（普通银行 / 城市银行）

- **法令 / 监管当局。** 银行法 普通银行牌照；由 FSA 主要行等・银行控股公司室监督，并作为货币政策对手方接受日银现场监测。
- **牌照 + 数量。** 银行法 普通银行。SMBC / MUFG / 瑞穗三大核心加上里索纳 / 埼玉里索纳，作为大型银行通道位于 [[banking/INDEX|banking index]] 内部。
- **资本下限。** 银行法规定普通银行的最低实缴资本为 ¥2  亿日元；大型银行的运营水平远高于此，并以 BIS 巴塞尔 III 集团资本运营 — 普通股权一级 Tier 1  ≥ 4.5%，Tier 1  ≥ 6%，总资本充足率 ≥ 8%，外加 G-SIB / D-SIB 资本缓冲附加层级（MUFG、SMFG、瑞穗承担 G-SIB 附加；里索纳被视为 D-SIB）。
- **流动性。** 完整的巴塞尔 III LCR ≥ 100% 与 NSFR ≥ 100% 在合并基础上适用。
- **存款保险。** [[financial-regulators/yokin-hoken-kiko|DICJ]] 承保；标准存款保险框架下本金 ¥10  万日元加利息，结算用存款全额保护。
- **许可业务。** 存款、贷款、FX、结算、债券与国债承销、通过子公司开展的证券关联业务、衍生品、托管、资产运用、租赁、消费金融、控股公司战略职能。
- **跨境。** 完整国际银行业务：海外支店、外国银行子公司、USD / EUR 清算接入、跨境贷款、项目 / 贸易融资、互换额度资格。
- **治理。** TSE 优质上市的股份公司；银行控股公司结构获许可并被采用（MUFG、SMFG、瑞穗 FG、里索纳 HD）。
- **地理。** 无法定地理限制；全国支店与海外存在。
- **M&A / 控股公司。** 经 FSA 事先批准许可银行控股公司；银行法下交叉持股与整合路径开放。
- **日银对手方。** 直接日银往来账户；JGB 的一级交易商；接入日银市场操作。
- **近期监管变更。** 银行法 改正（令和 3  / 2021）扩大了银行关联公司范围与数字 / 地区振兴投资范围；围绕银行发行数字货币与电子结算等取扱业的更新对此层影响最直接。参见 [[financial-licenses/bank-license-and-baas-boundary|bank license and BaaS boundary]]。

## 层级 2  — 地方银行（地方银行）

- **法令 / 监管当局。** 银行法 普通银行牌照；在 FSA 的中小・地区金融机构监督指针下监督；许多地方银行在都道府县 FSA 财务局介入时也落入 [[banking/regional-bank-consolidation-pattern|regional bank consolidation pattern]]。
- **牌照 + 数量。** 银行法 普通银行牌照，第一地方银行协会 会员；全银协的分组将其视为位于大型银行与第二地方银行之间的"地方银行"层级。
- **资本下限。** 同样的银行法 ¥2  亿日元最低额；大多采用 BIS 标准法，仅最大者采用 IRB；无国际业务的银行适用国内基准 4% 核心资本比率。
- **流动性。** 对国际活跃银行适用 LCR / NSFR；国内基准银行在 FSA 国内框架下采用更简单的流动性要求。
- **存款保险。** 与大型银行相同条件的 [[financial-regulators/yokin-hoken-kiko|DICJ]] 承保。
- **许可业务。** 存款、贷款、FX、结算、通过子公司开展的证券关联业务；通过单独牌照开展信托业务（部分地方银行持有信托附随职能）。
- **跨境。** 有限；较大的地方银行运营海外代表处与部分支店，但大多以国内为中心。
- **治理。** 股份公司；许多为 TSE 上市；银行控股公司结构日益普遍（[[banking/regional-bank-consolidation-pattern|regional bank consolidation pattern]]）。
- **地理。** 无法律地理限制，但特许经营集中于一至数个都道府县。
- **M&A / 控股公司。** 经 FSA 批准许可银行控股公司；FSA 经营统合框架下的整合是主导 M&A 路径。
- **近期监管变更。** 2021  银行法 改正扩大了许可关联公司业务，并使地方银行更易于投资地区振兴、数字银行子公司与咨询子公司 — 对 [[banking/regional-bank-consolidation-pattern|regional bank consolidation pattern]] 中追踪的整合弧线至关重要。

## 层级 3  — 第二地方银行（第二地方银行）

- **法令 / 监管当局。** 银行法 普通银行牌照；相同的 FSA 中小・地区金融机构监督指针。在法律上与地方银行无法区分 —"第二"标签是行业团体会员资格。
- **牌照 + 数量。** 银行法 普通银行牌照，第二地方银行协会 会员；历史上是在 1989  改革下从相互银行（mutual savings banks）向普通银行的转换。
- **资本下限。** 银行法最低额与地方银行相同；BIS 制度相同。
- **流动性。** 与普通地方银行相同；对非国际活跃银行采用国内基准流动性。
- **存款保险。** 完整银行法条件下的 [[financial-regulators/yokin-hoken-kiko|DICJ]] 承保。
- **许可业务。** 与地方银行相同；实务上特许经营更偏 SME，资产负债表更小。
- **跨境。** 极少；几乎完全是国内特许经营。
- **治理。** 股份公司；许多情况下为 TSE 上市；许可银行控股公司。
- **地理。** 单一都道府县特许经营比第一层级地方银行更典型。
- **M&A / 控股公司。** [[banking/regional-bank-consolidation-pattern|regional bank consolidation pattern]] 中描述的整合流程常将第一层级地方银行与同一都道府县的第二地方银行配对于一个控股公司内部。
- **近期监管变更。** 与普通地方银行相同的 2021  银行法 改正范围；FSA 对整合的支持措施（系统成本补贴、治理灵活性）在此层级是有意义的推动。

## 层级 4  — 信托银行（信托银行）

- **法令 / 监管当局。** 双重牌照：银行法 普通银行牌照 **加上** FSA 之下的信托业法 信托业务牌照。这种双重牌照形态使信托银行在法律上区别于普通银行。
- **牌照 + 数量。** 公开会员资格被捕捉于信托协会的信托银行名册中。运营公司锚点包括 [[banking/INDEX|banking index]] 信托银行路径加上 JapanFG 的信托银行页面；运营比较表层为 [[banking/trust-bank-custody-operating-comparison|trust-bank custody operating comparison]]。
- **资本下限。** 适用银行法最低额；额外的信托业法能力与运营要求（分别信托账户、受托义务）叠加于其上。
- **流动性。** 银行业务账簿侧适用银行法 LCR / NSFR；信托资产在单独受托会计下为表外。
- **存款保险。** 银行业务账簿存款适用 [[financial-regulators/yokin-hoken-kiko|DICJ]] 承保；信托账户在同一 DICJ 框架下，对受保本金金钱信托通过元本补填信托路径受到保护。
- **许可业务。** 银行业务（存款、贷款、FX、结算）**加上** 信托业务（金钱信托、证券投资信托、年金信托、托管、证券结算、不动产信托、资产管理）。双重牌照层级在吸收存款领域拥有最广的业务范围。
- **跨境。** 大型银行信托部门（MUFJ 信托、SMBC 信托、瑞穗信托）运营全球托管与资产管理网络；专门资产管理银行（日本万事达信托银行、日本托管银行）运营机构基础设施层。
- **治理。** 股份公司；几乎总是在银行控股公司之下。
- **地理。** 无法定限制；全国。
- **M&A / 控股公司。** 许可银行控股公司；信托银行子公司常与大型银行集团内的普通银行子公司并列。
- **日银对手方。** 直接日银往来账户；托管银行信托部门参与日银网 / JGB 结算基础设施。
- **近期监管变更。** FSA 对受托义务原则的扩展、资产运用业务改善命令，以及关于信托业务改革的 [[financial-licenses/INDEX|legal-financial-licenses]] 轨道重塑了运营范围；电子结算等取扱业下的银行发行数字货币中介也落到银行侧信托中介上。

## 层级 5  — 网络银行（网络银行）

- **法令 / 监管当局。** 银行法 普通银行牌照。**无单独法定类别** —"网络银行"是运营性的，而非法律性的。牌照与大型银行的相同。
- **牌照 + 数量。** 银行法 普通银行牌照；运营领域在 [[banking/japan-net-bank-competition-map|Japan net bank competition map]] 中追踪（生态系统零售、全银行 BaaS、法人 API、资产形成、地区数字）。
- **资本下限。** 银行法最低资本；大多采用 BIS 标准法，许多采用国内基准 4%；部分网络银行（乐天银行、住信 SBI 网络银行）已成长为中型资产负债表。
- **流动性。** 标准银行法 LCR / NSFR 规则，对非国际活跃运营者采用国内基准。
- **存款保险。** 完整银行法条件下的 [[financial-regulators/yokin-hoken-kiko|DICJ]] 承保。
- **许可业务。** 完整普通银行范围；部分为伙伴运营全银行 BaaS（[[banking/japan-baas-operating-models|Japan BaaS operating models]]），部分运营法人 API 银行业务，部分运营证券邻接的零售银行业务。
- **跨境。** 有限；FX 零售与海外汇款常见，完整海外支店罕见。
- **治理。** 股份公司；许可银行控股公司；许多网络银行位于金融科技 / 电商 / 电信 / 证券母集团内部，而非银行控股公司之下。
- **地理。** 无传统意义上的支店网络；全国数字渠道。
- **M&A / 控股公司。** 许可银行控股公司；运营模式问题（母生态系统、伙伴 BaaS、证券邻接性）在 [[banking/japan-net-bank-competition-map|Japan net bank competition map]] 中。
- **日银对手方。** 一旦作为普通银行获得牌照即直接日银往来账户。
- **近期监管变更。** 2021  银行法 改正的银行关联公司范围扩展是对此层最直接重要的变更之一；电子结算等取扱业与 [[financial-licenses/bank-license-and-baas-boundary|bank license and BaaS boundary]] 中追踪的 BaaS 边界监督影响运营自由度而非牌照形态。

## 层级 6  — 外国银行支店（外国银行支店）

- **法令 / 监管当局。** 银行法第 IX 章下的银行法 外国银行支店牌照；在 FSA 主要行等与外国银行监督路径下监督。
- **牌照 + 数量。** 银行法 外国银行支店牌照；FSA 公布外国银行支店清单并定期更新。该模式在 [[banking/foreign-bank-japan-retreat|Foreign-bank Japan retreat]] 中追踪。
- **资本下限。** 适用支店资本等价规则（资产等价处理、依赖本店资本充足）；并非国内法人化银行通常的 ¥2  亿日元实缴资本下限。本店的巴塞尔 III 地位很重要。
- **流动性。** 支店特定流动性期望；本店 LCR / NSFR 通过本国监管者框架传导，在国内业务重要时附加 FSA 上乘要求。
- **存款保险。** [[financial-regulators/yokin-hoken-kiko|DICJ]] 承保 **不会** 像对日本银行那样自动适用于外国银行支店 — 在银行法第 IX 章下，外国银行支店存款不受日本 DICJ 体系承保；存款人依赖本国保护与支店资产 / 日本资产质押规则。
- **许可业务。** 银行业务、FX、通过持牌子公司的证券关联；业务限于牌照所授权者加上受日本监管等价性约束的本店能力。
- **跨境。** 原生；牌照的全部要点在于支持法人、批发、托管、市场、FX、贸易与亚洲走廊业务 — 参见 [[banking/foreign-bank-japan-retreat|Foreign-bank Japan retreat]] 中的板块地图。
- **治理。** 海外法人实体的支店；并非日本股份公司；由本店董事会与 FSA 期望下的日本支店治理所治理。
- **地理。** 通常为东京 / 大阪的一至数个支店。
- **M&A / 控股公司。** 不能在日本银行控股公司内部作为母公司化；重组通常经本店进行。
- **日银对手方。** 外国银行支店可持有日银往来账户并参与日银网；并非所有外国支店都如此。
- **近期监管变更。** FSA 围绕 AML、可疑交易监测、治理等价性与本店风险控制的外国银行支店监督指针更新仍是活跃边界。

## 层级 7  — 信用金库（信用金库）

- **法令 / 监管当局。** 信用金库法；在 FSA 的中小・地区金融机构监督指针（关于协同组织金融机构的第 V 编）与财务局 / 福冈财务支局 / 冲绳综合事务局路径下监督。
- **牌照 + 数量。** 信用金库法 牌照；据 [[banking/shinkin-bank-registry-japan|Japan shinkin bank registry]] 有 254  家持牌信用金库，加上中央机关 [[cooperative-banks/shinkin-central|信金中央金庫]]。
- **资本下限。** 信用金库法 普通信用金库的最低 出资金 为 ¥100  万日元（指定地区为 ¥200  万日元）；BIS 式资本充足为 FSA 国内基准 4% 核心资本框架。
- **流动性。** FSA 国内基准流动性期望；来自 [[cooperative-banks/shinkin-central|信金中央金庫]] 的中央机关流动性支持。
- **存款保险。** [[financial-regulators/yokin-hoken-kiko|DICJ]] 承保以相同的银行法等价条件适用。
- **许可业务。** 会员存款、会员贷款、结算、FX、证券投资；对非会员贷款按比例限制（50% 非会员上限为历史锚点，有豁免）。
- **跨境。** 一般无；部分较大信用金库通过代理行路径运营适度的 FX / 汇款。
- **治理。** 协同组织 — 基于会员、一人一票治理，以 出资（会员出资）而非可交易股权。
- **地理。** 与市区町村 / 都道府县挂钩的法定营业地区；会员须在该地区居住、工作或经营业务。
- **M&A / 控股公司。** 无银行控股公司路径；合并在信用金库法手续与 FSA 批准下于信用金库间进行；跨信用金库联盟经 [[cooperative-banks/shinkin-central|信金中央金庫]] 运营。
- **日银对手方。** [[cooperative-banks/shinkin-central|信金中央金庫]] 是信用金库体系的日银对手方；各信用金库经信金中央金库清算。
- **近期监管变更。** 扩大关联业务范围的信用金库法修订与映射银行法扩展的 2021 时代协同组合金融改革是主要更新；背景参见 [[banking/cooperative-banking-japan|Cooperative banking in Japan]]。

## 层级 8  — 信用组合（信用组合）

- **法令 / 监管当局。** 中小企业等协同组合法 加上 协同组合金融事业相关法律；由 FSA 中小・地区金融机构路径与都道府县当局协调监督。牌照授予信用协同组合。
- **牌照 + 数量。** 据 [[banking/credit-cooperative-registry-japan|Japan credit cooperative registry]] 有 143  家持牌信用组合，加上中央机关 全国信用协同组合连合会。
- **资本下限。** 协同组合法最低 出资金 加 FSA 国内基准资本充足；下限小于信用金库。
- **流动性。** FSA 国内基准流动性；来自全信组连的中央机关流动性支持。
- **存款保险。** 适用 [[financial-regulators/yokin-hoken-kiko|DICJ]] 承保。
- **许可业务。** 会员存款、会员贷款、FX、结算；比信用金库更严格的会员资格规则（会员按业种、地区、职域或属性受限 — 业域 / 地域 / 职域）。
- **跨境。** 一般无。
- **治理。** 一人一票与 出资 会员出资的协同组织。
- **地理。** 法定营业地区；许多组合服务于特定职业或业种而非都道府县（例如历史上的商工组合中央金库、在日韩国 / 中国系商工组合、运输业组合）。
- **M&A / 控股公司。** 无银行控股公司路径；信用组合间合并在中小企业等协同组合法手续下许可。
- **日银对手方。** 全信组连 是信用组合体系的日银对手方。
- **近期监管变更。** 与信用金库相同的协同组合金融改革范围；关联业务范围扩展使地区振兴 / 咨询 / 数字子公司投资比 2021 之前稍易。

## 层级 9  — 劳金（劳动金库）

- **法令 / 监管当局。** 劳动金库法；由 FSA 与 厚生劳动省 **共同监督**。这种双重监管当局形态在吸收存款领域独一无二。
- **牌照 + 数量。** 据 [[banking/rokin-bank-registry-japan|Japan rokin bank registry]] 有 13  家持牌劳金（中央劳动金库、近畿劳动金库、东海劳动金库等地区劳金），加上中央机关 劳动金库连合会。
- **资本下限。** 劳动金库法最低 出资金；适用 FSA 国内基准资本充足。
- **流动性。** FSA 国内基准流动性；来自劳金连的中央机关流动性支持。
- **存款保险。** 适用 [[financial-regulators/yokin-hoken-kiko|DICJ]] 承保。
- **许可业务。** 会员存款、贷款、抵押贷款、结算；会员为劳动组合、消费生活协同组合及隶属于会员组织者。贷款集中于家庭 / 劳动者金融（抵押贷款、教育贷款、人生事件金融）。
- **跨境。** 无有意义规模者。
- **治理。** 协同组织；会员组织而非个人拥有劳金；劳动组合代表是结构性的。
- **地理。** 地区营业地区；13  家地区劳金共同覆盖日本而不重叠。
- **M&A / 控股公司。** 无银行控股公司路径；劳金可在劳动金库法手续下合并。
- **日银对手方。** 劳金连 是劳金体系的日银对手方。
- **近期监管变更。** FSA / 厚劳省 共同监督意味着除通常的 FSA 银行监督更新外，职场与消费者保护更新也渗入劳金监督。

## 层级 10  — 农林中央金库 / JA / JF 体系（农林系统金融）

- **法令 / 监管当局。** 农林中央金库法（针对农林中央金库）；农业协同组合法（针对 JA / 信连信用事业）；水产业协同组合法（针对 JF / 信渔连信用事业）。由 FSA 与 农林水产省 **共同监督** — 又一个双重监管当局制度。
- **牌照 + 数量。** [[cooperative-banks/norinchukin|農林中央金庫]] 作为中央机关位于顶点。其下：都道府县信连（JA 信用联合会）与信渔连（JF 信用联合会），以及底层信用事业 JA 与 JF 组合。JA Bank 体系与 JF 海洋银行体系路径参见 [[banking/INDEX|banking index]]。
- **资本下限。** 农林中央金库法为作为特殊法人的 [[cooperative-banks/norinchukin|農林中央金庫]] 设定特别框架；会员组合在各自协同组合法下运营。
- **流动性。** 对 [[cooperative-banks/norinchukin|農林中央金庫]] 适用银行等价审慎框架（鉴于其市场活动的 BIS 式资本与流动性）；会员机构以协同组合法审慎标准运营。
- **存款保险。** [[financial-regulators/yokin-hoken-kiko|DICJ]] 一般通过协同组合金融路径承保 JA / JF 存款等价请求权；农水产业协同组合储金保险机构（农水产业协同组合储金保险机构）为农业 / 渔业协同组合储金提供与银行 DICJ 轨道分离的平行保险机制。
- **许可业务。** 会员存款 / 储金（在 JA / JF 中技术上称 储金 而非 预金）、贷款、结算、FX（通过 [[cooperative-banks/norinchukin|農林中央金庫]]）、大规模证券投资、资产运用。[[cooperative-banks/norinchukin|農林中央金庫]] 按授权是全球活跃的固定收益 / 结构化信用投资者，在日本吸收存款机构中极为独特。
- **跨境。** [[cooperative-banks/norinchukin|農林中央金庫]] 运营大型海外投资组合并拥有海外代表处；会员 JA / JF 组合则无。
- **治理。** JA / JF 层面为协同组织；[[cooperative-banks/norinchukin|農林中央金庫]] 层面为特殊法人。会员组织治理与农业政策接近性是制度设计的一部分。
- **地理。** JA / JF 组合本地化于其市区町村 / 都道府县地区；信连 / 信渔连为都道府县单位；[[cooperative-banks/norinchukin|農林中央金庫]] 为全国。
- **M&A / 控股公司。** 无普通银行控股公司路径；体系整合经协同组合法合并、联合会重组与中央机关重组进行。
- **日银对手方。** [[cooperative-banks/norinchukin|農林中央金庫]] 作为体系中央机关持有对日银往来账户的直接接入。
- **近期监管变更。** 围绕资本结构 / 监管资本的农林中央金库法修订、关于市场风险与集中管理的 FSA-MAFF 共同监督更新，以及在 [[cooperative-banks/norinchukin|農林中央金庫]] 页面追踪的关于组合损失 / 重组的持续 2024  后披露。

## 如何阅读本矩阵

牌照层级矩阵是一个公开表层工具。它不为机构排名，不估计资产负债表健康度，也不预测整合。它的存在是为了在询问战略、渠道或特许经营问题之前，能够一致地分类 [[banking/INDEX|banking index]] 条目、[[banking/japan-net-bank-competition-map|net bank]] 条目或 [[banking/foreign-bank-japan-retreat|foreign-branch]] 条目。

阅读任何单一机构页面时：

1. 从 **法令** 列开始 — 它决定几乎所有其他字段。
2. 检查 **存款保险** 列。外国支店与 农水产业 协同组合是默认 [[financial-regulators/yokin-hoken-kiko|DICJ]] 框架的例外。
3. 检查 **日银对手方** 列。协同组合金融机构经中央机关而非直接清算。
4. 检查 **控股公司** 列。协同组合层级不能使用银行控股公司；外国支店也不能使用。
5. 使用 **治理** 列判读一人一票、股份公司上市，还是外国法人实体支店的措辞适用 — 它决定可用哪些治理、披露与 M&A 表层。

## 边界情形与注意点

- **网络银行 vs 普通银行。** 网络银行在运营上不同，但在法律上与任何其他普通银行相同。牌照层级与大型银行相同。参见 [[banking/japan-net-bank-competition-map|Japan net bank competition map]]。
- **信托银行 vs 普通银行。** 信托银行在 **法律上** 不同（双重牌照）。拥有信托子公司的普通银行与信托银行不同。
- **信用金库 vs 信用组合。** 两者都是具有单独法令的 协同组织 协同组合金融机构；会员资格规则与监督财务局不同。参见 [[banking/shinkin-bank-registry-japan|shinkin registry]] 与 [[banking/credit-cooperative-registry-japan|credit cooperative registry]]。
- **劳金 vs 其他协同组合金融。** 劳金独一无二地与 厚劳省 共同监督，并具有劳动组合会员结构。参见 [[banking/rokin-bank-registry-japan|rōkin registry]]。
- **农林系统 vs 普通协同组合金融。** 该体系使用 储金（savings）而非 预金 的术语，并拥有与 [[financial-regulators/yokin-hoken-kiko|DICJ]] 平行的自有储金保险机构；监督为 FSA / 农水省 共同。
- **外国银行支店 vs 日本法人化的外资银行。** 外国银行支店是第 IX 章下的支店。日本法人化的子公司（例如 SBJ 银行等 [[banking/INDEX|banking INDEX]] 实体）是外资所有的 普通银行，而非外国银行支店。存款保险处理不同。
- **银行代理 / BaaS 表层 vs 银行牌照。** 伙伴品牌 app、BaaS 服务或银行代理渠道 **不是** 单独的银行牌照 — 存款债务人是持牌银行。法律栈 vs UX 栈映射参见 [[financial-licenses/bank-license-and-baas-boundary|bank license and BaaS boundary]]。
