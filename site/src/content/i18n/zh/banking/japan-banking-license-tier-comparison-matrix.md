# 日本银行牌照层级比较矩阵

## TL;DR

日本的吸收存款体系并非单一牌照。《银行法》、《信託業法》、信用金庫法、中小企業等協同組合法、労働金庫法及農林中央金庫法各自划出不同的牌照层级,具有不同的法规、监管机构、资本下限、存款保险制度、许可业务范围及治理形态。本矩阵汇集了横跨**大型银行 / 地方 / 第二地方 / 信托 / 网络 / 外国分行 / 信用金库 / 信用合作社 / 劳动金库 / 农林中金-JA-JF** 的公开可核实维度,以便在品牌或资产负债表问题之前先回答牌照形态问题。

The regulatory boundary is [[JapanFG/legal-financial-licenses/bank-license-and-baas-boundary|bank license and BaaS boundary]] inside [[JapanFG/legal-financial-licenses/INDEX|JapanFG legal / financial licenses]]. The deposit-insurance system is [[JapanFG/yokin-hoken-kiko|預金保険機構 (DICJ)]]. The cooperative central institutions are [[JapanFG/norinchukin|農林中央金庫]] and [[JapanFG/shinkin-central|信金中央金庫]].

## 为何层级矩阵重要

「日本的银行」这样一个笼统的说法掩盖了一个事实:牌照层级几乎控制了对分析至关重要的每一个公开界面字段:

- 该机构归属于哪部法规;
- 哪家监管机构对其进行检查(金融厅;劳动金库由厚劳省联合监管;JA / JF / 农林中金由农水省联合监管);
- 存款保险对手方是 [[JapanFG/yokin-hoken-kiko|DICJ]] 还是 JA共済 / 合作保护制度;
- 该机构能否持有证券牌照、信托牌照,或仅限普通银行业务;
- 它能否纳入银行控股公司之内;
- 它是否拥有作为对手方对日本银行的直接 [[banking/INDEX|banking-domain]] 接入;
- 以及会员资格 / 地理是否构成硬性法律约束。

没有牌照层级,一家 [[banking/japan-net-bank-competition-map|net bank]] 看起来只是又一家零售银行,一家 [[banking/shinkin-bank-registry-japan|shinkin]] 看起来只是又一家小型地方银行,一家 [[banking/foreign-bank-japan-retreat|foreign branch]] 看起来只是又一家批发参与者。

## 层级 1 — 大型银行(普通銀行 / 都市銀行)

- **法规 / 监管机构。** 《银行法》普通銀行 牌照;由金融厅主要银行及银行控股公司司监管,日本银行作为货币政策对手方进行现场监测。
- **牌照 + 数量。** 《银行法》普通銀行。SMBC / MUFG / 瑞穗 三家核心加上 りそな / 埼玉りそな 位于 [[banking/INDEX|banking index]] 内,构成大型银行通道。
- **资本下限。** 《银行法》普通银行最低实缴资本为 ¥2 亿日元;大型银行远高于此,并按 BIS 巴塞尔 III 集团资本运作 —— 普通股权一级 1 ≥ 4.5%、一级 1 ≥ 6%、总资本比率 ≥ 8%,再加上 G-SIB / D-SIB 资本缓冲附加层(MUFG、SMFG、瑞穗 承担 G-SIB 附加;りそな 被视为 D-SIB)。
- **流动性。** 完整的巴塞尔 III LCR ≥ 100% 及 NSFR ≥ 100% 在合并基础上适用。
- **存款保险。** [[JapanFG/yokin-hoken-kiko|DICJ]] 保障;在标准存款保险框架下本金 ¥10 百万日元加利息,决済用預金 受全额保护。
- **许可业务。** 存款、贷款、外汇、结算、债券及国债承销、通过子公司开展的证券关联业务、衍生品、托管、资产管理、租赁、消费金融、控股公司战略职能。
- **跨境。** 完整的国际银行业务:海外分行、外国银行子公司、美元 / 欧元清算接入、跨境贷款、项目 / 贸易融资、互换额度资格。
- **治理。** 在东证 Prime 上市的股份制公司;银行控股公司结构获许并被采用(MUFG、SMFG、瑞穗 FG、りそな HD)。
- **地理。** 无法定地理限制;全国网点及海外存在。
- **并购 / 控股公司。** 经金融厅事先批准可设银行控股公司;交叉持股与整合路径在《银行法》下开放。
- **日本银行对手方。** 直接的日本银行往来账户;JGB 主交易商;接入日本银行市场操作。
- **近期监管变化。** 《银行法》改正(令和 3 / 2021)扩大了银行关联公司范围及数字 / 区域振兴投资范围;围绕银行发行数字货币及 電子決済等取扱業 的更新对该层影响最直接。参见 [[JapanFG/legal-financial-licenses/bank-license-and-baas-boundary|bank license and BaaS boundary]]。

## 层级 2 — 地方银行(地方銀行)

- **法规 / 监管机构。** 《银行法》普通銀行 牌照;在金融厅 中小・地域金融機関 监督指针下监管;当都道府县金融厅财务局介入时,许多地方银行也落入 [[banking/regional-bank-consolidation-pattern|regional bank consolidation pattern]]。
- **牌照 + 数量。** 《银行法》普通銀行 牌照,第一地方銀行協会 会员;全銀協 的分类将其视为位于大型银行与第二地方银行之间的「地方银行」层。
- **资本下限。** 同样的《银行法》¥2 亿日元最低额;多数采用 BIS 标准法,仅最大者采用 IRB;无国际业务的银行适用国内标准 4% 核心资本比率。
- **流动性。** 国际活跃银行适用 LCR / NSFR;国内标准银行在金融厅国内框架下采用更简单的流动性要求。
- **存款保险。** [[JapanFG/yokin-hoken-kiko|DICJ]] 保障,条件与大型银行相同。
- **许可业务。** 存款、贷款、外汇、结算、通过子公司开展的证券关联业务;信托业务需另行牌照(部分地方银行附带信托辅助职能)。
- **跨境。** 有限;较大的地方银行经营海外代表处及少数分行,但多数以国内为重心。
- **治理。** 股份制公司;许多在东证上市;银行控股公司结构日益普遍([[banking/regional-bank-consolidation-pattern|regional bank consolidation pattern]])。
- **地理。** 无法律地理限制,但业务集中于一到数个都道府县。
- **并购 / 控股公司。** 经金融厅批准可设银行控股公司;在金融厅 経営統合 框架下的整合是主导的并购路径。
- **日本银行对手方。** 直接的日本银行往来账户;有资格参与日本银行市场操作,规模小于大型银行。
- **近期监管变化。** 2021 《银行法》改正扩大了许可关联公司业务,并使地方银行更容易投资于区域振兴、数字银行子公司及咨询子公司 —— 这对 [[banking/regional-bank-consolidation-pattern|regional bank consolidation pattern]] 中追踪的整合脉络至关重要。

## 层级 3 — 第二地方银行(第二地方銀行)

- **法规 / 监管机构。** 《银行法》普通銀行 牌照;相同的金融厅 中小・地域金融機関 监督指针。在法律上与地方银行无从区分 ——「第二」的标签是行业协会会员身份。
- **牌照 + 数量。** 《银行法》普通銀行 牌照,第二地方銀行協会 会员;历史上是在 1989 改革下由 相互銀行(互助储蓄银行)转换为普通银行而来。
- **资本下限。** 《银行法》最低额与地方银行相同;BIS 制度相同。
- **流动性。** 与普通地方银行相同;非国际活跃银行采用国内标准流动性。
- **存款保险。** [[JapanFG/yokin-hoken-kiko|DICJ]] 保障,完全适用《银行法》条款。
- **许可业务。** 与地方银行相同;实践中其业务更偏向中小企业,资产负债表更小。
- **跨境。** 极少;几乎全部为国内业务。
- **治理。** 股份制公司;许多情况下在东证上市;获许设银行控股公司。
- **地理。** 单一都道府县业务比一线地方银行更为典型。
- **并购 / 控股公司。** [[banking/regional-bank-consolidation-pattern|regional bank consolidation pattern]] 中描述的整合流程常将一线地方银行与同一都道府县的第二地方银行配对纳入同一控股公司。
- **日本银行对手方。** 直接的日本银行往来账户。
- **近期监管变化。** 与普通地方银行相同的 2021 《银行法》改正范围;金融厅对整合的支持措施(系统成本补贴、治理灵活性)在该层是一项有意义的推动。

## 层级 4 — 信托银行(信託銀行)

- **法规 / 监管机构。** 双牌照:《银行法》普通銀行 牌照**外加**金融厅下的《信託業法》信托业务牌照。正是这种双牌照形态使信托银行在法律上区别于普通银行。
- **牌照 + 数量。** 公开会员身份收录于 信託協会 信托银行名册。运营公司支柱包括 [[banking/INDEX|banking index]] 信托银行路由加上 JapanFG 信托银行页面;运营比较界面为 [[banking/trust-bank-custody-operating-comparison|trust-bank custody operating comparison]]。
- **资本下限。** 适用《银行法》最低额;在其之上叠加额外的《信託業法》能力与运营要求(隔离信托账户、受托责任义务)。
- **流动性。** 银行账簿一侧适用《银行法》LCR / NSFR;信托资产在单独的受托会计下属表外。
- **存款保险。** [[JapanFG/yokin-hoken-kiko|DICJ]] 对银行账簿存款保障;信托账户通过 元本補填信託 路径,对受保本金货币信托在同一 DICJ 框架下受保护。
- **许可业务。** 银行业务(存款、贷款、外汇、结算)**外加**信托业务(货币信托、证券投资信托、养老金信托、托管、证券结算、不动产信托、资产管理)。该双牌照层在吸收存款版图中拥有最广的业务范围。
- **跨境。** 大型银行信托部门(MUFJ 信托、SMBC 信托、瑞穗信托)经营全球托管与资产管理网络;专业资产管理银行(日本マスタートラスト信託銀行、日本カストディ銀行)经营机构基础设施层。
- **治理。** 股份制公司;几乎总是位于银行控股公司之下。
- **地理。** 无法定限制;全国。
- **并购 / 控股公司。** 获许设银行控股公司;信托银行子公司常与普通银行子公司并列于大型银行集团内。
- **日本银行对手方。** 直接的日本银行往来账户;托管银行信托部门参与 BoJ-NET / JGB 结算基础设施。
- **近期监管变化。** 金融厅对受托责任原则的扩展、资产管理业务改善命令,以及信托业务改革的 [[JapanFG/legal-financial-licenses/INDEX|legal-financial-licenses]] 轨道重塑了运营范围;電子決済等取扱業 下的银行发行数字货币中介也落在银行侧信托中介上。

## 层级 5 — 网络银行(ネット銀行)

- **法规 / 监管机构。** 《银行法》普通銀行 牌照。**无单独的法定类别** ——「网络银行」是运营性的,而非法律性的。其牌照与大型银行相同。
- **牌照 + 数量。** 《银行法》普通銀行 牌照;运营版图追踪于 [[banking/japan-net-bank-competition-map|Japan net bank competition map]](生态圈零售、全功能银行 BaaS、法人 API、资产形成、区域数字)。
- **资本下限。** 《银行法》最低资本;多数采用 BIS 标准法,许多采用国内标准 4%;部分网络银行(乐天银行、SBI 住信网络银行)已成长为中型资产负债表。
- **流动性。** 标准《银行法》LCR / NSFR 规则,非国际活跃运营者采用国内标准。
- **存款保险。** [[JapanFG/yokin-hoken-kiko|DICJ]] 保障,完全适用《银行法》条款。
- **许可业务。** 完整的普通银行范围;有些为合作伙伴经营全功能银行 BaaS([[banking/japan-baas-operating-models|Japan BaaS operating models]]),有些经营法人 API 银行业务,有些经营证券毗邻的零售银行业务。
- **跨境。** 有限;外汇零售与海外汇款常见,完整海外分行不常见。
- **治理。** 股份制公司;获许设银行控股公司;许多网络银行位于金融科技 / 电商 / 电信 / 证券母集团之内,而非银行控股公司之下。
- **地理。** 没有传统意义上的网点网络;全国数字渠道。
- **并购 / 控股公司。** 获许设银行控股公司;运营模式问题(母生态圈、合作伙伴 BaaS、证券毗邻)在 [[banking/japan-net-bank-competition-map|Japan net bank competition map]] 中。
- **日本银行对手方。** 一旦取得 普通銀行 牌照即拥有直接的日本银行往来账户。
- **近期监管变化。** 2021 《银行法》改正对银行关联公司范围的扩展是对该层最直接重要的变化之一;追踪于 [[JapanFG/legal-financial-licenses/bank-license-and-baas-boundary|bank license and BaaS boundary]] 的 電子決済等取扱業 与 BaaS 边界监管影响的是运营自由而非牌照形态。

## 层级 6 — 外国银行分行(外国銀行支店)

- **法规 / 监管机构。** 《银行法》第九章下的 外国銀行支店 牌照;在金融厅主要银行及外国银行监管路径下监管。
- **牌照 + 数量。** 《银行法》外国銀行支店 牌照;金融厅发布外国银行分行名单并定期更新。该模式追踪于 [[banking/foreign-bank-japan-retreat|Foreign-bank Japan retreat]]。
- **资本下限。** 适用分行资本等值规则(资产等值处理、依赖总行资本充足);并非对境内注册银行适用的普通 ¥2 亿日元实缴资本下限。总行的巴塞尔 III 地位至关重要。
- **流动性。** 分行特定的流动性预期;总行 LCR / NSFR 经由母国监管框架贯穿,境内业务重要时附加金融厅附加要求。
- **存款保险。** [[JapanFG/yokin-hoken-kiko|DICJ]] 保障**不会**像对日本银行那样自动适用于外国银行分行 —— 根据《银行法》第九章,外国银行分行存款不受日本 DICJ 体系保障;储户依赖母国保护及分行资产 / 日本资产质押规则。
- **许可业务。** 银行业务、外汇、通过持牌子公司开展的证券关联业务;业务限于牌照授权范围加上经日本监管等同认定的总行能力。
- **跨境。** 与生俱来;该牌照的全部要点在于支持企业、批发、托管、市场、外汇、贸易及亚洲走廊业务 —— 参见 [[banking/foreign-bank-japan-retreat|Foreign-bank Japan retreat]] 中的分部图。
- **治理。** 境外法律实体的分行;不是日本股份制公司;在金融厅预期下由总行董事会及日本分行治理管理。
- **地理。** 通常在东京 / 大阪有一到数家分行。
- **并购 / 控股公司。** 不能被纳入日本银行控股公司之下;重组通常经由总行进行。
- **日本银行对手方。** 外国银行分行可持有日本银行往来账户并参与 BoJ-NET;并非所有外国分行都如此。
- **近期监管变化。** 金融厅外国银行分行监管指针围绕反洗钱、可疑交易监测、治理等同性及总行风险控制的更新仍是活跃的边界。

## 层级 7 — 信用金库(信用金庫)

- **法规 / 监管机构。** 信用金庫法;在金融厅 中小・地域金融機関 监督指针(关于合作金融机构的第 V 部分)及财务局 / 福岡財務支局 / 沖縄総合事務局 路径下监管。
- **牌照 + 数量。** 信用金庫法 牌照;据 [[banking/shinkin-bank-registry-japan|Japan shinkin bank registry]] 有 254 家持牌信用金库,外加中央机构 [[JapanFG/shinkin-central|信金中央金庫]]。
- **资本下限。** 信用金庫法 最低 出資金 对普通信用金库为 ¥100 百万日元(指定区域为 ¥200 百万日元);BIS 式资本充足为金融厅国内标准 4% 核心资本框架。
- **流动性。** 金融厅国内标准流动性预期;由 [[JapanFG/shinkin-central|信金中央金庫]] 提供中央机构流动性支持。
- **存款保险。** [[JapanFG/yokin-hoken-kiko|DICJ]] 保障按相同的《银行法》等同条款适用。
- **许可业务。** 会员存款、会员贷款、结算、外汇、证券投资;对非会员贷款按比例受限(50% 非会员上限是历史锚点,有除外条款)。
- **跨境。** 通常无;部分较大的信用金库通过代理行路径开展小规模外汇 / 汇款。
- **治理。** 合作社 / 協同組織 —— 会员制、一人一票治理,采用 出資(会员出资)而非可交易股权。
- **地理。** 法定业务区域(営業地区)与市町村 / 都道府县挂钩;会员须在该区域居住、工作或有业务。
- **并购 / 控股公司。** 无银行控股公司路径;合并在 信用金庫法 程序及金融厅批准下于信用金库之间进行;跨信用金库联盟经由 [[JapanFG/shinkin-central|信金中央金庫]]。
- **日本银行对手方。** [[JapanFG/shinkin-central|信金中央金庫]] 是信用金库体系的日本银行对手方;各信用金库通过 信金中央金庫 清算。
- **近期监管变化。** 信用金庫法 扩大关联业务范围的修订,以及 2021时代呼应《银行法》扩展的合作金融改革,是主要的更新;语境参见 [[banking/cooperative-banking-japan|Cooperative banking in Japan]]。

## 层级 8 — 信用合作社(信用組合)

- **法规 / 监管机构。** 中小企業等協同組合法 加上 協同組合による金融事業に関する法律;由金融厅 中小・地域金融機関 路径与都道府县当局协调监管。牌照授予 信用協同組合。
- **牌照 + 数量。** 据 [[banking/credit-cooperative-registry-japan|Japan credit cooperative registry]] 有 143 家持牌信用合作社,外加中央机构 全国信用協同組合連合会。
- **资本下限。** 合作社法最低 出資金,采用金融厅国内标准资本充足;下限小于信用金库。
- **流动性。** 金融厅国内标准流动性;由 全信組連 提供中央机构流动性支持。
- **存款保险。** [[JapanFG/yokin-hoken-kiko|DICJ]] 保障适用。
- **许可业务。** 会员存款、会员贷款、外汇、结算;会员资格规则比信用金库更严(会员受行业、区域、职业或亲缘关系限制 —— 業域 / 地域 / 職域)。
- **跨境。** 通常无。
- **治理。** 合作社 / 協同組織,一人一票及 出資 会员出资。
- **地理。** 法定业务区域;许多合作社服务于特定职业或行业而非某一都道府县(例如历史上的 商工組合中央金庫、在日韩裔 / 华裔商业合作社、运输业合作社)。
- **并购 / 控股公司。** 无银行控股公司路径;信用合作社之间的合并在 中小企業等協同組合法 程序下获许。
- **日本银行对手方。** 全信組連 是信用合作社体系的日本银行对手方。
- **近期监管变化。** 与信用金库相同的合作金融改革范围;关联业务范围扩展使区域振兴 / 咨询 / 数字子公司投资比 2021之前略为容易。

## 层级 9 — 劳动金库(労働金庫)

- **法规 / 监管机构。** 労働金庫法;由金融厅与 厚生労働省(厚生劳动省)**联合监管**。这种双监管形态在吸收存款版图中独一无二。
- **牌照 + 数量。** 据 [[banking/rokin-bank-registry-japan|Japan rokin bank registry]] 有 13 家持牌劳动金库(如 中央労働金庫、近畿労働金庫、東海労働金庫 等区域劳动金库),外加中央机构 労働金庫連合会。
- **资本下限。** 労働金庫法 最低 出資金;适用金融厅国内标准资本充足。
- **流动性。** 金融厅国内标准流动性;由 労金連 提供中央机构流动性支持。
- **存款保险。** [[JapanFG/yokin-hoken-kiko|DICJ]] 保障适用。
- **许可业务。** 会员存款、贷款、住房贷款、结算;会员为工会、消费生活合作社及隶属于会员组织的个人。贷款集中于家庭 / 工人金融(住房贷款、教育贷款、人生大事融资)。
- **跨境。** 无有意义规模的跨境业务。
- **治理。** 合作社 / 協同組織;由会员组织而非个人拥有劳动金库;工会代表是结构性的。
- **地理。** 区域业务区域;13 家区域劳动金库共同覆盖日本而不重叠。
- **并购 / 控股公司。** 无银行控股公司路径;劳动金库可在 労働金庫法 程序下合并。
- **日本银行对手方。** 労金連 是劳动金库体系的日本银行对手方。
- **近期监管变化。** 金融厅 / 厚劳省 联合监管意味着,除普通金融厅银行监管更新外,职场与消费者保护更新也渗入劳动金库监管。

## 层级 10 — 农林中金 / JA / JF 体系(農林系統金融)

- **法规 / 监管机构。** 農林中央金庫法(适用于 農林中央金庫);農業協同組合法(适用于 JA / 信农连 信用业务);水産業協同組合法(适用于 JF / 信渔连 信用业务)。由金融厅与 農林水産省(农林水产省)**联合监管** —— 又一个双监管体制。
- **牌照 + 数量。** [[JapanFG/norinchukin|農林中央金庫]] 作为中央机构位于顶端。其下:都道府县 信农连(JA 信用联合会)与 信渔连(JF 信用联合会),以及底层从事信用业务的 JA 与 JF 合作社。JA 银行体系与 JF Marine Bank 体系路由参见 [[banking/INDEX|banking index]]。
- **资本下限。** 農林中央金庫法 为作为 特殊法人 的 [[JapanFG/norinchukin|農林中央金庫]] 设定特别框架;成员合作社在各自的合作社法下运作。
- **流动性。** 鉴于其市场活动,对 [[JapanFG/norinchukin|農林中央金庫]] 适用银行等同的审慎框架(BIS 式资本与流动性);成员机构按合作社法审慎标准运作。
- **存款保险。** [[JapanFG/yokin-hoken-kiko|DICJ]] 通常通过合作金融路径覆盖 JA / JF 的存款等值债权;農水産業協同組合貯金保険機構(农水产业合作社存款保险机构)为农业 / 渔业合作社储蓄提供平行保险机制,与银行 DICJ 轨道分离。
- **许可业务。** 会员存款 / 貯金(在 JA / JF 中技术上称 貯金 而非 預金)、贷款、结算、外汇(通过 [[JapanFG/norinchukin|農林中央金庫]])、大规模证券投资、资产管理。[[JapanFG/norinchukin|農林中央金庫]] 按其使命是全球活跃的固定收益 / 结构化信贷投资者,这在日本吸收存款机构中极具特色。
- **跨境。** [[JapanFG/norinchukin|農林中央金庫]] 经营庞大的海外投资组合并设有海外代表处;成员 JA / JF 合作社则没有。
- **治理。** 在 JA / JF 层为合作社 / 協同組織;在 [[JapanFG/norinchukin|農林中央金庫]] 层为 特殊法人。会员组织治理与农业政策的接近性是机构设计的一部分。
- **地理。** JA / JF 合作社属其市町村 / 都道府县区域;信农连 / 信渔连 为都道府县级;[[JapanFG/norinchukin|農林中央金庫]] 为全国级。
- **并购 / 控股公司。** 无普通银行控股公司路径;体系整合通过合作社法合并、联合会重组及中央机构重组进行。
- **日本银行对手方。** [[JapanFG/norinchukin|農林中央金庫]] 作为体系中央机构持有对日本银行的直接往来账户接入。
- **近期监管变化。** 農林中央金庫法 围绕资本结构 / 监管资本的修订、金融厅-农水省 在市场风险与集中度管理上的联合监管更新,以及 [[JapanFG/norinchukin|農林中央金庫]] 页面中追踪的关于投资组合损失 / 重组的 2024 之后的持续披露。

## 如何阅读本矩阵

牌照层级矩阵是一个公开界面工具。它不对机构排名,不估算资产负债表健康度,也不预测整合。它的存在是为了让一条 [[banking/INDEX|banking index]] 词条、一条 [[banking/japan-net-bank-competition-map|net bank]] 词条或一条 [[banking/foreign-bank-japan-retreat|foreign-branch]] 词条,能在被问及战略、渠道或业务问题之前得到一致的分类。

在阅读任何单一机构页面时:

1. 从**法规**列开始 —— 它决定几乎每一个其他字段。
2. 检查**存款保险**列。外国分行与 農水産業 合作社是默认 [[JapanFG/yokin-hoken-kiko|DICJ]] 框架的例外。
3. 检查**日本银行对手方**列。合作金融机构通过其中央机构而非直接清算。
4. 检查**控股公司**列。合作社层级不能使用银行控股公司;外国分行也不能使用。
5. 用**治理**列来判断适用的是一人一票、股份制上市,还是境外实体分行的表述 —— 这决定了可用的治理、披露及并购界面。

## 边界案例与注意事项

- **网络银行对比普通银行。** 网络银行在运营上有别,但在法律上与任何其他 普通銀行 完全相同。牌照层级与大型银行相同。参见 [[banking/japan-net-bank-competition-map|Japan net bank competition map]]。
- **信托银行对比普通银行。** 信托银行在**法律上**有别(双牌照)。带有信托子公司的普通银行不等同于信托银行。
- **信用金库对比信用合作社。** 两者都是具有不同法规的 協同組織 合作金融机构;资格规则与监管财务局不同。参见 [[banking/shinkin-bank-registry-japan|shinkin registry]] 与 [[banking/credit-cooperative-registry-japan|credit cooperative registry]]。
- **劳动金库对比其他合作金融。** 劳动金库独特地由 厚劳省 联合监管,并具有工会会员结构。参见 [[banking/rokin-bank-registry-japan|rōkin registry]]。
- **農林系統 对比普通合作金融。** 该体系使用 貯金(储蓄)而非 預金(存款)术语,并拥有与 [[JapanFG/yokin-hoken-kiko|DICJ]] 平行的自有储蓄保险机构;监管为金融厅 / 农水省 联合。
- **外国银行分行对比日本注册的外资银行。** 外国銀行支店 是第九章下的分行。日本注册的子公司(例如 [[JapanFG/INDEX|JapanFG]] 等实体,如 SBJ Bank)是具有外资所有权的 普通銀行,而非外国银行分行。存款保险处理不同。
- **银行代理 / BaaS 界面对比银行牌照。** 合作伙伴品牌的应用、BaaS 服务或银行代理渠道**不是**单独的银行牌照 —— 存款债务方是持牌银行。法律堆栈对比 UX 堆栈的映射参见 [[JapanFG/legal-financial-licenses/bank-license-and-baas-boundary|bank license and BaaS boundary]]。
