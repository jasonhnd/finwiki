# 日本信托银行 对 全球托管行 比较矩阵

## TL;DR

将日本的信托银行系托管运营方 (全服务信托银行、资产管理专营、中小信托银行) 与在日本运营的全球托管行 (BNY 梅隆、道富、摩根大通、花旗) 进行横向比较。两大阵营并非直接替代关系: 日本的信托银行在 信託業法 (信托业法) + 銀行法 (银行法) 的双重路径下主导国内 JGB / 股票 / 公的年金 (公共养老金) 托管，而全球托管行则承担面向外国投资者的次级托管委托，以及经由 ICSD 网关的跨境资产服务。正确解读该矩阵，可避免一个常见错误，即将「信託口」(信托账户) 名义人当作仿佛银行本身就是经济上的投资者。

| 主要资产范围 | 国内株式 / JGB / 国内投信 / 公的年金資産 (うち国内分) | 海外株式 / 海外債券 / 邦銀の海外運用資産 / ADR / 外国年金資金の日本投資 |
| 结算基础设施 | [[securities/japan-securities-depository-center|JASDEC]] / [[securities/japan-securities-clearing-corp|JSCC]] / BoJ JGB システム 直接接続 | グローバル ICSD (Euroclear / Clearstream) + 日本側は JASDEC / JSCC へ |
| 受益人集中度 | GPIF + 大手投信 + 公的年金 (国内ベース) | 邦銀グローバル + 外国年金 / SWF + 海外投信 (国際ベース) |
| 表决权 / 治理角色 | 「信託口」名義 — 議決権指図は委託者 (運用機関 / 年金) | sub-custodian として通過、議決権は最終 beneficial owner / global custodian の委託者 |

| 近期整合 | **2020-07 JTSB + TCSB → CBJ** に類する集中化が継続 | グローバル統合は完了済 (BNY + Mellon 2007、State Street 単一系統、Citi / JPM は内部統合) |

## 万事达信托 对 日本托管银行 — 2020 整合的意义

**2020-07-27 JTSB (日本トラスティ・サービス信託銀行) + TCSB (資産管理サービス信託銀行) → [[trust-banks/custody-bank|日本カストディ銀行]]** 这一整合，是将日本托管基础设施的 2 强体制明确化为 **MUFG 系 [[trust-banks/master-trust-bank|日本マスタートラスト信託銀行]] vs SMTB / Mizuho / Resona / 第一生命系 [[trust-banks/custody-bank|日本カストディ銀行]]** 的事件。^[extracted]

作为整合理由，公开资料显示如下:

1. 国内被动管理扩张 + 公共养老金资产规模扩大带来的信息处理能力要求
2. 系统重复成本的压缩 (JTSB + TCSB 各自持有独立的投信计算 / 托管系统)
3. 应对表决权电子行使 / 公司行动电子化的投资集约

## 定价权动态

| 侧面 | 国内信托银行 | 全球托管行 |
|---|---|---|
| 客户议价力 | 凭规模效应 + 关系，Master Trust / CBJ 较强 | 凭全球必备功能性全社皆强 |
| 可替代性 | 因国内基础设施集约，在国内有限 | 凭母国规制 + 母国系统连接性，邦银替代在结构上困难 |
| 交叉销售 | 不动产 / 养老金 / 继承 / 投行关系 (全服务信托银行) | 全球管理 / 证券业务 / 主经纪业务 |
| 监管资本成本 | 以国内 RWA 为主 | 全球综合资本 (含 G-SIB 缓冲 — BNY Mellon / State Street / JPM / Citi 全社获 G-SIB 认定) |

## 研究清单

1. 在得出所有权结论之前，使用 [[banking/japan-trust-bank-custody-map|trust-bank custody map]] 确认对托管银行的提及究竟是 **实体 / 股东名义人 / 委托持有人**。
2. 对于专营银行，核查其究竟是 **银行法 + 信托业法 下的正规信托银行**，还是纯粹的 信托公司 — 唯有银行路径方能直接参与 DVP / JGB 簿记。
3. 对于全球托管行，识别其日本运营是拥有 **自身的信托银行牌照** (BNY / State Street / JPM)，还是仅经由 外国银行支店 + 证券子公司 的路径运营 (花旗模式)。
4. 将 ICSD 网关接入 (Euroclear / Clearstream) 视为真正的全球托管行与国内专营方之间的 **分水岭** — 国内专营方通常将外国资产经由全球托管行重新路由。
5. 对所有 AUC / AuA / 受託資産 数值标注出处日期，因其随市场价格与新委托而变动; 万事达信托的 ¥823万亿 (2025) 与 CBJ 的 ¥696万亿 (2025-03) 锚点为官方披露的概要点，而非实时数值。
6. 经由 [[securities/japan-asset-manager-landscape-matrix|asset manager landscape matrix]] 交叉链接客户一侧，以理解委托流向。
