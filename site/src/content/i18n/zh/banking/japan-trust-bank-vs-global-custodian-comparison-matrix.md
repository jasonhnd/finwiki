# 日本信托银行 vs 全球托管机构比较矩阵

## TL;DR

将日本信托银行托管运营方（全能型信托银行、资产管理专业机构、较小型信托银行）与在日本运营的全球托管机构（BNY Mellon、State Street、JP Morgan、Citi）并排比较。两大阵营并非直接替代品：日本信托银行在信托业法 + 银行法双重路线下主导国内 JGB / 权益 / 公的年金 托管，而全球托管机构为外国投资者持有次级托管委托，并经由 ICSD 网关进行跨境资产服务。正确解读本矩阵，可避免一个常见错误——把「信託口」名义人名称读成仿佛该银行本身就是经济上的投资者。

## Master Trust vs Custody Bank of Japan — 2020 統合の意味

**2020-07-27 JTSB (日本トラスティ・サービス信託銀行) + TCSB (資産管理サービス信託銀行) → [[JapanFG/custody-bank|日本カストディ銀行]]** という統合は、日本のカストディ・インフラの 2 強体制を **MUFG 系 [[JapanFG/master-trust-bank|日本マスタートラスト信託銀行]] vs SMTB / Mizuho / Resona / 第一生命系 [[JapanFG/custody-bank|日本カストディ銀行]]** として明確化したイベントである。^[extracted]

統合理由として公開資料は次を示す:

1. 国内パッシブ運用拡大 + 公的年金資産の規模拡大による情報処理能力要件
2. システム重複コストの圧縮 (JTSB + TCSB は別個の投信計算 / カストディ システムを保有していた)
3. 議決権電子行使 / コーポレートアクション電子化への対応投資集約

## Pricing power dynamics

| 側面 | 国内信託銀行 | グローバルカストディアン |
|---|---|---|
| 顧客交渉力 | 規模効果 + リレーションで Master Trust / CBJ は強い | グローバル必須機能性で全社強い |
| 代替可能性 | 国内インフラ集約により国内では限定的 | 母国規制 + 母国システム接続性で邦銀代替は構造的困難 |
| クロス販売 | 不動産 / 年金 / 相続 / 投資銀行リレーション (full-service trust bank) | グローバル運用 / 証券業務 / プライムブローカレッジ |
| 規制資本コスト | 国内 RWA 中心 | グローバル統合資本 (G-SIB バッファ含む — BNY Mellon / State Street / JPM / Citi 全社 G-SIB 認定) |

## Research checklist

1. 在得出所有权结论之前，使用 [[banking/japan-trust-bank-custody-map|trust-bank custody map]] 确认某个托管银行的引用究竟是**实体 / 股东名义人 / 委托持有人**。
2. 对于专业银行，检查它们是**银行法 + 信托业法下的正规信托银行**还是纯 信託会社——只有银行路线才能直接参与 DVP / JGB 簿记。
3. 对于全球托管机构，辨明其日本运营是拥有**自有的信託銀行牌照**（BNY / State Street / JPM），还是仅经由 外国銀行支店 + 証券子会社 路线运营（Citi 模式）。
4. 将 ICSD 网关接入（Euroclear / Clearstream）视为真正全球托管机构与国内专业机构之间的**分界线**——国内专业机构通常将外国资产经全球托管机构重新路由。
5. 为所有 AUC / AuA / 受託資産 数据标注来源日期，因为它们随市价与新委托变动；Master Trust ¥823tn（2025）与 CBJ ¥696tn（2025-03）的锚点是官方披露的汇总点，而非实时数字。
6. 经由 [[securities/japan-asset-manager-landscape-matrix|asset manager landscape matrix]] 交叉链接客户侧，以理解委托流。
