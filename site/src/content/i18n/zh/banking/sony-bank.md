### 索尼银行在母公司内部的位置

```
Sony Group Corporation (东证 6758, 最终母公司)
  └── Sony Financial Group (经由 TOB 于 2020 重新并表;经由部分分拆于 2025-09-29 在东证 Prime 重新上市,Sony Group 保留约 20%)
        ├── Sony Bank Incorporated (本页所述条目)
        ├── [[life-insurers/sony-life|Sony Life Insurance]]
        ├── [[non-life-insurers/sony-insurance|Sony Insurance (P&C)]]
        └── 其他金融子公司
```

Sony FG 此前曾上市;2020 Sony Group 发起 TOB,将 Sony FG 私有化。2025-09-29 Sony Group 经由日本首例「部分分拆」在东证 Prime 市场将 Sony FG 重新上市 —— 把其持有的 Sony FG 股份中超过 80% 以实物方式分配给 Sony Group 股东,同时保留约 20% —— 此举为索尼银行的持股经济学重新引入了公开市场参照 (Sony FG 开盘价 ¥205)。索尼银行本身仍是 Sony FG 之下的非上市银行运营公司。

### 多币种存款 + 外汇核心

相对于更广泛的 [[banking/japan-net-bank-competition-map|net bank universe]],索尼银行的外币存款产品是其最具特色之处:

| Feature | Sony Bank approach |
|---|---|
| 币种菜单 | 依官方 MoneyKit 外汇页面,共 12 种外币 (USD、EUR、GBP、AUD、NZD、CAD、CHF、HKD、BRL、CNH、ZAR、SEK) |
| 外汇成本 | 相对巨型银行柜台零售外汇,面向客户的点差较窄 |
| 借记卡外汇 | **Sony Bank WALLET** 在海外直接以外币余额支付 —— 刷卡时无须日元 → 外汇的换汇环节 |
| 定期存款外汇 | 收益挂钩海外利率的外币定期存款 |
| 存款安全 | 日元存款由 預金保険機構 承保;外币存款不在存款保险承保范围内 |

外汇借记功能,是面向频繁旅行及侨居类用户的客户获取杠杆 —— 这一客群因点差更高、开户流程更慢,巨型银行柜台无法干净利落地服务。

### 投资产品分销

MoneyKit 平台分销 投信 (投资信托),并支持证券账户的邻接业务。相较于 [[regional-banks/ssnb|SSNB]] 与 [[securities-firms/sbi-securities|SBI Securities]] 的深度关联,或 [[banking/au-jibun-bank|au Jibun Bank]] 与 [[securities-firms/au-kabucom-securities|au Kabucom]] 的关联,索尼银行的证券关联相对偏薄 —— 但其外汇强项与 外貨投信 分销形成互补。

### 卡与支付邻接业务

| Product | Role |
|---|---|
| Sony Bank WALLET | 多币种借记卡 + 现金卡;旗舰产品 |
| 信用卡合作 | 索尼银行与外部发卡机构合作,而非自营大型信用卡业务 |
| 直接支付 | 经由 全銀システム 进行索尼银行账户间转账 |

### 索尼银行的竞争领域

| Axis | Position |
|---|---|
| 外汇 / 外币存款 | 在无需高净值私行门槛即可面向零售的日本银行中**领先** |
| 房贷发放 | 强 (一家显著的网络银行发放方) |
| 多币种旅行 / 海外使用 | 经由 Sony Bank WALLET 领先 |
| 资产形成 / 投信 分销 | 稳健,但不及 SBI / SSNB 生态那般深入 |
| 工资账户转化 | 较 [[banking/rakuten-bank|Rakuten Bank]] / [[banking/paypay-bank|PayPay Bank]] / [[banking/au-jibun-bank|au Jibun Bank]] 生态覆盖更轻 |
| 卡 / 支付频次 | 较轻 —— 没有可与 楽天 / PayPay / au 相提并论的积分经济生态 |

### 客户细分

索尼银行的客户基础偏向于:

- 具有海外旅行或海外资产敞口的都市专业人士;
- 以外汇分散来构建有意义投资组合的家庭;
- 以利率和在线申请便利性为比较依据的房贷借款人。

## 4. KPI 形态与业务规模

索尼银行的公开披露 (「ディスクロージャー誌」系列与季度财务业绩) 显示出数万亿日元规模的存款基础 —— 截至 2025-09-30 存款总额为 ¥4,536.9 亿日元 (日元 ¥3,800.6亿 + 外币 ¥736.2亿) —— 同时客户数持续增长,且房贷账册可观。颗粒度的时点 KPI (账户数、房贷余额、ROE) 归于定期披露周期;本页追踪的是业务的形态,而非逐一列出每个数字。

### 近期披露周期的模式

| Metric | Pattern |
|---|---|
| 存款余额 | 数万亿日元区间;通过日元与外币产品双双增长 |
| 外币存款占比 | 显著高于巨型银行或其他网络银行同业 |
| 房贷余额 | 网络银行中较高的房贷账册之一 |
| 账户数 | 小于 Rakuten / PayPay / au-Jibun / SSNB;反映其富裕客群聚焦 |
| 资本 | 远高于 銀行法 下限 |

## 5. 监管边界

| Layer | Statute / regulator | Implication for Sony Bank |
|---|---|---|
| 银行监管 | 銀行法、金融庁 監督指針 | 标准的普通银行监管 |
| 外币业务 | FSA 项下的外汇相关规则 + 日本银行统计 | 外汇披露与 AML 适用于多币种产品 |
| 存款保险 | 預金保険機構 | 日元存款受承保;外币存款明确不受承保 |
| 证券产品销售 | 金融商品取引法 —— 用于 投信 分销 | 标准的适当性与披露义务 |
| AML / CFT | 金融庁 + 警察庁 + 国税庁 | 相对仅日元银行,跨境支付 AML 要求更高 |
| 房贷发放 | 銀行法 + 貸金業 相关法规 + 住宅ローン 相关指引 | 标准的银行侧房贷监管 |
