# 日本外资银行收缩与再聚焦时间线矩阵

## TL;DR

外资银行在 2000-2026 之间并没有「离开日本」。它们**收窄了**。数家全球银行在 1980s-2000s 建立起来的零售重心存在，让位于一种聚焦的组合：法人银行业务、市场业务、托管 / 资产服务、ICSD 网关、财富管理（往往经由日本 JV），以及亚洲走廊分行。本矩阵将每家主要外资银行母公司置于同一时间线上，包含**日本足迹峰值年份**、**收缩 / 拐点触发因素**（Lehman 2008, Mizuho 関連 纠纷、AML 执法、中国敞口转向、ESG、2023 CS / 俄罗斯退出）、**当前足迹**（CY 2024-2026）、**当前业务组合**及**计划退出 / 重组**。它是 [[banking/foreign-bank-japan-retreat|Foreign-bank Japan retreat pattern]]（定性模式地图）与 [[foreign-financial-institutions/foreign-bank-branches-japan-index|Foreign bank branches in Japan registry index]]（FSA 57-行登记面）的跨走廊对应版本。

## Why this matrix matters

「外资银行 X 离开了日本」是日本金融体系评论中最被夸大的说法之一。FSA 外资银行分行登记显示截至 2026-05-13 有 57  家活跃分行——因此持牌存在的一侧在原始数量上并未萎缩。萎缩的是面向消费者的品牌面（Citibank 零售 → 2014, HSBC Premier → 2012）以及某些外资 IB 在 2000s 维持的高接触财富 / 国内 IPO 特许经营。

具体而言，本矩阵呈现：

- **哪些收缩是品牌退出**（Citibank 零售 2014-09；[[foreign-financial-institutions/hsbc-japan|HSBC Premier]] 2012-03；Barclays 股票 2016-17 亚洲收缩）**而哪些是完整的法人退出**（极少——大多数银行保留了一个分行）；
- **哪些收缩由全球母公司危机触发**（Citi TARP 2009；CS → UBS 2023-06；Wells Fargo 资产上限时代）；
- **哪些收缩由日本特有事件触发**（CS Mizuho 相关纠纷；AML 执法；特定本地问题）；
- **哪些存在实为向更窄批发垂直领域的再聚焦**（Deutsche 2019-07 全球 IB 裁减级联至日本市场与 IB；Goldman / Morgan Stanley 联盟 / JV 配置；BNP cardif 保险与 BNP IB 分离）；
- **哪些亚洲银行在同期扩张进入日本**（[[foreign-financial-institutions/dbs-bank-japan|DBS]]、[[foreign-financial-institutions/ocbc-bank-japan|OCBC]]、[[foreign-financial-institutions/uob-bank-japan|UOB]] 均持有活跃分行；中国五大行与韩国银行也扩张了）；
- **哪些业务变得更集中**（托管 → [[foreign-financial-institutions/bny-mellon-japan|BNY Mellon]] + [[foreign-financial-institutions/state-street-japan|State Street]] + [[foreign-financial-institutions/jpmorgan-japan|JPM]]；ICSD 网关 → Euroclear + Clearstream Japan）；参见 [[banking/japan-trust-bank-vs-global-custodian-comparison-matrix|trust-bank vs global-custodian matrix]]。

缺少时间线视图，收缩叙事要么过于戏剧化（「外资银行离开了日本」），要么过于轻描淡写（「什么都没变」）。两者都错失了实际发生的结构性再聚焦。

| **2019-07 Deutsche IB リストラ** | Deutsche | Cut equity IB / Japan equity desk significantly |
| **2016 Wells Fargo 偽口座 事件 → 米連邦準備制度 資産上限 (2018-2024+)** | Wells | Constrained parent balance sheet → Japan presence stays niche |
| **Barclays 2016-17 Asia equity 縮小** | Barclays | Cut Asia equity but kept fixed-income / FX in Japan |
| **GPIF 海外運用比率上昇 (政策決定 2014-)** | BNY Mellon, State Street, JPM, Citi (custody) | Structurally expanded foreign-custodian Japan business — opposite direction from "retreat" |

### Citigroup → narrowest US retreat case ([[foreign-financial-institutions/citigroup-japan|Citigroup Japan]])

- **1902** National City 东京支店 开设 (邦银史上 最初期 外资 presence).
- **1980s-2000s** Citibank 银行 零售 峰值 (24h ATM / 外汇存款 / 富裕层零售).
- **2007-08** 峰值 Japan presence: 日兴 Cordial 证券 (2007 收购) + Citibank 银行 零售 + Citi 法人银行 + Citi GTS.
- **2008-11** TARP $25b 注入 → 实质国有化局面.
- **2009-10** 日兴 → SMFG (现 [[securities-firms/smbc-nikko-securities|SMBC 日興]]). Citi 日本 IB 的最大品牌资产出售.
- **2014-09 → 2015-11** Citibank 银行 零售 → [[trust-banks/smbc-trust-bank|SMBC 信託銀行]] (PRESTIA 品牌延续).
- **Current (2024-2026)** 法人银行 (NA 东京支店) + IB / 市场 (CGMJ) + GTS + custody. 零售完全撤退.

### HSBC → retail exit template ([[foreign-financial-institutions/hsbc-japan|HSBC Japan]])

- **1866** 横滨支店 开设 (HSBC 创业次年, 邦银以前最古级外资银行).
- **1990s-2000s** HSBC Premier 零售展开 (在日外国人驻在员 + 富裕层).
- **2007-11** 峰值 Premier 据点 (东京 + 大阪 + 名古屋 + 福冈).
- **2012-03** HSBC Premier 零售 日本撤退 → 顾客移管至 SBI 信托银等. 转向 法人 + 机构投资者服务专属.
- **2015** 在 Asia-Pacific-Pivot 战略下重新定义 Japan 角色.
- **Current (2024-2026)** 东京支店 + 证券公司 + AM + 信托 (缩小). 无零售.

### Credit Suisse → UBS absorption case ([[foreign-financial-institutions/ubs-japan|UBS Japan]])

- **2007-08** CS Japan IB 峰值.
- **2023-03-19** CS 经营危机: AT1 ~160 億 CHF 减记归零, 瑞士政府 / SNB / FINMA 主导 紧急救济决定.
- **2023-06-12** UBS 完成紧急收购 (取得对价 ~30 億 CHF; CS 1 股: UBS 1/22.48 股).
- **2024** 旧 CS 日本据点 (CS 证券) 与 UBS 证券 的整合工作持续.
- **Current (2024-2026)** UBS 整合: 旧 CS 法人 消灭. UBS 日本 = UBS 证券 + UBS 银行东京支店 + UBS AM + [[trust-banks/sumitomo-mitsui-trust|SMTB]] JV (UBS SuMi TRUST WM, UBS 51% / SMTB 49%).

### Deutsche Bank → 2019 IB cuts cascade case ([[foreign-financial-institutions/deutsche-japan|Deutsche Japan]])

- **1972** 东京支店 开设.
- **2007** 峰值 Japan IB / 市场 presence.
- **2016-2019** Deutsche 母公司经营困难 (股价底值 / 配当停止 / 罚款).
- **2019-07** 全球 IB 大规模重组 (全球 18,000 人裁减). 日本 equity IB / 股票营业大幅缩小.
- **Current (2024-2026)** 东京支店 + 德意志证券 (债券 / FX 集中) + DWS Japan (PB).

### Asia banks expanding into Japan (opposite direction)

- [[foreign-financial-institutions/dbs-bank-japan|DBS]] 东京支店 (1977 开设) — Singapore wholesale corridor.
- [[foreign-financial-institutions/ocbc-bank-japan|OCBC]] 东京支店 — Singapore corridor.
- [[foreign-financial-institutions/uob-bank-japan|UOB]] 东京支店 — Singapore corridor.
- [[foreign-financial-institutions/standard-chartered-japan|StanChart]] 东京支店 — Asia EM corridor.
- China-five (Bank of China / ICBC / CCB / ABC / BoComm) — RMB / 日中贸易 settlement.

这些亚洲 / 亚洲-EM 走廊银行在同一 2000-2026 窗口内是净扩张者，平衡了美国 / 欧盟零售收缩的叙事。

## How to read this matrix

1. **以峰值年份作为基线。** 大多数美国 / 欧盟银行在 2007-08 达到峰值；比较的是「当时日本足迹是什么样」与「现在是什么样」。
2. **区分收缩与再聚焦。** 本矩阵中几乎每家银行都保留了其 FSA 持牌的日本存在。改变的是业务组合与品牌面。
3. **关注 Citi / HSBC 零售模板。** 两家银行都为「撤退零售 / 保留批发」开了先例。这一模板是被引用最多的「外资银行离开」参照，但实际描述的是品牌面变化，而非法人退出。
4. **追踪 JV / 战略合作叠加。** [[securities-firms/morgan-stanley-japan|MS]] / MUFG 经由 MUMSS + MSMS；[[foreign-financial-institutions/ubs-japan|UBS]] / [[trust-banks/sumitomo-mitsui-trust|SMTB]] 经由 UBS SuMi TRUST WM。这些 JV 嵌入很深——退出在结构上很难。
5. **与托管矩阵交叉链接。** [[foreign-financial-institutions/bny-mellon-japan|BNY Mellon]] / [[foreign-financial-institutions/state-street-japan|State Street]] / [[foreign-financial-institutions/jpmorgan-japan|JPM]] / [[foreign-financial-institutions/citigroup-japan|Citi]] 托管是一个动态非常不同的独立轴——参见 [[banking/japan-trust-bank-vs-global-custodian-comparison-matrix|trust-bank vs global-custodian matrix]]。GPIF海外运用扩大 是这四家的顺风。
6. **亚洲走廊银行是净扩张者。** [[foreign-financial-institutions/dbs-bank-japan|DBS]] / [[foreign-financial-institutions/ocbc-bank-japan|OCBC]] / [[foreign-financial-institutions/uob-bank-japan|UOB]] + 中国五大行 + 韩国五行 不符合「收缩」叙事。

## Retreat-claim audit checklist

在引用任何「银行 X 从日本收缩」的说法之前，请逐项核查：

1. **FSA 牌照是否仍活跃？** 在 [[foreign-financial-institutions/foreign-bank-branches-japan-index|Foreign bank branches in Japan registry index]] 查询当前分行状态。
2. **该变化是品牌退出还是牌照退出？** Citibank 零售（2014-09）是品牌退出；Citigroup 牌照集团保留。HSBC Premier 同理（2012-03）。
3. **该变化是日本特有还是全球性的？** Deutsche 的 2019-07 IB 裁减是全球性的，级联至日本；CS 2023-06 是全球母公司的消失。
4. **是否有日本伙伴接手了该业务？** Citi 零售 → SMBC Trust / PRESTIA；CS Japan → UBS Japan。业务并未消失，而是转移了。
5. **业务层面的影响是什么？** 裁减股票 IB ≠ 退出日本，只要固定收益 / FX / 法人 / 托管全部延续。
6. **母公司的 G-SIB 地位是否在变化？** 母公司退出 G-SIB 地位会改变日本资本配置；稳定的 G-SIB 通常会在某种规模上维持日本存在。

## Boundary cases and caveats

- **「収缩」是一个被过度承载的词。** 它可以指：品牌退出（Citibank 零售、HSBC Premier）、业务线退出（Deutsche 股票 IB）、JV / 联盟重组（MS-MUFG JV 随时间调整）、完整法人退出（极少），或仅仅是人头精简。每一种都在性质上不同。请使用上方的触发因素分类法来厘清所指的是哪一种含义。
- **外资银行分行仍有 57 家**（[[foreign-financial-institutions/foreign-bank-branches-japan-index|2026-05-13 FSA registry]]）。牌照数量叙事不支持「外资银行离开」的故事；只有面向消费者的品牌面才讲述那个故事。
- **托管 / 资产服务的集中 ≠ 收缩。** [[foreign-financial-institutions/bny-mellon-japan|BNY Mellon]] / [[foreign-financial-institutions/state-street-japan|State Street]] / [[foreign-financial-institutions/jpmorgan-japan|JPM]] / [[foreign-financial-institutions/citigroup-japan|Citi]] 的 4-集群随 [[trust-banks/master-trust-bank|MTBJ]] / [[trust-banks/custody-bank|CBJ]] 国内侧集中而结构性扩大。完整托管图景参见 [[banking/japan-trust-bank-vs-global-custodian-comparison-matrix|trust-bank vs global-custodian matrix]]。
- **MS-MUFG JV 不是收缩。** 它是一种长期战略运营模式，Morgan Stanley 使用 MUFG 的日本零售 / DCM 渠道，MUFG 使用 MS 的全球 IB 能力。2  家独立牌照实体（MUMSS = MUFG 60% / MS 40%；MSMS = MUFG 49% / MS 51%）反映的是刻意的业务分割，而非收尾。
- **Wells Fargo Japan 在结构上是 niche，而非收缩。** 它从未为日本零售而建。「小规模日本存在」反映的是战略定位，而非近期退出。
- **2023 CS 事件是此窗口内唯一真正的大规模母公司消失**，涉及一家主要的日本 IB。UBS 吸收了 CS Japan；CS 作为品牌已从日本 IB 消失。
- **2024-25 Wells Fargo 资产上限解锁**是一个前瞻性变量，可能支持 Wells Japan 的适度扩张，但本矩阵不预测具体结果。
- **AML / governance 执法是全行业的**，而非银行特有。它提高了运营 FSA 持牌外资银行分行的成本，但并未单独指向某个特定退出。
- **引自母公司年报的数字**（G-SIB bucket 分配、母公司总资产、母公司 AUC/A）是最近 FSB / 母公司 10-K 披露周期的公共领域汇总锚点；它们逐年变动。
- **JV / 财富结构会变动。** [[foreign-financial-institutions/ubs-japan|UBS]] / [[trust-banks/sumitomo-mitsui-trust|SMTB]] JV 股权比例与运营公司配置可随全球母公司战略而变化。引用时请与最新的 JV 侧披露交叉核对。
- **本矩阵排除**保险分行外资公司（BNP 旗下的 Cardif 生 / 损保；AIG / Allianz / Aflac / Cardif / MetLife / Manulife），因为保险的收缩 / 再聚焦模式实质性不同——那一侧参见 [[insurance/INDEX|insurance domain]]。
